＜今日のピークパフォーマンス方程式＞

  【　真の変容は、感情変化から　】

───────────────────────────────────



■メルマガを毎日、書くようになってから、
　9年3ヶ月が経過しました。


　最初のうちは

　「書き続けるのに必死」

　だったのが、いつしか

　「苦もなく文章が書ける」

　ようになりました。



■「苦もなく文章が書ける」ようになった今は、
　一つハードルを上げて、

　「読んでくださる方の感情を、
　　どれだけ動かすことができるだろう」

　といったところに焦点を合わせています。


　いつも書いていることですが、

　「人は心が動き、そのあとに身体が動く存在」

　だからです。



■人は心が震える（感動する）ことによって


　「もっと勉強しよう」

　「もっとご縁を広げよう」

　「もっと人に尽くそう」

　「もっと自分を躾けよう」


　といった行動につながります。



■いくら、


　勉強が大事だよな、

　ご縁が大切なんだよな、

　自分を躾けることが必要だよね、


　などと分かっていても、心底、

　「かくありたい」

　と望まない限り、

　一時的に盛り上がることはあっても、
　継続した行動にはつながりません。



■そう考えると、

　人さまに良い変化をもたらすお手伝いを
　しようと思ったなら、

　人さまの行動にではなく、感情にフォーカスを
　当てなければなりません。

　感情に訴求することによって、初めて行動に
　つなげて頂くことができるようになるのです。



■変わってもらいたい、と思ったとき、

　「ええい、変われ！変われ！変われーー！」

　といくら吠えたり、叫んだり、脅したり、
　奇声を発したりしても、

　相手は心から変わろうと思うことはないだろうし、
　
　従って、本当の意味で変化することは
　ありません。



■真の変容は、感情の変化から起きるのです。

　だからこそ、

　「人さまの変化のお手伝い」

　を標榜し、喜びとし、かつ生業ともしている私は、

　「人の感情はどういったときに動くのか」
　
　について関心をもたずにはいられないのです。


　長くなりそうです。明日に続けます。
　　

　今日も人生とビジネスを楽しみましょう！
