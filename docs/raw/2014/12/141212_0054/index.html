▼今日は個別コンサル３件（＆高井伸夫先生事務所）

■こんにちは、鮒谷です。


  今日は、

　弁護士の高井伸夫先生の事務所にて、
　
　来年度、主催されるセミナー登壇させて頂くための、
　打ち合わせ、その他。


　個別コンサルは
　午後３時から、夜９時まで、３件、

　でした。

　

■話は変わりまして、

　昨日、コンサルさせて頂いた、
　山本憲明税理士事務所 山本憲明さんより、

　下記のようなご感想を頂戴いたしました。


　（ここから）
　----------------------------------------------------------------


　昨日のコンサルティングには、失礼なことに
　何も用意して行きませんでした。

　本来、こちらの現状をお話して、悩みを解決
　してもらうのが趣旨かもしれません。


　しかし昨日は、鮒谷さんの生き方とか過ごし方を
　吸収できれば、という思いだけを携えて臨みました。

　結果的にはそれが大正解。


　鮒谷さんの生き方、考え方、過ごし方を多角度から
　吸収することができ、脳内のある部分が書き換わる
　ような感触さえもありました。


　なかでも印象に残ったのは、

　「効率化を極めて、非効率を楽しむ」

　という言葉です。


　鮒谷さんの仕事の進め方は、効率化の極みとも
　言えるやり方で、

　これなら「忙しい」といった状態から脱して、
　時間を生み出せるなあ、と実感。

　
　さらに、その効率化で生み出された時間を、
　読書などの非効率なものに充てることで、

　今の鮒谷さんが形作られているということが
　良くわかりました。


　私も職業作家を目指す身、

　鮒谷さんの生き方を見習い、
　効率化を進めて非効率の時間を生み出し、

　存分に楽しんでいこうと決意しました。


　これからの人生が楽しみになりました。
　ありがとうございました。


　　　　　　　　　　（山本憲明税理士事務所 山本 憲明さま）

　「発想の転換」

　「イノベーションの種」

　をもたらすこともあるようです。


　今回の話が山本さんにとって、
　多少なりともお役に立てたようでしたら、
　大変嬉しく思います。

　これからも引き続き、よろしくお願いいたします。

 【 一人で食いたきゃ、一人で食うな 】

───────────────────────────────────



■今日のお昼、

　「一人で食えるもん！」

　と題して、

　「大阪での放談会＆個別コンサル」

　のご案内をいたしました。

　

■早速、お申込みくださった皆さま、
　ありがとうございます。


　少数精鋭の皆さまに、

　最新の知見を、時間の許す限り、
　大放談したいと思っております。

　
　ご期待ください！



■ところで、今回の放談会では、

　「一人で食えるもん！」

　というタイトルをつけ、


　「一人で食っていける実力を身につけよう」

　そんなテーマで話をしますと書きましたが、
　実のところ、

　「一人で食っていける」

　という言葉は誤解が生じやすい。

　
　したがって少々、補足が必要です。



■正確には


　「一人で食えるようになりたければ、
　　一人で食えるようになろうと思わない」


　というのが正解です。


　意味がわからない？

　それでは、以下のブログをお読み下さい。


　＜無形資産による垂直立ち上げの凄さを体験した話＞

　http://blog.livedoor.jp/keisukeiwano/archives/1015318412.html



■ちなみにこの方は、

　「３日間セミナー → 年間プログラム → 継続個別コンサル」
　
　とコンボでお申込みくださった方（笑）

　
　バックナンバーを通して
　お読み頂ければ分かる通り、
　
　とても頭がいい、文章うまい。

　そしてまだ、28才（ですよね？）。


　将来有望な若者！？です。


　
■ここにも記されているように、
　応援してくれる人があれば、何を行うにしても

　「垂直立ち上げ」

　が可能になります。

　
　応援してくれる人の力が上昇気流のように作用し、
　一気に高い高度まで持っていってもらえるのです。



■しかしながら、言うまでもなく、

　「この力は願えば、祈れば、与えられる」

　といった類のものではありません。
　

　応援してもらう前に、
　応援を惜しまないからこそ、

　一朝、事が起きたとき、
　周りのみんなから応援してもらえるのです。



■なのに、なのに、
　
　応援してもらうことだけ考えて、
　応援することについては無頓着な人の、
　
　如何に多いことでしょう。


　まず渡す、さらに渡す、さらにさらに渡す、
　そんな繰り返しの中から、

　「もらう」

　ことも、ときどき、ある。


　
■そんなことを繰り返しているうちに

　「多くの人との、深いつながり」

　が生まれ、


　そのつながりによって

　「一人で食えるもん！」
　
　と胸を張って言える、
　そんな世界に出られるようになるのです。



■一人で食えるようになりたければ、

　一人で（食事を）食べていてはいけません。


　一人で食えるようになりたければ、

　皆と食べ（つながり）、
　皆に貢献する必要があるのです。


　
■ということで、

　本日は、この難儀で先行き不透明な世界を
　
　「心の平安、平穏、安定、余裕、ゆとり」

　を持って渡っていくための、
　ほんの少しのヒントをお伝えしましたが、


　こんな内容を、

　「これでもか！これでもか！これでもか！」

　と叩き込み続けるのが、


　先ほどご案内した、

　＜超・少人数制 「一人で食えるもん！」大放談会（個別コンサル付き）＞

　の企画です。

　（タイトルは、
