6903号 「知っていた世界」が事後的に実現されるという世界

■昨日の年間プログラムにおいて
　行動を阻害する要因として

　「自分を信じられない」

　ことを挙げました。



■意識に乗る形でセルフトークが
　行われているかどうかはともかく

　「どうせ私なんか」
　「恐れ多い」
　「自分には無理」
　「そんなの出来るはずがない」

　という言葉が繰り返し
　頭の中を回っていたら、

　そこから強化される自己認識が
　ボトルネックとなり行動が妨げられる。



■それゆえ、これらの言葉に気づいたら
　一つ一つ、

　「その瞬間に、
　　セルフトークの上書き」

　を重ねる必要があるのです。



■具体的には

　「私だからこそできる」

　「私にはその資格がある」

　「それを実現して周りの人を
　　ビビらせられたら愉快だな」

　「私がやらなければ誰がやる」

　といった言葉を回すこと。



■ここでのポイントは

　「その言葉を信じているから
　　回せる」

　のではなく

　「回しているうちに
　　信じられる」

　ようになっていくところ。



■こうして

　【明るい未来を信じられる】

　ようになり、さらに続けていると、
　ついには

　【将来はどんどん良くなっていくと
　　『知っている』】

　というところまで

　【強度の思い込み、錯覚、勘違い】

　が生じます。



■ここまで自分の将来に対して
　確固たるイメージを持てれば

　【（成果が出るのを）長い時間、
　　待てるようになる】

　でしょう。


　いつもいう言葉でいうと

　【重心が低くなる】

　ということ。



■この世界に入れば、長い雌伏の
　時間にも不安は生じないし、

　右往左往することなく、
　どっしりと腰を据えて、

　積極的モラトリアム期間を過ごし、
　地力を蓄えていけるのです。



■むしろ、この期間が延びるほど
　ますますポテンシャルが高まり

　「あとはその潜在能力＆エネルギーを
　　開放してやるだけ」

　「それはいつ訪れるだろう」

　という期待や喜びや興奮が
　生まれることになるはずです。



■この状態に入ることなく、
　未来に向けた取り組みを行っても

　「自身の将来を信じられず、
　　時間が経過するほどに
　　焦りや不安や恐怖が増幅される」

　から、

　「短期的思考に逃避する」

　こととなります。



■これが重心の高さに繋がり、

　「あれこれ、つまみ食いして
　　結局なにも身につかず」

　それがために
　デフレスパイラルのごとくに

　「より一層、焦りや不安や恐怖が高じ、
　　すぐ手に入る成果を求める精神」

　につながり、それは容易に

　「一攫千金思考」
　「起死回生願望」

　へと転換されることとなります。



■こうした思考や願望から生まれた
　行動によって大きな損害を被ったり、

　ときに致命傷を食らってしまう人が
　後を絶たない、

　そんな悲劇的な事態が
　生じるわけですが、


　それもこれも

　【将来はどんどん良くなっていくと
　　『知っている』】

　という心の状態を
　作り出せていないから。



■誤解している人があるので
　何度でも強調しておきますが、

　すでに結果を出しているから

　【将来はどんどん良くなっていくと
　　『知っている』】

　という世界に生きられるのでは
　ありません。



■結果を出す前から

　【将来はどんどん良くなっていくと
　　『知っている』】

　という心の有り様を
　練り上げている人だからこそ、


　じっくりと、歩留まりの高い

　「（効率的ではなく）
　　効果的な取り組み」

　を継続して
　行い続けられるようになり、


　その結果、

　生きるにしたがって未来は、
　あらかじめ知っていたとおり、

　ほんとうに
　どんどん良くなっていくのです。



■要するに、

　行動するから
　未来が開けるのではなく、


　未だ経験していない
　将来であるにも関わらず

　「既に明るい未来を知っている」

　（という自己認識、確信）から
　重心の低い、継続行動ができるし、

　その結果、

　【知っていた世界が
　　事後的に実現される】

　こととなるのです。



■常識（現在→未来）をひっくり返し、

　「未来→現在」

　の時間の流れを生きることで
　行動が変わり、未来が変わります。


　この勘所を掴めるかどうかが

　【人生を大きく変化させられるか
　　否かの分水嶺】

　といって良いでしょう。



■以上、

　昨日の年間プログラムに
　ご参加下さった方へのフォローアップ、

　ならびに、本日、収録音源を
　お送りする皆さまに向けての

　「先出し、チラ見せコンテンツ」

　としてお届けいたしました。


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】

　■将来良くなると「知っている」人だけが
　　将来をほんとうに良くできる。

　■知っている＝信念、につながるから、

　　信念に従い、行動し、思っていた通りの
　　世界にほんとうに出られるようになる。

　■未だ到達していない世界にも関わらず
　　「知っている」から行動できるのであり、
　　行動するから分かってくるのではない。

　■現在と未来の時の流れを入れ替えることに
　　よって、人生が激変するのである。
