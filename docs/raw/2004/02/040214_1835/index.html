■いろんな企画書や作成依頼物、コンサルに入っているクライアント様の
　セールスレターやWebページのチェック、企画書、これから行うプロジェクト
　で必要になってくる文章の執筆や校正、などなど、朝から晩まで、
　アポも入れずにPCに向かっています。

　昨日も朝６時から夜９時まで、食事をしているとき以外はひたすら
　キーボードを叩いていました。あっという間に昼になり、夕方になり、
　夜になった、という感じ。

　作成した文章は膨大な分量です。


■結構、文章を書くのは早いほうですが、特にクライアントさまからの
　依頼文などはやはり相当の時間をかけて赤入れをしますから、かなり
　時間を使います。

　その後、食事にいって、また新しく開拓した夜１１時までやっている
　喫茶店で読書。（私はタバコを吸わないので、禁煙スペースがきっちり
　取られているところでないと思考が働かなくなるのです。そんな
　条件を満たした満足度の高い喫茶店でした）
　
　一章、目を通しては、ぼけ～と考えるでもなく考える。
　また一章、目を通しては、同じくほけ～と考えるでもなく考える。

　ちょっと頭を休ませ、クールダウンしないと脳天がオーバーヒートして
　しまいそうな感覚を覚えたので。

　そんなときは、こんな本の読み方をしています。


■その間に広告をご出稿して下されている方からお電話があり、１時間
　ほど、原稿のことやその他のことについてお話しさせて頂きました。
　

■帰宅後、早く休もうと１２時頃、床につくもいろいろ考えていると
　やっぱり興奮してきて、そろそろと布団から抜け出し、結局また朝５時
　まで書き物をやっていました。

　ほぼ２４時間運転でした。

　【　　読者さまからのご感想。　　】

───────────────────────────────────


■本日、貴重なご意見を頂きました。

　重要なメッセージだと思いましたので、今日、書き始めていた
　「平成・進化論。」原稿を一旦全て削除して、本日は、この頂いた
　メールについてのお返事をさせて頂きたいと思います。


（頂いたメールここから）
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

鮒谷様へ

いつも素晴らしいメルマガありがとうございます。
おかげさまで、様々な情報や良書の情報、仕事に対する姿勢など
こんな有益なメルマガを無料でいただいていいものかと思うほど、
素晴らしく思います。

さて今回メールしましたのは、平成進化論の素晴らしさを認めつつも
鮒谷様に少し反抗してみようかと、大それたことを考えてメールさせて
いただきました。

それは、鮒谷様がおっしゃる「情報格差」というものです。
たしかにＩＴ業界をはじめとする社会構造の変革はすさまじく
ものすごいスピードで変わっているようです。

しかし、その中でも根元になる木の幹の部分は、常に同じような気が
します。
いくら枝葉の変化が、ものすごいスピードで成長、変化しても、
木の幹の部分が、変わることはありません。もし変わるとすれば
その木が老いて朽ち果てるときです。

数百年、数千年前から、伝わる秘伝や、古文書、生活の知恵的なものは
木の幹を描かれたものであり、それを抑えることができれば、
枝葉の部分は自然にわかってくるような気がします。

勿論このような思想的なことに、これが正解だなんて
ものは無いと思いますので、これが正しいなんて大それたことを
言うつもりもないですが、鮒谷様のいつものご活躍の中で、少しでも
ご参考になれればと思いメールさせていただきました。

今後も素晴らしいメルマガがんばってください。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
（ここまで）


■メールどうもありがとうございました。
　こういったご感想は本当に有難く受け止めさせていただかねばと
　思っております。


■本当にこの方のおっしゃるとおりだと思います。

　いつも、毎日気が付いたことで、皆様のご参考になればと思う
　ことを配信させていただいているのですが、ともすると自覚のない
　ところでノウハウテクニック論に走っていたのかもしれません。
　こちらは、どんどん新しいことも登場して、ネタがつきることも
　ありませんし。


■自分自身のことはどれだけ客観視しようと思っても、客観視する
　ことは出来ません。だからともするとメルマガ配信も自己満足に
　陥ってしまうこともなきにしもあらず。


■たとえば、主観を離れて第三者的なポジションをとって今の自分を
　評価してみようと思っても、その第三者的なポジションにいる自分
　もやはり、自分の主観の息がかかっていますから、厳密な意味での
　客観評価は不可能です。

　さらに、その第三者的なポジションを客観的に見る第四者的？な
　ところから見てみてもやはり、その第四者も主観から逃れること
　はできません。

　主観←（第三者的）客観←（第四者的）客観←（第五者的）客観←　…

　といった無限連鎖にならざるを得ません。

　この図でいうところの「客観」は実はどこまでいっても「主観」
　なんですよね。


■そんな中で、ぽーんと外部から投げかけてくださる疑問や質問、
　ご感想は非常に有難いものです。

　出来るだけ、配信する前に自分の考えを中立で眺めてみて、簡単に
　チェックはしているのですが、やはり外から頂く感想は驚くと
　ともに勉強になったり、参考にさせて頂くことが多いです。

　現在、なかなかお返事をお返しする時間がとれませんので、
　積極的にご感想を募っているわけではないのですが、なおさら、
　そんな中でメールアドレスを探し当てて、ご返信してくださる方
　に感謝しています。


■さて、本題に戻りまして、根本と枝葉の問題。

　自分の中でこのことはバランスをとって配信するよう、意識して
　いたつもりなのですが、やはり主観から離れることが出来ず、
　伝わっていなかったようですね。

　平野先生言われるとおり、『伝わらなければ意味がない』の
　世界です。


■メルマガを配信する中でいつも考えていることは、やはり読んで
　いただいてこそのメルマガですので、どうしたら読んでいただき、
　また読者さまに結果を出していただけるか、ということです。

　そうするとどうしても、テクニック、すぐに結果の出る方法、
　書籍のご案内、などを優先してしまっていたのかもしれません。


■ただ、いつも考えていることは、悪いことをして良い結果は返って
　こない。種まきしないのに、楽して稼げる・儲かるなんてことも
　存在しない。たとえ短期でそういうことがあっても、必ず長期で
　見るとバランスされてくるものである。


■よりよい結果、運命を欲しいのであれば、まじめにきちんと、
　世のため人のため、の行いが必要であるということ。
　短期的には見えづらくても、長期の視点で見ると必ずこの原理原則
　が生きている。

　行間に、そういう根本の想いを入れていたつもりでした。
　これは枝葉ではなく、本質だと思っています。
　
　
■そのままの文章にすると「そんなの当たり前じゃん」の一言で
　片付けられそうなので、味付けの仕方を変えて、そんなメッセージも
　お伝えしていたつもりなのです。

　ほんの、一例ですが。
　
　でも、伝わっていなければ意味はありませんもんね。
　ちょっと反省です。


■このあたりのバランス、非常に難しいです。
　全ての読者さまを満足させることは実質的に不可能です。

　一方で本質論を求める方があり、かたやもう一方で、すぐに結果の
　でる方法論を求める方もあります。

　そして私はどちらも大事だと思っています。
　これからもうまくバランスをとりながら、自分なりに精一杯の情報を
　配信させていただければと思います。

　ただ、決して、本質、根幹、根本の部分を疎かにしているわけ
　ではなかったことをご理解いただければ幸いです。


■こういう、ご丁寧なご意見、本当に感謝しておりますし、
　今後もいろいろお聞かせいただければと思っております。

　お答えになったかどうか分かりませんが。。。
