【（鮒谷よりご案内）始め、続け、身につけ、変われる人となるために】

2013/05/28 17:14 
■いつもありがとうございます。鮒谷です。

　いつもメールが長くなってしまうので（汗）、
　今日は短めのご案内。


　表記のタイトルの通り、

　自分（の人生やビジネス）を良い方向に変えるために、


　「自身を変化、変革させるための良い習慣を、
　　始め、続け、身につけられる人になりたい」


　と思われる方にのみ、参加頂きたいセミナーを
　ご案内させていただきます。



■メルマガ配信を通して、
　毎日、たくさんのご感想を頂戴しますが、

　少なくない（というよりも、とても多くの）方が
　

　「何をやればいいか分かっているにもかかわらず、
　　始められず、続けられず、身につけられず、

　　したがって変われない」


　ことに悩まれているのが、
　ひしひしと伝わってきます。



■そんな

　「変わりたいのに変われない」

　方に向けて、これまで過去12回にわたって


　【 変わりたいのに変われない「３つの理由」セミナー 】

　
　を開催してきました。


　これは

　「３日間セミナーのダイジェスト版」

　のような位置づけで開催しているものです。



■スケジュールに都合がつくようでしたら、
　ぜひご参加いただければ、

　と思い、今回は珍しく早目にご案内いたしますね。

　（といっても、数日後なのですが 汗）


　※以前に３日間セミナーに参加下さったことの
　　ある方も復習としてご活用頂けると思います。
　　　
　　前回も数名の方がお越しくださいました。
　　


■今回も積極的にご案内するつもりはありませんので
　セミナーの内容をくどくど、説明することはいたしません。

　以前、参加下さった方のご感想をご覧頂き、
　ご判断くださいませ。

　　そちらをクリックしてご覧ください）
