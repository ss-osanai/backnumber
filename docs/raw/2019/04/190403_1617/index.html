【4月6日（土）開催！】そもそもなぜ、あなたは変わらなければならないのか放談会

2019/04/03 16:17 
■今日は（も！？）、急遽のご案内。


　いつもながら急なご案内で恐縮ですが、
　３日後に以下の放談会を開催いたします。

　
　あまりにも日程が近すぎて、お客さまが
　お越し下さるかどうか、甚だ心配ですが、
　
　たとえお一人のご参加でも、
　全力でお話しいたします。



■４月６日（今週土曜日）、
　午後１時半から４時まで、

　２時間半、ノンストップ、一本勝負で、
　
　「そもそも、なぜ、あなたは
　　変わらなければならないのか」

　について、放談いたします。

　（たぶん、音源の販売はいたしません）



■毎日、

　「変わりたい、変わらなきゃ」

　と思っていながらも、


　現実には何をするわけでもなく、
　時間の経過に流されているだけ、

　という人は少なくないように思われます。



■来るべき

　「令和の時代」

　に向けて、
　
　【そもそも、なぜ、あなたは
　　変わらなければならないのか】

　について、


　（もし、そのヒントを求められている方が
　　いらっしゃるのであるならば）

　ガツンと一発、活を入れるための
　強烈な放談を、と思い、
　
　急遽、開催することとした次第。


　
■放談する内容についてはタイトルの通り。


　この問いに明確に答えられない人が、
　具体的な行動を始められないのは、

　当然のことと言えるでしょう。
　

　そもそも

　「変わるための理由」

　を持っていないのですから、、
