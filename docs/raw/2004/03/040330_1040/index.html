■「平成・進化論。」読者の方には、下の本はご関心があれば、買われたら
　どうかな？という感じですが、上の本（カリスマ体育教師の常勝教育）
　は絶対に買って（買うだけでなく、読んで）頂きたい本です。


■いずれの本も、人間としてやるべきことをきっちりと行わないと
　結果を出すことはできないよ、と当たり前だけれども今の時代に
　忘れ去られていることがたくさん書かれています。

　

■ほんとに特に、「カリスマ体育教師の常勝教育」は、自分自身の
　ためにも読んで頂きたいし、また、これからの日本を担っていく、
　中高生を持つ、親御さんにも読んで欲しいと思うし、けっこう多読家だと
　思っている私も四半期に一冊くらいしか、出会えないくらいの名著だと
　思いました。


■総集編を書きながら、改めて反省するところ、多々あって、
　恥ずかしい限りです。はい。
