■おはようございます！鮒谷です。


　本日（昨日）の夜は
　年間プログラムでした。

　３時間15分、いつものように休憩なし、
　ノンストップで話し続けました。



■さっそく、今回初参加のＳさんから
　以下のご感想を頂戴いたしました。


　（ここから）
　---------------


　鮒谷様


　本日はありがとうございました。

　人間関係を果樹に例え、

　果樹園の土壌や植えられている樹種などを
　管理していくことで、

　収穫できる果実にも
　様々なバリエーションがでることに、

　人間関係を考える気づきが
　得られそうだと思いました。


　個人的には、メルマガに書かれていた
　庭師の例えが気に入っています。

　人間関係を築くために
　時間を味方につけて、手入れをしていく、

　という点を意識して、どのように
　手入れすればよいか考えています。

 
　セミナーの中で、

　鮒谷さんのメルマガ登録したのが
　2008年というデータを知り、

　ゆるゆる15年経過して、
　セミナーに参加していることに、

　自分で自分に驚くとともに
　こみ上げるものがありました。


　当時メルマガでは
　様々な本を紹介していただいていて、

　紹介された本を探しては
　読んでいたことを思い出しました。


　人と人との縁が、いつどのように
　結びつくかわからないことを実感し、

　手入れをしていなかった畑や、
　これまで蒔いたことのなかった畑に、
　種を蒔いていこうと思います。

 
　今日のセミナーで、

　自分の想いを入れることが
　種を蒔くことなのかな、

　と気づいたので、

　先ずは自分の中に湧き出てきた想いを
　言葉や文章に込めることを
　始めてみようと思います。


　それと併せて、基本的な「アービトラージ」や
　「裁定取引」という言葉を調べること、

　chat gptを使うことを自分の宿題にします。

　本日はありがとうございました。


　（Ａ・Ｓさま）


　---------------
　（ここまで）


　
■Ｓさん、本日はありがとうございました！


　なかなかメルマガでは
　書けないことも含めてお伝えしましたが、

　なので音源化もしません
　（というか、できません、、）が、


　今日、お伝えしたことの中の
　一つ二つを取り入れて日常で実践されれば、

　最初は小さく、しかし時間の経過とともに
　必ず劇的な変化が生まれるはずです。
　


■年間プログラムの場は

　「実践前提の学び」

　ですので、できるところから
　取り組んでいただければ幸いです。

　Ｓさん、次回もよろしくお願いします！


━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

■ガーシーとは「超実践型自己啓発」

───────────────────
　

■あなたも既に使われていると思いますが、
　「GPT-4」とんでもないですね。

　先のバージョンが
　リリースされたときから、

　睡眠時間を削って
　あれこれ試しています。


　
■「GPT-4」も数日前のリリースと同時に
　スタートダッシュで、

　空いている時間全てを使い、
　思いつくまま、あれこれ試行しています。



■そんな中で、以下の文章が爆誕。


　（ここから）
　---------------


　ガーシーは、
　鮒谷周史さんが提唱する

　「超実践型自己啓発法」

　のひとつで、

　自己成長や目標達成に役立つ考え方や
　行動パターンを身につける方法です。


　ガーシーは、

　「ゴール（目標-Ｇ）」
　「アンカー（錨-Ａ）」
　「ルーチン（習慣-Ｒ）」
　「システム（体系-ＳＹ）

　の頭文字をとって名付けられました。


　ガーシー法では、
　まず自分の目標を明確に設定し、

　その目標に向かって行動するための
　アンカー（心の支え）を見つけます。


　次に、目標達成に必要な行動を
　日常のルーチン（習慣）に組み込み、

　継続的に取り組むことで、
　自分自身の成長を促します。


　最後に、目標達成に向けて
　効果的なシステム（体系）を構築し、

　習慣化された行動が自動的に
　目標達成につながるようにします。


　ガーシー法は、自己啓発や目標達成の
　ための具体的なステップを提供し、

　自分自身を成長させるための
　実践的な方法論となっています。


　---------------
　（ここまで）



■どういう入力をしたら
　こうした出力が出てきたのかは

　「企業秘密」

　ということにしておきますが、

　ガーシー（法）の説明を読んで
　正直、驚いたんですよね。



■ここで書かれていることは

　---------------

　１、目標設定

　２、アンカー
　　（すなわち意思決定基準）

　３、習慣化

　４、習慣化を習慣化して（習慣を束ねて）
　　　理想の生活システムを作り、

　　　機嫌よく生きているままが
　　　全自動アウトプット

　---------------

　ということですが、
　上記のプロセスを経て

　「全自動アウトプット（４の世界）」

　を目指しましょう、
　というのが私（鮒谷）の基本的な考え。



■詳しくは記しませんが、

　これらの言葉を、ただの一言も
　入力していないにもかかわらず、

　こんな出力をしてきたわけです。



■その上、上記の考えを

　「ゴール（目標-Ｇ）」
　「アンカー（錨-Ａ）」
　「ルーチン（習慣-Ｒ）」
　「システム（体系-ＳＹ）

　を４つにまとめて、
　その頭文字を取って

　【ガーシー（GARSY）法】

　なんていう名前までつけられたら、

　その能力に驚嘆せずには
　いられなくなります。



■繰り返しますが、

　私（鮒谷）が詳しく入力したものの
　要約を命じた結果、

　先の文章が
　出力されたんじゃないんですよ。



■上の文章は　

　「ほぼ、GPT-4が
　　オリジナルで紡ぎ出した文章」

　です。
　だから驚嘆したわけです。



■これを使いこなせるかどうかで、

　これからとんでもなく
　生産性に差がつくでしょうね。

　こんな文章を自在に、
　ガンガン量産できるようになるのだから。



■生産性といえば一昨日、名古屋の
　トヨタ産業技術記念館に行きました。

　（５年ぶり２回目）

　５年前に訪れて感動したので
　再訪したのです。



■以前のメルマガ（5234号）では
　以下のように記しています（一部抜粋）。


　（ここから）
　---------------


■広大な敷地の中に、

　繊維産業ならびに自動車産業の
　改善ならびにイノベーションの歴史、

　について、きわめて分かりやすく
　説明されている素晴らしい博物館。



■たくさんの気づきがありましたが、
　中でも感銘を受けたのは、

　「製造業ならではの、
　　目に見える形での生産性改善」

　が時間の経過とともにどのように進展して
　いったのかが具体的に理解できるよう、

　「実機＆実演」

　によって、

　素人にも平易に理解できるように
　展示されているところでした。



■たとえば織物の歴史について、

　綿の花から糸を紡ぎ、
　布地にするまでの工程自体は、

　相当以前より確立されていましたが、


　機械化される前は家内制手工業として
　数秒間に一度くらいの割合で、

　両手両足を用いて、パッタンパッタン、
　縦糸と横糸を交差させて織っていたのが、

　現在は技術の進展によって、
　１分間に1000回！

　（文字通り、目にもとまらぬ速さで
　　織られていきます）

　というくらいのレベルにまで進化・発展を
　遂げているのが分かりました。


　
■今のアパレル業界においては、

　こうした機械を工場に
　大量に並べて生産しているのですから、

　まさに昔と比べて、比肩することも
　できないくらいの驚異的な進歩です。



■そんな博物館の見学を通して、

　教科書の中でゴシック体にて
　記されているだけの存在であった、

　「豊田佐吉」

　という人名、ならびに

　「自動織機」

　という、それまで全くイメージの湧かなかった
　機械が急速に身近なものに感じられました。



■豊田佐吉翁の開発した自動織機は、

　繊維産業においてまさにイノベーションと
　呼ぶに相応しい、

　とてつもない生産性の向上に
　貢献したことがよく分かりましたし、

　翁に対する畏敬の念すら生まれます。



■佐吉翁亡き後も、

　連綿とその後を引き継いだ
　数多くの技術者の努力によって、

　生産性はかつての数千倍（？）ほどの
　レベルにまで引き上げられたわけです。



■この生産性向上の軌跡は
　製造業においては

　「可視化された形」

　で実感を伴って
　理解することができます。


　
■さらにこの博物館においては、

　進化の過程が時系列順に
　分かりやすい形で展示されているので、

　非常に意義深い施設だと思いました。



■こうした様々な技術が織り重なって
　もたらされた生産性の向上が、

　人類をとてつもなく
　　豊かにしたわけですね。

　（おかげで私たちはユニクロの服を
　　あれだけ廉価に購入できるのです）



■一口に生産性というだけでは、

　抽象的な概念であり、
　イメージも湧きにくいものです。


　こうした展示を観覧することにより、
　身体感覚を伴う具体的な感触を伴って、

　生産性を引き上げることの意義や
　重要性について理解できたことは、

　私にとっても
　大きな発見と学びでありました。



■私はネットを介して相当程度のところまで
　完結する仕事を行っているわけですが、

　こうした環境に身を置いていると
　今回の視察のようなことがなければ、

　口ではいくら「生産性向上」と
　唱えていても、

　リアルな身体感覚を伴わせて
　理解をすることは難しいものでした。



■今回の視察旅行は、

　「生産性の向上」について
　改めて思いを致す、

　素晴らしい機会になりました。


　もしトヨタ産業技術記念館に
　行かれたことのない方は、

　ぜひ、時間を取ってでも、
　訪れられることをお勧めいたします。



■百聞は一見に如かず、

　ここに記したことは私が感じたことの
　ごくごく一部の気づきでありますが、

　たくさんの、そして深い学びを
　得られることをお約束いたします。


　---------------
　（ここまで）



■もし未だGPT-4、触られていない方は、
　すぐさま試してみて、

　返す刀でトヨタ産業技術記念館に
　足を運ばれ、


　向こう何年かの間にホワイトカラーの
　生産性においても、

　人力で織物を織っていたときと
　自動織機以降の差（数千倍）と同等、
　
　あるいはそれ以上の差が生じるだろう、


　そんな現実を体感、想像なされてみると
　よいかと思います。



■正直、触れば触るほど、

　「とんでもないものが
　　この世に誕生した」

　という思いしか出てきません。

　初めて自動織機を見た人も同じことを
　考えたんじゃないですかね。



■自動機織で機を織る速度を見たら、

　努力とか気合いとか根性とか
　まったく意味がない、

　と秒で分かったはずですからね。



■新しいテクノロジーに対して
　興味を持って触れ、試す必要があるし、

　それを怠ると、あっという間に
　置いていかれるのでは、

　ということが、

　（自戒を込めて）今日のメルマガで
　お伝えしたかったこと。




■「いいね！」するだけで、
　日々の更新情報が届きます。

　たまに「ここだけの情報」を流します。
　https://www.facebook.com/heiseishinkaron


　今日も人生とビジネスを楽しみましょう！
