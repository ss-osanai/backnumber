■（あと２日）
　大量の行動＆学習＆振り返りのみが
　「身体に紐付いた実践的な知恵」を
　もたらしてくれる

───────────────────


■これまでの人生において
　誇張ではなく、実際に、

　数百回、数百日では
　きかないくらいの回数、

　（おそらく累計で1000回超、
　　数千時間以上）

　さまざまなセミナーや勉強会、
　講演会等に参加し、

　数千冊ではきかないほどの
　本を読んできました。



■メルマガを配信しはじめて
　10年くらいの間は、

　セミナーや勉強会参加、会食をしたら
　会食場所等も可能な限り書いていたので、

　その頃からメルマガを
　お読みくださっている方は、

　これは本当のこととご理解頂けるかと。



■もちろん、

　メルマガのバックナンバーは
　第一号からすべてデジタルで保管し、

　それに加えて
　既に膨大な量になっていますが、

　全てを紙で出力もしているので
　証拠保全（？）も完璧です。



■また、そうした学びとともに、

　起業前後から
　何千回ではきかないくらいの、

　意思決定と行動を
　繰り返してきました。


　
■メルマガ配信、という意思決定
　＆行動だけでも今日時点で

　「7,288回目」

　ですから、

　少なく見積もってもビジネスに関連する
　意思決定は小さなものまで含めると、

　これまで少なくとも数万回、
　ないし10万回以上、

　行ってきたといえそうです。



■こうした

　【大量学習＆大量実践＆大量振り返り】

　によって、

　うまくいったことや
　うまくいかなかったことが、
　
　次第に浮かび上がり、
　自分でも理解できるようになりました。

　

■その成果物の一部は
　各種音源でも発表してきましたが、

　具体的には、
　
　こうすると（傾向として）
　うまくいくことが多そうだ、

　こうすると（傾向として）
　よくない結果になることが多そうだ、

　といった、

　ビジネスやキャリアにおけるルールや
　法則がある程度見えるようになってきた、

　ということですね。



■こうした、

　累積してきた数万回単位の
　トレーニングの集積が、

　危機に強く、同時に、
　確度高く機会を獲得するために必要な、

　基礎的な体力を培ってくれたように
　思われます。



■また、

　何ごとによらず、大量の学習と行動と
　振り返りを重ねなければ

　【身体に紐付いた実践的な知恵】

　は手に入らないのではないか、
　とも考えています。



■時間をかけて、身体を通して獲得する
　知恵だからこそ、

　安定的な成果をもたらしてくれるし、
　簡単に失われることはないし、
　すぐに崩れることもないし、　

　応用も利くし、
　自信や誇りも湧いてきて、

　幸福感やQOL（人生の質）に良い影響を
　与えてくれることとなるのでしょう。



■これまでは主として、

　商売やキャリア構築の分野に特化し、
　鍛錬を重ねてきましたが、　

　（まだまだ始まってもいないと
　　思っていながらも）


　それでも、起業当初と比べたら、
　
　「当時は想像もできなかったことを
　　実現させられた」

　という思いもあります。



■今は、こうした経験を通して得られた

　【学習やトレーニングのノウハウ】

　を

　「投資や資産運用の分野」

　に集中的に投下し、

　対象は異なるけれども、
　上記と全く同じことを行っています。



■ここ数年（特にコロナ後）、

　意図的にそちら方向に資源投下を
　シフトさせてきた感じでしたが、

　現在はさらにアクセルを踏み込み、
　可処分時間の過半を、

　（投資＆資産運用の）
　学習、実践、振り返り、

　に使っている状態です。



■破滅しないことを大前提にしつつも、

　取っているリスク以上の
　リターンを積極的に取りに行き、

　（これから長い時間をかけながら）

　リスクとリターンの最適解を
　身体感覚でつかむための訓練を、

　ひたすら行っていくつもり。



■一時的、瞬発的でなく、
　運やまぐれにも依存せずに、

　納得のいく結果を出せるようになる
　までの能力や実力がついた、

　と実感できるようになるためには、


　これまでの経験から、

　「最低でも５ないし10年から、
　　長ければ20年くらい」

　の時間はかかると思っています。



■しかし人生100年時代において、
　たとえそれだけの時間をかけてでも、

　新たな、そして身につけておくことが
　必須と思われる能力を強化しておくことは、

　決して無駄にならないどころか、

　更なる人生の質を高めつつ、
　危機に対する耐性が高まるものと思っています。



■なので、

　長い時間軸を意識しながら、
　足元をきちんと固めていくべく、　

　評価が高く、読むべきであると
　言われてきた古典＆定番の本を、

　毎日、0.5冊ないし、
　2冊のペースで読み続けています。


　（○○するだけで儲かる、的な本は
　　一切遠ざけています。

　　要するに文字が
　　ぎっしりと詰まった本しか読まない）



■分かってはいましたが、

　好きになった対象には
　いくらでも時間を投資できますね。

　まさに、好きこそ物の上手なれ。
　


■もともとお盆に帰省や旅行の予定を
　あえて入れず、

　例年のようなセミナーやコンサルの
　集中投下もあえて行わず、

　この期間に徹底的に勉強してやろうと
　思っていましたが、

　予定以上の進捗です。



■こうした学習と実践の成果が、

　言語化されて、
　外に現れてくるのは、

　経験上、早くても数年ないし、
　5年以上経過してからですが、


　そこまでいけば、

　これまでに培ってきた商売（ビジネス）や
　キャリア構築とともに、

　投資や資産運用におけるパラダイムや
　思考、方法論を統合した形での

　【大統一理論（！？）】

　が浮かび上がってくるのではないか、

　そんな期待に
　胸踊らせているところでもあります。



■ちなみに、
　とはいえ、現在も、投資・運用に、

　時間や認知資源を含めたリソースの、
　
　100あるうちの100を
　全振りをしているわけではなく、

　10ないし20程度はこれまで通りの分野にも
　継続して投資を重ねています。



■それらは、

　私（鮒谷）の今の生活や事業の基盤を
　作り出してくれた学習対象ですから、

　陳腐化せぬよう、さらには
　一層、磨きをかけていくべく、

　継続して更新をかけていく
　（トレーニングを重ねる）のも、また当然。



■そうした一つの取り組み
　（学び、更新）の機会として、

　先日よりご案内の通り、

　私（鮒谷）が以前より
　詳しくお話を聞きたいと思っていた、
　
　松澤さんにお願いして、
　お話しいただくことになりました。



■キャリア構築や、
　それに伴いビジネス立ち上げについての、

　さまざまな気づきやヒントをいただけるものと
　今から楽しみにしています。



■これは昔から書いてきたことですが、

　私（鮒谷）は自分が学びたいこと、
　かつ学ぶべき価値があると考えたもの、

　しか学びません。


　
■もし、このメルマガの内容に、

　多少なりとも共感なり共鳴なりして
　くださっている方であるならば、

　大いに学べるところがあると
　思いますよ。
