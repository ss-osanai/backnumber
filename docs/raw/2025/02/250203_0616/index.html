───────────────────


■今日は

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------

　ドン・キホーテ創業者・安田隆夫さんの
　原点は、

　「俺はこのまま何も成し遂げず、
　　誰にも知られずに朽ち果てていくのか」

　という思い。

　身悶えするような苦悩こそ、
　奮起を促す最高の燃料。


　著書『運』の中で、

　不遇時代の寂寥感と孤独感は
　その後の根源的な活力の素になった、

　と語られている。

　分かる。
　縮みが大きいほど、後で伸びる。

　----------

　人の言動がいちいち気になる人は、
　自分の軸を確立できとらんのよ。

　「私はこうした信条と共に生きる」

　という確固たる価値観が築かれていたら、
　自分は自分、人は人、となるから、

　心穏やかに過ごせるよ。


　SNSは他者を過剰に意識させる方向に
　導くプラットフォームやから、

　罠にハマらんように注意するんやで。

　----------

　心穏やかに過ごしたいなら、目指すべき
　遠い未来だけを見てればいいのよ。

　焦点距離が近くなるほど
　心が揺れ動くから。


　同じ環境にあっても
　焦点をどこに合わせるかで、

　平和に生きられるか、
　悩みに押し潰されるかが決まるんよ。

　お互い、到達したい理想を掲げ、見つめ、
　足元を踏み固めながら生きていこうやで。

　----------

　模倣の目的は「模倣不可能な世界」を
　確立するためやからな。

　無数の、遠い世界の参照が
　模倣困難性、さらには代替不可能性を
　もたらしてくれる。


　替えの効かない存在になれば、
　自由度の高い人生、ビジネス、キャリアを
　選択できるようになるんだわ。

　ちなみに創造的模倣と劣化コピーは
　まるで別物やで。

　----------

　一度、顧客基盤を作れば
　信頼を裏切らん限り、
　安定した収益がもたらされるよ。

　最初は大変やが、
　後からどんどん楽になる。


　手を抜いて成果を出すのがエライ、
　的な風潮に騙されてハンパな手抜きをするより、

　前倒しでやり切り、積み上げた方が
　トータルでパフォーマンス良くなるよ。

　----------

　「今すぐお申込みを！」がない通販は
　ないんだわ。

　バカみたいやけど、あれ本当に効くから
　どこも使ってんのよな。

　してもらいたいことあれば遠慮せず、
　「今すぐ●●を」構文を使うこと。

　----------

　経営者はDカップがお好き、と
　ワイの友人が言っとった。

　PDCAではなく、Dから始めろと。
　それでDCAP。マジでそれ。


　まず動くところから
　歯車回り始めるんよ。

　ワイも動きが止まりそうになる度、
　Dカップを思い浮かべとる。

　----------

　どこを切り取っても単調で、
　変化のない人間は面白くない。

　興味持ってもらうには
　多面性を強調すべきなのよ。

　つまり意外感が大事。


　ダイヤモンドのブリリアントカットは
　58面体やけど、人間も同じで
　58もの多面性があればキラキラ輝く。

　時間をかけて一つ一つ、
　面を削っては磨いていくことよ。

　----------

　「世界の見え方を一変させる」

　そんな可能性を持つ言葉のプレゼントは
　究極の価値提供やと思ってる。


　言葉はパラダイムを変え、
　更新された認識は、
　
　より大きな価値を生む物やサービス、
　仕事の仕方や日常の振る舞いにまで
　影響を及ぼすわけだから。

　----------

　もらいたければ与えなさい。

　何かの聖典にでも出てきそうな話やけど、
　真実よ。

　この世は常にバランスなんよ。

　もらうばかり、与えるばかりだと
　バランスが崩れる。

　----------

　躊躇なく、恥ずかしがらず押し切る力。
　それが営業力。

　遠慮している限り、欲しい物は手に入らん。
　欲しい物は欲しいと言え、いうことよ。


　ただ、その前に徹底した下準備で

　「買っていないが買っている」

　状態を作ってからのこと。

　----------

　本気で動く人を見て、
　バカにしたり叩いたりする風潮あるじゃん。

　だからチャンスなのよ。

　みんなビビって固まってる隙に、
　ゴリゴリ動ける人間だけが頭角を現せるのよ。

　ボーナスステージ到来やで。

　一時、盛大に恥をかいても
　喉元過ぎれば熱さを忘れるから、

　乗るしかない、
　このビッグウェーブに、だわ。

　----------

　物を売るには

　「売らずに売りつつ、売って売る」

　を心がけるといいよ。

　「売ってないけど、
　　何か売られたら買わずにおれない」

　という心境になるまで
　価値提供を徹底した上で、

　この機会に買って下さいと
　「売って売る」わけよ。

　両方が整うとエレガントに摩擦なく、
　熟した柿が落ちるように売れていく。

　----------

　「社長！経費でおごって下さい」と
　後輩から冗談半分で言われたことがあるんよ。
　しかもしつこく何回も。

　その瞬間、ワイの中で
　彼の存在はなくなったな。


　距離感を掴んでないのもそうやが、
　何よりワイにとっては経費じゃなくて
　ワイの金なんよ。

　根本的かつ致命的な認識の相違やで。
　世間を知らずは疎まれる。

　----------

　「win-winでいきましょう」

　と初対面で言ってくる人物の
　テイカー確率の高さは異常やな。


　当たり前すぎて、
　わざわざ言う必要のない言葉。

　分かっている人ほど
　態度で示そうとするけど、

　搾取系の人間には
　便利な営業トークの一つなんやろな。

　こちらも人物判定のリトマス紙として
　使えるからええけどな。

　----------

　親を泣かせてきた人間は
　いつか目覚めて、底力を発揮する。

　どこかのタイミングで深い懺悔の念が
　起爆剤となって成果を上げ、

　罪滅ぼしと親孝行に立ち上がるんだわ。


　ワイもそのクチで、ここ15年ほどの間、
　毎月新幹線に乗って実家に帰り、

　親と往時を懐かしみつつ、
　いろんな出来事を解凍しとるけど幸せよ。

　----------

　これからは一億総商売人の時代やで。

　組織に寄り掛かって生きていける、
　お気楽な時代はとうの昔に終わりを告げたんよ。


　何があっても己の食い扶持は己で稼ぐ。
　自身の生活は自分で守る。

　たとえ大組織に属していてさえ実態は、
　板子一枚、下は地獄の生活者。

　そんな自覚を持っておいた方がええと思うよ。

　----------

　スポーツの世界と比べたら、
　ビジネスの世界はぬるま湯にも程がある。

　真面目な話、ほとんどの人は
　真剣に仕事しとらんからな。


　スポーツに打ち込んでる中高生の方が
　はるかに本腰入れて、熱心にやっとるで。

　つまりちょっと本気を出すだけで、
　突き抜けられる可能性の高い楽勝競技いうことよ。

　----------

　会社勤めしてるときに気づいたのは、
　このままやと永久に経済的に不自由なまま
　という現実。

　ワイは本を読んだり、勉強するのが
　昔から好きやったけど、
　サラリーマンでいる限り、

　満足に本も買えなければ、
　講演会に行く金も足りんと気づいたのよ。

　勉強するにも多額のカネがかかる。
　それで独立を決めたんよ。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
