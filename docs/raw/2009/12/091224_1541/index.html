■こんにちは、鮒谷です。


　今日は一点、どうしてもお伝えしたいことがあります。


　私の古くからの親友であり、悪友？であり、兄貴分であり、
　先生であり、相談相手であり、

　毎年、２人で温泉合宿に行ったり、
　旅行に行ったり、


　（去年は道東を車で回って、北海道温泉行脚をしました。
　　※けして怪しい関係ではありません、、）


　私の人生に（良くも、悪くも？）
　大きな影響を与えてくれた今村暁さんが新刊を出されました。


　【今日のお勧め本　１分間の日記で夢は必ずかなう！
　　　　　　　　　　　成功と幸せを引き寄せる！「感性」を刺激する習慣】
　
　 　 http://amazon.co.jp/o/ASIN/489451379X/2ndstagejp-22/ref=nosim
　



■いうまでもなく、

　単に友人知人だからといって、
　誌面を使って書籍の紹介をすることはありません。

　（キリがないので、ご依頼されても全て、100％、
　　お断りしています）
　
　ただし、例外があります。

　

■それは

　古くからの付き合いで、
　私が大きな恩義や影響を受けた人であり、
　
　同時に、

　その人の著書の内容を私自身が担保出来るし、
　薦められるものであること。


　そういう人の本の場合、依頼を受けなくても
　勝手紹介することがあります。



■恩義に報いたいので。

　そして、「合う合わない」はあっても
　一般的な読者さんにとっては、役に立つとおもうから。
　

　そして今回の今村さんの本は、完全に上記の条件に
　合致している、ということ。


　なので勝手紹介させて頂きます。



■今村暁さんは

　
　----------------------------------------------------------------------

　メンタルリスクマネジメント株式会社 代表取締役
　アシスト株式会社 代表取締役

　北海道大学法学部卒業後、日本長期信用銀行を経て独立。


　「感性豊かで自立的で積極的な人材の育成を!」
　「夢、希望、感動、自信、勇気、誇りの持てる感性教育を!」
　「すべては良い習慣づくりの習慣教育から」

　をモットーにし、「感性教育」と「習慣教育」「能力開発」の
　研究と実践を続け、
　
　自らも人材育成をテーマにした会社経営を行うとともに、
　数多くの企業のコンサルティング、経営者・管理職の
　コンサルティングを行う。


　著書に、

　『３分間日記』(かんき出版)、
　『親が子どもに教える一番大切なこと』(三笠書房)、
　『子どもの成績を決める親の習慣』(PHP出版)などがあり、
　
　著書は日本、韓国、台湾、タイで10万部を超えるベストセラーに
　なり、NHK でもそのノウハウが特集される。

　子ども達の教育現場や経営者ネットワークを通じ、
　うつ病などのメンタルヘルスの問題が急増していることに直面し

　「これからの時代はますますメンタルクライシスの時代になる」

　と警鐘を鳴らし、

　「個人と組織の活性化」「メンタルヘルスのリスクマネジメント」
　
　を行うメンタルリスクマネジメント株式会社を設立。


　----------------------------------------------------------------------


　という活動・経歴を通して、

　これまで1000名を越える経営者、ビジネスパーソン、
　一流アスリートに影響を与えてきました。



■ちなみに私（鮒谷）も、

　今の自分の人生の、あるいはビジネスの
　少なからぬ部分が今村さんの影響を受けて形成されています。

　最近はお互い、忙しくなり、
　そんなに頻繁に旅行や合宿、あるいは長電話（笑）
　
　もできなくなりましたが、

　２，３年前までは一旦、電話したら１時間２時間くらい
　平気で話すこともよくありました。

　（今でもたまにあります）

　そうしていく中で、いろんなヒントや発見、
　物事の見方、捉え方を学ばせてもらったものです。



■今村さんはそんなふうに様々な知見やノウハウ、方法論を
　お持ちですが、

　今回の本のタイトルは、
　
　「１分間の日記で夢は必ずかなう！」

　ということで、文章（日記）を書き続けることの重要性を
　説かれています。

　そして、これはホントのこと。



■私のメルマガも、有り体に言えば、
　日記の一種といってよく、

　そしてそれを毎日毎日、愚直に書き続けることで
　たとえゆっくりであっても、少しづつでも進化を遂げてきた、
　
　嘘偽りのない事例だと思っていただいて
　結構かとおもいます。


　
■そしてそんな事例を莫大に持たれているのが今村さん。

　その知見がこの本に収められている、
　というわけですからお勧めできないハズがありません。

 【　無駄打ちこそが近道　】

───────────────────────────────────



■昨日は

　先日、お目にかかって長い時間、お話を伺った、


　元・三井物産副社長、日本ユニシス社長、

　現在、住宅金融支援機構（旧 住宅金融公庫） 理事長

　の島田精一さん


　【今日のお薦め本　一瞬で自分を勇気づける 仕事に必要な言葉 
　　　　　　　　　　　　　　　　　　　　これから生き抜くために】

　　　　　　　　　　　　　　　　　　　　　　　　　島田 精一 (著) 

　 http://amazon.co.jp/o/ASIN/4761266473/2ndstagejp-22/ref=nosim



　の、


　「人は言葉で変わり、言葉に支えられる」


　という「言葉」をご紹介させていただきました。



■話は変わりますが、

　たまたま今日、大和ハウス工業の代表取締役会長の
　樋口武男さんの


　【今日のお勧め本　熱湯経営─「大組織病」に勝つ】

　 http://amazon.co.jp/o/ASIN/4166605860/2ndstagejp-22/ref=nosim


　を再読しておりました。


　（こちらもまた名著、です。
　　
　　樋口会長の薫陶をじかに受けてきた
　　大和ハウスの関連会社の社長さんを存じ上げている
　　のですが、

　　この方も本当に素晴らしい方で、
　　この経営者にして、この部下あり、

　　を目の当たりにする思いです）

　

■樋口さんは「熱湯経営」を標榜し、
　大和ハウスのＶ字回復を果たした立役者として有名な方。

　機会があれば、改めてこの本についても書きたいことが
　山ほどあるのですが、この本の中にも、まさに

　「力を与えてくれる言葉」

　が満載されています。



■ちなみに私は、書籍の価値は


　「どれだけ人生にインパクトを与える言葉が
　　その本のなかに埋め込まれているか」


　で決まると思っています。

　一冊一冊の本のなかには「金（きん）」の含有量の多い
　ものもあれば、僅少なものもありますが、

　それもこれもすべて、宝探しだと思えばいい。



■当たり前ですが全部の本に、
　純度の高い金が混じっているわけではありません。

　がんがんあたっていくなかで、
　含有量の多い本に当たるもの。

　そういった発想を持つことで「無駄打ち」を恐れなく
　なるわけです。じゃんじゃん無駄打ちすればいい。

　というよりも、無駄打ちすればするほど、
　発掘可能性が高まります。



■所詮、鉱脈の発掘なんて確率論。

　無駄打ちを恐れることで、
　かえって最も、人生にとっての無駄を犯している、

　ということもあるわけです。
