■鍵山秀三郎氏の、この本。

　ぜひとも勉強熱心な「平成・進化論。」読者のあなたには
　お読みいただきたい本です。

■毎日毎日はわずかな、小さな種まきであってでも、積み重なると

＜今日のピークパフォーマンス方程式。＞

　【　恐竜が絶滅した原因を考えれば、ルールに依存しない方程式が必要  】

───────────────────────────────────



■昨日の話を続けたいとおもいます。



■最近、特に大企業、優良企業、優秀ベンチャーともてはやされた
　会社が一瞬にして瓦解するケースが見受けられます。


　実は、わたしがかつて勤めていた外資系の会社も、フォーブス誌で


　【 世界の優良企業ベスト30位 】


　に入っていたことがありました。

　なんと（確か）あのトヨタよりも上位に位置していたのですから、
　びっくりです。



■パーテーションで仕切られたおしゃれなオフィスで、
　高級オフィス家具に囲まれ、
　アメックスのコーポレートカードが全社員に配られ、
　昼食を食べに行くのにタクシーを利用する。
  食べるランチは3000円。

　株価もうなぎのぼり、
　会話で出てくるのはストックオプションの話。

　いやー、（実は英語も満足に話せないのに）いい会社に潜り込めた。（笑）

　なんて浮かれていたのが会社倒産のわずか１、２年前。

　まさか、まさか、豪華客船だと思っていた船が、泥舟だった
　なんて、そのときはまだ知る由もありませんでした。。



■そしてその後、たった数年で、

　アメリカ史上最大といわれた倒産劇の只中に放り出されたわけ
　なのです。



■ＩＴの進展、終身雇用の崩壊といったように、われわれを取り巻く
　社会構造は激動しています。

　かつて地球上をわがもの顔で闊歩（かっぽ）していた恐竜が
　突然歴史の舞台から姿を消してしまったように、

　今まで興隆（こうりゅう）を誇っていた大企業が、ある日突然
　倒れることが珍しくなくなりました。

　まったく予期せぬ事態が、当たり前のように起こる世の中なのです。



■ここ10年くらいのあいだに、社会構造の変化についていった企業は
　躍進し、それができなかった企業は倒壊しました。

　そのサイクルは、今も、どんどん速くなってきています。



■こういうときだからこそ、

　たとえば、

　大手企業につとめているから、
　学歴があるから、
　○○としう資格を持っているから、
　今の会社に○年以上も勤務してきた、

　といった、

　ある枠組みの中でだけ勝てるルール、
　今までの状況の中でのみ通用したルール、

　そういうルールにしがみつくのではなく、


　すべてのルールを手放して、より本質的な勝利の方程式に根ざして
　自分の人生を設計しなければなりません。



■今まで自分を守ってくれていたルールを捨てるのは怖いことです。

　例えるなら、それまでプールでお母さんの手にひかれて泳いで
　いたのに、急にその手を離すようなものでしょう。


　初めは、手足をばたつかせて苦しい。水も飲むでしょう。

　しかしその苦しみを経験しなければ、自分ひとりで泳げるように
　なれません。



■３歳児とか４歳児なら、お母さんにしがみついていてもいいでしょう。

　しかし、10歳になってもしがみついている。
　20歳になってもそのまんま。

　30、40、50になってもお母さんの手を離そうとしない人がいたら、
　あなたはどう思いますか。

　お母さんはもう80ですよ？


　わかりやすくいえば、そういうことです。
　（わかりにくいかもしれませんね。すみません　笑）



■恐竜が絶滅した原因はいまだに諸説定まりませんが、

　とにかく何らかの環境変化があり、その変化に対応できなかったから、
　という見方が多いようです。

 （もちろん私は専門家でないので、詳しくは分かりませんが）



■そういうわけで、この「平成・進化論。」ではスキルやノウハウ的な
　ことはほとんどお話ししていません。

　スキルやノウハウといったものは、ともすると、今あるルールに
　依存している、ないしは付帯しているケースが往々にしてあります。


　それは否定しませんし、なにより私自身、スキルやノウハウを
　学ぶためにさまざまな投資をしています。


　ただ、それと同時に、そのスキル・ノウハウを動かすための
　ビジネスＯＳを鍛える必要がある、

　ということを、つねづね申し上げているのです。



■目先、身を助けてくれる既存のルールから手を離す。
　それは、最初は怖いかもしれません。

　しかし、いついかなる環境に放り出されても泳げる、
　勝ち残れる、

　そんな普遍的なピークパフォーマンス方程式を自分の中に確立する。


　これこそが、遠回りなように見えて、結果的には最も成功する
　確率が高くなり、

　安心感、安定感を持って仕事に取り組めるようになる道だと
　確信しているのです。
