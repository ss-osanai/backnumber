6853号 垂れ流される情報を鵜呑みにする人と、自ら情報を仕入れ、頭で考える人

■昨日もご案内いたしましたが、
　このたび当社では、

　日本経済新聞社政治部長
　（政治・外交グループ長）

　吉野直也さんをお招きしての
　プライベート勉強会を開催いたします。



■吉野さんは、

　（以下、日本経済新聞社のプロフィールより）
　--------

　現職は政治部長(政治・外交グループ長）、

　政治記者として細川護熙首相から
　岸田文雄首相まで15人の首相を取材し、

　財務省、経済産業省、金融庁など
　経済官庁も担当しました。


　2012年4月から17年3月まで
　ワシントンに駐在し、

　12年と16年の米大統領選を
　現地で報じました。


　著書は

　「核なき世界の終着点
　　オバマ対日外交の深層」
　（16年日本経済新聞出版社）

　「ワシントン緊急報告 アメリカ大乱」
　（17年日経BP）。


　TBS「サンデーモーニング」や
　BS-TBS「報道1930」、
　BSテレビ東京「NIKKEIニュースプラス9」
　などに出演

　--------


　という方で、
　ますますご活躍されています。



■同社入社以来、30年以上にわたって
　政治記者一筋、

　国内の総理大臣を含む大物政治家を
　担当されてこられたり、

　ワシントンに駐在時には、
　オバマ大統領（当時）の広島訪問をスクープし、
　特ダネとなったこともありました。

　（その経緯は

　「『核なき世界』の終着点
　　: オバマ対日外交の深層」

　　という書籍にもなりました）



■その後、現在のポジションに就かれましたが、
　そんな履歴を通して、

　国内外の政治情勢についての
　機微に触れる情報にアクセスできる立場で、

　長らく活動してこられた方。



■先日も、食事をご一緒しましたが、

　国内外の生々しい政治ネタについて
　楽しく、興味深く、

　お話を伺いました。



■ところで、このところ、

　政治においても外交においても
　懸案、問題が山積し、

　さらには参院選も間近となり、


　国内外の政治について
　強い関心を持たれていらっしゃる方も、

　少なくないのではないかと思います。



■そんなこともあり、
　このたび、

　今、国内外で何が起きているのか、
　何が起きようとしているのか、

　について、

　政治の世界という
　インナーサークル内のキーパーソン、

　として活動してこられた吉野さんより、


　（一方的なセミナーという形式ではなく）

　昨今の情勢を前半でお話し頂いた上で、
　ふんだんに質疑の時間も設け、

　皆さんと一緒に、さまざまなことを
　お聞きいただきたいと思います。

　（質問が出ないようなら、私が聞きたいことを
　　聞きまくります笑）



■いつもお伝えしているように、

　自分の人生やビジネスやキャリアは、決して
　取り巻く環境と無関係ではありません。


　むしろマクロ（社会環境全般）は
　ミクロ（個人）と不離の関係にあり、

　個人の人生や生活に
　直接・間接に大きな影響を及ぼします。



■人生、ビジネス、キャリアを大切に
　取り扱おうとされる方こそ、

　より遠いところ（外交や政治の世界）に
　まで関心を示し、

　今と将来において、
　どんなリスクや可能性があるのか、

　学び続ける必要があるのでは
　ないでしょうか。



■ちなみに
　少しでも歴史に関心を持つ人であれば、

　先の大戦の前後において、

　報道を鵜呑みにしていた人と、
　多様な情報源を持ち、自分の頭で考えた人、


　あるいは、

　耳に入ってくる情報だけで意志決定し
　（＝流されるままだった）人と、

　自ら（不都合な聞きたくないことも含めた）
　情報を手にしようと努め、
　その上で意思決定した人、


　その両者の差が生死を分けた、
　財産形成において決定的な差が生まれた、

　という話は、
　聞かれたことがあるでしょう。



■というよりも、

　戦前・戦中世代が主人公であった、
　少し前までの『私の履歴書』は、

　そんな話のオンパレードでした。


　私（鮒谷）も、そうした幾多の例を
　見聞きしてきたからこそ、

　より開かれた目で世界を捉えたいと
　考えてきました。

　こうした勉強会を開催しているのも
　その一環です。



■戦前でさえ、そうであったのに、

　今の時代、これだけ世界中が
　緊密につながるようになり、


　人生にしても、生活にしても、
　キャリアにしても、経営にしても、

　ビジネスにしても、資産運用にしても、

　国際政治はもとより、国際政治と無関係に
　成り立つものなど何一つ、なくなりました。



■風が吹けば桶屋が儲かる、の話には多分に
　「こじつけ」的なところもありますが、


　たとえばウクライナ情勢が
　我々の思いもよらぬところで、
　思いもよらぬ形で、

　私たちの人生や生活やキャリアや
　経営やビジネスや資産運用に、

　大きく反映されてくる、などというのは
　こじつけでもなんでもなく、

　既に現実に起きている話です。



■ということは、もはや

　【国内政治はいうまでもなく、
　　国際政治に無関心でいられる人は、

　　自分の人生や生活、
　　キャリア（以下略）にも、無関心な人】

　と言い切ってしまっても良いのでは
　ないかとすら思えます。
