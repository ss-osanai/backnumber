6686号 何を学び、何を学ばないかは自分が意思決定するしかない

■人生において目標を持つのは
　とても大切なこと。


　将来、どんな存在でありたいかを
　意識しているからこそ、

　「未来の私」と「今の私」の間に存在する
　ギャップが明確化され、

　足りないものを埋めるために努力する
　エネルギーが供給されるのです。



■理想（目標）なきところに努力なし。


　このままじゃまずい、
　なんとかしなければ、

　と思いながらも、
　だらだらと惰性の日々を過ごしてしまう、


　そんな方がもしあったとすれば、
　その人は間違いなく

　「理想の将来像」

　を持っていないか、

　持っていても
　時々刻々に振り返る、

　そんな機会を自らに与えていない人。



■ここまでの話は、

　【目標設定＆達成放談会音源】

　や、

　このメルマガにおいて同テーマで
　のべ何百通以上（？）は発信してきたはず。

　なので、これ以上は記しません。



■ただ、目標を掲げたは良いけれども

　「では、具体的な一歩を
　　どのように踏み出せば良いのか」

　について、

　なかなか明確な形で提示することは
　難しいなとは思っておりました。



■不確実性の世界において
　期待する成果を出すためには、
　
　果たしてどのような思考＆行動様式で
　臨めばよいのか、

　まさにこのテーマは私（鮒谷）の
　ライフワークであり、


　その一部は

　【鮒谷周史の、圧巻！
　「卓越のコミュニケーション戦略
　　＆プラットフォーム構築」で
　　わらしべ長者になる！音源】


　や

　「人生は鉄火場（賭場）と考えるとうまくいく」
　放談会音源
　～鉄火場に学ぶ、リスクを極小化しつつ、
　　安定・逓増人生を送るための７つの心構え


　などの音源の形で
　世に問うてきました。



■いずれも私（鮒谷）の
　お気に入り音源であり、
　
　かなり密度の濃いコンテンツである、
　そんな自負があります。



■ただし、

　これら音源を含めた、これまで私が
　お伝えしてきたコンテンツ全般は

　「ある部分に特化」

　したものであり、必ずしも全体の

　「見取り図」
　
　的なものではありませんでした。



■ところが、大学院で

　「エフェクチュエーション」

　に出会い、

　これまでお届けしてきたコンテンツを
　俯瞰して捉えていただくためには、


　なにをさておいても

　「エフェクチュエーションという
　　思考＆行動様式」

　を先に学んで頂きたい、
　そう思ったからこそ、

　先日来、
　強くご推奨してきた次第です。



■なかなか一口で説明しにくいので

　「とにかく聴いて下さい」
　
　としか言えなかったのですが、
　
　そんな私の言葉を信じて
　お越し下さった皆さまが、
　

　このようなご感想
　https://ssdlfiles.s3.amazonaws.com/testimonials_effectuation.pdf
　
　をお送り下さっているのを
　ご覧頂ければ、


　私（鮒谷）が口からでまかせを
　言っているのではない、

　と信じて頂けるかと思います。



■私としては、

　各種音源をお聴きいただくのは
　とてもありがたいことではありますが、

　その前にぜひ

　「エフェクチュエーション」

　を学んで頂ければ、

　人生やビジネスの見取り図を
　手に入れた上で、

　最短距離で行動し、結果を出せる、

　と思ったからこそ、
　繰り返しご案内している次第です。



■これまでのメルマガでも記してきたように

　「全ての学びの価値は
　　事後的にしか分からない

　（＝事前にその価値を
　　　判断することはできない）

　からこそ、

　力を尽くして案内したあとは
　究極的には
　
　「信じて下さい」

　としか言えません。



■そこから先は

　「あなた任せ」

　なんですよね。
　どんな学びであれ。



■その学びから得られる結果は、

　私の人生ではなく、
　あなたの人生に直結するものであり、

　だからこそ、最終的には

　「自分が何を学び、何を学ばないか」

　については各人が自身の直感を
　信じるしかありません。



■なので

　「エフェクチュエーション」

　と聞いて

　「いらんわ、そんなもん」

　と直感的に判断されたのなら、
　その判断に従われると良いでしょう。



■反対に

　「それは聴いておいたほうが
　　良いかもしれない、、、」

　と思われたのなら、
　その感性に従うのが良いかもしれません。

　（私ならそうします）



■いずれにしても、あらゆる学びは

　【一人ひとりが
　　自分で意思決定するしかない】

　のです。

　そして、その意思決定、ならびに
　そこから生まれる行動の集積が、

　あなたや私の人生を導きます。



■これ以上は言いません。

　参加したほうが良い、と思われたら
　以下よりお申込み下さい。

　既に多くの方が参加され（前回）、

　今回も多くの方が
　お申込みくださっています。


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】　　　
　　　　　　　　　　　　　　　　　　　　　
　■あらゆる学びの価値は、事後的にしか　　
　　分からない。　　　　　　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■その学びが学ぶ価値のあるものであるか　
　　どうかは、　　　　　　　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　　究極のところ、学んだ後でしか　　　　　
　　分からないのである。　　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■だからこそ、あるテーマについて、　　　
　　学ぶべきか、学ばざるべきか、　　　　　
　　　　　　　　　　　　　　　　　　　　　
　　この意思決定は最後の最後は　　　　　　
　　直感に頼らざるを得ないのだ。　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■自分の人生に関わることだからこそ、　　
　　自分が断を下す以外にない。　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■そんな主体的な学びの意思決定の集積が　
　　あなたや私の人生を形作る。
