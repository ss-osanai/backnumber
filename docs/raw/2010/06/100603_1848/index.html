 【　好奇心と知識欲の無限循環　】

───────────────────────────────────



■日経新聞、2010年4月24日（夕刊）のインタビュー記事によると、
　建築家の安藤忠雄さんは、

　中学３年のとき唐招提寺と東大寺南大門を見て感動し、
　建築の世界に動かされていったそうで、


　「人生を組み立てるときに、感動からスタートする」


　と語られていました。



■安藤さんは24歳のとき、

　西洋建築を見ようと、
　ナホトカからシベリア鉄道経由で欧州へ一人旅に出ます。

　京都大学に進学した友人から
　「建築家を志したら古代ローマのパンテオン神殿を見ておけ」
　と言われていましたが、


　「だが、いざパンテオンを見てもよく分からない。
　　好奇心が湧かない。知識も必要だと思った」


　と振り返る。



■知識ではなく感動で人は動く、
　だから知動とは言わず感動という、

　という話を昨日はしましたが、

　「感動を裏付けるものは知識である」

　こともまた、否めない事実でありましょう。



■同じものを見ても、
　その背景に関する知識の有無によって、
　捉え方が全く変わってきます。

　知識があり、その上で、関心を持ったものを見たならば、
　よく深く理解し、深く感じることができるように
　なるものです。



■安藤忠雄さんは

　「好奇心が持てれば、知識を得る力にもなるし、
　　そうして得た知的財産の総量が、また新たな好奇心を生む」

　とも言われていました。



■つまり、


　【 好奇心 → 知識欲 → 好奇心 → 知識欲 → 好奇心 → 

 　　知識欲 → 好奇心 → 知識欲 → 好奇心 → …… 】


　この無限循環によって、

　一分野を切り開き、時代を代表する建築家と
　なっていかれたのが安藤忠雄さんという人物なのでしょう。



■私たちも、最初の好奇心の芽を大事にし、
　知識で裏付けることで感動を深め、

　自分の世界を切り拓いていきたいものです。

【今日のつぶやき】



＜構造的空隙にビジネスチャンスあり＞


ネットワーク上に多くの「すき間」やギャップが存在すると
起業の機会が生み出される。

ネットワークのすき間とはある人が別の二人とつながりを持ち、
その二人はお互いに知らないという状況を意味する。

価値が生み出される潜在的な可能性が
そこにあることは明白である。


　　　　　　　　　　（ウェイン・ベーカー「ソーシャル・キャピタル」）



「ネットワーク上のすき間、ギャップ」を
構造的空隙（くうげき）、structural holes、と呼ぶそうです。


以前、この欄で、

「私（鮒谷）は自身のネットワーク作りにおいて多様性を目指す」

という話を書いたことを記憶していますが、

多様性のあるネットワークは、
必然的にたくさんの構造的空隙をもたらしてくれます。


そしてその構造的空隙がビジネスチャンスを
もたらしてくれることはいうまでもありません。

「すき間」「ギャップ」を埋めるところに「価値」が発生するもの
だからです。ここに起業、あるいはビジネスチャンスが生まれる。


だからこそ、自身（自社）の多様性と、その大きさ、
この両方をこれまで追求してきましたし、
おかげさまで相当額の利益も確保してきました。

たった本一冊の知識だけでも大変な利益をもたらして
くれるもの。


「自分の境涯、境遇に文句をいうヒマがあったら
　ただひたすらに勉強しろや」

昔の上司の言葉が今更ながら身に沁みます




■では、またあした！
