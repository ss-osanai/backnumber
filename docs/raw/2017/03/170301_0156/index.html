　「その程度のビジネス」

　「一定のリスクを取り、

【 ビジネスで必要なことは、すべて競輪選手が教えてくれた 】

■人生や経営、ビジネス、キャリア構築等において
　満足のいく結果が欲しいのであれば、

　「成果が上がることが、ほぼ約束されている
　　マイルールを持っておく」

　ことが必要です。


　ひとたび

　「成果が上がることが、ほぼ約束されている
　　マイルール」

　すなわち、

　「自身に最適化する形で確立された
　　意思決定の基準」

　を確立することができれば、


　ある状況を迎えたとき、
　いちいち考えることなく

　「自動的に成果の上がるルール」

　を適用するだけで一定の結果が生まれます。



■たとえば私（鮒谷）なら


　「後悔先に立たず」

　「転ばぬ先の杖」

　（だから何ごとによらず、出来る限り
　　前倒しで準備しよう）」

　「お金を惜しんで、
　　もっと大切なもの（時間）を失わない」


　こうしたことは、
　大切にしてきたマイルールの一つ。



■このルールが、

　定例化された人間ドック入りや、
　歯のクリーニング、

　何かに取り組む前に、まず学ぶ、

　そのためのコンサル、セミナー、書籍、
　会食等にかかる時間、金銭を惜しまない、

　といった行動につながっていいくのです。


　これからのことは分からないけれども、
　少なくともこれまでは、

　このマイルールに従っていれば

　「人生もビジネスも逓増してきた」

　ので、これからも同じことを繰り返していきます。



■こうしたマイルールをたくさん持てば持つほど、

　安定的な結果を享受し続けることが
　できるわけですから、

　一つでも多くのルールを持っておくことが
　大切。

　
　そうすると、

　意思決定にかける時間も省略でき、
　迷うストレスもなくなり、
　
　「淡々と生きているだけで
　　勝手に成果が生まれてくる」
　
　こととなります。


　
■昔、

　「高い生産性を誇る工場は喧騒がなく、
　　
　　（むしろ大丈夫かと思うくらい）
　　粛々と流れている」

　という話を聞いたことがありますが、


　一定のルールを持たず、混乱と混沌の中、
　都度、賑やかに意思決定していくのは、

　「仕事をしている気」

　にはなるかもしれませんが、
　実際には、

　「ただバタバタして、右往左往している」

　というだけの話なのかもしれません。



■起業も同じです。


　起業やビジネスにも

　「マイルールの確立」

　が必要で、
　
　精緻化されたルールを持っているほど、
　
　「安定的に成果を出せるようになる
　　（高い生産性を発揮できるようになる）」

　わけです。



■とはいえ、
　
　起業したことのない人が、
　それをイチから作るのは大変ですから、

　お薦めは

　「パッケージ化された意思決定コンテンツ」

　を外部から導入し（守）、

　
　それを自身に適合するように
　必要に応じて、適宜、改変し（破）、

　それを繰り返していくうちに、気がついたら
　オリジナリティが確立されていた（離）、


　というプロセスを経ることです。


　
■ゼロからオリジナリティを確立するなんて、
　
　「前前前世」

　から取り組んだって
　間に合わないかもしれません（汗）
　
　
　省力化のために
　パクれるところはパクって、
　
　ある程度、「型」を作ってから、
　そのあと一気に独自性を、

　というのが最短の道ではないでしょうか。


　
■私（鮒谷）はいつも、
　その方法でやってきました。
　
　風圧を受ける（＝イチから意思決定基準を作る）のは
　莫大なエネルギーを消耗するから嫌なので、

　「（競輪でいうところの）
　　追込方式」

　と名付け、励行してきたのです。


　ギリギリまで他者からの学びを導入し、
　あるレベルまでは一気に持っていき、

　そのうえで、独自色を発揮して、
　ぶっこ抜く、

　そんなイメージ。



■そんなわけで、

　「起業におけるマイルール」

　をいち早く確立し、とっとと守破離の道を
　歩まれたい方には、
　

　「鮒谷が十数年かけて学び、試行錯誤し、
　　確立してきた起業マイルールを、
