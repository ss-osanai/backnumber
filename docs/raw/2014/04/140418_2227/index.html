＜今日のピークパフォーマンス方程式＞

 【　成功の秘訣は「意識の幽体離脱」から　】

───────────────────────────────────



■成果は目線の高さに比例します。

　目線とは、夢、目標、志のこと。

　目線を上げるほど、日常の一挙手一投足に緊張感が
　生まれてくるでしょう。


　ここでいう緊張感は、いわゆる

　「緊張でガチガチになって」

　という状態ではなく、


　「リラックスしているけれども、
　　（心と体の）姿勢はピン！と張っている」

　という意味での緊張感。



■その状態に入る必要があるから、

　やり方を考えるよりも先に、
　目線を上げておく必要があるのです。

　
　そして目線を上げるためには、
　目線を上げてくれる人に親近することが一番。

　

■こうして高い目線を持ち、ゴールを意識し続け、

　さらには無意識レベルでも
　常に願い、求めるレベルに至れば、

　行住坐臥、寝ても覚めても、
　目標達成に至る道筋を考え続けることになるでしょう。


　これは能動的にそうしよう、と思って、
　実践しているといった世界ではなく、

　無意識的、自動的に

　「（スイッチが入り続けている）状態」
　
　といったほうが適切かもしれません。



■そんな状態に入ることさえできれば、

　意識のレベルに応じて、行動のレベルも上がるし、
　行動のレベルが上がるから、結果のレベルも上がる。


　必然の帰結として、

　「目線の高さに応じた成果が現れる」

　こととなるのです。



■とするならば、

　「どうすれば、高い目線（目標）に
　　リアリティを感じることができるのか」

　について考える必要があります。


　目指す世界に実現可能性や臨場感を感じられれば、
　感じられるほど、

　「その気になれる」
　
　から。



■一つの解としては、

　「その世界をすでに体得、体現している人と
　　至近距離で接する」
　　
　ことではないかと思います。


　私も、その時点における理想の世界に出たいと
　思ったときには、

　まずはその世界にすでに至っている人に

　（どれだけハードルが高くとも）

　弟子入りすることを心がけてきました。



■こうして接点を持ち、
　さらには親しく交わらせて頂く中で、

　その世界の空気や温度感を臨場感とともに
　感じることによって
　
　「あ、オレもその世界の住人になれそうだ」
　
　という勘違いを自らに起こさしめる。



■その勘違いが

　「（向こう側の世界への）意識の幽体離脱」

　を起こし、


　向こう側の世界（未来の自分）から、
　こちら側の世界（現在の自分）に向けて、

　「おーい！この世界に早く来いよ！」

　と呼びかけられることとなるのです。



■これをもう少し、危なくない話（？）として
　書くならば、

　冒頭に記した
　
　「目線を上げるほど、日常の一挙手一投足に緊張感が
　　生まれてくる」

　という無難な表現となるわけです。



■こんな風に、自らの変化はあくまで

　「未来から現在に向かって働きかけられるところから
　　生まれる」

　のです。


　けして、

　「現在から未来に向かって働きかけるところから
　　生まれる」
　
　ものではありません。


　未来から現在であって、
　現在から未来ではないのです。



■そのためにも、

　あなたがもし本当に変わりたい、と思うのであれば、
　できるだけ早く、


　「（あなたにとって）理想の姿を体現、体得した人」
　
　に接し、間近に感じ、

　未来に意識を飛ばす必要があるでしょう。
