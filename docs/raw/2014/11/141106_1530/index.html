▼今日から大阪出張！

───────────────────────────────────



■こんにちは、鮒谷です。


　今日から大阪出張。

　東京にいるときより予定パツパツの日々が
　しばらく続きます。


　
■仕事もできず、誰からも相手にされず、
　孤独だった（涙）

　ダメサラリーマンの時代を思い出すにつけ、
　
　これだけ引き合いを頂いていることに
　毎日、感謝、感激しております。

　
　スケジュールが詰まっていることをこそ、
　喜ばなければなりませんね。
　
　大阪の皆さん、ありがとうー！！



■話は変わりまして、

　先日、個別コンサルさせて頂いた、
　IAM Corporation社長の沢辺 裕平さんより、

　以下のようなご感想を頂戴いたしました。


　（ここから）
　----------------------------------------------------------------


　鮒谷さま


　本日はありがとうございました！！


　頂いたお言葉で特に心に刻まれたフレーズは、


　・未来の自分から見て(理想の自分からみて)、何を今やるべきか
　
　・即時行動をすること。その一点のみ。

　・かき集めて圧縮


　この3つでした。


　これらを行うためにも、
　時間の管理を次のレベルに引き上げ、
　
　「変な客」に全力を注げるような状態を
　構築しないといけないと肌で感じました。


　これらの言葉はメルマガや本などで、
　聞いたことはあったかと思いますが、

　実際に鮒谷さんのお言葉で聞くと伝わる威力が違いますね。


　早くリトマス試験紙をあてにゆく時間も
　作らないといけませんし、

　無重力状態になれるように、
　やらなければいけない行動は尽きないと体感しました。


　このメールもシステム化出来るように
　今日から行動を取りたいと思います。

　本日はありがとうございました！


　　　　　　　　　　（IAM Corporation代表取締役　沢辺 裕平さま）


　----------------------------------------------------------------
　（ここまで）



■ドラッカーの言葉に

　「成果をあげる者は仕事からスタートしない。
　　時間からスタートする」

　というものがあります。


　この言葉に触発され、

　徹底した（自分仕様にカスタマイズした）
　時間管理法を試行錯誤の末、確立できた瞬間から、
　
　それまでと比較にならぬくらい、
　大きな成果を出すことができるようになりました。

　

■沢辺さんもぜひ、このドラッカーの言葉を胸に、
　
　「時間管理のレベル」

　を一気に引き上げて頂ければと思います！


　この度はありがとうございました！！



■それでは

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。

＜今日のピークパフォーマンス方程式＞

 【 お客さまが考えていることは、一から十まで◯◯◯ 】

───────────────────────────────────



■商売に携わる者が絶対に忘れてはならぬのは、


　「お客さんが興味があるのは、
　　一から十まで自分のこと」


　であるという、厳粛な事実。


　お客さまは決して、断じて、

　「あなたのビジネス」
　
　に興味があるのではありません。



■だからこそ、
　商売するにあたって経営者、起業家は

　「一から十まで
　　お客さんの（利益の）こと」

　を考える必要があるのです。


　というよりも、

　「一から十まで
　　お客さんの（利益の）こと」

　だけを考えていればいいのです。


　これが徹底されれば結果として、

　「あなたの売上」

　へと繋がっていくわけです。



■そのからくり、仕組み、構造を
　分かっていただくために

　「裏を見せ、表を見せて、散る紅葉」

　ではありませんが、


　私（鮒谷）はあえて、

　当社のマーケティングにおける原則や実際を
　包み隠すことなく、一切、開示しています。



■つまり

　「お客さまに尽くしていけば、
　　お客さまに良くして頂けるんですよ」

　という証拠、証明を、

　出来る限り、メルマガ上で示していこうと
　考えているのです。


　（その実現のためには、

　　強烈な能力的、精神的、時間的、その他の
　　プレッシャーがかかってくるのは当然のこと）



■この姿を率先垂範してお見せすることで、
　ビジネスに携わる人に対し、

　「お客さまの利益」

　について考えてみては、

　という問題提起をさせてもらっているわけです。


　もちろん、そんな問題提起をするままが、
　
　「自身への更なるプレッシャー」

　につながってくるのは、
　言うまでもありませんが、それは

　「望むところ！」

　です。



■そして現実に

　「お客さまの利益」
　
　について考えれば考えるほど、
　結果としてどういうわけか
　
　（というか、当たり前の話なのですが）

　自動的に

　「商売繁盛」

　するようになっているのです。


　
■あなたや私が携わるビジネスは
　かくあるべきではないか、

　との問題提起、ならびに自戒、
　さらには自らを奮い立たせるために、

　今回のメルマガを記させて頂きました。


　今日は短めに。




■「いいね！」するだけで、日々の更新情報が届きます。
　たまに「ここだけの情報」を流します。
