7275号 「己の魂を蘇生させる」ために、今、私たちができること
7277号 クールヘッド＆一見、ぶっ壊れたようにすら見える意思決定

7276号 金儲けが趣味の人（0.1％）、趣味が実益になっている人（4.9％）、それ以外の人（残り95％）

■既に副業、起業されている方は
　もちろんですが、

　この時代、

　「副業しよう、起業したい」

　という人でなくても、
　何が起こるか分かりませんから


　「もし副業、起業するなら
　　どんな感じになるのかの思考実験」

　をあらかじめ行っておいても
　バチは当たらないでしょう。



■逃げずにあれこれ考えてみることで
　気づくこともあるかもしれません。


　自分の将来を考えるときの
　一つの視点として

　▼やりたいことがない
　▼やりたいことがある

　と

　▼マネタイズ（収益化）できない
　▼マネタイズ（収益化）できている

　の両者の組み合わせを
　考えてみるのはありかもしれません。



■そうすると、

　-----------

　１、やりたいことがなく
　　　マネタイズもできていない

　２、やりたいことはないけれども
　　　マネタイズできている


　３、やりたいことはあるけれども
　　　マネタイズできていない

　４、やりたいことがあり
　　　それでマネタイズできている

　-----------

　の４パターンができますね。



■このうち、一番いいのは

　-----------

　４、やりたいことがあり
　　　それでマネタイズできている

　-----------

　でしょう。

　「趣味と実益を兼ねて」というやつ。



■さらにうまくいくと、

　「趣味で楽しくやっているのに
　　儲かって仕方ない」

　ということになるかもしれませんね。



■反対に、一番キツいのが

　-----------

　１、やりたいことがなく
　　　マネタイズもできていない

　-----------

　でしょう。


　楽しみがない上に、
　お金にも恵まれない。

　これはなかなか辛そうです。



■次にどれがいいかは
　意見が分かれるかもしれません。

　-----------

　２、やりたいことはないけれども
　　　マネタイズできている

　-----------

　あるいは

　-----------

　３、やりたいことはあるけれども
　　　マネタイズできていない

　-----------

　のどちらがいいのか。



■「２」の人、つまり、

　「やりたいことはないけれども
　　マネタイズできている」

　人は、数は少ないですが
　たしかに存在しています。



■実は、この種の人は、

　「やりたいことはないけれども」

　といいつつ、
　その本質は

　「金（＆金儲け）そのものが好き」

　なんですね。



■なので、

　別にやりたいことがなくても
　構わないし、

　儲かりそうなものを見つけては
　脈絡なくなんでも金に変えていく、

　こうして収益を上げるのを
　楽しんでいる人。



■この人は、カネになるのなら、
　何でもやりたいと思っているし、

　またそれが強い動機となって
　本当に金も儲かるという、

　要するに

　「金儲けそのものが趣味」

　の人なのです。



■こうした人はたしかにありますが、
　世の中全体から見ると極めてレア。

　ほとんどの人には無縁の精神性と
　言えるでしょう（私（鮒谷）もそう）。

　なかなかに模倣困難です。



■一般的には、ほとんどの人は

　「それが好きだから、
　　やりがいがあるからというところに、

　　喜びを見出し、仕事をして
　　お金ももらいたい」

　と考えるもの。



■とはいえ、現実問題として

　-----------

　３、やりたいことがあるけれども
　　　マネタイズできていない

　-----------

　というのも、厳しいもの。


　やりたいことがあり、

　それを仕事にしたい、
　あるいは既に仕事にしているけれども、

　それがお金に変わらない。



■すでにそれを仕事にしている人も、

　お金が続かなければ、
　商売を畳むしかありませんし、

　いつか仕事にしたいと願う人も
　お金がなければ生きてはいけません。



■したがって、
　この道を選択するのが許されるのは、

　（ほぼ失業の可能性がなく、
　　ときには残業もあまりない）

　地方公務員の人、

　もしくは、

　実家が太いとか、
　大金を相続したとか、
　巨額の宝くじをあてた人、

　くらいなものでしょう。



■こうした方が

　「趣味に生きているうちに
　　ときに大化けする」

　のを、

　なにかの記事で見ることもありますが、
　これはこれで高いハードルです。



■綺麗事を抜きにすれば、

　普通の人は、好きなことがあって、
　表面的には楽しくても、

　経済的な不安を抱えている限り、
　不満とか、不安が消えることがない。



■ここまでの話で、すでに

　-----------

　４、やりたいことがあり
　　　それでマネタイズできている
　　（趣味が仕事になっている）

　-----------

　あるいは

　-----------

　２、やりたいことはないけれども
　　　マネタイズできている
　　（金儲けそのものが趣味で
　　　うまくいっている）

　-----------

　人は、もう満足しているから
　おいておくとして、


　問題は

　-----------

　１、やりたいことがなく
　　　マネタイズもできていない

　-----------

　３、やりたいことはあるけれども
　　　マネタイズできていない

　-----------

　という人です。



■前者は、

　【好奇心がもたらしてくれるパワーや
　　エネルギーの大きさを思い出す】

　必要があるでしょう。


　後者の人は

　【没頭没入の対象をお金に変えるための
　　考え方やノウハウやテクニック】

　を学ぶ必要があるかもしれません



■そう考えてみると、今度、開催する


　【ニッチジャンルで圧倒的１位を確立！

　「独自すぎるビジネス」で食っていく方法が
　　身につく超密度セミナー】


　は、まさにこれらの方々にとっての
　福音となるかもしれません。



■お招きする講師を改めて紹介すると、
　以前からの友人で、

　その振り切った生き様に
　私（鮒谷）がリスペクトしてやまない、

　松澤茂信（まつざわ・しげのぶ）さん。



■以下、松澤さんのプロフィール。


　（ここから）
　-----------


　1982年東京都生まれ。

　上智大学経済学部卒。
　国内旅行業務取扱管理者。


　2011年より日本中の変わった観光地
　（珍スポット）を紹介するサイト

　「東京別視点ガイド」

　を始める。

　日本全国＆アジア各国1,000ヵ所以上
　巡った珍スポットマニア。


　2018年合同会社別視点を設立。

　来場者20,000人を超えるイベント
　「マニアフェスタ」、

　マニアによる専門的ガイドが楽しめるツアー
　「別視点ツアー」

　などの体験型イベントを数多く実施。

　2023年に学芸出版社より
　『マニア流!まちを楽しむ「別視点」入門』
　を出版。


　-----------
　（ここまで）


　松澤さんという「異常者（笑）」が、

　好奇心が自らの内に眠る
　パワーやエネルギーを、

　どのように開放してくれるのか、


　さらには、

　開放されたパワー＆エネルギーが
　どのようにマネタイズされているのか、

　について、語り尽くして下さる予定に
　なっています。



■以下はセミナー内容の、
　事前チラ見せとなります。


　（ここから）
　-----------


　年間2万足を売る
　便所サンダル専門通販サイト、

　ない街の地図を作るプロフェッショナル、

　軽妙な下ネタトークでリピーターを
　生み出しまくる大人の博物館……。


　世の中には驚くべきニッチなジャンルで、
　圧倒的1位のポジションを確立し、
　独自すぎるビジネスを営む人間がいる。


　商売と生き様が、たぎるような熱量のもと
　一体化する、いわばマニアな人々。


　講師は合同会社別視点代表の松澤茂信。

　マツコの知らない世界・タモリ倶楽部に
　出演経験があるマニアが多数出展し、

　例年1万人以上の来場者が押し寄せる
　人気イベント「マニアフェスタ」の主催者で、

　月間100万ＰＶ・Twitterフォロワー
　10万人の珍スポット紹介ブログ
　「東京別視点ガイド」の運営者でもある。


　日本中のあらゆるマニアと交流し、
　自身もマニアな事業を展開する経験をもとに、

　事例とエピソードトークをたっぷり交えて、
　ニッチジャンルで独自ビジネスを
　生み出す方法をお伝えする。


　-----------
　（ここまで）



■以上のような話を聴ける機会が、

　【8月20日（日）13:30～18:00】

　にありますが、冒頭の話を受けて、

　「これは聴いておいたほうがいいかも」

　と思われたあなたのような方にこそ、
　参加してもらいたいと思っています。



■このセミナーに参加して、
　ぜひ、あなたも松澤さんのような


　-----------

　４、やりたいことがあり
　　　それでマネタイズできている
　　（趣味が仕事になっている）

　-----------


　の世界に到達するためのヒントを
　手に入れて下さい。



　今日も人生とビジネスを
　楽しみましょう！

　【今日のピークパフォーマンス方程式】

　　■世の中には

　　　１、やりたいことがなく
　　　　　マネタイズもできていない

　　　２、やりたいことはないけれども
　　　　　マネタイズできている

　　　３、やりたいことはあるけれども
　　　　　マネタイズできていない

　　　４、やりたいことがあり
　　　　　それでマネタイズできている

　　　の４通りの人がある。

　　■２と４の人は機嫌よく生きているので
　　　勝手にやってもらっていればいいが、

　　　もし自分が１か３に該当すると思ったら

　　　（２の人は特別なので参考にせず）
　　　４の人から学びを得て、

　　　「好き」の感情を思い出し、燃料にして、
　　　マネタイズする方法を学んでおくことは、

　　　これからの時代において
　　　悪いことではないだろう。
