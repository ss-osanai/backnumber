7266号 観るべし！読むべし！B'z35年の軌跡を振り返る独占インタビュー

■上述のように、先ほど、

　クローズアップ現代の
　B’z 独占インタビューを視聴。


　（番組では紹介されなかった部分も含め、
　　全文の文字起こしが以下のサイトで
　　掲載されています）

　https://www.nhk.jp/p/gendai/ts/R7Y6NGLJ6G/blog/bl/pkEldmVQ6R/bp/pYjllzG99b/



■思わず笑いがこぼれてしまう瞬間が
　随所にある、

　そんなインタビューでしたが、
　中でも印象に残った部分がいくつかあり、

　それらを以下にご紹介。


　（ここから）
　-----------


　おふたりにとってライブは
　どういうものでしょうか？


　稲葉：

　基本的には音楽をまず作って、
　CDや配信で世間に発表するのが
　スタートになるんですけども、

　結局ライブという生の現場が、
　聞いてくださる方と面と向かって、
　曲の熱量を直に確認できる場所なので。


　僕らの場合はそれをやることで
　次の作品を作るエネルギーにもなってきたし、
　そのサイクルでずっとやってきてますね。


　松本：

　僕はスタジオ作業も
　すごく好きなんですけれども、

　それって僕たちの一方的な思いで
　創っているじゃないですか。


　ライブだと生で演奏して、
　お客さんたちの反応を

　「この曲に対してはこんな感じなんだ」

　と見られるのは本当に良いですよね、
　次への活力にもなります。

　-----------
　（ここまで）


　私（鮒谷）もメルマガを書いて、
　一方的に配信しているだけではなく、

　コンサルや年間プログラム、
　セミナー等を通して、

　エネルギー交換をしている感覚が
　あります。



■インタビュー中にあったように
　ライブだからこそ、

　「熱量を直に確認できる」
　「次への活力」

　につながっていくところも
　共感できました。



■また、こんな話もありました。


　（ここから）
　-----------


　──時代と音楽の関係性というものを、
　　　どう考えていますか？


　松本：

　35年も同じメンバーでやっていますから、
　そうそう形は変わらないと思うんです。

　それに無理に流行を取り入れても、
　たぶん良い結果にならないと思うので。

　ある意味変わってないんじゃないですかね、
　僕らは。


　稲葉：

　僕ら自身は、めちゃくちゃ器用なわけじゃない
　ので、自分のルーツからスタートして、

　お互いに好きなスタイルを
　模索しながら作っています。


　時代によって
　音楽を聞く形態が変わっているので、

　「そこに合わせて僕らも」

　という風にできればいいんですけど、

　自分たちのやっていることがチグハグに
　なると本末転倒というところもあるので。

　-----------
　（ここまで）


　これもなるほど、その通りだなと。

　結局、自分の軸がぶれてしまったら、
　「チグハグ」になるんですよね。

　自分が何者であるかを
　理解するところから、

　持続可能な創作も発信が
　始まるのかなと。



■さらに、こんなやり取りも。


　（ここから）
　-----------


　──「努力」という言葉は
　　　しっくりきますか？

　松本：

　努力？…あんまり大変な思いをしてる
　感じは自分にはないんです。

　月並みですけど、
　継続は力なりというか。


　ほとんど毎年、
　ツアーとアルバムリリースだとか、
　ソロ活動をしている時期もありますけれど、

　互いに音楽活動を
　35年間休んだことがないですね。


　──その原動力は何ですか？

　松本：

　原動力ね。
　僕は好きなんだと思います、きっと。


　──稲葉さんは、トレーニングを
　　　相当激しくしていましたね。

　稲葉：

　僕も努力をすごくしている意識は
　あんまりないです。

　例えば、身体のコンディションを
　保つための努力はもちろんありますけど…。


　もちろん大変だと言われれば
　大変なところもあるんですけども、

　でも好きでやってることでもあるし、
　あんまり恥もかきたくない
　みたいなこともあるし（笑）


　──好きが大前提なんですね。

　稲葉　：そうですね、やっぱりそこは。


　-----------
　（ここまで）


　やっぱり、論語に出てくる

　「これを知る者は
　　これを好む者に如かず。

　　これを好む者は
　　これを楽しむ者に如かず。」

　は真実ですね。



■好きだから続けられるし、上達する。

　「努力」している時点で、
　方向性が間違っている、

　のかもしれません。



■幸いなことに私（鮒谷）も、

　B’zの35年に比べたら
　ひよこみたいなものですが、

　それでも20年間、一日も休まずに
　メルマガを書き続け、

　さらに話し続けてきましたが、
　そこに

　「努力している」

　という感覚は一切ありません。


　だからいろいろあっても、結局、
　休まず続けられているのかなと。



■メルマガを書いているときの思いも


　「もちろん大変だと言われれば
　　大変なところもあるんですけども、

　　でも好きでやってることでもあるし、
　　あんまり恥もかきたくない
　　みたいなこともあるし（笑）」


　と結構、近い感じです（笑）



■あと、これは声を出して
　笑ってしまったところ。


　（ここから）
　-----------


　──15年前の番組では、
　　　体を冷やさないように夏でも
　　　鍋を食べていることが放送されました。


　稲葉：

　そうなんですけど、別に我慢して
　鍋を食べているわけじゃなくて、
　好きだから食べているっていう（笑）


　番組のなかでは鍋を食べながら
　ウーロン茶を飲んでいたりするんですけども、

　そのあとツアー以外のときにお店に行って
　ビールとかを頼んだら結構驚かれるとか（笑）


　「飲むんですか？」って、なんかちょっと
　がっかりしたような感じで（笑）


　-----------
　（ここまで）

　最後のくだりなんか、
　笑わずにはいられませんでした（笑）



■以下は継続の力について。


　（ここから）
　-----------


　──それでも、基本的には維持しないと
　　　いけないという意識はあるんですね。


　稲葉：

　やっぱり創作するのは波があるというか。

　俗に言われるマンネリだとか、
　なかなか新鮮味がないなっていう時期は
　当然あるんですけども、


　それでも互いに努力して続けていると、

　「あれ！これ新しいね！」

　と思えることができたり、
　そういうことがあるんですよ。


　-----------
　（ここまで）


　これまた、めちゃくちゃ
　よく分かります。


　いつも同じ話ばっかりしか書けないな、
　という時期が長い間続き、

　それでも辛抱して、
　あれこれ異質のインプットを心がけたり、


　「待つ」ことを大切にして
　とにかく続けていると、たしかに

　「あれ！これ新しいね！」

　と思えることが生まれることが
　あるんですよね。



■このサイクルを何度も繰り返して、
　ようやく

　「踊り場の過ごし方」

　こそが大切だと
　理解できるようなったし、

　そこで焦らず、腰を据えて、
　時間を味方につけられるようになりました。



■それによって、

　以前より、ますます折れにくく、
　くじけにくくなった、

　ようにも思われます。



■創作についてはこんな感じみたいです。


　（ここから）
　-----------


　──楽曲を作るなかで
　　　「いや、ここは譲れないよ」という
　　　部分はあるんじゃないですか？


　松本：

　そこはね、
　僕たちはずっとやってきたんですけど、
　試しもしないで

　「いや、それは良くないよ」

　というのは、
　ウチは絶対ないんです。


　とにかく互いの意見を必ず形にして試して、
　そうすると、どっちがいいかって
　一目瞭然になるじゃないですか。


　どっちもいいよねという時は
　いろいろな人の意見も聞いてみたりとか、

　日をおいてから聞いてみたりとか
　いろいろしますけれども、

　必ず互いのアイディアは
　試して形にしてから決めますね。


　-----------
　（ここまで）


　試すことの重要性について
　語られています。

　「必ず形にして試して、
　　そうすると、どっちがいいかって
　　一目瞭然になるじゃないですか」

　これは心がけていきたいですね。

　ついつい、守りに入ってしまうことが
　ありますから。

■やっぱり、

　どんな分野であれ
　長く第一線で戦い続けてきた人の話は、

　含蓄に富んでいますね。

　いろいろと自分に引き当てて
　考えるところがありましたし、


　大切なことを再確認したり、
　新しいインスピレーションをもらえました。



■さらに、

　二人を間近に、リアルに感じ、
　研ぎ澄まし、気づきを得るためにも、

　（そしてなにより自分を開放して、
　　楽しむために）


　私（鮒谷）は８月に２回、
　35周年ライブ

　「B’z LIVE-GYM
　　Pleasure 2023 -STARS-」

　に参戦してきます。


　久しぶりのB’zライブ、
　今から楽しみです。


　今日も人生とビジネスを楽しみましょう！

　　引き当てて考える機会は、

　　大切なことを再確認させてくれたり、
　　新しいひらめきをもたらしてくれるもの。

　■そうした人の話を聞ける機会があれば、
　　積極的に求めていきたいものである。
