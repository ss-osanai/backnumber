　舞台役者・声優としての活動を経て、
　ファッションの世界へ。

　　すべてのビジネスは変わる」

　ファッションビシネス科講師を務める。

　TBS や MXTV 等、
　メディアへのスタイリング提供も多数。

　阪神百貨店梅田本店で
　定期的にスタイリングイベントを開催。

　ニュースメディア「NewsPicks」では
　プロピッカー、トピックスオーナーとして活動。

　企業の顧問スタイリストとして社員の
　ファッション研修や印象改善などにも関わる。

　参加 3 ヶ月で閲覧数歴代２位の記事を執筆。


　またフランス大使館でのセミナーへの登壇、
　フランスブランドのスタイリングを行うなど
　パリとの縁も深い。

　ファッションの力で人やサービスを
　ブラッシュアップすることを目標に
　活動を行なっている。

　【あなたのビジネスを成功に導く、

■その昔（かれこれ十数年前）、
　ある人から

　「自分を変えたければ
　　思い切って服装を変えよ」

　と言われたことがあります。



■自らの自己認識を変え、
　殻を破り、

　新しい自分になるための
　最も手っ取り早い方法が

　「いつもと
　　まるっきり異なる服を着る」

　ということでした。



■この話を聞いたとき、

　「たしかにそういった側面は
　　あるかもしれない」

　と思ったんですよね。


　
■ただし、思ったはいいけれども
　実践しなければ、

　何もしないのと同じですし、
　現に何も変わりませんでした（汗）



■そんな中、今年に入ってから

　（あくまで人と会うときだけですが、、）

　着る服を大きく
　変えるようにしたんですよね。



■そうしたら確かに変わるんですよ、
　意識が。

　普段から服装に気をつけられ、
　さらにはお金をかけている人からすれば

　「常識」

　なのかもしれませんが、
　

　（服装に限りませんが）
　いかに生活にカネをかけないか、

　を追求してきた私（鮒谷）には
　新鮮な発見であり、気付きでした。



■そして十数年ぶりに思い出したんですよ、
　
　「自分を変えたければ
　　思い切って服装を変えよ」

　という話を。



■服装の内面に与える変化は
　思いのほか大きいものですね。

　実際にやってみて初めて、
　言われていたのは

　「こういうことだったのか」

　と体感できました。



■何ごとによらず、

　聞いたことがあるのと、
　知っているのと、
　
　さらには、
　体験して体感するのとは、

　まったくの別物ですね。



■もしあなたが、

　とにかくなんでもいいから
　自分を変えたいと願われるなら、

　あるいは、

　着る服で内面に大きな変化が
　与えられるのは承知しているけれども、

　具体的にどんな服を着たら良いのか
　分からない、

　などといった疑問があるならば、
　ぜひセミナーに参加してみて下さい。



■どんな対象であれ、

　少しでも興味を持った対象については
　どんどん試してみることで、

　将来において活きる、

　知識や経験や知恵や教訓、
　気づきや学びが手に入るもの。

　

■これが複利で効いてくるから、

　一事が万事、チャレンジする人と、
　しない人の差がどんどん開いていき、

　やがては取り返しがつかないくらいの
　広がりになるわけです。

　

■そんな真実に改めて思いを馳せる
　きっかけとなったのが、

　今回、講師にお招きする

　神崎裕介先生
　（オーダーメイド スタイリスト）　

　との出会いからでした。



■私（鮒谷）は、

　文章で第一印象を形成してもらう
　ことに力を尽くしてきましたが、

　神崎さんは服装で第一印象を
　マネジメントするプロフェッショナル。

■もともと、

　きちんとした服装を身に着けていた人は
　もちろんのこと、


　もしあなたが
　過去の私（鮒谷）と同じように（！）

　服装に無頓着で、
　それによって損をしてきた、

　あるいはそのあたりに自信がないから
　前に出ていけない、

　そんな自覚があるのであれば、

　今後の長い長い人生のためにも
　ぜひ参加して下さい。

　（事前のレジュメを拝見しましたが
　　かなりボリュームがある上に
　　密度が濃いです）

　この価格で参加できるのは
　本当に破格だと思います。

　　本当は２日間通しのセミナー受講ですから
　　
　　「48,000円（税抜）」

　　とする予定だったんですよね（泣）



■それはともかく、
　こういう機会でもなければ、

　普通の人は真面目にファッションやら
　自分の着る服に、

　がっつり時間をかけて
　深く学ぼうと思わないでしょうし、


　そのまま私のように
　５年、10年、20年、、、

　と時が過ぎ去っていき、

　あとになって、
　とんでもない機会損失になっていた、

　と気づくこととなるかもしれません。
