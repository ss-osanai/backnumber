━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

　【　あれから一年　】

───────────────────────────────────



■あれから一年が経ちました。


　その間、世の中にも、自身の身辺にも
　さまざまなことが起こり、
　
　これまで考えてもみなかったようなことを
　考える機会も増えました。


　（いつか書こうと思いますが、
　　大学院受験を決意する一因ともなりました）



■あの出来事によって、
　生きていくにあたって

　「可視化・数値化できる（経済的）側面」

　と

　「可視化・数値化できない（非経済的）側面」

　の二つとどのように関わっていくべきか、
　改めて強く意識させられたのです。



■可視化できるものばかりに焦点を
　合わせ続けていると、

　可視化されにくい（できない）側面を
　見落としてしまう、

　（そしていつか、手痛いしっぺ返しを
　　食らうこととなる）

　であろうことが、身体感覚を伴って
　想像できるようになったのです。


　
■といっても分かりにくいですね。
　

　言葉を換えると、

　
　「この世の様々な種類の価値に対し、

　　経済的な価値への置き換えを
　　（ほとんど無意識のうちに）
　　
　　習慣的に行なっていた」
　

　という恐ろしい事実を、

　震災とそれに伴う身の回りの変化を
　通して気づいた、

　ということです。



■ここでいう、

　「可視化・数値化できる（経済的）側面」

　とは主として金銭に換算できる価値。


　「可視化・数値化できない（非経済的）側面」

　とは金銭に換算できない価値。

　文字通り、プライスレスで希少な価値を
　持つものです。

　たとえば愛情や友情といった人間関係や
　豊かに過ごす時間などが該当するでしょう。
　


■今の時代においては、経済的、金銭的側面を
　無視して生きていくことはできませんが、

　金銭換算できない価値を無視して
　生きていくことはできます。

　（きっと永続しませんが、一時的には）



■お金は汎用性が高く、
　多くのものに交換でき、
　
　愉悦や快楽をもたらすための即効性を
　持っているがゆえに

　「中毒化の罠」
　「依存症」

　に陥る危険があります。



■その落とし穴に落ちてしまうと
　
　「可視化できない価値」

　を疎かにするようになるのです。

　短期的に「成功」したと言われる人が
　特に陥りやすいようですが。



■そうして人生まるごと、
　
　経済的な側面に囚われ、
　取り込まれてしまっても

　
　「可視化できない非経済的側面

　　（いつもメルマガで使っている言葉では
　　　無形の資産）」

　
　が完全に毀損されていることに
　気がつくのは一定の時間が経過してからのこと。

　失っていることが分かったときには既に手遅れ、
　修復不能となっているケースが多い。



■もちろん生きていくには、
　経済的側面を無視して生きることは
　できません。


　しかし、

　世の中にあるものの価値を図る
　単一の尺度、唯一の指標として

　経済的側面のみを用いるという哲学で
　生きていると、

　いつか取り返しのつかないことと
　なりそうです。

　
　そんなことを自らに問い直す契機となったのが、
　先の震災でありました。




　今日も人生とビジネスを楽しみましょう！
