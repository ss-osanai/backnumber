【進化論】（あと18時間）人は誰でも「自分は他人とは異なる存在である」と思いたい

2019/09/08 05:00 
■おはようございます、鮒谷です。


　私（鮒谷）は、


　人間には誰でも、

　【自分は他者とは異なる存在である】

　と思いたい、


　そんな潜在的な欲求を持っていると
　考えています。

　もちろん、私もそう。



■「他者とは異なる存在」

　とは、
　
　「人と異なる思索をして、
　　人と異なる行動をしている人」

　のこと。


　人と同じものを見聞きして、
　人と同じものを体験し、

　人と同じことを考えて、
　人と同じことをして、それでいて

　「他者と異なる存在」

　になれるはずがないのは
　子どもでも分かります。



■将来、自分がどんな存在になるのかは

　「今、何に触れて、何を思い、何を考え、
　　どんな行動をしているか」

　によって規定されるのです。
　


■だからこそ、

　人とは異なる多様な、
　そしてたくさんの経験をして、


　自分の脳内において高頻度で
　
　【核分裂・核融合、からの核爆発】

　を起こし続ける必要が
　あるのではないでしょうか。


　（もし、あなたが、

　　「他人とは異なる存在」

　　になりたいと
　　願うのであるならば）



■私（鮒谷）は
　こうしたことを意図的に行い

　【新しい概念や表現】

　を生み出しては記述し続け、
　その集積によって


　【オリジナルな思考の体系を確立する
　　（＝その総体が信念体系となる）】
　
　ことを意識して行ってきました。



■日々、大量の新しい概念＆表現を
　量産し続けていると、やがて、


　「脳内に存在している概念が
　　細分化（精緻化）されたり（＝核分裂）
　
　　ぶつかりあって、より飛躍した
　　別種の概念へと昇華したり（＝核融合）」


　することなり、

　【時間をかけて、頭の中の配線構成が
　　作り変わっていく】

　こととなるでしょう。


　さらに長い時を経て、それは

　【全く別の脳】

　へと進化をとげることとなるのです。



■そのプロセスを、私（鮒谷）は

　（過去、お読みになられたり、
　　聴かれたりされた方もあると思いますが）

　【術 → 説 → 論 → 学 → 道】

　という表現で表してきました。

　（詳しいことは今日は記しませんが）



■こうした思考体系の総体（＝信念）が
　
　「時々刻々の行動」

　を生み出すこととなり、最終的に

　【知行合一】

　のレベルに到達すると、


　それはもはや

　【（生きる）道が確立された】

　といっても差し支えないでしょう。



■ここに至って
　自信や自尊心が確立されるし、

　他人の言葉で心が
　千々に乱れることもなくなり、

　自らの道を貫き、生きていける
　ようになるのです。



■ありきたりではありますが、
　耳障りの良い言葉で表現するならば

　【軸を持つ】

　【ブレない生き方を確立する】
　
　ということになるのかも
　しれませんね。



■他者との違いを
　明確に意識できるようになり、

　その状態に誇りを持てるように
　なるために欠かせぬことは


　【日々、大量の刺激を受け、
　　大量に思索＆行動し、
　
　　そこからの大量の閃きを、
　　大量に記述し続ける】


　ことではないかと考えています。



■それによって

　【自分オリジナルの言葉】

　も生まれてくるし、
　

　自らの内側から湧き上がる言葉は
　自身の行動を決定づける

　「意思決定基準」

　となっていくでしょう。



■こうした

　【自分独自の意思決定基準】
　
　が増え、
　さらには精緻さも増していき、

　それらが

　【集積され、
　　閾値超えを果たしたとき】

　
　まさにそのときこそが、

　外部からみても、それと分かる

　【自分流の生き様】

　が確立されるとき、

　と言えるでしょう。



■別にそんな生き方、生き様に
　憧れることもないし、興味もない、

　というのなら、ここから
　先の話は読まなくても結構です。


　でも、あなたが心のどこかで、
　ほんの少しでも、
　
　そうした生き方に
　惹かれるものがあるならば、


　ぜひ、その世界に至るための
　
　【最強の手段たるところの、
　　日記（正確には「時々刻々の記」）】
　
　に意識を向けて頂きたいのです。



■これまで、さんざん、

　【日々の気づきや学びや発見や決意を
　　時々刻々に記述し続けること】

　の価値をお伝えしてきましたが、


　これが本当に腹落ちできれば

　「日記を書かない人生」

　など、まったく想像できなくなります。

　

■正直に告白すると、


　日記を書かない人は、普段、
　何を考えて生きているのだろう、

　日々の貴重な学びや思考の
　ハイライト、さえも記録に残さず、
　
　時の経過とともに、流れ去っていくのを
　もったいないと思わないのだろうか、


　と考えることが
　しばしばあります。



■日記放談会の音源をお聴き下さり、
　日記を書き始められた方が、

　数え切れないほど、同じことを
　言われるのを耳にしてきました。


　過去を振り返ってみると
　何も蓄積してこなかった、

　そんな

　「無為に感じる時間」
　
　を長く過ごしてこられた方で
　あるほど、

　
　「日記を書くことによって
　　得られる価値」

　「日記を書いてこなかったことに
　　対する今さらながらの後悔」

　
　を強く感じていらっしゃるように
　思います。



■私（鮒谷）自身、
　日記を書いていなければ、

　今、このような仕事をしていることなど
　絶対にあり得ませんでしたし、


　今頃、どこかで野たれ死んでいたと
　しても不思議ではない、

　くらいに思っています。



■もちろん、ここまで記してきた話は

　信じてもらっても、
　信じてもらえなくても、


　（私（鮒谷）の人生とは
　　何の関係もありませんし、
　
　　嫌われるのも
　　本意ではありませんので）


　どのような解釈をしていただいても
　まったく問題ありませんが、


　既に日記を適切な形で
　記述し始められ、

　自らの人生を
　作り変えていらっしゃる方、

　であるならば、完全に同意くださる
　のではないでしょうか。



■私（鮒谷）が、

　考え、実践し続けてきた
　日記の書き方の集大成たる

　【日記放談会音源（本編）】

　ならびに

　【日記放談会音源（Ｑ＆Ａ編）】

　【本日の23時59分
　　（あと18時間）】

　【30％引き】

　でお求め頂けます。
