【 （あと５時間）チコちゃんに罵られたくない人は… 】

───────────────────────────────────



■いつも書いてきたことですが、
　

　（人生を完全に理想通りに
　　コントロールすることなど、

　　できるはずがないのは
　　いうまでもないにせよ）


　それでもなお、


　「こうした状況のときには、
　　こういう風に意思決定すると、

　　確率論的に（期待値を考えると）
　　より良くなる可能性が高い」


　と思われる、そんな

　【良質な意思決定基準】

　をたくさん持てば持つほど、
　
　目指している人生に（傾向としては）
　近づいていくのではないか、

　と思っています。



■もちろん、

　そうしたルールや法則が
　どんな場面においても絶対に機能する、

　などというつもりはありません。


　ただ、

　「何も考えることなく
　　テキトーに意思決定する」

　よりは、より良い方向に展開していく
　可能性が高まるのは間違いない、

　と考えているのです。



■たとえば、

　何か（大小の）トラブルに
　巻き込まれるたびに、


　「私はダメだ」
　「自分は愚かだなあ」

　「何かするたびに事件が起こるのであれば、
　　首をすくめてじっとしているのが一番」

　「あいつを恨んでやる」
　「こいつを呪ってやる」
　「どうやって復讐してやろうか」


　といった問いを自らに投げかける
　（という意思決定を行う）

　と、その人の人生はどうなるでしょうか。



■おそらく、こうした

　「自らの価値を低く見積もる」
　「行動しないのがベスト」

　あるいは

　「他者への怒りや憤りや恨みつらみを
　　心の中で繰り返す」

　といったことを重ねているうちに、


　気づかぬところで自らの心が蝕まれ、
　前に向かって挑戦する気概も失われ、

　結局、その人の人生は、時間が経つにつれ
　より悪くなる（可能性が）高まるのでは、

　そんな風に考えています。


　
■そうならぬように
　問題が生じるたび、

　「来た！トラブル＝成長の機会、
　　だからラッキー」

　と（無理やりでもいいので）
　考える癖（意思決定基準）を持てば、

　
　面倒事に巻き込まれるたび、

　「来た！トラブル＝成長の機会、
　　だからラッキー」
　
　「来た！トラブル＝成長の機会、
　　だからラッキー」
　
　「来た！トラブル＝成長の機会、
　　だからラッキー」

　と考えながら、


　この言葉を前進するための杖として、
　学び、改善を加え続けることによって、

　【より戦闘力の高い
　　（＝変化対応力のある）人間】
　
　へと成長していけることでしょう。



■あるいは、

　「周囲の人は私に気遣うべき」

　と思っていて、その前提でさまざまな
　行為を行う（意思決定をしている）人は、

　思い通りにいかぬのを他の人の責任に
　転嫁し、疎んじられたりするものです。



■反対に、


　「そもそも周囲の人に気遣いを
　　求めても仕方ない

　　（自分中心に考えて振る舞うのが
　　　人間なのだから、期待しない）。

　　ただし、私は可能な限り、周囲の
　　期待を越えるための行動をしよう」


　といった意思決定のルールを持ち
　生きている人との間では、

　やがて大きな差がつくように考えるのは
　私（鮒谷）だけでしょうか。



■上に上げたような
　汎用性の高そうな意思決定基準から、


　もっと細かな（具体的な）ところでは


　「同行の年配の方が重たいカバンを
　　持たれていたら、必ず

　　『おカバン、お持ちします！』

　　と声をかけ、実際にカバンの取っ手に
　　強引ではない程度に手をかける」


　「その後の反応を伺い、

　　任せて頂けるようならカバンを持ち、
　　結構だよ、と言われたらスッと引く」


　という意思決定のルールを持っていると
　するならばどうでしょう。



■こうしたルールを発動することによって、

　もちろん人によっては、
　こうした行為を不要と思う人もあります、

　（でも、それを弁えた上で振る舞えば、
　　不快な感情はもたれない）


　が、その逆に、
　こうした言葉を発することによって

　「彼（彼女）は気が利くなあ」
　「この人物には好感が持てる」

　という人がいらっしゃるのも
　また間違いないこと、

　ではないかと思います。


　
■とするならば、


　人生、長い目で見たとき
　年長者との同道という機会に、
　
　少ない人でも数十回、
　多い人だと数百回あるいはそれ以上、

　遭遇するわけですが、


　こうしたルールを持ち、
　行動していると、

　その分だけ

　「好感残高（！？）」

　は積もっていき、

　ひいてはその蓄積が新しいチャンスを
　もたらしてくれる、

　ということもあるのではないでしょうか。



■こうした、


　【大小さまざまな意思決定基準

　（＝汎用性の高いものから、
　　　具体的極まりないルールまで）】


　を数十個、数百個（あるいはそれ以上）、

　頭の中に叩き込み、さらには
　無意識レベルで実行できるようになった人と、


　こうしたことを学ぼうともせず、
　思いつきで生きている人と、

　果たして同じような人生になるでしょうか、
　と問いかけたいのです。



■ちなみに、

　数日前よりご案内してきた

　【（3日間セミナー参加者向けに案内していた）
　　フォローアップ音源・全6巻セット】

　の音源には、


　これまで膨大な学びを重ね、
　さらには実践してきた中で、

　私（鮒谷）自身、どのようにして
　自分の人生を作り変えてきたか、

　を振り返り、中でも


　【人生を逆転させるために役立った
　　大小合わせて100個単位の意思決定基準】


　について、

　16時間以上に渡って、語り続けた
　セミナーの内容を収録しています。



■なので正直に言うと、
　定価であるところの

　「76,800円（税抜）」

　ですら、破格の値段と思っているのに、


　何を血迷ったのか半額の

　「38,400円（税抜）」

　にてご提供するなど、

　ボランティアか、
　はたまた、狂気の沙汰か、

　と自分でも思われるくらいです。



■でも、一度ぜひ、聴いて
　もらいたいんですよね、

　私（鮒谷）の「作品」を。


　私（鮒谷）の、
　ある時点における
　
　「こうした意思決定基準を持ち、
　　地道に生きていくと、

　　必ず、人生は
　　こんな風に好転しますから」

　という実践研究の発表の形で
　世に問うた「作品」です。



■ここまでの話を読んで
　

　「そこまで言うのなら、
　　一度、聴いてみるか。

　　考えてみれば、音源を聴き、生涯を通して
　　大きな波及効果をもたらしてくれる、

　　ほんの一つか二つの、
　　質の高い意思決定基準（＝行動規範）
　
　　を知り、実践するようになれば、

　　あっという間に、この程度の金額の
　　元は取れるにきまっているじゃないか」


　と思われた方があれば、
　ぜひお求め頂ければと思います。



■反対に、

　「意思決定基準とか、
　　わけわからんし、
　　どうでもいいわ（ハナホジー）」

　という方にお求め頂く必要は
　一切ないと思っています。


　価値を理解下さる方のみに、
　お聴き頂ければと思っています。

　私（鮒谷）の大切な「作品」ですから、
　鑑賞（？）に耐えうる方にのみ、

　鑑賞頂きたいと願っています。
