■（あと３日）原価のかからぬプレゼント＆商品である
　「言葉」を量産する　

───────────────────────────────────



■私（鮒谷）は、
　自分の専門分野の話であれば、

　丸一日だろうが三日だろうが、
　一週間だろうが、

　ぶっ通しで話し続けられる
　自信があります。



■実際に、

　毎月一度、開催している
　年間プログラムにおいては、
　
　朝10時から夜9時までの11時間、
　話し続けてきましたし、
　

　それ以前には三日間セミナーを
　行っていて、

　これも朝10時から夜6時まで
　３日間、ぶっ通しで話をして、

　プラス最終日に懇親会、

　という形で、


　時間にすると20数時間にわたり、
　レジュメもなく、

　ひたすら放談し続けることも
　行っておりました。



■他にも、
　
　朝10時から夜6時までの
　ワンデイコンサルを継続的に行い、

　あるいは最盛期には
　来る日も来る日も

　「２時間×４本
　　（最高は２時間×５本）」

　の個別コンサルを行っていたことも
　ありました。



■これだけ放電してもネタが枯れない
　（＝鮒谷が放談し続けられる）

　のは、ひとえに

　「毎日のトレーニング」
　
　あったなればこそ、
　と思っています。



■日々、

　人と会ったり、本を読んだり、
　ぼーっとしている時に閃いたり、


　あるいは、

　人に話をしているときに
　これは！というネタが降りてきたら、

　即座に、その内容の骨格を
　書き留める（メモをする）、


　それを時間のあるときに
　（もちろん、その日じゅうに）

　それなりにボリュームのある
　文章にまとめて記録＆保管する。



■毎日、これを淡々と
　繰り返すことによって、

　いつの間にやら、自在に
　話をできるようになりました。
　

　つくづく思うのですが、この

　「自在に話ができる」

　というのは、

　生きていく上において
　本当に大きな武器となるのです。



■今、ご案内している

　------------------------
　
　【鮒谷周史の圧巻！
　　「ことばで未来を創造する技法」
　　放談会音源

　　～ことばを操り、自他に影響力を
　　　行使する「意思決定基準＆物語」の
　　　実践的、かつ、超効率的な量産法】

　------------------------

　の音源の冒頭でも
　お伝えしていることですが、


　適切な形式で整えられた言葉は、

　タダで人さまにプレゼントしては
　喜ばれたり、

　なんなら、

　お金をもらって販売することすら
　できてしまう、

　それくらいの価値があります。



■そして、

　喜ばれるプレゼントを作るのに
　必要な原価はほぼゼロ、
　
　あるいは、

　販売するために商品を作る原価も
　これまた、ほぼゼロ。



■そんなコンテンツをばらまけば、

　喜ばれたり、
　儲かったりいたします。


　おかげさまで私（鮒谷）は、毎日、
　多くの人に喜んでもらっているし、

　毎日、多くの方々から
　お金を頂戴しています。

　（皆さま、ありがとうございます）



■先日、

　メルマガ号外として一度だけ告知した
　弁護士・高井伸夫先生の勉強会、

　すぐに二十数名のお申し込みを頂き、
　翌日のメルマガで
　
　「締め切りましたよ」

　と言っているのに、
　まだダラダラ？とお申し込みが入って、


　結局、現状、私（鮒谷）からの
　たった一度のご案内だけで、

　30名以上の方が
　お越し下さることとなりました。

　いちおう12万円という、
　それなりの参加費なわけですが。


　メルマガ一本400万円弱の売上げです。



■もちろん、
　皆さんがお申し込み下さったのは

　高井先生の能力や経験や実績、

　それが一番の理由であることは
　もちろんですが、


　さはさりながら、
　メルマガを通して私（鮒谷）が、

　高井先生から学んだことを言語化し、
　メルマガを通して
　
　「言葉をプレゼントし続けてきた」

　から、ある意味、
　その返礼？として、

　こういった形でお返しを頂戴している
　のではないかしらん、


　などと思ったりもするわけです。



■こんな経験を20年近くに渡って
　行ってきたので、

　その方法論

　（プレゼントできる言葉の量産化
　　＆必要に応じてマネタイズの仕方も）

　をお伝えした方が、
　そのとおりに実践すると、


　たちどころに、

　その方の周りにいる
　多くの人が喜ばれたり、

　それがキャッシュを生み出す
　資源となったり、

　ということが起こるようになりました。



■当然のことだと思っています。

　皆さんが長い時間かけて構築してきた
　経験や能力、スキルやノウハウは、

　それだけの価値があるのですから。



■ただ、惜しむらくは
　それらの資源が

　「言語化されていない」

　すなわち

　「パッケージ化されていない、
　　標準化されていない」

　ことによって、

　潜在価値が顕在化されていない人が
　あまりにも多いのです。



■今回リリースした音源は、
　
　「自分が自分で感動する言葉を
　　量産する方法」

　についてお伝えしているわけですが、


　ひとたび、
　この方法をマスターすれば、

　「死ぬまでに価値の源泉を
　　量産し続けられる」

　ことになるでしょう。



■量産したコンテンツは、

　無償でバンバン、
　プレゼントしても良いし、

　有償で販売することだって
　できるし、


　私（鮒谷）のように、

　「両者をハイブリッドで行う」

　ことも可能になるでしょう。



■どんな環境を整え、
　どんなことに心がけ、何を行えば、

　そのようなことが
　可能になるのか、

　そしてそれが我々の人生に
　何をもたらしてくれるのか、

　当音源をお聴きになれば、
　その疑問が氷解いたします。



■ここまでの話を読んで

　「！！！」

　あるいは
　
　「？？？」

　となった方はぜひ聴いてみて下さい。


　きっと多くの気づき、学びを得られ、

　さらに、その世界に出ようという
　決意を固め、行動に移される、

　と思っております。



■もしあなたが、
　自分の能力を全開放させて、

　大量の価値あるコンテンツや
　表現を量産し、

　（これは経済的、非経済的な
　　報いをもたらし、
　　影響力の源泉ともなります）


　人さまに喜んでもらえるように
　なりたいと思われるのであれば、

　ぜひ、このたびの音源


　------------------------

　【鮒谷周史の圧巻！
　　「ことばで未来を創造する技法」
　　放談会音源

　　ことばを操り、自他に影響力を行使する
　　「意思決定基準＆物語」の実践的、
　　かつ、超効率的な量産法】

　------------------------


　を聴いてみて下さい。



■内容的には、私（鮒谷）としても
　満足いくものができあがりました。


　ただし、ある程度、
　高度な話をしておりますので

　「聴く人を選ぶ音源」

　であることは
　間違いありませんので、
　
　以下の方限定で販売いたします。



■具体的には

　「日記放談会音源」

　をお聴き下さった方限定、

■前置きが長くなりましたが、

　私（鮒谷）は長期に渡って、
　日記を書き、メルマガを書き、

　自分の人生を変革させ、他の方の
　人生変容のお手伝いもしてきました。



■そんな中で、

　これはもう間違いがない、
　と確信しているのが

　「未来はことばによって創造できる」

　ということ。


　その思いが強すぎて

　「ことば未来研究所」

　という会社を作ったくらい。

　ことばを作り、

　この具体的なプロセスならびに
　内面の状態について、
