＜今日のピークパフォーマンス方程式＞

【　そういう愛を、私は、持ちたい　】

───────────────────────────────────



■成果を上げる人となるには、

　「異常者」

　にならねばなりません。


　というよりも、より正確には

　「異常者でなければ、
　　成果を出すことができない」

　と言うべきかもしれませんね。



■敏腕経営者として知られる、元インテル会長兼CEOの
　アンドリュー・グローブ氏は、

　「偏執狂だけが生き残る」

　と言っていますが、

　それもまた、同じことかもしれません。

　
　
■成果を出そうと思ったならば、

　「選択と集中」

　またの名を、

　「廃棄と決断」

　を行わなければなりません。

　
　それは、必然的に、

　「異常者、あるいは偏執者を生み、育てる」

　ことになるのです。



■批判を受けることを承知であえて記すなら、
　もし、自分が今、

　「何かに対し、異常レベルで執着していない」

　と思うのであるならば、

　きっとその人は、思うような成果を
　出していない人でありましょう。



■私の知る限りにおいて、

　何らかの結果を残している人は、
　必ず、ある部分において、

　異常な側面があり、取り憑かれたように
　何かに偏執しています。


　それでいいのです。

　それくらいでなければ
　人を先んじるような成果を上げることなど、

　できるはずがありません。


　だとするならば、大切なのは、

　「異常を極め、異端になることを恐れない」
　
　こと。



■異常者になり、偏執者になり、異端者になる
　決意と覚悟ができ、

　「選択と集中」
　
　「廃棄と実行」
　
　ができた人から順番に、

　「成果を出せる人」

　となっていくことでありましょう。


　
■反対側からいえば、


　なにかの専門サービスを受けようとするのであれば、
　当該分野における　

　「異常者、偏執者、異端者」

　に依頼するのが間違いありません。



■なにせ、朝起きてから、夜寝るまで、
　一分一秒を惜しんで

　「偏執している世界」

　に捧げているのだから

　（だからこそ「異常者、偏執者、異端者」なのです）


　依頼する側としても、
　少なくともその世界について指南を受けるにあたって

　「絶対的な安心感」
　
　があるわけです。


　
■そもそも、

　他人さまから命の次に大切なお金を頂戴する
　プロフェッショナルであるのであれば、

　その分野における

　「異常者、偏執者、異端者」

　であるべきです。


　その認識を持っていただくためにも、
　自分がどれほど

　「異常者、偏執者、異端者」

　であるかを、延々と、

　１年でも２年でも３年でも、５年でも、
　10年でも、20年でも、30年でも、

　語り続けるがよいでしょう。


　それができるのが「愛の証」です。


　
■途中で断念し、語り続けられなかったのであれば、
　それは、その世界に対し、

　「至誠と愛を持てなかった」

　ということ。


　真に愛あれば、

　「何時間でも、何十時間でも、何百時間でも
　　何年でも、何十年でも、語り続けられる」
　
　はずなのです。


　そういう愛を、私は、持ちたい。
