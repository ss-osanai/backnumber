▼引き続き、英語の勉強

───────────────────────────────────



■こんにちは、鮒谷です。
　

　昨日は親類に勧められ、こんな本を読んでみました。

　
　【今日のお勧め本 最高齢プロフェッショナルの教え】

　 http://amazon.co.jp/o/ASIN/419863081X/2ndstagejp-22/ref=nosim


　世の中には80代、90代、そして100歳を越えても、

　現役でバリバリ活躍されている方が
　たくさんいらっしゃるんですね。



■以前も書いたことがありますが、


　＜3008号 非凡を平凡に行える人＞

　http://www.2nd-stage.jp/backnumber/ppf_full/2011/12/3008.html


  私は人生のピークは100歳（あるいはそれ以降）に
　来ればいい、と本気で思っているので、

　※促成栽培より、安定・逓増・右肩上がりが一番です

　こういった方々の存在・活躍は励みになりますし、
　イメージトレーニングにもなります。



■常にここに登場されるような方をイメージして
　長期視点で捉えれば、

　たかが5年、10年程度の遅れとか遠回りなんか
　はっきりいって

　「誤差」

　ですね。



■道を踏み外して10年くらい迂回していたと
　しても、全く問題なさそうです。

　100歳をピークにしようとおもったら
　たかが十分の一ですよ。

　現状の自分がどうとかこうとか、
　こういう人たちの事例を見たら吹っ飛びます。

　とにかく、一つのことを好きになり、
　続けることが大切なのでしょう。



■これからもこういう「人生事例」にたくさん触れ、
　自分の中にストックしていきたいものです。

　それが「思考、認識の枠組み」を提供してくれ、
　生きる力になってくれるように思われます。


　
■この件についてはもっともっと書きたいことが
　あるのですが、

　誌面にも限りがあるので、
　日を改めてお届けいたします。

　　
　元気出る本ですよ。

　【今日のお勧め本 最高齢プロフェッショナルの教え】

　 http://amazon.co.jp/o/ASIN/419863081X/2ndstagejp-22/ref=nosim

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　人前でビビらなくなる訓練　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。

＜今日のピークパフォーマンス方程式＞

  【　人前でビビらなくなる訓練　】

───────────────────────────────────



■昨日は

　「思考のシャドーボクシング」

　と題して、

　　
　「文章を書くことによって
　　当該テーマについての想定問答を行う」


　ことの効果をについて記しました。

　今日も続けます。


　
■特に不特定多数に向けて
　あるテーマについて文章を書くときには、
　
　さまざまな疑問・異論・反論が向けられる
　であろうことを前提として
　

　【 あらかじめ想定され得る疑問・異論・反論に
　　
　　　　　　　　   対する回答を文中に埋め込んでおく 】


　必要が出てきます。



■あるテーマについて、


　▼何も書かない
　
　よりも

　▼自分の思考記録を文字にして残す

　さらには

　▼不特定多数に自らの思考過程を公開する


　ことによって、上述の理由から

　「できるだけ多面的に物事を捉えなければ」

　という思いが強くなってくるでしょう。

　　　　

■この訓練を続けるうちに、
　
　「当該テーマについて、
　　さまざまな角度から眺める習慣」

　を知らず知らずに身につけることが
　できるはずです。

　
　たとえば私は、このメルマガで

　「伝わる文章を記すには
　　どうすればいいか」

　について、おそらく数百回以上、
　書いてきたとおもいます。



■その文章は、

　不特定多数の人の目に晒すことを前提として
　書いた文章ですから、

　読んで下さる方が増えれば増えるほど、
　大きなプレッシャーがかかることとなります。



■それだけの圧力がかかるからこそ、

　「より多方面からの想定問答を行う」

　必要がでてきます。

　
　（実際に投げかけられるかは
　　ともかくとして）

　さまざまな質問や異論、反論が返されることを
　意識しながら、文章執筆を行うわけです。



■そういった自問自答を繰り返しながら
　何百回という回数、

　「伝わる文章の書き方」
　
　について自分の考えをまとめていく過程が
　
　「思考のシャドーボクシング」

　となるのです。



■それだけやっても


　「思いもよらぬ方向からのパンチを受けて
　　よろめく、あるいは打ち倒される」

 すなわち、

　「そういう考え方があったんだ」

　「そういう風に捉えられる方も
　　いらっしゃるんだ」

　「これは完璧に私の認識違いだった」


　ということが、しばしば起こります。


　
■けれども、このプロセスを反復して繰り返す
　ことによって、

　「伝わる文章を書くには、というテーマにおける
　　大きな穴、中くらいの穴は潰せたかな」

　すなわち、
　
　「頻繁に聞かれる、あるいはよく問われる
　　質問については、予め返答を用意しておける」
　
　というくらいのレベルには至るように
　なるのです。


　
■そうして抜け・漏れの穴はゼロにはできないまでも、
　どんどん小さくなっていくのです。
　
　このプロセスを繰り返すうちに人前で話した後、
　公開環境で質疑応答をすることになっても、

　「ビビらなく」
　
　なることでしょう。


　なぜなら、

　あらかたの質問とそれに対する回答は、既に頭の中で
　何度もシミュレーション済みのことだから。

　

■即応が求められる公開コミュニケーションの場では、

　このシミュレーションが完璧に繰り返せていないと
　聴衆から

　「あ、こいつ、誤魔化しとるな」

　ということがすぐにバレてしまいます。



■そんなことにならないよう日常生活そのものを

　「常在戦場の精神」

　で過ごし、

　「思考のシャドーボクシング」

　を徹底的に繰り返しておくのです。


　
■その過程を経ることによって、

　はじめてリング
　（＝たとえばセミナーなど公開の場での質疑応答など)

　にあがっても余裕を持って、

　どんな質問が飛び出しても打ち倒す（！？）
　ことができるようになるのです。



■出たとこ勝負でリングに上がっても勝負になりません。
　
　「余裕、安心は、毎日の地道なシャドーボクシングから
　　もたらされる」

　ものなのです。


　
■もしあなたがなにかの専門分野でご飯を食べて
　いきたいと考えているのであれば、

　今から、その分野の情報について不特定多数に
　向けて発信していくことです。
　
　（たとえ辛くても。というか辛いですよ。
　　たくさん矢弾も飛んできますしね 笑）

　けれどもそれによって、どんどん思考は多面的になり、
　深く掘り下げられていくことになります。



■その修練の過程において徐々に厚みを増していく

　「想定問答集」

　は、

　いざ、あなたがその分野でご飯を食べていこうと
　したときの

　「門外不出の書」
　「秘伝の巻物」
　
　あるいは

　「絶対的自信の源泉」
　「圧倒的差別化の根源」　

　ともなっていることでしょう。
　

　
　今日も人生とビジネスを楽しみましょう！
