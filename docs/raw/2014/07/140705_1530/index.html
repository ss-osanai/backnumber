＜今日のピークパフォーマンス方程式＞

 【  「守破離」の脳内プロセス  】

───────────────────────────────────



■幼少期に刷り込まれたものは別としても、

　思春期以降に、自分を変えようと思ったならば、
　
　
　▼尊敬する人
　
　▼あこがれる人
　
　▼かっこいいと思う人
　
　▼目指すべき人
　
　
　に

　「脳内常駐」

　いただく必要があると思われます。
　
　
　
■これらをひとまとめに、ここでは、
　
　「脳内モデル」
　
　と名付けようと思います。
　
　
　「脳内にロールモデルを住まわせる」
　
　「脳内に住んでもらっている、
　　理想の人、あこがれの人」
　

　といったような意味合いで、あえて
　
　「脳内モデル」
　
　と、仮に（適当に）名づけた次第です。
　
　
　
■ところで自分を変えようと思うなら、
　常に自分の
　
　「怠けたい心」
　
　と
　
　「脳内モデル」
　
　を戦わせ続ける必要があります。
　
　
　
■自分の「怠け心」が強く、
　「脳内モデル」が弱ければ
　
　（初期の頃は大抵そう）
　
　自分の「ズボラ心」が勝ってしまい、
　行動を起こすことはできません。
　
　
　だからこの「脳内モデル」を、せっせと強化し、
　怠け心に打ち勝たせないといけません。
　

　この状態に入ったとき、
　脳内に現れている言葉としては、
　
　「この人（強い脳内モデル）のようになりたければ
　　こんなことをしている場合ではない」
　
　という言語が回っていることでしょう。
　
　
　
■そんな風に
　
　「脳内モデル」
　
　を強化しようと思ったなら、
　

　自分が「これ」と思った人と
　できるだけ長く接触し、
　
　この脳内の人物にどんどん強くなって頂く
　必要があるのです。
　
　
　弱い「脳内ロールモデル」だと、
　頑強で、怠惰な自分自身に打ち勝つことができません。
　
　
　
■この
　
　「脳内モデル」
　
　を強化するために最も有効なのは、
　繰り返しになりますが、
　
　「外部にいる現実のモデルとの接触時間を
　　極大化する」
　
　以外にありません。
　
　
　
■そもそもあなたは、

　脳の外（外の世界）にそういったロールモデルを
　持っているか、

　
　さらにそのロールモデルに脳内駐在してもらい、

　かつ、その人にもっと強くなってもらうために、
　
　「理想とする人」
　
　に長時間、触れる努力をしているか、
　
　考えてみられてもよいのではないでしょうか。



■これを繰り返していると、最初、脳内モデルは

　「一時派遣」

　だったのが、

　「長期派遣」

　に、さらに

　「短期駐在」
　
　「長期駐在」

　遂には、

　「永住」

　することとなり、

　やがて、気がついたら存在自体が当たり前になって、
　意識することもなくなっていた、

　という境地に入っていることでしょう。


　以上がいわゆる

　「守破離」
　
　の、脳内におけるプロセスです。
