　新しい世界が立ち上がる

　　　　　鮒谷周史（ふなたにしゅうじ）

━━━━━━━━━━━━━━━━━━━

＜鮒谷の活動ダイジェスト＞

■新たなシステム構築を開始します

───────────────────


■こんばんは、鮒谷です。


　食生活＆健康管理については、
　数ヶ月の試行錯誤の末、一定水準までは

　「システム」

　として自動で回るように
　なってきました。

　

■基本的な数値を血糖値センサーと
　体組成計で確認しつつ、

　食事や運動について変化を加えては
　観察を重ねて、PDCAを回す、


　そんな流れが定着し、
　習慣化してきたので、

　さらなる改善を図りつつも、
　次のシステムづくりに入りたいと思います。



■そんなわけで、
　次のシステム化ターゲットは

　【X（Twitter）アカウントの強化】

　です。


　先日、開催したセミナーで
　ご一緒する同志もいらっしゃるので、

　まず隗より始めよ、
　率先垂範、

　ということで、
　強烈な大量改善活動を始めます。



■皆さん、来月、
　晴れ晴れとした表情で会えるよう、

　いちばん厳しい、ゼロイチの時期を
　乗り越えたところにある、

　次のステージを
　いっしょに目指しましょう。

　
━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

■恥も外聞も捨て去るところから
　新しい世界が立ち上がる

───────────────────


■当面、Twitter運用に全力を尽くすので、

　メルマガはTwitterでのつぶやき
　（ダイジェスト）を中心に、

　お届けいたします。

　

■新しい行動を開始するときは
　躊躇しながらではなく、

　とにかく恥も外聞もなく　

　【振り切る】

　ことが肝要です。



■振り切っている間に、他人の目も
　気にならなくなっていくもの。

　中途半端な着手は、
　人目が気になったままになるから、

　最後までやりきれず、
　結局、モノにもなりません。



■以下は昨日、X（Twitter）でつぶやいた
　あれこれ（の一部）。


　まだまだ投稿しています。

　全文、読まれたい方が
　いらっしゃるようなら、

　ぜひご自身のTwitterアカウントから
　（なければ作って！？）

　ご覧ください。



■以下のような内容を、
　朝６時から夜12時まで

　「１時間に一本ペース」

　での配信を目指します。


　ツイッターアカウントはこちらです。
　フォローお願いいたします↓

　https://twitter.com/shinkaron



■フォローと合わせて、

　「リポスト（旧リツイート）」

　や

　「いいね！ボタン（？）」

　についても、恥も外聞もなく、
　お願いしておきます（笑）



■なんといっても
　Twitterセミナー参加者に、

　範を垂れなければ
　格好がつきませんからね。

　

■つぶやいている内容は
　以下のような感じ。

　これまでのメルマガの復習（？）や
　こうどうのきっかけ、着想ネタとしても、

　ご活用いただけるのではないかと
　考えております。



■どうぞお読み下さい。


　----------


　【動けば動くほど悪くなっていく人】

　「筋の悪い手」に力を注ぐほど、
　事態は悪くなっていきます。

　営業でいえば飛び込みセールスとか、
　電話営業とか。


　今の時代、そんなことは
　やればやるほど嫌われ、悪評が立ち、
　
　お客さんが離れていく上に
　自尊心まで削られる。

　まずは「筋の良い手」を学ぶことが先。


　----------


　【石にも目がある】

　目に沿ってノミを打ち下ろせば、
　大きな岩石も一撃で砕けるが、

　適当にノミを振るっていても、
　細かな破片が削り取れるだけ。

　労多くして功少なし。


　仕事も同じ。頑張ればいい、
　というものではない。

　「目」の場所を見つけるのが
　仕事の９割。

　あとはそこに一撃を加えるだけ。


　----------


　【馬鹿の一つ覚え】

　期待する成果が表れてもいないのに
　何も変えず、同じことを繰り返す行為。

　その努力は偉くもなければ尊くもない。
　資源の無駄使い。


　安定した成果をもたらす
　資源投入の仕方を、

　実験を重ねて確立できた人だけに
　許されるのが「かしこの一つ覚え」。

　圧勝できるに決まっている。


　※ツイッターアカウントはこちらです。

　　フォローやリツイート、いいね！を
　　お待ちしております

　https://twitter.com/shinkaron


　----------


　【構造や流れにしたがう】

　流れに逆らうのは豊平川を遡上する
　サケに任せておけばいい。

　彼らはそれが仕事。


　どのみち我々は構造や流れに逆らえない。

　圧倒的な成果は、事象の構造や流れを把握し、
　味方につけることでしか得られない。

　見えている世界を具象ではなく、
　抽象で捉えましょうということ。


　----------


　【勝ちパターンは枯れるまで続けよ】

　過去、月商1000万円（ほぼ粗利）を、

　広告ビジネスやコンサル、音源ビジネス
　それぞれで短期に達成した秘訣。


　うまくいっても浮気しないこと。

　一度、確立した勝ちパターンは飽きずに、
　陳腐化するところまで全力投資。

　新しいことを始めたくなるのを
　グッとこらえて。


　----------


　【センスは量稽古から生まれる】

　大量の試行錯誤からのみ、
　因果に関する大量データが得られる。


　世の事象は変数が多く、
　因果関係が１対１対応しないので、

　膨大なテスト回数を経ないと
　因果の傾向が見えてこない。


　この過程を経て（一定程度まで）
　因果の関係性を掴めた人が
　センスある人と呼ばれる。


　※ツイッターアカウントはこちらです。

　　フォローやリツイート、いいね！を
　　お待ちしております

　https://twitter.com/shinkaron


　----------


　【勝てなくてもいいから、
　　負けないことを優先】

　ビジネスにおいては
　「負け確の手」をいかに避けるかが
　必勝の一手を探すよりも優先される。


　必勝パターンを探している間に
　敗着確定の手を選んだら次の瞬間、
　ゲームオーバーとなるから。

　決して負けないように。
　その上で勝ちにいく。

　それがビジネス。


　----------


　【夢中になれない対象は、
　　モノにならない】

　猿のマスターベーション並みに
　中毒化が進み、

　没頭できる分野だけがモノになり、
　そこに勝機が生まれる。


　「これを知る者は
　　これを好むものに如かず。

　　これを好む者は
　　これを楽しむ者に如かず（論語）」

　義務感で取り組んでいる対象に
　未来は開かれない。


　----------


　【「夢中」からしか、　
　　創造性は生まれない】

　夢中で取り組む＝当事者意識を持ち、
　コトに当たるということ。

　100％、自分ごととなっている状態が
　没頭、没入。

　そこにたどり着いた人だけに
　無数のアイデアが降りてくる。


　義務感の人に着想は生まれない。

　夢中になれること以外に
　時間を使いたくない理由。


　※ツイッターアカウントはこちらです。

　　フォローやリツイート、いいね！を
　　お待ちしております

　https://twitter.com/shinkaron


　----------


　【夢中になれば、時間を忘れる】

　時間を気にしている限り、
　ポテンシャルは開放されない。

　我を忘れ、ひとたびゾーンに入れば、
　時間感覚がなくなり

　【集中力×創造性×長時間】

　のレバレッジが効いてくる。


　いやいや働く人と、
　楽しみながら働いている人、

　後者のパフォーマンスが
　圧倒的に高くなる理由。


　----------

　
　【仕事が遊びで、遊びが仕事】

　人生や仕事をゲーム化できれば、
　ストレスがなくなる。

　コツは「見える化（数値化）」にあり。


　障害を乗り越え、設定した勝利条件（数字）を
　クリアするゲームだと最初に定義する。

　時々刻々、数字を確認しつつ、
　創意工夫を楽しむ
　リアル人生＆ビジネスゲームとなる。


　※ツイッターアカウントはこちらです。

　　フォローやリツイート、いいね！を
　　お待ちしております

　https://twitter.com/shinkaron


　----------


　、、、まだまだ、
　大量につぶやいています。



■とにもかくにも、量稽古。

　必勝方程式（成果を出せるパターン）が
　見えてくるまでは、

　上述の通り、恥も外聞もなく、
　大量行動で突っ走ります。



■そんな試行錯誤
　（PDCAサイクルの回し方）も、

　このアカウントを追いかけていると
　体感いただけるかもしれません。



■これまでは
　妖怪人間ベム、ベラ、ベロのように、

　闇に隠れて、
　こそこそ生きてきましたが、

　これを機に「くらいさだめ」を吹き飛ばし、
　表の世界（？）に出ていこうかと。



■とりあえず、
　わけもわからず、

　とにかく、馬鹿みたいに動き、
　大量の発信を行ってまいります。

　ご期待（ご容赦？）下さい。


　※ツイッターアカウントはこちらです。

　　フォローやリツイート、いいね！を
　　お待ちしております

　https://twitter.com/shinkaron
　


■今日も人生とビジネスを
　楽しみましょう！
