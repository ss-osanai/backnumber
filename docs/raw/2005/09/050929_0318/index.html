　【　ビジネス界の「フェルマーの最終定理」はきわめてシンプルだった。 】

───────────────────────────────────


■今日までに、「ピークパフォーマンス方程式」、すなわちビジネス
　の世界において、最高の結果を叩きだす為の原理原則、方程式、

　をこのメールマガジン上で、一日も休むことなく、

　実に700以上、世に問うてきました。


■しかし、それらは私が勝手にひねり出したものではなく、


　＜ビジネス界のトップアスリートたちが実践している成功方程式に
　　私のつたない体験を加え、それを私の言葉で語ったものである＞


　と昨日お話ししました。


■世にビジネスが存続する限り、これからもさまざまの
　複雑な方程式が現れてくると思います。

　しかし私は最近、ずっと考えていたことがありました。

　すべての方程式には、なにか共通することがあるのではないかと。


■そしてついに、ビジネスの世界に携わる人、全員が知りたかったこと、

　いわばビジネス界の「フェルマーの最終定理」が解けたのです！


 （ご参考までに「フェルマーの最終定理」とは、

　　3 以上の自然数 n について、
　　x^n + y^n = z^n となる 0 でない自然数 (x、y、z) の組み合わせはない、
　　という定理のことで、300年間、誰も証明できずにいたものです。

　　ちなみに ^n は n乗ということ）


■誰もが知りたかった、ビジネス界におけるフェルマーの最終定理
　とは何か。


■先日も紹介させていただきましたが、作家の浅田次郎氏は、


　【運の存在を認めないことが運を支配する唯一の方法】


　であると言っています。


　「商売の成功と失敗をわけるのは絶対に運じゃない。
　　努力と才能ですよ。
　　
　　何事でも成功した人は運なんて関係なく、それなりの才能が
　　あって、それなりの努力をしているんです。（以下略）」


■しかるべき結果が欲しければ、しかるべき努力をせよ、
　ということですね。


　そんなの当たり前じゃないかと思われるかもしれませんが、

　その当たり前のことをおろそかにし、ないがしろにする風潮が
　近年特にひろがっているように感じます。


　そういう間違った風潮は、やはり断固として打ち破って
　いかなければなりません。


■みんな努力を嫌って何もしなくなったら、そもそも社会が
　成り立ちません。

　みんなが与えられた立ち位置で、お客さんなり、友人なり、
　家族なり、周囲の関係する人に何かを提供していく。

　それによって価値が生み出され、その価値の総和として
　社会が良くなっていくのです。


■「情けは人のためならず」といいます。

　ときどき、この言葉の意味を間違えておぼえている人がありますが、
　正しい意味は、


　　　人に情けをかければいつか巡り巡って自分が情けをかけて
　　　もらえるときがくる。

　　　だから、人にかけた情けはその人のためではなく、自分のためになる

　
　ということです。


■であれば、ケチケチしないで力いっぱい人に情けをかけたら
　よいのです。
　自分のできることを全力で提供すればよいのです。

　そうすることが、ひいては社会に価値を付加することになり、
　やがては自分に返ってくる。

　そうしているうちに自分の能力も高まるし、評価も上がる。

　世の中がよくなり、自分のところにも結果が返ってくるのです。


■本来、こんなことは言わずもがなの話なのでしょうけれども、

　そんな、言わずもがなのことがあまりにも忘れられているのが
　現代という時代ではないのでしょうか。


■チルチルミチルは、幸せの青い鳥を求めて旅立ちました。

　ところが、探し求めたあげくの果てに、青い鳥が見つかった場所は、
　ほかでもない旅立ったわが家の中でありました。


■同じように、探し求めたビジネス界のフェルマーの最終定理は、
　見つかってみればきわめてシンプル。

　それは証明のための長い行数を要するものでもなく、
　難解な数式でもない。


　　【 まいた種は、生える 】


　この世は努力が必ず報われる世界なのです。

　誰もが分かりきっているはずの、この、ごくかんたんな原理原則
　だったのです。


■細かい成功の方程式はたくさんあります。

　しかしこのことを忘れて、一切の方程式は意味をなさなくなる
　ことを、よくよく知るべきでありましょう。
