【  圧巻のセールスは、高度に洗練された「伝統芸能」のごとし  】

───────────────────────────────────



■芸は身を助ける、という言葉があります。

　営業力は、一芸の中でも、
　最も身を助けてくれる可能性の高いものの一つ、

　と言ってよいでしょう。


　昨日も書きましたが、

　何が起きても、すべてを失っても、
　徒手空拳から再起できる力、

　それが

　「圧巻の営業力」

　です。



■モノを売る力さえあれば、
　経済的にも精神的にも肉体的にも安定いたします。


　「何かが起きても、
　　何かを売ればいいだけ」

　という圧倒的な自信が身につくわけだから。


　そういった意味で、我と我が身を助けてくれる
　最強の武器となり得るのが

　「営業力」

　です。


　
■そして、以前も書いたことがありますが、

　圧巻のセールスは高度に洗練された、
　それ自体がある種の、

　「伝統芸能」

　に近い世界です。


　それゆえ１冊、２冊、本を読んだからといって
　身につくものではなく、
　
　芸を極めるためには、これはと思った、
　然るべき師に長年付き、一挙手一投足を観察し、

　ときに独特の言語で表現された世界を
　口伝で学ぶより他にありません。


　職人世界特有の、薫陶、薫育的学習によってしか
　身に付けられない性質のものなのです。


　
■だからこそ、私（鮒谷）自身も、

　師につき、側に仕え、長年に渡り、
　同じ空気を吸いながら、

　「伝統芸」

　を学んできました。


　学ぶだけではなく、自分なりに
　体系化ならびに言語化を図ることによって、


　▼たった一人で営業を担当し

　▼在庫を持たず

　▼すべて前金で頂戴し

　▼特定顧客への売り上げ依存はなく
　　（最大でも１％程度）

　▼粗利率が限りなく100％に近いサービスを販売し

　▼毎月毎月、安定的に

　▼４ケタ万円の売り上げ（＝ほぼ利益）を立て
　
　▼そのうち既存顧客からの売り上げは約半分

　▼ここ５年ほどの間に受けたクレームはゼロ


　というビジネスを一から作り上げました。


　さらには、数多くの個別コンサルを通して、

　「営業の体系化ならびに言語化」
　
　の精度を高めることにも注力、尽力してきました。



■このビジネスの

　（営業的側面から見た）

　理論的背景について、ぜひお伝えしたいわけですが、

　残念ながらそれはメルマガでは難しい。


　別にケチケチしているわけではなく、
　実際問題として、

　「営業の空気感」
　
　は、文字だけで伝えきることはできないので。



■空気感、臨場感とともに、営業の真髄をお伝えする。


　これが今週末（日曜日）に行う、
　
　＜少人数制 圧巻セールス入門セミナー＞

　を開催する目的です。


　この、すぐれて芸術的な

　「伝統芸能」

　に触れ、目覚め、極めんとされる方が一人でも多く
　現れ出んことを、

　それが私の願いです。
