■おはようございます、鮒谷です。

　引き続き、新しいプロジェクトに
　没頭中。



■あと、お気づきの方もあると思いますが、

　数日前からさりげなく（実験的に）
　メルマガの文体を変えています。

　

■一人称が「ワイ」とか、
　文末が「なんよ」になったりしてますが、

　こっちの方が精神を開放し、
　より率直に文章を書ける、

　（正直、こちらの方がリアルな私、
　　もといワイのキャラに近い笑）

　ので、しばらくこの運用で
　試してみようと思います。


　すぐに慣れると思うので
　お付き合いください（笑）

＜今日のピークパフォーマンス方程式＞

■最高のマーケティング/24時間、まるまる練習/
　2種類の自転車操業/「全てを帳消し」の満塁ホームラン/
　楽して儲かる/孟母三遷/エビで鯛を釣る

───────────────────


■今日も、

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　-------------
　
　最高のマーケティング、
　かつ参入障壁となるのは

　「人生を賭けている姿」

　を周囲に見せること。


　本気の3乗で取り組んでる姿を
　発信し続れば、独自世界が立ち上がり、

　必ず足を向けてくれる方が出てくるよ。

　これが至高のマーケティングであり、
　本物の差別化。

　----------

　「耳障りの良い、それっぽい言葉」を
　いくつか入手するだけで人生が変わる、

　とか大間違いやで。


　波及効果の高い言葉を数え切れんくらい持ち、
　それを臨機応変に活用できる。

　この2点をクリアして初めて
　人生が豊穣なものとなっていくんだわ。

　ショートカットは無理やから、
　時間が応援団になってくれるんよ。

　----------

　言葉を持たん人が
　成果を出せるはずがないんだわ。

　言葉を収集し、創造し、さらに自在に
　使いこなせるようになったとき、

　人生は一気に楽しく、
　軽やかなものへと変貌する。

　言葉は、ただの
　コミュニケーション手段ではない。

　思考を深め、可能性を広げ、目標を
　達成するための最強のツールなんよ。

　----------

　倫理・道徳を抜きにしても、
　悪口を言うのは止めたほうがいい。

　世間は狭く、
　いずれ当人の耳にも入るもの。

　悪口を言われていると分かって、
　なお好感を持ってくれる人など
　いるはずもないだろ。


　なのになぜ悪口を言うかといえば、
　愚かにも

　「ほんの一瞬だけ
　　自尊心を（擬似的に）高く保てる」

　からなんよ。

　----------

　「動きたいのに動けない」の解決策は、
　自身を人さまの監視の目に置くことよ。

　慣れぬうちは
　自分を他人の目に触れさせるのが
　ストレスに感じるかもしれんよな。


　けど威力を実感できるようになると
　ストレスは軽減され、
　やがて快感にさえ変わるのよ。

　自分を律する環境の活用が、
　継続成長の鍵なんやで。

　----------

　圧倒的な練習量が
　本番での自然体につながるよ。

　起きているときも、
　寝ているときさえ

　「24時間、まるまる練習」

　という没我の境地に至れたとき、

　緊張という言葉と
　無縁の世界に出られるで。


　意識せずとも体が動き、
　雑念なく集中できる。

　自分と一体化するまで訓練し続けた
　者だけが出られる特別な世界。

　----------

　売上げは、心と頭と体で立てるもの。

　売上げを上げるには、
　お客さまの役に立とうとする心を
　練る必要がある。

　同時に、顧客に価値提供するために
　知力や想像力、加えて行動力を
　総動員せねばならない。


　それなくして
　お客に喜んではもらえない。

　心で思い、頭で考え、
　体を動かして初めて

　売上げは上がるのよ。

　----------

　儲かってる経営者は

　「波及効果の大きな行動と
　　そうでない行動」

　を峻別し、前者に集中するために、
　後者への意識と時間の割り振りを

　（あえて）省略したり、
　無視することに長けとるのよ。


　それゆえ一見、いい加減な人間に
　感じることもあるかも

　（外から見ると
　　そこがイラつくとこでもある）

　やけど、

　真に成果をもたらす核心の部分に限っては
　徹底した集中とこだわりを見せるんだわ。


　同一人物の中に
　真逆の姿勢が同居しとるのは

　「長期視点で見た全体最適」

　を追求しているところから
　来るものなんやで。


　限られた時間と資源を効果的に配分し、

　価値ある決断に集中して大きな成果を
　引き出そうとしているということ。


　こんな風に考えたことのない人が
　ほとんどやと思うけど、

　チマチマした成果じゃなく、大きな成果を
　出したい人には参考になるはずやから紹介しとく。

　若干、難しい話に感じるかもしれんけど、

　使いこなせるようになったらマジで
　生産性、爆上がりするからな。

　----------

　時間は有限なんよ。
　その中で結果を求められる。

　全てをまんべんなく、丁寧に、
　とか不可能やで。


　大きな成果を上げたければ、

　必然的にほとんどを無視するか、
　最低限の対応にするしかないんだわ。

　真に大事なことだけに集中する
　（あとは全部捨てる）。

　それが真に大事なことやで。

　----------

　「良い自転車操業」と
　「悪い自転車操業」があるんやで。

　忙しくても充実感のない
　「悪い自転車操業」は

　疲れるだけで何も得られない。


　「良い自転車操業」は、
　予定ギチギチで活動してるものの、

　続けるうちに目に見えて
　人生が楽になっていく状態。

　こちらは未来投資として機能する、
　歓迎すべき自転車操業よ。

　----------

　優秀なサラリーマンと経営者は
　別の生き物やからな。

　求められる資質が異なる、
　もしくは真逆まである。

　そつなく、万事無難に、長きに渡って
　失点せず、がサラリーマンや公務員。


　経営者は大量失点後も、
　一打逆転満塁ホームランを放てば

　「全てを帳消し」

　にしてもらえる。

　弱者が起業に夢を持っていい理由。

　----------

　新しいことを始めるにあたって
　対象が積み上がるものか、

　積み上がらないものか精査することは
　決定的に重要やで。

　ただ頑張ればいいんじゃない。


　積み上がらない忙しさは
　悪い自転車操業。

　積み上がり、さらに複利が効く
　プロジェクトでスケジュールが詰まるのが、
　理想的な自転車操業よ。

　----------

　「楽して儲かる」という言葉を嫌う人
　多いが、単なる嫉妬やで。

　より効率的に価値提供できてる
　言い換えに他ならん。

　貶される筋合いなんかあらへんよ。


　大勢に、効果的に役立つシステムを作れば
　儲かるに決まっとる。

　「楽して儲かる」全肯定で
　何の問題もない。

　文句いう馬鹿には
　生産性向上と言っとけばいい。

　----------

　「文句いう馬鹿には
　　生産性向上と言っとけばいい」

　と先に書いたが、ただ本当に思ってる
　わけやないから堪忍な。


　とはいえ、

　楽して儲かるを毛嫌いしとるクセに
　生産性向上とか宣ってる、

　知的活動に携わる人ｗ

　が滑稽やなとは正直、思っとる。

　同じと分からん？

　----------

　言語化能力とは

　「高低差の大きな抽象思考」、

　「メタファー（比喩）創作能力」、

　「エピソードトーク
　（テーマに引きずり込むための、
　　面白おかしい具体例）の卓越性」

　の３つで構成されるんよ。


　それぞれの説明は省くけど、
　この掛け算で

　納得してもらえる話を通して、
　影響力を生み出せるようになる。

　----------

　経営者は

　「自分にしか、できんこと」

　以外の全てを
　アウトソーシングすべきなのよ。


　コスト削減できる等の理由で
　あれこれ自分でやってるから、
　いつまで経っても儲からん。

　たとえできたとしても、

　絶対に勝ち切れる「絶対領域」
　以外のことは絶対にしない、

　と決断するところから
　世界が動き始めるんだわ。

　----------

　経営者は

　「一を聞いて十を知る」

　人との時間を大切にすべき。

　そんな人と話をすると
　高高度の抽象世界で、

　ドッグファイト（空中戦）しとる
　感じになるんだわ。


　身体感覚だから分かる人には分かるが、
　分からん人にはまるで分からん世界やろうな。

　ただこの感覚を掴めんと、
　あるところから先には進めんよ。

　----------

　効率化は大事やが、
　真の効率化は

　「無用の用
　（一見役に立たんように見えて
　　重要な役割を果たす）」

　からしか生まれんよ。


　狭い観測範囲の中で模倣を重ねても、
　たとえは悪いが近親交配に過ぎん。

　全く別の世界との掛け合わせによる
　イノベーション（新結合）だけが、

　非線形の効率化を実現させるんだわ。

　----------

　どれだけ頑張っても勝てなかったのが、
　場所を変えた瞬間、

　適当にやってるのに大勝利、
　みたいなことがあるんだわ。


　世界は広い。
　人生はポジション選択が９割。

　特に凸凹差がデカいADHD体質の人は、
　これだけ考えればいいくらいの
　死活的テーマやで。

　凸でハメれば無双できる。
　ワイもそれで突き抜けたったわ。

　----------

　ワイもサラリーマン時代、

　客に怒鳴られ、
　出入り禁止を喰らいまくったわ。

　百人ほどいるフロア中に響き渡る怒声で、
　上司から叱責されたことも複数回。


　そういえば営業成績悪すぎて、
　日常的に蹴られてたな笑

　適材適所の意味は
　誰よりも身に染みて分かっとるつもりやで。

　----------

　孟子が子どもの頃、
　墓地の隣に住んでたら墓掘人夫のマネをした。

　母親がこれはイカン！と市場の隣に
　引っ越せば商人のマネを。

　これもイカン！と学校の隣に
　引っ越したら勉強し始めた。

　孟母三遷の由来。


　かくも人は環境の影響を受けやすい。

　変わりたければ、
　かくありたいと思う人や場に近づくところから。

　----------

　会ったこともないのに
　一発目のメッセージで営業してくる人おるけど、
　あれなんなん。

　出身地や出身校が同じです。
　元勤務先が一緒、共通の知人がいます。

　知らんがな。
　そんなん無限におるわ。

　挙げ句の果てに一切不要なサービス。

　調べてこい。
　仮説立てろよ。

　想像力のない人物には
　何をやらせてもダメやろな。

　----------

　エビで鯛を釣る、
　いうことわざあるけど、

　エビを付けんと針だけで
　鯛を釣ろうとする人がいるんだわ。


　もしそんな釣り人おったら
　頭イカれとるやろ。

　せめて先にエビを出せ、いう話やで。


　後で鯛をゲットできるなら
　安いもんやとワイは思うけど、

　エビすらケチって、もらうもんだけ
　もらおうとする人が多いんよ。

　----------

　というか下調べもせず、

　とりま数打ちゃ当たる、
　みたいなメールセールスしてる時点で

　客を客と思ってない、
　要するに相手を舐めてるんよ。


　舐めた態度は相手に伝わるから
　売れるはずがない上に
　永久にブラックリスト入り。

　なんならネットで回覧されるまである。

　コピペでのフォーム営業も同罪やで。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。

　
　今日も人生とビジネスを
　楽しもうやで！
