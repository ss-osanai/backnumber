■今日は

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------

　「これ、エロいやん！」

　って脳みそスイッチ入った瞬間、
　男子中学生はそれしか考えられなくなる。


　「ヤりたくてタマらんのう（脳）」

　に変わった瞬間、
　脳のプログラムが書き換わるんよ。


　男中生に限った話でなく、
　人生の中間目標は「やらされ脳」を、

　「よだれダラダラ○○中毒脳」に
　魔改造するところにある。

　----------

　右肩上がりの人生の秘訣、
　それは3つの時間の使い方にある。

　「目の前の作物を育て、
　　刈り取る時間」

　「未来への構想が湧くのを待つ時間」

　「その構想を実現するための
　　準備行動の時間」。

　この3つを、状況に応じてスムーズに
　切り替えられる人が、
　年々増えていく豊かさを手に入れる。

　これぞ人生の無段階変速やで。

　----------

　「これは絶対ウケる！」思って、
　渾身の力を込めて話したり書いたりしたんが
　空振りになること、あるやん。

　そんな経験を重ねて、
　人は大人になるんやろうな。

　要するに自分が固く信じてる認識（思い込み）と、
　他者評価はまるで異なる現実を学ぶ、いうこと。

　ホームラン狙ったのにキャッチャーフライ、
　とかザラにあるからな。

　----------

　人生はカーナビと一緒なんよ。

　「正しく」目標設定さえすれば

　旧式の、ちょっとバカなナビでさえ、
　目的地付近までは連れてってくれるやろ？


　人生も同じ。

　「人生ナビ」のメカニズムを信じて
　任せてみる。

　そしたら不思議と、行きたいとこまで
　たどり着けるようになる。

　これが目標実現の本質よ。

　----------

　目標の達成プロセスは
　実は「メカニズム」なんよ。

　つまり再現性がある、いうこと。

　離陸するたびに頭と体の理解が深まって、
　次はより上手く飛べるようになる。

　いわゆる「つよくてニューゲーム」状態やで。


　レベルを上げ、アイテムを手にした状態で、
　新たな冒険を始められる。

　これが経験と成長の本質。

　----------

　商売繁盛の秘訣はシンプルやで。

　「行列」と「お客様の声」の可視化。
　でも勘違いしたらあかんで。

　そもそも行列できるほどの品質と集客が
　伴わず、感想も寄せられんのやったら、

　可視化もへったくれもない。

　「廃業する」か
　「心を入れ替え、本気で仕事する」か、

　の二択だわ。

　----------

　「上から目線の愛なき言葉」と
　「雑に見えても愛ある言葉」の二種類があるんよ。

　いずれも印象に残るけど、
　受け取る側の気持ちが全然違う。

　ワイはいつも後者を発する人間で
　ありたいな。

　この違い、分かる人には分かるはず。

　コミュニケーションで
　大事にしたいと思っていること。

　----------

　本気になれる目標に出会うと、
　人は変わるんよ。

　「可能かな？」じゃなく
　「絶対に形にする」が脳内ループし始める。

　結果的に時間も肉体もフルベット。


　逆に心震える目標もないのに、
　頭でいくら考えても何も始まらへん。

　ためらいの輪廻から抜け出せへんのや。

　----------

　最強の目標の見つけ方、それは
　感性の赴くまま時を過ごすこと。

　「～せなあかん」から離れて、
　童心に帰る。

　すると不思議と
　本来の自分が目覚めてきて、

　「せずにはいられん」

　の感情が勝手に溢れ出す。

　そこでようやく
　自分のための人生が動き出すんだよ。

　これが最高の人生の作り方。

　----------

　「せなあかん」から始まる目標は、
　うまくいかんのよ。

　最善シナリオで人並み、
　ほとんどは苦行で終わる。


　大切なんは、心の底から「やりたい！」が
　爆発するまで感性を育てる時間やで。

　そこから始まる人生は
　「苦行やなくて冒険」になるけど、

　全てを諦めた瞬間、
　「死ぬまでデスマーチ」が約束される。

　----------

　マジで効率とか一旦、捨てたらいいと思う。
　人生を取り戻したいのなら。

　とはいえ現実問題、
　難しい人が多いのも承知してるけど。


　幸いにもモラトリアム環境にある人は
　遠回りを恐れぬ精神があってよく、

　その環境になければ自らの意思で
　サバティカル（長期休暇）を取得するしかない。

　----------

　ワイが行ってきたビジネス戦略。

　まず一山当てて、
　数年分の軍資金を確保。

　次に「確実に来る未来の市場」を
　リサーチして、バーンと全力投資。

　回収期に一気に刈り取りつつ
　システム化。


　浮かせた資源を
　「より先の確実に訪れる市場」に向け、
　再び全力投資。

　この反復で右肩上がりさせていく。

　これが無敗の人生＆商売攻略法。

　----------

　「毎日が最高」となる秘訣は
　シンプルなんよ。

　忙しくても「せずにはいられん」対象に
　傾ける時間を、

　投資信託の積立みたいに
　少しづつ天引きすることや。


　この時間の確保と蓄積がやがて実をならせ、
　心を開き放ち、不安や焦りを消してくれる。

　滋味豊かな土壌から、
　かけがえのない人生の花が咲くんやで。

　----------

　安易な「他人の人生コピペ思考」はあかんよ。

　たくさん動き、深く考え、
　ようやく見えてくる「私だけの人生」。

　その人生を生きる覚悟を決めた時、
　全てが変わり始める。


　他人の人生をモノマネしたところで
　満たされず、楽しくもならんよ。

　「たった一人の長期戦」を貫き通した先に、
　本物の充足が待っとるで。

　----------

　「閲覧注意の詐欺メイク動画」は学びになる。

　過去も現状も変えられんけど、
　見せたい自分に魅せる技術があるんだわ。


　キャリア形成も一緒やで。

　ありのままのダメ姿を
　我田引水的な物語として編集し直す。

　人生における「詐欺メイク」と
　言ってもええかもな。


　最初期の自分を
　包み隠さず見せること推奨。

　一切の嘘をつくことなしに、
　自らを輝かせられる。

　----------

　全力で生きられないのは、自分で自分に
　「呪い」をかけてるからなんよ。

　子どもがめちゃくちゃ行動的なんは、
　まだ呪いがかかっておらず、
　感情フルパワーで己を解き放っているから。


　大人も呪いを解けば無邪気さが戻り、
　好きなことに全身全霊で打ち込める。

　これが「童心パワー」の秘密やで。

　----------

　人の意識は真空を嫌うんよ。

　意識から「長期視点」「継続力」「胆力」
　が抜けた瞬間、

　「その場しのぎ」「目先の利益」「逃げ癖」
　が入り込んでくる。


　そうなると（投資家ではなく）投機家と
　同じ末路をたどることとなる。

　意識の空白は悪い方向に
　埋められていくことを覚えておくといい。

　----------

　何にでも、
　ええとこと悪いとこがあるんだわ。

　悪いとこばかり見て、
　愚痴っても運気が下がるだけ。


　それよりも、
　いいとこ探しの達人になればいい。

　学びが深まり、なにより人生が楽しく、
　気持ちよくなる。

　ええもんしか目に入らなくなるからな。

　視点を変えるだけで、
　世界の景色も変わるんや。

　----------

　植物も人間も同じやで。

　一度にドバッと水をやっただけで
　急速に育つなら世話はない。

　毎日コツコツ水をやり、
　生長に適した環境を作り続け、
　ようやく大輪の花を咲かせられる。


　短期での派手な成功物語に
　惑わされたらあかんよ。

　手入れを欠かさぬ庭師の如く、
　着実に自分を育てることやな。

　----------

　人生に良い影響を及ぼす言葉を
　いくつ持っているかが将来を決めるんよ。

　最初は他人の言葉の輸入でええ。

　
　そのうち、輸入した言葉を編集し、
　自分バージョンにカスタマイズ。

　最後は完全オリジナルの言葉を
　作り出せるようになる。

　そこまでたどり着ければ、
　もはや後戻りできんくらいに面白くなっとるで。

　----------

　魂を燃やせる対象がないと、
　たちまち退屈が襲ってくる。

　人間はそんな生き物。

　ちなみに「没頭できる対象を見つける力」は
　実はスキルやで。


　妄想を膨らませるところに始まり、
　新しい世界に夢中になれるところまで
　己を導く技術。

　技術だから獲得できるし、身につけたら
　人生は無限のゲームフィールドになる。

　----------

　「知識はあるのに結果を出せない人」
　には原因があるんだわ。

　他人の目が気になりすぎて、
　思い切った行動ができんのよ。


　でも本気で理想の未来に恋焦がれたら、
　人の目なんて気にならなくなる。

　「恋は盲目」いうやろ。

　そんな状態になった人だけが、
　蓄えた知識を武器に行動し、
　卓越した結果を出せるんよ。

　----------

　人生って実は綱引きゲームなんよ。

　「未来」と「今」で綱を引っ張り合っとる。

　そしてデフォルト設定では
　「今」が圧倒的に強いんよ。


　しかし「未来」を勝たせたいなら、
　援軍を送ればええ。

　援軍の正体は「想像力」と「妄想力」。

　両者が暴走し始めたら、未来チームの
　勝利は約束されたものとなる。

　----------

　「将来そうなる」と確信してる人だけが、
　そこに到達する。

　そうなると「知っている」から
　大量行動できるのよ。

　確信が現実を生み出す。

　理想の未来を

　「掲げてもない」
　「信じてもない」

　のに、頑張れるはずがないわな。

　目標の現実化は「現在から未来」でなく、
　「未来から現在」の時間の流れで起こるんやで。

　----------

　「未来、そうなると分かっている状態」を、
　現在の一つ一つの行動で回収していく。

　これが目標が実現されるプロセスよ。

　時の流れを逆に捉えている人が
　ほとんどやけどな。


　このメカニズムを腹に落とせた人から順に、
　人生変わっていくんだわ。

　未来は「願う」ものやなく
　「既にそうなると知っている」ものなのよ。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
