■こんにちは、鮒谷です。


　今日は午前中に個別コンサル１件。


　お昼は協会ビジネスモデルを提唱者の、
　前田出さんと久しぶりのランチ。

　いつもながら楽しい時間です。
　気がつけば、もう10年ほどのご縁となりました。


　ビジネスの基本について、
　改めて思いを致す貴重な機会となりました。

　前田さん、ありがとうございました！



■午後は高井伸夫先生の事務所にて、

　藤沢周平さんの番組を手がけられたこともある
　NHKのプロデューサーさんとご面会。


　先日、高井先生と一緒に
　鶴岡の藤沢周平記念館を訪ねたこともあり、

　久しぶりに藤沢作品に触れたくなりました。


　『蝉しぐれ』は言うまでもありませんが、
　高井先生の最近の絶賛作品は『三屋清左衛門残日録』。

　『わたしの藤沢周平』も是非、よろしければ。
　　


■その後、個別コンサル１件。

　Ｋさん、久しぶりにお会い出来て嬉しかったです！
　Ｍ＆Ａ関連の件、ご検討くださいー。


　夜は、待ちに待った

　『舞台 北斗の拳－世紀末ザコ伝説』

　http://www.clie.asia/hokuto_zakobutai/

　でした。


　アフタートークは武論尊＆原哲夫！

　武論尊氏いわく

　「（舞台登場キャラの）雑魚の最下層感たるや」

　「原先生は雑魚を書かせたら世界一」

　など、名言連発ｗ



■今日もおかげさまで充実の一日でした。

　ところで、このところの北朝鮮情勢が気になります。

　リスク回避のために北朝鮮の建国記念日の
　9月9日と、翌10日は東京を離れるつもり。


　本をたくさん持っていって、
　どこかの山中に引きこもる予定。

　為替も神経質な展開が続いてますね、、



■話は変わりまして、
　先日、ブックライター上阪さんの、

　【10倍速く書ける 超スピード文章術】

　セミナーを開催しましたが、

　参加くださった皆さまから
　熱い感想をたくさん頂戴しています。


　本日はそのうちの一通をご紹介。
　　

　（ここから）
 　-------------------------------------


　上阪さん、鮒谷さん、
　スタッフの皆様、

　先日はありがとうございました。


　初めてセミナーに参加して、
　「直当たり」の意味が体感出来ました。


　これまで音源をいくつか聴いておりましたが、

　実際にセミナーに参加することで
　上阪さんの表情や身振りを目の当たりにし、

　同じ部屋で空気を共有している感覚は、
　完全に音源とは別格でした。


　普段の職務で従業員に、

　スライドや紙資料を使って
　話をすることが多いのですが、

　中々こちらの意図が
　伝えられていませんでした。


　何かヒントが得られればと思い
　参加したセミナーでしたが、

　上阪さんのおっしゃる

　「真の目的」も
　「ひとりターゲット」も定めずに、

　資料やシナリオを作っていたことに
　気づかされました。


　これでは相手に

　「ベネフィット」

　を提供できるはずもなく、
　興味を持って聞いてもらえなくて当然でした。


　これを機に、新しい意識を持って行うプレゼンが
　自分でも楽しみです。


　文章を書く上での技術的なことを軸に、

　多彩なエピソードを交えながら、
　最終盤には

　「何かをすることは他人の役に立つ」

　「幸福とは他人の役に立つこと」

　といった人生論にまで昇華された講演は
　興奮を覚えるものであり、

　予定を上回る５時間以上の時間が
　本当に短く感じられました。


　交通費も含め、少なくない出費ですが、

　「お金で時間を買う」初めての体験が、
　今後病みつきになりそうです。


　今後ともよろしくお願い致します。


　（岡本昭夫さま）


　-------------------------------------
　（ここまで）


　岡本さん、ご感想ありがとうございました！

　（また、遠いところよりご参加くださり、
　　ありがとうございました！）



■お書きくださったとおり、

　（文章を書く上での）

　「真の目的」
　「ひとりターゲット」
　「ベネフィット」

　を意識するのはとても大切ですね。


　さらに

　「何かをすることは他人の役に立つ」
　「幸福とは他人の役に立つこと」

　といったお話にも感銘を受けました。

　文章を書くことで、

　「自他を豊かにできる」

　ことは、私も日々、実感しております。

　

■ぜひまた「直当たり効果の体感」のためにも、

　ご都合がつきましたら、
　セミナーに足をお運び頂けましたら幸いです。

　
　そしてもし、セミナー参加が難しい場合には、

　「次善の策としての音源！？」

　もご用意しておりますので、
　皆さまもぜひ、ご活用下さいませ（笑）


　
■現在、

　【10倍速く書ける 超スピード文章術セミナー】

　の音源は鋭意、編集中ですが、


　その前段とも言える、

　【ブックライター上阪徹さん音源＜聞く力＆書く力編＞】

　　https://shinkaron.buyshop.jp/items/5362691

　において基礎・基本を学んだ上で、


　近々リリース予定の音源内容（10倍速で書く方法）を
　聴き、実践し、身につけられれば、
　
　「圧倒的な伝達力を求めるビジネスパーソン」

　としてまさに

　【鬼に金棒】

　となるのではないでしょうか。




■それでは、

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。

＜今日のピークパフォーマンス方程式＞ 

【 「言うだけ番長」それは見苦しく、滑稽で、恥ずかしく、情けない存在 】 

───────────────────────────────────


　
■人に動いてもらおうと思ったならば、

　それ以上のことを自分がやっていることが
　前提となるでしょう。

　できもしないことを、
　人に伝えていても、何の説得力も生まれません。


　子どもに

　「勉強しろ、勉強しろ」

　「たくさん本を読め」
　
　といくら言っても、


　肝心の親が一切、勉強をしておらず、
　蔵書もなければ、本も読まず、

　ずーっとテレビを見ているばかりで、
　その子どもが勉強したり、本を読むはずがありません。



■子どもに、

　「しかめっ面をするな」

　「無愛想になるな」

　「笑顔になれ」

　といくら言っていても、肝心の当人が、

　しかめっ面、無愛想、笑顔なし、
　であれば、

　子どもが変わることなどないでしょう。



■子どもを部下と置き換えても、

　あるいは（たとえばコンサルタントであれば）
　お客さま、

　と置き換えても同じです。


　できもしないこと、
　やってもいないことを、

　子どもや部下やお客さまに言うな、
　
　ということです。



■人に動いてもらおうと思ったなら、

　それ以上のレベルで
　自分がやっていなければなりません。


　リーダーシップを発揮するためには

　「口先ではなく、
　　体を動かしていなければならない」

　ということです。

　
　言っている通りに、
　体を動かしていてこそ、

　説得力も影響力も生まれてくるでしょう。



■もし、

　自分の発言に耳を傾け、
　尊重してもらいたいと思っているのに、

　周囲から一向に相手にされないとするならば
　それはきっと

　「伝え方が悪い」

　といった些末な（！？）問題ではなく、
　そもそも

　「言うだけ番長」

　になってしまっているのではないか？
　と疑ってみる必要があるかもしれませんね。



■やるべきことをやってもいないのに、

　他人を都合よく動かそうと思っているところに
　問題があるわけです。


　「やってもいないこと、できてもいないことを、
　　偉そうにいうな」

　「おまいう（お前が言うな）」

　という、アレですね。


　だからこそ、率先垂範が大事。



■とはいえ、こうした実践は

　「言うは易く行うは難し」

　ですから大変です。

　
　こんなことを書いている私自身、

　自信を持ってリーダーシップを発揮できているとは、
　もちろん、思っておりません。

　（どころか、まだ始まってもいません）



■ただ、少なくとも、

　「口だけの人間にはならないように」

　とは思っています。


　ですから

　「誰よりも真剣に、真面目に、一生懸命に、
　　そして圧巻の成果を」
　
　と思っています。
　
　そうじゃないと自分が語る言葉に
　説得力が生まれてこないから。



■マネジメントにおいては、

　やることやってるから、言うことを
　聞いてもらえるようになるのではないでしょうか。

　
　商売においては、

　やることやってるから、売っているものを
　買ってもらるのではないでしょうか。


　今の時代においては

　「背中を見せる」
　「背中で語る」
　「背中で示す」

　といった考えは古いのかもしれませんが、

　私自身は、この姿こそが、
　リーダーシップの理想の姿だと考えています。



■この道は、

　茨の道であるし、
　永遠に完成のない世界ですが、
　
　目指すからこそ、一歩づつでも、
　そこに近づいていけるのではないでしょうか。

　
　お互い

　「言うだけ番長」

　だけにならないようにいたしましょう。


　
■「言うだけ番長」、それは、

　見苦しく、滑稽で、恥ずかしく、情けなく、
　陰口を叩かれる存在。

　そんな人間に私はなりたくないので、
　死ぬまで、我と我が身を躾けていくつもり。


　※修行の道は遠くとも、
　　千里の道も一歩から、ですから。
