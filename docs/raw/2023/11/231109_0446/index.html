■昨日のメルマガでは、

　これからの日本の将来に対する
　一つの見立て、可能性として、


　-----------------------
　　
　ある時点から、全体として
　底が抜けた感じになると見ており、

　そんな状態においても
　生き延びるためには、


　国に依存することなく、
　円に依存することなく、

　特定の組織やキャリアや
　職業や専門能力にも依存することなく、

　それらから切り離されても、なお、
　生き延びられる準備しておく必要があります。

　-----------------------

　という話をいたしました。



■もちろん、上記は、あくまで
　私（鮒谷）の主観に過ぎませんが。


　ただ、もしそんな未来が
　ほんとうに現出するのであれば、

　そのときに備えて、

　【サバイバル能力を
　　今から高めていく必要があるのでは】

　といった提案もいたしました。

　

■今日も続けます。


　私（鮒谷）は

　【サバイバル能力
　＝身軽さ
　＝変幻自在＆融通無碍に生きられる力】

　と言い換えられると
　考えています。



■環境の激変時においては、
　
　【身軽であるほど
　　生き延びられる可能性が高まる】

　のではないでしょうか。

　（逆に言うと発展期、安定期においては
　　その逆の戦略が有効であるとも）



■そんなことを考えたのは、
　先日


　＜NHKスペシャル
　　恐竜絶滅　ほ乳類の戦い＞

　http://amazon.co.jp/o/ASIN/B003QVV9HS/2ndstagejp-22/ref=nosim


　を見たから、ということも
　あります。



■この番組では、

　恐竜絶滅前から、絶滅時、
　さらにその後の哺乳類の繁栄について、

　CGがふんだんに活用されつつ、
　描かれていました。



■これを見て、

　【ある時点までの圧倒的強者が
　　環境変化によって、瞬時に滅亡し、

　　入れ違いに、弱者が繁栄する可能性が
　　現実に存在する】

　ことを改めて理解したんですよね。



■同時に、そのときの訪れに備えて、
　
　「弱者」

　は、どのような準備をしたり、
　心構えを持っておくべきか、

　についても考える機会となりました。
　


■これから起きる（かもしれない）、

　【隕石衝突クラスの、
　　大規模な社会変動】

　に際してのサバイバルにおいても、

　参考になるのではないかと
　思った次第。

　※ご興味ある方は、
　　ぜひ番組視聴されてみて下さい。
　


■番組によると、

　私たちの先祖は恐竜全盛時代、

　彼らに食べられないため、
　隠れるように生きていたわけですが、
　

　巨大隕石の衝突による恐竜絶滅後に
　生き残ることができたのは逆説的ですが

　「弱かったから」

　であったとのこと。



■隕石が激突することで、

　火球が発生し、
　森林火災や津波が引き起こされ、

　巻き起こされた大量のチリで
　10度ほども気温が下がり、

　それが何年もの間、続きました。



■その結果、

　植物が育たなくなり、大量の食物を
　必要とする大型生物が絶滅した、

　という流れが起きたわけですが、

　その変化が身体の小さな哺乳類にとって
　有利に働いたのです。



■この話から、

　個人であれ、組織であれ、

　【大変動の時代にあっては
　　（自重で滅びぬために）、

　　固定経費の極小化が
　　生存に有利になる】

　という教訓を導き出せるのでは、
　
　たとえば、そんな連想にも
　つなげられそうです。



■さらに、続きがあるのですが、

　恐竜絶滅後において、
　
　すぐに哺乳類の天下が訪れたのかというと
　必ずしもそうではなかったようで、

　鳥とワニとの間で、
　三つ巴の覇権争いを行っていたようです。



■そんな中でも、

　たとえばワニであれば、
　ワニは水中での生活に対応できるように
　
　【特殊化】
　
　していたが故に、

　ワニの仲間は水辺を離れられず、
　そこから生存領域を広げられなかった、

　とのこと。



■一方で、（ネズミのような）哺乳類は、
　
　「特殊化」

　していない、

　つまり、あらゆる方向に向けて
　進化するための

　【多様な進化を遂げるための可能性
　（柔軟性や選択肢）】

　を持っていた、と。

　それゆえ、環境に適応する形で
　各々の場所で自由な進化を遂げました。

　

■こうして生き延びた哺乳類同士、
　
　ここからさらに、
　過酷な生存競争を行うわけですが、


　力の強い動物に殺されないために、

　社会性を発達させ、集団を形成し、
　脳を極限まで発達させた有胎盤類、

　としてのヒトが
　現れることとなりました。



■ここまでの話をまとめると、
　要するに、

　哺乳類と、その末裔であるところの
　「ヒト」がここまで生き延びて、

　さらには繁栄できた理由は

　-----------------------

　１、小さかった
　　（必要とする食料が少なくて済んだ）

　２、特殊化していなかった
　　（多様な進化を遂げるための可能性）

　３、社会性の獲得と
　　　脳の極限までの発達

　-----------------------

　といった要素があったから、
　だったようですね。



■既存のルールがひっくり返り、

　新しい社会が勃興する際に
　私たちが生き残るヒントとして、


　以上、３つの要素は、

　【社会情勢が激変した際の、
　　未来を思索するためのアナロジー】

　として、

　何らかのヒントや教訓を得られるのでは、
　と思いましたので、

　こんな話を記しました。

　ご参考にして頂けるところがあれば
　嬉しく思います。
