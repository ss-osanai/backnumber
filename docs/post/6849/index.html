6849号 心身ともに豊かになる

■我田引水音源を聴かれたＫさんから
　ご感想を頂戴いたしました。

　（Ｋさん、
　　いつもありがとうございます！）


　（ここから）
　--------------------


　今回の音源は、

　自分に対しての赦しと、
　勇気を与えてくれた、

　私にとってはそんな音源でした。


　それによって、

　より重心低く安定した感情で
　日々を過ごすことができるかもしれない、

　そんな気になりました。


　誰しもが、他人と自分を比べてしまい、

　「できない自分」

　を責めてしまうことがあると思います。


　自分は、どちらかというと、
　ポジ思考を持っている方だとは
　思っているのですが、

　最近は、どうしてもできないことがあり、
　それに直面することで

　「できない自分」

　を実感してしまう、
　そんな事がありました。


　あまりに、そのようなマイナスの感情に
　出くわす頻度が多いため、

　いつしか、

　「あの人も、自分のことを

　　『できない人間である』

　と考えているに違いない、」

　と（そんなことはないにも関わらず）
　考えてしまうようになっていました。


　そのような状態は、
　QOLを下げてしまうことにつながり、

　客観的に見れば恵まれているが、
　主観的には貧している、

　そのような状態かなと思っています。


　そのような背景の中、
　今回の音源を拝聴させていただき、

　我田引水、という比喩の中に、すべての
　事象をポジに解釈するという意味があり、

　それによって、人生に対する認識を
　変えられる事を知りました。


　自分は、これまでそのような解釈を
　したこともありませんでしたので、
　目から鱗でした。

　今回も素晴らしいパラダイムを
　ご教授いただいたと思っています。


　もちろん、すべての事象に対して、

　様々な角度から我田引水が
　できるようになるためには、
　練習が必要だとは思うのですが、

　このパラダイムを頂けたことは、
　これからの人生をよりよいものにしていく
　第一歩になったんじゃないかと感じています。


　今回も素晴らしい音源
　ありがとうございました。

　よろしくお願いいたします。


　（マーケティング Ｄ・Ｋ さま）


　--------------------
　（ここまで）



■幸福感と感情の安定には
　密接な関係がありそうです。


　自分のことが
　外からどのように見えているかと、

　自身が幸福と感じられているか
　どうかは全く別物。



■それゆえ、外から見て分かる装飾、
　たとえば、

　立場とか、役職とか、評価とか、
　評判とか、所得とか、財産とか、、、

　これらはないよりはあったほうが
　いいものもあるかもしれませんが、


　究極のところ
　それらの有無よりも

　「どんな状況にあっても
　　幸せを感じられる力」

　のほうがはるかに価値があるのでは
　ないでしょうか。



■客観的には恵まれているのに
　不幸せな人と、

　恵まれているようには見えないけれども
　幸せを感じられている人、

　どちらがいいかといえば
　私（鮒谷）は後者のほうがいい。



■もちろん、

　さまざまな有形・無形の
　価値あるものに恵まれつつ、

　その上さらに、

　幸せを感じられる力を持っていれば
　いうことありませんけどね。



■我田引水とは、ここでいう

　「幸せを感じられる力」

　といって良いでしょう。

　我田引水力が高まるほどに
　幸福感も高まる、

　これは間違いないことのように
　思われます。



■ここまでの話は自身に対して

　「我田引水力」

　を働かせるということですが、


　他者に対して
　我田引水力を発揮すれば、

　もしそれがほしいと願うのならば

　立場とか、役職とか、評価とか、
　評判とか、所得とか、財産とか、、、

　といったものも、
　手に入りやすくなります。



■もちろん、

　これらがあれば、幸せになれる、
　というものではありませんが、

　得たいものが
　手に入りやすくなるとすれば、

　それはそれで、
　求めてみるのもよいかもしれません。



■いずれにしても

　「我田引水力」

　は、身につけておいて損はない、
　そんな能力であるでしょう。



■お互いに、日々、
　トレーニングを重ねて、

　有形・無形の手に入れたいものを
　手に入れつつ、

　同時に、

　幸せを感じられる力も
　強化してまいりましょう。



■より詳しく学ばれたい方は

　「我田引水音源」

　をお聴き下さい。
