◎本メールマガジン（平成進化論）は、

　複数の会社を経営し、IT、人材、環境、飲食、ホテル、海外事業など
　20社以上のベンチャー企業に出資する投資家としても活動する筆者が、

　　------------------------------------------------
　   骨太かつ卓越した＜本物の教育サービス＞を通して
　  「変容の喜び」を提供し続ける
　　------------------------------------------------

　ことを目的とし、3000日以上にわたってお届けしている、
　個人発行として日本最大級のビジネスメルマガです。

　熟読するも読み流すも、自由な形でご活用下さい。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜昨日の活動ダイジェスト＞

▼メルマガへのご感想を頂戴しました

───────────────────────────────────



■こんにちは、鮒谷です。

　一昨日のメルマガに対してご感想を頂きました。
　ご紹介させて頂きます。

　
　（ここから）
　--------------------------------------------------------------


　毎日貴重なメルマガ配信を有難う御座います。
　◯◯で御座います。

　早いもので「平成進化論」を愛読し始めて8年目になります。
　いつも滅多に感想をお出し出来ない事に反省する次第です。


　早速ですが、本日のメルマガ“嫌煙”に対して
　少し感想を書かせて頂きたいと思います。

　確かに思考や行動の原理原則があって、
　鮒谷様には過去にそう言う歴史と背景があって
「タバコが苦手なのだな」とよく分かりました。

　と同時に相手を慮る大切さを改めて認識致しました。


　そこでひとつ思ったのですが・・・
　誤解を恐れず、鮒谷様の立場も承知した上でお話しを
　させて頂きますが、

　私は元喫煙者で3カ月前に2回目の禁煙に成功致しました。
　（勿論、現在も喫煙していません）

　1回目の禁煙は30歳から35歳での5年間でしたが残念な事に
　周りからの誘惑に負けて元に戻ってしまい喫煙を
　再開してしまいました。


　それから17年今回胸部レントゲンを撮って頂く機会が
　ありまして、

　結果は「肺の下部に炎症が起きている」との事で、
　「出来れば禁煙をお薦めします」とお医者様から言われました。

　　また「タバコは百害あって一利なしですよ」とも言われ、
　禁煙を致しました。

　3ヶ月経った現在は禁断症状も無くなり身体の全ての機能が
　すこぶる快調で本当に禁煙して良かったと思っております。


　鮒谷様が冒頭でおっしゃられた、
　読者の方で喫煙者（愛煙家）の方が大勢いらっしゃる事を
　承知した上でと、お話を始められましたが・・・

　確かに喫煙者の立場にも

　「一人ひとりの思考や行動の原理原則には
　　必ず歴史や背景や文脈が埋め込まれている」

　と言う事は理解しなければならないとも思いますが、
　それを含めすべて理解した上であえて言わせて頂ければ、

　本当に相手の事を慮り思いやる心があるのならば
　強く禁煙を勧める事も大事ではないかと私は思うのです。


　正直な話、自分が止められたからと言うのも少しはありますが、

　（中略）

　鮒谷様のお立場からは「平成進化論」に書けない事を充分に
　承知した上で今回は、このような大変に失礼な感想を書かせて
　頂きました。

　これからもお身体を大切にされてご活躍される事を
　お祈り申し上げます。


　--------------------------------------------------------------

　
　配慮に配慮を尽くしたご感想、
　ありがとうございます！
　
　（「文脈」を読んで、頂戴するご感想は
　　殊の外、嬉しいものです）
　


■喫煙・禁煙論争は深入りすると危険（！）なので
　これ以上、あまり突っ込んだことは申し上げませんが、

　今回頂戴したご感想をご紹介したということで
　私の胸中を斟酌していただければ幸いです。（汗）



■専門としている特定の主題以外については、
　必要以上に立ち入らない、

　ということは創刊当初からの編集方針として
　固く自らに戒めていることであり、
　今回のテーマについても同様に扱います。


　
■ただ、本当のところは、こういいたい、
　ということは、

　政治、経済、その他あらゆるテーマについて、
　持っています。

　（勿論、ごく親しい仲間とは語ります）



■ただ何ごとであれ、一たび発信すれば、
　そこに社会的責任が発生するわけですから、

　「自分が守備範囲と決めたこと以外は
　　黙して語らない」

　ことをポリシーとしているのです。

　

■そのようなわけで今回は

　「内角高め、ギリギリのところ」

　をご感想の紹介という形で、
　投げ込みました。


　この件についての本メディアにおける発信は
　これが限界、これでお終い、

　ということにしておきます。

　（他にも様々なご意見を頂戴したのですが）

　Ｈさま、皆さま、ありがとうございました！


　皆さまが思考するネタとしてメルマガを
　ご活用いただけましたら、嬉しく思います。





■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　コミュニケーションにおける自己修復システム　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

  【　コミュニケーションにおける自己修復システム　】

───────────────────────────────────

　

■昨日まで、


　「ある人の反応は、

　　『その人独自の歴史、背景、文脈』

　　から形成された反応装置から生み出される」


　というお話をいたしました。



■とするならば、

　「その人独自の歴史、背景、文脈」を
　100％理解することなど不可能ですから、

　意図せず、不用意に「地雷」を踏むことが
　日常茶飯に起きるのです。


　他者の前で言葉を発したり、
　なにかの振舞いを行うだけで、

　「地雷を踏む」
　（＝想定外のマイナスの反応が返ってくる）
　
　リスクが常にある、ということです。



■このことが分かればどうすればよいか。

　「常時更新の反省」（リアルタイムの反省）
　
　と
　
　「過去を振り返る反省」（時間をおいての反省）

　という二種類の反省が必要であることに
　気が付きます。



■「常時更新の反省」というのは、

　相手とコミュニケーションをとっている時、
　相手の反応を注意深く

　「リアルタイムで」

　行い、

　先方の表情が曇ったり、
　言葉に張りがなくなったりしたら、

　すぐさまそこで

　「地雷を踏みかけているかも」

　と一旦立ち止まる、ということです。



■「過去を振り返る反省」というのは、

　ひと通りのコミュニケーションを終え、
　相手と別れたとき、振り返って

　「あれでよかったのだろうか」

　と反省すること。

　
　ひょっとすると、自分は気持よく話をしていた
　けれども、

　相手は顔や態度に出さず、
　けれども内心、不快に思っていた、

　ということも、少なからずありそうです。
　その可能性を考えてみるのです。
　


■大切なのは、このように心がけ、
　常に反省（あるいは省察）し続けること。

　絶対に、自分が正しいという立場に
　立たないこと。


　自分が大切にしている背景・文脈と同等
　（あるいはそれ以上）の重みを持った背景・文脈を、
　相手も必ず持っている、

　ということに意識を向けることです。



■そのようにして

　「コミュニケーションにおける
　　自己修復システム」

　を自分の中に埋め込んでおきましょう。


　そうすれば、

　気づかずに断ち切られそうになった関係性
　をすんでのところで回復できた、

　ということが可能になるのです。
　


■「コミュニケーションにおける
　　自己修復システム」

　とは言葉を換えると、

　「解釈の数を増やして、
　　関係性の冗長性を高める」

　ことだともいえるかもしれません。



■「常時更新の反省」と「過去を振り返る反省」
　をぐるぐると回し続けていると、

　ある時点で不快に思えた人に対しても、
　
　ひょっとしてただ一つの解釈によって、
　相手の存在を固定しているだけなのではないか、

　と気づくことがあるのです。



■意識してこのサイクルを回し続けておかないと、
　あっという間に

　「ある人に対する観念」
　
　は固定化し、その認識から離れられなくなって
　しまいます。

　（そうしてせっかくの関係が切れてしまいます）



■出会う人に対して開かれた解釈、多様な解釈を
　持つことで人間関係の冗長性を確保する、という

　「コミュニケーションにおける
　　自己修復システム」

　をぜひ、自分の内にインストールしておかれると
　よいのでしょう。


　（なんていいながら、
　　まず私がインストールしなければ 汗）




　今日も人生とビジネスを楽しみましょう！
