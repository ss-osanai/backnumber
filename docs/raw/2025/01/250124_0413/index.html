■今日は

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------

　厚さ2センチの氷の上を跳ね回ってる
　大人がいたら、ただのアホだろ。

　でもたくさんいるんだわ。

　無意識に現状が続くはず、と
　勘違いしとる人が。


　実際は健康を損ねたり、
　交通事故、リストラ、勤務先の倒産、

　親の介護に大災害など
　一寸先は闇。

　「備えよ常に」はボーイスカウトに
　限った話やあらへんで。

　----------

　人生は山登りに似てるよな。

　軽い気持ちで登り始めると、
　気づいた時には危険な場所にいたりする。

　プロの登山家は常に立ち止まり、
　地図で現在地を確認する。


　人生も同じやで。

　今、目指す場所に近づいてるんか、
　道を踏み外して遭難しかけとるんか。

　確認作業を怠ってると、
　気づいた時には手遅れになる。

　----------

　一日18時間、
　働き続けても全く苦にならんから、
　正直、負ける気がせんのよな。

　しかも異常な集中力まである。

　ただし、ピンポイントでハマれる以外の
　99.9％は何をやらせても、
　まるで使い物にならん。


　分野選択が決定的に大事いうこと。

　ワイに限らず、どんな人でも
　そういうとこはあるんちゃうかな。

　----------

　人はみな、無意識に描いている

　「自分の物語」

　通りに生きようとしとるんよ。


　ほとんどの人は、その存在に気づかぬまま
　忠実になぞり、生きている。

　人生を根本から転換させたければ、
　無意識の物語を新しい物語に
　強制上書きするしかない。

　それには憧れや渇望、希求といった
　感情を伴う、強烈な出会いが必要。

　----------

　起業すべきでないのは、他人の評価や
　格好の良し悪しばかりを気にする人。

　ピカピカの経歴を誇り、
　酔ってるあなたは、

　日中、なんとかヒルズで
　下界を見下ろしながら仕事、

　夜は映える名刺を持って
　港区で合コンしてたらええ。

　
　それは地獄、くだらない、汚れ上等
　という奇特な人こそ、

　起業の世界にウェルカムや。

　----------

　言葉を操れる人は、
　結果を出せる可能性が高いんよ。

　ワイの知ってる
　長期繁栄系の経営者はみんなこれ。

　言葉は自分を動かし、
　誰かに影響を与える道具やからな。


　道具を雑に扱う人は、
　受け身で生きるしかない。

　ヤバい、キモい、ウザい、の3語だけで
　生きてる人間の未来がどうなるか、

　推して知るべしだろ。

　----------

　考え込むタイプは
　その姿勢自体を考え直した方がいいよ。

　立ち止まってる間にも、
　前に進む奴はフットワーク軽く、
　どんどん先に行ってるんやで。


　悔しいなら、致命傷を負う可能性の
　あること以外は全部やる、

　くらいの勢いで決断し、
　動けばいい。

　たくさんの人見てきたけど、
　マジであっという間に人生変わるよ。

　----------

　遊園地のアトラクションで、
　予め敷かれたレールを走りつつ、

　銃座だけ自由に動かせる
　乗り物あるやん。


　人生に例えると、
　レールは無意識に生きている物語。

　銃を操るのは意識的な頑張り。

　ただ、いくら頑張っても
　根本のルートそのものは動かせん。

　人生丸ごと変えたければ、
　新たなレールを敷き直すしかない。

　----------

　そう。「強烈な」出会いのみが
　無意識の物語を書き換えてくれるのよ。

　それは単なる出会いではないので、
　「邂逅」と呼ぶ方が
　個人的にはしっくりくるかな。

　100人に1人どころか、
　1000人に1人レベルやと思うが、

　そうした人と出会い、
　感情が揺さぶられ、

　無意識の物語が上書きされ、
　人生は新たな章に突入する。

　----------

　言葉の持つ力は、
　まず最初に自分が気づくんよ。

　その威力に驚愕し、
　言語化の達人の道を歩む決意が固まる。

　
　やがて実績が出始めると、周囲も放って
　おかなくなり、耳を傾けるようになる。

　そのときには既に
　言葉の匠（たくみ）となってるから、

　聞く耳を持ってもらえるようになり、
　外の世界に影響が波及する。

　----------

　たかが1回や10回や100回の行動で
　自分を変えられるとの思いが、
　どだい間違ってるんよ。

　ガキの頃から何千何万回やってきた習慣が、
　そんなに簡単に上書きされるかよ。


　歯の矯正をするイメージで、

　長期にわたる言葉による習慣矯正、
　くらいの気持ちで取り組まずに、

　自分を変えられるはずがないんだわ。

　----------

　カネを払うにあたっての
　明確な判断基準を持っている人は、
　実は少ないんよ。

　みんな雰囲気で判断してるだけ。


　何にとは言わんが、わけのわからん
　カネの使い方をしてる人、多いだろ。

　情緒や印象に流されず、
　正しく価値を見極め、

　思い切った投資決定できる人だけが
　長期繁栄、長期逓増していくのよ。

　----------

　想像と現実の差は100倍どころか、
　1000倍以上あるんだわ。

　現場を見ずに、想像や憶測だけで
　動きを止める人が多いのよ。


　その間にもとりあえず一歩踏み出し、
　五感を通して現実を感じて、
　さらに前に進んでいる人がいるんだわ。

　ダメなら戻ればいいだけなんだから、
　バンバン前に進むべき。

　----------

　「私は凄い」と信じ切ることよ。

　凄いと信じ込めれば、その思いが
　行動を促し、成果をもたらす。


　実績を多くの人に喧伝すると、
　大勢から凄いと言ってもらえるようになり、

　さらなる「自分は凄い」の信念が
　強化される。

　そこまでいけば、みなぎる私の自信に
　触れた人に一瞬で影響を及ぼせるようになる。

　----------

　半分冗談やけど、
　半分本気で、

　Ado『私は最強』を起床の音楽に
　することを勧めてた時期がある。


　多くの人から自己暗示の効果が
　あったと言われたから、

　騙されたと思って
　試してみてもええかもな。

　
　あと中高年向けにはSHOW-YAの
　『私は嵐』という曲を勧めてた笑。

　「私は嵐、私は炎、私は奇跡」

　とシャウトせよ。

　----------

　ワイはガキの頃から自分は別格、
　と信じてたよ。

　根拠なんかなんもない。

　
　幼少期から本をたくさん読み、
　非日常の物語世界に遊び、

　中高時代はインタビュー記事や自伝を濫読し、
　彼らの世界を疑似体験してた、いうのは
　あったかもしれんけど。

　思い込みの力に導かれ、
　規格外の世界に導かれつつあると思ってる。

　----------

　絶対的な自信は、
　虚勢や傲慢や増長とはまるで別物やから
　勘違いせんようにな。

　真の自信は謙虚さを生むから
　物腰はいたって丁寧。


　でも話をしたり、文章を書かせたら
　自信のほどが伝わる、

　という感じになるはずや。

　というか、ワイの文章や振る舞いが
　そのように見えとるとええんやが。

　----------

　ポジティブな言葉で脳を埋め尽くせ。

　ネガティブな言葉が入り込む隙のないくらい
　ギチギチに詰め込むのよ。


　前向きになるに決まってるし、
　それ以外の発想は浮かびようがない。

　こんな話を忌み嫌う人いるけど、
　無理する必要ないで。


　ネガティブな言葉で
　脳を埋め尽くせばどうなるか、

　自分で試してみたらいい。

　----------

　キラキラしたところに金をかけるほど
　廃業時期が早まるんよ。

　ドロドロしたところに手間と時間を
　かけると後から効いてくる。

　みんな泥臭いことはしたくないから
　参入障壁になるんだわ。


　それをやり切れる、気合いが入り、
　根性の据わった奴はほとんどおらんから、

　ビジネスで負けるはずないと
　昔から思ってる。

　----------

　お金に困らない生活をしたければ
　「稼ぐ」の定義を多方面から
　検討、更新することやね。

　以下は例。

　工夫を求められる楽しい他者貢献。
　真摯な価値提供の対価。
　需給ギャップの裁定取引。
　認知獲得競技の成果。
　高度な知的遊戯のスコア。
　人の嫌がることをする代償、とか。

　多様性があるほど知恵が湧く。

　----------

　朝から晩まで、たとえば

　「規格外」「規格外」「規格外」、、

　と100回唱えてみ。

　ほんとに規格外の存在となる方向に
　人生が動き始めるから。


　反対に

　「つまんない」「退屈だ」「腹立つな」

　とか、

　わざわざ自分で人生をつまらなく
　する呪文を詠唱しとる人は、

　それが現実になるんだわ。

　唱える呪文の更新をかけることよ。

　----------

　失敗を恐れるのは愚か者の特徴やで。

　知恵ある者はたとえ失敗してさえ、

　突っ込んだ方が経験その他でトータルで
　得すると分かっとるから行動する。


　あとはメンタルブロックを
　破れるかが障壁やが、

　5,6発殴られれば、
　すぐに痛みに慣れるから笑。

　パンチを恐れててはリングに立てず、
　デビューすら叶わんで。

　----------

　知人の、軽いノリでの

　「独立したら発注するよ」

　ほど当てにならん約束もないから、
　真に受けたらアカン。


　一方で確度100％に近い、盤石の発注確約を
　起業前に固めておくと危うげなく離陸できる。

　選挙と一緒やで。

　投票日（独立）前に浮動票を
　いかに自分の側に引き寄せ、

　確実なものにしておけるかよ。

　----------

　集客は恥を感じなくなってからが本番。

　選挙で恥ずかしがってる候補者とか
　見たことないやろ。

　振り切れば、そういう人間だと
　みなされるようになる。


　そこまで到達できれば無双やで。

　大量行動、大量実験、
　データ蓄積、仮説精度向上、

　無数の改善、大数の法則、
　バンドワゴン効果など、

　全部が一気に効いてくる。

　----------

　初対面の人にはクレクレ言わずに、

　徹頭徹尾、相手の役に立つことだけ考え、
　提供するんやで。

　できなくても姿勢を見せれば、
　分かる人は分かってくれる。


　特に年長者や実績を残している人。

　見るべき人は見てくれ、
　評価してくれる。

　けれど二度目はないから、
　初対面に命を賭けるのよ。

　後から大きく報われる。

　----------

　自己暗示、バカにする人多いけど
　バカにできない効果があるよ。

　やってる人だけ知っている。

　ただ、うさんくさい人認定されんために
　言わない人が多いだけ。

　ワイはうさんくさい言われても、
　メルマガ読者さんのために、

　世界の中心（メルマガ笑）で
　自己暗示について叫んどる。

　----------

　ワイの情報発信は過去現在を通して、
　体系化と一貫性を意識しつつ、

　あえてその構造を表に出さない形で
　発信してるんよ。


　ワイが影響を受けた駿台の
　故・伊藤和夫先生の教育理論に基づいとる。

　『予備校の英語』という本は
　英語教師のみならず、

　教育に関わる全ての人に
　読んでもらいたい名著。

　----------

　自己暗示と立てた目標の通りに
　人生、進んでいくからな。

　ネガティヴな暗示を自分にかけ、
　その上、目標もなければ、

　北風に煽られて側溝に吹き寄せられる
　落ち葉みたいになるに決まっとるやないか。

　----------

　ポジティブな言葉を使う人は
　ポジティブな言葉を使う人同士、

　ネガティブな言葉を使う人は、
　ネガティブな言葉を使う人同士、

　結びつくんだよ。

　磁石はN-N、S-Sだと反発するけど、
　人間はP-P同士、N-N同士が結びつき、

　共に良くなり、共に悪くなる。

　----------

　ネガティヴな言葉を封印するというよりは、

　肯定的な言葉で上書きして
　粉砕するイメージの方が良いな。


　封印したものは、何かのきっかけで
　解放される可能性があるからな。

　粉砕して二度と復活することのないよう
　トドメをさしておきたい。

　----------

　「キチンとしろ！」言う人おるよな。

　でもADHDのワイみたいに、
　それが出来たら苦労せん、

　いう人間もいるんだわ。


　そんな人に言いたいのが

　「99％はテキトーでいい。
　　ただし巨大な成果を見込める1％に
　　全精力を注ぎ込め」。

　その成果で得られる金や評価で、

　残り全ての肩代わりや失敗の帳消しを
　してもらうのよ。

　----------

　恥をかくことが快感に変わるところまで
　感覚を破壊できれば、勝利は目前よ。

　姿かたちを変えたSMと言って良い。

　----------

　発達障害（ワイはADHD）＝ステータス全振り、
　という解釈をしとる。

　天賦の資源に恵まれ、
　一点集中で勝ち切ることを約束された
　スーパーエリート。

　周りの人間を羨ましがらせたる、くらいの
　気持ちで毎日、生きてきたったわ。

　----------

　ワイは毎日、脳に浮かび上がってくる言葉に
　感動しっぱなしなんよ。

　コーヒー飲みつつ本を読んだり、
　ぼーっとしとるだけで、

　脳に電流走る感じの震える瞬間が
　不定期で訪れる。


　刺激を求めてパチスロ行く必要ないし、
　金もかからん。最高の道楽や。

　その感動のおすそ分けのつもりで
　毎日こうして発信しとるんよ。

　----------

　言葉の定義とか、
　脳内で結びついている単語同士の連想体系は
　人生を決定づけるほどの重みがあるのに、

　みんな適当なのよ。


　たとえば

　「恥＝社会的制裁、切腹モノ」

　と定義する人と

　「恥＝すべらない話のネタ、成長の糧」

　と意味づける人とでは
　同じ行動になるはずがない。

　一つ一つの言葉に丁寧な定義づけを。

　----------

　中途半端なつながりに、ほとんど意味ないで。
　まして名刺交換しただけとか。

　がっしりとお心を頂戴してこその、
　コミュニケーションの閾値超え。


　それをリアルで行いつつ、
　私はオンラインでもお心を頂戴できないかと

　熱烈熱狂バーチャルコミュニケーションを
　長年、遂行してきたわ。

　活きている縁こそ、大事。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。
