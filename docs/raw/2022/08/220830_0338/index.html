■一泊20万円の部屋（花火鑑賞プラン）に
　泊まりましたー

───────────────────


■上述の通り、先週の木曜から日曜まで
　４日間、大阪に行っておりました。

　３年ぶりに開催された
　
　「なにわ淀川花火大会」

　を鑑賞するためです。

　花火鑑賞できるホテルの部屋を取り、
　お客さまと一緒に楽しみました。



■清水の舞台から飛び降りるほどの
　覚悟とともに

　「10年に一度」

　くらいの贅沢のつもりで予約を入れた
　一泊20万円の部屋。

　
　
■にもかかわらず

　「ご案内早々、多くの申し込みを頂き、
　　満室になりました」

　とホテルスタッフの方からお聞きし、
　度肝を抜かれました。

　あるところにはあるんですね、、


　
■ちなみに私（鮒谷）の取った部屋は
　もちろん一番安いところ。

　それでこの価格。
　（スイートとかだと当然もっと高い）



■夕食、ついてないですよ。
　
　シャンパン、ちょっとしたスイーツ、
　翌日の朝食ビュッフェ、以上！


　これで20万か、と目を剥きましたが
　歯を食いしばって

　「これは投資的浪費なのだ」

　と自らに言い聞かせた次第。



■これだけ盛況であったのは
　コロナ禍を経ての

　「強制貯蓄＆反動消費」

　の影響なのかもしれませんね。


　それにつけても、ここだけ時空が
　歪んでいるのかと疑いましたが、

　需給のバランスに最適化された
　価格設定なのでしょう。

　現実に満席になっているわけですし。



■それはさておき、

　こうした非日常経験も意識すれば、
　一つの学びになり得ます。


　常識はずれの価格帯の商品やサービスを
　体感することで、

　常識はずれの価格帯の商品やサービスを
　自身も提供する許しが与えられる、

　ということはありそうです。



■現に私（鮒谷）は、

　こうした高額サービスが販売され、

　しかも早々に満室になり、
　さらに滞在している人はみんな笑顔、

　という姿を目の当たりにして、
　
　
　うっかりすると

　「安く、もっと安く」

　という精神に陥りがちなところを、


　やっぱり

　（もちろんお客さまが納得くださる
　　品質であること前提で）

　「高く、もっと高く」

　という姿勢で良いのだという思いに
　立ち返ることができました。



■非常識なほどの高額サービスが
　早々に売り切れ、かつ、顧客も満足、

　こうした世界をリアルに感じることで
　新しい世界認識が立ち現れる、

　そんなことがあるのです。



■ここまで書いて思い出したのですが、

　そういえば、私（鮒谷）もこんな音源を
　販売しておりました（笑）

■（今回の花火鑑賞もそうですが）

　私（鮒谷）のこれまでの高額商品購入の
　経験と学びを織り込みつつ、

　「安く、もっと安く」

　そんな値下げマインドから逃れられない
　あなたに痛撃を食らわせ、

　改心（？）を促すための、
　刺激に富む音源に仕上がっております。

　

■この音源を聴けば、

　価格交渉でビビり続けてきた
　（遠慮し続けてきた）、

　それゆえ貧乏暇なしから
　抜け出せなかったあなたが

　【値上げ一択】

　つまり

　【価値に見合った請求を行い、
　　適正な利益を得られる】
　
　ことによって、生活と精神にゆとりが
　生じるようになるもはずです。



■あなたが仕事を続ける限り、
　当音源の効果・効能が持続する、

　と理解できれば、

　その価値は
　音源価格の何十倍、何百倍、
　
　ひょっとすると何千倍にも
　相当するかもしれない、

　とお思いになられないでしょうか。

　
　
■たとえばですが、弁護士さんや
　税理士さん、コンサルタント等の方が、

　全顧問先への「一律２割」の
　値上げ要請を通すことができれば、

　今後の数年、十数年、、、の累計で
　どれほどの利益の絶対額が手元に残るか
　想像してみてください。
　

　
■もちろん（継続ではなく）
　単発の商売においても、

　価格を引き上げても、
　購入率が変わらなければ、

　それ以降、同じく利益の絶対額が
　そのまま上積みされることとなります。



■それゆえ

　【値決めは経営】

　と言われます。


　価格設定は、
　いい加減にやるものではないし、

　日和（ひよ）っていいことは
　一つもありません。
