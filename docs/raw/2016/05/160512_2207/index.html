■個別コンサル３件

───────────────────────────────────



■こんにちは、鮒谷です。


　昨日は終日、セミナーに参加。

　今日は個別コンサル３件、でした。




■話は大きく変わりまして、
　

　先日、個別コンサルをさせて頂いた
　Ｆさんより、

　以下のご感想を頂戴いたしましたので
　ご紹介いたします。
　

　（ここから）
　--------------------------------------------------


　この度は、個別コンサルティングを
　行っていただき、ありがとうございました。

　コンサルティングの感想を拝読していると

　「完膚無きまで叩きのめされる」

　という感想を頻繁に目にしているように
　感じていました。


　また、丁度先日の2日間セミナーで一倉先生の

　コンサルティングのワンシーン
　（「殴る」からの「抱擁」）

　をお聞きしたこともあり、

　お金をお支払して受ける側であるにもかかわらず、
　受ける前は緊張しておりました笑

　実際は、非常に大きな學の時間になりました。


　さて、本題のコンサルティングですが、
　私の中では「暫定1位」が、最も心に残りました。

　（おそらく、本筋のお話ではないかと
　　思いますが・・・笑）


　鮒谷さんは

　「何をもって憶えられたいか」
　「所を得た人生」

　平たく言えば、

　「目標」

　が明確になったからこそ、圧倒的な行動力に
　移すことができたのだと考えていましたが、

　意外にもまだまだ不明瞭なところがあるようで、

　「実態は違うのか」

　と感じました。


　だからこそ、

　「暫定1位」

　で行動することの重要性を
　感じられたのだと思います。


　今思うと、メルマガで拝読したか、
　CDやセミナーで拝聴したかで

　「目標の仮置き」

　のお話を読んだり聞いたりしていたかと
　思いますが、

　人によって同じ内容でも心への刺さり方は
　違うのだな、と感じました。

　それぐらい言葉の重要性を改めて
　「漆塗りのごとく」感じました。



　次に強く感じたこととしては

　「相対的に進んでいる」

　があります。


　完璧主義志向で

　「自分にとっての理想」

　が再現できなければ商品にならないと
　考えていましたが、極端な表現をすると

　「劣化版〇〇さん」

　でも需要はあることは、記憶の奥底から
　表面に引出されたと感じています。


　実際のところ

　「完全版〇〇さん」

　は高額だったり多忙だったりするため、

　劣化版でも安いし、
　すぐにサービスを受けられる、

　ことに対して需要があるのだと考えています。



　三番目のお話としては

　「福原の変化率は高い」

　と称していただいたことです。


　「変化率が高い = 変化して結果も出した」

　まで含まれると考えていました。


　絶対値としてはまだまだだけれども、

　「憧れの世界の住人」

　の1人である鮒谷さんに直接仰って
　いただいたことは、非常に励みになりました。



　最後に、本筋のお話としては
　「惑星直列」が頭に入れておくべき
　キーワードだと感じました。


　福原から見た惑星直列の世界観、

　言うならば惑星を包み込む「銀河観」とでも
　言うのでしょうか。

　次回までの課題である

　「メニュー」

　「プロフィール」

　を作込み、「銀河観」を提供していきます。


　最後になりますが、今回のコンサルティングで
　DCAPの感覚が掴めたのでご感想に入れさせて頂きます。

　D・・・スマートフォンでボイスレコーダーを使用
　C・・・再生してみて不明瞭な箇所が多い
　A・・・要対策
　P・・・鮒谷さんの近くに置かせて頂く、
　　　　ボイスレコーダーを買う、など


　次回以降も、引続き、よろしくお願いいたします。


　（視点提供家 / メンズファッションコーチ
　　　　　　　　　　　　　　　　 　福原 宗之さま）


　--------------------------------------------------
　（ここまで）



■福原さん、長文のご感想をお送りくださり、
　ありがとうございました！

　種々気づきを得ていただくことができたようで
　大変嬉しく拝読させていただきました。



■舌足らずであったかもしれませんが
　一点だけ補足（フォローアップ？）しておくと、
　
　「目標が不明瞭」

　であるわけではなく、


　--------------------------------------------------

　圧巻、極める、破天荒、

　日本一の存在、ブチ抜ける、

　プロフェッショナル、他の追随を許さぬ、

　他の追随が不可能な、超越、卓越、超絶、

　無敵、敵無し、戦わずして勝つ
　他者が戦意喪失するようなレベル
　
　市場の創造、競わない、孤高、独自の道を征く

　--------------------------------------------------
　

　という目標については、極めて明瞭、明確です。

　（今どうか、という話ではありませんので
　　念のため。
　　
　　目指すのは勝手です）


■この目的を達成するための手段については

　「仮置き」

　している、そんな話でありました。

　念のためー。

　また疑問があれば、次回に質問して下さい！



■福原さんの変化率や将来性については

　「太鼓判」

　を押しておきたいと思います。

　これまでたくさんの方を見てきた中でも、

　地頭の良さ、理解力、行動力、
　コミュニケーション能力、等々、

　いずれも際立ったレベルです。


　あとは的確な人生、キャリア、ビジネス戦略を
　携え、時間を味方につけることですね。

　引き続き、お手伝いさせて頂きます。
　これからもよろしくお願いいたします！

【 私（鮒谷）が仕事をする理由を、改めて考えてみたところ、、、 】
　　　　
───────────────────────────────────



■私（鮒谷）が
　仕事をするのはどうしてなのだろう、

　という根源的な問い（！？）が
　先日、唐突に生まれました。


　改めて考えてみたところ、
　一番の理由は

　「恩返しであり、罪滅ぼしのため」

　という理由が出てきました。


　これまで両親をはじめとして、
　さまざまな人から恩を受け、

　同時に、言葉に尽くせぬほどの
　大きな迷惑をかけてきたようにも思います。



■恩返し、あるいは罪滅ぼしを
　直接できる方もあれば、

　すでに鬼籍に入られた方や
　縁遠くなってしまった方もあり、


　さまざまな理由で、もはやそれも
　叶わぬ人たちがたくさんあるわけです。


　そうした方々から受けた恩を返し、
　あるいは罪を滅ぼすために、

　「恩送り、あるいは罪滅ぼし送り」

　ではありませんが、

　仕事を通して、世の中に対して、
　多少なりとも自分が役に立ち、

　最終的に少しでもプラスに転じ変わった状態で
　生を終えることができれば、

　と思うようになりました。


　（人として成熟したということでしょうか！？）



■さらに考えてみると、

　他にももう一つ、理由があるように
　思われました。


　それは

　「昔の情けない自分」

　を振り返ると思い出される、

　当時の中途半端さや、いい加減さ、
　妥協、怠惰、無精、横着、

　といった、

　長年にわたる不完全燃焼を通して
　溜まりに溜まった莫大なエネルギー、

　こいつを世の中にぶつけて、
　昇華させたい、

　という思いから来るもの。



■このような思いとともに、

　関わる人に全力でプラスの痕跡を残そうと
　して臨むからからこそ、

　一定の評価を頂けるようにもなり、
　それが励みとなってますます頑張れる、

　ということもありそうです。



■もちろん、

　私（鮒谷）が仕事をしているのは、
　上に記した動機が全てではありませんが、


　仕事を行う動機のある一定の部分

　（あるいは、
　　ひょっとするとかなり大きな部分）

　を占めているように思われます。


　付け加えると、
　
　「昔、手ひどく振った彼女を見返すため」

　というのもありそうです、

　というのは冗談ですが（笑）



■いずれにしても、

　ただ、利益だけのためには
　あるレベル以上には集中して、

　仕事を行うことなどできないように
　思われます。


　仕事を行う第一の理由が利益ではなく、
　他のところにあるからこそ、

　妥協なく、
　あくなき追求を行なうことができ、
　
　結果としてビジネスも回るし、
　利益もついてくる。



■そんなことを、

　冒頭の問いに自問し続けるうちに
　改めて気づき、

　一層の価値を提供するために
　さらなる高みを目指し、
　　
　ますます仕事に邁進しようと決意する
　機会ともなりました。
