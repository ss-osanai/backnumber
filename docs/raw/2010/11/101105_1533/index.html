▼価値の循環、元気の秘訣

───────────────────────────────────



■こんにちは、鮒谷です。

　この数日、

　「ホーソン効果」
　「フィードバックを頂く重要性」
　
　について記したメルマガに対して
　たくさんのご感想を頂きました。



■その中に

　「鮒谷さんはどんなフィードバックを
　　毎日受けているのですか？」

　というご質問があったので、
　一部をご紹介いたします。


　
■感想を下さる方は、

　拙文に理解、共感して感想をお送り下さるわけですから
　きっと記憶の定着が高まることとおもいますし、

　私自身も先にも記したように、有り難いフィードバックと
　なっており、更に頑張ろう！という気持ちになる、

　ということで、ご感想を頂くことで

　「価値の循環」

　が生まれるのを実感しております。



■ぜひ、引き続き、多くの皆様からのご感想を
　お待ちしております。

　あなたのために、そして私のためにも（！？）

　（こちらからお送りいただけます）
　→ http://clap.mag2.com/slosliolew?2609



　（ご感想ここから）
　-----------------------------------------------------------------

　いつもお世話になります。

　今日の「ホーソン効果」はとてもわかりやすく、納得できました。
　
　なぜ成果の上がる人と上がらない人がいるのだろう・・・と
　ずっと考えてきていたのですが、鮒谷様のメルマガでスッキリしました！

　本当に感謝いたしております。
　いつも素晴らしいヒントを頂きましてありがとうございます。

　-----------------------------------------------------------------
 
　今日のホーソン効果の話は子育てにも通じるなと感じました。

　私は割と仕事や自分の事が忙しく普通の母親のように教育熱心だったり
　送り迎えをしてあげたり、子供にものすごく執着
　（と言うと言葉悪いですけど）した生活をしているとは言えないのですが
　
　「いつも見ているよ（考えているよ）」と言う事に気づいてもらえれば
　接点の多い少ない大きい小さいに関わらず良い親子関係が築けるのかなと
　ふと思いました。

　今日は娘は修学旅行に行ってしまって会えないけど大切な息子と娘に
　「見ているよ」を感じてもらえる何かをしようと心に決めました。

　-----------------------------------------------------------------

　きょうのメルマガは一番良かったと思います

　このメルマガを読んでいてよかった。
　「見られている。注目されているということがわかると
　がんばるのが人間です」ということは人間の心理を突いていて

　理解すると自分を動かす時や相手を動かす時など
　いろいろな場面で活用できそうです。

　-----------------------------------------------------------------

　人間はどんな決意をして物事にまい進しても途中でマンネリ化をする。

　そこでやはり絶えず自らを鼓舞する為に刺激策を自ら考えなくては
　ならない。本日のテーマは良かったと思います。

　-----------------------------------------------------------------

　なるほど、と納得しました。

　それにしても一日も欠かさず発信しておられるのは驚きです。
　病気で寝込まれたことも無いのでしょうか？

　-----------------------------------------------------------------
　（ご感想ここまで）


　日々、このようなご感想をたくさん頂戴していると、

　落ち込んだり、やる気がなくなったりということに
　ならないというのがお分かり頂けるのではないでしょうか。

　「顧客（読者）の声」こそが前に進む原動力となるのです。



■ちなみに最後のご質問を下さったＳさんの

　（Ｓさま、いつもご感想ありがとうございます！
　　なかなかご返信できなくてスミマセン、、）


　「病気で寝込まれたことも無いのでしょうか？」

　に対するご回答ですが、

　これは一言、

　「ありません」

　です。



■どういうわけか対外的に情報を発信するように
　なってから風邪一つひかなくなりました。

　気の張り、緊張感、あるいは気合からだと
　私は解釈しているのですが、

　実は小学校から学生時代くらいまでは
　本当に病弱で、しょっちゅう39度とか40度くらいの
　原因不明の高熱を発しては倒れこんでいたものです。

　
　ところが社会人に出てから、

　「自分の体は頑強だ！ものすごい耐久性能がある！」

　などと強力な自己暗示をかけ、

　さらにはメルマガを配信し始めた頃から

　「風邪一つひかないと決めた（！）」

　ことにより、本当に風邪一つひかなくなったのです。


　そんなこともあり、もちろん全ての病が、とはいいませんが、
　実体験から、おそらく相当程度、「病は気から」というのは
　本当だと思っています。



■以上は余談的なこととなりましたが、
　こういうフィードバックを毎日、膨大な量、
　頂くことにより、

　非常に大きな力を頂いているわけで、
　ご感想を下さる方には本当に感謝しつつ、
　
　その価値を私も有用なメルマガコンテンツとして
　お戻ししたいと日々、考えています。


　読者の皆様、ご感想を下さる皆様、
　いつも本当にありがとうございます！




■それでは今日のピークパフォーマンス方程式（ビジネスにおいて、
　最高の結果を発揮するための方程式）をどうぞ。


　今日は、


 【　触媒的な人  】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　触媒的な人　】

───────────────────────────────────



■何度か書いたことがありますが、

　私はプライベートコーチというのか、

　種々、身の回りに起こる出来事について
　定期的に（月２回）相談できる方と、

　アドバイザリー（顧問）契約を結んでいます。



■この方は以前、ある上場企業の社長を務められた方で、
　プライベートでの友人でもあるのですが、

　あえて「契約」という形を取ることによって、
　普通の会食、飲み会、仲間内の話、あるいは与太話？に
　終わらぬよう、

　ビジネス的な関係を取り入れることにより
　制限を設けて密度の濃い時間を過ごさせて頂いている
　のです。



■この形式で進めてみて、
　その威力の凄さを改めて実感しています。

　毎回、非常に有益なミーティングになっています。


　なにが凄いかといって、
　この方のような

　「触媒的な人」

　を相手に話をすることにより、


　１、自らの中にある、モヤモヤとした考えや概念、言葉が
　　　外部世界に引きずりだされ、

　２、相手の思考や世界観と融合、混合され、再編集され、

　３、グレードアップ、ブラッシュアップ、パラダイムシフトを
　　　もたらされたされた思考を、改めて自らの内にインストールする
　　　ことができる


　ということ。
　


■告白すると、私はこれまで

　コーチ、あるいはコーチング、といわれるものについて
　正直、その価値について分からなかったのですが、

　その価値を改めて強く感じているところなのです。

　（対価の何倍、何十倍の価値が与えられていることを
　　実感しています）



■自分の頭だけで考えていては
　　
　自分の枠が超えられなかったり、
　勝手に限界を設けてしまったり、

　という制限を取り払っていただいたり、
　
　あるいは、

　他者から見た自分の見え方、捉えられ方、
　自らの可能性を新しく知らされたり、

　といろんなメリットがあるのです。



■ただし、ここで極めて大切なことが一つありそうです。
　
　それは相手をしていただく方が
　誰でもいいかというと決してそうではなく、

　
　「相互のレベル感、信頼関係、相性、
　　言語化能力、実績、感性」


　などが合致することが大前提である、
　ということ。



■そういう方がもし身近にいらっしゃるなら、
　あるいは、いないなら、探してでも、そういう

　「触媒的な人」

　と定期的に会うことで

　自らの可能性が引き出され、
　新しいチャンスが生まれるのでは、
　
　そんなことを感じています。



■さらに言えば、つまるところ、

　そういう「触媒的な人」を人生の中でどれだけ持てるか、
　が幸福感とか成長感に直結するのではないかと考えます。


　いつも書いているように、日経新聞に連載されている
　私の愛読コーナーである

　「私の履歴書」「交遊抄」

　を読んでいても、結局、人生に大きな影響を
　与えるのは人との出会い、人との語らい、

　であり、その過程の中で人は変わっていくものです。
　


■そして、これは私の使命として明確に意識している
　ことですが

　わたしも

　直接的に（実際にお会いすることにより）、
　間接的に（メルマガ等でを通じて)、

　関わる全ての人に対して

　「触媒的な人」

　にこそなりたいもの、
　と考えています。
　

　すなわち

　「（仮に何かを与えられなくて、であってでも）
　　何かを気付かせられる人」

　でありたいと思っているのです。
