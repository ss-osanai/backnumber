【 （あと２日）（あと２日）「妄想を現実化させる能力」の基礎となる力 】

───────────────────────────────────



■「人生を劇的に変える」ために、
　必要な能力が２つあります。


　それは

　--------------------------------

　１、自分を主人公とする物語を作る力（妄想力）
　２、その物語を現実化する能力（具現化能力）

　--------------------------------

　の２つ。



■納得のいかない人生を転換させ、
　機嫌良く生きていけるようになるためには、

　この２つの能力が必要ではないか、
　と考えています。



■前者（妄想力）は、またの名を

　【目標設定力】

　ともいいますが、


　せっかく、類まれなる目標設定力を
　発揮し、

　【血湧き肉躍り、実現を目指さずには
　　いられない素晴らしい目標】

　を立てることができたとしても、


　その物語（目標）を現実化させるための
　諸能力が欠けていれば、

　「妄想は妄想のまま終わってしまう
　　（絵に描いた餅となる）」

　のです。



■餅の絵すら書けない人
　（＝妄想できない、目標設定できない人）は、

　そもそも始まっていないので、まずは
　

　--------------------------------

　【目標設定音源】
　https://shinkaron.buyshop.jp/items/5162715
　
　ならびに

　【物語放談会音源】
　https://shinkaron.buyshop.jp/items/6536488

　--------------------------------


　をお聴きになることをお勧めします。



■しかし、これらの音源をすでに聴いて、
　内容に納得もされているのに、

　人生や生活や仕事上に
　何の変化も起こっていないとするならば、


　その理由は明白で、
　上記２つのうちの後者、すなわち
　
　【物語（妄想）を現実化する能力】

　が欠如しているから、
　ということになりますね。



■ここでいう

　【物語（妄想）を現実化する能力】

　の基礎をなすのが

　【時間管理の能力】
　
　です。


　時間管理の巧拙が、

　不可能を不可能のままにするのか、
　不可能を可能に変えるのか、

　の分水嶺となるのです。



■たとえば、

　「休日に 日がな一日 ユーチューブ」

　などと一句ひねり、

　「我ながら、なかなかよく出来た」

　などとのたまいながら、
　動画を見続けていたところで

　「妄想を現実化させる」

　ことなどできるはずがありません。



■きちんと


　「未来の目標と、現在の状況の間にある
　　ギャップを意識し、

　　そのギャップを持ち時間を使いながら
　　埋めていく」


　そんな精神を持ち、
　適切な時間の使い方を行ってこそ

　【（妄想）世界の創造、実現の達成】

　が近づいてくるのです。


　願うだけ、妄想するだけでは
　何も変わることはありません。



■私（鮒谷）自身、
　あるときから、この

　【当然過ぎるほど、
　　当然な理（ことわり）】

　を理解できるようになったので、

　時間の使い方には大きな注意を払うように
　なりました。



■それゆえ、

　昨日（多くの人が寝てたり、遊んでいるで
　あろう日曜日）の午前から、

　今日の夕方にかけても、およそ30時間、
　都内の某ホテルに籠もって、　

　【丸一日、集中して経営計画を立てる会】

　と銘打った会を、
　親しい友人経営者ととも行い、

　今後の経営計画を立案していました。


　あとはこの計画を粛々と
　実行に移していくだけです。



■こうした企画立案ならびに、
　企画を形にするための時間は、

　自らの意思によって作り出すより他、
　ありません。（誰も用意してはくれません）　


　自分が意識して取り組まなければ、
　
　「永遠に、流されるままの人生
　　（＝代わり映えしない人生）」

　が続いていくだけなのです。



■きちんと自分の持ち時間を把握し、

　可能な限り、残り時間を有効活用しようと
　決意し、実行する人から順番に、

　【妄想人生を現実人生へと作り変える】

　ことができるでしょう。



■理論、理屈は分かった。


　でも、具体的に
　どうしたらいいか分からないし、
　
　今ひとつ、感情も盛り上がらない、

　という方がもしいらっしゃるのなら、


　そんなあなたにこそ、
　ここしばらくお伝えしてきた

　【タイムマネジメント音源各種】

　をお勧めいたします。

　※そんなあなたに向けて作成した音源です。



■たいていの人は、
　こうした話を理解しても、なお

　「そうかもしれないけれども、
　　自分が取り組むのは億劫だ、面倒だ」

　となるのが普通ですが、


　こんな考えに触れ、より深く知り、
　さらには自らも実践してみたい、

　と思われた奇特な（！？）あなたには、

　ぜひ、以下の音源をお聴き頂ければと
　願っております。
