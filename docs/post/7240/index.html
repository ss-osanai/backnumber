■人生をどんどん作り変えている人は、

　当人に自覚があるかどうかはさておき、
　必ず

　【知る→想像する→妄想する→本気になる】

　の過程を通って、
　結果を出しているものです。



■あなたもご存知の通り、

　平均を大きく上回る、
　さらには尋常ならざる成果は

　【本気になった人】

　にしか生まれません。



■本気モードに入ることさえできれば

　（目標の大きさによっては相当の時間が
　　かかることもありますが）

　たいてい、実現するものです。



■本気から生まれる行動は、

　複利で転がり、
　思わぬ出会いやきっかけを通して、

　想像を超える
　化学反応が生まれることがあるし、

　それが現状の世界からの跳躍をもたらし、
　日常生活から雄飛させてくれるから。



■ところで、ここで誤解のないよう、

　【本気になる】

　と

　【本気を出す】

　の根本的な差異について
　触れておくべきかもしれません。



■本気になる（なっている）とは

　【内発的な、本気状態
　（どこどこまでも自然体）】

　に入っていることを言います。



■一方、本気を出すとは、

　【無理やり本気になろうとしている状態
　（不自然な力がかかっている状態）】

　と言えるでしょう。



■より分かりやすく言い換えれば

　【本気になる
　　＝自然体だから持続する】

　【本気を出す
　　＝無理があるから持続しない】

　ということです。



■もうお分かりのように、
　なので

　「本気を出す」

　と言っている人が
　結果を出すことは決してありません。

　（かならず三日坊主に終わるから。
　　かならず、です）



■反対に、本気になっている人は

　「いちいち口に出して言わなくても
　　行動を見れば分かる」

　「一つの対象に向けて
　　集中し続けている」

　ので、

　この人は、次々に新しい世界を
　切り開いていくでしょう。



■だからこそ、
　今現在、夢中になれるものがなく

　「早く本気になりたい！
　（妖怪人間ベム風に）」

　と願う人は、

　「本気になっている」状態が
　どのように作られるのかを、

　知っておくべきなのです。



■ここで再び、冒頭で紹介した

　【知る→想像する→妄想する→本気になる】

　の流れを確認いたしましょう。

　本気になりたければ、
　この流れを逆にたどればよい。



■すなわち、

　本気になっているには
　妄想スイッチが入っている必要があり、


　妄想スイッチが入るためには、

　（妄想のタネであるところの）
　想像を膨らませる過程が必須。



■想像を膨らませるためには、

　知識がなければなりません
　（知識がなければ想像を膨らませようがない）

　ので、

　【本気になるには、
　　まず知るところから】

　ということになるのです。



■もちろん、

　【知ったものの全てに対して
　　本気になれるわけではない】

　ですが、同時に

　【知ったものの（率は少なくとも）
　　一定割合で本気になれる対象が見つかる】

　のも、また真実。



■この確率を高めていきましょう、

　そんな提案を
　「目標設定音源」において

　【リトマス試験紙を当てる】

　という比喩を用いて
　行っていたわけです。



■見る、聞く、学ぶ、交わる、
　さらには体験する、、、といった行為を通じて、

　私たちはリトマス試験紙に当てて、

　「そこで色が変わるかどうか
　（対象に夢中になれるかどうか）」

　を判断すればよいでしょう。



■そこで色が変わらなかったものは、

　「素直に捨てる
　＝それ以上、突っ込まない」

　「拘泥しない（こだわらない）」

　という姿勢を貫き、


　「次行こ、次！」

　を繰り返していると、やがて、

　「これは面白いかもしれない」
　「もう少し掘っていきたい」

　という対象が見つかるはず。



■ときおり（というか、頻繁に）

　「そういう対象がなかなか
　　見つからないんですけれども、、、」

　つまり

　「本気になれる対象がない」
