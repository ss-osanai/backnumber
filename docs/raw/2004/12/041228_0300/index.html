＜昨日の活動ダイジェスト　優秀な人材を囲い込むための営業を。＞
───────────────────────────────────


■いよいよ年の瀬が近づいてきましたが、結局、今年もゆっくりする時間は
　とれなさそうですね。

　決して望んでやってるわけでもないのですが、結果としては不眠不休で
　活動してます。


■今日は赤坂のニューオータニのロビーで待ち合わせの為にソファに座って
　本を読みながら待機していたのですが、不覚にも読書中、意識を失って
　後ろのガラスに思いっきり頭を打ちつけてしまいました。
　（凄い音がしました）

　周りの方が一斉に振り返ってとっても恥ずかしい思いをしてしまいました。


■31日までは、なんとか気力で乗り切って正月三日間は、眠り続けようと
　思ってます。


■昨日は、将来どういう形でかはわかりませんが、一緒にビジネスをやりたい
　と思っている前途有望で優秀な若い人材（東大法学部卒→某一流企業在職中）
　の確保のために、さりげなく営業をしてました。

　学歴どうこう、ではありませんが、さすがにとても高いポテンシャルを
　感じさせるところと勉強熱心なところ、人間的に魅力のあるところなどが
　ポイントです。

　
■いまいまでなくても、いずれどこかで仕事で組みたいと思っているのです。
　そのうち何か一緒にできればいいなと思ってます。

＜今日のピークパフォーマンス方程式。＞

　【　ビジネスでは愛を持つことを心がけよ。　】

───────────────────────────────────


■昨日、ビジネスに【政治力】は必要、といいましたが、同時にビジネスには
　【愛】も必要なのです。

　矛盾するように思われる方もあるかもしれませんが、この二つは決して
　自家撞着（つじつまの合わないこと）ではなく、両立するものだと考えて
　います。


■ともすると、ビジネスというものは切った張ったの世界だと、そんなイメー
　ジをする人がいるかもしれません。

　ですが、実は、ビジネスにおいて最も重要なキーワードのひとつが、
　愛だと思っています。


■今までは、愛という言葉を使うことが恥ずかしいといいますか、照れがあり
　まして、真顔で語ることはあまりなかったかもしれません。

　しかしながら顧客に対する愛であったり、ビジネスパートナーに対する愛で
　あったり、あるいは自分の周りを取り巻く人たちに対して、愛をもって
　接することというのは、とてもとても大事なことだと思っています。


■ビジネスというものは決して、一方が勝ったならば一方は負ける、ゼロサム
　ゲームではないと思っています。あくまでWin-Winの関係です。

　周りにいる人たちに、何か自分がして差し上げられることはないかと、常に
　アンテナを立て続ける必要があります。


■偉そうなことを言っていますが私自身も、あまりできていない、どころか、
　まったく至らないことばかりで日々反省させられるわけですが、改めて
　こうやって原稿を書きながら、自分自身が忘れないようにしたいものだと
　思います。


■愛というものは、与えれば必ず報いられるものだと思っていますし、
　また、愛を与えてくれた人に対して、愛をもってお返ししたいと思います。

　やはりそういう関係の中で仕事をしていてこそ、本当に仕事が楽しくなるの
　ではないでしょうか。


■切った張った、勝った負けた、取った取られた、奪った奪われた。
　そのパラダイムも、絶対に持ってはいけないということではなくて、時に
　必要なことがあるのかもしれません。

　それでも原則としての立ち位置は、愛。
　常に愛に基づいた行動というものが重要だと思っています。


■もし、愛という言葉がしっくりこない方がおられたら、それは誠実さという
　言葉に置き換えてもいいかもしれません。

　あるいは、信義であったり、思いやりであったり、貢献したいという気持ち
　であったり、助けてあげたいという感情。

　そうした、人間らしい思いを総称して愛というとするならば、ビジネスに
　おいて愛というキーワードは、必要不可欠だと思うのです。
