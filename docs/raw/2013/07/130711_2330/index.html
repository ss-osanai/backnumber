＜今日のピークパフォーマンス方程式＞

 【　「一晩寝て、翌朝起きたらさらに強くなる」のは子供だけではない　 】

───────────────────────────────────



■将棋・囲碁の棋士の自伝やインタビュー記事などを
　読んでいると、

　幼少の頃、メキメキと実力がついていく様を

　「一晩寝て、翌朝起きたら、
　　またさらに強くなっているように感じた」

　という表現に出会うことがあります。



■そういえば、これは棋士だけの話でなく、

　野球やサッカーといったスポーツ選手の口から
　語られるのを聞くこともありますね。


　伸び盛りの子どもに見られる、
　急激な成長から来る、

　「若き才能の煌めき（きらめき）」

　は、誰しも一度は経験するものなのかも
　しれません。
　
　（というか、きっと経験するのでしょう）



■なにを隠そう、この私にも、
　そのような経験がありました（笑）
　

　小学校低・中学年～高学年にかけての
　ほんの一時期、

　見るもの聞くもの読むもの、
　すべてが記憶の倉庫に格納されていく、
　
　という輝かしい才能が発揮された一瞬間が
　あったのです。


　周囲の大人たちからは

　「神童」

　と持ち上げられ、讃えられたもの。


　（そんな輝かしい才能は、
　　わずか数年間たらずで消え失せてしまいましたが 涙）


　
■今さらながら考えてみると、

　私が今、身につけている知識の大半は
　この時期に得られたものであり、

　「そのときの貯金で食べている」

　とすら思われます。


　
■当時、

　故事ことわざ辞典に載っている言葉など、

　目に触れるものの一切が、
　刹那に記憶されましたが（やや大げさ）


　逆に言うと、今、メルマガで使っている
　言い回しや表現など、
　
　当時からほとんど進歩がありません（汗）。



■とするならば、

　「若き才能の煌めき」
　
　が輝く、ほんの一瞬を捉えて、
　
　そのタイミングで生きていく上で必要な知識を
　徹底的に叩きこむ、

　という教育方針はもっと見直されても
　いいのでは、

　と思ってみたりする次第。
　


■などと、長い前置きをいたしましたが、
　実は今、私に

　「若き（！？）才能の煌めき」

　が宿り、

　再び強烈な光芒を放ち始めたようなのです。


　ただし、前回発揮された

　「才能の煌めき」

　と様相が異なるのは、

　
　以前は、文字通り、時々刻々、知識が増加していく、
　という感じでしたが、

　今は、リアルタイムに認識が書き変わっていく、
　というような感じ。



■この数ヶ月間というもの、


　あまりにもたくさんの刺激的出会いや
　魂の震える経験がありすぎて、

　最近はいちいちメルマガに書きません
　（というか書ききれません）

　ので書いていないのですが、
　毎日毎時、

　
　「内面世界が翔んでいく、という感覚」
　　

　を覚えているのです。



■このような経験を通して分かったのは、
　人間いくつになっても

　「才能の煌めき」

　が発揮され得るということ。


　もう一つは

　幼少期を過ぎてからの
　
　「才能の煌めき」

　は

　「知識そのものの強化」

　という形ではなく

　「世界の捉え方の変化」

　という形で現れるもの、ということ。



■ただし、

　「しかるべき外部（の人や知識や情報）との接触が
　　絶対的に必要」

　ということだけは、いずれにも共通いたします。


　私であれば、幼少期の知識吸い上げに
　最も高い効果を発揮してくれたものは、


　▼日本ならびに世界の文学全集
　
　▼学研の各種図鑑

　▼少年朝日年鑑

　▼現代用語の基礎知識

　▼各種、故事・ことわざ・慣用句辞典

　▼その他、父親の書棚に格納されていた本


　といったものでした。



■長じて、おとなになった今、
　認識の書き換えに貢献してくれているものは、


　▼直接的な対人経験（じかあたり）
　
　▼自伝

　▼エッセイ

　▼インタビュー記事


　といったもの。

　
　
■幼少の頃であれ、
　長じておとなになってからであれ、


　日中のしかるべき出会いからくる刺激と、

　その刺激を自分の内に確実に取り込むための
　深く充実した睡眠、


　この二つが組み合わさることによって、
　一挙に異なる世界に

　「翔ぶ」

　ことができる、そんな風に思われるのです。
