　　　　　　　　　【 ソーシャル・キャピタル 】


　信頼や規範、社交ネットワークといったソーシャルキャピタルの
　蓄積は、自然に強化され、累積されていく傾向を持っている。

　ある取り組みにおける協力が成功すると、それによって
　人脈と信頼が構築される。

　これがさらに、将来、別の無関係な取り組みにおける協同を
　促す社会資産となる。従来の資本と同じように、
　ソーシャル・キャピタルを持つ人々は、ますますそれを
　蓄積していく傾向を示す。

　【　成果主義の時代とは喧伝することを要求される時代　】

───────────────────────────────────



■今日お話しするのは、

　わたし（鮒谷）のビジネスの先生である、
　弁護士の高井伸夫先生の勉強会で学んだことになります。



■月に一回、参加させていただいている勉強会で、
　高井先生より


　【 成果主義の時代とは、

　　　　 成果を外部に露出する、表現していく、
　　　　 ないし、喧伝(けんでん)することを要求される時代

　　　　　　　　　　　　　　　　　　　　　　　　ということ 】


　と教えていただきました。

　この言葉はたいへん印象に残りましたので、ぜひみなさんにも
　シェアしたいとおもいます。



■たとえなにがしかの成績を残したとしても、
　それを喧伝していなければ、そもそも評価の対象になりません。

　中小企業であるならば、社長自らが広告塔にならなければ
　ならないこともあるでしょうし、

　企業に勤めているのであればなら、成果を上げて、かつ
　その成果が社内（あるいは社外）に伝わるような
　何かしらの努力をする必要があります。



■以前、この「平成・進化論。」は私自身のプレゼンテーション
　だと申し上げたことがありましたが、

　これは「喧伝」と言い換えてもいいでしょう。

　

■私（鮒谷）という人間は何を学んでいて、どんなことができて、
　どのような人とお付き合いをして、いかほどのアウトプットを
　出しているのか、

　等々を、文字通り「口やかましく」お伝えしているわけです。

　ただそれを無為無策にやったらいけないのでして、
　難しいところは

　「自慢にならないように成果を告知すること」

　だと考えます。



■とはいえ、率直にいって、
　告知することイコール、自慢することです。

　たとえ意図していなくとも。

　そこで、告知する側としては、受け取る側の許容量を考え
　なければなりません。

　あまりにもやり過ぎると、単なる自慢主義になります。



■逆に、なんにもやらなければ奥ゆかしいかもしれませんが、
　実績を挙げていても、その事実がまるで人に伝わらないことに
　なってしまう。

　このあたりの絶妙なバランス取りが必要です。



■私はこの「平成・進化論。」によって23万人の読者さん
　全員から評価していただこうとは最初から考えていません。

　評価してほしい人に評価してもらえればと考えています。

　もちろん、

　一部の読者さんにかたよらず最大公約数的なところに
　照準を当てて情報発信をしていく、

　ということは心がけていますが。



■先日、

　このことを私自身のプレゼンテーションだと言いましたが、
 「喧伝する」という言葉がドンピシャリであると思ったので
　また今日あえてご紹介させていただいております。

　だいじなのは、
　いかにそれを、自慢の臭みを取り除きながら自慢するか。

　これ、ほんとの話です。



■これが成功するかどうかでお客さんが集まってくださるか
　否かが決まる。

　発信不足も過剰発信も、失敗すれば逆に埋没していきます。

　ことに最近はみんなが情報発信の手段を持つことに
　なりましたから、いっそうこの傾向には拍車がかかって
　きています。



■ちょっとこれだけだと分かりにくいかもしれませんので、
　この話はまた明日に続きます。
