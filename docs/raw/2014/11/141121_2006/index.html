　ぼんやりとした私の将来の夢の話についても
　色々とアドバイスを頂き、ありがとうございます！


　お話頂いた内容が、冷酒のようにジワジワと効いてきまして、

　今回コンサルティング頂いた中で印象に残った点として、

　・鍵穴が合わないなら場所を変える。

　→　場所を見つけるために、外に出て人と会うことが大事。

　　アクセルベタ踏みでないと効果が出ない。

　→　断続的に行動するのではなく、一気にアクセルベタ踏みで
　　　行動しないと人は変われない。


　ノーポチの精神で予定を徹底的に詰めて、どんどん行動する。

　一度、破たんするレベルまでやってみると
　色々と捨てるべきものなども見えてくる。


　他にもメタ認知の件等も印象深かったのですが、
　現在、鋭意消化中です(笑)。

　また、全体を通じて実感したのは、私は社外の方と会うといった、
　これまでの習慣にないことを少し始めただけで、
　満足してしまっている状況にあることが分かりました。


　前回のコンサルティングの際にそのような事例を色々と
　お伺いしたにも関わらず、このような状況に嵌まり込んで
　しまっているのは非常に恥ずかしい限りです。

　今回の反省を踏まえ、次回のコンサルティングまでに、
　現在の比較的時間に余裕のある状況を生かして、宿題目標の
　超過達成を目指し、もっと行動を重ねてゆきたいと思います。

 【 熟練の喜びを知らぬ、不幸な人 】

───────────────────────────────────



■同じ種類の仕事を、愚直に、地道に、
　
　そして圧倒的に行い続けることで、
　
　「ある瞬間に、仕事の質が一気に高まる」

　という経験をしたことのある人は、
　どれくらいあるでしょうか。


　
■一度この経験をすると、

　次から、同じ体験を繰り返し、
　起こしやすくなります。


　仕事の質を一気に高めるためのプロセスを
　体で理解することになるから、

　再現性高く同じことを繰り返すことが
　できるようになるのです。



■こうして、何度も何度も、

　「一気に仕事の質を高めることによって
　　自分を作り替えていく」
　
　ことで、


　「入門者」が「初心者」となり、
　
　さらには「一人前」となるでしょう。


　※ここでいう「一人前」とは、
　
　　ある仕事を最低限、
　　一人で成し遂げられる、
　　
　　というくらいの意味での「一人前」。



■さらに、この過程を繰り返すと、


　「中堅どころ」から「手だれの者」「いぶし銀」へ、
　
　ついには、
　
　「匠（たくみ）」「達人」「名人」

　と呼ばれる世界に出られることとなるでしょう。


　この世界に至ると、もはや

　「技術」

　ではなく、

　「芸術」

　の領域といえるかもしれません。



■これとは逆に、

　過去の人生において一度も

　「ある瞬間に、仕事の質が一気に高まった
　　（≒自分が作り替えられた）」

　という経験を持たない人は、
　

　いつまでたっても入門者、初心者、
　一人前レベルをうろうろして、
　
　次の段階に進むことができません。



■仕事の一番の醍醐味（おいしいところ、楽しいところ）は、

　「仕事を通して
　　熟練、熟達のプロセスを味わえる」

　そこのところにあるわけですから、

　その過程を、通らない（通れない）人は
　本当にもったいないし、


　さらに誤解を恐れずにいえば、

　「不幸」

　であるとさえ思います。



■ここまで読んで、

　「実は、ここまでの話に思い当たるフシがある、、、」

　という方は、


　まずは今取り組まれている仕事について、

　愚直に、地道に、圧倒的な量を、
　覚悟を決めて実践してみられてはどうでしょう。

　
　そして、ただ量をこなすだけでなく、

　「より、人さまの役に立つ」
　
　という思いを持って実践することが大切です。



■もちろん私（鮒谷）も、
　ただ、人さまに言うだけではなく、
　
　自らも、現在、
　個別コンサルティングという領域において、

　愚直に、地道に、圧倒的な量を、

　「より、人さまの役に立つ」

　という思いとともに、
　
　クライアントさんに対して
　実践させて頂いているわけです。
