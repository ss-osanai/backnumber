　【　ビジネスの目的は売ることにあらず、友達になることにあり　】

───────────────────────────────────



■私は今、同時並行でいくつかのビジネスに携わっていますが、
　そのいずれの事業においても、心がけていることがあります。

　それは、「ムリな押し売りは絶対にしない」こと。

　それどころか、できることなら

　「お願いしますから売ってください」

　と言われて初めて売る。そんなスタンスでいきたいと思っています。



■「この人はカッコいい営業マンだなあ」と感じる人を拝見していると、
　自分から率先して商品の説明を行っていることは少ないです。

　それよりも、まずは相手と友達になることを優先する。
　
　やがて先方が興味を持って、

　「ところで、それはどうすれば買えるんですか」

　「いくらお支払いすればよろしいですか」

　と切り出されて、ようやく営業に入る。

　すでに購買意欲が十分に高まっているところで販売するから、
　スムーズに商品・サービスとお金とが交換されます。



■ところがそんな友達関係、信頼関係が構築されることもなく、

　「この商品は、かくかくしかじかだから、絶対に買って損はありません、
　　買ってください」

　と言っても、それで成果に結びつけることはむずかしいでしょう。



■私などは、服を買いたいと思ってお店に入っても、
　いきなり店員さんが寄ってきて

　「これはどうですか、あれはどうですか」

　とやられると引いてしまうのですが、

　擬似的にでも友人関係（のようなもの）を築かれてしまうと、
　むげに断るわけにもいかなくなって、話を聞いているうちに、

　「じゃあせっかくだから、あなたから買いましょうか」

　ということになる場合があります。
　きっと、みなさまもご経験があるのではないでしょうか（笑）

　これは洋服であれ、車であれ、住宅であれ、保険の販売であっても
　同じことでしょう。



■私もたまに放談会の告知をさせていただくことがありますが、
　それほど積極的にお誘いをしなくても、非常に多くの方にお集まり
　いただき、本当にありがたく思っています。

　来られた方といろいろお話をさせていただくと、私がこの
 「平成・進化論。」でプライベートな話を含めて情報発信をしている
　ことで、擬似的な友人関係を築くことができていて、
　そのうえでお越しくださっているということを感じます。



■ところが、たとえ仮想的であっても、そういった友人関係を
　構築することなしに、

　「参加されればこんなにメリットがありますよ」

　とご案内しても、今の3分の1、5分の1の方もご参加にはならない
　のではないかと感じます。



■「放談会」と銘打っているくらいですから、その場で思いついたことを
　お話しするという企画。
　したがって、来ていただくまで何が得られるのかも分かりません。

　そのうえ決して安い金額でもないのに、多くの方にお集まり
　いただけるということは、

　擬似的にでも友人関係ができあがっていなければあり得ないことだと
　考えています。



■ものを売る前に、人間関係を築く。

　それが結果としてビジネスになる、という１つの事例として
　ご紹介させていただきました。



■「急(せ)いては事をし損じる」

　「急がば回れ」

　ということわざもあります。

　みなさんの中には、直接営業に携わっておられる方もあれば、
　その他の部門で間接的に販売にかかわっている方もいらっしゃる
　でしょうけれども、

　ものを販売する、その前段階として、まず信頼関係、友達関係を
　築きあげることを心がけてみられてはいかがでしょうか。
