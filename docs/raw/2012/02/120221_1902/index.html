 【 没入の果てに生まれる肥沃な土壌 】

───────────────────────────────────



■先日のメルマガ


　＜3072号 学びの蟻地獄（ありじごく）理論＞

　http://www.2nd-stage.jp/backnumber/ppf_full/2012/02/3072.html


　では、

　「何かに没頭、没入することで
　　　　　　　知の系統樹が広がっていく」

　という話をしました。


　今日は、没頭することで身に付くのは
　知識だけではない、

　というお話しをします。



■私（鮒谷）は文章を書くことが好きで、
　趣味、道楽としてこれを行っています。

　下手の横好きという話もありますが。（汗）


　しかし、たとえそうであったとしても、
　3000日以上の間、

　毎日のように一定量以上の文章を
　書き続けていくと、


　直接的な効果（文章が上達するとか）
　だけではなく、

　想像した以上の世界へと広がり、深まり、
　繋がっていくことを実感しています。



■継続して書くことによって、
　もちろん文章は上手くなりますが、

　書き続けていると、やがて

　「文章道」

　ともいうべき世界に入っていきます。

　
　
■数日、数週間程度ではなく、

　「何ヶ月、何年」

　といった単位で書き続けるうちに、
　
　予期も期待もしていなかった
　能力や考え方が身についてくるのです。


　それはまさに

　「一つの道に没入することによって
　　豊穣な広がりが展開されてくる世界」

　とでも呼べるもの。



■例えば私の例でいえば、

　そうやって広がってきた世界（＝能力）
　として、


　「情報収集力」

　「メモ術」

　「集中力、忍耐力」

　「文章を書くためのタイムマネジメント」

　「書き続けるためのやる気の管理」

　「概念把握、概念理解の力」

　「多面的なものの捉え方」

　「語彙力や表現力」


　といったものをはじめとして、

　ざっと考えても挙げきれないくらいの
　波及効果が得られました。



■ところで、ここまで、

　文章執筆から得られる効果効能の広がりに
　ついて記してきましたが、

　実は今日は、

　「だから文章を書きましょう、書いて下さい」

　という話ではありません。


　何か一つで良いので

　「これ！」

　と思ったことを掘り下げ、

　その道を没頭、没入することによって
　それまで想像もしなかった世界の広がりを
　体験頂きたい、

　というご提案をしたいのです。



■いつもメルマガで書いているように、

　「没頭、没入の境地」


　言い換えれば、

　「◯◯道」

　を極めようとする道のりにおいて
　得られる副産物は想像以上に大きい、

　そんなことをお伝えしたいのです。



■当該分野における知識はいうに及ばず、
　それ以外にも手に入るであろう様々な広がりは

　「人としての豊かな土壌」

　を形成することにつながります。


　そして、この

　「肥沃な土壌から生まれる果実」

　は、やがて、社会に対する価値提供として
　還元されていくのでしょう。


　明日に続けます。


　今日も人生とビジネスを楽しみましょう！
