　となってしまう理由

───────────────────

　（実質6800円（税込））にてご案内中の

　【「身体に編み込む勉強法」音源】

　「あと数日」

　ご希望の方はお早めにどうぞ。



■Twitterで以下の文章を投下しました。


　--------------
　
　「塊魂（かたまりだましい）」という、

　小さなカタマリを動かし、
　いろんな物を巻き込み、

　巨大化させるゲームがある。

　カタマリの表面積が増えるほど、
　より多くの、重たいものを
　くっつけられるようになる。


　学びも同じ。

　知識の総量が増えるほど、学習は加速的に
　広がり、深まり、知恵へと昇華される。

　前倒し学習が重要な所以。


　【鮒谷周史＠弱者の生存戦略】
　https://twitter.com/shinkaron

　--------------


■ここしばらくの間に伝えてきたことと
　矛盾するのでは？

　と思われる方も
　あるかもしれません。

　けれども一向に矛盾するところは
　ありません。



■順序の問題だからです。


　「塊魂」でいうところの、

　【いろんな物を巻き込み、
　　くっつける力】

　を身につけた人は、

　見聞きするあらゆるものを
　高い歩留まりで咀嚼し、
　
　自家薬籠中の物としていきます。



■「くっつける力」とは、

　【当事者意識を持ち、
　　自分ごととして学ぶ習慣】

　なのですが、これは

　【人生におけるOS的な位置づけ】

　にあるものです。



■学校の勉強がなぜ身につきにくいか、
　というと、

　「当事者意識が欠如しているから」

　です。

　なぜ勉強しているのかも分からず、
　英語を勉強していても、

　学ぶの効率は
　極めて悪くなります（経験者談）。



■しかし、

　どうしても外国人の彼氏、彼女を
　作りたい！

　という強烈な動機があれば、

　当事者意識バリバリですから
　すごい勢いで上達するでしょう。



■人生にテーマを掲げれば、

　見るもの、聞くもの、経験するもの、
　すべてが

　「テーマに関係あるか？」

　という自問自答が発動されるので、
　
　そこからの気づきや学びは
　大きなものとなるはずです。



■この状態

　（姿勢とか態度とか
　　精神といっても良いでしょう）

　に入ることなく、

　あれこれ見聞を増やしたところで
　自分ごととして受け止めないから、

　一向に定着することはありません。


　これを

　【スケートリンクを滑っているような
　　ツルツルの学び】

　と称しています。



■つまり、

　【人生を主体的に生きる】

　と決めた人が

　「学びの塊魂モード」

　に入れば、

　前倒し学習をするほどに
　どんどん加速していく、

　ということです。



■OSが整っていない人には

　--------------
　
　小さなカタマリを動かし、
　いろんな物を巻き込み、

　--------------

　という接着力がないから、

　【学んで学ばず】

　の状態が続くこととなる。



■そうならぬよう、

　まずはOS、
　次にアプリケーション、

　の順番を遵守すべきでしょう。



■それほど重要なOSについて、

　深く理解し、
　更新を図って頂くことを目的で、

　【「身体に編み込む勉強法」音源】

　をお届けしています。



■この音源を聴き、
　ある程度までOSの更新が進めば、

　そこから先はバンバン、新たな学びを
　取り入れていくと良いでしょう。


　そこで初めて、以下の世界が
　実現されることとなるはずです。


　--------------
　
　「塊魂」という、

　小さなカタマリを動かし、
　いろんな物を巻き込み、

　巨大化させるゲームがある。

　カタマリの表面積が増えるほど、
　より多くの、重たいものを

　学びも同じ。

　知識の総量が増えるほど、学習は加速的に
　広がり、深まり、知恵へと昇華される。

　前倒し学習が重要な所以。

　--------------

　目的としてリリースしたもの。

　（つるつるして定着しない）学びではなく、

　身体に編み込む
　（基本的な所作を確立する）学びを。
