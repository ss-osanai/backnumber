■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------
　
　自分を鼓舞する言葉、
　何個持っとるけ？

　ワイは瞬時に何百個でも出てくるで。
　長年、日記を書いて蓄え続けてきたからな。

　言葉を持たん人はすぐ折れる。

　でもワイは、無尽蔵の言葉の援軍を
　送れるから異常にタフなのよ。

　----------

　定型規格から外れ、
　生き辛さを感じている人を
　言葉の力で元気にする。

　それがワイのライフワークよ。
　昔のワイがまさにその対象やったわ。


　でもそこから抜け出せた今、

　経験者だからこそ同じ状況にある人を
　応援できると信じてる。

　そして言葉の力を用いれば、
　それほど難しいことではないんだわ。

　----------

　「もう年やから」
　「自分には無理」と思っていれば、

　思っている通りの人生になるんやで。

　31才が

　「もう、おっさん、おばさんやから」

　って言えば
　本当におっさん、おばさん化する。


　80才が「ワイは80才児やで！」と
　胸を張れば、

　好奇心旺盛に動けるんよ。

　言葉を武器にして生きようや。

　----------

　相撲の世界では

　「３年先の稽古をせよ」

　いう言葉があるそうやが、
　キャリアデザインにおいては

　「10年先を見て、努力せよ」

　やと思うわ。


　現役期間が長い分だけ、
　より長期の目線で生きるべき。

　「10年なげー、だるー」

　思うかもしれんけど、
　過ぎてしまえばあっという間よ。

　30代半ば以降の人なら分かるよな。

　----------

　「ADHD＆繊細さん＆勇気
　（行動力とも解釈転換力とも）」

　のトリプルコンボを、

　専門性を発揮できる場所で
　キメられたら、

　マリオのスター状態に入れるで。


　簡単ではないけど、

　その状態に入るために
　10年遠回りする価値はある、

　くらいに思って生きてきた。

　そして賭けに大勝利したったわ。

　----------

　緊張状態ではIQが50くらい下がる。

　人によるやろうけど経験ある人、
　多いんちゃう？

　ワイはIQ下がるを通り越して、
　CPU自体が固まって思考停止になる
　感じやったな。

　持てる知力や技術をフル発揮させるためには
　そのための環境も整えなきゃならん、

　いうことやね。

　----------

　モチベーション高いときに
　動けるのは当たり前。

　やる気があるんだから。

　やる気が湧かんときでも、
　バレることなく期待以上のパフォーマンスを
　発揮し続けるのがプロ。


　そのための仕組み作りまで含めて
　やってきたからプロフェッショナル
　なのであって、

　やる気のみに依存してる時点で
　素人よ。

　----------

　初期値の差が、
　時間とともに雪だるま式に効いてくる。

　だから最初が肝心なのよ。

　あとになって頑張ったところで、

　初期の頑張りを挽回するのは
　（不可能ではないが）かなり難しい。


　それゆえ、どうせ努力するなら今、
　やるべきなのよ。

　あるときから過去の「配当」
　「金利（複利）」で食えるようになる。

　----------

　「●●が●●してくれん」
　「●●よこせ」が口癖の人は地雷やで。

　角が立つから伏せ字にしとくが、

　ワイはそんな風に考えたこと、
　ただの一度もないわ。


　権利や報酬は
　自ら掴み取りにいくもんやろ。

　口を開けば食べさせてくれるのが
　当たり前、はさすがに甘え過ぎ。

　その姿勢がますます自分の首を
　絞めるのよ。

　----------

　多動傾向（ADHD)を

　「自分は今すぐ、その場で
　　やり切るしかない人」

　と定義するとええで。

　この定義のおかげで
　ワイの行動力は劇的に上がったんよ。


　ピッタリくる言葉を編めるように
　なると、人生は随分ラクになる。

　それに言葉は、未来を切り開く
　最強の武器にもなる。

　言葉を使って、困難を突破せよ。

　----------

　あなたが社会不適合なら、
　無理して組織に属さなくてもええんやで。

　一人ビジネス、スモールビジネスでも
　今の時代、普通にメシが食えるからな。


　最初はリスクのない副業からでOK。

　月に5万、10万と増やすうち、
　独立の瞬間は自然と訪れる。

　難しく考えんと、まずは
　小さく一歩を踏み出すことよ。

　----------

　まずは1000円稼ぐこと。
　すべてはそこから始まるんよ。

　大企業の部長とかでも
　給与以外の、月1000円の稼ぎ方が
　分からん人は多いからな。


　だから、それができれば
　自信を持ったらいいんだわ。

　筋トレと一緒で続けるほどに
　筋肉がつき、稼げるようになる。

　最初の一歩を踏み出すところから。

　----------

　ワイほど言葉の力を自覚的に使い、
　運命を変えてきた人間は、

　そうはおらんと思ってる。


　生活習慣を立て直し、

　社会に適合する場所を
　見つけられたのも、

　コミュニケーションの問題を
　解決したのもそう。


　事業を立ち上げ、多少のお金に
　恵まれるようになったのも

　「言葉を操る力」

　を強化したからなんよ。

　----------

　感謝の気持ち、忘れとらん？

　感謝は幸せを引き寄せる
　最強の武器やで。


　感謝するたび心が豊かになり、
　周りとの関係も良くなっていく。

　それに感謝の気持ちは、
　自尊感情も大幅に高めてくれるのよ。


　仕事に忙殺されて
　見落としがちやけど、

　目の前の小さなことに感謝する。

　人生の流れを変える、
　大切な一歩やで。

　----------

　決断力は筋肉のようなもの。
　鍛えんと衰える。

　周りに流され、惰性で生きてる限り、
　決断力は育たんよ。


　失敗していいから
　外食メニューや来週の予定、

　慣れたら旅行先選びなんかも
　じゃんじゃん即決していくんやで。


　決めないことが気持ち悪くなる頃には、

　人生に変化が起き始めているのが
　実感されるはず。

　----------

　催眠にかかってない人なんか、
　この世に一人もいないんよ。

　映画館や某遊園地にいる人は、
　どっぷり催眠にかかってる。

　で、そこから出ると我に返るんよ。
　つまり催眠が解ける。


　「できない」「もう年だから」

　が口癖の人も同じく催眠よ。

　----------

　誰でも生きていくのは大変で、
　苦しいこともある。

　でも困難から逃げとるうちは、
　影のようにどこまでも追いかけてくる。

　だから戦うしかないんだわ。


　覚悟を決めて立ち向かえば、
　困難も怖気づいて去り、

　以後、同種のことは起こらなくなる。

　戦うか逃げるか、
　毎回の選択次第で未来が変わるんよ。

　----------

　人生は強みと弱みが配られる
　カードゲームみたいなもの。

　強カードはレア、
　弱カードは山ほど配られる。


　面白いのはカードの組み合わせ次第で、
　大逆転も可能になるところ。

　ルールさえ分かれば
　誰でも楽しめるゲームやが、

　戦い方が分からんまま
　やみくもにプレイしていたら、

　つまらん運ゲー、クソゲーにしか
　ならんのよ。

　----------

　どれだけネガティブな出来事にさえ
　「良い意味」を見出す思考トレーニングは、

　慣れると中毒化して
　クセになる習慣やで。


　これが日常になると

　「自分は運がいい」

　という信念が育まれ、
　自尊感情が高まる。


　未来の可能性を信じられ、
　行動力もついてくる。

　「日々、起こる出来事の意味」

　の探求を楽しむといいよ。

　----------

　ルールは「従う側」やなく
　「作る側」に回ることやで。

　キャリアでも事業でも、
　これが分水嶺になる。


　ここで気をつけたいのは、

　手前勝手なルールを押し付ける
　のではなく、

　説得力ある物語と論理で
　相手の心を動かすこと。

　ルールメイキングの能力を有する者が
　ゲームをひっくり返せるようになる。

　----------

　実は世のほとんどの人が「規格外」。

　みんなバレんように必死で頑張っとるが、
　うまくいってるように見える人も

　ストレス抱えて幸せと程遠かったりする。


　そんな人はまず
　「私は規格外」と認めること。

　次に自分独自の規格を作り、
　「規格外の存在」になればいい。

　これが真の意味で、
　自分らしく生きるということよ。

　----------

　好きなことで日銭を稼ぎつつ、

　同時に安定した経済力と
　余剰時間を生む仕組みを作る。

　面白いことに、
　この仕組みづくりそのものが、

　軽やかな人生への準備運動に
　なってることも多いもの。


　まずは、そんな世界に生きてる
　自分を妄想してみる。

　これが理想とする、
　幸せな人生の一歩目になるかも
　しれへんで。

　----------

　人生こそ最高の名作ゲームやと思っとる。

　ただ攻略法を知らんうちは
　理不尽な鬼畜ゲーにしか見えんのよ。


　学びを広げ、深めて
　攻略手段を一つずつ増やし、

　組み合わせる中で
　環境適応力が付いてくる。

　こうして攻略本が厚くなるにつれ、

　「鬼畜ゲー」は「神ゲー」へと
　変わっていくんだわ。

　----------

　「平時の備え」が
　人生を決めることがある。

　公私にわたる様々なリスクを想像し、
　脳内シミュレーションを重ねる。

　それが本当の危機管理。


　緊急事態は一刻を争うのに、

　そんな時に
　「考えてから動こう」じゃ遅すぎる。

　平和な今こそが、
　準備するのに最適なとき。

　明日になったら遅いかもしれへんで。

　----------

　「今いる場所で頑張る」より先に、
　「向いてる場所を探す」方が賢い選択やで。

　向いてない場所では、いくら気合いを
　入れても全力を出せんし、能力も発揮できん。


　回り道に思えても勝てる場所を
　見つけることを優先する。

　そこを見つけた後、
　全資源を集中的に投下する。

　この順番を間違えんことや。

　----------

　以下はワイが目指してきた生き方。

　まず確固たる人生の基盤作り。

　経済的側面と非経済的な側面の
　両方を整える必要がある。

　次に不測の事態への万全の備え。


　以上と同時並行で、

　ささやかな日常に幸せを感じられる
　「詩人の感性」を涵養する。

　それなりにうまくやれてきたように
　思っとる。

　----------

　平凡に生きることすら
　難しい時代やからこそ、

　まずは「平凡」の土台を
　築くことが大事。

　「平凡」を極めた先に
　「非凡」への道が開かれると思ってる。


　非凡への近道を探すんやなく、

　平凡の先にしかない
　本物の非凡にたどり着く。

　ワイはこの思想と行動に
　フルベットしてきたが、

　30年後の勝利を確信しとる。

　----------

　不確実な時代やからこそ、

　原理原則や王道に
　立ち返る必要があると思ってる。

　一瞬間の成功事例に飛びつくのではなく、
　普遍の定石を理解したい。


　たまたま成功した人を真似て、
　短期で破滅する人があまりに多いのは
　今の時代の落とし穴ちゃうかな。

　目先の派手さに惑わされず、
　本質を見定める目を持ちたいな。

　----------

　調子がええ時こそ要注意やで。

　成功のリズムに乗ってるときほど、
　もう一度兜の緒を締め直すべき。

　不思議なもので、
　備えを怠る人に限って、
　来てほしくない不測の事態が訪れる。

　人生の皮肉な法則と言えそうよな。


　いずれにせよ、どんな時も
　重心は低く、備えは厚く。

　これが生き残りの鉄則よ。

　----------

　汚い言葉はあんまり使わん方がええで。

　一時的に効いても、
　強い副作用があるのよな。

　吐いた言葉が催眠的に
　無意識に染み込み、
　悪影響を及ぼすことがあるんだわ。


　短期の成果が長期のダメージに
　繋がることもある。

　言葉の吟味、選択は
　慎重に行うべきなんよ。

　----------

　マウンティングされたら適当に

　「すごいですね」
　「さすがっすね」

　って適当に相槌打っておけば
　いいんだわ。

　感情を揺らす必要は一切なし。

　確固たる軸があれば

　「やっとるなー」
　「いるよな、こういうの」

　と眺めていればいい。

　どっしり構えて、
　自分のやるべきことをやる。

　それだけでええんやで。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。

　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
