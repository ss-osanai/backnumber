7014号 「中流崩壊」に思うこと

■上述の通り、

　【NHKスペシャル
　　“中流危機”を越えて】

　の１回目、２回目を続けて見ました。



■かつて「一億総中流」と
　言われていた日本において

　「中流層が底抜けしつつある」

　といった趣旨の内容でした。



■番組を見た感想を端的にいえば、
　今の世の中は

　【希望欠如社会
　（未来に希望を見つけにくい社会）】

　なのかもしれないということ。



■私が本気で起業しようと思ったのは、
　今をさかのぼること1997年。

　ついこないだのような気がしますが、
　なんと25年前。


　それまでも漠然と

　「このままではいけない」

　と思っていたけれども、起業スイッチが
　完全に入ったのがこの年。



■97年に三洋証券、北海道拓殖銀行、
　山一證券と立て続けに経営破綻。

　その翌年（98年）には
　長銀と日債銀が破綻。


　いよいよこれはいかん、まずい、
　日本沈没だ、、、

　そんな不安と焦りと恐怖を感じ、震えて
　いたことを番組を見て思い出しました。



■おそらくは番組に登場された方々と
　同種の不安や恐怖を抱えていたのです。


　その当時から既に「中流崩壊」の
　兆しはありましたし、

　その流れを止めることはできないのは
　自明でしたから。

　今の「中流完全崩落状態」は、
　当時の延長線上のことでしょう。



■能力も経験もほとんどないのに
　人生や社会を舐めていた私にとっては、

　このまま日本が底なし沼に
　沈んでいくと、

　その渦に巻き込まれて自分も沈み、
　そこから浮かび上がれないのでは、

　と恐れました。



■その恐れは

　【希望の欠如（絶望）】

　につながりそうになりましたが、
　すんでのところで、

　【ショック療法】

　みたいな効果が発動されました。

　（「自己啓発」のおかげです。
　　それがなければ、まず間違いなく
　　一緒に沈んでいたでしょう）



■そのわずか数年後、今度は
　自分の勤務する会社が潰れて、

　さらなるショックが訪れるとは
　当時は夢にも思っていませんでしたが、

　それも

　【二度目のショック療法】

　みたいな感じとなりました。



■都合２回、強烈な電流を
　心の臓に流され、

　さすがに鈍感な私（鮒谷）も
　飛び起きましたが、


　あれほど強い衝撃を受けなければ
　本気になれなかったはずなので、

　一連の出来事があって今がある、
　と思っています。



■今、もしあなたが、

　未来に希望が見えないとか、
　不安や恐怖や焦りがあるとするなら、

　考えようによっては
　悪いことではないのかもしれません。



■人間、順風満帆の状態のときには、
　ハングリー精神は生まれません。

　大逆転は本気になって全力を傾ける
　ところからもたらされるのです。



■自身を顧みても

　【逆境にあることこそが
　　最大の資源】

　であったと思います。



■たまたま、逆境にある人は
　恵まれている人に比べて

　【爆発的な行動力をもたらす
　　エネルギーの源泉を持っている】

　ということですから、
　むしろそこを喜んでもいいくらい。



■その状況にあるときには
　そんなこと決して思えないものですが、

　一番苦しいところを乗り切った後、
　振り返って必ずそう思えるもの。


　恵まれていない環境にいること自体が
　最大の武器になることがあるのです。

　“中流危機”万歳です。


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】

　■希望が失われつつあるときが
　　最大のチャンスとなる。

　■未来の不安や心配、恐怖や焦りが
　　一発逆転をもたらす本気を促すからだ。
