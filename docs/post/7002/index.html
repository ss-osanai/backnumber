7002号 毎日が深呼吸

■最近、

　「引いてみる」

　ことを覚えました。

　いや、覚えました、という表現は
　正確ではないですね。

　もともと意識していたことでは
　ありますから。



■しかし、ここ何ヶ月かの間で

　「とりあえずは人生を一周」

　の境地に到達したことで、


　自分の人生（将来）や
　世の中について、

　より一層、俯瞰して捉えられる
　ようになったように感じています。



■なんでもそうですが

　【近寄りすぎると
　　かえって分かりにくくなる】

　ものですね。

　離れようと務めていたつもりでは
　ありましたが、

　それでも今までは、
　いろんなものに

　「近寄りすぎていた」

　のかもしれません。



■もちろん、離れ過ぎてしまっても


　【せっかく見えないものが
　　見えるようになったのに、

　　地に足がついていないから
　　実生活はなにも変わらない】


　ということにもなりかねないので
　やっぱり

　【両方の視点が大事】

　と言えそうです。



■とはいえ、

　私たちはどちらに偏りがちかといえば
　それはもう、圧倒的に

　「現場（近寄り過ぎ）」

　のほう。

　なんといっても、ほとんどの人は

　【一日の大半、
　　身柄を「現場」に拘束されている】

　からです。



■仕事や家庭、それらにまつわる
　あれこれ（子育てとか資格取得）など

　【至近距離で凝視しなければ
　　期待される成果を生み出せない】

　対象に注力するために、

　時間と空間の自由をかなりの程度、
　占有されている、

　ということ。



■もちろん、生きていく上で
　こうしたことは重要、

　というよりも、
　そうしなければ生きてはいけない、

　それがこの世でありますが、


　それでも現状を追認するだけではなく、

　一日のうちの、
　ほんのわずかな時間だけでもいいので、

　「現場」

　から引き剥がし、

　【異なる時間軸と空間軸で
　　自分の座標を捉え直す】

　重要性を訴えたいのです。



■いったん「今、ここ」から目を離し、

　「人生」とか「未来」とか「世界」
　といった観点から、

　自身を再認識、再評価するということ。



■昔から私（鮒谷）は、

　このモードを発動するための
　スイッチとして

　「静寂空間（耳栓）＆ゆったりした椅子
　　＆深呼吸」

　を提唱してきました。



■そこまでメルマガでは
　書いてこなかったかもしれませんが、

　以前の年間プログラムとか
　個別コンサルの場で聴かれた方は
　多いと思います。



■それはともかく、
　深呼吸をきっかけとして、

　リラックスモードに入って
　自分をはるか上空から眺める、

　そんな意識を持たれると、


　今の悩みがバカバカしくなったり、

　どこにボトルネックがあるのか
　見えてきたり、

　ボトルネック解消の道筋が
　捉えられたりするものです。



■なにはともあれ

　【浅い呼吸は最悪】

　です。

　目先の心配に追い回され、
　不安や緊張で将来を考えられなくなる、

　そんな「悪循環モード」に
　導かれてしまいます。

　【浅い呼吸に気づいたら、
　　即、深呼吸】

　をお勧めする次第。



■有り難いことに私（鮒谷）は、
　最近、わざわざ深呼吸しなくても

　【毎日が深呼吸】

　みたいな生活をするようになって、

　今まで以上に、いろんなものが
　見えてくるようになりました。



■こうした経験を経て
　人間、いくつになっても

　【新しい視点、感覚、感情を
　　開拓できるもの】

　だと、ますますこれからの人生が
　楽しみになっている今日この頃。



■そんな境地に到達できたのも、

　「現場」に張り付いているときから
　できる限り「俯瞰視点」を意識して、

　深呼吸しまくり、
　遠くから自分を捉え直し続けたから、

　だと思っています。



■以下の音源は

　「あなたの人生、生活を
　　俯瞰して捉える」

　ためのツールになると思います。


　深呼吸したあと、お聴きになると
　なお一層効果的（？）、

　かもしれません。
