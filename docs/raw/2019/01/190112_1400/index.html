【あと一両日】鮒谷が実演販売のオヤジに教わった「圧巻セールス話法」とは

2019/01/12 14:00 
■こんにちは、鮒谷です。


　私（鮒谷）の、
　興味・関心の在り処（ありか）の一つに


　「販売のプロセスそのものを
　　楽しんでもらいながら、

　　ときに感心し、ときに驚き、
　　ときに喜び、ときに苦笑し、
　
　　ときに失笑さえしながらも、
　
　　販売している商品やサービスを
　　購入してもらうメカニズム」


　があります。



■寅さん（映画）を見たことのある人なら
　すぐにイメージできると思いますが、

　テキ屋の口上は、それ自体に
　上質のエンタメ要素が含まれていますよね。


　私（鮒谷）の売り方は、見た目、
　テキ屋の口上そのものではなく、

　というよりも、

　むしろ全く異なるものですが、
　狙うところは近いものがあり、


　極力、嫌味、臭みのあるセールス色を
　漂わせることなく、

　楽しみながら、学びながら、
　
　気がついたら買っていた、
　申し込んでいた、

　そんな状態を作り出すことを
　目論んでおります。



■すなわち、

　「なぜだか分からないけれども、
　　知らない間に申し込んでしまっていた」

　という手品をかけられた人のような、
　

　あるいは、

　「『我田引水』的売り方に釣られて、
　　思わず、その売りに乗ってしまった」


　という心理を作り出すための

　「あれやこれやの仕掛け」

　を意図的に埋め込んでいるのです。


　
■そして、

　ある種の人たち
　（＝ビジネスで成果を上げている人）

　は、

　そうした仕掛けの発見自体に
　気づきや喜びを覚えられるもののよう。


　仕掛けにわざとハマって（＝買わされ）、
　そこに喜びを見出す傾向もあるようです（笑）

　※当然といえば当然ですが、

　　そうした人ほど、さらに
　　ビジネスセンスが磨かれていく模様。



■とはいえ、

　私（鮒谷）が仕掛けたタネの全部を
　見破ることは難しいので、

　仕掛け（マジック）の種明かしは
　セミナーか個別コンサルでいたしますよ、

　というと、

　さらに一定の割合で、お越し下さる方が
　いらっしゃいます。

　（ありがとうございます）

　※圧巻セールスセミナーなどは
　　まさに

　　「手品の種」

　　を包み隠さず、お伝えするセミナー。



■こんな風に、

　どうして買わされてしまったのかを
　実地体験（購入）する、

　そこから始まるお勉強、

　という世界にいざなう、


　こうしたエレガントな導き（！？）も、
　ある種の

　【ストーリーテリング技術の発動】

　といって良いかと思います。


　
■嫌味、臭みのあるセールス色を
　漂わせることなく、

　楽しみながら、学びながら、
　気がついたら買っていた、申し込んでいた、

　そんな状態は

　（すでに第一線からは退かれましたが）

　ジャパネットたかたの、高田明さんから
　モノを買ってしまう感覚と近い！？

　のかもしれません。



■商品販売それ自体に
　エンターテイメント要素を入れ込むのは、

　「古くて新しい、モノの売り方」

　です。


　温故知新、不易流行の精神で、

　今のような供給過剰の時代だからこそ、
　むしろテキ屋の口上のような、

　「エンタメ的販売要素」

　を入れ込んでみてはどうでしょう。



■この御時世、
　そんなことは嫌がられるのでは、
　
　ではなく、むしろ

　そうした意表を突く売り方
　（楽しい売られ方、学べる売られ方）

　だから売れていく、

　というのが今の時代。


　だからこそ私（鮒谷）自身、

　そうした「技芸」を
　まだまだ磨き抜く必要がある、

　と思っているのです。



■今だから明かしますが、
　実は私、その昔、

　各種便利グッズを展示会を行脚し、鬼のように
　実演販売で売りまくるオヤジに付き従い、

　セールスの片棒を担いでいた、

　そんな時期がありました。



■彼のセールストークを
　注意深く観察していると、

　まさにテキ屋の口上的な、
　面白おかしく、興味をそそる、
　
　「基本フォーマット（？）」

　に加えて、
　
　「当意即妙、臨機応変の口上」

　が加わるから、天下無敵。



■隣で販売の手伝いをしていた
　私（鮒谷）もまた、その口上に聞き惚れて、

　気がついたら、必要もないのに
　なけなしのカネをはたいて、
　
　「襟、袖の汚れを一瞬にして落とし、
　　真っ白にする魔法のドイツ製チューブ石鹸」

　とか、

　「いつまで経っても切れ味の衰えない、
　　研ぐ必要のない包丁」

　とか、

　「洗濯用多機能ハンガー」

　とか、

　「つけると水が浄化（？）される
　　シャワーヘッド」

　とか、

　手伝いとして付き従っているはずの
　私自身、
　
　オヤジの口上につられて、
　ついつい、買わされたものでした。


　「襟、袖の汚れを一瞬にして落とし、
　　真っ白にする魔法のドイツ製チューブ石鹸」

　なんか、

　ただ、汚れが周辺に広がるだけの
　ひどい欠陥商品でしたけれどもｗ

　
　
■こうして、

　口上を聞かされ、心を動かされ、
　ダメな商品だと分かっていてさえも（！）

　幾度も買わされながら体で覚えた

　【一連の感覚変容プロセス】

　が、

　まさか後に営業の現場で
　生きてくることになるとは、

　その当時は、夢にも思いませんでしたが。


　何ごとも経験しておくべきものです。

　無用の用。

　やってて無駄なことなど
　何一つとしてありません。



■少々、話が脱線いたしましたが、

　供給過剰、供給過多の
　今の時代だからこそ、

　【売り方を「芸」のレベルに高める】

　という発想は、

　もっともっと、強調されても良いように
　思われます。



■そんな

　【販売という芸】
　
　の中でも、

　【物語の持つ力を活用する】

　という切り口は、
　これからの時代においては必須、

　と言えると思います。
