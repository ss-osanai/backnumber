＜今日のピークパフォーマンス方程式＞

 【　利害打算で情報発信！　】

───────────────────────────────────



■一昨日のメルマガで、


　＜好感度100　←←・・・・・・・・・・・→→  嫌悪感100＞


　という図（？）を紹介しましたが、


　いったん左に転がり始めれば、
　基本的には、どんどん加速度がついていきます。


　反対に、何かの拍子に右側に転がれば、

　これもまた、ちょっとやそっとでは止められぬほど、
　亢進していってしまうのです。



■だからこそ、繰り返しになりますが、
　
　特に初対面の出会いにおいては、


　「細心の注意を払い、念には念を入れ、
　　左側に転がるように、仕掛け（事前準備）をしておく」


　ことが大事です。


　
■さらに、秘中の秘をお伝えするならば、


　私（鮒谷）がメルマガを書いているのは、
　
　「そもそも、出会う前に印象形成しておこう」

　という目的あってのこと。


　「なんと打算的な！！！」

　と軽蔑される方もあるかもしれませんが、
　その通り、

　「120％、打算」

　です。



■でも、それでいいと思っています。


　少なくとも私にとって、

　「打算なし、
　　出たとこ勝負で人に嫌われ続けてきた前半生（？）」

　と比べて、


　「打算まみれで、しかるべき準備と注意によって、
　　そこまでは嫌われなくなった後半生（！）」

　どちらが快適かと問われれば、
　
　明らかに後者になったときのほうが100倍、
　幸せになれたから。

　
　かつ、補足すると、ご一緒する相手もその方が、
　おそらくきっと、気持ちいい。



■好感度形成ツールとしてのメルマガ、
　
　その意図が期待通り、
　実現できているかどうかは定かではありませんが、

　当人はいたってまじめに、
　来る日も来る日も、シコシコ、シコシコ、

　打算まみれで執筆を続けております。



■個人でメディア
　（ブログ、メルマガ、フェイスブック、ツイッターなど）

　を運営、運用されている方は、
　
　「何も考えず（＝思いつくまま）発信、更新する」

　ではなく、


　少しなりとも

　「（事前にどんな影響が及ぶか想像した上で）
　　印象形成戦略の一環として発信、更新する」

　ことに留意してみられてはいかがでしょう。
