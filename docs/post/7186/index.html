　今も全く変わっていませんし、

　こうした生き方、生き様を選択してきて
　正解だったと思っています。



■どんな分野においてもそうですが、
　結果を出すには

　---------------

　１、筋の良い戦略
　２、戦略に基づく実行

　---------------

　の両方が大事であり、

　いずれか、あるいはいずれもが欠ければ
　期待される成果を出せない、

　ということになるのだとすれば、


　過去を振り返って、

　１、まあまあ筋の良いと思われる戦略を
　２、それなりの実行力で遂行

　してきたのではないかとも思われます。



■万人にとって、
　とまでいうつもりはありませんが、

　ある種の人にとっては
　最良の戦略ではないか、

　とは考えているところです。



■そんなわけで、これからも

　【バカにされても愚直に前進】

　の姿勢を堅持してまいります。
