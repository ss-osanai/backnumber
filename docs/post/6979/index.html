6979号 あなたは「う○こ」しているときに何を考えていますか？

■先日来、

　「オムニバス音源」

　のご案内をしてきました。

　３つの音源をお聴きくださったＨさんから
　以下のご感想を頂戴いたしました。


　（ここから）
　-----------------


　鮒谷さん・・・

　今回、音源の案内がはじまったとき、

　こ、これはぁ！？
　ほしかったやつやぁあああああああああ

　ってなってしまったのです。


　ほしい音源がたまっているので
　まとめ買いしよ～♪

　と考えていたのに
　思わずポチってしまいました。

　カナダにいたときに

　「ひたすら元気がでる音源、
　　ありませんか？」

　とお尋ねさせていただいたことが
　あったくらい、

　ほしいな、とずっと思っていたからです。


　とうとう音源化されてしまい、
　これは買わずにはおれねぇ・・・(?´ω`? )

　となったのに
　販売まで焦らすパターンｗ

　やっと聴けました。


　それにしても

　本気の恋ってなんなん？

　って思っていたんですが、

　ずっと、どうすれば？と思っていた
　問いの答えがありました。


　それが・・・

　"行動を起こすための
　 問いの質をあげるには？"

　その答えがこの音源には
　つまっていました。


　それを一言で答えてくださっていたのが

　「願いなし、問いなし、動きなし
　　（＋成果なし）」

　と

　「問いの質と数、そして頻度」

　です。

　強烈な願いがないと
　そもそも問いは発動しない。

　しかし一度強烈な願いが設定されたら
　問いが発動し、数と頻度がともなう。

　そしてそれが質になる
　（という風に理解をしたのですが）

　質的変化を起こすほどの
　問いかけの"頻度"が起こってないな
　ということに気づきました。


　頻度がともなっているか？の指標が、
　う〇こしてるときも考えてますか？

　（＝何もしてないときも
　　　願いについて考えてるか）

　という問いかけがありましたが・・・

　うん〇してるときは
　考えてなかったわ、と。

　頻度が足りてないのですね、と。
　気付きました。


　しかしながら、頻度は自力や仕組みで
　ひねり出すにも限界があります。

　これらの仕組みをもっとわかりやすく
　比喩に落としてくださったのが

　「恋に落ちた」

　あの感覚で想いつづけてますか？
　と問いかけてくださったことです。


　恋した相手のことは、
　朝も昼も晩も、トイレの中でもｗ

　何度も

　「好きなことを思い出し」
　「感情をもってかれ」
　「動き出す」

　ことが、不可抗力で起こり、

　思い出すたびに、
　また君に～恋してる～となりますが、

　今の仕事に対し、
　この恋した状態に自分は入れているか？

　を考えたとき
　まだ恋焦がれてないな、

　ということに気づきました。


　そもそも

　「"行動を起こすため"の問いの質を
　　どうすれば・・・」

　って思ってる時点で、
　今の立ち位置ではまだしっくりきていない、

　というかしっくりこさそうとさせていた
　自分がいたことに気づきました。


　つまり、まだ恋の対象に
　出逢えてないとするなら、

　これから冒険するたびに、今以上に
　素敵なところに出逢える可能性があるのか！

　と嬉しくなったのでした。

　（これが今回の音源でえた
　　一番大きなふるえる気持ちです）


　元気が・・・

　（恋音源のとき）ムクっ

　（実現した未来音源のとき）むくむくっ

　（熱狂音源のとき）
　MUKUMUKUMKUUUUUUUU！

　と元気が手を変え品を変え
　わいてきました。

　この音源を聴いて、すぐに実行できることは

　「"わたしはできる"を強化することや」

　と思いました

　跳び箱　できるできる！
　https://youtu.be/AO2Yrcoe5iA

　みたく、ちょっと仲間を集めて
　できるできるをひたすら言い合う会を
　ひらこうと思ってます。第二領域（笑）


　鮒谷さんの音源は結局何を聴いても
　元気はでるんですが、

　今回の音源はアンパンマンが
　新しい顔になるときみたいな、

　元気がリンリンに湧きあがってくる
　ような音源でした。

　今回もありがとうございました！

　（ここまで）



■Ｈさん、いつもありがとうございます！


　音源（特に「本気の恋」音源）の内容に
　触れてくださっていますが、

　つまるところ

　【対象と恋に落ちる】

　ことがなければ心が動かず、
　身体も動かず、

　結果として何も起こらない、
　ということですね。



■恋愛ほど人の心を動かすものは
　ありませんから

　（そこからストーカー騒ぎや刃傷沙汰に
　　なることすらあるわけで）

　それと同じほどの莫大な
　エネルギーを善用させられれば、

　たいていの願いは
　叶うに決まっている、

　ということ。



■とするならば、極論、

　「恋に落ちる対象が見つかる前は
　　何をやっても無駄」

　となるわけです。



■たとえるならば、

　恋に落ちてもいない人に
　夢中になろうとしても、

　それは無理な相談ですよね、
　ということです。



■生理的に無理だと思った人には、

　なにをどう頑張って
　好きになろうとしても無理、だし、

　その人に近づいてこられるだけで
　セクハラ＆ストーカー認定、

　となるように、

　心が拒絶していたら
　その先に進めることはありません。

　そしてそれは
　恋愛に限った話ではありません。



■私（鮒谷）はこれまでの人生で
　巷間、

　「学んでおいたらいい」
　「身につけておいたらいい」
　「手に入れておいたらいい」

　と言われている、
　さまざまなものを捨ててきましたが、

　以上のような考えあってのこと。

　心が震えない対象には
　時間を割くだけ無駄だから。



■こうした考えは
　私（鮒谷）にとっての

　【至高の生存戦略】

　となりました。



■世間一般の考えと真逆ですが、

　いちはやく、この戦略に従ったことが
　奏功し、すなわち

　【戦略が当たって】

　今のような、機嫌よく毎日を
　生きられる世界に出られました。



■何をやるかよりも
　何をやらないかのほうが大事、

　とも言われますが、

　やらないと決めたことを
　バンバン捨てていったほうが、

　残った時間投入対象の純度は
　ドンドン高まるというものです。



■こうした生存戦略について
　言い表されたのが

　「好きこそものの上手なれ」

　とか

　「これを知る者は
　　これを好む者に如かず。

　　これを好む者は
　　これを楽しむ者に如かず（論語）」

　といった言葉です。



■一旦、好きになる対象が見つかれば、
　（恋に落ちれば）

　ほうっておいても、なんなら

　「う○こ」

　しているときですら
　想ってしまうもの。



■それくらい強い思いを傾けられる
　対象が見つからないうちは、

　我慢してある程度までは頑張れても
　そこから先には到達できない、

　ということですね。

　まあ、「頑張る」とか言ってる時点で
　ハナから勝ち目はないんですが。



■夢中になれる対象がないときには
　無理をせず、

　【恋に落ちる対象を探す時間
　（モラトリアムとも）】

　を大事にしたほうがいい、
　急がば廻れですよ、

　と私（鮒谷）がお伝えしているのには
　理由があるのです。



■心動く対象がなければ、

　まずはあれこれインプットに努め、
　いろんな情報やメッセージに触れて、

　そうした対象を探し求め、
　見つけること。



■そうして好きになった対象が
　見つかることで、元気が

　ムクっ
　むくむくっ
　MUKUMUKUMKUUUUUUUU！

　と湧いてくるものです。
