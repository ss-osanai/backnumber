＜今日のピークパフォーマンス方程式＞ 

【 「自信を持ちつつ、自惚れず」の姿勢を堅持する 】 

───────────────────────────────────



■継続は力なり、

　という言葉がありますが、
　本当にそうだと思っています。

　
　日々、小さな取り組みでいいので、
　継続していくと


　【 毎日の小さな取り組み

　　 → 時間経過による蓄積

　　 → 複利効果】


　のコンボのパワーが発動され、

　時間の経過とともに、想像以上の
　価値がもたらされることとなるのです。



■実のところ、

　私（鮒谷）自身、毎日当たり前のように
　このメルマガを、

　粛々と14年以上、継続配信してきましたが、
　それが時間の経過とともに、

　よもや、これだけの成果を
　もたらしてくれることになるとは、

　夢にも思っていませんでした。



■それが、あるとき、

　メルマガの執筆と配信を通して
　淡々と継続＆蓄積してきた知識やスキルの強化、

　あるいは、

　粛々と広げてきた人とのつながりが
　とてつもない財産となっており、

　さらには複利効果が
　発動されていることに気づいた瞬間、

　「そこはかとない自信」

　が

　「絶対的な自信」

　へと切り替わることとなりました。



■このように

　【継続＆蓄積の力】

　は、あるライン（閾値）を超えると、

　【絶対的な自信のバックボーン】

　となるもののようです。



■もちろん、

　その自信が自惚れに変わっては
　危険なので

　「（自信の）よい部分」

　は残しつつ、同時に


　【まだ足りない、まだ足りない、まだ足りない、
 　 と思って、

 　 （一日も休むことなく）

　　投入できる限りの最大限の時間を
 　 対象分野に投下し続ける】

　そんな意識を大切にしたいと考えています。



■ここまでの話をまとめると

　「自信を持ちつつ、自惚れない」

　という状態を維持し、
　継続し続けられる姿勢を堅持することが、

　長期的かつ、安定的に、
　自らを作り変えていくための秘訣、

　といえるのではないか、
　ということになりますね。

　

■ぜひ、あなたにも、
　ある程度の時間はかかるでしょうけれども、

　【 毎日の小さな取り組み

　　 → 時間経過による蓄積

　　 → 複利効果】

　の破壊的なパワーを体感していただきたい、
　そんなふうに考えています。
