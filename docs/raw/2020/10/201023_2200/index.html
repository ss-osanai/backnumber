　（ここから）
　-----------------------


　常に自分自身を
　モニタリングする言葉として、

　「気を入れているか」
　「気が入っているか」

　といった言葉を一回も回さない人、
　一日1回、回す人、
　
　3回も5回、10回も回す人とでは
　違う結果になるのは当然。


　-----------------------


　もちろん人間だから、忘れてしまった、
　集中できなかったということもあろう。

　しかしそれでも
　戻ってこれるポイントがあればいいが、
　
　なければ
　延々迷走してしまうから、

　まず、そうした言葉を持ち、
　基本姿勢を確立することが重要。


　-----------------------


　ここで
　
　「あなたは震えていますか」

　という、もう一つの問いを
　お渡ししたい。


　不感症（感動しない）が
　常態（いつもの姿）となってしまうと、

　震えていないのに
　それが当然と思ってしまうことになる。


　-----------------------


　あらゆるエネルギーの源は
　感動にある。

　凄いな、自分もやってみたい、
　こうなりたいな、かっこいい、美しいな、、、

　そんな風に「感情が動く」から
　「感動」するのである。


　-----------------------


　感動や震える、という言葉が
　（脳内）辞書に入っているか。

　辞書に入っていても
　久しく開いたことがない、

　という人も多いのでは。


　-----------------------


　自分が何をやりたいのか、
　何に喜びを見出せるか、

　もはやわからなくなっている。

　何をやりたい、
　と尋ねてもやりたいことはない。

　それは内発的動機がないということ。


　-----------------------


　自分がどこに行きたいか、

　はっきりと定まっていなければ、
　どうにもならない。

　原動力がない
　＝スイッチを切っている、

　ということ。

　「気、云々」よりも
　いっそう深い話。


　-----------------------


　「気」を増大させるには

　「内発的動機」
　「自分を躾ける」

　の両方とも、
　高い水準にあることが必要。


　-----------------------


　内発的動機は行動の核。

　自らを躾けようとしても
　目指すべきところがないと、

　「何のために
　　やっているんだっけ？」

　となる。


　-----------------------


　人生を通して、

　こちらの方向に進んでいきたい、
　これを実現させたい、

　というところに紐づくからこそ、

　「気が入る」

　こととなる。


　-----------------------


　目標は一回決めるだけではだめで、
　つねに更新し、問い続ける必要がある。

　目標のことを考えるだけで
　興奮する、高揚する、
　未来のことを考えるだけで興奮する。

　それが常態、それがこそが
　「その気になった」状態。
　

　-----------------------　


　没頭没入モードの人は、

　夢遊病者、あるいは
　白昼夢を見ている状態。


　未来に生きている、

　他から見たらあの人なにやっている、
　と思われるかもしれないが、

　未来が臨場感を伴う現実のものとして
　感じられるからハマっている。


　-----------------------　


　今、震えるものがないとしたら、
　自分の中にはない、ということ。

　皆、何か一個の対象が
　ポコッと見つかったら、

　急にやる気が湧いてくると
　勘違いしているのではないか。


　-----------------------　


　僕（鮒谷）の感覚では
　そんなに安易なものではなくて、

　「内面にタネがあっても
　　閾値を超えていなくて、
　　盛り上がらない」

　という状況がある。


　-----------------------　


　小さな感動のタネを集めましょう。

　集め始めても当座は
　何も起こらないが、

　その習慣を持ち、集め続けると
　結びつき始める。


　自分の意識が持っていかれれてしまう
　分野はおよそこのあたり、

　ということは、
　大量の小さな感動を集めないと見えてこない。

　
　-----------------------


　感動体質が失われている人に、

　「ある時、急にスイッチが入った」

　ということは
　絶対に起こらない。


　-----------------------


　記述しないとセンサーが
　バカになって感動できなくなる。

　センサーの感度は鈍るし、
　心が死んでいくし、ゾンビ状態になる。


　-----------------------


　「連想」が発動され、

　「感動のコンボ」

　が生まれるようになると
　何を見ても感動出来るようになる。


　しかし

　「感動のコンボ」

　を日常的に起こせるようになるには
　時間がかかる。


　-----------------------


　感動の素材がない人は
　一から始めるしかない、という都合な真実。

　結局、感動し続けている奴が
　最後に勝つ。


　毎日、震えている人間と
　そうでない人間、

　どちらの話を聞きたいか、
　どちらに仕事を頼みたいか。


　-----------------------


　気を入れる、

　さらには感動によって
　風船に空気を送り込むがごとく、

　思いをパンパンに張ることが大事。


　-----------------------

　　
　気の総量を増やしつつ、
　粛々とシステム化を進めつつ、

　の合わせ技。

　徐々に負荷をかけていくうことで、
　ゆっくりと拡張されていく、、、


　※以上、音源内容の、
　　ごくごく一部をチラ見せ。

　　この続きは音源でお楽しみ下さい。


　-----------------------
　（ここまで）



　ここまでの話を読んで

　「知っている」
　「どんな話か想像できる」

　と思われる人も
　あるかもしれませんね。

　変わり、人生が激変し始めたので、

　　何度もお聴きしているので）

　絶対の自信と確信があります。
