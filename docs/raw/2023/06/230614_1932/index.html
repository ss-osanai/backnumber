■昨今、当たり前のように
　中小企業においても

　【会社のM&A】

　が行われるようになりました。

　

■私（鮒谷）も十数年前、

　当時経営していた会社を某上場企業に
　譲渡したことがあり、

　当時はまだ今ほど
　盛んではありませんでしたが、


　現在は、友人知人、
　コンサルのお客さまも、

　会社を売る側、買う側、それぞれの立場で、
　M&Aを行われることもしばしばとなり、

　一昔、二昔前とは
　完全に時代が変わった感があります。



■会社を売却し、思いもよらぬほど
　大きな個人資産を築かれた方もあれば、

　反対に買い手側として何社も会社を買い、
　成長を加速させている方もあります。



■資本主義社会においては、

　M&Aは使い方次第で
　強力な飛び道具となりますし、


　現状、どのような状況にあっても、

　経営者、資本家、ならびに
　その方向でキャリアを考えている人は、
　
　【知っておくべき基礎教養】

　といえるのではないかと思います。



■そうした

　【基礎教養＝道具＝武器】

　を手に入れて、
　使いこなされている方もあれば、


　（単なる思い込みに過ぎませんが）

　【自分とは一切無関係】

　ということで興味を示されない方も
　少なくありません。



■そのような、

　自社を売るなんて考えたこともない人が
　M&Aの話を聞いて、

　「うちのような会社でも
　　売れる可能性があるのか」

　とか

　「売れても二束三文と思っていたら
　　そんな高値で売れるかもしれないのか」

　と驚かれることもしばしばです。
　


■ちなみに私（鮒谷）も、
　友人が会社を売却したと聞いて
　
　「ぶっちゃけ、どれくらいで売れたのか、
　　差し支えなければ教えてー」

　と、下世話な質問をして、
　こっそり教えてもらうことがありますが、

　
　失礼ながら（本当に失礼なんですが汗）

　「あの会社が、
　　そんな価格で売れたんだ！？」

　と驚かされることもしばしばです。



■ただ、

　【自社を高く売るためには
　　然るべき知識やネットワークが必要】

　なんですよね。

　出たとこ勝負では勝負にならず、
　話にもなりません。


　
■あらかじめ、

　いざ、チャンスが来たときに
　飛びつけるよう、

　準備を整えておく必要があると
　いうことです。



■また、逆から言うと、

　【他社を安く買いたいと思うなら、これまた
　　然るべき知識やネットワークが必要】

　ということも言えますね。


　要するに、何ごとであれ、

　【丸腰で臨むのは無謀の極み（愚か者）】

　ということです。



■そんなわけで今回、


　「会社を売りたい、買いたい」
　「でもどうやって売るの？買うの？」
　
　「自分の会社は売れるのかな」
　「売れるのならば高く売りたい」

　「M&Aにあたって、
　　心がけておくべき点は？」
　「どんなところに落とし穴があるの？」


　そんな思いを持つ友人やお客さまの
　ご要望にお答えし、
