【３ヶ月で時価総額1.5億円のNFTコレクションを作った男】

2023/03/22 19:30 
■こんばんは、鮒谷です。


　今日のメルマガでは、

　最近、私（鮒谷）が
　考えていることについて、

　シェアしたいと思います。



■それは、

　「結果が予見できることには
　　取り組むけれども、
　　予見できないものは避ける」

　という考え方は、

　資産やキャリアを築いていく上で
　長期的に障害になり得るということ。



■以下にそう考える理由を述べますね。


　結果が予見できることのみを選んで
　チャレンジするという考え方には

　【リスク回避の姿勢】

　が含まれています



■リスクを避けること自体は
　悪いことではありませんが、

　過度にリスクを避けていると
　新たな成長や収益の機会を逃します。

　（過去を振り返って、思い当たる節の
　　ある方も少なくないのでは）



■人間は放っておくと

　【安全圏に留まる傾向】

　があるものです。


　しかし、自分の慣れ親しんだ範囲内でしか
　活動しないようになると、

　新しい発見やスキルの習得が
　難しくなり、

　結果として自らの人生、ビジネス、
　キャリアに制限がかけられることになる。



■ですから、人生やキャリアにおいて、

　予測できない対象に対して　
　前向きに受け止め、

　チャレンジを重ねることが大切です。



■また、

　予見できることばかりを行っていると
　時間の経過とともに

　【世界が閉じていき、
　　競争が激しくなる】

　のは自明です。


　
■多くの人は安全な選択を好むため、

　その道には多くの人が集まる
　（＝激戦区になる）こととなるので、


　自身が差別化を図ったり、
　特徴を際立たせることが難しくなり、

　飛躍的な成長を
　望めなくなりがちです。



■さらに、予見できないものを
　避け続けているうちに

　【決断力が鈍っていく】

　というデメリットもありそうです。



■人生やビジネスにおいて、

　予測できない状況（ときに絶好の機会）に
　直面することが時折、訪れます。


　そんな環境下において柔軟に対応し、
　適切な決断を下す力は、

　ここぞ、というタイミングにおいて
　飛躍を目指す際に要求される、

　とても重要な資質です。

　しかし、つねに安全な選択をしていると
　決断力が徐々に失われていくのです。



■これら幾つかの項目は「資産運用」と
　共通するところがありますね。


　資産運用の観点からも、

　結果が予見できる投資だけに
　集中していると、

　リターンの面で限界が生じます。



■未来を確度高く予見できる投資は、
　多くの場合リスクが低いため、

　リターンもそれに比例して
　低くなることが一般的です。

　
　典型例が、日本における
　銀行預金ですね。

　リスクは極小、
　しかしリターンも極小。



■あるいは、

　決められた報酬の範囲内で
　社内で与えられた仕事だけをやり、

　それ以上の挑戦をしなかったり、
　投資や副業にも取り組まなかったり、
　
　とかも、

　失敗はしないけれども
　リターンは限定される、

　一つの例と言えるかもしれません。



■適度なリスクを取り入れた
　ポートフォリオを組むことで、

　より良いリターンを目指すことが
　可能になりますが、

　予測できない投資を避けていると
　機会を逃し、

　個人も資産も
　成長が阻害されるのです。



■ですから、

　資産運用においてはもとより、
　人生やビジネスについても、

　適度なリスクを取り入れつつ、
　自己成長を図りたいもの。



■結局のところ、

　自分の人生、キャリア、資産を
　より富ませるためには、

　リスクを限定しつつ
　予測不能な分野にチャレンジし、

　その中で学び、経験を重ねることが
　肝要です。



■リスクとリターンの
　バランスを考慮しながら、

　柔軟な思考と行動力を持って、
　未来に向かって、

　お互いに進んでまいりましょう。



■ここまでの話に関連して、
　急な話ですが、
　
　来週4月1日（土曜日）に

　【NFT DREAM 
　　～３ヶ月で時価総額1.5億円の
　　　NFTコレクションを作った全て～】

　というタイトルのセミナー開催を
　いたします。

　「NFT」についての講座です。



■もちろん私（鮒谷）が話すわけでは
　ありません（話せません）。

　当該実績

　（＝3ヶ月で時価総額1.5億円の
　　　NFTコレクションを作られた）

　を持つ講師の方をお招きして
　お話しいたします。



■これまでいろんなコンテンツを
　提供してきましたが、

　たまにはこういう系（？）の
　生々しい話を提供する場を設けても、

　それはそれでニーズがあるだろう、

　というよりも、むしろ
　そちらのニーズのほうが強い（！？）

　と考えて、
　開催することにいたしました。



■正直、私（鮒谷）にとって、

　「NFT」

　と聞いても、概念は分かっても
　具体的なことはあまり分からない、

　「未来の予見できない投資対象」

　です。



■しかし、

　よくわからないからと言って
　触れようとしないのは、

　上述の話でいうところの

　「アップサイドの方向における
　　高いリターンを取り損ねる」

　というリスクもあるわけです。



■取り組まなかったがゆえの、
　逸失利益が生じる可能性がある、

　ということですね。


　だからこそ

　「抑えておくべき」

　だと思って、即、自分が学ぶために
　こうしたセミナーを企画しました。


　
■講師のルキフェルさんは、

　信用している、高校の超優秀な後輩から
　紹介されたのですが、

　その起業家的資質や
　コミュニケーション能力は群を抜いており、


　「３ヶ月で時価総額1.5億円の
　　NFTコレクションを作った」

　という実績はもとより、その実績を支える
　そうした資質にも興味を持ち、

　セミナー登壇を依頼した次第です。
　


■NFTと聞いて、
　私（鮒谷）と同じように

　「正直、よくわからん」

　けれども、無視することによって

　「アップサイドの、
　　高いリターンとを取り損ねる恐れ」

　を抱かれた方はぜひ、ご参加下さい。

　　

■激しく変化する世界を
　キャッチアップするための

　「とっかかり」
　
　となるはずですし、


　NFTについて学ぶための
　機会でありつつ、

　講師のルキフェル先生が語る

　「3ヶ月で時価総額1.5億円の
　　NFTコレクションを作った生々しい事例」

　を通して、起業家精神も学べると
　思います。



■なんといっても評論家ではなく、

　実践して大きな成果を現実に上げている
　ガチなプレーヤーから話を聞けるのは、
　
　めちゃくちゃ貴重な機会です。
