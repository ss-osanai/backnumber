＜今日のピークパフォーマンス方程式＞

 【  全ての知的プロフェッショナルに要求される、必須のリテラシーとは  】

───────────────────────────────────



■混同されがちですが、

　「あることを知っている
　　（コンテンツを持っている）」
　
　のと、

　「それを渡す技術」
　
　は、実のところ、全く別物です。



　「ある知識を知っている」
　
　のと、

　「（集団であれ、個人であれ）
　　その知識を求めている対象を理解し、

　　届く（すなわち、理解し動いてもらう）
　　ように伝えられる」
　
　のとは、全く別のものなのです。



■このことは、
　
　毎日、毎日、コンサルティングを
　たくさんさせて頂く中で、
　
　つくづくわかったこと。

　
　コンサルティングの回数が右肩上がりになるにつれ、
　明らかに、

　「伝える、届ける技量」

　が上がっているのがわかります。



■コンサルを重ねるにつれ、

　「伝えるべきコンテンツそのものの理解が
　　深まってくる」

　ということも、もちろんあるけれども、

　
　それ以上に、

　「当該コンテンツを伝える技量」

　が、劇的に変化したように思われるのです。



■こうして時間をかけて熟成、体得された力は、

　リーダーシップやマネジメント、
　営業、マーケティング、

　あるいは一般的な対人関係などなど、


　あらゆるところに波及する、

　「汎用性の高い力」
　
　であるということが分かります。



■今の時代において、
　コンテンツを持っていることは重要ではありますが、

　分野によっては、

　「たくさんの知識を持っている、
　　優秀と目される人」

　はゴマンといて、


　既に知識そのものが差別化の要因となることは
　ほとんどない、

　ということもあるように思われます。


　さらにいえばGoogleさんも、

　知識の優位性だけで生きていこうとする人を
　駆逐してやろう、

　そんな方向性で頑張っているわけで。
　


■だからこそ、

　コンテンツを持っているのは大前提ではあるけれど、
　そこに付加して、
　

　「自身のコンテンツを、

　　伝える対象とする人に、
　　最も役に立つ形で伝えられる力

　　（多くの場合、知識を伝えるのみならず、
　　　動いてもらえるまで導く力）」


　を持つことができれば、それは大いなる
　差別化の要因となるでしょう。



■であるからこそ、伝える力は、

　「差異化の源」
　
　になるがゆえに、もはや、

　「全ての知的プロフェッショナルが身に付けるべき
　　必須のリテラシー」

　とさえ言えるのではないかと
　思っているのです。



■それが分かったからこそ、
　私（鮒谷）は、

　顧客のためであると同時に、
　自分のためにも、

　「伝える力（伝わるように伝える技術）を磨き抜く」

　ための量稽古を、
　来る日も来る日も重ねているわけだし、

　これからも、重ね続けていくのです。


　
■こればかりは机上の空論では身につかず、

　数多くの実践によってのみ
　身体知化されるものだから。


　したがって、この活動は、
　私の将来に重大な影響を及ぼす

　「投資活動」

　であり、

　「武者修行」

　でもあります。



■だからこそ、日々のコンサルにおいては、
　一切の手を抜かず、妥協なく、

　お客さまへの価値を提供することに
　コミットメントしているわけです。


　（手を抜いたり、妥協していては、
　　投資活動にも武者修行にならないので）
