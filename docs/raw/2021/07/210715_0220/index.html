■人生において

　「この意思決定の基準だけは
　　早めに確立しておいてよかった」

　と思えることがあります。


　それは

　-----------------------

　【未来にプレゼント音源】

　「あなたは将来の自分に、
　　どんな世界を贈りたい？」放談会音源　　

　https://shinkaron.buyshop.jp/items/39557757

　-----------------------

　にてお伝えしたように、


　毎日の行動において

　【将来の自分が受け取ると喜ぶであろう
　　プレゼント（行動）だけを行う】

　こと。



■より具体的には


　「未来の自分が
　　楽をできるであろう」
　
　「未来の自分が
　　楽しめるであろう」

　「未来の自分の苦痛を
　　取り除けるであろう」


　と思われること「のみ」を
　行うということ。



■経済的な問題についても、
　非経済的な側面においても、
　
　上記の基準で判定し

　「今、これを行うべきか、
　　行わざるべきか」
　
　だけを考えて、一つ一つ、
　丁寧に意思決定するのです。



■これを心がけると、結果として

　-----------------------
　
　【時間の天引き音源】

　鮒谷周史の「圧巻！人生は『時間の天引き』から
　しか変えられない」放談会音源

　https://shinkaron.buyshop.jp/items/11202803

　-----------------------

　でお伝えしたように、

　【先送りから、前倒しの人生】

　へと舵が切り替わるはず。



■これにより、船の舵と同じように、

　舵を切ったからといって即座に
　航路が変更されるわけではありませんが、


　時間の経過とともに、ゆっくりと、
　しかし確実に、

　船の進路が切り替わります。



■切り替わったら、
　そこから先は早いもの。
　
　弧を描き、あるところから一気に
　角度がついてきます。

　急激に航路が変更されるのです。


　人生もこれと同じ、そんなイメージを
　持たれると良いでしょう。



■あなたが

　【「未来にプレゼント＆前倒し」の習慣】
　
　を身につけられれば、
　必ず人生は変わります。


　この習慣が完全に身についたら、
　次に問うべきは

　「では、どんなプレゼントを前倒しして
　　贈るのが最も効果的か」

　という質問です。



■大事な自分の将来に向けて
　どうせプレゼントを贈るなら

　「しょぼいものではなく、
　　価値あるもの」

　を贈りたいものですね。



■そんな贈答品リストとしての

　-----------------------
　
　【各種音源、取り揃えています】
　https://shinkaron.buyshop.jp/

　-----------------------

　であるわけですが、

　まずはプレゼント習慣がなければ、

　贈答品リストがあっても
　なんの意味もありません。



■だからこそ

　自分の未来にプレゼントしていくと
　こんないいことが起こりますよ！

　とお伝えしているのが
　先に揚げた

　
　-----------------------

　【未来にプレゼント音源】

　「あなたは将来の自分に、
　　どんな世界を贈りたい？」放談会音源　　

　https://shinkaron.buyshop.jp/items/39557757

　-----------------------

　【時間の天引き音源】

　鮒谷周史の「圧巻！人生は『時間の天引き』から
　しか変えられない」放談会音源

　https://shinkaron.buyshop.jp/items/11202803

　-----------------------


　の両音源となります。



■正直、人生において
　この２つの音源をお聴きいただくだけで

　「面舵いっぱい」
　「取舵いっぱい」

　レベルの航路の変更が可能になると
　思っていますので、

　これはぜひお聴きいただいたいと
　願っているわけです。



■その上で、

　「ではどんなプレゼントを
　　未来の自分に贈ろうか」
　　
　となったときにお勧めしたいのが


　-----------------------
　
　【鮒谷周史の、圧巻！
　　稼ぎたければこうして値上げせよ！音源】

　～中小企業＆フリーランスは値上げ一択。

　　長年かけて磨き上げてきた門外不出の
　　マル秘ノウハウをついに完全公開！

　-----------------------

　の音源となります。



■なぜなら、

　こうした意識＆具体的な実践能力を
　未来の自分にプレゼントすれば、

　間違いなく

　【生涯獲得年収】

　が増えることになるからです。



■自ら商いをする人であれば

　「自分の扱う商品、サービスを
　　高く販売する」

　力は必要不可欠ですし、

　組織に所属する人であれば

　「自分を高く売る」

　能力は必須です。
　


■前倒しして、

　こうした意識を身につけ、
　さらには具体的な能力を手に入れ、

　それを磨き上げながら
　将来の自分に襷（たすき）を渡せば

　「未来のあなた」

　はきっと喜ぶに違いないと
　思われませんか。



■私（鮒谷）は

　そんなプレゼントを
　過去の自分から贈ってもらい、

　そのプレゼントによって現在、
　生活させてもらっています。



■反対に、もし過去の自分が

　「安売り精神
　　＆自分を高く売る能力の欠如」

　を申し送ってきたならば、
　
　今の生活は相当、
　厳しいものとなっていたはずです。



■そういう意味で

　「昔の自分の意思決定に感謝！」
　
　しているのです。


　これが冒頭に記した

　-----------------------
　
　人生において

　「この意思決定の基準だけは
　　早めに確立しておいてよかった」

　と思えることがあります。


　それは毎日の行動において

　【将来の自分が受け取ると喜ぶであろう
　　プレゼント（行動）だけを行う】

　こと。

　-----------------------

　と記したことの真意です。



■私（鮒谷）は長い間

　「どうしたら自分、ならびに自分の販売する
　　商品・サービスを高く売れるか」

　という問いを立て、
　
　20年近く、試行錯誤と経験と学習を重ねて、
　自分なりの知見を確立してきました。



■その集大成たる

　-----------------------

　【鮒谷周史の、圧巻！
　　稼ぎたければこうして値上げせよ！音源】

　～中小企業＆フリーランスは値上げ一択。

　　長年かけて磨き上げてきた門外不出の
　　マル秘ノウハウをついに完全公開！

　-----------------------

　を、

　今から10年前、20年前に
　手に入られる環境にあったなら、
　
　何を差し置いてでも
　真っ先に入手していたはずです。

　
　それだけで数年から10年分くらいの時間は
　ゆうに縮減できたはずだから。



■そんな

　「試行錯誤と経験と学習と、
　　そこからの知見」

　をわずかなコストで手に入れられる
　価値をご理解頂ける方に、

　ぜひ当音源をお聴き頂ければと
　願っております。



■経営者、個人事業主、起業志望者は
　もとより、

　組織にいる人でも、自分を高く
　売りたいという思いを持つ人であれば、
　

　同じく商売人ですから
　必ずや当音源を聴いて、

　得られるもの大でありましょう。

■ちなみに、
　自分で言うのもなんですが、

　すでに十二分な利益（億単位の収益）を
　上げている人ならまだしも、

　未だ、そうではなく、

　でも、曲がりなりにも商売で
　利益を上げたいと思う人であれば、


　あるいは、

　組織に所属しているけれども
　自分の年収に不満のある人であれば、

　聴かない手はないと
　思うんですけどね、、、



■まあ

　「すべての教育的価値は
　　事後的にしか分からない」
　
　わけで

　「価値を理解する前に購入する」

　という意思決定が難しいのは
　先刻、承知していますが、


　だからこそ

　「価値を理解する前に購入する」

　という冒険をすることが習慣化されている
　（ごく、わずかな）人だけが、

　早期に、かつ安定的に
　結果を出していかれんですよね。



■こうした本能に逆らう意思決定を
　行える人が少ないからこそ

　「石橋を叩いて
　　橋そのものを壊してしまう人ばかり」

　なのだと思っています。



■そんなわけで

　「高い価格をつけて、
　　それでも向こう見ずに、

　　えいや、とお求め下さる
　　私（鮒谷）のお客さまは皆、
　　
　　早期に、かつ安定的に
　　結果を出していかれる」

　こととなるわけですが、


　こうした構造を理解できる方とだけ
　お付き合いしていれば、

　私（鮒谷）も儲かる上に、
　毎日、機嫌よく仕事ができ、

　お客さまも価値を形に変えていかれて
　みんなハッピーになれるのです。



■こうした世界に生きていられるのも

　「高い価格設定をしているから」
　
　なのですが、

　そんな気づきを与えてくれたのもまた、
　過去の私（鮒谷）自身なのでありました。
