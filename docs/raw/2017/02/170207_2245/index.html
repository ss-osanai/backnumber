【コミュニケーション能力向上は、もうこれ一本でOK！】予定した音源モニター数に到達し次第、販売終了するので、お急ぎを。

2017/02/07 22:45 
■こんにちは、鮒谷です。


　皆さま、お待たせしました！


　多くの方から、いつ音源販売が始まるんだと
　尋ねられていた、

　「ブックライター上阪徹さんのセミナー音源」

　ついにリリースです。

　

■以下に上阪さんのこれまでの実績を
　列挙させて頂きますが、

　これまで約30年に及ぶ、執筆実績ならびに
　インタビュー経験に目を通していただけば、

　それだけで上阪さんにはとんでもない

　【聞く力＆書く力（＝コミュニケーション能力）】

　が備わっていることをご理解頂けるでしょう。


　
■そのとてつもないコミュニケーション能力が
　どこから来るのかの

　「（手品の）タネ明かし」

　について、

　先日２日間、９時間以上に渡り、
　大いに語って下さった

　「秘中の秘」

　がついにベールを脱ぐこととなりました。



■販売している私（鮒谷）が、
　こんなことをいうのもなんですが、

　この音源（＝この価値）が
　この価格で手に入るとか、正直、あり得ない、

　と言っておきます。


　私（鮒谷）ももちろんセミナーに参加し、
　２日間（９時間）の間で、

　数十個どころではとても収まりきらぬ、
　100個単位での

　「示唆、刺激、気付き、学び、発見」

　を頂きました。

　　
　すでにひそかに！？実践しており、

　目覚ましくコミュニケーション能力が
　上がっている実感があります。



■ここ5,000日ほどの間、

　毎日、いろんな人と出会い、
　話を聞き、

　さらにはそこでの学びを記し、
　を繰り返してきた、

　「この私（鮒谷）が聴いてさえ」

　ですよ！（といってしまいます）



■それほど、勉強になりました。

　そして、ほとんどの人は

　「コミュニケーション能力向上」

　について、体系だった学びをしたことなど
　一度もありませんから、
　
　上阪さんの30年ほどに及ぶ経験が圧縮、凝縮された
　この音源を聴かれれば、文字通り、

　「目からウロコが100枚くらい落ちる」

　「周りの人に大いに差をつけるチャンスとなる」

　のではないかと思います。



■なんといっても、

　「論より証拠」

　以下の実績ですよ。
　

　このレベルの仕事が殺到している人に

　「聞く力が乏しい、書く力が乏しい
　　（コミュニケーション能力が難あり）」

　ということが、果たしてあるでしょうか。

　あるわけないですよね。



■その能力をどのように身につけてきたのか、

　あるいはそれを再現可能な形で
　お伝えして下さっている、

　貴重な上阪さんのセミナーが
　いつでもどこでも何度でも聞ける音源です。
　
　「聞く力が乏しい、書く力が乏しい
　　（＝コミュニケーション能力に難あり！？）」

　と自覚されている方であれば、
　必聴の音源であると確信しています。


　ただし！！

　今回のお求めには「条件」があります。

　詳しくは、本メールの最下部をご覧ください。
