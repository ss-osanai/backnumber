6894号 私は金持ちだ。金持ちはそんなことはしない。

6893号 お金のために働くのではなく、私のためにお金に働いてもらう

■似たような方も少なくないと思いますが
　私（鮒谷）が最初に

　「お金を稼ぐ」

　ことに強く意識を向けられたのは

　『金持ち父さん 貧乏父さん』

　という本でした。



■ご存じの方も多いと思いますので
　どんな内容かは詳しく触れませんが、

　一貫して同書を貫く主張は

　「私はお金のために働くのではなく
　　お金が私のために働いているのだ」

　というものでした。



■当時、そんなことを考えたことも
　なかったので驚いた、

　というか、
　より正確にいうならば

　「青天の霹靂」

　でありました。

　今から22年前、この言葉に触れた時の
　衝撃は今でも忘れられません。



■とはいえ、

　「では早速、お金を私のために
　　働かせよう」

　と思ったところで、
　先立つもの、つまり

　「種銭」

　が必要です。



■経済的に、未だ

　「ギャンブル後遺症」

　から抜け出せなかった私（鮒谷）は
　種銭がなかったため、


　ビジネスを立ち上げるために
　真剣に学び、

　同時に起業を視野に入れつつ、
　副業に邁進することとなりました。



■結果として、

　個人的にはこの順序であったことは
　とても良いと考えているのですが、

　-----------------

　１、個人事業主として
　　　お金を稼げるようになり

　２、労働とは切り離された、事業としての
　　　収益も上げられるようになり

　３、その蓄えを運用に回し、自分のために
　　　お金を働かせられるようになる

　-----------------

　ところまで到達できました。



■この３つを鼎立させられれば、

　-----------------

　１ができなくなっても
　２、３でフォローできるし、

　２が稼働しなくなっても
　１、３で補填できるし、

　３で損失が生まれても
　１、２によって生活を賄える

　-----------------

　という、

　生活の安全、安定が確保でき、
　平穏を保てるようになります。



■これが双発機（いずれか２つ）であれば
　不安定だし、

　（＝やや不安）

　単発機であれば、
　緊急事態が生じたとき、

　推力を生む機構が一つしかないので
　そのまま墜落するしかありません。

　（＝不安しかないことになる）



■そんなわけで私（鮒谷）は、

　自分の労働力を稼ぎに変え、
　事業で収益を上げ、
　運用でお金を回す、

　鼎発機（そんな言葉はありませんが）
　という状態に満足しています。



■ただ、「鼎発機」は飛び立つときに
　大きな負荷とリスクがかかります。

　つまりスムーズに
　商売が立ち上がるかどうかは、

　多分にやってみなければ分からない
　ところもあり、

　そこでそのまま墜落する人も
　少なからずあるのも現実です。



■ですから、許容限度を超える
　リスクは取りたくない、

　あるいは、

　そこまでの推進力（動機）が
　湧いてこない、内在していない、

　ということであれば、


　現実的な理想解としては

　「双発機（労働＆投資）」

　ときに、上記とは異なる意味での

　「（２馬力＝ダブルインカムによる）
　　鼎発機」

　で生活を賄う、ということではないかと
　思います。



■つまり、会社や組織に所属し、

　リスクを抑えつつ
　（ゼロにはならない）、

　その上で、節約を行いつつ、
　浮かせたお金を運用に回す、

　という両刀遣いでの生活原資の
　獲得です。



■勤務先からの（安定）収入によって、
　生活のダウンサイドを限定しつつ、

　余裕資金の範囲で投資を行うことで
　経済的なゆとりを追求するのは、

　堅実な生活設計の一つとして
　検討に値すると思われます。



■実際に私（鮒谷）の大好きな
　雑誌やネットで紹介されている

　「家計相談記事」

　を見てみると、

　上手にこれを行って
　安全領域まで資産を増やしてきた、

　という人を（多数）見かけますので、
　そこまで難易度は高くなさそうです。



■それに対して、

　「労働収入のみでの生活」

　は上記に比べると、

　やっぱりちょっと怖いですよね、
　不安になりますよね。



■かつて私（鮒谷）が経験したように

　「勤務先が倒産する」

　こともあれば、

　「リストラされる」
　「年収の大幅減額」

　といったことも、今の時代、
　珍しくはありませんから。



■だからこそ、

　もし適性がある人であれば、
　10年計画くらいで

　-----------------

　１、個人事業主として
　　　お金を稼げるようになり

　２、労働とは切り離された、事業としての
　　　収益も上げられるようになり

　３、その蓄えを運用に回し、自分のために
　　　お金を働かせられるようになる

　-----------------

　という状態を作ることを志されると
　良いかと思いますし、


　そこまでリスクは負いたくない、
　負荷をかけたくない、

　と思われるのであれば、せめて

　「双発機（労働＆投資）」

　での飛行（＝生活）を目指されると
　良いのではないでしょうか。



■共通するのは

　「蓄えを運用に回し、自分のために
　　お金を働かせる」

　ところです。

　まさに「金持ち父さんの教え」
　ですね。



■私（鮒谷）は

　-----------------

　１、個人事業主として
　　　お金を稼げるようになり

　２、労働とは切り離された、事業としての
　　　収益も上げられるようになり

　-----------------

　にあたっては、

　一切、我流で行うことを否定し、
　徹頭徹尾、学び、

　「守破離の精神」

　でもって模倣し、相応の結果を
　残せたと思っています。

　（そのために時間もお金も
　　めちゃくちゃ投資しました）



■そして、

　-----------------

　３、その蓄えを運用に回し、自分のために
　　　お金を働かせられるようになる

　-----------------

　状態をより確実なものにするために、

　今、あら貯めて一切、我流で行うことを
　否定し、徹頭徹尾、学び、

　「守破離の精神」

　でもって模倣し、相応の結果を
　残そうと思っています。



■ただ、どうせ学ぶのであれば、

　（一人ではなく）
　皆さんとともに、

　と思いましたので、

　以下の企画を立て、
　ご案内してまいりました。



■そんなわけで、もしあなたが

　「お金のために働くのではなく
　　お金を私のために働かせたい」

　という気持ちがあるのであれば、


　そして特に自らの事業を持ち、
　事業が収益をもたらしてくれる、

　そんなバックアップも
　セーフティネットもない方はなおのこと、

　資産運用について
　学ばれるべきではないかと思います。
