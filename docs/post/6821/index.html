6821号 先送りによってもたらされる不利益、２つの側面とは

■今日は

　【人生クマムシ化音源
　（＝破滅回避音源）】

　をお聴き下さった、大橋さんから
　頂戴したご感想の紹介を。


　（ここから）
　---------------------------------


　どこかで読んだ本に
　書かれていたのですが、

　イチロー選手は
　毎回の守備が終わるたび、

　アンダーシャツを
　着替えていたそうです。


　その理由は、

　「汗が冷えることで
　　風邪をひくかも
　　しれないから。」

　というもの。


　普通の感覚であれば、
　毎回着替えるのはめんどくさい。

　と考えてしまうかと
　思いますが、

　「めんどくさい」

　よりも、

　「風邪をひかない」

　ことの方がはるかに
　大切だからこそ、

　毎回着替えることが
　できるのかもしれません。


　この時、この二つの考え方の
　違いは大したことないと
　思ってしまいがちですが、

　実のところ、

　人生そのものに大きなインパクトを
　与えるほどの大きな違いである

　と考えています。


　すなわち、

　「今やっておくのか」

　それとも

　「何かが起こってから
　　はじめて動くのか」

　によって人生が大きく変わる
　ということです。


　今回の音源のテーマの一つは

　「危機対応」

　になるかと思いますが、
　危機対応は

　「今やっておくのか」

　それとも

　「何かが起こってから動くのか」

　によって結果が
　大きくかわります。


　これはイメージがわきやすいかと
　思いますが、

　これを普段の私たちの生活に
　置き換えてみるとどうでしょうか。


　例えば、

　「今読書をするのか」

　それとも

　「必要に応じて
　　読書をするのか」。

　長年読書を続けてきた人は、
　全く読書をしない人と比べ、

　長い時間軸では圧倒的な差が
　ついていることは間違いありません。


　でも、悲しいかな、今の時間に

　「本を読む」

　か、

　「スマホを見る」

　という選択肢があるのならば、
　スマホを見てしまうんですよね泣


　今回の音源は、

　「破滅」

　という人が本能的に避けたい
　（感情が動く）テーマが
　ベースになっているからこそ、

　音源を聞くことによって、

　「今動かなきゃ」

　という気持ちになることができる
　のではないかと考えています。


　そして、音源を聞いた後に
　行動した時の勢いを維持し、

　「他のこともやっておこう」

　という気持ちになったら
　儲けもの。


　例えば、
　音源を聞くことで

　「今の空いている時間に
　　読書しておかなきゃ！」

　という気持ちになれたのなら、

　この音源から得られる価値は
　計り知れないと思います。


　そういった意味では、
　今回の音源は、

　「危機対応しましょう。」

　ではなく、

　「人生をより充実させましょう。」

　ということが暗に含まれるメッセージ
　なのではないかと考えています。


　ですので、

　今日もスマホを見て一日が
　終わってしまった。

　という方は本音源を
　ぜひとも聞いて頂きたいです。

　（スマホをほとんど見ないで
　　終えた一日は爽快です！）


　改めまして、
　今回も素晴らしい音源を
　ありがとうございました！

　引き続き、よろしくお願い致します！


　（株式会社CODOR代表取締役
　　大橋亮太 さま）

　（ここまで）



■大橋さん、
　いつもありがとうございます！

　音源について書いてくださったことは
　まさに

　「破滅回避」

　に限らず

　「あらゆることを前倒しする習慣を
　　身につければ幸福感が逓増する」

　ということかと思います。



■私たちは、
　朝起きてから、夜寝るまでの間に

　（自覚の有無はともかくとして）

　「それを今やるのか」
　「後回しにするのか」

　のいずれかの決断を下し、
　実行に移すか否かを迫られています。



■今やる、と決めた人は、

　ほんの少しづつではあるけれども
　しかし確実に、
　
　未来は良くなっていくし、
　安全で安心な世界に近づきます。



■反対に、後でやる、
　と流してしまった人は、

　その一つ一つの意思決定

　（惰性で流されるのも、
　　結果として一つの意思決定）

　が短い時間軸では
　目に見える影響も少ないですが、


　時間の経過とともに
　だんだんと、しかし確実に、

　未来を不安と恐怖に満ちた、
　悪いものへと近づいていくでしょう。



■先送りの結果として、
　こうした

　「じわじわと生活を蝕む」

　というダメージが
　蓄積されていくわけですが、


　もう一つの恐怖として

　「備えていない状態のまま、
　　ある日突然、破滅に見舞われる」

　可能性が存在することも
　挙げられます。



■つまり、先送りの弊害は

　１、時間をかけて徐々に蓄積されていく
　　　ダメージが複利で回る

　側面と

　２、事が起きたとき、準備ゼロのまま
　　　緊急事態に向き合わざるを得なくなる

　側面の２つがあるということです。



■どちらにしても、悲劇的で
　悲惨な未来は確定するのだから

　「あらゆる意思決定と行動を
　　前倒し」

　する習慣を骨の髄まで
　刻み込みたいもの。



■この習慣を身につけられれば、

　人生が逓増していく
　（生きていくほど楽になる）

　こと、請け合いです。

　

■そんな姿勢を会得して頂くために
　いくつもの音源をお届けしてきました。

　たとえば以下のような。
