7520号 グループダイナミクス（集団力学）の力を借りて変身せよ

■断続的に

　「新しいことを始めよう！」

　と思う機会が提供される環境に
　身を置いていると、

　一瞬、動きが止まることはあっても
　すぐに立ち上がれます。



■一方で、そうした場がなければ、

　ひとたび座り込んでしまうと
　次に起動する機会は随分先になり、

　下手をすると年単位で
　ボーッと過ごしてしまう、

　といったことにもなりかねません。



■手前味噌ではありますが、

　私（鮒谷）が月に二回、主宰している
　年間プログラムは、まさにこの

　「新しいことを始めよう！」

　という思いや機会を、

　おおむね二週間に一度、
　お渡ししている場、

　といっても良いかもしれません。



■年に24回の機会があれば
　どれほど怠惰な人であったとしてさえ、

　毎回とはいわなくとも、最低でも
　何度かはスイッチが入るもの。



■現実には毎回、
　新しいことに取り組む人はありませんし、

　それぞれ日常のこともあるので
　そこまでの余裕もありませんが、


　ただ、継続して
　こうした場に身を置いていると

　「ちょうど良いタイミングで、
　　新しいチャレンジを行うきっかけが
　　渡される」

　ことがあるんですよね。



■人間、どうしても、
　そうした環境がなければ、

　「気持ちがあっても
　　ズルズルと先送りしてしまう」

　ことがよくありますが、


　こうした場を活用すれば、

　それぞれが
　それぞれのタイミングで、

　時期は少しづつずれつつも、新たな
　取り組みを始めていかれますから、

　それに触発されて、行動するきっかけが
　与えられて動き始めるし、

　その姿を見た周りの人が、
　さらに再加速を始める、

　なんてことが日常的に
　起こるようになるのです。



■これぞまさに、

　【グループダイナミクス（集団力学）の
　　極み】

　といって良いかもしれません。



■人の思考、行動は
　周囲の人から影響を受けますし、

　それを受けて考え、行動したことが
　今度は逆方向に作用して、

　周囲の人に影響を及ぼずことに
　つながっていく。

　こうした相乗効果によって、
　参加者全員が加速していきます。



■そういえば、

　以前からの読者さんは
　ご存知であるはずの

　（FXで全財産を溶かした上に
　　莫大な借金を背負うこととなった）

　例の船津さん（笑）が本日をもって
　晴れて会社を退職、

　４月より独立されるのですが、


　船津さんもまた、
　グループダイナミクスの力によって、

　七転び八起きを繰り返しながら、ついに
　起業に至るところまで到達されました。



■結局のところ、環境なんですよ。

　どれだけ優秀な人であっても、

　環境を選ばなければ
　その才能が開花されることはありません。



■こうした構造を理解していない人は
　いつまで経っても

　「自分一人で懸命に頑張って
　　なんとかしようとする」

　そんなパラダイムから逃れられず、

　結果として同じところをぐるぐる、
　周り続けることになるんですよね。



■そんな人こそ、年間プログラムに
　お越しになられればよいのに、

　とは思いますが、

　無理に案内したり、
　営業するつもりは一切なく、

　「もろもろについて、
　　心から納得された方のみ」

　お越しいただいているので、


　めちゃくちゃ空気感が良い上に

　「こうした場のメカニズムを
　　理解されている人」

　だけが参加くださっているのだと
　思います。



■もちろん参加者の中には、

　すでに圧倒的な結果を
　残されている人もあれば、

　ポテンシャルはありつつも、
　開花はこれから、

　という方もいらっしゃいますが、


　いろんな背景を持つ、
　多くの方々が交わることによって、

　不思議な化学反応や
　相乗効果が生み出され、

　新しい取り組みや協業が次々、
　生まれているのでしょう。



■最近、成り行きに任せるばかりで、
　まったくご案内もしてこなかったので、

　思いつきの、
　久しぶりのご紹介ではありますが、

　以下に年間プログラムのご案内を
　貼り付けておきます。



■ご関心のある方はぜひお申し込み下さい。


　あるいは、いきなり、

　わけのわからないプログラムに
　申し込めるわけがないだろ！

　という方には、

　「年間プログラム「個別」相談会」

　メニューもご用意しておりますので、

　参加をご検討下さっている方は
　こちらからお申し込みくださいませ。


　（絶対に営業しないことがウリなので、

　　この場において安心して疑問や不安や
　　その他について自由にお尋ねください）
