＜今日のピークパフォーマンス方程式＞

 【　悩みのないのが悩み　】

───────────────────────────────────



■こんなことをいうと怒られるかもしれませんが、
　以前、私には深刻な悩みがありました。


　それは

　「悩みのないのが悩み」

　というもの。


　「悩みのないのが悩み！？」

　なにをバカなことを言っているんだ、

　と思われるかもしれませんが、
　当人にとっては、とても深刻です。

　
　「目指すところがなくなった」
　
　という、恐るべき悩みなのです。



■実は考えようによっては、

　悩みがあるというのは、
　とても幸せなこと。
　

　なぜなら、理想と現実のギャップが
　存在するところに悩みが生まれるわけですが、

　悩みを持つ人には、その巨大なギャップを
　埋める過程を楽しむという歓びがあるのです。



■反対に、

　悩みがないという状態には、
　現状に一定の満足を覚えてしまい、
　
　ギャップを埋める歓びが希薄になる、
　もしくは、消えてなくなる、

　という側面もあるのです。



■だからこそ、

　「もう少ししたら、
　　この悩みは解消されそうだ」

　という状況が見えてきたら、

　「お悩み解決した後の、次のお悩み」

　を事前に用意しておくとよいように
　思われます。



■私も「悩みがない」という状態に陥ったとき、
　遅ればせながらではありますが、

　「お悩み解決した後の、次のお悩み」

　を自らに用意いたしました。

　おかげさまで再び、大きな悩みに
　満たされるようになりました（笑）


　ちなみに

　「次なるお悩みを用意する」

　とは、

　「新しい挑戦的な目標を用意する」

　ということです。



■これによって再び、

　「強烈な満たされていない感」

　が発生し、

　その欲求不満な状況を、一歩一歩、
　解消するためのプロセスそれ自体が、

　「くるたのしい（＝苦しくも楽しい）」

　状態となりました。

　不思議な事に、悩みのなかった時代より、
　はるかに心地良いのです。



■どうやら人生においては


　「悩みがある」→「悩みの解消」→「新しいお悩み」

　→「悩みの解消」→「次なるお悩み」→「悩みの解消」
　　
　→「更なるお悩み」→「悩みの解消」→　…………

　
　このサイクルを繰り返すことによって、

　次々に新しい場所に連れて行ってもらえ、
　新しい景色を見せてもらえる、

　という楽しみ、歓びがありそうです。



■そう考えるとやっぱり、


　「悩みのないのが悩み」

　「悩みのあるのが歓び」


　といえるのかもしれません。
　


　今日も人生とビジネスを楽しみましょう！
