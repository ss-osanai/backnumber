■昨日のTwitter投稿より。


　----------

　サラリーマン時代、壁一面に

　「願えば叶う」
　「意志あれば道あり」
　「未来は僕らの手の中」

　などの言葉を記した、
　何十枚もの紙を貼り付けていたのだが、

　たまたま家を訪ねてきた
　父が見てビビっていた。


　朝晩、眺めながら歯を磨き、
　着替えていたのだけれど、

　あれが潜在意識に
　刷り込まれたのだろう。

　----------



■何度も何度も
　同じ言葉に触れていると、

　その言葉の方向に引き寄せられて
　いくのは間違いありません。

　自分を変えたきゃ、
　触れる言葉を変えよ、です。



■ただし、

　それまでに自分が何千、何万と
　回し続けてきた言葉を上書きするのは、

　それなりに大変。



■だからこそ幾度も
　繰り返さなければならないし、

　目から、耳から、（自分の）口から、
　そして最終的には、

　脳内で回し続けている無意識の言葉を
　上書きする必要があります。



■自身に、全能感
　（という言葉が言い過ぎであれば）、

　絶対肯定感がなければ、
　新しいことなんか始められません。

　やればできる、と信じているからこそ
　始められるのです。



■新しいチャレンジの起点となる

　「全能感」
　「絶対肯定感」

　は、一定の年齢に到達したら、
　自分で育んでやらねばなりません。

　（ほかの誰も育ててくれません）



■私（鮒谷）は、サラリーマン時代に

　「願えば叶う」
　「意志あれば道あり」
　「未来は僕らの手の中」

　といった言葉を、

　それこそ何千、何万回も
　自身に与え続けることによって、

　いつしか、それが

　「全能感」
　「絶対肯定感」

　へと、つながっていったのだと
　理解しています。



■Twitterだって
　目標を掲げていますが、当然

　「俺が本気を出せば
　　かならず達成できるに決まっている」

　と思っていますし、


　（多分、ここでは書きませんが）

　５月から、まったく別ジャンルの
　大きなチャレンジを開始するのですが、

　それだって

　「俺が本気を出せば
　　かならず達成できるに決まっている」

　と考えているから、

　無謀と思われるような挑戦だって
　できるんですよね。



■たとえ、チャレンジの途上で
　やる気が出なくなったとしても、

　「やればできるけれども
　　やる気がなくなったから、もうええわ」

　と一切の挫折感もなければ、
　恥ずかしさも感じることもく、

　切り替えて次にいけるのだから、
　全能感最高（笑）



■大事なのは、

　自身に対する絶対の信頼から生まれる
　挑戦心を失わないことで、

　それさえあれば、

　いつも自信満々に始められ、
　ときに自信満々に辞められます（笑）



■私（鮒谷）は
　自分を取り巻く環境を

　「願えば叶う」
　「意志あれば道あり」
　「未来は僕らの手の中」

　といった言葉で
　埋め尽くしてきたし、


　今ももちろん、
　埋め尽くしているから、

　それ以外の言葉の棲（す）む場所が
　どこにもなくなって、

　頭の中から
　完全デリートされたわけですね。

　■絶対肯定感を削るような言葉は、目や耳に
　　触れるところから完全に遠ざけよ。

　■絶対肯定感を強化してくれる言葉だけを
　　触れられる環境を自らに用意せよ。

　　「願えば叶う」
　　「意志あれば道あり」
　　「未来は僕らの手の中」

　　さらには潜在意識を埋め尽くすのだ。
