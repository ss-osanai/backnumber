7497号 1・2・3・ダァーッ！の法則

■一昨日、昨日と
　少々、立て込んでおり、

　Twitterについては
　頻度少なめの更新でした。

　今日からまた加速していきます。



■いずれにしても、

　個人が大勢の方に対して
　メッセージを発信できる、

　という意味で、

　力を拡張させてもらえる
　強力なツールだと思います。



■先月、講師の先生をお招きして
　Twitterセミナーを開催し、

　そこで新たに開始された方も
　大勢いらっしゃるのですが、

　みなさん、もはやTwitter更新が
　日常のものとなっているようです。



■人によってフォロワー増加のペースは
　それぞれ異なりますが、

　それでも多くの方が
　やった分だけ増加しているようで、

　皆さん、ますます大量発信を
　行われています。



■講師の方が目標として
　まずは目指すようにと言われていた

　「3ヶ月で1000人のフォロワー
　（そこをクリアすると後が楽になる）」

　も順次、達成していかれることでしょう。



■そんな皆さんの姿を見るにつけ、
　私（鮒谷）が勝手に名付けた

　【1・2・3・ダァーッ！の法則】

　は、ここでも有効であったかと
　思わずにはいられません。



■この法則はどういうものかというと、

　何を始めるにしても
　最初は腰が重かったり、

　あるいは、

　周りの人がどう思うかなど考えては
　恥をかくのを恐れたりしますが、


　「1・2・3」

　つまり、最初の３回（あるいは３日）を
　クリアしさえすれば、

　あとは慣れが生じて、大量行動が
　可能になる現象について名付けたもの。



■３回（３日）、恥をかいたら
　そこから先は、なぜか

　「恥はかき捨てでいい」

　と思えるようになる
　（端的に言えば、どうでもよくなる）

　から、どんどん
　動けるようになるのです。



■私（鮒谷）も久しぶりにTwitterを
　さわり始めたときには、

　メルマガと、いろいろ勝手が違い、

　「恥ずかしいな、こりゃ」

　とか思ったりもしましたが、
　あっという間に慣れました。

　どうせメルマガと一緒で、誰も
　真面目に読んでいないでしょうしね。



■ということは、つまり

　「やったもの勝ち」

　です。

　その延長線上に築かれていくであろう
　メディアはうまくいけば、

　一生食いっぱぐれのない発信ツールとして
　機能するようになりますから。



■後から見返したら、

　恥ずかしく思われる投稿も
　日々、たくさん残していくわけですが、

　そこまで真剣に見てくれる人なんか
　どうせいないのだから、


　恐れずに

　（アントニオ猪木さんの顔や声や
　　立ち居振る舞いを思い出しながら）

　「1・2・3」

　で助走をつけて、
　そこから勢いをつけて本格的に

　「ダァーッ！」

　と突っ走っていきたいもの。



■これはなにもTwitterの開始に
　限った話ではありません。

　あらゆる、新しい取り組みにおいて

　【1・2・3・ダァーッ！の法則】

　は有効ですし、

　積極的に活用できれば、
　フットワーク軽く、

　いろんな夢を実現していけるように
　なるはずです。



■引き続き、
　X(Twitter）投稿を重ねていきますが、

　「各投稿についての、24時間後の
　　平均インプレッション数10,000達成」

　の目標達成まで、あと55日！

　＜鮒谷Twitter＞
　https://twitter.com/shinkaron



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■なにかに初めて取り組むにあたっては

　　【1・2・3・ダァーッ！の法則】

　　を上手に活用すると良いだろう。

　■１、２、３で助走をつけて、
　　心と体を新しいことに慣れさせ、

　　恥をかくのを恐れなくなった頃合いを見て
　　ダァーッ！と大量行動するのである。
