　【　Winner takes all の市場を勝ち抜き、圧倒的勝利者になる方法　】

───────────────────────────────────



■ Winner takes all（ウイナー・テイクス・オール）という
　言葉があります。

　アメリカで1979年から93年にかけて所得調査をした結果、
　上位２割は18％上昇しているのに対し、下位２割は15％減少して
　いることが分かったそうです。

　その調査結果をみて、

　「 Winner takes all の市場が原因」

　と分析した人たちがいました。



■その分析者は、 Winner takes all の市場（WTA市場）とは

　「少数の売り手だけに大きな利益を与え、残りの多くの売り手には
　　非常に小さな利益または損失しかもたらさない市場」

　だと定義しました。

　さらに加えて、このWTA市場は改善されるどころか、
　情報化社会がすすむにつれて今後ますます加速すると予測した
　のです。



■そしてまさに今、インターネットの発展にともなって
　WTA市場は驚異的なスピードで拡散しているのはご承知のとおりです。

　「このサービスは、ここがいいらしい」

　と情報が伝播（でんぱ）すれば、日本じゅう、世界じゅうの顧客が
　１ヵ所に集中する。

　ナンバーワン、オンリーワンでなければ意味がないといわれるのは、
　ここに理由があるのです。

　２番目、３番目に思い出してもらえるという状態では、
　そのお客さんから依頼がくることはまずありません。



■となれば、「このジャンルならこの人」とあなたが認識される
　ことによって、
　日本中（ことによると世界中）の需要に対する供給者となることが
　可能になります。

　あなたの存在が、当該ジャンルにおいて、
　見込み客のうちの何パーセントの「マインドシェア」を占めているか？

　ここを真剣に考えなければなりません。



■だからといって「売りこみ臭」をプンプンただよわせ、
　営業につぐ営業をしかけて嫌われるのは、
　これは悪性のマインドシェアを高める以外のなにものでも
　ありません。

　会社の名前を告げただけで「いらない」とソッポを向かれて
　しまうようであれば、逆にその悪名が拡大していくので
　最悪のケースとなります。



■ある問題が発生したとき、それを解決できるのはあなた、
　ないしはあなたの会社だけである。

　見込み客に、そのように認知してもらわなければなりません。



■では、そのために必要なものは何か。

　それは売り込みではなく、情報発信です。



■私も「平成・進化論。」では営業はめったにしません。
　必要なとき、ほんの少しだけ告知するにとどめます。

　基本的には、有益な情報の発信に徹するという姿勢をつらぬいて
　いるつもりです。

　そうすることによって、毎日売り込むよりも、セールスにおいて
　はるかに高いパフォーマンスを発揮することができるのです。

　これは、将来わたしとビジネスをしてくださるであろう（あるいは
　私がお付き合いしたいと思っている）お客さまにとって有益な情報の
　発信につとめているからなのでしょう。



■「わたし（の会社）は、このような有益な情報をお伝えして
　います。ご利用できるところがあれば、どうぞご利用ください」

　と、それだけでいい。

　そのかわり、書いて、書いて、書きまくる。時間の許す限り、
　書きまくる。
　そして配信をつづける。これが重要です。



■そうすればそれほど遠くない未来に、売り込み営業時代からは
　考えもしなかったほど、あなたはひっぱりだこになるでしょう。

　 Winner takes all の市場における勝利者となるのです。

　これは、やった人にしか分かりません。
　実践してみてはいかがでしょうか？

　ただ注意点もありますので、明日は、そのことについて
　考察を加えたいとおもいます。
