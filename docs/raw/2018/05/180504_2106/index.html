【 人生に「相転移」がもたらされるまでのプロセスとは 】

───────────────────────────────────



■人生（運命）は

　【意志決定の段階で全て決まってしまう】
　
　ものであると考えています。


　「意志決定」と聞くと、難しいイメージを
　持たれるかもしれませんが、
　
　一言で言えば

　【何をして、何をしないかを決めるのが、
　　意志決定】

　であると理解すると良いでしょう。



■少なかならぬ人は、

　日常的に、意志決定なんていう
　小難しい言葉を用いることもなく、

　日々を生きているようにも
　思われますが、


　私（鮒谷）は

　【意志決定なき人生

　（＝あらゆる行動に対し、それを行うべきか、
　　　否かを明確化させていない人生】

　イコール

　【意志もなく、意図も持たず、
　　ただ流されるだけの人生】

　になると考えています。



■たとえば、今週末（日曜日に）

　【丸一日、本当に大切なことだけに集中する会】

　という会を開催いたしますが、

　あるいは、上述の通り、
　現在、

　【鮒谷周史の、圧巻！
　「題名のない放談会（平成30年4月8日開催）」】

　の音源をご案内していますが、


　こうした企画に参加する（または、しない）、
　あるいは音源を購入する（または、しない）、

　といったことはすべて
　
　【意志決定】

　と言えるでしょう。



■こうした意志決定を、一つ、二つ、
　行ったところで人生が変わるはずがありません。

　ただし、意志を持ち、
　何かを決定するたびに
　
　【ほんの一ミリ、二ミリづつ、
　　変わっていく】

　のは間違いありません。

　
　こうした一ミリ、二ミリを小さく
　積み重ねているうちに、あるとき

　【臨界点を超える】

　こととなるのです。



■臨界点を超え、

　【相転移】

　が起きると、そこからあたかも

　【氷が水になるように、
　　水が気体（蒸気）となるように、
　
　　まったく別な存在になる】

　のです。



■かくいう私（鮒谷）も、もちろん

　【相転移の経験】

　があります。


　毎日の、単調な、さらには
　
　【果たしてそれが、
　　どう未来につながっていくのか、

　　ハッキリとはわからない
　　一つ一つの意志決定】

　を無数に行うことによって、
　あるとき突然、


　【意思決定→行動の臨界点を超えて、

　　回っている洗濯機の中に
　　放り込まれたような】

　激流人生が始まるのです。


　※これがサナギの時期を卒業し、
　　変化・変容が本格的に始まる合図です。



■私（鮒谷）だけでなく、

　【つまらない、単調で、退屈で、

　　「もてあましてるFrustration」

　　を抱えていたところから劇的に変化した人】
　
　は、すべてこのプロセスを経て、
　生まれ変わっているのです。


　
■この過渡期にある人は例外なく、皆、
　
　【果たしてこれでいいのか、
　　不安で不安でたまらない時期】

　を過ごすことになりますが、
　あえて言いましょう、

　【それでいいし、そういうもの】

　なのです。



■大切なのは、

　【ほんの一ミリ、二ミリづつ、
　　変わり続ける、という強い思いを持ち、

　　小さな意志決定を弛まず行う】

　こと。

　この繰り返しによって

　【相転移の世界まで導かれる】

　のです。



■ただ、一口に

　【小さな意志決定を弛まず行う】

　と言われても、理解しづらいかも
　しれませんね。


　「そもそも『意思決定の基準』を持たなければ、
　　決断の下しようがないじゃないか」

　ということになりますし。



■そんな疑問に答え、明確に、

　【この基準で意思決定すれば、
　　何をどうしたって成果が出るに決まっている】

　そんな

　【意思決定の基準】
　
　について詳述し、


　さらに、
　
　【意思決定から実行までの期間を極小化する
　　（もっといえば、思いついたら即実行できる）】

　習慣を身に付けるための、
　あれやこれやについて、


　豊富な、具体的極まりない、
　生々し過ぎる私（鮒谷）の事例について、

　徹底してお伝えしているのが
　以下の音源となります。



■ぜひ、この機会に

　【意思決定を下して、
　　意思決定について学ぶ】

　ことを行われてみてはいかがでしょう。
