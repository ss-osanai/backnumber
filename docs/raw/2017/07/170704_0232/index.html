■その昔（平成23年2月8日）、


　＜2705号 一期一会の精神で＞


　というメルマガのバックナンバーで、
　以下のような文章を書いたことがあります。


　--------------------------------
　

■１日３人の人の心に、プラスの痕跡を残したら、

　「年間1000人、5年で5000人、10年で10000人」

　の人の心に、プラスの痕跡を残すことと
　なります。


　（10年で10000人という）周囲の人の心に
　プラスの痕跡を残しながら、

　人間関係で悩むとか、
　経営がうまくいかないとか、
　お客様がいなくて困るとか、

　そんなことは起こりようがないのでは。



■反対に

　１日３人の人の心に、マイナスの痕跡を残したら、

　「年間1000人、5年で5000人、10年で10000人」

　の人の心に、マイナスの痕跡を残すことと
　なります。


　そうなったら、いくら頑張っても頑張っても
　底なし沼に足をとられるがごとくに、

　何をやっても人生が好転することは
　難しいかもしれません。



■面倒で時間のかかることほど、
　あたかもボディーブローのように、
　後々になって効いてくる。

　プラスの側面にもマイナスの側面にも。
　いずれも側面とも、私は過去、経験してきました。

　素晴らしい人たちに恵まれ、
　同時に、手痛い失敗もしてきたのです。



■だからこそ、今は、

　一期一会の精神で、
　目の前にいる人に対して、

　「この時、この場、この人に全力を傾注しよう」

　と思っています。

　（言うは易く、行なうは甚だ以て難し、
　　なのですが、、、でも想いは常にこのようで
　　いたいもの）


　--------------------------------


　と記したことがあります。



■このときに記した思いは、
　今でも、変わることはありません。


　来る日も来る日も、このメルマガでは、

　音源やコンサル、セミナーや放談会、
　年間プログラムや、メルマガそのもの、

　に対して頂くご感想を紹介していますが、

　
　実は、ここで紹介しているご感想は
　
　【（日々、頂戴しているご感想の）
　　ごくごく一部】

　です。



■メルマガでご紹介している感想は、
　
　基本的には、毎日、一通、
　多くても数通、

　ですが、それだけではとても紹介しきれない
　くらいの数の感想を、

　それこそ毎日、頂戴しています。


　あまりにもたくさん頂戴して、
　ご紹介しきれないこともあり、

　http://ssdlfiles.s3.amazonaws.com/170412_testimonials_mokuhyou.pdf

　
　や、

　
　＜日記の書き方放談会の音源をお聴きくださった方から
　　お寄せいただいた133ページにおよぶご感想紹介＞
　
　http://ssdlfiles.s3.amazonaws.com/170630_testimonials_nikki.pdf
　

　や、


　＜物語の持つ力放談会の音源をお聴きくださった方から
　　お寄せいただいた100ページにおよぶご感想紹介＞

　http://ssdlfiles.s3.amazonaws.com/170627_testimonials_monogatari.pdf　


　や、


　＜起業放談会の音源をお聴きくださった方から
　　お寄せいただいた80ページに及ぶご感想紹介＞
　　　
　http://ssdlfiles.s3.amazonaws.com/170621_testimonials_kigyou.pdf　


　も作成いたしました。



■あるいは、


　（とてもご紹介できないくらい、
　　毎日、メルマガのご感想を頂くので）

　もうかなり昔に更新を止めていますが、
　その時点までに頂戴した


　＜読者さまからの3229の感想メッセージ＞

　http://www.2nd-stage.jp/customer/


　もまた、

　【私（鮒谷）にとっての宝物】

　であり、

　【なによりも価値ある仕事の報酬】

　です。



■もし、たとえ、

　瞬間的に落ち込むようなことが
　あったとしても、

　上にご紹介したような感想に
　5分でも10分でも目を通すと、

　【俄然、元気が湧き上がってくる】

　ものです。

　そして、もっと世の中にとって
　価値ある人間になろう、

　そんな向上心も生まれてくるものです。



■おかげさまで、５年ほど前に


　--------------------------------

　１日３人の人の心に、プラスの痕跡を残したら、

　「年間1000人、5年で5000人、10年で10000人」

　の人の心に、プラスの痕跡を残すことと
　なります。

　--------------------------------


　と記したことは、
　
　今、ある程度まで実現できているのでは、
　という、

　それなりの自負も持てるようになりました。

　（実際に、そのように心がけてきたことでも
　　ありますし）



■日々、頂戴するご感想の数は、
　

　【毎日、どれだけの人にプラスの痕跡を
　　残してきたかについての、

　　（定量的ならびに定性的な）

　　これ以上ない客観的な指標】


　となり得るものだと考えてきましたし、
　だからこそ、


　【ますます多くの人に
　　プラスの痕跡を残したい】

　という思いで毎日を過ごしてきました。

　（今も過ごしています）



■ですから、

　こうしたお声を頂戴することこそが、
　私（鮒谷）にとっての
　
　【なによりのご褒美】

　であり、

　【なによりのモチベーションの源】

　であり、

　【なによりの自分を客観視するための
　　指標】

　ともなり、

　【なによりの財産】

　となっているわけです。
　


■このような取り組みを、

　長期間、毎日、地道に、飽きることなく、
　弛まず、続けていれば、
　
　【自ずから「逓増的人生」が開けてくる】

　と確信していますし、


　これからもますます、

　関わる方々にプラスの痕跡を残すべく
　全力を傾注する、

　そんな気持ちで頑張ってまいります。



■そして、ここまでお読み下さった
　あなたにもぜひ、

　【周りの方々から、

　　自分の提供する商品やサービス、
　　あるいは発信する情報、

　　などに対してフィードバックをもらう】

　というチャレンジをして頂ければと
　願っています。


　このシステムが稼働し始め、
　一定の期間が経過すれば、ある時点で

　【人生が劇的に変わる体験】

　をするはずです。



■もちろん、

　（テキトーに生きていくのと異なり）

　毎日、とてつもないプレッシャーが
　かかりますが、

　そのプレッシャーと向き合い、打ち克つべく、
　努力・精進しているうちに、

　【いつの間にか、不可能だと思っていたことが
　　できるようになっていた】

　ということが本当にあるのです。



■ある段階から、

　【雪だるま式に、それまでの取り組みが
　　報われ始める】

　といったことが起こるのです。


　それが嘘か本当かは、
　実際にやってみた人だけが分かります。
