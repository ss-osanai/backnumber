7121号 パチンコ、パチスロでの困窮経験のある人がそこをこそ喜ぶべき理由とは

■その昔、パチンコ、パチスロを
　やっていた頃の話。


　給料日前、
　どうしても金がなくなり、

　空腹を辛抱していたときに
　いつも思っていたのが

　「あの金さえあれば、、、」

　というものでした。

　ギャンブルに手を染めた人は皆、
　経験したことがあるはずです。



■目先の快楽や欲求のために使う金は
　一瞬で蕩尽されますが、

　そのあとの報い、責め苦は
　長い時間、続きます。


　パチンコ、パチスロで数万円を
　溶かすのに大した時間はかかりません。

　しかし、その数万円が
　手元にないという事実は、

　数日ないし一週間くらいの間、
　自分を苦しめます。



■未来を想像できないために目が曇り、

　意思決定を誤って
　安易な道に走ることで、

　将来、長きに渡って続く不幸へと
　足を踏み入れる人が少なくありません。



■ここまでは「お金の話」ですが、

　実はこれ「時間の話」に置き換えても
　同じことが言えますね。


　時間もまた、

　目先の快楽や欲求のために、
　一瞬で蕩尽され得る貴重な資源。

　休日の午前中にむさぼる惰眠、
　目的なき動画視聴、

　などは、
　まさにその典型といえるでしょう。



■消費は一瞬ですが、

　後になって、家庭や仕事などからの
　時間要求が厳しくなって

　「ああ、あのときの時間があれば」

　と後悔しても、あとの祭りです。



■ここまででお分かりの通り、

　お金についても、
　時間についても、

　つい一時の感情に任せて
　いたずらに浪費されがちですが、


　そうなりかけたときに

　「それらの資源が尽きた後の世界
　（困窮や切迫から生まれるストレス）」

　をぜひ、想像してもらいたいのです。



■こうすることによって

　「異時点間選択におけるバイアス」

　を回避できるようになりますし、

　人生における致命的な陥穽に
　はまりこむことも防げることでしょう。



■ちなみに、

　「ギャンブル経験や
　　時間の無駄遣いによって、

　　のちに大変な目に
　　遭ったことのある人」

　は、その黒歴史こそを喜ぶべき、
　と考えています（我田引水）。



■私（鮒谷）は
　冒頭に記したように、

　「パチンコ、パチスロで大負けして
　　熱くなった結果、

　　一時の気の迷いで財布の中の金を
　　すべて溶かして給料日前はいつも貧乏」

　という世界を、
　何度もリアルに体験してきました。



■いっときの感情に身を委ねての
　意思決定が、

　のちの生活に持続的ダメージを与えると
　身体に刻まれた経験があるので

　【当時の痛みを思い出すたびに
　　適切な意思決定（前倒し）を
　　行えるようになった】

　のです。



■なのでもし、あなたが

　「あの、情けない感情とともに
　　湧き上がる後悔」

　を経験されたことがあるのなら、
　そこを喜ぶべし、

　とお伝えしたいところです。

　（無駄な経験など何一つありません）



■、、、ということで、

　【人生（運命、幸福感）は
　　時間の使い方で決定される】

　ことになりますが、


　一日は24時間、

　この時間配分を戦略的に行うのか、
　成り行きに任せるのかが、

　数年後、十年後、数十年後の
　人生を決することとなるでしょう。



■この機会に、

　古くて新しい「時間戦略」について
　今一度、顧みられては、

　という問題提起を、
　ここ数日のメルマガで行った次第。



■「時間戦略」について学びたい方は、

　全部で６つの、違う切り口で
　多方面からお伝えしている
