　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

  【　行動するから、成果が生まれる　】

───────────────────────────────────



■今日は、

　当たり前だけど、
　理解している人が少ない、

　「行動するから、結果が変わる」

　という話をします。



■日々、いろんな方とお会いする中で、

　「人生を、ビジネスを変えるには
　　どうしたらいいですか？」

　と聞かれることがよくあります。

　
　この問いに対する
　一番シンプルなお答えは

　「行動すること」

　です。



■こういうと身も蓋もないのですが、
　当たり前の話です。


　去年と同じ事をやっていて
　来年、違う結果は現れません。

　先月と同じ事をやっていて
　来月、違う結果は現れません。

　先週と同じ事をやっていて
　来週、違う結果は現れません。

　昨日と同じ事をやっていて、
　明日、違う結果は現れません。



■来年、来月、来週、明日、
　現在と異なる結果を得たいのであれば、
　　
　今年、今月、今週、今日、
　そして「今」

　の行動を変えなければならないのは
　いうまでもありません。



■勉強しすぎ、考えすぎの弊害は
　行動しなくなる（できなくなる）こと。

　もちろん一定レベルまでの勉強は
　必要だと思いますが、

　そこから先は、やった人でなければ
　本当の意味で理解できない、

　ということも多いものです。

　（というよりも、
　　そういうものがほとんど）


　
■今と違う人生を送りたいのであれば、

　ある段階まで学んでから先は、
　とにかく、がむしゃらに動くしかありません。

　そうやって死に物狂いで動く人に
　学びが身体化されてくる（身につく）のです。


　
■泳ぎが上手になりたいと思って、

　水泳の教科書やら参考書やらを
　どれだけ買い込み、100時間読んでみても、
　
　プールに放り込まれたら
　その人はまったく泳げないはずです。

　
　
■逆に、プールに入って泳ごうとして、

　思い通りに泳げないことが体で
　理解できると、

　本に書いてあったことが、本当の意味で
　

　「ああ、あそこにあのように
　　書いてあったことは、
　　このことだったのか！」


　分かるようになるのです。

　そしてその学びを活かして、
　行動に微調整をかけ、泳ぎが上達します。



■この過程は泳ぎを上達するとき
　のみならず、

　人間のあらゆる営みにおいて
　有効に機能するものだと理解しています。



■だから、何かを始めるにあたって、
　最低限の学びは必要ですが、

　そこから先は、

　まず初めてみること、
　まず動いてみること、

　が重要です。



■そうやって、

　「（一定レベルまで）学び、行動に移し、
　　さらに学んでは実践して微調整、を繰り返す」

　というサイクルが回り始めると、

　そこから徐々に人生が、そしてビジネスが、　
　変化し始めるのです。

　

■このメルマガの読者さんの少なからぬ方々が

　「学びすぎて頭でっかち」
　　
　になっている感じを受けるので、
　あえて、今日はこのようなお話をさせて
　いただきました。



■といいつつ、

　ともすれば私も同じような傾向が
　あるので、


　最近は、学びすぎ、考えすぎに
　陥ることを戒め、

　「行動、行動、行動」

　と呪文のように唱えながら、
　毎日を過ごしています。
　　


　今日も人生とビジネスを楽しみましょう！
