7159号 置かれた場所で咲こうとするな

■私たちが生きるこの世界は、
　無限の可能性に満ちています。

　しかし、日々の暮らしを営む私たちは
　自分が知る範囲の世界に囚われがち。



■目の前に広がる世界が
　全てであるかのように錯覚し、

　その枠内での最善策を
　追求してしまうのです。

　しかし、その先にあるものは

　【部分最適化された解】

　に過ぎません。



■大きな飛躍を求めたいのなら、
　未知の世界への挑戦が不可欠です。

　そこには必ずしも成功が
　約束されているわけではなくとも、

　新たな可能性の芽を
　見いだせるかもしれません。



■一方、未知の世界から
　距離をおいていると、

　【非線形の進化】

　は望めません。


　今、生きている世界での、
　ひたすらな最適化（＝部分最適）

　を追求するのと同義ですから
　当然の帰結といえるでしょう。



■この構造を理解した上で、

　【リスクとリターンの最適化】

　を追求することが、

　人生やキャリア、ビジネス、投資における
　最善の戦略となるのです。



■そのような背景や考えから、

　昨日、私（鮒谷）は
　NFTセミナーを開催しました。


　私自身、NFTについて深い理解や
　造詣があったわけではありません。

　だからこそ、むしろ積極的に、

　確実な有名・無形のリターンが
　保証されていなくとも、

　知らなかったことから生じる
　機会損失を恐れたわけです。



■私たちが意思決定をするにあたって、

　【リスク＆リターンを勘案した上で、
　　所有資源における投資比率の最適化】

　を適切に行えれば、

　人生や資産における
　「喪失下限」を限定しつつ、

　合理的に「上振れするリターン」を
　追い求めることができます。



■こうした投資姿勢は、
　時間軸を長く取れば取るほど、

　豊かな人生が築かれる可能性を
　高めてくれるに違いありません。



■こうした生き方は
　言葉を換えれば

　【部分最適志向から全体最適志向への
　　パラダイムシフト】

　とも言えますね。



■今の環境や状況に固執している限り、
　部分最適にしかなりませんから、

　今の環境が自分の諸要素と
　照らし合わせて不適合だと思うなら、

　置かれた場所で咲こうとしている
　場合ではありません。



■広い社会においては、

　より自分に適合する環境が
　ほぼ確実に存在するはずであり、

　ただ、そうした場所を
　知ろうとしないから

　「相対的に苦しい場所」

　に留まっているだけのことが
　多いように思われます。



■たとえば、
　これは私（鮒谷）の例ですが、

　サラリーマン生活に執着していたら
　そこで咲こうと悪戦苦闘しているうちに、

　「枯死」していたであろうことは
　容易に想像できます。



■引いて、眺めて、旅して、

　咲けそうだと思った
　土地に根を下ろしたからこそ、

　（ささやかながらもなんとか）
　花を咲かせられました。



■こうした

　【引いて眺めることで
　　初めて理解できるパラダイム】

　は、実にシンプルです。

　しかし、この事実を自分の身体で理解し、
　実践できるようになるまでには、

　長い時間がかかりました。



■ただ、長い時間がかかりましたが、

　このルールに気づくことができたことは
　とても幸運だったと思います。


　気づかぬままに
　一生を終えてしまっていた、

　そんな可能性だって
　十分にあったのですから。



■そんなわけで、

　以上の文章が、あなたが
　未知の世界への挑戦に踏み出し、

　（リスクを限定しつつ）

　人生において大きな飛躍を遂げる
　きっかけになれば幸いです。

　そして、その先に広がる新たな可能性を、
　あなたの目で確かめていただきたいのです。
