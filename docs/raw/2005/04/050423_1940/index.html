＜昨日の活動ダイジェスト　読了後の再読は、記憶の定着に必要不可欠。＞
───────────────────────────────────


■以前も書いたことがありますが、私は結構本を読むほうだとおもいます。

　そしてその記憶が脳内に留まる定着率もそこそこ高い部類ではないかとも
　思っています。


■でも、以前はそうではありませんでした。
　読んだそばから、どんどん忘れていくんですよね。

　なんでだろう、なんでだろう。いつも悩んでました。


■それがある方法を導入した途端に、そういうモッタイナイことが半減した
　のです。


■その方法とは、至って簡単で、

　自分が本を読んだときに、ぴん！！ときたところをがんがん赤ペンで
　線を引く。


■本を読むときにきれいに読むな、というのはたしか、立花隆氏の本
　から学んだように記憶していますが、とにかく、スピード優先で
　線がゆがもうが、ずれようが、気にせずどんどん線を引っ張っていく
　のです。


■そして読了したら、そのあと１回、２，３分でよいので、赤線を
　引いたところだけ再読してから眠るのです。

　
■そしてその後、翌日に赤ペンで線を引いたところを同じく数分かけて
　もう一度読み返す。

　でも、まだ、その本は書棚にしまわないで自分のデスクの横においておく。


■さらに一週間後くらいに、もう一度、数分～10分くらいかけて
　読む込むと、もうその本の骨格や重要なポイントはほぼ全て頭の
　中に入っているのです。


■忘却曲線というのは最初の24時間、48時間、ないし１、２週間で
　一気に落ちていくものですから、その最初の記憶の減衰期に一気に
　脳みそに喝を入れるのです。

　重要事項を叩き込む！！という感覚でしょうか。


■忘れ去ろうとしていたものが、再読することで

　【短期記憶】⇒【長期記憶】

　に落とすことを手伝うんです。


■一度、長期記憶に落ちたら、あとは数ヶ月に１回、数年に１回程度
　のサイクルで読み返してみても、鮮明にその内容や感動を思い出す
　ことができるようになります。


■人間の脳は反復されてインプットされたものは

　「これは重要なものだ！」
　
　と認識するので、そうやって自分の脳みそをだましだまし、きっちりと
　必要なことを入れてやるのです。


■ちなみにこれは読書だけの話ではありませんよ。
　セミナーに行ったり、人の話を聞いたら必ずメモをとらなければ
　なりません。

　そしてそれを寝る前に読み返してみるのです。


■さらに、翌日、

　朝、出社してからコーヒーを飲みながら、トイレに入りながら（失礼）、
　サラダをほおばりながら（編集後記参照）、道を歩きながら（周りに注意）、
　タクシーに乗りながら、電車の到着を待ちながら、

　など隙間時間を利用して、再読する。


■一週間くらいはこんなちょっとした時間を無駄にせず、ちょこちょこと
　カバンの中にしまったメモ帳を取り出して眺めてみることです。

　あなたの記憶力が一気に改善することを保証します。






　・・・


　って、なんだか

　【今日のピークパフォーマンス方程式】

　のような文体になってしまいましたね。

　明日の方程式にとっとけばよかった！？（笑）


■そんなわけで、昨日は今までに読んでた本の中でもこれは！！という
　本の赤ペンを引いた部分を久しぶりに読み返していました。

　こういう読み方をすると、数十冊分の読書が半日で出来るように
　なったりするからお勧めですよ。

　新たな感動もよみがえります。


■それでは今日の方程式をどうぞ。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式。＞

　【　食べることは忘れても読むことを忘れるな。　】

───────────────────────────────────


■何かの本で読んだことがあるのですが、

　「食べることは忘れてもいい。だが、読むことを忘れるな」

　と言った人がいました。


■ひるがえって、今の自分自身を生み出した一番の原因は何かと
　尋ねられたならば……

　当然のことながら、私自身の思考・行動パターンはさまざまなもの
　から影響を受けていると思われますが、

　中でも大きな習慣はやはり、本を読む習慣だと言えるでしょう。


■これについて、私は両親に非常に感謝しているのです。

　うちは両親ともに本をよく読みますし、たくさんの蔵書がありました。


■幼稚園に上がるか上がらないかという時期から、読み聞かせを
　してもらっていましたし、

　小学校では童話、伝記などをどんどん読んで、
　そこからやがては小説であったり、それが長じて実務書なんかも
　読みはじめるようになりました。


■冒頭に述べました、

　「食べることは忘れてもいい。だが、読むことを忘れるな」

　との言葉は、これは本当に真実だと思っています。


■自分が何かしらの決断や選択を迫られたとき、その決定を下すためには
　必ずその決定を導くまでの自分のバックボーンが必要になります。


■そのバックボーンは自身の経験であったり、
　あるいは人と会うことによって得た知識であるとか、
　あるいはセミナーで学んだ内容かもしれませんが、

　時と場所を選ばず、しかも短時間・低コストで大量の情報収集を行い、
　自分の骨組みを強固にする。

　その効率性という点では読書の右に出るものはないと思います。


■そういうわけで、そんな習慣を植え付けてくれた両親に対して、
　私は多大な感謝の念を持っているのです。


■では、この「平成・進化論。」読者さまのような、お忙しい
　ビジネスパーソンのあなたは、どんな読書をしていったら
　よいのでしょうか。

　私が考えている読書習慣を明日、提示したいとおもいます。
