7362号 コミュニケーション能力はテクニックであり、スキル

■昨日のメルマガでは

　【コミュニケーションに難のある人は
　　徹頭徹尾、干されていく】

　という（残酷な）話を書きましたが、


　私（鮒谷）自身、実際に
　そんな仕打ちにさんざんあってきて、

　ようやく真面目に人間関係の作り方に
　ついて学び始めました。



■学んでどうなるかは分からなくとも、
　学ばないことには

　【永久に干され続ける】

　ことだけはハッキリしていたので、

　天は自ら助くる者を助く、の精神で
　勉強し、試行錯誤を重ねてきたつもり。



■そんな今、改めて思うのですが、

　「干している側は
　　干しているつもりはなかった」

　んですよね。


　ただ、

　親しくしたいと思っている人と
　優先的に付き合っているうちに、

　（認知能力も、時間も、
　　お金にも限りがあるので）

　そこまで思っていない人には
　それらの資源を割くことができず、

　結果として

　【干すー干される、の関係】

　が生じていたのでしょう。



■つまり、おそらくは、

　干している側、干されている側の
　両者とも無自覚のうちに

　【「干すー干される」関係】

　が自然に形成されていたということに
　過ぎません。



■だから

　「干されている側」

　が自分は不遇だとか
　恵まれていないといって、

　誰かのせいにして恨んだところで、


　リア充さんたちにとっては
　意図的に何かをしたわけではないので

　「は？何を言っているのだろう？」

　くらいにしか思わないわけですね。

　当の本人は、誰かを干しているつもりは
　まったくないのです。



■とはいえ、こうして自然発生した

　【「干すー干される」関係】

　が社会全域に広がっていくと、
　どうしても、

　さまざまな面で恵まれる人と、
　あらゆる方面で徹底的に干し上げられる人、

　の二方向に開いていってしまうのは、
　構造上の問題なのかもしれません。



■私（鮒谷）自身、長らく

　「完全に干される側」

　でありましたから、

　誰にも相手にされず、干され、
　隅っこに吹き寄せられていく、

　その不安とか恐怖、さらには痛みは
　普通の人よりも遥かに強かった、

　と思います。



■だからこそ、自分自身を助けるために
　コミュニケーションの勉強をしました。


　どれくらい学んだかというと、

　社内で管理している
　過去に配信をしてきた、

　号外分まで含めたメルマガのバックナンバー
　（１万通弱）において、

　以下のワードで検索をかけてみると
　それぞれ

　「コミュニケーション　1620記事」
　「コミュニケーション能力　483記事」

　と出てくるくらいには
　意識を払って自分を教育してきた、

　のだと思います。



■その結果は顕著で

　「干されている」

　という感覚は
　とっくの昔になくなりましたし、

　さらには

　「コミュニケーション能力のおかげで
　　心身ともにどんどん豊かになっている」

　ことを実感しています。



■そしてここが重要なところですが、

　学び、トレーニングを
　重ねて分かったことは、

　【コミュニケーション能力は
　　テクニックであり、スキル】

　であったということです。

　だから学べるし、
　学ぶ価値があるですよね。



■これが先天的な能力であるならば、

　学ぶ意味はないし、
　諦めるより他、ありませんが、

　そうではないところに
　希望のともしびが灯ることになるのです。



■さらに嬉しいことには
　（そして多くの人が誤解していることは）

　「声がでかい、オラオラしている
　＝コミュニケーション能力が高い」

　とは決して言えないということ。


　なんならむしろ、

　　「コミュニケーション能力の低さを
　　声のデカさで糊塗している」

　まであると思っています。



■私（鮒谷）は
　まったく真逆のタイプですが、

　昔と違って、今、改めて
　間違いないと思う

　「声がでかい、オラオラしている
　＝齢を重ねるにつれ、各種資源が逓減する」

　という傾向が顕著であるということ。

　ざまーみろ。



■声が小さく、

　いつも子羊（こひつじ）のように、
　ビクビク、オドオドしている人のほうが、

　案外、人の気持ちがよく分かるから
　コミュニケーション巧者になれる素地がある、

　というのは、
　およそ間違いなさそうです。



■もちろん、

　「声がでかい、オラオラしている」

　人がうまくいっているケースも
　一部、あるのかもしれません。


　しかし、たとえそうでも、

　声の小さな人が
　そのやり方を真似てもうまくいかないし、

　彼らを真似るよりも、まったく違う道を
　模索したほうが良さそうです。



■その上で、そうした人とは

　「死ぬまで接する機会もなければ、
　　接する必要もなくなる」

　という環境を作り（バリアを張って）、


　その最高位居心地の良い環境で

　死ぬまでオラオラした人間とは
　触れ合うことのないように距離を起きつつ、

　まともにコミュニケーションの取れる、
　幸せな人同士で付き合っていれば、

　それでよいのではないでしょうか。



■なので、もしあなたが、

　「声がでかい、オラオラしている」

　系の人ならば、どうぞそちらの方向で
　頑張って下さい（適当）



■反対に、『同床異夢音源』の内容は、

　「声の小さな子羊系」

　で、つねい

　「言いたいことも言えない
　　こんな世の中じゃPoison、、、」

　と嘆いているような人にとっては
　大いなる福音となるはずです。



■この話の詳細について
　さらに知りたい方は、ぜひ

　『同床異夢音源』

　をお聴き下さい。



■とはいえ、
　これ以上の内容も分からないのに

　「聴いて下さい」といわれても
　「聴けるかよ！」と思われるのは、

　極めて真っ当な反応でしょう。



■ですので、以下に、

　音源をお聴きくださった方から
　お寄せいただいた感想を紹介するので、

　それをご覧になった上で、

　音源の内容や品質について
　ご想像、ご確認の上、お求め下さい。

　　お求めは忘れないうちに、今すぐどうぞ。



■ちなみに、
　たとえ音源を聴かれなくても、

　以下にご紹介している大ボリュームの
　感想一つ一つを読むだけで、

　音源でお伝えしている内容の
　かなりの部分まで推察できるかと。



■というか、
