■上述の通り、昨日の夜から、

　リスクが極小化された実践＆実証済みの
　方法論をみんなで学ぶ不動産勉強会、

　のご案内を開始しております。



■勉強会の正式名称は
　以下の通りとなります。
　　

　--------------------

　【不動産投資 未経験者のための、
　　知識ゼロからはじめる騙されない&カモられない
　　リスク極小・クマムシ式 不動産投資勉強会】

　～不動産投資と聞くだけで
　　腰がひけてしまっていた、

　　そんなあなたの認識を180度書き換え、
　　勇気と希望とキャッシュフローを手に入れる

　　「クマムシ式 不動産投資術」

　　とは

　--------------------



■あなたもご存知の通り、

　何千万、何億と借り入れを膨らませ、
　うまくいくことを願いながらも

　「死なばもろとも」

　という形の不動産投資もありますね。



■そうした投資を行っている方の、
　さらに上を行く方は、


　何億、何十億と借り入れしながらも
　リスクを限定させている、

　（なにが起きても詰まないように
　　状況をコントロールできている９

　手練れの、というのか、
　プロフェッショナルの、というのか、の

　「熟達した不動産投資家」

　と呼べる人が（少数ながら）
　存在するのも、また事実。



■私（鮒谷）の友人、知人にも複数、
　そのタイプの人がありますし、

　こうした人は

　「内包するリスクには目をつむり
　　死なばもろとも、的な破壊的事象は
　　絶対に起こらないと願う」

　のではなく、


　「めったに起こらないけれども
　　ひとたび起きたら甚大な損害を被る
　　ブラック・スワン系リスク」

　にも適切に備えて、何が起きても
　生き残れる態勢（耐性）を整えつつ、

　同時に大きな収益を上げています。

　さすがというのか、凄いというのか、
　いつも羨ましく、指を加えてみています、、



■ただし、こうした方は、
　可処分リソースの相当の部分を、

　長きに渡り、
　不動産投資に投下してきた方であり、

　何ごとによらず、ではありますが、
　簡単に模倣できるものではありません。



■もしあなたが、こうしたタイプの
　投資家でいらっしゃるのであれば、

　今回の勉強会の対象ではありませんし、
　参加しても時間とカネの無駄になるので、

　お申し込みになることは
　お勧めいたしません。

　というか、来ないで下さい。



■ただ、もしあなたが
　こうした備えをきちんと行える

　「熟達した不動産投資家」

　でもないのに、


　不動産投資に興味を持ち、関連する業者や
　コンサルタントの過剰セールスに興奮し、

　高いレバレッジをかけて希望的観測で
　何も起こらないことを願う人であるならば

　「転ばぬ先の杖」

　を確保しつつ、ならびに

　「リスクを極小化させて投資する
　　こうした手法もありますよ」

　という知識を、あらかじめ、
　入れておかれてはどうでしょう。
　


■タレブ流にいうならば

　--------------------

　川の深さが平均で
　4フィート（約120センチ）なら
　渡ってはいけない。

　※あくまで平均だから水深8フィートの
　　足のつかないところがあり、
　　溺れる可能性がある（鮒谷注）
　　
　--------------------

　の教えに反し、
　　

　これまでは（そして、みんな）
　これまで大丈夫だったのだから、
　私も大丈夫だろう、

　と信じ込んでいると、

　不確実性に満ち満ちた世界において
　足元を救われる可能性があります。



■実は、こうしたことはなにも

　「不動産投資」

　に限ったことではありません。


　▼私の勤める会社は
　　退職するまで安泰だ、とか、

　▼私は80歳（90歳、100歳）まで
　　健康を維持し続けられる、とか、


　▼家族みんな元気で健康で
　　向こう数十年、大丈夫、とか、

　▼なにかあっても最後は国が
　　なんとかしてくれるだろう、とか、


　▼保険や年金などの社会保障制度に
　　身を委ねていれば安心だ、とか、

　▼大災害に遭う人もあるかもしれないが
　　私だけは避けられる、とか、

　▼日本国も日本円も、未来は怪しい
　　けれども生きている間は問題ない、とか、


　思い込んでいる人があります。



■でも、果たしてこうした希望的観測に、
　どんな根拠があるのでしょうか。

　不動産投資以外の、
　あらゆる投資において無数の

　「川の深さが平均４フィート」

　的な思考（＝無思考）で生きている人も
　少なくないように思われます。



■生きるということは、
　
　（自覚の有無にか関わらず）
　なにかに投資をしている、

　ということ。


　自覚がなくても

　▼勤務先に投資

　▼円に投資

　▼社会保険制度に投資
　
　▼既存の社会システムに投資

　▼籍を置いている国家に投資

　しているのです。



■そのことを自覚し、
　何が起きても死なないように

　「人生や生活について
　　クマムシ的にヘッジをかける」


　※クマムシについては以下の通り。

　「水生の無脊椎動物であるクマムシは
　　地球上で最もたくましい動物と
　　考えられている」
　　
　（ナショナルジオグラフィックHPより）
　

　ことは不測の事態において
　不意打ちをくらわぬために、

　平時においてこそ
　準備をしておくべきでしょう。



■昨今のように不確実性が
　ますます高まっており、

　明日のことも読めぬ時代にあっては
　明日、何が起きても生き残れる、

　そのための準備を整えておくべきであり、
　準備の一環として、

　これまで各種の具体個別の備えについても
　個別コンサル等でお伝えしてきました。


　このたびはその中でも
　強力なオプションの一つとして

　「クマムシ式 不動産投資術」

　をご案内している次第です。

　

■このクマムシ式 不動産投資術、

　確かに、いざ、実際に始めるときには
　元手が500万円ほど必要になりますが、

　逆に言うと、不動産投資において
　（借り入れなしで）その程度の初期費用、

　というのは、


　相応の所得や貯蓄のある会社づとめの方や
　フリーランス、経営者にとって

　「許容可能な損失」

　でありつつも、相応に確度の高い
　リターンの見込めむことができる、

　真剣に取り組んでも良い対象と
　言えそうです。



■講師の長田さんご自身も、

　会社づとめ（現在も）のシングルマザー、
　ということで、

　時間もなければ
　過度のリスクを取れない中

　「クマムシ経営的不動産投資」

　を追求され、


　投資開始からわずか５年で、
　所有不動産からのキャッシュフローを、

　約1000万円/年、まで
　持っていかれることに成功されました。


　まさにクマムシ経営の鏡。
　クマムシ経営の不動産投資版。



■以下、そんな長田さんの簡単な
　プロフィールとなります。


　--------------------

　日系企業勤務後、
　米国のアイビーリーグ大にてMBAを取得。

　帰国後は外資系企業に転職するも、
　ブラックな職場環境や、雇用の不安定さを実感し、
　給与所得のみの生活にリスクを感じ始める。

　ネットビジネス等、様々な方法・手段を模索し、
　最終的に不動産投資に落ち着く。


　その後、リスクを抑えつつ順調に規模を拡大し、
　投資開始から約5年で、

　所有不動産からのキャッシュフローが
　約1000万円/年となる。

　また、母子家庭の貧困率約50％の日本で、
　シングルマザーとして、子供を私学に通わせ、
　都内人気エリアのタワーマンションに在住。

　--------------------



■不動産投資経験がまったくない人を
　対象として、ご自身が実践してこられた

　「リスクを極小化しつつ、
　　不動産投資で収益を上げる」

　方法をコンテンツとしてまとめあげ、
　私たちに教えて下さいます。

　（私（鮒谷）も生徒として学びます。
　　一緒にお勉強いたしましょう！）

■人生の岐路は、思わぬところから

　私（鮒谷）は長田さんが
