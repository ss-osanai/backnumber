7512号 目的と手段の主客転倒は容易に起こり得る

■昨日はTwitterで


　----------------

　幸せが目的、お金は手段。
　幸せが目的、起業は手段。

　幸せが目的、ビジネスは手段。
　幸せが目的、キャリアは手段。


　金があれば、
　起業すれば、

　ビジネスで成功すれば、
　立派なキャリアを積めば、

　幸せになれると勘違いしている人、
　多すぎない？

　幸せになる為に最適化された
　人生システムの構築が最終目標。

　https://twitter.com/shinkaron/status/1770638973473640620

　----------------


　という投稿をしました。



■まあ、読んで字の如くなんですが、

　手段に没頭しているうちに
　目的を忘れてしまい、気づかぬ間に

　「主客転倒」

　していることってありますよね。



■幸せになるために
　ビジネスを始めたはずなのに、

　幸せになるために
　お金を稼ごうと思ったのに、

　幸せになるために
　懸命にキャリア構築してきたのに、


　いつの間にやら、

　「ミイラ取りがミイラになる」

　ような感じで、

　手段のために苦しみ、
　なぜそれで苦しんでいるのか、

　当人も分からなくなってしまう。



■そうならぬよう、つねに

　「自分は幸せになるために
　　これ（今、取り組んでいること）を
　　しているんだよね」

　「本当に今、
　　これがやるべきことなんだよね」

　と自問自答を重ねて、
　確認する必要があるでしょう。



■こうして、ある程度まで
　その分野で頑張ってきて、

　幸せになることを阻害していた要素を
　一通り取り除いたら、


　いったん、自分の人生を
　「引き」で眺めて、

　惰性でその場に留まるのではなく、

　他の、もっと大きなボトルネックを
　取り除くための旅に出立するべきでは、

　などと思うのです。



■ある世界にどっぷりと入り込み、

　そこで称賛されたり、お金なり、
　名声なりの魔力に取り憑かれると、

　それ以上、そこを掘っても
　幸せになれる要素がなかったとしてさえ、

　「掘らずに入られなくなる」

　といった、ある種、依存性のある
　中毒状態に陥りがち。



■そうならぬよう、いつも心の片隅に

　----------------

　幸せが目的、お金は手段。
　幸せが目的、起業は手段。

　幸せが目的、ビジネスは手段。
　幸せが目的、キャリアは手段。

　----------------

　という言葉を置いておき、

　ことあるごとに脳内で
　回し続けるべきなのです。



■こうした心がけによって

　「初心（＝幸せでありたい！）」

　を忘れずに済むのではないか、
　とも。



■今、なにかに過集中していて、

　それでも、どういうわけか
　幸せを感じられずにいるとするならば、

　「主客転倒の隘路（あいろ）」

　にハマり込んでいる可能性が、
　相応にあるはずです。



■そんなときこそが、


　「深呼吸して、

　　自分と対象物やテーマ、分野の
　　双方を『引き』で見て、

　　幸せになるための（部分最適ではなく）
　　全体最適のための次の一手はなにか」


　をゼロベースで自らに問うべき
　タイミングなのでしょう。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■油断していると簡単に
　　目的と手段の主客転倒が起こる。

　■手段に没頭している間に、何のための
　　手段であったのかを忘れてしまうのだ。

　■そうならぬために、

　　「深呼吸＆引きで見る」

　　その上で

　　「全体最適から見た次の一手を検討する」

　　というプロセスを繰り返す必要が
　　あるのではないか。

　■すぐに目的を見失ってしまい、目の前の
　　事象に入り込んでしまう私達だからこそ。
