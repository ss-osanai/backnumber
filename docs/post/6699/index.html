6699号 課金＆僥倖待ち、という大っぴらには語られにくい真実

■このメルマガは

　「エフェクチュエーション勉強会」

　大幅値上げ前の
　最後の配信となりますので、

　歯に衣着せず
　率直な思いを記しますね。



■以下は

　--------------------

　【重課金音源】

　最速でビジネス無双を確実なものとする
 「重課金戦略」放談会音源
 
 ～知っている人だけがフル活用、
　 表立っては言いにくい、
 　人生/ビジネスにおける不都合な真実とは

　--------------------

　でもお伝えしたことですが、


　積極的に有料コンテンツに
　アクセスしている人は、

　期待する成果を手に入れる確率は
　格段に高まります。



■ただ、これをそのまま言ってしまうと

　「カネを使わないと
　　成果を挙げられないのか（怒）」

　とか

　「学びにカネをつかってないけど
　　うまくいっている人もいるではないか」

　という反撃の台詞が
　返ってきそうです。



■でも、少なくとも

　「カネを使わないと
　　成果を挙げられないのか（怒）」

　という怒声や罵声や叱責に対する
　私（鮒谷）の答えは

　「残念ながら
　　そのようになっているようです」

　というものになります。



■もちろん、

　これを聞かれると不快な気持ちに
　なる方もあるとは思いますし、

　そうした方々のお気持ちを
　逆撫でしたくはないので、


　できることならば

　「そんなものは関係ありませんよ
　（ニッコリ）」

　なんて言いたいところなのです。

　

■でも、

　そんなことを言ってしまうのは
　むしろ不誠実、

　と思っています。

　少なくとも現実は
　そうはなっていないから。



■綺麗事をいっている人だって、
　多くの人が（たとえば）我が子に、

　お受験とか、
　中学受験とか、
　高校受験とか、
　大学受験とか、
　お稽古ごととか、

　その他、あらゆる学びに対して
　みな「課金」してますよね。


　
■これは自身に対しても同じで、

　語学とか、
　資格とか、

　その他、さまざまな学びに対して、
　「課金」しています。

　そういうことですよ。


　
■私（鮒谷）は、

　建前とか綺麗事は嫌いだから
　はっきり言いますが、

　なにかの分野において
　上達したかったり、

　結果を残したければ、

　「可能な限りの課金をすべし」

　だと思っています。



■支払先の相手だって、

　課金しているから
　真剣に取り組んでくれるわけで、


　一切の課金なく、善意やボランティアで
　やってくれる人があったとしても、

　長続きしないし、
　気を緩めることだって出てくるはず。



■ですから、

　真剣勝負の学びによって
　持続的な成長を遂げたいのなら

　「課金一択」

　と思ってきました。　



■まあ、これはあくまで
　私（鮒谷）個人の考えでありますが、

　少なくとも私は
　社会人なりたての頃からずっと、
　

　周囲の人たちが車やら時計やら
　洋服やら自宅やらを買ったり、

　カラオケやら旅行やら
　キャバクラ＆風俗通いや、

　合コンなんかをしているとき、


　可処分所得（＆時間）の
　ほぼすべてを学びに投下してきました。

　おかげで未だに借家住まいですが。



■こうした考えや生活を強要するつもりは
　一切ありませんが、

　振り返ってみて、こうした

　「前倒しの学習」

　が、

　現在の高いQOL（平穏な精神）に、
　大いに寄与していることは間違いない、

　とは思っています。



■そんな経験があり、

　しかも私（鮒谷）が一緒に学び、
　切磋琢磨してきた課金仲間？たちも、

　時間の経過とともに、どんどん
　心身共に豊かになっていき、


　相応に満足している生活を
　送っているのを見て

　「やっぱりそういうことだよな」

　とは思います。　



■今日は詳しく記しませんが、
　これと「格差」の問題は別物です。

　それはそれとして
　解決されるべき社会問題であっても
　
　「課金からもたらされる
　　学びの成果」

　とはまるで別の問題ということ。
　


■そしてもう一つの、
　よくある異論、反論、叱責として

　「学びにカネをつかってないけど
　　うまくいっている人もいるではないか」

　というものがあるのですが、
　これに対するお答えは以下の通り。

　「おっしゃる通りですね」



■たしかに

　「学びにカネをつかってないけど
　　うまくいっている人」

　も、世の中にはたくさんいるのです。


　でも、そういう人は、
　必ずどこか別の場所で

　「（その辺ではとても学べないような）
　　高いクオリティの学び」

　を得ています。



■多くの場合、それは、

　「高い能力を持つ、
　　自分を可愛がってくれる人、
　　あるいは相性の合う人」
　
　からの学びです。

　

■私（鮒谷）であれば

　「自分を可愛がって下さる数名の、
　　人生、ビジネスの大先輩」
　
　とのご縁をいただくことによって
　劇的な変化が生じました。



■でも、これらの出会いは
　「僥倖」であって、
　
　狙ったからといって
　そんなご縁をいただけるとは限らず、

　またいつ、そのようなご縁に
　恵まれるかも分かりません。

　（下手をすれば一生、出会えない
　　ことだってあるかもしれません）



■こうした出会いと比べると、

　「課金」

　は対象さえ間違えなければ
　確実に自分を向上させてくれます。



■しかも「許容可能な損失」の
　範囲内での課金であれば、

　リスクを限定した状態で、

　学びから生まれる高いリターンを
　狙いにいけるわけですから、


　そんな投資すらできなければ
　（そんなリスクすら負えないのなら）

　資産全額を普通預金口座に
　寝かせたままにしているのと同じで、

　リターンを得られなくても当然だろう、
　と思ってしまうのです。



■私（鮒谷）も、
　社会人なりたての頃は
　
　「なけなしのカネ」

　しかありませんでしたが、

　学びに対しては
　最初から積極運用に次ぐ積極運用、


　たとえていうならば

　普通預金口座には一円も入れずに
　全額を米国ETFにぶち込んでいる、

　（＝貯金せずに、全額を自己投資）

　そんな感じでした。



■最初は確かにきつかったですが
　やがて現実の資産運用と同じように
　
　「積立＆複利」の力によって、
　
　学びの成果を実生活に
　反映させられるようになりました。



■なので、人生戦略においては

　（各人の年齢や環境を
　　総合的に勘案しつつ）

　--------------------

　１、多少なりとも経済的余力があるならば
　　　許容可能な範囲での課金一択

　２、経済的余力がほとんどないならば
　　　出会いを重ねて「僥倖」を求める

　--------------------

　のいずれか
　（もしくはいずれも）しかない、

　と考えています。



■繰り返しますが、これはあくまで
　私（鮒谷）の個人的な考えです。

　ただ、私自身はまさに

　「１」を最優先しつつ、
　そうすると手元にカネがなくなるので、

　次に余剰時間を全て投下し、
　「２」も同時に進めていきました。



■これによって、
　
　「最初は少しづつ、
　　あるときから一気に」

　人生が変わり、
　大逆転を遂げられました。



■ここに至って、心の底から

　（自分が何の成果も
　　出していない段階で）

　車やら時計やら洋服やら
　自宅やらを買ったり、

　カラオケやら旅行やら
　キャバクラ＆風俗通いや
　合コンなんかをしなくてよかった、

　（前倒しで学び続けていて良かった）
　
　と思えるようになりました。



■そんなわけで、もしあなたが

　--------------------

　学びによる無双状態を目指したい

　--------------------

　そんなあなたにこそ、この度の

　「エフェクチュエーション勉強会」

　強くお勧めいたします。
