【 （あと２日） セールスとは詰将棋のようなもの】

───────────────────────────────────



■たとえは悪いかもしれませんが、

　（ある意味において）

　【セールスとは詰将棋のようなもの】

　といえるでしょう。



■時々刻々、投了（受注）までの
　道筋（シナリオ）を描き、

　相手（お客さま）の反応を見ながら、
　微調整を加えつつ、

　「投了（受注）」

　にいたるところまで、

　一手一手を進めていくところが、
　セールスが詰将棋に酷似しています。



■もし、あなたが将棋の出来る方であれば、
　一人で将棋をされたことがあるでしょう。


　自分と相手の立場を交互に入れ替えつつ、
　一手、一手、駒を進めていくわけですが、

　こちらが一つ手を進めれば、
　相手も対抗する形で一つ、手を進める。

　
　そんなシミュレーションの過程の中で、

　こう打てば相手はこう対応するだろう、

　そこを抑えるために
　次にここに、こうした手を打とう、

　といった未来予測を行いながら、
　手を進めることとなります。


　このプロセスが営業と詰将棋が
　とても似ているのです。



■将棋においては、

　最終的に他に打つ手がなくなり、
　玉が詰まされてしまったときが、

　「投了」

　となるわけですが、


　それが、セールスにおいては

　お客様が納得も得心もいき、
　断る理由も一切なくなり、

　申込書にサイン、あるいは申込フォームに
　必要事項を記入してクリックしたとき、

　ということになるわけです。



■つまり、

　誤解を生じかねない表現かもしれませんが、
　あえて言うならば、
　
　「詰将棋もセールスも、投了（受注）まで
　　順を追って手を進めていくゲーム」

　のようなものと言えるのかもしれません。



■セールスも将棋と同じで、

　相手のことを考えることなく、
　自分の駒だけを見て、手を先に進めても、

　期待する結果（勝利、受注）に
　至ることはできません。


　あくまで相手の動きを眺めつつ、

　その先を読み、考えられる手を
　全て出し尽くした上で、

　断る理由を全て封じることによって
　勝利（受注）が転がり込んでくるのです。



■これが出来る人は昇級・昇段していくし、
　（＝売れるセールスになっていくし）

　できない人はいつまでたっても
　ヘボ将棋（ダメセールス）のまま、

　ということになるでしょう。



■営業は、彼我の現状を把握し、

　一手一手のプロセス管理を
　詳細にできる人のみが成果を手にできる、

　そんな厳然たる力の差が表れる、
　シビアな世界です。



■このプロセスを、

　次回のセールスセミナーでは事細かに
　初級者、初心者にも分かる形でお伝えし、

　最速で

　「有段者」

　にまで導くことをお約束いたします。



■あと「２日」で早期割引がなくなります。
　

　そして、年明け以降、
　新しいプロジェクトを立ち上げるので、
　
　当面（あるいは二度と！？）、
　この内容でのセミナーは行いません。


　お申込みはお早めに。
