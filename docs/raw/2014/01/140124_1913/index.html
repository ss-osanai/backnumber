 【　一ヶ月に50-100冊の本を買い、速読も熟読もしない、鮒谷の読書法  】

───────────────────────────────────



■よく本の買い方、読み方について尋ねられます。

　以下、ご参考までに。（あくまで私の方法です）

　
　私は、20代の頃から本の購入にあたっては

　「迷ったら、買う」

　を心がけてきました。

　（もちろん、そのときどきの、許される可処分所得の
　　範囲内ではありますが）

　

■迷ったり、悩んだりするヒマがあったら、
　さっさと買ったほうが早いし、

　どうせ我慢できず、欲しくなって買うんだったら、
　迷う時間を最小化したほうが、

　トータルでみたとき、安くつきます。

　なにより価値の高いのは「自分の時間」だから。
　


■もちろん、

　残念ながら、それをすべて読むことができるほど、
　時間が潤沢にあるわけではありません。

　
　というよりもむしろ、

　一日の可処分時間すべてを投入しても
　とても追いつかないほどの本が、

　多い時は、時間差で日に３度くらいのペースで
　陸続とAmazonから事務所に送り届けられます。
　

　けれども、そもそも全部読めない（読まない）ことを
　前提としているからそれでOK。



■ただ、１日に数冊といったペースで
　着々と積み上がっていく本ではありますが、
　
　熟読でも精読でも速読でもなく、

　「とりあえずパラパラとめくり、眺める」

　ようにはしています。


　長年これを繰り返していると、

　一冊の本にかかれていることの

　「勘所」

　みたいなものが分かるようにもなってくるもの。



■購入するときに、さらっとAmazonの要約やレビューも
　眺めているし、

　本が届いてからもプロフィール読んだり、
　まえがきやあとがき、
　
　書籍によっては推薦文や解説文、
　増補版出版にあたっての補遺、

　など、いろいろ周辺情報に目を通すので、
　それらを頭に入れた上で、

　「短いときには、本文をせいぜい数分～５分程度」

　で読み流す（眺める）のです。



■けれども、この程度であっても

　「読まないよりも読んだほうがいい」

　わけであって、思いのほか、頭のなかにも
　残ってくれるもの、

　というのが実感。
　


■毎日10分、20分程度の眺め読みの蓄積が、

　「印象に残る、いい言葉の断片」

　を脳内に残してくれたり、

　「不案内の分野に対する苦手意識の減少」

　であったりという褒賞をくれることとなり、


　知識の網の目を「疎から密」の方向に向けて、
　働きかけてくれるのです。



■そして、このような大量の眺め読みを行う中で、

　「これは！！」

　と思った本を、ひもとき、
　ここで初めて熟読、精読するのです。


　こうした流れを、精度を高めつつ、自身の
　情報処理システムの中に組み込むことで、

　ある段階から、知識のつながりが急激に増加することが
　体感されることとなります。



■ひとたび、このシステムが完成し、稼働し始めると、

　「一気に世界が広がっていく」

　感覚を覚えることでしょう。


　これを毎日、そして長い間、繰り返すことで、

　それまでの自分とまったく異なる世界に
　羽ばたけるようになるのです。

　
　とはいえ実のところ、（書籍の保管場所含め）
　結構コストもかかるので、　

　このシステムを整えるためにビジネスしている
　くらいな感じ（汗）。



■ちなみに、たとえば私が、

　今年、お正月が明けてから３週間ほどの間に買ったものを
　Amazonの購買履歴で検索してみると大体、こんな感じ。

　「あとから、いろいろとつながってくる」

　真剣に読まなくても、目だけは通しておく、
