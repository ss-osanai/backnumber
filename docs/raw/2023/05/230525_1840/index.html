　【失うものがなにもない
　（格好をつける必要が一切ない）】

　【中高６年間の男子校生活において

　【どんな真面目なお題を振られても
　　瞬時に下ネタに切り替えて返す】

　毎日、続けていました。



■期せずしてこれが

　【競争優位性のある資源（？）】

　につながっていったわけですね。


　まさに

　【Connecting the dots!!!】

　です。

　【失うものがなにもない
　（格好をつける必要が一切ない）】

　【下ネタの量産化が可能

　あるいは控えめに言っても
　強みにはならなさそうな資源を、

　【アテンションを集める】

　という、今の時代における

　【重火器レベルの武器】

　に転換できました。

　「パズルが解けた」

　ということでもあります。

　よくあるYouTuberみたいに、

　迷惑系になって炎上させたり、
　他者をぶっ叩かなくても、

　注目を集められること。

　きっとあるでしょうけれども、

　誹謗中傷にさらされたり、

　ということはありません。
