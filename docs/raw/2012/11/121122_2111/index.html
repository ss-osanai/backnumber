＜今日のピークパフォーマンス方程式＞

  【　生活を振り返り、文章を書き続ける習慣　】

───────────────────────────────────



■今日は

　「日々の生活を振り返り、文章を書き続けること」

　の効果、効能、効用について
　考えてみたいと思います。



■私はこうやって10年近くにわたって
　一日も休まずメルマガを書き、

　約20年にわたって、
　一日も休まず日記をしたためてきました。

　

■そんな生活を続けてきて気づいたことが
　あります。

　それは、毎日、一定量の文章を書き続けていると
　
　「日々、いかにたくさんの気づき、
　　学びを得ているか」

　あるいは

　「一日生きて、いかに少ない気付き、
　　学びしか得ていないか」

　に気づくようになる、ということ。

　ちょうど「定点観測」をしているようなもの。



■要は、

　毎日、感じたこと、考えたことを書き連ねようとして、
　日々、定点観測を繰り返していると、
　
　いずれにせよ、必ず何かに気づく、
　
　ということです（笑）



■極論すれば、

　一日の気付きの内容が乏しくても、
　あるいは「なく」ても構いません。

　なぜなら、一日生きて、記録できるほどの
　学びも気付きも得られなかった、

　という気付きを得られるから。



■実のところ、この事実に気づくというのは、
　大変なことで、

　（昔の私もそうであったように）

　少なからぬ人は１日、24時間を過ごして、
　特筆すべき体験、あるいは経験を持たなくても、

　そんな１日であったことを自覚するための
　システム自体を持っていないように思われます。

　（だから自分の生活密度における
　　変化の自覚が生まれない）



■「見える化」ではありませんが、

　現実を視ることなしに、
　現実を改善することなどできません。


　私（鮒谷）にとって、
　
　「日々の学習、経験の質・量を、
　　見える化するシステム」

　とでもいえるのが、

　毎日、メルマガを書いたり、日記を書いたりという
　習慣なのです。
　
　

■この繰り返しによって、
　一日を振り返り現状を把握し、
　
　次の一手をどのように打つかのアイディアが
　出てくるのです。



■ところで、ブログや日記を（強制ではないですが）
　書かれるといいですよ、とお伝えすると、よく

　「書こうとするけれどもネタが出てこない」

　「毎日書いているけれども、質・量が安定してこない」

　というお話（悩み？）を聞きます。



■けれども、私はこれは「良い兆候」であると
　捉えています。

　
　なぜなら、自分が日々考えていることを
　いったん外に出してみなければ、
　
　「自分の思考の質・量」
　　
　がどのレベルにあるのか、
　見える化されないわけですし、


　見える化されて

　「書こうとするけれどもネタが出てこない」

　「毎日書いているけれども、質・量が安定してこない」

　という状況にある、

　ということが分かること自体が
　一つの大きな成果だと考えているからです。


　
■もしこの事実が自覚されなければ、

　「自惚れるほどの実力もないのに自惚れたり、
　　満足すべきでない状況に現状に満足したり」
　
　という状況に陥ることすら起こりえます。

　これでは成長もできず、進化も遂げられません。



■だから私も、

　その日一日がどんな状態であってでも、
　とにかく毎日、

　「生活を振り返り、文章を書き続ける」

　ことにしています。




　今日も人生とビジネスを楽しみましょう！
