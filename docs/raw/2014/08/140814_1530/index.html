＜今日のピークパフォーマンス方程式＞

 【  「亀の甲より年の功」マーケティング  】

───────────────────────────────────



■長い間メールマガジンを書いてきましたが

　我ながら先見の明があった（！？）と思うのが、

　「陳腐化していくコンテンツは扱わない」

　と決めたこと。

　
　「旬のコンテンツ」
　
　あるいは、
　
　「流行り廃りのある分野」
　
　には手を出さず、
　他の人におまかせしてきた、
　
　ということです。

　
　
■これが、

　10年以上前にメルマガを創刊したときから、
　変わらぬ目指してきた方向性。


　それを

　「永遠の定番」

　「偉大なるマンネリ」

　「目新しいけど、
　　目新しくないコンテンツ」

　と言ってきたわけです。



■この、

　「目新しいけど、
　　目新しくないコンテンツ」
　
　を創造し続けるのは、
　
　簡単なようでいて難しい。

　
　しかし、だからこそ、
　チャレンジする甲斐があると思いました。



■なぜなら、
　
　「目新しさのみを競い、
　　次の瞬間には陳腐化していくコンテンツ」
　
　をビジネスの土俵に選んだ瞬間、

　「不毛な最先端争い」
　
　に巻き込まれ、

　「血で血を洗う、
　　血みどろの勝者なきマーケット」
　
　に参入する、
　
　ということを意味するから。



■そんなマーケットに入ろう、
　あるいは、入りたい、とは、
　
　ハナから、考えてはいませんでした。


　そのような世界で戦っても、
　やればやるほど疲弊するから、
　
　戦う地は、
　
　「生きていくほどに戦いやすくなる」
　
　という

　「『亀の甲より年の功』が生きてくるマーケット」

　を主戦場として選んだのです。



■もちろん、そのような市場に向けて

　「年の功」

　を持たない人間が参入し、

　「橋頭堡」

　を築こうとしても、
　初めは厳しい戦いを余儀なくされるでしょう。



■しかし一旦、橋頭堡を築き、

　そこから時間をかけて微差を積み重ね、
　絶対差を生み出すことを目指していけば、

　後々、楽になる、というのが、

　当時立てた遠大な戦略であり、
　目標でありましたが、

　振り返ってみると、その戦略がまさに

　「あたった！」

　そんな風に思われます。



■短期的に見れば、
　
　「目新しく、瞬間最大風速を捉えて一気に盛り上がり、
　　爆裂、躍進しているように見えるマーケット」
　
　を追いかけたほうが、
　儲かりそうにも見えるものですが、


　考えてみると、

　一瞬浮かれては、あっという間に沈み、その都度、
　退場していく人が量産される世界で戦うのは、
　
　本当に大変なこと。



■ならば、
　
　「経験の蓄積がものをいう世界」
　
　に、陣を敷いたほうがいい。

　
　そんな仮説を立ててメルマガを始めたわけですが、

　ここにきて急に大きな価値を生み出し始めた
　ことに気づき、ますます、

　「陳腐化せず、
　　10年、20年経っても有効なコンテンツ」

　にこだわり続けたいと思っています。



■そのためにも、
　
　「温故知新」
　
　「不易流行」
　
　といった言葉を反芻しながら、


　「故（ふる）き」
　
　あるいは
　
　「不易」
　
　の世界を深めるべく、

　「古典」ならびに「古典的なもの」
　を再評価する機運が、
　
　（私の中で）急激に高まっているのです。



■「人の行く 裏に道あり 花の山」
　
　といった格言は、

　なにも株式の世界においてのみ
　有効なのではありません。


　ある側面を捉えれば、
　人生（ビジネス）もまた相場。


　人が飛びつかない（飛びつきたがらない）ところに
　こそ、
　
　「豊穣なマーケット」
　
　があるものです。
