【合法的「産業スパイ」でビジネスを加速させる】

2018/11/08 11:00 
■こんにちは、鮒谷です。


　私は普段からいつも

　「ウルトラＣはない」

　というお話をしてきました。


　人生やビジネスにおいて

　「なにか一発、秘伝の方法を知るだけで、
　　一気に状況は好転する」

　といったことはないからです。



■けれども同時に、

　「知らなかったが故に、
　　イバラの道を歩いてしまう」

　ことを防ぐための、


　「それぞれの世界における、
　　一連のノウハウ、スキル、テクニック」

　を事前に知っておくことが大切であることも、
　また、強調しておく必要があるでしょう。



■いつもお伝えしている言い方でいうと、
　あくまで

　「あり方」

　が一番大切なわけですが、同時に

　「やり方」

　をおそろかにして、
　成果を出すことはできませんよ、

　という話。



■以前に、書いた話でいえば、

　【圧倒的なノウハウの集積】

　があるほうが、
　ビジネスを有利に進められるのは間違いなく、


　ここから

　【圧倒的な参入障壁と成果と自信】

　がもたらされることとなります。



■ですからビジネスにおいて、

　「ウルトラＣはなくても、
　　近道を通りたい」

　と思うのであれば、


　何をさておいても、

　【他人が圧倒的な物量を叩き込んで
　　手に入れたノウハウを、入手する】

　ことに力を尽くすべきでしょう。


　これによって目的地到着に必要な時間を
　一気に短縮できるのです。



■だからこそ、昔も今も、

　「産業スパイ」

　が跳梁跋扈するわけです。
　

　倫理的な面はともかくとして、

　いつの時代にも
　産業スパイが存在する理由は、

　【秘匿されたノウハウを盗めば、

　　そのノウハウを確立するためのに
　　かけたコストや時間を浮かせられるから】
　
　以外にありません。



■ところが、この

　「産業スパイ」

　を合法的に行うことができるとしたら
　どうでしょう。


　・・・・・？

　と思われる方があるかもしれませんが、

　セミナーに参加する、
　コンサルを受ける、
　本を読む、

　これらは

　【完全に合法的な産業スパイ】

　と言って良いでしょう。


　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　┃
■他の人が、

　数年、十数年、あるいはそれ以上、
　といった途方もない時間、

　さらには大きな金をかけて試行錯誤し、
　編み出したノウハウやスキルを

　「数時間、ないし、せいぜい数日程度」

　の投資で手に入れられる、
　ありえない投資なのですから、

　私から言わせれば、
　やらない理由が見つかりません。



■私（鮒谷）は、過去20年近くの間に、

　都内に戸建ての家が建つくらいの金額を
　自らの学習に投下してきましたが、

　それもひとえに、

　「合法的産業スパイ活動
　　（セミナーやコンサルへの投資）」

　のため。


　（それゆえ、
　　いまだに借家ぐらしなんですが 汗）

　でも、その活動によって、

　自分でも驚くほどの成果を
　上げ続けられるようになりました。



■こんな風にして集積してきた
　知識、スキル、ノウハウ、

　ならびに圧倒的な時間をかけて
　実践してきた実践の知恵、


　それらの中でも

　【営業ノウハウ】

　に特化し、集約し、凝縮したものを
　お伝えする機会が、

　このたび開催する、

　「圧巻セールスセミナー」

　ということになるわけです。



■特に、

　経営者、経営幹部、営業に携わられている方、
　ぜひ、合法的にノウハウを盗んでいって下さい。
