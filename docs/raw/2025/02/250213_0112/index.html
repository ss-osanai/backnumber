■言葉を変えるだけで未来が開かれる/
　規範とする言葉を磨く/促成栽培の量産型/
　魚は水に気づけない

───────────────────


■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------

　「何をすべきか」は後でええ。

　その前に
　「どんな自分でありたいか」を
　言葉で書き表そう。


　行動に移る前に、
　規範とする言葉を磨くのよ。

　それが、あなたのこれからの
　行動を決め、結果を生み、
　人生を形作っていくんだわ。


　表層ではない、
　深いところから変えること。

　人生の変革も急がば回れ、やで。

　----------

　「どうせ無理」
　「今さら遅い」
　「もう歳だ」

　無意識に回してる言葉が、
　限界を作るのよ。

　こんな言葉が、知らぬ間に
　自分の可能性を封殺する。


　それぞれ

　「やれるに決まってる」
　「ここからがオモロイ」
　「経験こそ最大の武器」

　と書き換えれば、どうなると思う？

　言葉を変えるだけで、
　未来が開かれるんよ。

　----------

　日記を書くとき、ただの記録で
　終わらせるのは勿体ない。

　心が震えた瞬間を、
　自分の言葉で深く掘り下げていく。


　なぜその言葉、その出来事に
　心奪われ、魂が震えたのか。

　「なぜ」を追いかけ続けることで、

　新たな自己との出会いや
　創造が生まれてくるよ。

　----------

　コミュニケーションにおいて
　意識すべきは

　「自分の利益」じゃなく
　「相手の期待」。

　「どう見られるか」ではなく
　「何を届けるか」。

　この転換ができたとき、
　意味のある対話が始まるのよ。


　人は誰かのために何かを語るとき、

　思いもよらない力が
　引き出されることがあるんだわ。

　----------

　人生で迷ったら、
　時間軸を伸ばしてみるといい。

　3年後やなく、
　10年後、20年後の自分を
　想像する。

　その視点に立つと、
　今、取るべき選択肢も変わってくる。

　短期の成果か、長期の可能性か。

　この選択が人生を左右する。


　近くの実りを摘むのか、
　遠くの果実を育てるか。

　ここで覚悟が問われるわけ。

　----------

　時間軸を短く取ると、
　地に足ついた成長は期待できん。

　半年一年での成果を求めたら

　「促成栽培の量産型」

　にしかなれんのよ。


　でも本当の才能は、
　5年10年という時間の中で、
　ゆっくりと育まれるんだわ。

　時間をかけて培われた地力こそが、

　長きに渡り、大輪の花を
　咲かせ続ける土台となる。

　----------

　魚は水に気づけない。

　同様に、人は自分の
　思考パターンに気づけない。

　自分にとって「当たり前すぎる」から。


　だから外の世界に出て、
　違う考え方に触れることが必要。

　そこで衝撃を受け、
　自分の殻が見えるのよ。


　殻の中の「当たり前」が、

　人生を小さな箱に
　閉じ込めていたことに気づくんだわ。

　----------

　「新たに始める」より
　先にするべき大きな決断がある。

　それは「捨てる」こと。


　何も捨てず、新しいものばかり
　増やしていくと、

　あっという間にパンパンになり、
　行き詰まる。


　痛みを伴う過去の廃棄からしか、
　次に進むための余剰は生まれないのよ。

　廃棄なくして進化なし。

　それが次のステージに進む道。

　----------

　ワイは20代から、

　周囲に何を言われても
　30年、50年先のことだけを
　考えて生きてきたよ。

　短期的に厳しい状況に
　追い込まれることもあったけど、

　結果的にそれが大正解やった。


　当時のワイを虐げたり、
　貶めたり、苦しめてきた奴らに、

　ざまあみろと言いたいわ、、

　という気持ちもなくなった。

　幸せあるのみや。

　----------

　斜に構えてる人を見かけるけど、
　あれホンマ損してると思うんよ。

　そう伝え、逆ギレされても困るから
　触らぬなんとかに祟りなし、

　で近づかんけどな。

　ワイ以外にもそういう人、
　少なくないと思うわ。


　それで孤立し、ますます斜に構えては
　人が離れ、の悪循環。

　大きなお世話かもしれんけど、
　気になるな。

　----------

　人はいくつになっても
　間違える生き物。

　決断に迷ったら、

　視点を宇宙まで引いて
　眺めるといい。

　たとえば地球が見える距離まで。


　すると不思議と
　目先のこだわりが消えていく。

　時間軸も広げて、
　100年後から今を眺める。

　そうやって時空を拡げる
　イメージを持つと、

　決断の質が変わるから
　試してみるといい。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
