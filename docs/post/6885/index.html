6885号 私たちは知識と想像力によって危険を回避し、チャンスを掴めるようになる

■いつも自分に言い聞かせている
　言葉に

　【当事者意識（自責の精神）、ならびに
　　知識と想像力が全ての始まりである】

　というものがあります。



■自分の人生は
　誰かに委ねるのではなく、

　自身が手綱を握らなければならぬ
　（誰も面倒を見てくれない）

　ことを前提として生きるという
　決意表明でもあります。



■誰かが面倒を見てくれるというのは
　甘えであり、

　成人するまで、もしくはせいぜい
　学校を卒業して社会時になるまで、

　くらいなら、ギリギリ、
　甘えも許されるかもしれませんが、


　それ以降は徹頭徹尾、

　自分の身の上に起こることは
　自分の責任と受け止め、解決すべき、

　という信条。



■その前提に立つからこそ、
　不透明、不確実な未来を

　（完全に見通すことなど
　　もちろんできないにせよ）

　可能な限り
　見通しを良くしたいと考えます。



■そこから、

　今後起こりうる、いくつものシナリオを
　想像しては備え、さらには活用する、

　といった意識が生まれてくるのでは
　ないでしょうか。



■こうした意識を現実の生活に
　反映するためには

　【多様な分野にわたる
　　膨大な知識】

　が必要になります。



■どこまでいっても我々は

　「これまで見聞きしてきた
　　極めて狭い世界」

　に囚われ、その世界内でしか
　生きられないわけですから、

　せめて

　「境界線の一ミリでも
　　『向こう側』を知る」

　ことに努めたいものです。



■それにより、

　人よりも早く危険を察知し、同時に
　人よりも早くチャンスを見つけられる、

　ようになるのでしょう。



■そもそも

　「自分の知っていること以外は
　　なにも知らない」

　わけですから、

　どこに危険やチャンスがあるかを事前に
　分かってから学ぶことはできません。


　いつもお伝えしている言葉でいうと

　「すべての学びの価値は
　　事後的にしか分からない」

　のです。



■だからこそ、

　ときに無駄打ちがある可能性を
　承知していても、それでもなお、

　知らない対象について
　好奇心や興味や関心を示し、

　あらゆる方面に対する知識を
　求める必要があるように思われます。



■こんな風にして
　たくさんの知識を獲得するにつれ

　「想像するなと言われても
　　想像するようになる」

　でしょう。



■たとえば、

　南海トラフ巨大地震やら
　首都圏直下地震、
　富士山の噴火、

　についての知識を得るほどに
　想像しますよね。


　というか、
　想像せずにいられなくなります。

　想像するから備えるようにも
　なるでしょう。



■煽り運転する輩がどれほどいて、
　彼らがどんなことをしてくるのか、

　たくさんアップされている
　YouTube動画を見て知識を得れば、

　「想像するなと言われても
　　想像するようになる」

　はずです。



■想像するから、

　もし難儀なDQNに絡まれたときの
　事前シミュレーションもできるし、

　いざ、ことが起きたときも
　より正しい選択を取れる可能性が高まる、

　というもの。



■また、ガブブレイスの『大暴落1929』
　なんかを読めば、

　恐慌は宴の後に起こるもの、
　と理解できるようになるでしょう。



■理解すれば

　「想像するなと言われても
　　想像するようになる」

　に決まっています。


　株価急上昇期において、

　気が大きくなって
　リスク許容度を超えた投資を行う、

　といったことも
　なくなることでしょう。



■あるいは、

　知識があったので
　チャンスを手に入れられた、

　という話も
　至るところにありますね。



■たとえば、

　初期の頃の民泊（エアビー）や
　仮想通貨で大儲けした人は、

　少なくありません。


　この流れを初期に捉えられるかどうか、
　も知識に依ります。

　知らなければ
　投資のしようがありません。



■私（鮒谷）は、

　民泊においては出遅れてしまい、
　トントンくらいの着地でしたが、

　ビットコインを2017年くらい、
　未だ20万円いかないときに、

　「これからはビットコイン、
　　くるかもよ」

　という「知識」を友人から
　教わったことにより、


　もともとなかったお金だと思って
　張れば

　「大儲けできるかも！？」

　という想像力が生まれて、結果、
　それなりの利益を得られました。



■知識があることによって
　想像力が生まれ、

　それによって危機を回避
　（や直撃ダメージを一定程度、緩和）

　できることもあれば、

　有形無形の利益につなげられる
　ようにもなるということですね。



■知らなければ

　「つねに後手、後手」

　「ことが起きてから対応 （という名の、
　　行き当たりばったり」

　になるに決まっています。



■どちらがいいかと問われれば
　少なくとも私（鮒谷）は

　「知識と想像力を活用して
　　世の中を渡っていきたい」

　と思っているからこそ、


　【当事者意識（自責の精神）、ならびに
　　知識と想像力が全ての始まりである】

　と言い聞かせては、

　自分の人生は自分で決めるつもりで
　学び、知識を入れ、想像力を発揮し、

　これから行うべき行動を決断しては
　実行に移してきました。



■少なくとも今のところは

　「こうしたやり方を実践してきて
　　良かったし、

　　そうでなければ今、こんな生活は
　　させてもらっていなかったはず」

　と思っています。



■そんなわけで、

　あらゆる方面に興味関心を示したいと
　思ってきましたが、

　特に重点的に意識を向けてきたのは

　【将来の経済基盤を固めるための
　　知識と想像力の拡張】

　でありました。



■きれいごとを抜きに言えば

　「経済力は生きていく上での
　　最重要の土台」

　といって良いでしょう。



■そんな土台を、
　出たとこ勝負で生きた挙げ句の

　「成り行き任せ、運任せ」

　など、それこそ想像するだに
　恐ろしいこと。


　実際、

　「当事者意識（自責の精神）」

　を持つ人であれば、

　決してそんな生き方には
　ならないと思うんですよね。



■自身の経済基盤は、

　国や地方自治体や雇用主や
　家族や年金なんかあてにせず、

　「当事者意識（自責の精神）」

　を持ち、粘り強く生きてこそ、
　初めて強固にしていけるもの。



■このメルマガをお読み下さっている
　あなたも、

　きっと同じような精神、ならびに
　行動の姿勢を確立されているはずです。



■そんなあなたに

　「さらなる知識と想像力の拡張」

　をもたらし、

　未来の危険を回避し、
　チャンスを掴むための、

　「然るべき先生から学ぶ
　　貴重な学習の機会」

　として以下の勉強会を
　ご活用ください。



■正直に告白しますが、

　本勉強会は
　なにより私（鮒谷）自身が

　「さらなる知識と想像力の拡張を図り、
　　将来の危険回避と機会を捉える」
