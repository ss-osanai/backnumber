【２時間半の「（過去、最凶レベルに口の悪い）鮒谷放談」を無料でプレゼント】

2018/07/28 05:00 
■こんにちは、鮒谷です。


　現在、当社が提供している音源に

　【あなたが変わりたくても変われなかった
　　「３つ（＋７つ）の理由」セミナー】

　というセミナーを開催した際の音源が
　あります。



■当音源の内容は、以前より継続的に

　【結果を出せる体質になる】

　ことを目的として開催してきた

　【３日間セミナー（のち、２日間セミナー）の、
　　エッセンス凝縮＆行動変容促進版】

　となります。


　手抜きなし、手加減なし、妥協なし、
　
　最新の知見や考察を交えつつ、
　休憩なしで３時間半にわたって、

　全力でお伝えしたセミナーの音源です。



■当音源をお聴き頂ければ

　「変わりたい、変わろう、
　　変わらなきゃ、、、と、

　　何度も、何度も、何度も、何度も、
　　思いながら、

　　ここまで一向に変われなかった理由」

　がハッキリと分かるようになるでしょう。



■もちろん、

　「変わり方を（理論＆身体で）理解されている方」

　は、もう既に、変化・変容を遂げて
　いらっしゃるのみならず、


　（優れた方法論には再現性があるので）

　「今なお、変わり続けられている」

　はずで、

　そのような方は私（鮒谷）が
　とやかく言う必要もありません。
　


■ただ、もし、あなたが


　変革を遂げるための理論を持たず、
　したがって、自らを作り変えた経験もなく、

　変わりたいと願い続けてきたにも関わらず、
　なぜ変われなかったのかも分からず、


　でも、その原因を知り、
　ご自身の人生を作り変えたいのであれば、

　今回の話をお聞きいただければ

　「ああ、だから自分はこれまで変わることが
　　できなかったのだ」

　と明確にご理解頂けることでしょう。



■私（鮒谷）が過去20年以上の間、
　真剣に実践と検証を繰り返し、

　「変化・変容について積み重ねてきた知見」

　を３時間半（＋わずかな金額）で
　手に入れて頂けるとするならば、

　こんなお買い得な買物はちょっとないと
　（自社商品ながら）考えています。

■３日間セミナーを開催していたときには

　「30万円（＋税）」

　という金額を頂戴して開催しておりましたが、


　３日間セミナー開催当初から、

　遠方に住んでいる、仕事や家庭の事情等で
　家を長時間あけられない、

　あるいは、金銭的な事情から参加出来ない、
　といった声をさんざん聞いてきたこともあり、


　その全て、あるいはせめて重要な部分は音源化し、
　より安価に、お手軽に、

　お聴き頂ける形でご提供することは
　かねてよりの目標でありました。


　ようやく、上記の音源シリーズで
　実現した、ということですね。



■ただ、各音源の内容については
　それぞれ理解、納得頂けたとしても、

　それらがお互いにどのようにつながり、
　有機的に関連しあっているのか、

　より深く、理解頂くことによって、

　さらに確度高く、変化・変容・変革を
　遂げて頂きたいと願って、

　今回の音源を提供することとなった次第です。

　【フォローアップ講座・決定版】

　【上記三部作が完結する】

■最後に一つだけ補足させて下さい。

　今回の音源は、通常、当社がご提供している
　セミナーよりも安価な設定ですが、

　セミナーの構成において、
　当社が一年間を通して提供している

　【年間プログラム】

　の内容についてのご紹介が全体の３％程度、
　入りますので、その点のみ、お含みおきください。



■当然、

　「買ってくれとは絶対、言わない」

　ことを標榜している私ですので、
　営業的な話は一切、行っておりません。

　あくまで同プログラムの内容のご案内を
　ほんの一部、ご紹介しているのみとなります。



■ただ、普段から

　「ときおり、メルマガで出て来る
　　年間プログラムって何？？」

　と質問を下さる方も存外、多いので

　（というか、これまで横着して、メルマガ等でも
　　まともに内容について説明してこなかったので）

　今回は、このプログラムでは、
　どんな意図で何を行っているのか、

　具体的にお伝えしています。



■とはいえ、それはどこまでいっても
　「従」であり、

　お伝えした内容の主たる目的は
　今回のセミナーのテーマであるところの

　【あなたが変わりたくても変われなかった
　　「３つ（＋７つ）の理由】

　をお伝えすることなので念のため。

　行なってきた私（鮒谷）ですが、

　それらを通して、お客さまへの価値提供を
　コミットし、実践してきた中で

　「どんな挫折と蹉跌と後悔と懺悔があったのか」

　についての語りを通して、

　「なぜ私が年間プログラムを提供しようと
　　思うに至ったのか」

　その理由や背景にも迫ります。



■正直、その話自体が、

　「変化、変容についての深い理解と、
　　大きな気付き、学び」

　につながるはずだと確信しています。


　あなたさまからのお申込みを
　心よりお待ちしています。

■追伸その２：

　鮒谷が実際に話をしている音源（２時間半）、

■毎月そうなのですが、

　メルマガで書いているよりも、
　はるかにディープで、

　「数万円に相当する価格の音源」
