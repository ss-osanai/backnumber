■こんばんは、鮒谷です。

　いつもメルマガをお読み下さり、ありがとうございます！


　今、大阪出張で某ホテルに滞在しているのですが、
　突如、思い立って、

　お正月特別プライスでの、

　「２時間一本勝負の、個別コンサル」

　を３枠限定にて解放することといたしました。



■時間は、


　▼１月４日（水曜日）の正午（12時）から、午後10時までの、
　　いずれかの時間（２枠）

　もしくは

　▼１月５日（木曜日）の午前９時以降、13時までの、
　　いずれかの時間にて（１枠）


　という形です。



■コンサルフィーは、通常、

　「１時間12万円（＋税）」

　をご請求申し上げておりますが、

　今回は急な思いつきの企画ですので、
　
　「２時間12万円（＋税）」
　※特別価格（実質50％オフ）


　そしてさらに、11月以降、
　いずれかの音源をお求め下さった方限定で、
　
　「２時間98,000円（＋税）」　
　※音源ご購入者様限定・特別価格（実質60％オフ）

　としてご提供させて頂くことといたしました。


　※今から初めて音源をお求め頂き、
　　その勢いでお申し込み、

　　ということもありといたします（笑）
　


■今日のメルマガではありませんが、

　特に、新しいご縁を生み、育てる機会を増やすという
　目的も含めて行う、

　「人間関係の構築を目的に含めた、
　　長期的視点に立った、特別施策」

　です。



■また、直前、かつ、限られたスケジュールですので、
　
　「先着順」

　にて枠を確保させて頂きます。

　（とはいえ調整が容易になるよう、できるだけ
　　長い時間帯で、候補日時を頂戴できましたら幸いです）
