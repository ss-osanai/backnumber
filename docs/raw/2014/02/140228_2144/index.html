＜今日のピークパフォーマンス方程式＞

 【　「そもそも」は前提を覆す魔法の言葉　】

───────────────────────────────────



■「そもそも」

　という言葉は、難題に出会ったときに投げかけると、
　
　様々な問題の本質を切り出すための

　「魔法の言葉」

　となり得ます。
　
　
　
■例えば、

　「商売が儲からない」
　
　という難題に悩んだときには、
　
　
　「そもそも」
　
　お客さんにどれほどの価値を提供しているのか、　
　を問い直せばよいのです。
　
　
　
■また、

　「自分の周りに人が集まってこない」
　
　という難題に悩んでいるのであれば、
　
　
　「そもそも」
　
　人が集まるような魅力的な人間になる
　努力をしているか、
　
　
　について自らを顧みれば良い。
　
　
　
■あるいは、

　「クレームが多い」
　
　という難題が生まれた時には、
　
　
　「うっとうしいなあ」
　
　とか、
　
　「面倒くさいなあ」
　
　とか、
　
　「このクレーマーめ（怒）」
　
　と思う前に、
　
　
　「そもそも」
　
　自らの姿勢、あるいは
　自社の商品・サービスの質に問題があるのでは、
　
　
　と問いかけてみるのもいいでしょう。
　
　
　
■このように、行き詰まったときには、

　「そもそも」
　
　と自分に投げかけてみたら良いでしょう。
　
　
　そうすると、
　
　思いもよらぬ角度からの視点がもたらされ、
　
　思いもよらぬ解が導き出されることが
　数多くあります。
　
　
　
■私自身も、

　壁にぶち当たったり、悩みが出てきたとき、
　近視眼的思考に陥っていると思ったときには、
　
　問いを立て直すために、意図的に、
　意識してよく使っている言葉です。
　
　
　お気づきの方は既にお気づきかもしれませんが、
　このメルマガでも
　
　「『そもそも』が頻出ワードとなっている」
　
　のはそんな理由から。
　
　
　
■問題が起こったときには、

　「そもそも」
　
　何が問題の核心であるのか、
　
　改めて問い直されることをおすすめ致します。
　
　
　より本質的な問いを立てたときにのみ、
　より本質的な答えが導出されるでしょう。


　※ちなみに私は、毎日、
　
　　「そもそも、何を目的として、
　　　このメルマガを書いているのか」

　　について考え続けています。
