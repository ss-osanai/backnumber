6824号 ガチの資産運用勉強会に興味ある人はありますか？

■本日、決まった話ですが、

　【６月18日（土）の13時半～17時】

　の日程にて、

　講師の先生をお招きして
　資産運用の勉強会をいたします。



■もちろん

　「金融商品が紐付けられた勉強会
（＝金融商品の販売ありきの勉強会）」
　
　などではなく、

　そうしたものを一切排除しての、
　純粋な

　「原理・原則、王道系の
　　資産運用を学ぶための勉強会」

　となります。



■講師の方のプロフィールは
　改めてご紹介いたしますが、
　
　
　いかがわしい、インチキ臭い、
　（自称）投資のプロフェッショナルや、

　金融商品の販売ありきの
　セールスマン、

　ではなく、


　某メガバンクにて長年、
　エコノミストとして活躍され、
　
　（日経新聞等への寄稿やインタビュー、
　　講演等も多数）

　各国での海外駐在も長く、
　為替ディーラー等も経験してこられた

　「ガチのプロフェッショナル」

　です。



■なにより私（鮒谷）が、

　自身の資産運用
　（資産防衛ならびに運用）のために、

　一から勉強させて頂きたい、
　というところから

　「然るべき方」

　を探し求め、この方の存在を知り、
　コンサルを受け、


　（その方のアドバイスに従い、
　　資産運用方針を策定、実践しながら）

　ご縁を深めつつ、勉強会を企画した、
　そんな経緯がありますので、

　自信を持ってお勧めいたします。

　

■最初は、

　既に相応の資産（億単位）を
　形成されている方対象の勉強会、
　
　と思っていたのですが、


　本日いろいろご相談し、
　上述の通り、

　まっとうな考えをもたれる方が
　まっとうに資産形成を行うための

　「原理・原則、王道系の資産運用を
　　学ぶ勉強会」

　というコンセプトで
　開催することになりました。



■ですので、

　「１年で２倍、３倍、５倍、10倍になる
　　銘柄を見つけたい」

　とか、

　「数年で資産○億円を稼ぎたい」
　
　といった、


　（たしかにそういう人は例外的に
　　存在しないわけではないとはいえ）

　原理原則とはかけ離れ、
　王道でも保守本流でもない、

　荒唐無稽な資産運用の話を
　求めている方は、

　お呼びではございませんので
　他を当たって頂ければと思います。

　※そういう本やセミナーや講演会は
　　山ほどありますし。



■しかし今回の企画は

　「資産運用の基本的な考え方や姿勢を
　　身につけたい」

　とお考えの方にとって必ず有益な、
　そして一生ものとなる、
　
　そんな勉強会になろうと思います。



■ちなみに、そもそも資産運用について、

　私（鮒谷）自身が今更ながら、
　真剣に取り組む必要を感じた理由は

　「ここ数ヶ月ほどで、かつてないほど
　　不確実性が高まっている」

　という緊張感とか恐怖感が募り、
　じっとしていられなくなったから。



■そんな背景があって、

　ここ１、２ヶ月ほどの間、
　それこそ昼夜分かたず

　「資産運用」

　について目が潰れそうになるくらい、
　集中して学び、情報収集してきて、


　これはもう、やっぱり
　自分自身の目で確かめて

　「真摯、かつ、本物」

　だと思われる方から直接学ぶことが
　喫緊の課題、

　と思われて企画を立てた次第。



■もちろん、もう既に

　「投資については全てマスターして、
　　今更学ぶ必要はない」

　という方は、そのままご自身で
　取り組まれて頂ければと思いますが、


　そんな機会があるのなら
　ぜひ一緒に学びたい、
　
　とお考えの方があれば、今すぐ
　手帳（orスマホ）を取りだして

　【６月18日（土）の13時半～17時】

　のスケジュールをブロックしておいて
　くださいませ。

　※詳細は６月に入ってから
　　改めてご案内いたします。



■私（鮒谷）をして、

　ここ数ヶ月の、こうした取り組みに
　走らせた問題意識は、

　先日来、ご案内してきた

　【人生クマムシ化音源
　（＝破滅回避音源）】
　
　にて、詳細にお伝えしています。
