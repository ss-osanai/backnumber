6874号 過去の経験や学びが有機的に結びついている感覚のないあなたへ

■世の中にはいろんな生き方を
　している人がありますが、

　そんな中でも

　「確立された生き方を貫いている」
　「無駄のない一貫した生き様である」

　と思う人、ありませんか？



■そういう人に接したとき
　私（鮒谷）は、その生き方を

　「一つのシステム」

　として捉えて
　観察？してしまいます。



■大切にしている信条、価値観に従う
　生き方を実現させるためには、

　既に手元にある、あるいはこれから
　身につけるスキルや能力や知識等を

　「一個のシステムとして機能させる」

　必要があります。



■私（鮒谷）は、

　時間管理もコミュニケーションも
　営業もマーケティングも
　情報発信もモチベーション管理も、

　その他、これまで身につけてきた
　あらゆる技術やテクニック群を

　「自身に最適化された形で
　　統合しようとしてきた」

　つもりですが、


　これなどは自分自身の生き様を

　「統合された
　　一個のシステムとして機能させる」

　ために実践してきたこと。



■逆に言えば、

　「幸せになるために大切にしたい信条、
　　価値観 （＝その集積が生き様）」

　がハッキリしているからこそ
　何を学ぶべきかも明確になるし、

　学んだことの歩留まりも
　高まります。



■ここでいう「学んだこと」とは、

　ただ頭の中に詰め込んだ
　知識のことではなく、

　身体に定着する形で
　行動に伴う能力や知識のこと。



■それらが定着し、統合されるのは

　「そもそもどんな人生、生き様を
　　志すのか」

　があってのこと。


　こうした目標なく、

　その場しのぎで一生懸命に生きても
　労多くして功少なし、

　となるように思われます。



■だからこそ

　「そもそも
　（私はどこに生きたいのか）」

　の言葉を頻繁に回しながら、

　目指す世界を忘れず、
　さらにはブラッシュアップを重ね、


　そこに近づくための

　「各種資源が統合されたシステム」

　を構築されると
　良いのでないでしょうか。



■少なくとも私（鮒谷）は、

　（こうしたことを最初は無自覚に、
　　ある時期からは自覚的に）

　繰り返すことによって、

　目指す世界に自動的に
　近づいていくシステムに身を委ね、

　現実にその場所に
　着実に近づけている、

　そんな実感を持っております。



■以上のような話を書こうと思ったのは、

　菅智教さんからこのような
　ご感想（↓）を頂戴したからです。


　（ここから）
　--------------------

　ジョギングしながら、まず聴きました。

　使っていなかったと思いますが、

　これからは
　所詮という言葉を使う度に、

　一気に上空へ飛んでいき、
　今の自分を俯瞰してみられる、

　最強ワードを自分の中で
　変換できたと思います。
