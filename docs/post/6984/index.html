■結局のところ、
　人生に対する満足度、幸福度は

　【確率 × 試行回数】

　で決まってくるものです。



■もう少し詳しく説明すると

　【ある行動によって
　　人生の満足度を高めてくれる結果が
　　得られる確率】

　と

　【その行為を行った累計回数】

　が最終的な幸福感を決する、
　ということ。



■ちょっと乱暴な意見に聞こえるかも
　しれませんが、

　本質的な部分は外していないのでは
　ないかとも思います。



■たとえば、

　「他者にプラスアルファの価値を
　　提供しようと心がけ、向き合う」

　ことを行動指針とし、


　その思いを込めた行為を受け取った人の、

　仮に３割の人が感謝してくれ、
　６割の人は当たり前と受け流し、

　１割の人が誤解して怒り始めると
　するとどうなるか。



■たしかに

　「良かれと思ってやったことが
　　負の結果（怒声や罵声）を生むこともある」

　可能性もあるわけですが、


　総体としては回数を増やすごとに
　人間関係が豊かになり、

　それにともない、いろんなものに
　恵まれるようになるはずです。



■ただし、最初の一人目が、

　せっかく好意でして差し上げたことを
　誤解して怒り始め、それを受けて

　「もういいわ、こんなことになるなら
　　やってられるか！」

　となったら、

　回数を増やすことによって、
　その後、得られたはずの価値を
　享受できなくなります。



■そうした一つ一つの行為に対する
　結果について過度に振り回されず

　上げ下げを繰り返しながらも、

　長い目で見れば、
　良い方向に連れて行ってもらえるはず。



■だからまず

　【学ぶ必要がある】

　のです。


　学ぶことによって短時間で

　【人生の満足度を高めると思われる
　　確率の高い行為】

　について理解し、
　実践できるようになるからです。



■もちろん、そうした学びを一人で
　手に入れることも不可能ではありません。

　しかし経験からそれを得るには
　膨大な時間がかかります。

　その時間をショートカットしたければ
　先人の知恵を学ぶしかありません。



■こうして、
　行うべき行為を定めた上で

　【目先の結果を気にせず、
　　大量行動し続ける】

　必要があるわけですが、
　このときに必要になるのが

　【元気】

　です。



■ある行為を行い続けることが
　トータルで見たときにプラスになる、

　と理解していても、

　「他者の視線やフィードバックが
　　気になって身体が動かない」

　人は少なくありませんが、


　なぜそんなことになるかというと
　端的に言って

　【元気がない】

　からです。



■「元気」には特徴的な要素があり、

　既に「元気」がある人は、
　その元気をエネルギーにして、

　さらなる元気を
　内製（再生産）できるけれども、


　元気が枯渇した、
　あるいは枯渇寸前の人は、

　内製に要するエネルギー自体が
　失われているから元気になれない、

　ということになりがちです。
