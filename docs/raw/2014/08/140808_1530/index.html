＜今日のピークパフォーマンス方程式＞

 【  寝不足＝不機嫌＝低生産性  】

───────────────────────────────────



■「寝ないことを誇りとする」
　
　のではなく、
　
　「寝ていることを誇りとする」
　
　ことが、
　
　成果につながる考え方ではないかと思っています。



■往々にして、

　「寝ていないことを誇りにする人」
　
　が多いようにも見受けられますが、


　ある認識を持って生きている人が
　たくさんいるからといって、

　その認識が成果を生み出すとは限りません。



■むしろ、その常識により、

　成果を出す方向に対して
　反対側への力が働き、

　「普通の人を、
　　普通の人たらしめている元凶」

　になっていることも多いもの。


　
■睡眠に対する思いも、それに近いものが
　あるかもしれません。


　「寝ないことを誇りとする人」

　が99％であるとすれば、

　その99％の人が考えていることの　
　当然の帰結として、


　寝不足が生じ、不機嫌になり、

　思考が混乱、混沌とし、

　正常な意思決定もできず、
　
　
　成果を生み出しにくい体質になっていくのかも
　しれません。



■反対に、
　
　「寝ていることを誇りとする」

　そんな1％に入ることによって、


　常に感情が安定し、

　思考も明晰な状態を維持し、

　適切、的確な成果を生み出す意思決定が
　できるようになり、

　
　「普通の人」

　から抜け出すことができるように
　なるのかもしれません。


　そんな風に考えているから、

　私（鮒谷）自身、何があっても
　睡眠をとることを最優先しています。



■そんな私ではありますが、

　先日、どうしても取り組まねばならないことが
　重なって、珍しく、寝不足状態に陥りました。

　（といっても4時間睡眠になってしまったことが
　　一日あっただけですが）



■久しぶりにそんな状態で翌日を迎えたのですが、
　当然のことながら、その日一日は普段の

　「明朗・溌溂・無邪気」

　からは程遠い状態（？）となり、
　思うような成果を挙げられず、
　
　反省することしきりでありました。

　
　生産性のこともそうですが、
　
　なにより「気分」が、いつもの絶好調な状態とは
　ほど遠いというのは良くありませんね。



■つまるところ、

　寝不足は人生の効率を高めるように
　見えて、その実、

　「人生を浪費させるもの」

　であるといえるでしょう。



■だからこそ、

　一日の時間の中でも
　寝る時間（睡眠時間）は最優先して天引きし、

　必ず自分に見合った最適な時間を確保する
　必要があるのです。


　今までも心掛けてきたことではありますが、

　先の反省を生かし、これまで以上に
　徹底していきたいと思います。
