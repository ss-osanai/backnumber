6743号 あなたはどれ？３種類に分かれる世の中の人

■世の中の人を大別すると

　-----------------------

　１、下りエスカレーターを頑張って
　　　昇っている人

　２、上りエスカレーターに乗って
　　　努力なく昇っている人

　３、上りエスカレーターに乗りながら
　　　さらに頑張って昇っている人

　-----------------------

　の三通りの人があります。



■「１」の人は、

　いつまで経っても楽にならないことを
　不思議に思いつつも、

　真面目に、一生懸命に、
　全力を尽くし、行動している人。

　ただし残念ながら、その人の努力が
　報われることはありません。



■「２」の人は、

　期待する成果がもたらされる
　可能性の高い行動を、

　無意識に、ほとんど自動的に
　できるようになった人。

　ここまでいくと、
　人生は相当、楽になりますね。



■「３」の人は、

　「２」の状態を作り、さらに
　プラスアルファの成果を出すために、

　力を尽くしている人。　

　この人の未来は、ますます明るくなって
　いくことが約束されています。



■上に挙げた

　「３の世界」

　は

　「習慣化を習慣化できる人」

　だけが到達できる世界、
　と言えるでしょう。



■世の中には気楽に生きていながら
　大きな成果をあげる人と、

　一生懸命、頑張っているのに
　一向に報われない人がありますが、
　
　これが理由です。



■こうした理不尽ともいえるほどの
　大きな違いが生じるのは、

　一般には見えにくい世の理（ことわり）を
　可視化し、意識に乗せ、

　そのメカニズムを
　味方につけようとしている人と、


　メカニズムの存在を知らず、

　それゆえ、当然のことながら
　味方につけようもなく、

　時の経過に流されてしまう人の差、

　と言えるのかもしれません。



■こうした一見、理不尽に見える、

　しかし、構造を俯瞰して眺めれば
　生まれるべくして生まれてくる差の正体、

　ここまで読まれたあなたであれば
　既にお分かりのことでしょう。



■不可視＝知りようもない、であれば
　努力のしようもないことを、

　可視化し、理解し、その力学に沿おうと
　心掛けている人とそうでない人の差は、

　時間の経過とともに

　「月とすっぽんくらいの差」

　へと大きく広がっていくのです。



■いうまでもありませんが、

　それがいいとか、悪いとか、
　問題だとか、問題ではない、

　といった話ではなく、

　「こんな状況だと、
　　必然的にこうなるよね」

　といった、
　ほとんど物理法則のようなもの、

　と私（鮒谷）は捉えています。



■であれば、

　どうせなら物理法則に逆らうより
　従う方が楽ちんでいいな、

　と私（鮒谷）などは思うので、

　思い切り、その力学を味方につけて、
　楽をしながら楽しく過ごさせてもらいたい、

　と思っています。



■ここまでに記してきた力は
　上述の通り

　「習慣を習慣化する習慣」

　によって手に入ります。
