7263号 言葉ひとつ足りないくらいで、こわれてしまうものがある

■B'zの『HOME』という曲に

　「言葉ひとつ足りないくらいで
　　全部こわれてしまうような」

　という歌詞があります。



■先日、

　この曲を聞きながら
　筋トレしていたのですが、

　ふと、これは
　人生にも当てはまることなのでは、

　と思ったんですよね。



■どういうことかというと、

　（ある）言葉を
　一つを知らなかっただけで、

　人生を台無しにしてしまう、

　そんな地雷のような危険が
　至るところに埋まっているのが、

　私たちの生きているところである、
　という意味において。



■卑近な例でいえば

　「リボ払いは絶対にやめておけ」

　という言葉ひとつ足りないくらいで
　（人生）全部こわれてしまうような、

　ことが起こりえます。



■あるいは

　「月利10％で回る運用なんて
　　あるはずがない」

　という言葉一つ足りないくらいで
　（人生）全部こわれてしまうような、

　ことが起こりえます。



■ほかにも

　「電信柱が高いのも郵便ポストが
　　赤いのも社長の責任だ」

　という、

　（故・一倉定先生の）
　言葉一つ足りないくらいで

　（会社）全部こわれてしまうような、
　ことが起こりえますし、


　「未来にプレゼントする」

　という言葉一つ足りないくらいで
　（人生）全部こわれてしまうような、

　ことも起こりえます。



■言葉を知っているからといって
　必ずうまくいくとは限らなくても、

　言葉を知らなければ、
　かなり高い蓋然性をもって、

　未来に不幸な事象が起きてしまう、
　ということが現実にあるのです。



■私たちは継続して学び続けるべき
　意味の一つには、

　こうしたこともあるのではないかと
　思っているんですよね。



■機嫌よく、楽しく、
　私たちが生きていくためには

　-----------

　１、危機を未然に回避する
　２、機会を掴んで飛躍する

　-----------

　の両方を一定程度、
　実現させる必要がありますが、


　順序としては

　【１、危機を未然に回避する】

　を優先し次に

　【２、機会を掴んで飛躍する】

　ことに取り組む、でしょう。



■なぜなら

　「一度、人生が詰んでしまっては
　　再起は困難（ときに不可能）」

　となるからです。


　人生を詰ませることなく、

　ときには厳しい状況にあっても辛抱し、
　粘り腰で、試行錯誤を重ねながら

　再起の機会を伺い続けることが
　大切です。



■そのために有効な言葉として

　「韓信の股くぐり」
　「臥薪嘗胆」
　「会稽の恥を雪ぐ」

　などを脳内事象に格納し、

　いつでも取り出せるようにして
　おくとよいでしょう。



■と同時に、

　追い込まれたときには
　これらの言葉は有効ですが、

　できることならば、これらの言葉を
　回さなくても済むような状況、


　つまり損害の極小化を図りつつ、
　小さく勝ちを刻む、大きな勝負はその後で、

　という態勢を目指すべきでしょう。



■そこでは

　【危機を未然に回避する言葉】

　を事前に用意しておくべきであり、


　「言葉ひとつ足りないくらいで
　　全部こわれてしまう」

　ようなリスクを極小化しておく
　ことが大事です。

■それぞれのご感想を一部抜粋しておきます。

　以下は

　【極小リスク、ミドルリターン起業放談会】

　についてのご感想。


　（ここから）
　-----------

　音声で起業の基礎を反復学習することは、
　知っていれば避けられる起業時の失敗項目を
　つぶすのに最適の時間で、

　いつもながら、最高の時間とお金の
　使い方であったと実感しております。 

　-----------

　ある人からは起業を考える際は
　最悪のシナリオを想像しろと言われています。

　この音源を聞き、その意味が
　魂レベルでやっとわかった気がします。 

　-----------

　久しく慣れで忘れていた、
　「生き延びる」この言葉をまた念頭に置いて、

　忘れていたあんなこともこんなことも、
　実現に向けて行動します。 

　-----------

　長い経験を積まれ、かつ実績を出され、
　今でも走り続けている鮒谷さんの
　お話を聴くことは、

　リスク軽減となり、
　金銭的損失回避にもつながります。 

　-----------

　音源を聞いていないまま起業していたら
　泥沼、落とし穴に、間違いなく
　はまっていたに違いありません。 

　-----------

　飛び出す前に

　「死なないビジネスモデル
　　=クマムシ経営」

　を熟考していきます。 

　-----------

　僕自身、まだ起業予備軍に
　属しておりますので、

　放談会の最初のお話においてこれから
　起業される方への転ばぬ先の杖としての、

　貴重な秘訣、奥義を教えていただき、
　誠にありがとうございました。 

　-----------

　固定費極小化は
　誰しも知っていることではあるけれど、
　それをどれくらいのレベルでやるのか？

　具体的であるからこそ、その姿勢を知れ、
　汎用性の高い思考が生み出されるのだと
　感じました。

　-----------

　商売の種明かしをして下さった
　鮒谷さまには、とても感謝しています。

　（心が狭い人なら、一生誰にも話さず、
　　秘密にしていると思います） 

　-----------

　どの音源からも、とてつもない量の学び、
　気づきが得られております。

　私は本腰を入れて学びを始めたのが遅く、
　知識量も乏しい身です。

　そのなかで、「音源」という形で
　鮒谷さまの知見をインストールすることが
　できるのは非常にありがたく思います。 

　-----------

　言われてみれば、至極その通りなこと
　なのではありますが、

　臨場感をもって、
　ライブで鮒谷さんからお伝えいただくと、

　現実味を帯びて、とてもとても
　肚落ちしてくるのが分かりました。 

　-----------

　私は創業して21年経過していますが、
　今回のセミナーをお聞きして、

　改めて出来ていた事出来ていないことの
　振り返りの良い機会になりました。

　-----------

　休憩中も含めて合間合間に話された
　鮒谷さんの実体験が生々しくて勉強になりました。

　-----------

　起業時に限らず、仕事や人生において
　あらゆる場面に応用活用出来ると思います。

　クマムシの様に、死なない方法で
　チャレンジする事が出来るので、

　あらゆる場面でチャンスが待っています。
　最強の音源です。 

　-----------

　鮒谷さんのこれまでの経営感覚や学び、
　人との付き合いのなかで培ってきた、

　ガチな感覚をもって的確な文字・言葉で
　もってしてそれを聴くことで、

　頭で考えて学ぶのとは異なる、
　まさに体得するという感覚を持って、

　起業について学ぶことができました。

■以下は

　【人生クマムシ化音源】

　についてのご感想。


　-----------

　破滅前に幸運にも
　本音源を聞くことが出来ました。

　この後の人生は、あらゆる負荷をかけて
　想定し、何があっても詰まない人生を
　歩んでいこうと思います！ 

　-----------

　この音源では、鮒谷さんがどういったものを
　非常時に備えて用意されているのか、

　といった、具体的な内容から、
　タレブの書籍を踏まえたリスクに対する
　意思決定基準にいたるまで、
　網羅的に話されているので、

　これを聴けば、来るべきリスクに
　備えるためのノウハウが一通り得られると
　思っても過言ではありません。 

　-----------

　不安定な世の中で、簡単に滑り落ちる
　可能性があるからこそ、

　こういう意思決定基準は
　ものすごく大事ですね。 

　-----------

　不確実性が増している現代において、
　（社会的に、あるいは経済的に）死なないための
　必聴の音源と言えると思います。 

　-----------

　リアルタイムで起きている戦争や円安、
　地震やパンデミックの世の中において、

　未だにのほほんと自らのビジネスだけに
　邁進している自分に張本さん張りの、

　いやそれ以上の
　「喝！」を入れていただきました。 

　-----------

　サブタイトルにもある通り

　「勝つことより、破滅回避」

　なので、いつも通り(いつも以上に笑)
　地味ではありますが、

　今現在もそうですが、この先ますます
　役に立つ内容だと感じました。 

　-----------

　今回、人生クマムシ化音源を聴き、

　先が見えない時代には、
　特に破滅的なリスクを回避することが
　最重要ということで、

　人生を詰ませないためにも、
　音源の中で紹介があった書籍等を活用して、
　目標を見直していきます。 

　-----------

　「破滅を避けること」
　「死なないこと」が大切で、

　やっぱりビビりとクマムシが
　最強なのかなと改めて思いながら、
　脇を締めたところところです。 

　-----------

　音源を聴かせていただき、

　「今のままでは、何か起きたら
　　人生が詰んでしまいますよ！」

　と、水をぶっかけられたような気持ちです。 

　最後の最後まで生き延びれるよう、
　常にリスクを考え物事前倒しで進めていきます。 

　-----------

　目を惹きやすい攻撃よりも、
　地味な守備型の音源であるのかもしれませんが、

　ここに目を向けることは
　音源を聞かれた方のその後の人生の意思決定に
　大きな影響を与えるものと思います。 

　テールリスクに対する具体例を示しながら、
　それに対する対応や心構えについて
　解説がされていて、

　聴きながらいろいろ想像力を巡らせて、
　自分の危機対応力について考えさせられました。 

　-----------

　情報弱者にならないように、また、
　人生が詰んでしまわないように目の前の
　5分、10分を大切にし、読書や勉強に勤しむことの
　重要さも教えていただきました。 

　-----------

　人によっては若干、劇薬のように
　思われる箇所もあるかと思いますが、

　この音源は理想の未来へ邁進するための
　ルート上に存在する陥穽や障害物を除去するための
　重機のようなものに思われました。 

　-----------
