【あと２日】「より大きな物語」に導かれていく喜び

2019/01/13 13:30 
■こんにちは、鮒谷です。


　人はみな、

　「自分の物語」

　を生きているわけですが、


　ともすると、

　自分が価値を認めている物語
　（＝自己の信念）のみが正しい、絶対、

　という思い込みに囚われることが
　あるものです。

　

■自分が正しく、他者は誤り、
　
　自分は賢く、他者は愚か、

　自分が善で、他者は悪、
　
　自分は物知り、他者は無知、

　などなど。
　


■こうした独りよがりに陥らないためにも、
　
　【自分と異なる物語を生きている人
　　（＝自分と異なる世界を生きている人）】

　との交流を積極的に図り、
　自らの相対化を心がけるべきでしょう。

　
　そんな取り組みの繰り返しが自己の文脈を
　拡大し、

　【より大きな物語】

　を紡ぐ、あるいは導かれていくことに
　つながっていくのです。

　

■こうした世界の拡張感覚は、そのまま

　【生きている実感】

　【成長、成熟の喜び】

　につながります。


　私（鮒谷）など、自分が
　
　「間違っていたなあ、愚かだなあ、
　　バカだなあ、無知だなあ」

　と知らされ、同時に、

　「こんな世界の捉え方があったのか」　

　と気付かされるのは、
　いつも他者との交流を通してでありました。
　


■逆に言うと、

　他の人との交流を遮断しているうちは、
　自分の殻の外に出ることはできない、

　そんな風に思います。


　そこから一歩出て、


　「そんな考え方もあったんだ」

　「その発想はなかった」

　「そんな生き方もあったのか」


　と思わせてくれる機会は、つねに

　【自分以外の物語】

　を生きている人から、もたらされました。



■それは、なにも

　「直接の出会い」

　からのみならず、

　「書物を通しても出会えるもの」

　であるようです。
　


■たとえば私（鮒谷）であれば、


　経済学者の青木昌彦さんの
　

　＜人生越境ゲーム＞
　http://amazon.co.jp/o/ASIN/4532353025/2ndstagejp-22/ref=nosim　


　を読んでみては、

　「活動する世界を一ヶ所に決める必要はなく、

　　どんどん乗り換えて、
　　自分の世界を拡げていっていいのだ」

　と自らに許しを与える契機を与えられたし、


　＜職業=田原俊彦＞
　http://amazon.co.jp/o/ASIN/4845421526/2ndstagejp-22/ref=nosim　

　
　を読んでみては、

　「自分の名前を職業にしていいんだ」

　という発想の斬新さに心打たれたし、


　＜関ヶ原＞
　http://amazon.co.jp/o/ASIN/B0186R6NI6/2ndstagejp-22/ref=nosim　


　を読んでみては、

　「義に生きる生き方は格好いい、
　　こんな生き方をしてみたいものだ」

　と感銘を受けたし、


　＜我れの名はシイラカンス　三億年を生きるものなり＞
　http://amazon.co.jp/o/ASIN/4532168333/2ndstagejp-22/ref=nosim　


　や

　
　＜さくら道─太平洋と日本海を桜で結ぼう＞
　http://amazon.co.jp/o/ASIN/4833130769/2ndstagejp-22/ref=nosim

　
　を読んでみては、

　「己の人生を貫くことの尊さ」

　に震える感動を覚え、
　自分がそのような対象を持っているか、

　と自問自答の機会を与えてもらう
　きっかけとなったし、


　＜ファーブル昆虫記＞
　http://amazon.co.jp/o/ASIN/4002002551/2ndstagejp-22/ref=nosim
　
　
　を読んでみては、

　「それまで気にしたこともなかった、
　　全く別の世界に対して、これだけ関心を持って
　　眺めている人がいたんだ」

　ということに驚きを感じたし、


　＜外交＞
　http://amazon.co.jp/o/ASIN/4532161894/2ndstagejp-22/ref=nosim


　を読んでみては、

　「綺麗事では済まされない、
　　力こそが正義、の国際政治の舞台裏」

　に触れて緊張感を覚え、
　それとともに、

　「好む好まざるに関わらず、私たちの生や生活は
　　世界の政治状況に委ねられている恐怖」

　のようなものも感じたし、


　＜苦海浄土＞
　http://amazon.co.jp/o/ASIN/4062748150/2ndstagejp-22/ref=nosim


　を読んでみては

　「自分が、あるいは身内が、このような状況に
　　身を置いたら、何を考え、どう振る舞っただろう」

　と思いを致す機会にもなったし、


　＜桶川ストーカー殺人事件─遺言─＞
　http://amazon.co.jp/o/ASIN/4101492212/2ndstagejp-22/ref=nosim
　
　
　やら、

　＜消された一家─北九州・連続監禁殺人事件＞
　http://amazon.co.jp/o/ASIN/4101368511/2ndstagejp-22/ref=nosim

　
　やらを読んでみては、

　「人生、現実にこのようなことも
　　起こりうるのだ」

　と知らされ、戦慄を覚えたものだし、


　＜駕篭に乗る人担ぐ人─自民党裏面史に学ぶ＞
　http://amazon.co.jp/o/ASIN/4087481646/2ndstagejp-22/ref=nosim
　

　を読み、さらに


　＜田中角栄全記録─密着2年半、2万カットからの報告＞
　http://amazon.co.jp/o/ASIN/4087800814/2ndstagejp-22/ref=nosim


　を読んで（眺めて）みては、

　「出世街道を歩むべし、
　　権力闘争望むところ」

　と高揚感を感じたこともあれば、


　＜菜根譚＞
　http://amazon.co.jp/o/ASIN/4569691064/2ndstagejp-22/ref=nosim


　やら


　＜漢詩をよむ 陶淵明詩選＞
　http://amazon.co.jp/o/ASIN/4140842229/2ndstagejp-22/ref=nosim　


　（の『帰去来辞』）を読んでは、

　「譲歩や寛容や中庸の精神」

　に惹かれたり、

　「俗世からの開放感」

　を感じてみたり、

　
　＜茨木のり子詩集＞
　http://amazon.co.jp/o/ASIN/4003119517/2ndstagejp-22/ref=nosim


　を読んでみては、

　「感受性豊かな人は、日々の小さなことに
　　喜びを覚え、幸せを感じられる特権を持つ人」

　であると理解し、
　そのような人間になりたい、

　とも思ったし、


　＜深呼吸の必要＞
　http://amazon.co.jp/o/ASIN/4794935269/2ndstagejp-22/ref=nosim
　
　
　の『あのときかもしれない』を
　読んでみては、

　「不可逆的な人生の流れ」

　に寂しさを覚え、

　今の一瞬一瞬を大切に生きることを誓う
　きっかけが与えられたし、


　などなど、

　これまで触れてきた無数の本によって
　文脈を拡大させてもらい、

　小さな物語から、より大きな物語へと
　舞台を移し替えていく機会を与えられ続けたきた、

　という風に思われます。


　
■「私という人間は、今まで読んだ本に
　編集されてでき上がっているのかもしれない」

　とは、

　資生堂名誉会長、
　福原義春氏の言葉ですが、


　確かに、自分の思いもよらなかった世界に
　生きている人の存在を知り、

　【自分のあり方、有り様】

　について思いを致す（＝編集される）機会は、

　出会いによって、読書によってのみ、
　手に入れることができる、

　と思われます。



■身体を物理的に移動させる旅行も
　素晴らしいものですが、

　と同時に、

　いながらにして新しい世界を見る機会を
　与えてくれる、

　人との出会い、書籍との縁も、
　大切にしたいもの。



■プルーストの、

　「本当の旅の発見は、
　
　　新しい風景をみることではなく、
　　新しい目をもつことにある」

　という言葉も、

　こうした喜びが内包されている
　言葉なのかもしれない、

　なんてことも思ったりいたします。

 
  これからも私（鮒谷）は、
　出会いや書物を通じて

　【新しい物語に触れる旅】

　を重ねていくつもり。
　


■あなたも

　「自分にとっての、より大きな物語」

　について思索する機会を積極的に
　確保してみられてはいかがでしょう。

　必ずや、人生をより意義深く、
　生きている実感を得られるはず。
