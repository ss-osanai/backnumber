【（緊急案内）大阪で個別コンサルティングを受け付けます】1

2014/05/23 13:30 
■お世話になります、鮒谷です。


　いつもながら唐突な案内で恐縮ですが、

　明日、5月24日（土）から26日（月）まで
　大阪に出張することとなりました。


　出張の際は、いつも予定をびっしり詰めることに
　しているのですが、

　今回は今のところ多少、時間に余裕があるので、
　急遽、
　
　【 個別コンサルティング 】

　の枠を幾つか設けました。



■ときどき、メルマガの読者さんより、

　「東京には行けないけれども、
　　ぜひ大阪でコンサル枠を作ってください」

　といわれることがあるのですが、
　

　関西近郊にお住まいの方、
　たまたま関西に帰省、出張されている方、

　これを機会にぜひご活用ください。



■以下、過去に、個別コンサルティングを
　受けて下さった方より頂戴したご感想です。

　内容、雰囲気など、
　参考にしていただければと思います。


　＜お客さまより頂戴した、個別コンサルティングのご感想＞

　http://www.2nd-stage.jp/consult/testimonials.html



■ちなみに通常、

　＜１時間　126,000円＞

　にてコンサルさせていただいておりますが、


　急なご案内、
　かつ大阪での一回限定コンサル（複数回お申込不可）のため、
　
　今回に限って特別に、


　＜90分  84,000円（実質55％オフ）＞


　にて承ります。


　
■これまでの相談内容の傾向としては


　＜法人様＞

    経営戦略
    ビジネスモデル設計
    マーケティング
    情報発信
    営業関連　　　など


　＜個人様＞

    キャリア相談
    独立起業に当たっての心構えや実践方法
    何らかの決意の後押し
    事業計画策定　　　　　など


　といった案件が多いです。



■現在の状況としては

　
　【１】5/25(日) 14:00-15:30

　【２】5/26(月) 15:00-16:30

　【３】5/26(月) 17:00-18:30
　　　　（※３.については多少、開始時刻を
　　　　　　後ろの時間帯にずらすことも可能です）


　の３枠のみ、空き枠があります。

　（詳細はお申し込み後、ご連絡いたします）



■先着順にて枠を確保させていただきますので、


　「第一希望から順番に、できるだけ多く、
　　希望される枠の番号を記して」

　
　＜担当：長内（おさない）＞
　
　office@2nd-stage.jp

　
　にあてて、


　【大阪個別コンサルティング希望の件】


　というタイトルで、メールをお送りください。

　詳細は追ってご連絡させていただきます。



■先ほどのメルマガでは、


　「ノリと勢い」

　「フットワークの軽さ」

　「思いったら、即行動」


　が大切、と書きました。

　
　メルマガを熟読いただいた後で結構ですので、
　お申し込みをお待ちしております（笑）



■追伸その１：


　メルマガではとても書けないココだけの話（！）を
　お伝えさせて頂くこともあるかもしれません。

　「そこまで話してくれるの！？」

　と驚かれることも、よくあります。



■追伸その２：


　「時は金なり」

　ともうします。
　
　
　個別コンサルの一番の意味は

　「お金で時間を買う」

　ところにあるのでしょう。



■追伸その３：


　「知らないリスク」

　は想像以上に大きなもの。


　さまざまな失敗経験と、ほんの少しの成功体験と、
　たくさんの講演、セミナー、読書遍歴を通して学んだ、
　
　当該分野における

　「知っておいたほうがいいこと」

　をお伝えしたいと思っております。
　


■あんまりダラダラ書いても仕方ないので、
　これくらいにしておきますが、

　久しぶりの出会いから、新しい出会いまで、
　どんな御縁をいただけるか、楽しみにしています！


　＜お客さまより頂戴した、個別コンサルティングのご感想＞

　http://www.2nd-stage.jp/consult/testimonials.html

　「第一希望から順番に、できるだけ多く、
　　希望される枠の番号を記して」

　
　＜担当：長内（おさない）＞
　
　office@2nd-stage.jp

　
　にあてて、

　【大阪個別コンサルティング希望の件】

　というタイトルで、メールをお送りください。


　先着順で枠を確保させていただきます。
