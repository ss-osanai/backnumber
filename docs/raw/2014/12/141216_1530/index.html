＜今日のピークパフォーマンス方程式＞

 【  商売繁盛の一丁目一番地  】

───────────────────────────────────



■何度か書いたことがありますが、
　商売をしている人なら
　
　「賑わい感」

　を出すことは本当に大事です。


　商売繁盛しているから
　商売繁盛しているように見えるのではなく、

　商売繁盛しているように見えるから
　商売繁盛する、

　ということ。



■古典的ですが、

　「サクラ」

　なんかはその典型ですね。

　
　今は鳴りを潜めましたが
　（といって、なくなったわけでもありませんが）


　昭和の時代には、

　「サクラ」

　を使った様々な商売があったようですね。


　たしか、この本にも出ていたような。

　
　【今日のお勧め本　草競馬流浪記】

　 http://amazon.co.jp/o/ASIN/4103226277/2ndstagejp-22/ref=nosim



■もともとない存在を、
　あることにするのは頂けませんが、
　　
　「ある存在を、あるという」

　のは全く問題ありません。


　だから私（鮒谷）ももちろん、メルマガの随所に

　「賑わい感」

　を出すようにしています。



■賑わっているから、賑わい感が出たのではなく、

　メルマガ創刊初期の頃から、
　賑わい感を出してきたから、

　本当に賑わうようになったのです。


　そして一旦、本当に賑わうと、

　あとはその賑わいを、
　そのまま持続させるために、

　さらに賑わい感を見せていく（笑）


　これが商売の秘訣です。



■あなたも私も、

　「閑古鳥の鳴いているラーメン屋さんに入る」

　なんてこと、したくないですよね。


　どうせなら、

　「超満員で、行列ができている」

　そんなお店にこそ、入りたいのです。

　
　このシンプルな原理原則を理解し、
　実践するのが、

　「商売繁盛の一丁目一番地」

　です。



■お客が欲しいのなら、
　お客が来ている証拠を出さないと。


　どうして分かってもらえないんだ、

　とか、

　察して下さい、

　とかいっても、


　お客さんには

　分かろうとする義務もないし、
　察する必要性もないのです。


　ならば無理やり（！）
　分かってもらうしかありません。

　察してもらうのではなく、
　両の目を開かせ、見てもらうしかありません。



■よく、セミナーやコンサルで

　「売ってないのに売れるはずがない」

　というのですが、

　
　文字通り解釈して、

　「買って下さい」

　ということだけが、売る活動ではありません。


　
■むしろそれは稚拙だし、
　過剰にやらないほうがよいわけで、

　そういうことではなく、私（鮒谷）が

　「売ってください」

　と言っているのは、

　「買ってもらうための仕組みを整えて下さい」

　ということ。


　賑わい感を見せる、というのも、

　売るための大事な（というよりも最優先の）
　仕組みであり、施策です。



■こういう細かなことを
　ひとつひとつ潰していかずして、

　モノなんか売れるはずがないんです。

　頭も身体も使わないと。


　今回は、

　（特に経営者の皆さまへの）
　個別コンサルのフォローアップを兼ねて。
