───────────────────


■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------

　ショボくても急所を突く
　ビジネスは案外、儲かるんよ。

　格好わるい上に小さな市場だから
　ガチのエリートは入ってこんし、

　キラキラさせる必要がないから
　カネもかからん。


　結果として細く長く、
　それなりの利益を永続させられる。

　人の行く裏に道あり花の山。

　虚栄心で派手な世界に
　行かんのも一つの見識。

　----------

　自分を無能と決めつける
　必要は全くないよ。

　そう思う人は未だ適所を
　見つけられてないだけ。

　「所を得る」いう言葉があるが、
　その場所を見つけるのが
　人生100年時代の前半戦。


　後半は、たどり着いた適所を
　擦って擦って、擦りまくって
　強みを拡張するフェーズ。

　それこそが人生の喜びであり、
　醍醐味よ。

　----------

　凡人も環境次第で化けるんよ。

　ワイもADHD特性から
　周囲に疎まれ、

　居場所を探しては
　新たな挑戦を重ね、

　ようやく飛び出たホームラン。


　合わぬ土地なら、そこを離れ、
　安住の地を探す旅に出ればいい。

　あれこれ場所を変えて
　成功したのは、

　転職数十回の『ナニワ金融道』
　青木雄二さん他、たくさんいるんやで。

　----------

　「気」に意識を向けると、
　いいことあるよ。

　元気、活気、人気、
　気力、気迫、気合い、
　気配りなど。

　気の管理の上手い下手によって、
　パフォーマンスに大きな差が
　出るんだわ。


　スピリチュアルな話でなく、

　普通に「気」の持ちようによって
　人生もビジネスも変わってくるから、

　意識するといいよ。

　----------

　人間関係のメンテナンスは
　しといた方がいいよ。

　「去る者は追わず」の
　姿勢も悪くはないけど、

　せっかくの縁を放置して
　疎遠になるのも勿体ない。

　それを防ぐためのSNS活用などの
　活用なんかも検討してはどうか。

　----------

　当事者にならん限り、
　本当の意味では学べんのよ。

　逃げられぬ環境に身を置けば
　広い範囲に目配り、心配り
　するようになる。


　傍観者でいるうちは
　どこまでいっても他人事。

　それでは学びの吸引力が
　狭い範囲で、小さくしか働かない。

　成長したければ
　手を挙げ、名乗りを上げ、

　主体性を持って
　臨むしかないんだわ。

　----------

　エリートに冷笑される
　ビジネスは案外、悪くないよ。

　ガチで賢い人間は入ってこんから、
　格好悪さを厭わなければ儲けられる。


　ついでに

　「戦わずして勝てる場所」

　を占有し続けてる自分こそ格好いい、
　と自己洗脳できれば完璧。

　そんな場所に20年以上の長きに渡り、
　居座ってきたのがワイ。

　----------

　カネの流れが枯れてたら、
　増やしようがない。

　でも細くとも流れさえあれば、
　そこから大きく育てていけるのよ。


　ビジネス経験者はみんな知ってるけど、

　初学者はそれが分からんから、
　細いカネの流れをバカにする。

　そこを乗り越えた者のみ、
　極太の奔流を作り出せるのよ。

　初心者はまず細い水路づくりから。

　----------

　人生ゲームは「テトリス」に似てるよな。

　課題を先送りしてると、
　あっという間に詰んでまう。

　それを避けるには、先手を打ち、
　潰すべき課題を最初期段階で
　潰しにかかること。


　上手くいかずに焦ることもあるが、

　焦って次もミスると一気に
　詰みの可能性が高まるから

　「落ち着き払って、着実に」

　やで。

　----------

　『達人のサイエンス』はお勧め。

　分厚い壁を超える挑戦の際、
　脳は自らの回路を再構築している、

　といった話が出てくるで。


　表立った変化はなくても、
　試行回数を増やすほど

　水面下で猛烈な再編成が施され、
　あるとき一気に突き抜ける。

　こんな話は刺激的で
　勇気をもらえるよ。

　機会があれば読むといいよ。

　----------

　毎日、高い目線で生きてると、

　心中に「！」や「？」が
　ひっきりなしに生まれるのよ。


　「！＝感動」が生まれた瞬間、

　なぜ心が動いたのかを
　掘り下げて記述する。


　「？＝疑問」が浮かんだときは
　即座に調べ、

　学んだことを
　忘れぬうちに記録する。


　自分だけの経験 / 学び / 感情
　の軌跡が記された、

　他に二つとない財産。

　----------

　目指す世界にどうしても行きたい！
　というプラス方向の動機。

　このままいくとマジで人生詰むから
　「どげんかせんと」という

　恐怖と不安と焦燥しかない
　マイナス由来の動機。


　この合計が閾値を超えたとき、
　人は本気になるんだわ。

　負からの脱却だけでも
　強いエンジンになるけど、

　両者が合わさると最強。

　----------

　生活を安定させる
　確実な方法を教えるで。

　真剣に仕事に取り組み、稼ぎ切る。
　顕示的消費（浪費）から距離を置く。


　必要なところにはお金を使いつつ、
　余ったお金は将来、価値が増すと
　思われる対象に投資する。

　やがて投資収益が生活の下支えとなる。

　長年月、ひたすらこれを
　繰り返すだけの簡単なお仕事よ。

　----------

　幸せになるのに大金はいらんが、
　不幸を避けるには
　それなりのカネがいるだわ。

　幸せは心の持ちようやけど、
　不幸や絶望から身を守るには、
　お金という盾がいる。


　これに限っては
　キレイごとでは済まんのよ。

　だから余裕がある時に、
　しっかり蓄えを積み上げておく。

　窮地に陥ってからでは手遅れやで。

　----------

　NHK『ドキュメント72時間』
　いう番組が好きで、

　オンデマンド視聴が可能な
　全250本くらい？全部見たけど、

　10回に1回くらい

　「昔は羽振り良かったけど、
　　今はこの有り様」

　みたいなことを言われる人が
　出てくるんよ。


　調子の良いときにこそ備えとけ、
　という生きた教訓を

　リアルな回顧と共に
　学ばせてもらったよ。

　----------

　過去に何度「踏ん張りどころ」を
　乗り越えてきたかで
　未来が決まるのよ。

　一度もなかったとすれば、
　今からでも挑戦することを勧めるよ。


　一方、複数回の勝負どころを
　乗り越えた経験のある人は、

　経験と自信という大きな財産があるから
　自らの力で未来を切り開けるし、

　リスク耐性が高いから安定感もある。

　----------

　独自性のある固有の「語録」は、
　真剣な実践からしか生まれない。

　経験と思索からしか
　生み出されない魂の言葉やからな。

　特有の「節回し」を操る人の
　言葉に重みを感じるのは、

　言葉自体の価値もあるけど、
　それ以上に

　フレーズが紡がれるまでの
　数々の経験と深い智慧の裏打ちを
　感じさせるからやと思ってる。

　----------

　人生は自分で作り上げるもの。

　「右肩上がり傾向にある人生」

　は作ろうと決めない限り、
　絶対に手に入らん。


　ただし一度

　「逓増＝成長を続けるシステム」

　を人生に組み込む決意を固めれば、
　あとは行動に移すだけ。

　その行動が新しい現実を
　作っていくんだよ。

　----------

　「幸せになりたい」から
　「幸せになる」への決断が
　全ての始まりなんよ。

　この決意あって初めて、
　求める世界が見えてくる。


　で、面白いことに
　真剣に幸せを追求すると、
　必ずお金の問題にぶつかる。

　きれいごとじゃ済まんから。

　本気で幸せを考える人は、

　必然的に経済的な側面も
　真剣に考えるようになるんだわ。

　----------

　人は皆、例外なく自慢したい生き物。
　商売人も一緒やで。

　ただし、本当に自信があって
　自慢する商売人と、

　自慢するところがないのに
　自慢しようと試みるエセ商売人が
　いるので要注意。

　
　我々は心から誇りを持って
　商品を語る人から買いたいもの。

　だけど、
　発信情報を注意深く吟味すれば
　区別はつけられる。

　----------

　「平凡な人生」を送るのですら、
　なかなか大変。

　経済面だけ見ても、
　最低限の基準を満たすだけでも大変。

　だからこそ、強い意思を持って
　未来を切り開いていかなあかん。


　「平凡な毎日」を続けるだけでさえ、
　それなりの覚悟と行動が必要。

　これが今の時代を生きる難しさであり、
　現実なのよ。

　----------

　「幸運は準備していた者にだけ微笑む」
　いう言葉がある。

　チャンスが来る前から、
　黙々と学びと経験を積み重ねた人だけが
　機会をモノにできるのよ。


　チャンスが訪れたのを見て、
　慌てて準備し始めたけど

　「時すでに遅し」で
　痛い目に遭う人があまりにも多いよな。

　何ごとも早め早めの手当てが大事、
　いうこと。

　----------

　「人生は下りエスカレーター」
　だと気づいたとき、

　物語の第二幕が動き出す。


　この現実を受け入れた上で、
　下る速度を抑え、

　さらには上りエスカレーターに
　乗り換える。


　そんな取り組みをいち早く始めた人だけが
　人生の荒波を乗り越えていけるんよ。

　大事なのはアキラメやなく、
　この事実と向き合う覚悟。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
