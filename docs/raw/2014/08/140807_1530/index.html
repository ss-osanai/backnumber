＜昨日の活動ダイジェスト＞

▼鮒谷が個別コンサルを行い続ける理由

───────────────────────────────────



■こんにちは、鮒谷です。

　
　昨日はねぶた祭。
　今は青森からの帰路にあります。



■最近、

　ますます多くの個別コンサルのお申し込みを頂き、
　嬉しい悲鳴をあげている今日この頃ですが、

　時折、

　「なぜそこまでコンサルを行うのですか？

　　ビジネス的にはレバレッジも効いておらず、
　　むしろやらないほうがいいのでは」

　という質問を頂くことがあります。



■まことにおっしゃるとおり！なのです。

　たしかに表面的に見れば、

　「個別コンサルティング」

　は究極の肉体労働であり、労働集約ビジネスであり、
　レバレッジも効かず、属人的でもあり、

　ということで、

　「やらない方がいい」

　という考えもあるでしょう。


　（というか、だからこそ、
　　私もある時期までは、頑なに拒んできたわけです）



■しかし、あるとき、たまたま
　
　『ストーリーとしての競争戦略』
　
　を読んで、その中に、


　「それだけでは"一見して非合理゛だけれども、

　　ストーリー全体の文脈に位置付けると
　　強力な合理性を持っている」


　という文言に惹かれ、思考実験の一つとして

　「他を圧する数の個別コンサルティング」

　を行い続けていったら、その先に何が生まれるか、

　について考えたとき、

　「あ、これだわ！」

　と閃くものがあったのです。



■詳しくは稿を改めて記そうと思いますが、
　日本において、

　「この価格帯で、これだけの数のコンサルティングを
　　行い続けている人間」

　は、おそらく日本中探してみても、
　ゼロとはいいませんが、かなり希少な存在だと思うし、

　もっともっと、
　その希少性に磨きをかけたい、
　
　という思いを持っています。


　
■私よりも大勢の人とコンサルティングを
　続けている人はあるかもしれません。
　
　もっと高額のチャージをされている方も
　ゴマンといらっしゃるでしょう。
　
　
　しかし、個人向けコンサルとして、

　この金額で、これだけの方とお会いさせて頂き、
　なおかつ仕事が途切れない、

　という状態を維持し続けるのは、

　模倣しようと思っても、
　おそらくかなりの模倣困難性が伴うと思われます。


　（提供する価値はもちろんのことですが、

　　同時に、相当レベルのマーケティング、ならびに営業、
　　さらには運用の仕組み化が必要になってくるから）
　
　

■というわけで、

　ひたすらに「肉体労働」をやり続けて、
　そこから抜けた先に、どんな景色が現れてくるか、

　今から楽しみに夢を見ているし、
　だからこそ、朝から晩まで働き続けられるのです。


　といいつつ、昨日は、

　「ねぶた祭」

　だったわけですが（汗）




■さて。


　話は変わりまして、以下は、

　先日、個別コンサルティングを行わせて頂いた
　渡辺さんから頂戴したご感想のご紹介。


　（ここから）
　----------------------------------------------------------------


　鮒谷さま


　先日個別コンサルを受けさせていただいた渡辺です。
　延長のお時間までいただき誠にありがとうございます！


　初めて鮒谷さんにお会いしたとき、
　メールマガジンの文章から受けるイメージよりもお若く
　とても気さくに感じたのが第一印象でした。

　初めてコンサルを申し込んだということもあり、
　最初は少し緊張もあったのですが、話し始めてすぐに
　鮒谷さんのお話に集中している自分がいました。


　持ち込んだのが少し曖昧な質問でしたが、
　広く深い知識から適切かつ噛み砕いた表現で説明いただき
　脳が心地いい刺激を受けたのを覚えています。

　目標を見据えつつキャリアを積んでいくに当たり、
　表面的な部分ではなく本質を追求していく、
　自分に問いかけていくという点は、
　抽象度が高そうに見えて、自分の生き方、軸となる部分となり、
　後に具体性を持った行動に変化していく起点となりそうです。


　また先人のパラダイムと鮒谷さんの日常的な例に
　照らし合わせたパラダイムは笑いを混ぜつつ
　自分に浸透していきました！

　教わった書籍も早速アマゾンで注文しました。
　コンサルの復習かつ忘却のストップに
　歯止めをかけたいと思います。


　遥か先を進んでいる目標点でもあり、
　くだけた話も出来る仲のいい先輩と接しているような
　信頼感と安心感のバランスを持った時間に感じました。

　教わった内容を自分に落とし込みしつつ、
　次回に申し込みさせて頂く時に深堀り出来ればと思います。

　今後ともよろしくお願いいたします！

　ありがとうございます。


　　　　　　　　　　　　　　　　　　（渡辺　健文さま）


　----------------------------------------------------------------
　（ここまで）


　渡辺さん、ご感想、ありがとうございました！



■コンサルさせて頂いてきて、
　しょっちゅう言われるのが、

　「メルマガのイメージよりも若く、
　　メルマガよりも固くない（きさく）」

　ということ。
　


■文章って不思議ですね。

　別に作っているわけではなく、
　むしろありのままに書いているつもりなんですが、

　どういうわけか、実際にお会いしてみると、

　「本人とはまったくイメージが異なる」

　と言われることが多いです。

　喜んでいいのか、悲しむべきかは分かりませんが（汗）



■それにしても、

　昔から見知っている、勝手に親近感を持ってきた会社と
　ゆかりのある方とこのようなご縁を頂けたのも、

　メルマガを書き続けてきたから、
　と、つくづく思います。


　コンサル、お役に立てたようでよかったです。
　
　渡辺さん、これをご縁に、
　これからもよろしくお願い致します！
　

　　

■それでは

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【  「安心、安全、安くつく」専門家への頼み方  】

───────────────────────────────────



■最近、つくづく思うのが、

　「個別コンサルティングとは、
　　人間理解の場である」
　　
　ということ。

　
　これまで、個別コンサルを通して、

　たくさんの人と、継続的な関わりを持たせて
　頂いてきたわけですが、

　その中には、

　
　▼すぐに変化していく人

　▼徐々に変化を遂げていく人

　▼頑強に変わることから抵抗しようとする人

　　（＝頭では変わりたいと思いつつ、
　　　　身体が全くいうことを聞いてくれない人）

　　
　などなど、本当に様々な方がいらっしゃいます。



■そんなお一人お一人からお話を伺い、
　見立てをし、処方箋を用意し、お薬を出す。

　
　そして、次の回に、
　先に出した薬が効いたか効かなかったかを見て、

　あるいは、

　どれほど効いたか、
　また、効きが強すぎたのか、弱かったのかを見ては、

　また処方箋を用意し、お薬を出す。

　
　この繰り返しです。



■回数を重ねるうちに、

　もちろん、完璧にはまだまだ程遠くとも、

　かなり精度が上がってきた（ように思われます）。


　「携わってきた数が、能力に変わっていく」

　という意味では、
　お医者さんと同じかもしれません。



■お医者さんの手術の技量は、

　それまでに執刀してきた患者さんの数と、
　およそ比例する傾向にあるのは間違いありません。

　
　少なくとも

　「初めて手術をする新米のお医者さん」
　
　よりは、

　「執刀してきた数が数千の単位を数える重鎮」
　
　にお願いしたほうが、

　手術の成功確度は、言うまでもなく、
　高まるのでありましょう。



■そう考えると、どんな分野であれ、

　複雑な状況に対応する必要に迫られて
　専門家に頼もうとする際には、

　「経験と実績が豊富にある人」
　
　に依頼したほうが、

　多少、価格が高くとも、

　あるいは、

　接触が難しければ、
　様々な「つて」を頼ってでも、

　そのような人にお願いしたほうが、
　良いように思われます。



■医師の事例で挙げた通り、
　どんな分野においても、

　「経験してきた数」

　は、

　「その他のこと（たとえば知識など）に
　　決して代替され得ない、

　　状況把握、ならびに最適解を提供する能力を形成する
　　最大の要素」

　であるからです。



■いうまでもなく、
　数多くの、さまざまな種類の事例に接してきた人ほど、

　
　▼複雑な要素が絡み合った文脈全体を理解し

　▼解きほぐし

　▼最適解を提示してくれる
　
　
　可能性が高まります。



■ちなみに私自身は、
　もちろん、まだまだ修行の身であり、

　そのような世界とは、
　程遠いところにあるわけですが、

　いずれ、
　
　
　▼お話を短時間うかがっただけで、一発で
　　症状と、ならびに治療法がわかり

　▼最短で結果を出せ、副作用もほとんどない、
　　適切な投薬ないし手術をおこなえる

　
　そんなレベルの医師？治療家？を
　目指したいと思います。



■それには、十年、二十年単位の
　歳月がかかるかもしれませんが、

　志高く、日々学び、経験を重ねてまいる
　所存です。
