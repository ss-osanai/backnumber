6700号 人生の更新には、前提の上書きが必要

■「世界にはお金があふれ、
　　豊かさが満ちている」

　そう考える人は、

　まだお金を稼いでいなくても
　世界は豊かなものに感じられ、

　やがて富を手に入れられるもの。



■反対に

　「世界は凍てつく
　　ツンドラ地帯のような場所」

　と捉えている人は、

　永久に殺伐とした世界に
　住まなければなりません。



■人は見た通りに世界を捉え、
　その認識に従って振る舞い、

　そこから生じる結果を
　良くも悪くも手に入れます。



■先の例でいえば

　「世界は凍てつく
　　ツンドラ地帯のような場所」

　と見る人は、


　あらゆる資源は限られており、

　限られているということは
　全員には行き渡らない、

　そんな風に考え、
　行動するから本当に

　「奪い合いの地獄（殺伐世界）」

　が現出することとなります。



■逆もまた真なりで

　「世界にはお金があふれ、
　　豊かさが満ちている」

　と見ている人は、

　そうした前提で振る舞うことで好かれ、
　周囲から配当（還元）を受け、

　本当にお金に恵まれるようになる。



■つまり、

　世界は豊かなのが本当か？
　枯れているのが本当なのか？

　と尋ねられたときの答えは
　いずれも本当であり、
　

　真実は、私たち一人一人が

　「自分の心で生み出すもの」
 　
　これが正解なのでしょう。


　
■とするならば、

　今、私たちが持っている世界認識
　（先入観・前提とも）

　を更新し、
　精度を高めることによって、

　行動の質を高め、
　運命を作り変えることができる、

　とも言えそうです。



■あるときから、こんな風に考え、

　いま時点で抱いている自身の認識は
　正しくないかもしれない、

　と、常に懐疑する姿勢を持つように
　なりました。



■こうした意識を持ち、

　更新したほうが良さそうな「前提」を
　探しては、試行錯誤を重ねるうちに、

　徐々に人生が変わり始めることと
　なりました。



■たとえば一つの例として、
　他者とのコミュニケーションにおいて、

　前提を「同床異夢」と更新して
　生きるようになってから、

　コミュニケーション能力は
　格段に上がりました。

　うまく取れず、

　取り巻く環境が
　望ましい状態にならない、
