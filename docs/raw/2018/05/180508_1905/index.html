　（ここから）
　---------------------------------


　言わずもがなですが、
　今回も価格が「安すぎ」ました。


　鮒谷さんがいつもおっしゃっている

　「代価の10倍返し」

　の意味も改めて体感させて頂きました。


　加えまして、

　今回のmy「肝中のキモ」は、
　大きく3つです。


　1.気づき発即断経由アクション行き
　（JR九州会長 唐池恒二より）

　・ドラッカーも「決断の多くは
　　実行されるまでの間に行方不明になる。」
　　と言っている。


　2.DCAP（Do Check adjust Plan)

　PDCAはもう古すぎる。
　時代は、DCAP（ディーカップ）

　これ、早速フル活用させて頂いております！笑


　3.第二領域しか仕事と呼ばない。

　ネタばれになりますが、

　より具体的に記述しますと、
　下記と理解しております。


　◆第二領域＝未来に価値を生むこと
　（将来資産となること）しか仕事と呼ばない。

　別の言い方ですと、

　【昨日と異なる、新たな創造を行い、
　　価値を継続的に生み出すシステムを作ること】

　のみが仕事。


　◆生産性とは？

　　＝単位時間あたりの生産量
　　＝未来を見据えた時間軸での生産量

　　であると定義する。

　・生産性を1時間当たりの生産量で考えても
　　あんまり意味がない。

　・基本的に、肉体労働は仕事ではない。


　◆どうしたら、未来の資産になる
　　行動ができるのか？

　自分自身にとって
　未来の資産となる行動とは何か？

　という問いを立て、
　行動をすることが大切。


　これほど、仕事を強く

　【「定義づけ」＝「言葉として自分の中に
　　「持っておく」こと」】

　が出来れば、自ずと変化がおこり、

　成果が出ざるを得ない体質になれるのでは？！

　と感じましたし、
　それを体現できる男になります。


　いつも、学びの提供を有難うございます。
　次回の音源も楽しみしております。

　（ここまで）

　
　さすがはＮさん、
　まさに私（鮒谷）のお伝えしたかった

　【肝中のキモ】

　を要約くださり、
　ありがとうございました。



■このご感想、ならびに私のコメントを
　読まれた方は、

　以上が

　【肝中のキモ】

　なのであれば、

　おおよそのイメージもつくし、
　別にもう、聴かなくていいか、

　と思われるかもしれませんね。



■もちろんそれでまったく
　構わないのですが、


　「上の言葉を読んで、
　　なるほど、なるほどと理解できたとして、

　　果たしてそれで、
　　体が動きますかね、

　　動くようになって、成果を湧き上がらせる
　　イメージは生まれてきますかね、、、」
　

　というメッセージだけは、
　お伝えしておきたいと思います。



■知っている、のと、やっている、

　の間には巨大な断絶があるのです。


　そして、この

　【理屈と実行の間に存在する、
　　巨大な断絶】

　を埋めるもの、それが

　【語りの力（＝感情を揺り動かす力）】
　
　であったりするのです。



■私の本分は、実にこの

　【語りの力】

　にあると思っていますし、

　むしろその能力を磨くために
　ひたすら毎日、

　【書き修行】

　を繰り返してきたつもりです。


　※以前もメルマガで記したことがありますが、

　　書くのが目的なのではなく、
　　話せるようになるために書いている、

　　のが本当のところです。



■語りを聴かれることなく、
　メルマガを読んだだけで

　【分かったつもりになりました、
　　理解したつもりになりました、

　　（でも、体は一切、
　　　動かしてはおりません、、）】


　というのは、

　【語りの職人】

　としては少々、寂しいものがあります。



■語りの世界にお越し頂きたいと
　願っているからこそ、

　そのきっかけづくりのために、

　【（本職ではない）記述の世界】

　において、

　毎日毎日、こうやって、しこしこと
　メルマガ発信しているわけでして、、、



■ぜひ以下の音源を通して

　【鮒谷の語りの世界】

　をご体験頂ければと願っております。
