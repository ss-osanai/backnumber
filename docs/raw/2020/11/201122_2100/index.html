■音源販売は、こうした欠点を克服します。
　（＝お客さまのニーズを満たします）

　お客さまのニーズを満たす、

　ということは、その分だけ
　売り上げも増大するということです。

　お客さまもよし、売りても良し、
　ということになりますね。



■つまり、

　専門知識を持った人はすべからく
　サービスの一つの柱として

　「音源化ビジネスを持っておくべき」

　ということです。



■これはお客さまのことを考えれば

　「持っておいたらいいですよ」

　という話ではなく

　「持っておくべき」

　ものといえるでしょう。



■こちらの都合ではなく、
　お客さまの都合を考えたら、

　「素晴らしいサービスを提供している
　　あなただからこそ」
　
　音源ビジネスを始めなければならない、

　ということです。



■私（鮒谷）は、

　音源販売を初めてより、幾たび

　「もっと早くからやってもらいたかった」

　という声を、お客さまより
　投げかけられたかしれません。

　（本当の話です）



■つまり、

　あなたがお客さまを
　大切にしようとされている、

　士業、コンサル、専門職の方で
　あればあるほど、

　音源販売は

　「待ったなし！」

　の喫緊の課題といえるのです。

　「長らく、お客さまへの価値の最大化」
