■今日も

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------
　　
　大きな成果を上げるより大切なのが、
　破滅リスクを避けること。

　いくら儲かっても
　一発で全てを失うような賭けをする人間は、
　やっぱり賢いとは言われんで。


　人生も仕事も投資も、まず先に

　「何が起きても絶対に詰まない」

　ラインを見極め、

　そこから逆算して考えてれば
　最悪の事態を避けられる。

　----------

　仕事人としての基本も身についてないのに
　高度な知識ばかり求めるのは、

　シャツの第一ボタンを留め間違えたまま、
　次々とボタンを留めていくようなもの。


　最後までいって「あれ？」と
　なった時には全部やり直し。

　先にまず、原理原則を学び、きちんと
　身につけることを優先すべきやで。

　----------

　努力してるのに実を結ばん
　と悩んでる人は、

　「基礎をないがしろ」

　にしてる可能性が高いんよ。


　どんな分野でも
　「基礎の上に応用」が鉄則。

　学問もスポーツも芸術も、全部そう。


　基礎をおろそかにして、
　目新しいことばかり追いかけても

　労多くして功少なし、になるで。

　原理原則を徹頭徹尾、
　先に刷り込むこと。

　----------

　初めての起業で
　「無借金」はマストやで。

　特に会社の看板で仕事してる人は、
　肩書きがなくなったときに

　どれほど無力か（相手にされないか）を
　身体で理解しとらんから、

　自分の能力を
　過大評価しがちなんだわ。


　借金あるのに売り上げ立たず、

　毎日カネが流出し続ける恐怖を
　感じたときには既に手遅れ。

　----------

　毎日カネが流出し続ける恐怖は
　IQを30くらい下げるんよ。

　借金があれば、さらにマイナス30。

　IQ120のぼちぼち賢い人も、
　金繰りに困るとIQが60になってまう。


　思考が停止し、
　投げやりな気持ちから悪手を連発。

　貧すれば鈍し、
　ジリ貧がドカ貧になるんだわ。

　----------

　初めての起業で
　「小資本」はマストやで。

　慣れとらんのに、
　失敗したら即詰みのリスクは
　冒すべきやない。


　「許容可能な損失」

　いう概念があるけど、

　許される範囲での失敗なら
　再起できるんよ。


　それを超えての致命傷は
　一生を棒に振る可能性まであるから、

　「命運を賭して」とかは
　絶対に止めるべき。

　----------

　資産を築くために、
　カネの流れを生み出せるようになってから
　借金するのはあり。

　けれど初心者なのに過度の借金をして、
　いきなりコケたら再起不能になるからな。


　資本主義下において語られる借金礼賛は、
　（正しい側面がありつつ）
　生存者バイアスもかかってるから、

　そこは理解しておいたほうがいい。

　----------

　サラリーマン時代含め、

　ワイは商売で「形あるモノ」を
　扱ったことは

　ただの一度もないんよ。


　形なきものは（知恵を働かせれば）
　高い利益率を実現させられると学んだで。

　もちろんキーエンスやディスコなど
　例外もあるけどな。


　利益率は最初の事業選択で
　あらかた決まってしまうから

　起業時、慎重に選ぶべき。

　----------

　形なきモノを売る
　＝目に見えないモノを売る、

　いうことやから
　難易度は格段に高くなる。

　それゆえマーケティングや
　セールスを行う際の

　「言語化能力の巧拙（上手いor下手）」

　がダイレクトに効いてくるんだわ。


　あなたが際立って言語化力の
　高い人なら、

　形なきモノを売る世界で
　大儲けできる可能性が高くなる。

　----------

　形なきモノを「売れる」と儲かるが、

　その一歩手前、
　形なき商品を「作れる」と

　もっと儲かる。


　でもMAX儲かるのは、

　形なき商品をイチから作り、
　自らそれを売りまくれる人。

　「高度な抽象思考と言語化能力
　　（マーケ＆セールス力）」

　を兼ね備える必要あるけど、

　資質ある人は大儲けが約束されてると
　言って良い。

　----------

　起業においてモノを扱わなければ、

　倉庫や物流、在庫管理なんかと
　無縁になる。


　その上、経理もシンプルになるから、

　複雑性を伴わない、軽くて分かりやすい
　事業を営みたければ、

　無形商材の販売は
　一つの選択肢としてありやで。


　そこで成功した後、形あるモノを扱う
　事業に移行するのも勿論あり。

　----------

　原価率の高い商材を扱うビジネスは
　小規模起業に向かんのよ。

　体力（資本力）勝負になるからな。


　理想は粗利100％に近い商売。

　必然的に形あるモノではなく、

　情報・ノウハウ・知恵・システム・
　ネットワーク仲介等になる。


　特定事業を否定する意図はない。

　あくまで極小リスクで
　離陸するための参考として。

　----------

　自己犠牲を前提にした経営とか
　ありえんよ。

　人の幸せにつながる
　ビジネスだったとしてさえ、

　持続可能じゃなければ
　全く肯定できないな。


　経営者が幸せでいてこその経営。

　そもそも自分が満たされてもないのに、

　他人を満たすところにまで
　思いを致すことなんかできないよ。

　----------

　利益率の低いビジネスは
　必然的に自己犠牲を伴うのよ。

　たとえば
　休む暇もないほどの自己犠牲の下、
　薄利で商売を営まれてる方がある。


　たとえお客さんに価値を
　提供できていてさえ、

　少なくともワイはその環境では
　働きたくない。

　持続可能じゃないからな。

　----------

　事業のキモ（利益の源泉）は
　内製化しておく必要があるけど、

　それ以外は外注を活用しつつ
　柔軟に運用することで

　固定費を抑えられるよ。


　能力が担保され、
　文化に染まった大勢のスタッフが
　利益の源泉なら難しいけど、

　事業形態によっては
　固定費極小化を図りつつ、

　収益最大化を実現させられる。

　----------

　ビジネスにおいて
　見込み客の確保は死活問題やで。

　カネを払って集客する企業もあれば、

　一円もかけずに
　行列を生み出す会社もある。


　知恵を絞らなければ前者になるが、

　うまく立ち回れば
　後者の事業を行える。

　両者の方向は真逆やが、

　どちらが儲かる上に楽しいかは
　言うまでもない。

　----------

　特に大企業サラリーマンに顕著やが、
　集客を甘く見てる人が多いんよ。

　独立すれば自動的に客が来る、

　と勘違いしとるんやないかと
　思うくらい。

　そんなはずあらへんで。


　どれだけ頑張っても
　客が来んのが普通やで。

　その状態を「行列が日常」に
　変えるのがマーケティング。

　-------------
　（ここまで）
