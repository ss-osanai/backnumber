6842号 人生という名のカードゲームの必勝法

■極論かもしれませんが

　「欠点がなければ我田引水できない」

　と思っています。

　
　そつなく、何でも上手にこなせる
　器用貧乏の人は、

　自分に都合のよい物語を
　編むことができないのです。



■あらゆる方面において器用なのは
　結構ですが、外から見れば

　「器用貧乏
　　＝面白みのない、退屈な人」

　「特徴がありそうで、ない、
　　役に立つけど面白みのない人」

　認定されかねません。



■そう考えれば、むしろ

　「たった一つ、
　　ブチ抜けた強みがあるけれども
　　あとはことごとくダメ」

　もしくは

　「全方位的に万能で
　　圧倒的な結果を残せる実力がありつつも、

　　たった一つ、
　　信じられない弱みがある」

　そんな人ほどが耳目を
　集めやすくなるでしょう。

　


■バランスの崩れた人、

　つまり、自らの持つ各資源を
　レーダーチャートで示すならば、

　凸凹が大きければ大きい人ほど
　関心を示してもらいやすくなりますが、

　
　上記二通りの人の中では特に

　「たった一つ、
　　ブチ抜けた強みがあるけれども
　　あとはことごとくダメ」

　こちらのほうが
　目指しやすいものです。

　（後者は先天的な多方面にわたる
　　優秀さが要求されるのでレアキャラ）
　


■しかし前者
　（一つの強み＆多数の弱点の人）は、

　選択と集中による
　資源の的確配分によって、

　実現可能性は大いに高まります。



■したがって私たちが目指すべきは
　一般的にはこちらのほうであり、

　この姿に近づけていくほどに
　物語を作りやすくなる、

　と知っておかれるとよいでしょう。

　

■この感覚を深いところで
　理解できれば

　「規格外、万歳！」

　となるわけですが　
　分かっていない人が多いから、


　自ら資源の逐次投入や分散を行っては
　大した魅力の感じられない

　「量産型」

　になりにいくのですから、
　努力の方向が根本的に間違っている、

　と断ぜざるを得ないのです。



■私（鮒谷）はよく、

　「量産型」のことを
　「金太郎飴」と呼ぶのも同じで、


　持って生まれた、

　我田引水物語の素材となる
　規格外要素をわざわざ型にはめて、

　果たして何をしようというのでしょう。



■そんな方を見るたび、

　自らポテンシャルの高い、
　独自性のある魅力を殺して

　「その他大勢」

　になりにいくのですから、


　その努力の仕方は、

　東京から大阪に行こうとしているのに
　全力で北海道に向かっている、

　という滑稽な姿に見えてしまうのです。
　
　（すみません。
　　でも、そう見えてしまうのだから
　　仕方ない）



■とはいえ、もちろん、

　先天的に（ある意味）優秀で
　天性の「量産型」「金太郎飴」的な、

　そんな能力をお持ちの方は、

　ずば抜けた「量産型」「金太郎飴」としての
　活躍の場はいくらでもあるでしょう。



■もともと持って生まれた各種能力が
　規格内に収まりやすい人にとっては、

　これがその方にとっての

　「正しい人生＆キャリア戦略」

　になると思われます。



■しかし問題は

　「天性の『量産型』『金太郎飴』的な
　　人物は、むしろレア」

　であるところ。

　ほとんどの人は、圧倒的に
　欠点、欠陥のほうが多いのです。



■多くの人はわずかに、１つか２つ、

　「荒削りだけれども
　　光るものがある」

　そんな手持ちカードから人生ゲームを
　スタートさせることになるはずです。

　（もちろん私（鮒谷）もそうでした）



■例えて言うならば、この

　「ごくわずかの強み（候補）」

　と

　「圧倒的多数の弱点」

　を組み合わせてカードゲームを行う、
　これが人生。



■このカードゲームは、

　各特徴を順序よく、上手につなげて
　注目を集め、価値を提供できる、

　そんな一連のカードを
　いかに素早く作れるか、

　というゲームです。



■そんな風に捉えるならば
　私（鮒谷）は、

　ポーカーやブラックジャックにおける
　カウンティングと同様に

　「手札を組み合わせ、（人生）ゲームに
　　勝てる物語を創造するための必勝法」

　があると認識しています。



■分かりにくければ一つ事例をあげると、

　私（鮒谷）であれば、
　ほとんど唯一と言ってよい、

　小学校時代以来の、大量の
　（広い範囲に及ぶ）読書経験に基づく

　「言語化能力」

　という自覚できる強みがありました。



■この能力を引き続き伸ばし、

　それ以外の大量の弱点や失敗や黒歴史を
　面白おかしく語ることによって

　「欠点を我田引水式執筆、
　　ならびにトーク」

　に転換させることによって
　「すべらない話」化して、

　お届けすることで頂戴する
　アテンションをマネタイズしてきた、

　といえるでしょう。



■これはあくまで
　私（鮒谷）の例であって、

　全員が全員、同じことができるとは
　毛頭思っておりません。


　
■しかし抽象度を上げて
　この構造を把握できるようになり、

　「ごくわずかの強み（候補）」

　と

　「圧倒的多数の弱点」

　を自在に編めるようになれば、

　全員が全員、同じことができるように
　なるのです。

　（言っている意味、分かりますよね）



■敷衍すると、

　世間一般で求められている規格から
　多くの要素が外れており、

　さらには外れ方が半端ない
　（偏差値30未満レベル）ようであれば、

　むしろラッキーであると
　喜ぶべきでありましょう。


　
■超越した規格外、つまり

　「ホームラン級のバカ」
　
　だったりすると、

　独自性が尊ばれる物語の素材としては、
　一級品のものとなるのですから、

　そのバカさ加減を愛でるべきなのです。


　
■あなたの持つ

　「極めてまれな長所」

　と

　「ホームラン級のバカ要素」

　そして、そこまではいかない
　
　「無数の短所や欠点や黒歴史」

　を上手に編集することによって、


　人様から関心を払ってもらったり、
　価値を感じてもらえる、

　生き生きとした、
　あなた独自の物語が立ち上がるはず。



■こうした物語は

　「物語の作り方を理解し、
　　物語を編もうとすることによってのみ
　　生まれてくる」
　
　ものであり、


　「物語の作り方も知らなければ、当然、
　　物語を編もうとする思いもなければ、
　　決して生まれてはこない」

　ものでもあります。



■これまで案内してきた我田引水音源は、
　ここでいうところの

　「物語の作り方を理解する」

　ことを助ける音源となります。


　あなたが、

　自分の所有資源を把握し、
　物語の作り方を理解し、
　物語を編もうと決意することにより、、

　そこから新しい人生が始まるのです。



■先のたとえでいうならば

　「デタラメに人生という名の
　　カードゲームを行っていた」

　ところから

　「カードゲームの勝ち方を理解した上で
　　ゲームをプレイできる」

　ようになるということですね。



■その価値は私（鮒谷）にとっては

　「プライスレス」

　だと思うのですが、

　（だって、今、行っている人生という名の
　　ゲームがどんなゲームかを理解して
　　プレイできるようになる、

　　という価値に
　　値段はつけられないでしょう）


　理解できない人にとっては
　理解できないと思いますし、

　理解できない人に
　無理やり理解してもらいたい、

　とも考えておりません。



■確かに使いこなせるようになれば
　プライスレスの価値がありそうだ、

　そうお感じ頂ける方があれば
　ぜひ聴いてみて下さい。
