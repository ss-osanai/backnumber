＜今日のピークパフォーマンス方程式＞

 【  自分だけのお宝を俳句で作る　】

───────────────────────────────────



■唐突ですが、来月から俳句を習おうと
　思っています。


　俳句は、

　「一定のルールのもと、
　　言葉を用いて情景、情感を結晶化させる」

　ことで生まれる「５・７・５の芸術」である、
　と考えているのです。
　

　わずか「５・７・５」の17文字の中に、
　繊細な感興や感慨をまるごと封じ込める、

　というのは、

　簡単なように見えて、その実、
　とてつもないことのように感じられます。



■実は、

　私の親戚のおばあちゃんが、
　句集を４冊ほど出していて、

　いま、84歳なのですが、
　すごく元気です。

　知的レベルが高く、頭も涼しく、
　好奇心も旺盛。



■もう何年も前になりますが、

　わたしが「仕事は、かけ算。」という本を
　上梓したときも、なんと赤線を引きながら
　読んでくれていたようです。


　84歳のおばあちゃんが、
　
　ビジネス書のどこに赤線を引いていたのか、
　いまだ、とっても気になっているのですが（笑）
　


■それは余談ですが、
　
　おばあちゃんが句集を見せてくれながら
　私に語ってくれた言葉で印象に残っているのは、


　「写真もいいけれども、

　　俳句は、17文字のうちに情景や情感と
　　いったものを封じ込めるものなのよ。


　　おじいちゃんとの思い出

　　（既に亡くなっていますが、全国各地いろんな所を
　　　一緒に旅行していました）

　　なんかも、その句を見返すだけで、
　　その時の景色と共に感情が甦ってくるんだよね」


　という言葉。

　いまも深く記憶に残っています。



■それを聞いて以来、いつか俳句を習おう、
　と機会を伺っていたのです。
　

　「俳句（短歌でもいいのですが）という形式に
　　のっとって、

　　日々、出会う景色やその感情を17文字に
　　結晶化し、封じ込める習慣」

　
　を日常生活の中に織り込むことができれば、
　それはとても豊かな人生になるのではないか、

　と思われます。


　それが何かの役に立つ、とか、
　そういった話ではなく。



■役に立つからやる、
　役に立たないからやらない、
　
　といった考えを乗り越えたところにある、


　「毎日の感興を17文字に封じ込め、
　　自分だけのお宝を制作していく」


　世知辛い世の中にあって、

　そんな習慣が、荒びがちな精神を和らげ、
　豊かで実りあるものにしてくれるのではないか、

　そんな期待もあるのです。



■ということで、

　「自分仕様の、美しい結晶を作り出す」

　ために、俳句の世界に入門することに
　いたしました。


　「美しい結晶作り」のセンスが
　もし身につけば、

　回りまわって、いずれ、がさつなこのメルマガの結晶も、
　もう少し、ましな（美しい）ものになるのでは、

　そんな風にも思っているのですが、
　どうでしょう。


　
■別にそういうことを狙って
　習いに行くわけではありませんが、

　（そういう姿勢には、意味がありません）


　もし、結果としてそのようにもなったら、
　私の座右の銘の一つである

　「無用の用」

　の精神の、

　面目躍如ということにもなるのかなあ、
　ということは、ふと思いました。
　


　今日も人生とビジネスを楽しみましょう！
