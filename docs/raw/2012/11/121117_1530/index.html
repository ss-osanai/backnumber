＜今日のピークパフォーマンス方程式＞

  【　何が何でも「初めの一回し」を成功させる　】

───────────────────────────────────



■昨日のメルマガでは

　「逆算思考でストレッチを行う」

　と題して、


　「できるか、できないかではない。
　　
　　やる、と決めるから、逆算思考が働き、
　　実現するためのアイディアがたくさん湧いてくる」


　というお話をいたしました。

　今日も続けます。



■実はこのような話は、

　勉強熱心な平成進化論の読者さんであれば
　幾度となく、聞かれてきた話ではなかろうか、
　
　と拝察いたします。


　けれども、残念なことには

　「やる、と決めた瞬間から人生が変わり始める、
　　というのは理屈では分かるけれども、

　　そこまでの踏ん切りが、なかなかつかない」

　と言われる方が非常に多いのです。



■短い人で数ヶ月、数年、
　長い人なら十数年、数十年（！）も、

　「いつかは、、、」

　と思いながら、踏ん切りがつかず、

　やってくる案件を順番にこなしながら
　毎日を生きていく生活を繰り返している、

　そんな方も少なくないように思われます。

　（といいつつ、起業前の私もそうでした）



■ところがあるとき、
　行動することを躊躇している私に

　「そんなことでいいのか！！！」

  と、

　そんな私のケツを蹴ってくれる人（＝Ｋさん）
　が現れたのです。

　

■不遇の時代を過ごしつつ、
　
　「このままでいいのか（いや、いいはずがない）」

  「いつかやらねば（でも、怖い）」

　と悶々としていた私は、Ｋさんに活を入れられ、
　遂に行動を起こすことを決意し、

　「60日間 メルマガ読者１万人獲得プロジェクト」

　を立ち上げることになりました。



■このときにやってよかった、と思うのは、

　その実現が如何に困難であるか、
　ということを知らずに（汗）

　「短期間でメルマガ読者１万人を獲得する」
　
　という荒唐無稽な目標を掲げたこと。


　しかも怖いもの知らずなことに、
　この目標を公言し、

　日々の活動状況や進捗状況をブログに
　記載し続けたこと。


　（これによって私のプロジェクトは
　　大勢の友人・知人の知るところとなりました

　　＝逃げられなくなりました）



■振り返ってみると

　「私のケツを蹴ってくれる人」

　が現れ、さらに

　「目標を公言することで自身を衆人環視状態に置き、
　　自ら、自身のケツを蹴った」

　この二つによって、

　「逃げられなくなって、行動し始め」

　「大量行動をするうちに徐々に成果が現れ
　　（＝読者さんが増え始め）楽しくなってきて」

　「行動することが中毒化し」


　遂には

　「１万人達成！」

　とまではいきませんでしたが、
　メルマガを創刊時には

　「6,000人ほどの読者さんからのスタート」

　を切ることができたのです。


　（もし積み上げ式の思考で行動していたら
　　とても到達できるような数字ではありませんでした

　　たぶん、せいぜい数十人～数百人程度で
　　終わっていたのではないでしょうか）



■この出来事は私にとっての「成功原体験」と
　なったわけですが、

　一度このプロセスを上手に回すことができれば、
　それ以降は、目指す具体的目標が変わるだけで、
　
　「本質的なところにおいては、
　　同じことを繰り返すだけ」

　になりました。



■だから、なんとかして
　（というか、なんとしてでも）

　まずは、このプロセスを一回しすることが大切、

　という確信を私は持っているのです。


　一度、この流れを体感できれば自信がつくし、

　やり方（方法論）もわかるので、
　あとは同じことを繰り返すだけ、

　となるからです。



■そういう意味で、

　３日間セミナーや情報発信セミナーの提供価値は、
　自分一人ではなかなか回しきれない、

　「初めの一回しのお手伝い」

　にあるのかもしれない、
　と考えることもあります。
　

　

　今日も人生とビジネスを楽しみましょう！
