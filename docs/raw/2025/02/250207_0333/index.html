■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------
　
　自分を劇的に生まれ変わらせる
　魔法があるんだわ。

　それは「私は○○だ」構文を使うこと。

　例えば「ワイは規格外や」と
　毎日繰り返し唱える。


　単なる自己暗示みたいやが、強烈な
　自己催眠（洗脳）で大きな効果を発揮する。

　驚くほどの速さで、現実が作り変わる。
　人生を書き換える強力な呪文やで。

　----------

　好奇心を駆動力に世界を
　広げていくと、

　やがて全てが結びつき、
　統合される時が訪れる。

　そのとき自身に固有、かつ
　独特の世界が立ち上がる。


　広げた後の調和からもたらされる
　「真の一貫性」と、

　広げることを諦めて手にする
　「偽の一貫性」は全くの別物。

　前者には信念や本気さが伴うが、
　後者には何もない。

　----------

　ありのままで生きたい気持ちは
　誰にでもある。

　ただし配慮も技術もなく、
　それをやってしまうと
　単なるわがままで迷惑な人。


　そこで身につけたいのが、
　摩擦を避けつつ受け入れてもらうための
　エレガントな処世の技術。

　自分らしさと周りとの調和の
　両立が極意。

　----------

　いついかなる場合でも、
　きれいごと「しか」言わん人間には
　要注意やで。

　美徳の仮面の下、
　何かが腐敗してるのよ。


　人間であれば誰もが持つ矛盾や
　欲望、不完全さを完全否定している者ほど、
　胡散臭いもんはない。

　本音を隠し、心の奥で悪意や欺瞞を
　育ててるケースを、まま見かける。

　----------

　ワイは長期にわたって

　「励まされる言葉、
　　心に留めておきたい言葉の丸写し」

　を地道に行ってた。

　これを続けると記憶に定着し、
　必要な時に取り出せるようになる。

　やがて態度や振る舞い全般に反映され、
　「遅効的に」未来の運命に良い影響が
　もたらされる。

　----------

　キャリアデザイン、とか
　格好ええこと言うてても、

　結局は居場所の問題に過ぎんのよ。

　能力磨くのも大事やが、その前に、
　働く環境と自分との相性の見極めが先。


　間違った場所で全力を尽くすのは最悪。
　報われん。

　正しい場所でテキトーに、
　の方がはるかにマシ。

　もちろん理想は
　「正しい場所で、全力で」やで。

　----------

　「好きで」「強みを発揮でき」
　「評価される」フィールドを探すのが
　キャリアデザインの本質。

　「嫌いで」「強みを出せず」
　「叱責しかされん場所」で仕事してたら、
　どんだけ優秀でも結果出せんだろ。


　一生懸命に働く、ではなく、
　一生懸命に働かずにはいられん場所を
　見つけるのが仕事選びの一丁目一番地。

　----------

　ワイはコピーライターやなかったけど、

　会社勤めの頃に数年、一緒に
　仕事する期間があったのよ。

　そのときハッキリ

　「言葉の持つ、とてつもない力」

　に気づいたの。


　当時、著名なコピーライターさんの
　著作やコピー名鑑を読みふけったな。

　あれも言語化力の土台を養う一助になったと、
　今にして思うわ。

　----------

　人を動かすには熱量、マジ大事やで。

　醒めた表情で、クールな言葉遣いを
　してても人は動かんよ。

　よく言われるけど、
　人を動かすにはまず自分から。

　----------

　年齢とともに
　好奇心の枯れていく人が少なくない。

　自分への期待が年々、
　小さくなるからやろな。


　反対に、何にでもなれると信じている
　子どもの好奇心の強さは異常。

　大人になっても
　あれが異常でなく平常運転になるよう、
　未来に希望を抱き続けたいもの。

　----------

　気合いでモチベーションをキープし続ける
　のは、人間である限り不可能。


　だから先回りして、
　
　テンション上がらん時にも
　やるべきことをやり続けられるシステムを、

　モチベーションの高い時に
　作っておくんだわ。


　そのタイミングでしか仕組みは作れない。

　自分のやる気など信じず、
　システムに身を委ねよ、いう話。

　----------

　「これを学ぶべき」

　と一般に言われてる能力を磨いても
　既にそこは激戦区ゆえ報われにくい。

　言わば「芋洗い状態」に
　自ら突っ込む形となる。


　その方面に適性ない人が
　かえってチャンスとなるのは、それが理由。

　流れに逆らい、
　誰も歩かぬ獣道を行くのは怖いけど、

　道を切り開けば
　市場を独占できるのよ。

　----------

　ADHDはADHDに向いた場所で戦うんやで。

　秩序、ルール、規則といった単語が
　聞こえたら、全力で逃げ出すこと。


　代わりに創造、革新、自由、
　開放、奔放といった言葉と
　親和性のある環境を求めればいい。

　そういう組織も探せばある。

　見つからなければ自ら起業すれば
　創造、開放、奔放の毎日よ。

　----------

　もがき苦しむ時期はあっていいんだわ。
　むしろ絶対肯定よ。

　そんな時期あってこそ
　弱者の気持ちを体感し、

　人間としての深みも生まれる。


　巨木は土中に立派な根を下ろしてるだろ。
　それと同じ。

　辛いとき苦しいときは
　根を深く、広く張り巡らせているんだと
　希望を胸に堪え忍ぶ。

　いつか必ず報われる。

　----------

　緊張や萎縮を強いられる環境からは、
　今すぐ逃げ出すべき。

　実力を発揮できんどころか、成長が止まり、
　なんなら能力の摩耗さえ起きるからな。


　のびのび、機嫌よく、屈託なく、
　オープンマインドでいられ、

　時を忘れて過ごせる場所こそ、
　あなたがいるべき居場所やで。

　----------

　人生は居場所探しの旅。

　石の上にも三年とか、
　置かれた場所で咲け、

　と言われてたのは過去、
　自由な移動が難しかったから。

　自らを慰めるための言葉が
　必要だったのよ。


　けど今となっては役割を終えた言説。

　そのとき意味があっても、
　時代は変わる。

　変化の潮流に対しては、
　適応進化を遂げんとな。

　----------

　「強みを活かせる場所探し」が人生。
　今いる環境に無理に合わせる必要はない。

　アドバンテージを最大限、
　発揮できる場所を探すのよ。


　弱みを補強し、全体を平均化するより、

　突出した能力を磨き上げ、勝ち切れる場所を
　見つける方が簡単でオモロイよ。

　キャリア観のコペルニクス的転回。

　----------

　ADHDと繊細さんの同一人物内コラボは
　挙動不審を生むけど、それがええんやで。

　ワイもそうやが「仕事ができない」タイプは
　観察力が異常発達する。


　普通の人が絶対に気づかん、
　微妙な心のアヤまで理解できるようになる。

　該当者は、この「圧倒的強み」を活かすのを
　キャリア戦略の中心に据えたらええよ。

　----------

　「人生は、にこにこ顔で生命がけ」

　京都大学の元総長、平澤興先生の名言。

　さらに

　「最善の努力とは、
　　どんな条件の下で続けても過労が出ず、
　　その中に笑いがあるようなもの」

　とも。

　渋い顔してイライラするんじゃなく、
　にこにこ楽しく、一生懸命やってこその、
　本物の努力やな。

　----------

　人間、40を過ぎたら「できんことはできん」
　と素直に認めるべき。

　40までは可能性を試して
　あれこれやるべきよ。

　その過程で自分の適性が見えてくる。


　それ以降は過去の取り組みの中で
　向いてたものに絞るんや。

　選択と集中の一点突破を狙うのが
　一番、確実性が高い。

　その先に道が開けるんよ。

　----------

　ADHDこそ一人ビジネス、向いとるで。

　気兼ねなく自分のペースで、
　ときに我々の最大の強みである

　「ハイパーフォーカス（過集中）」

　を発動できるから。


　常人にはとうてい及ばんものを
　作り上げられる。

　大事なのは他者評価で自分を測らんこと。

　自信さえキープできれば、
　人生いくらでも切り拓ける。

　----------

　ADHDは他者から否定され続け、
　自らマイナス評価を下している人が
　少なくない。

　けど、それはイカン。
　断固として拒絶するべき。

　自分の心は自分で守るんや。


　ただ「間違った場所」にいるだけなんよ。

　わざわざ拡大解釈して自分は全部ダメ、
　なんて思う必要は一切あらへんで。

　----------

　虐げられてるときこそ、
　不屈の魂が磨かれる。

　だから感謝するんやで。
　ぬるま湯の中で精神は鍛えられん。


　長い不遇の時代を過ごしてきたワイは
　その間、本を読み、人と会い、思索しながら

　「今に見とれや」

　と毎日、思い続けてたわ。

　ワイの土性骨は、
　逆境の中で鍛え上げられたわ。

　----------

　合わない環境に身を置くと
　悲惨なことになるからな。

　メンタルへのボディーブローが
　毎日続くんよ。


　それが嫌なら働く場所の選定を
　働き方より優先して取り組む必要がある。

　見つからなければ自分で
　作るしかない。

　そんなあなたに、みうらじゅんの

　『「ない仕事」の作り方』

　は、絶賛お勧めよ。

　----------

　多動傾向の人（ADHD）は
　マジメな話、選ばれし者やと思っとるよ。

　世間的には見下されたり、
　面倒がられる存在やけどな。


　でも我々には「豊かな発想力」に加えて
　「過集中」という伝家の宝刀があるんだわ。

　この宝刀は、抜く場所とタイミング次第で
　我々を大化けさせる。

　希望と可能性しかあらへんやん。

　----------

　仕事で成果を出したければ

　「仮説を立てて実行し、
　　結果を確認する」

　のプロセスを最速で回すのよ。

　チンタラやってても
　仮説検証のサイクルは絶対に回らんで。

　勢いが大事。


　全力ノリノリで没頭するから
　発想も湧いてくる。

　トロトロやってたら、
　アイデアなんか出るはずないから
　仮説も生まれない。

　----------

　やること詰め込みすぎて、
　可処分時間ゼロ。

　そこで初めて、自分の限界が
　見えるんよ。

　そして、もっと効率よくできないかと
　生産性に目が向くんだわ。


　現場の泥をかぶった経験もないまま、
　生産性を語る人物は100％、口だけよ。

　泥まみれになったことのある人間だけが
　説得力のある語りをできるのよ。

　----------

　努力せんでも結果を出せる
　「選ばれし天才」には何の興味も
　ないんだわ。

　崖っぷちから這い上がる
　復活劇こそ熱いんよ。

　日陰者でも努力と工夫で、
　必ず逆転できると信じとるし、

　ワイもその途上やで。


　動き、学び、逆境を克服できたときに
　本当の強さを証明できるんよ。

　お互い、反骨精神を大切に生きていこう。

　----------

　収入も貯金も売上も、
　カネの桁を上げることを意識するんやで。

　90万と100万、900万と1000万、
　9000万と1億、9億と10億、

　桁が変わるたびに意識も変わる。


　桁が切り上がると、
　扱うお金の感覚も変わるんよ。

　不思議やけど、そこからさらに
　稼げるようになる。

　嘘みたいな、ホンマの話やで。

　----------

　成果を上げる人間は
　例外なく言語化能力が高いんよ。

　言葉で自分をマネジメントする。

　他者を鼓舞し、動かし、
　未来を共創する。


　言葉は自己の可能性を開く鍵、
　他者を巻き込むエンジンよ。

　動物じゃないんだから、
　鳴き声や唸り声で意思の疎通はできん。

　人間にのみ使用を許された
　特権をフル活用するんやで。

　----------

　遅くとも30代までには
　名乗りを上げておきたいもの。

　その後の展開がまるで変わってくるからな。

　能力や評価、評判、人のつながりは
　複利で効いてくるから、

　あとから挽回するのは
　かなり大変なのよ。

　前倒しでやるべきことをやっておくと、
　中盤戦以降が本当に楽になる。

　今、20～30代のあなたへのアドバイス。

　----------

　ADHDに100％の自由時間を与えたら、

　時間感覚を失い、延々、
　やりたいことを集中して続ける。

　食事し忘れたり、
　他のタスクや義務の後回しの
　副産物もあるけど、

　ハマれば爆発的な成果を出せるのよ。

　----------

　不思議なもんで、
　金がないとき欲しかったものも、
　いつでも買えるとなると物欲がなくなる。

　無駄遣いしなくなって
　さらに金が貯まる。


　逆に金がないのに背伸びして
　カードでブランド品を買うような人は、

　いつまでたっても浪費から逃れられんし、
　金も貯まらんわ。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。

　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
