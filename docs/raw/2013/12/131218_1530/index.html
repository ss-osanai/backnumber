＜今日のピークパフォーマンス方程式＞

 【　学びの始まりは「師匠選び」から　】

───────────────────────────────────



■どんなことであっても

　「その道のプロ」

　から教わり、学ぶことが大事です。

　
　徒然草に、
　
　「何事も先達はあらまほしきことなり」
　
　と記されている通り。
　
　

■分かりやすくいえば、私（鮒谷）に

　「事業のグローバル展開をどうすればいいですか？」

　という質問をしてはいけない、ということ。

　「知りません」

　としか答えようがありませんから（笑）

　
　けれども案外、このような

　「お門違い」ならぬ「先達違い」

　のご質問をいただくことも多いもの。
　

　
■少し話がそれましたが、重ねて、

　「何を学ぶにしても、先生選びは最も重要ですよ」

　とお伝えしておきます。

　
　「自分が実現したいことを既に実現している人に

　　話を聞きに行くのが一番早い」

　のですから、

　「学びの始まりは師匠選び」

　からとなるのです。

　

■もちろん、私（鮒谷）もそのように心がけてきましたし、
　
　だからこそ、

　「師事した先生に（傾向として）近い形の人生」

　に近づいているようにも思われます。
　
　
　「自分の求めているもの」

　を手に入れた方のところに習いに行くのだから、
　
　当たり前といえば当たり前の話です。
　
　
　これからも、私は商売にかぎらず、
　何事においても、その道のプロから学び続けるつもり。
　
　
　
■まとめると、こういうことになるでしょう。
　
　
　１、まず、なりたい姿を既に体現している人
　　　（＝その道のプロ）を見つける
　
　２、その人に習いに行く
　　　（時間やコストをかけてでも）
　
　
　この二つが大切、ということ。
　
　
　これを実践していくと、やがて本当に

　「なりたい姿」

　に近づいていくものです。
　

　今日は短めに。
