　▼日刊を堅持
　▼日記とピークパフォーマンス方程式と編集後記は必ず入れる

　▼箇条書きにしない
　▼基準は決して下げない


　ということがあげられます。

　おります。
　
　※長すぎる、内容がなく事実の羅列になっている、といった点に
　　ついては、再考の余地が十二分にあると考えています。
　
　　あと、言い訳をしたり、忙しい、と表したりすることは不快感を

■今回、本当に読者の皆さまに教えられたのは、私が読者の考えを
　想定せずに、情報を垂れ流しにしていたと気付かされた事。

　毎日、配信しているのだからクオリティが高い日もあれば、低い
　日もあるのはしょうがないよな、的な甘えが、私の中にありま
　した。

　そうではなく、あくまで、読者さまにどのような情報を得て頂ける
　のか、を基準に考え、より多くのことをお伝えしなければなら
　ないのは当然のこと。


■ちょうど、今、出ている週刊誌「東洋経済」の特集に「お客の
　苦情が満足に変わるとき」という特集が組まれています。

　今回、私がアンケートをとらせていただいたのは、「お客様の苦情」
　ではなく、「平成進化論がよりよくなるための進化のご提案」
　でありますが、読んでいて考えさせられる点、大でした。


■顧客の声を聞く企業が成功する、という内容です。
　私の場合でいうと、顧客＝読者さま、です。

　本当をいうと、私が私の為に配信しているものだし、別に読者さまの声
　を聞く必要がないといえばない、という考えも成り立つのかもしれません。

　現に、友人のメルマガ発行者から、そのようなことをする必要はなく、
　時間の無駄であり、自由に配信すればよいのだ。
　しかもお金をもらっているわけではなく、無料で配信しているもの
　なのだから。嫌なら配信を止める人がいるだけだから、とも
　いわれました。


■しかしそんな中、あえて今回、アンケートをとって良かったと思うのは、
　ご意見をおっしゃって下さる方がたくさんいらっしゃったということ。
　厳しいご意見も含めて。そして厳しいご意見こそが参考になりました。
　
　言いにくいことであっただろうと思うのに。
　そして厳しい感想を書いてくださる方ほど、ご質問の第一項<このメルマガ
　のいいところ>の評価も非常に温かく、優しく、嬉しいものが多かった
　こと。

　これは本当に嬉しかったです。


■アンケートをとって、返信はこないけれども、どんどん登録者の
　方の数が減っていく、これがやはり恐ろしいことなのです。
　これをきっかけとしてそういう方の発生を未然に食い止めることが出来る
　ので良かったのです。


■無料だからこれでいいでしょ、ではなく、無料で、こんなことまで
　開示してくれるの、という驚きのクオリティを提供する決意を
　することで新しい様々なチャンスが開けてくるのだと思います。


■そんなわけで。

　昨日、今日とひたすら前進モードを一旦ストップして、立ち止まって
　周囲を見渡してみました。

　明日は、いよいよどのように改善していくかについての発表です。

　そして明後日からは再び、破竹の進軍を始めようと思っています。
　年末に、そして来年度に向けて。



【鮒谷・昨日の活動】


■以前からお知り合いだった食品会社の支店長さんとそちらの事務を
　なさっていらっしゃる方と、午前中～ランチの時間にかけて打ち合わせ。


■お昼からは平野先生と、来年以降のことについて打ち合わせ。
　平野先生の出版のスケジュールが遂に確定しました。
　
　
■１月２７日（火曜日）出版。
　ほぼOKとなった原稿を拝見させていただきました。思わず唸る、
　素晴らしい内容です。

　私は、平均的な人よりは本をたくさん読んできていると思いますし、
　目利きもできる自信があります。

　でも・・・これは凄い。

　出版社もすごい。手がける本が全てベストセラーになる、社長直々の
　販売プロジェクト。
　発売が待ち遠しいです。すぐに手元において、再読したくなる内容
　でした。

　また発売日が迫ってきたらご案内させて頂きます。


■夜は、今まで、ぜひ一度お会いしてみたいと思っていた、

　株式会社ハーストーリィの日野佳恵子社長とようやくご対面することが
　できました。
　ハーストーリィさまの忘年会ディナーパーティに出席。
　ご著書に掲載されている写真どおり、大変お美しい方でした。


■人と人とのつながりの大切さを私に教えてくれた人は何名かありますが、
　書かれた本を通して、そういったことを教えてくれた方が日野さん
　でしたので、以前から一度お会いしたいなと思っていました。


■お話を伺うと、やはり熱い思いと明確なビジョンを持っていらっしゃる
　ことに感動。
　ハーストーリィ様をご存じない方のために、ちょっとだけご説明
　すると、

　【ハーストーリィ様HP】　http://www.herstory.co.jp/

　日本全国で１０万人の女性のコミュニティを作られている会社です。
　１０数年でここまで来られました。女性のコミュニティを持たれている
　会社としては日本で最大級でしょう。

　なぜ、このようなビジネスを始められたのか、というご著書にも書か
　れていない話をお聞きし、創業するときに夢や想いは必ず必要なのだと
　思いを新たにしました。


■今日、お会いする際にはぜひとも、日野さんに聞きたい項目、という
　ことで５個ほど箇条書きで質問項目を手帳に並べていたのですが、
　その全部もお聞きすることが出来て、大満足。

　決して幸運が積み重なって、事業が回っているのではなく、様々な
　試行錯誤を繰り返して、成功されているのですね。
　すべての成功者にいえることですが。
