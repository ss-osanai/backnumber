■昨日は税理士さんとランチ＆打ち合わせ。
　まだまだ無知であることを気付かされました。


■その後、アタッカーズビジネススクールの同期受講生であり、
　「平成・進化論。」の読者でもある方と、新宿でお会いいたしました。
　
　まだ若いのに（大学を卒業したばかり）、その物怖じせず堂々とした
　ところと、未来を見据えて行動しているところがすごいですね。

　大学生といい、大学を卒業して数年、という人といい、最近出会う若手
　は非常に元気で、勉強熱心な人が多いようです。

　不況下に大学入学し、そして危機意識を持って、学生生活に取り組んで
　きているからなのでしょうか。

　こんな若い人でも、自己への教育投資として半期で数十万という金額を
　投じているのですから、見習わないといけませんね。

　　
■その後、友人の結婚披露宴に。

　新郎新婦は東大法学部と早稲田政経を昨年に卒業したばかり。
　高校時代から付き合ってたとの事で実に7年越しのお付き合いらしい。　

　二人ともと親交があるのですが、いずれも美男美女。
　勉強熱心で、コミュニケーション能力にも長けています。


■まだ若いとはいえ二人の人望ゆえか、たくさんの人が披露宴に参加
　されていました。

　ご両人とも日本を代表する企業におつとめで、やがて日本の将来を
　担っていくのでしょう。そんなポテンシャルを持っていて、既に
　大器の片鱗を表しつつある二人です。
　
　（このメルマガを読んでいるようですので、ちょっと褒めておきます　笑）


■二次会、三次会といって、帰宅は深夜2時半頃。

　めったにめったに二日酔いになることはないんですが、さすがに
　ちょっと酔いが残るくらいに飲んでしまいました。
　めでたい席なので、最初からそのつもりで参加してたんですが。

　カラオケの歌いすぎでのどの調子が悪いです。

 （From Publishers Weekly）
 
　『Liar’s Poker』や『The New New Thing』の著者ルイスが、メジャーリーグ
　球団の中でも選手の年俸が最低だったオークランド・アスレチックスが
　どのようにして2002年の輝かしい戦績を残したのかを分析した1冊。
　ボストン・レッドソックスやニューヨーク・ヤンキースを見てわかるように、
　選手の報酬は世間に公表されている。

　野球関係者やファンは、才能のある選手はそれに見合う最高の報酬を手
　にする権利があり、実際に手にしていると思っていることだろう。
　しかし著者ルイスは、あまり知られていない数字や統計こそ重要だと
　主張する。著者はビル・ジェームズを取り上げて、彼が毎年発行している
　データ分析『Baseball Abstract』や、試合に関わるその他の数学的な解析
　について論じている。

　驚いたことに、アスレチックスのジェネラル・マネジャーで、かつては
　選手でもあったビリー・ビーンを除いて、この研究に着目したマネジャーは
　ほとんどいなかったのだ。 

  ルイスによれば「2002年のシーズンが始まった当初、少ない元手で勝利を
　収めていたオークランド・アスレチックスは、メジャーリーグ・コミッショ
　ナーのバド・シーリング、つまりメジャーリーグそのものを困惑させた」
　という。
　
　チームの勝利は絶妙な運と選手の選択、そしてビーンの一流の交渉スキルに
　よってもたらされたものだった。ビーンは他球団でどの選手がトレードされるか
　を察知し、そのトレードがアスレチックスとは関わりのないものであっても、
　なんとかそこに入り込んだ。

　ルイスによれば、「トローリング」と彼が呼んでいるこの方法で、
　「絶え間なく人としゃべることによって、トレードを成功させるために重要
　な各種の情報に目を光らせている」のだ。

　これは、よい選手を探し出し、契約にまでもっていくビーンの類まれな能力
　に焦点を当て、ビーンの生き方そのものをまとめた1冊だ。しかもルイスの
　叙述的な文体によって、ビーンやその他の野球選手たちは、生き生きと描き出
　されている。

（ここまで）


■この本。一言で言えば、

　【弱小球団】

　（＝選手獲得の予算を潤沢に与えられることなく、ごく限られた予算の中で
　　　チーム運営をしなければならない球団
　　＝オークランド・アスレチックス）

　のＧＭが最少予算の中でデータを駆使して、多大な予算を持っている
　有力球団に打ち克つ（＝最大の効率を生み出すこと）

　ことを目指して、さまざまに考え、行動し、実際に結果を生み出すまで
　の軌跡を詳細に描かれた本。


■【ピークパフォーマンスとはなんぞや】を常に考えている私にとって
　きわめて興味深い研究対象であります。

　野球の本というよりは、捉え方によっては秀逸なビジネス書に変貌する
　でしょう。この考え方は、単に野球というジャンルにとどまらず、全ての
　ビジネスに通用する考え方だと思われます。


■いくつかの貴重な示唆をこの本から得ることができたのですが、


　▼異端はチャンス（無条件に信じられている常識こそを疑え。
　　たとえ異端と呼ばれ、笑われようとも。異端が勝利することがある）

　▼数学的、確率統計的な手法を取り入れることで投資効率を高める
　　ことができる（頭を使え）

　▼常識を覆す一見非常識な（それでいて実は緻密に計算された、常識
　　的な思考プロセスと実践）が大きな成功をもたらす


　などについて、改めて思いを巡らせました。


■「異端はチャンス」、ということですが、それまでの球界では
　ホームランを打つ選手が素晴らしい、チャンスに強いといわれている
　野球選手も有用だ。足が早い選手は重宝され、守備力のある選手は
　市場価値が高い。よって無条件で高額の年俸でやり取りされる。

　などの球界の常識に、すべてデータを緻密に分析して、真っ向から臨み、
　選手の能力のなかで、チームの勝敗にとってキーとなる要因は、一見
　上述のような派手に見える能力よりも、【出塁率と長打力】であると
　いう分析を行い、他の球団からの嘲笑を浴びながら、よそがとらない
　【地味な埋もれた、それでいて、結果に貢献する選手】【無名の実力派
　選手】をどんどんドラフトで安価に採用していったのです。


■この結果、すぐにプレーオフに４年連続進出するという素晴らしい結果
　を成し遂げたのですが、上述のようにここで、異端であることは
　チャンスである、というヒントを見出すことができると思います。

　でも、実は異端ではなく、【誰も検証しなかった非常識な常識】であった
　ということだけなのですが。

　世の中にもこういう埋もれた、それでいてすぐに結果を出すことの
　できる手法が眠っているはずです。

　常に、投資効率を最大化しよう、とあらゆる事象に仮説をたてること
　で、常識を覆す方法論を掘り下げ、実践することができるのでしょう。


■こうやってオークランド・アスレチックスはだれも、検証しなかった
　選手採用の投資効率を数学的に分析することで一見、球界の異端に見え、
　その実、科学的な検証を経て、他球団と自球団の採用基準のギャップ
　をついて安価に無名の実力派選手を採用することができたのです。


■こういう考え方は、あなたが熾烈な戦いを繰り広げている業界の
　中に採用することで、あっと今に市場をひっくり返す可能性がある
　ことを教えてくれますね。

　書いてて、いま、思い出したのですが、

　【今日のお勧め本　パラダイムの魔力】
　 http://www.amazon.co.jp/exec/obidos/ASIN/482274020X/2ndstagejp-22
　※私の今まで読んだ本の中で10傑に入る本です。今まで、この
　　メルマガでご紹介したことはなかったのですが、この本を読んでまさに
　「パラダイムシフト」が起こることが実感できると思います。

　と相通ずるところがありますね。


■『パラダイムの魔力』はパラダイムについての、一般的な考え方。
　それを実務レベル、分かりやすく解説してくれたのが『マネー・ボール』
　
　という捉え方もできるな、と、ふと思ったのですが、あながち
　間違いでもないように思います。


　【今日のお勧め本　マネー・ボール】
　 http://www.amazon.co.jp/exec/obidos/ASIN/4270000120/2ndstagejp-22
