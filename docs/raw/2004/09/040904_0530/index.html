■長期的な視野で人生を考えようと思っている人には、まず聞いていただき
　たいテープです。

　「将来計画を長く設定していればいる人ほど、社会的に成功している」

　逆に、
　
　「将来計画が短い人ほど、社会的には困難を極めている人が多い」

　と聞いたことがあります。

■確かに10年20年計画をたてて、その計画に基づいて行動する人のほうが、
　一日一日を生きるのが精一杯の人に比べて、社会的に成功する確率が高く
　なるのは自明な気もします。

■そんな当たり前のことを当たり前と気付かせてくれ、自らの体験を通して、
　それが正しかったことをおしえてくれる。そんな熊谷さんのテープです。

───────────────────────────────────


■さっきまで、ついついビデオに見入ってしまいました。

　家にテレビがない、と書いたことがありますが、事務所にはアンテナの
　つながっていないテレビがあります。

　（アンテナつながってないので、やっぱり一般放送を見ることはできない
　のですが）

　セミナービデオとか、そういったものを見る用においてます。


■事務所にはコンサートのビデオなんかもおいてて、たまに見たりします。

　自らを鼓舞したいときなどに、仕事をしながら後ろで流すのです。

　コンサートビデオなど見ていると、やはりリアルで10000人とか20000人とか
　を相手にするというのは、それだけのパワーを浴びて、さらにそれを跳ね返す
　だけの気力体力精神力が必要なんだろうな、とつくづく感じます。

　コンサート映像で見て、文字通り、そのプレッシャー（悪い意味ではない
　ですよ）を実感するのです。


■私もこの「平成・進化論。」を通して、毎日、数万人の方々に向けて
　ライブ、コンサートをやっているつもりです。

　体調が悪いとき、精神的なプレッシャーを浴びたとき、そんなときに
　今まで気力で押して均衡を保っていた圧力を一気に受けてしまいそうに
　なるときがあります。


■しかし、どんな状態に置かれていても、それを跳ね返していくのが
　真のプロフェッショナル。

　でも、なかなかそういうわけにもいかず、ついつい、その時々の精神状況
　が文章に表れることもあるのは、まだまだ修練が足りない証。。。
　（そんなところがよい、なんていう奇特な方もいらっしゃるのですが）


■いつもそんなことを考えながらメールマガジンを配信しているのですが、
　どのくらいの人数がネット回線の向こう側にいらっしゃるのか、時として
　実感を持てなくなる（というか忘れてしまう）のです。

　なので、たまにそれを自らに再確認させるためにもビデオを見ます。


■そんなわけで、今日のビデオはシャ乱Qのライブビデオ Vol.1 真剣勝負。

　【今日のお勧めビデオ　シャ乱Qライブビデオ Vol.1 真剣勝負】
　http //www.amazon.co.jp/exec/obidos/ASIN/B00005HU2Y/2ndstagejp-22

　（⇒まぐまぐとAmazonの契約関係？により、「http」と「//」の間に「:」が
　　　自動で削除されるように設定されるようになったようですので、
　　　ご興味のある方は上記の間に「:」を追加してご覧下さいね）

　このビデオ、今から８年ほど前のものなのですが。

■シャ乱Qのボーカル、つんく、はじめメンバーのみんなも大志を抱いて？
　東京に出てきて、武道館での初コンサート。
　そのときの様子を録画取りしたものなんですね。

　映像を見てみてもやっぱりみんな、若い！し、このまま一発屋で終わるか
　どうかは、皆様（観客）の応援次第、みたいなこともつんくは言ってました。

　デビュー第一発目の東京コンサートだったようです。


■このコンサートの中での、最後の曲が「上・京・物・語」。

　この曲があったから、解散せずにここまで来れた、なんていってましたが、
　私もこの曲、スキなんですよね。


■昨年、大阪から東京に出てきて、最初にこの曲を聴いたときになんだか
　ぐっとくるものがあったのを覚えています。
　（カラオケに行ったときに誰かが歌ってたのですが。前田君だったかな？）

　それから、１年強。
　なんとかおまんま、食べさせていただけるくらいにはなりました。


■おそらくつんくも、そのときから志、のようなものがあったのでしょうね。

　曲と曲の合間の語りにも、そのような思いを込めた言葉を随所に挟んで
　いましたから。（今、聞くからそのように感じるのかもしれませんが）

　今では押しも押されぬ、モーニング娘。はじめ、さまざまなユニットの
　プロデュースを手がけるプロデューサーですが、なるべくしてなった、
　のかもしれません。


■志と行動力。
　
　やはりいずれかの世界で頭角を現す人というのは、現れた当初から、
　なんらかの匂いを放っているもののようです。

　そんなことを、このビデオをみながら考えてしまいました。
　

■私も初心を忘れず、目的を確認して、日々を過ごしていかなければと
　改めて反省しました。


■しかし、このビデオ、96年発売のもので、96年1月のコンサートのもの
　なのですが、既にそれから8年。

　曲を聴いていると、脳の襞（ひだ）に折々の感情が埋め込まれているのかな、
　なんて考えてしまうことがあります。


■久しぶりに、シングルベッド、とかズルい女、とかそんな当時、ヒット
　した曲を聞いていると、そのときの感情？なり時代の空気みたいなもの
　であったりがフラッシュバックのように甦ってくるんですよね。

　とっても不思議です。
　たまにこういうのもいいかもしれませんね。そうして意味なく涙を
　流したりして。
　
　こういうのがカタルシスになってるのかな。

＜今日のピークパフォーマンス方程式。＞

　【　　苦しみ⇒経験⇒知識化⇒方程式。自分だけの方程式を持つ。　　】

───────────────────────────────────


■ベンチャーセーフネット会長の関口房朗氏の本を読みました。

　タイトルはそのものズバリ、「金持学」。
　サブタイトルには年収3000万円以上をめざすアナタのための成功哲学、
　
　ということでその度肝を抜く表紙とタイトルに、おもわず書店で
　手に取り、そのままレジへ。

　【今日のお勧め本　金持学】
 　http //www.amazon.co.jp/exec/obidos/ASIN/4796638253/2ndstagejp-22
　こちらも「:」を追加してご覧下さい。


■生き方としては毀誉褒貶ある方だと思いますし、拒否反応を示す人
　もたくさんあるかもしれません。

　しかし私は直接お会いしたことがないので、氏についてのコメントを
　することはしないし、できません。

　ただ、やはり一代でここまでのことをやった、というのは単純に
　凄いと思うし、その中で学べるところは学びたい、というつもりで
　本を読み勧めました。


■その中で印象に残った文章がいくつかあったのですが、そのうちの一節を。


　「経営者として、または馬主として、私は数多くの修羅場をくぐってきた。

　　苦しみは経験となり、経験は知識化され、そのうちのいくつか自分の
　　中で方程式にまで発展している」（p.115）


■「方程式」という言葉にぴくっと反応してしまいました。

　私がこのメールマガジンを書くに当たって、様々な経験や学びを通して、
　普遍的なピークパフォーマンス（最高の結果）を発揮するための原理原則を
　記録にとどめよう、そんな意図から始めたのが「今日のピークパフォーマンス
　方程式」のコーナーです。


■自分が学んだことを自分の言葉で言語化したい、と思って書き始めたの
　です。


■なので、この

　▼「苦しみは経験となり」（暗黙知。言語化できない知恵）

　▼「経験は知識化され」（経験がどんどんあつまってきて知識の断片となる）

　▼「そのうちのいくつかは自分の中で方程式にまで発展している」
　　（いつでも成り立つ普遍的な原理＝方程式に）

　という理解を私なりにして、なるほど、と思った次第です。


■苦しみ⇒経験⇒知識化⇒方程式、ということで自分のなかで方程式を
　たくさん持とうとすれば、苦しみを求めなければならない。
　あるいはできるだけたくさんの経験をしなければならない。

　経験の数に比例して、知識の断片が増えてきて、さまざまに結果をだすため
　の方程式が自分の中に蓄積されていく、といった感じでしょうか。


■私も、このメールマガジンを配信することにより、様々な体験をできるだけ
　重ねるようにつとめ、それを経験し、知識の断片を積み重ね、方程式という
　形で言語化することをつとめてきました。

　こういう作業をすることで見えてくるものがたくさんあるので、極端な
　ことをいえば、たとえ読者様がゼロであってでも、やっておくべき価値の
　あることだったと今更ながら思っています。


■最近は、ブログがブームで、いとも簡単に情報発信をすることができる
　時代です。そして自分のコンテンツをマネジメントすることも簡単に
　できるツールがそろっています。

　（※勿論、メールマガジンなんかでもよいのですが）

　
■来訪者が多かろうが、少なかろうが、自分の経験を知識化し、方程式として
　蓄えておく、というのは将来にわたっての財産になるとおもっています。

　まだ自分の情報発信手段を持たない方、あるいは今までの経験の暗黙知を
　形式知化していく作業をしてこられなかった方に、ぜひともお勧めしたい
　とおもいます。


■本当に、いますぐ、やることをお勧めします。
　きっと１年後には世界ががらっと変わります。

　私の友人たちにも、勧めているのですが、感謝の声？をたくさんもらって
　います。それだけ強力な作業なのでしょう。
　自分の体験を書き出し、それを自分の言葉で方程式化する作業というのは。


■そうやって、個々の出来事をまとめていくうちに、もっと深いところを
　流れる本質的なことに気付くことがあります


■それこそが自分の言葉で言語化された原理原則であり、既存の成功哲学
　（これも凄い威力を発揮するものだとおもっていますし、今まで手にとった
　事もない方は学ばれると良いかもしれません）が、さらにより、強力な
　自分独自の成功哲学が形成されてくるのです。


■作り上げたビジネス、築き上げた資産、人的なネットワーク、その他を
　すべて一瞬にして失ってしまったとしても、自分なりのピークパフォーマンス
　を生み出す方程式を持っていれば、再起は早いとおもいます。


■実際に関口氏も以前に設立して東証２部にまでもっていった会社を
　クーデターで乗っ取られてしまった経験があります。（さまざまにメディアを
　にぎわせた事件ですので、ご存知の方もおおいとおもいます）

　しかし、それから７年ほどで不死鳥のように甦りました。

　それも関口氏なりの【方程式】を確立していたなればこそ、のことなの
　でしょう。


■苦しみ⇒経験⇒知識化⇒方程式。自分だけの方程式を持つ。

　としっかりメモしてしまいました。


■関口さん、ご存知の方がいらっしゃいましたら、どなたかご紹介ください。
　ぜひ、一度、お会いしたい方です。
　自分にないものをお持ちの方ですのでたくさんの学びと気付きとインパクト！
　を得られるのではと思ってます。

　
■こうやっていっておけば、どなたかつないで下さる方が現れる可能性が
　あることがわかってきたので、とりあえずここで言っておきますね。

　いわなければ可能性はゼロなので。
　
　関口さんとつながりのある方、どうぞ宜しくお願いいたします！
