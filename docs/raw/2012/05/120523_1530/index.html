■こんにちは、鮒谷です。

　先日の３日間セミナーに来てくれたＦさんのブログです。

　（お客さまなので「さん」づけ。笑
　　セミナー当日に、後輩と知りました、、）



■例によって？ブログを
　書いていただいているのですが、

　短いながら、なるほどとおもった内容だったので
　ご紹介いたします。

　
　＜日常を非日常に、非日常を日常に＞
　
　http://ameblo.jp/22-yama/entry-11254366240.html


　※お仕事で世界各地に超巨大な建造物を
　　作られてます。

　　小学生のとき、図鑑の写真を見ては
　　心をときめかせていた、
　
　　そんなことを思い出しました。


　　将来、偉くなったら
　　ぜひ内部を見学させてください！

　　目標は言葉にして残しておくことに
　　しているので、一応書いておきます（笑）


　

■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　人生を大切にしない人　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　人生を大切にしない人　】

───────────────────────────────────

　

■一昨日、昨日と、

　「自分の内にある『状況の解釈装置』を
　　グレードアップすることで人生の質が変わる」

　というお話をいたしました。

　今日も続けます。



■「状況の解釈装置」とは、

　ある環境に置かれたとき、
　より学びを深められる、よりやる気を出せる、

　そんな解釈を生み出してくれる

　「内的な（自分の内側にある）システム」

　です。


　
■では具体的に、

　そのシステムはどのようにアップグレード、
　あるいはバージョンアップすることができるのか、

　皆さんも興味があると思います。


　通常のソフトウェアであれば、
　CD-ROMで（古い！？）、

　あるいはオンラインで最新版を
　ダウンロードすることができるわけですが、

　この内的なシステムを最新版にするには
　どのような手順を踏めば良いのでしょう。



■私は、

　自分の内側で稼働している「解釈システム」は

　▼言葉で構成されており

　したがって

　▼バージョンアップも言葉によって行われる

　と考えています。


　
■だから、

　自分にとって好ましくない（＝マイナスの）
　結果をもたらすシステム（のバグ）を、

　「プラスの結果を生み出す、質の高い言葉」

　によって
　
　「上書きインストール」

　するようなイメージで、
　バージョンアップは完了されるのです。



■出来るだけ質の高い言葉を、出来るだけ頻繁に、

　「上書きインストール」

　し続けることによって、
　内的システムは常に最新版でありつづけ、

　「軽くてメモリを消費せず、しかも高パフォーマンスを
　　発揮してくれるシステム」

　へと、どんどん進化していくことでしょう。



■そしてそのシステムのバージョンアップとともに
　明らかに人生が好転していくに違いありません。

　「どこまでいっても言葉」

　なのです。


　だからこそ私は、

　「言葉を大切にしない人は、
　　人生を大切にしない人」

　だと思っています。



　今日も人生とビジネスを楽しみましょう！
