　熟読するも読み流すも、自由な形でご活用下さい。

＜今日のピークパフォーマンス方程式＞

  【　「何かを売るために書く」の大間違い　】

───────────────────────────────────



■昨日のメルマガでは、

　「『せめて、もう少しマシな文章』を
　　　　　　　　　　　書けるようになるために」

　というタイトルで、

　「文章表現技法」
　（というほどに、大袈裟なことは書いてませんが）

　について記しました。



■今日は、

　「技法」

　ではなく

　「文章を書くに際しての心構え」

　について書こうと思います。



■自分の話で恐縮ですが、私は

　「毎日、文章を書きたくて書きたくて仕方ない」

　そんな気持ちでメルマガを
　お届けしています。
　
　義務感で書いている、
　とかそんなことは一切ありません。

　（義務感だけで、この過酷な行（ぎょう）は
　　続けられません）



■なるほど、たしかに稚拙な文章であることは
　素直に認めますが、

　それでも内側からほとばしる、
　

　「凄いことに気づいたよ」

　「こんな視点で世界を見てみたらどうでしょう」

　「こうすれば生きるのがラクになりました！」

　「教えてもらったことを実行したら
　　こんな結果が出た」

　「とっても面白い本を見つけたよ」


　といった、

　日々の感動とか発見とか興奮とか驚きとか、

　そういったものを言葉に載せて
　お伝えしてきたのです。



■また、

　そのような気持ちから筆をとったものでなければ
　目の肥えた受信者には伝わらない、

　とも思っています。


　熱い気持ちを持って書いてすら
　伝わりにくいのですから、

　そもそも伝えたいという気持ち自体が
　弱ければ伝わるはずがありません。



■経営者やビジネスパーソンであるならば、

　「何かを売るために書く」

　ということは、しばしばあるでしょう。


　けれども、あるときから私は

　「それは、そもそも間違っている」

　と思うようになりました。



■どうしても知ってもらいたいことがある！
　どうしてもお伝えしたいことがある！

　それを知ってもらうためには、
　目の前にいる人に話をするしかない。

　目の前にいない人には
　書いて届けるしかない。


　だから話す！
　だから書く！

　そうしたら結果として売れました。
　

　この流れが正しい姿だろうと思うのです。



■したがって、

　別に情熱を傾けて話すこともなく、
　書くこともない。


　けれども、

　本でもメルマガでも広告でも、
　売れてカネになりそうだから書いてみよう、

　といった考えは本末転倒です。


　手にとってしまったり、
　目にしてしまった人の迷惑を考えよ、

　ということ。



■商品であれ、サービスであれ、

　自分が提供するもので絶対の自信を持っている
　ものについては必ず、


　「自分の商品に徹底したこだわりがあり、

　　だからこそ、どうしても伝えたいことが
　　次々と湧きだしてくる」

　
　ということになるはずです。
　


■そのようなものが湧いてこないような
　商品はそもそも売らないでもらいたい、

　これが消費者の率直な気持ちのはず。

　私なら、そんなものを売りつけられたくは
　ありません。



■「うまく書く」

　の前に、

　「どうしても書きたい」

　という強い情熱があって、はじめて伝わる文章が
　書けるのではないでしょうか。



■また、お客さまもバカではありません。

　そういう情熱が込められた文章だと
　心で感じるから購買に至るのです。

　
　まず自らが感動、興奮することによって、
　お客さまにもまた、感動、興奮が伝染するのです。


　自社商品やサービスに思い入れがないのに、
　お客さまが行動してくれるはずがありません。



■どれだけこの思いが伝わっているか
　分かりませんが、
　
　すくなくとも私は、


　「感動する、感じ入る、感服する、感銘を受ける
　　心を打たれる、心を捉えられる、心に沁みる、

　　興奮する、胸に響く、胸がいっぱいになる、
　　舌を巻く、

　　わくわくする、どきどきする、ぐっとくる、
　　きゅんとする、どきりとする」

　
　という感情が毎日起きるような、
　そんな生活を送りたいと思っていますし、

　そのための環境を自らに用意しています。
　


■だからこそ、このメルマガを通して、

　毎日、毎日、尽きることなく感動のお裾分けが
　できているのだと思っています。


　（文章化した瞬間に、この感動が相当程度、
　　劣化してしまっているのは重々承知しておりますが）
　　
　

　今日も人生とビジネスを楽しみましょう！
