7396号 「異世界転生」の可能性を高めてくれる人

■まだ見ぬ、新しい世界は、新しい人や本、
　場所や対象からもたらされます。

　ただ、一般に、
　私たちは一人ではなかなか動けず、

　もし動けたとしても、新たな人や
　事物に巡り合う機会も限られるもの。



■だからこそ、

　新しい世界に通じる道を
　一気に押し開いてくれる、

　そんな人との縁に巡り会えたら、
　そこに感謝し、

　そのご縁を大切にすべきでしょう。



■幸いなことに私（鮒谷）は、

　新しい世界を開いてもらう機会を
　早い時期に完全に、ある意味、

　「丸投げ」

　することができました。



■そんな出会いに恵まれたのは
　当時の私にとって僥倖でした。


　その頃は自身の可処分時間に対し、

　お誘いいただく機会や、勧められる本、
　さらには勉強会などについて、すべて、

　「ハイか、イエスか、喜んで」

　の精神で可能な限り、

　勧められた人に会い、本を読み、
　場所を訪れ、、、

　といったことを繰り返していて、
　振り返る余裕もありませんでしたが、

　今になって考えると、
　これは本当に幸せでした。



■自分ひとりでは、

　とうてい拡張しえなかった世界を
　大きく広げて頂いたとともに、

　それを通して新たな世界の捉え方、
　補助線の引き方について、

　学ばせてもらえる貴重な機会となりました。



■そう考えると、

　一つ一つの出会いは大切ですが、
　中でも

　【新しい世界との出会いを
　　拡張してくれる人】

　との出会いは
　さらに大事にしたいもの。



■そんな人と出会えれば、
　その人とのご縁を喜ぶべきですし、

　その方を通じて「異世界転生」できる
　可能性も高まります。

　今日は短めに。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■新しい世界は。新しい人や本や場所から
　　もたらされる。

　■そんな世界の扉を押し開いてくれる人との
　　出会いがあれば、

　　そのご縁を感謝しつつ、勧められるままに
　　新たな世界を切り開き続けたいもの。

　■そのような毎日を繰り返すうち、
　　「異世界転生」の可能性も高まるのだ。
