■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------

　人生は意思決定の連続。

　でも95%は
　「無意識に刷り込まれた思考回路」で
　自動的に決めとるんよ。

　だから注意深く言葉を選び、
　回路を書き換えないと、

　延々と同じ決断パターンを繰り返す。


　良いパターンならええけど、
　反対なら最悪よ。

　下手すりゃ死ぬまで続くからな。

　バグを見つけては上書き更新するんやで。

　----------

　ノリと勢いで動いているうちに誰一人、
　気づいていない勝ち筋が見えてくる。

　そのルートに乗れば急加速する理由を

　（1）強固なロジック
　（2）裏付けとなる経験

　を通して理解する。


　完勝の確信は、異常なる感情高揚を
　もたらすから、振り切った

　「あたおか（あたまおかしい）行動」

　を続けられる。

　が、他の人には意味不明、理解不能。

　----------

　経営者やビジネスパーソンにとって

　【異常、異端、極端、変態、
　　破天荒、過激、狂気、錯乱、
　　型破り、常軌を逸した、振り切ってる、
　　頭おかしい、イカれてる】

　は褒め言葉。


　平均的な振る舞いからは、
　平均的な結果しか出ないのよ。

　圧巻の成果は

　「普通、平凡、退屈」とは遠く離れた
　場所からもたらされる。

　----------

　我々は幸せになるために生きとるのよ。

　その結果、後ろにできた道が人生履歴。

　キャリアは未来に築くものではなく、
　過去から連綿と紡がれてきた歩み。


　金、肩書、立場は
　ゴールに至る道のりの副産物に過ぎん。

　それ自体を求めて生きるのは
　目的と手段の混同、

　と気づいたときが
　集団催眠から目覚めるときやで。

　----------

　「べき」「ねばならない」から
　解放された瞬間、

　人生は劇的に変わるんよ。


　こんな抑圧的な言葉を使っている限り、
　本物の情熱は生まれんで。

　「せずにいられん」の欲求を爆発させて
　フルスイングした人だけが、

　規格外のホームランを打てるのよ。

　内なる情熱を殺す言葉は
　徹底撲滅、完全追放や。

　----------

　「思い込み、錯覚、妄想、勘違い」
　で我々の行動は規定されるんよ。

　であれば、思い込みの方向を
　意図的に選べばいい。


　後ろ向きの妄想で苦しむより、
　前向きな錯覚で突っ走る。

　どうせ何かを思い込むのが人間。

　ならば、どデカイ勘違い一択よ。

　その圧倒的な勘違いが
　人生を切り開く原動力になるんだわ。

　----------

　言葉の限界が思考の限界を
　決めるんよ。

　逆に言えば、

　新しい言葉との出会いが、
　新たな世界を切り開くきっかけにもなる。

　言葉を変えれば思考が変わり、
　行動が変わり、人生が変わる。


　だから意識して

　未だ出会ったことのない言葉を探し、
　選び、磨けば成長できるのよ。

　言葉なくして成長なし。成果もなし。

　----------

　「売らなくても売れていくシステム」

　を作るんやで。

　頑張って売るのは、もう卒業。


　一度ベルトコンベアに乗ったら、
　いつの間にか購入に至ってた、
　が理想の流れ。

　ざっくりとでも全体の仕組みを作った後、

　ボトルネックを潰しては改善、
　を繰り返すから、

　営業コストがかからなくなり
　大儲けできるんよ。

　----------

　カネを稼ぐ力がないと、

　安易な
　（リスクの高い、
　　場合によっては法に触れる）
　カネ稼ぎにハマって

　抜け出せなくなることすら
　あるんだわ。


　自分だけでなく周囲にも
　迷惑をかけてしまう、まである。

　「カネ儲けは汚い」

　と思い込むのは危険。

　正しくカネを稼ぐ力こそが、
　自分と周りを救う現実があるんだわ。

　----------

　没頭できる対象を見つけて、
　完全燃焼したい。

　みんなの願いはこれなんよ。

　けど日常生活に追われ、
　忘れたり諦めてる人が多いよな。

　本心はごまかせない。
　そこに巨大な潜在ニーズがあるんだわ。

　----------

　このご時世、
　会社強制の長時間労働はNG。

　それゆえ短期で能力を
　急激に高めたいなら、

　自分で精神や環境を整えるしかない。


　やるやらないは自由意志やが、
　やってる奴は見えないところで
　やっている。

　そして能力差が目に見える形で
　顕在化したときには、

　すでに取り返しのつかない
　巨大な差になっている。

　----------

　読書はすぐに役立たんけど、

　思いもよらぬ形で人生を豊かにして
　くれることがあるんだわ。


　ワイは読書を続けた結果、
　人見知りを克服できたんよ。

　濫読を通じ、
　どんな人とも話せる多様多彩な、
　膨大な知識を得たんだわ。

　読書の副産物が、人生を変える
　大きな力になることがあるんよな。

　----------

　相手がどれほど年上でも、
　立派な肩書があっても、

　マウンティングしてくる人間とは
　関係を続けるべきでないよ。


　彼らは自分が優位に立ち、得をしたり、
　気持ちよくなることしか考えとらんから、

　ワイらの貴重な時間やエネルギーを
　消耗させられるだけとなる。

　拒絶一択。

　----------

　有益コンテンツやオトク情報なんか、
　今どき無料でいくらでも手に入る。

　でも個人の魅力ある物語は、
　固有性があるから
　「そこでしか読めん」のよ。

　物語作家になったつもりで発信せよ、
　いうこと。

　----------

　「サービス2割、ストーリー8割」

　が正解やで。

　多くの人は比率が逆転してるか、
　なんなら

　「サービス10割、ストーリー0割」

　すらあるからな。
　それだと感情移入できないんだわ。


　売っているものの成約率を高めたい、
　さらには高く値付けしたければ、

　物語性を商品設計＆営業マーケに
　埋め込むのは必須やで。

　----------

　「成果は出して当たり前」

　そう思い込むところから
　人生が変わるのよ。

　根拠なんて要らない。

　ただ強烈に信じ込むだけで、
　未来が変わる。


　前進し、結果を出し続けている姿が
　当然で、停滞すれば驚かれ、心配される。

　そう思われる自分を目指すのよ。

　「根拠なき信念」が、
　人生のレベルを大きく引き上げる。

　----------

　物語あるところに共感者が集まるのよ。

　協力者として、お客様として
　集まって下さる。

　そこに集う人は皆、
　深いところで同じ世界を共有できる人。


　さらに言えば、

　そうした方々を惹き付けてやまぬ
　独自世界を発信せよ、

　ということ。


　それはつまり、

　自身の思想/哲学を確立する必要がある
　ということでもある。

　----------

　物語には「吸引力」があるんだわ。

　「昔むかし、あるところに」
　という出だしを聞いただけで、

　一気に持っていかれるよな。


　「以前、ワイの勤めてた会社が
　　倒産したんやが」

　と読んだだけで、
　続きが読みたくなる（よな？）。

　そういうこと。

　有益コンテンツは知識欲を満たすが、
　感情を満たそうとするのが物語。

　----------

　「分かったからやる」
　じゃないんだわ。

　分からないからやるんだよ。

　分かるまで待ってたら
　永遠に分からん。

　やって初めて、

　何が分かって何が分からんか
　理解できるのよ。


　最低限を学んだら、そのまま
　実地に飛び込むのが一番早い。

　そこで分からんことが出てくる度に
　学んだら、歩留まり10倍よ。

　----------

　没頭に目的なんていらんのよ。

　せずにはいられん。
　そこに理由はないんだわ。


　ただ純粋に打ち込む没入体験だけが、
　誰も創れない自分だけの世界を創造する。

　世俗の意図を捨て、心から楽しむ。


　それが深いところから湧き出る
　情熱の力で、自分の花を世界に咲かせる、

　遠回りに見えて最短の道やと思ってる。

　----------

　人間は思っている以上に

　「空気、気分、雰囲気」に
　影響されるんよ。


　目指す理想の姿があるのなら、
　その状態を実現させている人と
　時間を過ごすべき。


　環境を選び、周りの空気を変えることで
　未来が決まっていくんだわ。

　成り行きに任せるな、
　意志を持って環境を選び取れ。

　----------

　３年もあれば、人は想像できんほどの
　大変化を起こせるよ。

　でも、ほとんどは日常の引力に負け、
　同じところで足踏みをくりかえす。


　劇的な変化に必要なのは
　以下の二つ。

　（1）非常識なくらいに高い目標
　（2）切磋琢磨できる仲間

　目標を共有する信頼できる
　友だちが見つかれば、

　一気に両方手に入る。

　----------

　未来は古いものを手放す
　勇気から生まれる。

　愛着あるものを捨てる決断には
　痛みが伴うよな。


　でも、その痛みを乗り越えた人だけが、
　次の大きなステージに立てるんよ。

　大切にしているものを手放してでも
　実現させたい未来がある、

　そんな覚悟と決断から
　全てが動き出すんだよな。

　----------

　「ビジネスは真空を嫌う」

　いう言葉があるんよ。

　今やってることを捨てると
　真空状態が生まれるけど、

　不思議にもその後すぐ、

　新たなチャンスが訪れることが
　多いのよ。


　この感覚を掴めると案外、
　気軽に決断できるようになるけど、

　慣れるまでは手放すのに
　時間がかかったり、

　結局手放せないことも。

　----------

　規格外の目標が、なぜ大切なんか
　教えるで。

　常識はずれの目標が腹落ちすると、

　現状とのギャップがデカすぎて
　「巨大な真空」が生まれるのよ。


　そこで己の無力さが自覚させられ、
　自惚れも破壊され、本気スイッチが入る。

　この世は本気の人にのみ
　資源が集まることになっていて、

　そこから全てが動き出す。

　----------

　「やる気」は当てにならんのよ。

　大切なのは「その気」に
　なることやで。

　やる気は一時的やけど、
　その気は持続する。


　表層の動機ではない、
　深い意識の変容からもたらされる

　「その気」

　が人生を変革する原動力となるんだわ。

　これなくして、

　どれだけ知識やノウハウを蓄えても
　モノになるはずあらへんで。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
