　【　金もうけより人もうけ。
　　　　金銭は人間関係のバロメーターの結果に過ぎないことを知る　】

───────────────────────────────────



■「金もうけより、人もうけ」という言葉があります。

　お金を儲けたいなら、まずはお金に着目するのではなく、
　人に注目せよ。

　他者と関係性を強固にすることを最優先に心がけよ。

　そうすれば人から助けられて、ビジネスはうまくいく。
　お金が儲かるのはその結果にすぎない、

　というような意味です。



■私がお世話になっていて、いろいろと教えていただいてる
　後藤芳徳さんは、


　「ピンチもチャンスも人に乗ってやってくる」


　という趣旨のことを言われていて、なるほどそうだと
　思いましたが、最近、ことに人のご縁で生かされていることを
　つくづく感じます。


　【今日のお勧め本　チャンスと出逢うための人脈大改造
　　　　　　　　　　　　　  フツーの人でも成功できる実践法則】

　 ⇒ http://amazon.co.jp/o/ASIN/4774506648/2ndstagejp-22/ref=nosim



　「チャンスはたった一人の素晴らしい人間と出逢うことで
　　大きく開ける。」　

　でも

　「たった一人の悪党との出会いでピンチに陥る」 

　いやー、わたしも両方とも経験ありますよ。


　そんな経験を既に通ってこられた方も、幸いなことに、まだ
　経験されたことのない方も予防接種のために。


　【今日のお勧め本　チャンスと出逢うための人脈大改造
　　　　　　　　　　　　　  フツーの人でも成功できる実践法則】

　 ⇒ http://amazon.co.jp/o/ASIN/4774506648/2ndstagejp-22/ref=nosim



■振り返ってみると、人生の転機となることは、必ず人を
　介してチャンスをいただいてきました。

　新しい世界に進むときの水先案内人になっていただいたり、
　後ろから押してくださったり、引き上げてもらえたり。

　いろんな人と交わってゆくなかで、いろんな機会を与えられ、
　そのチャンスの芽を大きく育てるためのご支援を賜って
　きました。



■決して、ちゃぶ台に向かって端座（たんざ）し、お茶を
　すすっていて儲け話が来ることもなければ、

　大平原のまんなかで、一人ぽつねんと突っ立っていて
　チャンスがわいて出るものでもありません。



■自分一人でビジネスを成功に導くことはできないし、

　よしんば成功はできても、孤独な人生になるでしょう。

　たとえお金があったとしても、
　それで人の心が買えるわけではないからです。



■かりに人の心がお金で買えたとしても、
　はたしてその人間関係にどれほどの価値があるでしょう。

　本当の信頼関係を築けるでしょうか。
　親友になれるでしょうか。

　所詮、お金で買ったものには、お金以上の価値はないのです。



■ビジネスはお金を稼ぐことが目的ではありますが、

　お金は、「人とどれだけ協力できたか」を示す
　数値的バロメーターにすぎないのではないでしょうか。

　人間関係は数字であらわすことはできませんが、

　「自分がどれだけ人と関わり合いが持てたか」を
　分かりやすい形で教えてくれるのが通帳の額面であり
　資産総額といってもよいのかもしれません。



■「金もうけより、人もうけ」。
　この本末を転倒させてはいけませんよね。

　ビジネスパーソンは人とのご縁で生かされているのであり、
　人との関係が尽きたときに一切を失うのではないでしょうか。

　

　↓↓　ご友人や会社の上司・同僚・部下にも教えてあげて下さいね　↓↓
