7458号 幸運の女神の後頭部はハゲ

■昨日、シャワーを浴びているとき


　【妄想力は実現力】

　【妄想あるところにしか
　　可能性は生まれない】


　という言葉が降りてきました。



■すべては妄想から始まるんですよ。


　可能性は極めて低いかも
　しれないけれども

　「妄想が実現する可能性は
　　まったくのゼロではない」

　「もしかすると
　　いけるかもしれない」

　そんな思いが行動力の源泉となり、

　そこから生まれる行動が
　理想の未来を引き寄せます。



■以前に、日経のインタビュー記事か
　なにかで読んだのですが、


　ファーストリテイリングの
　柳井さんも、

　（検索して当時の記事を探しましたが
　　見当たりませんでしたので、

　　以下、あいまいなところもあるので
　　細部は正確ではないかもしれませんが）

　以下のようなことを
　言われていました。



■柳井さんが、山口県は小郡市において
　小さな服飾ビジネスを行っていた頃、

　グローバルで大きな商売をしている
　海外企業の社長にインタビューを読み、


　「自分と同年代の人が自分よりもはるかに
　　大きな規模でビジネスを行っている」

　「私がその人物に劣っているとは思えない」

　「自分でもできるんじゃないか」

　「ワンチャン、
　　世界一を狙えるんじゃね？」


　と思ったというのです（超訳）



■そして、世界一（妄想）を念頭に置き、

　「可能性は0.1パーセントかも
　　しれないけれどもゼロじゃないだろう」

　と考えているうちに
　フリースが大ヒットして、

　「可能性は１パーセントかも
　　しれないけれどもゼロじゃないだろう」


　さらに、グローバル展開を
　加速させていく中で

　「可能性は10パーセントかも
　　しれないけれどもゼロじゃなさそうだ」

　という風に、

　徐々に妄想が、
　よりリアルなイメージに近づいてきた、

　というのです。



■規模感はまるで異なりますが、
　この感覚、分かるんですよね。


　私（鮒谷）は、

　「人生すべて前倒し」
　「すべての学びは前倒し」

　と唱えながら生きてきましたが、


　そうした精神から必然的に、

　「生涯において必要なお金も
　　できるかぎり『前倒し』で稼ぎ切ろう」

　（そうすればあとは自由だ！）

　という妄想が導き出されました。



■勤務していた会社が倒産（！）して、

　なけなしの現金を握りしめて
　大阪を離れ、東京に出てきたのは、

　20年以上も前のこと。



■倒産する前のサラリーマン時代から

　「生涯において必要なお金」

　は果たしてどれくらいだろうかと、


　これから起こり得る人生イベントや、
　不測の事態に見舞われるリスクを考えながら、

　毎日のようにシミュレーションを
　繰り返していたものでした。



■そのたびに、

　失業者（＝私）がシミュレーションで
　明確化された金額を蓄えようと思っても、

　稼いだ分から
　税金や社会保険料が差し引かれ、

　さらに生活費その他のお金も
　出ていくのだから、

　普通に生きていたら幾らも残らない、
　という冷厳な事実を目の当たりにさせられて。



■こんなことでは、

　前倒しで生涯に必要とする資金を
　稼ぎ切ることはとうてい叶いませんから、

　収入の質・量の抜本的な変更が
　求められるのはいうまでもありません。



■その上で、

　目標金額を仮に５年で貯めるとすれば、
　あるいは10年で貯めるとすれば、

　一年あたりでどれだけ（税引前で）
　稼ぐ必要があるのか、


　来る日も来る日も脳内で、

　（エクセルが使えないので汗）
　ときには紙に落として、

　電卓を叩きながら
　具体的な数字を考え続けたものでした。



■今から考えると、それらすべてが

　「妄想を育てる営み」

　だったんですよね。



■私（鮒谷）にとっては、

　「いくらハードルが高くとも
　　前倒しで必要資金を稼ぎ切る」

　という選択を捨てることは
　ありえませんでした。

　妄想が膨らんでからは、
　なおさらのこと。



■それゆえ、

　「難しいかもしれないけれども、
　　できるとすればどんなアプローチがあるか」

　とひたすら考え続けているうちに、
　妄想の芽が

　『ジャックと豆の木
　（最近では『ジャックと豆のつる』？）

　のように、どんどん
　育っていくこととなりました。



■とはいえ、
　最初は柳井さんと同じく

　「可能性は0.1パーセントかも
　　しれないけれどもゼロじゃないだろう」

　くらいにしか思えませんでした。



■ところが、小さな成功体験を
　積み重ねていくうちに

　「可能性は１パーセントかも
　　しれないけれどもゼロじゃないだろう」

　「ワンチャン、あるんじゃね？」


　さらに、商売が発展していく中で

　「可能性は10パーセントかも
　　しれないけれどもゼロじゃなさそうだ」

　と考えられるようになったのです。



■そうこうしている間に、

　私（鮒谷）は生きていくのに
　大してお金を必要としない人間なので、


　「あれ？気づいたら、質素倹約を旨として
　　粗食で清貧生活を送る分には、

　　生きていく上で必要な、一生分の
　　お金は前倒しで稼ぎきったんじゃね？」


　という状態になっていたんですよね。



■もちろん額面だけで見たら、

　大金持ちでもなければ
　小金持ちですらなく、


　家も借家なら、車も持たないし、

　幸か不幸か（？）銀座にも六本木にも
　港区女子にも一切の縁もありません。

　（といいつつ、
　　むしろ持ちたくもないですが。

　　とくに徹底拒絶、断固排斥、
　　港区女子）



■着るものも、

　夏なんかＴシャツも短パンもパンツも
　靴下も全部ユニクロですから、

　着ているもの一式合計5,000円以下、
　みたいな生活です。

　散髪も1,500円。



■最近は、遅い時間にスーパーに行き、

　半額とか30％お値引きのシールが貼られた
　魚の切り身の塊（？）を買ってきて、

　それをマイ包丁で切って
　お粥と一緒に食べるのが、

　ささやかな喜びにもなっています。



■それでも、毎日が幸せだし、

　おかげさまで、何の心配もなく
　日々を暮らせているのは、

　本当にありがたいと思っています。



■さして、その程度の生活であっても
　今風に言えば

　「FIRE」ｗ

　できる状態にはなった、
　ということなのかもしれませんね。



■なにがいいたかったというと、
　妄想はめちゃくちゃ大事、ということ。

　妄想力は実現力ですよ。



■ちなみに私（鮒谷）は失業者から、
　微妙に

　「FIRE」ｗ

　できるくらいの状態にはなりましたが、

　こんな程度のことですら、
　失業当時には想像も及ばぬことでした。



■しかし、育成した妄想力が、

　アンテナを高く、広く伸ばし、
　さらには鋭敏な感度ももたらし、

　そこで（真空）状態が生まれた時に

　「メールマガジン」

　という自分メディアの存在を知り、
　その可能性に気づいたんですよね。



■そこから競馬用語風に言うならば

　「後方待機から、
　　直線一気のぶっこ抜き！」

　が実現されることとなりました。



■自分の運営するメディアというのは、

　それほどの潜在力や瞬発力や
　爆発力を秘めているものなんですよ。


　現在の私（鮒谷）など正直、

　【当時に築いた、メールマガジンという
　　名の余剰資産（残存者利益）】

　で生活させてもらっているようなもの。



■もし、今の時代において
　そんなメディアを

　「ゼロから」

　立ち上げるのであれば、
　私（鮒谷）はきっとメルマガではなく

　「X（Twitter）」

　を選択していたと思います。



■拡散力とかも半端ないですし、

　メルマガよりも取っ掛かりの
　ハードルも、はるかに低いですからね。



■これまで、時折、

　「自分のメディアを持つといいですよ」

　とはお伝えしてきましたが、


　イマイチ歯切れが悪かったのは

　「メディア立ち上げの難易度が
　　途方もなく上がっている」

　ことを認識していたから。



■それが今回、
　講師のＨさんと出会い、

　「あ、やるべきことをきっちりとやれば
　　今からでも、巨大な個人メディアを
　　ゼロから立ち上げることができるんだ」

　※Ｈさんがまさにその実践者

　と分かったので、


　これはお客さまやメルマガ読者さんの中の、

　そうした強いニーズをお持ちの方と
　共有できればと多い、

　早速、今回の企画を持ちかけ、
　実現させました（Ｈさんに感謝）



■２時間ほど前に、

　「セミナー申し込み解禁！」

　のメルマガを配信しましたが、

　深夜にも関わらず、既に十数名の方から
　お申込みを頂いております。



■実績のあるベストセラー作家の方や、

　不動産ビジネスで活躍されている
　超優秀な投資家さん、

　などもお申込みくださっています。

　さすが「分かっている方」ほど、
　動きが早いようです。



■幸運の女神の後頭部はハゲ、もとい
　幸運の女神には前髪しかない。

　即断即決即実行。
　すべての学びは前倒し。

　などについて、じっくりと
　お届けしていこうと思っていたのですが、

　どうやらその前に満席となりそうです。
