【そうだったのか！鮒谷が船舶免許取得を通して気づいたこと～「はじめの一歩」と「天引き概念」で、人生が変わる】

2014/07/08 14:33 
■昨日もメルマガで書いたとおり、

　今回、一級小型船舶操縦士免許を沖縄にて
　無事取得して帰ってきました。


　この新しい取り組みについて
　改めて考えてみたのですが、今回免許を取れたのは、

　「はじめの一歩」

　と

　「天引き概念」

　という二つの要因からでありました。



■いつか取りたい、どこかで取ろう、
　どれだけそんなことを考えていても、

　実際に身体を動かさなければ、
　永遠に免許を取得できることはありません。


　また、何かを学んだり、身につけたいと思うなら、
　必ず一定の「時間の天引き」が必要になります。


　（たとえば一級船舶免許なら、
　　
　　学科講習24時間　＋　試験2時間20分
　　実技（1名あたり）講習　4時間　＋　試験25分

　　という合計30時間以上の時間が必要です）



■わずか30時間、されど30時間。

　私（鮒谷）のような意志薄弱な人間にとっては、

　日常生活の合間を縫って、
　これだけの時間を確保するのは相当に困難。


　けれども、そんな私でも、
　
　「はじめの一歩」を踏み出し、講座を申し込み、
　前もっての「時間の天引き」で身柄を拘束してもらう、

　これを実行するだけで、
　（法的には）そこそこの大きさの船

　※総トン数20トン未満
　　(プレジャーボートで24m未満に限り20トン以上可)

　を操船できるようになりました。



■免許取得を誘われ、

　「取りに行こう！」

　と思う。

　
　そして、その勢いで申し込む、という

　「はじめの一歩」ならびに「時間の天引き」

　によって得られたものが船舶免許だったのです。



■たかが船舶免許を取ったくらいで何を偉そうに、
　と言われる方もあるでしょう。

　けれども、たかが船舶免許、されど船舶免許。
　
　
　船舶を操縦できるといった技能を毎月、
　１つづつでも自分の身に付け加えて行ったら、

　いずれ凄い能力（無形資産）が蓄積されていく、
　という風には思われないでしょうか。


　※この考え方で、

　　営業能力であれ、
　　コミュニケーション能力であれ、
　　
　　文章執筆能力であれ、
　　マーケティング能力であれ、
　
　　プレゼンテーション能力であれ、
　　マネジメントやリーダーシップといった力を、

　　毎月まとまった時間を投下し、
　　少しづつ高めていけば、

　　いずれ凄いことになるのです。
　　
　

■逆にいえば、

　はじめの一歩を踏み出し、
　時間の天引きをしなければ、

　ある程度、人さまのお役に立てるような能力は
　なんにも身につきません。


　だから、この２つ（一歩踏み出す＆天引き）は
　人生を変えていくにあたって必要不可欠な、

　「習慣の王様」

　と私（鮒谷）は名づけました。



■あるとき、そのことに気づいた時から、
　誘われたら絶対に断らないように、

　あるいは面白そうな分野を見つけたら、
　じゃんじゃん、躊躇なく、
　
　新しい世界に飛び込んでいくように
　なりました。


　一度その習慣が身についたらしめたもので、
　むしろ

　「世界が変わっていかないのが気持ち悪い」

　状態になります。



■結果としてそれが

　（皆さまご心配の尽きないであろう）

　キャリアの不安、あるいは経済的不安を
　乗り越えていく原動力ともなりました。

　
　考えてみれば当たり前の話で、
　

　「人がやっていないことにどんどん取り組み、

　　かつ、それぞれに一定の時間をかけているので
　　専門性が身につき、


　　結果として希少性が強調されたり、

　　話の引き出しが増え、
　　コミュニケーション能力も増強される」
　

　わけですから、うまくいかないはずがないのです。



■そんな風に、

　今件に限らず、何かを手に入れたいと思ったら
　フットワーク軽く、

　「一歩、右足（左足でもいいんですが）を踏み出す」

　これが本当に大切。



■一歩踏み出せば、

　二歩目も三歩目も四歩目も、、、

　というように歩き続け、
　最終的に目的地まで到達できてしまいます。

　踏み出す人と、踏み出さない人との差、
　これが大きいのです。


　
■「一歩を踏み出す」ための具体的な行動としては、

　私がさんざん用いてきて、
　本当に良かったと思うのが、
　
　「何も考えずに、申し込む」

　こと。


　セミナーであれ、勉強会であれ、
　お稽古事であれ、パーティであれ、

　申し込みさえすれば

　「そのための時間は自動的に天引きされる」

　ことになります。



■申し込む、天引きする、

　という形で自らに強制力を持たせた瞬間から
　世界が変わり始めるのです。


　そのことが分かった人から、
　このカラクリを利用して、

　勉強する人は、
　どんどん勉強を続けていけるようになるし、

　あるタイミングから、かかったコストも
　ジャカジャカ回収できるようになる。



■その回収した時間的、金銭的、精神的余裕が
　さらなる

　「躊躇なき投資」

　を促し、一気に大きく羽ばたいていくことに
　なるのです。


　そのためにもまずは、

　「はじめの一歩」

　を踏み出さないことにはなんにも始まりません。

　そう、文字通り、何も起きないのです。



■古来より

　「光陰矢のごとし」

　「歳月人を待たず」

　「少年老い易く学成り難し（本当、その通り、、）」

　と言われます。


　きっかけやチャンスが目の前に来たとき、

　即、はじめの一歩（具体的な行動）を取ることに
　よってのみ変化を生み出すことができるようになる。



■そこでためらうことが習慣化されてしまうと
　ただでさえ、

　「光陰矢のごとし」

　「歳月人を待たず」

　「少年老い易く学成り難し（本当、その通り、、）」

　なのに、

　ますますそれに輪をかけて、恐ろしい勢いで
　時間が過ぎ去っていってしまうのです。

　何も身についていないのに。



■そんな視点であなたの周りにいる

　「変化し続けている人」

　を見てみてはどうでしょう。


　きっと、

　はじめの一歩を踏み出す勢い、
　
　あらかじめ時間を天引きする覚悟、
　
　投資は先、回収は後、という信念、


　これらにおいて普通の人と圧倒的な差が
　あるのではないでしょうか。



■こうして

　「小さな一歩を踏み出す」

　ことによってのみ、

　時間の経過が味方となり、
　どんどん大きな成果となっていく。


　さらに、その成果同士が結びつき、
　より大きな雪だるまとなり、

　やがて、雪崩のごとくに、
　素晴らしいことが同時多発的に起き始めます。



■以上に記した通り、

　どれほど言葉や思考を弄んでいても
　それだけで何か変化が起きるのではありません。


　「雪崩のような成果」が欲しいのならば、

　雪崩を起こすための最初の一撃を加える
　（はじめの一歩を歩き出す）

　必要があるのです。
　


■その一歩が、

　やがて怒濤のように変わり続ける人生を
　約束してくれる、

　そう信じて、私（鮒谷）もますます、
　新しい一歩を踏み出し続けたいと思っています。


　そんな思いに共感し、共有頂ける、
　人生を変えたいと思っている方はぜひ、

　次の一歩（＝先にご案内した、年間プログラム）
　を踏み出してみられることをご検討ください。
