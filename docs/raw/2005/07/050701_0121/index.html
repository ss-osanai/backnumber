　【　自らのビジネスにおいても
　　　　　　　「仕事のジャマ」をしてしまう危険性を常に意識する。 】

───────────────────────────────────


■このメールマガジンも、いまや70000部を超えました。

　ひとえにみなさんのご愛読のたまものであり、本当に感謝にたえません。


■この「平成・進化論。」、号外も含めると実に月に60通も配信されて
　おり、平均すると、限りなく1日2回配信に近い状態になっています。

　しかもその内容がまた、やたらと長い。（われながら）


■ということは常に、一歩間違うとスパムメールになってしまう危険性を
　はらんでいることになります。

　そのことは重々、認識しておかなければいけないと思っているのです。


■きのうは、都議会選の選挙カーが毎日仕事のジャマをしにきて
　ほとほと迷惑しているという話をしました。


■「○○をよろしくお願いします」

　「あなたのために○○します」

　ひとつ覚えのセリフを一方的にわめき散らして去っていく無神経さに、
　本当に勘弁してくれと思わずにおれません。


■ところが、もし私がスパムメール扱いされるような内容を配信して
　しまったなら、どうなるか。

　あれほど嫌っている、迷惑行為の行為者（ほんとに迷惑なんです）に、
　自分自身が成り下がることになるのです。


　（本メルマガの場合はイヤなら勿論、ご自由に解除いただくこともできます。

　　選挙カーと違って自身で選べる＝逃げられる、ので、本メルマガ下部
　　記載の解除ＵＲＬにて、そちらからお願いします）


■そこで、あなたにも一度考えてみてほしいのですが、

　あなたの営業スタイル、マーケティングスタイルは、ともすれば迷惑行為
　と受け取られる状態になってはいないか、これを機会に振り返ってみられ
　てはいかがでしょう。


■「○○を買ってください」

　「○○をよろしくお願いします」

　「あなたのために○○します」

　そんな紋切り型の口調で自社の宣伝をしまくることは、お客さんにとって、
　あるいは見込み客にとって非常に迷惑がられる可能性もなきにしもあらず。


■営業（自社商品の宣伝活動）に熱心になるあまり、お客さまの心の中を
　土足で走り回っていないか。

　これを機会にチェックしてみられてもいいのかもしれません。

選挙カーの周辺の迷惑を顧みない宣伝振りに閉口している状況は、私も全く

こうした事象に出会った場合の、その後の行動・思考回路が鮒谷さんと全く
異なりました。

鮒谷さんは、
　行動→　うるさくても仕事に集中する。
　思考→　こうした迷惑行為を反面教師として、自らの仕事と結び付けている。

私は、
　行動→　外に飛び出し、「うるさくて迷惑だから止めてくれ」とクレームを
　　　　　付け、やいのやいのと押し返され、気分を害すると同時に、貴重な
　　　　　時間まで失った。その後も、腹が立った気持ちを引きずってしまった。

　思考→　うるさい、うるさい、うるさい、絶対票なんか入れてやらないよ！

この差は大きいです。
これが毎日、毎分？起きていて、1年・5年と積み重なっている・・・。

鮒谷さんのメルマガは、こんなことを気付かせてくれますので、
空気・水と同じくらい必要なんです。多くの方がそう思っていることでしょう。

これからも心してメール読みます。蒸し暑い日が続きます。ご自愛下さい。
