7293号 東京と大阪と名古屋のレゴランドを制覇した私（鮒谷）が、今あらためて思うこと

■言葉はレゴブロックみたいなもの、
　と思うことがあります。


　レゴは、

　（想像力と創造力を持つ人にとっては）

　たくさんの種類のブロックが
　手元にあるほど、

　いろんなものを作って楽しんだり、

　見る人を喜ばせたり、
　驚かせたりもできますね。



■東京のレゴランドも、
　名古屋のレゴランドも、
　大阪のレゴランドも、

　ぜんぶ訪れたことのある

　「隠れレゴファン」

　であるところの私（鮒谷）は、

　いつもそんなことを考えながら
　レゴブロックを眺めておりました。



■レゴと同じで、
　たくさんの言葉を持っていると、

　（想像力と創造力を持つ人にとっては）

　いろんなものを作って楽しんだり、
　人を喜ばせたり、驚かせられます。



■ちなみに、たまたま昨日は、

　こんな「とんでもない動画」を
　見たのですが、

　＜大王レゴマグロ解体
　　/ストップモーションクッキング&ASMR。
　https://www.youtube.com/watch?v=kRipvYbzsFM

　これなんか、まさに

　「想像力と創造力の発露」

　以外の何物でもありません。



■言葉のブロックを使って、
　このようなことを行いたいものですが、

　それにはまだまだ
　修行が足りませんね、、、



■とはいえ、私（鮒谷）にとっては、

　年間プログラムも、各種セミナーも、
　音源販売も、個別コンサルも、


　【言葉というブロックを用いた、
　　私（鮒谷）なりの創造性の発露】


　であることに間違いはありませんし、
　それを生業ともしています。



■それぞれのサービスメニューにおいて、

　与えられている時間や、
　事前準備をどこまでできるかなど、

　異なる制約条件が与えられますが、


　所与の条件のもとで、

　最高のパフォーマンスを発揮するために
　何ができるのかを考えるのは面白く、

　それを最も適切な形で実現させたい、
　というのが私の願いでもあります。



■コンサルは

　（誤解を恐れずにいうならば）

　大喜利みたいな、瞬発力のある
　ひらめきの力が求められるし、


　年間プログラムは
　それなりに長丁場なので、

　事前にある程度の準備と全体構成を
　イメージしておくことが要求されるし、

　手元にあるブロックは同じでも
　競技（？）が異なる、

　といった感じ。



■いずれにせよ、

　手元に多種多様、かつ、
　数多くのパーツがなければ、

　作りたいものも作れないのは、

　レゴ製作者の方々と同じではないかと
　思います。



■ところでレゴは、

　トイザらスなど行けば、いちどきに
　大量に仕入れることも可能ですし、

　レゴ・アンバサダーにでもなれれば
　ひょっとしたら無尽蔵に供給される、

　ものかもしれません。



■それと比べると、

　（言葉の）ブロックは、
　一括仕入れが不可能であり、

　「手に入れられる、そのときその場で
　　必ず入手するしかない」

　という、ある種、

　「稀覯本を集めているコレクター」
　「希少グッズの収集者」

　的な立ち回りが要求されます。



■こんな風に、

　毎日、一瞬一瞬の時間を大切にして
　積み上げていく、

　そんな日々のトレーニング習慣からしか、

　【言葉を組み上げ、新たなものを創造する
　　喜びを感じられる体質にはなれない】

　のではないかと思います。



■かくいう私（鮒谷）ではありますが、

　（書いたり話をしたりするのは
　　大好きですが、下手の横好きで）

　たいした話もできないし、
　たいした文章も書けません。



■にもかかわらず、

　ときおり、
　そんな私（鮒谷）に向かって、

　「どうしたら話せるようになりますか」
　「書けるようになれますか」

　さらには

　「文章を書いて
　　売れるようになりますか」

　「自分をマネジメントできるように
　　なりますか」

　などといった質問をくださる方が
　いらっしゃいます。



■こうした問いに対する、

　「完全なる正解」

　はお渡しできないかもしれませんが、


　それでもそれなりの年数、
　取り組んできた経験から、

　より本質的、かつ根本的な回答を
　行おうとすると、

　【毎日のトレーニングしかありませんね】

　という身も蓋もない、

　聞かれた方によっては
　がっかりするしかない、

　そんなお答えしか返せないんですよね。



■でも本当のことなのだから
　仕方がない。

　このプロセスを通らなければ、
　残念ながら

　【言葉を編んで創造する】

　この喜びを味わうことはできません。



■ときおりやってくる、

　「とはいえ、ときにはサボりたい」

　という誘惑に打ち勝ち、


　心と格闘しつつ、
　鋼のごとくに鍛え上げ、最終的には

　「厳しいからこそ、楽しい」

　という境地にまで
　自らをもっていくことができて、


　そこでようやく、

　【創造性の発揮を楽しめる
　　体制が整った】

　ということになるように思われます。



■こうしたある種の「辛気臭さ」は、

　格闘技で強くなるプロセスと
　同じかもしれませんね。


　先日、Netflixで話題の

　『サンクチュアリ -聖域- 』

　を一気観したのですが、


　相撲の世界で強くなるためには

　「毎日、ひたすら四股を踏み続けることで
　　下半身を強化する」

　しかないようです。



■言葉の世界で強くなるにも
　同じように

　「毎日、ひたすら、言葉を集めては
　　編み続ける」

　というトレーニングを重ねるより他、
　なさそうです。



■その先に、
　相撲の世界でいうところの強くなる、

　言葉の世界でいうところの
　言葉を自在に操れるようになる、

　という能力の開花が起こるのでは
　ないでしょうか。



■ただ、言葉の世界が
　相撲の世界と異なるのは、

　相撲の場合は「肉体」という形で
　可視化されている者同士がぶつかり合うので、

　見ていても

　「強靭さの源泉」

　すなわち

　「下半身の粘り強さ」

　が見ていて
　すぐに理解できるのに対して、


　言葉の世界は、

　可視化されにくい
　（というか、可視化できない）ので、

　「地道に取り組まなくても
　　強くなれる道のりがあるのでは」

　と勘違いする人が相応の割合で
　出てこられるところにあります。



■たとえていえば、

　「僕、強くなりたいと思って
　　通信教育で相撲を習っているんですよ」

　といっている人があるとすれば、
　その人は間違いなく

　【嘲笑の対象】

　となるはずです。

　自らの肉体と向き合うことを避けて
　強く慣れるはずがない、

　と、みんな理解しているから。



■しかし、言葉の世界において

　「日々、言葉と真剣に向き合っていないのに
　　言葉を自在に操れるようになりたい」

　といっても、

　そこまでひどく嘲笑されたり、
　罵倒されることもなさそうです。



■なんなら、上手に学べば
　それが実現可能なのではないか、

　と勘違いしてしまう人まで
　一定割合で現れられるのは、


　言葉の世界には、目に見える、
　物質的なものが存在しないから、

　という側面があるからかもしれません。



■でも

　「通信教育で
　　相撲を強くなりたいと思っている人」

　と

　「言葉と格闘することなく
　　言葉を操れるようになりたいと思っている人」

　の間には、ここまでで見てきたように
　本質的な違いはありません。

　（いずれも重心低く、腰を落としての
　　継続的なトレーニングが必要です）



■今日の文章は、

　こうした可視化されにくい
　言葉を操る世界を、

　（それがどこまで可能になったかは
　　ともかく）

　言葉を用いることで
　多少なりとも可視化し、


　それによって
　多少なりとも身体感覚を伴う形で、

　言葉を操ることの楽しさや難しさが
　意識し、理解されれば良いな、

　と思って記しました。



■ここまでの文章によって、

　なんらか認識の変容、
　ならびに行動の変容が起き、

　強く、たくましい、ビジネスパーソンに
　なるための一助としてご活用いただければ、

　そんな思いを込めて
　本日のお届けいたしました。



■、、、ちなみに、上述の通り、

　【言葉や概念のブロック】

　は一朝一夕に、
　大量入手できるものではありません。

　時間がかかります。
　しかし、かけた時間は裏切らない。



■どこかで頭角を現す人は、

　「そんな地道な努力や種まき」

　を長い間、
　水面下で続けているものです。



■それが表に現れたとき、
　はじめて周りの人は驚き、慌てますが、

　そこから追いつこうと思っても
　追いつくことは不可能です。



■数年、十数年、数十年という期間、
　先行して

　「人が厭う、人が嫌う、
　　人が避けたがる」

　そんな大変な取り組みを続けていた人が、

　簡単に誰かに追い抜かれるはずが
　ありません。

　毎日の積み重ねだけが、
　人物を形作っていくと信じています。
