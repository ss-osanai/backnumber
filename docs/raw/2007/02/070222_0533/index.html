■長谷川和廣さんは、

　日産の再建を果たしたカルロス・ゴーン氏にちなんで
　【もう一人のゴーン】と称されている方。

　マーケティング、プロダクトマネジメントを担当。

　その後、

　＜ケロッグジャパン＞＜バイエルジャパン＞などの
　外資系企業で代表取締役社長などの要職を歴任してこられました。

　

■1995年、世界一のメガネレンズメーカーであるエシロール社
　(フランス)の日本法人(株)バリラックスジャパンの代表取締役社長。

　2000年、(株)ニコンとエシロール社の合弁会社(株)ニコン・エシロール
　の代表取締役に就任。

　赤字だった同社を1年目で黒字へ、2年目で無借金経営に変貌させ、
　7年間の在任中に売り上げは２倍、シェアも２倍、毎年増収増益。

　当初計画以上のスピードで同社を好業績企業に引き上げたことは
　あまりにも有名な話です。



■現在は、ご自身の「会社力研究所」にて企業戦略、会社再生などの
　コンサルティングに従事されていらっしゃいます。

　海外企業からも引き合いが多数あり、国際ビジネスコンサルタント
　としてもご活躍中なのですが、

　「実るほど頭を垂れる稲穂かな」

　を思い起こさせる、とても謙虚で誠実な方で、ついついうぬぼれて
　増長しやすいわたしなどはいつも反省させられています。

　私などをお褒めくださっては（調子に乗りやすい私などは）
　おかげさまで、いつも元気とやる気を頂いております。

【　チャンスをつかむために、若いうちは徹底したハードワークを　】

───────────────────────────────────



■このメルマガでは、量稽古を積むことの大切さについて、
  これまで様々なテーマで取り上げてまいりました。

  今回は、少し視点を変えて、量稽古自体がもたらす効果を
　ちょっと別の角度からお伝えしたいと思います。



■仕事の量をこなそうと思うと当然ですが、残業が増え、睡眠時間が
　減ったりとハードワークになってしまいます。
 
　しかし、若いうちはそのくらいがちょうどいいのではないか、
  と私（鮒谷）は思います。



■その理由は、

　若いうちは【 頑張っていること 】自体が評価されるから。
　むしろ、「頑張ること」しかないのではないかとさえ思えます。

  これが30も半ばを過ぎ、40代、50代ともなってくると、
  頑張っているだけでは評価されなくなり、
  年齢、経験、キャリアに見合った成果が求められるもの。

　しかし特に20代のうちは【 頑張っていること 】自体が
　評価される、ビジネスパーソンにおける最初で最後の
　稀有な時期なのです。



■なんかよく分からんが、あいつよく頑張っているな、
  と上の方から目をかけていただき、
　引き上げていただくことがあるかもしれません。

  すると、同年代ではなかなかできないような経験をさせてもらえたり、
　普通に仕事をしていてはご縁の持てないような人を
　紹介していただくこともあったりするでしょう。

  【 頑張っていること 】がフックになり、
  上の世界を見せていただくチャンスを得られるのです。



■頑張っても給料変わらないからムダ、
  適当に仕事したふりして残業手当をもらった方が賢い、
  楽してても頑張っているあいつと給料一緒、

  そんなことを言う人がいるかもしれませんが、 それでは頑張ることに
　よって引き上げられる可能性を自分の中に根付かせることはできません。

  見る人は見ているし、そんな言動を繰り返していると、
  だんだんと地力が衰えていき、踏ん張りがきかなくなり、
  能力が身につかなくなるのです。



■多くの人たちを見てきた感覚では、社会人になって
　５年から10年あたりが、頑張りによって引き上げていただく
　絶好のタイミングのような気がします。

  読者さまの中にはとてもたくさんの若手ビジネスパーソンが
　いらっしゃいますが、人が見ていようが見ていまいが、
  とにかく一生懸命頑張って欲しい、

  ということを今一度申し上げたいのです。



■頑張っているあなたを、見ている人は見ています。

  そして、あるとき突然チャンスを引っさげて現れ、
  引き上げて下さることもあるでしょう。

  そこで同年代の人よりも一歩先に経験を積むと、
  一歩先の実力がつき、ご縁をいただく方のレベルも変わり、
　また新たな経験へとつながっていく。

  このサイクルが回り始めると、一気に軌道に乗り、加速し、
　どんどん他の人との差がつくでしょう。



■そもそも差がつくとっかかりは何なのか、よく考えてみると
　どうでしょう。

  能力や才能で頭角を表す人というのも少なくはないのですが、

  多くの人は、一生懸命頑張り、額に汗して地道に努力したことで
  評価され、相応の仕事を任され、その経験を重ねることで、
  後から能力がついてくる。

  その結果、こういった努力を怠った同年代の人たちに対し、
  大きな差がついているのではないかと思うのです。



■『　あれを見よ  深山（みやま）の桜  咲きにけり
    　　　　　　　　　　　　　  真心つくせ  人知らずとも　』

  誰が見ていなくとも美しい花を咲かせている、かの桜の木のように、
  裏表なく努力する。

　いつか見るべき人が見て、大きなチャンスをもたらして
　くれることを信じて、 いまは地道な種まきを心がけてみられては
　いかがでしょうか。



■勿論、わたしも、まだまだ徹底したハードワークを
　「平成・進化論。」で【 プレゼンテーション 】し、
