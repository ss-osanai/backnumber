```
■上述の通り、明日は

　【ゼロから始める！
　　X（Twitter）立ち上げ＆伴走セミナー】

　全三回のうちの、
　第一回目を開催いたします。



■十日ほど前に告知し、

　相応の参加費（88,000円＋税）で
　あったにもかかわらず、

　半日足らずで、
　30席が埋まりましたが、

　今回は会場の都合もあって、
　追加募集は見合わせました。



■そんなわけで、

　即断即決の精神で
　申し込んでくださった方とのみ、

　明日、お目にかかれることと
　なりましたが、


　改めて起こし下さる皆さまの
　顔ぶれを拝見していると、


　ベストセラー作家さん（複数）を含む、
　本の著者（６名ほど）

　年商1000億規模の企業の元常務。
　年商数百億企業の経営メンバーの方、

　年商数十億企業の経営者や
　不動産投資で大成功されている方、

　他にも、それぞれの世界で活躍されている
　ビジネスパーソンや専門家の方々など、


　多士済々の参加メンバーです。



■さすが、成果を上げている人は
　決断の早い人ばかりですね。

　参加者名簿を眺めていたら
　こんな言葉が降りてきました。


　「即断即決する人にしか、
　　未来は開かれない」



■まあ、当然といえば当然なんですが、

　チャンスがあったときに
　躊躇したり逡巡したりする人は、

　いつもそのパターンが発動するため、
　毎度毎度、

　躊躇、逡巡を繰り返して、
　けっきょく動けない（動かない）、

　ことが多いのです。



■その一方で、
　今回の皆さまのように

　「決断慣れ」

　している人は、

　同じ期間に二つも、三つも、四つも、
　五つも決断しているわけですね。



■それでは行動量も学びの量も、

　（その結果としての）
　当たりくじを引き当てる可能性も、

　違いが出てきて当然です。



■もう随分前に
　お亡くなりになられましたが、

　その昔、「社長の教祖」と言われた
　経営コンサルタントの、

　一倉定先生という方が
　いらっしゃいました。



■私（鮒谷）が勉強させて頂いてきた、

　特に昭和から平成にかけて活躍された
　先輩（大先輩）経営者の少なからぬ、

　というよりも、
　かなり多くの方が、

　一倉定先生のもとで
　学ばれていたことも思い出されます。



■そんな一倉先生の名言の一つに

　【優柔不断は
　　誤った経営よりなお悪い】

　という言葉がありました。


　これは、

　【もし決定が間違っていたとしても
　　決定しないよりも優れている】

　といった意味。



■また、

　【決定は、巧遅よりも拙速が良い】

　ともいわれていましたが、


　意思決定し、行動に移し、

　結果としてそれが誤っていれば、
　修正すればいいだけの話ですが

　決定しない限り、行動に移せないし、
　改善の機会も訪れません。



■さらに、行動しなければ、

　あるアイディアが正しいか、
　正しくないかについて、

　検証する機会も持てないので
　どんどんジリ貧状態に陥ります。



■だから決断、即実行が大事になるのです。

　ダメなら、すぐに修正、
　もしくは撤退、

　いけそうなら、
　一気に追加のリソースを投下する、

　という姿勢が大切になるということ。



■「考えてから走る」のではなく、
　「走ったあと、考える」のでもなく、

　「走りながら考えつつ、
　　加速度をつけていく」

　のが、

　ビジネスの拡大にとっても、
　自分の成長にとっても、

　もっとも効果的な方法です。



■そういえば、三国志にも

　【兵は神速を貴ぶ（郭嘉）】

　という言葉が出てきましたよね。

　思い立ったら、遅滞なく、
　迅速に動くことが大事。



■今日も人生とビジネスを
　楽しみましょう！

```


```
【今日のピークパフォーマンス方程式】

　■兵は神速を貴ぶ、という言葉がある。

　■決断慣れしていないと、意思決定を
　　どんどん先送りしてしまう。

　■その結果、行動量も学びも増えず、
　　改善の機会も生まれなければ、
　　ホームラン性の当たりも掴めない。

　■とにかく、致命傷になる可能性がなければ
　　決断し、動き続けること。

　■結果を出している人は、そんな行動特性を
　　当たり前のように身につけているもの。
```
