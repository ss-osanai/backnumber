　今回の音源を聞かせていただき、
　いろいろな学びがありましたが

　自身の中で「これだ！」と、思った点は

　「言葉で意味付けを与える」

　というところ。

　言葉の真意が初めて理解できました。


　当方はサラリーマンなのでいつも、
　同じ通勤経路を歩いています。

　正直な話、

　「いつもと同じ」
　「無味無色」
　「灰色のアスファルト」

　の上を、ただひたすらテクテクと歩いています。

　そこは「感情」のかけらもなく無機質で、
　歩いているだけでも足取りが重くなります。

　でも、視点を空に移すと

　「真っ青な空」があり
　「曇天の分厚い雲」があり
　「大粒の雲の涙」が降り注ぐ、、、


　視点を足元に移すと
　アスファルトから飛び出てくる雑草や
　黄色や赤のチューリップ
　茶色の土の上に菜の花畑が広がっている、、、

　と、いうように、

　「視点」の取り方一つで同じ「通勤経路」で
　あってもいろいろなものが見えてきます。


　真っ青な空を見て「元気」になったり
　真っ赤なチューリップで「気分がよくなったり」
　地面から突き出る雑草に「勇気」をもらう

　そういう、無味無色で灰色な景色に彩りを
　添えて、

　見る人に対して「意味」を与える、
　これが、

　「言葉で意味付けを与える」

　という本当の意味だと理解できました。


　言葉を変えると

　「鮒ストップ」

　というところでしょうか？笑


　どんな物やサービスにも
「意味」を「言葉」で添えてあげるだけで、

　無機質な物質や無表情なサービスが
　宝物に見えることもあるかもしれません。

　パソコンの傍に置いてあるペットボトルの
　水でさえ「物語」があります。

　商品名は伏せますが
　
　「天然中硬水
　　硬すぎず、柔らかすぎず
　　程よく飲みやすい中硬水です」

　と、記載されています。

　これだけだと、
　僕の中の意味付けは弱いですが、

　綺麗でピュアでナチュラルな水を飲むことで
　自身の肉体を生まれ変わらせる
　人生を一変させる

　「転生水」
　
　という意味づけを与えると

　ただのペットボトルの水であっても
　それだけで、人生にとってかけがえのない
　貴重な水に生まれ変わります。

　今回の音源で学んだ

　「言葉で意味を付与する」

　というのは、

　お客様が心の底から
　感情が動く「物語」を付与することで、

　誰にも負けない「付加価値」を創造することに
　他ならないと個人的には思いました。

　
　いつも、貴重な学びを与えていただき
　ありがとうございます。

　目標音源

　繰り返し、聴き込もうと決心をした次第です。


　長文になりました。

　ここまでお読みいただき、ありがとうございました。

【 「物語と教訓」をセットで持つことでリーダーシップ＆影響力を発揮する 】

■過去２、３回、

　【栴檀は双葉より芳し】

　あるいは、

　【大器晩成】

　といった言葉を通して、
　自らの状況に応じて都合よく物語を編みかえる、

　という話をしてきました。



■これまでは自分に力を与えてくれる
　物語の力、という文脈で語ってきましたが、

　
　数多くの良質の物語を持っていると、

　実は、気持ちよく、他者に思っている通り、
　動いてもらうことが出来るようになる、

　かもしれません（悪用は厳禁ですが）。


　自分と自分を取り巻く環境（文脈）に
　添うように、

　手持ちの物語の原型を、当てはめ、
　意味づけ、活用することで、それが可能になる、

　ということがあるのです。



■たとえば、シェイクスピアが、

　ローマの独裁官シーザーに対する陰謀・暗殺と、
　その死の余波が描いた戯曲

　『ジュリアス・シーザー』

　には、


　シーザーの死後に
　アントニーが追悼演説を行い、

　シーザーの遺体を見せるなどして、
　群衆の感情に訴え、


　最終的に、シーザーを暗殺したブルータスの
　演説に納得していた大衆を、

　反ブルータスへと一気に転向させるという
　有名な一幕があります。


　（この場面は、文字通り「圧巻」です＆必読）


　※ご存じない方は適当に検索して調べて下さい。
　　きっと、どこかに詳細な解説があると思います。



■こうした物語の存在を知ることによって
　たとえば


　【言葉の力によって、気づかぬうちに
　　意識が書き換えられてしまうことがある】


　あるいは逆に（誤解を恐れずに言えば）、


　【言葉の力によって、
　　人の意識は書き換えられるもの】
　

　であることを、まざまざと知らされます。


　こうしたことが理論的に語られるだけでは、
　ほとんど記憶に残ることはないでしょう。

　物語として記述されているから、
　記憶に定着するのです。


　
■たとえば、こうして

　【物語（と、それとセットになった教訓）】

　がひとたび、インストールされれば、


　（『ジュリアス・シーザー』の、この場面は
　　奸計のために用いられた説得力ですが）

　自分の置かれている状況に応じて
　アレンジ

　【この構文を「善用」する】

　ことによって、

　（当人も気づかぬうちに）
　人に動いて頂くことが出来るようになる、
　
　ということもあるかもしれませんね。


　
■けれども、もし、

　こうした物語の存在も知らず、当然、そこから
　導き出される教訓も持っていなければ、

　その構造や構文を換骨奪胎し、
　活用することなど出来ようはずもありません。



■だからこそ、

　もし、あなたがリーダーシップを発揮して、
　人に動いてもらえるようになりたいのなら、
　
　たくさんの物語を読むことによって、

　【人を動かす、物語の基礎構文】

　を自らの内に蓄えておく必要が、
　絶対にあるのです。

　
　物語の価値を理解せず（できず）、
　物語の持つ力に思いを致さず、
　物語をいい加減に扱う人は、

　結局、リーダーシップを発揮することは
　できないでしょうし、

　商売人であれば、お客さまに購入して
　頂くことなどできるはずがありません。



■とするならば、

　英語構文を学ぶのも大事だけれども、
　それよりもまず、

　【物語の基本構文】

　を学んだほうが、
　よほど実社会においては役に立つのでは、

　そんな風にすら思われます。

　（、、、これは、英語ができない人間の
　　「負け犬の遠吠え」ですので、真受けしないようｗ）



■それはさておき、あなたがもし、

　【リーダーシップを発揮して、
　　人に影響力を及ぼしたい】

　のであるならば、


　ぜひ、

　【物語】ならびに【物語の基本構文】

　に意識を払うようにして下さい。

　きっと人生、そこから変わり始めるはず。
