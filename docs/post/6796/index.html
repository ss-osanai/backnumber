■ビジネスにおいて
　何かを提案する際には最低限

　--------------------

　１、誰が対象？

　２、その商品、サービスを通して
　　　どんなベネフィット（便益）が得られる？

　３、具体的な内容は？

　４、誰が提供する？

　--------------------

　といった諸項目を
　明確に提示しなければなりません。



■これらが明瞭に提示され、かつ、
　納得のいくものであればあるほど

　「反応率が上がる」

　こととなるでしょう。


　反対に、不明確な点が増えるほど
　
　「反応率は下がる」

　ことになります。



■これは、

　「なにかを告知する際に絶対に
　　踏み外してはならない最低限のこと」

　であり、

　上記を説明することなく
　なにかを案内するとか

　「商売をまるで分かっていない人間」

　のすることです。
