　鮒谷さん、


　昨日は個別コンサル
　ありがとうございました！！

　また早々に、音声メッセージも頂き
　ありがとうございます！！


　お声は毎回の音源で拝聴しておりましたが、
　お姿を拝見するのは久しぶりでした。

　肩幅が以前よりも広くなったように
　お見受けしました。

　筋トレ筋トレとおっしゃっているから
　私がそういうふうに見てしまったのではなく（笑）
　本当にそのように感じられました。


　今その体型なら、

　コロナが落ち着く頃にはご両親も
　びっくりされる体型になっているのでは！？

　と思いました。

　やはり筋トレは裏切らないですね！


　またコンサルにおいては、

　いかに私に利益を渡そうかという姿勢が
　端々から感じられ、

　いろいろな補助金や
　いかに落穂を拾っていくかなど、

　その姿勢をみさせていただき、


　私もいかに相手に利益を提供するか、
　私にとっては患者様ですので、

　その方にこの薬局でよかった、

　ということを感じてもらえるような姿勢で
　日々仕事をしないければいけないなと、

　改めて気づかさせて下さいました。

　ありがとうございます！！


　今回の●●●●●●のビジネスについては、
　鮒谷さんが指摘される通り、

　まずはそこに対して
　私が割ける時間と気力が、

　おそらくそこまで盛り上がってないことを
　指摘していただき、実際にそうであるので、

　このまま突き進んでも、きっと残念な
　結果に終わることが予想されました。

　そのことを気づかせていただいただけでも
　非常に収穫のあるコンサルでした。


　また改めて思ったことは、

　お話しすることで自分の中の考えが
　色々と湧き上がってきたりだとか、

　まだ取り組んでなかったことに
　対してもいちど調べてみよう、

　という思いが湧いてきたことです。

　いかに自分が
　最後までやり切る行動してないか、

　に気づかされた次第です。


　このコンサルの中で一番、
　私のOSの書き換えられたことは、

　どの市場でやるかが1番大事で
　それが粗利率に反映される。


　やる土俵次第でビジネスが決まる。

　アゲアゲの業界に参入すれば
　利益を得ることも簡単だけれども、

　下り坂のエスカレーターを下から登るような
　サゲサゲの業界では努力しても

　それに見合った利益が出なかったり、
　撤退することになったりするから、

　まずはその業界の見極めをするのが
　大事であること。


　さらに自分がいる業界についても
　そのような状況に陥っていないか、

　常にリサーチする必要があるなと
　思いました。


　その上でおっしゃっていた
　
　●●●●●●や

　いち早く気づき、

　新しい事業を立ち上げていくのが、

　時流に乗るビジネスの探し方の
　1つなのかなぁと思ったりもしました。


　年間プログラムも非常に勉強になり
　毎回何かしら1つは身に付けていこう、

　という心持ちで聞いて
　実践しているつもりではおりますが、


　鮒谷さんと2時間という
　短い時間ではありますが、

　言葉のキャッチボールをすることによって
　学び、気づきは明らかに高いものになるなぁ、

　と改めて思った次第です。

　早川さん、
　先日はありがとうございました！

　久しぶりのコンサルとなりましたが
　お元気そうで、なによりでした。

　年間プログラム音源等でも
　引き続き、よろしくお願いいたします！



■個別コンサルにおいて、

　私（鮒谷）が自覚的に
　ご提供している価値の一つに

　「赤の他人視点からのアドバイス」

　というものがあります。



■自分のビジネスやキャリアを、
　
　自分で客観的に捉えることは
　なかなかに難しいもの。


　より厳密に言えば

　「不可能」

　と言えるでしょう。
　
　どうしてもそこに自己認識という
　色眼鏡がかかってしまうからです。



■そうならぬよう

　「100％、顧客のメリットに焦点を
　　合わせつつも顧客に迎合しない」

　ことに努めています。


　たとえばお客さまが

　「これ、やりたいっす！」
　
　と言われた事業に対して、

　何の根拠もなく、
　条件反射的に

　「いいっすねー！！」

　と返すのは簡単ですし、
　短期的には喜ばれたりするものです。



■こういう観点もあるのではと
　一時停止を促すと、

　かえって嫌われたり、切られたり
　することにもなりかねない。　


　けれども、だからといって

　「いいっすねー！！」

　では役目を果たせないのです。



■もちろん、根拠あっての

　「いいっすねー！！」

　であればよいのですが、
　いずれにしても

　「第三者目線での提案や対話」

　が行えなくなったときが
　コンサルのヤメ時です。



■早川さんが記してくださった通り、
　今回は、

　「新しいビジネスについての検討」

　についてでしたが、

　上記のような点で、
　役目は果たせたのかな、

　と考えております。　
　


■まあ、それなりの期間

　（起業して以来、20年近く、
　　副業時代まで含めると四半世紀）

　多様なビジネスに携わってきて

　「亀の甲より年の功」

　ではありませんが


　数多くの中小・零細・個人企業規模の
　ビジネスに触れてきて

　「さまざまな業界を俯瞰して見るための
　　第三者目線」

　の精度を上げてきたつもりです。



■いちおう、

　従業員数でいうと微粒子レベルの
　自社ビジネスではありますが、


　起業以来

　「自分起点の累計売上げ」

　で、のべ数十億円を上げてきて、
　（改めて考えると自分でもびっくり）、
　

　自身のビジネスだけではなく、

　のべ（たぶん）7,000件くらい？の
　個別コンサルを重ねてきましたので、

　これだけやれば、さすがにバカでも
　見えるようになってくるものです。



■変数の少ない中小・零細・個人事業
　レベルのビジネスは
　
　「思い込み打破のお手伝いをすることで
　　一気にレベルがあがる」

　ことも珍しくありません。

　だから正直に告白すると
　コンサルで結果を出すのは、めちゃ簡単。



■個別コンサルを検討下さっている
　お客さまから、よく

　「個別コンサルでは
　　何をやっているんですか」

　と問われることがあります。



■もちろん、お客さまのご状況により、

　多種多様な展開となるので
　一概には言えませんが、


　たとえば以上のような流れ、
　すなわち

　「お客さまの色眼鏡を外して頂く
　　お手伝い」

　ことが特に、


　「起業直前、直後、あるいは
　　起業後の急成長期」

　のクライアント様に向けた
　典型的なコンサルの型、

　となります。
