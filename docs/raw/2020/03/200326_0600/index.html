■あなたは幸せ不感症？それとも幸せ過敏症？

───────────────────────────────────

　
　
■人生のある時期において

　「暗黒時代」

　のあった人は
　幸せな人だと思います。


　なぜなら、苦しい状況を
　乗り越えられた人のみが

　「暗黒時代」

　を懐かしみ、喜びの源とできる
　ようになるからです。



■こんなことを思ったのは、


　今日、たまたまYouTubeで、

　やることなすことうまくいかない
　不遇の時代に何百回となく聴いて、
　
　（それが誰かは言いませんは）

　元気をもらっていたアーティストの、

　ライブ動画がアップされているのを見て
　当時のことを思い出したから。



■香りもそうですが、
　音楽もまた、一瞬にして

　「その時代」

　に自分をワープさせてくれる、
　魔法の道具ですね。


　
■その曲を聞いて、

　「あの状況からここまで、
　　よくぞ来られたものよ」

　と感慨深く振り返りつつ、

　自分で自分を
　褒めてやりたくなる、

　そんな気持ちにさせられました。



■話を戻すと

　「暗黒の時代」

　と

　「（暗黒時代を乗り越えられた）
　　今の時代」

　のギャップは、

　大きければ大きいほど、
　幸福度が高まるもの。



■生まれ落ちてよりこの方、

　容姿や才能、出自その他、
　あらゆる面に恵まれていて、

　何一つ不自由したことがなく、
　今なお恵まれている。


　そんな人があるとしても、

　外からは幸せそうに見えても、
　実はそうではないように思われます。



■なぜなら、先述の通り、

　自らの過去の不幸と、
　現在の幸福との「差」が、

　今の幸福感を生み出す源で
　あるからです。


　
■過去も幸福、今も幸福、

　であると
　矛盾するようですが、

　今の幸せを感じにくくなる
　（幸福についての感応度が逓減する）

　ので、

　幸せなのに幸せを
　感受できなくなるのです。

　

■要は

　「幸せに慣れてしまう
　　（から幸せを感じられなくなる）」

　ということですね。


　この状態を私（鮒谷）は勝手に

　「幸福不感症」

　と呼んでいます。



■反対に、人生のある時期において
　
　「悲惨極まりない状態」

　を経験していると、

　
　他の人が感応しないような
　小さな小さな幸福についても、

　敏感に感じ取れるようになり、

　「幸福過敏症」

　となれるでしょう。



■日々、ちょっとしたことにも
　敏感に反応し、


　「は～、
　　ほんとうに幸せだなあ」

　「いや～、
　　ほんとうに幸せだなあ」

　「ありがたいなあ」

　「ありがたや」


　と思えるようになるのですから、
　不思議なもの。
　


■たとえば私（鮒谷）の例でいうと、

　数日前、スーパーに行って、


　インスタントコーヒー
　（ブルーマウンテン）と、

　インスタントコーヒー（モカ）と、　

　インスタントカフェラテと、

　インスタント紅茶（アールグレイ）と、

　インスタント紅茶（ダージリン）と、

　インスタント紅茶（アールグレイ）と、

　インスタント紅茶（カモミール）と、

　インスタント煎茶と、

　インスタントほうじ茶と、

　インスタント烏龍茶と、

　インスタント生姜茶、


　を10パックづつ買い揃え、
　棚に横一列に並べてみる、

　といったことを行いました。



■といっても、すべて

　「インスタント」

　ですから、

　全部揃えたところで使った金額は
　せいぜい数千円といったところです。


　しかしそれらをズラリ、
　並べてみると、

　なんだか王侯貴族にでもなった
　ような豊かな心持ちがいたします。



■私（鮒谷）は、

　「不遇の時代
　（カネに恵まれなかった時代）」

　が長かったものですから、

　ささやかながらも、こうした
　乱れ買い？ができるようになった！

　というだけで、


　勝手に脳内で過去との対比が
　始まって


　「は～、
　　ほんとうに幸せだなあ」
　
　「いや～、
　　ほんとうに幸せだなあ」　
　
　「ありがたいなあ」

　「ありがたや」


　と脳内でなんだか
　分からないけれども、

　私（鮒谷）を心地よくしてくれる
　ホルモンみたいなものが、

　ドバドバ、ドクドク、
　溢れ出してくるのです。



■何も大金を使わなくても、

　日々のちょっとしたことで
　勝手に

　「幸せセンサー」

　が発動して、


　（「幸せ不感症」ならぬ）

　「幸せ過敏症」

　とでも言うべき状態が
　日々、続いてくれるのですから、

　ありがたい。



■こうした、
　
　他人と比較する必要がなく、

　日々のちょっとしたことで
　喜びを感じられる状態について、


　これまた私（鮒谷）は勝手に

　「重心の低い幸福」

　と呼んでいます。



■起き上がり小法師のように、
　何度倒しても起き上がってくる、

　つまり、

　どんな状況に転がされても、
　あらゆる局面で幸福を感じられる、


　そんな

　「重心の低い幸福」

　を体感できる体質になったほうが、

　人生を通して幸せになれそうです。



■この状態をある方が

　「幸せの沸点が低い
　　（すぐに幸せを感じられる）」

　という表現をされていたのですが、
　まことその通り、言い得て妙、

　と思います。



■それと対局にあるのが

　「重心の高い幸福」

　で、こちらのほうは

　「カネを使ったり、
　　見栄を張らなければ感じられない
　　幸福感」

　と言えるでしょう。



■たとえば、
　カネが使えなくなったり、

　見栄を頑張って、
　張っていることを見透かされ、

　こっ恥ずかしい気持ちに
　させられたとき、

　瞬時に雲散霧消する、
　安定感のない幸福感。


　安定感がないから

　「重心の高い幸福」
　
　と呼んでいます。



■前者の、

　「幸福過敏症」
　「重心の低い幸福」
　「沸点の低い幸せ」

　は、

　厳しい環境に身を置き、それを
　乗り越えた者のみが手に入れられる

　「ご褒美」

　といって良いのかもしれません。

　

■だから人生において
　一度や二度（あるいはそれ以上！）の

　「暗黒時代」

　を持つ人は幸せな人だと
　断言できるのです。



■この人は

　「あの頃と比べると、、、構文」

　を使うだけで、

　カネを使わなくても、
　他者と比較相対しなくても、


　その場で瞬時に、他の人が見れば
　取るに足らぬ小さなことでも

　「あの頃と比べると、、、」

　という構文を使って作文するだけで
　あら不思議！


　即座に、
　
　「心からの幸せ」

　を感じられるように
　なるからです。



■とするならば今、まさに

　「暗黒時代」

　を生きている人があったとすれば、
　それは喜ぶべきこと、

　と解釈できるでしょう。



■その理由は、

　ここまでお読み下さった賢明な
　あなたなら、既にお分かりの通り、


　この状況を乗り越えたとき、

　「幸福過敏症」
　「重心の低い幸福」
　「沸点の低い幸せ」

　という


　「生涯を通じてフツフツと
　　湧き上がってくる、

　　微細な、そして多様な幸せを
　　感じ続けられる精神」

　をプライスレスの財産を
　手に入れられることとなるからです。



■言葉を換えると、


　暗黒時代を経ることによって

　（なんだか女性誌の特集記事の
　　タイトルみたいで恐縮ですが）

　「幸せ体質になれる」

　ということです。



■既に

　「幸せ体質」
　
　になっている人は、
　そのこと自体で更に喜べば良いし、


　今、まさに暗黒時代を
　生きていらっしゃる方は、

　現状を乗り越えて
　体質改善できた暁には
　
　「幸せ体質」

　になれるわけですから、

　夢や希望を持ち、
　前を向いて生きていかれて下さい。



■こうした物語を持って生きるのと、

　物語を持たず、
　丸腰で生きるのとでは、

　人生のおける幸福度において
　大きな差が現れてくるはずです。



■これは一例ですが、
　
　私（鮒谷）が、私なりに

　「人生を機嫌よく生きていくための
　　物語の編み方」

　について徹底放談した音源が
　ございますので、


　もし、ここまでの話を読み、
　ご興味をお持ちくださいましたら、

　ぜひ聴いてみて下さい。


　
■人生に対する解釈や向き合い方が変わる、
　毎日が楽しくなった、
