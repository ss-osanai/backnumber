■7月12～14日まで、都内某ホテルのスイートルームを貸しきっての
　三日間のセミナー、無事に終了いたしました。

　ご参加くださった方、本当にありがとうございました。

　今回も刺激と感動に満ちた、素晴らしい出会いにあふれた三日間でした。


■セミナーではいつも本気モードで、気力・体力・精神力の限りを
　尽くして、食事のときも含め、持っているものをすべてお伝えしようと
　決意して臨んでいるので、セミナー終了後は、一気に「がくん」ときます。

　でも、そうなることは予測しているので、あえて翌日夕方までアポは
　入れてなかったんですが。


■そんなわけでひさしぶりに目覚ましなしで、翌朝（というか昼）
　まで睡眠をとることができました。

　おかげさまで完全復活。


■ちなみになぜか、今回のセミナーは首都圏からの参加の方は少数で、
　北は北海道から南は四国・九州、そして海外（ニューヨーク）からも
　ご参加、という遠方からのご参加の方が多かったです。


■いつもセミナーなどで話をさせていただくとき、あくまで自分の
　成し遂げたことについては絶対の自信をもってお話をしている訳です。

　と同時に参加された方のほうが、当然のことながらご自身のジャンルに
　おいては圧倒的に実績をお持ちになっておられます。


■そういう意味で、今回のセミナーはお互いの「思考」「知恵」「知識」
　「経験」「ノウハウ」を三日間かけて共有化し、遺伝子交換を行うことが
　できた素晴らしい場でもあったと思います。

　参加者の方同士でいくつかのビジネスも成立したようです。

　こういうことが高い頻度で起こるのもセミナー（特に高額、長時間、合宿系
　のセミナー）の醍醐味でもありますね。

　
■わたしもセミナーに参加するたびに（特に宿泊の入るものでは）必ず一つや
　二つは一緒に参加している人からなんらかの商売を成立させて、参加費を
　回収していきます。

　出自は、ただでは転ばない大阪商人ですから（笑）

　いつか回収、ではなく、その場で回収を原則にしています。


■それが結果的に数百万単位の利益を生むことになったことも一度や二度
　ではありませんし。


■今回ご参加の方も、それぞれの方が各ジャンルにおいて大きな実績を
　上げておられる方ばかりでしたが、三日間、朝昼夜の食事の時間も
　ノンストップで話し続け、のべ30時間以上、お話を致しました。


■セミナー終了後、

　とっても嬉しいご感想も頂いておりますので、ぜひみなさまにご紹介
　させていただければとおもいます。


■わたしがどんなお話をさせて頂いたか、一部セミナーの内容を汲み取れる
　ようなところもありますので、ご興味ある方は行間を読み取って？
　いただければとおもいます。
