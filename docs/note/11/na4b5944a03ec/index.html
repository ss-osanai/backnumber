人は物語によって動く。あなたの人生の難易度を下げる「虚構」の力とは

2025/11/19 04:25

かつてと比べて「生き抜く」行為そのものの難易度が、体感で数倍になっているように感じている。今は、社会全体に余裕のない空気が漂っている。

とはいえ、社会の大きな潮流や空気感を、個人が反転させることは難しい。

であるならば、こちらが順応するしかない。そのためには、 自分がどんな「物語」を生きるのか、枠組みを捉え直すことが大切 になる。

例えるなら、かつてはイージーゲーム寄りだった世界が、中難度、さらにはハードモードへと切り替わりつつある、といったような。

変化を嘆いても仕方ないのなら、 むしろ自分の技量を試す場 として捉え直してみると、精神的にも前向きでいられる。

学び、試し、サバイブする。そんなゲーム として扱ってみる。そう考えると、攻略の糸口が見えやすくなる。

もちろん、これはあくまで物語の例であり、万人に当てはまるものではない。 人にはそれぞれの状況があり、それぞれの環境がある 。

だからこそ、 各人が納得し、前へ進むエネルギーが湧いてくる「物語」 を編み直すと良いのではないか。

たとえそれが虚構に近いものであっても構わない。 人は物語によって動く生き物 なのだから。

それによって、ほんの少し生きやすくなるかもしれない。あるいは、 人によっては大いに楽しめる「ゲーム」に変換できる可能性すらある 。
