◎本メールマガジン（平成進化論）は、

　複数の会社を経営し、IT、人材、環境、飲食、ホテル、海外事業など
　20社以上のベンチャー企業に出資する投資家としても活動する筆者が、

　　------------------------------------------------
　   骨太かつ卓越した＜本物の教育サービス＞を通して
　  「変容の喜び」を提供し続ける
　　------------------------------------------------

　ことを目的とし、3000日以上にわたってお届けしている、
　個人発行として日本最大級のビジネスメルマガです。

　熟読するも読み流すも、自由な形でご活用下さい。

＜昨日の活動ダイジェスト＞

▼本の読み方

───────────────────────────────────



■こんにちは、鮒谷です。


　毎日、本をたくさん読んでいるのですが、
　
　悲しいほどに学んだ内容が、
　（ザルで水をすくうように）抜けていきます。

　特に自分にとって難解な本とか。



■いろいろ試行錯誤した末、
　やっぱり間を置かない復習が大切だと分かりました。

　たくさん読むのも大切ですが、
　一冊づつ固めていく、

　ということも必要ですね。


　理想は、

　「一冊づつ固めていきつつ、
　　たくさん読む」
　
　という状態なのですが（汗）




■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　自分勝手で、それでいい　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　自分勝手で、それでいい　】

───────────────────────────────────

　

■最近、私が気に入っている言葉に

　「自分勝手で、それでいい」

　というものがあります。


　金子みすゞさんの、

　「みんなちがって、みんないい」

　ではありません。

　私が勝手に作った言葉です。



■この意味は、


　「自分のやりたいことだけをやり、

　　それについて人差し指を立てて、
　　興味のある人がいれば、この指とまれ、

　　でいいんじゃないか」


　ということ。



■これまで私は公序良俗に反しない限りにおいて
　できうる限り、


　「自分の心に忠実であろう」

　「やりたいことだけやろう」

　「やりたくないことはやらない」


　と考えて、生きてきました。



■これは

　「忍耐」「我慢」「努力」

　といったものを否定するものでもなければ、
　単なる

　「わがままの勧め」

　でもありません。



■私にとっては、ある種の

　「忍耐」「我慢」「努力」
　
　はとても大切な徳目であり、それは

　「やりたいこと」

　でもあるのです。


　
■これは個人的な見解ですが、
　
　「（今風の、分かりやすい）
　　痛みは極力避けて、ラクのみを求める」

　という考え方には同意いたしません。


　「必要な痛みは積極的に求める」

　ことはとても大切であり、必要なことだと
　考えています。



■そういったことをお含み頂いた上で、

　「自分がやりたいことをわがままに追求する」

　そうして

　「その世界観を徹底して伝え、
　　賛同する人があれば集まってもらう」

　ということは、

　自分にとっても集まられた人にとっても、
　一番よいのではないか、

　そんな風に思うのです。



■「自分勝手」を追求していくと、

　徐々に世間の常識から隔絶、懸絶、逸脱した
　世界を生きることになっていくかもしれません。


　けれどもそれが

　「あいつ、ちょっと変わっているなあ」

　というところから、

　「あいつはああいう奴だから」

　という

　「その人独自の世界」

　にまで至ると、かならずそこに

　「引力」「磁力」
　
　といったものが発生してくるのです。



■人間は主体的に行動せず、
　周囲に流されているだけであれば必ず、

　「他者の引力」「他者の磁力」

　に巻き込まれるもの。


　なぜなら主体的に行動しない人は

　「他者を人を引き付ける力」

　を持ち得ないから。



■ところが、他の誰かから、

　「ちょっとそれ、どうなのよ」

　といわれても、

　「自分はどうしてもこれをやりたいんだ！
　　　　　　　　　　　　　　　（＝自分勝手）」
　
　というものがあれば、それに挑戦することによって

　「変わっていることによって
　　他者を人を引き付ける力が生じる」

　という現象が現れてくるのです。



■むしろ、他の人が否定するようなものであるほど、
　チャンスは大きい思って良いでしょう。

　そのポジションは

　「引力・磁力の空白地帯」

　となっている可能性が高いからです。

　

■世の中にはたくさんの人があり、
　多様な価値観を持って生きています。

　その中にはたとえ少数派でも一定割合で、

　「自分勝手に生きている、
　　そんなあなたの世界観」

　に惹かれて、集まってこられる方があるのです。



■さらに今の時代は、

　「自分独自の世界を、
　　人差し指を立てて紹介し続けていると」
　
　必ずそういう人が探し求められ、見つけられるもの。


　そしてその人を中心とした磁場が発生し、
　そこから渦が巻かれていくのです。



■せっかくやるなら、より強烈な引力、磁力を
　持ってみてはどうでしょう。

　今日はそういったご提案をしたいのです。


　それには

　「何よ、それ！」
　「そんなことやってる人は回りにいないし」

　と周囲の人が驚くようなことを
　やってみることです。

　
　
■それは、

　「自分が没頭、没入、没我の境地」

　に入れる、

　「自分勝手極まりない世界」

　であればあるほどよく、

　その世界観を他の人に提示するところから、
　強烈な巻き込み力が発生することでしょう。



■思うに

　「マーケットに合わせて」

　とか考えるからダメなのです。

　お客の相手の顔色を見ながら商売していても、
　肝心の本人が没頭していないのですから、

　インパクトのあるものは提示できません。



■とはいえ、


　「客のことなんか知らん！」

　「俺（私）は俺（私）のやりたいようにやる。
　　それでよければついてこい！」

　・
　・
　・
　・
　・
　・
　・
　・
　・


　だけではなく、


　「けれども、

　　とりあえず他の人からも分かる目印として、
　　人差し指だけは立てときますね」


　という、

　「自分勝手」だけど、いちおう
　「相手のこともおもんぱかっている」

　というくらいの微妙なスタンスが良いのでは、
　と思います。



■これはマーケットに合わせているのではなく、

　「完全に自分勝手にやっているけれども、
　　マーケットにも一応お伝えしておきましょう」　
　
　という状態。

　
　かくいう私（鮒谷）も
　
　「完全に自分勝手に」

　「でも、いちおう、
　　（メルマガ使って）人差し指は立てて」

　生きてまいりました。



■親の言うことは聞かん、周囲の期待には応えん、
　世間の常識に逆らう、世の流れにも逆行する、

　そうして、

　「こうすればもっと儲かりますよ」
　「こうすればもっと有名になれますよ」
　「こうすればもっとラクできますよ」

　という人があっても、全然興味がわかないものは、
　失礼にならない程度にスルーして、

　「完全に自分勝手に」

　生きてきたのです。



■けれども、そういう独自路線を歩めば歩むほど、
　結果として

　「その世界観を背景とした商品、サービスが
　　求められる、というマーケット」

　が生まれてきたのです。（驚きです）



■そしてこれはあんまり大きな声では言えないので
　（小声で）お伝えいたしますが、

　そこに陣を敷いて商売すると、

　「目立たずに」

　それでいて

　「尋常ならざる利益率」

　が約束されます。


　だって、求める人はあっても供給者が存在せず、

　「強烈な需給ギャップ」

　がそこに生まれるからです。



■たとえばセミナーとか開催するとして、
　はっきりいえば「空気」を売っているわけですが、

　参加費が場合によっては、
　ある方にとってのひと月分の収入に相当する金額、

　ということであっても、
　飛ぶように（？）売れるようになるのです。


　※もちろん、あまりに無理してこられても困るので、
　　当方から参加をお断りすることもあります。



■ある意味、

　このような考え方（みんなが進む方向と反対に進む）
　は世間の常識に反します。

　だからその道を進む人はわずかしかありません。


　マーケットがあるかないかわからないのに、
　自分勝手にやりたいことを極めていく、

　のは「恐怖の道のり」でもあります。



■けれども、その恐怖の道のりをクリアした人は
　

　「やりたいことだけをやり、
　　（やりたくないことは絶対に行わず）

　　かつ、それで十二分に食べていける」


　というご褒美を頂戴できるでしょう。



■「独自性」とか、「差別化」とか、
　「オンリーワン」とか、「エッジの効いた」とか、

　そういったものを目指せ、とよく言われますが、
　言うは易く、行うは難し。

　簡単に「独自性」「差別化」「オンリーワン」といった
　世界を生み出せればいいですが、そうは問屋が卸しません。


　
■それらに該当する商品なりサービスを
　提供できるのは、
　
　「恐怖よりも興味が勝り、
　　興味の力によって恐怖を乗り越えた人」

　だけだからです。



■今日のメルマガは、
　
　そのようにしなければならないんですよ、
　ということをお伝えしているのではなく、

　「自分勝手で、それでいい」

　という世界があり、そういう一つの選択肢の存在を
　提示させて頂いたに過ぎません。
　


■したがって、こういった私の考えをどのように
　捉えられても各人の自由です。

　ただ、一つの思考の材料として、
　参考にできるところがあればご活用頂ければ、

　というつもりで記させて頂きました。



　今日も人生とビジネスを楽しみましょう！
