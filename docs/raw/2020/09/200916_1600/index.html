　-----------------------


　聞く人を選ぶどころか、

　聴いて頂く対象を
　厳密に選定しないと、

　物議を醸す、
　問題になるかもしれないな、


　という話をあえて
　デッドボール覚悟で、

　しかしデッドボールにならぬように
　慎重に言葉を選びつつ、

　今から恐る恐る話を進めていきたい。


　-----------------------


　本題に入る前に一つだけ、

　いいわけではありませんが、
　あらかじめ、保険をかけておくと、

　今から言う話はある一面の真実であり、
　全面的な真実ではない。


　ある側面からは
　そうではないかもしれないけれど、

　さはさりながら、
　間違いなくこういう側面もあるよね、

　という不都合な真実を解き明かして、
　説明をしていきたいと思う。


　-----------------------


　何度でも慎重に言葉を選びながら
　お伝えいたしますけれど、

　人生における価値あるアイテムを
　得るための課金は裏切られることがない。


　-----------------------


　今、私(鮒谷）が昔と比べて多少なりとも
　生活に余裕があるとするならば、

　これまでの重課金、
　ならびにそれによって獲得したアイテム。

　たとえばパラダイム、スキル、
　知識の力によるところが大きい。


　-----------------------


　あるいは課金によってプレイ時間を
　擬似的に延長させることができた、

　ということもあります。


　お金で時間を買う、
　という言葉がありますが、

　職住近接によってプレイ時間を
　延長させることができたし、

　いろんな選択をする際に迷わない、
　迷ったら買う、

　という形でのプレイ時間延長も
　可能になった。


　なぜそれができたかというと、

　課金することによって
　時間を買ったから。


　-----------------------


　どこかに行くときにも、

　基本的に電車移動はせず、
　タクシーで移動する。

　これもお金で時間を買っている、
　重課金者的行動の一例。


　-----------------------


　私（鮒谷）ももちろん、

　人生のある時期まで、
　特に会社員時代はひたすら電車に乗っていて、

　プライベートでは
　タクシーは年に数回、

　3000円のタクシーは
　清水の舞台から飛び降りるつもり、

　これは一般的な感覚でもあるでしょう。


　-----------------------


　自分で商売をするようになって、
　ある程度収益が上がる予感を覚えた際から、

　タクシーに意図的に乗るようにした。


　タクシーに乗って
　10分、30分浮かせることで、

　浮かせた時間でより生産性の
　高いことをする必要がある、

　という決意を固める意味合いにおいても
　意図してカネで時間を買っていた。


　-----------------------


　重課金によって、

　獲得したアイテムであったり
　時間であったり、

　後で話をするが、

　そうしたものに経験や行動を
　付加する、付与することによって、

　実生活によい方向に対する
　影響が生まれてくる。


　-----------------------


　言葉を丁寧に選ばないといけないが、

　人生において課金することによって
　ショートカットできることは、

　間違いなくあります。


　などというような、
　あからさまなことを、

　今までメルマガで
　（間接的にはあったかもしれないが）
　直接、書いたことはないし、
　
　重課金というパワーワード（！？）を
　言ったこともなければ、

　こういう場で
　話をさせて頂くこともなかった。


　-----------------------


　「重課金」などという言葉は
　非常に繊細なワードであるし、

　ある種のポジショントークに
　思われるところもあると思うから、

　直接的に話をすることはなかった。
　自粛していた。


　-----------------------


　さはさりながら、
　睡眠不足だった折に、

　潜在意識から
　ぼこっと浮かび上がってきた、

　「人生は重課金でうまくいく」

　というパワーワードが出てきた。


　たしかに人生は、
　ある意味合いにおいては

　「課金ゲーム」

　みたいな側面が間違いなくある。

　
　-----------------------


　このことを

　いい悪い、是非、善悪の方向から
　語ろうとしているわけではなくて、

　現実を正面から見たときに
　一側面として、

　間違いなく、
　そういう側面はあるであろう、

　ということを、

　認識しているか、いないかの差は
　人生における結構な差になるであろう、

　と思われた。


　それゆえ

　「あえて、
　　言葉を慎重、かつ丁寧に選びつつ」

　お伝えしようと考えた。


　-----------------------


　ゲームの世界であるならば、

　無課金にこだわっていては、
　ゲームを前に進められない。


　時間制限もあるし、
　課金しないとゲームをし続けられない。

　小休止小休止でいちいちうっとうしい。
　課金することによって無双する。

　無双=無敵モード。
　

　-----------------------


　そういう無双の世界に出られる、
　課金すれば。


　ゲームの世界でもそうだが、
　人生においても誰もが表立って言いにくい
　不都合な真実としてある。

　しかし、こうしてまとまった
　コンテンツとしてお話することはなかった。


　メルマガで音源の説明をするときに
　これでブーストさせましょう、

　ということはあったけれども
　それ以上の刺激的な言葉は用いなかった。


　-----------------------


　こうした話を聴いて、

　「おお！」

　と思う人もいれば、
　
　「結構きついよね、そうだったら
　　すでに詰んでるじゃん」

　と思われる人もあるだろう。

　このあと、そう思われた人への
　アドバイスもする。


　-----------------------

　
　「人生は重課金でうまくいく」

　というのは強烈なパワーワード、
　恐るべき言葉。

　問題提起に過ぎない。
　正しいかどうかは分からない。


　ただ私（鮒谷）自身は間違いなく、

　人前で言うにはばかられる言葉だから
　言ってはこなかったが、

　ビジネスを始めた
　初期の段階で気づきました。
