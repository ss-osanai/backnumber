【話題のビデオ】躍進企業の戦略を活用する

  年間300社以上の会社の現場、売場、工場に自ら足を運び、生きた最新
　情報を発掘する上妻英夫氏が、発想や視点を変え、今までの倍以上の
　粗利、集客を実現させた好調企業の社長に徹底取材を敢行。その販促
　手法、お客様へのサービス…“儲けの仕組みと仕掛け”を映像で初公開。
  http://www.jmca.biz/tape2/genba/g1/top.htm
───────────────────────────────────
■現場主義を徹底し、生の売場・工場を通してみた経営の最新情報を発信
　する上妻さんのビデオです。
　好調企業の【儲けの仕組みと仕掛け】が映像で初公開されているので、
　このビデオを見るだけで多大な経営におけるヒントを掴むことができる
　と思います。

■このビデオのプライスを見て、正直、高いと思われる方もあれば、安すぎる！
　とびっくりされる方、二通りにわかれるとおもいます。
　そして利益を出す経営者は例外なく「安すぎる！」と驚かれる方。

■当然のことですよね。優れた教材を購入することで、明日から、
　即経営に反映させ、次の決算のときに数百万円、数千万円の利益を生み出す
　ことが出来ていたなら、それは安すぎる買い物ということになります。
　しかもこれは一年限りの事ではないのです。ここで得た知識や発想は永遠に
　企業経営に活かされるのですから。

■こんなこと、私（鮒谷）が口で言ってるだけ、と思っていますか？
　４月５月で私が入金した、セミナー等の代金、上位３位の金額を明示します。

　　有名人です。


■テッド小笠原さんから先日電話があり、ボッキーさんと、IT義塾・塾長さん
　をご紹介したい、と。（すみません、謎のハンドルネームが続きます）

　のサイトだけは欠かさず見に行っておりましたので、これは！と思い
　一も二もなく、是非！ということで、顔合わせの場を設定して

　同じく、パワーあふれる、激しい方です。たまに（まれに？）メールの
　やりとりをさせて頂いてます。（栢野さん、みなさん、勿論ご存知
　ですよね）

　今、栢野さんの日記とボッキーさんの日記が、ビジネス系を含めた
　楽天日記の二大巨頭です。

　この二人がトークをされるというのですから大変です。　


■このセミナーのタイトルは、

　【インターネットで運命を変えよう！セミナー】
　 http://www.motion-network.com/tokyoseminar01.htm

　最短で（というと語弊があるかもしれませんが）自分の運命を
　変えようと思ったら、インターネットというツールを使うのが
　もっとも早いかもしれません。

　かくいう私も、一昨年の12月に、会社倒産、事業所撤収の憂き目にあい、
　１年ちょっと前まで、スーツを着て外出するも、いくところがなくて
　数ヶ月間、毎日、喫茶店で12時間くらい、１日10冊ほど本読んでたんです。
　（ご近所さんの手前、スーツ着て出社する、って漫画みたいなこと
　やってました　笑）
　
　でも、インターネットのおかげで運命激変です。


■実はボッキーさんも、栢野さんも、インターネットでご自身の
　人生を激変させた方。詳しくはセミナーでお話をされると思います。（多分）

■その後は、アタッカーズビジネススクールへ。

　お豆腐屋さんで東証マザーズ上場を果たした「篠崎屋」の樽見社長の
　お話。

　豆腐会社で上場する、というのはすごいとおもいませんか？
　わたしはすげぇ！って思いました。


■１時間半のお話＆30分の質疑応答があったのですが、いろいろ
　お話をなされた中で非常に印象に残った言葉は、

　▼やって失敗するリスクを選べ。行動しないことのほうが、
　　現代にとっては非常に大きなリスクである。

　　定年間際になって、会社になにかあって放り出されても、
　　失敗を繰り返しながらやってきた人は強い。なにもやって
　　来なかった人が、いきなり放り出されるほうがはるかに
　　大きなリスクである。

　▼とにかく動く。人の三倍働く。
　　一日は24時間、みんな同じ。人の三倍密度濃く生きる。

　▼死に物狂いで動き回るなかで活路が見えてくる
　　最初から成功のゴールが見えているわけではなく、やって
　　いくうちに、考え続けるうちに、試行錯誤を続ける間に、
　　より良い一手を繰り出していける。
　　そして成功とはその積み重ねである。
　　人と同じ事をして、人より楽をして、事業がうまくいくなんて
　　ことはあり得ない。

　といったところでしょうか。

　いずれにしても、楽して成功するなんてあり得ない。
　と膝を打って、その通り！と改めて思った講演でありました。

＜今日のピークパフォーマンス方程式。＞

───────────────────────────────────


■ボッキーさんにしても、栢野さんにしても、樽見さんにしても、
　お会いしていると大変な自信をそこに感じるのです。

　
■なんでこの人たちはこんなに自信に満ち溢れているんだろう？
　ちょっと考えてみました。
　そして共通項を抽出してみました。

　すると、見つかりました！

　いずれもいずれも、他を超絶する量稽古をこなされているということ
　です。


■毎日、毎日全力投球で生きているのです。
　妥協なく。普通にのほほんと生きている人と較べると、おそらく
　その密度は５倍～１０倍くらい、詰まっているのではないでしょうか。

　わたしも、結構密度詰まっているほうだと思いますが、この方々
　と較べるとどうにもこうにも、恥ずかしくなってしまいます。

　【量稽古＝密度×延べ時間】で計算するとすると、この３人とも10年、
　20年の永きにわたり、【濃厚密度×超・長時間】で蓄積してこられた
　ものがありますから、ちょっとやそっとで、追いつけるものでは
　ないでしょうし、だからこそそれぞれの分野でオンリーワンを貫く
　ことが出来るのでしょう。


■いやぁ、反省です。
　
　いずれにしても、自信の源は圧倒的なる量稽古から来る。
　天才はどうだか知りませんが、秀才、凡才、落ちこぼれは、徹底して
　同じ事をやり続けるしかないのです。

　昨日感じたことであります。
