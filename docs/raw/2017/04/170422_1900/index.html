【 幸福感は「術→説→論→学→道」の歩みから、もたらされる 】
　　　　
───────────────────────────────────



■以前のメルマガで


　【術 → 説 → 論 → 学 → 道】


　という話を書いたことがあります。


　どんな世界でもそうですが、
　たとえば営業であれば

　「営業における（一つ一つの）術」

　を極めようとすれば、
　術の上位概念であるところの

　「営業に関する、
　　各テーマについての仮説（説）」

　を立てる必要が出てきます。



■仮説を検証していく中で、

　これはもう間違いない、と思われる
　中心的な概念が生み出されたら、

　そこからは「説」を卒業し

　「（営業）論」

　が生まれることととなるでしょう。



■営業における、それぞれのテーマにおいて、

　複数の確立された論が統合されてくると
　そこから独自の
　
　「営業学」

　が立ち上がってくることとなるはずです。


　そして最後、いくら「学」として
　体系立てられたものを構築したとしても、

　最後は「知行合一」ではありませんが、
　実践できているかどうかが問題になるのです。



■実践の道は、修業の道であり、
　これで極めた、ということはありません。

　さらには気を緩めれば、
　あっという間に転落する。


　そんな世界を歩むことになるから、
　最後は

　「営業道」

　に至るのです。



■ここまでは、たまたま一つの事例として

　「営業」

　についての話をしましたが、
　どんな分野においても同じで、


　何か一つのことを追求していくと、

　【術 → 説 → 論 → 学 → 道】

　の順に、

　はじめは進化、やがては深化していく
　こととなるでしょう。



■さらに突っ込んで考えてみると

　「幸福感」

　とは、一つのことに徹底的に打ち込み、
　自分なりの

　【術 → 説 → 論 → 学 → 道】

　の階段を上がっていくプロセスにおいて
　感じられるものなのではないか、

　そんな風に考えています。



■話は変わりますが、
　以前（今から４ヶ月ほど前）、


　＜4839号 各種媒体の有機的発信によって、
　　　　　 圧巻の教育効果を狙う＞

　http://www.2nd-stage.jp/backnumber/ppf_full/2016/12/4839.html


　という記事を書いたことがあります。

　
　この記事は私にとって、
　非常に思い入れのあるもので、
　
　よろしければぜひ、時間を取って、
　じっくりとお読みいただきたいのですが、


　なぜ、私（鮒谷）が

　「音源販売ビジネスを始めたのか」

　の理由を余すところなく、記しています。


　
■熟読頂ければお分かりになるかと思いますが、

　「音源販売」

　も、つまるところビジネスのために、
　というよりも、

　【術 → 説 → 論 → 学 → 道】

　のプロセスを進むために
　絶対に通らなければならぬ道である、

　そんなふうに考えているのです。



■上記のバックナンバーでご紹介した通り、

　伊藤和夫先生は、自らの追求すべき道が

　「英語教育」

　にあると定められたわけですが、


　長らく教壇に立ち続け、
　膨大な量のテキストを作成し続けたのも、

　ひとえに

　【術 → 説 → 論 → 学 → 道】

　の歩みではなかったかと
　思われてなりません。


　
■分野は異なれど、目指す世界は同じ。


　そう記すと、天下の伊藤和夫先生と
　お前（鮒谷）を同列に置き、

　対比して論じるとは何ごとか、
　と不快に思われる向きもあるでしょう。


　でも、本当にそのように思われるし、

　だからこそ、どれだけ時間がかかっても、
　私も、

　後世の、多くの人から評価されるに
　足る独自の
　
　【術 → 説 → 論 → 学 → 道】

　を打ち立て、歩みたいと考えているのです。



■ちなみに、
　同じく先のバックナンバーにおいて


　-------------------------------------
　　
　『ビジュアル英文解釈』における転換とは、
　　つきつめて言えば、次の点にあった。
　
　1.　「体系」を隠すこと

　2.　「構造」よりも「流れ」を重視すること

　3.　「現場性」の取り込み

　-------------------------------------


　その他、いくつかの文章を紹介しておりますが、
　私（鮒谷）が目指す世界も、

　まさにこの世界。



■というよりも、この文章に触発されて、
　そう決意した、

　というのが正解です。


　数多くのセミナーやコンサルを行う中で、

　言葉に載せられない、漠然とした
　自分の行っていることに対する疑問が、

　これら文章に出会うことによって氷解し、
　進むべき道が定まりました。



■現在、私が、

　放談会やセミナー、年間プログラム、
　各種音源、個別コンサル、

　といったサービスを複合的に提供する中で
　意識して行っているのは、


　ひとえにここで言われているところの


　-------------------------------------


　1.　「体系」を隠すこと

　2.　「構造」よりも「流れ」を重視すること

　3.　「現場性」の取り込み


　-------------------------------------


　読者の方は、一冊の参考書を学んで行く中で、
　無意識のうちに全体をらせん状にたどり、

　結果的に「建築物全体」を
　体験していればいいのである。


　-------------------------------------


　自己主張としての「体系」は、

　教える側がプログラムとして隠し持っている
　だけで十分なのであって、

　読者の方は、体系の順序によってではなく、

　実際に出会う英文にぴったりより沿って
　学ぶべきなのである。


　そのような考えの下に、
　『ビジュアル英文解釈』は書かれている。


　-------------------------------------


　といった世界の実現を、

　自分の専門分野において実現せんという
　思いから。


　
■以上、記したことを理解してもらいたいとは
　思っていませんが、

　とはいえ、

　「せめて、その匂いだけでも
　　感じて頂ければ嬉しい」

　と思って日々、価値の提供に努めているのも
　また事実。
