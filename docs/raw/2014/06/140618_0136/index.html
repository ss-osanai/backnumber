▼個別コンサル２件

───────────────────────────────────



■こんにちは、鮒谷です。


　昨日は個別コンサル、２件。



■以下、少し前に頂いた個別コンサルのご感想を
　ご紹介させていただきます。


　（ここから）
　----------------------------------------------------------------

　
　必要なのは「5%の覚悟」だけなのだ、
　と思いました。
　
　残りの95%は、
　すべて鮒谷さんが用意して下さいました。
　
　
　「自分がなりたい自分」を100%としたとき、
　
　個別コンサルに赴く前の僕は、
　覚悟という5%しか所持していなかったのです。
　
　
　というより、
　
　「そもそも何が100%なのか」
　
　もわかっていませんでした。
　
　
　たった90分のあいだに、鮒谷さんは
　
　「何が僕の100%なのか」
　
　を教えて下さり、さらに
　
　「足りない95%」
　
　のすべてを、言語化してお伝え下さったのです。
　
　
　僕がコンサル後にするべきことは、
　その"95%の野原"に、"5%の火"を放つことだけでした。
　
　まさに「燎原の火」のごとく、
　景色は明らかな熱とエネルギーを帯びて
　変容し始めています
　
　(コンサル後、まだ一週間と少ししか経っていない
　にも関わらず、です)。
　
　
　もしかすると今回のコンサルの一番の収穫は、
　
　「持ち物は、5%の覚悟だけでいい」
　
　と学んだことかもしれません。(笑) 
　
　
　それさえあれば、
　あとは鮒谷さんが「舞台」を用意して下さる。
　
　もちろん、その上で舞う準備(覚悟)は必要だけれども、
　未だ「舞台」の何たるかが見えていなかった僕にとっては、
　それは非常に衝撃的な気付きでした。
　
　
　一旦付いた火を絶やすことのないよう、
　
　これからも鮒谷さんのコンサルを
　受け続けていければと思います。
　
　
　鮒谷さん、セカンドステージの皆さん、
　今回も本当にありがとうございました！
　
　今後ともどうぞよろしくお願い致します！


　　　　　　　　　　　　　　　　（文筆家　Ｒ．Ｗ 様）

 【　「平々凡々、平穏無事の人生」を踏み外すとき　】

───────────────────────────────────



■こういう仕事（？）をしているので、
　これまでにたくさんの、

　「人が劇的に変わる瞬間
　　（正確には数ヶ月から数年にわたる、一定期間）」

　を見てきました。


　そこで分かったのは、

　「おれ、やります！」
　
　「私も、動きます！」

　という言葉は、

　たとえ、そのときの気持ちは本当でも、
　すぐに元に戻るので、

　信用できないし、当てにもできないこと。



■言葉だけでなく、

　現実の行動に移され、明らかにそれが継続しているのが
　認識できたとき、

　初めて、本気になられたことが分かります。


　こうして本気になり、行動に移し、
　その動きが継続するようになると、

　果たして何が起きるか。



■たくさんの仕事が入り始め、
　たくさんのプロジェクトが立ち上がり始め、

　いろんなことが同時並行で急激に、加速的に、
　増え始め、取り巻く世界が変わり始めます。


　（だからといって、
　　許されることではありませんが、

　　ここで幾多の不義理を重ねてしまうことが
　　往々にしてあります、、）



■この過程を経ることなく、
　次のステージにたどり着くことはありません。

　いくら一生懸命、本を読み、セミナーに通い、
　お勉強していても、

　行動しなければ、100年経っても、
　なんにも変わりません。



■けれども、

　いったん、取り巻く世界が変化し始め、
　さらに、さらに、行動を叩き込んでいき、

　ある程度、進んだとき、そこで初めて

　「平々凡々、平穏無事、月並み、ありふれた、ありきたり」

　といった人生から、

　（良くも悪くも）

　大きく逸脱していることに気づきます。


　そして過去の人生と訣別し、
　
　「二度と後戻りできない道」

　を歩み始めてしまった自分であることを知らされます。
