■先日来、

　ご案内するともなくご案内してきた
　音源をお聴き下さった松本さんより、

　以下の感想を頂戴いたしましたので
　ご紹介。
　

　（ここから）
　-----------------

　鮒谷さん


　早速、

　「●●●●●●」
　「●●●●●●●●●●」
　「●●●●●●●●●●●」

　を購入して読み終わりました。


　いままでも、自己啓発本は
　多数読んていましたが、

　この３冊は読んでいなかったのは
　もぐりだったのかもしれません。


　３冊とも気づきの多い内容で、
　やはり少年の立場になって、

　自分の立ち位置は今どのぐらいか
　という見方になりました。


　半面、音源で言われている
　老人の視点で見てみると、

　自分ができていることは
　そのように振舞えるし、

　できていないことは
　振舞えないと分かり、

　ものごとの理解のレベルを知る
　バロメーターとして使えると思いました。


　また、物を教える立場の人の感情を
　考えながら読み進めました。

　成功者が分け与えた得たいと
　思う気持ちや、若者を応援したい気持ち、

　成功者ならではの悩みがあるのだと
　感じました。

　そして、若者の成長を見ることが
　楽しみになってくるのでしょう。


　逆に、そのような振る舞いをしていれば、
　成功に近づけるのだと思います。

　少年の立場で見ると、
　本に書いてあることを読んでも
　実行しないことがよくあります。

　分かっていてもやらなかったり、
　言い訳してやらなかったり。


　でも、老人の立場になると、
　やるとかやらないとかではなくて、

　すでにやってきた状態になっているので
　それを真似するのは、

　未来を鮮明にイメージする、
　シーソーの向こう側に重りを置くことに
　直結するのではないか、

　と思いました。


　少年と老人の二人の視点で
　本を読み進めることによって、

　学びが倍増すると教えていただきました。
　ありがとうございます。


　（松本 慶吾 さま）


　-----------------
　（ここまで）



■松本さん、
　ご感想ありがとうございました！


　自己啓発の世界で
　昔からよく言われることに

　「成功者のふりをせよ」
　
　というものがあるのですが
　一理あるんですよね。



■できていない人が
　できるようになろうとするよりも、

　もし今、できているとすれば
　どのように振る舞っているか、と

　【未来から現在を見る視点】

　を持つことによって
　思考と行動を矯正するのです。



■そんなことに効果あるのかよ、
　と思われる方もあるかもしれませんが、

　それが大アリ、なんですよね。


　であれば、
　それを行えばいいわけですが、

　いざ、取り組もうとしても、
　具体的にどうやればいいのか、
　今ひとつよく分からない、

　というところが問題になってきます。



■その問題に対する一つの解として

　【自己啓発本に登場する
　　爺さんになりきる】
　
　というアプローチがあるわけです。



■別に婆さんでも構いませんが、

　なぜかこの種の成功哲学本に婆さんが
　登場するのを見たことがありません。

　（私（鮒谷）が知らないだけかも
　　しれませんが）



■どなたか、

　「爺さん、ときにオッサン、が
　　若者（男性）に成功譚を語る」

　という、
　ありがちな形式ではなく、

　「婆さん、ときにオバちゃん、が
　　若者（女性）に成功譚」

　という構造の、

　新しい形の成功物語本を
　書いてみてはいかがでしょう（適当）



■それはともかく、
　現在から未来を見ている限り、

　今の認識の枠組みから
　決して出られないので、


　もし今とは次元の異なる世界に
　雄飛したいと願うなら、絶対に、
　
　【うまくいった未来から
　　現在を眺める視点】

　が定位置となる、
　認識のチューニングを行うべき。



■めちゃくちゃ効果あります。

　というか、やっている人は、
　みんなやっています。


　ぜひお試し下さい。

　より詳しく学び、実践したい方は
　以下の音源をお聴き下さい。



■、、、そんなわけで（？）、現在、

　【フナタニ、秋の音源まつり】

　真っ只中の上に

　【音源リリース記念】

　ということで、


　いつもの音源よりも
　安価な定価設定をして、

　なおかつ、30％割引にて
　ご提供しているのでぜひ

　【リラックスモードでの、
　　休憩なし3時間11分ぶっ通しの放談】

　を聴いてみて下さい。

　最近考えたことを
　あれこれ語っています。

　（あと４日で値上げします）
