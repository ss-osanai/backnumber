【あと９時間】「良質三兄弟（！）」の次男坊をご紹介

2020/07/26 14:30 
■こんにちは、鮒谷です。


　私（鮒谷）は

　【人生で一番、
　　経験したくないのが後悔】

　だと考えています。

　きっとあなたも同じでは
　ないでしょうか。



■好んで、後悔したい人など
　あるはずもありません。
　
　しかし、過去を振り返ってみて
　後悔している人は少なくない、

　ようにも思われます。



■上述の通り、私（鮒谷）は昔から
　　
　【人生で一番、
　　経験したくないのが後悔】

　だと思って生きてきたので、

　
　「では、どうすれば
　　後悔の残らない人生を過ごせるか」

　を頻繁に自問自答してきました。



■その結果、

　【後悔するかどうかは
　　毎日の積み重ねによって決定される】

　という結論を導き出しました。


　いい加減な毎日を過ごしていたら
　後悔は大きくなるし、

　毎日、確信を持って
　生きていたら


　（後悔は絶対にしない、などと
　　確証を持っては言えないにせよ）

　少なくとも、
　毎日をテキトーに生きている人よりは
　
　【後悔する可能性を減じられる】

　という結論に至ったと
　いうことです。



■そんな充実感を感じられる日々を
　過ごすためにはどうすればよいか、

　という次の自問が生まれます。


　
■この問いに対して
　私（鮒谷）が出した答えが


　【良質の意思決定基準は
　　良質の自問自答をもたらし、

　　良質の自問自答は
　　良質の行動を促してくれ、
　
　　良質の行動継続が
　　良質の習慣を生み出すこととなる】


　というものでありました。



■こうした生活を現実のものとするためには
　まず最初に

　【自身が納得のいく目標設定】
　
　を行う必要はありますが、


　ひとたび感情の震える目標を定めたら、
　自ずから

　【その世界に早く到達するために
　　有効と思われる、意思決定の基準】

　に強い関心が
　生まれることとなります。



■ここから


　【良質の意思決定基準は
　　良質の自問自答をもたらし、

　　良質の自問自答は
　　良質の行動を促してくれ、
　
　　良質の行動継続が
　　良質の習慣を生み出すこととなる】

　
　という法則が導き出されたわけですが、


　このプロセスが
　絶え間なく行われるようになれば、

　それなりに満足のゆく（少なくとも
　後悔しない）人生を生きられるのでは、

　と思われます。



■反対に、

　上記が一部、あるいは全く行われて
　いなかったことを思い出したとき、

　後悔が残るのではないでしょうか。



■この観点でいくと、

　少なくとも私（鮒谷）は
　ここ20年ほどの間の行動について、

　一切の後悔はありませんし、


　そうした時間を過ごさせてくれる
　土台となった、それ以前の時間も

　「全肯定」

　できるようになりました。



■少なくとも過去の人生を省みて、

　このようなことを自信を持って
　言えるようになるかどうかは


　【良質の意思決定基準】
　【良質の自問自答】
　【良質の行動】


　という
　
　【良質三兄弟（！？）】

　をきれいに串で貫き、
　一直線上に並ばせられるか、

　にかかっていると考えます。



■先日来、ご案内してきた

　【自問自答音源】

　は、

　【良質三兄弟の次男】

　の位置づけと言えるでしょう。



■そして、あなたも既にお分かりの通り、
　
　【長男（良質の意思決定基準）】

　と

　【三男（良質の行動）】

　には様々なバリエーションが
　あります。



■たとえば、

　コミュニケーションにおける
　さまざまな意思決定基準と、

　そこから生まれる毎日の行動、


　あるいは、

　時間管理における
　さまざまな意思決定基準と、

　そこから生まれる毎日の行動、


　あるいは、

　セールスにおける
　さまざまな意思決定基準と、

　そこから生まれる毎日の行動、

　などなどなど、、、



■しかし、唯一

　【次男（良質の自問自答）】

　だけは、

　【単一の習慣】

　つまり、


　数ある、
　良質の意思決定の基準に基づき
　
　【頻繁に自問自答を繰り返しているか】

　だけを問題にしていて
　そこにはバリエーションはなく、

　ただ、できているか、どうか、

　の問題となっているということ。



■つまり

　意思決定基準と
　そこから生まれる行動については、

　原理的に無数のコンテンツ（音源！）が
　生まれることとなりますが、


　自問自答音源のみは
　
　【最初で最後の単一音源】

　になるという構造を
　お分かり頂けるのではないでしょうか。



■実にこれこそが

　【あなたに、なにがなんでも
　　自問自答音源を聴いてもらいたい理由】

　なのです。


　なので他の音源はともかく、
　ぜひとも

　【良質三兄弟】

　のうちの　

　【次男（良質の自問自答）】

　音源だけは、

　漏れなく、欠かさず、
　皆さまにお聴きいただきたい、

　そう願っております。



■さんざん、語ってきたので、
　もう、これ以上は語りません。


　当音源ですが「モニター価格」で
　お求め頂けるのは、いよいよ

　【あと９時間】

　となりました。
