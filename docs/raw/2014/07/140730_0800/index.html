【学問とは驚く能力】変わりたい人が変われるようになる「８つのポイント」セミナー

2014/07/30 08:00 
■こんにちは、鮒谷です。


　例によって直前のご案内（４日後！）となり恐縮ですが、
　今日は今週末開催のセミナーの紹介をさせて下さい。

　

　　＜【 変わりたい人が変われるようになる
　　　　　　　　　　　　「８つのポイント」セミナー 】＞


　というタイトルで開催いたします。　


　以前、行っていた３日間セミナーの総集編（？）
　のようなものでありつつ、

　さらにそこに、

　「これだけは、どうしてもはずせない」

　というコンテンツを付加した形でお届けします。



■３日間セミナーの復習として、
　
　あるいは、

　初めてお越し下さる方にも必ずたくさんの気づきを
　お持ち帰りいただきます。


　「変わりたいと思っているけれども、
　　どうしても変われない」

　という深刻な悩みをお持ちの方に、
　ぜひお勧めいたします。


　メルマガ１年読むより、いい話が聞けるし、
　大きなインパクトもあるはずです。きっと。



■どんな雰囲気で行われるかは、こちら（↓）


　＜過去のセミナー参加者さまのご感想＞

　http://www.2nd-stage.jp/seminar/


　にご掲載させていただいた、過去、頂戴した
　たくさんのご感想からイメージして頂けましたら幸いです。



■お申し込みは以下よりお願いします。


　セミナーに参加下さる方の成果に
　焦点をあわせるため、

　顔の見える、双方向、少人数で行うつもりです。


　なので、たくさん申し込みを頂いて
　お断りするのも申し訳ないので、
　
　まったく営業色のないご案内で恐縮（？）ですが、
　
　
　セールスレターの専門家が見たら怒り出しそうな、
　ほとんど何の案内にもなっていない、こんな文章を読むだけで

　「ぜひ話を聞きたい」

　という奇特な方がいらっしゃいましたら、
　ぜひお申込みをお待ちしております（汗）



■そんな奇特な方に対しては、

　私（鮒谷）もなおのこと、
　一生懸命、お話ししたくなるというものです。


　お申し込みはこちら（↓）からお願いします。
