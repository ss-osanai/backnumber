───────────────────


■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------

　起業のアイデアは
　模倣が一番やで。

　ショボい自分の頭脳で
　考えたところで

　大したプランは出てこんよ。


　人間関係を通じて、
　儲かってる事業の情報を
　いち早く仕入れ、

　核心を教わり
　（あるいは推察、洞察し）、

　小さく試した後、
　猛烈な速度で磨きをかけて、
　儲かる事業に育てていくんだわ。

　----------

　「その辺でブラブラしとるスモビジ
　（スモールビジネス）オーナー」

　案外そういうおっさん（に限らんが）が
　金持ってる、

　いう話が『隣の億万長者』に
　これでもか、

　いうくらいに出とるから
　未読の人は読むといい。

　----------

　昔、経営者仲間で話をしてたとき、

　「デキる経営者は、
　　変数を極小化する」

　いう話が出たのよな。

　変数が多いと
　計測が難しい上に混乱する。


　事業のカギとなる要素の少ない事業は
　コントロールしやすい上に、

　ハマるととてつもなく
　成長させられる。

　商売は大興奮を伴う急成長を目指す、
　知的ゲームと思うといい。

　----------

　高学歴エリートに対して、
　ビビる必要なんかあらへんで。

　（皆とは言わんが）
　ほとんどはフットワーク激重の上、

　恥をかくのを極度に恐れる
　雑魚やからな。


　失うものを持たん落伍者は
　むしろ有利に立ち回れる。

　格好つけてる連中が入ってこれない、
　入りたくない市場で大量行動をして
　負けるはずがないんだわ。

　----------

　夢中になれる対象がないと
　地味にHP削られるんよ。

　「やるべきことを処理してる間に、
　　今日も一日終わってもうた」

　が一週間、一ヶ月、一年と続けば
　ワイなら狂ってまう。


　今、打ち込めるものがないなら、

　頭の中を検索したところで
　「ないものはない」んだわ。

　外の世界を探索するより、他にない。

　----------

　目先のカネ欲しさに、

　ストレスに耐えながら
　小銭を追うのは止めたほうがいいよ。

　自由奔放に生きたほうがアイデアも
　湧くし、トータルで儲かるから。


　余計なストレスを抱え込むと
　思考能力が大幅に減衰するんだわ。

　それが分かってからというもの、

　とにかくストレスフリーな状態作りに
　専念してしてきたわ。

　----------

　この人に勧められたら何でもするし、
　何でも買う。

　過去、そんな人（除く詐欺師）に
　出会ったことのある人は幸せやで。


　なぜなら理想の顧客の心理を
　すでに体験し、理解しているから。

　あなたが顧客に対して
　その状態を逆算で作り出せば、

　自ずからモノは売れていく。

　その状態作りこそが大変なんやけど。

　----------

　商売において、

　かゆい場所をピンポイントで
　かいてくれる人と、

　かゆいところの周辺を撫でるようにしか
　かいてくれない人があるんだわ。


　顧客すら気づかなかった課題を
　浮き彫りにして解決策を提示する人と、

　寄り添うことなく、それらしい
　提案をしてお茶を濁す人、

　どちらが信頼され、
　儲かるかということよ。

　----------

　奇跡を起こすところに
　人生の醍醐味があるんだわ。

　ジャイアントキリング
　（番狂わせの大物喰い）

　いう言葉があるやん。


　その機会を狙いに行くから
　モチベーションが上がり、

　闘争心が生まれ、
　未来に希望を持てるのよ。

　最初からあきらめてたら不戦敗。

　それで楽しく盛り上がれると
　思う方がどうかしとるやろ。

　----------

　商売において、できることは
　「できる」と言っていいけど、

　できんことまで雰囲気に流されたり、
　仕事欲しさのあまり、

　「できる」と言ってしまうと
　後で大変になるから厳禁やで。


　できないことはハッキリ「できん」と
　伝える潔さが信頼につながるから、

　断るのが苦手な（昔の）ワイみたいな人も
　頑張ってやるんやで。

　----------

　「ウソから出たまこと」
　いう言葉あるよな。

　ワイはダメリーマン時代、

　社外の友人に「経営者になる」と
　ホラを吹いてたんよ。

　本業でも一向に成果を出せんのだから、
　もちろん全くアテはない。


　でも目標と宣言の力は偉大。

　数年後、

　「なんで、私が東大に！？」

　ならぬ

　「なんで、私が経営者！？」

　言うとった。

　----------

　たった一つ「興味の火種」が見つかれば、
　そっから人生は動き出す。

　強い関心を持つ対象があれば、
　そこを起点に世界を広げていけるのよ。


　今、全力で打ち込めるものがなければ、

　好奇心全開に許しを与えて
　「興味の種探し」をするところから。

　興味の連鎖反応が始まれば、
　自ずから本気スイッチが入るんよ。

　----------

　アンテナを広く張り、
　お試しチャレンジの数を増やす。

　これが「全集中できる対象」発見の、
　確度の高い方式やで。


　「ちょっとでも 心揺れたら 
　　即アクション」

　で、分からんものでも次々にGOを
　出すうちに没頭対象が見つかるのよ。

　そこを起点に一点集中、
　結果を出せば評価も高まり、

　人生が変わっていく。

　----------

　人生を貫く大テーマ発見の裏には、

　「日の目を見なかった
　　無数の興味の芽」

　があるんだわ。


　とはいえ「無駄撃ち」を恐れて
　挑戦をためらうのは大きな間違いやで。

　事後的に見れば無駄に思える
　挑戦の数々も、実は必要な過程。

　大量の探索があったからこそ
　没頭対象に出会い、

　人生の突破口が開かれるのよ。

　----------

　「分かるからやる」ではなく
　「分からんからやる」が正解やで。

　分からんからやる、は
　人生を変える魔法の言葉。

　「分かったら動く」は
　「分からんかったら動かない」の裏返し。


　動かん限り、対象の文脈を
　把握できる日は永遠に訪れんから、

　「分かったら動く」と言ってる人は
　永久に動くつもりのない人よ。

　----------

　身銭を切らん人は
　本当の意味では学べんよ。

　起業してからも（もちろん自腹で）
　数々の勉強会やセミナーに
　参加してきたけど、

　企業派遣の人のやる気の無さは
　毎度、際立ってたな。


　他人事やからどうでもええけど、

　「そんな姿勢では意味がないよ、
　　カネの無駄遣い」

　と思いつつ、
　ワイは真剣に受講しとったよ。

　----------

　「真剣やから身銭を切れるのか、
　　身銭を切るから真剣になれるのか」

　は、ニワトリと卵みたいな
　永遠の問いに見えるけど両方、正解よ。


　真剣な気持ちとお金を払う決断は、

　互いが良い方向に絡まり合い、
　どんどん本気度を高めていくんだわ。

　この循環に入った人は
　加速するに決まっとる。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
