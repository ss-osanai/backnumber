■積み重ねられた時間には

　【絶対的な価値がある】

　と思っています。


　私（鮒谷）が、

　日記を30年、メルマガを20年、
　毎日、書き続けてきた事実は、

　決して裏切りません。



■また、個別コンサルも概算ですが
　のべ9,000件以上、行ってきて、
　
　自分の中に編み込んできた経験や
　知恵もまた、

　私を裏切ることは
　絶対にありません。



■しかも、これらを決して漫然と
　行ってきたわけではないことは、

　少なくとも５年、
　ないし10年以上前から、

　メルマガを読まれている方であれば
　ご理解頂けると思います。



■そのとき、そのとき、

　自身が許容できるギリギリの
　負荷をかけ続けてきたので、

　その事実そのものが
　圧倒的、絶対的な自信になっています。



■この間の膨大な蓄積があるからこそ
　手ぶらで人前に出ても、

　（自分の専門分野であれば）

　３時間でも５時間でも10時間でも
　ノンストップで話し続けられます。



■実際に、十数年前から

　（タイトルは安直ですが、
　　中身はガチの）

　「３日間セミナー」

　という名のセミナーで
　３日間にわたり、

　「のべ30時間近く、
　　手ぶらで話し続ける」

　ということも行ってきました。

　（今も年間プログラムで
　　終日、話し続けています）



■要は長い時間をかけて

　【継続的に、かなりハードな
　　脳みその筋トレを行ってきた】
　
　ということです。

　だから自信があるのです。



■自分でこんなことを言うのも
　ちょっとアレですが、
　
　【学びの総量と自らの実践量と
　　コンサル事例の量】
　
　の半端なさは、
　比肩するものはない、

　くらいに思っています。


　リソースを全振りしてきましたからね。

　（その代わり、ほかは全て捨てたので
　　何もできません）



■論より証拠、口だけではなく、
　数多くのご感想も頂いてきました。


　各種セミナーまで含めると、
　おそらく

　「万の単位」

　で頂戴してきているはずです。



■まったくまとめきれていませんが、

　これまでの個別コンサル感想
　（の一部）だけでも、

　これくらいあります。

https://ssdlfiles.s3.amazonaws.com/testimonials_consulting_career.pdf

　コンサルについてのご感想＞
https://ssdlfiles.s3.amazonaws.com/testimonials_consulting_manager.pdf
　


■おそらく、

　これだけの数と質の感想を
　頂戴してきた人は日本にいないはず、

　つまり

　【（自身の専門分野については）
　　名実ともに日本一である】

　くらいに思っています。

　繰り返しになりますが、
　人生、捧げてきましたからね。



■そんなコンサルですから、

　相応のチャージをしても
　バチは当たらないと思っているし、
　
　後ろめたいところもなにもないから
　正々堂々とご請求できます。
　


■そんなわけで、
　通常の個別コンサルは

　「１時間12万円（税抜）」

　にてお受けしております。
　（正直、それでも安いと思っています）

　

■ただし、本サービスは
　誰でもお申し込み頂けるのではなく、

　過去、

　「目標設定放談会にお越し下さった方」

　もしくは、

　「目標設定音源を
　　お聴き下さったことのある方」

　のみに限定して、

　【１時間50,000円（税抜）】

　にて承っております。


　※Zoomでの実施となります

　※「目標設定」以外のコンサルは行いません

　※状況によって１時間を超えるケースも
　　ございますので、できましたら
　　後ろの時間に余裕をもたせて下さい

　※「特別価格」につき、終了後、
　　感想のご送付をお願いしています



■お申込み時に、

　目標設定音源（三部作すべて）を
　お聴き頂いているのが理想ですが、

　１本目（基本編）のみを
　聴かれている方でも大丈夫です。


　※この機会に初めて３部作を
　　お求め下さる方も対象といたします。

　（詳細は本メール末尾をご覧ください。
　　特別割引価格にてご提供いたします）



■ハッキリ言って、

　ド●・キホーテさんではないですが
　激安を通り越して

　「驚安」

　だと思っています。


　あるいは、
　往年のダ●エーさんものけぞるほどの

　「よい品をどんどん安く」

　の価格破壊。



■それでさえ、

　「こうしたテーマに関心のある方でも
　　申し込まれない方がいらっしゃる」

　という事実は、率直に言って

　「価値を価値と伝えきれていない
　　自身の力不足」

　以外の何物でもないと考えています。



■大口を叩いていると、
　言われるかもしれませんが、

　本当にそう思っているのだから
　仕方ない。


　※テーマに関心のない方は、
　　もともと縁のない方なので、

　　それはどうしようもないし、
　　そこについては何も思いません。

　--------------

　今回が一対一の
　コンサル初受講でしたが、

　鮒谷さんはフルスロットル、
　パワー全開で、

　Zoom からでも
　その熱が伝わってきましたので、

　良い形で新年のスタートダッシュを
　切ることが出来そうです。

　--------------

　今回目標設定コンサルを受けた、
　その冒頭、その瞬間で、

　バールとハンマーと金ダライで
　同時に殴られたような衝撃を受けました。

　--------------

　昨年、仕事のポジションが
　かわったことで

　目の前のこと（第一領域）をこなすのに
　精一杯だったのですが、

　鮒谷さんのコンサルではるか上空に
　意識を連れて行ってもらえました。


　（ご感想全文はこちら）
　https://ssdlfiles.s3.amazonaws.com/testimonials_consulting_mokuhyo.pdf

　--------------

　1 月 3 日より、闘魂を注入して
　いただきありがとうございました。

　また、最後の一押し（ひと蹴り（笑）？）も
　ありがとうございました。

　--------------

　今回のコンサルでは、日常では
　決して触れることのできない、

　私にあったピンポイントの事例を
　次々に提示して頂き、目から鱗でした。

　時間をお金で買うためにも、もっと
　がっつり稼ぎたい意欲が出てきました。

　--------------

　目標の設定方法に
　今ひとつ自信が持てない、

　そもそも
　目標の立て方が分からない、

　という方は、迷わずコンサルを
　受講されることをお勧めします。


　自分の中で迷走するよりは、
　言語化のプロの鮒谷さんの教えに沿うことが
　キャリア設計の近道になると思います。


　（ご感想全文はこちら）
　https://ssdlfiles.s3.amazonaws.com/testimonials_consulting_mokuhyo.pdf

　--------------

　鮒谷さんが放つ一つ一つの言葉と
　比喩にバンバン心を突きうごかされ、

　更に、たとえ話、リアルな話、
　こんな事メルマガで書けないですけど、

　の前置きの後の話など、

　聞けば聞くほど自分のしょぼさや、
　やらないといけない事をやっていない
　自分が脳内に湧いてきて、

　頑張ろうという気持ちの前に、
　やらないとこれヤバイわ！！

　という気持ちにさせて頂きました。

　--------------

　私は今までに
　５回コンサルを受けていますが、

　現在はそこにかけた費用の
　10倍以上稼げています。

　これが鮒谷さんの言う
　「10倍返し」なんだな、

　と事後的に実感し、自らの体験を
　通して勉強させて頂きました。

　--------------

　今回、このタイミングで、
　コンサルを受講させて頂いて
　本当に良かったと思っております。

（申込させて頂いた自分の判断は
　間違っていませんでした。

　このタイミングを逃すと、
　迷走期間がさらに長期化するところでした）


　（ご感想全文はこちら）
　https://ssdlfiles.s3.amazonaws.com/testimonials_consulting_mokuhyo.pdf

　--------------

　この機会を逃したら、
　もしかするとですが、

　あと数年、もしくは永遠に同じことを
　繰り返していたかもしれません。。

　--------------

　コーチングとも違いますし、

　答えは自分自身にあるけど、
　答えを簡単にはもらえない、

　それでいて脳内に
　ガツンとくる刺激がきますので、

　それを求めて定期的に
　コンサルをお願いしています。

　--------------

　鮒谷さんから俯瞰して見て頂くことで、
　見えなかった自分が見えてきますし、

　まだ見ぬ世界も
　コンサルの間にたくさん見せてもらい、

　世界が拡がり、目標が耕され、
　目標設定コンサルを受けさせて頂いた
　目的は期待以上に達成されました。
