■こんにちは、鮒谷です。


　
　昨日は朝から一日中、
　ダイヤモンド社のドラッカーセミナーに参加しておりました。


　「聞いたことのある話を繰り返し聞く」
　
　のは、本当に大切ですね。

　生涯、学び続けるつもりです。




■それでは

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　当社の経営指標は「お客さまから頂く感想の数と質」　】

───────────────────────────────────



■経営者にとって必要なのは、
　
　「自社の成果を測る指標を決める」
　
　こと。
　
　
　「売上・利益の数字を指標とする」
　
　ことはもちろんですが、それ以上に、
　
　「より、顧客に提供する価値に照準を合わせた指標」
　
　を持つべきではないかと思われます。

　（経営数字が、必ずしも顧客満足を反映していないことも
　　多々あるため）
　
　
　
■ですから、たとえば、当社でいえば、
　
　「お客さまから頂く感想の数と質」
　
　を指標として設定しています。
　
　（数はともかくとして、
　　内容については定性的なものとなりますが）


　お客さまから頂くご感想は、

　「マーケティング的に、
　　集めたら売上向上に役立つのでもらう」

　といった目的、つまり

　「もらうためにもらう」
　
　といった安易なものではなく、
　
　
　「自社の使命がどこまで実現されているかが
　　ダイレクトに分かるバロメーター」
　
　であるが故に、最重視しているわけです。
　
　
　
■そのような指標を掲げるとともに、
　頂いたお声を外部に公開することによって、
　
　「自らに対する縛りとする」

　という意味合いもあります。


　良いご感想を頂けば頂くほど、それをご覧になり、
　新しくお客さまになって下さった方が、

　「最低でも、ご感想と同程度の品質」

　を期待されることになる。

　これが強烈なプレッシャーとなるわけです。



■また、公開を前提とすることによって、
　より多く、より密度の濃い、

　そんなご感想を頂戴すべく、より一層、
　サービス品質を高める強い動機ともなる、

　ということにもなるのです。


　さらには、なにより

　「人さまのお役に立てている！」

　という人間の根源的欲求に基づく実感、歓びが、

　事業に対する取り組みを真剣なものと
　させてくれることともなります。



■ところで当社が過去、

　頂いてきたお客さまのご感想は、例えば、
　以下にてご紹介させていただいております。

　
　＜当メルマガに対するご感想（3229通）＞
　
　http://www.2nd-stage.jp/customer/
　

　※日々、たくさん頂戴しており、

　　更新が追いつかず、
　　もう何年も前から更新しておりませんが、
　　
　　この10年以上の間に、延べにすると

　　「万単位」
　
　　のご感想を頂いてきました。



■あるいは、
　　
　
　＜これまでに開催したセミナーのご感想＞
　
　http://www.2nd-stage.jp/seminar/

　http://www.2nd-stage.jp/houdankai/testimonials/henkaku.html
　
　
　＜個別コンサルティングのご感想＞
　
　http://www.2nd-stage.jp/consult/testimonials.html
　
　
　といったものも掲載させて頂いております。



■繰り返しになりますが、これら

　「お客さまの声」
　「お褒めの言葉」

　を義理ではなく、心からの文章を頂けるようにと
　真剣に仕事に取り組むことが、
　
　なによりも、お客さまのためになり、
　自らの歓びともなり、
　

　結果として、

　それが当社の信用や信頼という無形資産となって、
　業績にも跳ね返ってくるもの、
　
　そのような認識をしているのです。
　
　
　
■よくビジネス関連の本を読むと、
　
　
　「お客さまの声を頂きましょう。
　
　　そうすると、それにつられて他のお客さまも
　　やってきますよ」
　
　
　的な話が記されていることを見かけますが、
　本質を見誤り、表面だけを見てしまうから、

　「商売繁盛のためには、
　　お客さまの声を捏造せよ」

　といった本末転倒の話になるのです。


　（実際に「捏造」を推奨するコンサルタントさんの話を
　　聞いたこともあります）



■いうまでもなく、これはとんでもないことで、

　お客さまの声をもらうこと自体が
　目的なのではなく、
　
　
　【 自社の商品・サービスに対する
　　 お褒めの言葉や感謝の言葉を頂くような仕事をする 】
　

　ために、

　顧客からのフィードバックを頂戴するシステムを
　構築する、
　
　という流れが本道ではないかと考えております。
　
　

■その蓄積が結果として、

　いまだ商品やサービスをご購入いただいたことのない、
　潜在的なお客さまに対する

　「強力な営業支援」

　となるのは勿論です。
　

　これが「売らずに売る」ための強力な武器となる
　のはいうまでもありません。



■だから

　「情けは人のためならず」
　
　ではありませんが、
　
　「顧客への貢献は、顧客のためならず」

　ということもできるでしょう。　


　
■以前、弁護士の高井伸夫先生から
　
　「天下の大道を歩め」
　
　と教えて頂いたことがありますが、
　策を弄し、手練手管でお客さまを丸め込む、

　という姿勢ではなく、私も

　「お客さまに尽くし、お客さまに評価を頂く」
　
　という大道を歩みたいと考えております。
　
　
　
■このような理由から、当社では、

　「お客さまにどれだけ貢献できているかを
　　客観的に判断する指標として、
　
　　お客さまから頂戴するご感想の数ならびに質に
　　徹底したこだわりを持って取り組んでいる」

　わけです。
　
　
　これについてお話しすれば、まだまだ語りたいことが
　あるのですが、
　
　長くなりますので今日はこの辺で。
