　書いて×書いて×書き続け、

　＝【良き意思決定基準の方策】

　さらに大きくして安定的におろせるように、

　【聞いて→書いて→実行】

　この道を行くと渋滞だから、この道がいいですよと、
　
　具体的で実用的なアドバイスまであるのです。


　また、お声の強弱・抑揚・メリハリ・高低・緩急と
　変幻自在で、
　
　3次元ではとどまらず4次元の世界に導かれるようです。

　「早よ、行動しいや！」

　　改善していこー！＝（旨み）」

　【小さく継続して、大きく勝ち取る】

■自分にとって痛い、嬉しくない、
　不快な出来事があったときに


　【ありがたい、ありがたい、ありがたい、と
　　繰り返し、唱えましょう。

　　不思議なことに、
　　本当にありがたくなってきますから】


　といった内容のことを、
　過去に書いたことがあります。


　※以下、参考URLです。

　＜3428号 「ため息」の回路を遮断する＞
　http://www.2nd-stage.jp/backnumber/ppf_full/2013/01/3428.html

　


■今日は、同じような状況に陥ったとき、
　
　「『ありがたい』とは別に、
　　こういう言葉を用いるのもありではないか」

　ということをお伝えいたします。


　それは

　【なんでも勉強】

　という言葉。


　この言葉を用いると、本当に

　【なんでも勉強】

　になってくるから不思議なもの。



■面倒なことや厄介なことが起きたときに
　私（鮒谷）は毎回、

　【なんでも勉強】

　と唱えるのですが、最初はいまだ、

　「『なんでも勉強』とは言ってはみたものの、
　　何が勉強なのか、さっぱり分からん」

　というところから始まります。
　
　

■ところが、何遍も何遍も、

　【なんでも勉強、なんでも勉強、なんでも勉強、、、】

　と唱え続けているうちに
　どういうわけか本当に

　【何かを学んでいるような気になってくる】

　のです。



■そのタイミングで、

　【何を学んだんだっけ？？】

　と自分に問いかけると、


　たとえば、
　

　【今後、二度と同じことをしでかしてしまうことの
　　なきように、原因を元から断つ方策が閃いたり】


　【今後、同種のことが起きても、二度目になるから、
　　冷静に対処できるはずと、この経験をプラスに解釈したり】


　といったことができるようになってくるのです。
　　

　※だから、「ありがたい」と「なんでも勉強」は
　　併用すると、より一層の効果が見込めますね。



■鬱陶しいと思うことや、腹が立つことがあっても

　【なんでも勉強】

　です。

　
　こうした言葉をたくさん持っているほど、

　外部からの力によって
　精神の均衡が崩れそうになっても、

　即座に立て直せるようになってくるでしょう。



■のみならず、

　さらに上手に言葉を操れるようになると
　むしろすべての負のエネルギーを正に転じ替え、

　【何が起きても、それを燃料にして
　　前進し続けられる駆動力】

　を手に入れることすらできるようになるのです。



■人生において

　【重戦車並みの駆動力】

　を得たいのであれば、

　もっと、もっと、もっと、言葉をストックし、
　
　その上で、自在に操れるところまで修練を重ねる
　必要がある、

　ということになりますね。



■もちろん私（鮒谷）もまだまだ修行中の身ですが、

　【修行の成果が、比較的、短期間で、
　　明確な実感を伴って感じられる】

　という意味では、


　【言葉を操る訓練は、自己の変化、変革、変容に
　　向けての事始めとしてはピッタリの取り組み】

　であろうかと考えています。


　そのための手引き、水先案内としては、
　やっぱりこれ！（↓）をお勧めしたいところです（笑）

　時々刻々、言葉を操る訓練をしている、

■補足すると、コンサルを申し込むとは、
　鮒谷の時間を買うことではありません。

　それをきっかけとして、あなたの
　「より良き人生、より良き未来」を買うことです。

　だからこそコンサル終了後、

　「高いはずだったものが、
