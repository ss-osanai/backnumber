▼15時間耐久個別コンサル

■こんにちは、鮒谷です。
　

　昨日は、

　朝7時半から夜10時半まで、

　短い小休止を挟みつつ、
　15時間連続（！）、６件の個別コンサル。

　皆さま、ありがとうございました！
　


■話は変わりまして、

　先日、コンサルさせて頂いた大学教員のＩさまより、
　下記の通りご感想を頂戴いたしました。

　
　（ここから）
　----------------------------------------------------------------


　今回もありがとうございました。


　今回のコンサルで印象に残ったことは2つあります。

　1つは、自分は職業に縛られていたのかもしれない
　ということです。

　今まで何となく自分ができそうな方向に流れてきて、
　これからも現在から予測しうる未来に向かっていくのだと
　思っていました。

　そうしないと、今までやってきたことが
　無駄になってしまうのではないかという怖さもあるからです。


　私にとって今の職業はすごくがんばって
　（運の良さも手伝って）やっと手に入れたものだったので、

　それを間違っていたと思いたくない、
　という気持ちもあります。

　でもわれに返って現在の自分の勤務時間の中身を思い返してみると、
　理想とは異なっているのかもしれないな、と思いました。


　もう1つは、コンサル前日にしでかした大ミスについて
　何が原因だと思うかをお聞きくださり、それに対して

　「たぶんそれは直らないから、人に投げた方が良い」

　とアドバイスをくださったことです。

　できないことに対して克服するとか改善すると考えるのが
　私の中の常識だったので、

　「直らない」とハッキリ言っていただけると
　「そっか！直らないんだ！」と目から鱗が落ちました。

　これは時間を有効に使うことになっていくのだろうな、
　と思いました。


　次回までの活動目標についてはだいぶハードルを下げてくださって、
　生徒に応じた鮒谷さんの対応の幅の広さを感じました。

　いつも新しいことにチャレンジするときは
　ジェットコースターに乗る気分で、

　「走り出したら終着点につくまでは降りられない。
　　怖いけどなんとかやりきらなくては。」

　と思うので、

　今回もなんとかコースターが止まるまで
　（コンサルが終わるまで）はしがみついて活動しようと思います。


　でも今回は、コースターが戻るのは出発点ではなく、
　別の場所にたどり着けるような気がして、
　怖さの中にも少し楽しみな気持ちもあります。

　次回もよろしくお願いいたします。


　　　　　　　　　　　　　（大学教員 Ｉ様）


　----------------------------------------------------------------
　（ここまで）


　Ｉさん、長文のご感想、ありがとうございました！

　
　せっかく頂戴したご感想なので、

　今日はフォローアップを兼ねて、メルマガ本文で、
　ご感想を通して考えたことを記しますね。


　次回もよろしくお願いします！

＜今日のピークパフォーマンス方程式＞

【　ビジネスは、思い込みを捨てるほどにうまくいく　】

───────────────────────────────────

　　

■最近、よく考えるのが、

　「ビジネスは、思い込みを捨てるほどにうまくいく」

　ということ。


　無数にある、

　「思い込み」

　（上記の例でいえば、

　　気づかぬところで職業名に縛られていたり、
　　すべて自分で行わねばならぬという固定観念）

　を捨て、無意識のうちに潜む呪縛から
　解き放たれたとき、

　「とんでもない世界」

　に出られるようになるのです。



■私（鮒谷）でいうなら、


　「営業は客先にいって行うもの」

　とか
　
　「セールスは足で稼げ」

　とか

　「受注は接待から生まれる」

　とか

　「値下げしないとモノは売れない」

　とか

　「営業やセミナーやプレゼンでは
　　見栄えのする立派な資料作成が必須」

　とか

　「コンサルはお客さまのところに行くもの」

　とか

　「オフィスは一等地に構え、カネをかけて、
　　おしゃれに格好良くしなければならない」

　とか

　「仕事するのにスーツ着用は常識」

　とか

　「ビジネスパーソンはスマホを徹底活用」

　とか
　
　「メールは自分で打つものだ」

　とか

　「仕事は事務所で行うもの」

　とか


　そんな常識を捨て去れば捨て去るほど
　儲かるようになりました。



■どのように捨ててきたかというと、


　▼営業は客先にいって行うもの

　　→ 無理に売りに行くようなものではなく、
　　　 買いに来て頂くもの

　
　▼セールスは足で稼げ

　  → セールスは頭で稼ぐもの


　▼受注は接待から生まれる

　  → 接待はお客さまからして頂くもの（笑）


　▼値下げしないとモノは売れない

  　→ 値上げするほどモノは売れていく


　▼営業やセミナー、プレゼンでは
　　見栄えのする立派な資料やレジュメが必須

　　→ 手ぶらで行くから、資料作成の手間もかからず、
　　　 相手の集中力が資料に分散されることもない
　　　
　　　 相手の目を見て、真剣勝負のトークでストレートに
　　　 相手の心に入っていけるから、むしろ資料なんか邪魔


　▼コンサルはお客さまのところに行くもの

　　→ コンサルはお客さまにお越し頂き、
　　　 行うもの


　▼オフィスは一等地に構え、カネをかけて、
　　おしゃれに格好良くしなければならない

　　→ オフィスが利益を生むわけではない、
　　　 安く上げれば安くあげられるほど頭がいい


　▼仕事するのにスーツ着用は常識

　　→ 真夏や梅雨の季節、台風や雪の日にスーツ＆革靴着用など
　　　 ありえない、そもそも不快で生産性も下がる

　　　 その時々の気候に合わせ、一番楽チンで、
　 　　荒天時は濡れても汚れても大丈夫な、テキトーな服にする


　▼ビジネスパーソンはスマホを徹底活用

　　→ スマホの携帯など、思索と生産性を妨げる最たるもの


　▼メールは自分で打つものだ

　　→ キーボードなどという物理的な物体に向かって作業すると、
　　　 そこがボトルネックとなり、生産性が致命的に減殺される
　　　 

　▼仕事は事務所で行うもの
　
　　→ その時々、私がいるところが事務所（本社）


　といったような感じ。



■以上、ほんの一例ですが、

　これ以上生々しいことを書くと、

　いろんなところからクレームが来たり、お叱りを受けたり、
　うっとおしがられたり、刺されたりしそうなので（汗）、

　これくらいで。



■世の中の常識に従っていたら、
　常識的な成果しか上げられません。


　常識に従って、みんながやっていることを
　自分も行っているだけならば、
　
　どうしたって、

　「似たり寄ったり、五十歩百歩、どんぐりの背比べ、

　　的な、平均的な成果しか出せない」


　のは当たり前。



　【 非常識な成果が欲しいのならば、

    　　　　　　 常識（思い込み）を破り、

　　　　　　　　　　自ら新たなルールを創造するより他にない 】



　のです。
