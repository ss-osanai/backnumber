　いる人があります。

 【　以心伝心が有効なとき、有効でないとき　】

───────────────────────────────────



■「以心伝心」という言葉があります。

　言葉によらず、お互いの心を伝え合う、というような意味
　ですが、

　これが有効な場合と、有効でないときがあります。



■「襖（ふすま）の文化」の中で育まれ、

　近い距離で、長い時間をともに過ごした人同士でのみ
　「以心伝心」は有効となるのでしょう。


　やがては、

　ふすまで隔てられた隣の部屋にいても、
　言葉で伝えずとも相手の気持ちをくみ取ることができる、

　というような。



■あるいは家庭内だけでなく、会社内、その他のコミュニティで
　あってでも、

　ある程度、身体的な距離が近く、関わり合う時間が長くなれば
　「以心伝心」が機能するようになり、
　
　言葉を交わさずとも、相手の気持ちが分かるように
　なってくるのかもしれません。



■しかし、

　多岐にわたる人とお付き合いすることが必要となってきた
　現代社会においては、

　すべてのひとと「以心伝心」で通じ合うことはできなくなり、
　したがって、お互いに誤解なく、付き合うことが
　とても難しい時代になってきています。



■そんな状況のなかで私も、

　数多くのささくれだった状況を経験してきて、


　「なぜ、こんなに人間関係に疲れるようなことが頻発するんだ！」


　とブチ切れて、

　やがて編み出した（？）必殺技が、昨日、お伝えした、



　【 情報（＝自分の趣味・嗜好、あるいは好みでないこと）

　　 を徹底的に開示しつづけることで、


　　　自分を理解し、受け入れてくれる人が集まってきて、

　　  反対に趣味嗜好が異なることが分かり、離れていく人が出てくる

　　　　　　
　　　　　　　　　　　　　　　　　　　　　　　　状況を誘発させる 】

　
　ことだったのです。



■要は、はじめてお会いする前に、私の人となりを知ってもらい、

　その上で、

　「合いそうだ」

　と思っていただけた人とだけ、お会いできる環境を
　あらかじめ、こちらのほうから提供しておく、

　ということなのです。



■こういう場合の第一印象というのは、かなりの確率で
　あたるもので、

　「合いそうだ」

　とおもっていただいた人とは、やっぱり合いますし、

　「ちょっと自分とは合わないかも」

　と思われた人とは、合わないことが多いものです。



■すなわちこちらから人を選ぶのではなく、
　相手に選んでいただく、のです。

　最初に相手に選んでいただいたのであれば、すくなくとも
　よほどのことが起きない限り、

　そこから生まれた人間関係が破綻することは
　ありません。

　
　
■だからメールマガジンで可能な限りではありますが、

　わたしの日常も、お付き合いしている人も、読んでいる本も、
　将来の方向性も、趣味も、嗜好も、

　開示しているのです。

　そしてそこになんらかの共通項を見いだし、興味関心を
　お持ち下さった方とだけ、お付き合いができるように
　なりました。



■結果、
　

　「なぜ、こんなに人間関係に疲れるようなことが頻発するんだ！」


　という怒りの感情自体、

　もはや思い出せないくらいになってしまったのです。



■自分に合う人とだけ、お付き合いできるというのは
　何ものにも代え難いくらい、ありがたいことです。

　メールマガジン、ブログなどで情報を開示しつづける、
　ということには、こんなメリットもあります。



　今日もビジネスを楽しみましょう。
