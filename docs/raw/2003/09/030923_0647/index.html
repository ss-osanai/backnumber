メルマガ　「平成・進化論。」～勝ち組志向のあなただけに～

　　　　　　　　　　　　　　　　　　　　　　　　　　　　　＜００９号＞

　　　　　　　　　　　　ピークパフォーマンス・ナビゲーター　鮒谷　周史
■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■

※このメルマガは、外資系専門の現役ヘッドハンターが、意識の高い皆様へ、

「ピークパフォーマンス」（人生における最高の結果）を発揮することが出来
るように、年間３００冊以上の本を読み、１００日以上、勉強会・セミナーに
参加し、学び、得た、内容を編集してお伝えしていくものです。
このメルマガを読み続けることで、こんなことが出来るようになります。

●セルフプロデュース術（自分を売り込む重要なポイントを発見できます）
●自信を持って、前向きに行動することが出来ます
●自尊心を持って生きていくことができます
●得たい結果を最速で得るためのヒントが掴めます
●新しい発想のヒントを得られます
●モチベーションを維持する方法を学べます
　　　　　　　　　　　　　　　　　　　　　　　　　　　　などなど・・・

読み続けること。継続することが大事です。自分の中に落とし込めるまで。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━


※ショッキングなニュースが報道されています。

======================================================================

金融広報中央委員会が２２日発表した家計の金融資産に関する世論調査による
と、貯蓄のない世帯が全体の２１．８％と、１９６３年の調査開始時
（２２．２％）に次ぐ４０年ぶりの高水準となった。また、前年より貯蓄が減
ったと回答した世帯が５１．１％と、３年連続で増加した。リストラなどによ
る収入減で貯蓄を取り崩した世帯が６割に上ったため。

一方、金融資産を保有する世帯の保有資産の平均額は前年比３８万円増の
１４６０万円と過去最高を更新。平均額を下げていた少額貯蓄世帯が無貯蓄世
帯になって資産保有世帯から外れたり、高額保有世帯が増えたためで、持つ者
と持たざる者の二極化が進んでいることが浮き彫りになった。
　　　　　　　　　　　　　　　　　
　　　　　　　　　　　　　［毎日新聞９月２２日］ ( 2003-09-22-18:52 )　
======================================================================


一言で言うと、

富める者はますます富む。貧しき者はますます貧しく、という社会が到来しつつ
ある、ということ。
貯蓄なしの世帯が、とうとう２割を越す。貯蓄のある世帯は、貯蓄額がますます
多くなっている。


というわけで、今日のお題は、


【リスクを見極め、自分の将来を考える】


勝ち組と負け組、という表現が使われ始めたのはいつの頃からだろうか？
私の記憶では、９６，７年くらいからだったような気がするのですが・・・。

現実に中間層が少なくなって、少数の高所得者層と多数の低所得者層に二極
分化してきている現実が数字となって現れてきています。

あおるわけではありませんが、現実の問題として、右肩上がりの経済成長が
見込めなくなった今、限られたパイを、多くの人が奪い合うような状況とな
っています。
しかも国内の人だけでパイを奪い合うだけならよいのですが、国際化時代の
名の下に、アジア諸国ほか、競争力のある海外の国々と戦っていかなければ
なりません。

その結果が数多くのリストラの悲劇であったり、会社の倒産、その他、毎日
の報道で見せつけられる通りです。

★

大局的な視野で、今後の行く末を見る目が必要です。たとえ、それが怖い
ことであっても。

私も見るのが怖いです。将来を冷静に見据えるのは。

でも本当のリスクヘッジ、は「最悪の事態を想定して、その上でリスクを覚悟
で勝負をかける」ことなのです。

本日のランチ会でお招きした、日本人唯一のラスベガスでのポーカープレイヤー
として通算、何億円という勝負を挑み続けてこられた嶋崎先生が同じ事をいわれ
ておりました。

やっぱり「最悪、最低の事態を考えずに勝負してはいけません」って。

★

最悪の事態が来たときに、日本はどうなっているのか。そのとき私はどのように
なっているのか。
いつもその危機感を持ちながら、それでいて、明るい自分の未来を心に描けてい
たら、個人個人にとって、よい将来が必ず訪れます。

そのリスクを見ないようにして、享楽に走っていたら、その報いは必ず来るもの
です・・・。
その、もっとも分かりやすいメタファーが、アリとキリギリスのお話。


【今日のピークパフォーマンス方程式】

楽観主義は大事です。前向き思考を持っていなければなりません。
ただ、常に最悪の事態を想定して、覚悟を決めた上での楽観主義、前向き思考で
あることが前提です。

【編集後記】

ポーカーの嶋崎先生をお迎えしてのランチ会を開催しました。
飛び石連休の合間、中途半端な？時間帯、中途半端な？場所、にもかかわらず、
３０名の方々にお集まりいただきました。

友人・知人、友人の友人、日記の読者に、メルマガの読者様。
総勢３０名の方が一堂に会する機会となりました。
お越し下された方、どうもありがとうございました！

本文にもちょっと書かせていただきましたが、命がけで勝負を張ってこられた
方のお話にはやはり迫力があります。
自分以上の枠を持った方と接することで、自身が持つ思考の枠組みを破り、
新しい基準が生まれると感じております。

積極的に、自分より凄い、素晴らしい、人を見つけるための努力をし、そう
いった方々と触れ合う機会を持てるよう心がけてはいかがでしょうか？

今回、トークセミナーということで、嶋崎先生と私が、皆さんの前でトークを
するような形でお話しさせていただきました。
生の勝負師と直接お話しすることが出来たのは、私にとって貴重な財産となり
ました。

★
