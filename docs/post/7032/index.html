7032号 面倒なことを省いて、面倒なことをやる

■人生やキャリア構築においては

　「面倒なことをやらない」

　のを基本としつつ、

　「面倒なことだから、あえてやる」

　一つ、あるいはいくつかのものを
　選択する、その峻別が大事、

　だと思っています。



■名言っぽくいうならば、

　【面倒なことを省いて
　　面倒なことをやる】

　とでも言えるでしょうか。



■人生を成り行き任せにしていると
　やがて日常の行為の大半が

　「面倒ごと」

　と化していきます。

　時間をかけて、
　ツケが回ってくるというわけですな。

　（その反動として「自分へのご褒美」を
　　欲するようになる）



■そんな状態に
　はまり込まないためには

　【面倒なことを極力省きつつ、
　　価値を生み出す面倒ごと】

　を積極的に選び取っていく方向に
　舵を切る必要があります。



■世に求められているけれども
　多くの人にとっては面倒ごと、

　しかし自分にとっては面倒ではなく、
　むしろ

　「趣味、道楽」

　という対象を見つけられれば、


　時間をかけてそちらに
　資源の傾斜配分をしていくにつれ、

　楽しく生きられるようになるし、
　それがそのまま参入障壁となり、

　さらに、その障壁は日に日に
　高くなっていくでしょう。



■ただし、もしあなたが、

　「そんな世界があったのか」
　「できるものならそうしたい！」

　と思っても、いきなり

　「今やっていることは面倒ごとだから
　　やーめた」

　なんて切り替えられないし、
　やってもいけません。



■面倒に感じる仕事をして
　生計を成り立たせていたところ、

　それが嫌だからといって
　感情の赴くままに

　「いち抜けた」

　しても、困窮するのは自分です。



■あるいは、

　「いち抜けた」の代わりに
　「手を抜くよ」してみても、

　それは高井伸夫先生が
　言われるところの

　「権利の極大化と義務の極小化」

　につながる話ですから、

　当座はよくても、徐々に自分の首を
　締め上げることになりかねない。



■なので、

　【目指すキャリアの最終形態】

　としては

　【面倒なことを省いて
　　面倒なことをやる】

　が正解になりますが、


　一朝一夕にはその世界は実現できず、

　中期（数年）ないし
　長期（十数年）かけて、

　理想の状態を作っていくことを
　心がけるとよいでしょう。



■少なからぬ人が

　「そんなに待ってられるかよ」

　と思うから、

　「いち抜けた」
　「手を抜くよ」

　するのですが、

　それは短期（数日～せいぜい数ヶ月）の
　思考ですから、

　その瞬間は良くても、
　以降、長く後悔が続くこととなります。



■ここまでの話を
　私（鮒谷）の例で言えば、

　キャリアのはじめの頃は価値観、強み、
　やり方と仕事とのミスマッチで、

　「すべてが面倒ごと」

　に感じられたわけですが、


　相応の時間をかけながら
　徐々に軌道修正を図り、

　【面倒なことを省きながら
　（他の人にとっての）面倒なことをやる】

　世界を実現させました。



■具体的には、

　自分が面倒だと思う仕事を

　他者に迷惑をかけることなく
　時間をかけてゆっくり減らしていく代わりに

　「メルマガを書く」

　という、

　　多くの人が面倒だと感じるけれども
　　自分にとっては全くそうは思わない、

　「（他人にとっての）面倒ごと」

　かつ

　「一定の価値を提供できると
　　思われる行為」

　に資源を集中的に
　振り分けていったということ。



■この面倒なことを楽しみながら行い、

　信用とか信頼とか、
　積み上げてきた大量のコンテンツとか、

　その過程で仕入れた知識や
　身につけた言語化能力、

　などを獲得しました。



■なにごとによらず、

　素晴らしい価値のあるものは、
　一朝一夕には手に入らないもの。


　「面倒なこと」を省きつつ、
　「面倒なこと」にフォーカスして、

　20年やり続けてきたからこそ、
　得られた大きなものだと思っています。



■こうして獲得された資源は、

　今、私（鮒谷）に絶対的な安心感と
　安定感をもたらしてくれています。


　正直にいうと、

　そこまでに至るプロセスは
　口でいうほど簡単なものではないし、

　誰にでも真似できるものではないし、

　もっというとほとんどの人はできないと
　最初から分かっていたから、

　やり続けてきたんですけどね。



■それが人生、生活、ビジネス、
　キャリアにおける

　【盤石の基盤】

　に繋がっていったわけですが、


　ここまでの話は、なにも私（鮒谷）の
　回顧談のために書いたものではなく、

　【あなたにとっての、
　　私のメルマガに相当するもの探し】

　を行う際のヒントになるのでは、
　という思いから記したものとなります。



■既にお分かりのことと思いますが、

　「だからあなたも
　　メルマガを書きましょう！」

　という短絡的な話ではありません。



■あなたにとっての、
　私のメルマガ

　（つまり他の人にとっては面倒だけれども、
　　あなたにはそうは感じず、

　　むしろ楽しくそれを行え、
　　あなたにしかできない価値を生むもの）

　を見つけられたら、

　それ以外の面倒なことを省き、
　価値を生み出す面倒ごとに一点集中、

　いうことですね。



■私（鮒谷）の事例を、

　具体（メルマガ執筆）で見るのではなく、
　抽象の世界で考え、

　あれこれ思索するうちに見えてきたり、
　閃くものもあるかもしれません。

　（というか、
　　あればよいなと思って書きました）


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】

　■理想の人生、キャリアづくりには

　　【面倒なことを省き、面倒なことをやる】

　　と良いだろう。

　■より正確に言うならば、
　　自分にとっての面倒ごと、

　　年単位の時間をかけてゆっくりと、
　　他者に迷惑をかけずに、減らしていき、

　　他人にとっての面倒ごとだけれども
　　自分にとっての趣味道楽、

　　となる分野を探し当て、そこに
　　資源の傾斜配分を行っていくこと。

　■やればやるほど機嫌よくなり、
　　やればやるほど高い参入障壁が築かれ、

　　人生もキャリアも
　　安定していくこととなる。
