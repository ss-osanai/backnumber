　  「変容の喜び」を提供し続ける
　　------------------------------------------------

　ことを目的とし、3000日以上にわたってお届けしている、
　個人発行として日本最大級のビジネスメルマガです。

　熟読するも読み流すも、自由な形でご活用下さい。

＜昨日の活動ダイジェスト＞

▼英語の勉強

───────────────────────────────────



■こんにちは、鮒谷です。
　
　昨日は終日、英語の勉強。




■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　美しい方は美しく、そうでない方はそれなりに　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

  【　美しい方は美しく、そうでない方はそれなりに　】

───────────────────────────────────



■昨日は、

　「文章を書くときには、
　　締めの言葉に注意を払うことが大切」

　というお話をしてきました。

　今日も続けます。



■文章を読み終わった時の「読後感」が
　どのようなものであるか、

　メッセージを伝達する側の人間としては
　とても気になります。


　できることならば、


　▼実質的な気づき、発見、学び

　とともに

　▼なにかしらの感情変化

　　（たとえば納得感、学んだ感、身についた感、
　　　オトク感といった）


　の両方をお届けしたいと思っているのです。



■単に

　「役に立つ（かもしれない）コンテンツ」

　を送りつけさえすればよい、
　といったものではなく、

　「読後、どんな感情となられるか」

　というところまで考えながら
　メッセージは伝えられるべきである、

　そんな風に考えているのです。

　言うは易く行うは難し、なのですが。



■この数日の間、お伝えしてきた、

　「文章を作ることを体操競技に例えると」

　と絡めていえば、

　読者さんに伝えたいとおもっている
　ことの一つ

　▼実質的な学び

　は

　「ハイライト的な技」

　から生まれるものであり、


　▼なにかしらの感情変化

　については、少なからぬ部分、

　「着地の言葉（文章を締める言葉）」

　からの影響があるのではないかと
　考えています。



■そのようなわけで、
　私も長い間、文章を書いてきて、
　
　「うまい着地」

　はどうやったら身に付けることが
　できるのだろう、

　と、ずっと考えてきたのですが、
　試行錯誤の結果、
　
　「着地言葉の事例ストック」

　にまさるものはない、

　と今更ながら思われるのです。



■先に結論を言ってしまうと、

　先人の作った名文句を拝借させてもらうのは
　とても効果的。

　
　なかでも、

　「四字熟語」「ことわざ」「故事成語」「古人の名言」
　
　といったものは汎用性が高いし、
　使いでがあります。

　
　あるいは、それ以外の

　「人口に膾炙した（誰でも知ってる）表現」

　も、けっこう応用が効きます。



■それらを用いることによって

　「文章が引き締まり、それっぽく見える」

　ようになるし、それゆえ

　「感情への訴求力が高まる」

　ことも多いものです。
　


■たとえば文章を50行くらい書いて最後に、
　
　「これまで書いてきたように～であるのです。
　　まさに『先ず隗より始めよ』ですね」
　
　という流れで文章を締めると

　「それっぽく」

　なります。



■また、100行程度、書いた文末に

　「上に述べたように～であると思われます。

　　このことをルソーは

　　『すべての不幸は未来への踏み台にすぎない』

　　といっているのでしょう」

　という形で文章を終えると、これまた

　「それっぽく」

　見えますね。


　同じ内容の文章でも「５割増」になるのです。



■したがって大切なのは、


　▼（上に挙げたような）使えそうな言葉を
　　たくさんストックしておく

　ことであり、

　▼それらを自在に取り出し、活用できるように
　　訓練しておく

　こと。

　

■かくいう私も、以上のようなことを相当意識して
　メルマガを書いています。（気づいてました？）

　いちいちバックナンバーの紹介はいたしませんが、
　古くからの読者さんであれば、

　「そういえば、よく使っとるなあ」
　
　と思い当たることもあると思います。


　
■こんなテクニックを用いれば、

　たとえ下手くそな文章であっても、
　最後の文句（着地）を決めることで、
　
　「それなりの文章」

　に見えることも多いものです。


　
■昔のフジカラーのコマーシャルに

　「美しい方は美しく、
　　そうでない方はそれなりに」

　という名文句がありましたが、


　以上のようなことを心がけて文章を書けば、

　「美しい文は美しく、
　　そうでない文はそれなりに」
　
　程度の補正はしてもらえることでしょう。
　


　今日も人生とビジネスを楽しみましょう！
