＜今日のピークパフォーマンス方程式。＞

　　　　【　時間管理の前に目標をたくさん掲げること。　】

───────────────────────────────────


■時間管理の本がたくさん出ています。

　こういった関係の本とかセミナーに昔、ちょっとはまったことがあり
　ました。


■いろんな方法があります、時間管理には。

　読者さまの中でも、さまざまな手法を学ばれた方がいらっしゃると
　思いますし、多くのこういった関係の本を読んでこられた方も多いと
　思います。


■私自身もずっとやり続けて、「結論」とまではいいませんが、ここにきて
　大原則のようなものを発見してしまいました。

　それは、「時間管理」を時間管理単体で考えるな！ということ。


■よく（でもないけど、たまに）「時間管理マニア」の人種をみかけます。
　手帳をきれいに、丁寧に、びしっと使いこなし、いかにも手帳を
　使いこなしているように見える人。

　でも、こんな人に限って、あんまり密度濃く、内容のある時間を使って
　いるように見えなかったりします。スケジュールをマシンのように、
　淡々とこなすだけ。

　あまり情熱がないから、どちらかというと「仕事をこなす」ために
　時間管理をしているという感じ。


■ちょっと違う気がします。

　目標が１つであれば、その目標を果たすために時間の使い方を考えます。
　目標が２つであれば、その目標を果たすために時間の使い方を考えます。
　目標が３つであれば、その目標を果たすために時間の使い方を考えます。
　目標が10あれば、その目標を果たすために時間の使い方を考えます。

　目標が増えるほど、全部やりたいことだから、だんだんタイトな
　スケジュールになってきます。

　そのうち、どのようにあがいても、とてもとても間に合わないように
　なってくる。


■そのときになってはじめて、仕事を捨てる、委任する、2割8割法則を忠実
　に実行する、などなど、真剣に時間管理に取り組めるようになるのかな、
　という発見。

　
■その状態になると無駄に自分の５分間を奪う存在に怒りを覚えますし、
　自分の時間価値を真摯に考えるようになります。

　ここではじめて真剣に時間管理に取り組めるのではないかと思います。


■目標を掲げず、やりたいこともない。でも時間管理（に見えること）は得意。

　こういうのはやはり「時間管理マニア」というよりも、むしろ
　「時間管理オタク」。


■とはいえ。

　かくいう私も、昔は「時間管理オタク」でした。

　今は必要に迫られてタイムマネジメントの勉強をまじめにしようと
　考えているところです。
　それもこれも、掲げた目標の数がだんだん増えてきたから。


■とかく、時間管理、時間管理、と仕事の現場で叫ばれることが
　多いですが、目標、やりたいこと、あつい情熱、あふれる行動力、
　を抜きにして、時間管理と叫んでみても、栓のないことですね。

　順序としては

　▼目標⇒時間管理（たくさんの目標を掲げ、時間がないから時間管理）

　であって、断じて、

　▼時間管理⇒目標（時間管理を義務のように思い、淡々と目標に近づく）

　ではないと考えます。


■前者のほうは、熱い情熱、勢いが感じられますね。
　後者は、無機質なテクノクラートの匂いを感じます。（※）

　（※官僚及び官僚組織を否定しているのではございません。念のため…）
