■今日は

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------
　
　摩擦ゼロでスムーズに流れている、
　そんなビジネスや生活のシステムを作るんやで。

　努力しなくても成果がもたらされる
　仕組みを作れ、いうこと。


　その仕組みが完成するまでは、
　ゴリゴリに実験を繰り返す必要があるのは
　言うまでもない。

　----------

　地味な継続は閾値（いきち、しきいち）を
　超えた瞬間、一気に化けるんよ。

　そこまでは退屈な積み重ね。
　でも、ある日突然、開花するんだわ。

　それを信じられた人だけが
　相転移の瞬間を迎えられる。


　信じ切れず、耐えられなくなった人から
　順に近道や奇跡を求めるようになり、

　閾値超えルートから脱落する。

　----------

　本当のところ、
　有能な人も無能な人もおらんのよ。

　あるのは

　『向いとる人』と
　『向いとらん人』だけ。


　ある場所で有能でも、
　別に移れば無能になる。

　無能も場所を変えれば
　スーパーヒーローや。


　ワイは高速道路の橋脚補強工事
　やってたとき、

　現場のオッサンに無能扱いされ、
　蹴られたことまであったやで。

　----------

　チャレンジし続けんと劣化するんだわ。

　そして劣化の事実は数年経過の後、
　はじめて気づく。

　悪いことには、そのとき既に
　足腰が立たなくなっている。

　そんなケースをたくさん見てきたよ。

　----------

　怠惰で無能なワイ自身をどう導くか、
　試行錯誤の半生やったのよ。

　長年、一日も休まず振り返り、
　日記を更新し続けて出来たのが

　「無能再生レシピ」

　なんだわ（言い方よ）。

　----------

　言葉を操る力は生涯を通じての武器に
　なるから、磨いておいたほうがいい。

　言葉で己を鼓舞したり、他者に働きかけ
　影響力を行使できるようにもなる。

　交渉ごとにも必須よな。


　端的にいえば、言葉を操れないと
　どんどん苦しくなっていく。

　鍛えなければ鍛えられない力ゆえ、
　地道に訓練するしかない。

　----------

　自らを鼓舞するつもりで
　毎日、文章を書いとるよ。

　裏返せば、自身が鼓舞されぬ文章は
　書く意味がないと思ってる。


　ちなみに鼓舞には
　二通りの経路があり、

　一つは直接鼓舞とでもいうべき、
　即効性のある文を編むこと。


　もう一つは間接的、かつ時間をかけて
　ジワジワ効いてくる文。

　ワイの作文は後者が99％。

　----------

　「○○さんは○○が凄いですね！」
　と言われたことあるよな。

　謙遜し、頭から否定してまいがちやが、

　複数人から言われたテーマは
　掘る価値あるで。


　当人にとって当たり前すぎて
　ポテンシャルに気づいとらんだけの
　ケースがある。

　ワイは言語操作の力を
　何度も褒められてきたが、

　当初は社交辞令と受け流してた。

　----------

　言語化能力を高めたければ、
　出力の前に入力しなけりゃならんのよ。

　入力データは抽象度の高いものから
　具体性に富む事例や各種表現技法など、

　入力範囲はガチ教養からゴシップまで
　幅広く、が理想。

　当然、膨大なものとなる。


　その後、素材を当意即妙に
　組み合わせる訓練に進む。

　それは死ぬまで続くのよ。

　----------

　「言語化する力、どうすれば
　　高まりますか？」

　という質問を、たくさん受けてきたよ。


　偉そうに語る資格はないけど、
　数十年に渡って日記を書いてきたから、

　控えめに語るくらいは許されるかな
　と思ってる。


　累積思考量の多さが
　言語化能力を規定するんだわ。

　要するに近道は、ない。

　----------

　NHK『クローズアップ現代』の、
　ちゃんみなの特集を見た。

　周囲からの心ない言葉や評価で
　傷つけられてきた人は見るといい。


　彼女は音楽の力で自らを救い、
　さらには自身の存在を通して、

　いま苦しんでいる人を救おうとしている。
　素晴らしいな。格好いいな。

　見ていて涙、出てきたわ。

　※今ならまだNHKオンデマンドで
　　見られるで。（期限はあると思う）

　----------
　
　弱点やコンプレックスを隠そうと
　するから自信が失われる。

　逆張りで全面開示すればいい。

　ただし自分が納得し、かつメリットが
　生じるストーリーを編んでから。


　いきなりは難しいから
　過去→現在の物語編集に取り組み、

　開示可能になったタイミングで
　公開すれば、親近感や好ましいギャップが
　生み出される。

　----------

　本当に理解してることは
　「自分オリジナルの表現」に
　言い換えできるんよ。

　言い古された表現しか
　思い浮かばんとしたら、

　生半可な理解しか出来とらん
　いうこと。

　----------

　「つるむ」いう言葉があるけど、
　つるむ相手は厳選するべきよ。

　「朱に交われば赤くなる」

　は典型やけど、

　悪事に限らず日頃、一緒に過ごしている
　人間とはあらゆるものが似てくる。


　もっとも恐れるべきは、

　自分に対する期待や目線を
　引き下げるようなつるみ方。

　居場所選び、仲間選びは
　本当に大事やで。

　----------

　一生懸命に生きてれば、
　自己肯定感が高まるよ。

　自分でいうのもなんやけど、
　ワイは長い間、休まず生きてきた。

　その自信だけはあり、間違いなく
　それが高い自己評価につながってる。


　いい加減に生きてたら、都度

　「自分なんか、、」
　　
　という後ろめたさを伴う言葉が回り、
　動けなくなってたに違いない。

　----------

　人の可能性は無限やで。

　どれだけ窮地に陥っても、
　スイッチさえ入れば劇的に変わる。


　ただ、どれだけ凄いポテンシャルを
　秘めてても、スイッチが入らんと
　何も始まらん。

　ちなみにスイッチの入れ方は
　人生を激変させた人と実際に出会い、

　話を聞き、物語をリアルに感じること。

　これが最強の起爆剤となる。

　----------

　「得か損か」「無駄か価値か」など、
　細かく考えすぎると人生が窮屈になるよ。

　そんな計算だらけの毎日じゃ～ポイズン、
　やで。

　楽しめんし、何より大切な
　「予期せぬ出会い」のチャンスを
　逃してまう。


　大きな跳躍は、たいていの場合、

　計算外の行動が起点となって
　もたらされることは覚えておくといい。

　----------

　好奇心の正体、
　それは「自分への期待」なのよ。

　「何者かに必ずなる」という
　大きな目標があるから、

　自ずから多方面にアンテナが立ち、
　森羅万象への関心が生み出される。


　その強い思いは異なる分野を
　またいだ連想から生まれる発想や、
　学びの歩留まりを高め、

　その人を本当に「何者か」に変えていく。

　----------

　人生の醍醐味は

　「知らなかったことを知る喜び」
　「分からなかったことが分かる快感」
　「見えなかったものが見える興奮」

　にあるんだわ。

　興味の輪が広がり、
　好奇心が満たされる体験こそ、

　人生を面白くする原動力。


　このスパイラルに入ると、
　発見と驚きの連続となり、やがて

　「真の、目指したい目標」

　へと導かれていく。

　----------

　垂れ流される情報を鵜呑みにする人と、
　自分で仕入れて考え、動く人。

　この差が人生の明暗を分けるのよ。

　「なんとかなるやろ」
　「誰かがなんとかしてくれる」

　じゃないんだわ。

　自分が当事者として世界と向き合うこと。


　そんな能動的な姿勢が生活を守り、

　さらには攻めの姿勢で生きていく
　土台になるんやで。

　----------

　「至るところに破滅リスクあり」

　それが人生よ。

　破滅は辺境の地から、音もなく
　忍び寄ってくる。

　気づいた時には、すでに目の前まで迫り、
　手遅れになっているかもしれん。


　だからこそ遠い世界（たとえば
　政治、経済、外交など）にも興味を持ち、
　視野を広げ、アンテナを張ること。

　身を守るための必修科目やで。

　----------

　大昔、図書券いうのがあったんよ。

　袋に

　「少にして学べば壮にして
　　為すこと有り。

　　壮にして学べば老いて衰えず。
　　老いて学べば死して朽ちず」

　という、佐藤一斎『言志四録』の
　言葉が記されてた。


　小学生の頃は眺めてただけやったけど
　今にして本当にその通りやと思うのよ。

　繰り返し読み、暗唱するといい。

　----------

　「明日の自分」に
　学びのプレゼントを贈るといい。

　最高の贈り物は言葉。

　それは雪だるまみたいに、
　価値が大きく膨らんでいく。


　学ぶ人と学ばない人の差は、

　時間とともに驚くほど
　開いていくんだわ。

　恋人へのプレゼント選びのように、

　わくわくしながら「未来のあなた」を
　喜ばせる言葉を贈るといい。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
