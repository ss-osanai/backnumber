＜今日のピークパフォーマンス方程式＞

 【  「生きているだけで成果を出し続けられる体」になるために  】

───────────────────────────────────

　

■いつもお伝えしていることの一つに、


　「知識を学ぶことも大切だけれども、

　　『自分流のスタイル』

　　が確立されるところまで、
　　反復することはもっと大切」


　というものがあります。



■「自分独自のスタイル」

　さえ確立できれば、

　毎時、毎分、毎秒、
　そして無意識レベルで永遠に、
　

　他人からは努力しているように見えるのに、
　自らは努力している自覚など全くなく、


　「生きているだけで自動的に、

　　様々な（自分にとって価値をもたらす）
　　行為を繰り返すことができるようになる」


　からです。
　


■反対に、知識のみを追いかけ、
　聞いた知識を１、２回だけやったところで、

　大した成果は残りません。


　このように、

　知識のみ頭の中に入れて満足していても、
　
　実生活の改善に反映されてくることは
　ほとんどありません。



■たとえば、
　
　「笑顔が大切」
　
　と聞いて、１回、２回、
　周囲に、にこやかに接してみても、

　（悪いことではありませんが）

　そんなに大きな変化が表れるはずはありません。


　しかし、
　　
　「常に笑っているのが基本形」

　「デフォルトが笑顔」

　となるところまで笑顔を訓練し、
　身につけることができれば、
　
　自ずから世界も大きく開けてくるはずです。



■私（鮒谷）であれば、


　「あらゆるスキマ時間を見つけて本を読む」

　「最低でも一日、５時間くらいは
　　人に何かを話をしている」

　「メルマガを毎日、一本配信」

　「メルマガネタ５本分程度の日記を
　　毎日、記録」


　といったことがほぼ、自動化されたレベルで
　実行できるようになりました。



■こうなると有り難いことには、


　「よくそれだけの克己心がありますね」

　「凄い努力ですね」

　「モチベーションをどんな風に
　　保っているのですか？」


　などと聞かれるのですが、ぶっちゃけ、


　「克己心を発動することもなければ」

　「努力しているつもりもなければ」

　「モチベーション、なにそれ？
　　そんなものそもそもないけれど」


　みたいな感覚となるのです。



■つまり習慣も極まってくると、


　「自分にとっての自然体が、
　　他人にとっての不自然体」

　「自分にとっての日常が、
　　他人にとっての非日常」

　
　となってくるのです。

　

■ここまで来るとまずは一安心で、
　余程のことがない限り、

　「習慣が崩れることはなくなる」

　ようになるし、

　なにより、冒頭にも記したとおり、


　「生きているだけで自動的に、

　　様々な（自分にとって価値をもたらす）
　　行為を繰り返すことができるようになる」


　のです。　



■そのようなスタイルを確立するためにも、

　「自分は何をなすべきか」

　を吟味、検討するための準備としての
　知識はもちろん必要ではあるけれども、
　

　それよりも、

　「なにをなすべきかを決定したら、
　　それを習慣化するところまで落とし込む」

　ことのほうがもっと大切であることを
　何度でも強調したいのです。

　

■これが本当に理解できれば、

　「学びさえすれば、何かが変わる」

　というのは、


　「ありがちな、
　　しかしながら、致命的な認識の誤り」

　であることがよく分かるのではないでしょうか。



■実践と結びつかない学習は、
　机上の空論と言わざるを得ません。


　難しい話をしたり、
　たくさんの知識を持っているけれど、


　「で、それがどういう風に行動と結びついているの？」
　
　「で、それがどんな成果につながっているの？」


　と問われたとき、

　「えっ！？」

　と固まってしまう人は、


　「（知行合一人ならぬ）知行分裂の人」

　であり、

　「学んでいるように見えて、
　　本当の意味では何も学んでいなかった人」
　
　ということになるのかもしれません。



■せっかく学ぶのであれば、

　「活かすことを前提に学ぶ」

　という風に、学習に対する定義を書き換えてみては
　いかがでしょう。


　ひょっとすると、
　それこそが多くの人にとって、
　
　学びを最大限有効化するために
　もっとも大切かつ喫緊に必要なこと、

　であるかもしれせん。


　※個別コンサルやセミナー、
　　あるいは年間プログラムなどでは、

　　こんなこともお伝えしています。
