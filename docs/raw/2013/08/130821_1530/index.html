＜今日のピークパフォーマンス方程式＞

 【 ムフムフするまで、フムフムし続ける 】

───────────────────────────────────



■最近、


　「知るから、やる気になる
　　（＝知らないと、やる気にならない）」


　ということに気づきました。


　内発的にやる気が湧き上がり、行動している人は、
　

　「ある世界の存在を知る → やる気が出る

　　→ 行動する → 結果を得る」


　というプロセスが綺麗に流れています。



■反対に、

　未来の目標が不鮮明であったり、そもそも、
　まったく持っていない（＝知らない）のであれば

　やる気が湧き上がってこないのも
　当然のことといえるでしょう。



■だからこそ、まずは


　「こんな世界があるんだ～」

　「すげ～」

　「ほえ～」

　
　という憧れや理想の対象として捉えられる
　世界の存在を知らなければなりません。

　
　新しい世界の存在を認識しなければ、
　新しい世界に向かおうとは金輪際、

　思わない（思えない）のです。



■だから、

　いま自分が生きている世界（環境）を
　どうにかして変えたいと思うのであれば、


　最初に
　
　「フムフムの道」

　を歩まなければなりません。

　

■いろんな世界を見聞し、

　フムフム、フムフム、フムフム、、、、
　
　を繰り返していくうちに、

　だんだんイメージが鮮明になってきて、

　　
　
　「ムフムフの道」

　に入っていくのです。


　つまり、イメージしていた未来が実現された
　自分の姿を想像するだけで

　ムフムフ、ムフムフ、ムフムフ、、、、

　と、思わず知らず、笑みがこぼれてしまうような、
　そんな状態が、ムフムフの道。



■こうなると、

　自然にやる気が生まれ、決断し、行動して、
　やがては人生も変わっていくでしょう。


　言うは易く行うは難し、

　の、この困難なプロセスを分かりやすく、
　直感的に著したものが、

　「フムフムから、ムフムフへ」

　という言葉なのです。



■やる気や行動は

　「フムフムが、ムフムフに転じ変わったとき」

　一気に溢れ出してくるもの。

　
　だからこそ、
　外に出て、学びに行き、出会いを重ね、
　
　「ムフムフするまで、フムフムし続ける」

　必要があるのです。　

　
　ムフムフが始まった瞬間から、
　人生が変わり始めます。
