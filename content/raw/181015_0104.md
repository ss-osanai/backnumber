＜今日のピークパフォーマンス方程式＞

【 読書量（知識）が飽和点まで達すれば、行動できるようになると考える人 】

───────────────────────────────────



■本日のメルマガは、

　先週、「個別相談会」にお越し下さったＴさんより
　頂戴したご感想の紹介へのコメントという形で、

　お伝えいたします。


　（ここから）
　---------------------------------


　鮒谷様


　この度は貴重な機会をいただき、
　ありがとうございました。


　普段は地方にいますが、

　たまたま東京へ行く機会ができたタイミングで
　鮒谷様と個別にお会いし、

　今後につながるヒントをいただく機会を
　紹介されたので、

　「これは何かの運命に違いない」

　と勝手に思い込み、
　即予約させていただきました。


　結論から言えば、今回お会いさせていただいて
　本当に、本当によかったと思います。

　陳腐な表現ですが、それしか思い浮かびません。


　何が良かったかというと、自分自身自覚がない

　「骨がらみのインプット過多」

　を即座に喝破いただいた点です。

　もっと言えば、

　「勉強するな！」

　と言っていただいたことです。


　学生の頃からとにかくインプットが
　いいことだということを信じて止まず、

　社会に出るまでには多読を意識して、
　とにかく貪欲に知識を吸収しようという姿勢が、

　至極もっともだと考えていました。

　しかし、そうした姿勢でい続けても状況は良くならず、
　かえって先行きが暗くなる一方でした。


　ついには職を失う羽目にもなり、
　今まで何をしてきたのか・・・という状況で、

　鮒谷様から何か今の自分に役立つお話を
　聞けるのではないか、

　そして適したサービスを紹介いただこう、
　と考えたのが今回の応募動機です。


　鮒谷様と直接お会いして自分について軽く話し、

　即座にインプット過多であることを
　喝破された時には本当にショックを受けました。

　自分の基本姿勢が

　「インプットへ逃げている」

　状態であることを、ずばっと言われ、
　先行きが暗くなる理由がわかりました。


　さらに、「行動する」ということは
　何か大それたことをやらなければならないのか？

　という自分の思い込みも叩きのめし、

　「お役立ち行動」という
　非常にわかりやすい例えで、

　小さな一歩の踏み出し方を
　示唆してくださいました。


　「行動する」と聞いて、
　即座に自己中心的な発信をしようかと
　考えていた自分が恥ずかしくなります。


　また、尺取り虫の例えも音源では
　聞いていましたが、

　直接目の前でお話いただくと
　ここまですとんと腹落ちするものかと驚きました。

　1時間程度というお話でしたが、
　気づけば時間を大幅に超過していました。


　そこまで真剣に向き合ってくださり、
　自分について思うところを話してくださった機会に
　心から感謝したいと思います。

　お話しいただいたように、
　まずは社会に対してパンチを繰り出すとともに、

　周囲の人からのありがとうを積み重ねて
　自尊心と自信を高めていきます。


　あの後、友人と会った時には早速、
　相手にとってのお役立ち行動を意識してみました。

　その結果お礼を言われた時には
　普段とは違う気持ちを感じることができました。

　きっと、鮒谷様のお話を聞いた後だったので、

　人と接する際の心の入れ方に
　変化が生じたのではないかと思います。


　お会いさせていただいた時間の中で
　サービスの紹介はあまり多くなく、

　とにかくお話をいただいたので
　申し訳ない気持ちさえしました。


　まずはご紹介のあった音源を購入し、
　かつ是が非でも土曜日に時間を作るなど、

　紹介いただいたサービスを何がなんでも
　利用させていただきたいと思います。


　この度は本当に貴重な機会をいただき
　ありがとうございました。


　　　　　　　　　　　　　（Ｙ．Ｔ さま）
　

　---------------------------------
　（ここまで）


　Ｔさん、先日はありがとうございました！

　また長文のご感想もお送りくださり、
　嬉しく読ませて頂きました。



■Ｔさんより伺ったお話は、

　まさに多くの人が陥りやすい罠（インプット過多の病）
　ですので、お話を伺った瞬間、

　「ああ、あのパターンだな、
　　例の症状だな」

　と、すぐに分かりました（笑）

　これまで同じ症状の患者さん（！？）を
　さんざん見てまいりましたので、、、



■頭がよく、真面目な方に頻繁に見受けられる、
　珍しくない症状です。

　しかし、放置しておくと

　「重篤化する」

　可能性がありますので、
　早めに処置しておくに越したことはありません。



■現状から、人生や生活を変えたいと
　本気で願うのなら、

　（もちろん、最低限の勉強（読書）は
　　必要ですが）

　あるところまで勉強したら、
　あとは、行動中心主義でいくべきです。



■勉強（特に読書）は、

　努力しているように見えて、
　その実、行動していない自分に対する

　「格好の言い訳、逃げ道」

　となり得るので、


　そのメカニズムを理解し、

　「絶対に読書に逃げない
　　＝読書をあえて止める」

　ことが必要な人も少なくありません。

　
　
■私（鮒谷）は、

　メルマガ上で読書を勧めてきましたし、
　平均より読書量は多い方だとと思っていますが、

　それでもなお、

　「今、本を読むべきかどうかは、
　　その人の状況次第」

　であると考えています。



■ただ、メルマガにおいては、

　「一人ひとりにカスタマイズした、
　　コンサル的な発信はできないので、どうしても、

　　この号は、このフェーズの人に、
　　この号は、この段階の方に、

　　という形にならざるを得ない」

　のです。



■そうすると、

　ある号では（行動する前に）読書せよ、
　と言ったり、

　ある号では読書するな（行動せよ）、
　と言ってみたり、

　一見すると矛盾していることを、同じメルマガで
　伝えているように見えるかもしれません。


　でも、それは矛盾ではなく、

　「（個別対応ではない）
　　一斉同報の限界」

　と受け止めて頂ければと思います。



■話を戻しますが、

　「本をたくさん読んでいるのに、
　　結果がなかなか出てこない」

　という人は、
　きっと、心のどこかで、

　「読書量（知識）が飽和点まで到達すれば
　　行動できるようになる（はず）」

　と考えているものです。



■少なからぬ人が、こうした勘違いから、
　
　【以下の軌跡】

　を経て、最終的には

　「全ての気力を失い、
　　成り行き任せの人生となり果てる」

　こととなるのです。
　
　
　----------------------------------------


　【勉強熱心な人が、気力を失い、
　　成り行き任せの人生となるまでの軌跡】


　＜スタート！！↓＞


　→ 飽和点の訪れを期待して、本を読み続ける

　→ 知識欲が満たされ、充実感を感じられる

　→ 読書は楽だし、他者との関わりがないゆえ、
　　 摩擦もない

　→ それでいて（本を読まぬ）他者への優越感も
　 　感じられるようになる
　
　→ 楽ちんな上に、満たされるものがあるので
　　 行動を起こすのが、さらに億劫に

　→ でも 「知識は増えているから」という
　　 言い訳を、自分に対して立てられる

　→ 読み続ければいつか飽和点を迎えるはず、
　　 そんな思いが赦しとなり、逃げ道となる

　→ しかし、外界に働きかける取り組みは一切、
　　 していないのだから、当然、結果は出ない

　→ 人生や生活は変化せず、
　　 自尊心が徐々に毀損されていく
　
　→ 人生や生活は変化せず、
　　 自信が段々と蝕まれていく

　→ 自分より読書量が少ないのに、行動し、うまく
　　 いっている人を見て、不安や焦りが生まれる 

　→ このままではまずいとは思う、けれども
　　 行動に移すための自信もきっかけも、もはやない

　→ 遂には「心の拠り所、癒やし目的」の読書へと

　→ ますますインプットとアウトプットのバランスが
　　 崩れ、やがて学んだ知識が毒素を放つようになる

　→ 勉強しているのに、成果が出ない苦しみが
　　 強烈なストレスとなり、精神を蝕み始める

　→ やがては本を読むのすら嫌になり、かといって、
　　 全く身体が動かなくなるほどに病が重篤化する

　→ すべてを放擲し、成り行き任せの人生に…

　----------------------------------------


　ということになる方が少なくありません。



■ひとたび、こうした

　「負のサイクル」

　にはまり込んでしまえば、

　その流れを自らで止めることは、
　実は、はなはだ難しい。

　
　もはや自分を客観視する余裕すら
　失われるからです。



■そんな方に対して、
　私（鮒谷）が個別コンサルをする際には

　【本を読むことを止める】

　こと、ならびに、

　【日常生活において、今すぐできる、
　　ごくごくアウトプット活動を行う】

　ことを、強くお勧めしています。


　
■詳細は省きますが、ここでは
　
　「毎日のお役立ち活動」
　「小さな３連勝」

　といった言葉がキーワードとなってきます。

　
　一旦、勉強をやめ、
　周りにいる人に尽くす、

　ことによって、

　【人生の歯車を逆方向に回すための
　　一歩目を踏み出す】

　ところから、

　人生の逆転劇を演じられた方は
　少なくありません。



■それを一人で遂行できる方が少ないから、
　
　求められる方には、お手伝いをさせて
　もらっているわけですね。


　ただ、

　上記のメカニズムを理解し、
　ほんの少しのコツを会得すれば、

　あとは、自分でもその泥沼から
　離脱できないこともないでしょう。



■確実に、そして速やかに離脱したいのなら、
　個別コンサル＆年間プログラムを勧めます

　（多数の「離脱お手伝い経験」があります）

　が、

　「いやいや、
　　人には頼らず、自分でやるわ」

　という方には、
　ご自身でPDCAを回し、試行錯誤を重ねながら

　「コツの会得」

　に力を注がれるとよいのでは、
　とお伝えしておきます。



■いずれにしても

　【読書量（知識）が飽和点まで到達すれば
　　行動できるようになる（はず）】

　という認識を破壊し、


　【正しい方法で体を動かさない限り、
　　人生も生活も変わるはずがない】

　そんな新しい認識をインストールすることが
　喫緊の課題といえるでしょう、


　大量の本を読んできたけれども、
　一向に成果が生まれない、

　と嘆かれている方にとっては。
