7368号 ほこ×たて。絶対に挨拶する人vs.絶対に挨拶を返さない人

■私（鮒谷）は、
　借家（マンション）住まいなんですが、

　挨拶しても決して返事が
　返ってこない人があるんですよね。

　お一人だけなんですが。

　あれ、なんなんですかね？



■それはそれは凄いんですよ。

　私が

　「おはようございます！」
　「こんにちは！」
　「こんばんは！」

　と元気よく挨拶しても、

　表情ひとつ変えずに
　完全黙殺。



■そういうのが苦手な方ならば
　恥ずかしそうにされたりとか、

　申し訳ないという態度が
　どこかに現れたり、

　とかあるじゃないですか。



■あるいは、

　返事するタイミングを失した
　気まずさとかでも一切なく、

　完全なる

　「自覚して行われているガン無視」

　ですね。間違いない。



■あそこまで清々しく振る舞われると、

　その鋼の心（？）に対して、
　むしろリスペクトの感情が湧いてくる、

　すらありますね。



■気づかぬうちに、

　私（鮒谷）が何かをしでかして
　嫌われているのかな、

　と思ったこともあったけれども
　様子をうかがってみると、

　どうやら私に対してだけではなく、
　挨拶する人全員に対しての模様。



■マンションに住んでいる、
　その他の皆さんは、

　挨拶をしたら、必ず気持ちの良い
　挨拶が返ってきたり、

　なんならちょっとした会話、
　までありますが、

　表情も変えずに完全黙殺は
　ただ一人。

　メンタル、強すぎっしょ。



■途中で止めるわけにもいかなくなり、

　何年もの間、
　挨拶し続けているんですが（笑）


　先方も意地になっているのか、

　あるいは完全なる不感症なのかは
　わかりませんが、

　変わらず完全黙殺です。



■『ほこ×たて』、ではありませんが、
　エレベーターの中で、

　「絶対に挨拶をする人 vs.
　　絶対に挨拶を返さない人」

　ファイッ！！！

　みたいな感じになってます。

　が、何年経っても決着つきません。



■それにつけても、あれ、
　なんかメリットあるんですかね。


　「言葉ひとつ足りないだけで
　　全部こわれてしまうような、

　　か弱い絆ばかりじゃない」

　なんて稲葉さんも言ってますが、


　とはいえ、ほとんどのご縁は

　「言葉＆笑顔」

　がなければ、
　壊れてしまうものじゃないですかね。



■なんにしても昔は、

　こうした無愛想な人や、

　路上タバコの煙や、
　返す刀でのポイ捨てをする人、


　カフェで隣りに座って
　激しく貧乏ゆすりし続ける人とか、

　甲高く、耳障りな嬌声＆馬鹿笑い＆
　手を叩き続ける、のコンボ攻撃の人、


　新幹線の隣席で、バッチバチにノートPCを
　叩きまくってる人（うるさい！）とか、

　街路樹の金木犀の枝をはばかることなく
　ボキッと折って持ち帰る人、


　などに遭遇するたびに、
　いちいちイライラしてました。



■狭量と言われれば狭量なんでしょうが、
　腹が立つものは仕方ない。


　ただその度に、怒りのため、
　希少な認知資源が削られるのが腹立たしく、

　さらに怒りが加速する悪循環、、、



■それが、このところ、

　【怒りレベル 80％オフ（当社比）】

　となる脳内プロンプト（脳内言語）を
　作り出してからというもの、


　ほとんどの出来事が
　許容できるようになり、

　認知資源が削られることも
　ほぼなくなったのです。



■他者の無遠慮な振る舞いが
　気になってしまう人は、

　人生の中で多大なコストを
　支払わされているわけですが、
