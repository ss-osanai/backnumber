※このメルマガは、外資系専門の現役ヘッドハンターが、意識の高い皆様へ、

「ピークパフォーマンス」（人生における最高の結果）を発揮することが出来
るように、年間３００冊以上の本を読み、１００日以上、勉強会・セミナーに
参加し、学び、得た、内容を編集してお伝えしていくものです。
このメルマガを読み続けることで、こんなことが出来るようになります。

●セルフプロデュース術（自分を売り込む重要なポイントを発見できます）
●自信を持って、前向きに行動することが出来ます
●自尊心を持って生きていくことができます
●得たい結果を最速で得るためのヒントが掴めます
●新しい発想のヒントを得られます
●モチベーションを維持する方法を学べます
　　　　　　　　　　　　　　　　　　　　　　　　　　　　などなど・・・

【鮒谷・今日の活動】（←今日の、と書いてますが、書いてるのは昨日
　　　　　　　　　　　　なので昨日のことです　念のため）



■かなり配信が遅れました。すみません。

　今日（って昨日のことですよ）は、平野先生の東京セミナーにご参加
　下された読者の方と、銀座でお会いさせて頂きました。

　いつもお会いさせていただく方には、メルマガにどの程度まで取り
　上げさせて頂いていいですか？と確認させていただくのですが、
　今回はうっかり、確認を忘れてしまいましたので、差し支えない程度
　で書かせていただきます。

　
■お会いさせて頂いた方は、もともと通販業界にいらっしゃり、今は
　その世界でライターさんとしてご活躍なさっていらっしゃる方でした。

　ライターさん、というご職業と、「通販」というワードに惹かれて、
　私のほうからぜひお会いさせていただきたい、とお願いして、お時間
　をお取りいただきました。

　外部からは伺い知ることが出来ない、通販のビジネスの奥深さと
　面白さを教えていただきました。
　近々メルマガを発行なさるご予定もあるようですので、ご紹介させて
　頂くこともあろうかと思います。
　
　お楽しみに。


■その後、ドリームゲート主催のセミナーに潜入。
　やっぱり、あちらこちらに見知った顔が。

　非凡＆２１時から倶楽部の増澤さん、先日、ご紹介させていただいた
　マーキュリッチの野村副社長、フォレスト出版のＹさん、などなど。

　野村さんとは、帰りの電車の中でしばらくの時間でしたが、情報交換
　を図ることができました。


■その後、事務局長沼と打ち合わせ。

　平野先生・大阪セミナーの最終調整、人数確認、と今後の構想について。
　
　来月早々より、週一回早朝（７時～９時）に、

　【セカンドステージ・ユニバーシティ（ＳＳ－Ｕ）】

　を開講することに決定いたしました。

　
■しばらくの間は、事務局・運営スタッフの中心メンバー限定の参加として、
　軌道に乗ってきたら一般開放をしたいと思っています。

　（ただし、内容はあくまでセカンドステージドットジェイピー、という
　組織体の運営、それに伴う個々人の実力の蓄積、といったテーマです。
　ですから通常のセミナーとは全く異なります。
　
　ただ、今後の人生で必ず役立つであろう、内容を盛り込んでいこうと
　思っています）

　勿論、これは公開する際は無償にて公開予定です。
　
　※以前（だいぶ前ですが）ご案内したメルマガセミナーについては
　　私がやろうと思っていたのですが、それを専門に実績をあげられて
　　いる方へおつなぎしたほうがよいだろうと思い、そちらのほうへの
　　ご紹介をさせて頂くことが出来るようにいたしました。

　　一定基準をクリアされる方に対し、オンラインショップのコンサルの
　　第一人者である江藤政親さんのセミナーをご紹介させていただく
　　ことが出来ます。（通常のコンサル契約から比べるとほんとに破格の
　　安値で受講が可能となっております）
　　お断りさせていただくこともございますので、予めご了承下さい。

　　江藤さんはご著書を２冊お持ちで、今回のセミナー参加者で実績
　　があがったケースにつきましては、来年春に出版される書籍でも
　　取り上げられる予定です。
　
　（詳細はこちらを）
　　→ http://backno.mag2.com/reader/Back?id=0000114948

　　１０月３０日（１０時４０分配信分）のバックナンバーをご覧下さい。
　　一番下のほうに書かせて頂いております。


・・・


■セカンドステージ・ユニバーシティの話に戻りますと、内容は、

　《前半》が目標設定、モチベーションの維持管理、その週の振り返りと、
　次週の目標設定など、参加者各人の目標に対する進捗具合の確認など。

　《後半》が、実際に各種のプロジェクトを推進していくにあたっての
　具体的な実務（ノウハウ・メソッド・スキル）についてのノウハウ。

　これを週一回、定期的に開催することで、私の中にある暗黙知を
　どんどん形式知に落としていき、マニュアル化を進めていきます。
　そしてセカンドステージドットジェイピーに今後、参加されたい、
　という方がそれを読むだけで、最低限の、能力発揮の方法について
　の理解が出来るような形に落とし込むつもりです。


■最近、たくさんの方々から、セカンドステージドットジェイピー
　への何らかのかかわりを持たせて欲しいとのお話を頂くように
　なりました。

　ありがたいことです。そして嬉しいことです。
　ただ、現状、受け入れ側の態勢（＝私の時間的な余裕のなさの
　ため）なかなか対応できないような状況です。

　ですが、来年に向けて、セカンドステージドットジェイピーに関わる
　個々人が次のステージ（セカンドステージ）へ向けて、大きく成長
　できるような環境をどんどん整えていきたいと思いますので、
　もうしばらくお待ち下さい。

　来年に入ったら、当メルマガ上などにて、ご案内させて頂こう
　と思っております。

　
■帰宅後は、昨日、メルマガで取り上げた、
　
　【今日のお勧め図書　人生を変える80対20の法則】

　を再読しておりました。

■昨日のメルマガへも、たくさんの反響を頂きました。
　ありがとうございます＆大変嬉しいです。

　味をしめて（？）、続編です。


■たくさんの新しいパラダイムを提供してくれる本、という意味で、
　昨日、ご紹介させていただいた

　【今日のお勧め本　人生を変える80対20の法則】
　http://www.amazon.co.jp/exec/obidos/ASIN/4484981068/2ndstagejp-22
　
　は私にとって大当たりの本でした。

　全部をご紹介させていただくわけには参りませんが、この本のキモ
　の一部分、ご紹介します。


■お求めになられた方、詳細は５０ページをご覧下さい。
　勿論、こういう内容がそこかしこに散りばめられている良書です。


　▼努力の平均水準を上げるのではなく、努力を一点に集中する
　▼最小限の努力で人生の支配権を握る
　▼網を広げるのではなく、網を狭める（→耳が痛い　筆者註）
　▼日常生活で出来るかぎり、アウトソーシングを勧める
　▼重要な分野ではすべて、２０％の努力が結果の８０％に
　　つながるように調整する
　▼手当たり次第にチャンスに飛びつくのではなく、気をしずめ、
　　仕事量を減らし、ゴールへの最短距離に照準をあてる
　　（→同じく耳が痛い　筆者註）
　▼一番得意とすること、いちばん楽しいと思うことだけをやる


　などなど。
　これだけ読んだだけで、自分の仕事のやり方、その他時間管理
　の方法を反省させられないでしょうか？


■経済学を学ばれた方はご存知だと思いますが、比較優位説、という
　のがあります。
　実はこの比較優位説と8020法則、表面上は関連しないように見えて
　根っこの部分につながっているのよね、と昨日、再読して、はたと
　気づきました。

　比較優位説を徹底的に分かりやすく説明した、大変よい本があります。
　（小学生にも分かるように）


■小室直樹さんの本なのですが・・・。
　
　目からウロコ、という最近、いたるところで目に付く表現。
　ここまでいろんなところで使われると、どうもへそまがりな私は
　陳腐な言葉に聞こえてしまって、あんまり使いたくなくなったりしてる
　んですが、その本読んで、目からウロコが落ちまくりました。
　今からたしか７年ほど前の出来事です。
　
　機会があればご紹介します。
　ご紹介したい本は山ほどあるのですが、毎日たくさん紹介すると
　読者のみなさまも混乱されると思いますので、ここはぐっと辛抱
　します。

　
■というわけで今日はこのくらいで。

こんにちは。はじめまして。
ひと月ほど前から、メルマガ購読させていただいてます。
小学生の子供が２人いる３０歳の主婦です。

今まで２０代は１００％子育てに一生懸命になりすぎて、
自分のことをあとまわしにしてきたので、
すこしづつ、自分のこれからのことを考え始めてきたところです。

目標や、夢や、こうなりたい、こうしたい、というものがたくさん
ありすぎて、そのたくさんの夢をどうやってまとめたらいいか、
なにを切り捨てたらいいか、どう、行動にうつせばいいか、がわからず、
常に自問自答です。
資格もとったり、資金づくりのためにバイトもはじめたけど、なんか
ちがうかなー？というところです。
前向きになれるような自己啓発的な本、メルマガをいろいろ読んでいる
うちに必然的に「平成・進化論。」にたどりついてきました。

鮒谷さんの一日の日記には毎回、その充実度に感心させられます。
刺激になる人たちとあって話しする毎日ってとても素敵で魅力的です。
そしてピークパフォーマンス方程式も毎回、
大事なことにきづかされて、感動があって、刺激になって、とても前向き
になれるのです。
これは！とおもった言葉は手帳（フランクリン・プランナー）にかき
とめるようにしてあります。
おすすめの本も全部読みたい！のですが、
自分のペースで、少しずつ読んでいこうとおもいます。
先日トム・ピーターズさんの本を注文したばかりなので
原田監督さんの本は、次に、とおもっております。

たくさんの学ばせていただいたことから、自分がすこしづつ進化して
いってるように思います。
これからの子供たちの教育にも、とても役立つことでしょう。
今はまだまだ１００％自分中心に生活はできないので、バランスとりながら、
なので、何かを始めるには時間がかかるかもしれませんが、
これからも少しずつ、長い目でみて、自分を進化させていこうと思います。
昨日より今日、今日より明日。
先月より今月、今月より来月。
去年より今年、今年より来年！

鮒谷さんのメルマガに出会えたことにとても感謝いたします。

「平成・セルフプロデュース論」もとても楽しみにしています。

どこかで聞いたことはありましたが、
何かに役立てるという意識をもっていなかったので、
はっとさせられました。

視点で書かれていらっしゃることに共感しています。
また、尊敬の気持ちも持っています。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

《鮒谷より》
ご感想ありがとうございます。
今度の大阪セミナーにもご参加くださる方です。
８０２０法則は、大抵の方は聞いたことがあるのですよね。
でもなかなか実生活に意識して取り入れている人は少ないようです。
かく言う私がその典型なのですが・・・。

セミナー当日、お会いできることをたのしみにしております！

（最後のお一方ご紹介・一昨日のメルマガのご感想です）
＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

【完璧を求めるな　求めるべきは進化。】・・・・

 今の私に、これ以上、これ以外の助言はありませんでした。
 （すいません、日にち遅れで。。。パソ買い替えとダブリ、今まとめ読みです）

私は何をするにしても、まず納得がいくまで完璧を求め、
発進時には、完璧でなければ怖くて駄目だったんです・・・
実はごく最近、実務翻訳の世界に飛び込み、そのダイブ直前、
まだまだスキル不足の私を紹介してくれた知人に対し、
（時期尚早だよぉ）と、内心、愚痴をこぼしてました・・・

が、発進してから、苦労は多くても、鮒谷さんおっしゃる所の通り、
悪戦苦闘しつつも、進化の加速は自分でも信じられぬものとなりました。
これが、立ち止まってまだ、発進していなかったら今頃はまだ・・・・
ですね。

本当に　いつもいつも、適宜適切に、狙い済ましたようなアドバイス・・・
私だけでは無いと思います、このような感想を抱くのは・・・・。
何度目かの投稿になりますが、これからもずっとずっと読み続けて
進化の糧にさせていただきます、ありがとうございます。

＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝

《鮒谷より》
いつもありがとうございます。
皆さますべてのご感想をご紹介することは出来ないのですが、毎日、

みんなが次のステージへと上がれる、そんな交差点的な存在になって
いければいいなと思っています。
