＜今日のピークパフォーマンス方程式＞

【 最大にして最強の資産、それが「良質の言語体系」 】

───────────────────────────────────



■不透明な時代にあって、


　私たちが頼れるものには二つあり、
　それは

　「（一定以上の）有形資産」

　と

　「（一定以上の）無形資産」

　です。

　
　有形資産とは、目に見える資産。
　
　一番分かりやすいのは、

　「お金（ならびにお金に類するもの）」

　でしょう。

　
　これがあれば、多少は安心できるもの。



■きれいごとを言ってもしかたがないので
　ハッキリ言いますが、

　「世の中、
　　お金が全てではないけれども、

　　お金がないと、
　　どうにもならないことがある」

　のは間違いありません。


　それを公に言うかどうかはともかくとして、

　皆、口には出さずとも、
　心の奥底では分かっていることです。

　
　いざというときに、

　「カネがないのは首がないのと同じ」

　です。
　


■ですから、


　人間、若いうちはともかく、
　ある程度の年齢になると
　
　「（一定以上の）有形資産」

　は手元に置いておきたいもの。


　しかしながら、有形資産には、
　
　「いざというとき、
　　あんまり当てにならない（ことがある）」

　という性質があります。


　分かりやすい、一つのシナリオを考えるなら、

　「ハイパーインフレが来たら一発退場」

　となる可能性があるということ。



■それに比べると、

　「無形資産」

　すなわち、

　ブランド、人間関係、信用、信頼、
　スキル、能力、知識、経験、人格、


　その他、

　（たとえば私であれば）
　
　ウェブサイトとか、

　メルマガバックナンバーとか、

　これまでに出版した書籍（と出版した事実）、

　メールマガジンに登録下さっている方、
　

　などがあると、

　「一発退場のリスクは大幅に減じられる」

　こととなるでしょう。



■危機管理の要諦は、


　「最悪を想定して、最善を尽くす」


　「備えよ常に（Be Prepared）

　　＝いつなん時、いかなる場所で、
　　　いかなる事が起こった場合でも
　　　善処が出来るように、常々準備を怠ることなかれ

　　　　　　　　　　　　（ボーイスカウトのモットー）」


　ですが、


　人生において考えられうる最悪の事態は、

　「自分の人生が
　　自分でコントロールできなくなること」

　でしょう。



■それを防ぐためには

　「有形資産」

　を蓄えておくことと同時に、
　

　むしろ、それよりも優先して、

　「（有形資産を生み出す）無形資産」

　を蓄積する必要があるわけです。
　

　そして、私（鮒谷）が個人的に考える
　最大の無形資産は、

　「明確に言語化された、
　　質の高い意思決定基準」
　
　ではなかろうかと思うのです。



■たとえば、

　ここまでに有形資産、無形資産という言葉を
　記してきましたが、

　両者の価値の軽重について、


　「無形資産が有形資産を生み出すのだから
　　最優先して蓄積すべきは無形資産」

　という言葉を一つ持ち、
　それを心がけて５年、10年生きているだけで、

　人生は大きく異なってくるものと
　思われます。



■そして、こうした人生を形作る言葉を、

　一つどころか二つ、三つ、五つ、十、
　数十、そして数百、、、

　と増やし、


　さらには意思決定を迫られるたび、
　自在に使いこなすことができるようになれば、

　基本、

　「人生における喜び、楽しみ、安心、幸福感は
　　逓増（徐々に増えていく）のではないか」

　と考えているわけです。



■つまり、人生の質を高めるためには


　「良質の意思決定基準を持ち、
　　時間を味方につけつつ、

　　ゆっくりと、しかし着実に、
　　『自分がありたい願う姿』に近づく」


　ことが必要なのではないでしょうか。



■来月、行なう２日間セミナーでは、
　

　私（鮒谷）がおよそ20年の時間をかけて収集し、
　さらには、自らの生活を通して実験し、


　（中でも十数年前から記録は

　　すべてメルマガのバックナンバーとして
　　当社サイト上に掲載しています）


　その上で、間違いなく効果があると確信の持てた

　「右肩上がりの未来を約束してくれる
　　圧倒的なボリュームの、良質の言葉たち」

　のみを厳選して、お渡しさせて頂きます。
　

　
■それを、

　「わずかこの程度の金額」

　で買えるのだとすれば（個人的には）

　「ありえないくらい、安価」
　
　だと思っているのですが、

　その価値を価値と感じて下さる方のみに、
　お越し頂ければと思っております。

　高いと思われる方は無理にお越しに
　ならなくて結構です！（キリッ）　


　お申し込みの意思決定にあたっての判断材料は、
　毎日のメルマガでお届けしているつもりです。
