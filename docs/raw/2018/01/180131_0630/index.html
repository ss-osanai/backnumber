■今の時代にあっては

　「独自の経験や知識やノウハウ」

　は大いなる無形資産です。


　これらは今風の言葉でいえば（！？）

　「知的財産」

　ですし、

　然るべき知的財産を他に先んじて手に入れた人は、
　人生やビジネスを

　「一気にショートカット」

　することも難しいことではありません。



■問題は、

　こうした知的財産の価値を的確に評価でき、
　妥当と思われる代価を支払い、

　そこで得られた知識やノウハウを活用し、
　人生やビジネスをどんどん加速させていく人と、
　

　（おそらくは無形の財産は目に見えないから）
　
　知識やノウハウに大きな価値を見出せず、
　頑張って全部、自分でやろうとする人、

　の両者の差がどんどん大きくなっていることです。


　ただ、その意思決定は100％、
　各人の自由意志に委ねられるものなので、
　
　他人がとやかく言うようなたぐいの話では
　ないのですが、、

　

■話は変わりますが、


　（ご記憶のある方もおありかと思いますが）
　
　元通産官僚にして作家、
　経企庁長官も務めた堺屋太一氏の著した

　『知価革命
　　─工業社会が終わる 知価社会が始まる』

　という本がベストセラーになったのは
　1985年のことでありました。

　（あれから33年も経っているんですか、、、汗）


　そんな頃から

　「知識にこそ価値がある」

　時代に入りつつあったわけですね。



■今はますます、

　「知識を持ち、さらに求める者は
　　ますます加速し、より一層富み、

　　知識を持たず、求めようともしない者は、
　　はじめ緩慢な、やがて急速な衰退となり、貧に堕す」

　そんな時代になっているのかもしれません。



■このメルマガをお読みのあなたは、
　もちろん

　「知識を持ち、そしてさらに求める者は
　　ますます加速し、より一層富み」

　の側にいらっしゃる方だと思いますが、
　

　10年前、20年前、30年前、、、の昔と比べても、
　
　「学ぶ人は、圧倒的なレベルで学ぶ」

　そして反対に、
　
　「学ばない人は、、、（以下略）」

　そんな世の中になりつつありますね。



■つまり、昔よりもはるかに

　（ある一部の人にとっては）

　「それぞれの専門分野における
　　知識、ノウハウ、コツ、勘所」

　に対する需要が高まりつつあるということです。

　
　もし、

　「需要が高まる一方なのに、
　　供給がそこまで追いついていない」

　そんな市場があったなら、
　そこにおいては

　「高額を支払ってでも、自分を
　　加速、変身させてくれる知識を求めたい」

　という方が陸続として現れられるでしょう。



■たとえば私（鮒谷）であれば、あるとき、


　「大儲けはできないけれども、

　　リスクを極小化させ、潰したくても
　　潰せないほどに安全な事業を作り、

　　その上で、リスクに比して、
　　それなりのリターン（ミドルリターン）を狙う、

　　そんなビジネスの立ち上げ方」


　という

　「万人受けはしないけれども、
　　ある種の方にとっては買わずにおれない」

　そんな自身の経験から編み出した

　「知識、ノウハウ、コツ、勘所」

　を販売することとなりました。

　（決済手数料を差し引くと「46,810円」の音源が）、

　売れ続けているのをご覧頂けるかと思います。



■要約すると、

　「2017/02/08 13:48」から
　「2017/02/08 14:08」までのわずか20分の間に、

　「48,600円の音源が30本売れている」

　のです。


　もちろんこの前後、すなわち

　「2017/02/08 13:48」以前、ならびに
　「2017/02/08 14:08」以降も、

　同様のペースで売れ続けているわけですが。



■これは今のような形で音源を売り始めて、

　まだ２ヶ月経つか経たないか、
　くらいの時期における数字なのですが、

　そこから１年近くの時を経て、
　現在はどのようになっているのか。


　、、、実は販売ペースはさらに
　加速しているのです（驚）

　

■でも、（驚）とは書きましたが、

　本当のところをいうと、
　それは驚きでも何でもありません。


　最初は不安を抱え、躊躇しながら、
　おそるおそるお求め下さった方が、

　一旦、購入し、お聴きくださると、
　かなりの割合でリピート受注下さることとなり、

　より一層、音源の販売数が上積み、上乗せ
　されていくことなりました。


　（また（直近の）更に加速している別の画像を、
　　折を見てご紹介できればと思います）

　

■こんな風にして、

　私は自分の経験を体系化＆パッケージ化し、
　音源として販売し始めたわけですが、


　正直にお伝えすると、

　一本リリースするたびに、
　このくらいのペースで音源が売れ続け、

　（商品にもよりますが）最終的な売上数値としては
　およそ、

　【上場企業役員の平均的な役員報酬（年収）】

　くらいの売上（＝ほぼ利益）が立つことと
　なりました。


　繰り返しますが、一本、
　音源をリリースするごとに、です。

　しかも私が音源化にかけた

　「直接的な実労働時間」

　は、音源作成に吹き込む数時間のみ。



■サラリーマン適性（耐性？）はまるでゼロ、
　会社員人生を全うすることができず、

　戦場（企業組織）に放り込まれても
　数年間しか持たず、

　敵前逃亡を図って、土中に潜み、
　ゲリラ戦での戦闘を志向した私でしたが、


　ぐるっと一周回って、

　機嫌よく、好きなことだけをして、まあ、
　それなりにはご飯を食べられるようになり、

　「それにしても、よく頑張ったなあと、
　　自分で自分を褒めてやりたい」

　そんな気持ちになることもしばしばです。

　（自分で言うなという話かもしれませんが、
　　本当のことなのだから仕方ない）



■でも、実は私と全く同じことを

　（それぞれの専門性を持つ）
　あなたもまた、

　もちろんできるのです。


　ここまでお読み下さったあなたもきっと、

　私とは異なる、それぞれの分野における
　プロフェッショナルとして、

　経験を重ね、知見を得て、
　それを体系化し、言語化し、

　後進に伝えるためのコンテンツを
　持たれているはずだからです。



■でも、難しいのは、

　せっかくすぐれて優秀なコンテンツを
　お持ちで、外販する意志もあったとして、

　「何を、どんな順番で、どこからどのように
　　始めたら良いのか分からない」

　ということもあるのではないでしょうか。
