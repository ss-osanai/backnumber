■仕事する際に、余計なことを
　考える必要はありません。


　---------------

　１、勝ちパターン（成果の出る方程式）を
　　　確立し、

　２、必勝パターンを淡々＆延々と
　　　繰り返す

　---------------


　ただ、これを行うだけで、

　平均を大きく上回る成果を
　出せるようになるはずです。



■複雑に考えるほど混乱し、動けなくなり、
　結果を出せなくなるものです。

　シンプル・イズ・ベスト。



■思うように成果を挙げられない、
　そんな悩みがあるならば、

　仕事を上記の二つに「因数分解」して
　考えるとよいでしょう。

　自分に足りないものは何であるか、
　すぐに分かるようになります。



■すなわち、

　「勝ちパターン（成果の出る方程式）」

　が確立されていないから
　うまくいかないのか、


　もしくは、

　「勝ちパターン」は
　できているけれども、

　継続して行えないところに
　問題があるのか、

　原因を特定できるようになります。



■前者（勝ちパターンがない）が
　うまくいかない理由なら、

　さらに因数分解して

　---------------

　１、学びが足りない

　２、たくさん学んできたけれども
　　　自分の環境に落とせていない

　---------------

　のいずれに原因があるかを
　検討するとよいでしょう。



■そして学びが足りないのなら、
　然るべき学びを。

　学びを実装できていないのなら
　学び→実装、の意識と習慣化を。



■後者（方程式はあるが継続できない）が
　うまくいかない理由なら、

　こちらも因数分解して

　---------------

　１、継続のための仕組みづくりの
　　　方法が分からない

　２、仕組みづくりの方法は知っていても
　　　仕組みを作るのを先送りしてしまう

　---------------

　のいずれに原因があるかを特定し、
　解決すればいいのです。



■以前にリリースした

　【直進行軍音源】

　は、まさに

　【勝ちパターンを確立し、
　　それを持続させるための方法】

　について
　お伝えしている音源であり、

　中でも後者（大量行動を持続させる）
　ための方法を語っています。



■とはいえ、

　単なる方法論かといえばそうではなく、
　個人的には、むしろ、

　【仕事に向き合うに際しての
　　思想、哲学】

　について語っている、
　応用範囲の広い音源と捉えています。



■「直進行軍音源」は
　私（鮒谷）のお気に入り音源で、

　【自身の大量行動を支えるベースとなる
　　思想＆具体的方法】

　について語っていますが、


　ここでお伝えしている認識と、
　実務的な仕事の進め方がなければ、

　自身の生産性はおそらく

　「数分の一から十分の一以下」

　になっていたに違いありません。



■直進行軍は、時間をかけて上手に
　自分仕様にチューニングできれば、

　大量行動を行いながらも
　ほとんどストレスを感じなくなる、

　そんな状態に持っていくことすら
　可能になるでしょう。



■ところで、
　前回の年間プログラムにおいて

　【直進行軍音源】

　のアップデート版に相当する

　【続・直進行軍音源】

　的な話をいたしました。



■これは先の音源内容を踏まえた上で、

　より一層のストレス軽減と
　直進行軍を楽しくするための、

　「アップデート版」
　「アドバンス版」

　みたいな位置づけのコンテンツであり、

　お聴き頂ければ、
　さらに高い解像度で

　【直進行軍（大量行動）】

　が可能となるはずです。



■先の「直進行軍音源」と合わせて
　お聴きいただくことで

　あらゆる障害を乗り越えて、
　エネルギッシュにバリバリ行動して
　結果を出していくことが可能となり、

　効果倍増、３倍増となること
　請け合いです。
