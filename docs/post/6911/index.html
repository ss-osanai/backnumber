　前回リリースの音源（メタ認知音源）
　を聞いて、

　人生という道中、
　でっかく構える心持ちができたところで、

　より具体の「目標設定」に進んだ
　今回の音源、

　待ってました！

　とばかりに聞かせて頂きました。


　一般的に目標設定と聞くと

　目標を達成できるか否かに意識がいき

　未達を避けるために
　予定調和な目標を立てる

　みたいなことになりがちですが


　鮒谷さんの語る目標設定は

　感情が動く言葉だけを集める
　ワクワクするもので

　「最高に震える言葉集」

　を編纂するというか、

　心躍る生き様を主体的に選ぶ行為なんだと
　再確認できました。


　しかも、

　「手段の目的化」

　「べき論の目標」

　「できなかったことに
　　意識が向いてしまうときには」

　といった、陥りがちな窮地の脱出法は

　この落とし穴にハマりそうな自覚が
　あるだけにありがたかったです。


　一番ありがちなのが
　現状に流れ流されてしまうことですが

　それは織り込み済みリスクというか
　ネジは緩んでしまうものだという前提で
　お話しいただき、

　今回しっかりと巻き直しができました。


　音源を聞いてすぐ実行したことは

　「未来に臨場感を持つ」

　ということです。


　臨場感は、今までの経験からも
　その大切さを感じていて

　肌触りを持って未来をリアルに想像でき、
　こうなって当然だと思えたら
　現実になる可能性も高くなる気がします。


　ただ、目標に向かって歩き出したときは

　目標を実現した姿と
　目標を実現するまでのプロセス

　そのいずれも
　リアルに想像するのが難しかったりします。

　でも、目標がワクワクするものだと
　それを考えるのも楽しいので

　日々立ち返って
　精緻化していきたいと思います。


　ちょっと脱線しますが、

　臨場感と聞いたときに
　思い浮かべる映像があって、それは

　「アナと雪の女王」の
　「Let it go」の動画です。

　曲の後半で、エルサが
　何も無い雪原に一挙に氷城を築くのですが


　これを見た時に
　妄想を現実（築城）とするには

　城の構造といい装飾といい
　細部まで描かれていないと、単に

　「氷のお城に住みたいな～」

　では城は建たないな、

　と思った記憶があります。

　あれくらいの臨場感を目指します笑


　そしてこの臨場感という流れがありつつ

　「意思決定基準」

　についても語ってくださったので

　未来の私はどういう意思決定基準を持って
　どういう行動をとっているんだろうか

　あるいは

　「正のフィードバックループ」

　を語ってくださったので


　未来、あるいは未来へと行く過程で
　どういう人間関係を持っているのだろうか

　といったように可視化されにくいところまで
　目標について想像が膨らみました。


　目標（妄想）がより高解像度に、
　より広がりを持ち

　まさに令和アップグレードできました！

　貴重な機会をありがとうございました！

　　既にそうなると知っている】

　くらいの強さがあるのです。



■私（鮒谷）なら

　【今の生活が実現できているのを
　　とうの昔から知っていた】

　ということになります。

　（そして時をおいてその通りになった）



■あなたは、あなたの将来が
　どんな世界になるのかご存知ですか。

　既にイメージがあるとするならば
　（良くも悪くも）それがあなたの未来。



■多くの人は、

　たとえ口で
　いろんなことを語っていても、

　あるいは頭で理屈をこねて
　いろんなことを考えていても、


　最終的には

　「今、イメージしている未来に向けて
　　着実に一歩づつ近づいている」

　のです。



■これは素晴らしいことであるとともに
　恐ろしいことでもあります。

　楽しい、理想の未来が訪れると
　「知っている」人は本当にそうなるし、

　悲劇的な将来（老後）が訪れると
　「知っている」人もまた然り。



■現状にひきずられることなく、
　理想の未来を思い描き、

　その実現を微塵の疑いもなく
　信じている人は、

　「信じている」というよりも、

　「今から、
　　そうなると知っている」

　という強度で自身の未来を
　理解しているから、

　なんのためらいもなく動けるし、
　イメージを形に変えていきます。



■こうした

　「未来に対する絶対的な確信」

　は、躊躇なく、疑いなく、
　「私」を突き動かし、

　「大量行動」

　を促すわけですが、この状態こそ

　「確変モード」

　です。



■言葉を換えると

　「圧倒的な臨場感を感じさせる未来が
　　現在の大量行動を促す」

　ことになります。

　多くの人が考えているのと異なり
　因果関係が逆、なんですよ。



■みんな、

　「今、頑張ると将来、良くなる
　（理想が実現されて、臨場感が湧く）」

　と思っているじゃないですか。

　違うんですよ。
　違うも違うも大違い。まるで逆。



■未来に確信を持てない人に、

　頑張りも踏ん張りも
　生まれるはずがありません。


　先に妄想を膨らませよ、
　そしてあなたの未来を知れ、

　ということですね。

　■未来が訪れる前から

　　「そうなれると知っている」

　　人だけが本当にその世界に到達する。

　　本当にその世界を実現させてしまうのだ。

　■理想の未来を「知る」どころか、

　　「信じる」どころか
　　「掲げてすらいない」状態で、

　　それでも頑張ろう、といった精神に
　　なれるはずがないではないか。

　■「目標設定→達成」は、

　　「現在から未来」ではなく
　　「未来から現在」の時間軸で起こるもの。

　■最初からそうなると予め知っていた状態を
　　生きていく中で回収し続けていく、

　　そんなプロセスなのである。

　■このことが腹落ちできた人から順に
　　「変わり続けられる人」となる。
