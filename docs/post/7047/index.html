7047号 本気になりたければ絶望と希望の間のギャップを最大化させよ

■私（鮒谷）の好きな言葉に

　【悲観的に考え、
　　楽観的に行動する】

　というものがあります。



■好きな言葉であるとともに
　大事にしている言葉でもあり、

　この言葉の通りに
　生きてきたつもりですが、


　本当のところをいうと

　【絶望的に考え、
　　楽観的に行動する】

　といったほうが、
　よりしっくりきます。



■この反対、

　【楽観的に考え、振る舞い、
　　人生を絶望で終える】

　のは何が何でも避けたいもの。



■人間の動機には

　「楽しい、せずにはいられない、
　　上手くなりたい」

　というものと、

　「○○という状態にだけは
　　なりたくない」

　という、大別すると二種のものが
　あるように思われます。



■両方、ハマると
　強いエンジンとなり得ますが、

　最強なのは
　両方を自覚的に用いること。



■今日は記しませんが、

　両者それぞれ長所と短所があるので、
　そのあたりの理解をした上で、

　可能な限りそれぞれの思いを強くして、

　【プラス方向への願いと、
　　マイナス方向を避けたい思いの
　　ギャップを最大限、拡張する】

　ことです。



■この方法は

　「動きたいのに動けない」

　という悩みを持つ人にとって
　大きな力を発揮します。



■両者の間のギャップが
　一定の水準を超えると

　「俄然、本気モードが
　　発動されやすくなる」

　からです。



■もちろん、いずれか片方だけ、

　つまり、理想を描き、
　その世界を求めずにはいられない、

　あるいは反対に、

　絶望的な未来を想像し、
　なんとしてもそこから抜け出したい、

　という、いずれかの思いだけでも
　スイッチは入ります。

　ただし可能であれば両方使った方が早く、
　強力なのは間違いありません。



■こうすれば両者間のギャップの
　閾値を超えさせやすくなるし、

　スイッチ入ったあとの駆動力
　（＝行動力）、

　も半端ではない
　大きなものとなるでしょう



■後付けにはなりますが
　本日開催の

　「台湾有事勉強会」

　と来年年初に開催の

　「経済入門セミナー」

　は、それぞれ反対方向のベクトルの
　感情を引き出してくれるはず。



■すなわち、

　「将来の危機感と、未来の楽しみ」

　の両極の方向性。


　こうした両側面の学びを
　取り入れることによって、

　ますます人生に
　本気で正面から向き合える、

　ということは間違いなくありますし、

　私（鮒谷）自身も
　自覚的に用いてきた力。



■未来に向けたワクワクが
　喚起されるであろう、

　元三井住友銀行チーフエコノミスト、
　山口曜一郎先生をお招きしての

　--------------

　【人生サバイバルのために、全ての人が
　　学んでおくべき一生モノの経済入門セミナー】

　▼2023年1月14,28、2月11,25日
　　（いずれも土曜日） 13時半～17時

　--------------

　は明日から申込み受付けを
　開始する予定です。



■未来に対する明るい気持ちを持ち、
　本気で生きていくための、

　ひとつのきっかけ、材料となるかと
　思っております。


　正式な告知まで
　あと少しだけお待ち下さい。


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】

　■希望と絶望、両方のベクトルに意識を向け
　　両者のギャップが最大化されると、
　　そこで人は初めて本気になるのである。

　■未来への希望、未来に対する絶望、
　　いずれか片方だけでも相応の馬力は出る。

　■しかし理想的には両者の力を
　　フル活用したいもの。

　■それによっていち早く、強力な駆動力を
　　獲得できるようになる。
