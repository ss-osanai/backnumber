　メルマガの文体を変えています。

　文末が「なんよ」になったりしてますが、

　こっちの方が精神を開放し、
　より率直に文章を書ける、

　（正直、こちらの方がリアルな私、
　　もといワイのキャラに近い笑）

　ので、しばらくこの運用で
　試してみようと思います。


　すぐに慣れると思うので
　お付き合いください（笑）

■子どもの教育論/言語デッキ/起業の一丁目一番地/
　焼け野原からの復活/儲けるより儲かる/
　オッサン各種取り揃えてます/3階級特進

■今日も、

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------

　子どもに教育つけて、
　このレールに乗せて、

　みたいな戦略指南？が
　流行っとるみたいやが、

　確率論で子育てしたって
　うまくいくのは1割ないやろ。


　親の願い通り、確率通りなら、

　ワイも中2くらいまで財務官僚の目すら
　ゼロではなかったかもやけど、

　現実はこの有り様よ笑。

　人生なんて
　七転び八起きのダルマみたいなもん。

　----------

　人生のコントロール感は

　「豊かな未来を生み出す言語デッキ」

　を構築し、使いこなすことで得られるんよ。


　「良質の言語体系」という
　最大の無形資産を持てれば、

　意思決定の質を高められる。


　そこまで脳内辞書を作り込めれば、
　未来を切り拓く強力な道具になってくれるよ。

　「行動は思考のしもべ」やからな。

　----------

　度が過ぎる、とはいい言葉やな。

　なかなか度を越せんから
　平均値から出られんのやからな。

　規格外の成果を出したければ、

　常軌を逸したレベルで
　フルスイングせんといかん。


　常に全力が大変なら、

　数年ないし5年か10年の間、
　渾身の力を込めて、

　一気呵成に仕組みを作り、
　その後のことを考えればいい。

　----------

　起業相談よく受けるけど、
　一番大変なのは集客。

　そして賢い人ほど背を向けるのよ。
　集客は格好悪いからな。

　代わりに力を入れるのが
　映えるオフィス作りとか。


　必然的に起業直後から金が流出し続け
　即廃業、即出戻りで、
　もっと格好の悪いことに。

　起業前に集客ルートを作っておくから
　垂直立ち上げできるんよ。

　----------

　見込み客の少ない中、
　商売するとかマジで大変やで。

　というか想像するだけで恐怖する。


　ワイは長年、ご縁を温めてきた
　分厚い座布団のような顧客基盤があるから、

　大舟に乗った気持ちで
　ビジネスできてるわ。

　そのための大きな（金銭以外の）コストを
　めちゃくちゃ払ってる。

　でも十二分に報いを得てきたよ。

　----------

　「影響を 受けて与えて 人間界」

　突然こんな句が降りてきたんよ笑。

　我々は多くの人から影響を受けては与え、
　生きている。

　そこから逃れられんのが人間。


　とすれば、

　影響力がどのように生まれるかに
　無関心ではいられず、

　無関心なら
　自覚なく食い物にされるということ。

　無限に勉強すること出てくるで笑。

　----------

　ガチでビジネスしたことのない
　外野に限って

　「商売＝どうせ金儲けやろ」

　みたいなことを言うんだわ。


　真剣勝負で役に立たないと、

　（特に長期では）
　お金をもらい続けられないのだから、

　できるもんならやってみろ、やで。

　正しくお金を頂戴することが
　いかに大変なのかは、

　やってきた人間にしか分からんよ。

　----------

　事業で一番大切なのは顧客基盤やで。

　これさえあれば焼け野原からでも、
　即日復活できるんよ。


　それ以外の全てが揃ってても
　見込みのお客様がゼロならば、

　売り上げが経つまでに
　相当の時間がかかる。


　極論、顧客以外の全てはカネで買えるけど、

　どれだけカネを出しても決して
　買えんのが顧客名簿と信用と信頼。

　----------

　きれいごとが嫌いやねん。

　ビジネスの何が悪いんや。

　緊張感を持ち、顧客が求める価値を
　出し続けなきゃ、いずれ廃業になるんだわ。


　良いものを納得頂ける価格で提供するから
　クレームなく、長い間、お付き合い下さるんよ。

　当社は10年20年の
　ご縁の方が沢山いらっしゃり、

　利益を出すのを全肯定頂いとる。

　----------

　起業したいと妄想し始めてから
　実際の起業までには、

　おおむね平均数年はかかるんよ。

　パラダイム更新には
　それくらいの時間がかかるいうこと。


　起業したいモードに入っても
　なかなか離陸ができず、

　焦っとる人も少なくないと思うけど
　頭の中身はそんな簡単に切り替わらん。

　焦って焦らず、どっしり構えるべし。

　----------

　利益を出したければ、
　お客さまに尽くし、
　期待を超え続けなければならんのよ。

　それはほとんど修行であり、
　その対価としてのお金。


　ビジネス＝金儲け、と蔑む人がおるけど、

　商人の存在せん世界で
　一日でも生きてみろと思うわな。

　そして世の人が思うほど、
　簡単にお金は儲からんのだわ。

　特に長期では。

　----------

　結果が欲しければ、

　知識やノウハウより
　パラダイムとマインドが優先よ。

　急がば回れやで。


　一例やけど、

　私は金持ちになって然るべき、
　という強い思いと、

　それに必要なパラダイムがあれば、
　知識やノウハウは後からついてくる。


　だけど、土台もないのに、

　知識ばかり追いかけてる限り、
　成果は永遠に掴まれへん。

　----------

　「儲ける」という言葉を使いたがる人多いけど、
　ワイは「儲かる」の方が好き。

　前者はセールス、狩猟、労働集約的な語感があり、
　苦行の匂い（臭い）を感じるんよ。


　後者はマーケ、仕組み、農耕的な語感と
　正しき経営精神が含意され、

　高い持続可能性を予感させる香りがするから
　惹かれるものがあるんだわ。

　----------

　志を同じくする、同年代の仲間は大事やで。

　日常的にべったり付き合う必要ないけど、
　たまに食事したり、

　時に合宿して目線を高め合うのはお勧め。

　ワイも昔、各分野の日本一経験者と
　日本一合宿をやってたよ。
　
　基準が爆上がりして良かったで。

　----------

　若い頃の友達は、
　歳を重ねるごとに出世するから
　大事にしとくんやで。

　とはいえ結構な割合で
　不義理をしたり、商売を飛ばして
　消えていく人間もいるけどな。

　それでもワイの友達も一定の確率で
　上場したり、国会議員になったり、

　JTCで出世を極めたり、
　ベストセラー作家になったり、
　各所に散らばっとる。

　----------

　20～30代前半くらいで
　早めに頭角現すと、
　いろんなおっさんが近づいてくるんだわ。

　長幼の序、という儒教精神に
　過度に囚われると
　食い物にされることがあるから要注意やで。

　年長者が皆、聖人君子とは限らんからな。

　オラオラしたのから、
　いかがわしいのまで各種取り揃っとる。

　上手に回避するんやで。

　----------

　年長者に可愛がられると
　一気に世界が広がるから、

　愛されキャラであることは
　めちゃ大事やな。

　あなたがもし、
　そうじゃないと自覚していたら、

　周りの愛嬌ある人間を徹底模倣し、
　可愛がられ要素を身にまとった方がいい。

　一回り～三回り上の世代からの教えは
　同年代との付き合いで決して得られんよ。

　----------

　ワイの師匠は、

　二回りより上の人、
　一回りくらい上の人、
　同年代、
　一回りくらい下の人、
　二回りより下の人、

　の5つの属性の人たちと付き合うように、

　さらに歳を重ねるごとに下の年代と
　意識して交わるようにと言われてたな。


　その話を聞いた時は
　2回り下とか子どもやんけと思ったけど、

　今は（文字数

　----------

　集客さえできれば極論、
　残りガタガタでも何とかなるんだわ。

　売上げさえあれば金銭解決できるからな。

　集客できんと後ろピカピカでも
　1銭も金が入らんから潰れるしかない。


　数日前、コンサル会社の大量倒産が
　話題になっとったけど、

　見た目を映えさせるのに夢中で泥臭い
　集客を疎かにしとったんちゃうかな。

　----------

　当たり前やが
　年長者は年長者同士、仲がいいんだわ。

　紹介好きな一人の目上の方に可愛がられたら、
　短期で年長者コミュニティに混ぜてもらえる。


　そこで得られる知恵や経験や交流は
　文字通りプライスレス。

　カネ払ったから得られる、
　いうもんとちゃうからな。

　ワイの感覚やと
　一気に3階級特進した感じやったわ。

　----------

　カネ払ったら誰でも買えるような物とか
　経験なんかに大した価値はないんだわ。

　たとえ高くてもプライスがついてる時点で
　希少性なんか無いからな。


　カネ払っても会えんとか出来んとか見られん、
　そんな経験を若いうちからしておくといい。

　人間関係はまさにその典型。

　手元に何もないなら
　起業すれば手に入る。

　----------

　価値あるコンテンツを、継続的に
　発信し続けた末にご縁を下さった方は、

　商人にとっての顧客台帳のようなものと
　いっても良いかもしれないね。


　商売人であればそこから収益につなげられる
　（金を積める）かもしれないし、

　商売やっていない人にとっては
　金が積まめる代わりに徳（縁）を積まれる。

　----------

　ご縁があり、若い頃から
　年長者コミュニティに入れてもらってた。

　こういう書き方は微妙やけど、

　客観的に見れば政財官の、
　それなりの立場の人と
　日常的に食事をご一緒したり、

　面会をする機会をいただいてた。

　これでどこに行っても臆しなくなったな。

　イキり親父に遭遇しても、
　醒めて眺める目も養えた。

　----------

　20代の頃は、
　小成功でイキってるおっさん見て
　「凄い！」と思ってたけど、

　各分野で圧倒的な結果を残した人たちと
　触れてるうちに、

　まず自分に対して
　ショボいと思うようになったんよ
　（今でも思っとる）。

　次に、俺スゲーとガチで思い込んでる人は
　世間知らずのお山の大将と分かったな。

　----------

　儲けは信用が積み上がった末の
　副産物に過ぎんのよ。

　数年程度、小手先で儲けるのは簡単やけど
　10年20年選手は激減する。

　それくらい厳しい世界やで、商売は。


　短期でうまくいき、
　調子に乗ってる人の9割はマジで消えるから。

　一方、信頼残高の増加に注力してる人が
　メシを食えなくなることもあり得ない。

　----------

　起業とか商売と聞くと

　「金儲けwww」

　という反応をあからさまに、
　あるいは含み笑いと共にしてくる人は
　今でも少なくない。


　商人いないとなんぴとたりとも、
　一日も生きていけんのやけどな。

　でもパラダイムは簡単に変わらんから
　討論してもしょうがない。

　別の世界で生きてる人と思い、
　本業に邁進すればいい。

　----------

　年長者コミュニティに入れてもらってた
　お陰で、ある方と海外にご一緒する時は、

　日本の大使や公使と大使館や公邸で
　ちょくちょく食事する機会にも恵まれた。


　在外公館では天皇皇后御真影？が
　必ず掲げられてることや、
　専属シェフがいることもそこで知った。

　年長者とのご縁は
　新しい経験を運んでもらえるよ。

　----------

　年長者に可愛がられていると、
　いろんな人を紹介してもらえるようになるで。

　そういえば元東京地検特捜部長と、
　2回ほど会食する機会も頂いたな。

　「〇〇君（ワイのこと）も
　　検察関係の人は知っておいた方がいい」

　という親心？から笑。

　そもそも検察に
　狙われるような対象やないんやけど、

　と思いながら同席。

　----------

　世話好きな年長者は脈絡なく

　「あの人に会ってこい」

　と頼んでもいないのに紹介状を書いて、
　面会を取り次いで下さるのよ。

　有り難い話やけど困惑することも多かったな。


　ある時は船井幸雄さんに会ってこい、
　言われてオフィスに伺ったこともある。

　デカい水晶玉が部屋に飾られてたのを
　今、思い出した笑。

　----------

　船井幸雄さん知らん人いるかもやけど、

　船井総研（上場コンサル企業）の生みの親。

　亡くなられたけど流通コンサルと
　スピリチュアルで有名やった。


　そこで話をしたのは流通業界の成長に
　関する極めて真面目な内容やった。

　（ワイがひたすら質問し続けた）

　話の内容と部屋の雰囲気のギャップが
　興味深かったな。

　----------

　年配の方との縁を頂き、

　ご本人の名刺と紹介状を渡され、

　事前に連絡入れとくから会いに行け、
　という、

　昭和エスタブリッシュメント紹介文化の
　存在を知ったわ。


　あと、思い知らされたのが
　東大法学部閥のネットワークの影響力。

　現在とは比較にならん、張り巡らされた
　地下茎みたいな感じを受けたな。

　----------

　滅多やたらと顔の広い
　年長者の方がいらっしゃるのよ。

　ワイの師事したお二人も
　まさにその典型やった。


　ミス日本から、元横綱、代議士、県知事から、

　バブル期にフォーブス資産家ランキング
　10位内に入った方、

　さらにネパールのミュージシャン、
　謎の気功師その他、

　幅広い人たちと会食させてもらったわ。

　----------

　あと起業家は多数。

　一番インパクトあった会食は、

　旅行業界の雄と情報流通産業の雄と
　外資金融業界の雄と紹介者、そしてワイw。

　神楽坂の某フレンチにて。

　全員、日経登場の常連やったな。


　アメックスのポイント！で

　ライカの150万相当のカメラ＆レンズを
　無料ゲットした、

　某氏の話が一番印象に残っとる笑。

　----------

　功成り名遂げた年長者は、
　仕事のスタイルが確立されているんだわ。

　長く近くでご一緒する機会があれば、
　完成スタイルを丸ごと模倣できるのよ。


　自分のタイプと合っていればこれ以上ない、
　最高のOJTとなる。

　ワイはサラリーマンちゃうから
　異動・転勤なく15年ほどご一緒させてもらい、

　完全模倣させてもらったよ。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。

　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。

　
　今日も人生とビジネスを
　楽しもうやで！
