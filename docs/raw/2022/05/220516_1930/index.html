■先日リリースした「我田引水音源」の
　ご感想をたくさん頂戴しております。


　私（鮒谷）があれこれ語るよりも、

　ご感想をお読みいただいたほうが
　内容や雰囲気を掴んでもらえそうなので、

　これからしばらくご感想紹介を通して
　音源内容の説明をしていければと。
　


■本日はＴさんから頂戴した感想のご紹介
　

　（ここから）
　---------------------------------


　昔、自動車工場の試験機器の
　調整バイトをやっていた際に

　バイト先の社長によく言われたのが

　「自分の都合の良いように考えるな」

　という言葉でした。


　これは
　機器の不調やお客さんの都合を

　「大丈夫だろう」

　と自己判断でスルーするな、
　といった意味合いでしたが、

　そういったこともあり、
　我田引水という言葉は、いわば

　「だろう運転」

　のようなものと認識しており、
　いつかしっぺ返しをくらうものである、

　といったように、
　あまり良い印象を抱いていなかったところ、

　「秘中の秘、悪用厳禁！」

　との触れ込みで我田引水についての
　音源がリリースされましたので、

　私の認識している我田引水とは
　どう違うのだろうと思い、

　聴いてみることにしました。


　音源を聴いてみて、
　私の我田引水の認識とは
　全く異なることに驚かされました。


　私はどちらかというと、
　不都合な出来事などがあった際に、

　特段対処することなく、
　まあ大丈夫だろう、

　といったように、

　現実から目を背ける的な認識でしたが、
　音源でお話されている内容は

　きちんと現実に向き合った上で
　どう解釈するか、といったものでした。


　というわけで、

　そもそも私は我田引水についての認識が
　根本から異なっていたことに気づき、

　これまで非常にもったいないことを
　してきたと後悔した次第です。


　なぜかというと、

　こちらの方が日々、元気に楽しく
　生き生きと過ごせると感じたからです。


　これまでは何か問題が起こるたびに
　自責の念に駆られ、

　ひたすらに自らの悪いところに
　フォーカスし、
　それらを書き出して改善策を講じる、

　といったスタンスでしたが、


　これで行動できる場合も
　あるにはありますが、

　アクションを起こすたびに
　問題が起きた際のイヤな気持ちが再生され、
　心が重くなり、

　あたかも油の切れた機械のように
　身体がきしむ感覚に陥ることが
　しばしばでした。


　そのため、改善に取り組もうとするたびに
　イヤな記憶が再生され、

　それがどんどん強化されるという、
　負のフィードバックループが働く状況で、

　改善はおろか、むしろさらに
　苦手が増大してしまいがちでした。


　一方、こちらの音源で提示されていたように、
　そういった場合に

　「ありがたい！」
　「最高だ！」

　などと唱えてみたところ、
　まず、そこで軽くテンションが上がり、

　さらにその理由を脳が自動検索を開始し、
　理由を見つけ出す過程で
　さらにテンションが上がります。


　そして、
　それだけでもかなり気分が良くなり、

　問題への取り組む
　エネルギーが湧いてきますが、

　さらに文章に認めることで
　思考のみならず、

　視覚からもあらためてポジティブな
　解釈が飛び込んでくることになりますから、

　輪をかけてテンションが上がる感覚が
　します。


　どうせ問題を解決するならば、

　自責の念という、内側に針の生えた
　拘束着を身に着けて

　痛みに耐え、
　苦痛にまみれて行動するよりも、

　その先に見える自らの成長や
　明るい未来を思い描いて行動した方が
　はるかに建設的だ、

　と思った次第です。


　まだ取り組みを開始したばかりですが、
　これからどういった展開が生じるか
　非常に楽しみです。

　人生を大きく揺さぶるような音源を
　ご提供頂き、ありがとうございました。


　（Ｒ・Ｔ さま）

■Ｔさん、ご感想ありがとうございます！


　「我田引水」と聞くと、
　
　「フェイクニュース」的なイメージを
　持たれる方があるかもしれませんが、

　「フェイクニュース」は
　端的に言って「嘘」なんですよ。



■対する「我田引水」は、

　「事実」について、フォーカスする
　場所を意図的にコントロールし、

　自他にポジティブな解釈を与える、
　ということですから、

　断じて同じものではありません。



■コミュニケーションにおいて
　（セールスにおいても同様）

　-----------------

　１、着地させたいゴールを決め

　２、スタート地点を決め

　３、ゴールに至る道のりを設計する

　４、理解を助ける表現に落とす

　-----------------

　これが我田引水です。　


　※スタート地点が「事実」なのが
　　我田引水であり、

　　「虚偽」から始めるのが
　　フェイクニュース。

　　違いはこれだけ。
　　でも、ここが決定的に異なる。

　　あとは一緒。



■この順序を意識すれば、誰でも
　できるように思うかもしれませんが、

　上手に操れるようになるまでには

　１、理解する
　２、トレーニングする

　を地道に、根気よく、
　繰り返すしかありません。



■ちょっと思いついてやってみて、
　すぐに忘れて、

　みないなことを、ちょこちょこと
　繰り返してみたところで、

　身につくものではありません。



■それゆえ、定着させるためには

　「伴走者」

　が必要になってくるのです。
　

　つねに我田引水に
　意識を向けずにはいられなくなる、

　そんな「伴走者」を自らに
　用意してやることによって、

　ゆっくりと、しかし確実に、
　この能力はあなたのものとなるでしょう。

　ご用意しているのが個別コンサルです。

■そんな方に向けた次善の策として、

　我田引水音源、

　よいかと思っております。



■ちなみに今日のメルマガも

　-----------------

　１、着地させたいゴールを決め

　２、スタート地点を決め

　３、そこに至る道のりを設計する

　４、理解を助ける表現に落とす

　-----------------

　の公式に忠実に則っています。

　暇があったら分析して下さい。



■無自覚に、自動的にこうした構文が
　使えるようになれば免許皆伝。

　そこまでいけば

　「何が起きても食っていける」
　
　状態になっているはずです。



■勤務先が倒産して無職になった
　私（鮒谷）が

　「食っていける」

　ようになったのも

　-----------------

　１、着地させたいゴールを決め
　　　（起業して飯が食えるように）

　２、スタート地点を決め
　　　（資源ゼロであることを開示）

　３、そこに至る道のりを設計する
　　　（元気と行動量をで耳目を集める）

　４、理解を助ける表現に落とす
　　　（各種教訓やエピソードと
　　　　自身の日々をリンクさせる）

　-----------------

　という「我田引水」を毎日、
　強烈に意識して実践してきたから。



■このフォーマットに忠実に従い、
　時間を味方につけて渦を巻けば、　

　資源が乏しくとも、
　むしろ資源が乏しいからこそ、

　勝ち目が高まるに決まっています。
