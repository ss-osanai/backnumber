＜今日のピークパフォーマンス方程式＞

 【  目指す存在を、より具体的に定義し直してみてはどうか  】

───────────────────────────────────



■「器の大きな人」の定義として、
　以下の７項目を考えてみました。 
　
　
　１、感情が外的環境によって変動しない、
　　　もしくは、動いたとしても振れ幅が小さい
　　　
　　　（自分の内にスタビライザー（安定化装置）を備えているような）
　

　２、中でも特に負の感情を瞬時に打ち消せる力を持つ

　
　３、結果として自分の感情を外にあふれ出させない


　その上で、
　

　４、他者からの厳しい指摘も受け入れる、
　　　取り入れる、感謝する度量を持つ

　
　５、何が起きても、いつもニコニコ

　
　６、こうした安定化した感情の土台の上に、
　　　大局観を打ち樹てる
　

　７、大局観をもった上で、自信を持って一つ一つの手を
　　　確実に打っていける（行動し、形に変える）力を持つ
　　　
　　　いわゆる「着眼大局　着手小局」が常の姿勢
　
　

　「器が大きい」という人物について、
　とりあえず今の段階でざっと思いつく項目を挙げてみました。
　
　
　
■そもそも、
　
　「器の大きい人間になりたい（！）」
　　
　と思っても、
　　
　「自分なりの器の定義」
　　
　を定めなければ目指しようもありません。

　
　
■上に挙げたものは、

　もちろん現時点での単なる思いつきであるとともに
　定量的に計測できるものでもありませんが、
　
　随時、定義をアップデートしつつ、
　できるだけこの方向に向けて、

　きちんと自分を躾けていきたいと思っています。



■ちなみに、よく、


　「デキる人になりたい」

　「幸せになりたい」

　「金持ちになりたい」

　「豊かな時間を楽しみたい」

　「人間力を身につけたい」

　「感受性が欲しい」

　「稼ぐ力を手に入れたい」

　「格好よくなりたい」
　　
　「男の魅力を醸し出したい」

　「女子力をアップしたい」

　
　といった、

　「分かったような、分からないような目標？」

　を言われる方があります。


　
■もちろん、

　目標はないよりは、あったほうがいいのは
　いうまでもありませんが、

　その目標を、もっと具体的な定義にまで
　落とさなければ、目指しようもありません。


　
■今日は、このところずっと考えている

　「器の大きな人間になりたい！」

　という目標について書こうと思ったところ、
　そもそも

　「『器』って何よ！？」

　ということを自分なりに、暫定でも、
　定義付けておく必要があると思い、

　上の７項目を挙げたわけです。



■抽象的で分かりにくい目標も、

　極力、具体化な表現で定義し直すことによって、
　実現可能性はぐんと高まる、

　そんな風に考えています。


　「器」について書こうと思っていたのに、
　全然違う方向に話が逸れてしまいましたね。

　でも、大事なことだと思うので
　書いておきます。
