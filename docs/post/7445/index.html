7444号 「構造上、儲からざるを得ない状況」を作るのが商売の極意
7446号 NHK『ねほりんぱほりん』コンプリート＆いよいよ本丸の『100分de名著シリーズ』へ

7445号 永久機関的に働く、気づきや発見や成長を生み出すために

■先日来、ご案内してきた
　「構造音源」についての長文感想を、

　Ｋさんより頂戴いたしたので
　以下にご紹介。


　（ここから）
　-------------


　鮒谷さん、スタッフのみなさん、
　お世話になっております。

　構造音源の感想を
　送らせていただきます。


　「構造」「本質」という言葉の持つ
　本物感？が大好きで、

　「構造を利用する」
　「本質に迫る」

　という動詞になると
　定性目標にも引っかかってくるので、

　先月末に発売された瞬間に
　購入は決めていましたが、

　聴く時間が取れずで
　ズルズルときてしまいました。

　しかし今さらですが、
　買えば聴く時間は作れるものですね。


　ところが・・・です。

　楽しみにしながら聴きはじめると、

　冒頭から感想や文章についての辛辣な
　物言い(笑)に耳を塞ぎたくなる思いでした。


　私にとってはもはや「聴く」と
　「感想を書く」はセットになっており、
　切り離すことはできないものですが、

　自己肯定感がさほど高くはない私にとっては

　（お陰さまで「自尊心音源」以降、
　　少しずつ上がっている実感はあります笑）

　書く前から筆が止まるという感覚でした。


　でも、文章が上手になりたいという
　願望はあるので、

　まさかのプチ文章コンサルを受けたと考え、
　最低限、冒頭で語られていたことは意識して、

　震えながら感想を提出いたします。笑


　まず

　「構造」「仕組み」
　「仕掛け」「からくり」・・・

　のような言葉を
　意識して用いることによって、

　初めて裏側にあるものが見えてくる、
　と仰ってました。

　他の言葉にも応用できるのでは？と思うと
　これだけで元が取れたと感じました。


　思索を深めたい、あるいは
　深堀りしたい言葉を意識して用いると、

　見えてくるものが多いのではないかと思い、
　「構造」を活かしていきたいと思いました。


　そして「抽象度」という単語が連発されていた
　ことも思いがけない贈り物でした。

　というのも、ここ2年くらい

　「抽象度」

　という、
　わかったようなわからないような

　（ということは
　　わかっていないのですが・・・）

　やっかいな言葉が耳に入ってくるようになり、
　悶々としておりました。


　もちろん鮒谷さんの音源で
　頻出ワードなのは承知しておりますが、

　今回、また違う視点からの解説で
　「抽象度」という言葉そのものと、

　それを使いこなしていく
　（上げたり下げたり？）

　ヒントをいくつかいただくことができ、
　テンション爆上がりです。

　私も目に見えない世界で遊びたいです！


　あとは「知的負荷」という言葉も
　数回登場しました。

　とかく負荷のかからない／少ない、
　楽な方へと流れがちな私ですが、

　やはり自分に知的な負荷をかけてきた人の
　文章や語りは魅力的なので、

　少しでも近づけるよう、
　紹介されていた知的負荷をかける方法を
　日々実践してまいります。


　そして

　「負けにつながる構造を排除する」

　というのも言われてみればその通りですが、
　ハッとさせられました。


　健康になりたければ、
　栄養価の高い食べ物云々よりも、

　健康を害するものを体に入れるのを
　やめる方が先・・・と同じで、

　大体においてやってはいけないことって
　わかってるんですよね。

　単にそれをやめればいい、
　でもやってしまう・・・

　という「構造？」は不変ですね。


　あと

　「長い時間軸のゲームの方が勝ちやすい」

　という解説がありました。

　鮒谷さんの音源では頻出で
　私にもその体感はありますが、

　鮒谷さんのわかりやすい言語化のおかげで
　兵糧には最新の注意を払いつつ、

　より一層長期視点の意思決定を
　続けようと、改めて思いました。


　今回も質疑パートの方が長いくらいで、
　その中でも多くのことを気づかされました。

　年プロ参加者の皆さん、
　ありがとうございました。


　最後に、今回も鮒谷さんの膨大な経験を
　明瞭な言語に落とした、

　極めて実践的な音源を提供していただいて
　ありがとうございました。

　今後ともよろしくお願いいたします！

　　　　　　　（Ｋ・Ｋさま）


　-------------
　（ここまで）


　Ｋさん、いつも音源のお求め、ならびに
　感想のご送付ありがとうございます！



■音源の冒頭で話をしている

　「文章の書き方講座
　（最低限、これくらいは意識しましょうよ）」

　のコーナーは、
　そんな話をするつもりは全くなかったのに、

　話をしているうちに
　ドライブがかかってしまいました（笑）



■もちろんご承知の通り、
　特定個人を指しての

　「指弾（！？）」

　ではなく、どこまでも
　一般論として語ったものですが、


　聞かれる人にとっては、
　それぞれ

　「俺の（私の）ことを言ってるんじゃね？」

　と思われた方もあったかもしれませんが、

　皆さま等しく、
　Ｋさんと同じように心配されているので、

　ご安心（？）下さい（笑）



■正直、あの内容単体だけでも、
　ちょっとした価値があると思います。


　あそこで挙げたことを
　ほんの少し意識するだけでも、

　どんな文章を書くにせよ、随分と
　読みやすいものになるはずです。



■それはさておき、

　音源をリリースするたび、
　いろんな方から、

　さまざまな角度からのフィードバックを
　頂けることはありがたいもの。



■年間プログラムを含め、これまで
　数多くのセミナーや勉強会を開催し、

　さらには音源をリリースしてきましたが、


　都度、率直な感想を頂戴し、

　【発信 → 受信 → 更新 → 発信 → 受信
　　→（以下、反復）】

　のエンドレスサイクルが
　回り続けることによって、

　【永久機関的な、気づきや発見や成長】

　のプロセスを楽しめるのだなと
　改めて感じた次第なのです。



■そう考えると、
　抽象度を上げると（！）

　あらゆる分野において

　【発信 → 受信 → 更新 → 発信 → 受信
　　→（以下、反復）】

　のサイクルもまた、
　一つの構造であると捉えることもでき、


　なにを行うにしても、
　このサイクルを意識して組み込めば、

　気づきも発見も成長も加速するに
　違いありません。



■私（鮒谷）の知人でも、
　このサイクルを回すことで

　衆人環視の環境を活用しつつ、
　難関資格に合格した人もあれば、

　事業を大きく伸長させた人も
　たくさんあります。



■かなり応用範囲の広い構造だと
　思いますし、

　上手に利用できれば、自己実現のための
　大きな武器になりますよ。


　※今日は頂戴したご感想が長いので
　　その分、本文は短めに。

　　（手抜きではありません笑）



■、、、ということで（？）、

　以上に記したような学びも含め、
　生きていく、という営み、それ自体が

　【（広義の）投資】

　という側面があります。



■であれば、

　良好な投資成績を得たければ
　（＝幸せになりたいのなら）

　【世の理（ことわり）を透視する力】

　はどうしたって必要になりますね。

　「投資」の前に
　「透視」しなければなりません。



■そんな話をあれこれしている音源に、

　【構造、骨格、流れ、本質を透視して
　　投資する！放談会音源】

　というタイトルを付けて、この度、
　お求め頂けるようにしております。


　※ちなみに

　　【あと数日で、音源リリース記念の
　　　特別価格での販売を終了いたします】

　　ので、お申込みは今すぐどうぞ。



■当音源は、
　ややマニア向け？であるのと、

　もともと外販予定のなかった音源

　（それゆえ話がタイトルのテーマのみならず
　　いろんな方向に飛んでいる）

　を販売するということもあって、

　いつもの販売価格よりも
　安価に設定しております。



■繰り返し聴いているうちに

　「目に見えないもののほうが
　　はるかに大切」

　であることを気づかれるようになる、
　さらには活用しようと思われる、

　ようになられれば、
　これにまさる喜びはありません。


　※タイトルに
　　「透視して投資する！」とありますが、

　　金融投資のみならず、
　　人生全般における

　　「広義の投資」

　　についての話となりますので
　　お間違えのなきように。



■お求めは以下より今すぐどうぞ！


　--------------------


　【構造、骨格、流れ、本質を透視して
　　投資する！放談会音源】


　▼収録時間：約191分（約3時間11分）


　--------------------


　あと数日で、音源リリース記念の
　特別価格での販売を終了いたしますので、

　お申込みは今すぐどうぞ。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■どんな分野であるかにかかわらず、

　【発信 → 受信 → 更新 → 発信 → 受信  
　　→（以下、反復）】

　　のサイクルを組み込めば、
　　気づきや発見や成長が加速する。

　■これも一つの「構造把握」を活用した
　　応用の形と言えるだろう。
