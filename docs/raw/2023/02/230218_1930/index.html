■究極の前倒しは、即実行

■このメルマガでは以前から

　「前倒しの効用」

　についてお伝えしてきました。


　やるべきことを前倒しするほどに、

　人生は楽に、
　楽しくなっていきますが、

　目先の欲求に流されて
　先送りすることが習慣になると、

　人生はどんどん辛く、苦しく、
　大変なものになっていくものです。



■だから、

　【つねに未来のことを考え、
　　優先すべきことを優先する】
　
　ことを習慣化していくと
　よいでしょう。



■とはいえ、

　【つねに未来のことを考え、
　　優先すべきことを優先する】

　ことの大切さを分かっていてすら
　実行するのは難しいわけですから、


　この種の言葉を持っていなかったり、
　頻繁に脳内で回す癖をつけていなければ、

　安易な方向に流されて
　
　【楽に溺れて、のちに後悔】

　をひたすら繰り返すことになります。



■そうならぬように、
　先の音源のタイトルでいうならば

　【前倒しの習慣が
　　身体に編み込まれるところまで
　　意識して自分を躾けていく】

　必要があるのです。



■その方法についてお伝えしたのが

　--------------

　【「身体に編み込む勉強法」音源】
　
　～一生懸命、勉強しているのに、
　　人生にも仕事にも

　　「お勉強」を身体に編み込み、
　 　生きた学びに変えるための
 　　３時間半の特効薬 

　--------------

　なります。

■人生において、

　「上達したい」

　と思うのであれば、

　長期的に見て人生を良い方向に
　変えてくれる習慣を一つ一つ、

　継続的なトレーニングを重ねて
　身につける必要があります。



■それは

　「空手」「サッカー」「水泳」
　「書道」「絵画」「ピアノ」
　
　などなど、

　こうした例は挙げようと
　無数に挙げられるので、

　てきとうに端折って、
　これくらいにしておきますが、

　全部いっしょですよね。



■これらについて

　「上達したい」
　
　と思っている人が
　数時間とか数十時間やった「だけ」で、

　芸術的なシュートを
　放つことができるようになったり、

　自由自在にピアノを弾けるように
　なったりするはずがありません。



■それと同じで、

　「前倒し習慣」

　一つをとっても、

　長い時間をかけて
　自分と対話を繰り返しながら、

　ゆっくりと、しかし確実に
　身につけていくものです。



■こうして、

　一つ一つの良い習慣を
　身につけていくことによって、

　人生は好転していきます。

　面倒と思われるかもしれませんが
　ミラクルはありません。



■けれども一度、身につけたら
　それは一生モノの財産です。

　資産運用に例えるなら
　こうした習慣は

　「元本」

　みたいなものであり、

　早く手に入れるほど、その後の
　人生が楽になっていくでしょう。



■では、たとえば具体的には

　「前倒し習慣」

　を、どのように躾けていくか。


　いろんな言葉を利用可能ですが、
　一つの呪文としては

　【究極の前倒しは、即実行】

　という言葉とともに、
　身体をその場で動かす訓練を重ねる、

　なんてことを行ってみられては
　いかがでしょうか。



■「即実行」以外は、すべて先送りです。

　あとでやる、そのうちやる、
　今晩やる、明日やる、来週やる、

　これらすべては先送り。

　99パーセント、
　結局やりませんからね。　



■だからこその

　【究極の前倒しは、即実行】

　【いつでもどこでも、即実行】

　ですよ。



■数回、十数回程度、であれば
　定着まで至ることはありませんが、

　数十回、数百回、となるにつれて
　徐々に出来るようになり、

　数千回単位になると、だんだん身体に
　編み込まれていく実感を得られるはず。



■これが

　「身体に編み込む」

　ということです。


　先日、大勢の方が

　【「身体に編み込む勉強法」音源】

　をお求めくださったので、

　今日はそんな皆さまにフォローアップを
　兼ねて、したためた次第です。
　
　活用（実践）いただければ幸いです。
