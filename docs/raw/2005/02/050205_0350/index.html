　【　テキストに感情を埋め込むことができれば
　　　ネット上で気持ちのよいコミュニケーションをとることができる。　】

───────────────────────────────────


■私はこのようなメールマガジンを配信していることもあって、たまたま
　ではありますが、ご意見やご感想、あるいはビジネスのご提案などという
　かたちで他の人よりも圧倒的に多くのメールをいただく立場にあります。


■実はそれだけ大量のメールに触れていると、比較相対するつもりでは
　ありませんが、結果としてデジタルコミュニケーションにおける巧拙が
　如実に存在していることが分かってしまったのです。


■昨日は、現実の世界とネットの世界ではコミュニケーション方法に
　違いがあるという話をしました。

　現実世界では言葉のほかにも、声の調子や顔色、身振り手振り
　といった数多くの要素が相手に伝わる。

　それに対してネット上のコミュニケーションでは言葉がすべて、
　という話でした。


■つまりネット上ではこれまでに私たちが培ってきた対話術とは
　全く種類を異にするコミュニケーション術が要求されているのです。


■そのことをよく心得て、ネット上で非常に美しい立ち居振る舞いを
　している人もお見受けします。

　それはあたかも上質なスーツをいかにもさりげなく着こなし、
　それでいていやみにならない紳士ぶりを感じさせるような。


■ところが、中にはほとんど全裸に近い格好で、いちじくの葉っぱ！？のみを
　身にまとい、そのおかしさに気付かずに歩き回っている人もいらっしゃる
　のです。（いや、ほんとに）


■現実世界では比較対象となる人が周りにいますから、よほど特異な人を
　除いては、ある程度のレンジ（幅）でコミニュケーションの上手下手は
　収まります。

　ところがインターネットの世界では、時に暴走して、とんでもない格好で
　どこまでも歩いていってしまう人がおられることは否めない事実です。


■今までは現実世界の中でのみ生きてきたわけですから、対面の
　コミュニケーションスキルを身につける必要がありました。


■しかしこれからは、それと同じように、いや、それ以上に、

　簡単なようで奥の深い、このネットにおけるコミュニケーション術を
　身につけていかなければなりません。


■非常に抽象的な話に終始しましたが、ひとつだけアドバイスを。


■毎日怒濤のように受信するメールの中で、ひときわ心をとらえる
　メールとはどんなものか。そういうメールは、どこが違うのかと
　言いますと、

　【テキストに感情を埋め込む力が違う】

　といえましょう。


■読む人の心を捕らえるか捕らえないか、有機的な文章になるか無機的な
　感じがするか。

　それは「テキストに感情を埋め込む」ことを心がけるかにかかっています。


■その具体的な方法については、各人の性格なり気質がありますから
　一概にこれが良いという雛形を提示することは困難です。
　ですから一人一人考えなければなりません。

　しかしこのこと一つを意識することにより、まったく違った印象を
　相手に与えることができるようになるでしょう。


■【テキストに感情を埋め込む】ことに気を付けてデジタルコミュニケーション
　の世界に入ることにより、より豊かで心の通った意思疎通が可能になるの
　ではないかと思います。
