■起業・副業したいのにメドすら立たぬ人/
　ショボビジも悪くない/丸裸で特攻起業する人/
　起業の順序は「小資本・高粗利 ⇒ 大資本・低粗利」

■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------

　ビジネスの最初期は
　人間関係だけでもなんとかなるんだわ。

　それなりに親しく、
　かつ真っ当な人間と認めている人間から
　「これはいいよ！」と勧められ、

　多少なりともそこにニーズがあれば
　結構な確率で買うだろ。


　あなたもそんな人間になれば
　起業とまでは言わんが、

　少なくとも副業までは簡単よ。

　----------

　起業/副業したいのに、
　メドすら立たんとすれば

　１．人望
　２．能力（専門性）

　のいずれか、
　ないし両方の欠如を疑うべき。


　この準備が整ってない人が
　いくら「いつか副業したい、起業したい」
　と言ったところで永久に無理。

　大企業勤務であっても
　人間関係の構築が話にならず、
　始まってもない人が少なくない。

　----------

　たとえネット上のビジネスでも、

　人望（人間性）は透けて見えるから
　ごまかせんよ。


　商売の種類にもよるので、

　そうした要素が完全無関係なビジネスも
　ゼロじゃないけど、

　基本的には、人間関係もまともに作れん人が
　商売なんかできるはずもない。

　売る時だけじゃなく、貴重な情報の
　仕入れも人間関係からなのよ。

　----------

　出資を受けて事業を始める人いるけど、
　根性据わってて立派やと思うわ。

　ワイみたいな小心者は、

　カネを委ねられた時点で
　プレッシャーかかるから
　絶対やりたくない。


　資金溶かしてスンマセンの可能性が
　あるだけでモームリや。

　精神的/物理的/経済的に、
　ひたすら自由で居続けたいから
　スモビジ一択になった。

　----------

　ワイ、勤務先の倒産を
　新聞で知ったんよ。

　スマホのない頃の話。

　夕方、営業帰りに
　地下鉄売店に並んでた新聞各紙の
　一面トップ記事をたまたま目にしたんよ。

　「うちの会社、潰れとるやんけ！」

　が第一声や笑。


　年商が数兆ある会社でも、
　潰れるときは一瞬。

　その後の残務処理含め、
　貴重な経験させてもらったわ。

　----------

　スモビジ（スモールビジネス）ならぬ、
　ショボビジ（ショボいビジネス）も悪くない。

　バカにされながら実利を取ればいい。

　一人+αで全額自己資本。

　軌道に乗せれば精神的負担なしに
　数千万～数億の利益まで見込めるから
　夢あるで。

　ワイの友人にも敵のおらん場所で、
　のんびりショボビジ実践者が大勢おる。

　----------

　スモールビジネスを
　バカにする人あるけど、
　案外悪くないよ。

　『となりの億万長者』という
　アメリカ富裕層調査の有名な本
　にも出てるが、

　金持ちの多くは
　その辺に転がってる
　スモビジオーナーなのよ。


　目立ってイキって
　派手に商売しとる人より、

　重心低く、地味に密かに
　長らく儲け続ける人に
　惹かれたんだよね。

　----------

　スモールビジネスは売上より、
　粗利率と利益の絶対額を追求すべき。

　特に最初期は

　・粗利8～9割のビジネス
　・受注発注
　　（受注/入金を確実にしてから発注する）

　を徹底すれば、
　リスクほぼゼロでビジネスを行える。

　起業志望者はサラリーマンのうちに
　副業を通して、こうした商売に
　慣れておくといい。

　----------

　初めての起業では

　「小資本 / 無借金 / 高粗利 / 利益の絶対額」

　と朝から晩まで唱え続けるべき。

　会社の看板に頼らず
　商売したことがないのに、

　いきなり大金を借りたり、
　虎の子の何百万や退職金の何千万を突っ込み、

　丸裸で特攻起業する人が後を絶たんのが
　不思議で仕方ない。

　そんなことせんでもカネは稼げるよ。

　----------

　起業の順序は

　「小資本 / 高粗利 ⇒ 大資本 / 低粗利」

　やから間違ったらあかんで。

　特にゼロイチ未経験者が後者から
　始めるのは死ににいくようなもん。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
