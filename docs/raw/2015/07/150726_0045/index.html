【 醒めた感覚を持って行うコミュニケーション 】

───────────────────────────────────



■昨日、コミュニケーションについての話を
　書きました。

　今日も続けます。


　コミュニケーションで最悪な、
　でも、つい、やってしまいがちなミスは、


　「伝えるべき内容は伝えているのだから、
　　それでよし

　　（伝え方はどうでもいい、
　　　あとはこちらの事情を察してくれ）」


　という態度。


　そこには

　「相手がどんな感情を受けるのか」

　についての想像が、ありません。



■１対１、にせよ、１対多にせよ、

　そんな気持ちで
　コミュニケーションを行っていたら、

　そりゃ、人さまから好かれることも愛されることも
　なくなっていきますよね、

　という話。



■伝え方を工夫しなくても、
　伝えるべきことは伝えているのだから、


　「あとは（自分に都合の良いように）
　　察してもらいたい」


　という気持ちはどんな人にもあると思います。

　もちろん、私にも。

　
　けれども、そんな考えは、

　「中学生、あるいはせいぜい高校生くらいまでで
　　卒業しましょうよ」

　という話。



■みんな、自分のことで精一杯。


　中に、そんな状態にあっても

　「他者に対して、
　　つねに気配り・心配りができる」

　という、出来た人があるのは否定しませんが、

　他者にそんな水準を求めるのは
　過度の要求、

　というべきでしょう。


　
■そもそも、あなたや私が
　そんな高度なコミュニケーションをとれているのか、

　という話。


　あなたのことは分かりませんが、
　少なくとも私（鮒谷）は、そんな出来た人間ではありません。

　だから自分を棚に上げて、
　他人に過剰な期待を寄せることはありません。



■以上に記したように、

　他者とお付き合いをする際には、

　「伝える内容はもちろん、
　　伝え方についても、きちんと考える」

　ことを意識するのは当然のことですが、


　同時に、

　一瞬、気分を害してしまいそうになる、
　他の人からのコミュニケーションについても


　「（単に伝え方の巧拙の問題であり、
　　　他意はないことも、往々にしてあるので）

　　他者からの『伝えられ方』については、
　　過度に気にしない」


　　→そのたびに、いちいち感情を動かしていたら、
　　　仕事にならない 笑

　

　という、

　相手と自分の関係性を上空から捉えるような、
　実のところ、けっこう醒めた（？）感覚を持って、

　日常のコミュニケーションを執り行っています。

■「平成進化論」でお届けしている、様々なコンテンツを

■経営戦略、ビジネスモデル設計、マーケティング
  情報発信、営業といったビジネス直結の話から、
