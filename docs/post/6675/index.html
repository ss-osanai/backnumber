6675号 「流れ弾」に積極的に当たりに行く

■過去のメルマガで

　【積極的に流れ弾に
　　当たりに行きましょう】

　と書いていたことがあります。


　最近、あまり書かなかったのですが
　とても大事なことだと考えています。



■人間には、

　「自分の目に入り、耳で聞ける、
　　（＝手に届く情報）

　　の中からしか、
　　将来の選択肢を選べない」

　という習性があります。


　知らない世界については

　想像することも発想することも、
　そこに到達しようと願うことも、

　できないのです。



■人の持つ、もう一つの習性として、

　「今、生きている世界に
　　耐え難い不満を持たない限り、

　　現状維持バイアスが発動し、
　　思い切った行動を取れない」

　ということもありますね。



■これら二つの習性（要素）を合わせると


　「強烈な精神的飢餓を感じない限り、
　　能動的に現状を変えようとは思わず、

　　それゆえ、知らない世界に冒険し、
　　衝撃を受ける（流れ弾に当たる）、

　　という行動を取ろうとも思わない」


　ということになりますね。



■その必然的帰結として

　「漠然と、変わりたい、変わらなきゃ、
　　とは思っているけれども、

　　日常における行動は一向に変わらず、
　　自らの意思で変えようとも思わない」

　という状態が長々と続くこととなる
　のです。



■そして、

　「永続的に続いてきた、
　　こうした状態こそがストレスの源」

　となり、ボディーブローのように、
　自らの自尊心を毀損し続ける、、、



■この状況を突破するための
　第一歩は

　【無理やりにでも
　　流れ弾（外部刺激）に当たりに行き、

　　大いなる刺激や衝撃を受けて、
　　強烈な飢餓感を感じる】

　ところから。



■こうしたことを

　「ルール」

　として決めておかないと
　上述の通り、


　「強烈な精神的飢餓を感じない限り、
　　能動的に現状を変えようとは思わず、

　　それゆえ、知らない世界に冒険し、
　　衝撃を受ける（流れ弾に当たる）、

　　という行動を取ろうとも思わない」


　という態度が延々と続くこととなる
　わけです。



■それでいいと思うのなら、
　それでもいいのですが、

　少なくとも私（鮒谷）は
　こうした

　「自らの精神に対する、
　　自覚症状なきボディーブロー」

　を受け続けることをよしとは
　していないので、


　【無理やりにでも
　　流れ弾に当たりに行き、

　　大いなる刺激や衝撃を受けて、
　　強烈な飢餓感を感じる】


　ことをスケジュールに組み込み、

　つねに新しい世界（衝撃）を探し求めて
　ウロウロしています。



■こうした取り組みが

　数百回に一回くらいの頻度で、
　「ホームラン級」の意識革命、

　をもたらしてくれます。

　※大学院に進学したことで
　　「エフェクチュエーション」に出会いました。
　　
　　私（鮒谷）にとっては
　　特大ホームラン級。



■あるいは

　数十回～百回に一回くらいの頻度で
　「長打級」の日常を打破する意識改革、

　を促してくれることがあります。



■また、

　十回に一回程度の頻度で
　「単打級」の日常の改善を促す営み、

　へと導いてくれたり、
　ということにつながっていくのですから

　【流れ弾には当たりに行け】

　と声を大にしてお伝えしたいのです。



■流れ弾に撃ち抜かれるのは、

　「人との出会い」

　によってが100％です。


　もちろん理想は
　リアルに対面することですが、

　常にそれを行うことができなければ、
　音源や書籍、

　といったことでも代用できるもの。

　（私もさんざん活用してきましたし、
　　今も活用しています）



■もし、すぐに

　「人との出会いによる
　　流れ弾の被弾」

　を起こすことが難しい方は、

　「音源による流れ弾の被弾」

　を目的としていただくのもよいのでは
　ないかと思います。



■各音源についての

　「被弾体験談（ご感想ｗ）」

　をご覧頂ければ、

　おおよその内容についても
　イメージ頂けるのではないかと思います。



■「漠然と、変わりたい、変わらなきゃ、
　　とは思っているけれども、

　　日常における行動は一向に変わらず、
　　変えようとは思わない」

　という自覚症状のある方であれば、


　その症状が亢進し、手遅れになる前に、

　【無理やりにでも
　　流れ弾に当たりに行き、

　　大いなる刺激や衝撃を受けて、
　　強烈な飢餓感を感じる】　

　というルール（意思決定基準）を
　インストールし、


　その基準で行動を開始されても
　よいのではないでしょうか。


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】　　　
　　　　　　　　　　　　　　　　　　　　　
　■「流れ弾」に積極的に当たりに行くことで
　　人生が思わぬ方向に切り開かれる。　　　
　　　　　　　　　　　　　　　　　　　　　
　■予定調和の人生、生活を繰り返しながら　
　　予定外の人生（新しい世界）に出る、　　
　　　　　　　　　　　　　　　　　　　　　
　　などということが起こるはずがない。　　
　　　　　　　　　　　　　　　　　　　　　
　■自分の見えている、聞いている世界とは　
　　異なる存在を知ることによって、　　　　
　　　　　　　　　　　　　　　　　　　　　
　　初めて能動的な思索と行動が誘発される　
　　こととなるだろう。　　　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■このままでいいのかと思いつつも、　　　
　　強い飢餓感も感じないまま、ダラダラと、
　　　　　　　　　　　　　　　　　　　　　
　　の日々を重ねていて、現状の枠を超える為
　　の行動など、取りようがないではないか。
