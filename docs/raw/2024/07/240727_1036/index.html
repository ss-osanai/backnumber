■あなたは人生や仕事の
　「見取り図」を持っていますか？

───────────────────


■先日の年間プログラムのご感想を
　頂戴しました。


　以前からの読者さんはご存知（！？）、
　船津さんからお送りいただいたもの。

　※船津さんはその後、借金をすべて返済し、
　　死地を脱して、独立され、

　　現在、七転八倒されながらも（笑）
　　新しい世界で活躍しておられます。



■以下にご紹介。


　（ここから）
　---------


　先日もセミナーのお時間、
　有り難うございました。

　リアルタイムでは最後の方の参加で、
　中途半端な感想をお伝えしたのみでしたので、

　音源版をお聞きして改めて感じたことを
　感想にまとめました。


　「適度な緊張が、成果に繋がる」

　というテーマで、

　私も最近の取り組みを振り返りながら、
　セミナー音源を拝聴いたしました。


　4月からフリーランスとして独立し、
　研修講師を勤めた3ヶ月間。


　研修講師の案件では、
　60名近くの受講生を担当し、

　高い要求水準が求められる毎日で、
　ひたすら予習と準備に追われ、
　複業も減らしました。

　また、依頼元の企業とのすったもんだで、
　精神を消耗することもありました。

　
　しかし、その中で

　「どうしたら期待を
　　超えることが出来るか？」

　という問いを回し続けていたお陰か、

　これまで無かった能力が
　育った実感があります。


　毎日緊張の連続で、
　心が休まることがありませんでした。

　正直、自分が何を価値提供出来たのか？

　を些細に把握して
　言語化も出来ていません。

　勿体無い話ですが。。


　そして、最近のエンジニア案件では、

　稼働は定時内に収まり、
　複業も減らしたままで、

　夜に空白の時間が生じました。


　今回のセミナーで

　「小人閑居として不善を為す」

　と、鮒谷さんに戒めて頂きました通り、

　空いた夜の時間で
　不安に襲われる時間が増えました。


　久々のエンジニア案件で、簡単に出来ると
　思っていたことが出来ていなかったり、

　最初から即戦力として期待されている
　プレッシャーに潰されてそうになり、

　業務が終わった後に、
　自分を責める時間が増えました。


　その結果、業務中は過度なプレッシャーで
　顧客とのコミュニケーションで、

　息が詰まりそうになり、
　思考力も低下するような毎日でした。


　ここで、先日のセミナーの話を
　思い出して持ち直すことが出来ました。


　どんな仕事にも「見取り図」がある。

　その「見取り図」を知って
　メタ認知出来れば、
　
　自分が何をやっているのかが分かる。


　この話を音源で拝聴し、
　一気に脳の回路が繋がりました。

　出来ていないこと、
　出来ないことがあることを受け入れ、

　出来る部分（強み）にフォーカスしよう。


　その姿勢になり、少しリラックスが出来、
　程よい緊張、フロー状態となり、

　少しずつ成果を出して
　達成感を感じることが出来ました。


　メタ認知して緊張をコントロールし、
　その中の不安を突破すると
　達成感を感じることが出来る。


　「域値超え」の話と通じるか分かりませんが、

　そういったことも
　「見取り図」として知っていると、

　不安と緊張に晒されている時も、
　逆にチャンスに転換することが出来ることを
　再認識いたしました。


　改めて、有り難うございます。

　年プロの場はメタ認知としての
　「見取り図」をひたすらインストールする
　場であることを再確認できました。


　最近、自分が緩んでいるのか
　緊張しているのか分からない程、

　くだらない不安で潰されていたので、
　そこを糸口に復活いたします！

　今後とも宜しくお願い申し上げます。


　　　　　　　　　（船津 忠文さま）


　---------
　（ここまで）



■前回の年間プログラムでは

　「緊張と成果の関係」

　について話をいたしました。


　こちらの詳細については言及しませんが、
　船津さんが記してくださった

　「見取り図」

　について少しだけ。



■人生や仕事において

　「全体構造」

　と

　「今、全体の中のどこにいて
　　何のために何を行っているのか」

　が分かっていると安心して
　取り組めるんですよね。

　これは受験や資格取得のための勉強に
　ついても同じことが言えるでしょう。



■今、このあたりにいて、
　
　少し大変だけれども
　ここを抜けると次はこうなる、

　みたいなイメージを描けていると
　腰を据えて挑戦できますが、

　
　今、どの地点にいるのか理解できず、

　やっていることの意味がわからず、
　合っているのかどうかも分からない、

　これが最悪なんですよね。



■今、行っていることに対する
　自信を持てないから、

　最後まで完遂することが
　著しく難しくなったり、

　あるいは逆に、見切ることも出来ず、
　サンクコストの罠にハマるのです。



■試験や資格取得のための勉強であれば、

　見取り図を持たずに戦い、
　最終的に成果を出せなくても、

　長くてもせいぜい数年程度を
　捨てる程度のダメージで済みます。

　（それでも
　　もったいないですけどね）



■しかし

　【人生の見取り図】
　
　を持たずに、つまり、


　【現在の状態は、理想の状態と比較し、
　　どのあたりにいるのか】

　【理想の状態に近づくためには
　　何をどの程度、行えばいいのか】

　【その過程において
　　どんなことが起こり得るのか】


　等を把握することもないまま、
　とにかく動こうとしても、それは

　【暗闇の中で全力疾走しようとする】
　
　ようなものですから、
　不安でたまらなくなりますよ。


　結果、全力を出せなくなるし、
　途中で折れる可能性も高まります。
　


■年間プログラムでお渡ししているのは、

　まさに船津さんが
　おっしゃって下さったように

　【人生や仕事（ときにプライベート）に
　　おける見取り図】

　といって良いかと思います。



■そして最近、気づいたのですが、

　【見取り図】

　を都度、作っては、
　その精度を高めていくことが。

　私（鮒谷）自身の、
　何よりの趣味だったんですよね、



■それが先を手に入れられたから、
　
　自信を持って、
　自らが信じる道を歩み続けられたし、

　今なお、歩み続けられているのだと
　思います。



■自分が今、どこで何をやっていて、
　その過程において何が起こりそうか、

　を把握するための
　見取り図を持っていれば、

　感情を揺らさず、確信を持ち、淡々と
　未来に対して積み上げていけますよ。

　

■今日も人生とビジネスを
　楽しみましょう！
