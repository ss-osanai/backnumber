＜今日のピークパフォーマンス方程式＞

 【　鮒谷が「師匠選び」の際に心がけていること　】

───────────────────────────────────



■世の中に、あまた流通している
　
　「シンプルで耳障りの良い言説」

　は疑ってかかった方がいい、

　少なくとも鵜呑みにはしない方がいい、

　と思っています。
　
　
　たとえば、一番わかりやすいものとして、
　ビジネスや自己啓発の世界における
　
　
　「こうすればうまくいく」
　
　「こうするだけで成功する」
　
　
　といった類の、もの言い。
　
　
　
■各人の履歴も、置かれている状況も
　バラバラなのに、
　
　100％、万人に通用するシンプルな方程式に
　還元することなど到底できるものではありません。


　なのに、

　「100％うまくいく」

　「絶対に間違いない」

　と断言、断定している人があったとするならば、
　眉に唾つけて、話半分、

　どころか、

　話半分、八掛け、二割引、

　くらいしてから、聞いたほうがいいでしょう。



■逆にいえば、

　
　「この本を読んだら、確実に成功できますか？」

　「このセミナーに出たら、絶対うまくいきますか？」


　といった類の質問をしたくなる人があったとすれば、

　（心情は痛いほど分かりますが）

　その人は、世の中をあまりに単純化し過ぎていて


　「単純かつ安易な因果律を前提として、
　　
　　発展の様態を一直線上に伸びていく、
　　線形的展開として捉える信念」


　に毒されているようにも思われます。



■このような質問をされる方における一番の問題は、


　「単純かつ安易な因果律を前提として、

　　『一直線上に伸びていく、未来の線形的展開を
　　　固く信ずるパラダイムそのもの』

　　が、自身の変化を阻害する最大の要因となっている
　　ことに気づいていない」

　
　ところにあります。
　

　
■このパラダイムは、具体的には、

　簡単に、必ず、うまくいく、といった言説を信じるも、
　
　一人ひとり状況が違うので、
　そう簡単には結果が出ないことが知らされ、

　やる気が萎え、動けなくなる、といったような姿と
　なって外界に現れます。


　この場合、そもそも、
　　
　「簡単に、必ず、うまくいく」

　という前提自体が間違っているわけで、
　一番上のボタン（パラダイム）から掛け違えていたのです。

　世界はそんなに分かりやすいところではありません。



■本当のところ、
　複雑系の世界に生きている我々が、


　「成果とか成功を、

　　『こうすれば絶対うまくいく』

　　的な物言いを通して、

　　単純な因果律で説明できる線形的な結びつきとして
　　捉え、理解する」

　
　といった姿勢自体、おこがましいし、
　そんなことを、そもそもできるはずがない、

　だからはじめから疑ってかかるべし、
　と思われます。


　
■このことを大前提として、

　けれども、
　それを言ってしまうと実も蓋もないわけで、

　種々、研究と実践を繰り返しつつ、


　「なんにもしないよりは、
　　相対的に成果を出せるであろう、

　　そんな傾向（法則ではなく）程度のもの」


　をお伝えすることくらいはできるだろう、

　ということはあり得ます。



■法則ではなく、「傾向」程度のものしか
　伝達できないからといって、

　それを伝えるのが不誠実かというとそうではなく、


　「何もしないし、考えないよりは、
　　まず間違いなく、相対的に成果が出せる」
　
　という確信、確証があり、

　また、そこに価値を感じてくれる人があるならば、
　どんどん伝えていけばいいのです。


　必ずや、それを喜んでくれる方があるでしょう。


　
■ただし、

　「お伝えしていることは、
　　けっして絶対的なものではありませんよ」

　と外に向けて開示し続ける真摯さ、誠実さ、
　謙虚な姿勢だけは失わぬのが、

　　
　いわゆる

　「本の著者」

　であったり、


　「講演家」

　「セミナー講師」

　「コンサルタント」

　といった、伝える仕事をしている人にとっての

　「伝達者の良心」
　
　ではなかろうかと思うのです。
　


■ただ、現実問題として、これを行うのは、

　「ビジネス的には、相当マイナス」

　です。


　「これを読めば、絶対に儲かりますよ！」

　「これを聞けば、絶対に成功できますよ！！」

　「ここに来れば、絶対にうまくいきますよ！！！」


　などと煽り、焚き付け、駆り立て、

　圧倒的な自信を見せかけ断言するほうが、
　お客さまにとって受けがいい

　（＝売りやすい）

　のは言うまでもありません。



■けれども、繰り返しますが、

　「絶対にうまくいく」

　という物言いは、絶対に真実ではありません。


　従って、学ぶ側の立場として考えれば、


　１、このことを理解し、見込みのお客さまに伝えた上で、
　　　最善を尽くすのか、
　
　２、理解しつつも、あえて伝えず、
　　　収益が上がればそれでいいと考えるのか、

　３、そもそも、外部からの働きかけが、果たして
　　　どのような形で成果に反映されるか、
　　　　　
　　　のプロセスについて、深く考えたこともないのか


　相手の姿勢をよくよく観察し、見分ける必要が
　あるでしょう。



■このあたりを見極めるポイントとして、
　私（鮒谷）も、

　「各分野の師匠」

　を選ぶようにしています。


　
　今日の話は、イメージの湧かない人にとっては

　「？？？」

　な内容かもしれませんが、
　
　分かる人には分かっていただけるだろう、
　そんな風に思って書きました。
