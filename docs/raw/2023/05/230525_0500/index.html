■（明日で終了）
　「ジャイアント・キリング」
　「小よく大を制す」のための通過点

───────────────────


■当たり前ですが、
　自身が他人になることはできません。

　誰も自分以外の存在には
　なれないのです。



■ですから、

　願ったところで同じになれるはずもない
　他人を羨んだり、憧れる行為は、

　時間の無駄でしかありません。

　

■誰かと比べて、そうなれず、
　苦しむ人生を送りたくないのなら、

　まずこの大前提を
　理解しなければならないのです。



■とはいえ油断すると、すぐに

　「他者との比較」

　を始めてしまうのが人間ですから、
　
　うっかり「自分以外の誰か」に
　なろうとしていないか、

　継続的に振り返る必要が
　あるでしょう。



■やがて、こうした
　振り返りの習慣が身につけば

　「どうすればあの人と同じように
　　なれるだろう」

　という問いの代わりに、


　【どうすれば手元にある
　　自分固有の資源を活用して、

　　他の誰とも異なる
　　ユニークな人生を歩めるか】

　という問いが立つはずです。

　すべてはそこから始まります。



■この問いに慣れてくると、

　「隣の芝生は青く見える」

　原理を逆手に取って、


　【他の人と比べて見劣りするようにも
　　思われる資源を棚卸しして、

　　それらを組み合せ、独自性を持たせた
　　武器をどこの誰に提示するか】

　という、

　知的好奇心をくすぐるパズルを
　解いているような感覚にもなるものです。



■手元資源が潤沢にあれば、
　パズルは「初級編」ですが、

　資源がほぼない、
　という状況からのスタートは

　「中級編」「上級編」
　
　のパズルになり、
　それがまた楽しめるんですよ。



■脳漿を絞ってあれこれ考えて
　光明が見えてきたときには、

　大量のドーパミンが
　脳内でドバドバ放出されますよ。
　


■こうした思索＆実践は

　「ジャイアント・キリング」
　「小よく大を制す」

　の喜びや感動を味わうための、
　必ず通らねばならぬ通過点。


　
■ここまで書いて思い出したのですが
　高校時代、

　池波正太郎の『真田太平記』を
　読んで真田昌幸・信繁（幸村）父子の

　「ジャイアント・キリング」
　「小よく大を制す」

　の精神に震えたものでした。



■時間がある方、
　ぜひ読んでみてみて下さい。
　
　（全部で18巻の大部な
　　歴史小説ですが）


　人間に不可能はない。

　やってやれないことはない。

　あきらめたらそこで試合終了。

　そんなことを学べますし、
　勇気や希望をもらえます。



■真田父子もそうですが、
　世の中には
　
　【ないからこそ、できることがある】

　側面があるのを理解できれば、

　【「ない」というのも一つの資源】
　　
　と捉え直せるようになるはずです。



■資源があれば感謝して使えばいいし、
　目立った資源がなければ


　---------------

　▼資源がない＝失うものがない
　　　　　　　（大胆に振る舞える）

　▼資源がない＝恥をかくのも恐れない
　　　　　　　（圧巻の行動力が生まれる）

　▼資源がない＝知恵の絞り甲斐がある
　　　　　　　（追い込まれるから知恵が出る）

　---------------

　
　などと解釈を変えつつ、
　ないことを逆手に取ればいいのです。



■そうすれば

　【資源がない
　　＝資源がないという資源がある】

　という一見、詭弁に見えて、
　詭弁でもなんでもない、

　真実が浮かび上がってくるはずです。



■とするならば、

　「資源がないなどありえない」
　
　わけで、


　もしそんなことを言う人が
　あるとするならば、

　その人に私（鮒谷）は

　【ないのはあなたの想像力＆観察力
　　＆解釈力（全てまとめて我田引水力）】

　とお伝えしたいところです。



■こうした「我田引水力」さえ
　発揮できれば、

　あとはその力をフルに
　発動するために大量行動するのみ。



■とはいえ、

　大量行動を行うのがなかなか難しく、
　それを実現させるのが

　「直進行軍システム」
　
　という位置づけです。



■人生にこのシステムを導入する支援を
　するために、

　音源を用意いたしました。
