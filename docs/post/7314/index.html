7315号 商売人適性のある人、ない人の違いはここで明確に現れる

7314号 プロのデブ？空想地図作家？路上園芸鑑賞家？便所サンダルビジネス？珍スポットマニア？

■以前、ドラッカー著作の
　ほぼすべての翻訳を手がけられた、

　故・上田惇生先生をお招きして
　勉強会を開催したことがあるのですが、

　そのときに

　【変な客こそ、本命】

　と教わりました。



■ドラッカーの

　『イノベーションと起業家精神』

　について触れられたときに
　語られた言葉ですが、

　印象深い言い回しだったので
　記憶に残っているのです。



■同著には、有名な


　【成功したイノベーションを
　　分析したところ、

　　「予期せぬ出来事
　　（予期せぬ成功、予期せぬ失敗）」

　　から、もたらされたものが
　　もっとも多いケースであった】


　という話が記されており、

　私（鮒谷）も大きな影響を受けて
　いろいろな音源でお伝えしてきました。



■自分の頭でどれほど考えていても
　絶対に出てこない潜在ニーズが、

　他者からの思いがけない要望、
　ときにクレームから、

　顕在化することがあるものです。



■顧客からの想定外の要望は、

　「そんな目的のために
　　この商品を売っているのではない」

　「そんな面倒なこと、
　　やってられるかよ」

　といった怒りに
　つながることもあるものですが、


　上田先生は、むしろ

　「そうした客こそ本命になる
　　可能性があるのだから見逃すな」

　とドラッカーの言葉を引きながら
　語られたものでした。



■「変な客こそ、本命」

　の語呂の良さ（？）から、

　「変な客（失礼 汗）」

　が現れたときにも
　この言葉を思い出すようになり、


　追い返すのではなく、
　一旦、深呼吸して、

　この人の後ろ側に未だ顕在化していない
　巨大なマーケットが眠っているのでは、

　と考えるようにもなりました。



■つまり、

　おかしなお客さんが現れても、

　適当にあしらったり、
　バカにしたり、
　否定的に捉えたり、
　腹を立てたり、

　してはならないということですね。

　その向こうに大きなチャンスが
　開ける可能性があるのですから。



■また、その後、
　この言葉をアレンジして

　【変な人こそ、本命】

　という言葉を作り、
　脳内で回すようにもなりました。



■つまり、

　「平均的な人」
　「ありきたりな人」
　「どこにでもいそうな人」

　ではなく、


　「なんじゃこりゃぁあ！」

　と思わず叫んでしまいそうになる、
　そんな変な人と出会ったときにこそ、

　【キャリアにおけるイノベーションが
　　もたらされる可能性がある】

　という風に
　考えるようになったのです。



■こうして、
　それまでも無意識に行っていた

　【奇人・変人探しの旅】

　を意識的、積極的に
　行うようになりました。



■そうした中で生まれる出会いから

　【人生やキャリアを省みる機会】

　がもたらされるものであり、
　それこそ

　「なんじゃこりゃぁあ！」

　と叫ばずにはいられなくなることも
　時折、出てくるもの。



■その経験は直接、間接に

　「人生や取り巻く世界に対する認識の
　　大きな変容につながる」

　こととなり、

　けれども、こうした衝撃は
　常識人と出会っている限り、

　決してもたらされないものではないかと
　思われます。



■ここまで書いて思い出したことがあります。


　私（鮒谷）の中高の同級生に、

　京都大学を卒業し、
　某有名巨大企業に入った後、

　デイトレーダーとして
　巨万の富を得て、30過ぎで引退し、

　現在は仕事をせずに、悠々自適の生活
　（＝ぶらぶら）している友人がいます。


　たまにLINEでファーストクラスの写真が
　送られてきたりします笑



■そんな、ちょっと変わった、

　「なんじゃこりゃぁあ！」

　の変な（＝普通ではない）人ですが、


　そういう人生を
　自分は過ごしたいのかどうか、

　ということも含めて、

　触れるから、出会うから、話すから、
　考える機会が与えられるわけで、


　そうした人が周りにいなければ
　その世界を想像することもありませんから、

　目指すとか目指さないとか
　以前の話になり、

　結果として人生にイノベーションが
　起きる機会は生まれません。



■いうまでもありませんが、
　これはあくまで一例であって、

　なにもデイトレーダーになりましょうとか、
　私（鮒谷）もなりたいです、

　といった話ではなく、


　ただ、外部からの刺激がなければ、
　人生やキャリアや仕事について、

　これまでの延長線上以外のイメージを
　持つことができない、

　とお伝えしたいがゆえに
　例を出しただけです。



■そんなわけで、もしあなたが、

　今の人生に退屈しているなら、
　満足していないなら、

　不完全燃焼感があるならば、
　不安を感じているならば、

　なおのこと、

　「変な人」

　に積極的に交わりに行くべきではないかと
　ご提案したいのです。



■そのための、
　事始めの第一歩として、

　今月23日（土曜日）に開催する

　【ニッチビジネスフェスタ】

　に参加されてみられては
　いかがでしょうか。



■ご登壇くださる方々の、
　詳細なプロフィールは以下の通りとなります。

　いずれも

　【なかなかの奇人、変人（褒め言葉）】

　ばかりです。

　以下のような方々が一同に会して、
　話を伺える貴重な機会です。

　7歳の頃から空想地図（実在しない都市の地図）を
　描く空想地図作家。

　テレビドラマやゲーム上の架空の舞台の
　空想地図も受注製作する。


　代表作は「中村市」は現代美術として、

　東京都現代美術館をはじめ
　各地の美術館に出展している。


　過去の著書は「みんなの空想地図」（2013年）、
　「『地図感覚』から都市を読み解く」（2019年）、
　「空想地図帳」（2023年）他2冊。

　飯田正勝
　（合同会社中野フジヤマ堂 代表社員）


　会社員時代は中堅ソフトハウスにて
　10年間、法人営業を経験。

　失業保険情報サイトの先駆け

　「失業保険のもらいかた」

　　便所サンダル専門通販ベンサン.JP」

　「日本製一体成型サンダルの価値の再認識」

　天谷窓大（あまや・そうた）


　イベントディレクター・ライター・
　熱波師・焼き芋アンバサダー・レンタルデブ


　ITエンジニア、テレビ制作会社社員を経て
　2019年よりフリーランス。

　会社員時代の2013年、
　「過激な会社への行き方」を競う
　「エクストリーム出社」を提唱、

　ブームになったことから
　企画クリエイターとしてのキャリアを
　スタートさせる。


　2017年には日本初の
　都市型焼き芋フェスの企画運営を担当し、

　焼き芋アンバサダーとして焼き芋メニュー・
　焼き芋イベントのプロデュースを開始。

　その後、活動で生まれたつながりを経て
　2020年からは横浜市鶴見区

　「ファンタジーサウナ＆スパ おふろの国」

　でサウナ熱波師としても活動。

　サウナの技術を焼き芋に応用した
　「サウナ焼き芋」もプロデュースする。


　テレビ・ラジオの制作に携わった経験を活かし、
　ライターとして、

　「TVerプラス」、
　三才ブックス「ラジオマニア」を始め、

　エンタメ媒体でインタビュー取材・執筆も担当。


　さらに100kg超えの巨大な体型を活かし、

　”レンタルデブ”サービス「デブカリ」にて

　“プロのデブ”

　としてフリー素材モデル・メディア出演も行う。


　自らに関わるあらゆる要素をコンテンツ化し、

　そこで生まれたつながりを活かして
　新たな活動を開拓し続けている。

　松澤茂信（まつざわ・しげのぶ）


　1982年東京都生まれ。
　上智大学経済学部卒。国内旅行業務取扱管理者。

　2011年より日本中の変わった観光地
　（珍スポット）を紹介するサイト

　「東京別視点ガイド」

　を始める。


　日本全国＆アジア各国1,000ヵ所以上巡った
　珍スポットマニア。

　2018年合同会社別視点を設立。

　来場者20,000人を超えるイベント
　「マニアフェスタ」、

　マニアによる専門的ガイドが楽しめるツアー
　「別視点ツアー」

　などの体験型イベントを数多く実施。

　2023年に学芸出版社より
　『マニア流!まちを楽しむ「別視点」入門』を出版。


・マニアフェスタとは

　「サザエさんジャンケンマニア」
　「犬糞看板マニア」「路上園芸マニア」

　など、ニッチで唯一無二なマニアが
　毎回50組以上出展するイベント。

　2018年よりイベントスタートし、
　年々注目度を増していき、

　ジロジロ有吉、めざましテレビ、
　散歩の達人などメディアに多数出演。

　2023年の「マニアフェスタvol7」は
　ニコニコ超会議内で開催。

　2万人以上の来場者が訪れた。


・東京別視点ガイドとは

　日本中＆アジアの変わった観光地、
　飲食店（珍スポット）を1000ヶ所以上巡り
　レポートするサイト。

　Twitterフォロワー10万人超、
　最盛期は月100万PVを集めていた。
