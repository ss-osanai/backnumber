＜今日のピークパフォーマンス方程式＞

 【　汎用性の高い人生変革法　】

───────────────────────────────────



■３日間セミナーでは、

　セミナー終了後（というか、初日終了後から）
　いくつかの課題をこなすことを課しているのですが、

　その中の一つが

　「文章を書く（ブログを書く）」

　こと。



■特殊な技能を持たない、圧倒的多数の人に
　とって、再現性高く、

　「人生を変革させるための方法論」

　があるのだろうか？


　もしあるとすれば、それはどのようなものだろう、
　という問題意識のもと、

　自らの体を実験台としながら、
　その方法論を言語化し、発信し続けてきたのが、
　
　平成進化論発刊以来、
　2700号（＝2700日）の歴史でありました。



■そのようにして私自身が実践し、
　（自分なりに、一定程度には満足できる）結果を出し、

　さらにはその方法論に汎用性があるのかどうかの
　テストとして、友人知人、後輩にも試してもらってきた
　ものの中で、
　
　「相当程度に効果がある」
　
　と分かったことの一つが文章を書く、
　すなわちブログの執筆、だったのです。



■ただし、単にブログを書けばよい、というものでは
　ありません。

　いくつかのルールに従い、書く必要がありますが、
　そのルールを忠実に守りさえすれば、

　時間の経過と共に持続的、逓増的スパイラルに
　入ることが、かなり高い確率で実証されても
　きています。



■その理由については、いろいろとあるのですが、、
　今日は短めですが、明日に続けます。
