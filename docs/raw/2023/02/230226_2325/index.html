■「真の投資家」は
　一瞬のチャンスを逃さない

───────────────────
　

■人生を楽しみたいなら積極的に

　【偶然の力】

　を借りるべきでしょう。


　これまでと異なることに取り組んで、
　人生軌道が劇的に転換する期待を持つ、

　これが

　【偶然の力を借りる】

　ということ。

　「偶然性＝予見不可能」ですが、
　これこそが人生の醍醐味です。



■すべてが見通せて、
　予定調和的に完結する、

　そんなゲームが
　退屈でつまらないのと同様に、

　人生もまた、不定期に、
　予見不可能だった出来事が起こるところに、

　楽しみがあると言ってよさそうです。



■もちろん、

　人によっては、あまりに高い
　ボラティリティ（変動率）は耐えられない、

　という人もありますから、

　
　自身の「リスク耐性」を考慮に入れつつ、

　偶然性、予測不可能性を
　どの程度に設定するか、

　は各人それぞれが
　決める必要があるでしょう。



■しかし、

　あまりに保守的に、
　つまり低変動率を求めすぎると、

　安全かもしれないけれども
　実につまらない、

　「ひたすらにルーティンが
　　繰り返される毎日」

　がやってきます（当然ですね）。



■３年先、５年先、10年先、
　あるいはそれ以降においても、

　「おおむね、どのような人生を
　　過ごしているか予見可能」

　という何の期待も楽しみもない、
　しょぼくれた世界など、

　少なくとも私（鮒谷）は
　生きたくありません。
　


■人生における
　最大の娯楽であり、愉悦は、

　それまでに想像もできなかったような

　【質的な転換】

　だと思うからです。



■その経験をしたいと思ったなら、

　　　取りに行く

　などにおいて、
　お伝えしてきました。
　


■「クマムシ音源」で

　「勝つことよりも 100 倍重要！
　　破滅を回避する力を手に入れよ」
　
　とお伝えしているように、
　守りは大事です。


　「身体的にも経済的にも、
　　死んだら（詰んだら）終わり」

　ですからね。



■しかし破滅を回避することは
　もちろん最優先した上で、

　リスクを限定しつつ、他の人が
　しないような挑戦をする者だけが
　
　【人生における興奮や高揚感と
　　いった果実】

　を手にすることができるのも
　また真実。



■昨日と同じことを今日もやり、
　今日と同じことを明日もやる。

　そんな毎日を続けていて、人生に
　質的な転換が生じるはずがありません。
　
　
　この明白な事実に気づいた方が
　年間プログラムにお越し下さっています。

　ダウンサイドは参加費のみ。
　アップサイドは無限大。
　
　
　
■そして実際に、

　大きなアップサイドを獲得しつつ
　ある方がどんどん現れているのが、

　現在の年間プログラムです。
　


■どう悪く見積もっても、
　失われるものは参加費のみで、

　（それさえも、参加して
　「全く役に立たなかった」と言われたら、
　　即日、全額返金いたしますよ）


　それでいて、手に入れられる
　将来の可能性はプライスレス。

　人生が加速する感覚なんて、
　金額換算できない喜びですからね。


　
■たしかに
　相応の金額かもしれないけれども、

　「期待値で
　　考えてみたらどうですかね」

　「しょうもない宝くじを買うよりも
　　はるかに当選確率高いと思いますよ」

　とお伝えしたいところです。



■下方リスクを検討した上で、
　上方リスク（伸びしろ）を想定し、

　リスクが限定されつつ、
　リスク以上の高いリターンが得られる
　確度が高いと思ったなら、

　そこに躊躇なく投資できるのが
　
　【真の投資家】

　と言えるのではないでしょうか。

　そして【真の投資家】は、ここぞ、
　という一瞬のチャンスを逃しません。



■もしあなたが、

　「未来が見通せてしまう、
　　退屈でつまらない、
　　高揚感も興奮もない人生」

　から離脱したいと願われるなら、

　人生において一度くらいは
　そんなバクチをしてみても良いのでは、

　と個人的には思います。


　
■もちろん「年間プログラム」は
　一つの例であって、

　ありきたりでつまらない日常に
　「非日常」を投入できる対象であれば
　なんでもいいのですが。

　※ただし、期待値計算と下方リスクだけは
　　きちんと行なってくださいね。



■そんなわけで、このたび
　　
　【１対１での年間プログラム説明会】

　を開催することにいたしました。
　（Zoomで実施いたします）



■年間プログラムに
　関心はあるけれども、

　内容がわからない、
　不明点がありすぎる、

　という方にこそ、
　お申し込みいただければと思います。



■もちろん、参加したからといって
　
　「おかしな圧力をかけて、
　　その場で申し込みを強制、強要」

　するようなことはいたしませんので
　ご安心下さい。

　避けるため、

　伺いますので、

　「個別コンサル的な価値」

　も間違いなく
　お渡し出来るはず。
