【 映画『関ヶ原』を観て思い出した、小学４年生だったあの頃のこと 】

───────────────────────────────────



■今日、観に行った映画は『関ヶ原』。
　司馬遼太郎原作。

　出来映えについて批評できるほど
　映画に深い造詣のあるわけでもなし、

　以下、あくまで一個人の感想。
　

　「原作イメージとはまるで異なる映画に
　　仕上がっていた」

　「司馬遼太郎の『関ヶ原』の密度の濃さが
　　感じられなかった」

　というのが率直なところでした。



■もちろん監督の狙いは別のところにあったのかもしれず
　（というか、きっとそうなのでしょう）、

　あくまで私がそのように感じた、
　というだけの話ですが。
　

　原作となった同書は秀吉の死後直後から始まる物語。

　諸国大名を調略するために、
　家康と三成の間で繰り広げられる、

　政治、駆け引き、策謀、腹の探り合いが
　この本における最大の読みどころ。
　


■そんな最も肝心な部分が
　期待していたレベルで描かれていなかったので、

　肩透かしな感じがしてちょっと残念でした。


　近くで見ていた岡田准一ファンと思しき
　女子中学生？高校生？グループには、

　満足度高かったようですが（笑）
　


■それはさておき、

　私（鮒谷）がなぜこの映画を封切り早々、
　観に行こうと思ったかというと

　『関ヶ原』

　という小説に、
　ひときわ強い思い入れがあったから。



■私はおそらく、

　平均的な人よりは、活字を追う喜びに
　目覚めるのがほんの少し早かったのですが、

　実は活字が相応に詰まった

　「大人の読む本」

　を読む喜びを教えてくれたのが、この本でした。
　


■小学校低学年のとき、

　なにがきっかけだったのか、
　戦国時代に興味を持ち、

　４年生になり、父親の蔵書中に

　『司馬遼太郎全集』

　を見つけました。


　それがきっかけとなり初めて司馬遼太郎の

　『関ヶ原』

　を手に取ったのです。

　（それまでは「字の本」は、小学生に向けて
　　書かれたものばかりで、いきなりの跳躍となりました）
　


■振り返って考えると司馬遼太郎の著作中、

　初めての作品がこの本だったのは
　私にとって幸いでした。

　事前に、おおよその戦国情報（武将の名前とか旧国名とか）は
　既に頭の中に入っているので、

　好奇心と親近感を持って読み進められる。
　

　また、小学男子が盛り上がるような、
　
　血湧き肉躍る、格好良い戦闘場面の描写も
　ほどよく盛り込まれているので、

　政略一辺倒の話にもならず。
　


■中学受験の勉強を始めるために
　塾に行くようになったのは小学５年の夏からで、

　それまでは一切の習い事もしていなかったので、


　ヒマにあかせて辞書や漢和辞典を手元に置きながら、
　分からないところは調べ調べしていくのだから、

　だんだん内容も分かるようにもなっていきました。
　


■こうして何度も繰り返し読むうちに全体の構成も
　理解できるようになるし、

　初見だった漢字も読めるようになり、
　氏の独特の文体にも馴染み、

　読書の真の喜びにも目覚めるようになりました。
　


■これがきっかけとなり、
　途中、中学受験期の１年半のブランクを挟みつつ、

　以降、中１から高３までかけて
　司馬遼太郎全集を読破することとなりました。

　（幸か不幸か、学校までの通学時間が
　　往復3時間以上あったので、

　　それが格好の読書時間となりました）


　
■映画を観に行ったことをきっかけに、
　そんな過去のことを回想したのですが、そのときふと
　
　【好きこそものの上手なれ】
　
　という言葉が浮かび上がってきました。
　

　昨日、出身高校の同窓会に参加しましたが
　校長講演では、論語の
　
　「これを知る者はこれを好む者に如(し)かず。
　　これを好む者はこれを楽しむ者に如(し)かず」
　
　という言葉を取り上げられていました。
　これも同じ意味ですね。
　


■今から考えると、

　とはいえ小学４年生で司馬遼太郎の本
　（それも文庫本３冊分の長編）

　を読むというのは、それなりの集中力と持続力が
　なければ決してできなかったことだと思います。
　

　大人になった今でも、

　興味を持てない文庫３冊組の長編小説を読めと
　言われて、読み通す自信はありません。
　


■そう考えると大人だろうが、子どもだろうが、

　我を忘れて没頭できる対象を見つけたとき、
　強烈な学習能力が発動されるのだろうと思われます。
　

　考えてみれば
　
　『関ヶ原』

　にハマっていた一時期は、
　それに関する様々な本や雑誌を読み漁りました。

　
　今でもあるのか『歴史読本』とか『歴史と人物』と
　いった雑誌を父親に買ってきてもらい、

　大名、武将のエピソードとか各合戦の布陣図とか
　暗記するくらい、繰り返し読みました。
　


■小学校のテストを早々に解き終えたあとは、

　テストの裏に戦国武将の話やら、
　布陣図やら、のぼり旗などを書いてみたり。
　

　当時の先生（重松先生）はそんな自由を許してくれ、
　かつ褒めてもくれたので、

　ますますやる気になるの巻。
　


■知的好奇心はこうやって育まれていったんだなあと
　改めて思いますし、

　こうした主体的な意志なり意図を持った学習は、
　年齢に関係なく、

　驚くほど短期間での文脈形成、拡大を
　もたらしてくれたようにも思われます。
　


■これまた考えてみれば結局、

　『関ヶ原』

　を夢中になって何度も読み返し、
　その没頭の延長線上に司馬遼太郎作品への没入が始まり、

　それが今に至る、

　歴史、政治、地理、人物、文化、文物といったものへの
　関心を生み出してもくれました。
　


■こうしてやがて

　「学習の核（興味関心分野の中心）」

　が生み出されることとなり、

　この核を中心として、どんどん学びが広がり、
　そして深まっていきました。

　
　また、一つの「学習の核」には往々にして
　「核分裂」が起こります。

　ある分野に没頭すると、それが新たな興味関心の分野を
　開拓することにつながるのです。

　こうして新たな核が生まれること、
　これを

　「学びの核分裂」

　と私は呼んでいます。
　


■こうして次々、核分裂が起こり始まると、

　自分の守備範囲は自ずから拡大することとなり、
　ここにいたって

　「既知と未知とが驚くべき勢いで
　　結びつき始める」

　こととなるでしょう。
　


■ちょうどWikipediaで何かの分野を深く学習しようと
　するとき、

　既に土地勘のある分野の学びは
　急激に加速するようなもの。


　「未知に未知」の学びは右から左へと
　素通りするようなものですが、

　「既知に未知」

　は学びの歩留まりが急速に高まりますから、
　学習の効率化も大きく進むことになります。
　


■その際、その閃きや発見を言葉によって記述しておくと、
　さらなる歩留まり率の向上にも寄与することでしょう。

　既知のことが増えるほど、
　学びが発動される接地面も広がりますから、

　ある段階まで進むと、常時

　「既知に未知が結びつき続けている」

　という状態が生まれます。
　

　
■ここまでいくと
　
　「新しい情報 → 既知と結びつく → 気づく、学ぶ、閃く
　　→ 編集し、まとめる → 言葉で記述する」

　の一連の学習サイクルの、最後の部分、
　すなわち

　「言葉で記述する」

　ことが時間の制約上、
　追いつかなくなってくる

　（＝書ききれなくなってくる）

　のです。
　

　
■私はここで次善の策として、
　「話す」機会を積極的に設けることにしました。

　話すほうは、記述と異なり、

　学習の定着率が下がることと、
　論理の精緻さや明晰さに欠ける、

　という欠点があります。
　


■ただ、その分、圧倒的なアウトプットの機会を
　増やすことができ

　（集中力の問題もあり、私にとっては一日、
　　５時間とか８時間とか書き続けるのは困難であっても、

　　それくらいの時間、
　　話し続けるのに何の苦労もありません）、


　つまりは

　「気付き、学びを、編集し、まとめる」

　の時間が増加するし、

　学びを言語化する時間がとれずに、
　時間の経過とともに揮発することもなくなります。
　


■両者はトレードオフの関係にあることを理解しつつ、
　最適なバランスを追求することで、

　より一層、学びを加速させることも
　行えるようになるでしょう。
　

　一点、重要なことを補足しておくと、

　書いたり話したりした、
　自分の言葉に触れることによって、

　新たな触発が生まれ、
　気付き、学ぶ機会がさらに増えるので、


　特に新しい

　「体験」

　をした際には、都度、その体験を書く、あるいは
　話すことによって広げ、深め、総括し、

　自らの血肉となる

　「経験」

　に置き換えていくことが重要であると思います。
　


■いろいろ書いてきましたが、
　つまるところ私の考える自己教育の要諦は
　

　▼人生のできるだけ早い段階において、
　　一定の大きさを持ち、刺激に敏感に反応する

　　「既知の核」

　　をまずひとつ作る
　
　（できれば幼少期に一つは欲しいところだけれども、

　　それが叶わなければいつの時点からでもいいので、
　　自分の好奇心の核となる分野を見つけ、深めることが大事）
　

　に加えて

　
　▼効果的な核分裂（学びの核をどんどん分裂させる）の
　　方法論を確立する
　

　ことにあるのではないでしょうか。
　


■このサイクル（学びの高速増殖サイクル）が稼働すると、

　それぞれの「知識」がバラバラに存在するのではなく、
　相互につながりを持ち、

　実生活において活用し得る「知恵」へと
　昇華されることとなります。
　

　こうして知恵を範囲が拡大するにつれ、
　当然、学びの接地面も広がりますから、

　さらなる学びを呼び込むこととなります。
　


■ここにおいて、
　
　「学ぶ人は（もはや学んでいるという自覚すらなく）
　　学び続ける」
　
　し、
　
　「学ばない人（学べない人＝核のない人）は
　　そもそも学ぶという概念の、本当の意味が分からないから、
　　永遠に学べない」
　
　という、絶望的な学び格差が発生するのかもしれません。

　

■その格差は、やがてさまざまな分野へと波及し、
　特に「生命力」に直結していくのではないかと思えます。

　学習習慣が定着した人は
　
　「活きた知恵」

　を持つ人となりますから

　「生き抜く力」

　の強い人となり、また、

　「生きる歓びを感じられる力」

　を持つ人となりましょう。
　


■この世界に出ることが人生の醍醐味であり、

　仕事とか、キャリアとか、経済的な自立などは、
　その生活を根底から支えるものではあるけれども、

　手段に過ぎないのではないか、

　と考えています。
　

　こうして学びの高速増殖サイクルが発動された人は、

　あとは勝手に根を張り、芽を出し、幹を形成し、
　枝葉も生い茂るようになり、

　人生において豊かな果実を実らせることとなるでしょう。
　

　
■今日は、

　映画『関ヶ原』を観た、という刺激から、
　気付き学びの増殖サイクルが発動し、

　思いがけず長文を書いてしまうことになりました。

　でもこうした現象が時々刻々、起きてくれる
　（自動発火してくれる）こと自体が、

　私にとっての何よりの歓びであり、
　楽しみとなっています。



■ちなみに、

　ここまでに書いた内容をさらに広く、深い文脈で
　敷衍している

　「人生、如何にして学び、切り拓いていくか」

　についての話は、
