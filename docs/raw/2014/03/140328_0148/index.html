＜今日のピークパフォーマンス方程式＞

 【　渡世に役立つ「紹介してもらえる力」　】

───────────────────────────────────



■いつも書いていることですが、
　
　「人とのつながり」

　が新しい世界を開いてくれることは、
　よくあるものです。



■ある人との出会いによって、文字通り、

　「人生を（良い方向に）持っていかれる」

　ことが往々にしてあるのです。


　そんな出会いを「邂逅」あるいは「巡り合い」と
　呼ぶのかもしれませんね。


　そのようなご縁を求めて、

　「他者とのつながり」

　は、大切にしていきたいもの。



■そう考えると、渡世するにあたって、

　「自ら出会いを生む力」

　とともに、

　「紹介してもらう力」

　は、極めて重要な資質といわざるを得ないでしょう。

　
　ちなみに

　「紹介をお願いする力」

　と

　「紹介してもらえる力」

　をはき違えてはなりません。



■いずれも持っていると役に立つ力ではありますが、
　どちらかといえば、私たちは

　「紹介してもらえる力」

　をこそ磨く必要があるでしょう。


　紹介をお願いする人はあっても、
　紹介をしたくなる人は少ない。


　だからこそ、そんな

　「紹介したくなる人
　　（＝紹介してもらう力を保持している人）」
　
　になれれば、人生はとても渡りやすくなるのです。



■つまり、自問自答すべきは

　「どうしたら紹介してもらえるか」

　ではなく、

　「どうしたら紹介してもらえる人間になれるか」

　という問いについて。



■これについて、
　私（鮒谷）が考える３つのことがあります。


　一つは、愛嬌のあるなし。

　笑顔とか愛想の良さとか、親しみやすさ、とか、
　その人から醸しだされる雰囲気、

　とでもいえばいいでしょうか。
　

　これがあるからこそ、紹介してあげよう、という気にも
　なるというものです。

　気取っていたり、かしこぶったり、偉そうにしていたら
　ダメなのです。

　

■もう一つは、

　自分が先に何を提供できるか、を考える姿勢が
　身に付いているかどうか。


　出会って最初の段階から、
　自分は相手にどんな価値を渡せるかを考えることなく、

　「くれくれ、くれくれ」

　のみを言っている人には、

　最初の１、２回は何かを与えられるという僥倖に
　恵まれたとしても、

　以降、継続して与えられる可能性は
　とても少ないでしょう。


　こちらから先に役立とう、という気持ちを持つ人が
　
　「積極的に、紹介してもらえる人」

　の資質を持った人。



■最後に、

　たとえ価値をお渡しししたいと思っても、肝心の、
　
　「価値を提供する力」

　を持っていなければ、これも始まりません。

　
　他の人に喜ばれる、あるいは役に立つ能力や技術や経験、
　といったものがあって、はじめて、

　「○○さんは、こんな分野（能力、技術）において
　　卓越しているんだよ」

　という形で紹介をされることとなります。


　
■厳しいようですが、

　（あくまでビジネスに限った話ではありますが）

　「自分に何の価値ももたらしてくれない」

　と思われる人を紹介されるのは、
　紹介される側にとっては、単なる時間の浪費、

　と思われることが多いようです。
　


■とするならば、

　「（良くて）人畜無害な人」
　「（悪ければ）箸にも棒にもかからない人」
　「（もっと悪ければ）極悪非道な人」

　を紹介してしまったら、紹介者側も、
　大きなダメージを受けることになるでしょう。

　（だからこそ、
　　人の紹介は慎重に行わなければなりません）
　


■以上３点、いずれも、

　自分が身の回りにいる大切な人を紹介する立場に
　たったときには、

　「そりゃー、当然だよな」

　思えることのはず。


　
■ならば、自分が人さまから紹介してもらえる存在に
　なるためには、逆に、

　この３つの点を満たせるよう、
　普段から心がけていけばよいのです。


　なんて、言うは易く行うは難し、なんですけどね（涙）

　お互い、頑張りましょう！
