7303号 コスパお化け、タイパお化けを一旦は目指してみる、という選択肢

■ここ数日、

　「コスパ（コストパフォーマンス）」
　「タイパ（タイムパフォーマンス）」

　の功罪について記してきました。



■効率重視もいいけれど、
　行き過ぎると味気ない。

　効率無視は楽しいけれど、
　行き過ぎると生活が成立しなくなる。


　そんな二律背反の中を
　私たちは生きていかなければならない、

　なんていったら
　大袈裟に過ぎますかね。



■今日はここに

　「リスク管理」

　という観点からの補助線を引いて
　考えてみましょう。



■リスク管理において
　もっとも大切なのは

　「破滅しないこと」

　です。


　どれだけ高いリターンを狙えても、

　想定外のことが生じた際に
　受け止めきれないほどの損失が見込まれる、

　そんな投資は、
　投資対象として不適格。



■安心して高いリターンを
　狙いにいけるのは、

　自身の人生や生活、キャリアにおける
　耐久性や抗堪性がそれなりに大きくなり、

　一定程度までの損失を

　（破滅することなく）
　受け止められるようになった、

　そのときから。



■この状態に入りさえすれば、

　多少、冒険して失敗したところで
　後からの挽回も容易になるし、


　そのような、安全が担保されている
　状態で行う投資姿勢を堅持しての、

　リスクに比して高いリターンが
　見込める対象への投資を重ねれば、


　長い目で見れば、
　このゲームにおいて、

　（破滅リスクを極小化させつつ）

　満足のいく結果に近づけていける確率は
　どんどん高まっていくでしょう。



■であるならば、

　最初はどれだけつまらなくても
　割り切って、一時的に

　（おそらく、ある時点から伸びしろは
　　急激に小さくなると思われるが）

　【短期的には確度高くリターンを狙える、
　　コスパお化け、タイパお化けを目指す】

　というのは、

　検討の俎上に乗せる余地のある、
　一つの人生・キャリア戦略といえる、

　かもしれません。



■ただし、この道は

　「ミイラ取りがミイラになる」

　可能性がゼロにはならない、

　そういう意味でのリスクが
　内包されています。



■生活基盤を整えて（ホップ）、

　からのステップ、ジャンプ、
　を遂げるはずが、


　ホップの段階で足元を掬われて、
（＝目的と手段が入れ替わってしまい）

　コスパお化け、タイパお化けから
　抜け出せなくなって生涯を終えてしまう、

　そんな危険性すらあることは
　理解しておく必要がありそうですね。



■繰り返しになりますが、

　そうしたリスクを承知した上で
　割り切って、一時的に


　（おそらく、ある時点から伸びしろは
　　急激に小さくなると思われるが）

　【確度高くリターンを狙える、
　　コスパお化け、タイパお化けを目指す】

　のは戦略としては
　ありなのかもしれません。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■コスパ、タイパを、リスク管理の観点から
　　考えてみると、

　　割り切って、一時的に、コスパお化けや
　　タイパお化けを目指すのは、

　　一つの戦略としてあるのかもしれない。

　■ただし、これらには蠱惑的な魅力があり、
　　ミイラ取りがミイラになる可能性も
　　ゼロではなさそうだ。

　■そのリスクを承知した上での一時的な、
　　コスパお化け＆タイパお化け志向は、

　　検討の俎上に乗せる余地のある
　　人生戦略のオプションになり得るだろう。
