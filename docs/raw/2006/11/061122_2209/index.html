　【　何も持たなくても目上の方に大きな価値貢献をする方法がある　】

───────────────────────────────────



■この「平成・進化論。」でよく書いていることですが、
　ありとあらゆる世代の人と付き合うことが重要だと、私は
　思っています。

　ともすると私たちは、同じ年代とかたまりがちです。
　そのほうが気が楽ですから、放っておけばどうしても
　そうなるのです。

　しかし、それでは並の成長速度にとどまらざるを得ない
　でしょう。



■そこで、思い切り目上の人、
　あるいは自分よりはるかに年下の人とも積極的に交わって
　いくのです。

　今日は、そのうちの「目上の方」との時間の過ごし方について
　お話ししたいと思います。



■先輩と一緒にいれば、同年代の人と過ごしているよりも
　緊張するのは当然のことでしょう。

　年上の人であればあるほど、成功された方であればあるほど、
　プレッシャーは強くのしかかってきます。

　そのプレッシャーと戦いながら、大先輩の方に意識を合わせて
　いこうと、心の中で絶え間なく努力するわけですが、

　それがうまくいかなければただ単に恥をかき、
　相手の方には悪い印象を与えて終わるだけ。



■ただ、そんなプレッシャーがあるからこそ、学ぼうという
　気も起きるのですし、神経を張りつめようとも思うのです。

　その負荷が成長につながります。

　恥をかいたらかいたで、それがまた次の成長へとつながると
　思えば、そんな痛みも喜んで歓迎すべきというものでしょう。



■また、目上の方とお会いするからには何らかの価値提供を
　しなければいけないと思うのは当然のことで、

　「そんな価値提供、私にはできません」

　ということで、ついつい腰が引けてしまうという方も
　いらっしゃるかもしれません。



■そのお気持ちは充分よく分かりますし、私自身も感じることでは
　ありますが、
　最近思いますのは、

　こちらから、たとえば何らかの知識なり情報なり、
　あるいは人を紹介させていただいたりといった価値が提供
　できなかったとしても、

　【 目上の方に気持ちよくしゃべってさえいただければ、
　　それ自体がひとつの大きな価値である 】

　と捉えていただいてもよいのではないかと。



■実際、成功された方は、ご自身の体験を人に伝承したい、
　継承したいという内なる要求をお持ちの方が多くいらっしゃる
　ようにお見受けします。

　教育について関心を持たれている方が少なくないのも、
　そんな思いに基づかれてのことなのかもしれません。



■ですから、こちらから何も出すものがなくても、

　相手の方から教えていただくことを素直に受け止め、喜び、
　感動し、感謝し、お礼の気持ちを申し述べる。

　これがすでに、ひとつの価値貢献であり、価値の交換がなされて
　いるとするならば、それでいいのではないでしょうか。



■よく、

　「鮒谷さんはさまざまな目上の方と会われていますが、
　　一体どんな価値を提供しているのですか」

　ときかれることがあります。

　率直に言って、提供できるものなど何もございません。

　

■ただ私は、いつも素直に、謙虚に、心の頭をたれて学ばせて
　いただこうという姿勢だけは忘れないでいようと思っています。

　それによってその方が、

　「ああ、またこいつには教えたいな」
　「伝えてやりたいな」

　という気持ちを持ってくださる、ということはあるのかも
　しれません。



■目上の人と過ごす時間は、私は非常に大事なものだと思っています。

　ただ、だからといって肩ひじ張って提案だ、プレゼンテーションだ、
　価値の提供をしなければ、といって思い悩まなくても、

　素直に話を聞き、感動し、感謝すること自体が価値貢献である、

　という気持ちを持たれると、力を抜いて、人生の大先輩とのご縁も
　増えていくのではないでしょうか。
