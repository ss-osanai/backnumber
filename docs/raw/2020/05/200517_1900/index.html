【（あと５時間）「三冠王」の狙い方】

2020/05/17 19:00 
■こんばんは、鮒谷です。


　セールスはバッティングに
　似ていると思うのです。

　野球、やったことないから
　知らんけど。



■バットの真っ芯で
　球を捉える能力を体得すれば、

　パワーヒッターでなくとも
　柵越えを連発させられる。


　それと同じように、

　顧客ニーズをど真ん中で捉える
　力を身につけられれば、

　暑苦しい豪腕系セールスでなくとも
　受注を量産させられる。



■とはいえ、打者の理想形は


　【球を真っ芯で捉える能力を
　　持ちながら、

　　パワーもあるから打った瞬間、
　　場外ホームランだと分かる当たりを
　　連発させられる、

　　技巧派パワーヒッター】


　です。知らんけど。


　
■同様に、
　セールスにおける理想形は


　【顧客のニーズをど真ん中で
　　捉える力を持ちながら、

　　情熱があり、尋常ならざる
　　情報提供量を維持し続け、

　　営業した瞬間、
　　高額受注を連発させられる

　　技巧派情熱セールス】


　です。



■とはいえ順序としては、
　バッターであれば

　--------------------------------

　１、球を芯で捉えられる力がある
　２、その上でパワーがあればなお良し

　--------------------------------

　であり、


　セールスであれば

　--------------------------------

　１、顧客ニーズのど真ん中を
　　　推察、洞察する力がある

　２、その上で情熱と圧倒的な情報提供を
　　　行えればなお良し

　--------------------------------

　ということになるのでは
　ないでしょうか。



■たしかに芯を捉えなくても、

　尋常ならざるパワーがあれば
　投げられた球を柵越えに至らしめる、

　ということもないわけでは
　ないでしょう。


　でも自然じゃないですね
　（無理がある）。



■同じように営業において、


　顧客ニーズを全く捉えていなくても
　尋常ならざる情熱と行動量

　（夜討ち朝駆けとか、
　　買ってくれるまで帰らないとか、
　　俺を男にしてくださいとか）

　で商品の販売に成功してしまうことも
　あるでしょう。
　

　しかし自然じゃないですね。
　（無理がある）



■やっぱり理想的には、


　軽くバットを振り抜くだけで
　安定的に柵越えを続けられる、

　全く負荷がかかっていないのに
　継続的に受注が量産されている、


　そんな状態を
　キープし続けたいもの。



■私（鮒谷）が思うところの

　「圧巻セールス」
　
　は、そんな

　「売らなくても売れていく」

　状態を作り上げるところから。



■そこからさらに、

　パワーヒッター的要素
　（情熱＆情報提供量）

　を身につけられれば、


　「売らずに売りつつ、
　　売って売る」

　という最強セールス状態に
　入ることができるでしょう。



■とはいえ、

　いずれにしても
　まず先に身につけるべきは

　「球を芯で捉える能力」


　営業においては

　「顧客の潜在ニーズを
　　的確に掴む（推察＆洞察）能力」
　
　です。



■それができるようなった上で、
　次は、私（鮒谷）であれば、

　文章に情熱を載せ、

　さらに文章量＆発信回数に
　こだわりを持ちます。


　
■ただし、

　心を捉えていない
　（顧客ニーズを理解していない）

　状態で発信し続ければ、

　やればやるほど
　不快指数が高まるだけですから、


　そうならぬよう、相当、
　意識を払っています。



■野球の醍醐味は、

　絶対に打たれまいと思っている
　投手に対して、

　技巧とパワーで対峙する
　そんな大リーグ精神を感じるところ。


　知らんけど。



■セールスにおける醍醐味は、

　絶対に買うまいと思っている
　見込み客に対して、

　潜在ニーズに的確に訴求し、
　圧倒的熱量で対峙する、

　そんな圧巻セールス精神を
　発揮するところ。



■向き合う投手の球を、

　ニーズのど真ん中を捉えた上で
　圧倒的パワーで場外に叩き出す、
　
　そんなイメージで日々、
　セールス活動に勤しんでおります。



■そのためには、
　何をさておいても

　「動体視力の強化
　　（顧客ニーズの推察、洞察）」

　を先行させる必要があり、
　次に

　「筋トレ
　　（情熱＆情報提供量）」

　の順番です。


　
■筋トレは
　そこまで難しくないけれども、

　動体視力の強化は
　なかなかに難しい。


　でも、先にそこを強化せずして
　筋トレだけやっていても


　「たまに（まれに）
　　ホームランを打つことはあるけれども、
　　ほとんどは三振」

　という、

　安定感のない、
　それゆえ監督としては起用しにくい

　「一発屋」

　にしかなれません。



■やっぱりどうせ野球をやるのなら、
　連続して長い間

　「三冠王」

　のタイトルを守り続けるくらいまで
　いきたいじゃないですか。

　営業においても同じ。
