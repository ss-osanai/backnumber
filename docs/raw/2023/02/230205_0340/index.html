■おはようございます、鮒谷です。


　今日（というか、昨日）は終日、
　月に２回開催の年間プログラムでした。


　このところ参加者の皆さんが、

　場の力による相乗効果によって
　急激に変化を遂げられつつあり、

　これまで以上に

　【変化・変容のコミュニティ】

　として機能しつつあることを
　実感しています。

　

■一人で頑張ろう、
　運命を変えたいと思っても、

　残念ながらその思いを
　持続させるのは難しいもの。
　
　人生を変える冒険に
　本気で取り組みたい方は、
　ぜひご参加下さい。



■現在、参加下さっている方の背景も
　まちまちで、変化の程度も著しく、

　普段、いる場所とはまるで異なる
　刺激を与えられ、

　スイッチを入れるきっかけとも
　なると思います。



■ツイッター、引き続きやってます。

　今日は

　--------------

　弱者の携えるべき武器は「言葉」。

　卓越した言葉の使い手は
　新たな現実の創造者となれる。

　（後略）

　--------------

　などの話を書いています。


　詳しくは

　【鮒谷周史＠弱者の生存戦略】
　https://twitter.com/shinkaron

　にてご覧ください。

　（ここから）
　-----------------


　この度も大変有用な音源を
　ご提供頂きましてありがとうございました。


　先日、仕事が終わったあと
　いつものように書店に行きました。


　入口にはアンデシュ・ハンセン著の

　「運動脳」

　という本が山積みになっていて
　パブロフの犬が如く反射的に
　すぐに手に取って読んでいました。


　「ふむふむ、脳には
　　やっぱり運動がいいのか～」

　とドーパミン垂れ流しで、
　ページをめくっている時に気づきました。


　「あれ、この内容、数年前に
　『スマホ脳』で読んだやつじゃね？」と。


　その時に鮒谷さんの声なき声が
　聞こえてきました。

　「知っているのと
　　やっているのとでは全然違う」

　結局、その本は買わずに家に帰って、
　しばらくお休みしていた
　ランニングをしました。


　もし、鮒谷さんの言葉を
　仕込んでいなかったら、

　きっと、また本を買って読んで満足して、
　知識だけを入れている状態に
　なっていたことでしょう。


　今回の経験で

　「まずはやってみる」

　ことの感覚を身体で学べたのは
　大きな収穫でした。

　この度もありがとうございました。


　（ヒラノ さま）


　-----------------
　（ここまで）



■改めてご感想ありがとうございます！

　まさにこういった変化を期待した
　音源なんですよね。


　今日の年間プログラムでも
　話をしていたのですが、

　身体＆行動に編み込めてこその
　人の変化であり、

　どれほど知識を蓄えても
　実践されなければ意味がありません。



■ここにおいて

　【そこに格闘はあるんか？】

　という問いが生まれるのです。

　格闘なくして、変容なく、
　変容なくして、変化なし。



■知識を頭に入力するために
　格闘は必要ありませんが、

　身体に埋め込むためには、
　
　「気持ちを込めなければ
　（＝気を入れなければ）ならない」

　のはお分かりかと思います。
　


■とはいえ、思春期くらいまでに、
　たとえば
　
　「前倒し習慣」
　「周りの人に役立とうとする気持ち」
　「いつも明るく、いつも元気で」

　といった姿勢や態度や精神が
　身についていたら、

　こんな取り組みは不要です。

　（そういう羨ましい人が
　　ときおりいらっしゃるんですよね）



■しかし、

　一定の年齢に達し、
　おおよそ人間形成ができたところで、

　後天的に上書きしようと思ったら、
　やっぱり知識を入れるだけではムリで、
　
　気持ちを込めて、丁寧に
　身体に覚え込ませなければなりません。



■つまり、

　勝ったり、負けたり、
　負けたり、負けたり、
　負けたり、勝ったり、
　　
　といったことを繰り返し、

　（こうすれば
　　人生が良くなっていくだろう）

　という知識を、

　知識に留めるだけではなく、
　身体に完全に編み込めるところまで

　【格闘し続ける必要がある】

　のです。



■継続して戦い続うのは大変ですから、
　そこから逃避する言い訳としての

　【学び（お勉強）】
　　　
　に思い当たる節のある方は　
　少なくないはずです。



■学んでいる限り、
　なにか取り組んでいる感じはする、

　（＝自分に許しを与えられる）

　わけですからね。


　しかし、学んでいる感じがしたところで
　実生活は何も変わりませんし、

　当人も心の深いところでは
　
　「そうじゃない」

　と気づいているはずです。

　そして実際に
　「そうじゃない」のです。



■当音源もそうですし、
　年間プログラムはなおそうですが、

　こうした

　【自分一人では矯正できない、
　　知識の編み込み（格闘）】

　の機会や環境を提供するために、
　
　ご用意しているコンテンツや
　サービスということになります。
　
　

■つまり当音源は、
　
　多くの人にとっての
　学びに対する概念を根本から覆し、

　「お勉強」から
　「身体に編み込むための学習」へと、

　パラダイムと行動習慣の転換を
　目的としてリリースしたもの。


　
■スケートリンクの上を滑るような
　（つるつるして定着しない）学びではなく、

　身体に編み込む
　（基本的な所作を確立する）学びを。


　これが同音源を通し、

　休憩なし、３時間半以上の時間を使って
　私（鮒谷）がお伝えしたかったことの、

　中心的なメッセージとなります。
