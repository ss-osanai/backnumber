■こんにちは、鮒谷です。


　先日、初めて個別コンサルにお越し下さった
　加藤さんより、

　以下のご感想を頂戴しましたので、
　紹介させて頂きます。


　※最近、音源のご感想紹介を日々、
　　たくさん頂戴していることもあって、

　　個別コンサル終了後に頂戴したご感想紹介が
　　全く追いついておりませんが、

　　今ももちろん、かなりの密度で
　　個別コンサルをさせて頂いております。


　（ここから）
　-------------------------------------


　鮒谷様、お世話になります。


　今回はじめて個別コンサルティングにて
　じっくりとお話しする機会を得、

　鮒谷さんの時間軸の長さと、
　日々のメルマガも将来のために積み重ねている
　企みの緻密さにまずは驚きました。

　自分はこれまで行き当たりばったりだったと
　実感しております。


　時間軸にて、10年20年或いは生涯、
　という長さを持たれているからこそ、

　逆算して今何をすべきか？に加え、
　陳腐化しない顧客への提供価値が何なのかを
　トコトン突き詰めて考え、

　今の形になっていることが垣間見れ、

　翻って自分自身のビジネスへどのように反映、
　落とし込んでいけるのか、

　考えるキッカケとなりました。


　特に

　「上に行く横に行く」

　の段は目から鱗で

　「上に行ってから横に行く」

　という順番も、

　言われてみれば当たり前なのですが
　自分自身、できていなかった部分でもあり、

　これからの半年で、新しいビジネスの構造に
　組み替えていきます。


　この課題に対し
　何をどのような順番でやっていくのか、

　一緒に考えて頂きまして明確になりましたので、
　あとはやるだけ。

　実践して成長した後の2回目の
　個別コンサルティングが今から楽しみです。


　引き続きご指導宜しくお願い致します！


　（加藤 武さま）

　（ここまで）


　加藤さん、ご感想ありがとうございました！
　

■加藤さんは事業で大成功されている方で、
　むしろ私がいろいろ教えて頂きたいくらいですが、

　ご感想を拝見させて頂き、改めて

　「成功者は謙虚である」

　という言葉を思い出しました。


　次回、お目にかかり、またいろんなお話を
　伺えるのを楽しみにしております！

　※先に頂戴したメッセージも、
　　ありがとうございました！

■また、以前より継続してコンサルさせて頂いている
　京都パープルサンガの久保さんからも、

　以下のご感想をお寄せ頂きました。


　（ここから）
　-------------------------------------


　現在の課題と、今後のありたい姿をお伝えして
　コンサルがスタートしました。


　自分が住んでいる小さな世界では、

　考えても考えても何も見えてこなかったところに
　希望の光を差していただきました。


　具体的には伏せますが、

　過去やったことがあり、
　結果が出ずにすぐ止めてしまった取り組みも、

　適切な「意味」を付与していただくことで
　強烈な意欲が湧いてきましたし、

　今はとにかく早く再開したいという欲求に
　駆られています。


　期限を定めることや、
　小目標の設定もお話ししてくださり、
　
　あとは自分次第というところまで
　一気に持って行くことができました。


　セイフティネットとして、同時並行で
　やるべきことも具体的極まりなく、

　余計なことを考えずに粛々と動いていけそうです。


　行動することや考え方に
　適切な意味づけさえできれば、

　見える景色が見違えるように変わるということも
　再確認できました。


　大目標が新たに設定されましたし、

　時々刻々の記も怠らず、
　物語の主人公として再出発いたします。


　現状、悩ましいことも多々ありますが、
　環境のせいにせず、全ては自責だと真摯に受け止めて、

　人間関係の構築に全精力を捧げて、
　生き生きと前進していきます。


　昨日はありがとうございました。
　次回コンサルではいい報告をさせていただきます。

　引き続き、よろしくお願いします。

【エレガントに、美しく】売れないのではなく、売っていない

───────────────────────────────────



■昨日は、ビジネスに携わる人間であるならば


　【お客さまの脳内に、

　　自社の商品・サービスの連想体系の
　　拡大・拡充を図る】


　ことに力を尽くす必要がある、
　そんなお話をいたしました。

　今日も続けます。



■私（鮒谷）であれば、
　もちろん、このメルマガを通じて


　【お客さまの頭の中に、

　　自社のセミナー・放談会・音源・個別コンサル、
　　といったサービスについての、

　　連想体系の拡大・拡充を図る】


　ことに、これ努めているわけです。



■結局のところ、


　【連想体系がお客さまの頭の中に根を下ろし、

　　それが「意思決定の基準」となるところまで
　　説明責任を果たす】


　ことがなければ、

　今のような商品過剰、情報過剰の世の中で
　購買して頂くことなどできるはずがない、

　ということです。



■よく、商売をされている方が

　「物が売れない、物が売れない」

　と言われているのをお聞きする機会が
　あるのですが、

　よくよくお話を聞いてみたら、
　あるいはサイトを見てみたら、


　【いやいや、売れないのではなく、
　　単に売っていないだけじゃないですか】

　と言いたくなることが、しばしばです。



■とはいえ、単に、

　声高に、がなるように、自社の商品について
　叫びまくればいいのか、

　というと、そういう話ではありません。


　あくまで、


　【お客さまが求められている価値を
　　お渡ししているうちに、

　　スムーズに、エレガントに、
　　気づけば連想体系が構築されていた】


　という状態を作り出すのが
　（真の）商売人の仕事です。



■もし、あなたが、
　
　このメルマガをここまで
　お読み下さっているとするならば、


　それは私（鮒谷）が無理やり、
　読め、読め、言って、

　読んで下さったのではなく、
　
　【あくまで自発的に、
　　ここまで目を通して下さった】

　ことは間違いありません。


　
■なぜなら私はあなたの傍らに
　いるわけではないので、

　読むのを止めようと思ったら
　いつでもそのようにできるわけだから。


　にもかかわらず、

　先の箇所から読み進め、さらに
　ここまで目を通して下さっていたとすれば、

　それはきっと（私がいうのもアレですが）


　【最低限、目を通すための使う時間よりも、
　　目を通したほうが役に立つ、

　　と思い、メルマガの内容に一定の価値を
　　見出して下さった】


　からこそ、限られた可処分時間を割いて、
　ここまで読み進められたのではないでしょうか。



■こうして（自発的に）読み進めていく中で
　徐々に


　【連想体系がお客さまの頭の中に根を下ろし、

　　それが「意志決定の基準」となるところまで
　　説明責任を果たす】


　ことによって、ようやく、自社の商品が
　購入対象に選ばれることとなるのです。



■こんな自明のメカニズムを意識することなく、
　ただただ、

　「買ってくれ、買ってくれ」

　といくら連呼したところで、


　お客さまにとって、そもそも

　【あなたや私から買う義理は何もない】

　わけだから、
　黙って去っていかれるのみでしょう。


　それでは到底、

　【エレガントに、美しく】

　モノやサービスを販売できるはずがありません。

　今日は短めに。
