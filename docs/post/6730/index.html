6730号 『徒然草』における、部分と全体の関係性

■長い間、日記を書き続け、

　日々の生活の中で起こる出来事に都度、
　意味を与えているうちに、やがて

　「人生における主旋律（テーマ）」

　のようなものが
　立ち上がってくるものです。



■これを

　【混乱、混沌の中から
　　秩序がもたらされる】

　とお伝えすることもありますが、


　生きる意味について
　問い続けているうちに、

　はじめは曖昧、漠然としたものが、

　ゆっくりと、しかし確実に、
　ある一方向に向かって、

　価値観や意思決定の基準が
　収斂されていくことになるのは、

　長く文章を書き続けてきた人であれば
　必ず感じているはずです。



■長編小説やノンフィクションの
　あとがきを読んでいると、

　前作を書き終わった後、
　次回作のテーマについて、

　何にするかを模索する中で、


　さまざまな資料に触れ、人と会い、
　思索を重ねるうちに

　「次のテーマ（今作の主題）が
　　徐々に立ち上がってきた」

　そんな風に書かれていることが
　ままあります。


　人生を一編の文学作品と捉えるなら
　それに近い感覚のようにも思われます。



■作家は

　「次回作のテーマを何に定めるか」

　そんな目的意識を持って
　日々を生きるから、

　「次のテーマ（今作の主題）が
　　徐々に立ち上がってきた」

　といったことが
　起きるのでしょうけれども、


　私たちであれば

　「生涯のテーマを何に定めるか」

　そんな目的意識を持って
　日々を生きていくうちに、

　浮き彫りになってくるのでは
　ないでしょうか。

　

■逆にいえば、

　そうした問いを持たなければ
　何も立ち上がってこない、
　
　という風にも言えそうです。
　　


■少なくとも私（鮒谷）は、

　自分が果たしてどこに向かっているのか
　まるで想像もつかなかった頃から、

　それでも、休むことなく

　「生涯のテーマを何に定めるか」

　について、

　日常的に日記を書くことを通して
　自問自答を繰り返してきました。


　※ドラッカーにとっての、
　　その問いは

　　「何をもって憶えられたいのか」

　　であったのでしょう。



■この「毎日の背伸びの過程」を通して、

　あたかも仏師が木の中に埋まっている
　仏像を取り出す、

　それと同じような感覚で、


　自分の中に眠っている
　本当に到達したい世界や姿が、

　ゆっくりと明確化されてきたのは
　間違いありません。

　

■ところで
　まったく話は変わりますが、

　『徒然草』を読むと各段はそれぞれ
　単体で内容が完結しているとともに、

　全体を通して明らかに、

　吉田兼好の世界感が深いところで
　全段に通底しているのが分かります。


　
■長く深い思索の末、ある境地に至り、
　その到達した世界から、

　見聞きした一つ一つの事象を眺め、
　考えたことについて語っている、

　私（鮒谷）にはそんな風に
　思われるのです。


　
■つまり、吉田兼好自らが
　長い時間をかけて掘り出した

　「吉田兼好その人の、
　　世界認識や判断基準」

　が『徒然草』という文学作品として
　昇華された、

　という風に味わっています。



■それと同じように、
　日記、あるいはこのメルマガも、

　お読みくださっている方からは
　どのように見えるか分かりませんが、


　自分の中では、

　いまだ吉田兼好の到達した世界ほど
　確実なものではないかもしれないけれども


　「日記あるいはメルマガ記事が
　　単体として独立しているとともに、

　　深いところで主旋律としての
　　基調テーマが立ち上がってきている」


　のがハッキリと分かるのです。



■まさに

　「自問自答を重ねる中で
　　事後的に立ち上がってきた」

　そんな風に感じられます。



■こんな風にして各人それぞれの中で
　生まれ、育ちつつある物語、

　これを仮に
　あなたや私（鮒谷）にとっての

　「神話」

　と名付けるなら、


　「その神話を体現する人物になる」

　と決めて
　人生を送れるようになれば、

　生きる意味を感じながら
　日々を過ごせるようになりますから、


　毎日が充実し、幸福感を
　味わえるようになるでしょう。


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】　　　
　　　　　　　　　　　　　　　　　　　　　
　■「生涯のテーマを何に定めるか」　　　　
　　　　　　　　　　　　　　　　　　　　　
　　そんな問いを持って自問自答を重ねれば　
　　自ずから、　　　　　　　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　　自らの価値観や大切にしたいもの、　　　
　　意思決定基準が立ち上がってくるだろう。
　　　　　　　　　　　　　　　　　　　　　
　■それが「神話の原型」となり、やがて　　
　　あなたや私にとっての信念、姿勢、態度、
　　さらには人生そのものへと昇華する。　　
　　　　　　　　　　　　　　　　　　　　　
　■このプロセスが幸福感と無関係であろう　
　　はずがない。
