　-----------------


　タイトルが

　「本気の恋音源」

　ということでしたが、

　恋愛関連の教材の販売は
　されないはずだよな、と思い、

　逆に一体どんな内容なのか
　気になってしまい、
　思わず購入いたしました。


　いい意味で見事に
　期待を裏切られる音源でした。

　特に、冒頭のノウハウを伝えても
　実践しない人の話は大変耳が痛く、
　汗顔しきりでした。


　結局のところ、自らの中に
　知れば何とかなるだろう、
　といったパラダイムがあり、

　知ることで満足してしまい、

　得た知識を用いて
　世界に対して殴りつけることを
　していませんでしたから、

　それでは当然、
　成果が出てこないわけです。


　知る→殴る→結果を確認し、
　そこから得たい結果と
　現状のギャップを把握し、

　必要な知識を得て
　さらに殴りつける、以下同様、、、

　ということで、
　小さく知って小さく試す、
　あたりから始めてみようと思いました。

　小知小試とでもいえばよいでしょうか。


　PDCAのようなものですが、
　私はPDCAという言葉に対して
　ネガティブなタグがつけられているため、

　音源を通じて、
　やっと自らに抵抗なくPDCAを
　行える言語を生み出すことができました。

　ありがとうございます。


　また、音源を聴いていて、
　本気である分野に没頭していた時期を
　思い出すことができました。

　あの感覚は何物にも代えがたいもので、
　それを他の分野でも味わえるものなら
　何としても味わいたいと思います。


　そのためには音源で言うところの
　本気で恋する対象を、となりますが、

　正直なところ、
　現在は本気に恋するような対象がないため、

　こういった小さいトライを繰り返し、
　出会いを増やすことで

　ガッツリと情熱を燃やし、
　没頭できるようなものを
　見つけ出していければと思いました。


　いつもながら、某被害者の会から
　リスクマネジメント、変態番付に至るまで、

　広範な知見を通じ、
　言葉を尽くしてご説明頂く内容も
　さることながら、

　すっかり消し炭のようになって、
　火種など微塵もないと思っていた当方に

　情熱を燃やす感覚を思い出させ、
　まだまだ心の火種が消えていなかったことを
　認識させていただいた、値千金の音源でした。


　ありがとうございました。


　（Ｋ・Ｙ さま）

■Ｙさん、いつも感想のご送付、
　ありがとうございます！


　私（鮒谷）は長い間、

　「学び、実践し、成果を出すプロセス」

　について探求してきましたが、

　【主体的、能動的でない学びは
　（ほとんど）意味がない＝結果を残せない】

　が結論となりました。



■義務感や、やらされ感から行う学びは
　歩留まりが悪いし、

　それゆえ実社会において、
　何の役にも立ちません。



■たとえば歴史を学ぶ際、

　（対象に興味を覚えることなく）

　教科書のゴシック体で記されている
　単語を丸暗記したところで、

　学びは定着せず、試験を通過したら
　即日忘れてしまいます。



■しかし、歴史そのものに対して、

　あるいは、

　ある時代、ある地域、ある人物、
　あるテクノロジー等に思い入れを持ち、

　そこから探究心を持って学びを広げ、
　深めていけば、

　学ぶそばから身についていきます。



■さらに、その学びを今の時代や
　自身を取り巻く環境に当てはめて、

　自身の頭で考えることによって
　見えてくることもあるでしょう。



■たとえば、冒頭に記した

　--------------

　▼新・映像の世紀　第３集
　　時代は独裁者を求めた

　-------------

　でも紹介されていた、

　ヒトラーは（合法的な）政権奪取の後、
　如何にして版図を拡大していったか、

　のプロセスなど、


　そのまんま、今の時代における

　「力による現状変更」

　の問題と地続きであることが分かれば、

　当事者意識を持って
　注視せざるを得なくなるし、

　そこからこれからの行動選択が
　変わってくることすらあり得ます。

　たとえば居住地や居住国を変えるとか
　資産の運用に反映されるとか。



■しかし世界史の教科書で

　「チェンバレン 宥和政策 ズデーテン（地方）」

　と丸暗記し、試験が終わり次第、
　即座に忘れてしまうような学びでは、

　能動的に世界と向き合うことは
　できません。



■これは一つの例に過ぎませんが、

　強い関心を持って
　ある対象と向き合い、学ぶのと、


　ほとんど関心もないけれども
　学ばなければならないから、

　くらいの気持ちで学習するのとでは、

　学びの深さも価値も10倍、100倍の
　差が現れるのではないでしょうか。



■これがまさに

　【好きこそものの上手なれ】

　ということ。


　もちろん食わず嫌いは良くないし、
　触れなければ分からぬこともありますが、

　軽く触れて、それで心躍らぬようならば
　速やかに立ち去るとよいでしょう。



■だから私（鮒谷）は早い段階から、

　好きになるべき（懸命に学ぶべし）と
　言われても、

　少し学んではみますが、
　それで好きになれなかった対象に、

　それ以上の時間を割くことは
　一切してきませんでした。



■たとえそれが

　「今の時代を生き抜くための
　　必須の知識である」

　と言われても、

　興味を持てないものに向き合うのは
　時間の無駄。

　時間は有限なのに
　学びたいことは無限にあるのだから。



■見切り千両、
　そんな思い切りの良さこそが、

　これまでの意思決定の中で
　最も大きな報酬をもたらしてくれた

　と考えています。



■それを恋に見立てるならば

　【本気で恋をした相手にコミットせよ】

　ということになりそうです。


　一途に恋した（関心を持った）対象に
　コミットすれば、

　恋した相手は様々な形で
　報いを与えてくれるもの。



■そのあたりの機微について
