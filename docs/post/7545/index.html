　良い買い物をさせていただきました。

■話を戻しますが、

　『先入観コントロール音源』

　において
　本質的にお伝えしていることは、

　【コミュニケーションの成否は
　　コミュニケーションが開始される前に
　　既に決まっている】

　ということなんですよね。



■未来にプレゼント音源風に言うならば

　【未来の運命は
　　未来が訪れる前（すなわち現在の時点で）
　　ある程度まで方向づけられている】

　のと、感覚的には似たように
　思われます。



■もちろん両者とも、
　完全にコントロールはできませんが、

　現在の振る舞いが、未来の運命と
　無関係であるはずがありません。



■要するに、

　運命（人生）そのものが
　そうであるのと同様、


　コミュニケーションであれ、
　営業であれ、交渉であれ、

　どんな入り方でご縁が紡がれていき、
　どんな方向性で着地するかは

　【９割がた、
　　入り口（向き合い方）で決まるもの】

　なんですよね。



■もちろん、だからといって

　最初に想定していた
　コミュニケーションの形で

　「決め打ち」

　的に会話をはじめて、
　そこから逸脱することを許さない、

　みたいな硬直的なやり取りになっては
　本末転倒です。

　もっと柔軟であるべきでしょう。



■とはいいつつも、

　事前にリサーチを重ね、
　仮説を立てておき、

　その上で、対話を通して柔軟に
　話の展開を組み替えていく

　（と同時に、事前のリサーチならびに仮説と、
　　想定問答集は常に意識にとどめておく）

　ことが大切です。



■こうしたことがどれほど大切であるか、
　お互いにとって役立つか、

　を理解できていなければ、

　「出たとこ勝負の、成り行き任せ」

　で会話をスタートさせてしまい、


　イメージしていた展開とは異なる、
　消化不良的な流れになる可能性が、

　どうしても高くなるでしょう。



■現時点において、

　そんなコミュニケーションスタイルに
　近い形の方があったとするならば、

　その認識や態度、振る舞いに
　抜本的なメスを入れることで、

　あっという間に仲良くなったり、
　関係性が深くなる方向性で、

　ご縁を紡げるようになりますよ。



■ぜひこのアプローチを学び、

　さらには実践し、コミュニケーションを
　重ねることで豊かになる実感を得られる、

　そんな経験をして頂ければと
　願っています。
