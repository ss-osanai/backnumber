　【　巧遅より拙速を尊び、60点主義で即決する。 】

───────────────────────────────────


■先日もご紹介させていただきましたが、

　故・土光敏夫さんのご著書「 [新訂]経営の行動指針 」の中に
　こんな一節があるのを改めて読み返していて見つけました。

　それは、

　「60点主義で即決せよ」。

　という言葉。（これも改めてインストールしておきたい言葉だと思いましたが）


　【今日のお勧め本　[新訂]経営の行動指針】
　 ⇒ http://tinyurl.com/8lm6j


■「60点主義で即決せよ。

　　決断はタイムリーになせ。

　　決めるべきときに決めぬのは度しがたい失敗だ」


　とまで書かれています。


■この「平成・進化論。」でも、たびたびお伝えしておりますように、

　巧遅よりも拙速を尊（たっと）ばねばなりません。

　すなわち、いくらたくみに物事を行ったとしてもそれがタイミングを失して
　いては意味がありません。
　
　拙くても、精度は高くなくても、早い動きで行動することが重要です。


■100％の完璧を期して、石橋を叩きすぎて動きが遅くなるよりは、

　手数を増やし、次から次へと行動していくほうが、長い目で見た
　ときにははるかに多くの結果が出せる。

　私は、そんな信念を持っています。


■変化の速度が遅い時代においては、じっくり考えて行動に移すことも
　よかったのかもしれません。

　ところがこれだけ変化の速度がアップしてくると、
　じっくり考え、さあ行動に移そうと思ったときには、

　その前提となる事業環境自体が変わっていることが往々にしてあります。


■事業環境に即応し、思いついたなら、即、行動を起こす。


■即断即決。巧遅よりも拙速を尊（たっと）ぶ。

　その姿勢こそがビジネスに成功をもたらすひとつのカギなのではないか。


■ことのことを、土光さんの本の中では

　「60点主義で即決せよ」

　と言われているのかなと思います。


■重要なことは、【決定】【決断】をスピーディーにすること。


■決断をすれば必ず行動に移るわけです。

　行動に移すにあたって、「決断ができないこと」が一番のネック。


■完璧を求めて長い時間をかけるよりも、

　ある程度、最低限のところまで煮詰まったのであれば実際に体にかけて
　動いてみる。

　それが、流れの速い現代の事業環境においては非常に重要なことでは
　ないでしょうか。
