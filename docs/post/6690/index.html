6690号 目指す世界は明確なのに、そこに至る道筋が分からず折れてしまう人へ

■これまで接してきた、
　少なからぬ方に見受けられる

　「典型的な例」

　は以下の通り。



■ある時点まで、

　自分がどんな人間になりたいのか、
　どんな世界を生きたいのか、
　どんな生活を送りたいのか、

　明確でなかった人が
　なにかのきっかけで、目覚め（？）られ、

　高揚感が高まり、やる気が生まれ、
　猛烈に行動し始める。



■しかし、

　騎虎の勢いで走り始めたものの
　目指す世界に至るまでの道筋が見えず、

　少しの間、試行錯誤を繰り返すも
　そのうちエネルギーが枯渇し、

　元の木阿弥となってしまう。



■誤解を恐れずに言うならば、
　たとえば

　「目標設定音源」

　をお聴きくださった方の中には
　上記現象（？）に思い当たる節のある方も、

　いらっしゃるかもしれません。



■目指す世界
　（特にBeingレベルでの有りたい姿）

　に到達するためには、まずその状態を
　イメージするのが大事です。


　ただし、同時に

　「そこに至るまで堅持しておきたい
　　思考＆行動様式」

　があるのです。



■つまり、目標に近づくためには
　先に記したような

　「騎虎の勢いで走り始めたものの
　　目指す世界に至るまでの道筋が見えず、

　　少しの間、試行錯誤を繰り返すも
　　そのうちエネルギーが枯渇して、

　　元の木阿弥」

　となりかねない危機を乗り越え、


　目指す世界に至るプロセスが
　ハッキリしない中でも、

　動かずにはいられない人で
　あり続けられるように

　「思考＆行動様式を書き換える」
　
　必要があるということ。



■違う言葉でいうならば

　「パラダイムを入れ替える」
　「世界認識の書き換える」

　必要があるのです。


　こうした危機に陥る方に対して、

　私（鮒谷）が年間プログラムや
　個別コンサルを通して、

　行ってきたことはまさに
　「これ」だったといえるでしょう。



■その効果・効能や再現性には
　かなりなものがある、

　という自負はありましたが、


　そんな私（鮒谷）が
　先の状態に陥っている人に対して

　「何が何でも、これを勧めたい」

　と心の底から思ったのが
　エフェクチュエーションでありました。



■（あえて具体的には
　　語ってきませんでしたので）

　いまだ
　
　「エフェクチュエーション」

　が何か、ハッキリしない方もあると
　思いますが、
　

　一言でいえば
　これは知識ではなく、

　「身体に刻み込ませたい知恵」
　
　と言えるでしょう。


　
■もちろん最初は知識から入る
　（知識からしか入れない）わけですが、

　知識から得た思考や身体技法の反復に
　よって知恵化することで

　「目指す世界は明確だけれども
　　そこに至るルートがまるで分からない」

　という不安や心配を乗り越え、
　走り続けられるようになるのです。



■エフェクチュエーションを単に

　「知識」

　として理解するのみならず
　ガチンコで身体に覚え込ませて

　「知恵化」

　を図っていただく。



■こうして実践をより確実なものと
　するために

　「エフェクチュエーション勉強会」

　参加後に、希望される方には
　
　「応用＆実践編」

　もご用意しています。


　
■もちろん参加は任意ですし、
　一切の営業的なものは行いません。

　勉強会に参加くださった皆さまに
　シンプルな案内文のみお送りします。

　「各種音源」みたいな売り方は
　まったくしておりません（笑）



■そんな生暖かい（生ぬるい）案内で
　あるにも関わらず、

　前回参加者のうちの多くの方が
　「応用＆実践編」に申し込み下さったのは、


　まさに、未だ満たされていない

　（皆さまも気づかれていなかった）
　真の欲求に気付かされ、

　そのニーズを満たしてくれるのが

　「エフェクチュエーション」

　だと理解されたからであろうと
　確信しています。



■既に相当の実績を残してこられて

　「こんな人までが！」
　
　と思われる方まで
　（むしろ、そんな方こそ）

　お申し込みくださっているのも、

　この講座の一つの特徴として
　挙げられるかと思います。



■今月15日に開催する
　勉強会に参加なされれば、

　ここまでに記してきたことの意味を
　すべてご理解頂けるに違いありません。



■前回の勉強会

　（今、ご案内しているものと
　　同一の内容となります）

　に参加くださり、

　迷妄を破られた方々からのご感想を
　お読み頂ければ、

　ここまでの話をご納得頂けるのでは
　ないかと思います。


　※時間がとれず、編集しきれていませんが
　　暫定的に50ページ強にまとめています

　　さらっとでも目を通してみて下さい（↓）
　https://ssdlfiles.s3.amazonaws.com/testimonials_effectuation.pdf

　　参加しないことの大きな機会損失に
　　思いを致されるに違いありません。



　前回は150名ほどの方が参加され、

　（まるで営業らしいことはしていないにも
　　かかわらず）

　その中の50名以上の方が
　「応用＆実践編」へと歩を進められている、

　その内容の深さや重さに今一度、
　思いを致して頂ければ幸いです。


　今日も人生とビジネスを楽しみましょう！
