■ボーリングで優良鉱脈を探し当て、
　採掘し尽くすまで掘り続ける

───────────────────
　

■上述の通り、

　ゴールデンウィーク中は
　一日も欠かすことなく、

　毎日３件程度の個別コンサルに加えて、
　数件の打ち合わせやズーム面談。


　おかげさまで普段と変わらぬ　

　「巡航速度」

　をキープできています。

　

■本日のお客さま
　（Ｋさん/年間プログラムご受講中）

　からは、
　以下のご感想を頂戴いたしました。
　

　（ここから）
　---------------


　本日は個別コンサルティングを
　ありがとうございました。


　2022年12月に目標設定コンサルティングを
　受けさせて頂いてから、
　２回目の個別コンサルティングでした。


　私は会社員で、12月の
　目標設定コンサルティングでは、

　仕組み作り、
　組織作り、
　横や右斜め上下との関係づくり、

　の３点のアドバイスを頂きました。


　この４ヶ月間、

　会社でその３点を意識して行った結果、
　予定より３年早く昇進し、

　成果を形に残すことができました。


　今回の個別コンサルティングでは、

　会社勤めが昇り調子の時に、
　このまま惰性で進むのがもったいなく、

　もっと加速させるために
　できることがないのか、

　というところを相談させて頂きました。


　また調子が良いので
　このまま会社員として突き進んで良いか、

　というところに不安がありましたが、

　「勝っているところで勝った方が勝てる。
　　それができる人はそこで勝ったらいい。」

　と背中を押して頂きました。


　背中を押して頂いただけでなく、

　次に勝ち上がるためのキモとなる
　部分である、

　社内外のネットワーク作りと、
　それを見せること・使うこと・繋ぐことの
　重要性についての説明は、

　私にとっては目から鱗の話でした。


　全ての学びは前倒しですが、

　今日この話を聞いたことにより、
　10年後の価値は10倍どころではない、

　と確信しております。

　お薦め頂いた本も早速購入しましたので、
　届き次第読んでいきます。


　また、番外編でしたが、
　家族の相談にも親身になって
　アドバイスを頂きありがとうございます。

　選択肢の一つとして
　検討を進めていきます。


　2023年1月から
　年間プログラムに参加させて頂いてからの
　初めての個別コンサルティングでしたが、

　メルマガ、
　音源、
　年間プログラム、
　個別コンサルティング、

　を全て使うことでの相乗効果は
　想像の遥か上でした。


　例えば、４つの楽器を一緒に演奏する
　カルテット（４重奏）のようなイメージです。

　一つ一つの楽器でも音楽は成立しますが、
　４つの楽器が重なると全く次元の違う音楽が
　完成するといった感じでしょうか。


　今回はその４重奏を
　初めて体感した感動でいっぱいです。

　本日はありがとうございました。


　（Ｍ・Ｋさま）

　---------------
　（ここまで）



■Ｋさん、今日は（も）
　ありがとうございました！

　カルテット（四重奏）のたとえ、
　最高の褒め言葉です＆嬉しいです。

　これから勝手に使わせていただこうと
　思います（笑）



■それはさておき、

　Ｋさんは日本の誇る、
　超優良企業にお勤めの会社員として、

　大きな成果を挙げられ、
　上層部から嘱望されている幹部候補生。



■そんなＫさんに今日、お伝えしたのは

　【勝てる場所を見つけたら
　　持てる資源をそこに集中投下する】

　こと。

　

■多くの人が

　【勝つべくして勝てる場所さがし】

　に難渋している中で、


　せっかくそうした場所を
　見つけたにもかかわらず、

　わざわざ勝ち目が薄かったり、
　そこまで高い期待値が見込めない、

　そんな場所に突っ込んでいってしまう
　ケースが散見されます。



■おそらくは、

　勝ち続けるうちに刺激が乏しくなり、
　それが原因で飽きてきたり、

　あるいは、
　
　「隣の芝生は青く見える症候群」

　によって、
　他の場所に魅力を感じたり、
　
　という錯覚が生じるからでしょう。


　※かくいう私（鮒谷）も、
　　その落とし穴にハマったことがあります。
　


■そのような内在的傾向を知った上で、

　確実に勝てそうな場所を見つけたら、
　浮気したくなる誘惑に打ち勝ち、

　【その場に眠っている資源が
　　枯渇するところまで掘り続ける】

　ことが大事です。


　
■埋蔵資源がまだまだ眠っているのに
　採掘をいったん中止し、

　（あそこにも鉱脈がありそうだから）
　
　と、

　勝てるかどうかも分からない場所で
　わざわざボーリング調査を行うのは、

　あまりにもったいないと言わざるを
　得ないでしょう。



■にもかかわらず、

　それをやってしまう人が案外、
　少なくないんですよね。

　かくいう私（鮒谷）も
　その中の一人だったわけですが。



■今でこそ、そうしたことも
　分かるようになりましたが、

　はじめて金鉱脈を掘り当てたときには、
　全能感に満たされ、

　他の場所でもいくらでも当てられる、
　などという勘違いも生まれがちなので、


　その錯誤を戒めるべく、

　【勝ちパターンを
　　ひたすら繰り返しましょう】

　という話をＫさんにいたしました。

　

■もちろん、なかには、

　どんな場所に行っても
　成果を必ず出せる、

　そんな人もあるかもしれませんが、


　私（鮒谷）も含めて、

　おそらく、ほとんどの人は、
　（残念ながら）そうではなく、

　【ここだからこそ勝てる
　（＝ここでしか勝てない）】

　ということなのではないでしょうか。



■こうした現実を直視し、

　一発、鉱脈を掘り当てるだけでも
　大変なことなのだから、

　【鉱脈を大切に掘り進め、必勝パターンの
　　さらなる磨き上げに注力する】
　
　ことが大事だと思います。



■要するに、

　キャリアやビジネスにおいて
　心がけるべきは

　---------------

　１、ボーリングで優良鉱脈を探し当てる
　　（勝てる場所を見つける、所を得る）

　２、鉱脈を見つけたら枯れるまで掘り続ける
　　（勝てる場所に資源の一点集中）

　---------------

　であるということ。



■Ｋさんにはそんな話をした上で、

　さらにそこから先についての
　あれこれや参考書籍をお伝えしましたが、

　せっかくなので、その内容の一部を　
　こちらでシェアさせてもらいました。


　Ｋさん、ますますのご活躍も
　祈念しております。

　ご家族のことも良い形で
　着地なされることを願っています。



■、、、そんなわけで、よろしければ、

　あなたもぜひ、個別コンサルを人生や
　ビジネス、キャリア構築に投入することで、

　【二重奏、三重奏、四重奏】

　を体感頂ければ幸いです。

　
　※上記ご感想において、Ｋさんは
　以下のようにおっしゃって下さっております。


　（ここから）
　---------------

　2023年1月から
　年間プログラムに参加させて頂いてからの
　初めての個別コンサルティングでしたが、

　メルマガ、
　音源、
　年間プログラム、
　個別コンサルティング、

　を全て使うことでの相乗効果は
　想像の遥か上でした。


　例えば、４つの楽器を一緒に演奏する
　カルテット（４重奏）のようなイメージです。

　一つ一つの楽器でも音楽は成立しますが、
　４つの楽器が重なると全く次元の違う音楽が
　完成するといった感じでしょうか。

　---------------
　（ここまで）



■たった一度、
　ないし数度のコンサルだけで、

　行動の筋を良くして（精度を上げて）、

　以降の見当外れの努力や、
　意味のない試行錯誤を丸ごと省けるのなら、

　実に費用対効果の高い投資になると
　個人的には思っているんですけどね。
