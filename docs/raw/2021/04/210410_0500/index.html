■「微差を積み重ねて絶対差」の、
　もう少しだけ詳しい解説

───────────────────



■先日来、ご案内してきた


　「人生は鉄火場（賭場）と考えると
　　うまくいく」放談会音源

　～鉄火場に学ぶ、リスクを極小化しつつ、
　　安定・逓増人生を送るための７つの心構え


　ですが、あと少しで価格改定（値上げ）
　いたします。



■ここまで時間をかけ、いろんな角度から
　ご説明してきた当音源の内容ですが、
　
　とはいえ、タイトルからして
　中身が分かりにくいですよね、、、



■なんとなくでも音源のイメージを
　付けていただきたく、


　当社が音源を
　リリースするにあたって

　「インデックス（目次）」

　作成に用いている、


　音源内容の要約、把握、管理するための
　社内テキストから抜粋引用して

　（なので正確な文字起こしでは
　　ありませんので、念の為）


　以下に内容チラ見せ？の、

　「コンテンツ一部紹介」

　を行いますので
　ぜひ読んでみて下さい。



■以下、音源の一部チラ見せでの
　ご紹介となります。


　（ここから）
　-----------------------


　「人生＝ギャンブル」

　のメタファーが有効な局面と
　そうでない局面、

　いずれも、もちろんありますが、
　以下、有効な局面についての話。


　ギャンブルも人生も

　「手金（手持ち資源）が尽きたら、
　　その時点で終わり」

　であるから、

　「決して死なない」

　ことを大前提として、

　短期的勝利よりも長期的勝利を
　追い求めることが重要。

　
　-----------------------


　多くの人は感情に振り回されて
　自分の置かれている状況を俯瞰できず、
　　
　正しい意思決定をしているつもりで
　その実、自覚なく、

　（大振りするバッターのごとくに）

　運に任せた短期的勝利を狙ってしまう
　傾向にある。


　-----------------------


　それで一時的にうまくいったとしても
　その思考や姿勢や態度が

　「勝つための意思決定」

　にまで昇華してしまうと、

　短期的に勝つことはあっても
　長期的には必ず退場を迫られる。


　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------
　

　長期視点を持って、

　長くプレイし続けることを
　第一目的にするならば、

　
　避けて通れないのが、

　「勝率、期待値、確度」

　といったキーワード。
　
　早い話が「勝ち目」の話。


　結局、長く生きていたら、
　短期的な浮き沈みはあっても

　「最終的には私たちが行った行為すべてに
　　ついての期待値に収斂していく」

　こととなる。


　-----------------------


　人生においても長く生きていると、
　
　その行為が一般的、平均的に
　もたらしてくれるであろう結果

　（便宜的に、これを期待値と呼びましょう）

　に収束していくこととなりますが、

　より鮮明なイメージをつけてもらうためには
　ポーカーなどのゲームを想像して下さい。


　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------


　（前略）


　場合によってはプロが素人に負けることも
　あるかもしれないけれども、

　長い間、続けていくと、

　最終的には実力に応じた結果に
　近づいていくこととなるはずです。


　-----------------------


　つまり、長期視点を持つだけではなく、

　一つ一つのプレイ（一挙手一投足）の
　勝率を引き上げなければ、

　トータルで勝つことはできません。


　ここでいう

　「勝率の高いプレイ」

　を人生において約束してくれるのが
　私（鮒谷）がよく使う言葉で言うならば、
　
　「質の高い意思決定」

　ということになります。


　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------


　質の高い意思決定基準というのは、

　すなわち高い勝率（期待する結果）を
　もたらしてくれる行動規範、

　と言えます。


　-----------------------


　人生まるごとの勝率を
　一気に引き上げるのではなく、

　一つひとつの行動の精度を高めるからこそ
　長期的、全体的な勝率を引き上げられる

　（人生でいうならば
　　QOLを引き上げられる）

　ことを目指すべき、
　
　というのがポーカーゲームの
　メタファーから導き出せるのでは。
　

　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------


　勝率引き上げに寄与する
　
　「質の高い意思決定基準」

　には、実は、二種類あり、


　一つは（中略）、すなわち（中略）、
　があるだろうと思われるものです。

　これは基本的に（中略）と思われます。


　-----------------------


　もう1つは（中略）です。

　（中略）など、例えばビジネスであれば、
　（中略）ということがあります。

　たとえば（中略）、
　あるいは（中略）、
　そういった（中略）、

　たとえば（中略）
　あるいは（中略）、

　いずれの意思決定基準も
　引き上げることが必要ですが、

　単発では成果は挙げられません。


　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------


　複数の（より正確には相当数の）、
　質の高い意思決定基準に裏付けられた行動、

　を長期に渡り、
　継続するからこそ、

　私たちは長い目で見たときに
　QOLを逓増的に引き上げられる、

　このことを理解していない人の
　なんと多いことか。


　-----------------------


　だから多くの人は辛抱できないし、

　一攫千金的な魅力あふれるオファーに
　引き寄せられては、

　自ら地雷を踏みに行ったり、
　飛んで火に入る夏の虫化してしまう。


　こうして、、、

　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------

　
　、、、以上、音源出だしのチラ見せです。
　続きは音源でお確かめ下さい。




■以上はあくまで、

　音源で語っている内容の
　ごくごく一部ですが、

　【長期で見たときの、
　　人生における勝率】

　を高めたい方にとっては
　必聴と言える音源です。
