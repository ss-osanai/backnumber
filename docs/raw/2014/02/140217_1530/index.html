＜今日のピークパフォーマンス方程式＞

 【　ついつい相手を「断罪」してしまいがちな、あなたへ　】

───────────────────────────────────



■コミュニケーションを図るにあたって起きるトラブル
　（主として怒り・悲しみ）は、

　「双方の期待値の相違」

　から起こるもの。
　
　
　人は、大抵の場合、
　
　「自分から相手への期待値を高く持ち、かつ、
　　相手から自分への期待値を低く見積もるもの」
　
　であるようです。
　
　
　
■ここから生まれる、お互いの認識ギャップによって

　「憤り」や「失望」、「幻滅」、「悲しみ」
　
　といった感情が生起することとなります。
　
　
　このような気持ちを抱いて不快な気持ちとなり
　損をするのは、他ならぬ自分。
　
　
　
■このようなことを何度も繰り返し、
　ある時、はたと気づきました。
　
　
　そんなことなら初めから、
　
　
　「相手にかける期待値をできるだけ低く見積もり、
　
　　同時に、相手が私に期待しているであろうことを
　　できるだけ高く見積もる」
　
　
　ようにしてはどうかと思ったのです。
　
　
　
■加えて、相手と頻繁に会話を行うことによって、
　
　その人が大切にしている価値観（や地雷？）などに
　ついての、おおよその

　「相場観」

　を持っておくことも大切です。
　
　
　それだけ行っても、互いの見積もりに行き違いがあり、
　問題が起きた時には、それを契機として、
　
　
　「期待値コントロールの技術や、観察力の乏しさを自覚し、
　　鍛える余地がまだまだあることに意識を向ける」
　
　
　ことによって、成長、向上するきっかけとすればよいでしょう。
　
　
　
■コミュニケーション上の齟齬が生まれ、
　相手に対して、
　
　「ふざけるな、アホじゃないか」
　
　と思っている時には、
　
　きっと相手も同様のことを思っているのです。
　
　
　
■人は、それぞれの世界に生きています。

　そして、それぞれの世界に優劣はありません。
　
　
　自分が価値判断の絶対的な基準を持っており、
　それが正当であると信じているからこそ相手を
　
　「断罪」
　
　しようとするものですが、
　
　そもそも、その前提が間違っている可能性に対して
　疑いの目を向けてみてはどうでしょう。
　
　
　
■そもそも、

　自分の見ている世界が最も正統かつ正当で、
　論理的で、常識的で、妥当、絶対的な正義である、
　
　と思う根拠などどこにもないのです。
　
　
　相手の言うことをよく聞かず、言葉のキャッチボールもせず、
　相手を観察することもせず、慮りもなく、
　
　「自分絶対」
　
　の立場から言葉を発したところには、
　険悪なムードや喧嘩しか生まれてくることはありません。
　
　
　
■以上は、コミュニケーションに悩み続け、
　今も修行中の身である私（鮒谷）が、
　
　コミュニケーションについて深く考え、自分なりに

　「コトが起きる前」
　
　すなわち、普段から、気をつけていることであります。
