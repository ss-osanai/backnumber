■今日のメルマガの件名にあげたとおり、


　「時代がどこに向っているのか、目線を上げて確かめる」


　ことを、常に意識していきたいと考えています。



■先日、

　ピュリツァー賞を３度受賞した名ジャーナリスト、
　トーマス・フリードマンの


　【今日のお薦め本　グリーン革命(上)】

　　　　　　　　　 トーマス・フリードマン (著), 伏見 威蕃 (翻訳) 

　 http://amazon.co.jp/o/ASIN/4532314410/2ndstagejp-22/ref=nosim


　【今日のお薦め本　グリーン革命(下)】

　　　　　　　　　 トーマス・フリードマン (著), 伏見 威蕃 (翻訳) 

　 http://amazon.co.jp/o/ASIN/4532314429/2ndstagejp-22/ref=nosim


　を読んだのですが、
　
　この本を読んで、明らかに10年前、5年前、いや２年前と
　比べても、
　刻一刻と時が流れ、時代が変化していることを感じます。



■同氏の、時間・空間を俯瞰する視点、
　あるいはその先見性や、巧みな論理展開は



　【今日のお薦め本　レクサスとオリーブの木
　　　　　　　　　　　　　　 ―グローバリゼーションの正体〈上〉】

　　　　　　　　　 トーマス・フリードマン (著), 東江 一紀 (翻訳) 

　 http://amazon.co.jp/o/ASIN/4794209460/2ndstagejp-22/ref=nosim

　　　　　　　　　　　　　　 ―グローバリゼーションの正体〈下〉】

　 http://amazon.co.jp/o/ASIN/4794209479/2ndstagejp-22/ref=nosim


　で示されたとおりのグロバリゼーションの実態、
　あるいは、



　【今日のお薦め本　フラット化する世界 [増補改訂版] (上)】

　　　　　　　　　 トーマス・フリードマン (著), 伏見 威蕃 (翻訳) 

　 http://amazon.co.jp/o/ASIN/4532313775/2ndstagejp-22/ref=nosim


　【今日のお薦め本　フラット化する世界 [増補改訂版] (下) 】

　　　　　　　　　 トーマス・フリードマン (著), 伏見 威蕃 (翻訳) 

　 http://amazon.co.jp/o/ASIN/4532313783/2ndstagejp-22/ref=nosim



　で記された、世界の潮流などを読むことによって
　以前から感じていましたが、

　あらためて、「グリーン革命」に目を通し、
　その感を強くしました。



■詳細については同書をご覧頂きたいのですが、


　▼地球上におけるエネルギーの需給バランスの変化
　
　▼それに伴う政治上の力学の変化

　▼気候変動

　▼新しいクリーンエネルギーシステム

　
　などについて触れられています。



■いつも書いていることですが、

　私たち個人個人の生活と、
　地球、社会、政治、経済といった、私たちを
　取り巻く環境とは、密接不離の関係にあります。



■個人の生活と「無関係」ではないからこそ
　これらのことに「無関心」であってはならない、
　そう思います。


　目の前のこと（仕事等）だけに向いていると
　なかなか、広い視点を持ちにくいもの。

　ときに、このような世界全体を捉えるような書籍に
　意識的に触れることで、普段とは異なった目線を
　持つようにしたいものです。
