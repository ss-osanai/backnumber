【（あと13時間）プロじゃなくても売っていい、プロじゃないから売っていい】

2018/02/09 09:30 
■こんにちは、鮒谷です。


 昨日のメルマガでは

　「自分の経験を知識やノウハウとして
　　パッケージ化し、多くの人に届ける」

　ことには大きな価値があると
　お伝えいたしました。


　聴かれる方は自らの試行錯誤を省き、
　ショートカットできるし、

　販売者側はこれまでの労苦が
　売上という形で報われ、

　両者ともにハッピーとなるからです。



■だからこそ、

　もし今、あなたが価値あるコンテンツを
　お持ちであるならば、

　ぜひとも音源ビジネスを始められると
　良いのでは、

　そんなお話をしたわけです。



■ここまでお読みくださった方の中には、

　すでにご自身のコンテンツ販売に挑戦してみようと
　思われている方があるかもしれませんし、

　あるいは早い人だと、もう既に音源を聴いて、
　早速、取り組み始めているよ、

　という方もあるかもしれません。



■ただし、ここで注意しなければならないのは

　「マスターベーション的な
　　コンテンツ提供」

　であってはならないということです。


　仮に自分が自信を持っているコンテンツを
　お伝えするのであっても、

　ただ単に

　「私が伝えたいことを伝える」

　のでは、お客さまに価値は届けられません。

　（というか、クレームになります）



■私（売り手）が話したくなる内容と、
　お客さまが求められている内容は、

　「（大抵の場合）異なる」

　ことを理解しておく必要があるでしょう。

　ですから私は当音源において

　ということを強調しています。

　実際にそうなのですが。



■音源販売をこれから始めたいと考えられている
　多くの人（＝ど素人さん）は

　「私になんかできるのかな？」

　「なんだかやることがいろいろありそうで
　　難しいかな」

　と思っているわけですから、


　そうした方に対して、
　価値をお渡しするためには

　「勇気と希望と一歩を踏み出す具体的な方法論」

　をお渡しし、背中を押すことが一番大事です。



■だから、

　高度で高尚で格好良く、難易度は高いかも
　しれないけれども売上につながりますよ、

　みたいな話をしても、

　「そりゃそうかもしれないけれども、
　　俺には（私には）できないわ」

　みたいなことになっても仕方ありません。



■、、、ということをさらに深く考えると、

　コンテンツ提供者は、ぶっちゃけ

　「ガチのプロでなくてもいい」

　という結論が導けるかもしれません。


　たとえばあなたがスキーを滑ったことのない
　ど素人さんとして、

　スキー競技のオリンピック選手に指導を
　求めたいと思いますか？

　という話。



■ほぼすべての人は

　「いやいや、
　　そこまでのレベルは求めてないっすよ」

　となるのではないでしょうか。


　それよりもちょっと上手で、親しみやすく、

　それなりに教える技術を持っているおにーちゃん、
　おねーちゃんから教わったほうが、

　かえって楽しく取り組め、
　上手になれたりするものです。



■つまり、

　既に一定の経験があり、プロレベルの
　コンテンツ販売をしたいと思っている人は、

　そういう人に話を聞きに行けばいいし、


　音源販売は全く未経験、というど素人さんが
　コンテンツ提供したいと思うなら、


　「私（鮒谷）のような、ど素人に毛の生えた
　　ような人間が、音源販売を開始し、

　　一生懸命、試行錯誤したプロセスと、
　　そこで得られた知見をお伝えしているコンテンツ」


　のほうが案外、役に立つのかもしれません。



■どちらが良いとか悪いとかではなく、

　お客さんのレベルやニーズによって
　プロレベルのコンテンツを求められることもあれば、

　そうではなく、別の価値が求められることも
　ある、ということです。



■こんな風に考えたら、

　（あなたがそれぞれの世界において、
　　それなりに頑張ってこられたならば）

　「俺でも、私でも、自分のコンテンツを
　　販売するのもありかもしれない」

　という風に、気持ちも楽になるのでは
　ないでしょうか。


　そして実際に、

　とはいえ、それなりに、それぞれの世界で
　頑張ってこられたあなたは

　「売ってもよいし、
　　売る資格もある」

　のです。
