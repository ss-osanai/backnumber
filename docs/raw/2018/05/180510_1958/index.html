【進化論】「戦いすんで日が暮れて」ゆったりしている場合じゃない
【進化論】最高の人生＝人生そのものをエコシステム化する




【実質無料でご招待】「丸一日、本当に大切なことだけに集中する会（第6回）」5月22日（火）に開催。

2018/05/10 19:58 
■こんにちは、鮒谷です。


　最近、強く思うのが、

　人生において最も高い生産性（＝波及効果）を
　手に入れるためには

　【本当に大切な、
　　少数のことに取り組む時間】

　を大切にすべきであるということ。



■ここでいう

　【本当に大切な、少数のこと】

　とは、


　【緊急ではない（＝今すぐ、取り組まなくても
　　さしあたって問題はない）けれども、

　　先送りしていると、徐々に不幸を招き寄せる、
　　人生において重要な取り組み（＝第二領域）】


　のことを言います。
　


■にもかかわらず、
　私たちは往々にして

　【第一領域（緊急かつ重要な要件）】

　あるいは、

　【第三領域（たいして重要ではないけれど
　　急ぎの案件＝割り込みの電話やメール返信等）】

　に追われがちで、

　【真に大切なところに時間を割く】

　ことがなかなか困難です。



■それに加えて、

　「今やるべきことは、これではない」

　と思いながらも、

　分かっているけど止められず、
　惰性でスマホを見てしまったり、

　（＝緊急でも重要でもない、
　　　第四領域）

　などという人も少なくないのでは
　ないでしょうか。



■「小人閑居して不善をなす」

　という言葉がありますが、

　人間、一人でいると、
　ろくなことをやりません。


　だからこそ、

　大勢の、他者からの視線を感じられる
　環境に身を置くことが大切である、

　とこれまでも書いてきました。



■その重要性を理解し、

　限られた自由時間を意義深いものと
　したい方のために、

　先月に引き続き、
　今月も当社セミナールームを開放し


　【丸一日、本当に大切なことだけに
　　集中する会（第６回）】


　を開催することといたしました。



■この場に参加頂きたい方は、
　以下のような方々です。


　▼目標設定の大切さや波及効果は理解している
　　けれども、ついつい設定（更新）先送りしてしまう


　▼今後の事業戦略を立てようと思いながら
　　なかなか手をつけられない方


　▼新しいプロジェクトについて時間を取って企画を
　　立てようと思いつつ、いつまでも進められない方


　▼資格試験の勉強をしようと思いつつも
　　集中して学ぶ環境のない方


　▼読みたい本がたくさんあるのに、普段は
　　読書をする時間を取りにくい方


　▼せっかく購入した音源なのに、聴く時間や
　　環境を確保しづらいという方


　▼動画学習をしたいけれども、落ち着いて
　　視聴できる環境のない方


　こうした方が参加されれば、必ずや


　【最優先すべき事項に対して、
　　実際に時間を天引きし、

　　それが形になる喜びを実感できる】


　こととなるはずです。



■こうした喜び（快感）を、

　強制的にでも
　身体に覚え込ませたことのない人は、


　いまだ、その喜びを知らないのだから、
　まずい、まずいと思いつつ、


　【いつまで経ってもずるずると、

　　（大切なことを疎かにして）
　　どうでもいいことを続けてしまう】


　という惰性のループから
　離れられなくなってしまうのです。


　そんな悪循環の輪を解き放つための
　最高の一日（価値）を、

　この場でご提供できればと思います。


　＜ご注意＞


　※今回は「PC利用可能」といたします。

　　企画立案のみならず、思索したことを
　　形に変えるための機会としてご利用ください。


　※Wi-Fi環境はございませんので、必要な方は
　　恐れ入りますがご自身でご用意下さい。


　※動画学習をしたいという方は、
　　その用途でのみPC利用可（スマホ閲覧も可）

　　（いずれもイヤホンをご利用下さい）

　　Wi-Fi環境はございませんので、必要な方は
　　ご自身でご用意下さい。



■本来は、気心の知れた仲間とともに、

　温泉合宿、あるいはホテルでの
　缶詰合宿などができればいいのでしょう。

　（ちなみに「一人合宿」はお勧めしません。
　　理由は述べませんが、、、経験者談（泣））


　ただ、やりたくても、

　時間的、金銭的に実現できない、
　あるいは、一緒に行えそうな人もいない、

　といったことはありそうですね。



■そうした

　【強い、充足されていない潜在ニーズ
　　（声なき声）】

　には以前より気づいていたのですが、


　実際に過去５回、参加下さった方々から
　頂戴した、熱いご感想を拝見し、

　こうした機会を提供する価値の大きさに
　驚かされました。


　【「集中する会」にご参加頂いた方から頂戴した、
　　熱いご感想集】
　　https://ssdlfiles.s3.amazonaws.com/180224_testimonials_mokumoku.pdf


　※ぜひ、ご感想読んでみて下さい。ご感想自体が
　　「気づき・学びの宝庫」となっております。


　ご感想を熟読してみると、

　如何に私たちが普段、いろんなところに
　認知能力を奪われてきたかが、　

　よく分かります、、、



■繰り返しになりますが、
　そんな時代にあって

　【最優先すべき事項に対して
　　時間を天引きし、それが形になる喜び】

　を、一度でも体験したら


　【自らの意志で時間を天引きし、
　　一つ一つを形にしていく喜び】

　を身体で味わいますから、
　自分でもできるようになるはずです。



■それができない（その喜びを知らない）人は
　いつまで経っても、

　「ダラダラと、惰性の毎日を続ける」

　ことになる傾向にあるのです。


　そんな現状を打破し、新しい一歩を
　踏み出すための場をご用意することも、

　当社の提供できる一つの価値ではないか、
　と考えた次第。



■会場は当社セミナールーム（東京・飯田橋）
　にて、開催いたします。

　水、コーヒー、お茶、お菓子は
　無尽蔵に（！？）提供させていただきます。


　確保いただきたい時間は、

　【5月22日（火曜）の10時～18時】

　（9時45分開場で、退出は18時15分までに
　　お願いいたします）

　まで。



■昼食はお弁当持ち込み、あるいは、
　徒歩1分のところにコンビニがあるので、

　そちらでお求め頂くことも可能です。

　全体の休息は原則、取りません。
　各自、適宜、休息して下さい。


　※終了時に、短い時間ですが、参加者同士の
　　名刺交換の場も設けます。

　　他の参加者の方との交流の場として
　　頂けましたら幸いです。



■合わせて、せっかくですので、

　この時間を最大限有意義に
　活用していただくべく、


　はじめの30分ほどの間、

　【鮒谷が生産性を最大限高めるために
　　心がけてきたいくつかのこと】

　というタイトルで、


　【圧倒的な生産性向上のために役立つと
　　思われるテーマ】

　について放談いたします。


　正直、この話を聴きに来るだけで元が取れる、

　といえるくらいのお話をしようと
　思っています。



■その上、さらに、
　
　【お一人につき15分の、
　　鮒谷からのショートコンサル】

　の時間も確保いたします。

　
　限られた時間ではありますが、
　錯綜する問題も、

　【ほんの短い時間で、
　　根っこから断ち切ることができる】

　こともあるものです。


　これを機会に、
　ぜひご活用頂ければと思います。
　


■つまり、今回の

　【丸一日、本当に大切なことだけに
　　集中する会】

　▼鮒谷の生産性向上放談（約30分ほど）

　▼終日のセミナールーム貸し出し
　　＆集中思索・作業環境の提供

　　【集中できる環境で丸一日、時間を天引きし、
　　　そこから手に入れることができた成果物】

　　　ならびに

　　【時間を天引きし、大切なことを優先して
　　　行う喜びを体で覚えること】

　▼水、コーヒー、お菓子等はご自由に

　▼第二領域（緊急ではないが、重要なこと）の大切さを
　　理解している人（＝分かっている人）との出会いの場


　の全てを提供させて頂きます、

　ということです。


　それに加えて、
