7489号 「早稲田に行くんか、人生詰んだなあ」と憐れんだ眼で言われた話

7488号 26年前の手帳（写真あり）を見て、改めて考えたこと

■昨日のTwitter投稿より。


　----------

　サラリーマン時代、
　仕事で成果を出せない反動だったのか、

　「いつか見とれや！」の気持ちで
　夜な夜な、新聞記事を切り抜いては、

　手帳に貼り付けていました。


　マーカーを引き、
　何度も読み返しては、

　脳に刻み込もうとしたのです。

　----------



■これは「写真つき」にての
　投稿ですが、

　手帳の表紙に記された年（1998）から
　今から26年前のものだと分かります。



■つい、この前のことのように思えますが、
　光陰矢の如し。

　マーカーの色も薄れ、
　歳月の流れを感じさせますね、、、


　（Twitter投稿の実物。
　　写真４枚つけてます）
　https://twitter.com/shinkaron/status/1761669862856995171



■昨日も書きましたが、

　「頭の中を、どんな言葉で
　　埋め尽くすか」

　によって、
　人生は決まってきます。



■要するに、

　いい言葉に触れれば
　人生は良くなっていくし、

　よくない言葉に触れれば
　人生は望ましくない方向に進む、

　ということです。



■そして、ここがポイントですが、

　【どんな言葉に触れるかは
　　自分の意思で決められる】

　し、もっといえば、

　【一定の年齢を超えたら
　　自己の責任で決めなければならない】

　ということだと思います。



■先のTwitterリンク先に、

　杉村太郎さんの言葉も貼り付けてあり、
　懐かしく読み返したのですが、

　こんなことが書いてありました。


　----------

　会社は、国家や家族や恋人など、
　他のあらゆる組織と同様に、

　永遠に存在しつづけることが
　約束されたものではない。

　なくなり得るもの、壊れ得るものだ。


　が、コトが起こると、組織や代表者など
　自分以外のもののせいにする。

　そういう人が多すぎる。

　----------



■当時は山一証券が倒産して、
　大騒ぎになった時代。


　前段、そのことに触れた後、

　----------

　同情したくもなる

　　（中略）

　しかしその、ほとんどについて、
　インディペンデントでない人の甘さを
　感じさせるを得ないのも事実である

　----------

　とも記されています。



■さらに

　----------

　何が起ころうとすべての現象について
　他人など自分以外のもののせいにしない。

　何が起ころうと、すべて自分の力の
　足りなさのせいにする。


　意識を持ち続け、さらにその意識を
　貫き通して生きていくだけの実力を

　絶えず身に付けているものが
　インディペンデントな存在なのだ。


　----------

　とも。



■この切り抜きには、

　「３種類のペン
　（蛍光マーカー、太い赤ペン、細い赤ペン）」

　を使い分け、力強く、
　線を引かれています。


　当時の心境を振り返るに、それだけ
　自身に徹底したい内容だったのでしょう。



■また、

　映画評論家の淀川長治さんの言葉
　（私の履歴書掲載文）にも、

　マーカーを引いて、
　脳に刻もうとしていた形跡があります。


　----------

　「苦労こい」

　苦労から逃げても人間には
　苦労というものがあるのだから、

　むしろ「こい、こい」とこちらから
　手を出して、

　その苦労を食べてしまえ。

　すると、もっと丈夫になるよ、
　そんな意味も含んでいる。

　----------



■あるいは


　----------

　「他人歓迎」

　「わたくしは、まだかつて
　　きらいな人に、会ったことがない」

　「いやがられている人は
　　みんな孤独だ。
　　そんな人ほど愛にかつえている」

　----------


　なんて言葉のところにも
　マーカーが引かれていました。


　※それぞれについて詳しく読みたい方は
　　こちらの写真からごらんください。

　（Twitter投稿の実物。
　　写真４枚つけてます）
　https://twitter.com/shinkaron/status/1761669862856995171



■20代の頃は、

　（真面目に働いていなかったので汗）

　今よりはるかに多くの時間も
　体力もあり、


　空いている時間は、
　ひたすら活字を読み、

　さらには耳からセミナーや講演の
　テープ音源（！）を聴き、

　を繰り返していました。


　パチンコ、パチスロやめてからは（笑）
　それしかやってなかったくらい。



■大量の、いろんな人たちの
　素晴らしい同じ言葉に、

　何度も触れて、
　反復、反芻を繰り返すうち、


　血となり、肉となり、やがては

　「誰かの言葉であったのか、
　　もはや出典も分からなくなったけど、

　　それでも意思決定基準として
　　私のうちに宿り、生き続けている」

　ようになりました。



■こうした言葉、さらには
　意思決定の基準の数々が、

　私（鮒谷）の行動を支配し、やがて
　運命を変えていくこととなります。



■自分の内側に入れる言葉を
　大切にすることは、

　自分の運命を大切にする、
　ということ。



■受け入れる言葉の選定を誤れば、
　人生そのものを過ちかねない。

　そんな思いを持って、たくさんの言葉に
　触れると良いのではないでしょうか。



■最初は、

　「どの言葉を選んで
　　どの言葉を捨てるのか」

　が分からなかったとしても、

　大量の言葉に触れていくうちに、
　センサー精度も高まっていくもの。



■とくに、

　「長い人生を生き、経験を重ね、
　　素晴らしい人格と知恵を身につけられた」

　と思われた方からの学びは、
　ほとんど外れることはないですね。



■生きてきた分だけ
　時の試練にさらされてきて、

　なお有効であった、
　という証左でもありますから。



■かつて住んでいた家から
　サルベージしてきた手帳の写真を見て、

　改めて、当時、感じていた
　いろんな思いが蘇ってきたのでした。


　（26年前の手帳写真はこちらから）
　https://twitter.com/shinkaron/status/1761669862856995171



■今日もこれから、
　一筆入魂の精神で

　「熱い投稿
　（というか、かなり恥ずかしい投稿）」

　を写真付きで行いますので
　よろしければご覧ください（笑）



■引き続き、
　Twitter投稿を重ねていきますが、


　「各投稿について、24時間後の
　　平均インプレッション数10,000達成」

　※インプレッション数とは、
　　ユーザー画面への表示回数のこと


　の目標達成まで、あと64日！


　＜鮒谷Twitter＞
　https://twitter.com/shinkaron



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■頭の中をどんな言葉で埋めるかによって
　　その人の人生は変わってくる。

　■言葉によって意思決定を行い、
　　その意思決定は行動を、やがては運命を
　　形作ることとなるからだ。

　■一定の年齢を過ぎたら、自らの責任で

　　取り入れる言葉、取り入れない言葉を
　　分別しなければならない。

　■私たちが日々、食べるものと同じ。
