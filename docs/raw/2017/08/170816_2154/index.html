───────────────────────────────────


■今日は

　「アウェイに赴かずして、成長できるか」

　という話をいたします。

　人間、同質性の中に身を置くと、
　やがて、今いる環境が

　「当たり前」

　という感覚を覚えるようになるみたいです。



■ところがそんな世界から一歩外に出ると

　「いつもの当たり前とは異なる、
　　違和感あるいは緊張感」

　を感じることとなるでしょう。


　以下に私（鮒谷）が最近（というか昨日）、
　体験した例でお伝えいたします。

　これまで約14年、一日も休まず5,000日日以上に
　わたって
　
　『平成進化論』

　を発信し続けてきましたが、結果として、

　ある意味、極めて居心地の良い場所が形成され、
　長らくそこに自らの身を置いていた（安住していた）のです。



■以前も書きましたが、情報発信を重ねていると

　「引力と斥力」

　が同時に発生し、

　発信している内容に共感・共鳴下さる方と、

　こんなくだらないもの、読んでられるかよ、
　という方と、

　両方の方が現れられることとなります。



■それはある種、揺るがしようのない
　物理法則みたいなもので、

　こうした力が継続して、
　繰り返し加わっていくうちに、

　徐々に擬似的なコミュニティ（バーチャルコミュニティ）が
　形成され、

　自分にとって座りの良い場所が生まれることと
　なるのです。


　これはある意味、
　素晴らしいことではありますが、
　
　ある意味においては、
　緊張感が失われ（もっといえば弛緩し）、

　その環境に身を置き続ける延長線上に

　「いつもと同じ日常」

　が続いていくこととなるでしょう。



■ここで話は大きく変わりますが、私の父親は

　「平凡を長く続けられること、
　　それ自体が非凡」

　とよく言っていました。

　
　これはこれで一つの真理であり、
　価値だと思っているのですが、

　同時に、やっぱり（肉体的にも精神的にも）
　活力を持ち続けたいとするならば、

　自ら率先して

　「（平凡を離れた）非日常の世界」

　に身を投じていくことも必要だと思うのです。



■もちろん非日常の世界は上述の通り、
　違和感や緊張感を感じるアウェイの場となりますが、

　そこに存在することによって
　脳も活性化が促されるだろうし、
　
　新しい行動が誘発されもする、

　といったこともあるのではないでしょうか。



■だから私（鮒谷）は昔から

　「非日常を日常に」

　とお伝えしてきたつもりです。


　ただ、非日常が日常になると、
　それが平凡な生活となり、

　それが長らく継続されると
　非凡につながっていくわけで、


　考えてみれば、
　これは父親の教えに一切矛盾せず、

　ここにおいて、
　自分の中で両者が止揚されることとなりました。



■まあ、そんな話はさておいて、
　以上のような理由から

　「非日常を日常に」

　と他者に発信し続けることで自らを戒め、
　自らに言い聞かせてきた、

　という側面は間違いなくありますね。



■そんな私が最近、取り組み始めた「非日常」に

　「『アウェイの世界で』情報発信を行う」

　というものがあります。

　これまで主戦場としてきたメールマガジンは
　
　気心しれた、
　勝手知ったるホームグラウンド、

　という感覚で発信してきましたが、


　昨日もご紹介した「note」というサイトに
　おいては、

　誰がどのような経緯でお読み下さることになるのか、
　まったく予想がつかず想像もできず、

　それゆえ完全なアウェイの環境にもなるので、

　あえてアウェイの地にて、
　情報発信することにしたわけです。



■とはいえ、実はそちらで発信し始めて、
　いまだ数日。

　発信したコンテンツも
　正味、数個しかありません。


　でも、早速いろんなことが起こりました。

　個人的に一番インパクトあったのは、一生懸命、
　考え、記述し、発信したつもりの内容について、

　たくさんの酷評を受けたことｗ


　「これがアウェイの恐ろしさか（ガクブル）」

　とも思いましたが、

　幸いなことに、もともとメンタルはけっこう強く、

　さらには、長年の発信でも鍛えられているので、
　まったく問題なし。

　でもでも、それでも、やっぱり痛いっすｗ、
　というのが本音のところ。

　　お利口さんなメッセージを読むたびに、

　とか、いろんなご意見があることに触れて、

　さっそく、斥力、キターーーー、

　と思いつつ、


　一方で、いちいち、なるほど、
　そういう捉え方もあるのか、

　と大変、勉強になりました。



■やっぱり、こういう緊張感の中で、

　人は教えられ、鍛えられ、育てられ、そして
　成長と飛躍のチャンスを手にするものなんでしょうね。

　改めてそんなことを教わった
　昨日・今日の出来事でありました。


　
■温暖な海域でヌクヌクと泳ぐ魚のような生き方では、
　どうしたって、やっぱり弛緩系の方向に進んでしまうわけで、

　あえて冬の日本海みたいに冷たく、
　波も荒い場所に棲みつき泳ぐから、

　想像力とか、配慮する力とか、表現する力とか、
　寛容の力とかも身につくし、

　背景も文脈も異なる他者と交わる上での
　格好のトレーニングになるのでは、と思われた次第。



■そんなわけで未知の世界に飛び出すことは
　大切だよなあ、

　って感じたわけですね。

　とはいえ、やっぱりいつもいつも張り詰めていたら、
　それはそれで大変なので、


　-------------------------

　新しい環境に身を置くことによって生じる緊張

  → その環境に慣れることによって生じる弛緩

　→ それじゃまずいと思って、再び新しい世界に
　 　飛び立つチャレンジから来る緊張

  → 再び、慣れが生まれて緩んでしまう

　→ さらに新たな挑戦を（、、、以下繰り返し）

　-------------------------


　こんな風に

　「緊張と弛緩を繰り返す螺旋階段」

　を上りながら、

　一定のペースで非日常を日常に変えていきながら、
　成長を遂げていければいいなと思っています。



■「我以外皆我師也」とは吉川英治の言ですが、
　
　そんな精神を持ちつつ、
　新しい世界を切り拓いていこうと思いますので、
　
　生暖かく見守って頂けましたら幸いです。


　特に「はてなブックマーク」をご活用中の皆さま、

　私も切られれば血の吹き出る生身の人間、
　くれぐれもお手柔らかにお願いいたしますｗ



■ちなみに、

　「アウェイに赴かずして、成長できるか」

　という言葉は

　「アウェイに赴かずして、
　　イノベーションは起こせるのか」

　と言っても、意味が成り立ちます。


　これはまた、もっと時間をかけて、
　何回にもわたって連載したいところですが、　

　「それまで待てない！」

　と言われる方は、以下の音源をお求め下さい。

　この音源に語られているほど
