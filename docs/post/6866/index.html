　--------------------


　今回の音源との出会いが自分自身の人生を
　豊かにするきっかけの一つになりました。

　鮒谷先生の30年を超える
　体験と経験に基づく音源ということもあり、
　圧巻の内容でした。


　特に、現在と未来という異なる次元を
　越えるために武器であるパワーワード

　「人生、幸福、所詮、そもそも」

　を用いるという学びは
　目から鱗でした。


　言葉の用い方一つで意識を
　未来へ強制的に持って行くことができ、

　「現在」と「未来から見た現在」の
　二つの次元で物事を捉えられることができ、

　それを日々意識して問いを回すことが
　結果的に自身が求める未来へ近づけられると
　強く感じることができました。


　音源を聴き、自分の人生の問いに正面から
　向き合うことが何よりも重要である、

　と気づけました。

　おそらくこの音源を聴いていなければ
　流れに流され、

　人生について考えることも
　ほとんどなかったかもしれません。


　日々、人生について問いを繰り返す中で、

　日記を書き続けることは
　人生をより幸福にするために、

　必要不可欠なものであると強く感じました。


　現在、日記を書き続けて
　3500日あまりになりますが、

　これからの人生をより豊かにできるよう
　パワーワードを活用し続けてまいります。


　今回も、貴重な音源を
　ありがとうございました。

　　　　　　　　（Ｒ・Ｙ さま）

■また同音源をお聴き下さった

　（ここから）
　--------------------


　もし私が

　「幸福と人生について
　　語ってみてください」

　と尋ねられたらどうするだろう？


　最近の数カ月の各種音源をきっかけに
　OS絶賛書き換え中の自分は

　日々の…特にネガティブな出来事や
　過去の黒歴史を解釈力で全肯定しつつ、

　一気通貫した人生を
　自然体で語れるようにしたい、

　と思いました。


　いつか人前で語ることを頭の片隅におき、
　まずは書いてみて、

　抽象度が高いテーマにどう具体をのせて
　一人称で語れば良いのかを妄想しています。


　お金や社会的地位など、
　見える化できる具体的なものを目標にし、

　それなりに手にしている人が多い中でも
　その上位にあたる

　「幸福と人生」

　については問いを回している人、
　追い求めている人は多くないのだろうと、

　直近のメルマガでのフォローを読んで
　思いました。


　だからこそ、
　自分はそこを追い求めたい。


　「人生」という上位の層というより
　頂点でしょうか。

　これを考えることで空間軸と時間軸が
　一気に広がることがわかりました。


　大学受験に失敗したら詰む、
　就職活動で失敗したら詰む、

　会社でうまくいかなかったら詰む、

　といった狭く短い両軸の中で
　生きていたら苦しいに決まっている。


　自分が置かれている状況を
　俯瞰する上位から

　「そもそも」

　や

　「所詮」

　という言葉をつけて

　問題や悩みを考えてみると
　気持ちが楽になるかもしれません。


　私の高校時代や大学時代、
　現在の会社員時代で、

　もやもやしていた過去の自分に向けて
　この音源と自分のフォローを送りたいです。


　目標・なりたい自分と思っているものと、
　行ったり来たり出来る

　「幸福と人生」

　について常に言葉を回しながら、
　一生かけて追求していきたいと思います。


　今回も気づきの多い音源
　ありがとうございました。


　（会社員 Ｒ・Ｔ さま）
