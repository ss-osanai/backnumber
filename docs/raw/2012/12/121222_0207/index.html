■本日は、

　私の公私に渡る先生・師匠である、
　弁護士の高井伸夫先生についてのお話をいたします。


　ご存じない方もあるかもしれませんが、

　高井先生は日本における人事労務分野の
　トップ弁護士として活躍してこられた方であり、

　現在、高井・岡芹法律事務所会長、
　経営法曹会議幹事、社団法人日中協会理事、

　などを務められています。


　※過去には、

　　財団法人日本盲導犬協会理事長、

　　財団法人カシオ科学振興財団監事、
　　
　　財団法人労働科学研究所評議員、等々も歴任。

■８年ほど前、

　かんき出版の境社長（当時）より高井先生を
　ご紹介頂いたときより、熱烈なファン！？となり、

　時間が許す限り、セミナー、講演、勉強会に
　通うようになりました。


　そのうち、日常ではなかなかご一緒できないような方々との
　会食に数多く同席させていただくようになり、

　個人的にいろいろご相談をする機会も
　増えてきました。


　
■考えてみれば、

　ここ数日、メルマガでドラッカーさんのことを
　記してきましたが、

　そもそも私がドラッカーに関心を持ったのも
　　
　「日本におけるドラッカーの分身」

　といわれ、氏のほぼすべての著作を翻訳されてきた
　上田惇生先生を高井先生からご紹介頂いたから。



■そんなこんながあり、二階堂進の有名な言葉、

　「趣味は田中角栄」

　に触発され、私も
　
　「趣味は高井伸夫」

　を公言しているうちに、


　趣味が嵩じて（笑）高井先生のカバン持ち、付き人、
　私設秘書？として、

　日本国内はもとより、海外出張、海外視察のお供も
　させて頂くようになりました。

　（これは人生観、ビジネス観が激変させられた体験で、
　　現在の私の日常にも大きな影響を与え続けています）



■高井先生と間近に接する中で、
　一つ一つの動きに本当に

　「圧倒された」

　わけですが、中でも大きなインパクトがあったのが

　【 思いつく → 記録する → 実行する 】

　のサイクルを、

　【 驚嘆するほど大量に、驚異的に速い速度で 】

　回されているのを見た時です。



■打ち合わせ中はもとより、
　会食中や移動の車の中などらゆる場面でメモを取る、

　分量が多い等、メモが難しければ、
　ボイスレコーダーへ吹き込まれます。


　記録を先送りされることは決してなく、

　「常にその場で記録する」

　ことを習慣化されていました。

　
　このようにして記録したことを
　後ほど、事務所に電話を入れ、

　実行を依頼されるのです。
　


■会話の中で話題に出たこと、
　それをきっかけとして閃いたこと、

　あるいは本や雑誌を読んでいて思いついたこと、
　などはこうしてすべて、

　【 思いつく → 記録する → 実行する 】

　の流れに乗って処理されていきます。



■朝から晩までご一緒している中で観察？
　させていただいていると、

　大袈裟でなく、このサイクルが毎日、数百回は
　回っておりました。


　事務所に電話する回数も一日数十回、
　そこで伝えられる内容はおそらく全部で数百件、

　（海外にいても国内と変わらず、毎日、
　　最低2,30回は事務所に連絡されていました。

　　多い時だと３歩歩いたら電話する、みたいな感じ。

　　それくらいいろんなことがひらめき続ける、
　　ということですね）



■この場合は秘書さんに記録も依頼することで

　【 思いつく → 記録する → 実行する 】

　ではなく、

　【 思いつく → 実行する 】

　のサイクルになっているわけですが、

　その莫大な仕事を処理するために
　実に10名以上の秘書さんがいらっしゃるわけです。



■内容は、たとえば


　▼◯◯を購入する

　とか

　▼◯◯を調べる

　とか

　▼◯◯の情報を誰々さんにお伝えする

　とか

　▼◯◯さんと◯◯さんを引き合わせる

　とか

　▼◯◯さんに◯◯さんを紹介する
　　（紹介状と名刺を添付する）

　とか

　▼◯◯さんに手土産を用意する
　 　（手土産はこれ、とか）

　とか

　▼◯◯さんに◯◯をお送りする

　とか

　▼◯◯さんに人の紹介依頼を行う

　とか

　▼メールの文面案の伝達

　とか、

　から始まり、

　大量の仕事が早朝から夜遅くまで
　リアルタイムで進められます。
　


■ひらめきを都度、メモし、
　空き時間を見つけて電話連絡を入れ、

　その内容をスタッフ総掛かりで
　粛々と実行に移していく。


　あるいはメモの段階を飛ばして、即電話、
　思いついたことをそのまま秘書さんに伝える、

　という流れを通して、

　閃いたり、思いついたりする案件を
　決して先送りせず、

　その日じゅうに形に変えていかれる
　（＝実行される）姿はまさに圧巻。



■私もその姿を目の当たりにして

　「一つやれば、一つ進む」

　さらには、

　「その速度を倍速、３倍速、10倍速で回すと
　　その分、加速する」　

　ことを圧倒的な臨場感を伴いつつ、
　教えていただきました。



■高井先生はこの

　「一つやれば一つ進む」

　のスピードを秘書の皆さんの力を借りながら、

　常人の10倍速、20倍速、あるいはそれ以上の速さで
　進めていかれます。


　
■このような高井先生の日常に初めて触れたとき、
　その尋常ならざるスピード感に本当に驚きました。

　（これまでそんな速度で生きている人を
　　見たことがありませんでした）


　長くなりそうなので、明日に続けます。


　今日も人生とビジネスを楽しみましょう！
