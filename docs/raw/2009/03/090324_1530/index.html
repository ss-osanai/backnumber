■もし貴社が新規のお客さんを効率よく獲得したいとお考え
  でしたら、このご案内に注目してみられるといいかもしれません。

  「愛・地球博（愛知万博）の理念継承事業」としてスタートした
  というこの展示会、毎年、出展者さんからも大変好評なのだとか。

■「既に１５社から引き合いを頂いており、うち８社とは取引口座の
    開設など、具体的な話に進んでいます。予想以上の成功です」

  これは昨年の喜びの声の一例のようですが、その成功理由の
  一つに、展示会前後の万全のサポート体制があるようです。
  
   ▼事務局が商談を仲立ちし、マッチングを支援
   ▼船井総合研究所のコンサルタントさんが【無料】で個別指導
   ▼終了後のアプローチについて実践講座を開催

  などなど、出展経験の少ない貴社にとっても大変魅力的と
  言えるのでは。

■【無料】で参加できる説明会は「残席わずか」とのこと。
  東京・大阪にて開催されるようなので、急ぎ詳細をご確認ください。

  （新規顧客からの引き合いが押し寄せる日を想像しながら、
    今すぐこちらをクリック）

■10日ほど前まで、一人の「コーヒー馬鹿」の連載を
　行っておりました。
　
　じつは、あの時点で10話ほど続けてご紹介したのですが、
　あまりにも同じネタが続くと読者さんも
　飽きてしまうかもしれない、

　とおもい、一時的に連載を小休止しておりました。



■しかし、ほんとのところをいうと、
　もう少しご紹介したい（というか、わたしが感銘を受けた）
　点が更にいくつかあるので、

　そのご紹介をあと数日だけ続けさせてもらって
　このシリーズを終了したいとおもいます。
　


■（ここから再開です）


　あなたに、自分の存在を万人に知られたいという
　欲求がないのであれば、

　当然、万人に知られるための努力をする必要は
　ありません。



■ただひたすら自分の関わるひとつの世界に没頭し、
　深掘りしていけばいいのです。

　その世界で畏敬の念をもって「○○馬鹿」と呼ばれる
　ようになれば、すごいことが起こり始めます。



■先日来、ご紹介してきたこの本には、その一例として、
　畏怖すべき

　「コーヒー馬鹿」

　が、何人か紹介されています。


　【今日のお勧め本  コーヒーの鬼がゆく ―吉祥寺「もか」遺聞】
　
　                              嶋中 労 (著) 　中央公論新社
　
　　http://amazon.co.jp/o/ASIN/4120039994/2ndstagejp-22/ref=nosim



■きょうは、その中から一人だけ、ご紹介します。

　その人物は東京・神田の「Ｌ」という老舗コーヒー店を
　営んでいました。

　「いました」、と過去形になっているのは、
　残念ながらすでに店を閉めてしまったからです。



■店がまだ健在だったときは、


　「ここは喫茶店ではありません。
　　珈琲を味わっていただくためのお店です」


　と但し書きがしてあったといいます。

　ケータイ禁止、おしゃべり禁止、待ち合わせも
　ダメだったとか。

　ご紹介した本の著者によれば


　「わびさびた店内には、空気が震えるくらいの
　　緊張の糸が張りつめていた」


　というのですから、その雰囲気が想像できるいうもの。



■しかしこの店、というか店主のすごさは、
　まだまだ、こんなものではありません。

　店主いわく、


　「大勢の客の中には、必ず身の震えるような恐ろしい人が
　　いるもので、

　　私はそうした人のために全力をあげて迎え撃つ
　　準備も怠らない。

　　客とは一騎打ち、真剣勝負です」



■つまり「敵情視察」ないし「道場破り」のために
　やってくるお客さんが、ままあるため、

　そんな、いつやってくるか分からない人を
　迎撃するための

　「特別な豆」

　を、常時、用意していたというのです。



■コーヒー豆というものは、焙煎してからいつまでも
　もつものではありません。

　味も香りも落ちていきますので、商品として使えるのは、
　煎ってからせいぜい１週間から２週間。



■そのあいだに、その「恐るべき客」が来るのかどうか
　なんて分からない、

　というか、たいていは来ないわけですが、

　その、いつ来るか分からない、
　しかしいつかは必ず現れる敵（？）を破るため、

　１～２週に１度は、夜な夜な秘蔵の豆をとりだして
　全神経をとがらせて焙煎する、

　という行為をしていたらしいのです。



■あなたにとって、この行為はいったい何に
　あたるのでしょうか？

　圧倒的差別化をはかり、山よりも高い参入障壁を
　築くため、一考に値することではないでしょうか。
