■年間プログラムのご感想を頂戴いたしました

───────────────────────────────────



■こんにちは、鮒谷です。


　先週末に開催した年間プログラムの
　ご感想をたくさん頂戴しております。

　参加下さった鵜木さんより頂戴した
　ご感想を以下にご紹介。


　（ここから）
　---------------------------------


　年間プログラムは、
　都度タイムリーな気付きを、
　
　レーザービームの如く脳に
　強烈に焼き付ける作用がある。


　つまり、自身の判断基準の書き換えに
　有効なプログラムである。


　今月は以下の項目に、
　ロックオン＆レーザー照射されました。


■圧巻の人は普通(基準)レベルが
　違う事を知れ！！！


　圧巻レベルの人達は、
　普通にしている基準が違う。

　人は基準に合わせようとする習性がある。


　よって、自分自身が何処を基準にしているのか、
　都度確認する事が重要。

　確認するには、凄い人と比べ、
　リアルを体感するのが一番良い。


　その為の年間プログラム、
　凄い人達が集う場に身を置くと、
　自分の基準値が否応無しに引き上げられる。

　基準値が引き上げると行動が変わる。


■心震えた気付きは、言語化し、
　意思決定基準まで落とし込め！！！

　年間プログラム、人との出会い、
　書籍などから得られる心震える気付きは、

　放って置くと泡の如く消失し、
　自分の中に何も残らない。


　誰もが１週間前の読書内容を
　思い出せない経験が有るのでは？

　気付きの浮遊防止には、
　言語化するしかない。


　言語化のメリットは次の通り


▼心震えた事象の歩留り向上

▼良質言語が変な言語をブロックする
  その結果、成功体質になる

▼ネガティブな事象を言語により、
　ポジティブに変換できる

　言語化出来たら、更にもう一歩進め、
　意思決定基準、アクションリスト、
　そして、行動まで落とし込む。


　意識が変われば行動が変わる
　行動が変われば習慣が変わる
　習慣が変われば人格が変わる
　人格が変われば運命が変わる

　なお、心震える気付きを増やしたい場合は、
　名言集を活用する手段もある。


■「そもそも」をまわせ！！！

　そもそも、何のためにやってるんですか？
　そもそも、誰のためにやってるんですか？

　これを高速で回す事で、顧客(組織)に
　最大の価値提供が可能になる。


　人間は、してもらった事は
　バランスをとる習性がある。

　顧客への価値提供は、
　自身の信頼性向上、
　ネットワークの拡がりに繋がる。


■良い本の読み方とは？

　読書により出会った、心震える気付きは、
　意思決定基準にまで落とし込み、
　定着させる事で自分のものとなる。


　そのためには、心が動いた所を
　自分の言語パーツまで落とす

　→この言葉を人に話す

　を繰り返す事により、
　脳に定着させるのである。


■パクってパクってパクリまくれ！
  3年経てばオリジナル

　異業種からヒントを得ることがある。


　異業種の成功事例を、自身の業界に
　応用出来ないか考察する事が重要。


　鮒谷さんの場合、

　俺のフレンチの店舗スタイル
　→コンサルの形態

　クリーニング店クーポン
　→メルマガ音源クーポン

　などが異業種からの気付きで生まれている。


　目標を高く掲げ、現実ギャップとの真空状態で
　異業種を見ることで、
　イノベーションが生まれるのである。

　　　　　　　　（エンジニア・鵜木 様）


　---------------------------------
　（ここまで）


　鵜木さん、長文のご感想、
　ありがとうございました！



■多くの方から年間プログラムのご感想を
　頂戴しておりますが、

　共通する気づきもあれば、
　ある方のみに固有の気づきもあり、

　発信する側として、
　とても勉強になります。



■鵜木さんからは、鵜木さん独自の

　「年間プログラムの価値」

　を見出していただいておりますが、
　参加者お一人お一人にとって、

　プログラムの価値が最大化されるよう、
　これからも力を尽くしてまいります。



■ちなみに今回のプログラムにおいては

　「世間一般から見ると、高いと思われる基準を
　　『普通』レベルとして、思考し、行動する」

　ことの大切さが、皆さまに

　「刺さった」

　ように思われます。


　これからも、

　（知識、スキル、ノウハウ、テクニック、
　　さらには知恵、はいうまでもなく）

　大いに震え、大いに感奮し、
　結果に直結する行動が促される場、

　のご提供を行ってまいります。

　次回も引き続き、よろしくお願いいたします！

　
　
■それでは、

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。

＜今日のピークパフォーマンス方程式＞

【 「同調圧力」を上手に管理できる人が、結果を残せる人 】

───────────────────────────────────



■しつこく、しつこく、しつこく、
　しつこく、しつこく、しつこく、書きますが、

　「体を動かさず、
　　頭と口だけを動かしていても、
　　
　　結局、事態は何も変わらない」

　ものです。



■世の中に勉強好きな人がいるものですが、
　
　「勉強に、行動が伴われてこそ、
　　価値が生まれる」

　わけであって、

　「勉強のための勉強は、
　　ただの趣味」

　なのです。



■もちろん趣味として学ぶのを
　否定するわけではありません。

　学びには純粋に

　「知らないことを知る喜び」

　があることは、
　百も承知しています。



■しかし（特に仕事絡みの学びについては）

　仕事のアウトプットを高めるために
　勉強しているはず。


　であるにもかかわらず、

　「一向に成果の生まれぬ勉強
　　（＝行動の伴わない勉強）」

　を続け、

　人生や生活に結果が現れず、それゆえ
　現状について不満を持っているのでは、

　学んでいる意味がないと思うのです。



■私（鮒谷）は、

　（特に仕事上において）

　学ぶのはあくまで人生の、生活の、
　仕事上の不満を消し、

　楽しく充実した毎日を送るためだと
　理解しています。



■しかし、

　「学びを行動に変える」

　ことがなかなか難しいんですよね。

　
　理由はいろいろ考えられますが、

　多くの人と接してきて、
　分かったのは

　「同調圧力にやられて、
　　動けない」

　人が、極めて多いということ。
　


■一生懸命、勉強し、
　その成果をどこかで発揮したいと思っても、

　「自分の周りに現状の殻を破ろうと、
　　突き抜けた行動をしている人がいない」


　あるいは

　「現状維持を良しとする人に
　　取り囲まれた状態」

　であれば、

　「無言の圧力」

　的なるものに打ち勝てず、


　結局、

　「やらなければならないとは
　　思っているけれども、
　　
　　どういうわけだか体が動かない」

　そんな状況が続くこととなるのです。



■こうした現状を打破するためには、

　【突き抜けた行動をしている人ばかりで
　　自分の周囲を満たせば良い】

　のです。


　上述の「同調圧力」は、
　否定的な意味合いで用いましたが、


　上手に「同調圧力」を活用すれば、

　「動けない自分が動かずにおれない　　
　　環境に身を置くことによって、
　
　　当人の自覚なく、自然体で
　　動き始められるようにすらなる」

　ということが現実にあるのです。



■年間プログラム、という、
　
　平均的な水準から大きく（あるいは少し）
　逸脱して行動し始めている、

　そんな人たちに囲まれていると、
　自然と身体も動き始めるというものです。


　同時に、

　「これから、身体を動かそうとしている人の
　　居場所も確保されている場」

　であるから、これから、の人も
　安心して参加いただけますし、


　「先行者と追随者の双方が交わり、
　　相互に影響を与え合い、

　　いつの間にか、行動しているのが
　　自然と感じられる場と一体化する」

　こととなるわけです。



■もちろん別に「年間プログラム」という場で
　なければならないわけではなく、

　どんな場でもよいのですが、
　ただし、


　「先行者と追随者の双方が交わり、
　　相互に影響を与え合い、

　　いつの間にか、行動しているのが
　　自然と感じられる場と一体化する」


　そんな場に身を置いていないと、
　いつまで経っても、

　体を動かそうと思っても動かない状態から
　抜け出す、

　といったことが起きないであろうことは、
　確信を持って断言できます。
　


■結局のところ

　「どこに身を置いているのか」
　
　ということですね。


　たとえば、
　会社づとめの社会人であれば、
　
　Ａ社に勤めるのか、
　Ｂ社に勤めるのか、
　Ｃ社に勤めるのか、
　
　によって、それが３年、５年、10年と
　経過するうちに、

　（ほとんど自覚なく）所属組織の水準に
　合わせて振る舞うようになるのと同じです。



■３年で10年分の能力を伸ばそうと
　思っている人ばかりの環境と、

　10年かかって、３年分の能力伸長でいいやと
　思っている人ばかりの環境の、

　どちらに身を置いていたほうが
　成長できるし、行動も伴うか、

　考えてみたら分かりますよね。



■別の形で説明するならば、

　圧倒的に高い能力を持ち、
　さらに行動を重ねている人たちに混じり、

　最後尾を走る人と、

　
　低い能力で、
　さらには行動もしていない集団の中で、

　先頭を走っている人と、
　どちらが早く変化を遂げられると思いますか、

　と問題提起したいのです。



■どんな場でも良いと思いますが、

　もしここまでの話に共感されるところが
　あると思われましたら、
　

　高い能力と行動量を兼ね備えられた方が
　大勢、参加下さっている、

　「年間プログラム」

　にぜひお越しください。


　既に参加下さっている皆さまとともに
　あなたをお待ちしております。
