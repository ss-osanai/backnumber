【 成功体験のある者と、ない者の間で、格差は急速に拡大していく、その理由 】

───────────────────────────────────



■何ごとにおいても、

　「中途半端は最悪」

　です。

　何が最悪かといって、もったいないのです。


　何かを始めるのはいいのですが、

　せめて、一定のレベルに到達してから、
　次の山に向かうべきでしょう。

　

■どんなことでも、

　やり切らないと分からないことが
　あるのです。


　裏を返せば、

　「やり切った先のところでのみ、
　　何かを理解し、何かを身につけられる」

　のです。


　
■私はこれまでに、
　いろんなことをやり切ってきました。
　
　同時に、

　いろんなことを中途半端に終えてきました。


　振り返ってみて、

　やり切ったことから得られた学びと、
　中途半端に終わらせてしまった学びとを比較すると、


　（単位時間あたりでみてみると）

　「体感で、じつに10倍どころじゃない差がある」

　ように思われるのです。



■実際、あなたの周りにいる人を見ても、


　何かを一つでもやり切った人の言葉は重く
　（耳を傾けたくなる）

　どれもこれも中途半端だった人の言葉は軽い
　（耳を傾けようという気にならない）


　ということはないでしょうか。


　何かをやり切るということは
　一つのことを深く掘る、ということですから、

　当然、穴の深さに比例して、
　穴の直径も広くなっていく。


　この穴の体積が学びの総量となり、魅力の総量とも
　なっていくのではないでしょうか。
　
　この先に、とてつもない世界が開けていくのです。


　※以下、参考バックナンバー。


　＜3380号 「波乗りより、深掘り」が顧客の安心を生む＞

　http://www.2nd-stage.jp/backnumber/ppf_full/2012/12/3380.html



■何かをやり切った先にのみ、
　何かが手に入る、

　これはもう間違いのないことでしょう。


　しかしながら、

　一つのことをやり切るにあたって
　大きな障害となるのは、

　この道の延長線上に何が手に入るのかは、
　やり切った、事後にしか分からない、

　という現実。


　
■コンサルをしていても、

　やり切った先に何があるか見えないから
　やらない、あるいは、やれない、

　という人は想像以上に多いように感じています。


　しかし、幾度かの

　「やり切った」

　経験を持つものとしては、


　「やり切ると、価値ある何かを必ず、
　　手に入れることができるのだから、

　　信じて前に進んで下さい」

　というしかありません。


　こればかりは

　「体を通してのみ、知らされること」

　だから、どれだけ言葉を尽くしても、
　完全にお伝えし切ることができない世界なのです。



■やり切り経験のある人（＝成功体験のある人）
　にとっては、


　一つのことに集中して取り組めば、

　必ず、すばらしい宝物が手に入ることのは
　自明のことだし、


　その宝物が何であるのかは
　事後にしか分からない、

　というのも、これまた自明のこと。



■この自明のことを２、３回、経験すると、
　メタ的な認知を持つようになり、

　
　「この延長線上に何があるかは分からないけど、

　　そもそも、そういうものだから、
　　とりあえず、今回も、突き抜けるまでやり切るか」


　という発想で事にあたれるようになります。

　

■けれども、人生において、
　こうした経験を一度も持たない人は、

　毎度、毎度、

　「この延長線上に何があるかが分からないから、
　　私は前に進みません、あるいは、進めません」

　と、自分の行動にブレーキをかけてしまうのです。


　こうして、両者のギャップが永遠に埋まらない、

　というよりも、開く一方となるのです。
　

　こうして、成功体験のある者と、ない者の間で、

　格差は急速に拡大していくこととなるのです。



■このギャップを埋めるためには、

　外部の人間の力（手助け、応援）が絶対的に
　必要です。


　なぜなら、

　当人が前進することに対し、抱えている不安は
　当人では解消できないから、
　
　外部の人間が（極端にいえば）
　ときに脅し、ときに透かし、ときに宥め、ときに励まし、

　えっちらおっちら、向こう側の世界に導く必要が
　あるからです。


　その経験のない人にとっては
　自分の力だけでは、とてもじゃないけれども、

　「やり切った世界と、その後の至福の喜び」

　を生涯、味わうことができません。



■ただし、たった一度でも、

　「やり切った世界と、その後の至福の喜び」
　
　を体験した人は、

　あとは２度、３度と、幾度もなく、
　その喜びを求め、

　たった一人になっても、
　一筋に一つのことに集中することができるのです。



■ちなみに私（鮒谷）でいえば、今は
　
　「個別コンサル」

　に、ハマっています。


　コンサル道の奥の深さ、そして難しさに、
　毎日、苦しみつつも、感動しながら仕事をしています。

　この道を歩き切った先に、
　何があるかは分かりませんが、

　何かがあることだけははっきりと分かっているのです。



■そしてこの感覚は、
　これまでにも何度も経験した感覚。

　だから、いつかの段階で、
　突き抜けた世界に出られるのは間違いありません。
　
　
　そのときに見える世界が、
　筆舌に尽くしがたい、素晴らしい世界であるからこそ、

　苦難の道も歩き続けることができるのです。



■ですから、少なくとも、
　そこ（免許皆伝）に到達するまでは、
　
　「個別コンサル」

　のお仕事を続けるつもりです。


　もちろん、そこから先は、
　この過程で得られた経験や知見を元に、
　
　再び新しい道に進むことになるわけですが。
　


■最後に、せっかくなので（？）

　「突き抜けた世界」

　を垣間見せてくれる、
　素晴らしい参考書籍をご紹介しておきますね。
