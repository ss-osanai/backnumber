■無秩序バンザイ！混乱ウェルカム！
　混沌サイコー！カオスこそ至高！

───────────────────


■早速、「ふな祭り」参加の皆さまから
　たくさんのご感想を頂いております。


　コンサルもさせてもらっているＨさんから
　以下のご感想を頂きました。


　（ここから）
　------------------------


　ふな祭りっていうまたズルい名前の
　イベントしてるわね・・・

　と心がザワついてから（笑）

　ノリと勢い仲間の谷さんと
　ゲリラ参加しました。


　下記の文脈をえられただけで
　元値以上の価値をえれたと思います！


　その文脈が脳内にあるかないかで
　自身の選択の仕方が変わり、

　選択が行動を変え、
　最終的に行動によって
　もたらされる結果が変わるからです。


　事前に質問したいなと思っていたことは
　Ｉさんが質問してくださったのと、

　人間関係を好きな形に編む方法も質問でき
　スッキリして祭りをあとにしました。


　いちばん刺さったのは

　無→無秩序→秩序

　になるというところです。


　今まで、０→１を生み出す時の

　「１」

　というのは
　何か形になった（整形された）

　「１」

　を想像していましたが
　そうではなく

　「１」

　かどうかわからない、無秩序の

　「１」

　っぽいものがまずできる、
　ということでした。


　無秩序って
　まさしくクレイジーキルトのキルトが

　手の中に編集素材として集まってくるけど
　どう縫い合わさっていってるのか
　広げてみるまでわからない

　そんな状態を想起しました。


　カオスの中から秩序が生まれる。

　だから
　わけのわからないところに行ってみよう！

　やり散らかしてみよう！

　という楽しい文脈が
　あたまの中にインストールされたので

　いっそう機嫌のよいゲリラになれそうです。


　わけがわからないものを見つけよう。
　わけがわからないところに行ってみよう。

　その行動量が、何かになる。

　何かわからんけど
　何かたのしい

　まさしく祭りのノリだなって思いました。


　ふな祭りのノリをお借りして、
　友人のＴさんと共謀？し、

　８月７日に、Ｈ祭り
　10月３日に、Ｔ祭り

　をスケジュールにとりあえず入れる
　ということをしてみました

　何が起こるかわかりませんが
　何か起こると思います（笑）


　匍匐前進しながら、
　状況に合わせて目くばせだけで作戦変更できる

　ゲリラの強みを生かして
　のしのし進んでいきます～！

　ふな祭りの開催ありがとうございました！


　（Ｍ・Ｈ さま）


　------------------------
　（ここまで）



■Ｈさん、本日は
　ご参加ありがとうございました。


　おかげさまで私（鮒谷）も、

　朝昼晩、トータル８時間ほど
　思いつくまま放談する機会を得られ、

　楽しい時間を
　皆さまとご一緒できました。



■午前、午後、夜、
　それぞれの回において、

　いくつかの中心的なテーマを
　意識しながら話をしましたが、


　中でもＨさんがお書き下さった

　【無 → 無秩序 → 秩序】

　については、

　個人的にはめちゃ、
　重要なメッセージであった、

　と思っています。



■変化・変容について思いを致すとき
　多くの人は

 　【無 → 秩序】

　のショートカットを
　したがるんですよね。



■新しい世界に飛び立とうとする人

　（もっとも分かりやすい例としては
　　起業志望の人、などが挙げられますね）

　は、往々にして、

　無秩序状態が怖いとか、
　不安だとか、心配だとか、焦るとか、
　落ち着かないとか、

　いろんなことを言われます。

　

■それゆえ
　　
 　【無（現状）
　　 → 秩序（いきなりうまくいく）】

　という道を求めるのです。

　しかし現実には
　そんなルートはありません。



■試行錯誤を重ねてカオス状態に入り、
　そこで揉み合い、やがて時期が訪れて、
　
　陰が陽に転ずるがごとくに
　
　【秩序の原型】

　のようなものが
　立ち上がってくるんですよね。

　（これを目標設定音源では
　　「サナギ、からの羽化」で例えてます）



■このプロセスを理解できない人は
　必然の帰結として

　「すべてを理解し、
　　予測がつくようになってから
　　動き始めよう」

　ということになるでしょう。



■けれども実際のところは
　どれだけお勉強を重ねたところで、

　いつまで経っても、すべてを把握し、
　未来の展開を読めるようになった、

　なんてことは起こりません。



■にもかかわらず、

　完璧を期したり、
　過度にリスクを恐れるがあまり、

　無為に時間だけが過ぎていき、
　結局、踏ん切りがつかず、最後は

　「行動できなかった過去を振り返り、
　　悔恨の日々を過ごす」

　ことになるのです。



■これに対して、
　日々、変わり続けている人は、

　この人とはまるで異なる世界認識、
　ならびに発想を持っています。



■すなわち、
　
　「動かなければなんにも分からない。

　　動いていくうちに混沌、からの
　　秩序の原型が立ち上がってくる。

　　原型が生まれたら、それを確実に
　　機能するシステムにまで磨き上げる」

　という意思決定基準に従い、
　行動するのです。
　
　

■ここまでお読み下さったあなたは
　既にお分かりの通り、

　【無 → 無秩序 → 秩序】

　と

　【無 → 秩序】

　は似て非なる、どころか
　真逆の結果をもたらします。



■無秩序バンザイ！
　混乱ウェルカム！
　混沌サイコー！
　カオスこそ至高！

　と雄叫びを上げながら
　わけの分からぬ状態を求め、


　カオス状態から、どんな変遷を経て
　未だ見ぬ新秩序が立ち上がってくるのか、

　ドキドキ・ワクワクしながらを
　変化を楽しめる人だけが、

　新しい世界に到達できるでしょう。



■その過程における、肝心かなめの

　「無秩序状態」

　に入ることを恐れている人に
　変化は訪れない。


　
■変化し続けている人にとって、

　当たり前過ぎるほど
　当たり前なこの事実を、

　変化の経験がない人は
　理解できないので

　
　「無秩序？混乱？混沌？
　　結構です！滅相もない！
　　勘弁して下さい！」
　　
　と絶叫＆拒絶して、

　自ら変化・変容の機会を拒み、
　遠ざけているのです。



■こうした人が
　いくら一生懸命にお勉強したところで、

　結局、あるところまで進んでも、
　そこから先、恐怖に足がすくんで

　「不確実性に満ちた世界」

　に進めません。



■そのため、

　そこから立ち上がってくるはずの
　「秩序の原型」が永久に形作られず、
　
　ちょっと頑張ってみたところで
　元の木阿弥となる。



■やがて

　「確実性は担保されているけれども
　　平凡で、月並みで、退屈な世界」

　に舞い戻ることになるでしょう。



■今回の「ふな祭り」においては、

　このあたりの機微について
　相当、時間を割いてお話ししましたし、


　これによって世界認識に多少なりとも
　揺らぎや亀裂を入れられたとしたら、

　お祭りを開催した意味が
　あったというものです。



■ぜひ、積極的に
　「無用の用」の世界に足を踏み入れ、

　「不確実性に満ちた世界」に
　身を置いている予測不能のドキドキ感と、

　難解なパズルを解くワクワク感を
　味わい、楽しみながら

　【無 → 無秩序 → 秩序】

　の旅を堪能くださいませ。



■ちなみに私（鮒谷）にとって
　人生で最も興奮するのは

　【無秩序（カオス）の中から
　　秩序が浮かび上がってきた瞬間】
　
　です。


　
■この瞬間こそ生物進化の歴史における
　「眼の誕生」のようなもので、
　
　【カンブリア爆発
　（カンブリア紀大進化）キター！】

　という異常興奮状態が
　沸き起こります。


　
■無秩序状態を経ることなく、
　人生における

　【カンブリア爆発
　（カンブリア紀大進化）キター！】

　は起こらないことを理解できれば、


　来たるべきカンブリア爆発の土壌となる
　カオス状態そのものを喜べる、

　ようになります。



■つまり変化・変容を何度か経験すると、

　その経験に至る前、つまり
　カオス状態にあるときから既に
　
　【射精直前・臨戦待機モード
　（我慢汁モード）】

　になるわけです。



■にも関わらず、これを理解できず、

　【無 → 無秩序 → 秩序】
　
　の「無秩序」フェーズをすっ飛ばし、
　いきなり

　【無 → 秩序】

　を求める人は


　最も大切にしたいし、
　大切にすべき

　【射精直前・臨戦待機モード
　（我慢汁モード）】

　を経ずして

　「射精」

　のみを求める人、と言えるでしょう。



■それでいい人は、いいんですけど
　至高の快楽を追求する者にとっては
　
　「射精直前・臨戦待機モード」

　こそが一番、
　大事なんじゃないでしょうか。



■おそらくこの感覚は、

　人生における幾度もの射精、
　もとい

　「非連続の跳躍（変化・変容）」

　を重ねてきた
　歴戦の勇士であれば皆、

　同意されるはず。

　（女性については知りません）
