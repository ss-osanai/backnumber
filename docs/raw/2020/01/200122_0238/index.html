＜今日のピークパフォーマンス方程式＞

■ご縁がつながり続ける、
　自分仕様の仕組みを確立する

───────────────────────────────────



■長年、メルマガを配信してきて
　つくづく思うのが、

　「接触し続けることの大切さ」

　です。


　毎日一通、ときに２通、３通と
　配信されてくる中で、

　送りつけられるメルマガ全部に
　毎回、きちんと目を通している、

　という方は、むしろ少数です。



■多くは、

　メルマガ一通を数十秒くらいで
　超・流し読みされているか、

　ヒマがあるとき、
　興味のあるタイトルで届いたとき、

　など気が向いたときのみ、
　開封下さる方。



■今のような慌ただしい時代にあって、

　そういうものだと思っていますし、
　それでいいと考えています。


　ただし一つだけ意識していることが
　あるとすれば、それは

　「解除だけはされないように」

　ということです。



■たとえ何年間も
　お読みいただいていなくとも、

　解除さえされていなければ、
　数年後、十数年後に、

　「不死鳥のごとく、
　　突如としてご縁が復活する」

　ことが結構な頻度であるのです。

　（今まで何度もありました）



■つまり、

　人間関係においては
　たとえ関係性が薄く（弱く）とも、

　ゆるゆると
　つながり続けてさえいれば、
　
　いつか、そのつながりから
　再び新たな何かが生まれることがある、

　のですね。



■これが分かっているから、

　私（鮒谷）は倦まず弛まず
　メルマガを配信し続けてきましたし、

　これからも配信し続けて
　いくわけです。



■こうした話をすると、

　もしそうだとしても、毎日毎日、
　よくもメルマガを書けますね、

　と、しばしば尋ねられます。



■しかし私（鮒谷）に言わせれば、


　メルマガを書くことを止め
　（関係性をつなぎとめる努力を放棄し）

　せっかく築いた
　ご縁が時間の経過とともに劣化し、

　二度と修復できない関係に
　なるくらいであるならば、


　毎日、ほんの少しだけ、
　メルマガ執筆の時間を天引きし、

　配信し続ける以外の
　選択肢はないではないか、

　という偽らざる思いがあるのです。



■努力とリターンの間の因果関係が、

　これだけハッキリと捉えられる
　価値を確信できる取り組みは、

　ちょっと他には思い浮かびませんし、


　やればやるだけ人生が豊かになり、
　QOLは高まると分かっているので、
　
　もはや

　「やめられない、止まらない」

　状態に入ってしまっております。



■別に、だからあなたもメルマガを、
　などというつもりはありません。


　しかし上手にSNS等を活用しながら

　「自分仕様の、長期継続で
　　ご縁がつながり続ける仕組み」

　を作っておくと、
　

　人生を豊かに、楽しく、充実させ、
　さらにはより安定したものとなるのは、

　もはや間違いないことであると
　思っています。
