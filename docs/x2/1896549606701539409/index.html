朝起きた瞬間、どこに意識を向けるかで一日が決まるんよ。にもか

2025/03/02 22:59

朝起きた瞬間、どこに意識を向けるかで一日が決まるんよ。にもかかわらず、フォーカスしてはいけない場所に、ほぼ無意識かつ自動的に毎朝、焦点を合わせてる人が多いのよ。

これが毎日の憂鬱さや退屈さを感じさせる元凶になっている。

起床時に「今日も憂鬱だな」「面倒くさいな」「やる気が起きないな」と無意識に頭の中で言葉を回してる人がある一方で、

「今日はどんな楽しいことが起きるかな」「どんなことを学べるのか」「将来のために何を積み上げようか」と考えてる人も（少数ながら）いる。

同じ朝を迎えていても、全く違う一日が始まるということ。この差は小さいようで、実は決定的。

そして面白い（？）のは我々の日常生活を支配している「気分」の正体。気づいている人はほとんどいないけど実は「気分も習慣」なんよ。

起床時のみならず日々の出来ごとを肯定的に捉える言語を繰り返し脳内で回して、その言い回しが定着すれば、もう言葉としてはっきり浮かんでこなくてなってすら、「習慣化された気分」が感情を覆い尽くすようになる。

こうした形で感情が固定化された時点で「気分自体が習慣」になるわけよ。多くの人は自分の感情を「所与のもの」「変えられないもの」「コントロールできないもの」と思い込んでるけど、実はそうじゃない、いうこと。

これは、たとえ歓迎せざる気分に陥りそうになっても、言葉によっていくらでも気分を切り替えられる、いうことでもある。同じ事象でも捉え方（=言葉の扱い方）次第で感情を変えられるという話。

たとえば「腹が立つ」と感じたとき、「人の振り見て我が振り直せ」とか「以って他山の石とせよ」とか言い換えればいい。

あるいは「これは怒りをもたらす元凶とどのようにコミュニケーションを図るか、という対人関係能力向上のトレーニング」といった風に内側の言葉を変えるだけで、怒りの感情が和らいだり、むしろ前向きなものに変換されたりするんよね。

つまり言葉と感情は非常に密接な関係があるわけ。事象は一つやけど、それに対してどんな言葉を当てるかで気分や感情が全然違うものになる。

でも、普段から言葉を意識して操ろうと心がけていなければ、感情を適切にマネジメントすることができんのよ。それゆえ感情、気持ち、気分が所与のものというか、既にその状態にあってそれを変えることはできないと思ってしまう。

残念ながら、こうして感情に流されて生きてる人が圧倒的に多いんじゃないと思ってる。

自分の気分をコントロールできない、自分の感情は預かり知らぬところからもたらされたものだと思い込み、それに振り回される人生を送ってる人が多いように見受けられるんよ。実に残念なこと。

ただし、その「悪感情習慣」を自在に切り替えるのは、やはりいきなりは難しい。一日や二日で変わるもんではないということ。

いきなり切り替えられないのであれば、どうすればいいかというと

「無意識的無能⇒意識的無能⇒意識的有能⇒無意識的有能」

のプロセスを順に進んでいくと良い。

最初は気分に支配され、「これは変えられない」「これは所与のものである」と思い込んでる状態。これが無意識的無能。

変えられると思ってないし、そもそも上記のようなメカニズムについて意識もしてないから、当然それに対して何らかの改善を図ろうという思いが出てくることはない。

次の段階は、「あ、この気分は内面で回している言葉から来てるんや」と気づきはする。けれども、それを上手くポジティブ方向に転換させらない段階。これが意識的無能。意識には乗ってるけど、対応できない（無能）いうこと。

さらにトレーニングを重ねると、意識的に言葉を置き換えられるようになる。これが意識的有能。意識しているときには変えられる。ただし油断するとすぐに元通りになってしまう。そういう意味で未だ完全に習慣化されているわけではない。

最後には意識せずとも自然とポジティブな言葉が浮かび、いつも明るく楽しい気持ちで満たされるようになる、そんな無意識的有能（習慣化された状態）の段階へと進んでいくんやで。

いずれにしてもすぐには変われんから、毎朝の気分を少しずつ書き換える訓練から始めるといい。

まずは「今日はどんな発見があるだろう」「今日は誰に喜びを届けよう」「今日はどんな成長があるだろう」といった問いを意識的に回すところから。

やがて朝起きた瞬間から自然とポジティブな言葉が浮かび、それに伴って気分も明るくなり、一日が楽しみになる日々が訪れるようになる。

そんなわけで明日の朝から騙されたと思って、自分の言葉に意識を向けてみるといいよ。それが未来の自分への最高のプレゼントになるからね。

ワイは上記を長年続けて、完全に習慣化されてるから毎日、楽しく生きられてる。
