■耐性菌、という言葉がありますね。

　細菌やウイルスが
　薬に対する抵抗力を持ってしまい、

　薬が効かなくなる、という
　アレです。

　どんな抗生剤を使っても効かぬほどの

　「スーパー耐性菌」

　大きな問題となっているようです。



■これは医療の話ですが、
　自己啓発の世界（？）においても

　「耐性」

　はキーワードの一つ。


　思考を良い方向に変えてくれたり、
　鼓舞してくれる、

　そんな効果・効能のある言葉も
　何度も触れているうちに

　「耐性を持ってくる」

　ところが
　耐性菌とそっくりなのです。



■耐性菌は、

　「効果があるから」

　ということでバンバン、
　抗生剤を用いられたところから、

　耐性がついてきました。



■菌からすれば、
　薬剤は自らの命を脅かす天敵ですから、

　外膜変化やDNA・RNAの変異、
　バイオフィルム等の技（？）を用いて、

　身を守ろうとして、
　結果、耐性を身につけていきます。



■同様に、私たちは

　「変わりたくない」
　「このままでい続けたい」

　という、

　ホメオスタシス維持（？）の如き、
　防御のための方策を様々に取ります。



■頭では「変わりたい」と思っていても
　身体は「変わりたくない」と拒絶する。

　これがもともと、私たちに備わっている
　習性のようなもの。

　【言葉という薬の薬効】

　変化を助けるのが学びですが、

　問題は、

　【言葉という薬にも耐性が生じる】

　ところにあります。

　効果のある言葉を聴いても

　【言葉に慣れる】

　あるのではないでしょうか。

　【新薬の開発】

　を行うように

　【新しい言葉の開発】

　をするしかありません。



■昔と違って、
　今のような情報過多の時代にあっては、

　【生み出された新しい言葉に対する
　　耐性があっという間についてしまう】

　もの。

　学ぶほどに耐性化が亢進してしまう。
