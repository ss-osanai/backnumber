＜昨日の活動ダイジェスト＞

▼グローバル資本主義の中に身を起きつつ、考えること

───────────────────────────────────



■こんにちは、鮒谷です。


　ここ数年、私の中に伏流している大きなテーマの一つに


　「グローバル資本主義の世界に生きつつ、

　　グローバル資本主義に飲み込まれぬ生き方を」


　というものがあります。



■話は変わりますが、つい先ほど、


　＜Analyze + Summarize＞

　http://analyzesummarize.blogspot.jp/2012_10_01_archive.html


　というブログに鮒谷さんのことが紹介（？）されているよ、
　友人が連絡をくれました。


　
■早速、とても興味深く拝読させて頂きました。

　文章の中で少しメルマガのことにも触れて下さって
　おりました。ありがとうございます！


　またロンドン講演の際にも会場でお話しさせて
　頂いていたとのこと、

　改めてその節は、ありがとうございました！
　
　
　
■この方が先日、ご覧になられたという、


　＜モンドヴィーノ＞
　　
　http://amazon.co.jp/o/ASIN/B000EGDDMW/2ndstagejp-22/ref=nosim


　という映画は私も一度、見たことがあります。


　コロンビア大学の人類学部博士課程を修了された後、
　
　現在、早稲田大学琉球・沖縄研究所客員講師、
　早稲田大学などで兼任講師を務められている前嵩西一馬さんから、

　「ダーウィンの悪夢」

　とともに５年ほど前、紹介頂いたのです。


　それをきっかけにして、私も見たのですが、
　なかなかに考えさせられる映画でした。


　
■考えてみると、その頃から、

　「グローバル資本主義の世界に生きつつ、
　　グローバル資本主義に飲み込まれぬ生き方」

　というテーマに問題意識を持ち、
　以来、今に至るまで模索を続けてきたようです。


　当時はそんな言語化はできていませんでしたが。
　
　（ちなみにこの試行錯誤の延長線上に、
　　この度の大学院受験があったのですが、それは余談）
　

　
■そして上記のブログにおいては、

　私が漠然と感じながらも深めきれず、
　うまく言語化しきれずにいた思考について、
　
　「こういう流れに向かうといいなあ」

　と思わずにおれない、適切な表現をいただいた、
　という感覚を（勝手に）持った次第です。


　----------------------------------------------------------------
　

　ただ、長く栄えるためには、「売れていく」要素、
　つまり商品への注力が必要だ。

　キャピタリズムギラギラのワイン農家側にとっては、
　フランスの老舗と対抗していくために、
　当然ながら今は商品デザインで画期性を持たせる必要がある。

　だから最初はそこへフォーカスするのは当たり前だ。


　ただ長期的には、商品への注力と商品デザインの注力バランスの
　舵取りをうまくやったものが生き延びるのだと思う。

　だから、いつの日か、彼ら（の子孫）がフランスのワイン農家と
　同じようなことを語る日が来るのではないか。

　文化はそうやって育って根付いていくものだと思う。


　----------------------------------------------------------------
　

　ここのところ。（勝手引用、すみません）
　


■深く共感、共鳴いたしました。

　また、文体も素晴らしく、
　いずれこういう文章が書けるようになりたい、

　と思わずにおれませんでした、、
　

　どなたがお書き下さったものなのか分からず、
　したがってご連絡先も分からなかったので、

　こちらにて御礼、申し上げたいと思います。


　ありがとうございました！
　今後共よろしくお願いいたします。


　＜Analyze + Summarize＞

　http://analyzesummarize.blogspot.jp/2012_10_01_archive.html
　



■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　出会う人から通信簿をもらう　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

  【　出会う人から通信簿をもらう　】

───────────────────────────────────



■これまでに何度かメルマガで書いたことがありますが、
　私はいつも、

　「出会う人、出会う人に、
　　必ずプラスの痕跡を残していく」

　ことを自らの努力目標として課しています。


　会食のときもセミナーのときもパーティのときも、
　人と出会うときには必ず意識しています。

　ほんの少しでも、
　相手に価値を与えることができないか、
　
　といつも考えながらお会いしているのです。



■都度の出会いにおける、
　そんなちょっとした心がけの集積が、

　やがて雪だるまのように自分の人生を
　好転させてくれるのだと信じています。


　「出会いの場における、
　　ほんの少しのプラスイメージ」

　と

　「出会いの場における、
　　ほんの少しのマイナスイメージ」

　との間には、途方もない差があるはずです。



■１に0.99をかけ続けると、
　その数字は限りなくゼロに近づいていくでしょう。

　反対に、

　１に1.01をかけ続けると、
　その数字は無限大へと大きくなっていきます。


　こんなところから、毎日のちょっとした心がけの
　積み重ねが、いずれとんでもない差になる、

　という意味を感じて頂けるのでは
　ないでしょうか。



■田中角栄氏は、
　
　「頂上を極める為には、敵を減らす。
　　自分に好意を持ってくれる広大な中間地帯を作り上げる」

　ことが必要、と述べていたそうです。

　※角栄氏の秘書をしていた早坂茂三氏の著作に
　　記されています。


　【今日のお勧め本　オヤジの知恵】

　 http://amazon.co.jp/o/ASIN/4087474372/2ndstagejp-22/ref=nosim



■今からちょうど15年前、

　（将来の展望が見えず、苦しんでいた時に）

　この本の、この言葉に出会い、当時、愛用していた
　モンブランの万年筆で手帳に記し、

　以来、繰り返し眺めては、
　できる限り、実践するように心がけてきました。

　（言うは易く行うは難しで、
　　なかなか難しいものですが）



■すると最初は「一期一会」の出会いを大切にしようと
　するのですが、

　やがて、

　「出会った際に、自分にプラスの痕跡を与える力がない
　　（もしくは乏しい）」

　ことに気づくことになりました。

　そこから日常生活の作り替えが始めることに
　なったのです。



■出会いの場だけを頑張るのではなく、

　普段から努力し続けないと、
　肝心なときに相手の役に立つことができない、

　だから努力し続ける、
　という風に切り替わっていきました。


　
■この考えは最終的に、

　「他者へのお役立ち、貢献の軌跡を
　　継続的に可視化出来る形で確認していきたい」

　という思いに進化していくことになりました。

　
　メルマガやセミナーへのご感想をお願いしているのは、
　このプロセスを通して己に強烈なプレッシャーをかけるため。



■こうして今では、

　セミナーを開催したときに毎回、ご感想をいただくことを
　お願いしていますが、これには、

　「全力を出し尽くした上で、
　　参加下さった方からの客観的な評価を待つ」

　という巨大な緊張を伴います。

　リアルタイム通信簿のようなもの。


　
■ただ、その通信簿を頂戴することを恐れずに
　お願いしていくからこそ、

　よい評価をいただけるようになるために、


　「渾身の力を出し尽くす」
　
　「そのために日常生活においても努力し尽くす」


　という姿勢が醸成されていくのだと
　考えています。



■たとえば、

　先の土・日にセミナーを開催しましたが、
　たくさんの「通信簿」をお寄せいただきました。

　まだ一部しかアップできていませんが、
　
　こちらがそのときにいただいた、
　詳細な「通信簿」となります。


　＜9月29日開催セミナー＞
　http://www.2nd-stage.jp/houdankai/testimonials/testimonial120929d.html

　＜9月30日開催セミナー＞
　http://www.2nd-stage.jp/houdankai/testimonials/testimonial120930d.html

　
　（ご感想を下さった皆さま、ありがとうございます！）

　
　セミナー開催時には毎回、必ず、
　感想を頂戴することにしているのですが、

　このプレッシャーが自らに全力を出させるための
　原動力となるのです。　



■これは一つの事例ですが、
　ぜひあなたも、関わる人から

　「通信簿」

　をもらうように心がけてみられてはいかがでしょう。

　（具体的には外部からの評価システムを作る、
　　ということです）



■こうすることによって、

　自分の人格やら能力やら姿勢やらが客観的評価に
　晒されることになります。

　実のところ、これは凄く怖いこと。
　とてつもない緊張感を伴います。


　ただ、その恐怖に自らの身を晒し続けることによって、
　その分だけ成長も加速することだけは、

　間違いないのです。
　

　※毎回、前回よりももっとよい通信簿をいただこうと、
　　頑張っています。

　＜9月29日開催セミナー＞
　http://www.2nd-stage.jp/houdankai/testimonials/testimonial120929d.html

　＜9月30日開催セミナー＞
　http://www.2nd-stage.jp/houdankai/testimonials/testimonial120930d.html
