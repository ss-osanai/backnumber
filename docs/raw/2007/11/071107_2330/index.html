■人との約束は、とても大切。

　そして、自分自身との約束は、それ以上に大切です。

　他者との約束をだいじにしながら、
　自分との約束を最重要に扱っていく必要があります。



■ところが、この「自分との約束」、

　徹底して守ろうとすると、非常にむずかしいことが分かるのです。



■といいますのも、

　自分との約束は、たとえ果たさなくても、
　誰からも怒られません。

　誰も見ているわけでもなければ、命令されるわけでもない。

　監視できる人がいるとすれば、ただ一人、


　【 自分だけ 】


　です。

　「まぁいいや」

　と思ってしまえば、それっきり。



■ですから、


　【 自分との約束は、入れたなら守らなければならない 】


　という「鉄の決意」が必要となってくるのです。



■第一線に立って仕事をしていれば、
　次々と頼まれごともあり、面会のご依頼などもある。

　自分の生活時間を、すべて先様にあわせたら、

　24時間どころか48時間あっても自分の時間を確保する
　ことができなくなるかもしれません。



■しかし、どれほど忙しくても、

　「自分の人生は、自分でコントロールする」

　ことをあきらめてはなりません。

　
　そんな固い決意を持ちながら、自らの人生を律していかなければ
　なりません。
　


　今日もビジネスを楽しみましょう。
