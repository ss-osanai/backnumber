＜昨日の活動ダイジェスト＞

▼個別コンサル２件＋コンサル感想を頂きました

───────────────────────────────────



■こんにちは、鮒谷です。


　昨日は２件、個別コンサルでした。



■ここ数日の間にも、またたくさんの個別コンサルの
　ご感想を頂戴いたしました。
　
　今日はそのうちの一通をご紹介させていただきます。


　（ここから）
　----------------------------------------------------------------


　鮒谷様


　先日は個別コンサルをしていただき、
　ありがとうございます。

　１年ほど前からメルマガを読んでいて、セミナーに
　参加したいと思っていたのですが、毎回何かしらの予定が
　入っていたため行動に移すことができずにいました。

　今回初めて私の予定と個別コンサルの案内が
　見事にマッチングしましたので、即申し込みました。


　ところが、すでに枠は埋まってしまったと返信があり
　残念に思っていたのですが、特別に枠外で
　個別コンサルを受けてくださるという思いがけない展開に。

　改めて、行動（決断）すれば何かしらの結果が得られるのだと、
　申し込みの段階ですでに教えられました。

　しかも９０分コンサルがいつの間にか１２０分コンサルに。
　コンサルを受ける前から満足度１００でした。


　人生で初めてのコンサルだったので正直どうしてよいか
　どういうものかわからず、現在の悩み３つを準備して
　いったのですが、開始３０分でほぼ解決してしました。

　残りの９０分。コンサル慣れしていない私はどーなるのか
　戸惑いましたが、その９０分を使いわかりやすい例えを
　用いて、より深い気付きを与えてくれました。


　特に時間の使い方が刺さりました。
　１日２４時間の有限な時間をどこにどのように
　いつ投資していくのか。

　そして「今」本当に大切なことは何なのか。

　頭では漠然と思っていたけど、うまく行動に移せて
　いなかったことをズバリすっぱ抜かれ、
　思い切り背中を蹴飛ばされた気分でした。


　はい、やります。

　何よりコンサルの場で行動・決断を迫られる感覚が
　とても刺激的で、鮒谷さんが私を「お客様」ではなく
　１人の「後輩」として接してくれている感覚に陥りました。


　雲の上の大先生の言葉よりも身近な尊敬できる
　先輩の言葉の方が心に響くものですね。

　早速、コンサル当日に約束した３つの行動＋テレビやめる
　を実践しています。


　３日坊主にならぬよう、

　「上がったときに即決断」

　を実践して自分自身を気持ちよく追い込んで
　いきたいと思います。
　
　私のセカンドステージを楽しみにしていてください。


　ありがとうございました。


　　　　　　　　　　　　　　　　　　　　（小林 豊明さま）


　----------------------------------------------------------------
　（ここまで）


　小林さん、長文の、そして丁重なご感想を
　ありがとうございます！


　
■早速実践いただいているとのこと、嬉しいです。

　くれぐれも

　「３日坊主」
　　
　とならないよう、周りの方を巻き込みつつ、
　
　「逃げられない衆人環視態勢」

　を仕組みとしてつくり上げることが大切ですね。



■あと、お約束させていただいたとおり、
　目標達成された暁には、必ずご連絡下さいね。


　盛大に「祝賀会」を開かせていただきます。

　「セカンドステージ」に到達され、
　お祝いさせていただける日を楽しみにしています！




■それでは

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　千冊読め、全てはそこから  】

───────────────────────────────────



■先日、


　親しい友人である赤羽恵太さん（AKB、羅王）の父君で、
　
　「日本発の世界的ベンチャー」

　を1社でも多く生み出すことを使命として設立された、
　ブレークスルーパートナーズ株式会社の創業者で、

　それ以前は、

　マッキンゼーで14年、日本企業、韓国企業の経営改革、
　新事業創造に、パートナーとして取り組んでこられた

　赤羽雄二さん

　
　【今日のお勧め本　ゼロ秒思考 頭がよくなる世界一シンプルなトレーニング】

　 http://amazon.co.jp/o/ASIN/447802099X/2ndstagejp-22/ref=nosim

　※発売一ヶ月足らずで５万部突破とのこと


　のお話を伺う機会（プライベート勉強会？）がありました。

　
　
■その中で、参加者の一人から、
　
　
　「どんな本がオススメですか」
　
　「どんな本を読めばいいでしょうか」
　
　
　的な質問があったのですが、
　
　その時に赤羽さんが答えられた回答が、
　今日のメルマガタイトル、
　
　
　【 （四の五の言わずに）千冊読め、全てはそこから 】
　
　
　というものでありました。
　
　
　※正確ではないかもしれませんが、
　　このようなニュアンスでおっしゃられたように感じました。
　
　　まさに一刀両断（笑）
　
　
　
■でも、これは私もはっきり口に出して言ったことは
　ありませんが、常々思っていたことです。
　
　
　人生通しての累計で、

　例えば数十冊とか百冊二百冊程度しか
　本を読んでこなかった人が、
　
　「わずか数冊のお勧め本」

　を読んだ程度で変われるとはとても思えないのです。

　
　※ご質問された方が、そういう意図で質問されたのでは
　　ないことは十二分に理解しております。

　　断罪（？）しているわけではないので、
　　くれぐれも誤解なされないようにお願いしますm(__)m


　
■続けて、
　
　
　「小説なども読むように」
　
　「心の襞（ひだ）を作るように」
　
　
　と言われていたことも印象に残っています。
　
　
　
■おっしゃる通り、私も、

　偏ったジャンル（の本）のみを追いかけ続け、
　平板な人になってしまうことを恐れます。
　
　
　私も極力、時期ごとに異なったジャンルの本を
　読もうとしてきました。
　
　
　
■昔（特に中・高・大、そして社会人になりたての頃）は
　小説をよく読んできましたが、今は個人的な興味として、


　ノンフィクションであったり、
　
　数奇な人生を歩んできた人や、
　珍しい仕事をしている人の回顧録・回想録・自伝、

　
　などを読む機会が増えてきています。
