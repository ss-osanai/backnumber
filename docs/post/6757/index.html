6757号 「覚えめでたき存在」にならねば機会をもらえず、初速もつかない

■先のメルマガで

　「自分の名前を好印象とともに
　　憶えてもらうことはめちゃ大事」

　そんな話をいたしました。


　そのためにはお金の使い方も大切で
　どうせお金を使うなら

　「生きたカネ」

　にすることを意識したいもの。



■お金は、

　ここぞというときにケチらず、
　目立つように使うからこそ、

　生きてくる、
　ということがあるのです。



■手順を踏んだ値引き要請を
　一概に否定することはいたしませんが、
　
　場合によってはそうして浮かせたカネが
　信用、信頼の毀損によって

　「かえって大きな損失」

　になってしまうことも
　往々にしてあるから注意したいもの。
　


■話は変わりますが、

　いま、ある方から提案を受け、
　水面下で協業案件を
　進めている方があるのですが、


　そうなったきっかけは

　「長期（10年近く）に渡る
　　継続的な関わり」
　
　によって既に

　「だるまさんが転んだ方式」

　で距離が縮まっていたから、
　というのが多分にあります。


　
■瞬間風速ではなく、

　長らく継続的にお客さまとして
　お付き合いくださってきた方ですから、

　もちろん当社の売上にも
　長年、貢献下さっており

　（ありがとうございます！）


　その方からAsking（問いかけ）が
　あったからこその
　
　「ハイか、イエスか、喜んで」

　だったということです。



■もともとご縁がなかったり、

　あっても、印象に残っていない、
　さらには印象が良くなければ、

　決してそんな展開には
　ならなかったはず。

　その方は長期のお付き合いの上に
　好印象の残る方だったんですよね。



■協業するにあたっては、

　もちろん双方にメリットがある
　ことが大前提ですが、


　ただメリットがあればいいかというと
　必ずしもそういう話ではなく、

　その人との関係性が重要になってくる、
　ということ。



■これから書くようなことをお伝えすると、

　（そのつもりはないのですが）

　上から目線的な、悪いイメージを
　持たれそうで本意ではないのですが、

　それでもあえて記すと、

　
　自分だけの力で
　ゼロベースでなにかやるよりも、

　私（鮒谷）の持っているリソースを
　活用いただくことによって

　「０→１が早くなる」

　というのは
　間違いなくあるように思います。



■既にこれまでに蓄積してきた資源が
　たくさんあるので、

　その方の強みやご経験に
　私の所有する各種資源を組み合わせれば

　「立ち上がりも、加速も早まる」

　可能性が高くなるに違いありません。



■私（鮒谷）自身、起業直後の
　「０→１」のタイミングで、

　他の多くの方のお力添えや
　ご支援、ご協力によって

　「立ち上がりも、加速も早まる」

　ことを経験しているので、

　そうした力を貸してもらえるのであれば
　貸してもらうにこしたことはないでしょう。



■別に私（鮒谷）がどうこう、
　ということではなく、

　もしアウトプットの産出を
　ブーストさせてくれる人があれば、

　
　自分一人でなんとかしようとせず、

　然るべき人の力を借りたほうが
　スピードを早められますよ、

　という話です。



■ここまでの話が納得できれば

　「自分の名前を、ポジティブな
　　イメージとともに記憶してもらう」

　ことが極めて重要になってくる理由が
　よく分かると思います。



■自分の名前に良いイメージを
　持ってもらえるようになるほどに

　「力を貸してもらえる可能性は
　　どんどん高まる」

　こととなるのです。



■ただし、そのためには
　
　「前もって身銭を切っておく」

　必要があります。


　ここでいう身銭を切る、とは
　あくまで例えであり、

　お金を使いさえすれば良い、
　という話ではありません。

　
　
■ただし、

　うまく使えば「生き金」として
　有効に機能する場面が多々あるのも、

　一概に否定するつもりはありません。
　


■そんな比喩的にも実際においても
　
　「身銭を切る」

　姿勢の積み重ねが
　
　「自分の名前に力が付与される」

　すなわち、
　
　好感、ならびに、協力を頂戴するための
　強力な資源となるのです。



■この状態を地道に作り上げていくと
　なにを始めるにしても

　「初動の重さ」
　
　がなくなり、

　はじめからトップスピードを
　出せるようになりますから、

　しなくてもいい遠回りがなくなり、
　人生が楽しく、楽になっていくでしょう。



■だからこそ、特に若いうちは
　いざというときに

　「初速を出せる」

　ようになるための土壌、基盤を

　「身銭を切る」

　ことによって整えておくと
　良いでしょう。

　

■その手間を惜しむ人や、
　そもそもこうした意識を持たない人は、

　高い好感度とともに
　名前を覚えてもらうこともなければ、
　
　惜しみない協力を頂戴することも
　難しくなるから、


　初速がつかず、初動が重くなる
　（あるいは全く動かず）

　それゆえ元本を育てた先にある
　複利の力を味方につけることも出来ず、

　いつまで経っても同じところに
　留まる可能性が高まります。



■チャンスが欲しい、
　そのタイミングで初速をつけたい、

　と願うのであれば、

　（誤解を恐れずに書きますが）

　「どうすれば『覚えめでたく』の
　　状態に入れるか」

　の問いを回し続け、
　そのために身銭を切ることが大事、

　ということになるでしょう。


　
■いうまでもありませんが

　これを私（鮒谷）にしてください、と
　言っているのではありません。


　あなたから見て、
　これは！と思う人にとっての

　「覚えめでたき存在」

　になると人生変わりますよ、
　ということです。


　今日も人生とビジネスを楽しみましょう！
