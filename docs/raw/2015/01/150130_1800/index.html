＜今日のピークパフォーマンス方程式＞

【　三年勤め学ばんよりは、三年師を選ぶべし　】

───────────────────────────────────

　　

■何かを誰かから学ぼうとする際、
　私（鮒谷）が一番初めに確認するのは、

　「その人が本当に結果を残しているのか」

　という点です。


　たとえば本を読んで、
　それをそのまま読み上げているだけの人と、
　
　実際に自身で成し遂げた体験を持ち、
　語っている人との違いは、

　とてつもなく大きなもの。

　

■やった人にしか分からない、
　
　言語化することができない微妙なアヤ、
　あるいは機微、

　といったものがあるからです。


　それを学ぶには、それを経験し、
　さらには表現できる力のある人につくしかありません。



■成果を出していることが分かれば、
　次に見る点は、

　「その人が上げた成果は一過性のものなのか、
　　継続して上げ続けているものなのか

　　（最低でも10年以上、できれば20年、30年）」

　というところ。


　瞬間風速で結果を出すのは実のところ、
　そんなに難しいことではありません。


　しかし成果を継続して残し続ける、
　というのは並大抵のことではありません。


　だからこそ、

　一定期間以上、結果を出し続けている人（本物）から
　学ぶことが大切だと思っているのです。



■そんな方と出会うことができれば、
　それこそ人生は一気に飛躍、飛翔いたします。

　私もこれまで何名かの、
　そういった方々との出会いがあったからこそ、

　今の人生があると確信しています。


　自分の能力とか才能とか努力などではなく、
　ひとえに、


　「そういった方

　　（高井・岡芹法律事務所会長弁護士の高井伸夫先生や、
　　　かんき出版最高顧問の境健一郎さんはじめ、多くの方々）
　
　　との出会い」


　によって人生が一気に大逆転したからです。



■やはり、学びにおいては絶対に

　「誰から学んでも一緒」

　ではなく、

　「誰から教わるかに徹底的にこだわる」　

　ことが大切なのです。


　中国のことわざに

　「三年勤め学ばんよりは、三年師を選ぶべし」

　という言葉があるそうですが、


　この意味は、

　「三年かけて独学で勉強するより、
　　三年かけて師匠を探したほうが良い」

　といったようなもの。



■それくらい、何かを学ぶに際して、

　「誰から教わるか」

　は決定的に重要、ということですね。

　
　恥ずかしながら、

　この言葉を知ったのは、
　起業して随分と後になってからのことでしたが、


　耳にするよりかなり前から、
　けっこう意識して行っていたことで、

　「ああ、これで間違ってなかったんだ！」

　と思った記憶がありますし、

　そのおかげで今がある、と思っているのです。
