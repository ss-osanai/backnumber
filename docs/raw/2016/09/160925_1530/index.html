【 ポイズン！？「○○したいのに○○できない、こんな世の中じゃ」 】
　　　　
───────────────────────────────────



■ときおり、


　やりたいこともやれない
　こんな世の中じゃ、

　行きたいところも行けない
　こんな世の中じゃ、

　食べたいものも食べられない
　こんな世の中じゃ、

　飲みたいものも飲めない
　こんな世の中じゃ、

　学びたいことも学べない
　こんな世の中じゃ、

　読みたい本も読めない
　こんな世の中じゃ、

　買いたいものも買えない
　こんな世の中じゃ、


　「ポイズン！」

　ならぬ

　「仕方ない！」

　みたいな感じで、

　現状追認、諦めの境地に入っている人と
　出会うことがあります。
　


■これ以外にも、たくさんの


　○○したいのに○○できない、
　
　「ポイズン！」

　ならぬ

　「仕方ない！」

　の派生系はたくさんあるわけですが、

　「諦めたらそこで試合終了」

　です。
　


■現状を打破するためには、


　○○したいのに○○できない
　こんな世の中じゃ、
　
　「ポイズン！」（しつこいｗ）

　もとい

　「仕方ない！」
　
　なんていっている場合ではなく、


　やりたいこともやれない
　こんな世の中「で」、

　行きたいところも行けない
　こんな世の中「で」、

　食べたいものも食べられない
　こんな世の中「で」、

　飲みたいものも飲めない
　こんな世の中「で」、

　学びたいことも学べない
　こんな世の中「で」、

　読みたい本も読めない
　こんな世の中「で」、

　買いたいものも買えない
　こんな世の中「で」、


　「自分はどうするべきか」

　という問いを立てる必要があるのです。



■どうすれば、

　
　やりたいこともやれるようになるのか、

　行きたいところも行けるようになるのか、

　食べたいものも食べらられるようになるのか、

　飲みたいものも飲めるようになるのか、

　学びたいことも学べるようになるのか、

　読みたい本も読めるようになるのか、

　買いたいものも買えるようになるのか、


　文字通り、朝から晩まで考え続ける、
　さらには行動し続けるのです。

　
　天は自ら助くる者を助く、

　ではありませんが、

　こうした自助努力の末に、
　現状打破の突破口が開かれます。



■意に沿わぬことがあっても、

　「こんな世の中だから」

　ではなく、

　「こんな世の中で、どうすれば、
　　やりたいことをできるだろう」

　と脳内言語を置き換える。


　これを繰り返すうちに、
　行動パターンも変わるようになり、

　行動が変わるから結果（運命）も
　変わるようになるのです。



■無意識に回している脳内言語に
　バグがあれば、

　自分でも気づかぬ内に、
　思考が硬直してしまい、

　その思考が行動を規定し、
　行動が運命を決定づけるのです。


　
■○○したいのに○○できない、
　こんな世の中じゃ、
　
　「ポイズン！」

　ならぬ

　「仕方ない！」
　
　は、思考硬直の現れ。

　

■常に脳内で回している言語を

　ことさらに意識し、
　書き換えることによってのみ、

　「自らの人生を、
　　自らの手によって取り戻す」

　そんな気概を取り戻すことが
　できるでしょう。


　※以下のセミナーでは、
　　そんな話もしようと思ってます。

　　ポイズン！！！
