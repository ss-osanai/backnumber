【進化論】大阪コンサル案内！（枠数限定・先着順）個別コンサルティングを受け付けます

2014/07/16 16:22 
■こんにちは、鮒谷です。


　来週末、
　
　7月25日（金）から7月28日（月）まで
　大阪に出張することとなりました。


　出張の際は、いつも予定をびっしり埋めることに
　しているのですが、

　今のところ多少、時間に余裕があるので、
　
　【 個別コンサルティング 】

　の枠を幾つか設けました。



■ときどき、メルマガの読者さんより、

　「東京には行けないけれども、
　　ぜひ大阪でコンサル枠を作ってください」

　といわれることがあるのですが、
　

　関西近郊にお住まいの方、
　たまたま関西に帰省、出張されている方、

　これを機会にぜひご活用ください。



■以下、過去に、個別コンサルティングを
　受けて下さった方より頂戴したご感想です。

　内容、雰囲気など、
　参考にしていただければと思います。


　＜お客さまより頂戴した、個別コンサルティングのご感想＞

　http://www.2nd-stage.jp/consult/testimonials.html



■ちなみに通常、

　＜１時間　129,600円＞

　にてコンサルさせていただいておりますが、

　急なご案内、
　かつ大阪での一回限定コンサル（複数回お申込不可）のため、
　
　今回に限って特別に、


　＜90分  86,400円（実質55％オフ）＞


　にて承ります。


　
■これまでの相談内容の傾向としては


　＜法人様＞

    経営戦略
    ビジネスモデル設計
    マーケティング
    情報発信
    営業関連　　　など


　＜個人様＞

    キャリア相談
    独立起業に当たっての心構えや実践方法
    何らかの決意の後押し
    事業計画策定　　　　　など


　といった案件が多いです。



■現在の空き状況は、

　
　【１】7/25(金) 13:00-14:30

　【２】7/25(金) 15:15-16:45

　【３】7/25(金) 17:15-18:45

　【４】7/27(日) 9:00-10:30

　【５】7/27(日) 11:00-12:30

　【６】7/27(日) 16:00-17:30

　【７】7/27(日) 18:00-19:30

　【８】7/28(月)  9:00-10:30

　【９】7/28(月) 11:00-12:30


　の９枠です。

　（詳細はお申し込み後、ご連絡いたします）



■毎回、ご案内後、
　24時間以内、遅くとも48時間以内で満席になります。

　先着順にて枠を確保させていただきますので、


　「第一希望から順番に、できるだけ多く、
　　希望される枠の番号を記して」

　
　＜担当：長内（おさない）＞
　
　office@2nd-stage.jp

　
　にあてて、


　【大阪個別コンサルティング希望の件】


　というタイトルで、メールをお送りください。



■最近、随所でこの言葉を聞くようになりましたが、

　「ノーポチだけが人生を変える」

　のは真実です。


　（チキンレースではありませんが）

　躊躇なく、冒険の世界に飛び出していった人だけが
　どんどん新しい世界に飛躍していき、

　保守的になり、リスクを恐れ、
　前に進まなかった（進めなかった）人は、

　その場にとどまり続ける、

　双方の、たくさんの人物事例を見ての率直な感想です。



■だからこそ、私（鮒谷）も、
　目の前に流れてきたお誘いには


　「躊躇なく、先送りせず、その場のノリと勢いで」

　「乗るしかない、このビッグウェーブに」


　とばかりに飛びつくことにしてきました。


　その結果、想定通りの結果を得られず、
　撤退戦を戦わなければならないこともしばしばありましたが、

　全体としてみれば、だからこそ、
　世界を作り替え続けることができているように思えます。



■つまり、勢い、思い切り、を大切にして
　試行錯誤を繰り返してきたから、

　ようやくなんとか、

　「ある程度、コントロール感のある、
　　自由な生活を謳歌できる」

　ようになったのだと考えているのです。


　
■つまるところ、人生において、

　（安全と、部分最適を求めるのなら
　　冒険は止めたほうがいいけれど）

　全体最適を求めるのなら多少の失敗は恐れず、

　部分最適なんかクソ食らえ！とばかりに、
　決断し、冒険し、前に進み続ける、

　そんな姿勢が大切なのではないか、

　と考える今日この頃です。
　
　

■もしあなたが全体最適を求め、


　「躊躇なく、先送りせず、その場のノリと勢いで」

　「乗るしかない、このビッグウェーブに」


　という掛け声に乗って下さる方であるのであれば、
　ぜひ下記より冒険（お申込み）頂ければ幸いです。



■追伸その１：


　メルマガではとても書けないココだけの話（！）を
　お伝えさせて頂くこともあるかもしれません。

　「そこまで話してくれるの！？」

　と驚かれることも、よくあります。



■追伸その２：


　「時は金なり」

　ともうします。
　
　
　個別コンサルの一番の意味は

　「お金で時間を買う」

　ところにあるのでしょう。



■追伸その３：


　「知らないリスク」

　は想像以上に大きなもの。


　さまざまな失敗経験と、ほんの少しの成功体験と、
　たくさんの会食、講演、セミナー、読書遍歴を通して学んだ、
　
　当該分野における

　「知っておいたほうがいいこと」

　をお伝えしたいと思っております。
　


■あんまりダラダラ書いても仕方ないので、
　これくらいにしておきますが、

　久しぶりの出会いから、新しい出会いまで、
　どんな御縁をいただけるか、楽しみにしています！


　＜お客さまより頂戴した、個別コンサルティングのご感想＞

　http://www.2nd-stage.jp/consult/testimonials.html

　　希望される枠の番号を記して」
