「稼がねば」から「せずにはいられぬ」へ、という金儲けのコペルニクス的転回

2025/11/19 04:05

金を稼ぎたければ、「どうすれば稼げるか」ではなく、 「せずにはいられぬことを、どうすれば金に変えられるか」という問い に切り替えるといい。

私はこれを 「金儲けにおけるコペルニクス的転回」 と呼んでいる。

「どうすれば稼げるか」という問いの裏側には、「稼がねばならない」という生活の要請や義務感がある。この不純物が混ざるかぎり、長期にわたる集中は維持できない。

しかし「せずにはいられない」という状態には、 不純物が一切ない 。そこに あるのは、内側から突き上げる内発的欲求のみ 。

だから圧倒的な成果を出せるし、純度と強度が十分に高ければ、その分野の第一人者になることすら難しくない。

ちなみに日本に1億人いるとすれば、1000人に1人で10万人、10000人に1人でも1万人の見込み客がいる。 ニッチであるほど強烈なニーズが生まれるから、高額課金も成立 する。

そして今の時代、ネットを通じてその人たちの多くにリーチできる。 マニアは何が何でも検索し、辿り着く。

要するに、 純度高く一つのことを徹底的に突き詰めた人間が優勝する世界 に、我々は生きている。

この構造を理解せず、旧来型の「どうすれば稼げるか」という義務感モードで仕事していても、 息をするように対象へ15時間、18時間を投入してくる人間がいる以上、勝負にならない よ。
