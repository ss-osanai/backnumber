　あなたのビジネスのヒントになれば幸いです。

＜今日のピークパフォーマンス方程式＞

　【　異常値を取り、特長を生み出す　】

───────────────────────────────────



■先日の日経ビジネス誌（2012.2.6号）に、

　味の素社長、伊藤雅俊さんの
　インタビュー記事がありました。


　興味を持った内容だったので、
　ここにご紹介いたします。


　（ここから）
　--------------------------------------------------

　我社の場合、他社とは異なる高い比率で
　R&Dへの投資を続けることが特長を生み出しています。


　例えば研究者の数です。
　日本の味の素には研究者が1085人いて、
　その割合は全社員の約30％を占めます。


　しかも、そのうち約10％がドクターです。
　このような研究組織を持つ会社は、
　世界でも類を見ません。

　海外にも研究者を多く配しています。
　ロシアと中国に研究所があり、ロシアには113人、
　中国には95人の研究者がいます。


　食品業界の中で、これは異常とも言える割合です。

　しかし、異常なことをやらなくては、
　特長など生まれません。

　--------------------------------------------------
　（ここまで）



■特に、インタビューの最後にある


　「食品業界の中で、
　　これは異常とも言える割合です。

　　しかし、異常なことをやらなくては、
　　特長など生まれません」


　という言葉に感銘を受けたのです。



■伊藤さんのいわれている、


　「異常なことをやらなくては、
　　特長など生まれません」


　という言葉は企業のみならず、
　一個人でも同じことがいえるでしょう。

　
　ときおり私はセミナーを
　開催していますが、
　
　そこではいつも、

　「異常値を取りに行きましょう」

　という話をしています。



■よきにつけ、あしきにつけ、
　今という時代は、

　個人のスキルや能力といったものが
　インターネットを介して他者と比較相対され、

　「透明化」
　
　されていく時代。


　
■言葉を換えると、

　「半強制的に己（おのれ）の能力が
　　あぶり出される」

　世の中になってきていることは、
　紛（まご）うことなき事実です。

　平凡さは埋没につながります。



■かといって、その流れに背を向け、
　
　目立たず、ひたすらに暗がりに
　身をおいて、生きていこうとしたら、

　（それも一つの生き方ではありますが）

　現実問題として、チャンスの広がりは
　相当程度、遅くなるでしょう。


　なにもしなければ、発信している人に
　チャンスが優先して与えられるのが
　現実だからです。

　（同時にリスクも背負うこととなりますが）



■いずれにしても、

　自らの存在について発信している人が
　これだけ増えてきたとなると、

　そうこうしているうちに最終的には、
　多くの人が自覚なきままに、

　「（ネットを介して可視化された）
　　均一、同一の市場」

　の中で比較相対される世界に
　身をおかざるを得なくなってくるはず。

　徐々に逃げ場、身を隠す場所が
　少なくなってくるのでしょう。
　


■あらかじめ、
　そうと分かっているならば、

　腹を括り、覚悟を決め、

　「異常」を追い求め、
　「特長」を磨き、

　自分なりの「独自性」を追究していく
　しかなさそうです。


　長くなりそうなので、
　明日に続けます。


　今日も人生とビジネスを楽しみましょう！
