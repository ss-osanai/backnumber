　　到達したある結論

　▼気合、根性、やる気、全てゼロ人間が
　　動き続けるために必要なたった一つの決断とは

　▼なぜ、ただ作業するだけの
　　「丸一日、集中する会」にリピーターが多いのか

　▼先延ばしがあなたの脳内メモリを蝕む理由

　▼鮒谷が未来への投資として
　　意識を傾け、取り組んでいることとは

　▼実はビジネスが主目的ではなかった！？

　　「第二領域」となっているのか？

　　抜け出させるきっかけになるものとは

　▼鮒谷がランチ代に35,000円を投下してでも
　　会食を行う理由

　　語ってみた

　　大きく飛躍させた週末の過ごし方とは

　▼「果報は寝て待て」という危険な物語

　▼あらためて第二領域に取り組むための
　　考え方について

　▼やってもやっても成果が出ない、、、
　　そんな第二領域賽の河原状態のときにこそ、
　　持っておきたい、あるイメージとは

　　大いに削り取るものたち

　▼いやでも第二領域に取り組めるように
　　するための２つのポイントとは

　▼成果が出始めてきたあなたに知ってもらいたい、

　 人生においては「ドＭ」に自らを躾けた理由とは 】

───────────────────────────────────



■私（鮒谷）は、

　【時間を制するものが、
　　人生やビジネスを制する】

　と信じ、

　これまで時間管理については
　相当程度、意識を払って

　「システム」

　を構築してきたつもりです。

　

■さらにいうと、
　私（鮒谷）にとっての理想の生活は、


　【未来のことだけを考え、
　　未来を良くするためだけに、

　　時間投資をする生活】


　であると考えています。

　
　このような生活を実現させるために

　【人生まるごと、
　　第二領域化プロジェクト】

　と一人、勝手に称し、
　いろいろ工夫を凝らしてもきました。

　※第二領域＝緊急ではないけれども、
　　　　　　　人生において重要なこと



■現実問題として、

　こうした生活を完全に100%実行することは
　難しい側面もあるかもしれませんが、


　私（鮒谷）は、

　【日々、行っている一切の活動が、
　　未来を良くすることにつながっている】

　という状態を作りたいと
　考えています。



■このことについては、先の

　「年間プログラム」

　においても
　詳しくお伝えいたしましたが、

　本日は年間プログラムに参加下さった方への
　フォローアップも兼ね、

　以下に記します。



■人間は

　「よりよい未来」

　を創るための取り組み（投資）の
　大切さを理解しつつも、

　放っておくと、ついつい「今」ばかりを
　見てしまう傾向にあるように思われます。



■今、未来の事を考えると、
　ときに不安になったり、焦ったりするけれど、

　そんな感情を内包しつつも

　「今、楽しい、
　　今、楽ちん、
　　今、満足」

　という生活への欲求に抗（あらが）うことが
　困難な動物なのです。



■ただし、

　目先の欲求を充足させるのための日々を
　どれだけ送っても、

　（一時のストレス解消には
　　なるかもしれませんが）

　中・長期的に見たときには
　ジリ貧状態に陥っていく、

　ということは皆さん、
　既にお分かりのことと思います。



■後先を考えず、
　今さえけ楽しければそれでいい、

　という考えで、

　やりたい放題、放蕩三昧に過ごしていれば、

　右肩下がりの人生や生活になっていくのは
　必然的帰結。



■そんな生き方とは反対に、

　「今、苦しい、
　　今、辛い、
　　今、不満足」

　ではあるけれども、


　今の苦しい、辛い取り組みが、

　確実に明るい未来を生み出してくれると
　信じられる限りにおいて、

　
　ある時点においては
　マイナスに感じられる感情も、

　【長期的に見れば、今の苦労や努力が
　　実るときがかならず訪れる】

　と信じられるものとなるはずです。



■こうした回路が脳内に形成されれば、

　「今、辛い、
　　今、苦しい、
　　今、不満足」

　といった状態にあるからこそ、


　むしろ、この状況がもたらす感情が、

　将来に対する希望や可能性を
　感じさせてくれることともなるのです。



■こうした思考ができるようになると、
　今、辛いことが

　「辛（つら）楽しい」

　こととなり、

　今、苦しいのが、

　「苦（くる）楽しい」

　こととなり、

　今、不満を感じているのが
　
　「不満（ふま）たのしい（！？）」

　という世界に出ることとなるでしょう。



■こうした解釈が自然とできるようになると、

　【目先の欲求が押さえつけられている
　　今の状況に、

　　かえって喜びを見出せるようになる】

　のです。


　自己の内にこうした物語を埋め込むことが、

　【自分の世界認識や思考態度を躾ける】

　ことにつながる、
　そんな風に私（鮒谷）は考えています。



■仮に

　「今、目先の欲求に逆らう」

　ことを、蓄財に例えていうならば、

　「今、使ってしまいたい思いをこらえて、
　　元本を積み立て、さらには殖やす」

　ということになるでしょう。


　
■当座、少ない元本を大切に扱い
　（無駄遣いせず）、

　さらには金利の力を活用して、複利で
　回していくことによってタネ銭が膨らみ、

　やがてどこかの未来において、
　
　「大きな富（＝充足感や幸福）」

　をもたらしてくれることに
　なるはずです。



■反対に、

　今、手元にある元本を、

　「どうせ大した金額でないのだから、
　　パーっと今の楽しみに使ってしまおう」

　などと考え、
　無軌道な生活を過ごしていると、


　ただでさえ少ない元本は
　あっという間に枯渇し、

　未来に対して投資するべき
　タネ銭自体が失われる

　（＝お金や時間に追われる人生となる）

　のではないでしょうか。

　

■つまり、

　元手（未来を豊かにするための原資）を
　無駄に使ってしまうことほど、

　愚かな行為はないわけであり、


　それゆえ、

　【将来を犠牲にして、
　　楽しみを追求するのは愚の骨頂である】

　と言わざるを得ないのです。



■こうした認識が定着し、

　【今の苦労や努力が大きいほど
　　喜びを感じられる「ドＭの世界」】

　にまで至ると、


　自らを痛めつければ
　痛めつけるほど、

　明るい将来が約束されると
　認識できるようになり、


　傍目には
　
　「よく、あれだけ頑張れるものだ」
　「よく、あれだけ働けるものだ」
　「よく、あれだけ本を読めるものだ」
　「よく、あれだけ人と会えるものだ」
　
　といった風に言われるくらいの、
　圧倒的な行動量（負荷）を、

　何の苦もなく（というか、むしろ喜びながら）
　できるようになるのです。



■世間一般に「努力」と思われることを
　「努力」と感じることなく、

　むしろ喜びを感じながら
　平気で行っている人はすべて、


　脳内配線において
　
　「上記の回路」

　が日常的に回っているから、

　【他の人にとっての非日常が、
　　その人にとっての日常】

　となるのです。



■この世界に生きている人は
　
　【世間一般には「楽しみ」と呼ばれている
　　現実逃避行動（＝ごまかし）】

　とは一切、無縁であり、


　【未来に投資し続けていることから生まれる
　　ふつふつと湧き上がってくる喜び】

　を感じられるようになりますし、


　さらには、時間の経過とともに本当に、
　
　物質的にも、非物質的な面からも、
　満たされた世界に到達できるため、


　【一見、苦労しているように見える
　　今も良し、

　　今の果実を収穫する時期を迎えた
　　未来においても良し】


　という生活が実現なされることと
　なるはずです。



■以上のような理由から、
　私（鮒谷）は

　（プライベートにおいては「ドＳ」ですが
　　→どうでもいいｗ）

　人生や仕事においては
　「ドＭ」でありたいと思っていますし、

　自身を躾けているうちに本当に
　「ドＭ」になってしまいました笑
