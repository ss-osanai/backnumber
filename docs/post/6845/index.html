6845号 自分らしく生きる＝わがままな嫌われ者？

■昨今、

　「ありのままの自分で生きたい」
　「自分らしく生きていきたい」

　という人が増えているようです。
　私（鮒谷）も賛成です。


　本来の自分ではない人間に
　なろうとすることほど、

　苦痛なことはありませんからね。



■とはいえ、

　「ありのままの自分で
　　生きていきたいです！」

　「自分らしく生きたいっす！」

　といって、

　なんの配慮も工夫もなく、ありのまま、
　自分らしく生きている人があるならば、

　その人は単なる

　「わがままな人間」

　ですから嫌われ、疎んじられます。



■挙げ句、主義を貫こうとするほどに

　「ありのままの自分、
　　自分らしい自分とは程遠い状態で
　　生きねばならなくなる」

　でしょう。



■実は先日来、ご案内してきた
　「我田引水（技術）」とは、

　ありのままの自分を貫きつつ、

　摩擦なく、エレガントに生きていく
　ために欠くべからざる

　「生活技術」

　ともいえます。



■素の自分のままに
　生きていこうと思ったら、

　そんな（ある意味、わがままな）自分を
　他者に受け容れてもらう必要がある。

　そのための方法が
　「我田引水」ということ。



■こうした技術を持たぬまま、

　ありのままに生きようとすれば
　ただの、わがままな人ですし、


　だからといって、

　自分らしく生きることを放棄したら
　その瞬間から、生活のために己を殺し、

　「標準品」
　「規格品」
　「汎用品」
　「量産型」
　「金太郎飴」

　に成り果てた状態を目指すしか
　なくなります。



■一度、この道を選択すれば、
　死ぬまでやりたいことを辛抱し、

　他者から求められることに
　合わせにかかるより仕方ない。



■それでいいのであればいいですが、
　それが嫌ならば、

　自ら、自身の運命を切り拓き、
　幸せに生きるために何がなんでも

　自らの田んぼに
　水を引いてこなければなりません。



■ここまでお読みくださった方の中には
　なにかの音源でもお伝えした

　「鍵と鍵穴」

　の話を思い出された方も
　あるかもしれませんね。



■今日は詳しくは記しませんが、

　ここまでの話を
　「鍵と鍵穴」のたとえで表すなら、

　「（かんたんにスペアキーを作れる）
　　共通規格系・ピッキングされ放題の鍵」

　でいいのか、


　それとも

　「（スペアキーの作成自体が不可能な）
　　絶対にピッキングされない鍵」

　を目指すのか、
　とも言えるかもしれませんね。



■後者のような存在を目指し、

　自分らしく生きる人生を
　謳歌したいのなら

　「我田引水音源」

　を聴かれることをお勧めいたします。
