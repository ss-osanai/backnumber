6729号 認知の歪みをハックすることで人はいつからでも生き直せる

■どんな人も

　「自分の芝生は枯れていて、
　　隣の芝生は青々と茂っている

　（＝自分の持っているものは
　　　他人の持っているものよりも
　　　劣っている）」

　ように見えてしまうという、

　抗おうと思っても抗えぬ、
　根深い一般的傾向を持っています。

　特に若いうちは
　より強く発現する傾向がありそうです。



■他の人が持っている資源

　（知識、スキル、能力、性格、
　　人のつながり、経験、学歴、職歴、
　　閨閥、資産などなど）

　は過大に評価し、

　反対に自分の持っている強みは
　過小評価してしまう、

　そんな認知の歪みは
　どんな人にでもあるということ。



■この習性は卑下、卑屈、萎縮、

　場合によっては、
　嫉妬や羨望、妬みといった感情さえ、

　育み、増幅させる土壌と
　なり得ます。



■幸いにも、

　早期、かつ、相対的にキャリアや生活に
　恵まれる世界に出られた人は、

　他者から相応の称賛を得られ、それで
　心のバランスが取れたりするものです。


　あるいは肯定的な評価に
　レバレッジをかけて、

　正のフィードバックループに入る人も
　見かけますね。



■しかし、巡り合わせの関係で
　そうはならなかった人も、

　一定の、そして少なくない割合で
　生まれます。


　そうした人は心理的なバランスが
　最初はゆっくりと、やがて一方向に崩れ、

　認知の歪みが加速度的に強化される
　方向に進みます。



■その結果、

　他の人はあんなに優秀なのに
　自分はダメだ、

　と未来の可能性の芽を自ら摘み、
　卑下し、卑屈になったりもしますし、

　萎縮し、闊達に振る舞えなくなる人も
　出るでしょう。



■こうした人は
　ますます結果を出せなくなり、

　より一層、悲観し、卑下し、卑屈になり、
　萎縮し、闊達さが失われ、、、

　の地獄行きのサイクルが音を立て
　回転数を上げながら、

　急激に回り始めることとなります。
　（負のフィードバックループ）



■当人の資質とは無関係に、
　単なる「初期設定」の問題で、

　適材適所ならざる環境に身を置き、
　早期に成果を残せず、

　その上、心の制御を上手に行えなかった
　がゆえに起こりがちな悲劇。


　本来、可能性に満ち溢れた素晴らしい
　ポテンシャルがあっても発揮できない、

　そんな歯がゆく、情けない状態に陥る
　ことは、残念ながらしばしば起こるもの。



■この状態にはまり込むと視野が狭まり、
　余裕が失われてしまいます。


　自分を徹底的に痛めつけ、
　絶望に追い込み、

　同時に、他者の活躍を見ては恨み、
　憎しみ、人を呪わば穴二つ、

　という閉じられた地獄世界から
　抜け出せなくなってしまう人まで
　生まれることもあるのです。



■その根本原因は

　「（自分の芝生は枯れていて）
　　隣の芝生は青々と茂っている」

　と捉えてしまう人間普遍の心理から。

　ネットを見ていると不幸にも、
　こうした自縄自縛状態にはまり込んだと
　思われる方も散見されますね。



■ここまで行ってしまうと、
　当人が気づいていないだけで、

　新たな可能性を広げることに
　寄与するであろうと思われる、

　たくさんの手元にある強み
　（人生や仕事、キャリアを
　　より良いものにする編集素材）があっても、

　価値を正当に評価できなくなります。



■あらゆることを悲観するようになり

　「毒を食らわば皿まで精神」

　が発動されて、

　すべてにおいて投げやりで、
　怒りに満ちた姿勢となることすら
　あるかもしれません。



■こうした状態に落ち込まれた方も
　最初からそうだったわけではないでしょう。


　どこかのタイミングで抜け出す方途も
　あったはずなのに、

　その機会をたまたま
　手にできなかっただけ、

　と思いたいし、逆に言うと、

　いつでもそこから抜け出せる、
　と信じています。



■なぜなら私（鮒谷）自身も一度、

　地獄の窯を覗くに至るまでになった
　状態から、

　あれこれ試行錯誤した末に
　抜け出すことができたから。



■起死回生の生き直しの第一歩は

　「（自分の芝生は枯れていて）
　　隣の芝生は青々と茂っている」

　という認知の歪みの存在を知り、

　そのメカニズムをハックするための
　最善手を見つけたところからでした。



■つまり、

　私が他人の芝生を実態以上に青く捉えて、
　その物語に自分自身が編み込まれ、
　飲み込まれてしまっていたように、

　他の人もまた、
　見せ方次第で私の芝生を見て、

　同じように捉える可能性があることに
　気づいたのです。


　この構造を逆手に取って

　「ことさらに自分の芝生が
　　青く見えるようなプレゼンテーション」

　を反転・反攻戦略の一丁目一番地に
　据えました。



■生まれ落ちたときから
　今に至るまで挫折もなく、

　順風満帆な人生、キャリア、
　生活を過ごしてきた人は知りません
　（興味もありません）。


　しかし、人生のどこかで踏み外し、
　後のなくなった人に対しては、

　たとえ

　「あざとい」

　と言われようとも、
　ひとたび負の循環にはまり込み、

　絶望的な戦いを強いられる状況から
　抜け出すために、


　こうした認知バイアスが
　存在することを理解し、

　そのメカニズムを攻略するところから
　反転攻勢をかけてみては、

　と提案いたします。


　今日も人生とビジネスを楽しみましょう！
