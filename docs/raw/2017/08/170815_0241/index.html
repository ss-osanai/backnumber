■セミナーに参加下さった天野さんから、

　セミナー終了後、
　以下のようなご感想を頂戴いたしました。


　
（ここから）
　-------------------------------------


　この度は、井上達彦先生のセミナーを
　開催していただき、

　井上先生、鮒谷さん並びに関連業務の皆様、
　この様な機会を与えていただき、

　誠にありがとうございました。


　色々なワークを交えてのセミナーで、
　上や下を何度も見ながらの、

　あっという間の時間でした。


　「模倣の経営学」の一節に

　「本当の旅の発見は新しい風景を見ることではなく、
　　新しい目を持つことである。」

　マルセル・プルースト(フランスの作家)の
　言葉がありました。


　自分探しの、新しい目を持つ為に
　受講させていただきました。


　守・破・離を考えるワークで

　もっとよくしたい、もっと高みに到達したい、
　と強く思うからこそ

　アンテナを立て続け、お手本として、それを
　現在行っている人物や組織を探しあてることができる。


　近いところから模倣し、それでも事足りず
　遠くに模倣のターゲットを探す。

　その模倣から新しい価値が生み出される。


　模倣出来ない仕組みですら、
　模倣から生み出されている。


　ただ、その仕組みを生み出すには、
　鮒谷さんが目標設定で仰っている、

　強烈な引力のある行きたい場所を
　目標設定することが大切だ。


　ということを再認識いたしました。


　どうしても到達したい目標に、
　到達してない不満足な現状があるからこそ

　時間を使い、お金を払ってでも
　そこへ行く必要があるのであり、

　その様な目的がなければ
　そこまでの行動力は発動されない。


　お手本は外にしかいないのだけれども、
　目に見えるものはもはや模倣困難になっており、

　どこにお手本があるのかわからず、
　歩行困難な状態であると痛感しました。


　散歩ついでに富士山に登る人がいない様に、

　現状と違う世界に行くには、
　それなりのパワーと覚悟が必要ですので、

　自分を外に連れ出してくれる誘いには
　積極的に参加する。


　そして、新しい風景を見ることではなく、

　新しい目を持つことが今の自分には大切だと
　気付きました。


　自分を外に連れ出してくれる鮒谷さんの誘いには
　ハイか、イエスか、喜んで！

　自分の内側からは発見もなければ、
　模倣の対象は見つかるはずはないですね。


　強く明確な臨場感を抱く為には、
　自分の外からの刺激と

　高等なるフィクションを打ち立てて
　QOLを高めて行きたいと強く思いました。


　こうやって違った角度から生み出される気付きを
　いつも頂いてありがとうございます。

　これからも、どうぞよろしくお願い致します。


　（天野 郁 さま）

　天野さん、いつもありがとうございます！

　スミマセン（笑）



■でも、まさにおっしゃって下さっているように

　「自分を外に連れ出してくれる誘いには
　　積極的に参加する」

　私自身も、新しい世界を見せてくれるお誘いには
　都合のつく限り、

　「ハイか、イエスか、喜んで！」

　の精神で、出来る限り、その誘いには
　乗るようにしています。

　
　
■この繰り返しによって

　「世界が広がっていく」
　
　のだと思います。

　
　ここまで書いて、
　今日のメルマガネタが閃きましたので、
　
　続きは以下を御覧ください（笑）


　天野さん、これからも引き続き、
　よろしくお願いいたします！

【 「知らないから学ばない」ではなく「知らないからこそ学ぶべき」 】

───────────────────────────────────



■イノベーションと聞いて、

　「知らん、聞いたこともない」

　という人もあれば、

　「聞いたことはあるけれども、　
　　よく分からない」

　という方もあるかもしれません。



■知らないことは恥ずべきことでも
　なんでもありませんが、

　知らないから興味ない、
　分からないから学ばない、

　そのような姿勢は、

　「（個人的には）あんまりよろしくない」

　と考えています。



■なぜなら、

　何ごとによらず
　（「イノベーション」に限った話ではなく）

　一事が万事、

　学ぶ姿勢全般において、
　同様の習慣が身に付いているはずだからです。


　けれども、本来は、

　「知らないからこそ、学ぶ必要がある」

　のではないでしょうか。



■知らないから触りだけでも学び、

　その上で、このテーマについて、
　これ以上掘り下げて学ぶ必要がない、

　という風に思えば、
　その時点で撤収すればいいのです。



■学ぶ前から学びを放棄する方向に進んでいくと、

　時間とともに
　興味・関心の幅はどんどん狭くなっていき、


　最終的には、
　自分の身の周りのことしか興味を持たない、

　（あえて言ってしまうと）

　「話題の乏しい、魅力に欠け、
　　したがって、人も集まってこない人間」

　となってしまうことを、
　私（鮒谷）自身も危惧しているからこそ、

　【知らないからこそ、学ぶ必要がある】

　と考え、

　【そんなテーマが出てきたときこそ、

　　はいか、イエスか、喜んで！の
　　合言葉を発動し、行動に移す】

　よう心がけてきました。　
　


■人間、

　興味や関心や好奇心を持ち、
　自分を中心とする半径を広げ続けられる人と、

　その逆に世界を閉じていき続ける人が
　ありますが、


　現在の自分を取り巻く円の面積をより一歩でも
　向こう側（未知の世界）へと広げていこう、

　そんな姿勢の有る無しで、

　生きている世界が広がるか、
　閉じていくかが決まっていくのだと思います。



■ですから、たとえば、

　「イノベーション」

　の話で言うならば、

　この概念をある程度知り、学び、
　理解している人は、

　もはや学ばなくても良いのかもしれません。


　けれども、不思議なことには、
　あるいは、それが当然のことかもしれませんが、

　音源をお求め下さった方のお名前を
　拝見いると

　すでに十二分に学び、理解し、

　「もういいんじゃないですかね」

　という方ほど、ご購入下さっています。

　
　だからこそ、

　【常に成果を出し続け、さらには
　　学び続けられる環境にいられる】

　ということなのかもしれませんが。



■学ばない人は学ばないし、

　学ぶ人は、さらに深掘りして
　学んでいかれｒわけですから、


　長年の歳月をかけて生じる蓄積の差は、
　いずれ、いかんともしがたいほどの差となり、
　
　【巨大な学び格差】

　となって現れてくるのです。


　こうして生まれた学び格差は
　必然的に行動の格差、

　ひいては成果の格差に
　つながっていくこととなります。



■つまり、

　ある人と、別のある人の成果に、
　大きな格差があるとしたら、

　そこには

　【学びに対する根本的な姿勢に
　　起因することにあることを疑う必要が

　あるのではないでしょうか。


　あなたの周りにいる、圧巻の成果を残している人が、
　どのような態度姿勢で学びに対して取り組んでいるか、

　今一度振り返って見られるとよいかと。


　【わからないから学び、わかっているからなお学ぶ】

　これが圧巻の成果を出す人に共通する、
　心的ならびに、行動的習慣です。



　※以下、イノベーションという言葉を聞いて、

　　「知らん、聞いたこともない」

　　という人や、

　　「聞いたことはあるけれども、　
　　　よく分からない」

　　といった方にこそ、ぜひお聴き頂きたい。
