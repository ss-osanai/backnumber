6793号 「もてあましてるfrustration」を解き放て！

■古い曲になりますが、BOOWYの
　『MARIONETTE』の一節に

　「もてあましてるfrustration」

　という言葉があります。


　昔は曲を聴いて、
　この歌詞のところに差し掛かるたび

　「ああ、俺のことや、、、」

　と思ったことが
　懐かしく思い出されます。
　


■当時の私（鮒谷）と同じように、

　まさに持て余しているエネルギーの
　投下先を見つけられず、

　「何かしたいのだけれども、
　　何をしたら良いのか分からない」

　そんなフラストレーションを
　溜めまくっている人も、

　少なくないのかもしれません。



■こんな風に、
　もし毎日、忙しいけれども、

　単調で、退屈で、刺激がなく、
　未来の展望も見えてこない人があれば、

　それは単純に

　「未来のお楽しみのタネが
　　不足している」

　んですよね。



■大化けすることもあれば、
　腐ってしまうかもしれないけれども、

　継続的にタネをまいていれば
　一定の割合で開花し、結実します。



■今、没頭できる対象のある人は
　過去のある時点において

　「未来に向けて、お楽しみのタネを
　　まいていた」

　はずです。


　そうしたものの中の多くは腐って、
　発芽すらしなかったかもしれませんが、

　一部は芽が出て、膨らんで、
　花が咲いて、果実としての

　「今のお楽しみ」

　が生まれたのです。



■もしあなたにとって、

　今が単調で、退屈で、
　未来もこのままの予感がするならば

　「未来のお楽しみにつながりそうな
　　タネを大量にまく」

　しかありません。



■こうした地道な取り組みの末、
　一つ、二つ、
　
　「数百時間、数千時間、
　　場合によっては数万時間」
　
　といった時間投資の対象を持てたら
　楽しく幸せになれますよ。

　私（鮒谷）はこれによって
　充実した毎日を過ごせています。



■そうなったら
　
　「単調で、退屈で、刺激がなく、
　　未来の展望も見えてこない」

　どころか

　「楽しくて楽しくて時間がなくて
　　今の２倍、３倍の時間が欲しくなる」

　ものです。



■では、ここからは

　「具体的なタネのまき方」

　についてお伝えいたしますね。


　結論としては

　【（楽しさが）分かったらやる】

　ではなく

　【（楽しさが）分からないからこそ
　　やってみる】

　ことです。



■毎日を楽しく生きている人は
　日頃からフットワークが軽いもの。

　もちろん、フットワークが軽い、の
　裏側には　
　
　その分、時間がかかる、
　カネが出ていく、

　という側面もありそうですね。



■また、せっかくどこかに出向いたり、
　何かに取り組んでも

　「結局、何にもつながらなかった」

　なんていうことも
　しばしばあるものです。



■しかし、過度にそうしたコスト負担を
　恐れた結果として

　「一定期間経過の後、結果が分かる、
　　お楽しみ宝くじ」

　を買わなければ、

　人生から興味の対象が失われていき、
　　ジリ貧状態となり

　「気づかぬうちに
　　ゆっくりと世界が閉じていく」

　こととなるでしょう。



■このような経過を経て、
　
　世界がだんだんと閉じていき、
　あるところにまで至って、初めて強烈な

　「退屈や、単調さ、
　　刺激の乏しさへの痛み」

　を感じるようになるはずです。



■ここにおいて、それまで、

　フットワークの軽さから生まれる
　あれこれを「コスト」「リスク」と認識し、

　避けて遠ざけてきたことに対する
　しっぺ返しが倍返し、３倍返しとなり、

　我と我が身に
　襲いかかってくるのです。



■そのときようやく、遅ればせながら

　「一定期間経過の後、結果が分かる、
　　お楽しみ宝くじ」
　
　を一瞬、買おうと思っても、


　既に人生がフラストレーションに
　まみれている中で

　「宝くじの当選発表は後日」

　という現実を知って、
　諦めて堕ちていく人もあります。



■もちろん、そこから復活して、

　子どもの頃のように
　キラキラと目を輝かせながら、

　今が楽しい、明日も楽しみ、
　もういくつ寝るとのトキメキも思い出した、
　
　そんな世界に戻れる人も
　無きにしもあらずではあります。



■けれども、できることならば
　そこまで追い込まれて

　「人生が退屈」
　「人生がつまらない」

　となる前に


　「閉塞した人生に風穴を開ける
　　可能性のある、お楽しみ宝くじを、

　　時間や経済的余力があるうちに
　　一枚、また一枚と買い足しておく」
　
　べきではないでしょうか。



■もちろん、ここでいう

　「お楽しみ宝くじ」

　とは、本当の宝くじではありません。


　あんなものを買っていても

　偽物の夢と、虎の子のお金を
　交換しているだけですから、

　どんどん貧乏になるだけです。



■そうではなく、上述の通り

　「一定期間経過の後、結果が分かる、
　　お楽しみ宝くじ」

　をひたすら買い増していくのです。

■もちろん、世の中には無数の

　「一定期間経過の後、結果が分かる、
　　お楽しみ宝くじ」

■ただ、あなたがもし、冒頭の

　「もてあましてるfrustration」

　と聴いて、ギクッとし

　「俺のことじゃね？」
　「私のことじゃないかしら？」

　という思いが、

　たとえ一瞬間であっても
　よぎったとしたならば、
