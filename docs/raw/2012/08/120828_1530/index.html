  【　「てへぺろ」「雰囲気てへぺろ」は金輪際やめること　】

───────────────────────────────────



■昨日は

　「知らなきゃ調べろ」

　というテーマで、


　言葉を知らなければ
　自分を取り巻く世界は変えられない、

　知らない言葉に出会ったら
　すぐに調べることが大切だ、


　そんなお話をしました。
　今日も続けます。



■特に20代の若手ビジネスパーソンに向けて、

　日常的に用いられると思われる言葉を
　適宜、織り交ぜつつ話をしているとき、

　「？？？」

　という顔をされることが
　よくあります。



■あるいは、

　いかにも分かった、というような顔をしながら
　頷いているので大丈夫と思いつつ、
　
　念のために

　「この言葉、どういう意味か分かってるよね？」

　と確認してみると、

　「てへぺろ」

　されることもあります。

　
　（本当に「てへぺろ」されることはなくても
　　「雰囲気てへぺろ」状態になることはよくあります）



■それに対して、私（鮒谷）は

　「『てへぺろ』とかやってる場合じゃないんだよ。
　　馬鹿野郎！（怒）」

　なんてキャラでは残念ながらないので、

　その言葉を解説したり、
　すぐに調べるように、と言うわけですが、
　
　それにしても語彙の貧困さには
　驚かされます。

　（繰り返しますが、昔の自分のことは
　　ぜーんぶ、棚に上げておきます）

　

■当たり前以前の言葉と思って使っている

　「ことわざ、故事成語、四字熟語」

　をはじめとする言葉。

　知らなきゃ知らないでそれが現実ですから
　仕方ないのですが、
　
　分からなかったとき、その場で聞かなかったり、
　調べないことが一番の問題だと思っています。

　
　
■空気を読んでいるのかもしれませんが、
　本当に自分の将来を大切に思うなら、


　多少、空気をぶち壊したとしても

　「聞くは一時の恥 聞かぬは一生の恥」

　ですから

　「その場で聞いて、
　　その場で頭に叩き込むべき」
　
　と私は考えているのです。



■せめてそれくらいしないと、

　「博覧強記」

　の、この人にも、あの人にも追いつけず、
　永遠に引き離されていく一方じゃないですか。


　だから私も分からない言葉が出てきたら、
　
　「すぐに、その場で」

　を原則として、調べ、頭の中に刷り込みます。

　「あとで、そのうち」

　は絶対にやりません。



■まさかメルマガ読者さんの中には
　上に書いた
　
　「博覧強記」

　が分からない方はないと思いますが、

　（冗談抜きで、このレベルの単語に
　　「？？？」されることがけっこうあるのです）


　もし、万一、

　「？？？」

　という方があれば、

　「すぐに、その場で」

　ググるなり、辞書を引くなりして、
　頭の中に叩きこんでくださいね。


　
■この習慣が身につくと、
　時間の経過を味方につけて

　「語彙能力が複利で経年優化」

　していくこととなります。

　明日に続けます。


　（Ｙくん、この号は３回熟読するように）

　

　今日も人生とビジネスを楽しみましょう！
