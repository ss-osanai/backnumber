■年末年始もフル稼働

───────────────────────────────────

　　

■こんにちは、鮒谷です。


　おかげさまで（？）年の瀬になっても、
　全く年末感がありません。


　今日も先ほどまで、

　「アルトコインで大儲け（！？）」

　された方との個別コンサルでした。

　（と言いつつ、真面目なコンサルでしたよ）

　

■明日（30日）も朝から、

　毎年恒例の、年末に行っている、
　若手社会人向け勉強会、

　その後、コンサル（受ける方）へと続き、


　明後日（31日）も朝から個別コンサル、
　年が明けての1月2日も個別コンサルです。


　※幸か不幸か、

　　今年は1月1日の個別コンサルは
　　今のところ、入っておりません。

　　スタートダッシュをかけたいあなたからの
　　お申込みをお待ちしています（笑）



■このスタイルが定着してしまったので、

　今となっては、昔のように


　「年末だ！ → 気持ちの緩み →  

　　→ モチベーションの低下 → 昼夜逆転

　　→ 惰性の毎日 → 未来に繋がる行為ゼロ

　　→ 自尊心の毀損 → さらなるモチベーション低下

　　→ そろそろ休暇明け → 惰性の仕事始め
　
　　→ 軌道に乗ってくるのは1月10日過ぎ

　　→ 圧倒的な後悔、悔悟の念」


　という

　「残念極まりない10日間」

　を過ごすよりもはるかに良いと思っています。



■もし、昔の私（鮒谷）みたいな
　年末年始にしたくない！

　という方がいらっしゃるようでしたら、
　先日来、ご案内してきました

＜今日のピークパフォーマンス方程式＞

【 インフラが整った今だからこそ、手に入れておきたい２つの能力とは 】

───────────────────────────────────

　
　
■いい時代になったとつくづく思います。


　それは

　
　「どんな人とお付き合いをしたいかを決めて、
　
　　その人が価値を感じて下さるであろう
　　文章を書き、発信することができるなら、

　　本当に、望んだ通りの世界が現出する」


　ということが現実に起きるようになる、
　そんな時代が訪れたのだから。



■それもこれも、多くの人とつながり、出会える
　インフラ（＝ネット環境）が整ったからですね。


　自分の思いや考えを発信することさえ
　できれば、あとは勝手に

　「自動増殖」

　とでもいうべき勢いで、
　
　
　これまで対面したこともなかった、そして
　想像もしなかったような人たちと、

　簡単に結びつきが生まれるようになりました。



■たとえば私であれば、


　「好奇心や向上心旺盛で、勉強熱心、
　　そして人間関係を大切に思われる方、

　　さらに最低限、一定以上の文字を読める人で
　　ビジネスに少なからず関心のある方とつながり、

　　ズルをして、横着しようとする人は
　　近づいてこないでもらいたい」


　みたいなことを決めて、
　その思いが形になるように、

　想定読者対象に刺さると思われる
　メッセージを発信することで、

　本当に願っていたようなたくさんの方と
　お目にかかれるようになりました。
　


■そんな実体験を通して、
　私（鮒谷）などは、

　「このインフラなかりせば、、」

　といつも思っていますし、
　感謝もしています。


　ただし、せっかくのインフラも、
　
　「ある種の力」

　がなければ宝の持ち腐れとなります。



■ここでいう「ある種の力」とは、

　「発信するためのコンテンツ」

　ならびに

　「発信するための力」

　の二つを指して言っています。


　
■「コンテンツ＆発信力」は
　話し言葉でもいいですし、
　
　身振り手振りを交えた映像でも
　いいのですが、

　多くの人にとって、一番、取り組みやすいのは
　やっぱり文章ではないかと思います。

　
　然るべきコンテンツを持った上で、
　然るべき発信をすることができれば、

　世界は一気に広がります。



■あなたの友人知人にも、一人や二人、
　（あるいはそれ以上）
　
　実際に、生きている世界を大きく拡張された方が
　いらっしゃるのではないでしょうか。


　もしあなたが、今、こうした取り組みを
　行われていないのであれば、

　あるいは行っていても、そこまで本格的には
　取り組んでいない、ということであれば、

　ぜひ、そのための

　「方法論・お作法」

　を学ばれると良いのではないかと思います。


　
■具体的には

　「一次情報のネタを仕入れ（聞く力）、

　　自分なりの観点を持ち、
　　発信する能力（書く力）」

　といってもよいでしょう。

　私（鮒谷）がメルマガ創刊以来、
　やってきたことは、ぶっちゃけ、これだけです。
　

　この年末年始は、こうした能力を磨く、
　絶好の機会となるのではないでしょうか。

　この機会にぜひお求めを検討下さい。


　※休日、休暇のときこそ、

　　「勉強している人とそうでない人の差が
　　　最高潮に開くとき」

　　ですね。

　　だからこそ、私は世の中が長期休暇に
　　入ったとき、最高潮に盛り上がります。
