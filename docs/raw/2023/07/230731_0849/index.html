■金儲けが趣味の人（0.1％）、
　趣味が実益になっている人（4.9％）、
　それ以外の人（残り95％）

───────────────────


■既に副業、起業されている方は
　もちろんですが、

　この時代、

　「副業しよう、起業したい」

　という人でなくても、
　何が起こるか分かりませんから
　

　「もし副業、起業するなら
　　どんな感じになるのかの思考実験」

　をあらかじめ行っておいても
　バチは当たらないでしょう。



■逃げずにあれこれ考えてみることで
　気づくこともあるかもしれません。


　自分の将来を考えるときの
　一つの視点として

　▼やりたいことがない
　▼やりたいことがある

　と

　▼マネタイズ（収益化）できない
　▼マネタイズ（収益化）できている

　の両者の組み合わせを
　考えてみるのはありかもしれません。



■そうすると、

　-----------

　１、やりたいことがなく
　　　マネタイズもできていない

　２、やりたいことはないけれども
　　　マネタイズできている


　３、やりたいことはあるけれども
　　　マネタイズできていない

　４、やりたいことがあり
　　　それでマネタイズできている

　-----------

　の４パターンができますね。



■このうち、一番いいのは

　-----------

　４、やりたいことがあり
　　　それでマネタイズできている

　-----------

　でしょう。

　「趣味と実益を兼ねて」というやつ。



■さらにうまくいくと、

　「趣味で楽しくやっているのに
　　儲かって仕方ない」

　ということになるかもしれませんね。



■反対に、一番キツいのが

　-----------

　１、やりたいことがなく
　　　マネタイズもできていない

　-----------

　でしょう。


　楽しみがない上に、
　お金にも恵まれない。

　これはなかなか辛そうです。



■次にどれがいいかは
　意見が分かれるかもしれません。

　-----------

　２、やりたいことはないけれども
　　　マネタイズできている
　
　-----------

　あるいは

　-----------

　３、やりたいことはあるけれども
　　　マネタイズできていない

　-----------

　のどちらがいいのか。



■「２」の人、つまり、

　「やりたいことはないけれども
　　マネタイズできている」

　人は、数は少ないですが
　たしかに存在しています。


　
■実は、この種の人は、

　「やりたいことはないけれども」

　といいつつ、
　その本質は

　「金（＆金儲け）そのものが好き」

　なんですね。


　
■なので、

　別にやりたいことがなくても
　構わないし、

　儲かりそうなものを見つけては
　脈絡なくなんでも金に変えていく、

　こうして収益を上げるのを
　楽しんでいる人。



■この人は、カネになるのなら、
　何でもやりたいと思っているし、

　またそれが強い動機となって
　本当に金も儲かるという、

　要するに

　「金儲けそのものが趣味」

　の人なのです。



■こうした人はたしかにありますが、
　世の中全体から見ると極めてレア。

　ほとんどの人には無縁の精神性と
　言えるでしょう（私（鮒谷）もそう）。

　なかなかに模倣困難です。

　

■一般的には、ほとんどの人は

　「それが好きだから、
　　やりがいがあるからというところに、

　　喜びを見出し、仕事をして
　　お金ももらいたい」
　
　と考えるもの。　



■とはいえ、現実問題として

　-----------

　３、やりたいことがあるけれども
　　　マネタイズできていない

　-----------

　というのも、厳しいもの。


　やりたいことがあり、

　それを仕事にしたい、
　あるいは既に仕事にしているけれども、

　それがお金に変わらない。



■すでにそれを仕事にしている人も、

　お金が続かなければ、
　商売を畳むしかありませんし、

　いつか仕事にしたいと願う人も
　お金がなければ生きてはいけません。
　


■したがって、
　この道を選択するのが許されるのは、

　（ほぼ失業の可能性がなく、
　　ときには残業もあまりない）

　地方公務員の人、

　もしくは、

　実家が太いとか、
　大金を相続したとか、
　巨額の宝くじをあてた人、

　くらいなものでしょう。



■こうした方が

　「趣味に生きているうちに
　　ときに大化けする」

　のを、

　なにかの記事で見ることもありますが、
　これはこれで高いハードルです。



■綺麗事を抜きにすれば、

　普通の人は、好きなことがあって、
　表面的には楽しくても、

　経済的な不安を抱えている限り、
　不満とか、不安が消えることがない。



■ここまでの話で、すでに

　-----------

　４、やりたいことがあり
　　　それでマネタイズできている
　　（趣味が仕事になっている）
　
　-----------
　
　あるいは

　-----------

　２、やりたいことはないけれども
　　　マネタイズできている
　　（金儲けそのものが趣味で
　　　うまくいっている）

　-----------

　人は、もう満足しているから
　おいておくとして、


　問題は

　-----------

　１、やりたいことがなく
　　　マネタイズもできていない

　-----------

　３、やりたいことはあるけれども
　　　マネタイズできていない

　-----------

　という人です。



■前者は、

　【好奇心がもたらしてくれるパワーや
　　エネルギーの大きさを思い出す】

　必要があるでしょう。


　後者の人は

　【没頭没入の対象をお金に変えるための
　　考え方やノウハウやテクニック】

　を学ぶ必要があるかもしれません
