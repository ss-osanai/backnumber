7131号 中学受験はゲーム感覚で

■厳しい物言いになりますが
　世の中には

　【何を言っても無駄な人】

　がいらっしゃるんですよね。
　それも少なくない人数。



■「その気」になっていない人には
　何をどう伝えても無駄。

　結局、当事者意識がないからです。
　何かを伝えても意味がない。

　その典型が

　「子どもの勉強」

　でしょう。



■私（鮒谷）は中学受験組でしたが、

　受験に向けての勉強は
　本当に楽しかったんですよね。


　日々、膨大な知識がインプットされて
　いくのを感じられるのが嬉しく、

　既に身につけたことを土台に、より高度な
　問題にチャレンジする喜びもある。



■学校に行っている時間以外、

　起きている間は朝から晩まで
　ずっと勉強していましたね。

　それがまったく苦にならない。

　楽しくて楽しくて仕方なかったので
　なんならもう一度、
　あの頃に戻りたいくらい。



■今となっては信じられないのですが、

　大人でも解けないような、
　中学受験特有の、あの難解な算数も、

　パズルを解く、くらいの感覚で
　夢中になって向き合っていました。



■それに毎月、校舎内で順位が
　掲出されるのもよくできた仕組みでした。

　そこにライバルの名前を見つけては
　喜んだり、悔しく思ったり。

　まるでゲームです。
　楽しくないわけがありません。



■これこそが当事者意識です。
　完全に「自分ごと」になっていました。

　自分のことだからこそ、
　全力傾注するし、没頭もできます。

　だから成績もみるみる上がる。



■反対に、

　「勉強なんかをやる意味、
　　どこにあるんだろう」

　「つまらない、辛い」

　「一刻も早く、
　　苦行から開放されたい」

　と思っていたら、

　どれだけ頑張らせようと思って
　無理やり、机に向かわせても、

　持続するはずがありません。



■そして、この部分については

　【子どもも、大人も同じ】

　なのです。


　究極のところ、

　人生に対して、仕事に対して
　問題意識を持っていない人には、

　何をどう伝えたところで
　やる気がないのだから、

　やらないし、
　やっても続くことはありません。



■だから冒頭に

　「何を言っても無駄な人がいる」

　と書いたのです。

　その人がその気になるまでは
　外部からの働きかけになんの意味もない。

　機が熟すまで
　「放置」するしかありません。



■反対に、当事者意識のある人は、

　主体的精神を持って
　問題解決に取り組みます。


　それでも、うまくいかず、
　ときに落ちることもあっても、

　それでも再起動をかけて
　なんとか巻き返そう、

　との精神は持続するものです。



■その精神は己を再び立ち上がらせ、
　行動させ、前進に導くもの。


　この繰り返しによって
　少しづつ前に進んでいき、

　やがてあるところから、
　変化している自身に気づき、

　それが喜びになり、自信ともなり、
　さらなる駆動力を得る。



■やがて、時々に小休止を交えても
　最終的には

　「別人に作り変わっていく」

　こととなるでしょう。

　そんな方を
　これまで何人も見てきました。


　その姿を見て

　【変化・変容においては
　　当事者意識の有無が決定的に重要】

　であると知ったのです。



■当事者意識のない人は、

　「人生において
　　課題を持っていない人」

　と言い換えることもできますが、


　そうした人は、

　「大休止」して座り込んだら最後、
　前に進むことを諦めますから、

　何を言ってもやっても、
　無理やり掴んで歩かせようとしても、

　テコでも動きません。



■昔から

　「馬を水辺に連れて行くことはできても
　　水を飲ませることはできない」

　と言われる通りです。



■あえて言ってしまいますが、
　年間プログラムは

　【当事者意識を持たれている方のみが
　　参加している場】

　です。

　そうでない人が参加されても
　私（鮒谷）は何もできませんし、

　お金と時間の無駄だから
　参加されないほうが良いでしょう。



■一方、

　「現状はどんな状況であれ」

　人生、このままでは絶対に
　終われないと思っている人には、

　お役に立てるであろうあれこれを、


　さまざまな角度から
　徹底的にお伝えしていきますし、

　周りの参加者も喜んで
　サポートしてくれることでしょう。

　（そういう文化が育まれています）



■結果のところ、

　当事者意識だけは自分で用意しなければ
　何も変わらない、

　ということですね。

　身も蓋もない話ですが。



■自分がどちらのタイプであるかは、

　胸に手を当てて考えれば、
　すぐに分かるはず。


　よく

　「こんな私でも年間プログラムに
　　参加する資格はありますか」

　とありますが、
　現状の状態などなんの関係もありません。



■昔の遺産（？）で、

　それなりのトラックレコードを
　持っている人であっても、

　今、人生や仕事、キャリアにおいて
　問題意識がゼロの人は、

　お越しいただいても困りますから

　「参加資格はナシ」

　と申し上げます。



■しかし、

　今、特筆すべき能力や成果、実績、
　キャリアが一切なかったとしても、

　今、人生や仕事、キャリアにおいて
　問題意識がある方は

　「参加資格は、大アリ」

　とお伝えいたします。



■そして、おそらく間違いないことは、

　このメルマガを
　お読みくださっているあなたは、

　誰がなんと言おうと

　【人生と主体的に向き合っている人】

　です。



■そうじゃなきゃ、

　こんなメルマガを、
　しかもこんな下のほうまで（！）、

　読まれることなど
　あるはずがありませんからね。

　つまり、そんなあなたのような方こそが
　参加対象。



■私（鮒谷）が

　「変化・変容の旅」

　に出たときも、まさに

　「今、特筆すべき能力や成果、実績、
　　キャリアが一切なかったとしても、

　　今、人生や仕事、キャリアにおいて
　　問題意識がある人」

　でしたからね。



■そんな人間でも、

　当事者意識を持ち、
　全てを自分ごととして捉えられるなら、

　適切な教育を受けられれば、
　みるみる変わっていくものです。


　私（鮒谷）にとっての教育は

　「高井伸夫先生との出会い」

　から始まったわけですが。



■もちろん私（鮒谷）は、

　高井先生ほどの、
　高い水準の教育を行えませんが、

　「（自称ではありますが）
　　直系の弟子」

　として、


　模倣と研究と実践に次ぐ、
　模倣と研究と実践を重ねて、

　（私の中学受験時代のような）

　「人生がどんどん作り変わっていく、
　　そんな喜びや感動や興奮や高揚感を
　　感じていただく」

　ためのお手伝いを再現性高く、
　行えている、

　という自負があります。

　「とはいえ、年間プログラムが
　　どんな内容か、よく分からない」
