■ピュアな好奇心、探究心、知りたい、
　学びたい欲求が、学びには一番大事

───────────────────



■一昨日のメルマガでは、
　良い言葉を

　【ロールプレイングゲームにおける
　　アイテム】

　に例えて、

　「アイテムがあるほど、
　　人生は楽に、楽しくなる」

　という話をしました。



■さらに昨日は、

　「ヒスイ探し」と「言葉探し」には
　共通する６つのポイントがある、

　とお伝えしました。


　※６つのポイントは、これ↓

　-----------

　１、現地に行かないと採れない

　２、現地に行っても
　　　強い意思や意欲がないと採れない
　　　
　３、意欲はあっても
　　　目利きができないと採れない

　４、探索のための創意工夫が必要

　５、努力も要求される
　　（時間や精神力や体力が求められる）

　６、最後は運

　-----------


　今日も続けます。



■昨日紹介した番組
　（ドキュメント72時間）の


　「新潟 ヒスイ探しの海岸で」
　https://www.nhk.jp/p/72hours/ts/W3W8WRN8M3/episode/te/ZNNN57G66Z/


　（以下、番組紹介）
　-----------
　
　初夏の日本海を臨む新潟県の
　とある海岸。

　実はここ、宝石のひとつ
　ヒスイの原石が拾える特徴がある。


　まるで天然の宝探しで、

　すぐに見つかることもあれば、
　１日かけても見つからないことも。


　ヒスイか見分けられないまま
　探す家族連れや、

　ヒスイ探しが趣味と
　定期的に遠方から通う常連組。


　ここに通う中で、家にこもりがちだった
　生活が変わったという女性もいる。

　みんなどんなワケでやってくるのか。
　３日間、波打ち際で耳を傾ける。

　-----------
　（ここまで）


　を改めて見返してみたのですが、
　やっぱり良かったです。



■趣のある言葉もいろいろと。

　いわく、

　「大人の宝探し」
　「冒険感がある」

　など。

　言葉探しに通ずるものがありますね。



■午前３時からヒスイを探している
　年季の入ったおじさんが登場、

　「短期間で結果は出ない」
　「努力しかない」
　「ないときはない」

　言葉探しも、また然り。



■毎回、車中泊２～3泊して、
　新潟の海岸を訪れる人も登場。

　工夫をこらした自作道具で
　ヒスイ探しをしている人あり。
　
　どれがヒスイかわからない
　素人さんもたくさん登場。


　プロと素人さんが混在するのも
　言葉探しの世界と同じ。



■また、

　ヒスイに見えて、ヒスイではない、
　キツネ石というものもあるそうで、

　両者の見極めは初心者には難しく、
　鑑定が必要というのも、

　言葉探しに似ていますね。



■それっぽく（正しく）聞こえる、
　おかしな言葉を掴んで、

　それで痛い目（？）に遭う人がいるのは
　さながら

　「キツネ石をヒスイと勘違いした
　　初心者」

　と言えるかもしれません。



■投資の世界は、

　キツネ石をヒスイと騙して
　売ろうとする輩もたくさんいる、

　百鬼夜行の世界であり、

　キツネ石をヒスイと信じたい
　願望強め（情弱とも）の人もいたりして、

　カオス化していますが、


　新潟の海岸は

　「たとえキツネ石であっても
　　思い出が残ればそれでいい」

　という平和な感じなので
　そこは唯一、異なる点かもしれません。



■話を戻して、

　老若男女、
　みんな楽しんでいるのも、

　ヒスイ探しの世界と
　言葉探しの世界の共通点。


　小さい子どもから、
　おじいさん、おばあさんまで、

　みんな一緒に楽しめる。



■また、

　心の調子を崩したけれども
　ヒスイ探しで元気になった、

　という方も登場。


　言葉探しで心が元気になる人も
　たくさんありますから、

　こんなところも両者、
　相通ずるところが。



■石ころばかりの中、

　まれにヒスイが見つかる
　（まれにしか見つからない）

　ところもまた、
　言葉探しと同じですね。

　

■ヒスイ探しは
　やったことがありませんが

　言葉探しもきっと、
　おんなじくらい楽しめますよ。


　一生物の道楽になり得ますし、
　趣味が実益を生むことがあるところまで、

　ヒスイ探しと言葉探しが
　似ているところですね。



■、、、そんなわけで、

　以下は、今度開催するセミナー内容の
　事前チラ見せとなりますが、


　学ぶに際しては、

　儲かるとか、出世するとか、
　もちろんそういう目的もありですが、

　ときには純粋な喜びを求めての学び、
　というのも良いものですよ。


　
■ヒスイ探しも、

　目の色変えて、
　ヒスイを見つけてカネに変える、

　みたいな、直接的な利益を
　求めている人はほとんどおらず、


　いたとしても、

　楽しんでいるうちに
　いい石見つけたら売ることもあるよ、

　くらいな感じの人が
　少しあったくらい。
　

　
■言葉探しにおいても、

　これくらいの、ピュアな精神からの
　学びの結果として
　
　「現世利益（？）」

　につながることもあるものです。



■むしろ、私（鮒谷）なんかは
　そっちのほうが多いですね。

　言葉に惹かれているうちに
　なんだか利益にもつながったぞ、と。


　
■おそらく、というか間違いなく、
　講師の松澤さんも、

　そちらタイプ（？）だと思います。

　「趣味と実益を兼ねて」の世界へ、
　ようこそ！


　（セミナーの内容、ここから）
　-----------


　年間2万足を売る
　便所サンダル専門通販サイト、

　ない街の地図を作るプロフェッショナル、

　軽妙な下ネタトークでリピーターを
　生み出しまくる大人の博物館……。


　世の中には驚くべきニッチなジャンルで、
　圧倒的1位のポジションを確立し、
　独自すぎるビジネスを営む人間がいる。


　商売と生き様が、たぎるような熱量のもと
　一体化する、いわばマニアな人々。


　講師は合同会社別視点代表の松澤茂信。

　マツコの知らない世界・タモリ倶楽部に
　出演経験があるマニアが多数出展し、

　例年1万人以上の来場者が押し寄せる
　人気イベント「マニアフェスタ」の主催者で、

　月間100万ＰＶ・Twitterフォロワー
　10万人の珍スポット紹介ブログ
　「東京別視点ガイド」の運営者でもある。

　
　日本中のあらゆるマニアと交流し、
　自身もマニアな事業を展開する経験をもとに、

　事例とエピソードトークをたっぷり交えて、
　ニッチジャンルで独自ビジネスを
　生み出す方法をお伝えする。

■やっぱり、
　学びにおいて一番大切なのは、

　「ピュアな好奇心、探究心、
　　知りたい、学びたい欲求」

　だと思いますね。


　
■雑念（現世利益の追求）だけだと、
　手段とかテクニックばかりに走って、

　楽しくない上に、結局、
　儲かりもしないんですよ。

　あなたの知的好奇心にヒットしたのなら、

　ただ、もう、それだけで
　参加するべき理由になると思います。
