　「お礼を言われない仕事は
　　仕事ではない」

　と思っているのです。


　お礼を言われるような
　仕事をしていれば、

　自尊心が高まり、
　誇りも持てるし、

　当然、経済的にも潤う。

　精神の安心、安定が約束され、

　人生、生活をより一層、豊かで
　安全なものにしてくれるでしょう。



■ですから

　「どんな心がけで
　　仕事をすればよいですか？」

　と尋ねられたら、
　間髪をいれず


　「お礼を言ってもらえるような
　　仕事をして下さい」

　（そして同じ言葉を
　　自分にも言い聞かせています）

　なんだかいやらしい感じを
　持たれる方もあるかもしれませんが、

　なんら恥じることはありません。


　お礼を言われるような仕事をしたら、
　感謝され、評価され、

　それが最終的に自分に
　（様々な形を取って）返ってくる、

　という、ただそれだけの話です。



■ここでのポイントは

　「お礼を言われるために」

　他者に尽くそうとすると　
　必然的に

　「そのためには、
　　どんな価値を提供するべきか」

　という自問自答が生まれる、

　この順序が大事だということ。



■最初に

　「どんな価値を提供するべきか」

　という問いを
　持ってくるのではなく、


　「どうすれば
　　お礼を言ってもらえるだろう」

　と問い、そのためには
　
　「どんな価値を提供するべきか」

　という順番で
　自らに問うていったほうが、


　圧倒的に臨場感が生まれ
　当事者意識を持てるようになる、

　からです。



■なんといっても

　「どうすればお礼を言われるか」

　の問いのほうが、
　生々しいですからね。

　リアリティを持って、
　真剣に考えるようになります。



■これが

　「どんな価値を提供するべきか」

　という問いになると、
　途端に臨場感が失われます。


　大事な問いではあるのですが、
　まずは

　「お礼を言われる」

　というイメージをつけやすい
　目的を持ち、

　そのためには今、
　なにをなすべきか、

　を考えると良いでしょう。

■変化していることが常態である。

　何も変化していないように見えて、
　時代は常に変化している。


　変化のスピードが速いのか
　遅いのかの違いでしかない。


　変化のスピードが遅い場合は、
　
　少しだけ努力すれば
　時代の流れについていける。


　しかし、

　新型コロナウイルスにより
　強制的に引き起こされた
　時代の変化のスピードは、

　変化の方向を考えて、

　自分を変えていかないと、
　淘汰されることになる。

　変化を拒絶すれば、
　あとがどんどんきつくなっていく。


●感想

　変化のスピードが急激に
　なっていることがよくわかる。

　昨日まで常識であったことが
　今日は非常識になっている。


　特に非常事態宣言が出てから、
　いろいろなことが変わってきている。

　テレワークに関してはこの時初めて
　対応することになった。

　移動時間が無くなり非常に効率的に
　時間を使える。


　なぜ利用しなかったのか。

　変化の流れの中に
　自分を置いていなかったのではないかと
　反省するばかりである。


■今までは、ありえないことが
　あまり起こらない、

　変化がゆっくりした時代であった。

　これからは変化のスピードが
　加速していく。


　この変化のスピードは
　世界のパワーバランスを変えていく。

　そして、先が見えない時代に
　なっている。


●感想

　高度成長の時代から
　昭和の終わりくらいまでは、

　過去の状況から、未来を
　予想することが難しくなかった。


　平成の時代になると
　変化のスピードがだんだん速くなり、

　今は二次曲線のように
　変化のスピードが速くなっていると思う。


　そのように考えると
　今までの経験から未来を予測することは
　不可能であり、

　今までやってきたように
　過去の延長線上で未来を予想したにしても、

　変化のスピードが速すぎて
　対応ができていない。


　それはまるで、
　１次曲線と２次曲線の差のように、

　時間が経過するほどに
　差が開いていくようだ。


　多くの人はそこに差があることが
　見えていない。

　あるいは見えていても
　見ないようにしているのではないか。


　今回の話を聞いて
　その認識を新たにした。


■先が見えないのであれば、
　行動するしかない。


　会社にしがみついて
　守りに入っている人は大変である。


　国が、会社がと文句を言って、

　しがみついて
　安定を求めていたはずなのに、

　それが原因で不安定になっている。


　変化の流れとして、個人が
　副業を行うことが一般になってきている。


　会社の中だけではなく、
　自分の経験を生かした副業を持ち、
