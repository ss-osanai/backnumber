【 「シャセリオー展」のタペストリーを鑑賞して気づいた、
　　日記を記す効果・効能 】
　
───────────────────────────────────



■先日、

　【日記の書き方放談会音源】

　をお求め下さった松永さんより、
　以下のご感想を頂戴いたしましたので、
　
　ご紹介させて頂きます。


（ここから）
　--------------------------------------


　鮒谷様、


　この度は貴重なお話しを聞かせて頂き
　ありがとうございます。


　私は5年前、仕事で悩んでいるときに、

　3日間セミナーに参加したことのある
　前職の先輩のすすめで、

　平成進化論のメルマガ読者となりました。


　鮒谷さんのお話に感銘を受け、
　ブログを書いてみたり

　（これは続きませんでしたが・・・笑）、

　積極的に人に会いに行くなど、
　思考や行動がほんの少しながら変わりました。


　しかしその後、その先輩のアドバイスや
　平成進化論がきっかけで転職し、

　その転職がうまくいったと思い、
　読むのを中断してしまいました。


　自分の実力ではなく、たまたま運が良かった、
　また、自分の本当の人生の目的が達成していない、

　むしろ具体的な目的が
　定まっていなかったのにも関わらず、

　失礼ながら、きっと自分は大丈夫、
　もう必要ないだろうと、

　勝手に勘違いをしていました。。。


　その後2年弱が経過し、
　
　やはり再度仕事や人生で悩むことがあり、
　また平成進化論の読者となり、

　ついにこの度、「日記放談会」の音源を購入しました。

　有料コンテンツを購入したのは
　今回が初めてです。


　「日記放談会」音源を聞いた率直な感想は、
　もっと早く購入しておけばよかった、

　いや、今このタイミングで
　購入の決心がついてよかった。

　（これ以上先でなくてよかった）でした。


　時々刻々記すことの重要さを、
　なんとなく頭でわかっていたつもりが、

　それはただ

　「知っていた」
　「理解した-つもり-」

　だったに過ぎず、

　やっと今、自分の中に落とし込み、
　本当に理解できたように感じています。


　不思議なもので、内容自体は同じであっても、
　人から聞く、文章を読むのではなく、

　鮒谷さんの言葉で聞くことで、
　自分の中に浸透するレベルが全くもって別物でした。

　本当に言葉の重要さ、そして凄さを
　痛感しています。


　今回の音源がきっかけとなり、
　日（時々刻々）記を書き始めたことで、

　自分の人生ベースでの目的や目標にしっかりと
　向き合い始め、具体的に考えるようになりました。


　そして大きな目標、成功のためには、
　行動を変えなければならない、

　そのためには、
　思考を変えなけらばならない、

　そのためには、
　継続して意識しなければならない、

　そのためには、
　自分の言葉で明確に表さなければならない、

　そのツールとして日記を書く＝時々刻々記す。


　日記を書くことの大切さや力が、
　心に深く深く突き刺さりました。。。


　音源を聞いて2週間弱の現在、
　まだまだ成果や結果は出せていませんが、

　それでも既に、

　常にアンテナを張る、
　「！」や「？」に敏感になる、

　延いては自ら「！」や「？」を探しに出る
　（これはまであまり出来ていませんが）、

　というように意識が確実に変わってきました。


　今では、新しい「！」や「？」を
　早く仕入れたい、知りたい、

　そしてそれを時々刻々、記したくて記したくて
　ウズウズ、ワナワナしています。


　今後も日記を継続し、常にベクトルを未来に向けて、
　日々楽しく精進していきたいと思います。


　事情により一呼吸おいてからになっては
　しまいますが、

　どうやらいつの間にか鮒谷さんに
　秘孔を突かれていたようですので、

　「日記放談会」に続き別の音源である、
　「目標設定3部作」にも期待しております！


　この度はありがとうございました！！！


　（松永 和也 さま）


　--------------------------------------
　（ここまで）


　松永さん、長文のご感想を下さり、
　ありがとうございます！



■3日間セミナーに参加下さった先輩というのは、
　Ｔさん（？で）しょうか。

　もしそうであれば、また機会があれば
　よろしくお伝えおき下さい。


　せっかくなので、頂戴したご感想に触発され、
　今日の本文を記したいと思います。



■日記を書けば書くほど、頭は良くなります。
　どうやらこれは間違いなさそうです。


　もちろん、自分の器に見合ったレベルでしか、
　良くはなっていかないものですから、

　「過去の自分と比べて」

　ではありますが、

　少なくとも、書けば書くほど、

　昨日よりも今日、今日よりも明日、
　間違いなく、頭が良くなっていく、

　そんな実感を覚えるもの。


　おそらく松永さんも、

　音源を聴き、日記をつけられてから
　２週間ほどの間で、

　既に感じられていることなのでは
　ないでしょうか。



■これまでにも書いてきましたが、

　私（鮒谷）は25年の間、
　文章（日記）を書き綴ってきました。


　何をどのように考えて、日記を書いてきたか、
　そして日記を進化させてきたか、

　については、

　【日記放談会の音源】

　にて散々に語りつくしているので、
　そちらに譲りますが、今日はその補足を。



■すなわち、
　
　【学びとは、「既知と未知が結びついた瞬間」に
　　生まれる】

　わけですが、

　過去に自らが学んできたことに
　外部からの未知の刺激が加わることで、

　【「！」や「？」】

　が生まれます。


　
■後述の通り、

　【「！」や「？」】

　が生まれた理由に思いを巡らし、
　日記を書くことによって


　【感情変化をもたらした新しい刺激
　　（＝未知の刺激）を、
　
　　自分の人生・生活文脈に織り込んでいく】

　ことが可能になるはずです。


　これが

　【頭がよくなっていく実感】

　の正体です。



■先日、国立西洋美術館で開催されている


　＜シャセリオー展＞
　 http://www.nmwa.go.jp/jp/exhibitions/2017chasseriau.html


　を観に行ったのですが、そこで
　巨大なタペストリーが展示されていました。


　それはそれは素晴らしいもので、
　思わず見入ってしまったのですが、

　そこで、

　【学びとは、「既知と未知が結びついた瞬間」に
　　生まれる】

　という言葉が思い浮かんだのです。



■もちろん、私に

　「タペストリーの織り方」

　は分かりません。


　至近距離から眺めてみても、
　どうやってこれを織っていったのか、

　皆目、見当がつかず、
　ただただ、感嘆するばかり。



■とはいえ、

　膨大な時間はかかるかもしれないけれども、
　毎日、毎日、ほんの少しずつでも織り進めれば、

　最初は小さなマット程度のものが、
　いつかやがて、巨大なタペストリーになる、

　ということくらいは私にも分かります。



■同様に、自らの経験や知識、能力も、
　はじめは

　【非常に狭く小さなマット程度】

　から始まるわけですが、


　積極的に未知の領域に飛び出し、
　「新しい色の糸」を縦横に織り込むことで、

　やがては

　【とてつもなく巨大なタペストリー】

　にまで成長させることができるのでは
　ないでしょうか。



■この例えでいえば、

　【機織り機に相当するのが、日記を書く行為】

　といっても良いのではないでしょうか。


　おそらくは時代が下るとともに

　【手織りから機織り】

　になり、タペストリーの生産効率は
　とてつもなく上がったのではないかと思いますが、

　　
　それと同じように

　【記憶から記録（その場限りで流してしまう
　　のではなく、日記を記述する）】

　によって、周囲に起きた出来事を
　自分の文脈に織り込む速度が加速し、

　それに伴って、成長速度もとてつもなく
　上がるように思われるのです。



■今までの自分の生きてきた履歴（文脈）
　（＝既知の世界）

　と新たに出会った世界を
　結び付け、関連付け、意味付けて、
　
　言葉によって、既知のものへと取り込んでいく
　プロセスを、


　時間を味方につけつつ、

　倦まずたゆまず、あくなき実践
　（＝「時々刻々の記」を記し続ける）によって、

　どんな凡人であってもそこから抜け出し、
　「巨人」になることができるのではないか、

　そのように信じているのです。



■そして、もう一つ思ったのは、

　やがて織物が大きくなり、一定サイズ以上の
　タペストリーになってくると、

　【タペストリーの外縁部の全長、すなわち、
　　既知と未知との境界線の長さが長くなる】

　ということ。


　ということは、つまり、

　【同じ経験をしても、
　　世界と触れ合う機会が増大しているから、

　　それまでよりも気付き、学び、発見の量が
　　激増する】

　こととなるのです。



■日記を書き続けているうち、
　このメカニズムが発動し、
　
　知らず知らずの間に、
　興味・関心の幅が圧倒的に広がって、

　ある段階から、カンブリア爆発もびっくりの、
　「！」や「？」が生じるようになるはずです。


　そのとき初めて、学びがとてつもない速度で
　加速し始めていることを、

　身体を通して実感されることになるでしょう。



■あくまでこの世界を体験できるのは、

　【そこまで折れずに書き続けられた人のみ】
　
　ということになるわけですが、


　無事、その瞬間を迎えることができさえすれば、
　そこから先は

　【学びの高速増殖】

　が永遠に続くようになるでしょう。

　（これは日記を書き続けた人だけが得られる
　　巨大な快楽であり、醍醐味です）



■そんな、

　【時々刻々、気付きや学びが爆発し続ける】

　という、とてつもない体験を
　あなたにも体験して頂きたいと思い、

　お届けさせて頂いているのが、
　日記放談会音源ということになります。



■私（鮒谷）が、なにゆえ、このような音源を

　「しつこく、しつこく」

　お届けさせて頂いているのかの理由の一つを
　本稿において記させて頂きました。


　ぜひ、あなたにも、この境地に
　到達して頂きたいと願っております。
