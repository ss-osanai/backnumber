■朝からPCの前に張り付きっぱなし。
　その合間にひっきりなしにかかってくる電話。こっちからもがんがん電話
　を入れる。

　とにかく、人との接触をあらゆる手段を使って持とうと思っています。
　
　
■電話、メール、その他、いろんな媒介ツールを利用して、皆様との
　接触回数が増えました。
　実際に会って話をするのが一次情報を得るという意味では最善ですが、
　最善が無理なら、次善の策としての電話、メールというのはいいですね。


■おかげさまで、先月の携帯電話の利用料金をWebで調べてみて（まだ明細
　はきていないようですが）腰が抜けそうになりました。

　でも、これも必要経費ですね。（と自分自身を納得させようとしている…）


■でも実際にお会いする、電話させていただく、メールを入れる、という
　ことでビジネスはどんどん発展していきますから。

　ただ、物理的に、思考の時間とキーボードを叩く速度
　（＝いずれも私のリソース）が完全にボトルネックになっています。

　早く、このサイクルから抜け出さないと。


■同じ思いを抱いているあなたに、このメルマガ。

　私の大好きな先輩、兄貴、九鬼さんが発行されてます。
　そしてまたその後ろに控えているのが錚々たるメンバーたち。

　ゆったりしながら、自分のビジネスを成長させる法。

＜今日のピークパフォーマンス方程式＞

　　　　【毎日、外部情報に接し続ける。】

───────────────────────────────────


■仮にもピークパフォーマンスナビゲーター、と名乗っているからには
　自分のビジネスについても、最高の結果を出していかねばなりません。

　ピークパフォーマンスを発揮できない人間からピークパフォーマンスの
　話を聞こうという奇特な人はあんまりいらっしゃらないと思うから。

　▼お金儲けが出来ていない人からお金儲けの話は聞きたくありません。
　▼人間関係づくりが出来ない人からコミュニケーション能力を学ぼうとも
　　思いません。
　▼起業していない人から起業の話をききたい人もありません
　
　そういう意味で、自分が利害関係者すべてをハッピーにして、
　自分のビジネスも成長させていく、そんな結果を残さずして
　人さまに語ることもできません。


■まだまだ私自身がピークパフォーマーだなんて、こっぱずかしくて
　とてもとても言えないです。

　だから、「ナビゲーター」「水先案内人」なんですよ。

　ただ、ピークパフォーマーを目指して、試行錯誤して、痛い目にあって、
　悩んで、苦しんで（できるだけそういった内容はメルマガ上では
　書かないようにしていますが）、
　
　その中、ある種、自分を実験材料として供してきて、実践してきて
　うまくいったことだけをメルマガ上で皆さまにお伝えしております。

　
■非常に短い時間で、

　▼セルプロデュースしていっていること
　▼このメルマガがビジネス路線を歩み始めてたったの二ヶ月で成長軌道
　　にのっていること（順調に収益をもたらしてくれること）
　▼人との出会いを求めて始めたメルマガが、新しい人との出会いを
　　次々と導いてくれる
　▼新しいチャレンジングな仕事をたくさん頂けるようになってきたこと

　などなど、ある程度、多少は語れる内容も出てまいりました。


■前口上が長くなりましたが、今日は、そんな実体験を通しながら、
　ほんとうに大事だなぁ、とおもったことを少々。

　それは、毎日、自分の中に新しい情報を取り込み続けるということ。
　
　▼毎日、毎日、本を読む。
　▼テープを聞く。
　▼セミナーに出来るだけ出る。
　▼人と会い続けることかもしれません。
　
　いずれにしても、ひとりでぼ～とする時間を極力作らないということ。


■これは結果を生み出すために非常に大切なことであるとおもわれます。
　私にとって、情報の入手先は、本とテープ、CD、セミナー、人。
　上記あげた全てです。

　いろんな結果を出している人を見ていると、このいずれもすべて大事、と
　寸暇を惜しんで新しい情報をインプットしようと日夜、動いているようです。
　傍目にはどう見ても、遊んでいるようにしか見えない社長さんですら。


■こういう方、たまにありますよね。でも会社の業績は絶好調。

　でもこれは、遊んでいるのではなく、遊びの中から新しいアイディアや
　インスピレーションを得るための立派な仕事なのです。
　（※ただし、遊びを「仕事」と思った瞬間に、発想が閉じてしまう
　から難儀なことなのですが）

　肩に力をいれずに、遊ぶまま、趣味のままが仕事になる。
　ちょうど上述の九鬼さんのような姿がモデルケースかもしれませんね。
　

■ということで、こういう習慣をもし、今、もたれていないならば
　少なくともインプットすることが習慣化するまで、行い続けましょう。

　習慣化することは上記全てでなくてもいいのです。

　▼ある人にとって、それは本を読むことかもしれません。
　▼ある人にとっては、テープを聞くことかもしれません。
　▼また別の人は、ランチは毎日違う人と取ることかもしれません。


■一度、習慣化できると、あとは自動操縦です。
　自分自身が無意識に、その行為を行い、自動的に情報を収集しはじめます。

　テレビを見るのを習慣化するのか、
　ベッドにひっくり返って、ぼけ～とするのを習慣化するのか、
　マイナス発想を習慣化するのか、
　
　習慣の内容と質で、人生が大きく変わるように思います。
