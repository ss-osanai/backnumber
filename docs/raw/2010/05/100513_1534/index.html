■こんにちは、鮒谷です。

　夜にあったイベント？参加の時間を除いて、
　終日、親孝行。

■この本を読んで、触発されたかもしれません。


　仮に親が現在60歳とすると、

　20年間（親の残された寿命）×６日間（１年間に会う日数）
　×11時間（１日で一緒にいる時間）＝1320時間、

　つまりあと55日しか残されていない。


　「残された時間を後悔しないために」

　なんていわれると。


　
■明日以降も引き続き、親孝行デー。

　そんなことくらいでこれまでの親不孝の数々！？が
　消し去られるわけではありませんが、、、



■年を経て、経験を重ねることで、

　受けてきた親の恩を骨身に感じるようになり、

　あるいは、

　かけてきた心配や迷惑が痛みを伴って知らされる、

　ということがありますね。


　これからも

　「親孝行が趣味（＝一緒にいられる時間を極力長く取る）」

　を心がけていきたいと思ってます。

　
　
■昔は何か言われるたび、
　
　話が長くなるたび、
　　
　はたまた、
　必要以上に心配されたり、気を遣われたりするたびに、


　「うるさいなぁ！（怒）」

　「黙っといてくれ！（怒）」

　「うっとおしい！（怒）」

　「そんなことくらい分かっとるわい！（怒）」


　なんて思っていましたが、
　
　（今も時折、思うこともあります、、、はぁ、情けない）


　しかしながら、
　
　出来る限り、そういう心を消し去り、
　暖かく、ゆったりと接しよう、

　そんな風に頑張ってます。

　（頑張らなければならない時点で、
　　まだまだなんですけどね）

＜今日のピークパフォーマンス方程式＞

 【　柔よく剛を制する人生行路　】

───────────────────────────────────



■よく

　「自分を主張できない」

　という人がいます。


　けれども、

　人生という一幕の劇場は、


　「自分の主義・主張をこちらから発信し続けていないと、
　　なかなか思い通りにコントロールできない
　　
　　　　　　　　　　　　（＝他者にコントロールされがち）」


　そういうところなのです。



■実は、


　「私は、これをやりたい。
　　けれども、あれはやりたくない。

　　こんなことに興味があるが、
　　こっちには興味がない。


　　こんな人と付き合いたいが、
　　こういう人とは付き合いたくない。

　　こんなことをされると心地いいけれど、
　　こういうことをされると不快になる。


　　これから、こんなことをやろうとしている。
　　そしてこの部分は、これから捨て去ろうとしている」


　こういう事前情報が分かっていて、
　それでもその人が嫌がることをやってやろう、

　なんて人は、まずいません。



■不快な仕事をもらったり、
　イヤな頼み事をされたり、
　望んでもいない人のつながりができていくのは、

　これらの情報を事前に相手（あるいは周囲）に
　渡していないから。



■自分は何を望んでいて、
　何を拒んでいるのか。


　これを明確に外部に発信し続けることによって、

　無理な力を出すことなく、
　自分の人生が思い通りになっていく。



■つまり

　【 情報を発信していると、コントロールされにくくなる 】

　のです。


　これが

　「エレガントに」

　自分の主義主張を通していく方法。


　だから、自分についての情報は
　ある一定レベルまでは（少し過剰かな？と思うくらい）
　供給してもいいのです。



■私がメールマガジンでやっていることは
　まさにそういうことなのです。


　「これをやりたい、
　　こんなことはしたくない、

　　こんなことに興味がある、
　　こんなことには興味がない

　　・・・・」


　という事前情報を、陰に陽に発信し、

　（すなわち、言葉に表したり、行間にメッセージを埋め込んで）

　周囲に渡すことによって、
　結果として自分の人生が思い通りになっていく。


　
■この一連のプロセスを、私は


　【 柔よく剛を制する人生行路 】


　と呼んでいます。

　
　剛腕で（敵をたくさん作りながら）自分のやりたいことを
　ガンガン押し通していくのも一つの方法ではありますが、

　こちらのほうが、よりエレガント、だとは思いませんか？


　そして実際のところ、

　遠回りに見えて、最短距離を歩むことになる。
