7171号 考えがまとまらないから動けないのではなく、動かないから考えがまとまらないのだ

■昨日のメルマガでも書きましたが、

　昨日まで３日間、
　九州に行ってました（福岡＆鹿児島）。

　先月の日経新聞、R九州の唐池さんの
　『私の履歴書』に触発されて。



■昔から大切にしている信条に

　【結論が見えていないからこそ
　　動く】

　というものがあります。


　ある時期までは

　「結論がハッキリしてから動く」

　ようにしていたのですが、
　これは良くありませんでした。



■あなたも経験があると思いますが、

　「考えがまとまったら取り組もう」

　なんて思っていても、

　一日、一週間、一ヶ月、なんて時間は
　あっという間に流れていきます。



■その間、何も実のあることは
　実現できず、

　ただ、なんとなく手を動かし、
　作業を繰り返していただけ、

　そんな日々を過ごして、結果、
　後悔することになるのです。



■そんなことを繰り返しているうちに、

　短い一生はあっという間に終わる、
　と思うようになってから、

　行動規範を書き換えました。



■惰性の人生に抗うためには、

　【考えがまとまらないうちから
　　とりあえず新しいことを始める】

　ようにすると良さそうです。



■まずは動き、初速をつける。

　そこからのフィードバックを得ながら
　次の行動は何かを考えているうちに、

　思考は（ときに軌道修正を交えつつ）
　徐々に明瞭化されてくるものです。



■明瞭化の前には

　【思考の発火】

　とでもいうべき現象が伴います。


　「日記放談会音源」でお伝えした
　言葉でいうと

　「！」とか「？」が頭の中に生じる、
　ということですね。



■これらを言語化による回収する、
　つまり自身の文脈に組み込む、

　言葉を換えると、

　自らの来し方行く末に絡める形で
　当該事象に意味を与える、

　ことによって、
　考えが鮮明になっていくのです。



■ですから、

　もしあなたに、人生やキャリアや
　ビジネスについての悩みが生じたら、

　（必要以上に考えることなく）

　とにかく動くこと、
　とにかく刺激を入れること、

　ですね。

　そこから頭の中で何かが発火します。



■煮詰まったら、
　きっかけはどんなことでもいいから、

　とにかく、とりあえず、
　動くことをお勧めいたします。

　煮詰まる前だと、なおいいですね。



■私（鮒谷）もそのために、

　【思い立ったが吉日】

　の精神で、毎日を生きています。


　日々の生活に、

　【思いつきから決めた行動】

　はもちろんのこと、

　【瓢箪から駒、的な動き】

　まで意識して取り入れてきました。



■今回、福岡、鹿児島に行ったのも、

　【思いつきから決めた行動】

　であることは間違いないのですが、


　そこに至る前に、
　コンサルのお客さんとのやり取りの中で

　「今回の唐池さんの『履歴書』は
　　素晴らしいですね。
　　鮒谷さんも読んでますか？」

　と問われたので、

　「もちろんですよ！
　　あれを読んで触発されたので、近々、
　　九州に行くことを決めてます」

　なんて、そのときまで一切、
　思ってもいないことを口走ってしまい、


　その結果、実際に行かないわけには
　いかなくなってしまった、

　という展開となりました。



■これなども、ある意味においては

　【瓢箪から駒、的な動き】

　といって良いでしょう。


　こういう出来事についても
　ポイティブに捉え、

　すぐに自分の行動に
　変えていければいいですね。

　それによって自己更新、自己刷新が
　さらに加速しますから。



■今回は、行くと先に決めた「あと」で、


　▼福岡のコンサルクライアントさんと
　　博多でラーメンを一緒に食べる

　▼博多から鹿児島まで
　　「３６ぷらす３」という電車に乗る

　▼桜島を間近で見る
　　（桜島が正面に見えるホテルに宿泊）

　▼鹿児島から指宿まで出ている
　　「指宿のたまて箱」という電車に乗る

　▼指宿を視察
　（指宿駅前の指宿中央名店街を通り抜け、
　　海岸まで歩く）

　▼指宿から鹿児島駅までバスで戻る中で、
　　鹿児島の町並みを観察

　▼途中、知覧で途中下車、
　　2006年5月9日以来、17年ぶりに
　　「知覧特攻平和会館」を見学


　といったことをメインの目的にし、

　そこからなにかの意味を
　見出してこよう、

　と決めるという順番でしたが、
　めちゃくちゃ充実した旅となりました。


　どこかでこの旅での気づき、学びを
　シェアしたいと思っています。



■そんなことも、こんなことも、

　普段の生活圏の外に出たからこそ、
　得られた学びであり、感情、

　といって良いでしょう。


　もし今回、面倒がって
　九州を訪れていなければ、間違いなく

　【振り返っても何の記憶も残らぬ、
　　いつもと変わらぬ３日間】

　を積み上げていただけのはず。



■こうした

　「いつもと変わらぬ
　　○時間、○日、○週間、、、」

　をいかに減らすか、

　つまり日常にどれだけ新しい挑戦を
　組み込めるかが、

　長期的に見た、人生やキャリアの達成や
　充足感に関係しないはずがない、

　と思っているんですよね。



■そして実際、

　その通りであったと、これまでの
　体験を通じて、感じてきましたし、

　だからこそ、日々、

　【ルーティン的な活動の中に
　　新しいイベントをいかに組み込むか】

　を工夫してきたつもりです。

　その巧拙がQOL(人生や生活の質）に
　ダイレクトに関わってくる、

　それが私（鮒谷）の、
　これまでに生きてきての結論です。



■今日の話があなたのご参考に
　なりましたら幸いです。


　もしあなたが、

　そんなことを言われても、
　すぐに行えそうな

　【ルーティン的な活動の中に
　　新しく組み込めそうなイベント】
