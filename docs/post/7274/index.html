■結局、幸せに近づくためには

　【幸福を阻害するボトルネックを
　　一つづつ潰していく】

　しかないんですよね。



■ミラクルは存在しない、
　ということ。

　地味で、地道な歩みでしか

　【数ある、不幸せ要因】

　を除去できないということでも
　あります。



■たとえるならば

　【不幸せ要因を取り除くのは、
　　地雷除去みたいな取り組み】

　なのかもしれません。


　短期間で、一網打尽で
　大量の地雷を取り除く、

　などという方法は
　聞いたことがありません。



■不測の事態が起きないように、

　丁寧に、細心の注意を払って
　一つづつ、取り除いていくものです。


　要するに


　「宝くじさえ当たったら」
　「この株が暴騰してくれれば」

　「人間関係をリセットしたら」
　「良い会社に就職（転職）さえできれば」

　「このサプリさえ飲めば」
　「○○だけ食べていれば」

　「この器具を使って
　　トレーニングをしさえすれば」
　「彼氏、彼女ができさえすれば」


　一発で人生まるごと幸せになれる、

　なんてことはないということ。



■もちろん、瞬間風速でなら、

　不幸せな状態から離脱できることも
　あるかもしれませんね。


　宝くじあたったら、
　一時的には手元資金は分厚くなるし、

　副作用を考えなければ、謎のサプリを
　飲んで体重激減することもある。



■恋人ができたら、結婚したら、

　数週間か、数ヶ月（ないし数年）
　くらいは幸せになれる？

　かもしれません。



■けれども、

　インチキサプリや
　インチキ情報商材に書いてあるような、

　「これ一発で大丈夫！」

　なんてことはありません。

　自分自身すらも、

　【不安や不満や不快をもたらす要因を
　　都度、一つづつ潰していく】

　ことだけが、

　結果として、
　幸せに近づくための近道となるのでは、

　「たまたま運が良かっただけ」
