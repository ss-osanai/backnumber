こんばんは、鮒谷です。

会場にて、グーグルの特設サイト経由で日本赤十字社にお送り
いたしました。

自分が今できることを、気負わず、肩肘はらず、
淡々と行っていきます。


皆さまのご協力により３回の開催で合計「290,700円」の
義援金をお送りしております。ありがとうございます。


明日（というか、もう、本日ですね）以降も
開催いたしますが、

第４回開催→3月20日（日曜）
第５回開催→3月21日（月曜）

ともに、本日と同じく、同じ時間、同じ場所にて
開催いたします。

　お送りすることができております。後藤さん、ありがとうございます。

●中古車輸出業　Fさま


鮒谷さま

中古車輸出をしている○○です。
先日はありがとうございました。

お話しは大変興味深く、沢山の話題と数々のキーワードを使い
聞き手に分かりやすいように話されていました。


平成進化論を読み出して4年近くになりますが、
日刊メルマガと同じ思想の言葉が耳で入ってくる感覚に
最初はミーハーな気分で緊張すらしていましたが、

メルマガ読者なら「うんうん！」というキーワードが
出てきて楽しかったです。


今回の武者語りの大テーマ？であった

『セーフティーネットと保険』

を話されていましたが、確保できた人間が次にする事を
行動で示されていたようにおもいます。


今何ができるか？　を考えるのは誰でもします。
長所を生かして、行動する！　という事が一番印象に残りました。

また機会があれば、お願いします。

●栗本 大介さま
　ファイナンシャルプランナー
　株式会社家計の総合相談センター大阪オフィス代表


本日、大阪での武者語りの会に参加させていただきました。

鮒谷さんとは、昨年4月4日に大阪での少人数放談会で
お会いして以来でしたが、改めて「直接息遣いを感じる」ことの
大切さを感じました。

今回のお話の中のポイントは、
「有形資産よりも無形資産が大切である」という点だったと
思うのですが、そのための具体的な日々の取り組みとして、

１、情報リテラシーを高めておくこと
２、人とのご縁を築いておくこと

がいかに重要かということを痛感いたしました。


特に情報に関してお話しいただいた、
「情報の三角測量」のお話は、
以前にもお聞きしたことがあるはずなのに、
そういえば、ちゃんとできていなかったな・・と反省しきりです。

時の経過とともにできなくなっているということは、
つまり「習慣」にまで落とし込めていない証拠ですよね。

習慣化されるまでは「しっかりと意識して取り組む」
という姿勢を忘れないようにしなければ。


また、私はファイナンシャルプランナーとして、
お金に関する相談をお受けすることが多いのですが、
「老後の生活資金の心配」をされる方は結構多いです。

でも、今日のお話で

「長生きすると、人的なネットワークが若い時より圧倒的に
 広がるわけだから、本来ならリスクが少なくなっていくはず」

という言葉をお聞きし、
「お金」という面にとらわれすぎていたのかもしれないな、
という想いを持ちました。


そしてもう１つ心に響いたのは、
『「術」から「道」へ』というお話です。

仕事だけに限らないと思いますが、
人間って何かに取り組むとき、
当然ながら「術（＝スキル）」を身に付けようと訓練します。

でも、どこかのタイミングで「道」にすることで、
自分の軸をしっかり持たないと、
安定したパフォーマンスを出すことができない、というのは、
非常に自分の腹にストンと落ちる内容でした。

結局人間って、最後は自分自身の心のありようなんですよね。

それがブレていたら、いつまでたっても
周りに振り回される不安な状態から抜け出せないのかもしれません。


また、今回の震災に関連するネット情報の話題もあり、

「情報を正しく取捨選択して、その情報に従って行動する力」

というものの大切さは、私もとても感じています。


情報に対する感度を高めておくこと、
情報を取捨選択できるだけのリテラシーを持つこと、
そして、これと思ったらすぐに行動に移せること、

どれが欠けていてもダメですよね。

わずか2時間半ほどの時間でしたが、
確実にスイッチを入れていただけたと思います。


私自身、今年40歳という節目を迎えることと、
ビジネス的にも次の10年に向けてスタートした年であることで、
このタイミングに大きな元気をいただけたことに感謝しております。

今後の武者語りの会のご盛会をお祈り申し上げます。


（今回の内容について記していただいています）
http://blog.livedoor.jp/kurisuke701/archives/51805922.html

●清水 省三さま
　アメリカン ライフ インシュアランス カンパニー
　コンサルタント


鮒谷 様

昨日は、大変貴重なお時間を頂きまして、誠にありがとう
ございました。メルマガを読ませて頂き５年になります。

５年間、毎日、毎日、刺激を頂き、読み始めた頃の私にとって、
鮒谷さんは雲の上の存在でした。

しかし、毎日、毎日、のメルマガを通して、
鮒谷さんの読まれている本や、鮒谷さんの会食された方の本などを
読む事を行い、少しでも鮒谷さんのライフスタイル・思考レベルに
近づきたいと思い、色々な経験やチャレンジをさせて頂く中で
徐々にですが、周りから少しづつ信頼される自分になって行きました。


情報発信の価値などもメルマガを読むにあたりヒシヒシと
感じていて、なかなか始める事が出来なかったのですが
思い切って去年の年末に開始しました。

そうしたら、今まで知識レベルで分かっていた情報発信の
素晴らしさが、経験値で感じる事ができ、この自分メディアのすごさも
徐々に肌で感じる事が出来てきました。


そんな中で、今日お会いする機会を頂けたのは、
本当にすごいタイミングと縁を感じてなりません

雲も上の存在と思っていた方とお会いできたという事は、
自分のその方と会えるくらいの器になれたのかなと言うよりも
メルマガを通して、その領域(まだまだ全然下のほうですが、)まで
上げて頂けたのかなと勝手に思っています。

５年ほぼ毎日読ませて頂き、自分自分の思考の変化に
とてもビックリするほど、変わってきている事を感じます。

今の自分の思想のベースには、
進化論の各テーマが一杯蓄えられています。


本日お会いできて、今までの文字のつながりが、これからは、
立体的にとらえていく事が出来るようになったと思っています。

自分が情報発信を始めて、色々と苦労したからこそ、
今回鮒谷さんにお会いできて、お話を理解でき経験から
いろいろな学びを感じさせていただく事ができました。

・言わない所に真実はある。

・情けは人の為にならず、自分の為になる。

・結局は誠実が大事

・そして、メルマガを身体感覚で扱う

このような事はお会いして、お聞きして初めて奥の深さが
見えてくるものだと思いました。


貴重なお話本当にありがとうございました。
自分ももっともっと成長して、成功します。

成功の秘訣は？と聞かれた時に、平成進化論が僕の成功の秘訣です。

と答えれる日を迎えたいと思います。


これからも、日々勉強させて頂きます。


感想遅くなりましてすみません。
また、お会いできる事と確信しています。
