6969号 「欲しくて欲しくてたまらない（ぷるぷるぷる）」が営業の極意

■今日は、あなたに

　【営業の極意】

　をお伝えいたしましょう。


　営業の極意を一言で表すなら

　【辛抱たまらん】

　です。

　これだけだと
　なんのことか分かりませんね（笑）



■Weblio辞書によると

　【辛抱たまらん】

　とは、

　-----------------

　関西弁で、もう我慢できない、
　耐えられない、といった意味の表現。

　-----------------

　ということです。


　ちなみに、この言葉は、

　私が花の中高６年間の男子校時代、
　多用した思い入れのある言葉（笑）



■それはさておき

　「無理やり売る」

　のではなく

　「欲しくて欲しくてたまらない
　（ぷるぷるぷる）」

　状態になって頂いた後、


　「ようやくお求めが可能になりました！」

　とご案内することによって、

　「やっと買える！！」

　となってご購入に至るのです。
　（理想形）



■ここ数日のメルマガにおいては

　【元気注入音源（チラ見せ）】

　をご案内してきましたが、

　その意図、目的は
　実にここにありました。



■たまたま今、

　コンサルのお客さまで、
　いいサービスをお持ちなのに

　「なかなかお客さまに
　　お求めいただけない」

　そんな悩みをお持ちの方が
　いらっしゃるのですが、


　その方に向けて、

　「なるほど！
　　それでは私（鮒谷）が売れる流れを
　　お見せいたしましょう」

　ということで現在、
　実演販売（の下準備）を行っています。


　※どこまでいっても顧客志向。
　　（自画自賛）



■チラ見せ開始から
　まだ三日ほどしか経っていませんが、

　元気不足の方からすると
　ひょっとすると、そろそろ

　「欲しくて欲しくてたまらない
　　（ぷるぷるぷる）」

　となられているかもしれません。

　というか、
　そうなっていてもらいたいもの。



■そんな

　「ぷるぷる感」

　が、いよいよもって

　【辛抱たまらん！！！（暴発寸前）】

　となったタイミングを見計らって
　ドカンと告知するのです。



■そうすれば、

　【後先、考えることなく
　　お申し込み】

　となってもらえるに違いありません。


　今日はコンサルクライアントの
　Ａさんに向けての

　「誌上フォローアップコンサル」

　を兼ねて、お届けいたしました。
