　復習するのは非常に重要。


　-----------------------


　行動変容において
　自在に使いこなせなければ意味がない。

　何を自在に使いこなすかというと、
　意思決定基準。

　それを臨機応変に使いこなせるように。

　聞いて知っている、では意味がない。


　即座に取り出せない、
　意識もしてないし、

　言われてみたらそういえば、
　というのは実生活に反映されない。

　だから勉強をしていても
　結果につながらない。


　-----------------------


　知識を増やすことに
　血道を上げるより、

　知識を自在に使いこなせるようになる
　ことに意識を傾けることのほうが、

　100倍大事。

　
　考えなくても、
　勝手に体が動いている、

　そこまで徹底できてこその
　真の学び。

　
　-----------------------


　一番退屈で地味なのが
　OSのアップデート。

　退屈だがじっくりやると
　味が出てくる。

　生涯楽しめるようになる。

　その喜びが身体知化されるところまで
　反復、反復することが大事。


　-----------------------


　基礎の土台があるから
　応用が効くようになる。

　だからこそ基礎部分の学びが大事。
　

　学びを散らすとなんにも身につかない、
　焦るから、さらに短期の学びを求める。

　どんどん学びの時間軸が
　短期に寄ってくる。


　-----------------------


　氷の上を横滑りに滑る人のメタファー、
　穴を掘っても下に掘らない。

　焦れば焦るほど、
　違う場所に穴を掘ろうとする。

　あるいは、すぐに穴を開けられそうな
　小手先のミラクルを求める。


　-----------------------


　滑っている間は何も学んでいないし、
　時間が経ってもが、氷に穴は開かない。

　立ち止まり、覚悟して掘る。
　掘ったら絶対に結果が出る学び。

　（特に初期の頃は、そうした学びしか
　　してはならない）

　それをやりきる人が少ないから
　大きな成果を出している人もまた少ない


　-----------------------


　人生とは問いの質。
　目標が変わると問いが変わる。

　Beingが最重要。

　ここが明確になると、それを支えるブロックを
　どう作るかという具体的な問いが生じる。


　-----------------------


　変わらない人は
　問いを持っていない人。

　問いを持たない人は変わらない。


　-----------------------


　変われる人は問いを持つ人。

　さらに問いが変わり続ける人が
　成長し続けられる人。


　-----------------------


　「新しいことを学ぶ」

　のではなく

　「知っていることを、無意識で
　　出来るようになるまで躾ける」

　ことを学びと定義すると良い。


　-----------------------


　認知資源を浮かせては、
　新しい本質的学びに投下。

　それが無意識にできるようになれば
　認知資源が浮くので、
　
　そのリソースをさらに
　新しい本質的学びに投下、

　を繰り返す。

　これが「変われる人」である。


　-----------------------

　
　Beingは渡せない。

　死力を振り絞って
　自らが作らないといけない。

　（誰も作ってはくれない）

　自問自答の末に言葉が生じると
　人生が主体的に変わる。

　自分が何を以って覚えられたいか、
　何者であるのか、

　の答えを他者は渡せない。


　-----------------------


　学びにおいて「コンボ」が発動すると
　学習が楽しくなる。爽快感すらある。

　想像以上の結果を
　自在に出せるようになってくるからだ。
