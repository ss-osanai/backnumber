 【　天職に近づく３つの行動基準　】

───────────────────────────────────



■言うまでもありませんが、人とのつながりほど
　大切なものもないでしょう。

　ただ、それを無理に作ろう、広げようと頑張るのは
　精神的に辛いものがあります。

　また、相手にも迷惑がかかることもあるでしょう。



■そこで、

　「人脈をつくるんだ！」

　などと勢い込むのではなく、

　それよりも、自分の経験を増やすことに注力してみては
　いかがでしょうか。

　話題が増えれば、自然と仲良くなれる人も
　増えてくるというものです。



■そして昨日も言いましたが、その際、
　人のつながりを介した利害損得のことなどは
　考えないことです。

　急いで結果を求めるのはやめて、
　むしろ、成果は先へ先へと押しやろうとする。

　そのほうが、結果的にのちのち、より大きなものと
　なって戻ることになります。



■これは、実行するのがある種、怖いことであるがゆえに、

　（少なからぬ人、特に経営者や自営業をしていると、
　　人のつながりを介して、自分のビジネスにつなげたいと
　　考えがちであるから）

　ビジネスにつながらないことを是として
　継続的に、上記のことを実行できる人の比率は、

　実は非常に少ないのです。



■ですから、その恐れを乗り越えて、

　あなたがこの道をどんどん踏み込んでいけば、
　いずれ平穏な海に到達することでしょう。

　血の海を離れた、違う世界に出られるのです。



■とはいえ、

　「短期的な成果を捨てる」こともまた、
　無理してやっても、辛くなるでしょう。


　そこで、これは私の親友がいつも言っていること
　なのですが


　【 MUST 】（しなければならない）

　【 SHOULD 】（すべきだ）

　を行動の動機にするのではなく、
　あくまでも


　【 WANT 】（したい！）


　で動きましょう。



■あなたが


　▼やりたいことを

　▼徹底して

　▼利害損得を考えずに


　やっていくようにするのです。

　というより、

　本当に好きな、やりたいことであれば、
　自ずからそのようになってくるはず。



■ですから、今やっていることが果たして
　そういう条件をクリアしているか？

　つまり、さきほど挙げた３つの基準、


　▼やりたいことを

　▼徹底して

　▼利害損得を考えずに


　を満たしているか？
　と常に問うようにするのです。



■これらを満たした行動をとっているとすれば、
　天職に近づいていけるでしょう。

　逆に、もしどこか引っかかりを感じる、
　無理をしているとしたならば、

　いまやっていることを修正する必要があるのかも
　しれません。
