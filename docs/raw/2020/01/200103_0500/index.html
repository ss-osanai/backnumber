【人は「自分に期待する水準」までしか到達できない】

2020/01/03 05:00 
■おはようございます、鮒谷です。


　自戒を込めて記しますが、
　結局のところ、人は

　【自分に対して期待する水準】

　までしか、
　到達できないのでしょう。



■昨年末に書いた、


　自分の目指す「あり方」が
　明確になっていない状態で、

　どれだけ「やり方」を
　学んだところで意味がない、


　という話に通じること。



■いくら学んだところで、


　実際に身体を動かして
　行動することは、

　お勉強して知識を入れるより
　少なく見積もっても

　「10～100倍は大変」

　です。



■なにが大変かといって、

　お勉強しているだけであれば
　外部との軋轢は生じませんが、


　ひとたび行動し始めた瞬間から、

　たとえば物を販売するときに、
　ちょっと強めに出れば、

　摩擦が発生するし、


　かといって、

　乏しい熱量でゆるゆるの
　アプローチをしても、

　何の反応も生まれません。



■一例を挙げると、


　たかが営業、
　とまではいいませんが、


　営業という、

　人生やビジネスにおける
　たった一つの側面においてさえ、

　形にしようとしたら、

　いちいち軋轢や摩擦を恐れている
　場合ではありません。



■むしろ、

　そうしたものを積極的に
　発生させていきながら、

　ときに長い時間をかけて
　微調整を繰り返し、

　ようやく

　「仕組み、システム」

　が出来上がってくるのです。



■それは、

　相応にエネルギーを要求される
　営みですから、


　「ありたい姿、
　　なりたい姿、
　　目指すべき姿」

　が明確でなければ、

　わざわざ、痛みを伴う（自分以外の）
　外部に向けての取り組みを、

　行おうなどという
　思いが湧いてくるはずもありません。



■多くの人が

　「お勉強はしているけれども、
　　まったく身体を動かしていない」

　のは、実に

　「摩擦を恐れる」

　言葉を換えれば

　「行動後に生じる結果を
　　つつがなく回収できるかどうか」

　に自信を持てないから。



■では

　「摩擦を恐れない人」

　あるいは

　「行動後に生じる結果を
　　つつがなく回収できる人」

　であれば行動できるかというと、
　そうではありません。



■その理由は、

　そもそも、そんな人は
　存在しないからです。


　どんな人でも
　摩擦を恐れるし、

　あるいは少なくとも、

　面倒に思うし、


　また、

　ここでいう「行動」とは、

　それまでにやったことの
　ないことを指すわけですから

　「つつがなく回収できるかどうか」

　など、やってもいないのに
　分かるはずもありません。



■では、どういう人が
　行動を起こせるのかというと、


　「摩擦や軋轢が生じるのは
　　覚悟の上で、

　　目指す世界に到達するには、
　　そこを乗り越えなければならない、

　　と覚悟が決まった人」


　であり、


　「行動後にとんでもない結果が
　　生じる可能性もあるし、

　　それを回収できかどうかは
　　分からないけれども、

　　回収するしか
　　前に進めないのだから。

　　覚悟を持って事にあたることを
　　決めた人」


　であるわけです。



■ここまでの話をお読み頂ければ、

　きっと少なくない割合で
　存在するはずの

　「どれだけでもお勉強はできるけれども
　　一歩も前に踏み出せない人」

　の

　「一歩も前に踏み出せない理由」

　が明確に理解できるはず。



■ここで冒頭の


　----------------------

　結局のところ、人は

　【自分に対して期待する水準】

　までしか、
　到達できないのでしょう。

　----------------------


　の話に戻るのですが、


　激しく行動すればするほど
　いろんな面倒ごとに巻き込まれるし、


　それを想像すると怖くなるから、

　蛇に睨まれた蛙のように
　動けなくなります。



■その恐怖を乗り越えるためには、

　恐怖を乗り越えて
　一歩踏み出し、

　さらにはそこから先、
　歩み続けようとするだけの

　「動機」

　が絶対に必要になるのです。



■その動機は、

　他人に強制されたものではなく
　内発的なものでなければならず、

　それは徹底した自問自答からしか
　生まれてくることはありません。
