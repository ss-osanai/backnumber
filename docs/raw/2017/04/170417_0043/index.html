　【気付かれたことや深く学ばれたこと】

　鮒谷先生の個別セッションは、
　完全にカスタマイズされた内容で、

　さらに先生の多くのコンサル経験に
　裏打ちされた、

　その場での偶発的でありながら
　適切なアドバイスに感動させられます。

　自分自身がお客様に対してサービス提供を行う際も、
　こうありたいと感じる内容です。


　【特に印象に残ったこと】

　鮒谷先生のアドバイスは、

　「きれいごと」ばかりでない点がメルマガや
　セミナーと決定的に異なる点であると感じます。

　○○○のご説明の際に「○○○○」のお話を
　して頂き、

　自分がいかに、生活と仕事に追われ、
　近視眼的になっていたか深く気づかされました。


　【今後活用できると思われたこと】

　今回、コンサルテーションを受けるにあたり、

　「鮒谷先生との出会いから
　　個人コンサルを申し込むまで」

　どのくらいの金額を、
　どの商品に投入したかを検証し、

　できればマネしたいと考えています。


　セリングを不要にするマーケティングという
　考え方は、

　今後どのようなビジネスに取り掛かるとしても
　常に意識していきたいと思います。


　【決意されたこと】

　今後、100年戦略が必要になる時代にあって、

　30年、40年ビジネスの世界に身を置くことを前提に
　将来のある時点から現在の自分をみるというか、

　例えば「20年後の自分から自己紹介をする」ような
　ものの見方をしていきたいと決意しました。

　このようなパラダイム転換のきっかけを
　与えて下さり、感謝しています！


　【鮒谷の個別コンサルティングの利用を
　　迷われている方へのメッセージ】

　これまで鮒谷先生のセミナーや音源を聴かれた方も、

　完全にカスタマイズされた自分だけの時間を
　取って頂くことにはまったく違う意味があります。

　特に、ミーティングのなかで偶発的に繰り出される
　アドバイスは、

　セミナーで一方的に鮒谷先生のお話しをお聞きして
　いることとは全く異なる価値があると思いますので、

　「たくさん音源を聴いたのに
　　向こう側へ行く実感が持てない」

　方はぜひコンサルティングを受けられることを
　お勧めします！

【 商売人は「また買わされた（笑）」の言葉をもらってナンボ 】
　　　　
───────────────────────────────────



■私（鮒谷）にとって最も嬉しいのが、

　「お客さまから（笑いながら）、
　　またお前に買わされてしまったよ」

　と言われる瞬間です。


　-------------------------------------

　【笑いながら】（←ここ重要）


　「また買ってしまったよ」
　「また買わされたよ」
　「またハマってしまった」

　-------------------------------------


　と言われるということは、

　心の中では、


　「また買わされて、
　　お金も使わされてしまったけれども、
　　
　　まあ、それはそれでアリだよね、
　　結果としては、良かったよ」


　と思われている証拠（のはず！？）だから。



■もし、


　-------------------------------------

　「また買ってしまったよ」
　「また買わされたよ」
　「またハマってしまった」


　【ふざけんな（怒）】

　-------------------------------------


　という感情であったら、

　
　【笑いながら】


　そんな言葉を発することなど
　できないでしょう。



■冗談交じりに、
　そのような言葉が発せられた時点で、

　既に一定レベル以上の関係性が構築され、
　親近感を持って頂けている、

　ということです。


　ですから、

　「笑いながら、
　　（一見）ひどいこと（？）を言われる」

　というのは、

　サービス提供者にとっては
　大いに喜ぶべき出来事といえるでしょう。



■だから、もし、

　未だ、お会いしたことのないお客さまから
　そんなことを言われたとするならば、

　
　それは、たとえば私（鮒谷）にとっては


　「諸手を挙げて、おおはしゃぎ」

　あるいは

　「ガッツポーズで、快哉を叫ぶ」

　あるいは

　「爆発する喜びに、狂喜乱舞」


　くらいの大変な出来事。



■つまり、


　商売を行っている人であるならば、
　目指すところは、
　
　毎日、たくさんのお客さまから


　-------------------------------------

　【笑いながら】（←ここ重要）


　「また買ってしまったよ」
　「また買わされたよ」
　「またハマってしまった」

　-------------------------------------


　と言われることではないか、
　と提案したいのです。



■ちなみに私（鮒谷）は、毎日、毎日、
　来る日も、来る日も、

　大勢のお客さまから、直接、
　そのように声をかけられたり、

　お会いしたことのない方から、
　ご感想を頂戴したりしています。

　「商売人冥利に尽きる」とはこのことです。
　


■【日記放談会音源】についても、同様に

　-------------------------------------

　【笑いながら】（←ここ重要）


　「また買ってしまったよ」
　「また買わされたよ」
　「またハマってしまった」

　-------------------------------------

　と、さんざんに言われてきましたが（汗）
　
　日々、頂戴するご感想を読ませて頂く限り、
　ご満足頂けていているようで、

　安心しているところです。



■初めての音源としても、

　二度目、三度目、四度目、、、
　の音源であったとしても、

　ぜひお聴きになられてみて下さい。

　そして「また買わされたよ」の声も是非（笑）
