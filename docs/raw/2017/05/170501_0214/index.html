＜今日のピークパフォーマンス方程式＞

【 決断においては「波及効果」と「回数」の両方を意識する必要がある 】
　　　　
───────────────────────────────────



■私たちは、毎日、

　大小幾多の意思決定（決断）を
　行っているわけですが、

　意思決定にあたっては、

　【「波及効果」と「決断の回数」の両方を
　　同時に追求する】

　ことが大切です。



■すなわち、

　人生やビジネス、
　あるいはキャリア構築においては、


　【より波及効果が高いと思われる
　　意思決定を優先的に行う】


　ことが重要ですが、その一方で、


　【そこまで波及効果は高くないものの、
　
　　日々、決済していかなければならない、
　　数々の決定決断も大量に行なっていく】

　
　必要もあるのです。



■もちろん、本来、優先順位としては、


　-------------------------------------

　▼波及効果の高い、数少ない意思決定
　
　　↓

　▼そこまで波及効果の大きくない、
　　日常的に生じる、数多くの意思決定

　-------------------------------------


　の順番で進めていく必要があるでしょう。



■ただ、意思決定の水準を高める
　訓練のためには、

　ある決断を行い、行動し、
　結果を見ては、次の意思決定に活かし、

　再び、次の決断を行ない、行動し、
　結果を観察しては、次の意思決定に活かし、

　
　という一連の流れを

　【超速で】

　回し続ける必要があるのです。



■自らの決断のクオリティを引き上げるため、
　そんな意思を持って、

　無数の意思決定を繰り返す毎日を
　過ごしているうちに

　【それなりの高い精度の決断】

　が下せるようになるでしょう。


　こうして品質を高めた意思決定基準を

　【ここぞという決断（波及効果の高い決断）】

　に適用することによって、

　人生やビジネスにおいて、大外しせず、
　それなりに成果を出せるようになるのです。



■ここにおいて、ぐるっと一周回って、

　【より波及効果が高いと思われる
　　意思決定を優先的に行う】

　ためには

　【そこまで波及効果は高くないものの、
　
　　日々、決済していかなければならない、
　　数々の決定決断も大量に行なっていく】

　という一見、相矛盾する両者を
　両立させる必要が出てくるということ。



■波及効果は低くとも、気を入れて、

　【決断 → 行動 → 結果 → 観察】

　を数多く行い、
　意思決定の訓練を重ねるうちに、
　
　【決断の地力】

　が付き、そこまで数の多くない、けれども、
　きわめて波及効果の高い意思決定を、

　高い水準で、かつ的確に、迷い少なく、
　前倒しで実行できるようになるのです。



■本日の話を総括するならば、

　意志決定においては、
　波及効果の高いものと低いものがあり、

　もちろん、優先的に、かつ、極力、
　間違いのないように取り組まねばならぬのは

　「前者」

　であるけれども、


　同時に、そこまで波及効果の高くない
　意思決定だからといって

　「後者」

　も手を抜くことなく、時々刻々、
　気を入れて大量のPDCAを繰り返し、

　【意思決定筋】

　を鍛える必要があるということです。



■この両者が統合されたところで

　【圧巻のパフォーマンスが生まれる】

　こととなるでしょう。


　つまり、

　【圧巻レベルに到達したいのであれば、時々刻々、
　　片時も気を抜けない（抜いてはならない）】

　ということですね。
