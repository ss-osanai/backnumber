　説明してくれるというのでしょう。

　「こんなものが欲しかったんですよ！」

　と言ってくれるのを待つのでしょうか。

　そんなこと、
　起きるはずがありません。

　【商売人】

　だけなのです。

　【元気】

　「潜在的に」

　「あなたは

　と問いかけ、説明しながら、
　飢えていることに気づいて頂き、

　お届けしようとしているということに
　なるわけです。

　【毎朝、起きたときから
　　元気いっぱいで、

　　夜毎、充実感に満たされながら
　　床に就く】

　私（鮒谷）に言わせれば
　間違いなく

　「『元気』に飢えていらっしゃる」

　のです。



■いや、そんなはずはない、
　元気は足りてます、間に合ってます、

　と思われる方もあるかもしれません。


　しかし、おそらくは多くの場合

　【病気でないのを
　　健康と勘違いしている】

　のと同じように

　【元気が足りないのに、
　　あるように誤解されているだけ】

　なのです。



■身体に例えるならば、それは

　【未病】

　すなわち、

　【発病には至っていないけれども
　　必ずしも健康な状態とはいえぬ状態】

　と言えるでしょう。



■そんな未病状態の方に対して

　「それを未病というのですよ」

　と繰り返し、説明責任を果たし、

　真の健康とは、
　について語る必要があるでしょう。
