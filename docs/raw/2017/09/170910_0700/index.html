＜今日のピークパフォーマンス方程式＞ 

【 「必敗必至の竹槍営業」から脱却したければ… 】 

───────────────────────────────────


　
■最近、訳あって（？）

　「とある人材公募サイト」

　にて、優秀な人材を探しているところです。


　そんな話を昨日、いたしました。

　今日も続けます。

　

■昨日は

　【自分の目線・都合でのみ、提案をしていても、
　　それが通るはずがない】

　という話をしました。


　これが分かって、
　相手を理解することに徹している人は、

　多少、知識や能力が劣っていようが、
　不器用なところがあっても

　「売れる人」

　になれるのです。


　これが分かっていない人は、

　たとえその人に知識や能力があり、
　小利口に立ち回れる人間だったとしても

　「売れない人」

　のまま。



■自分が提案を受ける側なら、
　どういう人にお願いしようと思うだろう。

　こうした想像力を
　働かせ続けられる人だけが、

　物を売れるし、
　商売もうまくいく、

　という当たり前の話です。



■こうした

　「当たり前の世界」

　を体得している人を見たことがない人は、


　そもそも目指すべき方向が定まらず、

　あるいは、仮に定めても、
　たいていの場合、

　それが見当はずれなものとなることが
　多いように思われます。


　
■典型的な例は

　「何度も通えば、心を開いてくれる」

　といったような的はずれな努力系、

　（ただの迷惑です）


　あるいは、

　「私を男にして下さい」

　といった相手のことを一切、考えない、
　己に酔いしれている熱血根性系、

　（ただの迷惑です）


　あるいは

　「買ってくれるまで帰りません、
　　帰れません」

　といった居座り居直り犯罪スレスレ系、

　（ただの迷惑です）


　などがありますね。



■たまたま、

　カギとカギ穴が合うが如くに、
　この作戦がヒットすることもあるかもしれません。


　でもそれは、ほんとうにたまたまであって、
　
　【まったく普遍性はない】

　のですから、

　合わない人にとっては、
　時間を奪われ、精神衛生も害され、

　迷惑至極の存在となるでしょう。



■誤った、人さまに迷惑をかける方向に
　向かってどれだけ努力しても、

　永遠に、　

　【請われる、頼まれる、頼りにされる】

　といった世界に出られません。

　
　
■いい加減、気がつくべきなのに、

　一向に気づかず、ただひたすらに
　同じことを繰り返す人を総称し、

　私は

　【竹槍系の人】

　と呼ぶことにしています。


　延々と前に後ろに竹槍を突いては引き、
　引いては突き、

　を繰り返しているだけの、

　「芸も工夫もない人」

　という意味です。


　
■こうした人は、

　槍筋がハッキリと見える
　（＝ワンパターン営業）

　から、次々に打倒されていくのです。

　いくら営業を頑張っても玉砕続きとなる。


　そんなザコキャラになりたくないのなら、
　これを観に行くべし！

　
　『舞台 北斗の拳－世紀末ザコ伝説』
　http://www.clie.asia/hokuto_zakobutai/


　、、ではなく、


　圧倒的にモノを売っている人が
　何を考えているのか、何を行っているのか、

　徹底リサーチすべきでしょう。



■売れている人がどこにいるのかを
　調べようともせず、

　いることが分かっても、
　接しに行こうとせず、

　教えを請おうともしない。


　そして、いつまで経っても、
　自分勝手な無手勝流の、

　独自営業方式で押し通そうとしても
　うまくいくはずがありません。


　うまくいっていないのなら、
　うまくいっている人に教えを請う、

　何ごとによらず、これが一番の近道です。



■超弩級レベルで販売している人は、
　例外なく、


　【お客さまについて
　　徹底的にインタビューし、

　　お客さま以上にお客さまを理解し、
　　「痒いところに手が届く」以上の提案を、

　　何度も何度も繰り返し行い続けて、
　　圧倒的な信頼を獲得していった人】


　です。


　
■そんな人が何を考え、
　何を行っているのかを、

　間近で観察し、やってみて、
　うまくいかなかったら微調整を繰り返し、

　を重ねていくうちに、
　その世界が徐々に近づいてくるのです。



■何も営業に限った話ではありませんよ。

　人が集まってくる人は、みな、
　これを行っているはずですし、

　人が離れていく人は、みな、
　これと反対のことを行っているはずです。


　たまたま、
　営業についての話をいたしましたが、
　
　卓越したコミュニケーション能力を持つ人が
　営業を行ったら、そりゃ売れるでしょうよ、


　というだけの話。



■なにも営業に限った話ではありません。

　人生や生活の質をあげたいのなら
　万人が心がけるべきこと、


　それが、

　【周囲にいる人に対して関心を持ち、
　　徹底的にインタビューし、

　　当人以上に当人を理解し、
　　「痒いところに手が届く」以上の配慮を、

　　何度も何度も繰り返し行い続けて、
　　圧倒的な信頼を獲得していく】
　
　という行動なのではないでしょうか。
　

　私（鮒谷）も、もちろん修行中の身ですが、
　死ぬまで、この道を歩んでいくつもり。
