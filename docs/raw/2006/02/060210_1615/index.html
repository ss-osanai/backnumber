＜今日のピークパフォーマンス方程式。＞

　【　ビジネスパーソンは長期継続して評価し続けられる「芸人」であれ  】

───────────────────────────────────



■世に「芸人」と呼ばれる人たちがいます。


　広辞苑第五版によると「遊芸・芸能を職業とする人」がすなわち
　芸人とのことですが、

　観衆に対して何かの芸を披露し、その対価としておひねりをいただく。


　「お駄賃」ではなくて「おひねり」をいただくのが芸人だ、ビジネスに
　携わる人も同様である、ということを小阪裕司さんが言われています。

　わたしはこの内容に非常な感銘を受けたんですよね。


　【今日のお勧め本
　 　「惚れるしくみ」がお店を変える!―大繁盛のしくみづくり】
  ⇒ http://amazon.co.jp/o/ASIN/4894511045/2ndstagejp-22/ref=nosim

　【今日のお勧め本　失われた売上げを探せ】
　⇒ http://amazon.co.jp/o/ASIN/4894511215/2ndstagejp-22/ref=nosim


　※いずれも私の大好きな本です。
　　小阪さんの本をまだ読まれたことのない人、仕事観が変わる人も
　　出てくると思いますよ。



■芸人は、芸を磨いてはじめて、おひねりを投げてもらうことができます。

　芸のこやしという言葉がありますが、いろんな経験をすることに
　よって芸に磨きをかけ、その芸によって渡世するのが芸人です。

　

■ビジネスパーソンも本質的にはまったく同じ。

　商人として活躍し、それに対する対価を頂戴して、さらにその対価を
　自らの「芸＝ビジネスパーソンとしての提供物」を磨くために、
　資本投下する。



■そうやって得られたさまざまな知識を得て、経験を自らに施し、
　得られたものを、また改めて顧客に提供するわけです。

　そのさい、良い芸（顧客に対しての価値提供）をすれば拍手喝采され、
　おひねりが飛んできます。

　ところが悪い芸をすれば、芸人に座布団が飛んでくるようにクレームが
　きます。

　芸人と何ら変わるところはないのです。



■「何に対してお金を頂戴しているのか？」

　それを忘れてしまえば、そこから地獄が始まります。



■昔から言われますように、一発当てることは誰にでもできることで、
　維持することはその何倍も難しい。

　それを継続して発展させることは、さらに難易度の高い芸当です。

　それが証拠に、芸能の世界で一発あてた人は過去にゴマンといますが、
　それを維持している人はどうでしょう。

　グッと数が減るのではないでしょうか。



■一度つかんだポジションを維持している人のいかに少ないことか。

　みなさんも、去年は大ブレイクしたけれども今年はもう見ないという
　芸人さんの名前を何人も思いつかれるのでは。

　さらには人気を維持したうえに、毎年毎年、右肩上がりで高みに
　のぼっていった芸人といえば、雨夜の星となるのではないでしょうか。



■一発屋で終わることなく、長期的に評価され続けてこそ真の芸人。

　ビジネスパーソンも常に能力を磨き、現状維持のための情報収集や
　勉強を怠らず、余念なく芸を磨き続けなければなりません。



■わたしもビジネスを行っている中で、頂いたおひねりは自らの
　芸を磨くための投資に最優先にて投下しております。

　セミナーに行くのも、本を購入するのも、さまざまな教材を買って
　読むのも、毎日毎日、さまざまな方と会食を共にするのも、

　すべては自らの芸を「人間国宝級」に磨き上げるための一つ一つの
　地道な努力だと思っているのです。

　といっても、まだまだその道は始まったばかりなのですが。
