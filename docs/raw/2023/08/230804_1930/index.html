■十年一日の世界から抜け出したい人は

───────────────────



■十年一日で、
　
　10年前と（本質的に）
　おんなじことをやっている人、

　ありませんか？



■いや

　「おんなじこと」

　が、

　-----------

　１、やりたくてたまらない

　かつ

　２、やればやるほど深まる

　-----------

　ものであればいいんですよ。



■それは前号のメルマガで
　書いたところの

　【閾値超え＝超過利潤の源泉】

　となりますから。



■問題は

　-----------

　１、やりたくもない

　かつ

　２、いくらやっても深まらない

　-----------

　ものに資源（主として時間）を
　割いていて、

　-----------

　３、その状態に満足していない

　-----------

　とき。



■こんなときにはどうすべきか。

　以下の２つ、つまり

　-----------

　１、今やっていることを止める

　２、新しい世界に飛び込む

　-----------

　しかありません。



■最低でもどちらか片一方、
　理想的には二つともいっぺんに、

　実行に移すことですね。


　特に、あなたが若かったり、

　配偶者や子どもや老親や
　その他守るべき人がなかったり、

　それなりに大きな蓄えが
　あったりするのなら、

　-----------

　１、今やっていることを止める
　２、新しい世界に飛び込む

　-----------

　の同時選択もありでしょう。



■とはいえ、１は恐ろしいですね。
　
　現在の状況を一気に捨て去るのは
　リスクがあります。


　でも、思い切って前に進めば、
　案外、なんとかなるものです。

　ただし、そうならないことも
　あるから恐ろしい。



■そんなあなたのための、
　もう一つの選択肢としては、

　-----------

　（今やっていることを続けつつ）

　一定の時間を天引きして、
　新しい世界に飛び込む

　-----------

　こと。

　こちらのほうは相対的に
　小さなリスクでリターンを狙えます。



■このあたりは各人の状況が
　それぞれ異なるので、

　いちがいに、これが正解、
　なんてことはいえません。



■ただし、

　-----------

　１、今やっていることをそのまま続けて

　２、新しい世界には向かわない
　　（今いる世界に張り付く）

　-----------

　という意思決定をしながら、

　今と違う人生にしたい、
　今と異なるキャリアを歩みたい、
　
　と思っても、
　それは絶対に無理、

　ということだけは間違いありません。



■ということで、
　自身取り巻く状況を勘案しつつ、

　最適な選択肢を各人それぞれが
　選択されると良いでしょう。


　そんな中、

　もっとも安全に（リスクを下げた上で）、
　超過リターンを獲得したいのなら、

　-----------

　１、今やっていることを続けたまま
　２、新しい世界を探し始める

　-----------

　ことですね。



■新しい世界とは、

　今、あなたが生きている、
　半径３メートル、５メートルといった、
　
　意識していなくても
　勝手に触れている世界ではなく、

　【多少、頑張って、
　　跳躍しなければ決して触れられない】

　そんな世界のこと。



■そうした世界を覗き見るところから

　-----------

　１、イメージが湧き
　２、感情が動き

　３、なんらかの欲求が胎動しはじめ
　４、思考が揺さぶられ

　５、小さく行動を始めてみて
　６、意識の変容がもたらされ

　７、前進し続けるうちに世界が変化し
　８、気がついたら別世界に生きていた

　-----------


　という旅がスタートするのです。



■いきなり、なにもないところから

　-----------

　３、なんらかの欲求が胎動しはじめ

　-----------

　とか

　-----------

　５、小さく行動を始めてみて

　-----------

　とか

　-----------

　７、前進し続けるうちに世界が変化し

　-----------

　とか

　-----------

　８、気がついたら別世界に生きていた

　-----------

　などは起こりようがない、
　ということですね。


　
■まあ、そういうことです。

　とにもかくにも

　【新しい世界を覗き見る】

　という意思決定をしない限り、
　なんにも始まりません。



■以下は、今度開催するセミナー内容の
　事前チラ見せとなります。


　（ここから）
　-----------


　年間2万足を売る
　便所サンダル専門通販サイト、

　ない街の地図を作るプロフェッショナル、

　軽妙な下ネタトークでリピーターを
　生み出しまくる大人の博物館……。


　世の中には驚くべきニッチなジャンルで、
　圧倒的1位のポジションを確立し、
　独自すぎるビジネスを営む人間がいる。


　商売と生き様が、たぎるような熱量のもと
　一体化する、いわばマニアな人々。


　講師は合同会社別視点代表の松澤茂信。

　マツコの知らない世界・タモリ倶楽部に
　出演経験があるマニアが多数出展し、

　例年1万人以上の来場者が押し寄せる
　人気イベント「マニアフェスタ」の主催者で、

　月間100万ＰＶ・Twitterフォロワー
　10万人の珍スポット紹介ブログ
　「東京別視点ガイド」の運営者でもある。

　
　日本中のあらゆるマニアと交流し、
　自身もマニアな事業を展開する経験をもとに、

　事例とエピソードトークをたっぷり交えて、
　ニッチジャンルで独自ビジネスを
　生み出す方法をお伝えする。

　（ここまで）
