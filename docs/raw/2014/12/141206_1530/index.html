　   骨太かつ卓越した＜本物の教育サービス＞を通して
　  「変容の喜び」を提供し続ける
　　------------------------------------------------

　ことを目的とし、4000日以上にわたって毎日お届けしている、
　個人発行として日本最大級のビジネスメルマガです。

　熟読するも読み流すも、自由な形でご活用下さい。

＜昨日の活動ダイジェスト＞

▼今日は終日、年間プログラム ＋ 昨日のメルマガへのご感想

───────────────────────────────────



■こんにちは、鮒谷です。


  今日は終日、月に一度の年間プログラム。

　限られた人を対象に、
　朝10時から夜9時まで、ひたすら話し続けます。



■話は変わりまして、

　１年数ヶ月前、３日間セミナーに
　参加下さったＩさんより、


　昨日のメルマガ

　【 「やればできる感」さえ持てれば、
　　　　　　 やれるようになり、できるようになる 】

　について、以下のようなご感想を頂きました。


　（ここから）
　-----------------------------------------------------------


　こんばんは！
　３日間セミナー22期の◯◯です！

　いつもお世話になりましてありがとうございます。
　ご無沙汰しております。（汗）

　毎日メルマガを楽しく拝読させていただいております！


　本日のメルマガ、、、
　今の僕にグッと刺さりました！！


　・・・というのは、

　僕が鮒谷さんからいただいた無形資産は、
　まさに本日の

　「やればできる感」

　だと考えるようになってきているからです。


　もちろん、現状は、

　＞　１、短い時間で、小さな成果

　ではありますが、、、（汗）


　そして、このまま前進して、

　＞２、中くらいの時間で、中くらいの成果

　まで到達できる道をつくり続けていくつもりです。


　少なくとも、

　「自分から何らかのアクションを起こせば、
　　何かのリアクションがある」

　ということが、少しずつ身体で理解しかけているのでは
　ないか、という肌感覚です。


　まだまだ未熟で、先日のメルマガで表現された

　「遠慮気味に変化する人」（笑）

　ですが、

　昨年７月の３日間セミナーから１年４ヶ月で、
　小さな小さな

　「やればできる感」

　が育ってきております。


　それは、言葉をかえると、

　「未来への自分への期待」

　と言えるものです。


　あれほど何度も挫折したブログを、
　３日間セミナー以降500日以上継続できていることも
　少なからず影響しているものと存じます。

　本当にありがとうございます！ 
　

　-----------------------------------------------------------
　（ここまで）


　Ｉさん、ご感想ありがとうございます。

　そして勝手紹介、スミマセン（汗）



■実は、私（鮒谷）がセミナーを行っていて、
　一番嬉しく思うのは、
　
　こういったご感想を頂戴するとき。
　

　３日間セミナー（現・２日間セミナー）は、

　「このセミナーに出れば、
　　３ヶ月で収入が２倍になりますよ！」

　的なことは一切謳わない、
　
　したがって、

　極めて集客しにくいセミナーなわけですが、


　そう言っちゃったほうが集客しやすいと
　分かっていても言わないのが、

　「約束して、出来ないことは言わないという、
　　正しい大人の姿勢」

　であろうと思ってやってきたわけです。



■その代わり、
　そんなことよりもっともっと価値のある、


　「一生ものの財産

　　（パラダイムとか、思考習慣、行動習慣、
　　　あるいは自信とか自尊心、未来の可能性とか）」


　をお渡ししようと頑張ってまいりました。

　

■１ヶ月とか３ヶ月程度で、

　100％、起業を成功させるとか、
　劇的に収入が増やしてみせるとか、

　そんなことは一切言わない代わりに、

　
　生涯にわたって、価値を生み続ける財産を
　お渡しすることに焦点を絞り、

　そのこと一つを意識して、
　話をさせて頂いてきたわけです。



■だからこそ、

　ご縁を頂いてから半年、一年、
　あるいは数年が経過してから、
　
　こうした感想を頂くのは本当に嬉しいことですし、


　同じく、半年、ないし数年という単位で、

　再びセミナーにお越し下さったり、
　コンサルをお申込み下さる方があると、

　提供したサービスに対して一定の評価を
　頂いていることが分かり、安心もいたします。


　（そして実際に、
　　継続してお付き合いさせて頂く方とともに、

　　不定期で長らくお付き合い下さる方が、
　　たくさんいらっしゃることを誇りとしています）



■そんなわけで、

　今日はこのようなご感想を頂戴し、
　皆さんに紹介せずにおれなくなり、

　取り上げさせて頂きました。


　Ｉさん、ありがとうございました！

　そして、ご感想の最後にも記して頂いておりましたが、

　ぜひ来年、個別コンサルのお申込み下さることを
　楽しみにお待ちしております（笑）


　これからも引き続き、よろしくお願いいたします。

　


■それでは

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【 「カネをドブに捨てるような学習」をしたくない人だけ御覧ください 】

───────────────────────────────────



■年間のべ1000人を越えるペースで、

　他者の変容のお手伝い（個別コンサル）を
　続けているうちに、
　
　だんだん、
　
　「人の変わり方」
　
　が分かってくるようになりました。
　
　
　その事例（？）がたくさん蓄積されるほど、
　人が変容するに際して、
　
　「本質的に関係ある部分」
　
　と
　
　「一見、関係があるように見えて、その実、
　　ほとんど無関係、瑣末といっていい部分」
　
　の違いを、早く理解できるようになってきます。
　
　
　
■要は、

　多数の変化事例に触れることによって、
　
　「変化する人、変化できない人、それぞれに
　　共通するパターンを認識できるようになる」

　ということ。


　たとえば一番分かりやすい例でいうと、
　
　短期間で大きな変化を遂げる人は
　例外なく、


　「行動から入り、
　　その行動から生まれた結果を見て、

　　（そこで初めて）学んでから微調整する。


　　さらに行動してみては
　　その行動から生まれた結果を見て、

　　（そこで初めて）学んでから微調整する。


　　、、、以下、繰り返し」


　こうしたサイクルを回している人です。



■反対に、

　大量の本を読み、たくさんのセミナーに通い、
　

　成果を上げる方法について、
　　
　偉そうに、上から目線で、賢そうに、
　根拠らしきものを提示しながら、
　
　滔々と語られている人に対して、


　「なるほどですね。

　　では、実際のところ、
　　あなたはどんなことを成し遂げられましたか？」
　
　
　と問うてみると、


　「え、、、！？」

　
　と絶句してしまう人もあります。



■結局、

　成果を出している人は行動主体の人ですし、

　いくら勉強をしていても、
　行動していない人は100年経っても変われません。

　
　厳しいかもしれませんが、これが現実。



■こうしたことは、


　たくさんの

　「変わろうと思って、実際に変わった人」

　「変わりたくても、変われない人」
　　
　に触れることで初めて理解できたこと。



■でも、

　大量の事例に触れていなければ、

　「学習 → 行動」

　の罠に陥ってしまうのも、
　分かるような気もしませんか。


　実際のところ、

　「学習 → 行動」

　が正しいと信じ、
　学んでから行動しようと思いつつ、
　

　「学習 → 行動（しようと思っているけど、
　　　　　　　　　　　　　学びすぎて動けない）」

　となってしまっている人を
　たくさん見てきました。



■実際に、

　無数の様々な人たちとお会いすることに
　よってのみ、
　
　「論より証拠」

　で、

　「何が本質的なことで、何が瑣末なことなのか」

　が分かるようになるものです。


　そして、

　常識、思い込みと現実は全く別のもので
　あることもよく分かりました。



■上に挙げたような、

　一番、納得して頂きやすそうな例ですら、
　誤解している人がたくさんあるのだから、

　「それっぽいけど、それは間違い」

　という思い込みは、
　それこそ無数にあるわけです。


　私がこうしたことを理解できたのは

　「何より現場を重視した」

　からです。

　
　銀座や六本木に繰り出す代わりに、
　毎日、毎日、ひたすら個別コンサルを繰り返す、

　そんなストイックな生活（？）から
　本質が見えてくるようになりました。



■で、ここまで書いて、気づきます。


　やっぱり、何かを学ぶ際には

　「圧倒的な臨床事例を持った人から
　　学ばなければならない」

　ということに。


　私（鮒谷）は

　「人は如何にして変容するのか」

　「あっという間に変わっていく人と、
　　いつまでたっても変わらない人」

　という、

　限定された分野における事例蓄積の量については、
　人後に落ちない自信がありますが、

　それ以外の分野については
　まったくもって浅学非才、無知蒙昧。



■だから、

　それ以外の分野を学ぶためには、
　誰かから教えてもらう必要がありますが、


　その際、


　「流暢に立派な論を述べている人（机上の専門家）
　　よりも、

　　圧倒的な証拠（現場で見聞きしてきいた事例）を
　　持っている人から学ぶべき」


　だと確信してするようになりました。

　上に記したような、私自身の経験から。



■たとえば私（鮒谷）が

　「人生を変えたい」

　と思ったら、

　（実際には不可能ですが）

　自分にコンサルしてもらいたいです、本当に。


　ずーっと現場に張り付かなければ、
　
　絶対に見えてこない、分からないことが、
　たくさんあるから、

　そんな現場経験を持った私自身に
　教わりたいのです。



■そんなわけで、


　あなたが、何かを伝える立場であるならば、

　当該分野において、ずっと現場に張り付いて、
　圧倒的な事例蓄積に励むべきでしょうし、


　あなたが、何かを学ぶ立場であるならば、

　当該分野において、ずっと現場に張り付いて、
　圧倒的な事例蓄積をしている人をこそ、

　たのむべき、


　と考えます。

　＜120％、あなた仕様にカスタマイズ＞

　してお届けするのが、鮒谷の個別コンサルティング。


■経営戦略、ビジネスモデル設計、マーケティング
  情報発信、営業といったビジネス直結の話から、

  キャリア相談、独立起業に当たっての心構えや実践方法、
  何らかの決意の後押し、事業計画相談、

　などなど、経営、ビジネス、キャリアに関わるご相談、
　承っております。


■補足すると、コンサルを申し込むとは、
　鮒谷の時間を買うことではありません。

　それをきっかけとして、あなたの
　「より良き人生、より良き未来」を買うことです。

　だからこそコンサル終了後、

　「高いはずだったものが、
　　むしろ安かったと感じられるようになる」

　わけです。次に結果を出すのはあなたです。
