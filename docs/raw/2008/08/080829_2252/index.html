■昨日は

　コミュニケーションの事前準備として、

　「即効性」と「遅効性」の二つの準備があることを
　知りましょう、

　というお話をいたしました。



■あわせて、

　コミュニケーションにおいては「即効性」と「遅効性」の
　準備のコラボレーションで、

　撃墜（？）される確率を大幅に減少させ、
　次につながる話へと展開することができるようになる、

　というお話をしたかとおもいます。



■今日は、さらにその上を行く？、とっておきの裏技を
　ご披露したいとおもいます。


　以前、ある方からのご紹介で、

　ベネッセ・コーポレーションの社長さんと会食を
　ご一緒させていただく機会を頂戴いたしました。


　ベネッセさんといえば、教育業界の雄として、
　連結売上高、3845億円（2008年3月期）

　を誇る、押しも押されぬ大企業。



■その社長さんと会食の機会をいただき、
　
　あるビジネスについてのご提案の機会を頂戴する
　ことができることになったわけですから

　それは大きなビジネスチャンスの到来でもありました。



■そのときに、同社、ならびに同社社長との

　「共通の○○」

　は一体、どこにあるのだろう、


　と、高校の後輩、川俣君の尽力で

　いろいろ同社について分厚い資料を作ってもらって
　事前の予習をしたりしたものです。

　（川俣君、その際はどうもありがとう）



■そんななか、当時、

　わたしのやることについて公私にわたって
　いろいろ手伝ってくれていた浅見君
　（当時学生、現在、リクルートで新規事業に携わっています）


　からの提案があり、
　
　一緒に香川県の直島


　＜ベネッセアートサイト直島＞

　http://www.naoshima-is.co.jp/


　に日帰りツアーに行くことになったのです。


　
■じつは直島というのはご存じの方もいらっしゃると
　おもいますが、同社が

　「瀬戸内海の風景の中、ひとつの場所に、時間をかけて
　　アートをつくりあげていく」

　という活動の中で、

　直島の自然や、地域固有の文化の中に、現代アートや建築を
　置くことによって、

　どこにもない特別な場所と経験を創造しようとする
　試みが行われている島です。



■同社としても非常に思い入れの深いプロジェクト
　だそうで、

　われわれも、その取り組み事態にとても関心を持つと
　同時に、

　今度、社長にお目にかかるときに
　
　「共通の○○」

　として、直島を訪れようということになったのが
　お会いさせていただく一週間ほど前。


　日程の関係もあり、東京から日帰りの、
　かなりの強行軍でした。



■しかし、

　直島に展示されていたアートは非常に
　すばらしく、強行軍でも行く価値があったな、ということは
　もちろん、

　（あわただしかったのは残念でしたが）
　
　社長との会食の際に

　「つい先日、直島に行きました！」

　というお話をさせていただくと、
　とても喜んで下さいました。


　おかげさまで、ご提案させていただいた案件を
　担当部長に降ろしていただき、
　
　実務レベルの話に降りてくることとなったのです。



■そんなふうなことをこの何回かの連載のなかで
　思いだし、

　裏技、ウルトラＣとして、
　

　【 「むりやり共通体験」を作り出す 】


　ということも場合によっては、とおもった次第です。


　
■じつは、

　私自身、スケジュールの関係で、

　友人、知人である方、
　
　あるいは

　それらの方からのご紹介なく、
　人にお会いさせていただくことがほとんどできないのですが、


　ある

　【 むりやり共通体験 】

　を持ってこられて、

　そこまでされたのであれば、ということでお目にかかった
　ことがありました。


　これも（内容はあえて書きませんが）
　アッと驚く共通体験づくりをされて、一気に心を許した、

　という自分のなかでの経験として記憶、保存されています。



■ここまでの話をまとめると、
　

　【 むりやり共通体験を作る 】


　ということは、場合によって強烈なインパクトを与える
　出逢いの演出ではなかろうか、

　そんなふうにも思うのです。

　
　
■いやらしくなく、押しつけがましくなく、
　さりげなく、喜んでいただけることが想定される場合、


　（そのあたりの温度調整がたいへんに難しいところでは
　　ありますが）


　こういった方法もコミュニケーションの一つの
　オプションとして存在するのではないか、

　ということを私自身の備忘のためにも
　記録して、残しておこうと思います。
　


■「コミュニケーションについて」の参考材料を得るための
　情報源として、ご活用いただければとおもっています。
