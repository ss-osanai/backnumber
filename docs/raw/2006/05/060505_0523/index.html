───────────────────────────────────



■「成功法則」といわれるものがあります。

　これは、ひと昔前は、よく本を読む人であったり、セミナーに出る
　習慣でもなければ身近に接する機会はありませんでした。

　ところが現代は、成功に至るまでの考え方からスキルやノウハウに
　至るまで、一般に流布(るふ)し、手軽に、かつ安価に触れることが
　できるようになりました。

　インターネット上で簡単に、無料で、ある程度のレベルまで成功できる
　だけの考え方、スキル、ノウハウが手に入ります。



■毎日「平成・進化論。」で配信させていただいていることも
　その情報のひとつといえるでしょう。

　ここでお話ししていることは私のオリジナルの考えではなく、
　
　すでに実績を残してこられたさまざまな経営者やコンサルタントの方、
　お世話になってきた先生や、友人・知人から教わったものであり、

　かつ、自分自身がその教えに従うことで実際に結果を出すことが
　できた内容ばかりです。



■したがって「平成・進化論。」で申し上げている内容は、
　そのまま活用することによって、ビジネスパーソンとして求められる
　高い結果を出すことができるものと確信しています。

　こうした思考のプロセスには、「結果を生み出す再現性」が
　あるからです。



■再現性とは、いつ、誰が使用しても、ほぼ同様の結果を生み出せる
　ということ。

　そういう内容でなければ配信する意味がないと思い、
　記事の選定にはことのほか気をつけているつもりです。

　そもそも、一部の天才しか利用できない成功方程式を持ってきた
　ところで、それは私の方程式としてはとうてい活用不可能な
　ものであるのです。



■いまこの「平成・進化論。」でお送りしていることは、
　なにより私自身が5年前、10年前に知りたかったことばかり。

　正直、読者さまがうらやましいと思うことさえあります。

　これは私が配信している内容が優れているとかどうだとかいった
　ことではもちろんなく、

　勉強熱心な「平成・進化論。」読者様にとって、非常に有利な
　世の中になってきたという意味で申し上げているのです。



■さきほど申しましたように、ほんのちょっと前までは、
　成功法則やノウハウのごときものは、
　ごく一部の、マニアの間の話でしかなかったありませんでした。

　そんな情報が、世間一般に当たり前のように流布しはじめたことは、
　まったくもって画期的な、事件というべきもの。



■いまや数多くの成功法則はネット上に氾濫(はんらん)していると
　いっていいほどで、ともするとその貴重さを忘れてしまいがちです。

　もし、それが原因で「知っているのに実行していない」としたら、
　いかにもったいないことであるかお分かりになることでしょう。



■精度の高い結果を出すための方程式が、容易に入手可能な時代になった。

　これはすばらしいことでありますが、また同時に、

　知識レベルでは全員が横並びの状態になってしまったことも
　意味しています。

　では、こうなってきたときに、どこで差がつくのかといいますと…。

　この続きは明日お話しいたします。
