■こんにちは、鮒谷です。


　明日、日曜日に行われる、
　中学・高校の同級生Ｏ君の結婚式に出席するため、
　前日から横浜入りしています。

　（久しぶりの横浜！）


　
■中学・高校６年間を通して帰路が同じだったため、
　一緒に大阪梅田や西宮のゲーセン通いしていた
　悪友だったはずが、

　いまでは某省のキャリア官僚として
　日本と海外を行ったり来たり、立派に活躍しています。

　（えらく差がついてしまいました 汗）


　同級生一同と会場で会えるのも楽しみです。

　といっても、特に今年に入ってから何度も会ってる
　人間がほとんどなのですが、、





■それでは今日のピークパフォーマンス方程式（ビジネスにおいて、
　最高の結果を発揮するための方程式）をどうぞ。


　今日は、


 【　「流れ去る日常」VS「時間を積み上げる日常」  】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。

＜今日のピークパフォーマンス方程式＞

 【　「流れ去る日常」VS「時間を積み上げる日常」　】

───────────────────────────────────



■しばらく、こちらの本


　【今日のお薦め本 リッツ・カールトンで育まれたホスピタリティノート】

　　http://amazon.co.jp/o/ASIN/4761266872/2ndstagejp-22/ref=nosim


　を皆さんと一緒に読んできました。

　まだまだシェアしたい部分はたくさんあるのですが、
　最後にどうしてもご紹介したいのが、この項目。



■タイトルは


　【 流れ去る日常を、時間を積み上げる日常に変えていく 】


　です。


　------------------------------------------------------------


　自分の反省を踏まえてのことですが、

　人間というものは生まれつき怠け者なのではないか
　と思います。

　天性の努力家というのは、とても少ないのではないかと。



　ですから、ちょっと油断するとすぐに安易なほうに
　流されてしまいます。

　仕事でも、少し経験を積んでコツを覚えてしまえば、
　そこそこの努力でこなす方法を覚えてしまいます。

　しかし、そこに「怖さ」が潜んでいます。

　何となく作業をこなすような仕事を続けていくと、
　その日常は、やはり何となく流れ去っていきます。

　そこに進歩がない。

　それはつまり、周りが進歩していた場合、
　自分は後退していることを意味します。



　頭角を現す人は、今日の仕事を通じて、自分も周りも
　成長するにはどう働くべきかを、自分に問いかけます。

　働くことの意味を考え、他人のために何ができるかを
　考えながら１日を過ごす。

　その時間は流れ去ることなく、確実に積み重なり、
　力となっていきます。



　でも、そういう人生を送っている人は非常に少ない。

　だからこそ貴重なのです。

　そんなメンター（師匠）を必死で探して、その生き方に
　触れ、真似ることの意味はそこにあります。


　　　　　　　　　　　　　　　　　　　　　　（引用ここまで）
　------------------------------------------------------------



■時間というのは、ムダにするまいと努めていてさえ、
　アッというまに流れ去っていくもの。

　意識していなければなおさらです。


　「光陰矢のごとし」

　なのに

　「時間だけが経過し、そこに何も残っていなかった」


　という事態はなんとしても避けたいのだけれど、
　往々にして起きてしまうものです。



■油断していると、
　１時間が経っていて、

　気が付くと午後になっていて、

　今日も１日が終わっている。


　そんな調子で、
　１週間、１ヵ月も流れ去り、

　しまいに

　「あれ？　このまえ書いたと思っていたら、もう年賀状の季節か」

　となっています。

　（そういえばもう秋ですね）



■二度と戻らない一日一日を、
　いかに密度を濃くし、充実させていくか。

　そして、
　いかに自分の未来の可能性のために積み上げていくべきか。

　上記の高野さんの言葉は、
　そのための教訓であると思います。



■私が注目したいのは、

　高野さんが、なぜ時間は無為に流れやすいのか
　自らの反省を踏まえて分析を加えられているところです。

　すなわち、


　「人間というものは生まれつき怠け者なのではないか
　　と思います。

　　天性の努力家というのは、とても少ないのではないかと。

　　ですから、ちょっと油断するとすぐに安易なほうに
　　流されてしまいます」


　この自省があってこそ、

　安逸を貪り、惰性の日々を過ごしやすい自分の心を注視し、
　律していけるようになるのでしょう。


　明日、もう少し続きを書いて、この稿を終えたいと
　思います。

【今日のつぶやき】


＜若手ビジネスパーソンの素直さ、純粋さ、夢、志、想い＞


数ある高い意欲と能力を持つ若手ビジネスパーソンの中でも、
本当に親しく、かつ、たまたま声をかけるタイミングが合った人に
向けて、少人数・内輪向けの勉強会をほぼ毎日（！）開催しています。

収益度外視（というか完全持ち出し）でやってるのですが、
これが本当に楽しい。完全に趣味、道楽の領域です。


ただ、完全持ち出し、趣味、道楽の世界とはいえ、
継続して開催していると私自身も（結果として）凄く大きな学びが
得られているのです。


なにより、一番の学びであり、刺激となっているのは
若手ビジネスパーソンの

「素直で純粋で大きな夢・志・想い」

を直接肌で感じられること。


小知恵がついて、相応にうまく世の中を立ち回れるように
なってくると、徐々に保守的になったり、現状維持、守りの態勢に
入ったり、

「小さくまとまる」「落ち着く」「安定する」

という方向に傾きがちですが、

彼らと接していると、そういう心の安住の世界が
強制的に破壊されるようなインパクトがあるのです。


「素直さ、純粋さ、夢、志、想い」

これらはスキルやノウハウ、能力よりもはるかに大切なもの。


他の何ものを持たなかったとしても、これを備えている人間は
強い。従って、彼ら彼女らは皆、きっといずれ、何事かを為す人たち
となることでしょう。


そんな人たちと接することで、新鮮な心を取り戻させてもらっている、
という意味で、私が一番、得をしている、感謝しなければならないと
思っています。


みんなありがとう！！




■では、またあした！
