　【　いつも笑顔でいられるように。　】

───────────────────────────────────


■いつも笑顔であり続けることができればいいのはわかっています。

　にこにこ、明るい笑顔が絶えない人のところに人も集まるし、友達も
　できる。結果として幸せもやってくる。


■でも、「常に」というのはなかなか出来がたいことであったりします。

　人と会ったり、電話でお話をしたり、メールを送るときでさえ、自分の心
　の有り様がそのまま外に出てしまうことがあります。


■自分が厳しいときにこそ、笑顔を絶やさずに、にこにこしていられる
　ことができるかどうか。

　
■生きていく中で、常に楽しいとき、幸せなときばかりではありません。


■ときに、人には言えぬ苦悩を抱えながら、あるいは、様々な問題が一度
　にやってきて、心に余裕が持てないこと、など、自分の容量を超えるほど
　の状況がくることも当然のことながらあったりもします。

　ついつい、その心情、心境を外側にぶつけてしまいたくなることも
　しばしばです。


■そんなときに、なかなか笑顔であり続けること、相手を優先すること、
　は、できがたいことです。

　でも、だからこそ、それを行うことができると、周りから信頼され、
　信用されるものなのでしょう。


■苦虫かみつぶしたようなのはダメ。不快オーラを放つのもダメ。
　それでうまくいってる人って、あなたの周りにもいないですよね。

　少なくとも私の周りで、うまくいっている人はみんないつも笑顔で、明るく
　たくさんの心遣いをされていらっしゃるから人の輪が出来ています。


■いろんな問題が次々にやってくる。そんななかで笑顔を絶やさないことは、
　本当に難しい。

　でもそれをやりきる人こそ、プライベートでも仕事でもうまくいくのでしょう。


■そう考えると、私など、まだまだまだまだ（以下略）修練が足りないようです。
　ついつい感情が表に表れてしまったりします。
　
　お子さまなのでしょうかね。
　反省して、頑張ろう。
