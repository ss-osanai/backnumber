＜昨日の活動ダイジェスト＞

▼経営理念策定

───────────────────────────────────



■こんにちは、鮒谷です。
　　

　昨日は4.0時間、大学院受験の勉強。

　（３月累計：74.0時間）


　
■日中は、

　TOMAコンサルタンツグループの
　主催される
　
　「経営理念作成セミナー」

　に参加しておりました。

　
　講師は先日、ご縁をいただいた、
　藤間秋男先生。

　　
　
■これから、時間をかけて、
　ブラッシュアップしていきますが、


　------------------------------------------------

　骨太かつ卓越した＜本物の教育サービス＞を通して
　「変容の喜び」を提供し続ける

　------------------------------------------------


　という経営理念を策定いたしました。


　ここに決意表明することによって、

　この覚悟を引き受けて、
　日々、精進していく所存です。


　藤間先生、素晴らしいお話をいただき、
　ありがとうございました！

　


■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　「三本の矢」で自らを律する　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

　【　「三本の矢」で自らを律する　】

───────────────────────────────────



■今日は

　「三本の矢で自らを律する」

　というテーマでお届けします。


　いつもメルマガで書いている通り、
　人間の意思は思いの外に弱いものです。



■しかし、

　毛利元就の三本の矢の話ではありませんが、
　

　「意思の弱い人間でも３人集まれば

　　一本一本（一人ひとり）の気持ちは弱くとも、
　　弱い意思の力が糾合され、折れにくくなる」


　ということをお伝えしたいのです。



■もちろん、弓の数（人の数）が５本、10本と
　増えるにつれ、

　さらにその強度は増していくことでしょう。

　そしてそのように心がける必要が
　あるのです。



■したがって、

　「そういう（志を同じくする）人たちが
　　集まるような場を作る能力」

　が必要となってくるでしょう。


　自らが目指す方向と同じ方向を向いている
　前向きで、やる気のあるメンバーを

　「この指とまれ！！」

　と集めてくることができる、そんな力です。



■この能力を身につけることができれば、

　一人では到達することが難しかったことでも
　衆力を集め、衆知を借りつつ、

　自分一人では越えられなかった
　限界を越える、

　そんな体験ができるようになるのです。



■自らの力をたのむのではなく、
　衆の力をたのむとすると。

　そんな発想の転換をしてみると、
　違った世界が見えてくるかもしれません。


　明日に続けます。
　
　
　今日も人生とビジネスを楽しみましょう！
