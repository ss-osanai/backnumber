■実は、

　年末発売予定の新刊書籍に
　はさみ込む、


　ハガキ大の大きさの

　「メルマガ（平成進化論）紹介のご案内」


　という用紙に掲載するための、

　感想文、体験談をお送りいただきたい
　のです。



■今度の本は、

　「人生に変化を起こすために」

　というテーマで執筆しているので、

　
　このメルマガを読んでいることによって、

　「こんな変化が起こったよ」
　「こんないいことがありました」

　といったことがありましたら、

　その経験談や、率直なご感想を
　頂戴できればと思っているのです。


　（その文章を通して、新しく書籍をお求め
　　下さった方にメルマガをご紹介したいのです。

　　私の言葉だと説得力がないので 汗）



■といいつつ、
　まことに申し訳ないのですが、

　スペースが限られているため、
　頂いた方全ての文章を掲載できるかどうか、

　また、

　お送りくださった全文をご掲載できるか
　どうかは、まだわかりません。

　なんにもお礼もできません（汗）



■ただ、もし採用させていただいた場合、


　【 実名（もしくは匿名希望の方はイニシャル） 】


　で、


　一冊の本につき一枚、
　かならず挟み込まれますので、


　1000部売れたら、1000人の、
　10000部売れたら、10000人の、
　30000部売れたら、30000人の、
　
　さらに

　100000部うれたら、100000人の、
　
　（以下省略）


　人の目に触れるわけですから、

　　
　【 あなたのお名前と文章が全国区に登場 】


　することとなります。



■それに、

　よく考えたら、立ち読みされる方も
　あるわけですから、

　目に触れる実数はこの２倍、３倍に
　増えることにもなりそうですね。


　
■それを見た昔のクラスメートや
　昔の彼氏、彼女から
　　
　「ひょっとして、
　　これって、お前のこと？」

　なんて連絡があるかもしれません。（笑）


　（でも、こういった話は
　　本当によく聞きます）


　
■それはさておき、


　「メルマガ読んで、こんな風に変わったよ」
　
　「このようにメルマガを活用して、
　　こんな成果があがりました」


　といったコメントをいただけると、

　私もますます調子に乗って、
　もっといい文章をお届けしなければ！

　という気持ちにもなると思いますので、
　ぜひ、たくさんのご感想を頂き、

　調子に乗せていただければ
　とても嬉しいです。


　
■そして本をご購入くださった方々の中で、
　皆さんのご感想をお読み下さり、

　「俺も（私も）、メルマガ読んでみようかな」

　と、また新しいメルマガ読者さんとなって
　いただければ、
　　
　さらに私のやる気もかきたてられる、
　というものです。

　
　
■また、これをきっかけとして、
　どこかで

　「感想をお送りくださったあなた」

　と
　
　「未だ見ぬ、新しく読者となられる方」

　とのご縁が生まれるきっかけとなるかも
　しれません。

＜今日のピークパフォーマンス方程式＞

 【  覇気・活気・人気がリーダーシップの源　】

───────────────────────────────────



■先日、弁護士の高井伸夫先生から
　大変おもしろい話を聞かせていただきました。

　それは、
　
　
　【 リーダーシップは

　　　　「覇気」「活気」「人気」
　
　　　　　　　　この３つから生まれる　】
　
　
　という内容でした。
　
　
　
■まず「覇気」とは、

　個人の元気であり、活力であり、やる気であり、
　前に進む意欲であり、駆動する力であり、
　
　それらが一切のものに対する
　気の張りとなり、

　
　結果としてその気の張りが
　
　「覇気」という形でのオーラ、といったら
　言い過ぎかもしれませんが、

　ある種の雰囲気を漲（みなぎ）らせていく。
　
　
　
■次に「活気」。

　これは、前述の「覇気」が集団に
　伝播することによって
　
　生まれてくるのだと思います。
　
　
　それは、当該集団において、

　同じく元気や協同、あるいは協働の精神
　といったものをもたらし、
　
　それが集団における「活気」を
　呼び起こしてくれるもの。
　
　
　
■そして３番目に、

　そういった「覇気」「活気」が
　組み合わさっていくと、
　
　第三者からの評価として「人気」が
　生じてくる。
　
　
　そんなお話でありました。
　
　
　
■このようにして個人が集団に波及し、
　
　それが第三者からの評価されることで
　一つの大きな広がりがもたらされていく。
　
　
　そういう状況になった時に、
　
　「これはリーダーシップが発揮されている」
　
　と言えるのだ、
　
　そんなお話をされていました。
　
　
　
■振り返ってみて、

　自分自身に「覇気」があるか、
　自分の所属する共同体に「活気」があるか、
　
　さらにその自分やその共同体に対する
　「人気」があるのかどうか、
　
　
　これはそれぞれの立場でリーダーシップが
　発揮できているか否かの、

　一つのバロメーターになるのではないか、
　そんな風に思った次第です。



　今日も人生とビジネスを楽しみましょう！
