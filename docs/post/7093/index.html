7093号 「学び切る」という、強者だけが持つ習慣

■なんか、最近思うんですよね。


　ほとんどの人は、自分のことを

　「強者」

　もしくは、そこまでいかなくても

　「弱者ではない」

　と認識しているのでは。



■でなければ、もっと多くの人が

　【学び方を学ぼうとするはず】

　だと思うのですが、、、

　もちろん、名実ともに「強者」で
　あるならばそれでいいんですが。



■もしあなたが、

　私（鮒谷）と同じく、
　強者でないならば、

　もっと真剣になると思うのです。

　たとえば【学び切る】ということ、
　一つをとってみても。



■強者は、既に持っているんですよ。
　【学び切る】という習慣を。

　しかし、私（鮒谷）も含めた
　弱者は

　「学び切れない」

　から困っているのです。



■せっかく、あることを学んでも、

　自分のものにできなければ
　学んだことにはなりません。


　「つまみ食い学習」では
　能力が高まらない上に、

　時間が経っても変わらないから、
　自信喪失していきます。



■それは学習性無力感、
　つまり

　「どうせ、なにをやっても無駄」

　という意欲の喪失、行動量の減衰に
　まで波及していきます。



■反対に、

　学び切る力のある強者は
　学んだ分野の能力を着実に身につけ、

　それだけではなく、

　能力をつけたことによる自信を
　手に入れて行動量も逓増させていく。



■加えて、最も価値のある学びで
　あるところの

　【学び方を一通り、
　　経験したことによって獲得される
　　「学びのメタスキル」】

　まで手に入れるのですから、

　そこからより一層、
　学習速度も加速していくでしょう。



■こうして、

　強者はますます強者となり、
　弱者は弱者のまま捨て置かれる、

　という格差が生じることとなる。



■弱者の私（鮒谷）も、

　この繰り返しで悪循環にはまり込み、
　長い間、逃れられなかった、

　という経験があります。


　結果を出せなかったのは
　能力ではなく、ただ

　「学び方を知らなかっただけ」

　なのです（と、信じたい笑）。



■その環から抜け出せたのは、

　いったん、焦り、慌てる気持ちを抑え、
　時間軸を長く、重心を低くして、

　【学び切る】

　姿勢を身につけようと
　心がけたから。



■やがて、悪循環を断ち切り、

　最初はゆっくりと、
　やがて徐々に早く、

　自信と行動量の増加と
　獲得したメタスキルの深化、

　が行われるようになりました。



■この過程で身につけた

　【成果につながる学び方の極意】

　についてお伝えしているのが、

　先日来、ご案内してきた

　--------------

　【「身体に編み込む勉強法」音源】

　一生懸命、勉強しているのに、人生にも
　仕事にも大した変化が現れぬあなたへの、

　「お勉強」を身体に編み込み、生きた学びに
　変えるための３時間半の特効薬

　--------------

　となります。



■当音源は

　多くの人にとっての
　学びに対する概念を根本から覆し、

　「お勉強」から
　「身体に編み込むための学習」へと、

　パラダイムと行動習慣の転換を
　目的としてリリースしたものです。



■スケートリンクの上を滑るような
　（つるつるして定着しない）学びではなく、

　身体に編み込む
　（基本的な所作を確立する）学びを。


　これが同音源を通し、

　休憩なし、３時間半以上の時間を使って
　私（鮒谷）がお伝えしたかったことの、

　中心的なメッセージとなります。



■今回の音源は、音源お届け後、
　速やかにお聴きいただきたく、

　「３日以内にご感想を下さった方」

　には、さらに

　「3,000円引き」

　の音源クーポンを差し上げます。

　（つまり実質の音源価格は
　　6,800円（税込）となります）



■こんな時代だからこそ、
　この音源を聴き、

　【波及効果の高い、
　　一つ一つの意思決定の基準を、
　　確実に身体に編み込む学び】

　の価値と力を再発見し、

　地に足のついた実践を通して
　重心の低い姿勢を手に入れて下さい。


　今日も人生とビジネスを楽しみましょう！

　【今日のピークパフォーマンス方程式】

　　■弱者は能力がないから弱者なのではなく、
　　　ただ

　　　「学び方を知らない」

　　　からであることが、ほとんどではないか。

　　■学び方を学び、すなわち、学んだことを
　　　定着、実践させられるようになれば、

　　　自信も生まれ、行動量も増え、
　　　学び方というメタスキルを獲得し、

　　　人生はそこから良い循環が
　　　回りだすこととなるだろう。
