＜今日のピークパフォーマンス方程式＞

■狭義の能力を磨くことに加え、
　広義の能力も意識する

───────────────────────────────────

　
　
■昨年末、
　大掃除をしていた際に、

　起業当時の通帳が
　大量に出てきました。


　懐かしくなって
　眺めていたのですが、
　
　驚くべきことには
　起業２、３年で、


　平均の月商（≒利益）が
　2000万円に到達しておりました。

　（せっかく出てきたものなので、
　　どこかでその通帳写真を大公開？
　　するつもり）



■その数字を見て、
　改めて理解したことがあります。


　それは、

　ビジネスでうまくいくかどうかは

　（能力とまったく無関係、
　　とまではいうつもりはありませんが）


　能力があるからといって
　稼げるわけでもなく、

　能力がないからといって
　稼げないわけでもない、


　という事実。



■当時の私（鮒谷）は、

　私が今現在、身に付けている
　能力と比べると、

　間違いなく十分の一以下、
　ことによると百分の一以下だったはず。


　※なにせ、

　　サラリーマン時代の特筆すべき
　　実績は何もないし、

　　その程度の能力しかなかった
　　ということです。



■しかし今、当時と比べて、
　
　多少なりとも能力が伸びている
　（はず）だからといって、


　当時の十倍、百倍の利益を
　上げているかというと、

　とてもとても、そのレベルには
　至っておりません。



■これは一つの例ですが、

　能力を高めるのは
　いうまでもないこととして、


　それ以上にビジネスにおいて
　重要なのは、

　「時流、場所、ご縁」

　に恵まれることではないかと、
　改めて思うのです。



■「時流」とは流れに乗ること。


　どれだけ能力があっても
　時流に乗れなければ、

　本来の能力の十分の一しか
　成果を出せないこともありますし、

　時流に乗れば、数十倍、百倍の成果を
　叩き出せることもあるでしょう。


　起業当初の私（鮒谷）が、
　まさにそうでした。


　
■当時はメルマガの勃興期であり、

　放っておいても
　メールマガジンの読者さんが増え、

　皆さん熱心にお読み下さる、
　そんな状況。


　
■そのときは、これが俺の実力だ、
　と自惚れていたわけですが、

　今となってはとんでもない
　勘違いだったと分かります。


　時流に乗っているので、

　誰がやっても収益を
　あげられる市場であった、

　というだけの話ですね。



■「場所」とは、
　どこで戦うかですが、

　陣を敷く場所を間違えれば、
　本来の能力を発揮できません。


　強みに合致し、
　夢中になって取り組め、

　そこで成果を出せる場所（市場）、
　を見つけられれば、


　能力の十倍、百倍の成果を
　出せることもあるでしょう。



■私（鮒谷）の場合、

　20年近く前、メールマガジンという
　新しい媒体（場所）に参入しましたが、


　メルマガを書き、その場で
　ビジネスを展開することに、

　多少なりとも
　適性があったように思われ、


　他の場所では一向に輝けなかった
　私などでも、

　唯一、その場所においてのみ、
　成果を上げられたのでありましょう。

　
　
■これがもし、

　他のところであれば、どれだけ
　一生懸命に頑張ったところで、

　何の成果も出せなかったに
　違いありません。



■「ご縁」とは、

　これは！という縁に巡り合う
　僥倖に恵まれれば、

　それをきっかけに、
　
　能力の数十倍、百倍、
　引き出される、

　そんなことがあるということです。



■私（鮒谷）の場合は、

　かんき出版の
　境健一郎社長（当時）、

　あるいは、

　境さんからご紹介頂いた
　弁護士の高井伸夫先生との出会い、

　といったご縁が、
　まさにこれにあてはまります。



■ということは、


　能力には、いわゆる、
　専門性を磨く、といった

　「狭義の能力」

　という意味合いもありますが、


　専門性を持ち合わせていることは
　当然として、

　それらの能力に
　
　「時流、場所、ご縁」

　に巡り会える能力、


　すなわち、

　「広義の能力」

　を身に付けることが
　重要であると思われます。



■狭義の能力に
　レバレッジをかけるのが、

　広義の能力です。


　起業や経営、ビジネス全般に
　おいて、心がけたい考え方。
