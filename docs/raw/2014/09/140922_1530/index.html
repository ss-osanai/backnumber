 【  「あなたの潤いが、私の潤い」こそ、商売の極意  】

───────────────────────────────────

　

■私（鮒谷）が考える、

　「最もきれいなビジネスの形」
　
　は、

　「お客さまに勝ってもらうのを最優先することで
　　継続的に受注をいただける」

　という形。


　
■イメージしやすいよう、あえて、

　「勝ってもらう」
　
　という言葉を用いているわけですが、
　

　しっくりこないようであるならば、

　「喜んでいただく」
　
　「価値をお渡しする」

　「ありがというと言っていただく」
　
　などの言葉に言い換えてみてもよいでしょう。



■反対に、

　自身の商品・サービスを通して、お客さまが
　どんな価値を得るかにはほとんど興味もなく、

　「私は何を（あるいは、いくら）もらいましょう」
　
　にばかり焦点を合わせ続ける

　「搾取モデル」

　だと、商売は長続きいたしません。


　
■例えていうなら、

　「自転車操業的」

　というべきか、
　
　「狩猟型」

　というべきか、はたまた
　
　「焼き畑農業式」

　というべきなのか、


　そんな商売を繰り返していると、
　いずれ力尽き、倒れ、
　
　あるいは、
　
　マーケットを焼き尽くし、蹂躙し尽くして
　しまいます。



■何より、そんな商売をしていても楽しくないし、

　やればやるほど、

　「自分の存在自体が悪」

　という刷り込みを、
　自らの内に無意識に叩き込み、
　
　何よりも大切にすべき、
　自尊心をどんどん毀損させてしまうことになる。



■そうではなく、

　お客様より頂いた代価以上の価値を
　提供することにコミットし、

　私（鮒谷）であるならば、
　たとえばお客さまへ、

　「商売のアドバイス」

　をすることによって、
　勝ってもらう（＝儲けてもらう）ことに成功すれば、

　ますます大きな金額を、
　私に投資していただけることになるでしょう。



■すなわち、当社から見れば、
　
　「さらなる売り上げが計上される」

　ということになります。


　だからこそ私は、

　お客さまに勝っていただくために、必死で、全力で、
　持てる資源をすべて出そうと尽くすわけですし、

　それが、お客さまの有形・無形の利益につながれば、

　（ときに一定の時間差があることもありますが）

　巡り巡って、
　再び返ってくることになるのです。



■お客さまの手元にある
　大切な資源（お金）を搾取するのではなく、

　一旦はお預かりするけれども、
　その後、10倍返しで増やす術をお伝えし、

　
　新しく増えた資源の一部を再び

　「下さいなｗ」

　と言えるようにと心がけているのです。



■それが実現できている限りにおいて、

　お客さまも、違和感なく、納得感を持って、
　そして、もっと大きなものを得るために、

　「改めて差し出して下さる」

　のだと思うし、


　その期待に応え続けられる限りにおいて、
　私にも、

　「潤い（笑）」

　がもたらされることとなるのです。



■現に、

　お客さまの皆が皆、とまでは申しませんが
　かなりの割合の方々が、

　契約更新（？）時において、
　長期契約、年間契約の形でお申込み下さり、

　だからこそ、

　私も、より価値をお出ししなければ、
　と、身が引き締まる思いなのであります。



■まとめると、


　「あなたの潤いが、私の潤い」

　「先にあなたに、次に私に」


　これが商売の極意です。


　これがキレイに回り続ける限りにおいて、

　売り上げも利益も、右肩上がりの逓増を
　遂げていくことになるでしょう。



■この反対の、

　「くれくれ言うけど、与えない
　　（もしくは、そもそも与えられない）」
　
　商売人に、未来はありません。


　というよりも、

　早々に市場から立ち去るべき。



■なんて、

　連日のように、公の場でこんなことを
　言ってしまっているわけだから、


　私（鮒谷）自身も淘汰の波に飲み込まれ、
　マーケットから退場宣告をされないよう、

　毎日、毎日、強烈なプレッシャーの中、
　必死で生きているわけです。



■めちゃくちゃ厳しい世界だけれども、

　やり甲斐があり、楽しく、ときに嬉しく、
　さらには儲かるお仕事です。

　マーケットに居残り続けられれば、
　という前提ですが（おー、こわ）
