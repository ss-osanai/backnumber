　　　　【刺激を与える人に触れる、感じる。】

───────────────────────────────────


■プレジデント、という雑誌があります。
　実は私、結構『プレジデント』誌、好きなんです。

　毎回ではありませんが、気に入った記事が載っていると結構買っちゃい
　ます。

　今回、なにげに本屋さんにいったら、営業特集をやってたので、購入
　いたしました。営業特集ということですが、今回のは面白いですよ。
　
　
■私の好きなセブンイレブン、イトーヨーカドーの会長の鈴木敏文氏
　の取材記事も載ってましたし、各業界のトップセールス１２名の
　取材記事も出ていました。

　この雑誌はセールス系の人の取材記事を頻繁に行っているんですが
　実はこの取材記事が好きなんです。


■彼らの具体的な事例、というか方法論ははっきりいってどうでも
　いいのですが（というと言い過ぎかもしれませんが）それぞれの営業
　マンが徹底して自分のスタイルを貫いていて、自信を持ってそのスタイル
　を押し通し、結果を出しているところに惹かれます。

　そして、出ている人たちが案外同年代が多いということも刺激に
　なります。

　※具体的な事例がどうでもいい、というのは、それをマスターしようと
　　しても、それぞれが強みを最大限に伸ばした結果であり、一朝一夕に
　　身につけられるものではないから、ということです。

　　それよりも、そのノウハウの後ろにある、情熱や、根性や、行動力、
　　あるいはなぜ、それだけのことを彼らは行い続けることができるのか、
　　ということを学ぶほうが大事だと思う、ということの裏返しとして
　　申し上げていることです。為念。


■いずれの取材記事からも、身体、知性、感性をフル活用して結果を出す事
　に熱中している姿をうかがい知ることができます。

　これが感じられれば、もう550円の元はとったのです。

　目を通すのも、はっきりいって５分か、せいぜい10分くらいで十分です。
　その体温を感じることができれば。


■営業のノウハウは二の次三の次。それよりも、もっと熱い情熱のような
　ものに継続的に触れることで、私自身も負けていられないな、と心の中に
　炎が燃えてくるのです。

　人間だから感情の起伏は勿論ありますが、その感情の下り坂に差し
　掛かろうとするときに、こういう熱い情熱を持った人たちに触れること
　でうかうかしておれん！！とより一層頑張ることができるのです。

　ですから、こういう人たちに日々コツコツと触れ続けることが重要
　なのです。たまに触れるのではなく、モチベーションが下り坂に入る
　前から常に接し続け、モチベーションが下がる余地を心の中に作らない
　ことが重要なのです。


■勿論、刺激を与えてくれる人と実際に会って、話を聞く、ランチや
　ディナーを一緒にとる、ことができればいいですが、それが叶わぬ場合は、
　テープを聞いたり、本や雑誌を読むことで、心の中に熱い火を燃やす手助けを
　してくれるのです。


■刺激を与える人に触れる、感じる、ことが大事ですね。
