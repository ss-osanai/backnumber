　親孝行が「まぐれ」で
　習慣化される人もいれば、

　入れようとは思っていても
　油断するとすぐに

　「そういえば最近、
　　連絡を入れてないな」

　ということになる。


　-----------------------


　上京当初はできなかったが、

　ビジネスが落ち着いてきた15年ほど
　前からは毎月、両親と食事をしよう、

　すなわち毎月、大阪に帰ろうと。


　そのために行ったことは…

　既に「うっすら習慣」だったのが、
　そうやって習慣化、仕組み化されていった。


　年に1回は親と温泉旅行などに行く、

　２週間ほど、クルーズツアーに
　一緒に行ったりも。


　あるいは長年、母親に
　毎月●万円振り込んできた。

　今はいい加減自分で使え、と
　言われたので振り込みは止めている。


　-----------------------


　うっすらと親孝行習慣ができてきたが、

　この習慣をより強固にするために、
　どんな仕組みを作ったか。


　今までは親の誕生日その他に
　電話をしていたが、抜け漏れがある。

　それ以外にも４日に一度程度は
　電話を入れている。

　
　親孝行の思いはあっても
　油断するとすぐに忘れてしまうので、

　どういうシステムを運用しているか
　というと…


　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------


　親孝行は親のためのみならず、
　自分のためでもある。

　自己重要感、
　喜んでもらってうれしいな。

　恩返し。


　親が先か、自分が先か、
　もしどちらが先に逝くにしたところで、
　後悔したくない。

　そういう意味でも親孝行を行っている。
　
　習慣に頼っていると危ないから、
　親孝行については仕組み（システム）も
　組み込んでいる。


　-----------------------


　習慣としても強固なものとなっているが
　習慣とシステムが一体化されている。

　コロナ後は会食はできず、
　Zoomもしくは電話になっている。


　電話もタクシーに乗っている、
　あるいは歩きながら、
　筋トレしながらといった形で、

　それ単体で時間を使うわけではない。

　
　-----------------------


　即断即決も習慣といえば習慣であり、

　確かに内的な精神の筋トレ、
　という側面もある。

　先送りしたい、気乗りしないという
　感情を躾けて、決して先送りしない。


　-----------------------


　最初は言葉で躾けた。

　着手の早さが仕事の速さ。
　着手の早さが9割のファクター。


　言葉によって躾けて即断即決、

　私の場合は「高井伸夫」と唱えると
　即断即決モードに入る。

　先送りすると雪だるま式に辛くなる。


　-----------------------


　「うっすら習慣」ができてきたら、

　その習慣に、外部のシステムを
　接続する必要がある。

　即断即決で全てをやろうとしても
　現実問題、できないことがある。


　毎日メルマガを書く以外にも　
　個別コンサル、年間プログラム、
　筋トレ、親に電話などいろいろある。

　即断即決しないといけないが　
　現実問題、溢れることがある。

　これは無理だと思ったら…


　（後略
　　→詳しくは音源をお聴きください）　


　-----------------------


　こうして「うっすら即断即決習慣」に
　仕組みをぶち込んだ。

　これをやることによって
　即断即決習慣が強固なものとなる。

　これをやらないと
　忘却の彼方に流されてしまう。


　-----------------------


　正直言うと、ここまでやっても
　流れていくものがある。

　しょうがない、現実問題、
　やることを詰め込んでも、

　たとえばセミナー開催するにあたっては
　インプットする時間が必要。

　読書は継続的に行うし、
　先週もZoomのセミナーに参加していた。


　やりきれない、
　ということがあるにせよ、

　うっすら習慣から強固な習慣に変えて
　できない場合は、

　これはもう、
　しようがないよな、と。


　-----------------------


　こうやって習慣システム、強固な習慣化を
　行うからやっていけるのであって、

　これによって、少なくとも
　サラリーマン時代の仕事量の10倍くらい、

　はやっているかと思っている。


　-----------------------


　たとえば音源販売。

　能力があるから、あるいは儲かるから
　やるんだろうと思われるかもしれないが、

　やろうと思ったって出来ない、
　そう簡単には。


　儲かると思うのなら、
　やってみてください。

　できるのと、
　やっているのとは違う。


　-----------------------


　確かに音源制作＆販売できる能力は
　持っているかもしれないが、

　簡単に実行できないからこそ
　音源販売が半年空くことがあった。


　それでも「うっすら習慣」があったから
　3カ月なり半年に1回音源をリリースできた。

　でもそれ以上でも以下でもない。


　-----------------------


　音源リリースに
　加速度をつけることができたのは
　
　「確固たる習慣化＆仕組み化」

　を完成させたから。


　では具体的にどのように
　習慣化＆仕組み化を進めたかというと…

　（後略
　　→詳しくは音源をお聴きください）

　皆さん気付かれているか
　分からないが、
　
　敢えて言うと、な完璧フォーマット、


　フォーマットに見えないかもしれないが
　完全なフォーマットがあるから、

　回らないと仕組みを作り、

　仕組みがフォーマットとなり、
　習慣が強化された。
