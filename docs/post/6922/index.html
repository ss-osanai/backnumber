6922号 「解除呪文」であなたにかけられている呪いを解き放て

■先日来、ご案内をしてきた

　【目標設定音源
　（令和アップグレード版）】

　をお聴き下さった村田さんより
　以下のご感想を頂戴いたしました。



■ご感想を踏まえて本文を記しますので
　まずはお読みください。


　（ここから）
　-----------------


　ここ最近、個別コンサルティングを
　受けさせていただく中で、
　再度目標を見直す機会が来ておりまして、

　まさにちょうど良いタイミングでの
　目標設定音源のアップデート版に、
　ありがたさを感じています。


　以前の目標設定音源はもちろん
　聞かせていただいているのですが、

　どうしても、知らず知らずのうちに

　「かくあるべし！」

　といった目標になっており、


　まさに音源で語られていたように、

　知らず知らずに目標を見るのが
　苦痛になっているということに、

　大きくうなずいてしまいました。


　今回の音源の中で、
　感情が伴っていないものは辛いもので、

　これを

　「不純物」

　という表現で語られていたことが
　妙に納得でした。


　そしてこの感情が伴っていることを、
　子ども頃の喩えで表現していただいたことで
　すごく理解が深まりました。


　確かに、私が幼少期の頃に、
　漫画のキャラクターの絵を描くなど、

　無我夢中でなりふり構わず、
　ただ楽しいことをやっていたことが
　思い出されます。


　これがひと度社会に出て、
　社会の波に揉まれる中で、

　いつの間にか、予定調和的に
　鍵を削って常識というに鍵穴に合うが如く、

　自分をつぶして
　あらぬ方向に進んでいるという状況が、

　私の中で見て取れるところです。


　これは、何度も何度も
　軌道修正を繰り返さないことには

　私自身の悪しき修正のようなものが
　無意識に発動されており、

　少し油断するとあっという間に
　元に戻ってしまうように感じているところです。


　その中で、感情が動く言葉を
　目標にすることにより、

　またそういったことを
　日記などに書き留めることにより、

　日々畑を耕すが如く、目標を定着させる
　活動をすることを通じて、

　種から芽が出て目標達成という
　大輪の花が咲くことになり、

　そして、それに至るまでの道としての
　小さな花が咲いていく

　といったことなのでしょう。


　自分が経験した貴重なことであっても、

　これを記述しないことには
　消えてなくなってしまうことも、

　すごく納得いたします。


　このために日記が必要であり、
　日記で目標という作物を発芽させるべく

　耕し続けることが必要なんだな
　と気づかされました。


　また、ナビの例えも
　すごくピンとくるものがあり、

　目的地を設定したことにより、
　たとえ遠回りであっても、
　回り道をしても、

　プロセスはどうであれ
　目的地にたどり着けるんですよね。


　ただ、この目的地に
　強烈な杭を打たなければ、

　この目的自体を意識するのが
　難しくなるため、

　ここに感情が動く言葉を打ちつけるのが
　重要なのだろうと
　私の中では理解いたしました。


　長文になりましたが、私の中では

　【童心を取り戻す時間】

　を天引きすべきである
　ということが大変心に刺さり、

　まさにそのことが
　感情の動くことのように
　思っています。


　私にとっては、家族との時間が
　童心に返り、心底楽しく感情が動く時間
　となりそうです。


　今一度目標を見直し、
　感情が動く言葉で再度自らの心を動かし、

　目標達成に向けて
　邁進して参りたいと思います。


　この度も貴重な音源を
　本当にありがとうございました。

　　　　　　　　（村田佑樹 さま）

　（ここまで）



■村田さん、長文のご感想、
　ありがとうございました！


　音源では、

　「動けないのは
　　感情が動いていないから」

　とお伝えしています。



■感情が動く、
　この感覚を掴みにくいようであれば、

　子どもの頃を思い出してはどうか、
　そんなお話をしています。


　子どもは

　「やりたいと思ったことを
　　全身全霊、飽きるまで」

　やり続けられる巨大なパワーや
　エネルギーを持っています。



■大人になっても
　持ち続けられる人は少ないですが、

　加齢によってそれらが
　枯れるわけではないんですよね。


　外野の雑音（こうあるべきとか、
　こうすべきとか）に影響されて、

　子どもの頃の感覚を忘れる、あるいは
　封印されてしまうだけの話。



■長い間、封じ込められてきた、

　邪気のない（無邪気な）気持ちを解き放ち、
　童心に帰ることができさえすれば、

　どんな人でもとてつもないパワーを
　持っているわけですから、

　想像以上のことを実現できるのです。



■久しく

　「童心に帰る」

　ことを忘れていた人は、

　しばしリラックスして
　あの頃の気持ちを思い出し、

　どうしたらあの感覚を
　今に呼び起こせないか、

　頭や心や体と対話してみられては
　いかがでしょう。



■ちなみに、幼少期における
　エネルギーの全開放は、

　原動力はちょっと違いますが
　感覚としては

　「火事場のクソ力」

　に近いものがあるように思います。



■火事場経験はありませんが、
　いざ、そのような現場に遭遇したら

　「恥も外聞もなくなり、
　　持てる力を全集中、全開放」

　できるようになる感覚は
　分かるような気がします。



■自分が全力を尽くせないのは

　「自らにかけられている呪い、つまり
　　制限や呪縛（常識とも）が原因」

　であるので、

　それらを開放させることによって
　初めて自由な身になれるでしょう。



■あなたにかけられている呪いの
　解除呪文（音源）、

　それが

　【目標設定音源
　（令和アップグレード版）】

　だと解釈していただければ
　良いかと思います。



■、、、こうしたことも、

　口頭（放談）であれば
　何時間でも話せるんですが、

　どうしても
　文章だと限界があるんですよね。

■ある程度、

　伝わるように書くには膨大な時間が
　かかるし（なにより面倒くせえ）、


　よしんば頑張って
　書いてみたところで、

　超長文になった文章を
　読み通す人も少ないし。


　そこで

　【音源登場！】

　となるわけです。

　　一部切り出し」

【今日のピークパフォーマンス方程式】

　■日常の生活において全力を尽くせないのは
　　感情が乗ってこないから。

　■自ら呪いをかけることによって
　　パワーやエネルギーを開放することを
　　妨げているのである。

　■子どもには呪いがかかっていないから
　　童心ゆえのエネルギー全力開放が行われ、

　　それがあの圧倒的な行動力につながる
　　こととなる。

　■大人になっても呪いを解きさえすれば
　　いつでもどこでも無邪気さを取り戻し、

　　これは、という対象に向けて
　　全身全霊、尽くせるようになるだろう。
