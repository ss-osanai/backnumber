6974号 「真の元気」とは伏流水のようなもの

■ここ何日か、「元気」について
　お伝えしてきました。


　「元気」と聞くと、

　ロッキーとかトップガンの
　テーマソングを聴いて

　「テンション、上がってきた～」

　という、あの感じを想像される方も
　少なくないかもしれませんね。



■しかし、お伝えしてきた「元気」は
　そういった

　「一時的で持続しないもの」

　ではありません。



■かくいう私（鮒谷）も高校時代、

　学校の行き帰り、ウォークマンで
　トップガンの主題歌を聞き、

　やる気を高めようと
　試みていたことがあります。

　（実際、テンション爆上がり）



■あるいは、学生時代、
　さらに社会人になってからも、

　朝早くに起きるため、
　断続的にですが目覚めの音楽として、

　ロッキーの主題歌を起床音楽に
　設定していたこともあります。

　（たしかにシャキッと目覚められる）



■活力ある毎日を過ごすために
　そうしたことは意味がない、

　というつもりはありません。

　実際に一定の効果はありました。



■しかし、残念ながら
　続かないんですよね。

　音楽を聞いているときだけ
　ハイテンション。

　しかし音を止めた瞬間、
　元の木阿弥に。



■こういうのは

　【真の元気】

　とは言いません（言えません）。


　真の元気とは
　どんなときであっても

　【伏流水のごとくに】

　あるいは

　【通奏低音のごとくに】

　自身の深いところに流れていて、
　尽きることのない元気。



■持続的、かつ情熱的なエネルギーが
　こんこんと湧き上がってくる、

　そんな

　【気力と行動力の源】

　といってもよいでしょう。



■ここまでの話をまとめると

　【真の元気】

　を引き出すための

　【一時的なテンション】

　を否定するつもりはなく、

　むしろ、有効に機能させられれば
　充実感を与えてくれますが、


　あくまで

　【真の元気が主であり、
　　一時的なテンションは従である】

　ことを忘れてはならない、

　ということになりますね。



■往々にして
　両者を混同してしまう方があり、

　あるいは、

　少なくない方が「真の元気」の存在を
　知らず、体感したこともないから、

　一時的なテンションを
　元気そのものと勘違いしてしまう、

　といったことが
　生じてしまうのです。



■そんなわけで、

　身体を通して、双方の区別を
　きちんと峻別できるようになったときが

　【充実した人生を過ごすための
　　スタートを切れたとき】

　と言えるでしょう。



■両者の相違を頭で理解していても
　身体で理解できていなければ

　【まだ始まってもいない】

　のです。



■双方の区別をきちんとつけ、
　さらに体感していただくための

　【補助線】

　としての音源、
　それが昨日からご案内を開始した

　【オムニバス音源（３本セット）】

　となります。



■以上のような問い、すなわち

　【私は「始まっている」のか、否か】

　という質問が

　【聴いているだけで自動的に生まれ、
　　自問自答が発動され続ける】

　音源。



■さらには

　【両者の違いが峻別され、
　　「真の元気」がインストールされる】

　　根源は「真の元気」のみである。
