　　【　「営業する」⇒「求められる人に」と言い換えてみると。　】

───────────────────────────────────


■「営業」という言葉を聞いて、あなたは直感的にどのようなイメージを
　持つでしょうか？


■「営業」の語源がどこにあるのかは分かりませんが（ちょっと調べて
　みましたが突き止められませんでした）、どうも「営業」というと、
　必要もない人に商品やサービスを押し込む、というイメージが
　私の中にはあったようにおもいます。


■「営業」という言葉は一般的なので、私も日常で普通に「営業する」
　という言葉は使いますが、ある時から、

　口では「営業する」という言葉を使いながら、心の中では
　「求められる人になる」と言い換えてみると、驚くほど仕事がうまく
　回り始めることを経験しました。


■不思議なもので、やはり私の中に、「営業というのは悪なるもの」
　の潜在イメージがあったのかもしれません。

　そうではなく、

　「本当に必要としている人に、必要とされているものを」

　、また

　「私自身が求められる人になる」のを「営業」というのだと、自分
　の中で定義を変えた瞬間に、お客様を追いかける側から、お客様に
　追いかけられる側に変わりました。


■営業というと、やはり～のクロージング術、とか応酬話法、
　とか心理学を駆使して～、とか、欧米流のタフ・ネゴシ
　エーターになるための～、とか、そういう勉強はそれこそ
　死ぬほどやってきて、結局いきついたのは、口先でモノを売るよりも
　本当に必要な人に本当に必要なものをアドバイスし、提供する。

　こんなスタンスでやったほうが、関係する全ての人がハッピーに
　になることに気付いたのです。


■自分のできる範囲内でのことではありますが、これを心がける
　ようになってから、ビジネスで困ることはなくなりました。


■なんといっても向こうからお客様が押し寄せてくる。そんな感覚
　ですから、申し訳ないけれどもお断りせざるをえなくなる。
　
　そんな状況が現出するようになったのです。


■本屋さんにいけば、それこそ無限にいろんな営業本が次から次へと
　現れては消えているわけですが、そういう勉強は、勿論とっても
　有意義なものではありますが、まずは、

　「営業＝求められる人になる」

　といった原理原則を理解した後のことではないかと、ふと思った
　次第でありまして。。。
