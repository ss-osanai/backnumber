＜今日のピークパフォーマンス方程式＞

 【　誰に教わるかで、人生は大きく変わる（先生を選ぶ２つの基準）　】

───────────────────────────────────



■どんなことにおいてもそうですが、
　何かを教わろうとするときには、
　　

　食べるために行っているのではなく、

　自らの楽しみとして、さらには自分の人生をかけて、
　実践している人から教わりたいもの。



■先生選びにおいて、絶対に必要なことだと思います。


　午前９時から５時、ないし６時まで、
　仕事としては行っているけれども、

　それ以降の時間帯においては、

　「仕事終わったんだから、極力、その分野には近づかない」

　という人からは、個人的には教わりたくありません。



■また、
　
　「お金をもらっているから、その仕事を行っている」

　という人からも、教えを請いたいとは思いません。


　なぜなら、その人が、

　お金をもらえなければ
　そのことに取り組むつもりがない、

　といっている時点で、
　
　少なくともその分野において、きっと
　プロフェッショナルではないと確信できるから。



■真のプロは、お金のために動いているのではなく、

　「当該分野における強烈な好奇心を持ち、
　　追求する精神を持つ」

　ならびに

　「自らの挟持を守る」

　ことを自身の存在意義としているはず。


　もし、その人が金銭的に恵まれているのであるならば、
　結果として、お金もついてきている、
　
　というだけの話。



■何かを学ぶときには、上に挙げたような、

　
　▼仕事時間中以外は、
　　そのことについて考えていなさそうな人

　▼お金のためだけに、その仕事を行っている


　という人を外し、その逆の、


　▼朝起きてから、夜寝るまで、
　　そのことについて考えているように見受けられる

　▼金銭的報酬抜きでも自分の専門を極めようとしている人
　　（結果としてお金もついてきている人も多いけれども）


　という人からのみ教えを受ける、

　はじめから、そんな風に決めておきさえすれば、

　
　「ムダな時間（とムダなお金）」

　を大幅に節約することができるでしょう。



■安物買いの銭失い、というのは、
　モノの購入にだけ当てはまるものではありません。

　何かを教わるにも、然るべき人から教わらなければ、
　安「人」買いの銭失い、

　ということになるでしょう。

　言い過ぎならゴメンナサイ。


　でも、私は時間とお金のムダな投資を絶対にしたくないので、
　そのような厳格なルールを設けています。


　できることなら

　「２４時間戦っている」

　そんな人から学びたい。



■そんなわけで、

　私はどんな分野においても、学ぶときには
　以上の基準で然るべき師匠を探し、

　反対に、私自身がなにかを人に伝えるのであれば、
　常に先の条件を満たしておくべき、

　と思っています。


　
■肉体はともかくとして、心の中では

　「２４時間、戦っているべき」

　だとも考えています。
　

　そのような存在になって、はじめて、
　人になにかを伝えるための、最低限の資格が与えられる、

　（それを満たしていないと失礼にあたる）

　そんなルールを持ち、自らを律しているつもり。
