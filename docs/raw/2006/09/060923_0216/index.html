　昨日も、終日、打ち合わせ、ならびに会食続きでした。

　

■昨日のメルマガでも書きましたが、現在、アンテナを張っている
　ジャンルの一つが金融・証券。
　
　そんなわけで、
　午前中はスパークス・アセット・マネジメント投信さまに
　伺っておりました。
　
　＜スパークス・アセット・マネジメント投信＞

　http://www.sparx.co.jp/index.html

　※スパークスさまは、国内独立系の資産運用グループ。2001年12月、
　　投信投資顧問会社で初めて、JASDAQ市場に上場されています。

　

■元アクセンチュアの10年来の友人（後輩、超優秀です）が、
　こちらで働いているため、

　情報交換ならびに金融のお勉強をさせていただくために
　訪問したのですが、関連会社のスパークス証券の方も合わせて
　４名の方からお話をうかがいました。

　（みなさま、ご多忙の中、ありがとうございました）

　その後、ランチ。



■お昼からは、

　株式会社プロフェッショナルバンクの上田宗央社長とのご面談
　の機会を同社のM部長様からのご紹介で頂戴いたしました。

　（M部長さま、ありがとうございました）

　上田社長は、人材派遣のパソナの代表取締役社長を経て、
　同社を設立、代表取締役に。

　

■パソナという大きな企業のトップとしてご活躍の後、現在、
　人材マーケットにおいて、新しい潮流を生み出そうと大きな
　展望を描いて動かれていることを感じました。

　今後の人材市場についての見通し、そして同社の方向性などを
　身振り手振りを交えられながら、熱く語ってくださいました。

　同社と一緒にお仕事させていただく機会もありそうです。

　お忙しいところ、どうもありがとうございました！

　2ヶ月に一度程度、開催されている「2000年会」に参加。
　
　2000年に人生の転機を迎えた人という切り口で、ある方を

＜今日のピークパフォーマンス方程式。＞

　【　成功者の資質は即断即決にあり　】

───────────────────────────────────



■意思決定の速さと実行力。

　これは成功者にとって欠かせない資質でしょう。

　数多くの成功者といわれる方々とお付き合いを
　させていただく中で、彼らの即断即決のスピードの速さには
　舌を巻いています。



■なかでも驚かずにおれないのが、仲良くさせていただいている
　株式会社ウィルＰＭインターナショナル代表取締役社長の
　石田淳さん。

　ベストセラー『リーダーのためのとっておきのスキル』

　【今日のお勧め本　リーダーのためのとっておきのスキル】

　⇒ http://amazon.co.jp/o/ASIN/4894512130/2ndstagejp-22/ref=nosim
　

　の著者で、東海地方を中心にわずか数年間ですべて直営の（！）
　学習塾を約100教室展開させた実績を持たれるマネジメントの
　天才です。

　いまは学習塾の展開で蓄積されたマネジメント理論に磨きを
　かけられ、その手法（行動科学マネジメント）を普及させるための、
　もう一つ会社を立ち上げ、東奔西走されています。



■アッという間に事業を急拡大させてこられた経営者であり、
　その決断力と行動の迅速さには驚かずにおれません。

　私はひとつのすばらしいお手本として、学ばせて頂いて
　いるわけですが、石田さんと接していていつも思うのが

　経営者として最悪なのは


　　【「そのうちやる」という選択を残しておくこと 】


　なんだ、と痛感させられます。

　※石田さん、信じがたいほど、めちゃくちゃ行動が早いんです。。



■経営をしていますと大小さまざまなチャンスがやってきますが、
　GO なのか NOT GO なのか、スピーディーに処理をしていく
　必要があります。

 「そのうちやります」「そのうち決めます」という選択肢を　
　残していると、結局はズルズルと決断が先延ばしになり、

　いつのまにか「やらない」という選択肢になっています。



■やるならやる。やらないならやらないと決定する。

 「そのうちに」という選択肢は排除しなければなりません。

　私もまだ完全にはできていないのですが、常にそうありたいと
　おもっています。



■全盛期には「今太閤」と持ち上げられ、ロッキード以降、
　「キングメーカー」「闇将軍」、
　棺を覆うても、なお評価が定まらず、いまなお
　毀誉褒貶(きよほうへん)のある田中角栄氏ではありますが、

　尋常小学校卒の経歴で、日本という国においてある意味
　トップに立った人。

　そんな田中角栄氏は、またの名を、
　「コンピューター付きブルドーザー」と呼ばれていました。

　その彼の口ぐせが、「決断と実行」でした。



■私もさまざまなビジネスチャンスをいただきますが、
 「そのうちやろう」という決断はしません。

 「やる」「やらない」で決める。

　やるとなったら決断即実行で、次の瞬間には行動に
　落としていこうと思っているのです。

　なかなか難しいことではありますが、
　心がけとしてはいつもそのようにしています。

　さきほども言いましたが、「あとで決める」という選択肢は
　排除して考えているのです。



■「平成・進化論。」でご掲載させていただいている、
　メルマガ広告などもそうですが、日ごろのご案内なども含め、

　小さなところから即断即決、決断即実行の習慣をつけて
　いられるのもいいのかもしれません。（笑）



■直近では、何度か告知させていただいたものでは
 「日経デザイン」がありますが、

　あなたご自身のビジネスをアートに高めていく、
　あるいはデザイン材料を活用することによって収益性の高い
　ビジネスを展開していける予感のようなものがあるのであれば、

　決断即実行で、こちらから申し込むという選択をとられるのも
　いいかもしれません。

　　これを機会に、デザインに関係のある人もない人も、
　　一度このような雑誌に触れてみられるのもよいのでは？
