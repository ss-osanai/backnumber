■新しいことを始める際は

　「馬鹿にされてナンボ」

　です。

　馬鹿にされるくらいまで動くから
　馬鹿にされるのです。



■行動量が足りなければ、

　馬鹿にされたくても
　馬鹿にされません。

　無視されるだけ。



■一生懸命に動き、
　大量の行動をしていく中で、

　誰からから馬鹿にされたら、

　「大量行動を褒めてもらった」

　と喜び、感謝すべきでしょう。



■同時に、そこで

　「反骨精神」

　をかきたてるのです。

　ふざけんなよ、とか
　今に見とれや、とか。

　なにもせず、外野でヤジを飛ばし、
　中傷してきた人に対して

　「ざまあ」

　と、心の中で言えばいいのです。



■実際のところ、その時が訪れても、

　素晴らしい世界がすでに目の前に
　広がっているから、

　そんなことはどうでもよくなるし、
　忘れるし、

　思い出したとしても、頑張るための
　燃料を投下してくれたという、

　感謝の素材にしかなりませんが。

　大量投稿３日目を迎えました。


　たった３日、かつ、
　数十件程度の投稿ではありますが、

　トライアル＆エラーを重ねると
　傾向も見えてくるし、

　対策（新たな施策）も
　打てるようになってきます。

　しかもTwitterはすぐに成果
　（特にインプレッション）が、

　目に見える形で反映されるので
　システムの作り甲斐がありますね。

　お分かりの通り、

　たった３日ですが、
　されど３日。

　たった数十投稿するだけでも、

　投稿の体裁が当初の姿と
　まるで異なる形になりました。


　その分だけ進化を遂げている、
　洗練されていっている、

　と言って良いでしょう。



■もちろん期待する成果と比べると
　まだまだ始まったばかりですが、

　それでも遅ればせながらの
　Twitter投稿という戦場での反転攻勢の、

　緒（いとぐち）は
　確実に掴みつつある、

　そんな予感や気配は
　すでに感じ取れています。

　頑健なシステムに進化させてきた、

　そんな自信があるので、

　こちら（Twitter）についても
　ほぼ同じことができるかなと。



■こういったことは一事が万事なので。

　対象は異なりますが、
　やることは全く同じなので。
