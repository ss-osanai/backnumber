 【　人生をスーパーイージーモードで生きたい人へ　】

───────────────────────────────────



■一昨日、昨日と、

　「人を紹介されるようになり、
　　さらには人を紹介できる人になる」

　ことの効用についてお伝えいたしました。


　今日も続けます。



■いったん、上記のサイクルが確立されると、
　それ以降は、

　「人が人を呼ぶ状態」

　が生まれることでしょう。

　
　プライベートにおいても、ビジネスにおいても、
　この状態に入ると、一般論として

　「スーパーイージーモード」

　になるように思われます。



■なにせ必要とする資源は、
　みな、周りの人が集めてくれるのです。

　情報も、知識も、ノウハウも、技術も、
　営業対象も、場合によってはお金までも。


　その世界に出るまでは本当に大変ですが、
　一度、その世界に出ることができれば、

　「慣性の法則」

　が働くこととなり、


　以降、一定の速度を下回らない程度に
　人を紹介したり、されたり、

　を繰り返しているだけで、巡航速度を保つことが
　できるようになるでしょう。


　（もちろん、さらに加速させることも可能です。
　　そこは個人の自由）



■この世界に出るためには、前回もお伝えしたように


　１、愛嬌（人に好かれる能力）

　２、先に価値を提供しようとする姿勢　

　３、（具体的に）提供できる能力や知識や情報


　の３つが必要です。



■これは自分で用意しなければなりません。

　ただし、１、２は心がけ次第で、

　早ければ一瞬、ないし数週間程度で、
　一定のレベルまで持っていくことも可能です。


　しかしながら、

　「３（つまり、価値提供のための能力や知識）」

　だけは、相応の時間がかかります。


　これは早くても数ヶ月、場合によっては何年もかけて
　身につける必要があるものです。



■そして残念なことに、

　この能力を身に付けるのを、誰かが手助けして
　くれるわけではありません。


　ただただ自分だけが、

　リスクを引き受け、自分の時間、ときにお金を
　投資することによってのみ、

　獲得できるもの。


　そして残念なことではありますが、

　１、２もさることながら、
　特に、３は効果が目に見えるのが遅い、
　
　すなわち遅効性という特質があるがゆえに、
　継続して取り組むことが困難になるのです。



■けれども、ここに

　「大きな参入障壁（？）」
　
　があると考えてみるとよいかもしれません。


　多くの人が、

　一つの分野をマスターしたり、技能を習得するのに
　なかなか腰を据えて取り組むことができない、

　そんな状況にあるからこそ、


　時間をかけて継続していくと、いずれ確実に、
　当該分野において、頭一つも二つも抜け出して、


　「○○の分野に詳しい人、と紹介される人」

　「○○の技能を身に付け、卓越性を持った人、
　　と紹介される人」


　となることができるのです。



■そうやってたくさんの人とのコネクションができ、

　そこで知り合った人たち同士をつなげ、
　価値を生むことによって、

　さらにレバレッジを効かせることができるように
　なるのです。

　
　昨日も書いたとおり、

　然るべき人と、然るべき人をおつなぎする、
　
　という形での人の紹介は、
　ほとんどコストがかからないので、

　ここから先は、極めて費用対効果の高い、

　周りの人に対して喜ばれる投資が行えるように
　なるでしょう。



■世に、

　「お金を持っている人に、ますますお金が入ってくる」

　といわれるように、


　「知り合いの多い人は、ますます知り合いが増える」

　ということも言えると思われます。

　そのカラクリは以上のような点にあります。



■金持ちになるには、はじめに

　「タネ銭」

　が必要なように、
　

　人に恵まれるためには、はじめに

　「タネ人」

　が必要です。

　※なんだか語感が悪いですが、他意はありません。
　　ご容赦ください。


　そしてこの「タネ人」は、
　上述の１～３を心がけた人だけが恵まれるもの。



■ひとたび、

　「タネ人」

　とのつながりが生まれたら、

　あとはタネ人が喜ばれるであろう人を、
　次々とご紹介する。


　こうするとタネ人も、タネ人に紹介される側も、
　紹介者も、みなハッピーとなるでしょう。

　こうやって

　「価値逓増の高速増殖炉」
　
　を、能力一杯まで発揮させ、稼働させるのです。


　（お金儲けも一緒ですね。

　　まずは日々の生活で節約、節制し、
　　タネ銭を作るのに全力を尽くし、

　　その後、タネ銭が喜ぶ（有効活用される）投資先を
　　見つけてはぐるぐる回していくことで、

　　タネ銭も、投資先も、そして投資家にとっても、
　　価値が増殖し、お金が増殖していくこととなります）



■ということで、

　本気で人生を大転換させたい、と思うのであれば、

　「（上述の）１～３」

　によって、タネ人との出会いを求める。


　さらに、そこから更にもう一段階、歩を進め、

　「人に紹介する人となる」

　という過程に入ることによって、


　つながっている人、全員に価値を渡しつつ、
　結果として、自分も恵まれる、

　そんな流れが加速されることとなります。
　

　これが

　「人生スーパーイージーモード」

　に入る秘訣です。
