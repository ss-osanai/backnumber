【 「マスターベーション職人」と「本物の職人」の簡単な見分け方 】

───────────────────────────────────



■昨日は、

　「売り上げは、心と頭で立てるもの」

　というタイトルで、

　
　売り上げを立てるには、

　「まず心を整え、次に頭を使う必要がある」

　
　すなわち、まず最初に

　「お客さまが求めている価値を渡す」

　という強い意志を持ち、


　その上で、

　意志を実現するための頭をつかう必要がある、

　という話をいたしました。



■さらに、


　「私はお客さまのためを思い、
　　一つのことに打ち込んでいる職人である」

　と半分、誇りを持ちつつ、　
　しかしながら、ビジネスとしては成り立っていない、


　という人を

　「自己満足の、マスターベーション職人」

　と記しました。


　
■なぜなら、


　「私はお客さまのためを思い、
　　一つのことに打ち込んでいる職人である」

　という姿は、
　
　一見、素晴らしい、
　敬意を表すべき姿勢のようにも見えますが、
　

　厳しい言い方をすると、
　ビジネスとして成り立っていない、ということは、
　
　現実問題として、

　世の中に対して、大した影響力が行使されておらず、
　（ほとんど）社会の役に立っていない、

　とも言えるかもしれないから。



■口ではお客さまのため、といいながら、
　実際のところ、

　「分かる奴だけ分かればいい」

　という姿勢をとっている人があったとするならば、
　それは説明責任の放棄です。　
　

　お客さまに対して、するべき説明をせず、
　それがゆえにお客さまに価値を渡せていないのならば、

　本当にお客さまのことを思っている、
　なんていうのは

　「きれいごと」

　もっとハッキリ言えば、
　
　「嘘」

　になるわけです。



■さらに言うと、

　「本物の職人」

　になりたいのであれば、


　「売る → 顧客の声を聞く → 改善する

 　 → 売る → 顧客の声を聞く→ 改善する

 　 → 売る → 顧客の声を聞く→ 改善する

 　 → 売る → 顧客の声を聞く→ 改善する

 　 → 売る → 顧客の声を聞く→ 改善する

 　 → 売る → 顧客の声を聞く→ 改善する


　　　　　　（、、、以下、エンドレスに続く）」 


　を回す必要が絶対にあるのです。



■本気で

　「お客さまの役に立つ職人」

　になりたいのであれば、
　
　圧倒的なお客さまの生の声を聞き続ける
　必要があるのです。

　
　商品、サービスの説明責任を果たさず、
　売る努力もせず、

　（当たり前のように）モノも売れず、


　という姿勢のどこに、

　「お客さまの役に立とう」

　という気持ちがあるのでしょうか。



■本当にお客さまの役に立ちたいのであれば、

　「売って、売って、売りまくり」


　同時に

　「毎回、毎回、お客さまの反応を観察し、
　　お客さまの声を聞く」

　ことに絶対的な執着が生まれるのではない
　でしょうか。


　その姿勢がないということは
　
　「職人と名乗るのも、おこがましい」

　とさえ言えると思っています。



■ビジネスがただのカネ儲けなら

　「売って、売って、売りまくり」

　で、留まってもいい（？）わけですが、


　本物の職人でありたいと思ったら、
　圧倒的に高速度での改善を図りたいと考えるはずだし、　

　それにはお客さまの観察が必要だし、
　それにはお客さまへの感想募集も重要です。



■だから私（鮒谷）は、


　コンサルを行なうにせよ、
　セミナーを行うにせよ、

　絶対にご感想を頂戴するようにしていますし、

　有り難い感想については、
　さらにより良いサービス開発の参考にするし、

　耳に痛い、口に苦い感想についても、
　同様に、改善のヒントとさせて頂いています。



■ここまで書いたことを私（鮒谷）自身も
　実践している、嘘偽りない証として、

　日々、頂戴するお客様の声を毎日、
　メルマガの冒頭に掲げています。


　それによって自らに圧倒的なプレッシャーも
　かけ続けるわけだから、

　当然のことながら職人としての技量も上がるに
　決まっています。


　というよりも、

　職人としての技量をあげたいのであれば、
　それくらいのことは、

　「呼吸をする」レベルで行ない続ける必要があると
　考えているのです。



■こんな考えを持って仕事に取り組んできましたので、
　
　「ウチの商品が売れないのは客が悪い」

　「分かる奴だけ分かればいい」

　という姿勢で（自称）職人を貫こうとしている人が
　もしあったとするならば。


　私は世界の中心で、

　その人は

　「本物の職人ではなく、
　　マスターベーション職人である」

　と声を大にして、叫びたいと思います。
