＜昨日の活動ダイジェスト 伊豆より東京に帰還。新規事業リサーチ＞
───────────────────────────────────


■こんにちは、鮒谷です。

　昨日は、一昨日よりの三島⇒熱海⇒伊豆、のスケジュールを経て、
　昼過ぎ、東京に戻ってまいりました。

　帰りの新幹線の中でいくつかの事業プランがひらめいたので、
　ネットを使ってのリサーチと、友人、知人、関連事業者あてに、
　電話して情報収集を行っておりました。

　（結局、モノにならなさそうだったのでボツにしました）



■それでは今日のピークパフォーマンス方程式（ビジネスにおいて、
　最高の結果を発揮するための方程式）をどうぞ。


　今日は、

　【　１日24時間の枠を越えるには　】

　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式。＞

　【　１日24時間の枠を越えるには　】

───────────────────────────────────



■「平成・進化論。」読者さまの中には、

　「１日が48時間あれば」
　「なんで１日は24時間しかないんだ」

　なんて思われている、ご多忙な方もいらっしゃるでしょう。
　私もいつもそう思っています。

　しかし、１日は24時間。これは覆しようのない現実です。



■でも、今まで１日ではできなかったことが
　１日でできるようになるとどうでしょうか。

　２日でやっていたことを１日で、
　１ヶ月かかっていたところを１週間で。

　そうすると、１日が48時間、１週間が30日、
　などと考えることはできないでしょうか。

　つまり、時間の密度を高めていくことで、
　時間の単位の枠を越えることができてしまうのです。



■では、どうやったら時間密度を上げることができるのか。
　これはわたし（鮒谷）も日々意識していることですが、

　「アンテナの数を増やす」

　ことが非常に有効なのではないかとおもいます。



■１日の時間密度は、立てた「アンテナの数」に
　比例するのではと考えています。

　自分の興味のあるジャンルがいくつあるかによって、
　キャッチできる情報量も変わってくるのです。

　１つしかアンテナが立っていない、つまり、興味のある分野が
　１つしかないとなると、１日はゆったりとしたものになるでしょう。

　それが３つ、５つ、１０ともなると、見聞きするちょっとしたこと
　でも情報として引っかかるようになり、アドレナリンがどんどん
　放出され、脳ミソがフル回転をし始めるのです。



■最初は24時間がゆったりだったのが、
　アンテナの数を増やすことで情報量が増えていきます。

　それをもとに行動しようとすると、だんだんと時間不足という
　現象が起こり始めます。

　１日の時間には限りがありますので、行動が追いつかなくなり
　やがて「もう限界だ」となるでしょう。



■すると、

　「どうやったらさらなる時間を生み出せるか」
　「どのようにしたら効率よくこなせるか」

　という工夫をし始めます。

　そこで時間当たりの密度を濃くするためのエネルギーが
　生み出されてくるようになるのです。



■なぜこんなことを考えたのかと言いますと、わたし自身が
　以前にも増して、興味関心のあるジャンルが増えてきたから。

　数年前までほとんど関心のなかった分野も数多く含まれます。

　具体的に言いますと、

　税制、投資、M&A、海外展開、これらについての関連法規、
　健康管理、採用・マネジメント、ケータイビジネス、
　リスク管理、

　などなどなど。



■ただでさえ時間がないないと言っていたのに、
　ますます足りないという状況に自らを追い込んでしまいました。

　しかし、これらの状況に陥ったときに現れた変化が、
　先程申したような、権限委譲、委任、委託、ライフハックなど、
　より時間効率を高めるような行動を取れるようになったこと。

　そして、１日が24時間という感覚は消え、36時間、48時間に
　増えているような感じがしてくるのです。



■本日お伝えしたかったのは、自らの実体験を通してわかったこと
　なのですが、

　たくさんのアンテナを立てることにより、密度の濃い時間を
　生み出すことができ、これがビジネスを行ううえでの大きな武器と
　なるのではないかということ。

　時間が足りないとお嘆きのあなたにとって少しでもご参考になれば
　幸いです。

【編集後記】


■ひとつのことに没頭すると、やがてその向こう側にある
　興味あるジャンルがどんどん現れてきます。

　
■たとえると、山を越えるとまた新しい山脈が現れてくるようなもの。

　ひとつの山も越えなければ、その向こうにある山に興味の持ちよう
　もありません。

■どうしたら興味ある分野を増やせるんですか？

　と聞かれることがあるのですが、ひとつのことを徹底的に行えば
　やがて関連するあらゆる物事にもその興味が派生するのでは
　ないかと思います。
