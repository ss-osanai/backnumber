　サッカーの比喩で自分が今どの位置にいるのか、

　そして今いる場所は客観的に見ると
　どこに位置しているのかが理解でき、

　ユースでもきわめて有望な方、

　出てくる数字に平然を装うのがやっとでした。汗


　・強みについて

　他人に自分の強みを聞くだけでは不十分で、

　自分で内省し、咀嚼していかなければ、
　絶対に用途開発などされないこと。

　最近違和感を感じていた点でしたので、
　改めて自分の思考の浅さを認識することができました。


　・価値について

　これも上記の強みに繋がる話として、
　潜在・顕在ニーズをどのくらい深掘りし、

　どのくらい考えているのか、強く問いただされました。

　四六時中、価値について向き合う事にします。


　・顧客リスト

　開いた口がふさがらないとは
　この時のためにある言葉だと実感しました。

　実物を見る事が何よりも驚きでした。

＜今日のピークパフォーマンス方程式＞

【 然るべき言葉の徹底コピーは、
　　　　　　　　　　　　 言語体系（成果を出す脳）のコピーにつながる 】

───────────────────────────────────



■最速で、

　「結果を出せる男（女）」

　になるための方法があります。


　それは、

　既に結果を出している男（女）の使っている
　言葉を

　「完コピ（完全コピー）」

　すること。


　
■既に結果を出している人が
　よく使っている言葉のみを選択し、

　さらには徹底的に自らに刷り込む。


　すると、その人の思考に近い、

　「結果を出せる脳」

　が形成されるようになるのです。



■言葉というのは、


　意識的、あるいは無意識に脳で考えていることが
　表出されたものだから、


　成果を出している人の言葉を自らに刷り込み、
　かつ、積極的に使用するように意識してゆくと、

　結果として、

　「その言葉を紡ぎだしている脳の回路
　　（＝成果を出す脳）」

　をコピーすることにつながってゆくのです。



■こうして形成された脳は、
　
　やがてあらゆる場面で、必要に応じて、
　適切な言葉を返してくれることとなるでしょう。


　この言語体系が自らの行動を規定することとなり、
　自ずから、成果も上がるようになるのです。



■たとえば、


　私（鮒谷）の場合、営業においては、
　下記に記してきたような、

　営業系の言葉を（先人から）

　http://goo.gl/Kg5V8n

　徹底的にインストールしてきたわけですが、


　これらの言葉が自らの営業においける行動規範となり、
　営業活動を支配することとなり、

　短期間に億単位の売上（利益）を
　創出することにつながりました。


　この売上（利益）は、ひとえに

　「過去にインストールしてきた言葉」

　がもたらしてくれたものなのです。



■営業に限らず、その他の分野においても
　たくさん言葉を収集し、さらには編集してきましたが、

　それをタダでばらまきましょう、
　おすそ分けしましょう、

　という大盤振る舞い、太っ腹なことをしてきたのが、
　この平成進化論なのです。


　嘘のような本当の話。

　あなたがその価値に気づいているか
　どうかはともかくとして。


　今日は少々上から目線で（？）記してみました（笑）



　※そうした成果の出る言葉（言語セット）を、

　　完全にあなた専用にカスタマイズし、超短期間で、
　　インストールするお手伝いをいたしましょう、

　　というのが、

　　「（近々、値上げ予定のｗ）個別コンサルティング」

　　です。


　　分かる人には分かる価値でしょうし、

　　分からない人に、無理やり分かってもらおうとも
　　思っていない価値。

　　迎合するつもりも売り込むつもりも全くありません。


　　ただ、
　　　
　　「価値を価値と分かる人」

　　と話をするのは最高に楽しいので、
　　そうした人からのお申し込みは大歓迎しています。
