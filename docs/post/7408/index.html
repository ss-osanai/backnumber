7408号 思いの強さが行動に現れ、結果をもたらしてくれる

■唐突ですが先日、３日間、
　神戸に行きました。

　中・高時代はもちろん、
　社会人になってからも、

　ちょくちょく訪れている、
　懐かしい場所。



■今回の神戸訪問は、私（鮒谷）を

　「ふなちゃん扱い」

　してくれている

　（ずいぶん年次が下の）
　同じく高校の後輩で、

　年商100億企業の後継者、
　Ｙくんの関係でした。



■Ｙくんは現在、

　神戸大学のMBA（名門）に通っていて
　なんとそこで、以前、

　「エフェクチュエーション」

　のご講義をお願いしたことのある、

　神戸大学の吉田満梨先生の
　授業を受けていたらしい。



■そのつながりもあって
　神戸で講演会を企画したので来ないか？

　という案内をもらったので、


　それは行かねばならんと、

　神戸に在住の年間プログラム参加者の
　お二人にも声をかけ、参戦。



■改めてエフェクチュエーションの
　復習ができたのですが、

　（やっぱり復習は大事ですね）

　積極的に活用して、
　機会を広げていきたいと思いました。



■そんなこんなの神戸滞在でしたが、
　せっかく神戸に行くのだからと、

　これまた、以前（2012年4月）、
　※先のＦくんと一ヶ月違い！

　３日間セミナーに参加くださり、


　その後、個別コンサル等でも
　お付き合い頂いてきた、

　EMP税理士法人代表税理士
　あべき光司さん

　と久しぶりにお会いして、
　食事をご一緒しました。



■あべきさんは、
　現在、創業８年にして、

　士業経営者（オーナー士業）という形で、

　グループ全体で100名の従業員を
　雇用するまでに成長させた、

　超やり手の経営者さん。



■創業20年以上経って、

　いまだ零細企業のままの私（鮒谷）とは
　えらい違いです（泣）


　私とはまるで異なる
　（というか真逆の）スタイルの、

　大勢のスタッフを抱え、
　マネジメント力を発揮されることで、

　徹底したチーム化＆オーナー化を図り、
　大きな成果をあげていらっしゃいます



■そんなあべきさんが、

　先日、ここまでに至るプロセスについて
　解説された

　『オーナー士業になって
　　たちまち年商１億円を突破する方法』
　http://amazon.co.jp/o/ASIN/B0C9385BV8/2ndstagejp-22/ref=nosim

　という書籍を上梓されました。



■この本を読んで思い出したことが
　あるですよね。


　あべきさんとは、古くからの思い出が
　いろいろとあるのですが、

　「願えば叶う」

　より正確に言えば

　【思いの強さが行動に現れ、
　　結果につながっていく】

　ことを、まさに
　体現されている方なんですよね。


　そして、ここまで
　やりきる人がほとんどいないから、

　大きな成果を出している人もまた、
　少ないという、、、



■いろんな成功法則が出回っていても結局、
　これ（思いの強さ）に尽きるんですよね。


　当時は、たしか

　「未来日記（３年先の日記）」

　というものを
　ブログで書かれていたのですが、

　その頃から、未来に思いを馳せ、一歩一歩、
　着実に歩まれていたことを思い出します。



■学び、行動され、慌てず、焦らず、
　時間を味方につけられ、

　その延長線上に、
　今の結果が生まれてきたわけで、

　まさに思いの強さが
　現実を引き寄せられた、

　と言えるでしょう。



■お寿司を食べながら、
　あれこれお話を伺っていると、

　このところ、M&Aやら何やらで、
　さらに動きが加速されていて、

　成功が成功を呼ぶという、
　理想的な展開ともなっているようです。

　羨ましい。



■先の本で、
　あべきさんは士業・コンサルには

　「レベル０からレベル５までの６段階」

　があると書かれていたのですが、

　まさにご自身が卓越したマネジメントや
　コーチングの能力を活用されて、

　「レベル５（エヴァンジェリスト）」

　の領域に到達されていらっしゃいます。



■未来に夢を持ち、行動しつづけたら
　大きな夢も実現される、

　ということを、

　改めて、これまでの
　あべきさんの履歴、足跡を伺うことで、

　再確認させてもらう機会を頂きました。


　やっぱり目標を掲げ、
　イメージし続けることが大事ですね。

　成果を出すための一丁目一番地。



■それにしても、
　あんな話を聞いたら私（鮒谷）も

　『オーナー士業になって
　　たちまち年商１億円を突破する方法』
　http://amazon.co.jp/o/ASIN/B0C9385BV8/2ndstagejp-22/ref=nosim

　を読み込み、

　スタッフをじゃんじゃん採用し、
　M&Aを駆使しての

　「攻めに攻める、攻めダルマ経営」

　を一度はやってみたい、
　という気にもさせられました。



■無能の上にセンスもないので
　叶わぬ願いですが（泣）

　あべきさん、ありがとうございました！
　ますますの快進撃を願っております。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■結局のところ、思いの強さが、
　　目標実現の一丁目一番地。

　■思いの強さは必ず行動に現れるし、
　　持続もするし、

　　時間が味方になってくれ、
　　結果をもたらしてくれるのだ。
