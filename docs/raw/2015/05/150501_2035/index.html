＜昨日の活動ダイジェスト＞

■明日から２日間セミナー

───────────────────────────────────



■こんにちは、鮒谷です。

　
　今日は午前中から先ほどまで、
　連続して個別コンサル４件でした。

　皆さま、ありがとうございました！



■引き続き、明日、明後日は２日間セミナーです。

　ゴールデンウィークにもかかわらず
　わざわざ２日間まるまる潰してお越し下さる

　「22名の奇人・変人たち」

　にお会いできるのが楽しみです（笑）
　
　お申込み下さった皆さま、ご期待下さい！

　
　それにしても仕事し過ぎて、
　毎日、大興奮状態で最高です。



■話は変わりまして、


　一ヶ月前、初めて個別コンサルさせて頂いた
　上野大介さんより、

　個別コンサルについて、
　以下のようなご感想を頂戴いたしました。


　（ここから）
　----------------------------------------------------------------


　第一回目の個別コンサルを受けてから
　約１ヶ月が経過しました。


　まさかセミナーを主催する立場になるとは
　予想もしておりませんでしたが、
　
　最終的に昨日２５日に、
　無事にセミナー＆懇親会を終えることができました。


　（いま考えるとお恥ずかしい限りですが）

　１回目コンサル時に、

　「セミナーを主催した先には何があるのでしょうか」

　という私の問いに対し、

　『どうなるかなんて分からない』

　と即答いただいたことが一番印象に残っており、

　「ならば、やってみてから考えよう」

　の心持ちでここ１ヶ月の間を過ごしました。


　結果、

　「やってよかった。」

　の一言に集約されます。


　実にさまざまな学びや気づきがあり、
　ここには記載し切れませんが、


　これまで、ほとんど使ってきていなかった、

　主催者という立場になって始めて分かる感じる
　視点や行動の一部を、

　実際の行動をとりつつ垣間見たような気がします。


　また、講師の谷さんには本当にお世話になりました。

　ご紹介いただきありがとうございました。
　私には最適の方だったと感じております。


　講師の谷さんは、「講師」でお世話になるというだけでなく、

　「セミナー主催者の心得の先生」

　的な面も非常にお持ちで、
　目からうろこのお話をたくさんしていただきました。


　そして特に、「三方良しの精神」（参加者、講師、主催者）が
　無意識レベルまで身に付かれている雰囲気に触れ、感じることが
　できたことは、かけがえのない経験となりました。

　また、磨いていきたいと思います。

　そして今後もこのご縁を大切に育てていけたらと感じています。


　※課題のひとつで、今回主催者という立場のことだけで頭が
　　ほぼいっぱいになっていた自分に反省です。

　　参加者や講師、その他についてもっと考えたりするなどが
　　必要だと感じています。


　第一回目の渦まき結果の総括としては、

　有形無形で考えますと、上述のように、
　得るところがずっと多かったと感じております。

　今後も

　「どうなるか分からない」

　ではありますが、

　継続の先には様々な光があるような予感がしています。


　　　　　　　　　　　　　　　　　　　　　（上野 大介 様）


　----------------------------------------------------------------
　（ここまで）


　上野さん、ありがとうございます！



■まさに

　「やっていただいたからこそ、
　　分かった生の声」

　ですね。


　谷さんとは、
　私も長らくお付き合いしてきましたが、
　
　「成長を遂げ続けている、魅力的な男」

　です。
　
　谷さんとのご縁もぜひ大切にして頂ければと
　思います。
　


■せっかくですので、

　頂戴したメールに記してくださった感想に
　関連する形で、

　フォローアップも兼ねて、
　以下の本文に続けたいと思います。

　
　次回のコンサル（といっても数日後ですが 笑）も、
　よろしくお願いいたします！



　
■それでは、

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。

＜今日のピークパフォーマンス方程式＞

【  「分かったから、やる」ではなく「やるから、分かる」  】

───────────────────────────────────



■よくコンサルの場で伝えるのは

　「分かったからやる、ではなく、
　　やるから分かるんです」

　という言葉。


　（受験戦争の弊害なのか？）

　少なから方が、


　「それをやったら、
　　どんな結果になるのか、

　　教えてもらって
　　納得できればやりますよ」


　ということを言われます。



■それに対して私（鮒谷）は

　学校の試験じゃないのだから、
　それが正解かどうかなんて

　「私にも分かるはずがない」

　とお伝えいたします。

　
　間違いなく分かっているのは


　「何かやったら

　　（それが何かは分からないけれども）
　　何かは起きるし、

　　何もやらなかったら、
　　今までどおり、何にも起きることはない」


　ということであって、だからこそ、


　「分かったからやるのではなく、
　　やるから分かるんです」


　「もし本当に変わりたいのであれば、

　　せめて昨日までと同じ活動をするのだけは
　　止めましょうよ」　


　という話をしているわけです。



■上にご紹介した上野さんのご感想にも、

　
　-------------------------------------------------


　１回目コンサル時に、

　「セミナーを主催した先には何があるのでしょうか」

　という私の問いに対し、

　『どうなるかなんて分からない』

　と即答いただいたことが一番印象に残っており、

　「ならば、やってみてから考えよう」

　の心持ちでここ１ヶ月の間を過ごしました。


　-------------------------------------------------
　

　と書いて頂いておりますが、

　ここに記して頂いているのは、
　まさにこのやりとりのことなのです（笑）　



■これまで行ったことのない世界に出ようと
　いうのであれば、

　行ったことがないのだから、
　どうしたって


　「分からないけれども、

　　というよりも、

　　分からないからこそ、前に進む」


　という心的態度が絶対に必要です。


　この姿勢が習慣化されない限り、
　永遠にその人は変われません。



■ぶっちゃけ、なんだっていいんです。

　これまでと違うことさえ
　やってもらえるのであれば。


　違うことをやったら違う結果が起きるし、
　それがたとえ

　「望ましくない形での変化」

　だったとしてさえ、
　退屈な日常から抜け出す契機となるものです。　



■こんな風なことを繰り返しているうちに、
　気がついたら

　「平々凡々、退屈な日常」

　から

　「ジェットコースターに乗っているかのような
　
　　不安と恐怖に満ちつつも、
　　しかしながら刺激と興奮満載の人生」

　へとシフトしていたことに気づくのです。



■ただし、

　「平々凡々、退屈な日常」

　を愛するのであれば、

　わざわざそこから離れる必要もないし、
　私も何も言いません。


　そして、あなたがもし後者の方であるならば、
　
　きっとこのメルマガも合わないので
　解除頂いた方が時間のムダにもならないかと思います。
　

　※どちらがいいとか悪いとかいう話ではなく、
　　あくまで価値観の問題に過ぎません。



■ただし、あなたが長らく

　「このつまらない日常から
　　どうしたら抜け出せるだろう」

　と懊悩、煩悶されているのであるならば、


　悩むのもほどほどに切り上げて、
　とっとと何か行動を起こすしかないのでは、

　と口を酸っぱくして
　お伝えしてきたわけなのです、



■そして

　「とっとと何か行動を」

　が私のセミナーや個別コンサルだったら
　いうことありませんが（笑）
　
　というのは冗談で、


　わざわざカネもかかるし、時間もとられる、
　そんなものに申し込まなくても、


　昔の友だちに連絡入れて久し振りに再会するもよし、

　旅に出てもよし、

　なにかのお稽古事にでも申し込むもよし、

　仲間を集めて勉強会でも開催するもよし、

　
　今までの自分にとって

　「非日常な何か」

　を始めればいいでしょう。


　何かを始めれば、何かが起こるでしょうし、
　何も始めなければ、何も起こりません。
