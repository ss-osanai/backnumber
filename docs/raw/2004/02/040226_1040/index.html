＜今日のピークパフォーマンス方程式＞

　　　　【勝ちグセをつける、ということ。】

───────────────────────────────────


■人生において、「勝ちグセ」をつけるというのは重要なことです。

　勝つパターンにはまるのと、負けパターンにはまるのは、当たり前
　ですが転地雲泥の差があります。


■ここでいう、勝つ・負ける、という言葉。
　これを他者との関係のなかで捉える人もあれば、自分自身との関係性と
　いうコンテクスト（文脈）で捉える人もあると思います。

　が、どっちでもいいです。都合のよいほうでお読みください。

　自分自身の関係性、で以下を読むなら、「克つ」としてお読みくださいね。


■両者の差は最初はたとえば１センチ上に進むか、１センチ下に落ちるか、
　の小さな差。

　とはいえ、これだけでいちどきに２ポイント、差が付いています。

　そしてちょっとした勝利、敗北がきっかけとなって、勝利・敗北のスパイラル
　状態に入ることが往々にしてあります。


■インフレのスパイラル、デフレのスパイラル。

　一回で２ポイントの差が付いているとしても、片方が階段をさらに上がり
　片方が階段をさらに下がると合計４ポイントの差が付きます。

　これを３回、５回、１０回、１００回、と重ねると、両者では取り返し
　ようのない差が発生してしまっているはずです。


■そして、その差はどこから始まっているかというと最初の第一歩。
　ちょっとしたきっかけが大きな未来を形作る、ってことがありますね。

　どんな小さなことであってでもいいから、成功体験を積むことが大事と
　いわれるのは、ここに理由があるのだと思います。


■最初の一歩を踏み出すことで、次の一歩が踏み出しやすくなる。

　この一歩を踏み出すのに大切なのが、

　「勝とう」

　という意思。
　
　どんな小さいことでもいいのです。どんな些細なことでもいいのです。
　ほんの小さなことから「勝ちグセ」をつけていく。
　それがどんどん大きく複利で回転することで、加速度がついていく。


■以前、何かの本で、

　勝つという意思が大事で、つぎに勝つための努力をし続けることが大事、

　ということを書かれているのを見たことがあります。
　

■「勝とう」と思わなければ勝つことができないし、最初の一回の勝利を
　大事にして、それを元手にどんどんチャレンジしていく。

　そう考えると、「勝つ意思」と「勝ち続ける思い」両方が同じくくらい
　大事なものなのかもしれませんね。

┃　そのためにはどんなに小さなことでもいいから、「勝とう」と　┃

┃　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　┃
