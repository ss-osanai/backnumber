6771号 心が震えていない＝生きながら死んでいる？といったら言い過ぎか

■なにかに集中して、
　行動し続けられる人は例外なく

　【魂が震えている人】

　です。



■己の魂が震えてもいないのに、

　人の魂を揺り動かす話はできず
　行動にも落ちてこないし、

　まして、人さまに動いてもらうことなど
　できるはずもありません。



■仕事にはもちろん

　【人に動いてもらう営み】

　という側面があるわけで、


　特にビジネスを
　なされている方であれば

　【血と汗と涙の結晶であるところの
　　命の次に大切なお金】

　を頂戴しようと思うのであれば、

　それを差し出して頂けるだけの「震え」を
　お客さまに提供しなければならない、

　と私（鮒谷）は考えています。



■自分が感動もしていないものを、

　人さまが感動するはずがないでは
　ありませんか。


　自分が感動もしていない商品や
　サービスを売りつけて、

　それを見た人、触れた人が、

　「感動しました！！」

　なんてことになるはずがありません。



■自分が、毎時、毎秒、

　「ブルブルと震え続けていて」

　さらには

　「なぜ『私』がこんなにも震えているのか
　（震えずにはおれない理由）」

　を説き続けるからこそ、

　その驚きや興奮や喜びや感動が、
　他者に伝播していくのです。


　その、お客さまの震えが「購買」へと
　つながっていくのであれば、

　ビジネスは何も難しいものでは
　ないと思われませんか。



■そんなお前（鮒谷）は、どうかって？

　もちろん時々刻々に震え続けています。


　どころか、

　その震えの源を探るべく、

　震えた瞬間、「震えの源」を捉えるべく
　日記（正確には「時々刻々の記」）を、

　その名の通り、時々刻々、

　誰にも負けぬほどのボリュームで
　記述し続けているつもりです。



■その記録を、

　差し支えのない範囲でほんの一部、
　おすそ分けしている、

　それがこの「平成進化論」という
　メルマガ。



■こうして配信を続けていると、

　なぜ、お前のメルマガは
　こんなに長いのだ、

　とよく尋ねられます。


　あるいは、

　書こうと思っても、
　そんなに毎日、書くことはできないよ、

　書こうと思っても、
　そんな長文、書き続けられないよ、

　とも言われます。



■でも、私（鮒谷）は義務感で
　書いているわけではありません。


　日々の、驚きや興奮や喜びや感動を、
　すかさず捉え、記録に残し、

　中でも特に、

　「メルマガをお読み下さっている方に、
　　最も刺さるかな、と思われる記録」

　を、一日一通、
　お届けせずにはいられないから、

　お届けしているだけのこと。



■本当は、毎日、３通、５通と
　お送りしたいくらいですが、

　それだと
　ただでさえ少なくない

　「解除率」

　が、半端なく増大するのは
　間違いないので自重しているだけの話。



■以前、ある方が

　【震える＝没頭＝熱量＝想いがこもる＝
　　愛がある＝差別化】

　とおっしゃってくださったことが
　ありますが、

　とするならば、

　震えられる対象を見つけられれば
　勝負あった、

　なんですよね。

　
　であれば、

　震えられる可能性が高い場に
　身を置き続ける習慣を持てば良い。



■そんなわけで私（鮒谷）も、

　ブルブル、ブルブル、ブルブル、ブルブル、
　ブルブル、ブルブル、ブルブル、ブルブル、　

　ブルブル、ブルブル、ブルブル、ブルブル、
　ブルブル、ブルブル、ブルブル、ブルブル、　

　ブルブル、ブルブル、ブルブル、ブルブル、
　ブルブル、ブルブル、ブルブル、ブルブル、　

　ひたすら震え続けたいと願い、
　その対象を探し求める日々の中で、


　【月間で、のべ数百？の
　　震えの源の中から、

　　厳選した10ほどを
　　ご紹介している場】　　

　が「年間プログラム」となります。



■毎日、「震え」を求めて生きている
　私（鮒谷）が震えた話の中でも、

　ひときわ、大きく揺さぶられた
　ネタを厳選して月に二回、

　「11時間＋3時間」

　に渡って
　お伝えしているわけですから、


　受け取る側の方も、

　「多かれ少なかれ、
　　震えて下さるに決まっている」

　と思っています。



■こうしてお伝えしている驚きや
　興奮や喜びや感動が、

　参加下さった方の心に響き、
　刺さり、それがやがて

　【震える＝没頭＝熱量＝想いがこもる＝
　　愛がある＝差別化】

　という流れからの、
　圧巻の行動力につながることになれば


　【あとは勝手に成果が出始める】

　世界に出られるわけですから、
　そこまで進んでいただきたい、

　そんな思いを持ち、毎月二回、
　開催しているのが「年間プログラム」です。



■何度でも書きますが、

　日常、心がまるで震えていない人が、
　行動に移せるわけがありません。


　「心が動いてから、
　　体が動く（動かずにはいられない）」

　のが人間という生き物。

　久しく、
　がつんと心揺さぶられる経験がない、

　とするならば、
　それは極めてよろしくない状態。



■ぜひ「年間プログラム」に参加し、
　ぐるんぐるんに心揺さぶられて下さい。

　こうした大きな感情変化から
　人生が変わり始める、

　といったことが
　往々にしてあるものです。


　今日も人生とビジネスを楽しみましょう！

【今日のピークパフォーマンス方程式】　　　
　　　　　　　　　　　　　　　　　　　　　
　■心が震えて、体が動く。　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■心が震えていないのに、体が勝手に動く　
　　（動かずに入られなくなる）ことなど　　
　　起こるはずがないだろう。　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■最近、心が震えていない、とするならば　
　　そういう場を探し、そこに身を置くこと。
　　　　　　　　　　　　　　　　　　　　　
　■成果を生み出す行動をもたらす感情を　　
　　生み出してくれる場の選択こそが　　　　
　　もっとも大切である所以。
