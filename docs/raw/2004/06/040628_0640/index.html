【野村さま】

感銘を受けました。しかもそれを実際に行動に移しながら模索されている

られるような志みたいなものを忘れないようにしようと決意しました。

【情報は集まるところに加速度的に集まる。情報の中心に位置するという事】

───────────────────────────────────


■ご感想のご紹介が長くなりましたので、今日は短めに。
　（というか配信時間がかなり遅くなりましたので…）


■メールマガジンを書いていてつくづく思うのは情報というものは集約されて
　いるところに、さらに加速度的に集まるということ。

　情報発信するところに情報が集まる、というのは今まで散々書いてきた
　ことですが、付け加えるとするならば、情報が集まるから、その情報
　との交換？により、さらに情報が集まってくるという構図があるようです。

　どんどん雪だるま式に集まってくる情報が増えてきて、これは
　加速度をまして集まってくる（押し寄せてくる？）ようにも思えます。


■情報集積地、というか情報巨人、というか情報ターミナルというべきか、
　情報におけるハブのとしてのポジション、というべきか。

　各業種業界、あるいはビジネスにおけるそれぞれのジャンルでそんな
　ポジションが取れたとすると。


■今からの時代は「情報」が富を生み出す流れがますます加速している
　ので、まずは自らの中に情報を集積する、という意思、意欲、思いを
　もつことはこれからじゃ特に求められる考え方ではないかとおもって
　います。


　（本日短めに）

　世の中の声に従い、価格競争に巻き込まれてしまえば、もうその会社に
　未来はありません。

　大手企業だけです。ほとんどの中小企業で、価格競争を勝ち残る体力の
　あるところはないでしょう。
