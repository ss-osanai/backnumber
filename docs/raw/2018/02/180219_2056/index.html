【 苦笑・失笑されながらも売り続ける、私（鮒谷）の秘奥義を初公開 】

───────────────────────────────────



■私（鮒谷）はセールスにおいて

　【お客さまにとっての購買行動を
　　エンターテイメント化する】

　ことを強く意識しています。


　なぜなら基本的に、どんな人であっても

　「お金を払うことには本能的な忌避感情、
　　あるいは拒絶感情を持つ」

　ものだと考えているからです。



■こうした負の感情を乗り越え、

　「できることならば笑顔で購入」

　して頂くためには、
　どんなことだってやるのです。


　あなたも先刻ご承知の通り、
　私（鮒谷）は日々、音源を販売しています。


　提供している音源は、
　
　（一般的に見ると）
　決して安いものではない、
　
　ことも重々承知しています。


　※ただし内容との比較であれば、
　　「破格値」との確信を持って案内していますが。



■音源の価値を未だご理解頂けない方にとって

　「安いものではない（と感じている）」

　ということはあるでしょうし、


　そうした方にとっては、
　ただでさえ存在している

　「売られることに対する忌避＆拒絶感情」

　が、より激烈な形で現れます。



■そうした感情を緩和し、さらには逆転させ、
　最終的に

　「笑顔で購入」

　の世界にまでお越し頂くためには
　かなり難度の高いプロセス管理が必要です。


　このノウハウは秘中の秘であるわけですが、

　今日は、ちょっといいことがあって
　気分が高揚しているので（笑）
　
　お伝えしようと思います。



■このプロセスのキモは、一にも二にも

　【（自然な形での）我田引水】

　にあります。
　

　ここでいう「我田引水」とは、

　最初から最後まで、無理のない、
　緻密な流れで論理＆物語を展開していき

　「～なので、この音源のお求めを」

　というところに着地させること。



■そしてときおりは、

　お読み下さる方が想定されるであろう
　論理＆物語展開の範疇を、

　わざと大きく逸脱させ、ときに

　「苦笑不可避、失笑不可避」

　の
　
　「まったく音源とは無関係に思える、
　　真面目な論理＆物語展開」

　から急転直下、突然の、

　「～なので、この音源のお求めをｗ」

　といった構文を織り交ぜること。



■こうした

　「正統的」ときに「異端的」な構文を
　恒常的に発信することによって、

　「毎回、発信される、
　　論理＆物語の展開自体に関心を示して頂く」

　ことができれば、それがお客さまにとっての

　【購買の意思決定を
　　エンターテイメント化する】

　ことにつながっていくのです。
　　


■こうした流れを恒常的に行うことで、

　「お約束」

　にまで昇華されると、あたかも
　
　「水戸黄門の印籠が出てこない
　　水戸黄門などあり得ない」

　ように

　「音源販売の案内のない
　　平成進化論などあり得ない」

　というイメージが形成されることに
　なるでしょう（多分ｗ）。



■もう引退？されましたが、

　ジャパネットたかたの高田明さんだって、
　番組にあの顔（と声）が出てきたら、

　それを見ている人は
　
　「ああ、これから商品を販売されるなあ」

　とすぐに分かるのです。


　でも、なぜか見てしまう。
　
　それは

　「エンタメ化された販売
　　（客側からみるとエンタメ化された購買）」

　の域に達していたからではないでしょうか。



■私（鮒谷）も同じく、

　「エンタメ化された販売
　　（お客さま側からみると購買）」

　を目論んでいるということです。
　
　ただし、これは言うは易く行うは難し、です。


　『水戸黄門』において

　「コンテンツの構造は深層では同じ」

　であってでも

　「毎回、引き起こるあれやこれやの出来事は
　　各回、全く別のシチュエーション」

　です。



■同様に平成進化論においても

　「コンテンツの構造は深層では同じ」

　であってでも

　「毎回の事例や、そこから説き起こされる話は、
　　各回、全く別のシチュエーション」

　なのです。


　『水戸黄門』なら「印籠」、
　『平成進化論』なら「我田引水的結論（音源案内）」

　が着地、という、


　深層構造と着地は同じで、
　でも、各回の表面はまるで別コンテンツ、

　こうした発信が高頻度でなされると

　「構造のエンタメ的要素に虜になる
　　（人も現れる）」

　のです。



■ただし、

　表面は全く異なるように見えて、
　毎回、必ず同じ着地に落とす、

　そんな発信を恒常的に
　行えるようになるためには、

　そのテーマについて朝から晩まで
　考え続ける必要があります。


　『水戸黄門』の脚本を書いていた人も
　（これは想像ですが）きっと朝から晩まで、

　番組のことだけを考えて
　生きていたのだと思います。



■私（鮒谷）も同じで、朝から晩まで、
　『平成進化論』のことを考えているから、

　着地は毎回、全く一緒なのに、
　
　そこに至るプロセスはまるで異なるように
　見える（はずの）コンテンツを、

　ひたすら発信し続けることができるのです。



■華麗にモノを売りたいのであれば、
　ただ一つの商品（私なら音源）を売るために


　小泉今日子の『ヤマトナデシコ七変化』ならぬ、
　『商品販売七変化』

　江戸川乱歩の『百面相役者』ならぬ、
　『商品販売百面相』


　のごとくに、同じモノを売るのに、

　最低でも７つの切り口、
　できれば100ほどの切り口、

　が提示できるくらいでなければ本物ではない
　（自社商品について真剣に考えていない）

　のでは、と問題提起したいのです。



■そもそも、

　自分が愛している商品を、十年一日、
　たったひとつのポイントでしか語れないとか、

　情けないにも程がある。


　３時間でも、10時間でも、
　３日間でも、１週間でも、

　無尽蔵に、あらゆる角度、
　あらゆる切り口で案内できるくらいに、

　自分の商品を愛し抜き、自分の商品を
　考え抜く必要があるのではないでしょうか。



■音源という姿かたちなく、

　実態のない、
　しかもそれなりに値の張るものを


　▼銀だこのアルバイトさんが
　　たこ焼きを売るように

　▼球場の売り子さんがビールを売るように

　▼アイスクリーム屋台の人が
　　アイスクリームを売るように


　摩擦なく、次々と売れていく、という形で
　販売することができているとするならば、

　その理由はひとえに

　自分の商品を愛し抜き、自分の商品を考え抜き、
　その想いを発信しているからだと考えています。

　
　これくらい愛し、これくらい考えたら、
　これくらい売れて当然（というかまだ足りぬ）、

　と思っているのです。



■大げさではなく、人生をかけて
　自社の商品について考えていれば

　「お客さまの購買活動をエンタメ化する」

　ことなど造作もないこと（と思います）。


　こうした取り組みの継続によって
　商品購入に際して生まれる

　「忌避感情や拒絶感情」

　を乗り越え、

　読み手をときに呆れさせたり、
　苦笑、失笑されながらも

　「しょうがないなあ、
　　そこまでいうなら買いますわｗ」

　という流れになるのです。

　（というか、この流れになったら
　　売れるだろうと）



■この

　「しょうがないなあ、
　　そこまでいうなら買いますわｗ」

　を引き出し、お求め頂く瞬間が、

　私（鮒谷）にとって

　「エンタメ販売」

　が実現された瞬間となるわけです。



■言葉を換えれば、

　モノを販売するときに
　どうしたって必要なのは

　【物語的発想を駆使して、
　　売りたい商品を販売する】

　という思想、あるいは発想です。



■既視感のある、
　単調な物語ではモノは売れません。


　思わぬところから始まり、
　そんなところで着地するのか、
　
　との驚きを与えられたとき、


　その意外感はエンターテイメント性となり、
　さらにはカタルシスにつながり、

　そのエンタメ性やカタルシスに対して

　「意気に感じて（！？）下さる方」

　がお買上げ下さる、
　という流れができるのです。



■そう考えると

　「物語発想を持たぬ人は、
　　商売すべからず」
　
　と言っても言い過ぎではないように思われます。


　ここまでお読み下さり、

　【私は物語発想に乏しいから、
　　モノがあんまり売れないのでは】

　と思われた商売人のあなたには、

　　なった方がお求め下さる」

　ようになるのです（本当です）。



■センスを身につけた商売人は、
　物語発想を持ち、
　
　その上で、◯◯という一つの商品について

　「十数通り、ないし数十通りの物語」

　を用意し、来る日も来る日も


　「（毎度、表面は異なる物語＆論理展開）
　　→だからあなたには◯◯が必要ですよ」

　「（毎度、表面は異なる物語＆論理展開）
　　→だからあなたには◯◯が必要ですよ」

　「（毎度、表面は異なる物語＆論理展開）
　　→だからあなたには◯◯が必要ですよ」

　「（毎度、表面は異なる物語＆論理展開）
　　→だからあなたには◯◯が必要ですよ」

　「（毎度、表面は異なる物語＆論理展開）
　　→だからあなたには◯◯が必要ですよ」

　
　、、、と永遠に繰り返します。



■やがて、情報の受け手は、

　「常に目新しく、でも、
　　常に既視感のある構文」

　がクセになり、それは一種の
　エンタメ＆カタルシスとなり、


　ついには、商品そのものではなく、
　商品販売のプロセスそのものに対して

　「投げ銭的な形で購入下さる」

　ということすら起こるようになるのです。



■この世界に出た人は

　「販売、営業における免許皆伝」

　といって良いでしょう。


　なんといっても

　「商品の代価」

　ではなく、

　「商品の（エンタメ的）販売プロセスを
　　見せてくれたことに対する代価」

　として購入頂いているのだから。
　

　
■極端にいえば、売られた商品が
　クソみたいなものであったとしてさえ

　「限りなく、クレームが起きない」

　構造をここにおいて作り出すことが
　できるのです。


　ここにおいてはもはや、

　売っている商品ではなく
　売られるプロセスに対して支払いが発生し、

　その支払いの段階で
