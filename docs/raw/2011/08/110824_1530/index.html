＜今日のピークパフォーマンス方程式＞

 【　去る者は日々に疎し　】

───────────────────────────────────



■「去る者は日々に疎し」

　とは、よくいったもの。

　どれだけ親しくしていても、
　しばらくやりとりがなくなると、
　
　徐々に交流が少なくなり、
　やがては途絶してしまいます。



■むかし昔でさえ、

　「去るものは日々に疎し」

　なのですから、

　現在のように、これだけ情報流通のインフラが
　整ってくるとどうなってしまうのでしょう。



■あらゆる人から、あらゆる情報が
　洪水のように流入してきて、

　あっという間に、

　「他人の脳内における自分の存在」

　は、上書きされていってしまいます。


　昔とは比較にならぬほどの速度で、
　彼、彼女の脳内の深いところに沈殿していった

　「私という存在」

　は容易には浮かび上がってこない

　（＝思い出してもらえない、
　　　去る者は日々に疎し）
　
　ということになるでしょう。



■相手を責めても仕方ありません。
　自分の頭の中を覗き込めば、よく分かるはず。


　特に、

　日々、たくさんの人と会う方。

　日々、たくさんの人とネットを介して
　やり取りする方。


　もし、あなたがそういった人であるならば、
　すでに日々、体感されているように、

　昨日、今日、お会いしたり、やりとりした人との
　ことさえ、一晩、二晩、床に就いている間に、

　さらなる新しい情報が次々と押し寄せてきて、
　ほんの数日前の記憶が、曖昧、あやふやに
　なってしまっているはずです。



■これがいいことなのか、悪いことなのか、
　私には分かりません。


　しかし、どうやら、

　特別な立場や仕事をしているわけではない、
　ごくごく一般的、平均的な人にとってさえ、

　あまりにも、忙しく、慌しく、せわしない
　時代がやってきたことは間違いなさそうです。
　

　
■次々と押し寄せてくる会合、打ち合わせ、
　メールのやりとりをおこなっているうちに、
　
　ついさっき、考えていたことすら中断され、
　忘却の彼方へと押しやられていきます。

　
　ましていわんや、

　一週間、一ヶ月前の会合、会食のことなど、
　いつの出来事？太古の記憶？

　というくらいにスピードが速くなってきた。

　そのとき初めて出会った人のことも、
　きれいさっぱり忘れ去ってしまっているのかも、、、



■もしもあなたが、

　そうだそうだ、
　そういう風になりがちだよね、
　
　と頷かれるなら、
　　
　同様に、あなたもまた、
　相手に忘れ去られてしまっているだろう、

　ということになるでしょう。


　では、せっかく生まれたご縁を未来に繋ぎ、
　忘れ去られないようにし、

　さらには関係性を深めていくためには
　どうしたら良いのか。



■１週間に１人の新しい人と、
　新しい出会いが生まれるのであれば、

　１年間で50人、10年間では500人の人と出会います。

　
　１週間に５人の人と、
　新しい出会いが生まれるのであれば、

　１年間で250人、10年間では2500人の人と出会います。


　１週間に10人の人と、
　新しい出会いが生まれるのであれば、

　１年間で500人、10年間では5000人の人と出会います。



■この関係性の

　「歩留まり（率）」

　といったら叱られるかもしれませんが、


　人によっておおよそ、

　1年後の（関係性の）歩留まりは○％、
　3年後の（関係性の）歩留まりは○％、
　5年後の（関係性の）歩留まりは○％、
　10年後の（関係性の）歩留まりは○％、
　
　という数値が決まっているはずです。



■なにもこの「歩留まり」を
　無理やり高めようとする必要はありません。

　しかし、もし、せっかく自分に好意や共感を
　もってくれた友人・知人が、

　（何もしなければ、
　　10年後の歩留まりが１％のところ）

　ある行動
　（たとえば、お会いしてすぐのお礼メール）

　を取ることで2％に、
　
　あるシステムを導入する
　（たとえば、そういう人に年賀状を送る）

　ことで3％に、

　ある習慣を身につける
　（たとえば、承諾を得て、一週間に一度、
　　メールマガジンを送る）

　ことで5％に、

　
　と地道に歩留まりを高めていけば、
　時間の経過と共に、

　その「無形の財産」たるや途方もないことに
　なりはしないでしょうか？

　そういうことに思いを致してみては
　どうでしょう？
　
　
　そんな問題提起をしてみたいのです。
　
　

■もともと私は、

　不特定多数に向けたこのメルマガ（平成進化論）を
　発行する、さらにその数年前から、

　個人発行のメルマガを発行していましたが、
　この

　「無形財産の蓄積」

　をこそ、一番の目的、一番の眼目、
　として続けてきました。


　それは過去のメルマガでも、何度もお伝えして
　きたとおりです。

　たとえばこの記事。


　＜年賀状よりも「日賀状」を活用して関係性の維持につとめる＞

　http://www.2nd-stage.jp/backnumber/ppf_full/2007/04/1313.html


　
■「去る者は日々に疎し」

　ということにならぬよう、
　自分なりにいろいろな工夫を重ねてきたつもりで、

　もちろん反省すべきところもありますが、
　しかし、何もやらなかったなら、
　
　今、私を取り巻いている人間関係の大半は
　存在していなかったことと思われるのです。

　

■だからメルマガを書きましょう、

　という話ではなく、この話を通して、
　一人ひとり、なにをなそうか、

　あるいは、このままでいい、と判断されるか、

　そんな思考を行うための材料として、
　今日のお話を活用して頂けましたら幸いです。



　今日も人生とビジネスを楽しみましょう。

　でもあります。最近、一週間に一回くらいはさらっと目を通して、経営の
　原理原則についてとことん腹に落ちるよう、常に勉強させて頂いています。

　説かれています。
