　このたびは音源を提供頂き、
　ありがとうございました。


　年間プログラムで、音源化を念頭に置かずに
　「開陳」された内容を、

　Fさんをはじめ参加された複数の方が
　「今回は神回だ」との後押しを受けて、
　音源化されたとのこと。


　私にとっても
　刺さりまくる音源となりました。

　音源化を決断した鮒谷さんと、
　後押しして頂いた方には感謝です。


　ちょうど、私自身が、音源に没頭できる
　環境を整えたタイミングと重なり、

　この一週間、心を沈めて、
　「精聴」することができました。

　この音源で紹介をいただいた、
　「●●●●●●」も読みながら聴かせて頂くと
　相乗効果が高まるように思いました。


　自分は、これまで、自己啓発系の物語は、
　あまり読んできておらず、

　また、年齢的に
　「今さら」も感じていたのですが、

　なるほど、若者の立ち位置でなく、
　年長者の立ち位置で読むことを、
　音源で教えて頂きました。


　同時に、現状が物語の中の成功者と
　同じ状況とは言い難くても、

　内的に言語を回し、適切な自己肯定の
　プロンプトと信念さえあれば、

　自分の中に資源として眠っている
　これまでの膨大な経験や知見を、
　財産や武器に変えていくことができる、

　そうした勇気も与えて頂きました。


　この●●●●●●●では、
　厳しい生い立ちからスタートし、

　町に出て居心地の良い場所を
　やっと得られたところから離れ、

　より大きな成功の道を選択する際の、
　心の葛藤が描かれていますが、


　ここは、自分自身にとっても、
　この先の人生の自由度を高めるという
　目標を実現するうえで、

　最大の内なる敵と考えており、そこは、
　川の流れを変える治水工事をするが如き、

　強い気持ちで突破しなければならないと
　知らされました。


　「全ての悪戦苦闘の成果物」

　「信念が創造性を発揮してくれる」

　といった言葉の宝石が飛び交い、

　まだまだ、聞くたびに新たな発見のある、
　噛めば噛むほど味の出る、

　スルメのような音源であり、
　しばらくは繰り返し聞きたいと思います。


　それにより、自分の中に言葉を取り込み、
　ウォーミングアップを重ね、

　来週以降の「祭り」の日々、
　ストレスフルな毎日を、
　自分なりのゲーム感覚で乗り切ることで、


　感情を短時間で収斂させ、
　解決モードのテーブルについて、
　効率よくゲームを進めつつ、

　自分の目指すところへ行動をおこし、
　処を得るところまで、
　道を歩んで行きたいと思います。


　ちょっとまとまりませんが、今後とも、
　どうぞよろしくお願い申し上げます。

＜今日のピークパフォーマンス方程式＞

■10時間、ノンストップで没頭し続けられる
　対象を見つけられれば勝ちというゲーム

───────────────────


■いつもお伝えしていることですが、
　
　【せずにはいられない対象】

　を見つけられさえすれば
　人生は勝ちですね。



■ここでの「勝ち」は
　他の誰かと比較しての勝ち負け、

　という意味ではもちろんなく、


　納得できる生き方ができる前提が
　ここにおいて整う、

　といった意味で
　「勝ち」という語を用いています。



■没頭、没入できる対象が見つかれば、
　人間、ほとんど休息も取らずに、

　数時間はおろか、
　10時間とか12時間とか、
　
　ぶっ通しで一つのことを
　行い続けられるもの。



■ゲームに夢中になっている子ども
　（大人でもありますが）は、

　平気で半日、さらには丸一日、
　疲れ知らずで時間を投入できますよね。


　あれと同じで、
　たとえ年齢を重ねても、

　子供と同じくらいのみずみずしさをもって
　一つのことに集中できるのです。



■集中が極まれば

　「ゾーンに入る」

　ことになりますが、

　この世界に到達すれば、気づけば
　５時間、10時間が過ぎていた、

　ということが普通に起こります。

　

■ちなみに私（鮒谷）は今、
　ある対象について、

　現在進行系で毎日（深夜帯に）
　ゾーンに入る生活を送っています。


　
■（健康にはよくなさそうですが）
　没頭没入の合間に、というか、

　没頭没入状態のまま、
　ウーバーで頼んだ料理を食する、

　が日常の風景となっています。

　没頭状態だからやむを得ない。


　そんな感じで、絶賛、第何度目かの
　「モラトリアム期」を過ごしています。



■これくらい夢中になれれば、

　取り組んでいる時間そのものが
　幸せですし、


　さらにその対象は時間をおいて、
　数年も経てば

　【圧倒的な強みに昇華され、
　　当該分野の言葉も溢れてくる】

　ようになるはずです（経験済み）。



■ただ、こうした生活は、

　先行きが約束されていない対象に
　リソースのオールインを行うのと同義、

　ですから、

　キャリアとか経済的な側面において
　相当のリスクが伴います。



■さらに、

　必然的に他の人と違う生活を
　することにもなりますから、

　取り巻く環境から加わる同調圧力に
　逆らうことにもなりますね。



■リスクや同調圧力を過度に恐れると、

　フルベット没頭生活に憧れつつも、
　
　キャリアの断絶につながらないか、
　周囲の人から反対されないか、
　経済的に持ちこたえられるか、

　などと恐れて、

　保守的に生きる選択を取る人も
　少なくないように思われます。



■けれども、守りに入って
　保守的に生きようとするほど、

　毎日がつまらなくなる上に、

　生きるほどに可能性が閉じていく、
　希望が失われていく、
　
　といったことになりかねないように
　思われます。



■それが嫌なら、

　資源の傾斜配分を
　行わずにいられなくなるくらいの、

　具体的な対象を
　見つける必要がありますが、


　それは、あちこちに
　アンテナを張り巡らし、

　いろんなところに出歩かなければ
　容易には見つからないもの。



■さらには、それが見つかっても、

　「資源の集中投下」

　に腰が引けて、中途半端な形での
　逐次投入となってしまっては、

　モノにはなりません。


　
■どんな対象であれ、
　ブチ抜けようと思ったなら、

　キチゲージを振り切って

　「相転移＝閾値超え」

　の狂気の世界に到達しなければ
　ならないのです。



■そういう意味で、
　人生やキャリアも投資と同じように

　【リスクを取らなければ
　　リターンを獲得できない】

　のでしょう。

　

■以下に

　（不快に感じる方もあるでしょうが）
　最もわかりやすい例をあげましょう。


　首を切られる心配のない
　日系企業勤務のサラリーマンが、

　お金を失う危険があるからと
　副業はしない、投資もしない。


　そんな中、生活費の残りのお金を全額、
　銀行預金に回していながら、

　「老後不安を克服できる程度の
　　お金が貯まらない」

　と嘆く人があったとするならば、
　そうなってしまった原因は、

　一言でいうと

　【リスクを取ってこなかったから】

　です。



■その分、

　未来に対してリスクを取って
　動いた人よりも

　「（その時点における）
　　安全、安心」

　というリターンを
　得ていたということですから、

　バランスしているといえば
　バランスしています。



■もちろん、リスクを取らない生活を
　否定しているわけではありません。

　リスクを取って
　すっからかんになった人も、

　たくさん見てきましたからね。



■ですから、
　ただリスクを取りさえすればいい、

　という話ではありませんが、

　とはいえ、

　「リスクを取らなければ
　　リターンは得られない」

　この現実を理解した上ででなければ
　自分にとって最善の選択はできません。



■その上で、

　有形資産も無形資産も手にして
　人生や生活の質を高めたいのなら、

　破滅リスクをコントロールし、
　期待値を把握した上で、

　リスクを取って行動するのには
　一定の合理性があるのではないか、

　とは思います。



■もちろん「投資≠投機」ですから、

　戦略も持たず、ただやみくもに
　没頭できる対象にベットせよ、

　そうすれば必ず満足のいくキャリアを
　切り開けるとか、経済的に報われる、

　などといったことは言えません。

　

■ただし、

　適切なリスク管理の方法を理解した上で、
　リターンを追求すれば、

　破滅を避けつつ、保守的な運用以上の
　リターンを獲得できるのは、

　投資も人生、キャリアも
　同じではないでしょうか。



■そう考えて、

　私（鮒谷）は現在、
　人生の次のステージに進むために、

　破滅リスクを
　限りなくゼロに制御しつつ、

　再び大きな投資に
　邁進（没頭）しているところです。



■（メルマガに詳細は書いていませんが）

　かなりのリスクを取って、
　
　緊張感高く、
　毎日、冒険を続けている生活の中で、

　このところ、考えていることを
　お伝えしました。
