7453号 大量行動するほどに悪くなっていく人

7452号 15年前、防衛大学校の校長を前にプレゼンしたときの話

■日経新聞の夕刊に

　『こころの玉手箱』

　というコーナーがありますが、
　長年、愛読してきました。



■こちらでは、

　各界でご活躍されている方々が
　一週間（全五回）にわたり、

　過去の人生を振り返って、

　【昔の記憶と感情に密接に紐づいた
　　思い出の品々（＝こころの玉手箱）】

　を紹介されています。



■今週の月曜日から金曜日は、

　ある時期（およそ15年ほども前）、
　とてもお世話になった、


　前田建設工業株式会社、
　前田道路株式会社、
　株式会社前田製作所、

　などを傘下に持つ持株会社である、

　インフロニア・ホールディングスの
　岐部一誠社長が登場されており、

　毎日、楽しみに読んでいました。



■岐部さんとは、
　全５回の連載のうちの４回目

　＜あふれかえる自宅の本棚＞
　https://www.nikkei.com/article/DGKKZO77750190Y4A110C2BE0P00/

　の冒頭、

　------------

　読書量が2004年に急増した。

　会社を辞めるつもりでいたら、
　社長に研修を受けるよう勧められた。

　現在は「不識塾」という名の
　経営者育成講座だ。

　そこで膨大な課題図書を与えられた。

　------------

　の箇所にも登場する塾で
　ご縁をいただきました。



■私（鮒谷）が参加したのは
　今から15年前（2009年）、

　哲学、歴史、文化、宗教、倫理など
　リベラルアーツ教育を行う、

　経営者育成のための塾で、
　今とは異なる名称の講座でしたが、


　岐部さんはご自身が参加された
　2004年以降も、

　引き続き、運営？として携わられており、
　その流れでご一緒する機会を頂きました。



■今でもそうだと思いますが、

　参加者全員、日本を代表する
　大手企業の幹部社員や役員で、

　皆さん全員、当時の私より年長で
　居心地が悪かったのを覚えています（笑）



■私（鮒谷）は当時、たまたま読んだ
　書籍を通して勉強会の存在を知り、

　（立場的にも、年齢的にも、
　　対象外だったのですが）

　「どうしても参加したい！」

　と思い、塾のOBでいらっしゃった、

　旧知の品川女子学院の理事長、
　漆紫穂子さんにお願いして、


　塾長の中谷巌先生

　（マクロ経済学者、
　　多摩大学名誉学長、一橋大学名誉教授、
　　元ソニー取締役会議長、
　　元三菱UFJリサーチ&コンサルティング理事長）

　と面談の機会を頂戴し、
　特別に参加許可を頂きました。



■残念ながら（！？）、

　企業派遣ではないので、
　誰も参加費は出してくれないので、

　私費（参加費と研修旅費を入れると
　600万円ほどでした）

　を投じて丸々一年間、
　こちらで勉強させてもらいました。



■毎週土曜日、
　いろいろな先生がお越し下さり、

　皆さまの前でプレゼンする機会が
　何度か回ってきます。



■そんな中、

　当時、防衛大学校長でいらした
　五百旗頭真先生

　※その後、日経新聞の『私の履歴書』にも
　　登場されていましたね

　の回で、


　先生を前にしてプレゼンをする機会が
　与えられました。

　そのときにも岐部さんに内容について
　相談したことも覚えています。



■そんなこんなで、この一年は、

　仕事はしていても、大量の課題図書も
　読まなければならないし、

　ほぼすべての可処分時間を
　勉強会の準備と講座参加に充てていた、

　そんな記憶があります。

　中学受験以来、くらいの勉強量（笑）



■岐部さんも、そこで
　学ばれていた頃を振り返られて、

　コラムの中でこんなことも
　書かれていました。


　（ここから）
　----------


　どれもなじみの薄い
　宗教や哲学の書籍で古典も多い。

　オーストリア出身の経済学者、
　フリードリヒ・ハイエクの著書などは
　1冊読むのに1カ月もかかった。


　そこでまずは一気に読み進めるよう
　作戦を立てた。

　分からなかった箇所に線やメモを書き込んで、
　一度読み終わった後で読み直すようにした。


　特にフランスの哲学者デリダが掲げた
　「脱構築」には刺激を受けた。

　既存の枠組みや体系を解体し、
　新たに構築し直す方法論だ。


　（中略）


　自宅には本があふれ、
　新しい棚を買ってもすぐ埋まる。

　読み終えた本は処分しなければと考え、
　古本屋に持っていったこともあるが
　やめてしまった。

　自分にとって新たな世界を開いてくれる本。
　時たま読み返したくなる。


　----------
　（ここまで）



■これを読んで
　改めて思い出したんですが、

　たしかにめちゃくちゃ、
　大変だったんですよね。


　普段、読み慣れない本を読み、
　考え付けない考えに触れ、

　自分の中で咀嚼し、ときに発表する
　機会まであるわけですから、

　かなりの負荷がかかりました。



■でも、今から考えると、
　本当に貴重な機会だったなと思います。


　しかも（今も若いですが！）
　今よりも随分と若いうちに、

　そんな場に身を置き、そんな学びに
　身を浸せたことは良かったなと。



■もちろんそこでの学びが、

　直接的に何かのビジネスに
　つながったわけではありません。


　たとえば私（鮒谷）が、

　防衛大学校長（当時）の
　五百旗頭真先生を前にしてプレゼンしたのは、

　戦前の軍部の暴走についてであり、

　「統帥権干犯」

　とか

　「軍部大臣現役武官制」

　についての内容でした。



■もちろんそこでの学びが
　商売に結びつくはずはなく、

　（当たり前ですが）
　結びつけようと思ったこともありません。



■とはいえ、
　以上は一例に過ぎませんが、

　本業とは無関係の、
　いわゆるリベラルアーツの学びは

　「無用の用」

　ではありませんが、


　たとえ直接的に
　なにかに役立つことはなくとも、

　好奇心旺盛に、関心を持ったテーマを
　あれこれ学んでいくうちに、

　徐々に

　【考える力
　　＝生存するための基礎体力】

　のようなものが培われた、

　そんな実感は間違いなくあるのです。



■ここでの学びは、

　何を勉強したから、
　この力を手に入れ、生存能力が高まった、

　というような

　「１対１対応」

　では断じてありません。



■潜在意識下で、多様な学びが互いに、
　そして複雑に結びつき、

　それらが総体として

　【生きていく上での力】

　として昇華されてきたのではないか、
　というような実感。



■こうした蓄積は、
　視界をより遠く、より広くさせてくれ、

　さらには事象を深く認識するための
　助けともなります。


　やがてビジネスから社会、
　ひいては人生そのものについて、

　あれこれ思索を巡らせる貴重な材料、
　ならびに機会にもつながっていきました。



■結果として、

　リベラルアーツについて
　たとえ少しづつではあったとしても、

　今なお、学び続ける習慣が
　身につきましたが、


　振り返ると、この時期の学びは、

　ここしばらく
　お伝えした内容に絡めていうならば

　「構造、流れ、本質」

　を多少なりとも見通すための、

　これ以上ない訓練となっていたようにも
　思われます。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■リベラルアーツを学ぶことは
　　「無用の用」的な効用をもたらしてくれる
　　ことがある。

　■「１対１対応の学び」では断じてないが、

　　さまざまな分野の学びがいつの間にか、
　　相互に、そして深いところで結びつき、

　　やがて社会や人生を見通すための
　　補助線を与えてくれ、

　　それが生きる指針や力にまで昇華される
　　といったことがあるように思われる。
