■このメルマガ上で何度か書いていますし、またいろいろなところで
　目にし、耳にされることであると思いますが、

　【 「知っている」と「できる」は別物 】

　です。

　自身が進化することで、その深い意味がわかってきて、
　そしてはじめて「できる」ようになるのです。



■ただ単にできるだけではなく、自動的に体が動くまでに
　腑に落ちているか、ということが重要でしょう。

　だからこそ、良い本は繰り返し読まなければなりませんし、
　こういう内容だろうと予測できるセミナーや講演でも、
　新しい発見があったりするので、参加すべきだと思っています。



■皆さん既にお気づきのように？わたし（鮒谷）は
　勉強が大好きです。

　多くの本を読んだり、セミナーに参加したり、
　教材を買って勉強したりしましたが、

　実は多くの場合、それらの本質、原理原則は同じ。
　
　

■切り口が変わったり、見せ方が変わったり、
　言い回しが変わったりするだけで、ほとんど同じ内容を言ってたり
　するものです。

　もちろん、「平成・進化論。」も同じですし、拙著も同様です。
　別にオリジナルなことを書いているつもりはまったくございません。



■しかし、そこにこそ価値があると思っているのです。

　ある人はAという言い回しに反応し、
　ある人はBという伝え方で深い理解をし、
　ある人はCという見せ方によって腑に落ちる。

　だから、自分が一番ピンと来るものであれば、
　それを繰り返し何度も反復すべきです。



■わたしは、自分が学習し実践したことを自身が一番理解しやすい
　ように翻訳して（だから本質は同じですね）、

　成功に至る道筋を「方程式」としてなによりも自分のために
　記録・保存をしておこうとおもってメルマガを書いています。

　ただ、それだけだともったいないので、もし多少なりとも
　読者さんのお役に立つとするならばご自由にご覧ください、と。



■同じような内容をいろいろな人からさまざまな表現で聞いたり
　すると、

　ある時、その本質がストンと腹に落ちることがあります。

　これが「知っている」から「できる」への最短ルートでは
　ないでしょうか。



■同じことに取り組んでも、できる人とできない人がいます。
　その差の原因は、腑に落ちたかどうか、
　というところにあるのではないかと思うのです。

　わたしもこれからより良い考え方、
　より良いビジネスの手法を求めていくでしょう。

　みなさまもぜひ、ストンと腹に落ちるところまでそれぞれの
　ジャンルについての学びを深めていかれてはいかがでしょう。

　

■上述の通り、

 　【 「知っている」と「できる」は別物 】

　で、知っているだけではなにも結果は現れてきません。
　
　「できる」ようになるために、なんども重ねて同じことを
　学ばなければならないのでしょう。
