　能動的な行動＆好奇心

　という投稿をいたしました。



■上記の


　「人は能動的に行動することで
　　世界を知る。

　　好奇心も合わさると効率的に知る。

　　さらに主体的に世界に介入して
　　実験することで、

　　物事にどんな因果関係が
　　あるのか分かるようになる」


　を読んで、
　おお！と思ったんですよね。

　自分の実感と
　完全に一致していたから。


■私たちは、世界と

　「能動的」
　「受動的」

　いずれの姿勢で向き合うかによって、

　世界を知る速度も広がりや深みも
　変わってきます。



■さらにここに

　「好奇心」

　が加わると、より学びの歩留まりが
　高くなるに決まっています。



■付け加えて、

　「主体的に世界に介入して実験」

　つまり、仮説を立てては
　あれこれ条件を変えて実験することで、

　世の理（物事の因果関係）を
　理解できるようになる。



■これらを満たすと、

　学びの質は高まるし、歩留まりも
　高くなっていくに決まっていますし、


　反対に、この逆、つまり、

　当事者意識なく、受動的で、
　好奇心も持たず、

　それゆえ仮説も立てず、
　実験も繰り返さないのであれば、

　学びの質・量ともに高まっていくことは
　なさそうです。



■現状を振り返る際のチェック項目として
　活用できるのではないか、

　と思いましたし、

　これらの項目がすべて満たされ、
　積極的な発動される、

　そんな場と対象のみを選んで
　経験を重ねる、

　といった姿勢も大切ではないかと。



■そんなことを考えつつ、

　この内容はあなたとシェアすることで
　お役に立つところもあるのではないか、

　と思い、ご紹介いたしました。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■高い質、ならびに歩留まりを学びに
　　求めようとすれば、

　　主体性、当事者意識、好奇心、
　　仮説検証（介入＆実験）、

　　といった要素が揃わなければならない。

　■これらの要素を満たしているか、
　　現在の、学びの姿勢を振り返る、

　　そんな機会を、ときには持ってみても
　　良いのではないか。
