　「鮒谷周史の肩書きは鮒谷周史」

　一つ目は「憧れの再設定」、
　二つ目は「知見の獲得」

　鮒谷さんは「平成進化論」というメールマガジンから連なる
　芸術的なビジネスモデルを構築されていますが、

　個別コンサルを受け始めたりしたのも、

　そのようなビジネスモデルを土台に人生を謳歌されている
　鮒谷さんのお姿に「憧れた」からです。


　その鮒谷さんと直接お会いする機会は、
　月に一度のたった一時間半ながら、
　
　明らかに僕の原動力となっており、
　むしろ「二週間に一回」でないのが悔しいくらいです。


　月に一度の「憧れの再設定」によって、
　行動力という器にたっぷりと水を注いでいただいています。

　もちろん鮒谷さんのビジネスモデルの土台には、
　膨大な勉強量・行動量・知識量があります。

　そのおすそ分けをしていただくことも個別コンサルの目的で、
　これが「知見の獲得」です。


　これまでに起こしてきた行動のいずれにしろ、

　鮒谷さんの知見や影響がなければ（自分一人では）、
　なかなか起こせるようなものではありません。


　「少しのことにも、先達はあらまほしきことなり」

　の言葉のごとく、
　24歳の何者でもなかった僕にとって、

　非常に幸運なことでした。


　全体をまとめれば、

　僕にとって鮒谷さんは人生全般の「師匠」であり、
　個別コンサルは月に一度の「掛かり稽古」、

　というイメージです。


　12回目となった今回のコンサルでは、初めて「事前資料なし」で
　稽古を付けていただきました。

　いきなり師匠に防具なしで
　挑みかかったようなものでしたが(汗)、

　動きは最初のころと比べれば格段に良くなっていたようで、
　僭越ながらお褒めの言葉をいただくことができました。


　しかし、まだ下半身の鍛錬が圧倒的に不足しているので(汗)、

　さらに重心を低く、かつ華麗なフットワークを身に付けるべく、
　引き続き精進を重ねてまいります。

■昔、


　「チャンスは一瞬だけ、
　　後悔は一生もの。

　　勇気は一瞬だけ、
　　得るものは一生もの。

　　考える前にアプローチすることが肝心。」


　という言葉を聞いたことがあります。



■あまりに共感したので、
　そのまま流してしまうのは勿体無く、

　手帳の一番目立つところに
　でかい字で記しておきました。
　

　その文字が潜在意識に入ってきたのか、
　それ以降の私は、


　「チャンスは一瞬だけ、
　　後悔は一生もの。

　　勇気は一瞬だけ、
　　得るものは一生もの。

　　考える前にアプローチすることが肝心。」


　ということで、

　「考える前に決断し、実行する（決断即実行）」

　姿勢が習慣化されました。

　この習慣を身につけられたことが
　人生を大きく転換させるきっかけとなったのでは、

　という気がしています。

　すでに、

　　後悔は一生もの。

　　得るものは一生もの。

　　考える前にアプローチすることが肝心。」

　今の自分（と、自分のビジネス）がある、

　ということ。



■四の五の考えるよりも、
　飛び込んだほうが早い、

　「案ずるより産むが易し」

　という世界があるのです。


　こうした世界観の中で生きるのか、
　その反対に、　

　「石橋を叩いて壊す」

　という世界があります。

　どちらの世界観の中で生きるのか、
　いずれも各人の自由です。



■ただ、個人的にはやっぱり、


　「チャンスは一瞬だけ、
　　後悔は一生もの。

　　勇気は一瞬だけ、
　　得るものは一生もの。

　　考える前にアプローチすることが肝心。」


　という認識で生きるほうが
　楽しいし、結果も出るに決まっている、
　
　なぜなら、決断も実践もしないのに
　何かが変わるはずなどあるわけないだろう、

　と考えているわけです。
