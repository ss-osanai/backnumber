 【  鮒谷が親に感謝している、最大のこと  】

───────────────────────────────────



■私（鮒谷）が親に感謝していることは
　たくさんありますが、とくに、
　
　「教育環境を与えてくれたこと」
　
　には、どれほど感謝してもしきれないくらい、
　感謝しています。

　
　幼少の頃の読み聞かせより始まって、
　
　中学受験に際しての、塾その他の費用、


　私立の中学・高校への学費、ならびに通学費用、
　大学受験のために学習塾の費用を負担し、

　さらに東京の私立大学に進学させてもらい、
　仕送りまでしてもらっていたこと。

　（挙句の果てに、
　　留年までしてしまったわけですが、、汗）


　
■それ以外の直接、間接の体験、
　あるいは書籍購入費用、

　そんなものまで含めると、

　長い期間に渡る莫大な有形・無形の、教育支援が、
　現在の私を形作ってくれていた、
　
　と思うからこそ、
　大きな感謝の念を持っているのです。
　
　
　
■同時に、

　大学卒業後には、
　自分が自身に対し、教育環境を与えてきた、

　そんな過去に対しても
　自分に感謝（？）しています。
　
　
　それは、
　
　親が幼少の頃から少年期、青年期にかけて
　教育環境を整えてくれたのと同様、
　

　社会人となって以降、
　自分自身に対して同じく、
　
　その時々に所有していた、限りある資源（お金や時間）
　を使って出来うる限りの最大限、
　
　
　「学び」や「経験」

　あるいは

　「人のつながり」
　
　（これらは広義の教育環境と言っていいでしょう）
　
　
　といったものを自らに提供してきたからです。
　
　
　
■その間、

　特段の贅沢もしなかったし、
　人並み以上の遊びもしなかったし、

　
　食べ歩き、飲み歩き、テレビ見放題、映画見放題、
　休日休暇は眠り放題、

　キャンプ三昧、ドライブ三昧、
　バーベキュー三昧、旅行三昧、
　
　キャバクラ三昧、合コン三昧、
　ゴルフ三昧、スポーツ観戦三昧、
　

　などなど、
　享楽の限りを尽くしてきた同年代の友人と比較すれば、
　
　まったくもって、干からびた、
　つまらない、華のない、単調な、

　そんな人生に見えていたかもしれません。

　（ちなみに別に、以上に挙げたようなものを
　　否定する話ではありません。程度問題です）
　
　
　
■しかし、人生（時間）においては、
　
　「あることを行えば、あることは行えない」
　
　というトレード・オフの関係にあるものも
　また事実で、

　
　もし私がその時代に遊び呆けていて、
　そこでの学びのいずれが欠落していても、
　
　（自分の履歴の中でも
　　過去最高に恵まれていると思っている）
　
　今の状態に至ることはできなかったのではないか、
　　
　と考えているのです。


　つまりは、

　「今、遊ぶのか、今、学ぶのか」
　
　その決定は本人に委ねられ、
　その結果もまた本人の責任に帰せられる、

　ということ。

　
　
■このように、
　
　「自らに教育を施す大切さ」
　
　が分かっているが故に、

　
　これからもますます積極的に、
　
　学びや経験、人のつながりといったものに対して
　投資をしていきたいと思うし、
　

　それらを行えば行うほど、
　どんどん楽に、そして楽しくなってくることが
　既に実証済みなのだから、
　
　躊躇なく、惜しみなく、
　投資をしていくことが可能になるのです。
　
　（それがまた、新たな未来を切り拓いてくれるでしょう）

　
　
■反対に、

　もし、ここで調子に乗ってタガを緩め、
　羽目をはずし、教育投資を怠ったらどうなるか、
　
　思考実験することもあります。
　
　おそらく半年、一年、あるいは場合によっては
　数年くらいは今の状況を維持できるかもしれません。
　
　
　しかし、おそらくは緩やかに

　「逓減カーブ」

　を描いて、元の木阿弥になってしまうことは
　想像に難くありません。
　
　
　
■今のような知識社会の時代にあって、
　
　学びをやめることはすなわち、
　成長を諦めることであり、

　それはまた、

　自らの生活環境が悪化していくことに
　つながるのも自明のことと思われます。

　
　
■このような経験と確信から、
　
　「自らに教育を施すことの大切さ」
　
　を説き続けてきましたし、
　
　自らも誰よりも実践しているつもりです。
　
　
　また、その成果を皆さんに見ていただくことによって、
　
　「教育の大切さ」
　
　を体を張ってお伝えしていきたい、
　　
　とも思っているのです。
