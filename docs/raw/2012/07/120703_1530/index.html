■こんにちは、鮒谷です。

　
　【今日のおすすめ漫画 おっとりした人が最後に勝つ成功法則】

　 http://amazon.co.jp/o/ASIN/4492557148/2ndstagejp-22/ref=nosim

■タイトルからイメージされるであろう？内容とは異なる、
　実地調査に基づいた、まじめな分析が記されています。

　自分が

　「ガンガン系」「ガッツリ系」「イケイケ系」

　ではない（というよりも、そういうタイプはむしろ苦手）

　という自覚のある方にとっては、
　大いに参考になるところがあるのではないでしょうか。

　（「おっとり系」の私としては、大いに共感いたしました）



■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　ネット巡回の功と罪　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。

＜今日のピークパフォーマンス方程式＞

 【　ネット巡回の功と罪　】

───────────────────────────────────



■昨日は、

　今のように放っておいてもネットを介して　
　次々と情報が入ってくるような時代にあっては、

　無自覚にクリックし続け、
　時間を垂れ流し続ける状況が起こりうる、

　というお話をいたしました。

　
　今日も続けます。



■私たちは、日々たくさんのメールを受け取ります。

　そのメールには、ULRが記載されていることが
　しばしばあります。


　もちろん、メール以外の情報源として、

　ニュースサイトやら、まとめサイトやら、
　動画サイトやら、ソーシャルネットワークやらの、

　たくさんのURLを

　「お気に入り登録」

　されている方もあるかもしれません。



■けれども、

　それは非常に危険なことではないか、
　と思われます。

　このようなサイトは一度アクセスしたら、
　中毒性があるので、

　サイトのトップページを基点として、
　ぐるぐると

　「巡回活動、巡視活動」

　を始めてしまいがち、なのです。（経験者談 汗）



■もちろん、その中には自分の仕事や日常生活に
　役立つ情報もたくさんあるに違いありません。

　けれども、

　「ある一つの有益な情報が得られたぞ！」

　と喜んでみても、その情報を入手する
　対価として、たとえば

　「２時間の巡回活動、巡視活動」

　を支払っていたとするならば、
　費用対効果としてどうなのだろう、
　
　という疑問を私などは持ってしまうのです。



■それよりも、日々、たくさんの情報に触れ、
　分析することを生業としている人が、

　徹底して時間をかけ、
　手をかけてくれた、

　書籍であったり、
　特定のサイトであったり、

　（一概にネット上の情報を否定している
　　わけではありません）
　
　のみを厳選して、参考情報とさせてもらった
　ほうがはるかに効率的ではなかろうか、
　
　と思われるのです。



■と、ここまで書くと、

　「無目的の（ネット）巡回」

　を否定している、と思われる向きもあるかも
　しれません。

　
　矛盾するようですが、私はそのような活動について、
　完全に否定しているわけでもありません。
　
　そういう無差別巡回による

　「出会い頭の一発」

　的な情報に触れることにより、
　人生が変わることも往々にしてあるからです。



■ですから、

　必ずしもこうするべきである、とか、
　こうでなければならない、

　というつもりは全くありません。

　上述の理由から、そのような発言に対する
　責任を取ることもできません。



■ただ、そのような前提をお含みおき頂いた上で、

　「ネット巡回、ネット徘徊」

　という行為について考えてみたとき、
　やはり一般的には、

　「功よりも罪のほうが大きいのでは」

　と思わずにおれないのです。



■だから私のPC、スマホには

　ニュースサイトやら、まとめサイトやら、
　動画サイトやら、ソーシャルネットワークやら、

　の「お気に入り」「ショートカット」は
　おいておりません。すべて削除しました。

　（あれば、見ちゃいますからね）

　※ただし残念ながら、iPhoneに予め入っているYouTubeの
　　アイコンは削除したくても仕様上できません。
　
　　仕方ないので、一番最後のページ（？）に
　　置いてます。



■正直に告白すると、その昔、
　たくさん入れていた時期もあるのですが、

　それらを思い切って、
　ばっさり捨て去ったとき

　「巨大な時間的余裕、精神的余裕」
　
　が生まれたことに、驚愕したのです。

　長くなりましたので、明日に続けます。

　

　今日も人生とビジネスを楽しみましょう！
