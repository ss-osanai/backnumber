6762号 改めて、災害時の備えは万全に

■メルマガを書き終えて
　いよいよこれから配信、

　というところで
　大きな揺れに見舞われました。

　急遽、原稿を差し替えて
　配信いたします。



■震源は東北であったとのこと、

　まずはお住まいの皆さまがご無事で
　いらっしゃることを願っております。


　今、暗闇の中で
　メルマガを書いていますが、
　
　現時点で関東圏では209万軒が
　停電とのこと。



■以前より様々な災害に対する
　実際にシミュレーションをしており、

　その中で照明の確保は
　疎かにできないということで、

　以前より、すぐに取り出せるところに

　▼ランタン
　▼懐中電灯
　▼ヘッドライト
　▼蓄電池ならびに携帯用充電池
　
　等を置いていたので
　メルマガを書けています。



■各種想定をしてはいましたが、
　本当の停電とはやっぱり違います。

　月明かり以外、なにもないので
　事前の予行演習とは全く違う状況。


　ただ、たとえば照明器具であれば、

　すべてGENTOS(ジェントス) 製品を
　取り揃えており、

　特にランタンとヘッドライトが
　思っていた以上に役に立っています。



■もし、準備されていない方があれば、

　そしてこの度の地震とは直接、関係のない
　ところにお住まいの方であっても、

　（むしろそういう方こそ）
　至急、購入されることをお勧めします。

　スマホの明かりだけだと
　不便ですし、なにより不安です。



■私はそれぞれについて最上位のものを
　複数揃えていたので、

　部屋がかなり明るくなり、行動に
　支障がない上に安心が得られています。

　全ての電気が消えている中での明かりは
　精神衛生に大きな影響を及ぼします。
　


■そしてヘッドライトが
　予想外に役立つことも分かりました。

　こうしてメルマガを書けるのも
　ヘッドライトがあるからです。



■ランタンは部屋全体は照らしますが
　全方向なので、

　指向性のある光源を持っておくと
　大変便利。

　もちろん懐中電灯も指向性がありますが、
　手で持つ必要があるので不便です。



■その点、ヘッドライトはバンドで
　頭を巻けば両手を使えるので、
　
　絶対に備えておきたいもの。

　余裕があればヘルメットもあると
　良いかもしれません。



■あと、これは余談ですが、
　保管時は電池は抜いておく、

　もしくは年に一度は
　電池を入れ替える、

　といったことを行っておくべきです。


　以前、数年ぶりにテストした時、

　電池が劣化して結晶（粉）が吹き出ていて
　使い物にならなくなっており、

　ランタンを捨てたことがありました。

　（予行演習時で良かったです）



■こうしたことを防ぐには、

　照明器具と電池を
　同じ場所に保管しておくか、

　リマインダー設定をして、定期的に
　日を決めて電池の入れ替えを行うか、

　をしておくことを強くお勧めします。


　いざという時に使い物にならないと
　分かることほど残念なことはありません。



■果たしてこの停電がどれだけ続くのか
　分かりませんが、

　もし、このまま長引くようであれば
　蓄電池、充電池、乾電池等の準備を、

　やり過ぎるくらいに
　やっておいて良かった、

　というのが偽らざる思いです。



■ランタンも節約すれば、

　単１電池３本で（私の持っているものなら）
　最大350時間持ちますが、

　その分、薄暗くしか照らしてくれず
　気持ちも萎えます。


　それが嫌だからと最大光量にすると
　７時間しか持ちません。

　冬だと寝ている時間は消していても
　一晩持つか持たないか、くらい。



■大災害時に、

　こうした状況が一週間とか続くとすれば、
　電池は21本準備しておく必要がありますし、
　
　ランタン×２ならその倍、
　ランタン×３ならその３倍、

　必要になります。


　なので、長期戦に備えたい場合は
　大容量蓄電池＆エネループを準備、

　という形でも良いかもしれません。
　（私はそうしています）



■あとこれも補足ですが
　移動時にも何が起きるかわからないので、

　私（鮒谷）は懐中電灯とヘッドライトと
　充電池（スマホ７回充電可能）、

　ならびにスマホ２台、

　（あと、カロリー補給の飴など）

　を必ず持ち歩いています。

　こちらもご参考になれば。



■311のときにも同じ備えをしており、

　（当時のメルマガでも書きましたが）、

　それでワンセグを通して状況を把握し、
　通信状況が不安定な中で後輩と連絡がつき、

　車で迎えに来てもらって
　翌日の午前２時くらいにはなりましたが、

　家に帰ることができました。
　

　たまたまこの日はセミナー参加のため、
　東京の海沿いの会場にいたのですが、

　電源切れていたら、もっと大変なことに
　なっていたはずです。
　


■今、たまたま私（鮒谷）がいるところは
　現状、停電で済んでいますが、

　これを機に改めて全てを見直し、
　さらなる改善、刷新をしたいと思います。


　
■もし今回の地震に直接の影響がない
　地域にお住まいの方もこの機会に、

　照明、電力、水、食料の事前備蓄、
　家具の固定などなど、

　防災対策により一層の準備をなさって
　いただければと思います。


　
■この話が役に立ったという日が
　
　（来ないことを願いますが）
　いつか来るかもしれません。


　もしまだ準備が足りていないと
　思われた方は、

　あとでやろう、ではなく、
　今すぐやって下さいね。


　
■その姿勢がいざというときの、
　場合によっては生死まで決することがある、

　（後悔を未然に防ぐことがある）　

　と思っています。



■繰り返しになりますが、

　震源周辺の被害状況について
　まだ詳細は分かっておりませんが、

　皆さまのご無事を
　祈念いたしております。


　今日も人生とビジネスを楽しみましょう！
