■戦後の混乱の中から立ち上がり、
　一大企業を作り上げられた方ですが、

　そこに至る過程には強い「気」が
　あったと思うんですよね。

　前向きな「気持ち」がなければ
　物事を進めていけません。



■私（鮒谷）は、

　公私に渡り、長くご指導頂いてきた
　弁護士の高井伸夫先生から

　「元気、覇気、活気、本気、人気、
　　勇気、根気、気は心」

　といった言葉を繰り返し語られ、

　「気を入れて生きなければならない」

　と教わりました。



■その教えは今でも生きており、
　時折（頻繁に？）緩むことはあっても、

　おかげさまで緩み切る、
　ということはなく、すぐに

　「気を張っている状態」

　に戻れるようにもなりました。



■仕事はもちろん、
　毎日の生活においても

　「気を入れて生きる」

　ことを心がけてきましたが、


　そんな（地味だけれども）
　実はとても大事な心構えについて、

　語っている音源、それが

■繰り返しになりますが、
　地味ですが大事なんですよ。

　ボクサーにとっての、

　ロードワークとか、
　縄跳びとか、
　シャドーボクシング、

　みたいなもの。



■スパーリングとか試合が
　楽しいんだと思いますが（多分）、

　それだけでは通用しないからこそ
　基本が重視されるわけで。



■あるいは相撲でいえば、

　四股とか鉄砲も地味だし、
　退屈そうだけれども、

　同じくそれやらないと
　強くならないんですよね。



■日々、「気を入れる」のは、

　ビジネスパーソンが
　安定的に成果を出すための

　【基礎トレーニング】

　だからです。



■それだけで何かができるように
　なるわけではありませんが、

　継続して訓練していなければ
　何もできるようにはなりません。

　それくらい大事なもの。


　気が抜けていたり、
　気を張っていなければ、

　何も身につかないし、
　成長もできません。



■ところで、

　「気を入れる」

　ためには、

　あらかじめ注入するための「気」を
　用意して置かなければなりません。



■具体的には、そもそも

　「元気、覇気、活気、本気、
　　勇気、根気」

　といったものが内在していなければ、

　「気を入れよ」

　と言われても

　「入れるための気」

　がないのだから、
　入れようがありません。



■つまり、

　元気や覇気がなければ、
　そもそも気を入れられない、

　ということ。



■また、

　「元気や覇気はあるけれども
　　気を入れ忘れる」

　これは結構あるものです。


　しかし全力投球していないことを
　自分で気づいているから、

　その姿勢が続くと、小さなダメージ
　（自尊心の欠如や、不完全燃焼感）

　が日々、蓄積されていくでしょう。



■つまり、

　【元気や覇気が溢れている人が、
　　日常においても気を入れての
　　行動や対応を心がけている】

　これがベストということに
　なるわけです。



■とはいえ、

　いつも元気である、
　この状態を作るのが大変な上に、

　さらに、必要な場面において、
　都度

　「気を入れる」

　という、ある種、気合いのようなものを
　発動させる必要があるから、

　簡単なことではありません。



■この２つの関門を突破して
　（＝自身を躾けられて）初めて、

　「仕事ができる」
　「成果を出せる」

　人物となれるのでしょう。


　元気も覇気もない上に、
　いつも気が抜けているし、
　気を入れようとも思っていない、

　そんな人が何かを実現できるはずも
　ありません。
