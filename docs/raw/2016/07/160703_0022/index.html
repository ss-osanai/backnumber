【 潜在的な悩みを解消するから、ビジネスが爆裂する 】
　　　　
───────────────────────────────────


■今日は私（鮒谷）が

　「営業」

　について考えていることについて
　記そうと思います。


　物を売る行為の裏側には、

　もちろん見込みのお客さまのニーズが
　存在するわけですが、

　まずはニーズを的確につかむことが
　必要です。



■ただし、

　ニーズを掴み、
　ニーズに対応するのがビジネス、

　とはいえ、

　顕在ニーズに対して、そのニーズを
　埋める手段を提供したところで、

　大したビジネスにはなりません。



■なぜならば、


　顕在化しているニーズは、

　既に、誰にも分かる形で見えている
　ニーズであるから、
　
　そこを目がけて、商品を投げ込めば
　一定の売上が立つ、


　そんな風に皆が皆、思い、大勢の人が
　そこに向かって突っ込んでいくのです。

　こうして、
　血で血を洗う戦いが繰り広げられ、

　みんな傷つき、
　倒れていくわけですが。



■そのような

　「分かりやすい（そして厳しい）」

　世界で戦いたくないのなら、


　ほとんど誰も気づいていない、
　したがって言語化もされておらず、
　
　それゆえお客さま自身、自分の内に
　そんなニーズがあることを知覚していない、

　そのようなニーズ（＝潜在ニーズ）を
　掘り起こす必要があるのです。



■お客さまの立場に立ち、
　お客さまも気づいていなかったような

　「声なき声」

　を汲み上げ、言語化して
　お渡しすることができれば、

　「私が欲しかったのはこれだった（！）」

　という反応を示してもらうことが
　できるでしょう。



■そこにおいて初めて、顧客の側に

　「その商品、あるいはサービスを
　　どうしても手に入れたい」

　という欲望が頭をもたげてくることと
　なるわけです。



■その状態がさらに強化されると、

　（実際の購買が起きたかどうかに
　　かかわらず）

　「意識の上では、
　　もう買ってしまっている」

　という

　「お前はもう、買っている」

　という北斗の拳営業に入るでしょう。


　
■ここまでは理論編ですが、

　では、たとえば私（鮒谷）であれば
　どのように実践で行っているかというと、

　一つ例を挙げると、


　これまでにコンサルやセミナー、
　会食その他を通して、

　大勢の方とお会いする中で、
　少なくない方が

　「本当のところ、仕事が楽しくない」

　という悩みを抱えていらっしゃることに
　気づきました。



■そして、

　（口では、ワークライフバランスだとか
　　なんだとか、いろいろ言われていても）

　心の奥底では、

　「プロジェクトＸ」

　ばりに真剣に、仕事に取り組みたい、
　魂を燃やしたい、

　何か一つのことに没頭没入したい、

　
　、、、なのに実現する方法を持たない、
　なにから始めたらいいかも分からない、

　そんな人が少なくない
　（＝巨大な潜在ニーズ）

　があることに気づきました。


　
■このニーズについて
　さらに深く考えると、

　少なくない人にとっての
　悩みの根源であるところの

　「仕事が楽しくない」

　という思いそのものを
　解決するソリューションを提供できれば、

　文字通り、

　「巨大なマーケットを開拓できる」

　と思ったのです。



■実際のところ、


　個人的な感覚としては、
　仕事ほど楽しいものはないし、

　どれだけ没頭しても飽きないし、

　私（鮒谷）はその世界について
　いくらだった語れるし、

　その世界への到達の仕方、させ方、
　も知っている。
　
　（なぜなら自分が通ってきた道だから）



■ならば、私（鮒谷）にこそ、

　「仕事に打ち込みたいのに打ち込めない、
　　という潜在的な悩み」

　を解決する方策を提供する資格が
　あるのではないか、


　そんな風に考え、
　この潜在ニーズを解決するソリューション

　（たとえばセミナーであり、
　　たとえば個別コンサルであり）

　を提供すれば、
　爆裂するのではないか、
　
　との仮説を立て、検証したら、


　、、、本当に爆裂した、

　ということです。



■顕在ニーズではなく、
　潜在ニーズへのアプローチであるから、


　「他に提供する人がありそうだけど、
　　なかなか見つからない」

　「いないわけではないはずだけど、
　　目に入る範囲には鮒谷しかいない」


　という真空状態（ブルーオーシャン）が
　生まれるわけであり、

　そこから

　「売らなくても売れていく」

　あるいは

　「継続的に受注が生まれる」

　といった現象が起きることに
　なるわけです。
　　


■これはほんの一例に過ぎませんが、

　一つ一つの潜在ニーズを
　丁寧に掘り起こし、
　
　その解決策を提供していけば、
　自ずからモノ（サービス）は売れていく。


　こうしたことを考えては（仮説を立てては）
　実践する（検証する）、

　これを繰り返すうちに
　ビジネスはどんどん強化されるでしょう。

　
　私（鮒谷）が

　「営業とは究極の知的遊戯」

　と考える所以でもあります。
