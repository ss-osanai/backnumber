■ビジネスパーソンにとって重要なスキルはいくつもありますが、
  中でもこの「数字力」は筆頭にあげられることが少なくないようです。
  
  このところ様々なところで言われていますので、見聞きされた方も
  多くいらっしゃることでしょう。

  会計の知識が身につく構成になっているとのこと。
  
  一話あたり15分ほどで読めてしまうとのことですから、通勤途中などの
  ちょっとした時間に取り組めそうですね。

■できるビジネスパーソンほど忙しく、数字力の大切さも分かって
  いるけれどなかなか勉強する時間が取れない…。

■年間で定期購読している、2008年7月号の「日経ベンチャー誌」が
　先日、届きました。


　今回の号の特集は


 【　「まね」から始める経営革新　】


　ということで、


　▼酒巻久キヤノン電子社長　「コピーは飛躍に通じる」

　▼正垣泰彦サイゼリヤ社長　「洞察は儲けに通じる」

　
　などといった巻頭の記事がありました。



■またそれ以外にも


　▼異業種を応用する

　▼勝ち組をマークする

　▼名経営者になりきる

　
　といった非常に興味深い記事が掲載されています。



■これは私自身も経験がありますが、

　早く成功を手にしようと思ったならば、
　うまくいっている人から学ぶ（模倣する）のが一番早いです。

　
　新しいものを創造することが成功の秘訣、といわれる
　こともいわれたりすることもありますが、

　この世の中でそんなに新しいことが次から次へと
　創造できるものではありません。



■してみると、

　あなたが経営者として、あるいはビジネスパーソンとして
　さらなる活躍を求めようと思うのであれば、

　

　【 すでに成功している人の思考のエッセンスなり、ノウハウなり
　
   　　仕事術なりを、既存の自分のものに付加することで加速度をつける 】



　というのがもっとも素早く成果を出すための秘訣といえるのでは
　ないでしょうか。



■では、どこでそれを学んだらいいのか、というご質問が
　聞こえてきそうですが、
