　-----------------------


　私たちはどんな感情で満たされたいのか
　想像する必要があるとともに、

　その感情をもたらすのに要求される
　（具体的な）資源があるわけですから、

　そのリソースを未来の自分に対して
　贈りたい。


　-----------------------


　求めている感情を未来に手に入れたい

　（今の私が、将来の私に
　　期待される感情を贈りたい）

　わけですけれども、


　ある感情を未来を贈るといっても、

　感情そのものを
　プレゼントすることはできない。


　-----------------------


　それゆえ、

　ある感情を味わうために必要な資源を
　特定し、それを送る必要がある。


　つまり

　「Being Doing Having」

　を一つのセットして
　未来に贈りたいということ。


　「Being Doing Having」のセットを
　将来の自分にプレゼントする、

　そんな意識をしてみてはいかがでしょう、

　今日は、以上のような提案を
　これからしていきたいということ。


　-----------------------


　私（鮒谷）がいつも言う言葉に

　「人生に補助線を引く」

　言葉がありますけれども、


　未来に対して積んでいきましょう、
　というのも確かに補助線ではあるが、

　その補助線がぼやけているというか、
　ちょっと切れ味が悪い部分もあるように、

　最近、感じている部分があり、


　「未来の自分へのプレゼント」

　と呼んだ方が明確になるし、
　臨場感も沸いてくるし、

　イメージがつきやすいのではないか、
　というところで、

　今日はこうした補助線を
　まず冒頭でお渡しさせていただきました。


　（ここまで、まだまだ
　　音源の入口部分に過ぎません）


　-----------------------

　
　どんな感情が欲しいかを意識すると、

　それに対して必要な要素っていうのが
　浮かび上がってくる。

　だからBeing目標は 
　Being目標だけで完結しないという話を、

　ピラミッド音源でしつこくお話しした。


　-----------------------

　
　「目標設定音源」の内容は、

　非常に良かったと、皆さん
　おっしゃってくださいます。

　ところが、その後出てくる質問で
　よくあったのは、さはさりながら、

　スイッチは入ったけれども、
　日常がなかなか変わっていかない、

　と。


　意識だけは飛んでいるけれども、
　日常の行動は変わらない、と。


　-----------------------


　それを補うというのか、
　意識と行動を架橋するという意味合いで、

　時々刻々の記、つまり
　日記放談会音源で語ったことであったり、

　物語放談会音源というものも　
　お届けしてこれで事足れり、

　ということで完結したのかと思いきや、


　特に個別コンサルなどでお話を伺う中で
　なんか、俺が思っているのと違うよな、

　ということが結構あったわけ。

　まさにその欠如していた部分が
　ピラミッとして音源に結実した、

　ということになるわけです。


　-----------------------


　上のbeingの目標、

　すなわちこいつは状態であったり、
　そこから生み出される感情、

　という話ですけれども、

　そこを満たそうと思ったら当然、

　Doing、Havingはピラミッドの、

　逆三角形のピラミッドって、
　ありえなくて、


　基本的にピラミッドは当たり前ですけど、
　三角形というか、
　四角錐みたいな感じになるっていうか、

　「錐」ですから、上が小さくて、
　下が土台としての広がりが出てくる。


　だから、常に上に、
　どんな感情を欲しいかがあり、

　下にそれを満たすためには
　何が必要かという構造となる。


　-----------------------

　
　もう僕（鮒谷）なんか、
　いっぱい常に回ってるというか、

　ほとんど無意識的有能の領域だと
　思うんですけれども、

　基本的につねにこうしたことを
　考えてるわけですよ。

　日々、こうしたことを考えているから
　やることがたくさん出てくる。


　-----------------------

　
　可処分時間に比して
　やることの方が圧倒的に多いわけ。

　そのウェイティングリスト、
　それはそのまま第二領域。


　第二領域、
　つまり緊急ではないけれど、

　自分の人生、自分の未来に対して
　重要なことであり、


　自分の未来をより良くする、
　さらに正確に言うならば、

　将来、より豊かな感情、
　より幸福感を感じさせてもらえる、


　そんな状態に入るためには、

　これがあった方がいいよね、
　あれもあったほうがいいよね、

　こんなリソース（資源）があった方が
　いいよねという風になってくるから、

　それを獲得しようと思って、
　具体的な行動につながっていく、

　ということです。


　-----------------------


　なのでまず、改めて、
　どんな感情が欲しいんだ、ということを、

　今日はより鮮明にしてもらいたいと思って、

　先に挙げた2、30ぐらいの感情は
　あくまで一つの例ですが、

　を挙げさせて頂きました。


　-----------------------


　それを聴くことによって

　「あ、この感情は欠けてかけておったな」

　と気づき、


　味わいたい感情の
　レパートリーリストに追加し、

　さらにそれを実現するための資源を
　一つ、一つ積みましていくと、

　より幸せな気持ちに
　なれるんじゃないかな、

　そんな問題提起を今日はしたいのです。


　-----------------------


　その豊かな感情のレパートリーを
　言い表す言葉をたくさん持てると良いですが、

　これを逆にしてしまうと、
　私たちは乏しく、貧しくなるわけです。


　-----------------------


　どの感情にフォーカスを当てるのか。

　自分が得たい感情にフォーカスするのか、
　望ましくない、好ましくない感情に
　フォーカスするのかは、

　どこまでいっても各人の自由。


　-----------------------


　明確なBeing目標が
　フォーカスを当てるべき感情を規定し、

　それが定まることによって
　どんな感情を得たいのか、

　さらにそのために私は今、
　何をすることによって、

　未来の私にどんなプレゼントを贈るのか、
　まで規定されてくる。


　-----------------------


　将来、私がどんなプレゼント
　（感情と、それを満たす各種資源）を、

　今の自分から手渡されると喜ぶか、

　リアルに想像すればするほど
　欲しくて欲しくてたまらなくなる。


　-----------------------


　そこまでいけば

　「欲しいものは、欲しい」

　わけだから当事者意識が生まれて、


　求める感情（＆それを支える資源）を
　手に入れるために

　「何を、どの順番で、どのようにして、
　　手に入れようか」

　という問い（自問自答）が
　生まれることとなる。

　
　-----------------------

　
　言葉を換えれば

　感情がないから、
　感情が乏しいから、
　感情の振れ幅が小さいから、

　当事者意識が生まれてこないし、
　行動もしないし、


　それゆえ

　ある感情を手に入れるために必要な
　各種資源（具体的なDoing、Having）

　を求める気持ちも湧いてこないし、
　なにかが実現されることもない。


　つまり、

　いつまで経っても
　欲しい感情を手に入れられない、

　ということ。

　そもそも、
　欲しい感情を明確にしていないんだから

　「そりゃ、そうなるよね」

　という話ではあるんだけれども、、、
