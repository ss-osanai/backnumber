■今日も

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------
　　
　商売とはつまるところ、

　血と汗と涙の結晶（カネ）を張り、
　高利回りを追求する営み。


　事業を始めると、
　じっとしとるだけでカネが出ていくから、

　毎日「身を切る」痛みを
　味わうこととなる。


　それゆえ真剣にならざるを得ず、
　その当事者意識が大きなリターンを
　生み出すのよ。

　リスクを取るところにリターンあり。

　----------

　「儲け話」の95%は怪しい話やで。

　とはいえ

　「リスクを限定しつつ、
　　勝ちが見込める投資案件」

　は、きちんと探せばゼロじゃない。


　注意すべきは二つ。

　一つは信頼できる人間関係からの
　案件を精査する。

　もう一つは、いきなり大勝負をせず、
　素早く小さくテストを重ねる。

　騙されずに、金鉱脈を掘り当てよう。

　----------

　「意思決定しないこと」が
　最大のリスクやで。

　結果として間違った決断だったとしても、
　事前に損失を限定しとけば死ぬことはないし、

　失敗経験から
　勝ち筋が見えることすらある。


　でも決断を避け続けると、
　間違いもない代わりに成功もない。

　現状維持は後退やから、
　ジリ貧状態が続くこととなるんだわ。

　----------

　商売人として長年生き残っとる人は、

　「死なばもろとも」
　「乾坤一擲」

　なんて言わへんな。


　むしろ、めったに起きんが
　起きたら致命傷になる

　ブラック・スワン的なリスクまで
　想定して備えてる。


　そんな生き延びる態勢を整えた上で、
　収益を上げている。

　潔さも華々しさもないけど、
　これが正しい生存戦略やで。

　----------

　「川の深さが
　　平均4フィート（約120センチ）なら
　　渡ってはならん」

　いう、『ブラック・スワン』著者の
　タレブの教えを大切にしとるのよ。


　あくまで「平均」4フィートやから

　足のつかん場所があり、
　溺れる可能性もある、いうこと。

　統計学的楽観に基づき生きてたら、
　足元をすくわれることがあるから要注意。

　----------

　「ただ生きてるだけ」
　と言ってる人でさえ、

　それ自体が大きな投資を
　行っているのと同じなのよ。

　気づいてないだけで、
　皆さまざまな対象にベットしとる、

　いうこと。


　無自覚なまま、

　ヘッジなしで危険な投資を
　継続している状態が危ないのよ。

　リスクゼロにはできんが、

　冗長性確保（多重化）で
　即破滅は防げる。

　----------

　収入源は複数あった方がええで。
　今みたいな不確実な時代には特に。

　ただし中途半端な取り組みは禁物。


　まず一本の柱をしっかり育てる。

　それが安定したら、
　次の柱を立て始める。


　このルールを守りながら、
　収益源を一つづつ着実に
　増やしていくことよ。

　高収入を誇ってても
　一本足打法は危ないで。

　----------

　うちの会社は潰そうと思っても
　絶対に潰れんのよ。

　固定費極小、売り上げ＝ほぼ利益、
　のビジネス構造やから。

　その利益を原資に、

　松下幸之助のダム経営をモデルとして
　潤沢な内部留保を積み上げてきた。

　もちろん無借金。

　向こう10年、20年
　売り上げゼロでも潰れんから、

　精神的・経済的な安定感は半端ない。

　----------

　ワイは「クマムシ経営」を標榜して、
　長年の間、商売してきたよ。

　ナショジオのサイトに

　「水生の無脊椎動物であるクマムシは
　　地球上で最もたくましい動物と
　　考えられている」

　と書いてあったのに着想を得てのこと。

　確実に利益が出る上に、

　潰そうと思っても
　絶対に潰れん会社は無敵やで。

　----------

　会社をデカくするのも、

　絶対潰れん会社を作って
　高等遊民生活を過ごすのも、

　それぞれの価値観。


　他人の意見を参考にしつつ
　自問自答を重ねて、

　自分にとっての
　幸福の在り処（ありか）を
　見つけるといい。

　こんな会社を作る、と決めたら
　後は自動的にその方向に進んでいくよ。

　----------

　ワイは起業して最初期の頃から

　「潰そうと思っても
　　絶対に潰れん会社」

　「時の試練に耐えうる会社」

　「小回りの利く会社」

　「商売の発展が幸福や自由を
　　阻害しない会社」

　を作ると決めてたよ。

　ゴールを決めさえすれば、
　実現に向けて動き始めて形になる。

　目指す世界を明確にしてないと
　どこにもたどり着けんよ。

　-------------
