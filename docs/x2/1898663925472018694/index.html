「人生で起こるすべての結果を、丸ごと引き受ける」つもりで生き

2025/03/09 18:15

「人生で起こるすべての結果を、丸ごと引き受ける」つもりで生きている人と、「自分の人生がこうなったのは○○のせい」と考えている人とは永遠に交われない。

前者は「今、どんな状態にあっても試行錯誤を重ねてなんとかしよう」と思う。けれども、後者は「今、こんな状況にあるのは誰それのせい」であり、「より良い状態になるためには、誰かがなんとかしてくれる必要がある」と考える。

これは「習性、習慣」の問題であり、一事が万事、すべてにおける姿勢と言って良さそうです。結論からいえば、満足のいく人生を過ごせる可能性の高いのは前者（自責）の人。

どう考えても眼前の不幸やトラブルが「（自分ではなく）他の要因」であったとしても、原因を突き止め、追求しても、腹を立てたところでどうなるものでもない事象については、コントロールできることのみにフォーカスするのが合理的な判断であるから。

問題発生を契機として、それ以前よりも良い状態に保っていく気持ちで臨む人は人生が豊かになっていく。

逆に相手を責め立てたところで状況がもとに戻らないのなら、時間と精神の無駄遣いにしかならんよね。

なのに、怒りや恨みや悲しみで理性的な意思決定ができなくなり、ネガティブな感情で満たされたら行動もネガティブなものとなる。

ネガティブな行動はさらなるネガティブな結果と感情をもたらし、自分で自分の首を絞めることとなるから、やがてその解釈で作り上げた世界から逃れられなくなる。 　 自責も人生、他責も人生。それぞれの思いもあるでしょう。しかし道徳や倫理を抜きにして、純粋に「得か、損か」だけ考えても、どうしたって「自責の人」のほうが、さまざまな面で豊かになっていくだろうな、と思われる。

いつも自分の不幸にフォーカスし、他者の責任追及ばかりして怒り、呪い、妬み、悲しみ、苦しんで、他人の振る舞いに振り回されるほど、バカバカしいことはありません。接している人も離れていくでしょう。

ところで人によっては、自分をこんな目に合わせたあいつに痛い目に遭わせてやろう、復讐してやろうとか考える人もあるみたいだけど、この人は「目先の感情を解消するために、もっと大切な何かを失っている」のよね。

また、どうでもいい相手について思い煩えば、自分も相手と同じレベルに落ちている、ということにもなるからバカバカしいと思わんか？とも問いかけたい。

こうしたマイナス感情は、ただただ「それをプラスのエネルギーに転換して、飛躍の燃料とする」ことにのみ用いるべきであり、これ以上、エネルギー効率の高い燃料もないのだから、うまく使えばいいだけの話。

せっかくの良質の燃料があるのに、「無駄な使い方（他責の精神により、ネガティブ感情に満たされる）」のは、もったいない。そうしたエネルギーを上手に操れてこそ、人生も上手に渡れるようになるもの。

冒頭に記したように「自責の人と他責の人は、深いところでは永遠に交われない」のだとするならば、少なくとも私は、自らが「自責の人」となり、同種の人と、仲良く交わりたいと思って生きてきたんよね。

そしてその姿勢が、豊かさをもたらしてくれたのは間違いない。
