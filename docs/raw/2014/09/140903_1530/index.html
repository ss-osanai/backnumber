 【  生存戦略としての文章執筆力  】

───────────────────────────────────



■ビジネスに携わる人にとって、

　「文章を書くことができる」

　ことは相当な強みになります。


　イメージでいえば、

　「100メートル競争で、
　　50メートルくらいハンデをもらっている」
　
　ようなもの、あるいは、

　「競馬で、第４コーナーからとはいわないまでも、
　　第３コーナーからスタートさせてもらう」
　
　ようなもの、

　といっても言い過ぎではないかもしれません。



■特に、これからの時代、

　「一億総情報発信者時代」

　が到来するわけですから。

　（というよりも、もうすでに到来しているの
　　かもしれません）


　自分の存在を外部の人の目に留めてもらい、
　惹き続けることができなければ、

　あっという間に、

　周囲の人の記憶の奥深くに沈殿していって
　しまうのです。



■そんな時代に生きていることが
　幸せなのかどうか、良いか悪いかはともかく、

　現状、そのような状況にあるということは

　「客観的な認識」

　として持っておく必要があるのかな、

　そんな風に、個人的には思っております。



■「一億総情報発信者時代」が来た、
　ということは、

　ますます情報の流通量が増えていくことになり、

　その中で、

　日々、目に触れる情報の１％、あるいは
　せいぜい数％の中に入れてもらうための、


　１、触れた人の目を留めさせる力強い概念提示力

　２、そうして目に留めてくれた人を惹き続けられる力

　　　（「継続した情報発信力」「継続発信習慣」といっても
　　　　良いかもしれません）


　を持っていれば、

　ビジネスにおいて、
　成果を出すことが約束されている、

　といっても良いのかもしれません。



■といいながらも、

　「言うは易く、行うは難し」

　で、簡単にそのような文章が
　書けるわけではありません。


　しかしながら、これも、

　「訓練によって鍛えられる力」

　であり、かつ、
　
　陳腐化せず、漆塗りのように、

　「重ねれば重ねるほど美しく、
　　価値ある力と光り輝いてくれる」

　ものであるわけですから、

　今から取り組み始めても、
　遅いということはありません。



■実のところ、

　「いちいち言葉に出して伝えなくても、

　　自分の思いを分かってくれる人にだけ、
　　分かってもらえればいい」

　という考え方も一理あると思われます。


　というよりも、
　
　個人的にはむしろ、
　そんな世界のほうが良いとすら思っています。



■しかし、リアリストとして冷めた目で捉えると、
　
　そういうスタンスだと、
　今の世にあっては、

　どんどん、

　「忘却」

　あるいは

　「駆逐」

　されていってしまうわけですから、
　
　
　その中、

　「俺が私が、的な強烈な自己主張」
　
　は控えめにしつつも、

　しかしながら、常に微弱な電流を流し、
　読み手・聞き手の脳髄を刺激し続ける形で、

　「私はここにいますよーーー」

　という存在アピールをすることは、
　やはりどうしても必要ではないのか、

　という風にも思えるのです。



■繰り返しますが、

　別にそのようなやり方を全肯定している
　わけではありません。


　むしろ、心情的には、

　「良いものさえ作っていれば、
　　分かる人には分かる」

　という職人的心情のほうが深く理解できるし、
　共感も覚えるものです。



■しかしながら、
　そういう職人さん的、控えめな方々が、次々と

　（先に使った表現でいうならば）
　
　「忘却」「駆逐」されていく時代、
　事ここに至っては、

　「発信する力」

　もまた、身につけておく必要があるのではないか、
　
　と思えるのです。
