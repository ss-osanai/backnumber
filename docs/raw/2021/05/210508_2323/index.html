　（ここから）
　-----------------------


　「背中で語る」とは何かというと
　
　当然アウトプットを
　見られるわけですから、

　アウトプットを行っている
　外形的な姿もそうですし、

　かつ、もちろん外形的な姿を
　生み出すためのパラダイムであったりとか、
　
　意思決定の基準であったりとかの
　内的なものも含め、

　そこに独自性を生み出せ時に、
　それを一流と言っていいのではないか、

　と考えています。


　-----------------------


　もちろん、思い描くだけでなく、
　その世界を目指して頑張るのです。

　ポイントは、よく言われることですが、
　型を作って型を破ること。

　でたらめにやっても形は作れないのです。


　独自の型といえど、
　ベースとなる型は必ずあります。


　ただし、

　ベースとなる型は
　必ずしも単一ではなく、

　さまざまな種類の、結果を出せる、
　いくつもの型（雛形）があると考えています。


　-----------------------


　だから、

　どのタイプの原型を選び取り、

　さらに、速やかにそれを破り、
　離れられるか、

　というようなイメージを持たれると
　学習効率は格段に向上するはずです。


　-----------------------


　ただし、どういう形を選び取るのかと
　いう点において、

　自分の合う型、合わない型が
　どうしてもあるわけです。


　ある人が驚くほど大きな結果を出していて、
　そういう結果を出したいと思ったとしても、

　深いところで決定的に
　自分とタイプが違うと、

　これは模倣対象にはなりません。
　
　あるいは模倣対象にしては
　いけないということ。

　その型を模倣するのは
　無理があるからです。


　-----------------------


　（中略）


　もう一つは、

　似たタイプでありつつも、
　人間だれしも好き嫌いがありますから、

　自分に似ていて、プロトタイプ（原型）の
　相性も良さそうなので、

　その型をインストールしようと思っても、

　この人は嫌だと思っていたら、
　嫌いな人からは残念ながら人間は学べません。


　-----------------------


　確かに理想論としては、

　自分の感情と離れて学びましょう、
　という考え方は存在します。


　でも、ある部分において、

　最低限、妄信とか盲目的に大好き、といった
　ところまではいかなくても構いませんが、

　最低限、一定の好感とか
　一定のリスペクトとかがなければ、

　やっぱり学びの歩留まりは絶対に悪くなるし、
　斜に構えながら勉強しようと思っても、

　頭に入らないものです（経験談）。


　-----------------------


　だから、師を選ぶ際には、

　「似たタイプか否か＆好き嫌い」

　の二軸で捉えると良いのではないかと
　考えています。


　-----------------------


　一番良いのは、

　自分と似たタイプで結果を出していて
　そのシステムが確立されていて、

　さらに、その人のことが好きだと思える人です。

　（その上、言語化の得意なタイプだと
　　最高の学び対象となりますね）


　-----------------------

　　
　相性の合うタイプだが、

　本能的に忌避してしまい、
　リスペクト出来ない、

　という場合は、

　少なくともシステム全体を
　学ぶ対象としては不適切。


　仮に学ぼうとしても、
　学びの効率は悪いでしょう。

　（だから最初から止めたほうがいい）


　-----------------------


　また

　「自分とタイプは異なるけれど好き」

　というパターンもありますね。


　似ていないけれど、
　模倣しようと思うものの、

　そもそもタイプが似ていないため、
　模倣できないということがあります。


　これは人付き合いとしては
　もちろんありだけれども、

　中核システムのインストール、
　という意味での対象としては不適格。


　最後に、

　「タイプも違うし、
　　そもそも好きになれないタイプの人」

　これは範疇に入りません。
　×ですね。


　-----------------------


　いつもお伝えしていますが、

　自伝を読んでいると、

　全模倣、あるいは、
　この人から相当影響を受けた、

　という人が出てこない自伝は
　見たことがありません。


　-----------------------


　短めのものであれば、そうした箇所が
　端折られていることはありますが、

　ある程度のボリュームで
　文章で書かれているものにおいては、

　背中を見せてくれた人がいなかったので
　勝手に自ら編み出しました、

　という人は、
　全くと言っていいほど見かけません。


　必ず、誰かしらの影響を受けているのです。
　もちろんその影響の受け方に濃淡はありますが。


　-----------------------


　一切、他者から影響を受けていないと
　いう人はいません。


　影響を受けた人と
　疎遠になることがあったとしても、

　他者との関わりが全くなしに
　自分で型のないところから、

　自分がオリジナルの型を作りました、
　
　ということは誰もやっていませんし、
　私も一人も見たことがありません。


　-----------------------

　
　これらを踏まえると、

　やはり学びというのは
　インプリンティング、

　すなわち、初めて見た動いてるものが
　親鳥であると思うといった、

　刷り込みの話ではありませんが

　基本的に、学びに
　インプリンティング的要素があるとするならば、

　あるいは最近よく言われるように
　ミラーニューロンの話にも関係すると思いますが、

　やはり、見ているものの真似をする、模倣する、
　こういう話も関係してくるのかなと考えています。


　-----------------------


　見ることによって、さらに
　模倣することによってインストールされる、

　ということは明らかにある。


　ここまでの話は、
　読めば大した話ではなく、

　本当は「ほーん」という感じや、
　「そうかいな」って話ではないかと思いますが、

　これはあらためて思うに、
　実は凄い重要な補助線です。


　-----------------------


　何を学ぶにおいても、

　こういった補助線を引けば、学びという
　行為を捉え直せるようになりますが、


　こうした言葉を持たなければ、
　
　学びに対しての認識は、おそらく、
　モジュール的なものであったり、

　あるいは、色々なところから寄せ集め、
　無理やり整合性を取らせよう、

　などと思ったりされるのではないでしょうか。

　（そうして成果を出せずに長年、
　　苦しんできた人をたくさん見てきました）


　-----------------------


　こうした補助線なしに学びを進めると、

　場合によっては、自己の信念と
　意思決定基準の股裂き状態が起こったりします。


　そうやって成果が出ずに
　時間ばかりかかって苦しむのであれば、

　（ある人の）成果を出すための全体システムは
　膨大な思考と行動の集積から生まれたものなので、

　そちらを学習することを優先して
　資源を投下してみてはいかがでしょうか。


　-----------------------


　（中略）

　こうした補助線を引いた上でであれば、

　焦ることもなく、重心低く落ち着いて
　学ぶを勧めていくことができるもの。


　-----------------------


　ある人のスタイル（システム）というのは、
　さまざまなサブシステムの集積です。


　たとえば、その人なりの
　目標設定の仕方かもしれませんし、

　人との関わり方かもしれませんし、
　時間の使い方かもしれないし、

　家族との関わり方とか、
　学び方とか、仕事の進め方、

　いずれもいずれも人によって全然違いますが、


　そうしたサブシステム、あるい
　はサブシステムをさらにその下に支えるような、

　（骨組みや幹ではありませんが）
　大きな枝もあれば、小さな枝もあれば、
　葉っぱもありという形になっていますから、

　サブシステムを構成する
　サブサブシステムも色々と存在しています。


　-----------------------


　この人は、

　という自分に似たタイプ
　＆ある程度一定の基準、

　その部分においては
　最低リスペクトできる、好感度が高いとか、

　好きになれそうだ、という人を見つけて、
　一旦、幹と大きな枝だけ入れましょう。


　最低限のシステムが入ると、

　あとはさまざまな行動をする際に、
　是々非々で選び取れるようになります。


　自分で主体的に
　この（サブ）システムは取り入れようとか、

　このやり方は取り入れないとか、

　つまり、原型があるから思考ができ、
　取り入れたり、排除とまでは言わないですが、

　拒絶、却下したりする、
　ということができるようになるのです。


　例えば、色々な学びの対象がありますが、
　中には当然、矛盾も生じてきます。


　（ここからが一番、面白く、
　　価値ある話に入っていくわけですが、

　　残念ながら
　　ここでタイムアップとなりました、、、

　　続きは当音源にてお楽しみ下さい）
