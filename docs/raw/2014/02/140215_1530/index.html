 【　人生を台無しにしてしまう「感情暴発系の人」とならないために　】

───────────────────────────────────



■世に「感情暴発系の人」があるようです。
　
　
　「感情暴発系の人」とは、
　
　すぐにキレる、吠える、泣く、
　
　といった人たちのこと。
　
　
　
■感情を制御する力の弱い人、と言っても
　良いかもしれません。
　
　
　などとえらそうなことを書いておりますが、
　
　現実には、自らの感情を完璧にコントロールするのは
　大変に難しいもの。
　
　
　というよりも、絶対に不可能だと思っています。
　
　
　
■これをやってしまったら絶対にダメ
　
　（自分が損する、甚だしきに至っては、
　　全てを灰燼に帰させてしまう）
　
　と分かっていて、
　なお起こしてしまう可能性のあることが、
　
　この
　
　「感情暴発」
　
　なのです。
　
　
　
■けれども、

　人生においてこの回数をいかに減じられるか、
　あるいは皆無に近いレベルにまで持っていけるかは、
　

　「人生戦略（人生におけるリスクマネジメント）」


　において、たいへん大きなテーマとして、
　
　ときに俎上に上げてみても良いのではないかと思い、
　本日のテーマに持ってきた次第。
　
　
　
■ところで、

　感情の制御が上手な人と下手な人というのは
　実際にあると思います。
　

　（私自身、長年、あの手この手で、暴発しようとする
　　感情を制御する術を学び、躾けてきたつもりであり、
　　
　　感情をコントロールするのは上手ではないけれども
　　極端に下手でもないつもり）
　
　
　
■そんな私が、

　では具体的にどのように感情変化に対応しているのか、

　というと、以下の通り。


　自分の中で心がけているのは、
　
　感情が大きく揺さぶられる出来事に遭遇したときには、
　とりあえずその全てとはいわないまでも、
　
　できる限り

　「感情の沸点」

　を越えさせないように、
　
　
　▼とりあえず深呼吸したり
　

　▼リラックス、リラックスと心のなかで呟き、
　　リラックスすることを意識してみたり

　
　▼（何がありがたいかが自分でも分かっていなくても
　　　なにはともあれ）
　　
　　ありがたい、ありがたい
　　
　　と唱え続ける
　　
　
　ようにしています。
　
　
　
■このようにして、


　１、怒りや不満や悲しみの対象から目をそらす
　　　（凝視しない）ようにする
　
　
　次に、
　

　２、その出来事が自身にとっての価値となるような意義を
　　　見出すようにする
　
　
　具体的には、
　
　
　「ありがたいなあ、鍛えてもらっているんだ」
　
　「ありがたいなあ、訓練の機会を与えてもらっているんだ」
　
　「ありがたいなあ、自らを躾ける機会を頂戴しているんだ」
　
　「ありがたいなあ、自分が悪いことに気付かせてもらった」
　
　「ありがたいなあ、難度の高い状況のさばき方を身につける
　　実践練習をさせてもらっている」

　
　などなど。
　
　
　つまりは、
　
　「全てを成長の機会と捉える」
　
　結論に至るように論理を構築する。
　
　
　最後に、
　
　３、感情の波が、凪（なぎ）、とまでは言わないまでも、
　　　軟風・和風のレベルまで収まるところまで繰り返す
　
　
　これらを心がけることで、憤りや悲しみの大波に
　根こそぎ持っていかれることなく、
　
　徐々に緩和されていくことを実感されることでしょう。


　是非一度お試しください。
