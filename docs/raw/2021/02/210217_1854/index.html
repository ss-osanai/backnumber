　-----------------------


　これまで、

　「未来に対して積んでいきましょう」

　合わせて、対比というか、
　対応する言葉として

　「積まなければ詰みますよ」

　といった話をしてきました。


　未来に対して
　いろんな物を積んでいかないと、

　人生も生活も仕事も、
　詰んでいきますよね、

　という話。


　-----------------------


　未来に対して積む、
　積んでいくというのは、

　「未来の私に対するプレゼント」

　という風に言い換えてみられては
　いかがでしょうか。
　

　-----------------------

　
　未来の私に対して
　何をプレゼントするのか。


　プレゼントと言うと、

　物質的なものであったりとか
　あるいは経済的なものであったりとか、

　想像しがちであり、もちろんそれはそれで
　価値のあることだけれども、
　
　非物質的なもの、
　物質にあらざるもの、

　というところに
　意識を向けてみてはどうか。


　-----------------------

　
　過去の自分が現在に送ってくれた
　プレゼントの中で、

　一番嬉しい、有り難い、感謝せずには
　いられないプレゼントは何かというと

　「幸せの感情」

　ではないか。

　
　あるいは幸せを構成する感情、
　といってもいいかもしれませんが、

　感情こそが
　私たちのもっとも欲しいものではないか、

　という風に思われます。


　-----------------------


　よくビジネスの世界で言われることに、

　ビジネスの世界において
　価値を渡ししましょうとか、

　お役に立つとか、貢献するかという話が
　あったりするわけですが、

　究極、お客さんに何を渡そうとしているかと
　言うと、

　「お客様が望んでいる感情」

　を渡している。


　-----------------------


　たとえば私（鮒谷）なんかだと、

　いろんなものをお渡ししていると
　標榜しています。


　たとえば新しいパラダイムを
　お渡ししてますとか、

　新しいシステムの作り方を
　お渡ししてますよとか、


　新しい習慣のつくり方を
　お渡ししていますよとか、

　落穂の拾い方を
　お渡ししていますよとか、

　ですね。


　-----------------------


　いろんなものを、これまで私は
　お渡しさせて頂いてきたわけですが、

　別に、システムそのものが
　皆さん欲しいわけではなかったりする。


　パラダイムそのものが
　欲しいわけでもないし、

　落穂そのものが欲しいわけでもなく、
　それらはあくまで手段であって、

　欲しいのは

　「それらを通して得られる感情」

　です。


　-----------------------


　もうちょっと分かりやすい話にすると、

　車を売っている人は車について、

　燃費であったりだとか、
　加速がどうだとか、
　内装がどうだとか、

　2人乗りか4人乗り6人乗りなのかとか、
　右ハンドルだとか左ハンドルだとか、

　いろんなスペック的要素を
　説明すると思います。


　車を売っている人は、（油断すると）
　そのスペックを売っていると思いがちであり、

　買う側は買う側で、これまた油断すると
　スペックで比較してしまいがち。


　-----------------------


　しかし究極のところ、特にお客様サイドから
　見たとき何を求めているかというと、
　
　車を購入することによって、
　どんな感情を得ることができるかが大事であり、

　ということはつまり、
　感情を購入したいと考えているわけです。


　-----------------------


　スペックというのはすなわち手段。


　車のみならず、
　たとえば家なんかでもそう。

　大きさとか広さとか内装、間取りとか、

　（車同様、
　　家の事もよく分かりませんが）

　家は家でそれぞれスペックがあるわけです。

　防災性能であったりとかね、
　耐震性能であったりだとか。


　売る側はまた、タワマンだったら、

　何階建てですよとか、
　何平米ですよとか、
　何 LDK ですよとか、
　ここから花火大会が見ることができますよ、

　とか。

　共有ラウンジがあるとか、近所に
　こんな学校だとかスーパーだとかもある。


　そういう設備があって、
　近隣にそうした施設があり、

　いろんなこれは全部スペックで、
　買う側もスペックを見るわけですけれども、

　そのスペックを見て
　何を想像しているのか。

　そのマンションを購入したら
　何が得られると考えているかというと

　「どんな感情が得られるか」

　を考えている。


　-----------------------


　たとえば、車だって、
　
　そこにステータスとか優越感といった
　感情を求める人もあるでしょうし、

　家族とのつながりであったりだとか、
　絆であったりを求める人もあるかもしれません。


　心地よさ、気持ちの良さ、
　乗り心地であったりとか、
　
　という身体的に感覚からくる、
　心地よさという感情を求めている、

　ということもあるかもしれません。


　-----------------------


　それ以外にも、

　宝石を買うとか、
　旅行に行くとか、

　ホテルに泊まるとか、
　電化製品を買うとか、

　服を買うとか、

　買い物にあたって意思決定する時に
　我々は結局、何を求めているかというと、

　「ある感情が欲しくて、その感情を
　　満たしてくれるであろうものを購入する」

　という話。


　-----------------------


　実はこれは買い物に限らない。


　何か購入するということだけではなく、

　我々がいろんな日々の取り組みというか
　活動をしていく中で、


　それによって今の自分が、
　あるいは未来の自分が、

　「幸せを構成する感情を
　　手に入れられるかどうか」

　を無意識に意思決定の、あるいは
　判断の拠り所としているということ。


　-----------------------


　ここからどういう結論が
　導き出されるかというと、

　未来の私に向けて、今の私が
　何をプレゼントするかを考えるときに、

　「（具体的なスペックよりも先に）
　　まずどんな感情を手に入れたいか」

　に思いを馳せるべきではないか、
　ということとなる。

　
　そしてその「感情」をプレゼントすることに
　フォーカスを当てるとよいだろう。


　-----------------------


　では、

　「我々の幸せを構成する感情」　
　「私たちが幸せになれる感情」

　その感情には
　どんなものがあるかということを、

　あくまで挙げるものは一例ですが、
　以下にそれらを挙げていこうと思う。
