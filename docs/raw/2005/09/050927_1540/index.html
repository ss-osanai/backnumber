＜今日のピークパフォーマンス方程式。＞

　【　成功方程式を因数分解してみると。 】

───────────────────────────────────


■ひと口に「成功」といっても、いろんな成功のかたちがあります。


■では、自分はどのような成功のスタイルを求めればいいのか。
　
　あなたも困惑されたことがあるのではないでしょうか。


■私も思い悩んでおりました。

　そこで考えついたことが、
　ビジネス界のトップアスリートたちが実践している成功方程式から
　「共通の因数」を引っ張ってくることでした。

　どういうことかといいますと、、、


■多くのビジネス界のトップアスリートたちとお話をしたり、
　身近に接したり、
　さまざまな媒体を通じて学ばせていただく機会があるわけですが、

　そうした人たちは一人一人、じつにさまざまな個性を持って
　いらっしゃいます。

　そしてそれぞれが全く別の行動をとり、それぞれの世界で一流の
　結果を出しておられます。


■彼らの成功法則は、一人一人まったく異なるように見えます。

　ところが、こうしてずっとピークパフォーマンス方程式を配信して
　いく中で、あることに気がつきはじめました。

　それは、成功者の成功原理は、因数分解が可能なのではないか、
　ということ。


■たとえば、次のような方程式があったとしますね。

　　5ｘ^2 ＋ 10ｘ ＋ 25 ＝ 0 　（ ^2 は2乗を表す）


　この方程式は共通の因数である 5 でくくって、
　次のようにすることができます。

　　5（ｘ^2 ＋ 2ｘ ＋ 5）＝ 0


■ちょうどこれと同じように、

　ビジネス界のトップアスリートたちが実践している成功方程式にも
　共通の因数があるのではないか。

　その因数を取り出してくることによって、成功はどこからやって
　くるのか、という核心がみえてくるのではないか、と。


■ちょっと余談になりますが、

　いま申し上げましたように、
　いつも配信しているピークパフォーマンス方程式は自分で
　編み出したものではありません。


■ビジネス界のトップアスリートたちの書籍やＣＤやＤＶＤ、
　セミナー、あるいは直接の交流を通して学んだことに、

　つたないながらも自分の体験とを融合させ、

　それをあくまでも自分の言葉で語る、という方法で
　方程式を作ってきました。


■そして、きのうお話ししましたように、

　そうしてできた方程式は、
　あたかもコーヒーメーカーがボタンひとつで誰でも
　同じ高品質のコーヒーをいれられるがごとく、

　誰がインストールしてもただちにピークパフォーマンス
　（最高の結果）を叩きだせるようになる。
　

■この「平成・進化論。」は、そんな特効薬を１日１粒ずつ
　処方する、というポリシーのもとで執筆しています。

　そもそもこのメールマガジンを日刊にしようと思ったのも、
　処方には継続性が必要だからです。


　毎回毎回のピークパフォーマンス方程式が、１週間ごと、
　あるいはそれ以上の期間をあけて、忘れたころに配信されて
　きたらどうでしょうか。

　毎日毎日、継続的に、忘れずに飲むからこそ、
　その薬は効果・効能を発揮できるのです。


■話がそれました。

　とにかく、そうやってピークパフォーマンス方程式を
　つむぎ出すなかで、いつも心がけていることは、


　世に、どれだけ多くの成功方程式があろうとも、
　そこにはきっと共通の因数が存在するはずだ。

　その因数を抽出することによって、
　いわば精製したかたちで成功方程式をお届けしよう、

　ということです。


　中には私の力量不足、筆力不足により、精製の度合いが
　不十分のままお伝えしていることがあるかもしれませんが、
　それはご愛嬌ということで。。



■成功とは、さまざまな成功要因の複合の産物。

　しかしその根底には「共通する因数」が存在します。

　その共通因数を抽出できれば、いついかなる場面でも、
　どんな形態のビジネスにも適用できる方程式ができあがります。


■そして、そのこと一つを考え続け、

　なんと私はついに、最強の共通因数を抽出することに
　成功してしまったのです！！

　明日は、それについてお話しします。ご期待ください。
