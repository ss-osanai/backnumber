7470号 桃鉄（ゲーム）を徹夜でプレイした挙げ句に得られた学びとは

■先日、人生初の

　「桃鉄（桃太郎電鉄）※ゲーム」

　をやりました。

　それも横浜の、大画面テレビ備え付けの
　某ホテルに泊まりがけ＆徹夜にて。



■プレイしたのは最新版？の、

　「桃太郎電鉄ワールド
　　～地球は希望でまわってる！」

　でした。



■長時間、休憩なしで
　画面を見つめ続けて、

　最後、目が痛くて、
　開けなくなるくらいまで没頭、、、

　桃鉄、たのしいですね。
　ついつい、やりこんでしまいました。



■それはともかく、
　プレイしながら思ったんですよね。

　（以下は、桃鉄のゲーム設定とは
　　異なる話も出てきますが）

　あくまでプレイ中に
　触発されての思索も含み、

　その上で考えたこととなります。



■それは

　（頻繁に大きなイベントが訪れる
　　桃太郎電鉄とは異なり）

　多くの人にとって、
　人生の大半は

　「可でも不可でもない時期」

　が続き、ここで失点すると
　挽回はなかなか難しいということ。



■ほとんどの人は

　「可でも不可でもない時期」

　について最低でも、
　無難にやり過ごしている中で、

　自分だけが

　「下手を打ってしまう」

　と、そこからの復活は
　著しく難しくなるのです。



■なので

　「可でも不可でもない時期」

　には、

　守りを固めることを基本としつつ、
　（過度にリスクを取らない）

　その上で、

　攻めの要素も取り入れつつ、
　安全運転の中で好機を待つ、

　そんな姿勢が
　求められるのではないかと。



■こうした状態をキープしているうちに、
　桃鉄流に言うならば、まれに

　「絶好調（モード）」

　に入ることがあるんですよね。



■その典型が、

　【独自性＆排他性のある、
　　圧倒的に有利な勝ちパターン】

　で、

　（ビジネスやキャリアにおける）

　必勝モード、つまり何をやっても
　成果に繋げられる状態に入ったとき。



■生涯において、
　やることをやっていれば、

　最低でも一度は、
　そして、運がよければ何度か、

　こうした上昇気流に乗れる機会が
　訪れることでしょう。



■このとき、有頂天にならず、
　一過性のものであると強く自覚し、

　チャンスを逃さず、
　一気に資源投下を行って、

　取りこぼすことなく、
　獲得できるだけのリターンを獲得し、


　返す刀で、

　その状態を可能な限り、
　再現性のある形で固定化させる、

　ことを目指すべき。



■必勝モードが続いている間に、

　そんなシステム（再現性のある仕組み）を
　組み上げられれば、

　絶好調モードが去っても、
　それと同等、とまではいかなくとも、

　それに近い状態を無理なく
　キープできるようになるでしょう。



■ただし、

　システムを作るという発想がなかったり、
　あるいは、あっても作りきれなければ、

　そのうち元の状態に戻り、

　「可でも不可でもない時期」

　を再び、過ごすこととなる。



■また反対に、

　さらに絶好調モードと同じくらいに
　（レアな）頻度で、

　「絶不調」

　が訪れることがありますね。


　あたかも「キングボンビー」に
　取り憑かれたような。

　（これが言いたかった）



■そんな時には、一刻も早く、
　そこから抜け出したいもの。


　しかし同時に焦り過ぎたり、
　慌て過ぎてしまうと、

　おかしなものを掴まされ、もう一段、
　地獄に突き落とされることがあります。



■そうならぬよう、

　キングボンビーの襲来時
　（泣き面に蜂、弱り目に祟り目状態とも）

　には、壊滅的にさまざまな資産を
　破壊され続けても、


　（甚だ困難なことではありますが）

　それでも可能な限り、
　冷静に事態を平常状態に戻すための、

　突破口を探したいもの。



■その糸口さえ見つけられれば、
　あとはなんとかなるけれども、

　平静さを失い、焦って、慌てて、
　出口を見つけられなければ、

　やがて、すべてを失い、
　破局を迎えてしまいます。



■人生を俯瞰すると、
　この三つの時期、すなわち

　「可もなく不可もない状態」
　「絶好調モード」
　「キングボンビー襲来モード」

　における状態認識、
　ならびに対応策の解像度を上げていけば、


　人生という名のゲームに勝てる、

　イコール、

　幸せになれる可能性を
　高めていけるのではないかと

　（桃鉄をプレイしながら！）

　考えたわけでした。



■まとめると、そして若干の補足をするなら、
　こういうことになりそうです。

　平時は無理はし過ぎず、
　かといって安定も求め過ぎず、

　新たなチャンスの模索を繰り返す。



■そんな中、突如、訪れる
　「絶好調モード」には、

　乗るしかない、
　このビッグウェーブに！

　と、嬉しい感情を押し殺しつつ、


　フルベットに近い形で
　手持ちの資源を投資し、

　同時に確変モード終了時に備えて、
　早めに仕組み化を進めておくこと。



■また、「キングボンビー」の襲来に
　際しては、

　諦めず、脱出口を探すけれども、
　慌てると、そこでおかしなものを掴み、

　さらなる地獄に叩き落とされることも
　少なくないので、

　焦燥感の中でも冷静さは失わず、
　逃れる道を諦めずにサーチし続ける、

　といった感じでしょうか。



■前もって、

　こんな感じで自身を取り巻く環境認識と
　それぞれへの対処法について、

　「ルールとして確立」

　しておけば、


　迷うことなく、
　その時々を生きていき、

　いろんな事件が起きたとしても、
　上手に対応しつつ、

　長い目で見れば、人生も良い方向に
　向かっていくに違いありません。



■そんなわけで、

　人生に大きな気づきを
　もたらしてくれた桃太郎電鉄、

　さらに、その徹夜プレイに
　誘ってくれた友人に感謝！

　なのでした。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■平時のときは淡々と、極力、失点をせず、
　　同時に次の飛躍の芽を探す。

　■絶好調モードに入ったときには、
　　機会の総取りを目指して大きく張りつつ、

　　絶好調モード終焉時に備えた、
　　システム化を心がける。

　■絶不調モードのときには、
　　焦らず、冷静さを保ちながら、

　　突破口を見出すことに
　　全力を尽くすこと。

　■人生を俯瞰し、こうしたルールを
　　前もって定めておけば、

　　いざというときに慌てたり、
　　迷ったりすることもなくなるだろう。
