7103号 「脳内配線のつなぎ替え」について

■私（鮒谷）は長い間、

　【良い言葉さがし】

　をしてきました。


　言葉の持つ力に魅了され、
　日記を書き始めて30年。

　その間、ひたすら

　「良い言葉」

　を書き溜め、のみならず
　自身に編み込み続けてきた、

　そんな人生だったと思われます。



■「良い言葉」の定義は、いろいろ
　与えられるでしょうけれども、

　私（鮒谷）にとっては

　【脳内回路の配線を良い方向に
　　変えてくれる言葉】

　と言い換えられそうです。



■築いてきた回路がある言葉に触れ、
　配線が更新された、

　誰しも、そんな経験を
　お持ちのことでしょう。


　その瞬間のことは

　「目からウロコが落ちた」

　とか

　「開眼した」

　とか

　「霧が晴れる思いがした」

　と言われます。



■もちろん刺激の大きさ、
　配線替えの規模はまちまちですが、

　多い人は毎日、さらには
　一日になんども、こうした

　【認識の更新】

　が起こるもの。



■この瞬間、
　世界の捉え方が変わり、

　それはそのまま行動の変容に
　つながります。

　行動の変化はやがて未来の運命を
　変えていくこととなるでしょう。



■あなたは毎日、

　「ご自身脳の配線が
　　良い方向に更新されている」

　と実感されているでしょうか。


　更新の頻度や
　配線替えの程度はまちまちですが、

　私（鮒谷）は毎日、
　上書き更新され続けていますし、

　それゆえ人生が楽しく、充実し、
　生きている実感を味わえます。



■あるいは、

　脳に一定の負荷をかける機会が
　どのくらいあるのか、

　つまり

　「認識に混乱、混沌をもたらすような
　　言葉に触れていますか」

　という問いも投げかけたいと
　思います。



■世界の捉え方に混乱が生じる、

　そんな事態は一見、ネガティブに
　見えるかもしれませんが、


　混乱から跳躍は生まれ、
　混沌から新たな秩序が立ち上がる、

　と考えれば、

　そうした機会にはむしろ
　肯定的な評価を与えるべき。



■混沌を受け入れ、
　喜び、楽しがる姿勢があれば、

　「どこに跳躍の道筋があるのか」

　という問いが
　潜在意識に留まり続けます、


　こうした持続的な自問自答は
　やがて矛盾を克服し、

　止揚された答えを必ず
　導き出すこととなるでしょう。



■先日来、ご案内してきた

　【「身体に編み込む勉強法」音源】

　もまた、

　「脳内配線のつなぎ替えや
　　意図的な混乱を促す音源」

　としての役割も担う音源です。



■これを聴いて、
　ぜひ脳の配線を更新したり、

　跳躍をもたらす、
　前向きな混乱を生じさせる、

　そんな機会を獲得して下さい。



■当音源は、

　多くの人にとっての
　学びに対する概念を根本から覆し、

　「お勉強」から
　「身体に編み込むための学習」へと、

　パラダイムと行動習慣の転換を
　目的としてリリースしたもの。



■スケートリンクの上を滑るような
　（つるつるして定着しない）学びではなく、

　身体に編み込む
　（基本的な所作を確立する）学びを。


　これが同音源を通し、

　休憩なし、３時間半以上の時間を使って
　私（鮒谷）がお伝えしたかったことの、

　中心的なメッセージとなります。


■こんな時代だからこそ、
　この音源を聴き、

　【波及効果の高い、
　　一つ一つの意思決定の基準を、
　　確実に身体に編み込む学び】

　の価値と力を再発見し、

　地に足のついた実践を通して
　重心の低い姿勢を手に入れて下さい。


　今日も人生とビジネスを楽しみましょう！

　【今日のピークパフォーマンス方程式】

　　■あなたは日々、

　　　「脳の配線更新が促される言葉」

　　　にどの程度、触れているか。

　　■世界の捉え方が更新されないと
　　　行動も更新されることはなく、

　　　行動に変化がなければ
　　　運命の軌道を変更させられない。
