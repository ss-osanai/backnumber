＜　昨日の活動ダイジェスト　三井不動産さまにて講演。　＞
───────────────────────────────────


■昨日は、中堅～若手の将来有望な社員の方々を集められた勉強会の講師
　として、三井不動産さまよりお声かけ頂き、お邪魔させて頂きました。


■たまたま、同じ高校の先輩が、こちらのほうにお勤めになって
　いらっしゃり、お声をかけて下さったのがきっかけだったのです。


■今回10回目で最終回、ということでたまたま講師のトリをつとめさせて
　いただいたのですが、今までの講師の方はパイロットの方とか、高名な
　お医者さんだったり、東京大学の副理事？の方とか、さまざまな世界で
　実績を挙げられている方々ばかり。


■そんな一連の流れをぶち壊すような、明日にはどうなっているかも
　わからない、ぽっと出の若造？をご招待くださり、Ｙさん、どうも
　ありがとうございました！


■お話し自体は１時間半という時間だったのですが、その後の懇親会が
　大変盛り上がりました。

　いろいろ会社のこともお聞きしたのですが、さすがは日本の不動産業界
　のキープレイヤーの一社。

　売上高、社員数、採用人数を聞いてびっくりしたのですが、売上に比して
　従業員数、採用人数が非常に少ないんですね。


■本当にこれから自分たちが将来経営を担っていく、といった気概と
　高い能力をお持ちの方々ばかりで、非常に刺激を受けました。

　私が聞いていた中では日本郵船、といった会社もそういう気風がある、
　なんて耳にしていましたが、こちらの会社もまさにそんな感じが
　いたしました。


■世の中にはほんとに優秀な方々がたくさんいらっしゃると改めてびっくり
　した次第です。しかも一つところにまとまって、いらっしゃるんですね。

　最近はあんまりなかったのですが、やはり日本を背負って立つポジション
　にいる総合商社の人たちや、日本の技術をリードする某メーカーの方々と
　以前、一緒によく飲みにいったりしていたときに、同じ事を感じたのを
　思い出しました。


■１対１、あるいは１対数名、の優秀な方とお話をする、という
　シチュエーションはよくあることですが、多くの方々がいらっしゃり、
　その一人一人の全ての方が大変、切れ者、優秀、コミュニケーション能力
　も当然のことながら非常に高い。

　また将来、会社を担っていくという自覚や自負をもっていらっしゃる方々
　で構成された組織、というのは非常に強力だな、と参加くださった皆様
　を前にして身震いするような感動を覚えた次第です。


■こんな優秀な人たちが組織を構成されているのだから、日本の大企業は
　なんだかんだいってもやっぱり強いよなぁ、、、、と改めて感じ入った
　ひと時でした。

　やっぱり企業は人なり、なんですね、、、


■いろいろ取り仕切って下さいましたＹさま、またご参加くださった皆様、
　どうもありがとうございました！

＜今日のピークパフォーマンス方程式。＞

　【　　積極的に他流試合に臨んでみる。　　】

───────────────────────────────────


■今回、三井不動産さまでお話をさせて頂き、感じたことは他流試合の
　経験がまだまだ不足しているなぁ、ということ。


■限られた世界の中で、優秀な、あるいは卓越した方々と触れ合わせて
　いただくことはあっても、異質な世界の方々とはこういうきっかけでも
　なければ、交わることがないから、自分の立ち位置というものが正確に
　つかめない、と知らされました。


■今回は、今までお付き合いさせていただいている世界の方々とはやはり
　ちょっと異なる世界にいらっしゃる方々と接しさせて頂いたのですが、
　物事の捉え方、考え方、伝達の仕方、コミュニケーションのとり方、
　一つ一つが、自分とは違ったカルチャーで表現されているのです。

　それぞれが大変、勉強になりました。


■私がお伝えさせていただいたことはお伝えさせて頂いたこととして、
　きっと何かのお役に立てていただけるのではないか、と思って
　おりますが、同時に私も参加下された皆様からのたくさんのフィードバック
　や、考えをお聞かせいただくことで、自分の考えを改めて客観的に
　知ることができる、非常に素晴らしい機会でした。


■どんどん外に出て、異業種、自分とは違う世界の方々と触れ合うことは
　重要と改めて知らされました。

　特に、同じ殻、枠の中でそとに出ないような業種、職種の方々こそ、
　他流試合の経験を求めてしていただければと思います。

　きっと、そこに新しい発見があるのではないかと思います。
