```
■長い間、来る日も来る日も

　「どうすれば幸せになれるだろう」

　という問いを
　回し続けてきました。


　この問いは

　「どうすればビジネスで成功できるか、
　　理想のキャリアを重ねていけるのか、
　　大きなお金を手に入れられるのか」

　といった問いとは、
　似て非なる問い。



■もちろん、幸せな生活を支える
　一つの要素として、

　ビジネス（仕事やキャリア）、あるいは
　お金について避けては通れませんから、

　その種の自問自答も重ねてきました。



■しかし、

　長く、そして注意深く、これまで
　メルマガを読んできて下さった方なら、

　あくまで、
　私（鮒谷）のど真ん中の自問自答は

　「どうすれば幸せになれるだろう」

　であって、決して

　「どうすればビジネスで成功できるか」
　「どうすればお金を稼げるか」

　ではなかったことを
　ご理解頂けるかと思います。



■その一つの証拠（？）として、
　20年近くずっと、メルマガの末尾には

　「今日も人生とビジネスを
　　楽しみましょう！

```


```
【今日のピークパフォーマンス方程式】

　■幸せを本気で願えば幸せに近づける。

　■お金を欲しいと本気で願えば、
　　お金も手に入れられる。

　■これは一つの例だが、欲しいものを
　　明確にしていない上に、

　　その定義も曖昧なままであれば、
　　手に入れたくても動きようがなく、

　　日々の多忙に紛れて、惰性の生活が
　　続くことになるのではなかろうか。
```
