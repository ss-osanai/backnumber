━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

　【　開けていく人生と閉じていく人生　】

───────────────────────────────────


　
■こうやって毎日メルマガ配信していると

　「集めるのは難しく、離れるのは簡単」

　だなあ、とつくづく思います。


　新しい読者さんとご縁を頂く（＝集める）のは
　難しいものですが、
　
　既存の読者さんは簡単に、
　次々と離れていかれます。（汗）


　こういう表現をすると叱られるかもしれませんが、
　イメージはまさに

　「読者さんの新陳代謝」

　といった感じです。



■けれども、これはネットの内外を問わず、
　同じことがいえそうです。

　私たちは毎日、
　
　新しい人とのつながりが生まれては消え、
　生まれては消え、

　を繰り返しながら生きています。



■この新陳代謝のパターンが


　「新しい出会いの数 ＞ 離れていく出会いの数」

　「新しい出会いの数 ＝ 離れていく出会いの数」

　「新しい出会いの数 ＜ 離れていく出会いの数」


　のいずれであるかによって、


　「人間関係が豊かに広がっていくのか」

　「均衡状態が保たれるのか」

　「どんどん世界が閉じられていくのか」


　が決まってくるように思われます。



■これはビジネスにおいても同じです。


　新しいご縁を頂いたら、そのご縁を大切にし、
　温め、育んでいくことによって、
　
　その会社（あるいは社長）にとって、

　「新しい出会いの数 ＞ 離れていく出会いの数」
　　
　のパターンが定着していきます。



■反対に、

　新しいご縁を頂いても、投げやり、不躾な対応に
　終始し、関係を温めていこうという思いがなければ、

　「新しい出会いの数 ＜ 離れていく出会いの数」

　のパターンにはまり込み、

　いずれ、会社を畳まなければならないことに
　なるでしょう。



■綺麗事とかではなく、本当に

　「お客さまあっての会社」

　なのです。

　お客さまからお代を頂戴することができなくなれば、
　一日たりとも会社は存続できません。


　「お客さまあっての会社」
　
　だからこそ、お客さまの求めるものを理解し、
　その要望に応えることにフォーカスすることにより、

　会社を支える基盤が
　より強固になっていくのでしょう。



■最近、よくメルマガでも書いていることですが、
　そういう考えから、

　人生でも商売でも、長く安定した、
　右肩上がりの軌跡を求めるのであれば、　

　「広げるより深める。深まれば広がる」

　このパターンが完成させ、その上で、

　「深まりを広げる」

　ステージを求めるという順番が
　大切なのではないか、

　と考えているのです。


　いまは毎日、こんなことを考えながら、
　会社の仕組みづくりを行なっています。

　
　今日も人生とビジネスを楽しみましょう！
