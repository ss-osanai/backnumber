　-----------

人生は物語であり、自分自身を物語の主人公と見立て、
良い物語をコピーしたり、あるいは自ら練り上げたりすることが
重要であることを、他の音源等で鮒谷さんから学んできましたが、

物語の細部をより解像度高く詰めていく際に、今日のゲーム化の
「戦術」を存分に活用することができれば、人生の物語が
より一層豊饒なものになる、そのように解釈した次第です。

　-----------

今回は特に「依存」の持つ中毒性、不可逆性の
恐ろしさを再認識するとともに、

同じくその中毒性、不可逆性を利用した、鮒谷さん流
「人生逓増大作戦」をご開示いただいたと捉えております。

　-----------

どうやったら「依存」を良い方向に持っていくか、について
様々な視点や事例を元に語られており、

没入しやすい自分のような人間にとっては、
そのエネルギーを、どうやって人生の逓増に向けていったら
良いのか、という大きなヒントを得られた音源でした。

　-----------

依存を組み入れるメリット、そのバランスの調整、
具体的な設計思想などを具体的にお話いただいたものと
解釈しています

　-----------

同じ依存でも、負のスパイラルになる依存と
正のスパイラルになる依存では天と地の差がありますが、

依存という大きなエネルギーを良いスパイラルを
生み出すために使いたい、と考えている方は、
本音源を聞いてみると良いかもしれません。

　-----------

この音源をきっかけにして良い依存症となって、
人生ゲームのデザイナーとプレイヤーの両方を
極めてまいります。

　-----------

「クソゲー人生」を「神ゲー人生」へと変えるキッカケの
手引書と、ご解説をありがとうございました！

　-----------

「ゲームのように考える、とはどういうことなのか。」
そして、「目標、物語、ピラミッド音源の内容と
どう密接につながりあっているのか。」

色々な問いや言葉、気付きが頭の中を駆け巡り、
既知の知識との化学反応がバシバシ起きています。

　-----------

鮒谷さんのこれまでの経験を赤裸々に語って
いただいている部分がとても新鮮でした。

どのような過去があって今に至られているのか、
その歴史を垣間見て

「そうか、鮒谷さんにもそんな時期があったのか！
　そんなことやってたのか！！」

という驚きと、安心感（自分もコツコツ歩めば良いな、
という感情）の両方を感じています。

　-----------

細々とした仕事にゲーム性を見出すのではなく、
人生そのものがゲームであると捉えて、

ゲームの難易度を自分自身で調整し、自身が人生という
ゲームに没入・没頭する仕組みを構築することが
大切であるということに気づきがありました。

　-----------

今回の音源では、ゲーム化というメタファを通じて、
適切かつ圧倒的な行動を促すための
具体的な定義や方法を学ぶことができました。

　-----------

「カラーテレビ」に見えてきたかのように、

思考 OS が抜本的に変わり、気持ち的にも
モチベーションが沸き上がっております。

■今日は別に

　【人生ゲーム化音源】

　についてのご案内ではないので、
　これくらいにしておきますが、

　なぜ、このタイミングで同音源の
　紹介をしたのか以下に記載します。



■実は

　【人生ゲーム化音源】

　は、その名の通り、

　【人生をゲームに見立てましょう】

　という「見立て」についての
　話なんですね。



■先日、松岡正剛の『見立て日本』と
　いう本を読んだばかりで、

　思わず「見立て」という言葉を
　使ってしまいましたが、


　でも、先の音源は、やっぱり
　人生をゲームになぞらえて捉え直す、

　つまり「見立て」についての話です。



■以下、こんなことを書くと
　熱狂的なファンの方からクレームを
　いただきそうなので、

　先に謝っておきます、ごめんなさい。



■実は「見立てる」という行為は
　「催眠」と深い関係があるんですよね。


　たとえばディズニーランドは、

　私（鮒谷）のようなドライな商売人とか、
　あるいは株式投資をする人間からすれば

　【よくできた（＝収益性の高い）
　　商業施設】

　以上でも、以下でもありません。



■でも、運営会社(Ｏ社）さんは、

　かの商業施設の伝えたい側面を
　あの手この手で見せ（魅せ）続け、


　ファンに対して、現実にはコンクリートと
　遊具の複合体に過ぎないパークを

　「ここは『夢と魔法の王国』ですよ」

　と見立てさせることに
　成功しています。



■夢から醒められると大変
　（客が来なくなる）ですから、

　普段の嫌な仕事を思い出させるような（！）
　高層ビル群も見えなければ、

　煙をもくもくと吐き出す工場なんかが
　視界に入らないように、

　パークを設計しています。

　キャストの動きもまた然り。



■他にも、園内に弁当を持っていくと
　怒られますよね。

　（実際に怒られはしないでしょうが、
　　やめて下さいとは言われるでしょう）


　あの場所で弁当箱を開いて、

　日の丸弁当とか
　巨大な爆弾おにぎりなんかが出てきたら、


　当人はもとより、
　それを見た周囲の人も、

　「夢と魔法の王国」

　から一気に現実世界に
　引きずり戻されてしまいますから。



■そうならぬよう、

　【舞台装置やルールに
　　あれこれ工夫がなされている】

　わけであり、

　ハッキリ言ってしまうと、
　あの場所は確信を持って

　【催眠に支配された空間】

　として作り出されたわけですよ。



■その証拠に、

　パークを出て、京葉線に乗って
　東京駅に近づく頃には、

　（一部の熱狂的なファンを除いて）
　我に返り＝恥ずかしくなって、

　つけていたネズミのカチューシャを
　いそいそと取り外す、

　みたいな光景が
　繰り広げられるわけじゃないですか。



■あれこそが

　【催眠支配空間の磁力から離れて
　　夢から醒めた姿】

　といってよいでしょう。


　あの空間において生じさせられていた
　変性意識＆高揚感（＆謎の購買意欲）が、

　現実世界に触れることによって
　もとに戻った、

　ということです。


　※変性意識
　＝日常の意識状態とは異なった意識状態



■同じように、ゲームにも実は

　【変性意識を生ずる効果がある】

　んですよね。


　RPGなんかが特に分かりやすいですが、

　物語や世界観を通して
　変性意識が生じるように、

　ゲームが作られているわけですよ。



■だからドラクエでも、
　ファイナルファンタジーでも、

　ゼルダでもなんでもいいですが、
　その世界に没入できるし、

　何時間でも、人によっては
　10時間ノンストップ、

　とかですら、
　プレイし続けられるのです。



■他にもパチンコ、パチスロなんかも、

　カモ（もといパチンカス、
　もとい遊技機愛好者）を、

　意図的に変性意識状態に導き、
　カネを巻き上げる、

　もとい、金銭と引き換えに
　娯楽を提供しているわけですね。



■うまく催眠誘導させられると、

　それこそ開店前に店の前に並び、
　そのまま閉店まで、

　といったことも普通にありましたね。
　（今もあるのかな）



■ディズニーファン、
　ロールプレイングゲームのプレイヤー、

　【変性意識状態に入って、
　　没頭・没入しているところ】

■重度になると、その場を離れても

　「24時間、夢から醒めなくなる」

　のは、たとえばブギー・マンの

　『PACHINCO・MAN』

　の歌詞なんかを聞けば、
　一目（一聴）瞭然でしょう。



■ここに出てくる

　「目を閉じれば
　　スリーセブンの赤い文字」

　なんか、

　一定以上、パチンコ（パチスロ）に
　カネを突っ込んだ人間なら、

　全員、何度も

　「ある、ある」

　と頷かずにいられぬはず。



■さらには、あの

　「24時間、働けますか？」

　が許されていた時代もまた、

　「時代そのものが、多くの人を
　　変性意識状態＆ハードワークに
　　導いていた」

　わけですし、


　詳しくは述べませんが、

　恋愛もまた、ある種の催眠状態と
　言えるでしょう。



■それゆえ、

　ピックアップ・アーティスト
　（ナンパ師）もまた、

　催眠を使いこなします（悪用）。



■催眠とは、

　実はかくも身近なものであり、
　力を持つもの、

　とも言えるでしょう。



■こうした強力な力を持つ催眠
　（変性意識への誘導）を、

　自分のために「善用」できれば、

　自身を思い通りの方向に
　進ませることができると思いませんか？



■私（鮒谷）なんか、完全に

　【自分が楽しんだり、感動しながら
　　没入せずにはいられないワールド（沼）】

　にどっぷり浸かりきり、

　死ぬまで醒めないであろう

　【変性意識状態】

　に置かれているわけですが、

　実際のところは
　自らをその場所に導いてきたのです。



■今日は深くは記しませんが、

　変性意識状態と無関係の人は
　一人たりとも存在いたしません。

　（テレビドラマや映画やアニメを見ていても
　　小説を読んでも意識は変容する）



■気持ち悪いとか、怖いとか、
　怪しいなどという以前に、

　みんな自覚なく
　催眠にかかっているんです。

　今もですよ。



■今日は月曜日だから、
　会社に行かなければならない、

　などといった
　「当たり前」と思っていることですら、

　催眠かもしれません（催眠です）



■さらには、

　自分にはできない（難しい）とか、
　どうせ自分は○○だから、

　などという構文が
　勝手に脳内で回り始めるのも、

　催眠かもしれません
　（催眠です。ありがとうございました）



■なので、

　自分には関係ないとか、
　気持ち悪いとか、興味ない、

　なんて思っている人は

　【たんに自分のことを
　　客観視できていないだけ】

　なんですよね。


　自分の意識に
　全生活が縛られているのに、

　縛られていることに
　気づいていないだけ。



■悪しき催眠に支配され、

　悲惨な目に遭いたくなければ、
　先手を打って

　【人生が心身ともに逓増するような
　　変性意識状態に自らを導く】

　思います。



■というか、

　思うだけではなく、
　実践してきたのが、

　私（鮒谷）のこれまでの人生です。



■正確には、

　【人生が心身ともに逓増するような
　　変性意識状態に上書きし、自らを導けた】

　のですけどね。
