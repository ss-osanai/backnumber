頭の回転が速くなる「速聴」が手軽に取り組めるようになりました。
 
すでに100万人に指示されている加速学習法「速聴」はご存知ですか？
今までは音質が劣悪なものだったり、高額な「速聴」しかありませんでした。
 
ところが、「速聴」システムに惚れ込んだ、東京の吉井さまが「速聴」の商標
を持つ会社と交渉して、テープタイプの速聴機の販売権を取得され、高音質の
速聴機を格安で提供できるようになりました。
吉井さまは、販売チャンネルを中間マージンの必要がない、インターネット
販売に絞るとのことです。今なら、サイトオープンに伴い、価格の割引を
おこなっています。

＜今日のピークパフォーマンス方程式。＞

　【　　時間当たりのパフォーマンスを最大限に引き上げる秘訣。　　】

───────────────────────────────────


■時間だけは貯金することができません。
　当たり前のことですが、なかなか普段、意識していないこと。


■お金は貯金できるけれども、時間は蓄えることができないから、限られた
　時間をどのように最大限、有効に活かすかについては真剣に考えなければ
　なりません。

　言い換えると、単位時間あたりのパフォーマンスを如何にして最大化
　するか。


■恐縮ですが、私の例でいくと、必要以上にスケジュール、仕事を詰め込みすぎ、
　とよく周囲からいわれるし、それによってご迷惑をおかけすることもあります。
　（申し訳ありません）

　それでも、自分のキャパの120％、130％まで無理やりスケジュールを入れたり、
　いろんなご依頼に応えようとさせていただくのは、自分の容量（すなわち
　時間当たり価値）を最大限まで高めることができる訓練だと思っているから
　なのです。

　筋肉でも頭脳でも、負荷をかけなければ、怠けてしまいます。
　100％以上の負荷をかけることで成長がようやく始まるのです。


■訓練しなければ、時間当たり価値をより一層高めることはできません。
　
　訓練は厳しいものですが、その訓練を突破しなければ一騎当千の戦士
　にはなれないのです。


■私のなりたいイメージは「ビジネス版グリーンベレー」（アメリカ陸軍の
　特殊部隊）みたいな存在なのです。

　誤解を恐れずにいえば、常に最高のパフォーマンスを発揮し、期待され、
　恐れられ、尊重される、そんな存在。


■本を読んだだけ、セミナーに行っただけ、では一騎当千の強者にはなれません。
　勿論、型を覚える、型を身につける上で座学は非常に重要だと思いますし、
　私も常にお勧めしていることであります。

　ただ、最終的に実際に戦闘したときに「勝てるかどうか」は偏に実践、訓練の
　総量にかかってくることは間違いありません。


■限られた短い時間で自分自身を促成栽培しなければならないのですから、
　徹底的に負荷をかけていかなければなりません。


■最初にやっておいたら後は楽だと思っています。

　先に泣いて、あとで笑うか。
　先に笑って、あとで泣くか。

　わたしは圧倒的に前者がいいと思っているので、いろんなトラブルの発生や、
　肉体的、精神的負荷があったとしても、「ビジネス版グリーンベレー」になる
　ための強烈な訓練をいましばらくは続けていこうと思っています。
