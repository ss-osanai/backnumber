【抽象世界に遊びながら、飯を食う】抽象思考×コミュニケーション能力で空に翔びたい方へ

2017/03/27 09:30 
■先日、そうしたクライアントさんの
　お一人である、

　アイティビジネスサロン株式会社代表取締役の
　板橋力さんと久しぶりに会食をご一緒致しました。


　板橋さんは、世界最大の
　コンピュータネットワーク機器開発会社である、
　
　シスコシステムズにて長らく働いてこられた方。


　現在は独立されて、

　「外資系のIT関連ならびに、
　　コンサルティング会社に強い人材エージェント」

　の経営者として、ご活躍していらっしゃいます。



■突然ですが、私（鮒谷）は
　
　「原理、原則、本質、構造、普遍、パターン」

　といった言葉が好きです。


　ビジネスを行う上で、
　表層部分においては、時々刻々と

　「微細な改善」

　を繰り返し行っております。
　

　しかし、それはあくまで、

　「原理、原則、本質、構造、普遍、パターン」

　といったところに立ち返り、その土俵の上で行わないと、
　その場しのぎの小手先の取り組みとなり、
　
　結果として、労多くして功少なし、
　ということになりかねません。



■だからこそ、常日頃、
　
　「超速PDCA」

　などと言っていますが、実のところは、

　鳥の目で全体の構造をとらえてから、
　虫の目で具体個別の状況を見、改善を行い、

　その後、再び鳥の目に戻っていく、

　ということを幾たびも繰り返すことによって、
　ご飯を食べている、

　ということになろうかと思います。



■ここで、冒頭の板橋さんとの会食の話に戻りますが、
　
　特に

　「抽象度の高いことだけをずっと考え続けたい」

　という方も平成進化論の読者さんには
　数多くいらっしゃるようにも思われ、

　そんなところから

　「本質を押さえることの重要性」

　についての話題が出たのです。



■その話題の中で、

　私（鮒谷）の個別コンサルのお客さまで、

　本当は（具体的な話に寄り過ぎない）
　未来予測や技術動向などから導き出される、
　
　「抽象化された本質的なことをメインとした仕事」
　
　に取り組んでいきたいけれども、
　
　今は現場にいることを求められているし、
　また、（良くも悪くも）成果を出しているので、
　
　会社からも更なる活躍を期待されている。


　ただ、

　「本当にやりたい仕事はこれじゃないんだけどなぁ」

　と思い悩んでいる方が思い出されました。


　
■こうした方のように、

　世の中の流れやテクノロジーの動向などを
　含め、

　「将来、向かうべきところに向かい、
　　落ち着くべきところに落ち着いた後の世界」

　　は、どのような世界であるかといった、

　社会や歴史、技術動向等の本質を捉え、
　その仮説や根拠となる情報について、
　
　お客さまと話をするのが楽しくてたまらない、
　といった方にご活躍頂ける場がある、
　
　というお話が板橋さんからもたらされました。



■その具体的な求人案件について、
　板橋さんよりお話しを頂いたところ、

　「もっと抽象度の高い仕事をしたい」

　という悩みを持った
　私のお客さんにとっては
　最高の仕事となるのではないか、

　ということで
　こちらの方にご案内させて頂く運びとなりました。


　
■抽象世界に遊びつつ、遊びから得た知見を
　アウトプットすることに楽しさを覚える方には、

　またとないポジションとなるのではないでしょうか。


　ピンポイントの採用とのことで、
　もし、ご関心がある方があれば、

　急ぎ、ご応募なされてみられてはいかがでしょう。


　詳細は以下をご確認下さい。

　「高度な知的遊戯」
