 【　はじめに「その気」ありき　】

───────────────────────────────────



■昨日のメルマガでは書ききれませんでしたが、
　昨日の内容を一言でいうと、

　「成功の秘訣は、
　　自らをトランス状態に持っていけるかどうか」

　ではないかと思います。

　（誤解を招きそうな表現で恐縮ですが）



■理性ではなく、感情（本能）のレベルで、
　自分の信ずる道を歩む、

　そんな決意が生まれたとき、
　
　「他人の目線ではなく、自分の信ずる道を往ける」

　ようになるのです。



■よく催眠術をかけられた人が、

　鳥になって羽ばたこうとしたり、
　蝶になって舞おうとしたり、

　という話を聞くことがあります。


　これがどういう現象であるのか、
　私にはよく分かりません。



■けれども、もしこういったことが

　「やらせ」

　ではないとするならば、


　他人の視線を気にすることなく、
　鳥になったり、蝶になったり、

　それくらいのレベルの没入感で、
　取り組むべきことに取り組めれば、

　大抵のことは実現できるのではないでしょうか。
　


■とすれば、問題は、

　「いかにその状況に自らを持っていけるか」

　ということになるのかもしれません。


　昨日も少し書きましたが、

　「（理性で）これをやるべき」
　
　と思っていても、

　「（感性、本能のレベルで）やりたくない」

　と拒否しているとするならば、


　それはあたかも

　「アクセルとブレーキを両方踏んで、
　　車を動かそうとしている」

　のと同じことになります。



■そんな状態で、

　時速60キロ、80キロ、あるいは100キロ超を
　出そうとしても、それは無理というものでしょう。


　心の深いところで、

　「これをやる以外の選択肢は考えられない」

　という確信を持つことによって、


　「寸暇を惜しんで」

　「あらゆる犠牲を払っても」

　「他人の目なんかどうでもいい」

　「無我夢中で、時間を忘れて」


　という心的状態がもたらされることになります。



■私はそういった分野に不案内なので、
　以下の記述は正確ではないかもしれませんが、

　上に記したような世界に入ったときのことを

　「フロー状態」

　とか

　「トランス状態」

　というのかしらん、と思われます。



■考えてみれば、私も、

　勤務していた会社が倒産し


　※本棚を整理していたら、

　　倒産したときの新聞記事が出てきたので、
　　ウェブにアップしてみました。


　　地下鉄の駅構内の売店で売られていた、
　　この駅売り新聞の見出しによって、

　　勤め先が倒産したことを知りました（笑）

　　↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓↓
　　
　　http://www.2nd-stage.jp/profile/wc_hatan.html

　
　　
　その後、起業の道を歩むことになるわけですが、
　どういうわけか、

　「きっとうまくいく！という、根拠なき確信」

　だけはありました。

　

■この

　「根拠なき確信」

　とはすなわち、

　「フロー状態」

　あるいは

　「トランス状態」

　に入っている状態といっても良かったのではないかと
　思います。
　
　
　
■振り返ってみると、

　実は起業後の数年間については、
　ほとんど記憶が欠落しており、

　白昼夢を見ていたというのか、
　夢遊病者の状態というのか、

　はたまた催眠術の被験者（？）のような、
　というのか、

　そんな感覚しか残っておりません。


　まさに「催眠にかけられた」とでもいうような。



■こうして、

　自らの内なる声に導かれるようにして、
　ほとんど睡眠時間もとらず、

　没頭、没入して仕事に捧げていると必然的に、

　「集中力×投下時間」

　の積が最大化されることとなり、

　それに伴い、みるみるうちに成果も上がり始め、

　（今振り返っても、気持ちのいいくらいに）

　事業の垂直立ち上げが実現されました。



■そんなことを振り返るにつれ、
　結局のところ、


　（起業なり、なんなり、
　　なにかを始めるにあたって、必要なのは）


　経験でもなく、知識でもなく、スキルでもなく、

　テクニックでもなく、ノウハウでもなく、人脈でもなく、

　資金でもなく、能力でもなく、


　「その気（＝トランス状態）」


　になっているかどうか、が全てを決する、

　ということに気づいたのです。



■もちろん、

　「その気」

　だけで夢が実現するほど世の中甘くはありません。


　けれども、あらゆる資源が全て揃っていたとしても、
　肝心の、

　「その気」

　がなければ、夢が実現することはありません。


　
■はじめに「その気」ありき、であって、

　それさえ用意できれば、あとは必要に応じて
　「資源」は手に入れられるのです。

　いずれも必要ですが、順番がある、
　ということです。


　では、肝心の「その気」は、
　どうすれば手に入れられるのか。


　一言で語れるようなものではありませんが、

　折を見て、今後のメルマガでお伝えしていこうと
　思います。
　

　
　※３日から始まる３日間セミナーでは、

　　最近の研究成果（？）も含め、
　　出し惜しみなく、全てをお伝えしようと思います。

　　ご期待ください。
