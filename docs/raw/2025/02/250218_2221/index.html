■今日は

　「直近の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと。


　（ここから）
　---------

　売り込んでる時点で
　摩擦が発生しとるのよ。

　つまり有形・無形のコストが
　発生しているということ。


　セールスを生業にしている人から
　叱られるかもやけど、

　営業力自慢は「摩擦突破力」自慢と
　言い換えることもできる。


　でも、顧客を安全地帯に置いて
　「案内」するだけでスルリと売れていく
　商売の方が美しいよ。

　---------

　勤務先がいつ何時、
　潰れたり廃業するか、

　変化の激しい今の時代、
　分かったもんやあらへんで。


　だから、いざというときにも即座に
　独り立ちできるだけの、

　精神や能力、ネットワークは
　平時から強化しといたほうがいい。

　勤め先がどんな大企業でも
　潰れるときは一瞬やからな。

　----------

　相手をいくら愛していても
　フラれる時はフラれる。

　会社も一緒やで。


　勤務先が潰れることもあれば、

　大規模リストラだって
　もはや日常茶飯事。


　そのときブチ切れて

　「可愛さ余って憎さ百倍」

　となる人あるけど、

　もともとこの世に
　安全地帯なんかないんだわ。

　もうちょっと世界を引いて見る
　冷静さが欲しいわな。

　----------

　起業時には結構な
　ストレスかかるから、

　耐えきれずに脱落する人が
　後を絶たんのよ。


　特にそれまで出会わなかった
　タイプのモンスター
　（難儀な取引先や客）に、

　結構な頻度で遭遇する。


　でも会社勤めと違って
　誰も守ってくれんから
　慣れないうちは大変よ。

　だから前もって副業を経験し、
　予行演習しとくといい。

　----------

　大企業に勤めてる人には
　想像も及ばんような人間が

　ウヨウヨしてるのが
　起業後の世界やで。

　マジで

　「この世にはこんな
　　モンスター（魑魅魍魎）が
　　おったんか」

　と驚愕させられることもしばしば。

　ラダトーム城を一歩出た、
　ドラクエの勇者（古！）みたいな
　驚きを味わえる。

　起業志望者は副業で慣れておくといい。

　----------

　世の中には徹頭徹尾、

　今やってる仕事に
　適性のない人がいるんよな。

　何をやらせても使えない。


　向き不向きは努力でどうにもならんから
　諦めて、別の仕事に就いた方がいい。

　関係者に迷惑かけつつ、
　自尊心まで砕かれるからな。

　そんな人でも輝ける別の場所は必ずあり、
　あんがい副業起業で解決することも。

　----------

　もともと広い人間関係あるか、
　集客ノウハウを身につけてきた人は
　起業向き。

　両方ある人は今、
　この瞬間にでも起業できる。


　反対に、いずれも持たん人が起業するには、
　それなりの準備をせなあかん。

　立ち上げ期にいちばん大事なのは
　見込みとなるお客さまとの接点なのよ。

　これなくしては何も始まらんで。

　-------------

　販路さえあれば極論、売る商品の
　選定は後からでも間に合うのよ。

　つながっている人にヒアリングして
　顕在＆潜在欲求を把握し、

　ニーズを満たす商品やサービスを
　開発するか、外で見つけるかすればいい。


　その情報を販路に流せば売れるのよ。

　「はじめに商品ありき」に囚われ、
　苦戦している人は少なくない。
　
　-------------

　役に立たないクソ商品を煽り散らかし、
　売りまくり、多くの人に迷惑かけてる輩が
　少なくない。

　そんな連中を市場から駆逐するためにも、
　良い商品を作っている自信のある人ほど
　発信を強化すべき。

　
　職人気質の人こそ「世のため人のため」に
　マーケティングを学び、

　自社のビジネスを
　喧伝しなければならんのよ。

　-------------

　ワイ、小学生の頃からボードゲーム版の
　シミュレーションゲームの愛好者なんよ。

　昨日は1年ぶりに「ハリコフ攻防戦」を
　プレイしたことをきっかけに、

　バルバロッサ作戦（独ソ戦）をネットで詳しく
　調べ始めたら半日、潰してしまったわ笑。

　罪悪感なしに、こうした時間を自由に
　取れるだけで、起業して良かったと思ってる。
