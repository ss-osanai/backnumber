＜今日のピークパフォーマンス方程式＞

 【　万国のコミュニケーション弱者よ、団結せよ　】

───────────────────────────────────



■一昨日、昨日と、


　「コミュニケーション弱者のための
　　　　　　　コミュニケーション能力増進法」

　というテーマでお届けしております。

　今日も続けます。

　

■コミュニケーション能力が低い、

　と悩まれている方はたくさんいらっしゃる
　ようですが、その弱点を強みに変える方法論が、


　「コミュニケーション弱者のための
　　　　　　　コミュニケーション能力増進法」

　
　です。

　

■この方法は、以下の３つの項目にまとめられます。
　
　すなわち、
　

　１、自らの強みを自覚する（強みを決める）

　２、強みを発揮させ、学んだことを発信する
　　（読み手へのメリット提供を意識する）

　３、キャラを入れ込む


　この３つ。



■今日は、この２番目、


　「強みを発揮させ、学んだことを発信する
　　（読み手へのメリット提供を意識する）」


　について、お話しいたします。


　昨日、まず、あなたの内にある強み
　（これは相対的なものでよい）

　を見つけて下さい。自覚して下さい。決めて下さい。

　そんなことを書きました。

　

■次に大事なことは、

　その（相対的な）強みを外部に発信すること、
　です。


　自分の強みについて、ありとあらゆる角度から、
　それを言葉に載せて、プレゼンテーションするのです。
　
　なぜ言葉に載せるのか、
　それは言葉に載せなければ伝わらないから。

　（私を含めた、コミュニケーション弱者の場合）

　

■ノリとトークと雰囲気、押し出しで
　ガンガン攻めていける、

　「コミュニケーション強者」

　は、態度だけで、それっぽい雰囲気が出てくるもの。

　そのノリと雰囲気だけでおいしいところを、
　次々と刈り取っていくのです。

　

■その後を追いかけようとする

　「コミュニケーション弱者」

　の目の前には、強者によって徹底的に刈り取られ
　ペンペン草も生えていない不毛の大地が広がり、


　我々は、その光景を目の当たりにさせられ、
　呆然とするしかない、

　こんなことが過去、幾度も繰り返されて
　きたのです。



■しかし、

　コミュニケーションが苦手、
　初対面が苦手、
　
　という人でも、どんな因果か僥倖か、

　インターネット時代になって
　大きくチャンスが広がってきたのです。

　
　
■それは一人ひとりが低コスト（場合によってはタダ）
　で情報の発信が行えるようになったこと。

　このインフラを活用しない手はありません。


　時は来た！

　「万国のコミュニケーション弱者よ、団結せよ」

　なのです。

 　（別に団結しなくてもいいけど）



■我々はこのインフラにのっかって、
　ゲリラ戦を仕掛けていけるのです。

　そもそも「コミュニケーションが苦手」と
　いう人は、

　思考能力が劣っているわけではなく、
　言語化能力が欠けているわけでもない、

　ことがほとんどです。
　


■たまたま、


　ハイスピードで展開される会話についていく
　ための反射神経（運動神経？）が少々、遅れがち、

　とか、

　本来、誇るべきその繊細さが
　粗野で野卑な者たち（！）に蹂躙され、

　勢いに押され、二の句を告げなくなったり、


　という、ただ、それだけの理由のために、
　コミュニケーション能力が低い、

　というレッテルを貼られてしまう。



■そんな人たちに対し、弱者が逆襲に転じるには
　どうすればいいのか。


　リアルの会話では不利になりがちな、
　反射神経の鈍さを、逆手に取れば良いのです。

　粗野で野卑な者たち（！）が持たない、
　感受性の高さ、繊細さを武器にすれば良いのです。


　
■すなわち、

　ノリや反射神経のみで生きている人には
　競合できないレベルまで、

　自己との対話によって思索を深め、
　その思いや感じていることを文字にして発信する。



■さらに、

　その文章に、持ち前の繊細さを活用し、

　届ける相手への慮りを入れ込んだ文章を
　発信すればいいのです。

　
　
■これには、反射神経は必要ありません。
　
　（自分一人で行う作業だから）


　徹底的に、己の繊細さを文章に反映させればいい。

　（繊細さは、コミュニケーション弱者の武器）



■これによって、

　「弱みが強みに転じ変わる」
　
　のです。

　「最大の弱点が、最強の武器になる」

　のです。


　だからこそ、

　「万国のコミュニケーション弱者よ、団結せよ」

　と声を大にして、叫びたいのです。



■ふぅ。

　ちょっと興奮してしまいました。（笑）


　そういえば、今日の本題であった、


　「強みを発揮させ、学んだことを発信する
　　（読み手へのメリット提供を意識する）」


　について、まだほとんど語れていませんね。（汗）

　明日に続けます。
　
　


　今日も人生とビジネスを楽しみましょう。
