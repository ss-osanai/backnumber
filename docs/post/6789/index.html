■大別すると、世の中の人は

　「真剣になっている人
　（当事者意識を持っている人）」

　と

　「真剣になり切れていない人
　（当事者意識のない人）」

　のいずれかしかありません。



■厳密にいえば、両者はスパッと
　切り分けられるものではなく、

　どちらかといえば前者、
　どちらかといえば後者、

　みたいな感じでグラデーション状に
　なっているのかもしれませんが。

　
　
■そんな中、あえて、
　いずれか一方を選んで下さい、

　と問われたら、多くの人は
　
　「前者かな」
　「後者かな」

　と選択できるのではないでしょうか。



■前者を選んだ人は、相対的に

　「本気で生きている」
　
　のであり、後者を選択した人は

　「本気で生き切れていない」

　ことになるのかもしれません。



■どちらが成長するか、結果を出せるか、
　逓増するかというと、それはもう

　「真剣になっている人
　（当事者意識を持っている人）」

　に決まっています。



■真剣になっているからこそ
　身銭を切っているし、

　裏返せば、

　身銭を切ることによって
　より真剣さを高めていっている、


　そんな相乗効果が発動され、
　加速的に、螺旋階段状に、

　一気に成長を遂げていくのです。



■これはどんな分野においても
　当てはまりますし、

　おそらく、あなたも

　「真剣さ＆身銭を切る、
　　の相乗効果が発動されて、
　　螺旋階段状に、加速度的に成長」

　を遂げられた身近な方を一人や二人は
　ご存知なのではないでしょうか。



■真剣だから身銭を切るのか、
　身銭を切るから真剣になれるのか、

　の問いは、

　ニワトリが先か、卵が先か、の問いに
　近いものがありますが、
　

　答えは両方とも正解であり、

　両方が良い方向に絡まり合って
　本気度が高まっていくのです。



■この状態に入った人は
　パチンコ風にいうならば

　「確変モードに突入した」

　といえますし、そうでない人は

　「いいところまではいっても
　　結局、勝てない」

　状態が続くことになります。



■もしあなたが

　「真剣になっている人
　（当事者意識を持っている人）」

　の状態に入りきれないのであれば、


　もう一方の通用口である

　「身銭を切ることによって
　　真剣さを手に入れる」

　という方策を考えてみるのも
　一つの手かもしれません。



■一例を挙げれば、

　商売とはつまるところ、
　血と汗と涙の結晶であるカネを張って
　利回りを高めようとする営み。

　（商売を始めると、ジッとしているだけで
　　カネが流れていきます）


■まさに毎日のように

　「身銭を切る」

　痛みを味わい続けることに
　なります。

　
　
■それゆえ命がけにならざるを得ず、

　その当事者意識や真剣さや本気度が
　異常に高い学習効率をもたらし、

　高い学びの質と歩留まりは、平均を
　遥かに上回るリターンを生み出します。



■このような関係があるので

　「真剣に学べないし、取り組めない」

　という悩みをお持ちの方にとっては、


　【身銭を切りつつも、
　　確度高く勝てそうな土俵を見つけ、
　　
　　リスクを限定した上で
　　当該分野において勝負を仕掛ける】

　というアプローチは極めて効果的だと
　思われます。



■振り返ってみれば、

　私（鮒谷）もいくつかの音源で
　お話をしている通り、

　サラリーマン一年目のときから
　がっつり副業をやっておりました。

　（限定されたリスクを取り、
　　身銭を切ってあれこれやりました）



■そんな数々の経験が当事者意識や
　真剣さや本気度を引き出し、

　異常に高い学習効率をもたらし、
　

　マーケティングやセールスや
　コピーライティングやコミュニケーションや
　ビジネスモデル研究、

　歩留まりを約束してくれました。



■学びの集積は複利で効いてくるので
　蓄積されてくるにしたがって、

　平均を遥かに上回るリターンを
　もたらしてくれることとなりましたし、


　今のビジネスも、

　その頃から当事者モードで築いてきた
　基礎的な素養が土台となって、

　その延長線上に成立している、

■私（鮒谷）の場合は、
　本気になれたきっかけは

　「副業＆小商い」

　だったわけですが、

■つまるところ、
　きっかけはなんでもいいんです。

■なのでもし、あなたが

　「（リスクを極小化した形での）
　　不動産投資」

■であるならば、もう、

　「（リスクを極小化した形での）

　「大化けする可能性のある株を買う」
