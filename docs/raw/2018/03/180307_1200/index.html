　を開催いたします。



■私（鮒谷）は、これまでの半生を

　「人生・ビジネス・キャリアを、
　　どうしたら一大転換をさせられるのか」
　
　の研究に注ぎ込んできました。

　もはやライフワークとなっています。

　その研究記録は、5000号以上のメルマガの形で
　すべて公開もしています。



■もちろん私自身も、その研究成果を活用し、

　（大したレベルではありませんが）
　それでも社会人になりたての頃から考えると
　
　「想像もできなかったような環境」

　の中で生活＆仕事をさせてもらっています。



■その一番の理由を挙げると、

　「自己啓発に夢中になって取り組んできた」

　から、
　ではなくして（それは前提ではありますが）


　「自分の思いや能力や強みや考えを　　
　　発信することによって
　
　　【他の人と相乗効果が発動】

　　され、そこから人生が作り変わっていった」

　というのが本当のところです。



■ここは間違えられやすいところですが、

　どれだけ一生懸命、勉強して、
　努力して、頑張っても、

　「他者との出会い、関わり、つながり」

　なくして、変化することはありません。


　昨日のメルマガで書いた通り、
　そもそも人間、一人でいると肝心の

　「やる気、モチベーション」

　すら持続しないのです。



■だからこそ私（鮒谷）はこれまで、
　再三再四にわたって、

　「本当に人生を変えたいのであれば、
　　外に出て、出会いを求める必要がある」

　と、しつこくしつこく、お伝えしてきました。


　（そんな内容のメルマガバックナンバーだけでも
　　100や200はあるのではないかと思います）



■ただ、そうしてお伝えしてきたことも、
　
　「そのとき、思いついたことを単発で」

　お伝えしてきただけであり、

　「本当に人生を変えたいのであれば、
　　外に出て、出会いを求める必要がある」

　理由を体系的にお伝えしたことは
　ありませんでした。
