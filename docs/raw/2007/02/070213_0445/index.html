　【 プロローグ 】

　この本を出すことになって、一番初めにうかんだ父のことばは
　
　「人生は複利計算。ほんのちょっとずつでもよいから、毎日進歩せにゃ

　ということばでした。 

　複利計算の方法を知っている方ならば、おわかりになると思うのですが、
　この計算式での値を大きくするのに重要なのは、「時間」です。

　成長の度合いがたとえほんの少しずつだとしても、かける時間数が
　長ければ長いほど、最後にはものすごく大きな違いを生みます。 

　「時間」ときいて想うのは、これから輝く未来が待っている、
　子供達です。

　複利なのですから、早めに知っていればいるほど、もっとも大きな値が
　生まれるわけです。できればわたし自身も、今持っている感覚や知識を、
　高校生くらいまでには身につけておきたかった。

　そうしたらもっと早い時期に、望む未来をすべて実現できたことでしょう。 
　とはいっても、この本は単に「経営者」になるための本ではありません。

　わたしはもっともっと欲張りなのです。
　わたしは二児の母として、ふたりの娘には幸運を味方につけて生きてほしい。
　わたしの考える幸せとは、「自らを主人公として、人生を自在に
　コントロールする人」になることです。 

　かつてわたしにはとても良いお手本がおりました。
　その手本であるわたしの父の、強運を身につけるための言葉をひとつずつ

　彼は裸一貫、アルバイトから起業し、いま現在は700億円を売り上げる
　上場企業に育てた人。

　その強運に生きるコツには大人も子供も関係ありません。
　どうぞご自身のために、または大切なお子さまのために・・・

　どうぞあなたの人生にご都合よく、お役立てくださいませ。わたしの

＜今日のピークパフォーマンス方程式。＞

　【　コミュニケーションスキルにより、
　　　　　　　　　　　　　自分の求める結果を得るための環境を整える　】

───────────────────────────────────



■わたし（鮒谷）はメルマガ上でコミュニケーションのお話を
　よく書いていますが、これはビジネスに取り組む上で
　最重要と言っても過言ではないくらいのテーマであると思うので、
　自然と多くなってきているのかもしれません。

　全てのビジネスには、何らかの形でのコミュニケーションが発生
　しています。



■当然ですが、商売というのは人が営んでいます。
　決して木や石や水が勝手にお金を生み出しているのではないのです。

　お金はキャッシュの手渡しであったり、カード決済によるデータ交換で
　あったり、小切手や金券といった代替物を通してやりとりされたり
　するものですが、
　
　その端と端には必ず、人間が関わっているのです。

　人から人へお金が流れている、ということが、
　ビジネスにおける本質です。まずその点を理解することが重要です。



■その本質を理解しないまま、つまり、
　人の心の機微がわからないまま商売をするのは、

　高価なワインを無駄に一気飲みしたりするようなもので
　ちょっと理解するだけでその行動が価値のあるものになるのに、
　その権利を自ら放棄してしまっているようなものだとも思います。



■わたしは朝から夜まで、多くの方とお会いさせていただき、
　会話の機会が多いほうだと思いますが、１日の終わりにはいつも
　コミュニケーション下手にうなされてます。（そして今日も…）

　どれだけ勉強しても、どれだけ経験しても、

　「あの時こう答えればよかった」
　「なんでこんな返事をしてしまったのだろう」

　ということが少なからずあるのです。



■わたしはコミュニケーション能力が天性のものとして備わって
　いないので、日々悩み、苦しみなりながら勉強しています。

　なので、コミュニケーション能力を身に付けることに
　敏感にならざるを得なく、常にアンテナ立てており、
　メルマガでもその内容が多くなるのも必然のことなのかも
　しれません。



■コミュニケーションの種類にはいくつかあります。


　▼アナログでの１：１
　　　対面販売、面接、路上での会話など

　▼アナログでの１：多
　　　セミナー、講演、学校の授業、街頭演説など

　▼デジタルでの１：１
　　　個別送信メール、チャット、電話など

　▼デジタルでの１：多
　　　一斉送信メール、メルマガ、掲示板、ブログなど


　いずれのパターンにも、根っこにあるのは人と人。
　やはりコミュニケーションについては深く学ばなければならないでしょう。



■別のテーマの際にもお話しましたが、ある分野について学ぼうと
　思うのであれば、

　能力が自分にはない、下手だ、

　と自覚することが必要ではないかと思います。



■自覚があるからこそ、危機感から勉強をしようという動機が
　生まれるのだと思うのです。

　その能力が自分には充分ある、と傲慢になったり、勘違いをしてしまうと
　そこからさらに先へ行こうという意欲が不足し、それ以上の
　上達の足かせになってしまうのではないでしょうか。



■ビジネスにおける優れたコミュニケーションスキルというのは、
　

　【 自分の求める結果を得るために、
　　   　周囲の人たちが積極的に手伝ってくれるような
　　　　　　　　　　　　　　　　　　　環境を整えるスキル 】


　という言葉に置き換えられるのかもしれません。



■このスキルがもたらす結果は、ビジネスを営むあなたにとって
　大変喜ばしいものになることは間違いありません。


　▼いざというときには無理なお願いでも聞いてもらえる

　▼困ったときに相談に乗ってくれ、知恵や資源を貸してくれる

　▼有益な情報を積極的に提供してくれる

　▼自分のためにわざわざ時間を割いてくれる

　▼頼んでもいないのにお客を紹介してくれる


　まずはそういった結果を引き起こしやすい環境の整備が
　必要になってきます。



■これはテクニック的な部分もありますが、それを下支えする、
　思いやりのような気持ちも必要です。

　つまり、あなたの人生観や哲学も多少なりとも影響があるのでは
　ないかと思います。

　表面的な技術だけを学ぶのではなく、裏側に隠された本質を
　学ぶこともあなたのコミュニケーションスキルアップの力と
　なるでしょう。


■そのようなスキルを身に付けるうえで、頭に入れておくことで
　よい方向へ流れることができるポイントについて、
　明日、お話したいと思います。
