＜今日のピークパフォーマンス方程式。＞

　　　　【　若いうちから人脈開拓力を磨いておく。　】

───────────────────────────────────


■読みたい本をクリック一発ですぐに手に入れる環境になったので躊躇なく
　Amazonなどで本やらなにやら発注してしまうのです。

　巨大な箱が毎日のように到着してちょっと困っているのです。

　情報の海にわが身を浸す、というポリシーからすると好ましいことなの
　かもしれないけれど、でもせっかくだから、購入した本はさらっとでも
　目を通したいですね。今は事務所中が本やら資料やらビデオやらテープ
　やらで大変なことになってます。


■そんななか、速読も含めて毎日数冊づつ必ず読むように努めているのですが、
　ちょっと面白い本に出会ったのでご紹介。

　【今日のお勧め本　仕事のための12の基礎力】
　 http://www.amazon.co.jp/exec/obidos/ASIN/4822244091/2ndstagejp-22

　年代別にキャリアを切り開いていくために必要な能力を12にわけて
　詳述されています。

　もともとは、リクルートワークス研究所・大久保幸夫所長による年代別
　の「キャリアと能力」の育て方、という人気講演のエッセンスを本にした
　もの。


■年代ごとに、あるいはおかれている境遇、環境ごとに読んだ感想という
　のは異なるものなのだろうけれども、わたくし的には、

　第一能力　反応（リアクション）力
　第二能力　愛嬌力
　第三能力　楽天力
　第四能力　目標発見力
　第五能力　継続学習力
　第六能力　文脈理解力
　第七能力　専門構築力
　第八能力　人脈開拓力
　第九能力　委任力
　第十能力　相談（カウンセリング）力
　第十一能力　教授力
　第十二能力　仲介調整（コーディネート）力

　の中の、人脈開拓力、などもっと早い段階から意識して磨いておくべき
　だった、とつくづく思ったのです。


■今の学生さんとか見ていると動いている奴はほんとものすごい勢い
　で自前のネットワークを拡大していますからねぇ。

　時代背景による危機感からくる行動力、ITスキルを駆使したオンライン、
　オフラインを両用するネットワーク開拓力、情報の収集力にたけていて
　ものすごい勢いで、その年代のトップ集団は上の年代の、のほほん世代？を
　追い抜いてくるのではないかと多少の恐怖心をもってしまうくらいです。


■だから私は彼らと敵対せずに仲良くやっていこうと思っているんですが。
　みなさん、よろしくね。（笑）
　
　最近は、だからこの年代の人ともたまに飲んだり食べたりしてコミュニケー
　ションを図るようにつとめてます。
　情報収集は怠らないようにしないと…。

　ふんぞり返って年上を敬えという時代は遠い過去の話になったような。


■年代年代でこの本を読んで学びになることはあると思いますが、私などは、
　ここに書かれている「人脈開拓力」など、もっと早いうちに鍛えておくべき
　と繰り返しになりますがやっぱり思うのです。

　今にして思うとこれも一つのスキルなんですよね。

　
■「平成・進化論。」読者の中には学生さんもいれば、入社数年の、ほんとに
　これから、という人たちもたくさんいるようです。

　アタッカーズビジネススクールとか、いろんな集まりに顔を出して、
　若い人にあうと結構な確率でその年代の読者の方に会うのですが、
　そういう人にこそ、こういった本を読んで自らのキャリア形成について
　まじめに考え、まじめに取り組んでいってほしいとおもいます。
　
　何も指針なく動くよりも、きっと今年は○○の能力を身につける、
　来年は○○のスキルを磨くのだ、という目標のようなものをもっていた
　ほうが進化する速度は速いとおもいますよ。


■勿論、それ以上の年代の人にとっても、特にあまり自分のキャリア、
　将来のビジョンについて考えたことがない人にもお勧めの本ですね。


　【今日のお勧め本　仕事のための12の基礎力】
　 http://www.amazon.co.jp/exec/obidos/ASIN/4822244091/2ndstagejp-22
