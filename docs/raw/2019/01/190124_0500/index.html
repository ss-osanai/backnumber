【進化論】心のふるさと、小石川後楽園で私（鮒谷）が考えたこと
【進化論】「決意が実践につながらない」という、誰しもが持つ悩み




【実質無料でご招待】「丸一日、本当に大切なことだけに集中する会（第9回）」2月2日（土）に開催

2019/01/24 05:00 
■こんにちは、鮒谷です。


　最近、強く思うのが、

　人生において最も高い生産性（＝波及効果）を
　手に入れるためには

　【本当に大切な、
　　少数のことに取り組む時間】

　を大切にすべきであるということ。



■ここでいう

　【本当に大切な、少数のこと】

　とは、


　【緊急ではない（＝今すぐ、取り組まなくても
　　さしあたって問題はない）けれども、

　　先送りしていると、徐々に不幸を招き寄せる、
　　人生において重要な取り組み（＝第二領域）】


　のことを言います。



■にもかかわらず、
　私たちは往々にして

　【第一領域（緊急かつ重要な要件）】

　あるいは、

　【第三領域（たいして重要ではないけれど
　　急ぎの案件＝割り込みの電話やメール返信等）】

　に追われがちで、

　【真に大切なところに時間を割く】

　ことがなかなか困難です。



■それに加えて、

　「今やるべきことは、これではない」

　と思いながらも、

　分かっているけど止められず、
　惰性でスマホを見てしまったり、

　（＝緊急でも重要でもない、
　　　第四領域）

　などという人も少なくないのでは
　ないでしょうか。



■「小人閑居して不善をなす」

　という言葉がありますが、

　人間、一人でいると、
　ろくなことをやりません。


　だからこそ、

　大勢の、他者からの視線を感じられる
　環境に身を置くことが大切である、

　とこれまでも何度も書いてきました。



■その重要性を理解し、

　限られた自由時間を意義深いものと
　したい方のために、

　当社セミナールームを開放し、
　久しぶりに


　【丸一日、本当に大切なことだけに
　　集中する会（第８回）】


　を開催することといたしました。



■この場に参加頂きたい方は、
　次のような方々です。


　▼目標設定の大切さや波及効果は理解しているが
　　ついついを先送りしてしまう方

　▼今後の事業戦略を立てようと思いながら
　　なかなか手をつけられない方

　▼新しいプロジェクトについて時間を取って企画を
　　立てようと思いつつ、いつまでも進められない方

　▼資格試験、その他の勉強をしようと思いつつも
　　なかなか集中して学ぶ場を作れない方

　▼読みたい本がたくさんあるのに、普段は
　　読書をする時間を取りにくい方

　▼せっかく購入した音源なのに、聴く時間や
　　環境を確保しづらいという方

　▼動画学習をしたいけれども、落ち着いて
　　視聴できる環境のない方


　こうした方が参加されれば、必ずや


　【最優先すべき事項に対して、
　　実際に時間を天引きし、

　　それが形になる喜びを実感できる】


　こととなるはずです。



■こうした喜び（快感）を、

　強制的にでも
　身体に覚え込ませたことのない人は、


　いまだ、その喜びを知らないのだから、
　まずい、まずいと思いつつ、


　【いつまで経ってもずるずると、

　　（大切なことを疎かにして）
　　どうでもいいことを続けてしまう】


　という惰性のループから
　離れられなくなってしまうのです。

　
　そんな悪循環の輪を解き放つための
　最高の一日（価値）を、

　この場でご提供できればと思います。



■本来は、気心の知れた仲間とともに、

　温泉合宿、あるいはホテルでの
　缶詰合宿などができればいいのでしょう。

　（ちなみに「一人合宿」はお勧めしません。
　　理由は述べませんが、、、経験者談（泣））


　ただ、やりたくても、

　時間的、金銭的に実現できない、
　あるいは、一緒に行えそうな人もいない、

　といったことはありそうですね。



■そうした

　【強い、充足されていない潜在ニーズ
　　（声なき声）】

　には以前より気づいていたのですが、


　実際に過去８回、参加下さった方々から
　頂戴した、熱いご感想を拝見し、

　こうした機会を提供する価値の大きさに
　驚かされました。


　【「集中する会」にご参加頂いた方から頂戴した、
　　熱いご感想集】
　　https://ssdlfiles.s3.amazonaws.com/180224_testimonials_mokumoku.pdf


　※ぜひ、ご感想読んでみて下さい。ご感想自体が
　　「気づき・学びの宝庫」となっております。


　ご感想を熟読してみると、

　如何に私たちが普段、いろんなところに
　認知能力を奪われてきたかが、

　よく分かりますね、、、



■繰り返しになりますが、
　そんな時代にあって

　【最優先すべき事項に対して
　　時間を天引きし、それが形になる喜び】

　を、一度でも体験したら


　【自らの意志で時間を天引きし、
　　一つ一つを形にしていく喜び】

　を身体で味わいますから、
　自分でもできるようになるはずです。



■それができない（その喜びを知らない）人は
　いつまで経っても、

　「ダラダラと、惰性の毎日を続ける」

　こととなるのです。


　そんな現状を打破し、新しい一歩を
　踏み出すための場をご用意することも、

　当社の提供できる一つの価値ではないか、
　と考えた次第。

　お取りいただいても構いません。

■合わせて、せっかくですので、

　この時間を最大限有意義に
　活用していただくべく、


　お申し込みくださった方には、

　【鮒谷が生産性を最大限高めるために
　　心がけてきたいくつかのこと】

　というタイトルで、


　【圧倒的な生産性向上のために役立つと
　　思われるテーマ】

　について鮒谷が放談いたします。
