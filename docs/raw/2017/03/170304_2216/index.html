■話は変わりまして、

　昨日、ご案内した日記放談会についてですが、
　早速、お申込みを頂戴しております。


　昨年、同テーマで開催した会の終了後、
　頂戴したご感想を、
　
　内容のイメージを付けていただくために
　以下にご紹介。


　（ここから）
　───────────────────────　

　日記というものの定義の仕方が
　まったく私と異なっており、衝撃を受けました。

　───────────────────────


　このたびは貴重なお話をありがとうございました。

　ちょうど1年前から紙の日記を書き始めており、
　燻っていたところにセミナーの案内があったので、

　躊躇せず申し込ませていただきました。
　

　日記というものの定義の仕方が
　まったく私と異なっており、衝撃を受けました。

　「意思決定基準としての日記」
　また「感情のコントロールのための日記」

　を書くことができるよう、
　徐々にシフトしていくことにします。


　言葉の限界が思考の限界となってしまうので、

　まずは圧倒的にたくさんの言葉に触れることが
　必要だと感じています。

　そのためには読書も必要ですし、セミナーに出たり、
　外の世界に出て行かざるを得ません。

　そうなると、ムダなことに時間を費やしている暇も
　なくなりますから、

　言葉に対する意識を高めることで、結果として
　人生がよい方向に向かっていくといえそうです。


　脳内をいい言葉で埋め尽くし、

　人生のレールをとっぱらい、
　自由自在に人生を謳歌できるよう、

　言葉というものに対する意識を高く持って
　過ごしていこうと決意しました。


　また、

　「圧倒的な量稽古でしかパターン認識できない」

　ということを改めて鮒谷さんの口から
　聞くことができ、感無量でした。


　まさに私が受験生に向けて日々伝えていることと
　共通していたので、

　自信を持って伝え続けていくことにします。


　個人的には、一番最後にお話しされていた

　「社員がやる気にならないんです」

　と嘆く社長の話がものすごく刺さりました。


　言葉の話とは違うところですが、トップが800％になって
　初めて社員が100％になるというのは、

　様々なところに通じるものだなぁと思います。


　私も100％で仕事をしていますが、それだと半分か
　それ以下しか相手に伝わっていないのかもしれませんので、

　これからは800％のエネルギーを持って
　仕事に取り組むつもりでいきます。


　本当にありがとうございました！ 


　（公認会計士 尾崎 智史 様）


　───────────────────────
　（ここまで）


　尾崎さん、いつもありがとうございます！
　
　そして突然の、昨年のご感想紹介、

【 「理想のジョブチェンジ」は日記を記すことによって実現する 】
　　　　
───────────────────────────────────



■昨日のメルマガでは、

　「自分の金鉱脈（強み）は
　　日記を書くことによって発見される」

　という話をいたしました。

　今日も続けます。



■日記は書き続けていくうち、
　その性質上、必然的に

　「自分が何者であるのかを
　　問いかけ続ける仕組み」

　として機能するようになるわけですが、

　
　ここから生まれる

　「繰り返しの自問自答」

　が

　「自分再発見への旅」

　へと誘（いざな）い、

　自分という存在への理解を深めさせて
　くれることとなるでしょう。



■昨日も書きましたが、こうした

　「自分再発見の旅」

　を行う中で、

　「（無数の）強みのタネ」

　が見つかり始めます。


　そのタネを守り、育て、
　伸ばすことによって

　（ときにタネが腐ったり、せっかく出た芽が
　　枯死することもありますが）

　一つ一つの「強みのタネ」が
　「強み」になり、

　さらには「圧巻の強み」へと昇華する。



■こうして自分自身が
　
　「たくさんの強み（訴求ポイント）」

　を手に入れられるようになるのです。


　この流れを人為的、自覚的に作り出すに
　あたって、

　日記は極めて効果的なツールとして
　機能することは、

　今や疑いの余地がなくなりました。

　（私自身が長年、実践してきて
　　身体で理解いたしました）



■慣れてくると

　「日記は１日１回、夜寝る前に」

　という、

　誰が決めたか知らないけれども
　一般的には

　「そのようなもの」

　と思われているルールから離れ、
　
　「毎日、何回も日記を書く」

　ようになってくるかもしれませんね。


　
■なぜなら、そのほうが、

　より多くの「強みのタネ」を、
　より早く確実に「強み」に育て上げ、

　さらには「圧巻の強み」へと導けることが
　分かってくるからです。



■こうした習慣（武器）を手にして
　自らの意志で強みを増やそうとしている人と、
　
　強みは、いつかどこからか、
　主体的な意志とは別にもたらされるもの、

　と思っている人との間では、


　成長の速度は変わってくるし、
　手に入れられる結果も異なってくるのも、
　
　ある意味、当然のことかもしれません。



■付け加えると、昨今、

　対人コミュニケーション能力の欠如に
　悩んでいる人が多い、

　と感じているのですが、


　これなどはまさに（適切な方法で）
　日記を書くことによって克服できる、

　と考えています。



■正しい方法で日記を書いて、
　然るべき複数の強みを育てられれば、

　
　１、自信と自尊心が育まれるから、
　　　コミュニケーション上の態度に現れる

　　　（自信を持って他者と向き合える）


　さらには


　２、自ら求めてコミュニケーションを図らずとも、

　　　先方からコミュニケーションを求められ、
　　　合わせて頂けるようになり、

　　　（求める存在から求められる存在へと
　　　　ジョブチェンジ！？することとなり）

　　　人間関係構築が格段にラクになる


　ということも起こるでしょう。



■そんなことを考えると、
　
　「（日記を書くことによる）
　　強みの発見・伸長から生まれる自信・自尊心」

　と

　「コミュニケーション能力」

　とは密接不離の関係にあるのではないか、

　とすら思います。
　
　
　またこうして強化されていく
　コミュニケーション能力は、

　そのまま営業力や交渉力にも
　直結してくることでしょう。



■つまり、多くのビジネスパーソンが
　求めてやまぬ、

　「コミュニケーション能力や
　　営業力や交渉力」

　は日記を適切に書くことによって
　手に入れられる、

　ということになるわけです。


　（私自身が、意識して、意図して、
　　このようなプロセスを経て、

　　コミュニケーション系の能力を獲得し、
　　さらには磨き上げていきました）
　


■結論としては、

　「コミュニケーション能力や
　　営業力や交渉力を手に入れたければ、

　　日記を（適切な方法で）書きましょう」

　ということです。


　このメカニズムを知悉し、
　自在に操れるようになると、

　文字通り、人生が変わります。


　そして、これだけではありません。

　日記の効用は、まだまだあるのです（明日に続く）



■こうして私（鮒谷）が、
　約9000日かけて編み出した、

　独自の（！？）方法論に
　ご関心をお持ち下さる方がありましたら、


　私が確立してきた

　「根拠を伴った、実証済みの、

　　コミュニケーション能力を強化し、
　　人生を劇的に転換させる日記の書き方」

　についてお伝えする、
　以下の放談会にご参加頂けましたら幸いです。
