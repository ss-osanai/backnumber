■私（鮒谷）は、

　ビジネス（セールス）においては、

　【徹底的に説明責任を果たす】

　必要があると考えています。


　売れない、買ってもらえないのは、

　お客さまが悪いのではなく、
　理解力が不足されているのでもなく、


　ただただ、

　「売る側が、お客さまに分かって頂ける
　　ところまで説明していない」

　だけの話です。

　「もう、これ以上、
　　何も説明することはない」

　その上でお求めにならないのなら、

　「もともと、
　　そのお客さまには需要がなかった」

　ということですから
　諦めもつくというものです。

　なかなな売れない、
　買ってくれない、

　それは説明責任を果たさなかった者に
　原因があるのです。

　お客さまにお届けしようと思ったなら、

　理解頂けないし、お求めにもなられない、
　
　と思うから、

　【商売人としての説明責任を果たすため】

　説明を重ねてきたわけです。

　それでもお求め頂けない方が

　「事業の立ち上げや、自らのビジネス、
　　キャリアのイノベーション」

　といったテーマについて、
　
　ご興味をお持ちではないということですから、
　諦めもつくというものです。

　売れない、売れないと言っていても、

　「やりきった」

　わけではありませんから、
　気持ち悪さが残るもの。


　それが嫌だから、
　私（鮒谷）は、原則として、

　「ご案内する全ての商品、サービス」
