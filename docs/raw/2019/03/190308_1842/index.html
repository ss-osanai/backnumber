　ベンチャー業界の頼れる兄貴的存在
　スダックスの若き日の物語

　いまやベンチャー／スタートアップ界隈で
　著者の存在を知らない人はいない。

　それくらいの知名度と活躍ぶりの著者が若き日に
　ビジネス、恋愛の双方で猛烈に突っ走った自伝的物語。


　--------------------------------


　エンジェル投資家ならぬ
　「エンジェル労働家」として、

　ITベンチャー／スタートアップ業界で
　名を馳せるスダックスのデビュー作。


　ロスジェネ世代にとっての社会人生活について、
　公私のエピソードが実感をもって綴られていく。

　昨今、時代の要請も相俟って
　市民権を確実に得ている起業。

　そんな状況から離れたひと昔前の壮絶かつ
　抱腹な事態が次々と繰り出される。

　一方で、バランスをとるように求められた
　プライベート。


　観察力あふれる細やかな表現描写から、アドバイザーと
　しての力量も十二分に窺い知れるだろう。

　平成最後の早春に世に出た青春実録記、
　世代を問わず手にとってみるべき一冊である。

　一般人だったことがわかり勇気づけられる本
