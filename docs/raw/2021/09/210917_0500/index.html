■経営をされている方なら
　今さらの話だとは思いますが、

　税務について、

　「悪気はなくとも、

　　知らなかっただけで、何十万どころか、
　　何百万（ことによるとそれ以上）の損失」

　ということは、普通によくある話です。


　※私（鮒谷）も、
　　もちろん何度も痛い目に遭いました。
　
　　でも、

　　「不勉強だった自分が悪い」
　　「知らなかった方が悪い」

　　という話なんですよね、、

　
　私と同じように、
　すでに痛い目にあって、

　このことを分かり過ぎるくらい、
　分かっている方もあるはずです。



■そんなわけで、

　「一円だって損はしたくない！」

　という経営者・起業家（起業志望者）であれば、

　今回の吉澤さんプライベートセミナーの
　音源が

　「極めて費用対効果の高い投資」

　となるのは間違いありません。



■なんといっても「たったの28,000円」で、

　今後、場合によっては
　何十年も経営していくにあたって、

　【累計すれば、少なく見積もって数十万、数百万、
　　あるいは、きっとそれ以上】

　の、納める必要のなかった税金を、
　知らずに納めてしまっていた、

　といったことが激減するであろう音源です。



■特に「今だけ有効！」みたいな
　期間限定の話ではなく、
　
　【そもそも税金とは、節税とは、税務調査とは】

　といった本質的な話を徹底して語られているので、
　普遍性、汎用性がかなり高いものですし、

　実際に参加者からも圧倒的な高評価を頂戴した
　セミナーとなりました。
