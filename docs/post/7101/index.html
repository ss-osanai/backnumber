　（ここまで）



■改めてご感想ありがとうございます！

　まさにこういった変化を期待した
　音源なんですよね。


　今日の年間プログラムでも
　話をしていたのですが、

　身体＆行動に編み込めてこその
　人の変化であり、

　どれほど知識を蓄えても
　実践されなければ意味がありません。



■ここにおいて

　【そこに格闘はあるんか？】

　という問いが生まれるのです。

　格闘なくして、変容なく、
　変容なくして、変化なし。



■知識を頭に入力するために
　格闘は必要ありませんが、

　身体に埋め込むためには、

　「気持ちを込めなければ
　（＝気を入れなければ）ならない」

　のはお分かりかと思います。



■とはいえ、思春期くらいまでに、
　たとえば

　「前倒し習慣」
　「周りの人に役立とうとする気持ち」
　「いつも明るく、いつも元気で」

　といった姿勢や態度や精神が
　身についていたら、

　こんな取り組みは不要です。

　（そういう羨ましい人が
　　ときおりいらっしゃるんですよね）



■しかし、

　一定の年齢に達し、
　おおよそ人間形成ができたところで、

　後天的に上書きしようと思ったら、
　やっぱり知識を入れるだけではムリで、

　気持ちを込めて、丁寧に
　身体に覚え込ませなければなりません。



■つまり、

　勝ったり、負けたり、
　負けたり、負けたり、
　負けたり、勝ったり、

　といったことを繰り返し、

　（こうすれば
　　人生が良くなっていくだろう）

　という知識を、

　知識に留めるだけではなく、
　身体に完全に編み込めるところまで

　【格闘し続ける必要がある】

　のです。



■継続して戦い続うのは大変ですから、
　そこから逃避する言い訳としての

　【学び（お勉強）】

　に思い当たる節のある方は
　少なくないはずです。



■学んでいる限り、
　なにか取り組んでいる感じはする、

　（＝自分に許しを与えられる）

　わけですからね。


　しかし、学んでいる感じがしたところで
　実生活は何も変わりませんし、

　当人も心の深いところでは

　「そうじゃない」

　と気づいているはずです。

　そして実際に
　「そうじゃない」のです。



■当音源もそうですし、
　年間プログラムはなおそうですが、

　こうした

　【自分一人では矯正できない、
　　知識の編み込み（格闘）】

　の機会や環境を提供するために、

　ご用意しているコンテンツや
　サービスということになります。



■つまり当音源は、

　多くの人にとっての
　学びに対する概念を根本から覆し、

　「お勉強」から
　「身体に編み込むための学習」へと、

　パラダイムと行動習慣の転換を
　目的としてリリースしたもの。



■スケートリンクの上を滑るような
　（つるつるして定着しない）学びではなく、

　身体に編み込む
　（基本的な所作を確立する）学びを。


　これが同音源を通し、

　休憩なし、３時間半以上の時間を使って
　私（鮒谷）がお伝えしたかったことの、

　中心的なメッセージとなります。



■こんな時代だからこそ、
　この音源を聴き、

　【波及効果の高い、
　　一つ一つの意思決定の基準を、
　　確実に身体に編み込む学び】

　の価値と力を再発見し、

　地に足のついた実践を通して
　重心の低い姿勢を手に入れて下さい。


　今日も人生とビジネスを楽しみましょう！

　【今日のピークパフォーマンス方程式】

　　■変化・変容に必要な意思決定基準は、

　　　知識として蓄えられるだけではなく、
　　　身体に編み込まれてこそ、

　　　初めて期待した成果がもたらされる。

　　■そのためには「気を入れて、格闘する」
　　　ことが必要だ。

　　■多くの人はそれを面倒がったり、
　　　億劫に思ったりして

　　　「お勉強に逃避」

　　　することとなる。

　　■お勉強している限り、自分に言い訳が立ち
　　　許しも与えられるから。

　　■しかしそれで変われるはずがないと
　　　当人も深いところで理解しているもので、
　　　実際にその通りなのである。
