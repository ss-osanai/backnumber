■成果を上げたいのであれば、

　「すべてを自分でやろうとしない」

　と決めることが重要です。


　言葉を換えると

　「バカになれ」

　「バカにされることを恐れるな」

　ということになるでしょう。



■もちろん、

　組織に所属する人においては
　そんなことを言っていられない、

　と怒られる人もいるかもしれません。

　その状況も分かります。


　ただ、その上で、

　「できねーこと、いうな！（怒）」

　といって突き放すのではなく、

　現実に即した形で、
　できる限りであっても

　原理原則に忠実であろうとするほうが
　はるかに大きな成果を出せるはず。
　


■私（鮒谷）は、

　「すべてを自分でやろうとしない」

　どころか、さらに一歩先を行き（？）

　「自分にしかできないこと以外は
　　決してやらない」

　と決めています。


　
■詳しくどのように生きているかは、
　どこかのタイミングでリリースするであろう、

　「鮒谷周史の、
　　圧巻！タイムマネジメント放談会」
　
　の音源（笑）をお聴き頂ければ
　理解してもらえるかと思いますが、


　頑（かたく）なに、

　「自分にしかできないこと以外は
　　決してやらない」
　

　逆から言うと、

　「ほとんどの能力は、
　　平均的な人から見ても無能」

　であるという状態を、あえて、わざわざ、
　意識して目指してきたわけです。



■一例として、たとえば私（鮒谷）は、

　パソコンで、ファイルを保管したときに、
　一体、どこに保管したのか分からないし、

　どうせワードもエクセルもパワーポイントも
　使えないのだから、

　マイクロソフトさんに
　カネを払うのも嫌なので、

　ただで配られている「ビューワー」で
　ファイルが見られればそれでええわ！

　というのも、その一つ。

　（そんなわけで私のパソコンには、
　　MSオフィスは入っておりません）


　
■もちろん、パソコンをプロジェクターに
　接続してプレゼンしたことなど、

　人生において、

　「ただの一度も」

　ありません。

　できないし、やったこともないし、
　やる気もないし、金輪際、やるつもりもない、

　のないないづくしの、ダメ人間。



■こんなのは、普通の人から見たら無能の証。

　でも、別にいいんです。
　無能といくら言われても。


　できないし、そもそも、
　やりたくもないことをやるくらいなら、

　自分が圧倒的にブチ抜けられることだけを
　徹底してやりきったほうが、

　遥かに精神衛生上、よいし、


　なによりも、そうするから

　「ある一つの特殊な能力だけが
　　磨かれ、鍛えられ、突出する」
　
　こととなるし、結果として

　「異形の者」

　となることができるのです。



■これだけ競争の激しい社会において、

　他者に劣後せず、
　有利な立ち位置に身を置きたいのなら

　「異形の者」
　
　と認識されるのが一番です。


　異形の者を見たときに
　人は注目せずにはおれなくなるのです。

　平均的な人の中に紛れ込んでいる
　平均的な人、

　では、今の社会において最も価値ある

　「アテンション（注目・関心）」

　を集めることはできません。



■あなたが、このクソ長いメルマガを
　毎日であれ、たまにであれ、

　お読み下さっている理由の一つには、
　私（鮒谷）が

　「異形の者」
　
　であるからであり、
　
　気になってしまっているからであることは
　おそらく間違いありません。

　（もしそうなら、
　　私の術中（＝アテンション獲得戦略）に
　　陥ってしまっているということですね）
　


■「アテンション・エコノミー（関心経済）」

　という言葉があるくらいなわけだから、

　「アテンションを集めるために
　　異形の者を志向する」

　そんな生存戦略があったって
　おかしくないでしょう。


　どころか、今の時代にあって、

　「相当、生存率の高い、
　　スジの良い戦略」

　であると確信を持って、


　私自身が実験台となり、
　十数年の長きに渡って、

　「人のやらないことだけをやり、
　　人のやっていることは死んでもやらない」

　と決めて徹底して行ってきたわけです。



■その中間結果がでてきた頃ですが、
　今のところ、

　「狙いはズバリ、的中した」

　と断言していいかと思います。

　誰かにできる人にお願いしたり、


　場合によっては
　各人の専門能力について私と同じく、

　「異形の者」
　
　レベルで磨き上げた人があったとすれば、
　
　「物々交換ならぬ、スキルスキル交換」

　までできれば最高です。
