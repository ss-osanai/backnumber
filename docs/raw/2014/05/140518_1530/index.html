　Ｍさんご夫妻、ありがとうございました！

　（ここから）
　----------------------------------------------------------------


　本日はありがとうございました。

　先日の3日間セミナーにて鮒谷様とお会いして、まず

　「妻に紹介してみたい！」

　と思い、個別コンサルを申し込みました。


　今回私は、二人の話を聞くことに徹していたのですが、
　妻が、今までに聞いたことの無いようなことを
　活き活きと話す姿をみて、

　今回申し込んでよかったな、と思うとともに

　「妻は、こういうことを考えていたのだな」

　という発見がありました。


　私も、もっと「人の話が聞ける」ように
　ならねばならないと強く感じました。

　（幅広い教養、コミュニケーション能力、
　　言葉のセンスを磨かなければ！）

　いつも新しい気づきをありがとうございます。
　今後とも、よろしくお願いいたします。


　　　　　　　　　　　　　　　　（エンジニア　Ｍ．Ｍさま）


　----------------------------------------------------------------


　鮒谷様


　Ａです。

　本日は貴重なお時間を頂きありがとうございました。
　たくさんの収穫を得て、ほくほくと帰宅して参りました。

　今まで、私が興味ある分野・事柄は、ビジネスにするには
　収益性もないと漠然と考えていました。


　今日お話しできて、自分が先入観に囚われていたことに
　気づきました。

　核として足りるものがあると、
　肯定した頂いたこともですが
　
　ビジネスにチャレンジすることへの
　敷居を下げていただきました。


　興味がある分野への知識や体験を集めるだけ集め、
　燃えにくい豆炭になっていましたが、今日着火しました。

　明日から、何か一歩一歩、
　収益に向けて何ができるかを考え、

　今、自分に足りないものを洗い出して
　埋めていこうと思います。

　同時に、今日、お会いして
　お話しできたことが楽しかったです。


　これまで自分の考えていることを
　アウトプットすることはありませんでした。

　自分の考えていることを話すこと、
　それに対して反応を頂けること、
　意見を引き出していただくこと、

　どれもが新鮮で勉強になりました。


　話を聞く技術はまだまだこれからだと思うので、
　まずは人に伝えるのに十分な言葉を持てるよう
　言葉を書き溜めていこうと思います。

　帰りの電車では主人とたくさん得たものについて
　話をしました。
　
　まずは０を１へ。

　またお話しできるほど何かを進めて、
　お会いしたいと思います。


　ありがとうございました。


　　　　　　　　　　　　　　　（Ａ．Ｍさま）

　改めてありがとうございます。

　「今日のピークパフォーマンス方程式」

　　最高の結果を発揮するための方程式）

＜今日のピークパフォーマンス方程式＞

 【　ノウハウの全てを開示する、という大冒険　】

───────────────────────────────────



■一般には、

　「自ら知り得た、あるいは体得したノウハウは
　　極力、秘匿する」

　という考え方が普通でしょう。

　そのノウハウが競争力の源泉となるからです。

　
　今日は、

　そんな経済的利益の源ともなるべき、
　自らのノウハウ一切を開示したら、どうなるか、

　について一つの思考実験を行います。



■自らのノウハウを開示して、開示して、開示しまくると
　どうなるか。


　その答えは


　・

　・

　・

　・

　・

　・

　・

　・

　・


　「空っぽ」になります（汗）



■すべてのノウハウをぶちまけてしまうと、
　当然ながら、
　
　「他との差別化、差異化を図る要素が
　　ゼロになる（空っぽになる）」

　わけです。


　それは困る、ということで通常、

　「せっかく得られたノウハウは秘匿しよう」

　という力学が働くのです。



■では、ここで反対に、


　「あえて、全部ぶちまけて真空状態にしてしまい、
　　その吸引力を活用して、自らはその先を行き、
　　
　　他の追随を許さぬ、さらに高度、高々度の
　　ノウハウ開発に注力しよう」


　という考えを持ってみてはどうでしょう。



■自らの成長、という視点で考えても、
　純粋に仕事が楽しめる、という観点から見ても、

　個人的には、明らかに

　「後者に軍配が上がる」

　と思われるのですが、いかがでしょう。



■そもそも、

　自らのノウハウを秘匿して、
　なんて言っている時点で、

　それこそ己のケツの穴の小ささを
　全力で開示しているようなもの。



■そんな恥ずかしいことをするくらいなら、


　「ノウハウはすべて駄々漏れにするから、
　　もし模倣できるなら、模倣してみてはいかが？」
　
　くらいの自信や余裕や誇りが欲しいものですし、


　そんな姿勢こそが
　新しいノウハウを生み出し続ける、

　「（ビジネスにおける）横綱相撲」

　につながっていくのではないでしょうか。

　なによりもそんな姿勢は格好いい。



■こうした、
　一番一番の取り組み（横綱相撲）こそが、


　「高邁な理想を掲げ、妥協を許さず、
　　未来を志向し続ける存在」

　としての認知を高め、新しいビジネスを引き寄せる
　原動力ともなるはずです。
　


■なんて偉そうなことを書きながら、
　ついつい

　「せっかく編み出したノウハウなので、
　　これは秘密にしておくかー」

　なんて誘惑に負けてしまう、
　ケツの穴の小さな私（恥）



■せめて個別のコンサルティングでは、
　そんなことのないようにしようと誓っています。

■私（鮒谷）はこれまでこのメルマガ上で、

  「成功したいと思ったら行動を変えよう、そのためにはまず、
    思考レベルで変わることが大切」

  そんなことをたびたび記してきたかと思います。

■行動につながっている自らの思考、そしてその背景にある、
　「深く根ざしたもの」を知ることは、行動を変え、成果へとつなげて
　いくにはとても重要なステップといえるでしょう。

■堺屋太一さんのこのＣＤ、さっそく私も聞いてみましたが、
 「だから自分は○○だったのか！」と納得できる話が満載でした。

  自分とは無関係に思えた、歴史上のあの人物のあの行動が、
  何百年もの時を越え、こうも深く影響を受けているとは衝撃です。

■歴史を学びながら、自分を再発見できるという貴重なこのＣＤ教材、
  残念なことに書店には置いてないとのこと。

  果たして、堺屋さんの選んだ「日本を創った１２人」とは？
