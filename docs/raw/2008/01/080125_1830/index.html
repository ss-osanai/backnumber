■わたし（鮒谷）が、ビジネスで気をつけているのは、
　「自分のなかに成功者と同じエンジンを積んでおく」こと。
　ほかに気をつけていることはありません。これだけです。

■とてつもない結果を出している経営者から、話を聞く。
　これで、彼らと同じエンジンが、いとも簡単に手に入ります。
　すると自分の才能とは無関係に、思い通りの結果が出せてしまうのです。

　これに気がついてから、自分の資源の多くをセミナー参加に注ぎ込むように
　なりました。そしてこれが「最も効果的な投資」にして「最重要戦略」。

＜今日のピークパフォーマンス方程式。＞

　【　学習を行動に落とすことで次の学びに移ることができる　】

───────────────────────────────────



■ただ学んでいるだけは、向上することができません。
　学んだなら、それを行動に移す必要があります。

　また、その逆に、行動だけでもだめなわけです。

　精度の低い行動をいくらしたところで、
　労多くして功少なし、ということになるだけですから。



■行動の前に学び、学びのあとには行動する。

　学びと、行動と、その両方が必要なのです。

　昨日は、そんなお話でした。



■この、


　【 学び → 実践 → 学び → 実践 → 学び → 実践 … 】


　を繰り返していると、
　いいことが起きてきます。


　「学びの段階」

　と、

　「行動の段階」

　とがレベルアップしていくのです。


　例の図で表すと、こんなかんじ。



                              ←──────■ [ 第８段階 ]     
                                           学び                 
                                            ↑                  
 [ 第７段階 ] ■──────実践─────→┘                  
             学び                                               
              ↑                                                
              └←─────実践──────■ [ 第６段階 ]     
                                           学び                 
                                            ↑                  
 [ 第５段階 ] ■──────実践─────→┘                  
             学び                                               
              ↑                                                
              └←─────実践──────■ [ 第４段階 ]     
                                           学び                 
                                            ↑                  
 [ 第３段階 ] ■──────実践─────→┘                  
             学び                                               
              ↑                                                
              └←─────実践──────■ [ 第２段階 ]     
                                           学び                 
                                            ↑                  
 [ 第１段階 ] ■──────実践─────→┘                  
             学び                                               



■第１段階の学びを終えたからといって、

　すぐさま学びのレベルをアップさせることは
　できません。

　第１段階の「行動」が分からなければ、
　第２段階の学びに入れないのです。

　車の教習みたいなものですね。



■いくら第１段階の学科を丸暗記したからとっても、

　いざ実技になったとき、アクセルとブレーキをまちがえて
　踏んでいるのでは第２段階に進めませんし、


　「踏切の手前では窓をあけて音をきく」

　と頭で知っていても、

　運転のときにそれをスッ飛ばしてしまったら、
　やはり次の段階に進ませてもらえません。


　【 学習によって頭で覚えたことが行動によって身について

　　　　　　　　　　　そこで初めて、次の学びに移ることができる 】


　のです。



■学びと実践を繰り返し、

　あせらず、着実にレベルアップをはかっていくべき。


　今日もビジネスを楽しみましょう。
