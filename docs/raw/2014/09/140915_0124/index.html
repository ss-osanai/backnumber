■こんにちは、鮒谷です。
　

　昨日は終日、「年間プログラム」セミナー開催。

　朝から夜遅くまで11時間。



■話す側の私は一向に問題ないのですが、
　聞かれる皆さんは毎回、一日が終わる頃にはぐったり。


　そのぐったり度合いで、

　「その日、参加者の皆さんの脳を
　　どれだけ混乱させることに成功したか」

　を判断しています（笑）

　
　「混乱からしか、飛躍は生まれない」

　と信じているからこそ、昔から、

　参加者がぐったりするほど
　価値を提供できた、

　と思って、やってます。




■それでは

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。

＜今日のピークパフォーマンス方程式＞

 【  「イイハナシダナー」で終わらせないために  】

───────────────────────────────────


　
■量もこなせていないのに、

　「質を追求しよう」

　と、本気で思えるはずがありません。


　「質」の追求は、「量」を追求したあとに、

　「これじゃ、きりがない」

　「このままじゃ、ラチがあかない」

　と思って、抜本的な解決策を
　求め始めてから、気にかかってくるもの。



■そのエンジンにスイッチを入れるのは、

　「まず、量から」

　なのです。


　そのことを理解せず、

　「量を追求せずとも、
　　質さえ高めれば、ラクできる」

　という短絡的、横着な考えで、

　「効率化しよう」

　とか、

　「レバレッジをかけていきたい」

　とか、

　「一の労力で、十の成果を」

　とか、

　「最短、最速で」

　といった標語のみが上滑りをして、
　一向に成果が出ないことになるのです。



■言うまでもなく、これは、

　「効率化」

　「レバレッジ」

　「一の労力で、十の成果を」

　「最短、最速で」

　といった考えを否定しているわけではありません。


　むしろ、その逆で、

　それらの価値が分かるからこそ、

　その価値を分かるためには、
　少なくとも

　「最低限の量稽古（言葉に矛盾が感じられますが 汗）」

　くらいは、クリアしておく必要があるのです。



■そこを超えて、

　「これはきついな」

　「このままだとつらいな」

　と、身体に堪えるレベルで実感、痛感して初めて、
　本気で「質」を追求したいという欲求が生まれます。



■初めから体も動かさず、頭だけで

　「効率化、レバレッジ」
　
　といった語感のみに惹かれ、

　「ラクしたい」

　という考えでいても、


　その程度の活動では、
　そもそも「効率化」の必要性も感じられず、
　
　したがって、切実に希求する気持ちも湧いてこないから、
　真剣に取り組もうとする気持ちも生まれず、

　せっかく（効率化に）役立つ話を聞いても

　「イイハナシダナー」

　で終わってしまうだけなのです。



■一度は、慢性的に睡眠不足になるくらいまで
　体を動かす経験を通らなければ、たとえば、

　「現状と同じ、あるいはそれ以上の成果を出しつつ
　　睡眠時間を確保するにはどうすれば」

　といった「仕事の質」を追求するモードに
　切り替わることはありません。


　昨日の「年間プログラム」のフォローのためにも、
　今日は思うところを書きました。
