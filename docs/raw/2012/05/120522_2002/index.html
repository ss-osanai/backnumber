■こんにちは、鮒谷です。


　今日は、

　過去、７年ほどにわたって食事をご一緒したり、
　勉強会の場で学ばせて頂いてきた、

　CPI会社力研究所を経営されている
　長谷川和廣さんの

　「講演会ならびに懇親会」

　のご案内をいたします。



■長谷川和廣さんは、

　ケロッグジャパン、バイエルジャパンなどの
　外資系企業で代表取締役社長などの要職を歴任、

　2000年、(株)ニコンとエシロール社の合弁会社
　(株)ニコン・エシロールの代表取締役に。


　50億円の赤字を抱えていた同社を1年目で黒字へ、
　2年目で無借金経営に変貌させた経営手腕をお持ちで、

　これまでに関連企業を含めると2000社を超える企業の
　再生事業に参画し、赤字会社の大半を立て直して
　こられた方。



■もともとは、

　消費財のマーケティング責任者として
　数々の新商品の開発に従事し、

　名前を言えば「ああ、あの、、」と思われる商品を
　数多く、世に広めてこられ、

　そこから外資系企業の経営の世界に入り、
　さらに企業再生の専門家になられた、

　というキャリアを辿ってこられました。



■冒頭でも述べたように、

　私は長谷川さんのそのような過去のご経験から来る
　たくさんの学びを頂いてきましたが、
　
　今回、その長谷川さんの講演会、そして懇親会という
　貴重な機会をご紹介いたしたいと思います。

■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　1を聞いて100学ぶ人と、100聞いて1も学ばない人　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　1を聞いて100学ぶ人と、100聞いて1も学ばない人　】

───────────────────────────────────

　

■昨日は

　「どんな状況に置かれても

　　そこで喜ぶ人は喜ぶし、
　　不満を持つ人は不満を持つ」

　と書きましたが、


　同じ状況であっても人によって異なる反応が
　現れるのは、

　「状況の解釈装置」

　の性能によるものではないかと
　お伝えしたかったのです。



■これは「学び」についても同じことがいえる
　のではないかと考えています。

　つまり、

　「学びなど全く得られそうもない環境に
　　いるにもかかわらず100のことを学べる人」

　もあれば、

　「そんな素晴らしい環境にいるんですね！
　
　　と周りの人が羨むようなところにいても、
　　１つのことも学べない」

　という人がある理由も

　「状況の解釈装置のレベル」

　によるのではないか、ということ。
　


■このように

　１を聞いて10どころか100も得る人もあるし、
　100を聞いて１も得ない人がある、

　という現象が起きるのは一体どうしてなのかを
　考えてみると、

　これもその人の持つ「解釈装置の質」の問題で
　あるように思われるのです。



■よい解釈装置を持つ人は、どんなことからでも
　学びを得ることができるし、

　品質の低い解釈装置しか持たない人は、
　どんな素晴らしい環境にあっても
　学びを得ることはない、

　のかもしれません。



■たまに、

　「私がなかなか変われないのは、学べないのは、
　　このような環境にいるからだ。

　　（独立して、転職して）環境を変えれば
　　自分も変わる」

　と思っている人があります。


　しかしそのような物言いが通用する人は

　「既に高性能の状況の解釈装置を持っている人」

　だけであって、

　未だそのような装置を持たない人が
　独立して、転職して、別の環境に身をおいても、

　きっとその場所で同じことが繰り返される
　だけでしょう。



■自分の外側にある学びのネタの有無、多寡、
　というよりは、

　むしろ自己の内部にあるシステムに
　目を向ける必要があるのでは、

　と思われることも少なくありません。

　

■不平不満をいう人はどこにいっても
　不平不満を言うものです。


　「悪いところを見る」

　ことが癖になってしまい、

　「悪いところしか見られない」

　ようになるのです。


　
■さらに言えば、

　「本当に悪いところしかない環境」

　に身をおいていたとしても、
　「反面教師」という学び方もあるわけで、

　その場合、

　「悪いところが、よいところ」

　という解釈を自動的に発生されられるのが
　高級な（？）解釈装置でしょう。



■したがって私は順番として、
　まずは、

　「学びそれ自体よりも、
　　学びの解釈装置の精度を高める」

　ことが大切、と力説したいと思います。


　
■この装置の質を現時点で望みうる限りの最高レベル、
　と思われるところまでアップグレードして、

　そこではじめて、より多くの学びを得られる
　環境はないだろうか、

　という問いが有効なものとなってくるのでは
　ないでしょうか。



　今日も人生とビジネスを楽しみましょう！
