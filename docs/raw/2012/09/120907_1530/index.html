  【　経験とは、語ることができる体験　】

───────────────────────────────────



■人は毎日、たくさんの体験をしながら
　歳を重ねていきます。

　「生きること＝体験すること」

　と言ってもいいくらいですね。


　けれど、

　「体験と経験は別物」

　のようにも感じます。



■日々、いろんな体験をするわけですが、

　すべての「体験」が「経験」になるとは
　いえないように思われるのです。

　単に言葉の定義づけの問題なのかも
　しれませんが。



■私なりの（暫定的な）定義として、

　「体験とは、自分の中で咀嚼することができず、
　　時間の経過とともに、そのまま流れ去ってしまうもの」

　であり、反対に、

　「経験とは、『語ること』ができる体験」

　と位置づけています。


　すなわち、

　「自らの言葉で語れぬ体験は
　　『経験』とはいえない」

　ということです。



■たとえば海外旅行に行ったことについて
　語るとき、


　（極端に言えば）

　「アメリカに行ったことがあります。
　　オーストラリアに行ったことがあります。

　　イギリスに行ったことがあります。
　　南アフリカに行ったことがあります。
　　　　　　　　　　　　　　　　　　　　　以上！！！」


　こういった言葉しか発することができないのであれば、
　それは経験ではなく、体験です。



■確かに現地を訪れる、という体験は
　したのでしょうが、それ以上でも以下でもない。

　異国を訪れた、という事実が
　積み重なっていくだけです。



■ところが、そうではなく、


　たとえばニューヨークを訪れてみて、

　◯◯という事実に気づいたとか、
　感じたとか、考えたとか、


　そんな「語り」が内側から生まれるほどに
　心が動いたとき、

　体験が「経験」へと昇華されるのでは
　ないかと思われるのです。



■そういった観点から捉えたとき、

　日々、生活しながら積み重ねているのは、
　「体験」なのか「経験」なのか、

　改めて考えてみたいと思います。


　できることなら、

　心が動いたタイミングで、
　積極的に意識を働かせ、

　ついでに手も働かせ（＝メモを取り）

　「体験」を「経験」に変えて
　蓄積していきたいものです。




　今日も人生とビジネスを楽しみましょう！
