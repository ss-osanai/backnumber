■午前９時前。商社で働いている昔からの友人と久しぶりに再会、近況報告。
　海外と日本をいったりきたり、だそう。ちょっと羨ましいかも。
　でもちょっと治安の悪いところのようでした。

■午前１１時半。広告代理店の方と打ち合わせ。

■午後１時半。業者さまと打ち合わせ。

■午後２時半。銀行。

■午後３時。渋谷のビックカメラにて、今更ですがカメラ付き携帯電話を購入。
　なんと200万画素！一昔前のデジカメの最高スペックくらいですから驚きです。
　せっかく買っても、使い道はあんまり考えていないんですが。

■午後４時。セカンドステージスタッフの清水とお茶。

■午後５時。お世話になっているヘッドハンターと情報交換。

■午後６時。ソフトバンクパブリッシングさま。来月発行の「ビジスタ」誌の
　取材のため。思いつくままに適当にしゃべりまくったんですが、一体
　どんな記事になるのでしょうか？

■午後９時。ＰＲ会社を経営されている玉木さんと食事。情報交換。

■午後１１時。帰宅途上の地下鉄社内で、お世話になっている与信管理
　コンサルタントの牧野さんとばったり。
　こんなこともあるんですね。悪いことはできません。

■帰宅は午前０時前。

■それから溜まったメールの処理を開始しはじめるも、途中で断念。
　寝ました。

　　　　【脳に刺激を与え続ける。】

───────────────────────────────────


■犬も歩けば棒に当たる。歩かなければ、何にも当たりません。
　人間も同じ。
　

■いいチャンスに巡り合うためには、いろんなところに顔を出して
　チャンスを捜し求め続けなければなりません。

　すぐには結果にならない行動が１００のうちの９９かもしれません。
　でも残りの「１」が逆にすごい出会いだったりするのです。

　遠回りなように見えて、何かを行おうとするときには、実は案外
　これが一番の近道だったりするのかもしれません。

　
■さらにこれを確率論と単純に捉えることも出来ません。
　人と会うことが、

　《脳に刺激を与える》

　こととなるからです。


■何にも考えずに人と会う、話をするというわけにはいきませんよね。
　いろんなことを考えるし、いろんな発想、構想がその中で湧いてきます。

　▼昔から教えることは一番、自分自身が成長することとなる

　とか

　▼人と話をする、教壇に立つことで新しいインスピレーションが閃く
　
　とかいわれますが、これは間違いなくほんとです。
　だからセミナーに参加することなんかは一石二鳥です。
　
　新しい（別に新しくなくても）情報を入れることで脳にインスピレーション
　を与えてもらったり、そこで新しい人とも知り合えるし、いろんなお話を
　参加者とすることが出来るから。


■そういう行動をするままが、脳にびんびんと刺激を与えていること
　になるのでしょう。


　外からのインパクトがなければ、じっと考え込んでいても、決して
　何も浮かんできません。
　外部刺激があって、初めて行動につながるための「何か」のヒント
　やアイディア、きっかけをつかめるものなんですね。


■出会いを求める。これが、大きく人生が変革する第一歩と
　なるかもしれません。
