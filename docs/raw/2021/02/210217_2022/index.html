　-----------------------


　安心という感情があります。
　安心の反対は不安。

　不安であるよりも
　安心であったほうがいいと、

　その安心を支えるための何らかのものを
　我々は必要とするわけです。


　-----------------------


　安心というのも幅が広くて、

　食に対しての安心、
　健康に対しての安心、

　あるいは住空間に対しての安心、

　職、食べるのほうの食じゃなくて
　仕事・職場における安心、

　というのがあるかもしれません。


　-----------------------

　
　「安心（不安のない）」という感情を
　手に入れるためには、

　さらにそこからスペック的な、
　と言うべきか、
　
　Doing、Havingレベルの
　具体に下りていくことが必要になる。


　-----------------------


　安心というのは「Being」に関わる状態。

　安心という感情の状態を
　本当に手に入れたいのなら、

　その感情を手に入れるためには
　具体的に何が必要になるのか、

　必然的に考えることとなる。


　-----------------------

　
　ピラミッド音源を
　お聴きくださった方であるならば、

　ピラミッドの上部に
　この安心という感情が来て、

　下部に安心を構成する要素というか、
　支えてくれる手段的なもの、

　が色々とぶら下がってくることを
　ご理解いただけるはず。


　-----------------------


　たとえば地震が来た時、
　
　簡易トイレがなければどうなるか、
　みたいな話を同音源ではしたと思います。


　その状況を想像し、

　なければ不安だよねとか、不快だよね、
　といったマイナスの感情を消し去るために、

　簡易トイレ云々という話を
　させてもらった。


　上部から下部、上位から下位、、、
　
　（ここから先の話は
　　音源をお聴き下さい）


　-----------------------


　まず上位概念として、安心が来る、

　他にどんな気持ちが欲しいのかと、
　どんな感情が欲しいのかと考えると、

　安心に関係する言葉として、

　「安全である」

　という感覚、、、


　（ここから先の話は
　　音源をお聴き下さい）


　-----------------------

　　
　あるいは「変化を楽しむ」という
　感情もある。

　安心とか安全、という感情は
　一歩間違うと、

　退屈とか、マンネリの気持ちに
　つながってくるわけ。


　-----------------------


　私たちは、

　安心・安全を求めるとともに
　変化も希求するという、

　そういうニーズもあるわけ。


　では、変化、変化の感情というか
　欲求を満たしてくれるものとしては、

　果たしてどういうものが必要であるか、

　すなわち、未来の自分に
　変化の感情を楽しんでもらうためには、

　今の私は何をプレゼントすれば良いのか、
　ということを考える。


　-----------------------


　あるいは

　「他者からの評価にまつわる感情」
　
　もありますね。

　他者から承認してもらうとか
　評価してもらうのは、


　これは私たちにとって
　承認欲求がある限り、味わいたい感情。


　-----------------------

　
　他にも「自尊感情」も欲しいもの。

　他者の目を意識することなく、
　自分自身が自分を尊ぶというか、

　自尊感情だったりだとか、
　自己重要感といった感情、

　これもあるのとないのと
　どっちがいいかっていうと、

　もちろん、あった方がいいわけですよ。
　あったほうが幸せを感じられます。


　-----------------------


　いくら他者から評価されてるとか、

　安心な領域にいるとか、
　安全とかの欲求が満たされていたとしても、

　自尊感情が著しく劣っていれば、
　我々の幸せは著しく、阻害、毀損される、

　ことになるでしょう。


　-----------------------


　自信を感じる、というのも
　私たちが手に入れたい感情の一つ。

　誇り、というものも
　そうかもしれませんし。


　-----------------------


　あるいは「貢献の欲求」という感情も
　ありますね。

　誰かの役に立てたとか
　喜んでもらえたとか、感謝された、

　そこに喜びを感じたというのは、
　貢献の欲求が満たされたということ。


　-----------------------


　他にも、

　日々、自分が成長を遂げていると
　感じられる喜びもあれば、

　知的好奇心が満たされる、
　知識が増えていく、

　という感動もある。

　
　-----------------------


　知識が増える、に関連して、
　別の切り口からいうと

　生きていく上での知恵が身についた、
　という実感も嬉しいもの。


　それは例えばいろんなものを
　パターン化して捉えられるようになったとか、

　教訓が得られたとか、俯瞰して
　物事を見ることができるようになったとか、

　一を聞いて十を知られるようになった、
　とか。


　こういうのも一つの状態であり、

　そこから安心感とか自信とか
　ゆとりといった感情が得られるでしょう。


　-----------------------


　あるいはまた全然違う側面から。

　リラックスしていますというのも、
　一つの感情です。


　全てを忘れているというのか、
　余計なこと考えずに、

　ゆったりとか、
　リラックスしている、

　そんな感情。


　-----------------------


　また、自然を愛でられる、
　花鳥風月を愛でるといった、

　豊かさを感じられる気持ち。


　今日なんかも
　何気に昼ご飯を食べていたら

　僕は結構雲が好きなんですけれども、
　なかなかいい雲が出ておりましてね、

　雲見るだけでも幸せを感じられたら、
　まあ、安上がりでいいわけですよね。


　-----------------------


　あるいはたとえば、
　春とか初夏に、

　たくさん芽が出てくるとか
　夏だったらすごい勢いで毎日毎日、

　葉っぱが出てくるとか、
　枝葉が茂っていくというのを、

　見ることによって幸せを感じられる、
　これも一つの感情。


　-----------------------


　まだまだ、たくさんありますね。

　達成感を得られる、
　これも感情です。


　目標を掲げると、目標を目指して、

　ああでもない、こうでもない、
　と試行錯誤することによって、

　目標達成できたときに得られるのが
　達成感という感情。


　-----------------------


　そこに向けて全力投球していると、
　
　日々、充足しているとか、
　満足しているとか、充実している、
　
　そんな感情が生起するし、

　過去を振り返って
　感慨深い、という感情も生まれるし、

　その過程で人と繋がれている、
　というような喜びも一つの感情です。


　-----------------------

　　
　自由という言葉について考えると、

　それは精神的な自由というのもあれば、
　空間的な自由というのもあれば、

　人からの束縛からの自由であったり、

　人に限らず、例えば、家かもしれないし、
　地域共同体かもしれないし、組織かもしれない。


　そういうところからの
　しがらみということかもしれないし、

　そうした束縛から離れられた自由、
　という感情もあるかもしれない。

　
　-----------------------


　また、何かを追求していると、

　極めるとかこだわる、といった
　喜びの感情が生まれることもある。


　-----------------------


　そういったことを全部ひっくるめての
　豊かさの感情であったりだとか、
　
　他にも、人によってはわびさびの感情に
　対して幸せを覚える人もあるかもしれないし、


　あるいはチャレンジから生まれる興奮、
　なんていうのも一つの感情。


　-----------------------
　

　感情表現辞典というものがあります。

　無数の感情が　
　感情辞典に出ているけれども、

　似ているんだけど、たとえば、
　ニュアンスが少しづつ異なる、

　そんなたくさんの感情が紹介されている。


　-----------------------


　思うに、我々は単一、ある感情を
　求めているのではなく、

　無数のポジティブな方向の感情を
　味わいたい、楽しみたいのではないか。


　-----------------------

　
　ネガティブな感情ももちろん存在する。

　それは怒り、憤り、だとか嫉妬、妬み、
　嫉みであったり、憤りであったりだとか、

　悲しみであったりとか
　いろんなものがある。


　悲しみというのはちょっと微妙というか、

　ポジティブな感情になることもあるし、
　ちょっと微妙ですが、

　一番わかりやすいのは
　怒りとか恨みとか呪うとか、

　こういうのは幸福を感じるにあたって
　ネガティブ要素しかない感情。


　-----------------------


　つまり、

　どんな感情もあればさえいいと
　いうわけではなく、

　ポジティブな方向の、そしてできる限り
　多くのレパートリーの感情を、

　日々感じられるようになると、
　さらには自在ににチョイスできるようになると、

　けっこうな幸せを感じることが
　できるんじゃないかな、

　という風に思われる、、、


　（ここから先の話は
　　音源をお聴き下さい）
