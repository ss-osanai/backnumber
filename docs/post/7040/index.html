7040号 パンツをはく人、はかぬ人

■この数ヶ月、NHKの過去の番組を
　見まくっています。


　『のど自慢』もいいですが、

　『アナザーストーリーズ
　　運命の分岐点』

　もまた素晴らしいコンテンツです。



■このところ集中的に視聴しており、

　現在、アップされている視聴可能な
　番組のほとんどを見終わり、

　残すところ数本となりました。



■Wikipediaでの説明文を見てみると

　（ここから）
　--------------

　世界の歴史を揺るがした、
　様々な大きな出来事にスポットを当てる。

　その出来事が起きた時代に、それに
　かかわった人らは何を考えていたのかを、

　貴重な映像や写真などの文献、
　関係者へのインタビューを使いながら、

　その出来事に隠されたもう一つの物語
　「アナザーストーリー」を、

　3つの視点から浮き彫りにする
　「マルチアングル・ドキュメンタリー」である。

　--------------
　（ここまで）

　と記されています。



■45分の短い時間の中で、

　CMもなければ、
　もったいぶることもなく、

　数多くの証言や貴重な未公開資料が
　登場して、

　ある出来事についての当時の背景や
　真相を解き明かそうとする、

　NHKならでは実現されえなかった
　企画、と言えそうです。



■テーマは多彩で直近だとこんな感じ。

　いや、これ、全部興味をそそるし、
　面白すぎるでしょ。

　だから全部見たわけですが。


　そしてこれを「毎週」続けられる、
　NHKの底力に畏怖すら覚えます。

　NHK凄すぎ。



■そんなわけで、昨日は

　『天才激突！ 黒澤明VS勝新太郎』

　の回を見ておりました。


　芸能の世界には疎いので
　知りませんでしたが、

　映画『影武者』の主役には、もともと
　勝新太郎が起用されていたのだけれども

　「とある出来事」

　があって、前代未聞の主役降板と
　なったとのこと。



■勝新太郎については例の

　「これから、パンツをはかない」

　の名言（迷言？）くらいしか
　知らなかったのですが、

　こんな事件があったんですね。

　といっても1979年の出来事らしいので
　知らなかったのも無理はないか。



■ところで黒澤明監督といえば『羅生門』、

　『羅生門』といえば、
　ある一つの出来事が

　「三者三様の目線で語られる映画」

　ですが、


　アナザーストーリーズという
　番組自体がまさに

　「三者三様の目線で語られる番組」

　であり、

　その中で黒澤明が
　取り上げられているところに

　一種の入れ子構造的な
　面白さがあるな、

　と思ったものでした。



■それはともかく、

　同一の出来事でも見る人、見る立場により
　全く異なる世界が展開されることは、

　この番組を見て、
　なおよく分かるわけですが、


　この状況こそ、まさに

　【同床異夢】

　であると思わずにはいられない。



■特に

　『天才激突！ 黒澤明VS勝新太郎』

　の回については、

　両者のアシスタントや付き人が登場し、
　それぞれの見方がまるで異なっていて、

　まさに「羅生門的」でありました。



■人はどうしても

　「自分が見てきた世界のみが真実」

　と考えてしまいますが、
　この番組を見ていると

　【それぞれの目に映った、
　　それぞれの真実がある】

　と理解できます。



■そういう意味では、この番組は

　【物事はすべて多面的、と教える
　　コミュニケーション能力の基礎トレ】

　的な教育価値もあるのでは、
　などと思われた次第です。



■この番組を見ながら

　【ある事象に対する、
　　自分の捉え方が正しいとは限らず、

　　むしろ、その奥にはるかに複雑な
　　背景や文脈が広がっている】

　と自戒しなければと考えた次第。
