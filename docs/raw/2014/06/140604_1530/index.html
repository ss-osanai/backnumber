＜今日のピークパフォーマンス方程式＞

 【　人は、教育がつけばつくほど、ますます好奇心が強くなる（ルソー）　】

───────────────────────────────────



■前述のとおり、

　先日、たまたま、新聞だか雑誌だかの書評で
　見つけた、こんな本


　【今日のお勧め本　遣唐使 阿倍仲麻呂の夢】

　 http://amazon.co.jp/o/ASIN/4047035300/2ndstagejp-22/ref=nosim


　を読みました。



■この本は、

　Amazonの「内容紹介」や「レビュー」にも
　記されている通り、


　当時の時代（奈良時代）背景から始まり、

　遣唐使に乗って唐に渡り、遠い異国の地で学び、
　科挙に合格、
　
　宮廷内で社交を重ね、破格の出世を遂げながらも、

　故国を思う念、止みがたく、
　しかしながら結局、帰国叶わず現地で果てた、

　阿倍仲麻呂の望郷の念、

　がリアルに描かれています。

　
　この本に触発されて、何冊かの


　「奈良時代関連の本」

　「遣唐使関連の本」

　「科挙について記された本」

　「漢詩入門書」

　
　を読んでみたのです。



■このようにして、受験生時代以来、
　ほとんど久しぶりに、

　「奈良時代」

　に触れたわけですが、


　『遣唐使 阿倍仲麻呂の夢』

　の中で記されている、

　奈良時代という時代、当時の諸外国との関係、

　教科書で覚えた、

　吉備真備、鑑真、玄宗皇帝、
　聖武天皇、杜甫、李白、王維、

　などといった人物の相関、

　あるいはかの有名な、

　「天の原 ふりさけみれば 春日なる 
　　　　　　　　　　　　　三笠の山に いでし月かも」

　の歌の解釈、などから派生して、


　その時代に出てくるさまざまな単語が、
　一気につながり、広がり、

　「はるか昔の、関係ない、
　　ほとんど興味も湧かない時代」

　　から、

　「そこそこリアルにイメージできる時代」

　になったのは、大きな収穫（？）でありました。



■こうして、いつの間にか「奈良時代」が、私の

　「興味、関心の範疇」

　に入ってくることになったわけですが、


　今後、奈良時代のことに接したときは、
　今回、得られた知識やイメージが「核」となり、

　さまざまな知識がこれまでの何倍、
　あるいはそれ以上のレベルで、

　吸着されていくことになるはずです。
　


■考えてみれば、

　この過程こそが学びの歓びであり、
　この歓びを得るために、
　
　読んだり、聞いたりしているのか、
　と思うくらい。

　
　こんな風にして好奇心や知識が、
　次々に広がっていくさまを、以前


　＜みんな大好き塊魂＞
　
　https://www.youtube.com/watch?v=EDkKqyNLosg

　※注意！音が出ます！


　の動画を通して、

　「未知が既知になるプロセス」

　とお伝えしたこともありますね。



■人生に大きな歓びを持てるか否かは、


　「人は、教育がつけばつくほど、
　　ますます好奇心が強くなる。　（ルソー）」


　この言葉を実感として持てるかどうかに
　かかっているのではないか、

　とすら思えます。



■どこに向かうのか、分からないけれども、

　内発的な欲求（好奇心）に引っ張られるままが、
　どんどん新しい世界に連れ出される姿、

　そんな感動をこれからも重ねていきたいもの。

■私（鮒谷）はこれまでこのメルマガ上で、

  「成功したいと思ったら行動を変えよう、そのためにはまず、
    思考レベルで変わることが大切」

  そんなことをたびたび記してきたかと思います。

■行動につながっている自らの思考、そしてその背景にある、
　「深く根ざしたもの」を知ることは、行動を変え、成果へとつなげて
　いくにはとても重要なステップといえるでしょう。

■堺屋太一さんのこのＣＤ、さっそく私も聞いてみましたが、
 「だから自分は○○だったのか！」と納得できる話が満載でした。

  自分とは無関係に思えた、歴史上のあの人物のあの行動が、
  何百年もの時を越え、こうも深く影響を受けているとは衝撃です。

■歴史を学びながら、自分を再発見できるという貴重なこのＣＤ教材、
  残念なことに書店には置いてないとのこと。

  果たして、堺屋さんの選んだ「日本を創った１２人」とは？
