■多くの人が意識的にであれ、
　無意識的にであれ、

　「知らないからやらない」

　「分からないから近づかない」

　「知らないからこそやる」

■世界が拡張されないと
　どうなるかというと

　「時の経過とともに

　ようになるでしょう。



■これは

　「時間をかけて

　「知らないからやらない」のと
　「知らないからこそやる」の間に、

　数日や数週間、数ヶ月程度で

　どのようになっていくか。

　ご推察の通り、

　「世界の広がりを感じさせる人」

　と

　「閉じられている世界に生きていると
　　感じられる人」

　へと、二極分化が進むことと
　なるのです。



■両者の姿勢が、

　それぞれの方向性で
　顕在化したときには既に、

　【自身がどちら傾向の人であるかを
　　自己規定してしまっており、
　　周囲からもそのように認定されている】

　状態が固定されているでしょう。



■こうした自他の認識は、

　【自身の運命に
　　影響を与えないはずがない】

　と思われませんか。

　実際に大きな影響を与えるのです。



■もし未来の可能性を
　開いていきたいのであれば。

　最大限の好奇心を発揮して、
　自分の心も開放させて、

　【分からないからこそ、
　　とりあえずチャレンジ】

　する姿勢が大切だと言えますね。

　毎日を過ごしながら、

　「人生が変わらない」
　「人生を変えたい」
　「閉塞感で窒息しそうだ」

　その人を
　客観的に見ているあなたも

　「そりゃ、そうなるに決まってるでしょ」
