6942号 一泊20万円の部屋（花火鑑賞プラン）に泊まりましたー

■上述の通り、先週の木曜から日曜まで
　４日間、大阪に行っておりました。

　３年ぶりに開催された

　「なにわ淀川花火大会」

　を鑑賞するためです。

　花火鑑賞できるホテルの部屋を取り、
　お客さまと一緒に楽しみました。



■清水の舞台から飛び降りるほどの
　覚悟とともに

　「10年に一度」

　くらいの贅沢のつもりで予約を入れた
　一泊20万円の部屋。



■にもかかわらず

　「ご案内早々、多くの申し込みを頂き、
　　満室になりました」

　とホテルスタッフの方からお聞きし、
　度肝を抜かれました。

　あるところにはあるんですね、、



■ちなみに私（鮒谷）の取った部屋は
　もちろん一番安いところ。

　それでこの価格。
　（スイートとかだと当然もっと高い）



■夕食、ついてないですよ。

　シャンパン、ちょっとしたスイーツ、
　翌日の朝食ビュッフェ、以上！


　これで20万か、と目を剥きましたが
　歯を食いしばって

　「これは投資的浪費なのだ」

　と自らに言い聞かせた次第。



■これだけ盛況であったのは
　コロナ禍を経ての

　「強制貯蓄＆反動消費」

　の影響なのかもしれませんね。


　それにつけても、ここだけ時空が
　歪んでいるのかと疑いましたが、

　需給のバランスに最適化された
　価格設定なのでしょう。

　現実に満席になっているわけですし。



■それはさておき、

　こうした非日常経験も意識すれば、
　一つの学びになり得ます。


　常識はずれの価格帯の商品やサービスを
　体感することで、

　常識はずれの価格帯の商品やサービスを
　自身も提供する許しが与えられる、

　ということはありそうです。



■現に私（鮒谷）は、

　こうした高額サービスが販売され、

　しかも早々に満室になり、
　さらに滞在している人はみんな笑顔、

　という姿を目の当たりにして、


　うっかりすると

　「安く、もっと安く」

　という精神に陥りがちなところを、


　やっぱり

　（もちろんお客さまが納得くださる
　　品質であること前提で）

　「高く、もっと高く」

　という姿勢で良いのだという思いに
　立ち返ることができました。



■非常識なほどの高額サービスが
　早々に売り切れ、かつ、顧客も満足、

　こうした世界をリアルに感じることで
　新しい世界認識が立ち現れる、

　そんなことがあるのです。



■ここまで書いて思い出したのですが、

　そういえば、私（鮒谷）もこんな音源を
　販売しておりました（笑）
