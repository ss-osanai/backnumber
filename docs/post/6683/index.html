6683号 自分の軸を持ち、ブレない生き方を確立したいなら

■人間には誰でも

　【自分は他人とは異なる存在である】

　と思いたい、

　そんな潜在的な欲求を持っていると
　考えています。

　もちろん私（鮒谷）もそう。



■「人とは異なる存在」になりたいのなら、

　「人と異なる思索をして、
　　人と異なる行動をしなければならない」

　のはいうまでもありません。


　人と同じものを見聞きして、
　人と同じものを体験し、

　人と同じことを考えて、
　人と同じことをして、それでいて

　「人と異なる存在」

　になれるはずがないのは
　子どもでも分かります。



■将来、自分がどんな存在になるのかは

　「今、何に触れて、何を思い、何を考え、
　　どんな行動をしているか」

　によって規定されるのです。



■だからこそ、

　人とは異なる多様な、
　そしてたくさんの経験をして、

　自分の脳内において高頻度で

　【核分裂・核融合、からの核爆発】

　を起こし続ける必要があるのです。



■私（鮒谷）はこうしたことを
　意図的に行い

　【新しい概念や表現】

　を生み出しては記述し続け、
　その集積によって


　【オリジナルな思考の体系を確立する
　　（＝その総体が信念体系となる）】

　ことを意識して行ってきました。



■日々、大量の新しい概念＆表現を
　量産し続けていると、やがて、


　「脳内に無数に刻み込まれた概念が
　　細分化（精緻化）されたり（＝核分裂）

　　ぶつかりあって、より進んだ、
　　あるいは別種の概念へと昇華したり
　　（＝核融合）」


　することなり、

　【時間をかけて、頭の中の配線構成が
　　作り変わっていく】

　こととなるでしょう。


　さらに長い時を経て、それは

　【全く別の脳】

　へと進化をとげることとなるのです。



■そのプロセスを、私（鮒谷）は

　（過去、お読みになられたり、
　　聴かれたりされた方もあると思いますが）

　【術 → 説 → 論 → 学 → 道】

　という表現で表してきました。

　（詳しいことは今日は記しません）



■こうした思考体系の総体（＝信念）が

　「時々刻々の行動」

　を生み出すこととなり、

　【知行合一】

　のレベルに到達すると、
　それはもはや

　【（生きる）道が確立された】

　といっても差し支えないでしょう。



■ここに至って、

　自信や自尊心が確立されるし、
　他者からの言葉で心が揺れることもなくなり、

　自らの道を貫き、生きていける
　ようになるのです。



■ありきたりではあるけれども、
　耳障りの良い言葉で表現するならば

　【軸を持つ】

　【ブレない生き方を確立する】

　ということになるのかも
　しれませんね。



■他者との違いが明確化され、
　さらにその姿（状態）に誇りを持てる、

　そんな世界に進むために
　欠かせないことは


　【日々、大量の刺激を受け、
　　大量に思索＆行動し、

　　そこからの大量の閃きを、
　　大量に記述し続ける】

　ことではないかと考えています。



■それによって

　【自分オリジナルの言葉】

　も生まれてくるし、


　自らのうちから湧き上がってきた言葉は
　自分の行動を決定づける

　「意思決定基準」

　となっていくでしょう。



■こうした

　【自分独自の意思決定基準
　　（＝マイルール？）】

　が増え、
　さらには精緻さも増していき、それらが

　【集積され、
　　閾値超えを果たしたとき】

　こそが、


　外部からみても、それと分かる

　【自分流の生き様】

　が確立されるとき、と言えるでしょう。



■別にそんな生き方、生き様に
　憧れることもないし、興味もない、

　というのなら、
　ここから先の話は読まなくても結構です。


　でも、あなたが心のどこかで、
　ほんの少しでも、

　そうした生き方に惹かれるものがあるならば、
　ぜひ、その世界に至るための

　【最強の手段たるところの、
　　日記（正確には「時々刻々の記」】

　に意識を向けて頂きたいのです。



■これまで、さんざん、

　【日々の気づきや学びや発見や決意を
　　時々刻々に記述し続けること】

　の価値をお伝えしてきましたが、


　これが本当に腹落ちできれば

　「日記を書かない人生」

　など、まったく想像できなくなります。



■正直に告白すると、


　日記を書かない人は、普段、
　何を考えているのだろう、

　（いや、考えていないのかな、
　　日々の学びや思考を記録にも残さず、
　　流し去ってもったいないと思わないのかな）

　と思わずにはおれません。


　（腹を立てられた方がいらっしゃったら
　　ごめんなさい。

　　でも、本当にそう思ってしまうのだから
　　仕方ない）



■日記放談会の音源をお聴き下さり、
　既に日記を書き始められている方からは、

　数え切れないほど同じこと


（＝日記を書かない人生、って
　　どんな人生なんですかね、

　　あるいは、

　　日記を書き始めた今になって、
　　無為に過ごしていた過去を振り返ると、

　　本当に何も考えずに生きていたことが
　　分かって、もったいなく、残念に思う）

　という話を聞かされてきました。



■でも、それが皆さんの

　【偽らざる思い】

　なのでしょう。
　その感覚は私もよく分かります。



■私（鮒谷）自身、
　日記を書いていなければ、

　今、このような仕事をしていることなど
　絶対にあり得ませんでしたし、

　今頃、どこかでのたれ死んでいたとしても
　不思議ではない、

　くらいに思っています。



■もちろん、ここまで記してきた話は

　信じてもらっても、
　信じてもらえなくても、

　どちらの解釈をしていただいても
　まったく問題ありませんが、


　既に日記を適切な形で記述しはじめて、
　自分の人生を作り変えていらっしゃる方、

　であるならば、完全に同意くださるのでは
　ないかと思います。

　　と思いたいもの。　　　　　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■が、他人と同じものを見聞きし、経験し、
　　同じ考えで同じ行動をして　　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　　「他人とは異なる存在になりたい」　　　
　　　　　　　　　　　　　　　　　　　　　
　　というのは、筋が通らない。　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■もし本当にそのように願うのなら、　　　
　　　　　　　　　　　　　　　　　　　　　
　　他人と異なるものを見聞きし、経験し、　
　　異なる考えで異なる行動をすることだ。　
　　　　　　　　　　　　　　　　　　　　　
　■さらには、そこでの気づきや学びを　　　
　　言語化して「核爆発」を生み出すことで、
　　　　　　　　　　　　　　　　　　　　　
　　【独自性を持つ、個を確立している存在】
　　　　　　　　　　　　　　　　　　　　　
　　へと昇華されることだろう。　　　　　　
　　　　　　　　　　　　　　　　　　　　　
　■もし、あなたがそんな存在を目指すなら
