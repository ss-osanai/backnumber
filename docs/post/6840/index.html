6840号 手中の鳥＆我田引水によって弱点を武器に転換する

■先日リリースした「我田引水音源」の
　ご感想をたくさん頂戴しております。


　私（鮒谷）があれこれ語るよりも、

　ご感想をお読みいただいたほうが
　内容や雰囲気を掴んでもらえそうなので、

　これからしばらくご感想紹介を通して
　音源内容の説明をしていければと。
　


■本日はＴさんから頂戴した、
　ご感想の紹介です。
　

　（ここから）
　---------------------------------


　「我田引水」は、
　辞書、自分の解釈ともにネガティブな
　印象が強い言葉でありました。


　政治家や強い立場にある人の権力を
　私的利用する、

　という解釈でしたが当音源にて、
　パラダイムシフトがありました。


　「我田引水」は周囲に不利益は生まずに、
　自分を強くしていけるのだと思いました。


　既に強い人は必要ではない、

　何者でもない、
　多くの悩みある私たちにとって
　勇気の出る言葉です。


　エフェクチュエーションの
　紹介の頃から、

　「手中の鳥
　（＝私は何かを持っている）」

　ことを強く意識しだしました。


　例えば日記において、

　書ける時に書くのではなく
　毎日書くものとして、

　日々の出来事を全肯定する。
　ということがあり、

　自身も一度、二度ストップしてしまった
　日記に意味付けをして復活出来ました。


　そして「我田引水」の新たな解釈で
　全肯定することの意味が深く理解できました。


　当音源の内容では、YouTubeなどの
　トークを伴った内容に注意が行きました。

　一人称で語ることで自分自身の感動が、
　相手側に伝わる。


　エフェクチュエーションの教授と
　鮒谷さんの語りの対比にもあるように、

　自分がどちらに向いているか、好きか、
　ということを元にバランスをとっていけば
　良いのかと思いました。


　対比される物のバランスをとることを
　意識して人前で話すことがある時に
　活かそうと思います。

　今回も貴重な音源、
　ありがとうございました。


　（会社員 Ｒ・Ｔ さま）

　（ここまで）



■Ｔさん、ご感想ありがとうございます！


　お書きくださったとおり、
　一般的には以下の通り

　「我田引水」
　
　という言葉はネガティブな意味合いで
　用いられます
　
　-----------------

　他人のことを考えず、自分に都合がいいように
　言ったり行動したりすること。

　自分に好都合なように取りはからうこと。
　自分の田んぼにだけ水を引き入れる意から。

　（goo辞書より）
　
　-----------------

　が、

　もちろんそのような意味合いで
　使っているわけではございません。



■少なからぬ方が、自らの特徴を勝手に

　「短所」「弱点」

　と解釈して、可能性を閉じたり、
　狭められているように思われるので、


　それらを

　「長所」「強み」「武器」

　として扱うための方策として

　「解釈転換による物語創造」
　
　を我田引水という言葉を用いて
　提唱している次第です。



■エフェクチュエーション勉強会に
　参加くださった方は

　「手中の鳥」

　という言葉とともに

　「見逃していた自分の資源に
　　気づくことが事始めとなる」

　という話を強い印象とともに
　ご記憶くださっているかと思います。



■特筆すべき資源を持たぬ人こそ

　「手中の鳥」
　「我田引水」

　という、イメージしやすい、
　印象深い比喩を用いながら、

　
　一見するだけでは
　
　「長所」「強み」「武器」

　と思えぬ自らの持つ諸要素に
　ストーリーを付与して、

　自らの田（人生やビジネス等）に
　水（価値）を付与する、

　そのための方法論について
　お伝えしているのが

　「我田引水音源」

　となります。



■ここでお伝えしているトレーニングを
　日常的に行うことによって

　自在に我が田に水を引ける
　（＝人生やビジネスに意味を与えられる）

　ようになるはずです。
