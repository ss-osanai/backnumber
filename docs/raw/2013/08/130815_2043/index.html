───────────────────────────────────



■こんにちは、鮒谷です。


　昨日は

　（最近は回数が減りましたが）

　以前、頻繁に開催していた、
　若手向け勉強会の参加メンバーとランチ。


　＜シンガポール・シーフード・リパブリック 品川店＞

　http://singaporeseafood.jp/introduction/shinagawa.html


　にて。


　夜は別メンバーとの会食となりました。


　＜ストリングスホテル東京インターコンチネンタル 26F
　　　　　　　　　　　　　　　　　ザ・ダイニング ルーム＞

　http://intercontinental-strings.jp/restaurant/the_dining/


　にて。


　なんと、何年ぶりかのカラオケにも行き、
　夜中まで３時間も熱唱してしまいました（笑）

　たまにはこういうのもいいですね。



■話は変わりまして、

　数ヶ月間、継続してコンサルさせて頂いていたＡさんから、
　下記のようなご感想を頂戴いたしました。
　
　　
　
　（ここから）
　---------------------------------------------------------


　今回で３回目の個別コンサルとなりましたが、
　既に継続して５回の個別コンサルを申し込んだところです。


　こうして継続を決意した理由ですが、
　大きく２点あります。


　１、この３ヶ月間で急激な内的+外的な変化を実感できたから


　私自身の体験として、第２回目までの個別コンサルでは、
　時間とお金の使い方、行動と環境の変化がありました。


　まずは、日常の時間とお金の使い方は、

　投資なのか？消費なのか？浪費なのか？

　この時間の使い方を俯瞰的に考えることができるように
　なってきました。


　そして、第３回目以降では、人を巻き込んで行動していくことの
　大切さを教えていただき、直ぐに実践している自分がいました。


　第３回目の個別コンサル後、数時間後には人に連絡を取り、
　３日後には３日連続で人に会う約束を取り付けたほどです。

　以前なら、ここまで直ぐに行動に移すことは
　できませんでした。


　これも鮒谷さん自身が行動に移し続け、
　楽しそうにしている雰囲気を感じるからこそ、

　つい同じように真似をしたくなってしまうのです。


　２、１年後先の成長曲線が自分でも想像できないから


　最近、鮒谷さんはメルマガでも、セミナーでも、
　個別コンサルでも、

　「一日一決断で人生は大きく変わり始める」

　というお話をされています。


　実際にご自身も一日一決断を実行されていて、
　加速的に進化し続けていらっしゃるようです。

　
　本で学ぶ以上に、直接、鮒谷さんから、

　「一日一決断、さらには二決断、三決断と重ねていく重要性」

　をお伝えいただく方が腹に落ちます。


　こうしてご自身が楽しく日々を過ごされているので、
　お会いする度に心地良い空気感で包み込んで下さいます。


　その影響からか、第３回目の個別コンサルを実施していただき、

　第１回目よりも第２回目、第２回目よりも第３回目と
　確実に成長している手応えを感じています。


　これを後、5回実施していくことで、
　どのような変化が待っているのか、

　もはや想像がつきません。。。

　
　この２点の理由から、現在と未来の時間軸でワクワクしている
　毎日を過ごしているというのが率直な感想です。

　これからも私以上に加速し続けている鮒谷さんの空気感に
　接し続けることで、一回りも、二回りも大きく成長していきたいと
　思います。


　---------------------------------------------------------
　（ここまで）


　Ａさん、大変うれしいご感想を
　ありがとうございます！


　
■決断と実行を繰り返し始めると人生が大きく
　変化し始めることを実感いただいているとのこと、

　ここからさらに急激に右肩上がりに
　持っていけるかどうかが、

　次の大きな課題ですね。


　これから５回のコンサルで、
　完全に人生を作りかえてしまいましょう。

　ここからが面白いところです。




■それでは

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。



━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　「人生確変モード」はこうして始まる　】

───────────────────────────────────



■最近のメルマガでずっと書き続けていますが、


　「決断 → 実行 → 結果」
　
　のサイクルを回す（回し続ける）威力は
　本当にすごい、
　

　と思わずにおれません。



■人に言いまくっているからには、
　自分も

　「決断 → 実行」

　のサイクルを負けずに回していこう、
　と思い、


　「一日一決断」

　を実践するようにしていると、一ヶ月も経たない間に
　まったく別の日常となりました。



■そうこうしているうちに、

　一日一決断の力を思い知らされると同時に、

　
　「決断 → 実行 → 結果」

　の前に

　「新しい世界を知るor経験する」

　があってこそ、


　「（知るor経験する） → 決断 → 実行 → 結果」


　という流れが機能することにも気づきました。



■そもそも決断しようにも、

　なにを選ぶか、という選択肢がなければ
　決断のしようがありません。


　では、その選択肢はどうすれば手に入るのか。

　なんだか矛盾するようですが、
　

　「外に出る」

　「新しい世界に触れる」

　「未体験ゾーンに足を踏み入れる」


　ことを決断をするところからしか
　選択肢を手に入れることはできないのです。



■つまり

　「決断 → 実行 → 結果」

　のサイクルを高速回転させたいのに、
　選択肢不足で決断そのものができないのであれば、


　（そもそも存在しない選択肢の中から選ぶことなど
　　不可能なわけだから）


　「まずはどんなことでもいいから（→ここ、超重要）

　　とにかく今の日常を越える選択をする」


　以外に方法がありません。



■この

　「日常を越える選択」

　を繰り返すうちに、


　「この方向だったらより深く突っ込んでいっても
　　いいかなあ」

　「こっちの方角にも面白そうなことがあるなあ」


　と思える、複数の選択肢が生まれてくるのです。


　
■こうして


　--------------------------------------


　未体験ゾーンに突入

  → 複数の選択肢が手に入る 

　→ それらを順番に実行

  → 新しい選択肢が手に入る

　→ さらに複数の選択肢が手に入る 
　　
　→ それらを順番に実行

　→ さらに新しい世界が開ける


　--------------------------------------


　というサイクルが回転しはじめれば、しめたもの。


　これが数回転すれば、
　一気に人生は変わり始めます。



■けれども、もし最初の

　「未体験ゾーンに突入」

　する決断を下すのに躊躇して、

　
　昨日の延長線上の今日、

　今日の延長線上の明日、


　を重ねていけば、


　--------------------------------------
　　

　未体験ゾーンに突入することがない

  → 複数の選択肢が手に入ることもない

　→ 選択肢を実行することもできない

  → 今いる世界が強化される

　→ ますます選択肢が手に入らなくなる
　　
　→ 当然、新しい選択肢を実行することもできない

　→ さらに今いる世界が強固になる


　--------------------------------------


　ということになってしまいます。



■このように、


　「未体験ゾーンに突入するか否か」


　という最初期の選択が後々、大きな差となって
　現れてくることになるわけです。



■だからこそ、


　人生を変えたいと心の底から願うのであれば
　ただ人生を変えたいと願うだけではなく、

　
　「これまでの人生と異なる、
　　未体験ゾーンに突入する決断」


　を下すことです。


　この決断を下すことを恐れる限り、
　変わらない日常が延々と続いていくことでしょう。



■そうならないためには、


　「新しい取り組みをしても、
　　そのほとんどは自分の人生を何も変えてくれない」


　と、あらかじめチャレンジの失敗を
　織り込んでおくことです。


　これによって、もし、
　決断が変化をもたらしてくれなかったとしても、


　躊躇なく

　「次行こ、次」

　の精神で、新しい決断をどんどん繰り出していくことが
　できるようになるのです。


　
■この繰り返しのなか、


　「ちょこちょこ出る小ヒット、
　　たまに出る中ヒット、

　　そして

　　まれに出る大ヒット」


　によって、

　「持っていかれる感覚」

　を覚えたなら、そこからが


　「人生確変モード」


　の始まりです。
