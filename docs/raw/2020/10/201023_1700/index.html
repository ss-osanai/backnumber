　（ここから）
　-----------------------


　私（鮒谷）の人生を変えた気の話


　元気、活気、人気、覇気、根気、気合、
　気働き、やる気、その気、気力、気遣い、

　狂気、負けん気、気配り、気が乗る、気を張る、
　気のない、気は心、気の抜けた、など、

　日常生活において「気」を含んだ
　言葉が数多く用いられているが、

　当たり前のように
　溶け込んでしまっているので、
　
　「気」という言葉をことさらに
　意識して取り出すことはそうそうない。


　-----------------------


　気について意識を持つきっかけとなったのは
　弁護士の高井伸夫先生とのご縁。

　ことさらに「気を入れる」という言葉を
　いろんな場面で使われていて、

　特に、対お客さん、
　あるいは他者と接するときに

　「気を入れて接する」

　ことの大切さを教わった。


　-----------------------


　気を入れる、気を張る、といった
　言葉には、ある種の指向性がある。

　コミュニケーションであれば、
　向き合っている人に気を入れる。


　今、私（鮒谷）はオンラインで
　皆さんに向けてお話をさせて頂いている。

　気を入れて話をさせて頂くこともできれば、
　気の抜けた話になることもある。

　
　-----------------------


　「気を入れる、気の抜けた」は自らの意思。
　ここで大きく差がつくこととなる。

　気を強く持つこともできるし、

　持たない、あるいは緩む、あるいは
　気がない、気の抜けた、

　人によってまるで気の入り方は
　異なる。


　-----------------------


　高井先生がよく言われていたのが

　「尽くすべきは尽くす」
　「なおになおなお」

　という言葉。


　-----------------------


　脳内辞書に「気」という言葉（基本語）が
　どのように定義されているか、また

　「気にまつわる言葉」

　がどれだけ格納されているかは
　当人の振る舞いに大きな影響を及ぼす。


　-----------------------


　「気」あるいは「気を入れる」という
　言葉について、
　
　どういう定義を（脳内）辞書で
　与えているか、

　いうまでもなく、その定義は
　私（鮒谷）と皆さんで当然、異なる。

　その定義の違いが行動の違いとなり、
　結果にも差異をもたらすこととなる。


　-----------------------


　別の例を挙げると、

　「仕事」「お金」といった言葉
　（これも人生における基本語）に対しても、

　各人、異なる定義を持っている。
　それこそ同床異夢。

　
　もちろん、定義に正しい答えはなく、
　みんな違ってみんないい。
　
　だが、さはさりながら、
　できることならば、

　「良い結果がほしい、
　　幸せになりたい」

　と、みな、願っている。


　-----------------------


　その願いが叶えられるには
　何らかの行動が必要。

　私たちは日々、無数の行為・行動を
　行うわけであるが、

　その行為の集積が
　運命を築いていくこととなる。


　-----------------------


　その行為はどこから生じるかというと、
　認識から生じる。

　　
　仕事というものは楽しいもの、
　震えるもの、興奮するもの、

　関係者に対して
　価値を提供するもの、

　あるいは、

　頂戴する代価、期待値を
　はるかに超えていくように努めるべきもの、

　と定義している人がいる。


　-----------------------


　脳内辞書を紐解くと
　そういう定義が記述されているなら、

　その人は、そのような視点で
　世界を見ていることになる。

　その認識の集積が「仕事観」。

　そして仕事観が行動（仕事の進め方）を
　規定している。


　-----------------------


　同じ仕事という言葉でも、
　まるで異なる定義を与えている人もいる。

　極力やりたくない、
　出来ることならやりたくない、
　カネのために嫌々行うものである、

　と記述している人が
　あったといたしましょう。

　（後略）


　-----------------------


　あるいは、

　私（鮒谷）の場合、仕事について
　
　「何らかの代価を頂くためには
　　顧客の期待値を超えて驚かせるべき」

　と脳内辞書に記述されているが、


　別のある人は

　「期待を下回り、激怒されたり、
　　クレームになったり、
　　トラブルにならない程度の最小労力をもって、

　　ギリギリのハードルで顧客の
　　最低満足ポイントを超えるのが仕事」

　それをうまく行えるのが

　「必殺仕事人」
　
　という仕事観に基づき、
　仕事をしていたりする。


　良い悪いではないが、

　前者と後者で仕事のクオリティが
　どう変わるかは推して知るべし、

　ではないか。


　※以上、音源内容の、
　　ごくごく一部をチラ見せ。

　　この続きは音源でお楽しみ下さい）

　　何度もお聴きしているので）

　この音源で語っている話には、
　絶対の自信と確信があります。
