◎本メールマガジン（平成進化論）は、

　複数の会社を経営し、IT、人材、環境、飲食、ホテル、海外事業など
　20社以上のベンチャー企業に出資する投資家としても活動する筆者が、

　　------------------------------------------------
　   骨太かつ卓越した＜本物の教育サービス＞を通して
　  「変容の喜び」を提供し続ける
　　------------------------------------------------

　ことを目的とし、3000日以上にわたってお届けしている、
　個人発行として日本最大級のビジネスメルマガです。

　熟読するも読み流すも、自由な形でご活用下さい。

　★鮒谷周史プロフィール http://www.2nd-stage.jp/profile/

▼宮崎奈穂子さんの公演を開催

───────────────────────────────────



■こんにちは、鮒谷です！


　昨日は、

　昨年11月、武道館でのコンサートを実現されたアーティスト、
　宮崎奈穂子さん


　【今日のお勧め本　路上から武道館へ】
　 http://amazon.co.jp/o/ASIN/4806145416/2ndstagejp-22/ref=nosim


　【今日のお勧め本　路上シンガー宮崎奈穂子 武道館公演への軌跡】
　 http://amazon.co.jp/o/ASIN/4870354713/2ndstagejp-22/ref=nosim


　の公演（講演＋歌）を主催させていただき、
　大盛況のうちに無事、終了することができました。
 


■路上ライブの活動から武道館に至るまでの軌跡を話と歌で
　伝えて頂き、

　参加下さった皆さんにとっても、
　満足度の高い会となったのではないかとおもいます。


　私自身、とても感動、感激しました。
　特に私が感銘をうけたのは３つの点。


　１、夢は持ってもいいんだ！
　　　自分で自分の可能性に蓋をしないこと

　２、ご縁を徹底的に大切にすることが大事なこと

　３、とはいえ、圧倒的な行動量がなければ
　　　現状は変えられないこと


　いろいろ考えさせられましたし、
　元気、勇気をいただきました。
　
　私も頑張ろう。



■参加下さった皆さま、ありがとうございました！
　
　宮崎さん、スタッフの皆さま（＆当社スタッフ）には
　夜遅くまでお付き合いいただき、

　本当にありがとうございました！


　本日、3月2日（土）放送（23時～）の日本テレビ
　「心ゆさぶれ！先輩ROCK YOU」で、

　【トゥルーストー リー】

　というコーナーで、宮崎奈穂子さんが取り上げられるそうです。


　よろしければぜひ御覧になってみてください。
　（残念ながら私はテレビが家にないので見られませんが、、、泣）

　http://tv.yahoo.co.jp/program/67824049/

　まるでシンデレラのような歌姫…のようですが、
　正直最初は路上で歌うことが好きではありませんでした。


　「レコード会社の方針だから」、そこがスタートだったけれど、

　道行く人に歌うからこそ、ひとりひとりの心に届くように

 【　応援される「お人柄」  】

───────────────────────────────────



■昨日の宮崎奈穂子さんの公演で感銘をうけたのは、
　宮崎さんが

　「縁をとても大切にされてきたところ」

　です。


　応援されるお人柄、というものがありそうですが、

　同時に苦労しながら、
　そういう自分に作り変えて来られた、

　というようにも思われます。



■いずれにしても尋常一様ではない圧倒的な努力
　（ご本人は努力と思われていないかもしれませんが、、）

　を重ねてこられての、

　「御縁を紡ぎ、紡がれる力」

　であり、
　
　「応援される力」

　であったかと思われます。



■公演終了後、たくさんのご感想も頂いております。


　（ここから）
　-------------------------------------------------------------


　宮崎奈穂子さん、「夢への努力」が服を着たような方ですね。

　その懸命な姿に大勢の方々が共感してくださった結果が
　「フツーの女の子が武道館単独ライブ」に繋がったことが
　よおくわかりました。


　宮崎さんのライブを聞きながら
　その昔、渋谷のジァンジァンという小さなライブハウスへ
　津軽三味線の初代 高橋竹山さんのライブに伺ったときのことを
　思い出しました。

　彼の生のステージに接して
　
　「ああ、ここに集うお客様達は、単に演奏を聞きに来ているのではなく、
　　このオジサンに会いに来ているのだ」と感じたのです。

　宮崎さんのライブに集う方々もきっと同じではないでしょうか。


　また、今でこそ大御所的な扱いをうけている桂文枝さん
　（元 桂三枝さん）や立川志の輔さんも、
　実は宮崎さんと同じ努力型の成功者です。

　もれ聞いたお話では（古い記憶で、錯誤があったらお詫びします）
　文枝さんは若い頃、ラジオ局へ早めに行って、リスナーの葉書の山から、
　自分宛や自分について書かれたものをピックアップして山の上のほうに置き
　ディレクターやプロデューサーの目にとまりやすくするよう工夫。

　名刺を通常より大きくつくり、通常サイズと重ねてはみ出す部分に
　自分の名を印刷。

　立川志の輔さんも広告代理店勤務で得た知恵も活かしながら
　通常の落語家さんがしないようなことも含め、
　少しでもお客様が心地よいように
　できる工夫はありとあらゆる方法でする。

　という姿勢を貫いていらっしゃいます。

　（彼の事務所のDMはいつも凝った素敵なもので、
　　捨てたくなくて取ってあるものも多いです）

　もちろん本来の高座の工夫も大変緻密になさいます。
　声の高さをどうしたら、お客さんの耳に受け入れてもらいやすいか
　まで気をつかわれます。

　ご本人は左利きなのに、蕎麦を食べるお箸の仕草は右手です。

　その理由が
　
　「『あら、この人左利きだわ』などという余計な情報で
　　お客様の噺への集中力を削ぎたくないから」
　　
　だそうです。

　宮崎さんがこれからどのようにご活躍なさるのか
　楽しみにウオッチさせていただきます。

　このたびは素敵なライブを企画してくださいまして
　ありがとうございました。

　　　　　　　　　　　　 書家・田坂 州代（たさかくによ） 様


　-------------------------------------------------------------

　
　本日は大変貴重な機会を下さり、本当にありがとうございます。

　鮒谷さんからのメッセージを戴いた瞬間、反射的に体が動いて
　申し込みをしておりました笑


　昨年の１１月に鮒谷さんとお逢いしたとき、
　ちょうど鮒谷さんが宮崎さんの武道館ライブに行かれたすぐ後で、

　当日やメルマガでもその話をされていたので、
　是非私も逢いたいなぁ、歌と話を聴きたいなぁと思って書籍を購入し、
　より一層お逢いしたくなっていました。

　まさかその夢を鮒谷さんが叶えて下さるなんて、
　ダブルパンチで嬉しい限りです。涙


　そんなこともあり、また歌や話の内容もあり、
　今日は感極まって涙してしまいました。。。


　それほど今の私のモヤっとに突き刺さり衝撃的でした。

　宮崎さんは私よりも二歳お若く、ちょうど私の妹と同い年でした。
　年齢も近いこともあり、近いながら余計に身近に感じるものがありました。


　宮崎さんも鮒谷さんもおっしゃっていましたが、
　何よりも圧倒的な行動ですね。


　そしてご縁を大切に。これは私も「縁する人を幸せに」という夢？を
　掲げていますので、

　ご縁を大切にして「私」をつくって下さっている皆様に
　恩返ししていきます。

　
　正直、今回ほどの価格の講演？公演？に、なんの迷いもなく、
　一分もかからずエントリー完了したのは恥ずかしながら初めてでした。

　大きな声では言えませんが、正直まだまだ余裕もなく。。。笑


　しかし、今日のこの素晴らしい内容をあれだけの人数で、
　そして何よりもたくさんの氣づきを戴き、色々なお土産？もついて、
　この価格は正直ものすごく、ものすごく、ものすごぉーく、
　価値のあるものでした。

　何の迷いもなく即行動して、本当に、本当によかったです。
　
　これも鮒谷さんと宮崎さんのお蔭ですし、
　おふたりが教えて下さっていることなんですね。

　これからも、鮒谷さんから教えて戴いた「すべての学びは前倒し」を
　実践していきます。


　　　　　　　　　　　　　　　　　　　　　　桑原 孝太 様


　-------------------------------------------------------------

　
　富川一男です

　（本日参加者のうち飛びぬけての高齢者です
　　ー申込みを断られるかと懸念していました）。


　本日はお世話になり有難うございました。

　１、スタッフの方の好感のもてる対応
　　　（会場地図もわかりやすかった）

　　　そして鮒谷さん自ら自然体でにこやかに応対頂いたことに
　　　感銘を受けました。

　２、宮崎さんの声は温かく知的で若い歌手にしてはボリュウムが
　　　あると感じました。10年後の進化が楽しみです。

　３、お話しの中で鮒谷さんもご指摘になりましたが「縁の大切さ」と
　　　「全身全霊に神意が宿る」に共感しました。

　４、私はサラリーマンのあと大学教授そして７０歳から熱心に
　　　歌－シャンソン、カンッーネな　どーを勉強し年間２０回くらい
　　　ステージに立っております。

　　　客観的に見れば老人、余命いくばくもない身ですが本日のお話しを
　　　伺い業界のことも一部わかりましたし視野が広くなり
　　　勇気づけられた感がします。

　　　宮崎さんのますますのご活躍をお祈りします。　　　

　
　　　　　　　　　　　　　　　　　　　　　　　富川 一男 様


　-------------------------------------------------------------


　今回はじめて、鮒谷さんの開催されるイベントに
　参加させていただきました。

　 まず最初に感じたことは、

　「鮒谷さんって、めちゃめちゃ気さくな方だなー」

　ってことでした。


　会場に入って来られる参加者の方お一人お一人と談笑されていて、
　はじめてお会いする私にもとてもフレンドリーに
　接していただけました。

　スタッフの坂元さん・長内さんも、とても気さくに会話していただき、
　とてもリラックスした空間で参加することができました。

　ありがとうございました。


　そして、宮崎さんのお話と生歌は、本当に素晴らしかったです。
　これまでの体験談をシェアしていただきましたが、

　そのどれもが本当にチャレンジし続けてきた方であるから
　こその「本物さ」を感じました。


　そしてその本物さには、実際にその空間にいるからこそ
　感じとれることだとも思いました。


　行動を止めないこと、決してあきらめないこと、
　ご縁を大切にすること、感謝すること。


　宮崎さんの大切にしていることに触れ、自分自身も
　よりパワフルに挑戦し続けていこうと、決意を新たにしました。


　そして、路上から武道館へ、宮崎さんが夢を実現したように、
　私も、現場に立ち続け、現実に成果を創ることにチャレンジし続けます。

　貴重な機会を、本当にありがとうございました。


　　　　　　　　　　　　　　　　　　　　　　　  田口 弘和 様


　-------------------------------------------------------------


　鮒谷様、宮崎様

　本日はとても素敵なコンサートを有難うございました。

　生まれて○十年、これまでこのようなコンサートには
　一度も参加させて頂いたことがございませんでした。。。。。


　今回は、鮒谷さんのお声掛けだったからこそ、

　「自分の全く知らない世界を覗いて見るのもいいかも」

　と思わず申し込んでしまった（申し込んでしまうことができた）
　次第です。


　宮崎さんの、夢に向かう直向きさと飾り気のないトーク、
　柔らかで透明感のある歌声。

　これまで眠っていた脳ないしは心のどこかの部位が刺激され、
　心地よい余韻に浸っています。


　「路上から武道館へ」の歌には宮崎さんのこれまで「思い」が
　凝縮されているかと思いますが、

　「思い」を歌詞という言語にて表現し、さらにそれをメロディに
　載せて人に伝えることができるのは素晴らしいなと改めて感じました。


　宮崎さんと鮒谷さんの最後のお話もとても印象的でした。
　自分なりの解釈、理解を含めて記させて頂き、糧としていきたいと
　存じます。

　○「出会い」は、自らが出かけて行って会うこと（待っているものではない）
　○全身全霊のときのみ、神意が働く

　○自分の可能性に自分で蓋をしない
　○無数の手数を打ってこそ
　○改めてご縁の不思議さ、大切さ

　このような言葉は、活字で何百回・何千回読むよりも、
　実際に実践されている方から直接伺うと重みが全く異なる
　（腹に落ちる）ものですね。

　宮崎さんの益々のご活躍を楽しみにしております。
　（今度はやっぱり路上で歌う歌声を聴かせて頂きたいかな。）

　有難うございました。

　　　　　　　　　　　　　　　　　　　　　　　　　　Ｙ・I 様


　-------------------------------------------------------------


　お世話になっております。佐々木大樹です。

　昨日は貴重な時間をありがとうございました。
　
　鮒谷さんにも宮崎さんにもお会いすることが出来て、
　とても良い一日でした。


　宮崎さんの歌声はとてもきれいで、
　お話とともに心に響いてくるものを感じました。

　鮒谷さんも仰っていたことですが、宮崎さんは
　『自分には何もない』と言いながらも、圧倒的な手数と、
　ご縁を大切にする姿勢で多くの支援者を集めたのだと思います。


　圧倒的な美貌や歌声が仮になかったとしても、
　十分に持って生まれた才能を活かしていらっしゃる方だと
　感じました。

　それとともに、自分は仕事においてもまだまだ余力を
　残してやっているなと感じてしまいました。

　歩きながら、仕事しながら、寝る！くらいの勢いで
　仕事にも全力で取り組んでいくためのエネルギーをいただきました。

　ありがとうございます。


　　　　　　　　　　　　　　　　　　　　　　　　　佐々木 大樹 様


　-------------------------------------------------------------
　（ここまで）


　その他、たくさんのご感想を頂戴しているのですが、
　とてもここでは載せきれません、、

　ご感想下さった皆さま、ありがとうございました。

　お喜びいただき、主催者冥利に尽きるというものです。



■皆さんがお寄せ下さったご感想からわかると思いますが、
　つまるところ、何事かをなそうとするのであれば

　冒頭でも少し申し上げたような


　▼ご縁を徹底的に大切にすること

　▼応援しようとおもわれるほど、
　　圧倒的な努力、手数を積み重ねること

　（そうして応援、協力したいと思われる人になること）


　が大事なのだと思います。



■宮崎さんが路上ライブを終え、
　帰られるのはいつも終電だったそうで、
　
　歩きながら寝る（寝ながら歩く？）ことも
　しばしばだったそうです。


　私も会食しながら寝落ちしてしまったこともある
　起業直後のことなどを思い出しておりました。

　生ぬるい活動してる場合じゃないなあと
　喝を入れられました。
　

　夢を夢で終わらせないために、
　行動し、形に変えて行きます。
　

　宮崎さん、参加下さった皆さま、関係者の皆さま、
　本当にありがとうございました！！

　これからも宜しくお願いいたします！
