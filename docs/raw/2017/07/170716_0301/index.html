【絶対的な自信が欲しければ】圧倒的な自信は、圧倒的な継続からしか生まれない

───────────────────────────────────



■一日、あいだが空きましたが、


　先日（３日前、ならびに２日前）に記した
　メルマガにて、


　【 毎日の取り組み → 時間経過による蓄積 → 複利効果】

　のコンボのパワーを享受し、
　成果を上げるために、


　【継続を主とした、
　　思考習慣、行動習慣、生活習慣】

　をインストールしてみられてはどうか、

　と、お伝えしました。


　今日も続けます。



■２日にわたって

　「継続の力」

　についてお伝えしたわけですが、


　実のところ、私自身、毎日当たり前のように
　小さな積み重ねを粛々と継続してきて、

　やがてそれが、

　時間の経過とともに、これほどの成果を
　もたらしてくれることになるとは、

　夢にも思いませんでした。



■ただただ、未来を信じて、
　ひたすらに継続しているだけで、

　あるとき突如として、

　【継続＆蓄積してきた知識や能力、
　　スキルやセンス、人とのつながり】

　が、実はとてつもない財産と
　なっていたことに気づく瞬間が訪れ、

　
　その瞬間から、自らの内に
　
　【絶対的な自信】

　が生まれるようになりました。



■こうした経験を通して

　【継続＆蓄積の力】

　は、閾値（いきち、しきいち）を超えると、
　一気に成果がもたらされることとなり、

　その成果が

　【絶対的な自信】

　をもたしてくれると、今は断言できます。



■ただし、

　【閾値超えの世界】

　には未だ到達しない状態のまま、
　粛々と継続を行っている人は、


　往々にして、自らの継続に対して、
　ともすれば

　【過小評価してしまいがち】

　なのです（私もそうでした）。


　これが悪いほうに出れば、最悪、
　途中で心が折れてしまい、

　せっかくの継続を断ち切ってしまうことにも
　なりかねません。



■（そのような危険な側面はあるけれども）

　継続のプロセスを過小評価してしまう
　ことにもよいところがあって、

　それは、継続の力を小さく見積もってしまうことで


　【まだ足りない、まだ足りない、、、
　　との思い（過小評価の認識）が、

　  日々、投入できる限りのエネルギーを
　  対象分野に投下し続ける力がもたらされる】


　ということがあるのです。


　
■こうして過小評価し続けてきた蓄積が、

　ある段階で、本当に目に見える結果として
　現れるようになったとき、


　そこで初めて、

　【実は自らの中に、
　　とんでもない量の蓄積を行っていた】

　と気付く、そんな利点もあるものです。



■私（鮒谷）自身、

　読む習慣、としては30年以上、
　書く習慣、としては約25年、

　さらに、

　出来る限り多くの人と会い、
　対話を重ねていく、

　という習慣を約20年続けてきました。



■その途中には、幾度も

　「一体、こんなことを繰り返して
　　果たして報いはあるのだろうか」

　と不安に押しつぶされそうになったことも
　ありますが、


　こうした取り組みが、長い時を経て、
　
　（それが外から見てどのように見えるかは
　　分かりませんが）

　少なくとも、私（鮒谷）自身が自分に対して、
　圧倒的な自信や、安心感をもたらしてくれたのは、

　もはや疑いようのないことです。



■だからこそ、
　そのような経験を通して知らされた

　【ともすれば過小評価されがちな、
　　継続の力】

　について、

　【過小評価されているから、

　　普通の人はやりたがらない、
　　継続の力を味方につけるべし】

　と今一度、声を大にしてお伝えしたいと
　思うのです。
