■今までのビジネスでは考えられなかったほどの規模のことを

　【インターネット】

　を利用することによって加速的に成立させてしまうことができるのが
　現代という時代です。

　元失業者（＝私のことです;;）は、自らの身体を実験台にして現実に
　それを証明することができました。


■メールマガジンを配信し始めて、わずか1年半しか経過していないにも
　関わらず、ゼロから開始したビジネスとしては信じられないくらいの
　結果を短期間に残すことができました。


■ネットに参入する前からそこそこいける予感、というか根拠のない確信は
　あったのですが、その直感に具体的な戦略戦術を肉付けしたわけです。

　その戦略戦術を実践して形にすることにより、今のビジネス、そして
　ライフスタイルが可能になったわけですが、想像以上だったその威力に
　私自身が最も驚いています。


■しかしそれは方向性を間違えなければ、誰でも可能なことだと思います。


■メルマガではあまり記してこなかった、私のいまのビジネスモデルが
　如何にして生まれたのか。それをどうやって育ててきたのか。

　さらに、今後の展開は？といったテーマについてもこのセミナーで
　お伝えさせて頂きます。

　
■個別に私が手取り足取り、指導する、というスタイルはとりません。

　ただし、機関銃のように、利益を生み出すビジネスモデルを構築する
　ためのヒントをお伝えいたします。


■少なくとも数十、多い人で数百項目の新しい発見を持って帰って頂ける
　とおもいます。

　なにしろ何百万もお金をかけて、何千時間も時間をかけて勉強し、
　寸暇を惜しんで仮説と検証を繰り返したその2000日のエッセンスを
　3日に凝縮してお伝えしようとするわけですから、当然といえば当然の
　ことですが。


■たくさんお伝えする中の、一つでも二つでも自社の、あるいは自分の
　ビジネスに取り入れられれば、即日参加費の元をとることができる内容
　をお伝えいたします。


■ただし、あくまで、私が考えてきた思考のプロセス、実際の行動を
　ヒントにして、自身でビジネスを組み立てようと思っておられる
　主体的な方の参加のみお待ちしております。

　
■そういう意味において、私の話の中から自社の戦略戦術に落としていく
　ことが出来ない方の参加は今回はご遠慮下さい。

　あくまで私の事例をお伝えするだけですから、そこから何かを抽出して
　自社に取り込むことができないのであれば、ご参加いただいても
　成果物を得て帰っていただくのは難しいかと思いますので。


■いろいろ書いてきましたが、でも、期待値はくれぐれも「低く」参加
　くださいね。「またセミナーに無駄金使ったよ」といってのけるくらい
　余裕がある人にご参加いただければ最高です。


■聞く人が聞いたら大したことないなー、という話しかできませんから。
　立派な資料がついてくるわけでもありません。
　ほんとに何にもありませんよ。そもそもパワーポイント使えませんし。
　特別なお土産があるわけでもありません。

　それでよければ。


■「放談会」のスペシャル版みたいなものですので。
　その理解を前提として参加くださる方のサロンのような形にしたいと思って
　います。
　

■くれぐれも借金して参加する、これで結果が出なければ路頭に迷う、
　なんて方は参加はされないで下さいね。
　（そんな無茶な方はいないと思いますが）


■あなたとの、久しぶりの再会、あるいは新しい出会いを本当に楽しみ
　にしております。
　
　
■追伸：

　本年度中に同じ内容のセミナーを行うことはありません。

　
　【セミナーに参加する

■私は、先日ここで申し上げた



　【継続的学習によるアウトプットの最大化を図り、

　　　　　　　　　　肉体的・精神的・金銭的自由を得るライフスタイル】



　を提唱するというジャンルにおいて極めようとしているつもりですし、
　何よりそれが好きですし、日本一努力しているつもりです。

　その学んだことも日本一、アウトプットしていこうともおもっています。


■おかげさまでいくつかの携わっているビジネスがいずれも軌道に乗り
　つつあります。

　今後もますます加速することになり、セミナーの参加の時間や、
　本を読む時間、ＤＶＤを見る時間がさらに増えていくことと予測
　しています。


■こうした結果とそこから得られる圧倒的自信はどこから生まれてきた
　のかを振り返って考えてみました。


■あなたも寝食を忘れて没頭する趣味があるかもしれません。

　あるいは、このことについて話し出したら三日三晩止まらない、
　そんなものもお持ちかもしれません。


■そこまで没頭できるものが、あなたのビジネスにつながるのであれば、

　そのビジネスは誰にも負けない圧倒的自信をもって展開することができる
　ことができるし、

　そのことが他社（他者）を寄せ付けない参入障壁となるのです。



■私の場合は、「 学び ⇒ 結果 」のサイクルを回しながら、
　どんどん自分が自由になっていく、

　そんなライフスタイルを追求することにおいて、日本一であると
　言い切れるようになりたいと思っています。


■各人それぞれが切り開くジャンルにおいて、自分が圧倒的ナンバーワン
　であるとの自信を持って語ることができる、そのポジションこそを
　目指すべきであるし、

　そこまで徹底してやることによって圧倒的な自信と、顧客に媚（こ）びない
　姿勢が身に付きます。

　そしてそれ自体が強力なマーケティング戦略になり、強固な営業力となり、

　気がつけばあなたの前に、数多くの顧客、クライアントの行列ができて
　いることになるのです。


■くれぐれも繰り返しますが、
　
　あなたの経験の裏付けがある自信こそが、あなたのビジネスを加速させる
　重要なファクターです。

　ここにすべての資源を投入することが、成功への必須条件といっていい
　のではないでしょうか。

■いったい何が、町田さんにそのような偉業を達成させたのでしょうか。

■その重要なカギとなる本が、このたび満を持して発売されます。
　キーワードは「上司と部下の円滑なコミュニケーション」。

■どんなにあなたの能力が高くても、
　どんなにあなたの部下の能力が高くても、
　あなたと部下の「パイプ」がつながっていなければどうなるでしょう。

■逆に、ひとたびそこに、円滑なコミュニケーションというパイプが
　つながったならば、町田さんのような業績も夢ではないのでは。
