 【　サンドバッグみたいに殴られた体験はありますか？　】

───────────────────────────────────



■よく、

　「メールマガジンを発行していると、
　　たくさんの、そしていろいろな感想が届くんでしょうね」

　と言われます。


　おっしゃる通りで、
　実際、たくさんのご意見、ご感想を頂戴いたします。


　読ませていただくと、

　非常にありがたいご感想ばかりで、
　嬉しくなったり、心なごんだり、

　ときに涙が出そうになることもあったりします。



■その一方、

　数は非常に少ないのですが、

　発信者として、正直、読むのが苦痛になる反応を
　頂戴することもあります。


　それは、的を射ている指摘であることが
　自分でも分かっているがゆえに苦しくなるものもあれば、


　どれほど自分と切り分けて、客観的に読もうと思っても、
　思っても、思おうとしてみても（！）

　「この方は、他のところで受けたストレスを、
　　たまたま私にぶつけてこられたのでは」

　と思えてしまうような感想（叱責）を頂戴することも
　あります。



■昔は、今以上に人間できていなかったので、


　「お金もらって送ってるんじゃないんだから、
　　嫌なら、読まなきゃいいじゃないか！！（怒）」

　とか、思ったりもしたものですが、
　　　

　あるときから、そのようなご意見、ご感想の
　一切合切を引っくるめて、すべてを
　
　「ありがたいなぁ」

　と思えるようになりました。



■（今もそうですが）昔は特に、

　すぐに腹が立つし、すぐにショックを受けるし、
　すぐに感情が波立つ、

　といった塩梅でしたから、
　我ながら、これは驚くべき変化（進化）です。

　
　感情の振れ幅の大きかった私が、
　
　このように思えるようになったのは、
　ひとえに量稽古の賜物と考えています。



■11年近くのメルマガ配信で、
　日々、たくさんのご感想を頂戴してきましたし、

　累計では、万単位の感想を拝読してきました。


　読み進めたくない感想をいただいたときでも、
　とりあえず、たとえ心からは思えなくても、

　「ありがたいなぁ」

　と呟きながら、読み進めるようにしてきました。


　すると、後から、こんなことが思えてくることに
　気づいたのです。



■自分では

　「これが正しい」

　という認識で書いたのだけど、

　正しいか、正しくないかは、
　どこまでいっても個人の主観であって、


　自分にとっての「正しいこと」が、
　ある人にとっての「正しくないこと」であることは、

　それこそ日常茶飯にある、

　そんな当たり前の事実を素直に受け止められるように
　なりました。



■実のところ、たくさんの部数のメルマガを配信してきた
　からこそ、

　「それが世の中というところなんだ」

　という事実を、

　（頭ではなく）身体を通して学ばせてもらえたことが、
　今となっては、私にとっての一番の財産になっている、

　そんな風にすら思えます。

　

■つまりは、


　「どれだけ自分として冷静に、客観的に見て

　　『これが絶対に正しい』

　　と思っていることであっても、


　　それについて

　　『絶対に正しくない』

　　と思われる方は必ず存在するし、
　　その方々もまた正しい」


　そんなところが世の中であるのが分かった、ということ。



■これに気づいたとき、

　「痛いのに、痛みが感じられなくなる経験」

　という、矛盾を含んだ、不思議な体験をすることに
　なりました。


　この経験を通じて、
　他の人とコミュニケーションを取るにあたっての、

　「両者を俯瞰する視点」

　を無意識レベルで持てるようになったことは、

　人生の質を劇的に変えられた体験であったのです。

　

■こうして、

　「ありがたいなぁ」「ありがたいなぁ」

　と

　「ありがたくもないのに」

　辛抱して唱え続けているうちに、


　どんな感想や指摘を頂戴しても、上述の世界観を
　強化してくれる、本当に

　「ありがたい出来事」

　となりました。



■期せずして、お金では決して買うことができない、

　このような心的態度を持つことができたことは、
　メルマガを続けてきて得られた、

　一つの大きな財産です。


　「ありがたいなぁ」「ありがたいなぁ」「ありがたいなぁ」

　と唱え続けているうちに、本当に、

　「ありがたいなぁ！」「イヤッホー！」

　と世の中に対して、
　叫びたくなる気持ちまで湧いてきたのです。


　人の心って、本当に不思議。

　ときにサンドバックのように、たくさん、
　かつ一方的に、殴られてみるのも悪くありません。
