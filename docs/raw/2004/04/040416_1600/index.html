───────────────────────────────────


■昨日のメルマガ、なかなか反響よかったです。

　歓迎９割以上、

　といった感じでしょうか。ちょっと意外でした。

　あんまり生々しいのはやめてほしい、というメールが殺到するかと
　思いながら書いたのですが、実は、こういう情報がほしかったんだ、
　もっと開示して欲しい、なんてメールを大挙頂きました。
　
　こんなにたくさんメールを頂くことは、あんまりないですけどね。


■味をしめて、続編を。

　与えるところに、ビジネスはある。

　よく言われる言葉です。
　人さまのためにならないビジネスはビジネスではないし、
　だまし、詐欺、といわれます。


■あくまで、まず、お客様にできるだけのことをして差し上げる、という
　ことが大事ですね。

　と同時に、自分のキャッシュポイントはどこにあるのか、ということを
　明確に設定しておかなければなりません。

　一体、どこで自分は課金するのか、ということですね。
　これを抜きにしてビジネスは存在しえません。


■人さまに喜んでいただいて、お金を頂くから商人なのです。
　
　商人はまず与え、他を利し、その後、自分も残りの幸せを享受できるから
　幸せなのです。


■会社勤めであってでも、自営であっても、経営者であっても、
　すべからくビジネスパーソンは商人でなければならないと思います。

　喜んでいただき、対価を頂戴する。だから長続きします。
　喜んでいただくのはいいけれども、対価を頂かなければ長続き
　いたしません。

　ということは結局、それはお客様にとっても不幸なこと。


■私でいうとこのメルマガでの現在の一番のキャッシュポイントは、
　広告収入となっています。
　
　ですから、無償で掲載することは決して行いません。それはどれだけ
　親しい友人に対してでもそうです。
　きっちり対価を頂戴いたします。（どれだけがめつい！といわれても）


■結局、それはビジネスを存続させ、最終的に２回、３回、４回と告知
　させていただくことで、掲載していただいた方にもより喜んでいただく
　ことになると信じているからです。

　義理、付き合いで、たくさん掲載すると、それだけで誌面が埋まって
　しまい、肝心のコンテンツが載せられないということもありますし、
　それを行うことで、クライアントさまからの信頼を失うことが最も
　怖いです。

　そして媒体としての力がどんどん衰えていく…
　考えただけでも恐ろしい現象です。


■ですから、どれだけ親しい友人であってでも、心を鬼にして、対価を
　請求させていただきます。
　そしてここで真剣勝負にもなるんですけれどもね。

　甘えを廃した真剣勝負だから、結局、お互いにとってメリットがある
　のだと思っています。


■昨日、ちょっとメルマガでも触れた、読者を獲得するための大キャンペーン
　を行い、予算を投下する、というのもクライアントの方々の満足度を
　高めるためですし、さらに、予想を上回る結果をなんとか出させて頂きたい
　との思いからなのです。


■というわけで今日の内容は、


　▼まず与えよ、結果を出せ
　▼でもキャッシュポイントを明確に
　▼商人なんだから、与えて、その後、残りを頂く。
　　でも間違いなく、頂くものは頂かないと、長続きせず、社会にとっても
　　不幸なこと。
　▼だから商人は稼ぐことが重要。
　▼世のため人のため、でも稼げないのは存続しない。
　　結局、双方が不幸になる。
　
　といったところでしょうか。
