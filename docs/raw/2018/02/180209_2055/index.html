　（ここまで）


　ここでのポイントは、

　「この賞品が14ヶ月も前にリリースした
　　音源である」

　ということです。


　
■それに加えて、

　「これまでにもう、当音源は
　　散々、ご案内し尽くしてきた」

　ということも補足しておきましょう。


　それでもなお、ここにおいて

　【15,120円（※ここから若干の決済手数料が
　　引かれます）の音源が、

　　わずか2時間で32本、
　　約50万円の売り上げ（≒利益）】

　程度には売れているのは、


　（そして、上の数字は一部の切り出しなので、

　　この前後、数日にわたり、似たようなペースで
　　延々、売れ続けてきたわけですが）


　【たった一度、行った仕事が、
　　永続的に価値をもたらし続けてくれる】


　という具体的なイメージをかき立ててくれるに足る、
　十分な証拠といえるのではないでしょうか。



■ちなみに「目標設定＆達成」というテーマは、

　（数年やそこらで陳腐化するような
　　テーマではありませんから）

　一度、制作すれば、相当、長期に渡って、
　ご案内をし続けられるのが

　「ビジネスにおける、最大の利点」

　と言って良いでしょう。


　
■極端な話、なにかの拍子に私（鮒谷）が
　ポックリ逝ってすら、その事実を知られることなく

　「音源が売れていく」

　ことだって可能性としてはあるわけです。

　
　（遺される人がある方にとっては
　　「遺族年金化」することも可能かもｗ）



■繰り返しになりますが、これが

　【たった一度、行った仕事が、
　　永続的に価値をもたらし続けてくれる】

　ということの意味です。


　ビジネスが安定しない、
　常に自転車操業、

　という方は、ひょっとすると

　【たった一度きりの商売を、
　　延々と続けようとしている】


　言葉を換えれば
　　
　【自分の限界に挑戦！というビジネスを
　　展開しようとしている】

　ということになりはしないでしょうか。



■それだと、どこまでいっても永遠に

　「回転木馬のデッド・ヒート」

　もとい

　「回し車のコマネズミ」

　のように、いつまでたっても大変で、
　
　「はたらけど
　　はたらけど猶わが生活楽にならざり
　　ぢっと手を見る」

　そんな仕事や生活が続いてしまうことに
　なりかねません。



■それが嫌ならば、つねに

　【今の自分の営みは、
　　必ず将来につながる資産に変えていく】

　という発想を持って、
　仕事に取り掛かる必要があるのではないでしょうか。


　ビジネスにおける、根本的な
　パラダイムシフトを起こさなければ、

　「死ぬまでひたすら、時間に追われながら
　　肉体労働し続ける」
　
　ことになるのです。



■なにもそうした生き方がダメだ、
　と言ってるのではありません。
　
　それがダメなら、私（鮒谷）も
　個別コンサルとか、とうの昔に止めています。


　私が今もなお、コンサルを継続して
　行っているのは、

　それが楽しいからであり、
　学びになるからであり、

　修行、訓練、だと思っているからであり、
　人間関係構築の営みだと考えているから、

　です。


　
■だからこれからも、少なくとも当面は、
　続けていくつもり。
　

　ただし、ここで重要なのは

　【何か事情が変わったら、いつでもコンサルを
　　止められるという自由を持っている】

　ということです。


　こうした状態を確立していることは

　【精神的、ならびに経済的な安定や平安】

　に直結するのです。



■たとえばの話ですが、

　私（鮒谷）の身体の具合に変調が生じたら、
　両親の介護が発生したら、
　本気で取り組みたいと思う事業に出会ったら、、、

　「すぐにコンサルをやめて、
　　新しい事態に対応できる」

　そんな自由を持つことができています。


　それはひとえに、音源販売という

　【たった一度、行った仕事が、
　　永続的に価値をもたらし続けてくれる】

　仕組みを私が作りあげることができたから。



■そんなシステムを人生や経営の中に
　インストールするお手伝いをいたしましょうか？

　とお伝えしているにもかかわらず、


　（それが実現されたら、めちゃくちゃ、
　　価値があると思いません？？）


　悲しいことには、
　なかなかその価値が伝わらないのか

　【私（鮒谷）のリリースしてきた他の音源よりも、
　　圧倒的に売上が伸びない（泣）】

　のはどうしてなのでしょう。


　まあ、私の人生ではないので、
　どっちでもいいのですが（と、開き直ってみる）。



■これ以上はしつこくなるので、
　これくらいにしておきますが、


　【たった一度、行った仕事が、
　　永続的に価値をもたらし続けてくれる】
　
　
　そして、それによって、


　【精神的、ならびに経済的な安定や平安】

　
　を手に入れられる、
　その最短距離（？）であるところの

　「音源販売」

　にご関心をお持ち下さる方であれば、
　きっと
