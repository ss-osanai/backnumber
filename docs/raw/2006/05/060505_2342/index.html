　【　ネット革命後は情報ハイウェーをとことん利用することが
　　　　　　　　　　　　　　　　　　　　成功への第１ハードル　】

───────────────────────────────────



■現代は、インターネットを通じて、

　ビジネスで精度の高い結果を出すための方程式がすばやく、そして
　安価に入手できる時代となりました。

　これはすばらしいことでありますが、また同時に、

　今まではごく限られた人たちしか入手できなかったような情報が
　すべてオープンとなり、知識レベルでは全員が「横並び一線」の状態
　になってしまった、

　ということでもあります。

　ネット革命は、こんな社会構造の革命まで引き起こしています。



■こうなると、ではどこで本当の差がついてくるのかといいますと、

　「知っている」から「実行している」へシフトできた人だけが
　勝ち残っていけるのです。



■これまでは、成功に至るまでに２段階のハードルをこえねば
　なりませんでした。

　成功法やノウハウを「知る」ことがまず大変。
　勝利のための方程式を知るという高いハードルを乗り越えてから、

　次に、その方程式通りに「実行」していく、というさらなるハードルが
　待ち受けていました。



■ところが今は、「知る」だけならほとんどコストをかけず、
　労力もかけずにポーンと跳び越えることができます。

　これまでは門外不出といわれてきたようなノウハウですら、
　ごく普通に流通しているのが現状です。



■ほんのちょっと前までは、能動的に、積極的に、意識して
　取りにいかなければ手にできなかった情報が、

　ほとんど受動的に流れ込んでくるといっていいくらいのところまで
　下りてきました。

　成功法則を知るために右へ左へと苦労して探し回った時代を
　知っている人にとっては、
　現代はとてつもなく恵まれた時代であることがお分かりだと思います。

　私たちは、幸いにもそんな時代に生きていることをまず認識
　しなければならないでしょう。



■これは、たとえていえば免許とりたてのドライバーがいきなり
　高速道路に出たようなもの。

　いままでは、さんざん下道で練習したあげく、高い高い
　高速料金を払った人だけがハイウェーに出ることを許されて
　いました。

　それどころか、高速道路の存在を知る人自体がごくわずかだった
　のです。

　一般の人は、高速の入り口がどこにあるかすら知ることは容易では
　ありませんでした。



■ところが今は、誰もが最初から高速道路で運転しているような
　もので、

　流れに乗って運転しているだけで、きわめて速いスピードで
　ある一定のところまでは到達することができます。



■成功にいたる方程式は、もうみんな知っている。あとはやるだけ。

　やりさえすれば、ある一定レベルのところまでは容易に到達できます。

　その一定レベルのところからさらに上を目指そうとなったときに、
　本当の、激しい戦いが待っているのです。



■まずはせっかくの情報ハイウェー、とことん利用し、実行しましょう。

　勝ち残るための第１ハードルが、知ることではなく「実行する」
　ことに移行してしまったのですから。
