■「こんな大変な思いをしながら私は一体、何のために経営しているのだろう」

　こんなことを、まじめに経営に取り組まれている「平成・進化論。」読者
　の経営者、経営幹部、リーダー、起業家のあなたも、ふと考えたことが
　あるのではないでしょうか？（わたしもあります）

■「このために経営者をやっているんだ！」という熱い想いや信念、確信が
　あれば、きっとその想いを原動力として、前に向かっていけるに違い
　ありません。

■一生懸命に経営に取り組むほどに「経営道」「会社経営とは」「経営者とは」
　といった哲学的、本質的な命題に必ず突き当たるはずです。

■そんな疑問を氷解させるヒントが、この３名の講師陣から得られることが
　できるかもしれません。今一度、「経営者本来の役割」について再考して
　みる必要性を感じられるのであれば、5月23日、この講演会をチェックして
　みられてはいかがでしょうか。

───────────────────────────────────



■こんにちは、鮒谷です。



■ゴールデンウィークが明けて一週間、今週は限界まで自らの
　肉体と精神を鞭打って、ぎりぎりのスケジュールで頑張ってみました。

　ちょっと怖かったので、昨日（土曜日）はアポを入れなかった
　のですが、よかったです。

　前日の金曜日はトドメを刺す、20時間連続労働。

　朝の6時に倒れるように床に就き、夕方の7時半に起床。
　13時間半、昏睡状態に陥っておりました。



■いろんな生活パターンを試してきましたが、
　私の場合は、平日月曜から金曜までは数時間睡眠で、自分を
　追い込んで、週末に12時間ほど眠って回復する、

　というのが良いように感じています。



■集中力は一度とぎれると、トップスピードに戻すのが大変で、
　せっかく乗った波をいかに持続させるか、にとても神経を使います。

　（とはいうものの、ずっと集中しつづけるわけにはいかないので）

　月曜～金曜は、ぎりぎりまで追い込み、週末に疲れを取る。
　これが現時点でいろいろ試してきた上での、私にとっての
　ベストパターンです。

　最近、プロフェッショナルのライフスタイル、に強い興味を
　持っていて、いろんな人の生活パターンを研究しています。
　


■一人ひとり、そのスタイルには差異があるのですが、皆さん、
　試行錯誤の末、その生活習慣を確立されているようです。
　（そして現在もよりよいものを求めて更新中…）
　


■イチローも、押切もえも、食事や睡眠など、かなり注意を払っている
　ようですし、だからこそトッププロとして君臨できるのでしょう。
　
　スポーツでも、なんでもプロは、24時間、最高のパフォーマンスを
　生み出すために様々な試行錯誤を繰り返しています。

　ビジネスパーソンも妥協なく、高いパフォーマンスを叩き出す
　ための、最高の環境を求め続けなければならないと考えています。



■それでは今日のピークパフォーマンス方程式（ビジネスにおいて、
　最高の結果を発揮するための方程式）をどうぞ。


　本日は、


　【 すべてのビジネスパーソンは熱烈かつ雄弁な語り部たるべし 】


　というテーマで、どうぞ。


　あなたのビジネスのヒントになれば幸いです。

＜今日のピークパフォーマンス方程式。＞

　【　すべてのビジネスパーソンは熱烈かつ雄弁な語り部たるべし　】

───────────────────────────────────



■経営者、営業部門の方はもちろんのことですが、

　人事部・総務部・経理部などの部門に携わっておられる方も
　間接的に自社の商品を売って収入を得ていることは間違いありません。



■ビジネスに携わっているということは、イコール、自社の商品を
　販売していること。

　ということは、どんなポジションであれ、企業に属している限り
　すべての人は商売人です。

　自社商品のセールスマンであり、広報マンであり、
　アンバサダー（大使）なのです。



■だから、モノを売るのは社長に任せておけばいいとか、
　営業に任せておけばいいと言うことなかれ。

　一人一人が、誇りと自信を持って、自社製品・サービスの
　語り部、伝道者にならなければならないのは言うまでもありません。



■それが大前提であることを銘記したうえで、

 「平成・進化論。」の読者さまの中には経営者の方がたくさん
　いらっしゃいますので、あえて申し上げますと、

　最も活動的であり外部とも接する機会が多く、
　内外に影響力があるのは、やはり経営者。

　経営者はだれよりも雄弁な語り部にならなければいけません。

　今は、顔のみえる経営者が扱う商品のほうが、より購買に
　結びつきやすいようです。
　きっと親近感がわくからでしょうね。



■そういう意味で、私も自社の商品・サービスの語り部として、
　日々伝道活動を行っています。

　この「平成・進化論。」がその記録であるとするならば、
　もう少しで伝道活動は1000日になります。

　この伝道生活を通して、より多くの方々と交流を持たせて
　いただくことができました。



■これが、もし私が仮に一生懸命に本を読み、セミナーに行き、
　なにがしかの知識や能力を身につけたところで、

　部屋に閉じこもって何も語っていなかったらどうであったか。
　決して今のようなビジネスはできなかったでしょう。



■雨の日も風の日も、自らの思いに従ってメッセージを語り続ける
　ことにより、多くの方に興味をお持ち頂き、認知してもらえた。

　それによってビジネスが成り立ってきたことは疑いようの
　ない事実です。



■おかげさまで、これまでなんとか無事にビジネスを進めさせて
　もらうことができましたが、

　それはこうした語り部としての活動を続けたことによる
　ところがたいへん大きいと思います。



■「平成・進化論。」の読者さまには、中小零細・個人企業の
　経営者さまが多くいらっしゃいます。

　みなさま、さまざまな形で内外へのメッセージを配信なさっている
　ことでしょう。

　ぜひそこに、熱い思いと、魂を込めて、継続して配信なさって
　いただきたいのです。

　それによって御社の事業に新たな展開が生まれることは間違いない
　でしょう。



■そしてあなたが経営者や経営幹部の方でなかったとしても同じです。

　こうした活動していくことがビジネスパーソンとしての経験や能力を
　向上させ、また周囲からの認知を高めることになりますので、

　すべからくビジネスパーソンは自社商品・サービスの語り部たるべし、

　とお勧めする次第です。

いつも、お金に代えられない貴重な学びをありがとうございます。
