▼最近の読書傾向について

■こんにちは、鮒谷です。
　　

　昨日は2.0時間、大学院受験の勉強。

　（３月累計：76.0時間）



■この数カ月、

　哲学、倫理、教育、政治、歴史といった分野の
　書籍を読む時間が大幅に増えています。


　血肉になって自分の言葉に乗ってくるのは
　まだまだ先（おそらく10年後くらい？）のことだと
　思いますが、

　私たちが常識だとおもっている
　「ものの捉え方」の系譜に触れ、

　感嘆、驚嘆している毎日です。



■話は変わりまして、

　以前にセミナー参加下さった方のブログを
　ご紹介。


　某大手製薬メーカーで、
　新薬開発に携わられている方です。

　（素晴らしい内容です！）


　＜システム化中毒化への道！＞

　http://ameblo.jp/miyahi24/entry-11193006556.html


　＜逆算思考；薬を作るために＞

　http://ameblo.jp/miyahi24/entry-11188155649.html



■セミナーでお伝えした内容を
　参加下さった皆さんが、

　さまざまな表現で翻訳してくださるので、
　誰よりも一番、私（鮒谷）が勉強させて
　いただいております。

　いつも楽しみに読ませていただいています。
　ありがとうございます！

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　「この指とまれ！」で渦を巻く　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

　【　「この指とまれ！」で渦を巻く　】

───────────────────────────────────



■昨日は

　「三本の矢で自らを律する」

　というテーマでお届けしました。

　今日も続けます。



■前回は、


　「意思の弱い人間でも３人集まれば、

　　一本一本（一人ひとり）の気持ちは弱くとも、
　　弱い意思の力が糾合され、折れにくくなる」

　
　だから、

　「志を同じくする人たちが集まるような
　　場を作る能力」

　を持った人が価値を生み出す、
　という話をしました。



■また、これが出来る人ならば、


　「一人では到達することが難しかったことでも
　　衆力を集め、衆知を借りつつ、

　　自分一人では越えられなかった
　　限界を越える」

　そんな体験ができるようになる、


　というお話もしましたね。



■これはいつもメルマガで書いている、

　「渦に巻かれる人」から「渦を巻く人」に
　なった瞬間から世界が変わる、

　ということと同義です。


　渦を巻く首謀者になった瞬間から
　自他の力を結集できるようになり、

　自分のみならず、巻き込んだ人までも
　ともに一大飛躍の道を歩めるようになるのです。

　
　
■だからこそ、


　「自分の力を鍛え抜くことだけを
　　考えるのではなく、

　　（自身も含め）弱い一人ひとりの力を
　　大きく糾合することによって、

　　自他の限界を跳躍する」


　そんな発想を持ってみてはどうか、
　という提案です。
　
　　

■この考え方はある意味、

　「夢、理想を実現させるための
　　パラダイムシフト的発想」
　
　だと思っています。


　少なからぬ人は、

　自分一人の力をたのみ、
　自分一人の能力を磨き、
　自分一人の意思のみを頼ろう、

　としがちです。



■しかしそうではなく、

　「みんなでゴールを目指そうよ。
　　そのためにまずは私が率先して渦を巻くよ！」

　というのが、上述の考えの根底にあります。

　
　「この指とまれ！」と言うための
　何ものか（理念、指針、方向、決意）を持つ、

　そして、実際に勇気を出して

　「この指とまれ！」と言ってみる、

　ここから何かが動き始めるのです。


　明日に続けます。



　今日も人生とビジネスを楽しみましょう！
