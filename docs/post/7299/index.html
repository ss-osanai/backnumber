7299号 「長く沈み続けられる力」が理想の世界に導いてくれる

■昨日のメルマガに引き続き、
　今日も、

　先日、『ハンチバック』で芥川賞を取った
　市川沙央さんのインタビュー記事

　-----------

　【芥川賞受賞】
　文學界新人賞・市川沙央さん

　「なにか職業が欲しかった」
　ままならぬ体と応募生活20年の果てに

　https://book.asahi.com/article/14917541

　-----------

　を読んで、
　あらためて思ったことを記します。



■一番に感じたことは、

　「ままならぬ体と
　　応募生活20年の果てに」

　のタイトルにも書かれているように、


　自分が納得できる人生を送りたいのなら

　【長く沈み続けられる力】

　の涵養を避けては
　通れないのではないか、

　ということ。



■別の角度から表すなら、

　【１、理想の状態への、強い憧れ】

　ならびに

　【２、自身を強く信じられる力
　　　（諦めない粘り）】

　を持続させられさえすれば、


　挑戦心や張り合いを持って
　毎日を生きていけるようになるし、

　そこから生まれる継続的な取り組み
　（＝長く沈み続けられる力）

　は夢を現実のものとしてくれる、

　ということ。



■前者（理想の状態への、強い憧れ）に
　よって、

　なにがどうなっても
　匍匐前進をし続け、

　目指す場所に必ず到達してみせるという
　強い信念や執念が生まれます。



■また、後者（自身を強く信じられる力
　/諦めない粘り）によって、

　戦いを重ねる中での過程は
　大変ではあるけれども、

　途中で折れること自体、ありえないと
　自分を信じられる力がもたらされる。



■この両者が伴えば、

　【理想とする状態】

　に少しでも近づいていけるのでは
　ないでしょうか。


　そのための時間軸を最低でも

　【10年単位】

　と考えれば焦ることもないし、


　競輪用語的にいうならば
　人生中盤から終盤における、

　「捲り（まくり）」

　が実現される可能性が高まるように
　思われます。



■長い人生においての

　「先行逃げ切り型志向」

　は、あとで失速するかも、
　という心配からのストレスもかかるし、

　なにより爽快感も味わえませんが、


　後方から一気の「捲り」が成功すれば
　脳汁がドクドク出てきますし、

　その可能性が失われない限り、
　テンションを上げたままの走行も可能。



■考えてみれば
　小説、漫画、映画、ドラマを見ていても、

　「先行逃げ切りタイプの主人公」

　なんて、
　ほとんどいないんじゃないですかね。

　仮にあったとしても
　感情移入できる気がしません。



■ライバルや敵役として
　存在することはあっても、

　主人公としては
　物語が成立しないのではないでしょうか。



■であるならば、人生においても

　【「捲り」の人生】

　を肯定し、

　将来の展開を期待したり、
　楽しんだり、

　くらいの心的態度こそが
　QOL（人生の質）爆上がりの秘訣ではないか、

　と思います。



■人生の前半戦においては
　（100年時代とすれば60歳前後まで）

　【自身を意識して
　「後方」に配置しておく】

　ように心がけるのも、
　人生において幸福感を感じるための

　「ちょっとしたTips」

　となるのではないでしょうか。



■あまり語れることはありませんが、

　ひょっとしたら、
　多くの人が意識していない、

　「QOLを高めるための、
　　隠された裏技」

　かもしれません。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■「長く沈み続けられる力」は
　　幸福感の源泉となり得る力。

　■先行逃げ切り的人生は、失速の不安も
　　あるし、ストレスが溜まるもの。

　■一方、捲りタイプの人生は、毎日が
　　思索と創意工夫の喜びに満たされるし、

　　未来を想像するところから生まれる
　　「脳汁ドクドク」を感じながら
　　生きているので充足感が半端ない。

　■人生の喜びは長く沈み続けてからの、
　　「後方からのぶっこ抜き」にあるのでは。
