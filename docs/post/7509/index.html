7509号 何度も問われ直すところに意味がある

■長らくメルマガを書いていて

　「いつも同じことを言っているな」

　と思うことがあります。


　昔からの読者さんであれば、

　同じように考えている方も、
　きっと少なくないでしょう。



■昨日書いた、

　「備えあれば憂いなし」

　「来たる第一領域対応のために、
　　資源の余裕を確保しておけ」

　「そのためには平時から
　　第二領域にリソースを割くべし」

　なんていう話は、

　もう何度、お伝えしてきたことか
　分かりません。



■とはいえ、

　「聞いたことがある」
　「知っている」

　のと

　「やっている」

　のと

　「できている」

　のとは、それぞれ全く異なります。



■同じ話を何度も繰り返されるのは、

　あなたは、この話について

　１、知っているのか
　２、やっているのか
　３、既に完了しているのか

　どれですか？

　と、都度、問い直されているのと
　同じです。



■たとえば、一つの例ですが、

　もしあなたが突如として
　勤務先の倒産に見舞われたとき、

　「そういうこともあろうかと」

　と、

　十分な余裕とともに、
　次の一手を打ち、

　さらに次、次、次と歩を進めて
　新しい世界を切り開いていけるのか。



■あるいは、

　今、こんな問題も抱えているし、
　あんな問題にも悩まされているし、

　そういえば、このような問題にも
　同時並行で対応しなければならないのに、

　よりにもよって、
　このタイミングで、、、

　と絶望に陥るのか。



■それは

　「不測の事態が生じる前に
　　その可能性を予測し、手を打っておく」

　必要があったことを、
　前もって

　１、知っていたのか
　２、やっていたのか
　３、既に完了していたのか

　によって変わってくるわけです。



■こうしたことを、

　たとえば一度、私（鮒谷）から
　問われたからと言って、

　それだけで一気に

　「知る → やる → 完了」

　まで進めることはありません。



■早くて数年、
　長ければ10年単位の時間をかけても

　「既に対応完了」

　とはならない可能性のほうが、
　ひょっとしたら高いかもしれませんね。



■でも、だからといって

　「問われることに意味がない」

　ということにはなりません。


　あらかじめ備えていれば、

　（たとえ完璧ではなかったとしても）
　備えておいたところまでは、

　安全圏に近いところまで
　身を移すことが可能になるからです。



■だからこそ、私（鮒谷）は、

　言い回しは変わったとしても
　何度でも本質的には同じ話をし続けるし、


　それはお読みくださる方に対しての
　価値提供であるとともに、

　すぐに怠惰に流れる自分自身に対する
　戒めでもあると考えています。



■そんなつもりで、
　ただ読み流すのではなく、

　「実行に移すキッカケとしての
　　メディア」

　として、

　当メルマガをご活用いただければ、
　いつか

　「読んでてよかった」
　「助かった」

　と思って頂ける日が訪れることも
　あろうかと考えています。



■今日も人生とビジネスを
　楽しみましょう！

【今日のピークパフォーマンス方程式】

　■同じことを何度問われても
　　当事者意識があれば

　　１、知っているのか
　　２、やっているのか
　　３、既に完了しているのか

　　について都度、
　　ゼロベースで自らに問い直し、

　　気持ちを新たに取り組めるようになる。

　■同じ話、聞いた話、知っている話、と
　　スルーしてしまうだけだと、

　　いつまで経っても行動に変化が現れず、
　　成果につながることもない。
