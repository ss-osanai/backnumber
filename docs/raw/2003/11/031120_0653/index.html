「ピークパフォーマンス」（人生における最高の結果）を発揮することが出来
るように、年間３００冊以上の本を読み、１００日以上、勉強会・セミナーに
参加し、学び、得た、内容を編集してお伝えしていくものです。
このメルマガを読み続けることで、こんなことが出来るようになります。

●セルフプロデュース術（自分を売り込む重要なポイントを発見できます）
●自信を持って、前向きに行動することが出来ます
●自尊心を持って生きていくことができます
●得たい結果を最速で得るためのヒントが掴めます
●新しい発想のヒントを得られます
●モチベーションを維持する方法を学べます
　　　　　　　　　　　　　　　　　　　　　　　　　　　　などなど・・・

■今日午前中は、最近、連日メルマガ上でとりあげさせて頂いている、
　日本起業家教育研究所の今村社長から来年、株式公開予定の会社の
　社長をご紹介して頂いておりました。


■どこまで書いていいのか分かりませんので、詳細は伏せさせて頂き
　ますが、株式公開前の最後の増資のための、プレゼンテーションを
　今村社長が受けられるとのことで、私もそこに出資を前提に同席
　させて頂いたのです。


■わたしもこういう話に別に詳しいわけではないのですが、お話を
　聞き終わりまして、ぜひ出資したいと思いました。

　私にとって決して小さな金額ではありませんが、今までの会社
　経営のストーリー、商品開発にかける熱い思い、社会に対する
　貢献、従業員に対する思いやり、将来にかける夢、など、思いの丈
　を語られる全てが熱く、こういったお話をきかせていただき、私の心
　の琴線に触れました。


■技術開発系の会社で、正直、技術力についての話や、今後の営業
　展開の話については全く分からないのですが、社長の今まで生きて
　来られた数十年間の一貫した思い、に共感いたしました。


■よくよく聞いてみると、今まで個人からの投資だけで２００名
　以上を募り、数億の資金を調達し、最後の資金調達のための
　数千万円を募られているとのこと。出資されている方々に
　ついてお話を聞いても、技術は分からないけれども、経営者の
　思いに賭ける、という状況のよう。
　お話を伺って、今まで出資されてきた方々の気持ちがわかるような
　気がしました。


■こういう形での出資はしたことがないのですが、宝くじを買うより
　もはるかに、夢があるし、健全であるように思います。
　たったこれだけの話で１００万単位のお金を投じる決定をしたという
　のは、甘すぎると思われるかもしれませんが、やはり、こういう社長
　に成功して頂きたい、とおこがましいかも知れませんが、思ってしまう
　のです。

　勿論、あくまで自己責任、の世界で、塩漬けどころか、雲散霧消
　してしまう可能性もなきにしもあらず、のところが、未公開株の
　恐ろしいところではありますが、今日は、久しぶりに経営者の熱い
　思い、が伝わってくるお話をしていただいたので、その夢に
　私もかけたい、と思っています。


■あんまり詳細かけないのが残念ですが、また折を見てご紹介できる
　機会があればよいなと思っています。
　「思い」のある経営者からの話は本当に刺激になります。
　実際にカネもモノもヒトも、「思い」に引き寄せられて、集まって
　いる感があります。
　なんとか成功して頂きたいと心の底から思います。　


■その後、昼食を取る時間がなかったので、駅の売店でチョコレート
　を購入して（脳に当分補給）、そのまま次のアポイントメント先へ。


■次のアポイント先は、ジャパンインターネットコム、さん。
　ご存知の方も多いと思います。百式の田口さんも連載を持たれて
　います。
　なんと月間３００万アクセス！！のサイトをお持ちです。
　ＩＴ関連のポータルサイトです。

　独自配信のメールニュースも「平成・進化論。」とは勿論、比較に
　ならないほど大部数を配信していらっしゃいます。

　《ジャパンインターネットコム様のWebサイト》
　http://japan.internet.com/

　《田口さんのコラム》
　http://japan.internet.com/busnews/

　
■こちらの取締役の方と、お話をさせて頂く機会を得て、長時間に
　わたり、種々、打ち合わせをさせていただきました。
　これまた詳細をお伝えすることは出来ませんが、ひょっとしたら
　何かいいことが起きるかも！？　起きんかも。

　こちらの取締役の方、非常に頭の良い方で聞けばやはり事業企画
　系のキャリアをずっと重ねられてきたようです。前職も大手のＩＴ
　関連企業で事業企画を立てられるポジションにいらっしゃった
　ようです。

　こういった優秀な方と今後、是非、コラボレーション出来れば
　よいなと思っています。


■その後、駆け足で次のアポイント先へ。
　次にお会いした方はマーキュリッチ有限会社の副社長、野村さん。

　《マーキュリッチ有限会社》
　http://www.mercurich.com/index.htm

　数多くの研修や、上場企業の社長さんなどをお招きしてのセミナー
　開催をしていらっしゃる会社さんで、法人化される以前から、ぜひ
　一度、社長もしくは関係者の方とお会いさせていただきたいなと
　思っていたところ、ひょんな機会で、こういうご縁を頂くことが
　できました。

　《メルマガ・お金の女神との出会い方》

　を発行なさっていらっしゃいます。

　＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝
　豊かなお金持ちを研究し、その秘訣を探るマーキュリッチのメルマガ。
　企業経営者・投資家等、本当のお金持ちから伝授された、お金や投資、
　ビジネス・人生に関する正しい知恵・マインドを伝えていきます！
　（登録はコチラから→） http://www.mag2.com/m/0000084218.htm
　＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝＝


■数多くの成功者といわれる人たちからの取材をされていて、その
　取材の内容をペンネームで、非常に参考になる形で配信されている
　メルマガです。（５０００部以上の配信数がおありです）

　私もいつもなるほどなるほど、と、大変勉強させていただいている
　のですが、ぜひ皆さまも読まれたらいいのではと思います。

　このメルマガは筆者が「金田一さん」ということになっており、
　以前からこの方のことが気になっていたのですが、どなたが書かれて
　いるのですか？と聞いてみたら、実は、野村さんご自身だったと
　聞いて、親近感が湧きました。
　

■ここでいくつかのビデオを販売されているのですが、大変面白い
　ものが多いです。

　（特にこれが面白かったです）
　http://www.mercurich.com/video/carisma.htm

　３２歳でモーニングスターを上場させた、七海　秀之さんの講演
　が収められています。　

　これを見て背筋がぞくぞく来ました。これから見られる方のために
　詳細を書くのは控えますが、これは本当に値千金のビデオです。
　私も、こういうビデオはよく購入して見るのですが、だまされたと
　思って、お金に余裕のある方はご覧になられたらよいかと思います。

　--機関投資家として-- 
　多くの日本人投資家はババを引かされていることに気付いていない。
　「投資の間違った考え方」とは？
　バブル崩壊後もなぜ『外国人投資家』は強かったのか？

　→たくさんの内容が入っていますが、出だしの１０分、上記の内容
　だけで、私にとって衝撃の事実がたくさん含まれていました。
　かなり「怖い」話を話されています。

　
■今後のための、情報交換を２時間ほどさせていただき、帰宅。
　その後は食事をしながら、たまっていた留守電対応、連絡しなければ
　ならない件について５名ほどの方と、電話にて２時間ほど。

　相変わらず、メールの返信が「全く」追いついておりません。
　最近、特に外出していることが多く、出先でＰＣは見ませんので、
　ますますご迷惑をおかけしております。

　特に、次回の平野先生のセミナーに申し込み下さった方々へは、
　お一人づつ、お礼のメールを出させていただきたいと思っている
　のですが、現状ままならない状況です。お許し下さい。

■冒頭にあげた公開直前の会社の社長さん。

　今まで、多くの経営者を見てきましたが、やはり「思い」のある
　経営者は強いです。多くの経営資源が「思い」に惹かれて、集まって
　来ている事を感じました。

　しかもその「思い」は何年、何十年と一貫性を持って暖め続けてきた
　もの。だから数多くの人から信頼され、投資、という形で支持されて
　来られているのだと思います。
　
　
■私もこのメルマガ上で、「夢」「志」という言葉を良く使いますが、
　非常に軽く扱っていたなと反省です。
　ただ、それぞれ、思いの軽重はあれど、こういう言葉を語れなく
　なった人生はつまらない人生だと思います。


■常に、一流といわれる人と接し続けることで、自分自身を変えていける
　と思っています。

　「思い」を持っているか、その「思い」は本物か、もっともっと、
　素晴らしい出会いを求めて、自分を磨き続けていきたいと思っています。
