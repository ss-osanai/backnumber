───────────────────


■米津玄師さんのインタビュー記事を
　読みました。

　https://natalie.mu/music/pp/yonezukenshi26

　全編通してよかったですが、
　なかでも以下に痺れました。


　（ここから）
　---------


　最後にもう1つ聞かせてください。

　2024年は自分にとって
　どんな年になっていますか？


　1年中ずっと続くかはわからないですけど、
　今は準備を楽しんでいる感じです。

　この世の中ってほとんどが
　準備だと思うんですよね。

　結果が生まれるにしても、
　その結果に紐付いた準備が水面下にある。

　その準備の連続によって
　世の中が満たされている。


　ものすごく当たり前のことなんですけど、

　それをいかに楽しめるかというのは
　今の自分に重要なことなんじゃないかなと。

　準備を楽しむ人生がここから
　獲得できたらいいなと思っています。


　---------
　（ここまで）



■これ紹介したら、
　もう何も言うことないんですが（汗）

　でも、本当にその通りだなと
　思ったのです。



■相撲の世界で

　「３年先の稽古をせよ」

　と言われます。


　また、むかし、ある著名な起業家と
　対談する機会があったのですが、

　そのときに

　「５年前にやっていたことが
　　今の私（の人生）を作っている」

　と話されていたことも
　印象に残っています。



■米津玄師さんが

　「結果が生まれるにしても、
　　その結果に紐付いた準備が水面下にある」

　「準備を楽しむ人生」

　と言われているのも
　まったく同じことですね。

　

■準備なくして、
　いきなり結果が生まれることはなく、

　たとえ、そうしたことがあっても
　その運は持続させられません。



■これは個人的な、
　一つの例に過ぎませんが、

　そして大した稼ぎもなければ
　蓄えもありませんが、


　それでもこれまで、

　生き抜くために長年、
　お金の問題と真正面から向き合い、

　今さらながら思うのは、

　「持ち慣れないお金を
　　いきなり手にしても扱いきれず、
　　ロクなことにならない」

　ということ。



■宝くじの高額当選者
　（当選金1000万円以上）になったら

　「その日から読む本」

　という冊子が
　手渡されるそうですね。


　＜宝くじが当たった「その日から読む本」
　　謎に包まれたその中身は？＞

　https://newsdig.tbs.co.jp/articles/-/875450?display=1



■安かったら入手して
　中身を読みたかったのですが、

　メルカリで調べたら39,000円とか
　45,000円で出品されていて、

　さすがに高すぎて断念しました。

　どなたか手に入れて
　内容を教えて下さい（笑）
　


■それはさておき、おそらく、

　「調子に乗ったら
　　あっという間に破滅するよ」
　
　というような話が
　書かれているんでしょうね。



■でも、こうした冊子を読んだところで、

　実際のところは、当事者として
　ある程度のお金と向き合わなければ

　「読んで読まず」
　「知って知らず」

　で、相当の確率で
　やらかしてしまうのだろうな、

　と思ってしまいます。

　

■私（鮒谷）は子どもの頃から
　父親に

　「金は絶対に貸すな」
　「貸した金は決して返ってこない」
　
　と言い聞かされてきましたが、

　商売がうまくいき、儲かり始めると
　調子に乗ってしまうんですよね。

　

■それでさまざまな依頼を
　安請け合いして、

　金を貸して返ってこなかったり、
　出資を求められて溶かされたり、
　
　しかも感謝されるどころか
　音沙汰なくなったり、

　なんなら悪く言われたり、
　といったことがしばしばありました。

　人間不信にもなりますよ。



■あるいは、

　友だちと思っていた人から薦められた
　よくわからない金融商品に手を出したり、
　
　勘と度胸で株式投資をしてみて、
　詐欺られたり、大損ぶっこいて、

　といったことも
　何度か（何度も）経験し、

　「お金との付き合い方」

　を体で学ぶこととなりました。
　

　
■ただし、長い目で見れば、これも
　一つの準備だったのかもしれません。
　

　今は殖やせるかどうかはともかく、
　安易に溶かしたりはしないだろう、

　という自信というのか、
　確信がありますが、

　それは、身銭を切り、痛みを伴う
　いろんな経験を経て、学んできたからです。



■そんなこんなを振り返ってみて
　改めて思うのは、

　いつ、いかなるときであれ、
　米津さんが言われている通り、

　「結果が生まれるにしても、
　　その結果に紐付いた準備が水面下にある」

　「準備を楽しむ人生」

　という心持ちの大切さ。
　


■つねに

　「今は準備期間」

　だと思えるから、

　未来に可能性を感じられるし、
　希望も持てるし、

　たとえ失敗しても、それを
　学びに変えられます。



■反対に

　「今は成果を楽しむ期間」

　という思いが全面に出ると、そこで
　足元をすくわれるように思われます。

　そんなことを、先のインタビュー記事を
　読んで考えた次第。
