───────────────────────────────────



■「成長しよう」と決意することは、非常に怖いことでも
　あります。

　しかしその恐怖心に打ち勝って、進化する覚悟ができた時点で、
　もう半分は目的達成したようなものなのです。

　決意し、第一歩を踏み出し、最初のカベを乗り越える
　ところが最初にして最大の難関だからです。



■現代は、非常に変化の速い時代。

　その時代の変化に追随して自分が変化していくか、
　あるいは現状維持にとどまるのか、

　その２つの選択肢がありますが、

　もっと正確な表現をすると「変化するか、滅びるか」という
　選択をしなければならない時代がきているといっていいでしょう。



■これは決してみなさんを驚かせるために誇張して
　言っているのではなく、
　多くの人たちがそのように語っているのを私はそのまま
　申し上げているだけです。

　わたしが非常に影響を受けたトム・ピーターズ氏も、
　数ある著書のいたるところでそんな発言をしています。

　決意し、覚悟を決めて、はじめの一歩を踏み出すことが大切です。



■昨日までの話で言いますと、イモムシからチョウになるぞ
　という決意です。

　「イモムシのままではいやだ！」という心の叫びに
　真摯に耳を傾け、

　「オレもいつか大空を優雅に飛び回る、
　あのアゲハチョウになってやるんだ！」

　あるいは

　「キプリスモルフォチョウ」

　⇒ http://tmshop.fc2web.com/tmuseum/saibi.htm

　になってやるんだ（笑）

　と、自分に宣言し、行動を開始するのです。



■「千里の道も一歩から」という慣用句がありますが、

　じつは、０から１へ進むことがいちばんつらく苦しく、
　それから先の「１から10」「10から100」というのは
　思いのほかカンタンなものです。

　行動にも「慣性の法則」が働くのか、
　一歩目さえ踏み出せば、二歩目、三歩目は非常にラクに
　足が出るようになります。



■自分の中の心の叫びに耳を傾けたなら、ほとんどの人が
　「イモムシのままではいやだ！」
　という声が聞こえてくるはず。

　もし、この「平成・進化論。」の読者さんの中で
　まだ変化の決意をされていない方がいらっしゃったら、

　今日を、変化の第１日目としていただきたいのです。

　昨日と違ったことを、今すぐ始めてみられては
　いかがでしょうか？
