◎本メールマガジン（平成進化論）は、

　複数の会社を経営し、IT、人材、環境、飲食、ホテル、海外事業など
　20社以上のベンチャー企業に出資する投資家としても活動する筆者が、

　　------------------------------------------------
　   骨太かつ卓越した＜本物の教育サービス＞を通して
　  「変容の喜び」を提供し続ける
　　------------------------------------------------

　ことを目的とし、3000日以上にわたってお届けしている、
　個人発行として日本最大級のビジネスメルマガです。

　熟読するも読み流すも、自由な形でご活用下さい。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜昨日の活動ダイジェスト＞

▼石橋を叩いて渡る

───────────────────────────────────



■こんにちは、鮒谷です。


　昨日のメルマガでは

　「最悪を想定し、最善を尽くす」

　ことについて記しました。



■そういえば昔、似たようなことを書いたことが
　あったなあ、と思って調べたら、

　3年ほど前のバックナンバーに


　＜2169号 大勝ちするより大事なこと＞

　http://www.2nd-stage.jp/backnumber/ppf_full/2009/08/2169.html


　という内容をお届けしておりました。


　昔から表現が変わっているだけで
　言ってることは同じですね。

　「石橋を叩いて歩く」

　とか書いているのはご愛嬌。（歩く、って、、）


　＜業務連絡＞

　Oくん、週明けでいいので、

　「石橋を叩いて渡る」

　訂正しておいてもらえますかー。
　お願いします。




■それでは今日のピークパフォーマンス方程式

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。


　今日は、


　【　意識の解像度を上げる　】


　というテーマでお伝えします。

　あなたのビジネスのヒントになれば幸いです。


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

 【　意識の解像度を上げる　】

───────────────────────────────────



■昨日は、


　日々の経験や気づきを、毎日、文章に残すことを
　習慣にすると、

　思っている以上に文章を書けない
　（＝学びが蓄積されていない）

　ことによって、

　「いかに一日一日を軽く生きているか」

　が知らされる。

　
　その悲しくも厳粛な事実を知らされることが、
　反省、向上につながっていく、

　そんなお話をいたしました。

　今日も続けます。



■当社で時折主催している３日間セミナーでは、

　「日々の思考をブログに記すこと」

　を課題としてお出ししています。

　
　多くの人にとって、慣れるまでは
　これは相当に苦しいことのようです。



■けれども、この課題を通して、

　「これまでいかに、毎日を無自覚に、
　　流されるままに過ごしていたか」

　を痛烈に思い知らされることになります。

　
　間違いなくその日一日を生きたにもかかわらず、
　振り返ってみて、　

　「ほとんど書くことがない！」

　ことに気付かされるのですから。


　
■それには２つの理由があり、

　▼その一日、人生を切り拓く新しい体験を
　　しなかった

　もしくは

　▼新しい「体験」はしたけれども、
　　総括できず「経験」に昇華させられない

　このいずれかのために何も書けない、
　ということになるのです。



■まずはその現実を直視させられるところが大事。

　自分の「感度」がどの程度であるかは
　（毎日休まず）書くことによって知らされます。


　このプロセスを通して、己の感度を

　「毎日、定点観測」

　することになるでしょう。



■よく

　「日記を書く人は夢が叶いやすい」

　と言われるのを聞いたことがある人は
　多いでしょう。


　その理由はたくさんありますが、
　一つには上述の通り、

　「日記を書いていると毎日、
　　自分の活動を振り返らざるを得ない」

　ところからくる反省にあるはずです。

　そして反省するところにしか向上は
　ないからです。



■この振り返りの感度（＝センサーが感知する能力）
　が高まると、
　
　自らの行動を
　
　「毎日の行動の振り返り」

　として大雑把に捉えるところから、
　やがて、

　「一つ一つの行動の振り返り」

　のレベルで意識が向いてくるように
　なるでしょう。



■言葉を換えると

　「意識の解像度が上がる」

　のです。

　「『一日一日の振り返り』という25万画素の省察
　　（＝ぼやけた省察）」

　から

　「『個別行動の振り返り』という1600万画素の省察
　　（＝くっきり鮮明な省察）」


　へと画素数が向上するイメージ、
　といってもよいかもしれませんね。

　（余計、わかりにくい！？）



■このようにして意識の解像度が上がると

　日々の気づき、学び、発見が増え、
　それに従ってメモを取る回数、調べる回数、
　
　といったものが増えていくこと
　でしょう。

　
　同時に、それまでいかに

　「ぼーーーーーーーーっと」

　と生きてきたのかに気付かされるはずです。


　
■私は日記をつけ始め、メルマガを書くことにより、
　その事実に気づいたとき、愕然としました。

　このショックが

　「あんなにも学びのなかった日々とは、
　　金輪際おさらばだ！！」

　と前に向いて歩くための
　大きな原動力となったのです。



　今日も人生とビジネスを楽しみましょう！
