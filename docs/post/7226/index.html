　メンタルブロックの外し方セミナーの
　開催、ありがとうございました。


　非常に興味のあるテーマでした。

　セミナー中、特に印象に残った言葉は、


　▼人はイメージした以上にはなれない

　▼願望の強さ＝成長の量


　同じ願望でも
　小手先で行っている人と、

　両手両足を使っている人とでは、
　雲泥の差です。


　願望が大きいほど、
　強い動機があるほど、

　そこまでの道のりを必死に考え、
　模索し、行動し、改善し続けるので、

　実現にまで辿り着けることを
　再認識しました。


　▼転んでもタダでは起きない人生！と
　　決めて立ち上がり、物語を編む

　▼死んでもネガティブな言葉を
　　使わない！と決める


　ネガティブな事があれば、

　自分の周りで起きている
　すべての事柄が関係しているかのように、

　接続して考えてしまうところがあります。


　習慣、環境、現実の引力は
　とても強いですが、

　各種音源で語られているように
　1日のうち何回も自らを躾けて、

　ありたい姿、状態に
　近づいていこうと思います。


　今後も繰り返し拝聴して、

　心の状態や自分に投げかけている
　言葉を把握し、

　コントロールできるところまで
　自らを持っていきたいです。

　どうもありがとうございました。


　（中野 誉久さま）

　（ここまで）


　中野さん、長文のご感想、
　ありがとうございました！



■お書きくださった内容を
　一言でまとめるなら、


　-----------

　１、未来における理想の自分像

　と

　２、現状に安住する気安さ、気楽さ

　との間の「綱引き」において、
　どちらが打ち勝つか

　-----------


　という話なんですよね。



■音源では、

　【向こう側の世界への杭打ちと、
　　こちら側から引っ張るゴム】

　【シーソーゲーム】

　といった例えで示したことも
　ありますが、

　今日は「綱引きの例え」を用いて
　説明しようと思います。



■より良い人生を作るために
　今、全力を尽くせている人は

　【「未来」の理想に重きを置いている】

　んですよね。


　一方で、将来のために頑張りたいと
　思いながら、

　（これでいいのかと煩悶しつつ）
　惰性の毎日を過ごしている人は

　【「現在」の気安さや気楽さに
　　重きを置いている】

　わけです。



■そして、ほとんどの場合、
　放っておいたら

　【未来の理想 ＜ 現在の気楽さ】

　に流されるんですよね。


　なぜなら、

　【理想の未来の実現に至るまでの
　　投下資源の総量を考えると、

　　現状維持で気楽に生きていく方に
　　軍配が上がる（そちらを選択する）】

　からです。



■未来ー現在、の間のギャップを
　埋めるためには、当然

　「相応の努力」

　が要求されるわけですが、


　その過程を想像して、

　面倒になったり、
　億劫になったり、

　腰が引けたり、
　不安や恐怖を感じて、

　前に進めなくなるのです。



■この関門を越えるためには、
　内面が

　【未来の理想 ＞ 現在の気楽さ】

　に完全に切り替わり、

　現状に負荷がかかってでも
　未来の理想を求めずにはいられない、

　という状態に入る必要があります。



■実はこの過渡期において

　【未来と現在の間で
　　綱引きが生じる】

　わけですね。


　図示すると、こんな感じ。

　【未来←←←←　→→→→現在】



■そして、ほとんどの場合、

　現在（今いる環境）から受ける
　影響の方が圧倒的に大きいので、

　「現在」が綱引きに勝利するのです。


　図示すると、こんな感じ。

　【未来←　→→→→→→→現在】



■こうして

　（このままでは良くないと思いつつも）

　代わり映えのしない、
　惰性の日々が続くこととなる。



■一方、大多数の人とは逆に

　【未来の理想 ＞ 現在の気楽さ】

　という形で、

　綱引きに圧勝し、
　未来のために投資し続けられている、

　という少数の人があります。


　図示すると、こんな感じ。

　【未来←←←←←←←　→現在】



■これらの人はどのように

　「未来ー現在」

　の綱引きに勝利したのかというと、
　単純に

　【「未来」陣営に重くて、力の強い
　　人間が入ったから】

　と言えるでしょう。



■とはいえ、
　ほとんどの人の初期設定は


　【「現在」陣営に重くて、
　　力の強い人間がたくさんいて、

　　「未来」陣営には細くて、
　　非力な人間が少しだけいる】


　ような状態になっています。

　こうして

　【現状維持がデフォルトの生活】

　となるのです。



■そんな状態にありながらも、

　あきらめの人生に陥ることなく、
　逓増する人生にしたいと願う人は

　【たとえ困難が伴っても、
　　地滑り的勝利を収めるところまで、

　　未来陣営に援軍を
　　送り続ける必要がある】

　のです。



■最初の力が均衡していても、

　ひとたび流れが一方向に傾くと
　そこから先は一気に勝敗が決する、

　それが「綱引き」という競技。

　人生もそれと同じです。



■最初はたとえ「現在陣営」に
　持っていかれそうになっていても、

　強力な援軍を「未来陣営」に
　陸続と送り続けることによって、


　ひとたび形勢が
　未来陣営サイドに傾けば、

　そこから先は、流れを味方につけて
　一気呵成の勝利を収められます。


　そして、そんな

　【ジャイアントキリング】

　は決して実現不可能な話では
　ないのです。

　（だから希望が持てます）



■勝敗を分かつポイントは

　【ここぞというタイミングを見計らい、
　　その瞬間、一気に「力の均衡」を
　　突き崩せるかどうか】

　です。



■春秋の筆法、ならぬ、
　我田引水の筆法をもってすれば、

　（とくに年間プログラムの参加者の
　　皆さまは、体感でお分かりの通り）


　年間プログラムにおいて
　参加後、わずか数ヶ月のうちに、

　一気に変化を遂げられる人が、
　多々現れられるのは、ここまでの

　【綱引きのたとえ】

　で理解いただくことができるはず。



■具体的にいうならば


　【圧倒的に強力な大勢の援軍が
　　寄ってたかって一気に加勢して、

　「未来ー現在間の戦力バランス」を
　　突き崩し、雪崩現象を起こさせるから
　　急激な変化を遂げられる】


　のです。



■ここでいう援軍とは、

　（他にもいろいろありますが、
　　分かりやすいものをいくつか挙げると）

　-----------

　１、月に二度の大ボリュームの
　　　「未来投資の話」

　２、大勢の未来投資をしている人との
　　　日常的な交流

　３、大勢の未来投資をしている人が
　　　発信しているメッセージ

　（これらに継続的に触れることで、
　　自分も雪崩を起こせるに違いないという
　　自己暗示、からの信念形成を行う）

　-----------

　といったものであり、これらには

　【天引きによる、
　　物理的な時間拘束】

　の側面がたしかにあります。



■しかし、

　こうした大量の増援部隊の派遣を
　短時間に一気呵成に行うことで

　【あっという間に内面が書き換わる
　（＝綱引きに勝利する）】

　ことができるようになるのです。



■そうなればしめたもので、

　あとは情熱、本気が続く限り、
　湧き上がる行動力を味方につけるだけ。


　最初は小さな結果、やがて
　大きな結果へとつないでいくことで、

　【流れが完全に
　　未来陣営のものとなる】

　わけです。



■この状態に入れば、

　致命的な失敗をしない限り
　半永久的に

　【未来を生きている人】

　になれるでしょう。



■つまり、この人は

　【頭の中は常時、未来一色】

　となっていますから、極論をいうと

　「現在のことなど
　　どうでもよくなる」

　のです。



■現在のことなどどうでもよくなる、

　とは、

　未来を想像し、さらには創造する
　喜びに満たされるから、

　今の苦しみについて不感症になり、

　（世間的に見ると）
　苦行としか思えないことさえ、

　投資の喜びに変わります。



■たとえば私（鮒谷）のメルマガなんかも
　そうかもしれません。


　毎日１通、ないし２通、
　多い日には３通ないし５通も、

　それなりの分量のメルマガを
　20年に渡って一日も休まず、
　書き続けてきました。



■これなども、多くの人からは
　苦行（努力）にしか見えないでしょう。

　実際に、時間的、精神的な負荷が
　相応にかかりますし。



■しかし、

　【脳内綱引き】

　において、早期に

　【未来陣営のワンサイドゲーム】

　を実現させることができた
　私（鮒谷）にとっては、


　多くの人が、努力とか忍耐としか
　思えないはずのメルマガ執筆が、

　【現在の資源を
　　未来に向けて投資している】

　という感覚にしかなりませんから、

　一通、一通、メルマガを書く行為も
　純粋なる喜びがあるだけです。

　そして「止める」という選択肢も
　存在しなくなるのです。



■これはほんの一つの例ですが、
　これと同じ感覚を掴んで頂く為に、

　関わってくださるお客さまに対して、

　年間プログラムを実施し、
　個別コンサルを行い、
　各種音源をリリースし、

　さらには、

　外部の講師もお招きすることで、


　無敗を誇る「現在陣営」に対して、

　ジャイアントキリングを起こして
　いただきたいと願っているのです。



■巨人のV9時代を彷彿とさせる
　常勝軍団「現在陣営」は、

　いついかなる戦いにおいても

　【未来←　→→→→→→→現在】

　という形で勝利を収めてきましたが、


　我々は、
　誰もが想像もできなかったような

　【未来←←←←←←←　→現在】

　の大番狂わせを実現させ、
　自他を熱狂させたいところです。



■そのためにも、

　【強力、かつ多様な援軍を
　　未来に送り続けたい】

　ものであり、


　強力な助っ人としての

　【催眠（自己暗示）】

　は召喚する価値が大いにあり、
　と考えており、

　以下のコンテンツをご案内してきた
　次第なのです。



■、、、そんなわけで、
　ここまでお読みくださった方の中で、


　【私も未来陣営に向けて
　　最強の援軍を増派したい】

　と思われたあなたは、
　半信半疑でも結構ですので、


　ものは試し、
　虎穴に入らずんば虎子を得ず、

　ということで

　ぜひ動画＆音声コンテンツによって

　【増援部隊の拡充】

　に取り組んで下さい。


　まず最初に驚かれ、
　次に速やかに援軍を送り込みたい、

　となるに違いありません。
