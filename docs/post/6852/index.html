6852号 人間到る処 破滅リスクあり

■先月、リリースした

　「クマムシ音源」

　において、徹底して

　「破滅リスクに敏感になるべし」

　というお話をしました。



■「人間到る処青山あり」

　という言葉がありますが、
　この言葉にちなんで（？）

　「人間到る処 破滅リスクあり」

　といってもよいのではないかと
　思われるくらい、

　思わぬところに
　危険が潜んでいるものです。



■特に現代社会は、

　昔（それこそ昭和後期など）と
　比較してはるかに、

　あらゆる面において不確実性が
　高まっているのではないか、

　そんな気がしております。



■気のせいであればよいのですが、
　おそらくそうではなく、

　テクノロジーの発達とともに
　グローバリゼーションの影響で

　「地球が小さくなった」
　「混乱をもたらす変数が増大した」
　「混乱が大衆に波及する期間が短くなった」

　ことは原因の一端に
　挙げられるように思われます。



■コロナもそうなら、

　ウクライナ危機も私たちの生活に
　直撃しつつあり、

　ここから派生するインフレリスクや
　昨今の株価下落、
　さらには国際政治の不安定化は

　「私たちの生活圏外」

　からの不意打ちの可能性を
　大いに高めている状況です。



■我々は、

　知る知らないによらず、
　好む好まざるに関わらず、

　これらの影響をダイレクトに受ける、
　そんな時代を生きているわけです。



■とするならば、

　私たちが将来に向けて備えるためには
　遠い異国の出来事について

　「知らない」
　「興味もない」

　などと言っているわけにはいかず、
　無関心であってはならない、

　のではないでしょうか。

　自分や家族の人生や将来に
　関心があるのであるならば、、、



■とはいえ、

　もちろん各人の自由ですから
　別に無関心であってもいいのですが、

　そのことによってもたらされる
　自らの身の上に起きた出来事の責任は、

　いずれにしても
　自身が引き受けなければなりません。



■そんなわけですから、
　私（鮒谷）も昔から

　「遠い世界において今、
　　何が起こっているのか」

　を知ろうと努めてきたつもりです。



■来週、開催する、

　日経新聞政治部長（政治・外交グループ長）の
　吉野直也さん

　をお招きしての政治・外交勉強会も、

　ここ数年、
　年に数回ペースで開催してきましたが


　開催の意図するところはまさに

　「遠い世界において今、
　　何が起こっているのか」

　について学びたいからでした。



■また学ばせていただくからには、

　同じくこうしたテーマに関心を抱く人と
　ご一緒できれば、

　という思いでご案内してきました。


　今回も、これまで同様、
　ご関心をお持ちの方に参加頂ければ、

　と思っております。



■諸般の事情により、
　直前のご案内となり恐縮ですが、

　本日より、
　受付けを開始いたしますので、

　ご希望される方は
　以下よりお申し込み下さいませ。
