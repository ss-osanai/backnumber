＜今日のピークパフォーマンス方程式＞

  【　混乱したときこそ「石の目」を意識する　】

───────────────────────────────────



■これまでに何度か書いたことがありますが、
　　
　私の公私にわたる先生、師匠である、
　弁護士の高井伸夫先生から

　「石にも目がある」

　という言葉を教えていただいたことがあります。



■何十年も前、駆け出しの勤務弁護士として
　事務所に所属し、

　多くの仕事を任され、
　自由闊達に活動されていたとき、
　
　その事務所の所長さんから教わった
　言葉だそうです。



■石工はノミ一丁で、たいした力もかけずに
　大きな石を割っていきます。

　それは

　「石の目」

　めがけてノミを振るうからです。


　所長さんはこのような例えをもって、

　固い石を割るのと同じように、
　どれほど難題に見える問題であっても、

　まず核心を見極めることによって解きほぐし、
　最終的には解決できる、

　そんなことを伝えられたかったのでしょう。



■日々、私たちが行なっている仕事も

　「核心」

　と
　
　「その周縁部にあるさまざまな要素」

　が錯綜し、渾然一体となって構成されているように
　思われます。

　（したがって一見するだけでは、
　　どこに「石の目」があるのか、全くわからない）



■大きな案件になればなるほど、

　無数の条件が複雑に絡まり合い、
　解を見出すことが困難になり、

　途方に暮れることもしばしばです。
　
　ちょうど、難解に見える数学の問題を解く時のように。



■けれども一見、難しそうな数学の問題も、

　いったん突破口さえ開かれれば、
　あとは容易に解き進められるもの。


　方程式を力技で解こうとする。

　あるいは難しい幾何の問題を補助線を引くこと
　なしに解こうとする。


　それではいつまでたっても解の糸口すら掴めません。

　まずは問題の

　「突破口」「核心」「（適切な）補助線」「石の目」

　を探すことが大切です。



■重たい問題が眼前に現れたとき、

　手を付けられるところから始めるとかえって
　問題解決を困難にすることがあります。

　
　どこから手をつけて良いのかわからない問題に
　直面したとき、

　つい、打ちやすいところに安易にノミを
　振り下ろしたり、
　
　やみくもに手当たり次第、
　振り回しそうになったりするもの。



■それではおそらく、大きな石は割れず、
　小さな破片という新たな厄介事を増やすだけ。

　労多くして、功少なし、ということに
　なるでしょう。


　私が混乱、混沌の中に身を置くこととなったとき、
　いつも頭の中に浮かんでくるのは、

　高井先生から教わった

　「石にも目がある」

　という言葉なのです。



　今日も人生とビジネスを楽しみましょう！
