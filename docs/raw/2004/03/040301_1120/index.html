＜今日のピークパフォーマンス方程式＞

　　　　【求むべきは、金の卵か、金の卵を産むニワトリか。】

───────────────────────────────────


■「金の卵」と「金の卵を産むニワトリ」。

　みなさんはどちらが欲しいでしょう。

　私は圧倒的に「金の卵を産むニワトリ」が欲しいです。


■みんなお金を求めています。どんな人であってでも。
　お金がなければ生きていくことはできませんから、それはすごくよく
　分かります。

　そんな中でお金持ちとそうでない人の違いをよく見てみると、はっきり
　していることが一つあります。


■それは、日夜得ようとしているものが「金の卵＝おかね」なのか、
　「金の卵を産むニワトリ＝おかねを作り出す能力」か、という違い。

　
■お金をなかなか稼げない人ほど、焦点は「金の卵」に合っているようです。
　「金の卵を産むニワトリ」をどのように飼い、育てるかという発想が
　ないようです。


■反対に、お金儲けをしている人は、お金を産む資産を得るための投資に
　回します。

　それは自分のビジネスかもしれないし、不動産かもしれないし、株式投資
　かもしれないし、種類はいろいろですが、おかねを生み出す「ニワトリ」を
　育てる事に一生懸命です。

　
■そして彼らの行う一番の投資は自分の能力を最大限高めようとする
　時間的、金銭的投資。


■最初は小さい差でも、これが積もるとだんだん大きな差となって現れて
　きます。


■実は、私、このメルマガを使って広告収入でお金を稼いでいます。
　そして、これがわたしの生業（なりわい）の一つの柱となっています。

　いろいろ紆余曲折、失敗は多数ありましたが、なんとか広告掲載を受け付ける
　ようになってからわずか２ヶ月で広告ビジネスの月商は３ケタ台に乗りました。
　まぁ、かかってるのは私の人件費だけですから、非常に高利益率のビジネス
　ですね。

　⇒勿論、読者さまにも、有益な情報を配信するというミッションにそって
　　記事の一部としての位置づけで配信しています。ですからコメントもつけて
　　います。また、掲載する原稿は選んでいます。お断りするケースも多々
　　あります。

　⇒広告をお出し下されたクライアントさまにもほとんどのケースで、
　　ご満足頂き、リピート、ご紹介を頂けるよう、全力をそそいでいるからです。
　　決して安易におこなっているわけではないことをご理解くださいね。

　（詳しいことは、今月インプレスさんから発刊されている『インターネット
　マガジン』誌でお話ししていますのでご関心ある方はお読みください）


■こんな生々しい話をするのは本意ではないのですが、このメルマガを
　読まれている方はビジネスパーソン、トップ10％～20％を目指そうとされて
　いる意識の高い読者さまばかりだと思いますので、あえてご参考にして頂きたい
　と思い、お話しいたします。


■メルマガ広告で仮に月100万円を稼ぐとすると、メルマガ上での年間の収益は
　1200万円ということになります。
　
　
■実は、いろんなお金持ちの人を見ていて気付いたことがあります。
　それは収入をもたらしてくれる資産を持つことの重要性。
　
　私の場合は、お金を稼ぐための考え方、ノウハウ、スキルを身につけ、人的
　ネットワークを構築するために金銭・時間面を含めて過去何年ものあいだ、
　莫大な投資を行ってきました。

　みんなが遊びに行く、飲みに行く、その他レクリエーションに興じている
　ときに、ひたすら資産を生み出すための能力を磨く事に注力してきました。


■そういうわけで私の一番の資産は、自分がお金をうみだすための思考方法
　であり、そのための具体的な方法であり、さらに具体的にいうとメルマガ
　も一つの資産として含まれることとなります。

　これらはいずれも、単発の「金の卵」ではなく「金の卵を産むニワトリ」
　です。
　

■仕事をすれば、その報酬として「金の卵＝おかね」を得ることは出来ます
　が、問題はその「金の卵」を産み出す「ニワトリ」を飼って、育てること
　ができるかどうか、だと思います。


■仮にメルマガで収入が1200万円とれるとすると、それを仮に年利20％の利回り
　で割り戻すと6000万円の資産があることとなります。
　30％で割り戻しても4000万円の資産ということになります。
　（一応、くりくり動いているので、銀行の金利0.0○％で割り戻すのは
　適切ではないと思います）

　コミッションビジネスのように運良く一発、稼いで終わり、の種類のもの
　ではないからで、しかるべきクオリティのものを発行し、広告営業を行い
　続ける限りにおいては、それだけの収益を継続的に生み続けることと
　なります。


■そう考えると、メルマガ開始５ヶ月で、数字上は4000万なり6000万円なり、
　相当の資産を築くことができたということになりますね。
　（※もちろん、僕がけがをしたり、病気をしたりすると、そのままストップ
　してしまうビジネスですから、公式をダイレクト当てはめるわけにはいか
　ないですけれども）


■こういう発想は、私はある先輩経営者から教えてもらいました。
　ロバートキヨサキ流にいうと、「その資産はお金をポケットに入れて
　くれる資産か、お金がポケットから出ていく資産なのか」の違いですね。


■私にとって「平成・進化論。」は無形の、大きな資産だと思っています。

　でも、実はもっと大きいのは、そのメルマガから収益をあげることが
　可能となる考え方であったり、メルマガを書くための情報であったり、
　いろいろメルマガ発行していく上でアドバイスをしてくれる先輩、友人、
　後輩であったり、だと認識しています。
　
　またそういうものを蓄積しようとおもって今までやってきましたし、
　これからも行っていこうと思っています。


■これからの時代、いくつ「金の卵」を持っているかを競うよりも、
　「金の卵を産むニワトリ」をいかに大きく育てるか、がますます
　重要になってくる予感がします。

　卵をなくしても、ニワトリさえいれば、時間の経過とともに、新しい
　金の卵を産み落としてくれるのですから。


■やはり、常に勉強し続けることが大事だとつくづく思います。
　まさに自分自身が最大の資産であることを知れ！ですね。
