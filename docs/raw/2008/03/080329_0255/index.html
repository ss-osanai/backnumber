━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式。＞

　【　自分と相手を第三者の視点から俯瞰してみる　】

───────────────────────────────────



■思いがけず大連載となっていますが、

　コミュニケーションについてのお話をもう少しだけ
　続けたいとおもいます。



■コミュニケーションの基本は、

　相手を慮(おもんぱか) るとか、
　場の雰囲気を読むといったことが挙げられますが、


　私が特に心がけているのは


　【 自分をＡ、相手をＢとすると、

     ＡとＢを俯瞰(ふかん) するＣという視点を想定し、

     Ｃの視点に立ってコミュニケーションをとる 】


　ことです。



■図示すると、こんなかんじですね。



　　　　　　　　　 ┌─────┐　　　　　　　　　　　　　
　　　　　　　　　 │　第三者　│        　　　　　　　　　
　　　　　　　　　 └─────┘（Ｃ）　　　　　　　　　　
　　　　　　　　　　　／　　＼  　　　　　　　　　　　　　 
　　　　　　　　　　／　　　　＼　　　　　　　　　　　　　　
　　　　　　　　　／　　　　　　＼　　　　　　　　　　　　　
　　　　　　　　／　　　　　　　　＼　　　　　　　　　　　　
　　　　　　｜／　　　　　　　　　　＼｜　　　　　　　　　　
　　　　　　　￣　　　　　　　　　　￣　　　　　　　　　　　
　　┌─────┐　　　　　　　　┌─────┐　　　　　　
　　│　　私　　│　　　　　　　　│　相　手　│　　　　　　
　　└─────┘　　　　　　　　└─────┘　　　　　　
　　　　　　　　（Ａ）　　　　　　　　　　　　（Ｂ）　　　　



■いつもＡの視点ばかりで、

　つまり、自分優先のコミュニケーションを
　とってばかりいると、

　どうしても、我を通そうとする人になってしまいます。



■また、その逆に、
　Ｂ（相手）の立場にたつことのみに専念していると、

　こんどは、相手の都合で「ダンスを踊らされる」結果に
　終わりがちです。


　これは、お互いにとても不幸なこと。



■そこで、第三者の視点「Ｃ」を導入するのです。

　明日、もうちょっと詳しく説明しますが、

　このことにちょっと気をつけてみるだけで、
　コミュニケーションはだいぶ違ってきますよ。


　今日もビジネスを楽しみましょう。
