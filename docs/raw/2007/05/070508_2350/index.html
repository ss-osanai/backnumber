■長く続けてきたシリーズですが、今日で最後にいたします。



■先日も書きましたが、

　１日に10人喜ばせることができれば、1年で3650人を
　喜ばせることになります。

　それが10分の１でも返ってきたら、人生は、ビジネスは
　どれほどすばらしいものになるでしょうか。



■そうやってギリギリいっぱい人さまのために努めていると、
　どうしても「オペレーションの貧弱さ」に突き当たります。

　もっと先方の力になりたいのに、こちらの手際が悪いこともあり
　処理能力の限界をこえてしまう。

　その葛藤があります。



■そんなときとても参考になるのが、冒頭にもご紹介した
　「ザ・リッツ・カールトン・ホテル」のオペレーション。

　
　こういった書籍、


　【今日のお勧め本 リッツ・カールトン 20の秘密（ミスティーク）
　　　　　　 　　　～一枚のカード（クレド）に込められた成功法則 】

　 ⇒ http://amazon.co.jp/o/ASIN/4903721027/2ndstagejp-22/ref=nosim

　
　や雑誌、あるいは実際に泊まった人からの体験談などを
　見聞きされ、

　リッツ・カールトンに泊まろうと思われた方は非常に
　高い期待値を持ってやってこられることでしょう。



■そしてリッツ・カールトンのスタッフの皆さんは、
　その期待に見事、応えていかれるわけです。


　これは、


　　▼「お客さんの期待に応えたい、期待を超えたい」という思い

　　そして、

　　▼実際に期待に応え、期待を超越するオペレーション


　を兼ね備えているからこそ実現できる芸当ではないでしょうか。



■ここで学べるのは、

　顧客の期待に応えたいという「志」の部分と、
　それを実現するためのオペレーションの技術、

　その両方を高めることで、周囲に対する期待に可能な限り
　応えていかなければならない、

　ということ。

　

■志だけでは機能せず、オペレーションの技術だけでは魂は
　入らない。

　両方を同時並行で進めていけるところが、
　リッツカールトンホテルのリッツカールトンホテルたる
　所以なのでしょう。



■わたしであれば、

　そういった理念とオペレーションの双方を回しつつ、


　（直接的には）１日10人の人にお喜びいただく。
　（間接的には）１日23万人超の方の役に立つ。


　これをひとつの目標として自分に課してみようとおもっています。



■すると、物理的な時間の不足、能力の不足、が見えてくるように
　なります。

　もしそのギャップが生じないとするならば、
　そもそも目標を掲げていないか、目標の設定が甘すぎる、

　つまり自分に全然負荷がかかっていない状態だということになります。



■そうなったときに、

　もっとスタッフを充実させなければとか、
　自らに教育を施さなければ、
　あるいはさらに厳密な時間管理を、

　などとといった課題がみえてくる。


　
■そして採用活動を始める、外部のパートナーの力を求める、
　自分の知識、スキル、ノウハウ、ネットワークの充実を図ると
　いう実際の行動につながってきます。

　それが結果として、組織・個人の成長につながるのでは
　ないかと考えています。
