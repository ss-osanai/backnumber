　と改めて実感した次第。

＜今日のピークパフォーマンス方程式＞

 【　人生を総括する意味  】

───────────────────────────────────



■昨日も少し書きましたが、

　2ヶ月ほど前、私が東京に出てきてからの10年間を
　総括したメルマガを配信いたしました。


　＜鮒谷の上京10年物語＞

　http://www.2nd-stage.jp/backnumber/ppf_full/2013/02/3441.html

　

■自分の人生を総括してみて思ったのは

　「（少なくとも人生の節目において）
　　一人ひとり、過去を振り返って総括すると良い」

　ということでした。


　ただし、総括の仕方にもコツ？ポイント？があり、
　
　「すべての出来事をプラスの方向に解釈する」

　ことが大事です。



■たとえば私の場合、


　「会社が倒産した → 独立のチャンス到来！」

　「騙された → 若いうちに経験できてよかった！」
　　　　　　　　（老後に騙されたら大変だった）
　
　「人間関係が大変だった → コミュニケーション能力が上がった！」

　「投資失敗 → 戦場は得意分野に絞るべきと学べた！」

　「何をなすべきか分からず悶々としていた 
　　　　　　　　　　　　　→ 圧倒的なエネルギーを蓄えられた！」


　などなど、

　一つ一つの出来事にプラスの意味をつけたり、
　教訓を導く、

　という工夫を行うことにより、

　「一切の過去が全肯定」

　されることになりました。



■一切の過去を全肯定することができれば、

　自分の存在（過去、現在、さらに未来）について
　自信を持つことができるようになります。

　そして自分の人生を肯定できるようになるか否かは
　まさに過去の軌跡の解釈次第なのです。



■先日、講談社の編集者さんと食事をご一緒する
　機会があり、

　その方が編集を担当された、乙武さんの新刊


　【今日のお勧め本　自分を愛する力】

　 http://amazon.co.jp/o/ASIN/4062881985/2ndstagejp-22/ref=nosim

　
　を読んだのですが、

　（発売一ヶ月ですでに10万部売れてるそうです）

　
　この本に書かれていることなどまさに

　「出来事は一つ、解釈は無限

　　（だから自分にとって一番、
　　　力が与えられる解釈をしよう）」

　のお手本のようなものでありました。



■同じ出来事でも

　
　「会社が倒産した → 失業だ、これからどうやって生活していこう」

　「騙された → 死ぬまで恨み、呪い続けてやる」
　　　　　　　　
　「人間関係が大変だった → 私を苦しめた人間に天誅を！」

　「投資失敗 → 損害をもたらした関係者全員、
　　　　　　　　恨み晴らさでおくべきか」


　と、反芻し続け、苦しみ続け、苦悩のデフレスパイラルに
　陥ることもできるわけですが、

　そうするのではなく、

　（一般的に考えたら）マイナスにしか思えないことを
　プラスに解釈する、

　という言葉遊び（ゲーム）をしてみよう、
　と考えてみたらどうでしょう。
　　


■このゲームは、


　死ぬほど課金される（こともある）
　ソーシャルゲームと違って、

　一銭のお金もかかりません。
　

　一歩間違うと廃人になる（こともある）
　ソーシャルゲームと違って、

　うまく言葉を操ることができるようになるほど
　豊かな人生が送られる。


　そんな楽しく、大きな実益を伴う遊びなのです。

　（D社さん、G社さんをはじめとする関係者の皆さま、
　　すみませんね）



■こんなふうにゲームを行なっているだけで、
　段々と、自動的に、

　「自分の人生が自分で全肯定されていく」

　感覚を味わえるようになるでしょう。


　そして実は、

　（正しいやり方で行えば）

　日記をつける、ブログを書く、メルマガを書く、
　という行為それ自体が、

　人生の節目どころか、

　「毎日毎日、自分の人生にプラスになるような総括を行う」

　ことにつながっていくのです。



■マイナスと思える出来事があっても
　リセットし、ゼロにする、

　ということを超えて、

　プラスに転換させてしまう、
　という力が

　「（正しい）振り返り」

　にはあります。　



■このような効果効能のある、

　「振り返り」

　を上手に使える人が間違いなく持っている力を

　「総活力」

　と私は呼んでいるのですが、
　日々、この力を鍛えようと努めてやってきました。

　このメルマガにおいては3500回以上、
　その日の総括を続けてきたことになります。
　


■そしてこの力が強化されればされるほど、

　「自信とともに心の平穏が確保される」

　ことも間違いないと感じられるようになりました。


　ぜひ　節目節目、という不定期の形であれ、
　毎日、週に一回、月に一回といった定期的にであれ、

　（できれば頻繁に、定期的に行い、　
　　思考習慣にされるのをお勧めしますが）


　「総括の習慣を身につけることによって、
　　総活力を鍛える」


　ゲームに挑戦されてみてはいかがでしょう。
