■三泊四日の札幌滞在

───────────────────


■おはようございます、鮒谷です。


　メルマガには書きませんでしたが、
　先週は四日間、

　避暑を兼ねて札幌にいたのですが
　想像と違って普通に暑かったです。



■ただ、東京に戻ってきたら、

　暑いは暑いでも、こちらは
　異次元の暑さだったので、

　（相対的に）過ごしやすかった、
　とは言えるのかもしれません。

　実際の体感としては
　めちゃくちゃ暑かったですが。



■札幌は過去に何度も訪れていますが、
　
　今回、初めて行って
　ダントツで良かったのは、

　北大構内の中にある

　「北海道大学総合博物館」

　でした。

　

■北大の知の見取り図、
　みたいな感じで、

　文理を横断する、それぞれ興味を引く
　充実した展示はもとより、


　北大ならでは、ともいえる

　「北方生物圏フィールド科学センター」
　「スラブ・ユーラシア研究センター」

　「地球環境科学研究院」
　「サステイナビリティ推進機構
　　SDGs事業推進部門」

　といった組織の
　教育、研究発表の場としての
　
　「北大の探求心2024」
　
　という企画展も
　かなり面白かったです。



■時間の関係で２時間ほどしか
　いられなかったのですが、

　ゆっくり見れば丸一日、
　楽しめるくらいの内容でした。

　しかも完全無料！
　これから札幌に行かれる方はぜひ。



■他には札幌市街、ならびに
　豊平川を沿ってのサイクリングも。
　　
　これは普通に暑かったです。
　もう少し涼しい時期であれば最高ですね。



■普段は節制しているので、東京で
　ほとんど食べる機会はありませんが、

　せっかくなので
　
　「寿司、ジンギスカン、ラーメン、
　　ケーキ類」

　をフルに食する許しを自分に与え、
　
　（暴飲はしないけれども）
　暴食を重ねてしまいました。



■ただ、東京に戻ってきてから
　体重計に乗ったところ、

　体重も体脂肪も
　まったく変化なしだったので、

　サイクリング＆かなり歩いたのが
　奏功したのかもしれません。



■なにはともあれ、

　場所や取り巻く環境を変化させて
　刺激を与えることは大事ですね、

　悪性のマンネリ化を防ぐために。

───────────────────


■上述の通り、先週は４日間、
　札幌に滞在していました。


　そして今週月曜日（今日）からは、
　小泉淳作展

　＜小泉淳作画伯生誕100年
　　記念特別展開催のお知らせ＞
　https://www.kenninji.jp/news/?p=1896

　を見にいくため、
　京都を数日、訪れる予定でした。



■が、例の南海トラフ地震の件があり、

　キャンセル料はかかりましたが
　宿泊をキャンセルし、

　京都行きを延期することにして、

　何ごともなさそうなら来月頭くらいに
　改めて訪れようと思っています。



■いろいろネットを見ていると、

　こうした行動について
　肯定的な人もあれば、

　否定的に捉える（バカにする）人も
　あるようです。

　こんな奴がいるから経済が止まるんだとか、
　コロナのときの過剰反応と一緒とか。



■考え方は人それぞれあっていいと
　思っているし、

　そんな中、たまたま私（鮒谷）は
　こうした行動規範を持つ人間、

　というだけの話なので、

　私自身は他人の振る舞いについて
　とやかくいうつもりはありませんし、

　強要するつもりもありません。
　どうぞご自由に。



■と同時に、

　京都（関西、西日本）が危ないなどと
　危機を煽るつもりもありません。


　そもそも私（鮒谷）自身、
　今まさに
　
　「首都直下型地震のお膝元（？）」

　に住んでいるわけですしね。



■居住エリアによって発生確率に
　多少の差はあるかもしれませんが、

　日本は「地震の巣」ですから
　基本的にはどこにいても一緒です。


　
■ただ、

　「備えを可能な限りまで分厚くしてきた
　　自宅や事務所」

　と

　「スーツケースのキャパの範囲内で
　　備えを固めて出撃した旅先」

　とでは、

　どうしたって打たれ強さ（打たれ弱さ）に
　当然、差が出るわけですよ。

　あるいは、新幹線に乗っているときに
　なにか起きても逃れようがありません。



■ならば、

　多少なりとも激甚災害の発生確率が
　高まっているのは間違いなさそうで、

　かつ、今回の目的は不要不急、さらに
　特別展の開催期間はまだ先まであって、

　自分の意志と都合で
　ずらすこともできるのだから、

　今回あえて、
　無理をする必要もないだろうと。



■心配し過ぎとか、
　そういうことじゃなく、

　私（鮒谷）の行動原理はつねに

　「後悔したくない」

　というところにあるので、

　万一の事態に巻き込まれたとき、
　後悔する可能性があり、

　その可能性が平時よりいくらかは
　高まっていると考えてのこと。



■もちろん、真逆の展開、つまり、

　「私（鮒谷）が留まった場所で
　　想定外の悲劇的な事態が引き起こる」

　ことも十分に考えられるわけで、

　なにをどうしたって
　危険をゼロにはできませんが、


　それでも、今件に限らず、

　日々の小さな意思決定を
　十も二十も三十も積み重ねるうちに、

　（比喩的な意味でも
　　字義通りの意味においても）

　「生存確率は有意に高まる」
　
　とは思っているんですよね。



■それはタレブの、

　テールリスク
　（ならびにファットテール）についての、

　考え方と振る舞いに
　大いに触発されてきたからです。


　
■以下に

　『身銭を切れ
　 「リスクを生きる」人だけが知っている
　　人生の本質』
　　
　http://amazon.co.jp/o/ASIN/447810381X/2ndstagejp-22/ref=nosim

　で印象に残った、いくつかの節を
　紹介するので参考にしてください。

　
　（ここから）
　--------------------

　今まで生き延びてきたものは、
　ブラック・スワン的な出来事に対する
　頑健性を示している。

　しかし身銭を切るという行為がなければ
　そのようなふるい分けのメカニズムは
　働かない。


　身銭を切らなければ

　「時の知性」

（１、時は脆いものを排除し、
　　　頑健なものを生き長らえさせる

　２、脆くないものの余命は時がたつごとに
　　　長くなっていく、という性質を持つ）

　は働かない。

　--------------------

　合理的なものとは、
　集団（長く生き延びなければならない実体）の
　生存を可能にするものである。

　--------------------

　テールリスクを過大評価するのは
　どういう基準から見ても非合理的ではなく、
　生存にとって総合的に必要なことだ。

　--------------------

　長く生き延びてきたモノは、
　一定の頑健さを備えていることを
　事後的な形で示唆している。

　ただし、その間、
　害にさらされてきたという条件がつく。

　--------------------

　「リンディ的」なるものとは、
　逆向きに歳を取る者を指す。

　つまり、生存を前提条件として、
　時がたつごとに余命が長くなるものを指す。

　--------------------

　あるモノが生き残るには、
　必ずリスク面で首尾よくやる必要がある。

　つまり、死なないことに
　長けていなければならない。

　（中略）　

　その思想がリスク管理に長けていて、
　その思想の持ち主に害を及ばさないどころか、
　生存にとって有利になるなら、
　その思想は生き残る。

　--------------------

　実際、リンディ効果により、
　時に対する頑健さ　

　（つまりリスクを冒すような状況下で
　　行われる物事の頑健さ）

　は、生存の有無によって
　確かめられる。

　--------------------

　特定のテールリスクをものすごく非現実的で
　大げさな形で解釈しない限り、
　その人は生き残れない。

　たった一回の出来事で社会保障制度から
　脱落し、二度と戻れなくなる。


　このような限定的な強迫観念は、

　それを持たない人々や集団が最終的に
　死んだり絶滅したりするのであれば、

　非合理的と言えるだろうか？

　--------------------

　合理性とは、言葉ではっきりと説明できる
　要因によって決まるのではない。
　
　生存に役立つもの、破滅を防ぐものだけが、
　合理的なのだ。

　--------------------

　起こることの全てが、
　理由があって起こるわけではない。

　だが、生き残るものはすべて、
　理由があって生き残る。

　要するに合理性とはリスク管理なのだ。

　--------------------

　「成功するには、まず生き残るべし」
　という原則がある。

　（中略）

　私は

　「重要なのは順序であり、
　　破滅の可能性があるところでは
　　費用便益分析がまるきり無意味になる」
　
　という原則に従って一生を送ってきたと
　いっても過言ではない。

　--------------------

　繰り返しリスクに身をさらすことが
　余命を縮める

　（中略）

　これらのリスクがすべて累積し、
　被験者の態度に反映される。

　破滅は、破滅を引き起こすランダム性の
　要因とは切り離せない不変のもの。

　--------------------

　単なるリスクについて恐怖を煽るのは
　文字通りの煽りに他ならないが、

　破滅的なリスクについて
　恐怖を煽るのは違う。

　リスクと破滅は別のものなのだ。

　--------------------

　増殖的でシステム的でファットテールな
　リスクと、

　増殖的でなく特異的なシンテールの
　リスクを比べてはいけない

　--------------------

　破滅を完全に避けつつも、
　リスクを愛することはできる。

　破滅を含むような戦略では、
　利益が破滅リスクを帳消しにすることは
　決してない。

　破滅とそれ以外の状況の変化は
　全く別の生き物である。


　人間の冒すリスクは積もり積もって
　その人の余命を縮める。

　合理性とはシステムの破滅を
　防ぐことである。

　--------------------
　（ここまで）


　凄くないですか？

　私（鮒谷）は完全に
　ここに書いてある内容に従って、

　生きている（きた）だけ。



■いつもメルマガで書いてきたことは、

　上記の超簡易的な内容であると
　お分かりいただけるのではないでしょうか。

　今回の（京都行き延期の）意思決定も
　こうした意思決定基準に従ってのこと。



■何年か前、北朝鮮がバンバン、
　ミサイルを飛ばしていたときには、

　東京を離れて軽井沢に逃げていた
　時期があったくらいですし。



■観光産業に携われている方や、

　京都（近畿、関西、西日本）に
　お住まいの方などは特に、

　不快に思われるところがあるかもしれず、
　その点は申し訳ないと思いますが、

　他意はありません。



■とはいえ、
　どれだけ心がけたところで

　「死ぬときは死ぬ」
　「破滅するときは破滅する」

　ということもまた、
　百も千も承知しています。



■だからこそ日々（本当に毎日）、

　イメージトレーニングを重ねながら
　最終的には何が起きても受け入れる、

　という

　「諦観」
　「受容の精神」

　の涵養に努めていますが、


　それはそれとして、
　避けられるリスクは避けたいという、

　一種病的な人間の妄言、くらいに
　捉えていただいて良いかと思います。



■別にこんなことに
　正解はありませんからね。

　あくまで個々人の
　価値判断の基準に過ぎません。
　


■にもかかわらず、
　ここまであれこれと書いたのは、

　（誰も見たことがないと思いますが）

　あくまで鮒谷個人が、時々刻々、
　変化を続けるその時々の考えを、

　読者さんの「思考のきっかけ」として
　活用いただければと配信しているものであり、

　同意を求める、あるいはその考えを
　強要しようとするものでもありません

　（自身がそんなに立派な見識をもった
　　人間であるとも考えておりません）
