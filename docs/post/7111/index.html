7112号 便秘は万病のもととなる

7111号 草むらに隠れ続ける武士が可児才蔵に出会ったら

■昨日のメルマガで

　＜アンサーズ法律事務所＞
　https://answerz-law.jp/

　の代表として
　弁護士として活動されるとともに、


　上場企業２社の取締役も
　務めていらっしゃる、

　荒木さんの話を書きました。



■昨日のメルマガの内容を一言でいうと


　「名乗りを上げた瞬間から
　　すべてが変わる」

　「名乗りを上げたら、
　　いろんな人が応援してくれ、
　　協力してくれ、
　　世界が劇的に転換する」


　という話。



■問題は、

　「名乗りを上げ慣れている人」

　と

　「名乗りを上げるのに緊張する人」

　「名乗りを上げたことがないから
　　名乗り方が分からない人」

　の間に決定的な断絶があること。



■いくら頭が良くても、
　結果を出せない理由の一つは

　「名乗れない
　　（怖くて勇気が出ない）」

　ところにあるのです。



■戦国時代の武士とかも
　同じだったんじゃないでしょうか。


　普段から剣術の練習を重ねているのに
　怖くて名乗りを上げられず、

　戦いの行方が見えてくるまで
　草むらに隠れていた武士、

　絶対にいたと思うんです。



■そして私（鮒谷）が
　その頃に生まれていたら、

　絶対にそちらのタイプの
　人間だったんだろうな、

　ということも分かります（笑）



■ただ、たとえそんな人であっても、

　百戦錬磨の可児才蔵みたいな
　槍の名手と一緒に戦ったら、

　「名乗りを上げる勇気」

　も出てこようというものです。



■要するに

　【勇気がなければ、一緒にいて
　　勇気の出る人と戦えばいい】

　のです。



■しばらく戦い続けているうちに

　【勇気が慣れに変わる】

　ようになるものです。


　行き過ぎて、

　慣れが慢心にまで至るのは
　考えものですが、

　それでも

　【毎日の定例行事として、
　　名乗りを上げられるようになる】

　くらいにまでなれば、
　世界は変わりますよ。



■今では

　「呼吸をするみたいに、毎日、
　　何かを売っている私（鮒谷）」

　でも、

　昔は、なにかを売ろうとするたびに
　ビクビクしていたものです。



■まさに

　「戦が終わるまで
　　草むらに潜んでいた武士」

　みたいなもの。

　怖くて、恐ろしくて、
　名乗りを上げられなかったのです。



■しかし、

　歴戦の勇士の皆さま
　（先輩経営者の皆さま）

　と長い時間、
　ご一緒し続けることによって

　「呼吸するようにものを売る」

　ことができるようになりました。



■戦い方の手本を示し、かつ、
　守り、アドバイスを下さる、

　そんな諸先輩方がいらっしゃらなければ、
　今の私はありませんし、

　早々に討ち死にしていたに
　違いありません。



■そういう意味では、私（鮒谷）が、

　「（年間プログラムや個別コンサルを）
　　申し込みませんか？」

　というのは、

　「（可児才蔵ならぬ）
　　鮒谷周史と一緒に戦いませんか？」

　と言っているのと一緒です。



■うぬぼれが過ぎる、
　と言われるかもしれませんが、

　そろそろ、この程度のことは
　言ってもいいくらいには、

　相応の履歴も残してきたつもり。



■討ち死にを回避し、

　【勇気が慣れに変わる】

　ところまで進みたい方があれば、

　いつの日か、年間プログラムか
　個別コンサルにてお会いいたしましょう。


　音源だけでは決して見ることができない
　世界を見せて差し上げます。


　今日も人生とビジネスを楽しみましょう！

　【今日のピークパフォーマンス方程式】

　　■名乗りを上げる勇気がない人は、
　　　一緒にいて勇気の出る人を探すべし。

　　■そうした人と一緒にいるだけで、
　　　勇気が出てくるし、

　　　勇気を出し続けて、
　　　継続的な戦果（成果）も上がってくると、

　　　名乗りを上げる勇気が（良い意味で）
　　　「慣れ」に変わって恐れがなくなる。

　　■こうして名乗りを上げ続けられる人
　　　（＝自分の渦に他者を巻き込み続ける人）

　　　になるのである。
