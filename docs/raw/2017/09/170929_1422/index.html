■こんにちは、鮒谷です。


　おかげさまで（？）
　昨日メルマガにてご紹介した、

　　
　【（大量画像あり）
　　数十年ぶりに「パンドラの箱」を開けてみた】
　
　https://note.mu/funatani/n/n447faab3bede


　に大きな反響を頂きました。



■いろんな感想を頂きましたが、

　ミャンマー在住の桂川さんがお寄せ下ったような


　-------------------------------
　
　ブログ等でおっしゃっていた、
　リアルのび太 伝説。。

　心のどこかでは

　「関西人、大げさに盛ってるんだろうな。」

　と、信じていませんでしたが、
　本当だったんですね。笑　

　大変、失礼いたしました！

　-------------------------------

　
　という類のご感想が一番多かったです（笑）



■マジのガチの本当の話であったことを
　これでご理解頂けたかと思います。


　これまで

　「経歴詐称」

　ならぬ

　「デキが悪かった詐称」

　を頻繁に疑われてきたので、

　これで疑念を完全に晴らすことができたかと
　思います（笑）


　「高校入学以来何をしていたのか」

　と書かれたテスト答案ですが、
　まさしく、何もしていませんでした、、

　https://note.mu/funatani/n/n447faab3bede



■話は変わりますが、

　今日の夜は久しぶりに不動産投資の
　プロフェッショナル、沢孝史さんと会食。


　今でこそ「サラリーマン大家」さんは
　一大ブームとなっていますが、

　当時、まさにこの流れの、
　はしりのはしり、といった形で、

　不動産投資を始められていた沢さん。

　今では総投資額25億になるそうです。
　すげー！

　http://www.rakumachi.jp/news/column/212135/4



■そんな沢さんとは、


　十数年前、

　不動産投資サイト「楽待」を運営し、
　今では東証一部上場企業になった、

　ファーストロジックの坂口直大社長に
　紹介してもらい、

　以来、ちょくちょく３人で
　お会いしていたものでした。
　


■ざっくばらんでとっても楽しい方なので、
　久しぶりにお会い出来るのが楽しみです。

　沢さん、そして本日ご一緒する
　北野さん、コテツさん、

　今日はよろしくお願いいたしますー！



■そして明日は再び大阪に向かいます。
　

　こちらのセミナー

　【大阪会場】少人数制・あなたが変わりたくても
　　変われなかった「３つ（＋７つ）の理由」セミナー

　https://shinkaron.buyshop.jp/items/8336153

　は、すでに満席となりましたが、


　その後、行うこちらの放談会

　【大阪会場】少人数制「比喩の力で人生を作り変える」
　放談会

　https://shinkaron.buyshop.jp/items/8381902

　は、あと１、２名なら、なんとか入れると思います。


　もしよろしければ、ぜひお越し下さい。

　リラックスモードで超ゆるい感じで行うつもり。
　質疑も自由です。

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。

【 「天職の芽」は、あなたの過去の中にある 】 

───────────────────────────────────



■昨日もご紹介の通り＆上述の通り、

　大阪の実家にて、パンドラの箱を開け、
　いろんなお宝を掘り出してきました。
　

　【（大量画像あり）
　　数十年ぶりに「パンドラの箱」を開けてみた】
　
　https://note.mu/funatani/n/n447faab3bede


　改めて思ったのが、

　「三つ子の魂百まで」

　ということ。



■上のブログにも記した通り、

　（今に至るまで
　　まったく記憶になかったのですが）

　どうやら『ビッグ・トゥモロウ』誌を
　愛読していたようです。　

　
　ただ実物（雑誌）を目にすることで、

　勉強についていけなくなり、
　深い悩みを抱えて

　「起死回生の一打」

　を模索していた当時の記憶が解放され、
　急速にそのときの気持ちが蘇ってきました。



■今はどうか分かりませんが、
　少なくとも当時の同誌は

　「（やればできる、と思っていながら）

　　なかなか成果を上げられず、
　　周囲からデキが悪いと思われている人間」
　
　が対象だったのではないかと思います。


　その証拠に、
　同誌の特集記事を拾い上げてみると

　『学校秀才がどうにもかなわない
　　落ちこぼれたちの生きる頭脳』

　『会社、学校のダメ人間が
　　たちまちできる奴を追い抜いた笠巻式プラス・ワン』

　といったタイトルが目白押し。



■そんなタイトルに惹かれて、

　『学校秀才』
　『できる奴』

　をギャフンと言わせたい、

　そんな鬱々とした思いが
　蓄積されていたのでしょうね（汗）


　それにしても同誌の発行年月日を見て逆算すると
　私が高校一年生のときに買っていたようですから、

　「早熟な自己啓発マニア」

　だったことは間違いなさそうです、、


　
■それ以外にも、
　写真を撮ってくるのを忘れましたが、
　
　（これも自己啓発の王道とも言える）

　たくさんの

　「三笠書房の本（知的生き方文庫）」

　も発掘されました。


　たしかに高校１年生のときから学生時代、
　さらに新社会人になってからも

　「狭義においても、広義においても
　　自己啓発本」

　を読み漁っていたわけで、
　そういう意味では、

　今、私がこうしたメルマガを発信しているのも、
　いろんなセミナーを行っているのも、


　長い人生を通して見てみると、

　【すべては一貫した営みであった】

　といえるのではないかと、
　今さらながらに思った次第です。



■その昔、

　中・高時代に考えていたり、
　取り組んでいたことに、

　「天職の芽が眠っている」

　なんていう話を聞いたことがありますが、

　その言葉はあたらずとも遠からずだったのかも
　しれません。



■あなたがもし、今、

　「自分が取り組んでいることが
　　いま一つ、しっくりこない。

　　『コレジャナイ感』
　
　　を抱えながら、毎日仕事をしている」

　という感覚を持たれているのであれば、


　一度、中学高校時代（あるいは、それ以前の
　小学校時代）に立ち返り、

　当時、何を考え、何を行い、
　どのような感情を抱えていたのか、

　を振り返られると、ひょっとすると何かの
　ヒントが得られるかもしれませんね。



■少なくとも私（鮒谷）は、
　今行っている仕事に対して

　【さらなる正当性の確信】

　を手に入れることができました。


　こうした自分の信念に対する

　【首尾一貫性】

　が強固になればなるほど、
　より大きな力をもたらしてくれるものですし、

　そのためにもますます

　「過去・現在・未来」

　をつなぐ、

　もっと自らを鼓舞する物語の発見、
　さらには創造、

　を行っていこうと思います。



■こうしたことを考えるという、
　内的な営み（思索）は、

　【（大量画像あり）
　　数十年ぶりに「パンドラの箱」を開けてみた】
　
　https://note.mu/funatani/n/n447faab3bede

　の記事

　「自分の身の上に起こる幸・不幸にまつわる
　　あらゆる出来事をポジティブに解釈し、

　　バリバリ呑み込み、文脈を拡大していく」

　ということになるでしょう。



■こんな内的営み（＝省察）のプロセスやメカニズムや
　効果・効能について、

　圧倒的な質・量でお届けしているのが
　以下の２つの音源となります。　
　
　
　【鮒谷周史の、圧巻！『物語』を自在に操れる者だけが、
　　人生を思い通りに生きられる」放談会音源】
　　https://shinkaron.buyshop.jp/items/6536488

　【鮒谷周史の圧巻！「転落続きの人生を、
　　日記を書いて蘇生させた話」放談会音源】
　　https://shinkaron.buyshop.jp/items/5955812

　
　もし、未だお聴きになられていない方が
　いらっしゃいましたら、ぜひ聴いてみて下さい。



■私の数十年に及ぶ試行錯誤の集大成であり、

　【首尾一貫した人生】

　を送ることができる楽しみや喜びについて、
　理解を深めて頂き、

　さらにはそうした人生を過ごしたい、
　という方にとっての一助となるであろうことを、

　確信しています。
