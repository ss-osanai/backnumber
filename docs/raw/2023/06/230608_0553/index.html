■先週土曜日に開催いたしました

　---------------

　【潜在意識のプロが教える、

　　思い通りの人生を送るための
　「マインドブロックの外し方」セミナー】

　---------------

　に参加くださったＴさんより
　以下のような長文の感想を頂戴しました。

　

■下記にてご紹介した後、
　関連する話を本文に続けます。


　（ここから）
　-----------------


　お世話になります。

　今回はマインドブロックの外し方、
　ということで、

　催眠をテーマに機会をいただき、
　ありがとうございました。


　私自身は、

　催眠というものに対して、
　半信半疑…どころか二信八疑、

　くらいの気持ちでありました。


　ではなぜこんな私が
　このセミナーを受けたのかというと、

　今にして思えば、藁にも縋りたい
　気持ちだったのだと思います。


　他の人と比べて、
　自分の足りないことに目がいって、

　自信を無くしたり、卑屈になったり、
　やる気がなくなってしまったり、

　努力する気力がなくなってしまったり、
　エネルギーが感じられなかったり、、

　そんな毎日がこれからも続いていくと
　考えたら、全然足に力が入りませんでした。


　そんな毎日でした。


　また、そんな時には、
　いつも自分の中には

　「今までもそうだったし、
　　きっと今回も変われない」

　そう思っている自分に
　気がついてもおりました。


　そして、そんなもう1人の僕を
　なんとかしなければ、

　何をやっても
　空回りしてしまうのではないかと、

　心のどこかでは打破できるきっかけを
　探し求めていたのではと思います。


　さて、セミナーを実際に受けてみての
　感想ですが、

　私が考えていた眉唾なものでなく、
　右脳的でありながら、
　大変にロジカルな説明で、驚愕でした。笑


　最後の鮒谷さんの放談の中でも
　話がありましたが

　「いっていることは同じ、
　　アプローチの仕方が違うだけ」

　ということで、

　鮒谷さんが今まで言われてきたことを
　より立体的に、より深く理解・実践する、

　そんなセミナーだったと感じています。


　私の中で特に心に残った内容について、
　記載してみますが、

　もし誤りがありましたら
　教えていただければ有難いです。


　●脳の使い方を知るということは、

　　（以下、内容に関連するため略）

　　潜在意識にはものすごいパワーがある。


　●潜在意識のパワーはものすごいため、
　　見たいもの・聞きたいものしか見えないし、

　　（以下、内容に関連するため略）

　　それをノンバイアスにしていくことが
　　催眠やマインドフルネスの目的。


　●それまでの自分のバイアス・信念はどうか、

　　（以下、内容に関連するため略）

　　などなど・・・

　　この信念によって見え方・聞き方が
　　変わってしまう。


　●だからそれを書き換えていくと
　　どうなるか。

　　（以下、内容に関連するため略）

　　それくらい強力な作用を持っている。


　●器の法則。

　　（以下、内容に関連するため略）


　●この器はセルフトークによって
　　育てることができる。

　　（以下、内容に関連するため略）

　　脳は敏感であり、
　　気をつけなければならないことが２点。

　１ 脳は思い浮かんだ言葉、

　　（以下、内容に関連するため略）

　２ 脳は思い浮かんだ言葉、発した言葉が

　　（以下、内容に関連するため略）
　

　●潜在意識の基本性能「自動思考」

　　（以下、内容に関連するため略）


　●脳はリアリティが好き

　　（以下、内容に関連するため略）

　　そのためには
　　成長記録・成長実感をつけることが有用。


　●何が心のブレーキになっているか

　　（以下、内容に関連するため略）


　今までにも、鮒谷さんの言葉による書き換えを
　音源で聞かせてきていただきましたが、

　信念の塗り替えに至っていなかっただけで、
　全く無駄になってはいなかったということが
　知らされて、それも嬉しくなりました。


　自分が変わる最初の一歩である

　「目標を立てる」

　ためには、

　なんの制約もない状態を想像して、
　したいことやりたいこと、ありたい状態、

　なんでも１００は最低書いてみましょうと
　よく言われます。


　私は書けず、書けず、書けず、
　すごく打ちひしがれたワークになり、
　すきじゃない（笑）のですが、

　そもそも、これが書けるバイアスがない
　脳の状態だったのだろうかと、

　今までにない疑問が生まれました。


　したいこと、書いてくださいと言われ、
　考える脳が、

　果たしてよく機能してくれていたのか、
　第一歩である目標設定の中でもさらに、

　最初の腕を引き、腰を捻る、
　そんな当たり前になってしまっている
　動作を意識させられ、

　そこに違和感があるなら
　まずはそこもなんとかしましょうよ、


　という大変重要なセミナーだったのでは
　ないかと感じました。

　（おそらくもともとは目標に向けて
　　エネルギー全開で進んでいくために、
　　という内容だったかと思うのですが、

　　感想を書いていて、
　　私にとっては全然ちがう受け止めに
　　なっていて、面白いなと感じました。笑）


　相手に向ける言葉も、
　自分に向ける言葉も

　もっともっと大切にしていきたいし、
　していきます。


　他の音源で
　鮒谷さんがお話しされていた、

　「みなさんもっと言葉を
　　愛されればよいのにな、

　　みなさんもっと言葉を
　　愛でられればよいのにな」

　という言葉が、
　急に色づいて思い出されました。


　この潜在意識というものの持つパワー、
　イメージが生み出す世界を
　現実化する能力を活用し、

　一層明るい日々を送れるよう
　精進していきます。


　この度は本当にありがとうございました。

　引き続きよろしくお願いいたします！


　（Ｉ・Ｔ さま）

　（ここまで）


　Ｔさん、長文のご感想、
　ありがとうございました！



■せっかくお書きいただいたのですが、

　直接、内容に関わる箇所については
　私（鮒谷）のコンテンツではないため、

　勝手に紹介するわけにもいかず、
　
　（以下、内容に関連するため略）

　の記載をいたしました。

　なにとぞご容赦ください、、、汗



■実は今回、Ｔさんより頂戴した
　ご感想を読んでいるときに、

　日経新聞の夕刊に連載中の、

　朝井リョウ
　『イン・ザ・メガチャーチ』

　に、少し前に書かれていた

　【脳の回路が開通】

　という言葉を思い出したんですよね。


　
■とても印象深い言葉だったので、

　これはいつでも使えるように、
　脳みぞに刻み込んでおこうと思い、

　何度も繰り返し復唱（？）していた
　言い回しが突如、蘇ってきたのです。



■用いられている文脈は
　もちろん異なるんですけどね。


　参考までに（？）
　「推し」をテーマにした物語である

　『イン・ザ・メガチャーチ』

　に記されていた、
　
　【脳の回路が開通】

　の前後を以下に引用。


　（ここから）
　-----------


　倫太郎が眉を下げて申し訳なさそうに
　しているほんの十数秒の映像。

　そこから読み取れる情報量を
　遥（はる）かに超えた解釈に勝手に励んで、
　勝手に胸を打たれる。

　こういうところを、その気質のない人種から
　嗤（わら）われていることくらい、
　自分でもよくわかっている。


　でももう、そういう脳の回路が
　開通してしまったのだから仕方ない。


　それに、名前も顔も知らない
　"一般常識人"みたいな概念に嗤われたところで、
　本当に、あまりにもどうでもいい。

　それよりも、感情に合わせて動く眉毛さえ
　愛しく思える対象に出会えた幸運に、
　日々感謝が止まらない。


　-----------
　（ここまで）　

　

■いかがですか。さすがですね。

　この、

　【そういう脳の回路が
　　開通してしまったのだから仕方ない】

　の、

　「声に出して読みたい日本語」感。



■それはともかく、これは、


　ひとたび回路が開通したら、
　他の一切がどうでもよくなって、

　「推し」の一挙手一投足から、
　他の人が想像もできないような、

　圧倒的なボリュームの情報を
　取り出せるようになる、


　といった話ですが、

　願望実現の世界においても、
　同じような現象が起こるんですよね。



■つまり

　「ある願望を実現したいという
　　回路が開通してしまったら、

　　他のことがどうでもよくなる
　（それ一つに集中するようになる）」

　ということ。



■そう考えると、

　かたや「推し」の話、
　かたや「願望実現」の話、

　という、
　まったく別のものではありますが、
　

　ひとたび【脳の回路が開通】したら
　そのこと以外に考えられなくなる、

　そんな意味で、両者に通底するものが
　あるように思われます。



■おそらく、

　Ｔさんの感想を読んだ瞬間に
　この言葉が思い出されたのは、

　無意識にそんな共通項を
　見つけ出したからではないでしょうか。



■そして確かに、

　人生が変わる、
　その前には間違いなく

　【脳の回路が開通】

　した瞬間があるのです。



■それは私（鮒谷）だけの話ではなく、
　
　【人生を質的に作り変えた人】

　であれば、
　必ず経験しているはずの出来事。


　そこから世界の捉え方が変わり
　（パラダイムシフトが起こり）、

　行動が変わり、
　運命、人生が変わっていくのです。



■私（鮒谷）は長年、

　あの手この手を使って
　この体験を提供していただくために、

　言葉を尽くしてきたわけですが、


　先日のセミナーにお招きした
　坂上さんは

　【催眠（自己暗示）】

　という技術を用いることによって、

　【脳の回路を開通させるお手伝い】

　をしていらっしゃる、

　と言えるのではないかと思います。
　


■もしあなたの脳が、既に
　
　【開通】

　しているのであれば、とっくに

　「願望実現以外の
　　他のことがどうでもよくなる」

　状態に入っているはずであり、
　当然、そのための行動も行っているはず。



■が、今、
　そうなっていないのであれば、

　【未開通】

　なんですよ、きっと。


　開通しているかどうかが分からない、
　という方も未開通。

　開通したら、
　必ずそれと分かりますからね。
　


■、、、そんなわけで、
　ここまでお読みくださった方の中で、

　【自分は現状、未開通。
　　でも没頭没入の回路を開通させたい
　（というか、どんな体験！？）】

　と思われたあなたは、

　ぜひ動画＆音声コンテンツで
　
　【脳の回路の開通のさせ方】

　を学んでください。
