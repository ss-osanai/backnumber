━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
＜今日のピークパフォーマンス方程式＞

  【　狭量が広量に切り替わり、度量が生まれる　】

───────────────────────────────────



■昨日のメルマガでは、


　「自分の軸が定まらないから動けない、
　　というのは間違いで、
　
　　動くからこそスタンスが固まってくる、
　　というのが正解」


　というお話をいたしました。

　今日も続けます。



■どんどん外に出て、さまざまな人や考えに出会い、
　そこで混乱、葛藤が生まれたら、

　なぜ、そのような葛藤が生まれたのか、
　理由を考える習慣をつけたいものです。
　

　さらに理由を考えるだけでなく、
　
　その事態を自分の内側で統合し、飲み込めるように
　なるためには、

　自分の認知システムを変えてやる必要が
　出てくるかもしれません。



■正しくこのプロセスを繰り返せば、

　「狭量が広量に切り替わり、度量が生まれる」

　ものなのだと考えています。
　
　
　「一回り大きくなる」
　
　というのは一つの比喩表現ですが、
　
　この過程において文字通り、認知の枠組みが

　「一回り大きくなる」

　のだと思われます。



■これまでに何度も書いてきましたが、

　これを強力に促成、推進してくれる
　極めて有効な手段となるのが、

　「考えを文字にして表し続ける」

　というもの。

　
　「文字にして表す」

　ではなく、

　「文字にして表し続ける」

　というところがポイントです。



■来る日も来る日も、

　否応なく人生と向き合わせられ、
　様々な事象の意味付けをしていく、
　
　という苦しみが、

　「自分を一回り大きくしてくれる」
　
　のではないでしょうか。


　そんなことを信じて、毎日の日常を振り返り、
　反省し、総括し、

　記録することを続けてきました。



　今日も人生とビジネスを楽しみましょう！
