7220号 セールスやマーケティングにも効く「催眠＆自己暗示」

■今週月曜日から

　---------------

　【潜在意識のプロが教える、

　　思い通りの人生を送るための
　「マインドブロックの外し方」セミナー】

　---------------

　をご案内してきましたが、
　いよいよ、これで最後です。

　（お申し込み締め切りまで
　「あと４時間」となりました）



■前々回、前回のメルマガで

　【道具箱に、
　　各種道具を揃えましょう】

　という話をしましたが、
　今回も続けます。



■コミュニケーションという
　限定された分野においても

　【道具はたくさんあったほうが良い】

　という法則は、
　とうぜん当てはまります。



■コミュニケーションにおいて、

　相手が変わっているのに頻繁に
　同じ（望ましくない）結果が起きるのなら、

　それは

　【相手が悪いのではなく、
　　自分が悪い】

　んですよね。



■よりハッキリいうならば、

　コミュニケーションにおいて
　使える道具が一種類しかなければ、


　たとえ相手が誰であっても、

　【同じ道具を用いて、
　　同じ結果が引き出される】

　ことになるわけです。


　道具が一緒なのだから、

　そこから生じるアウトプットも
　同じものになるのは当然です。



■たとえば私（鮒谷）であれば、

　短かったサラリーマン時代に
　営業職に就いていましたが、

　誰に営業しても

　【買ってもらえない】

　という同じ結果を
　引き出しておりました。



■なんなら、

　営業に就いている人の中でも
　直接言われることなど、

　ほとんどないであろう

　【出入り禁止の申し渡し】

　を、短い期間にもかかわらず、

　人を変えて、
　何度も言われたことがあります。

　（再現性、高すぎ汗）



■お客さんから

　「二度とうちの敷居をまたぐな」

　という反応を高い確率で
　引き出してきたわけですが、


　今から考えると
　そうなってしまっていた理由は明確で

　【道具箱に
　　適切な道具が入っていなかった】

　だけなんですよね。



■客先に行っても、
　世間話から入ることができず、

　「いきなり営業資料を取り出しては
　　相手の目も見ず、読み上げ始める」

　という竹槍コミュニケーションを
　繰り返す。



■そりゃ、客も

　「まず自社のことを聞けよ」

　と思うでしょうし、
　怒るに決まっていますよね。



■今、私（鮒谷）が
　そういう営業を受けたら、

　それこそ相手に

　「出入り禁止を申し渡す」

　に違いありません。



■当時の私（鮒谷）は、

　自分が持ち歩いていた
　「営業の道具箱」を覗いてみても

　「いきなり営業資料を取り出しては
　　相手の目も見ず、読み上げ始める」

　という名のトンカチしか入っておらず、

　相手が誰であっても
　トンカチで相手の頭をぶっ叩いては、

　そのたびに
　ブチ切れさせていたわけです。



■さすがに今となっては、

　【再現性高く（！）、
　　出入り禁止を食らっていた理由は、

　　ただ道具箱に、入っているべき
　　道具が入っていなかっただけ】

　と分かります。



■だからこそ

　【さまざまなシチュエーションに
　　対応できるよう、
　　数多くの道具を取り揃える】

　ように心がけてから、

　お客様から激怒されることも
　なくなりました。


　（どころか、再現性高く、
　　いろんなものをお求め頂けるように
　　なりました。

　　道具箱にたくさんの道具が
　　取り揃えたからですよ！）



■そういうことなんですよね。

　相手が変わっても、

　いつも同じこと
　（あるいは同じ傾向のこと）

　が起こるとするならば、


　その理由は
　自分の道具の種類が乏しく、

　【コミュニケーションが
　　ワンパターンに堕している】

　ことを疑うと良いでしょう。

　そしてたいていの場合、
　それが正解です。



■たとえば、誰と対応しても、

　「笑顔をみせてくれない」
　「無愛想なまま」
　「不機嫌になる」

　「ものを買ってもらえない
　「クレームばかりを受ける」
　「苦情対応するほどに炎上する」

　といったことが起きるのなら、

　道具箱に限られた道具しか
　入っていない

　（＝そのとき必要なコミュニケーションの
　　　道具を持っていない）

　ということになるわけです。



■誰が相手でも、

　「カナヅチでぶっ叩くだけ」

　では、そりゃ相手も怒るでしょう。


　違う例では、

　【ハサミを持った子どものような】

　という例えもありますが、
　これなども、まさにそうですね。



■知能がいまだ、
　十分に発達していない子どもが、

　ハサミという単一の道具を持ったら、
　何をみても切り刻んでしまうのも、

　仕方がないことでしょう。



■親や周囲から叱られても、
　また切り刻む。

　それしか道具がない上に、
　知恵も乏しいのだから

　「分かっていても、やってしまう」

　ところに問題があるのです。



■コミュニケーションや営業において

　「不快な出来事が何度も続く」

　ようならば、

　まず自身の道具箱を開いて、
　どんな道具が入っているかを、

　確認しなければなりません。



■そして、

　それぞれの状況に応じ、
　対処するための工具が足りなければ、

　急ぎ、調達してきて、さらには
　扱い方に習熟する必要があります。


　こうした場合、

　【道具を増やす以外に、
　　根本的な解決策はない】

　のですから。



■だかこそ、

　【大急ぎで学び、大急ぎで試し、
　　大急ぎで使いこなせるようになれ】

　ということですね。


　人生を逓増させたい人は
　この原則を頭に叩き込み、

　常日頃から
　実践し続けることが大事なのです。



■ちなみに、

　明日（土曜日）開催のセミナーで
　学ぶことができる

　【催眠＆自己暗示】

　も生きていく上での、

　極めて有用な
　（つまり是が非でも揃えておきたい）道具、

　と言えるでしょう。



■落ち込んだり、
　自らの歩みを止めさせたり、

　といた障害を克服するための
　セルフマネジメントに、

　圧倒的に役立つ道具ですから

　【一家に一台】

　ならぬ

　【一人に一つ】

　くらいの感じで、
　実装しておきたい道具、ベスト３、

　にはまず間違いなく
　ランクインするはずです。



■さらに、あえて
　誤解を恐れずにいうならば、

　【催眠＆自己暗示】

　はセルフマネジメントに
　使えるとともに、

　セールスやマーケティングにも
　応用が効くんですよね。

　いうまでもなく、
　悪用厳禁ではありますが。



■自己催眠と他者催眠は
　本質的には同じことですからね。


　私（鮒谷）が長年に渡って、
　どんなものでも売り続けられるのも、、、

　あ、恐ろしいことを
　いっちゃいましたかね（汗）

　今すぐ、忘れて下さいませ。



■ただ、

　こうした能力を「善用」すれば、

　お役に立ちたいと思った相手（顧客）に
　結果を残してもらえるし、

　感謝をされるようにもなるでしょう。



■さらにいうと、
　お子さんのいらっしゃる方であれば、

　みんな気になる、

　【お子さまの教育】

　なんかにも
　応用できるかもしれません。

　（というか、応用できるでしょう）



■子どもは特に、

　親や先生や周囲から、
　ただひたすら無防備に

　【催眠や暗示を
　　かけられ続けている存在】

　なわけですから、

　ネガティブな催眠を遠ざけ、
　ポジティブな暗示を与えられたら、

　素晴らしいことだとは
　思いませんか。



■何はともあれ、この技術については

　【大急ぎで学び、大急ぎで試し、
　　大急ぎで使いこなせるようになれ】

　を実践いただきたいものです。


　言葉を換えれば、
　これまでさんざん書いてきたように

　【すべての学びは前倒し】

　ということになるわけです。



■ちなみに、

　私（鮒谷）も20年以上前から道具箱に
　この道具（催眠＆自己暗示）を入れて、

　ことあるごとに、用いてきました。


　それほど汎用性のある道具だし、

　だからこそ、いろんなセミナーや音源でも、
　直接・間接にこの道具について、

　語ってきたわけです

　※一番はっきりお伝えしているのは
　　『自尊心』ではないかと思います。



■ぜひあなたも、

　大工さんの道具箱にカナヅチやカンナが
　入っていなければ仕事にならぬように、

　プロの人生大工（？）になりたいのなら、

　真っ先にこの道具は道具箱に
　入れておかれるべきだと思いますよ。



■長々と書いてきましたが、
　以上のような背景もあり、

　ここ何日かの間、
　各種道具の中でも、

　特に汎用性の高い、
　すなわち応用範囲の広い、

　【メンタルブロック解除のための
　　催眠、自己暗示という道具】

　のご案内をしてきた、
　ということになります。



■人生はオープンワールドで、

　せっかく自由な遊び方が
　許されているわけですから、


　どうせなら私（鮒谷）は

　【各種道具を
　　できるだけたくさん獲得しては、

　　それぞれの使い方の研究をして
　　自分の生活に役立てる】

　生き方を選びたいと思って、
　実践してきました。



■そのような（人生）ゲームをプレイし、
　20年にわたって毎日

　【（人生）ゲーム実況】

　を行ってきた、
　ということになりそうです。



■一度しかない人生。

　せっかくなので、
　たくさんの道具を手に入れて、

　あれこれ組み合わせては
　使ってみて、

　人生というゲームを
　一緒に存分に楽しみませんか。



■「メンタルブロック解除の道具」

　は数ある道具の中でも、
　最優先で手に入れておくべき道具、

　だと思います。

　（人生）ゲーム攻略サイトにも
　そう書いてありましたよ！


　今日も人生とビジネスを楽しみましょう！

　【今日のピークパフォーマンス方程式】

　　■コミュニケーションにおいて、

　　　望ましくない結果が何度も繰り返し、
　　　現れるのであれば、

　　　それは相手が悪いのではなく、
　　　自分が悪いのだ。

　　■自分の道具箱に一つしか
　　　道具が入っておらず、

　　　誰を相手にしても同じ道具を用いて
　　　同じ（望ましくない）結果を引き出す、

　　　ただ、これを繰り返しているだけだろう。

　　■この場合、道具を増やす以外に、
　　　根本的な解決はないのである。
