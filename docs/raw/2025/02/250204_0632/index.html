■今日は

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------

　動くからエネルギーが湧いてくる。
　動かないとますます動けなくなる。

　動く人はどんどん元気になるし、
　動かない人は一層腰が重くなる。


　このメカニズムが働き、
　動く人と動かない人の両者は

　「複利と活気の相乗作用」

　によって凄まじい差が生まれ、
　さらには加速する一方になるんだわ。

　----------

　他人に期待するから裏切られて苦しむ。
　過剰な期待はせんことよ。

　代わりに自分の未来に
　望みをかければいい。

　期待し放題の上に、望めば
　望んだ以上のものを返してくれるから。


　穏やかに生きたければ、
　自分以外に過度の期待をせんことと、

　期待外れや裏切りまでも
　あらかじめ織り込んでおくことよ。

　----------

　弱者の戦略はご縁が全て。

　一つの武器を磨き抜く。
　ご縁を通して見出してもらう。

　全力を尽くして貢献する、役に立つ。


　評価と実績と信用が生まれる。
　それを元手に新たなご縁がもたらされる。

　これを高速で雪だるま式に回していくと、
　あっという間に別世界に出られるよ。

　----------

　平均を嫌え。
　異常値を取りに行け。

　これが今後のキャリアの基本戦よ。


　バランス感覚を重んじ、
　全方位に資源を割り振るから金太郎飴化し、

　「お前なんかどっちにしろ、
　　いてもいなくてもおんなじ」

　と言われるようになる。


　取り替え可能な汎用人材
　（≒中間管理職）から順番に、

　市場から追い出される。

　----------

　今に見とれや、の精神は大事よ。

　ニーチェやないけど、
　力を持たぬ者のルサンチマン精神は

　大量行動力をもたらしてくれるからな。


　ネガ感情をポジに転換させ、
　燃料を燃やし尽くせばいい。

　ワイの20～30代なんか全部これ。

　ってか死ぬまで燃料切れは
　なさそうやから、

　ワイを虐げてきた強者には
　感謝しかないわ笑。

　----------

　「Win-Win」に限らず、
　やましい気持ちがある人は

　余計なひと言を付け加えたがるんよね。

　営業とか交渉の場で

　「それ、わざわざ言わんでええやん」

　という、
　気になる言葉を使う人があったら、

　その直感は当たっているから
　その場で席を立てばいい。

　----------

　若者で可愛げないのは
　マジで致命的。

　賢ぶるより可愛がられよ、やで。

　知ってるからといって
　自慢げにペラペラ喋ると嫌われるんよ。


　笑顔/素直/向上心の間に、

　知性を感じさせる
　切れ味鋭い発言を差し挟む。

　ワイの感覚やと85:15くらいが黄金比。

　比率が逆転してる奴は、
　頭は良くてもチャンスをもらえない。

　----------

　自信と増長とは違うんよ。

　経験豊富な年長者は
　簡単に見透かすからな。

　自信がありつつ謙虚で、
　愛嬌とフットワークのある若者。

　これが最強。
　先行投資したくなる若者の代表格。

　----------

　自分が何をしたいのか、
　何を発信したいのかも定まってないのに、

　ノウハウに踊らされて、
　既視感しかない定型発信してても、

　読みたがる人なんかおらへんよ。


　どこまでいっても

　「誰が→何を→どのように」

　の順番やで。

　読み手の思考と感情を
　想像しながら発信することやな。

　----------

　フットワークの軽さが
　習慣化するところまで、

　ノリと勢いを至上の決断基準に据えて
　動き続けば必ず人生変わるよ。


　数週間から数ヶ月もあれば
　行動様式は完全に書き換えられる。

　そのハードルを乗り越えた経験のある人が
　少ないだけで、

　やれば必ず習慣化できるんだわ。

　----------

　資源は有限だから一点集中、

　一つの分野で100点を取りにいき、
　実現させられれば自信もつくし、
　評価も高まる。


　そこで獲得した評価やご縁は
　雪だるま式に転がり、広がっていくから、

　そこまでいけば足りないところを
　他人に補ってもらえればいいし、

　補ってもらえるようになるんだわ。

　----------

　適性ある人なら
　『私の履歴書』登場目指して、

　会社人生全ツッパのフルスイングも
　アリやと思うのよ。

　正直、できることなら
　ワイもそうしたかった笑。


　しかし、適性のない人が
　苦行にのたうち回ってる姿もよく見かける。

　かくいうワイもそうやった。

　そんな人にとっての福音、救済、
　蜘蛛の糸がスモールビジネスよ。

　----------

　チマチマやってる限り、
　永久に変革は生まれんよ。

　気持ちもないまま
　無理してもうまくいかんけど、

　どこかで思い切って踏み出さないと
　現状から抜け出せないのも間違いない。


　だから、そこまで気持ちを
　引き上げてくれる人との出会いに
　価値があるんだわ。

　自分一人では
　そこまで高められんからね。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。



■ぶっちゃけ毎日、こんな事を考え、
　日記を書き続けたら

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
