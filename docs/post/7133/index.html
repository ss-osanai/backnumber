7133号 「乗りかかった船」効果を味方につけよ

■乗りかかった船、
　という言葉がありますね。

　（ネットで）意味を調べてみたところ、
　こんな説明がされていました。

　--------------

　《乗って岸を離れた船からは
　　下船できないところから》

　物事を始めてしまった以上、
　中途でやめるわけにはいかないことのたとえ。

　乗り掛かった船（のりかかったふね） の
　意味・使い方 - goo辞書

　--------------



■年間プログラムに参加する、とは、
　まさにこれに近い感じです。


　高いカネも払ったし、

　仲間がいて、
　変わると宣言してしまったし、

　応援されたり、
　協力もされてしまったし、

　今さら後戻りもできない。



■ときに弱気になることがあっても、
　結局

　「乗りかかった船」

　から降りることはできませんから、
　がむしゃらに前進するうちに、

　いつしか最初は小さく、やがて大きく、
　結果を出していかれることになる。

　コミットメント効果の後押しで。

　大きなイベントを立ち上げ、無事、
　開催までこぎつけられました。



■次はあなたの番ですよ。

　人生を変えたいと思ってもいない人は
　乗ったら後悔する船です。


　でも、自分の可能性を
　信じている方であれば、

　「船に乗る
　（年間プログラムに参加する）」

　意思決定をすることを
　検討してみられてはいかがでしょう。



■人生において、

　乗る船によっては

　【決断は一瞬、
　　波及効果は一生涯】

　となることすら、あるものです。

　※ただし、
　　船はあと数日で出航いたします。

　　どんな内容か、よく分からない」

　個別の相談会を設けています。
