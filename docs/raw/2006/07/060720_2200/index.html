　【　情報発信を行うことで思いもよらなかった展開が広がる　】

───────────────────────────────────



■電卓を叩いてみて、恐るべきことに気がつきました。
　少し、お付き合いください。



■今までに私はメールマガジンを1050日間、一日も休まずに
　配信を続けてきました。

　一口に1000号（1000日以上）といいますが、

　毎号、平均１時間の執筆時間で書き上げたとすると
　1000時間（＝約42日間）、

　平均２時間の執筆時間で計算してみると、
　2000時間（＝約83日間）

　飲まず食わず、そして睡眠もとらずに書き続けたのと同じだけ
　の力を注ぎ込んできたことになります。

　

■あわせて、純粋に執筆に費やした時間は上記の時間となりますが、

　その背後にはそのネタを収集するためのセミナー参加
　であったり、会食であったり、パーティであったり、
　読書であったりという「仕込みの時間」が存在しています。

　これらの時間までカウントすると膨大な時間が
　「平成・進化論。」の執筆に投入されてきたことが分かります。



■それくらいの力をいれて情報発信を続けているといろんな
　面白い現象に出会います。

　今日はその中でも、

　情報発信をしていく私の周囲で興味深い現象が立て続けに
　起きていることから説き起こしてシェアします。



■このメルマガを発行して初期の頃は数千人から、そして
　現在では15万人以上の方にご購読いただいているわけですが、

　情報発信をしていると狙ってもいなかった、考えていなかった、
　ような出会いが次々と生まれ、

　私が知恵をいくら絞っても決して浮かんでもこなかったで
　あろう、素晴らしいご提案をいただくことがあるのです。



■たとえば「情報発信が今のように簡単になる前」の時代を
　例にとって申し上げます。

　以前、私は直販セールス、チャネル（代理店）セールス、と
　ずっと営業畑におりました。

　その当時の営業手法としては狙いをすまして、
　
　この会社のトップに、
　あるいはあの会社のこの部署のこの担当者に、
　
　どうやって食い込んでいくか、とそんなことばかりを
　考えていたわけです。



■自分の出身校のネットワークを使ってみたり、師匠、先輩、
　友人知人のつながりをたどっていったり、直接、電話したり、
　メールしたり、手紙を出したり、パーティや懇親会で知り合った
　つながりを活用しようとしてみたり、

　その他、いろんな手段を使ってアプローチしていくわけですね。
　


■そして、その目的はただ一つ、そのトップなり担当者なりに
　○○というモノを売る。

　狙いも一人なら、目的も一つ。

　ここにはなかなか発展性はありませんし、次の展開が
　次々広がっていくということはほとんどありませんでした。



■ところが、いまはまるで違うプロセスを通って、どんどん
　新しい展開が生まれてくるのです。
　
　新展開が新展開を生む、とでもいえばよいのでしょうか。



■どんどん自分をさらけだし、情報発信していくと、
　その中である一定割合の方が私に対して興味を持って
　くださり、当社サービスの販売につながることもあります。

　あるいは提携先となることもあれば、新規事業の提案を頂くこと
　もあれば、その他思いもよらなかったような面白いお話が
　次から次へと持ち込まれるようになってきたのです。



■昔、リクルートで営業していたとき、
　「アントレ」という雑誌の広告販売を担当していたことが
　ありましたが、実はこのときに同じ経験をしました。



■ある商品の代理店を募集するための広告をクライアントが
　告知したとき、

　当初、目的としていたのはもちろん「代理店開拓」なわけ
　ですが、

　その「代理店開拓」のために、取り扱い商品を詳しく、
　こと細かに誌面で説明するわけです。



■すると、当然、代理店になりたい、という方からの資料請求や
　問い合わせが入るのですが、

　その合間合間に、こんな用途で使えないか、こんなところに
　販売することはできないか、という提案が次々に押し寄せて
　きたと、そのクライアントがおっしゃられるわけです。

　

■彼ら自身が想定していた用途とは違うものが情報発信をして
　「衆知を集める」ことによって、新たなビジネスチャンスを
　見つけることができたと喜ばれていたのが印象的でした。



■自ら所有する情報を不特定多数の他者にさらす、というのは
　結果として、

　自分の小さな、そして狭い考え方を覆す意見やアイディアを
　いただくための「魔法のランプ」となってくれることが
　往々にしてあります。



■わたしも、自らのメディアを持ち、毎日、恥をしのんで
　多くの情報をさらけ出すことで、誤りを指摘されたり、
　次につながる人を紹介いただいたり、あるいは自らの枠の中から
　はとても出てこなかったような、ビジネスを飛躍させるための
　情報を頂戴したり、

　と情報発信していなければ決して目にすることができなかった
　であろう、さまざまな出会いを毎日のように経験できる
　ようになってきたのです。



■まず、自らが結晶の核となって、その核の周囲に、
　「衆知をまぶしていく」というような感じでしょうか。

　「結晶の核」となるのは、勇気もいるし、ときに恥もさらすし、
　非難の対象となることもあるわけですが、
　
　それを差し引いても、素晴らしいリターンがあるわけです。



■逆に言うと、そのようなことを恐れて、情報発信する人が
　少ないからこそ、勇気を出して発信し始めた人に大きな
　リターンが配分される、といってもいいのかもしれません。



■需要と供給のバランスでみると、受信の需要は多いのに、
　発信の供給はまだまだ少ない、ということから考えても
　発信し始めたものから手厚いリターンを得ることができる
　といってもいいのかもしれません。

　オンラインにおいてある一定以上の規模で情報発信
　されていらっしゃる方であれば大いに頷かれることでしょう。



■「何もないから情報発信できない」のではなく、

　「何もないからこそ、（衆知を頂戴するために）情報発信を
　　していく」

　術を身につけることこそが、これからの時代における
　（不特定多数との）重要なコミュニケーションスキルになる
　ことは間違いないのではないでしょうか。
