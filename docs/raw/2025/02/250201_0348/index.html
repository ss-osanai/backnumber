■今日は

　「昨日の日記（の抜粋版）」

　の中から、いくつかを
　ピックアップしてご紹介いたします。

　ご参考にして頂けるところが
　一つか二つはあるのではないかと、、、


　（ここから）
　---------

　追い詰められて絶望してる状況でも、
　外から見れば大したことない場合も
　多いんよ。

　でも追い込まれると視野が狭くなり、
　思い込みが激しくなる。


　そんな時こそ自分視点のロックを解除し、
　客観的に状況を眺められる技術が
　必要になるんだわ。

　精神の安定を保てるし、
　より良い判断もできるようになる。

　----------

　四六時中、自分に問いかけ続ける人にだけ
　開かれる扉があるんだわ。

　特に

　「幸せ」「自由」「充実」「豊かさ」

　みたいな、あいまいな目標こそ
　問い続けることが大事やで。


　自問しなければ、定義が定まらず、求めようも
　ないから気づかぬうちにどんどん遠ざかる。

　人生の盲点と言えるわな。

　----------

　人生を安定させる秘訣は、自身に固有の
　「行動規範」を確立することよ。

　これにより迷いやブレがなくなり、
　心の平安が訪れる。

　ただそんな規範は簡単には手に入らん。


　毎日

　「人生とは？」
　「幸せとは？」

　と問い続け、
　膨大な自問を重ねる必要があるんだわ。

　地道な営みの延長線上に、
　揺るがぬ信念が育まれる。

　----------

　仕事好きで、
　よく働く奴だけが生き残るという真実。

　小手先ノウハウで短期的成功は収められても、
　長期で見ると水が高きから低きに流れるように、

　物理法則にしたがい、落ち着くべきところに
　落ち着くのは当然のことよな。

　----------

　「人生」「幸せ」について
　正面から考えることを避けとらん？

　「そんな難しいこと、考えんでええやん」
　思うかもしれんな。

　でも、この問いを立てなければ、
　人生を豊かにする方法や、
　幸せになる道筋を考える機会が生まれない。

　結局、漫然と時を重ね、
　後悔する人生になってしまいかねん。

　----------

　人は本当のことを言われたときに
　腹を立てる。

　故・小室直樹先生の名言。

　腹を立ててる人を見たら
　「言われたことが図星やったんやな」
　と思えばいい。

　逆に、自分が立腹したときには、
　こうした観点から振り返り、
　反省や向上、改善の材料とすると良い。

　----------

　変化を願う人に伝えたい
　「百聞は一見に如かず」という言葉。

　頭でわかれば全てができる、
　その考えは大間違い。

　「頭でわかる」と「体でわかる」は、
　まったく別物なんよ。


　人や現場に触れることなく、知識だけ
　入れても腹底からの欲求や動機は生まれん。

　ゆえに行動に移れず、
　結局、何も変わらない。

　----------

　勤務先がいつ潰れても動じない程度の
　稼ぐ力と蓄えのいずれか、

　理想的には両方を手に入れておくべき。

　想定外の事態で、
　突如会社が傾くこともあれば、

　突然リストラの憂き目に遭うことも
　日常化しつつあるしな。
　
　----------

　『失敗の本質』の姉妹編、
　『戦略の本質』の終章、

　最後のまとめ的な一節で
　フロネシス（賢慮）について語られている。

　戦史、ならびに戦略論の愛好家のみならず、
　実践知について理解を深めたい人にも
　必読の本やと思ってる。

　終章を読むだけでも震えるで。

　----------

　時間を売ってカネに変える仕事をしている人は
　本気で考え、備えておくべき命題よ。

　目指すべき最終形態は
　ビジネスの仕組み化（非・肉体労働化）と
　運用収益の二本立てで、

　勝手に資産が増えていく世界。

　----------

　ワイ、勤務先の倒産経験者なのよ。

　全国紙全ての一面トップを飾り、
　株式市場に影響を及ぼすほどには
　ド派手な倒産やったで。

　ある日突然、報道で自社が潰れたのを
　知ったんや笑。


　だからワイが、会社潰れるいうてるのは
　煽ってるわけやなく、実体験からの話やで。

　説得力しかない忠告やからな。

　----------

　ビジネスで一番大事なんは
　見込み客集めやで。

　こうした活動をバカにしたり、
　一段下に見る「お利口さん」が多いけど、

　みんなそこでつまづき、
　カネが回らず廃業に至るんよ。


　マーケ＆営業を笑う者は
　マーケ＆営業に泣くんだわ。

　むしろこれさえできれば極論、何でも
　売れるからマジで食いっぱぐれなくなるよ。

　----------

　他人を気にしすぎるから病むんだわ。

　30年後の理想の自分に近づくことだけ考えたら
　他人の目とか気にならなくなるよ。

　人生はマラソン、と理解できれば、
　細かなことはマジでどーでもよくなる。

　ただ、頂（いただき）だけを見とけば良い。

　----------

　一応補足しとくと、
　他人を気にするなということではなく、

　圧倒的に高い基準を掲げていれば、
　他人の目なんかマジでどーでもよくなる、
　という話。

　因果の順序の理解が大事。

　----------

　書き手の価値観、性格、能力等、
　すべて晒されるのが文章やで。

　ワイも発信する言葉を通して、それら
　全てが見られてると思いながら書いている。


　見る人が見れば
　全部モロバレやからキツいけど、

　あえて自らをそんな環境に置いて
　成長したいと思ってる。

　尊敬する人に読まれてるつもりで書いてるよ。

　----------

　正直言うと、
　どうしたらお金を儲けられるかより、

　どうすれば幸せになれるかの方に、
　昔から関心がある。

　ただ幸せになるには、
　（キレイごと抜きに）

　稼ぐ力はどうしても必要、
　という結論に至ったわけ。


　少なくとも、不幸を回避したり
　ダメージ軽減にもお金は要るからね。

　だから真剣にビジネスに取り組んだ。

　----------

　2回り、3回り上の、
　人生の大先輩とのお付き合いを通して

　数多くの知識や知恵を
　伝えて頂いてきたのはもちろんだが、

　究極的には

　「人はどのように老い、
　　そして逝くのか」

　について、身をもって
　教えてもらってきたのではないか、

　と今にして思うのよ。

　----------

　どれだけお金が儲かるとしても、
　やっている仕事に社会的意義を
　感じられなきゃ幸せとは言えんよ。

　仕事自体に意味と喜びを見いだせたら、

　「散財や豪遊で（かりそめの）幸せを買う」

　的な発想にはならんから、
　実際問題、大してお金を使う機会もなくなり、

　生活基盤も盤石になるんだわ。

　----------

　人生は数十年後の未来をどこに定めるかで、
　おおむね決まるのよ。

　意識すべきは自身の理想の未来像と
　現状の間に横たわるギャップであって、

　少なくとも目の前のことだけを追いかけ、
　走り回っている他人

　（ごく一部、例外はあるけど、
　　ほとんどといってよい）

　ではない。

　----------

　先に、金の不安を解消するための
　仕事から入るのは間違いではない。

　生活に困窮してたら、
　次の一手を考えられんからな。


　ただし、ある程度の余裕ができたら、

　仕事の意義や喜びを犠牲にしてでも
　更なる金儲けに走るのか、

　幸せになるための仕事にシフトするのか、

　立ち止まって
　真剣に考えるべきやと思ってる。

　----------

　経済的自立なしに
　まともな社会生活は送れんから、

　まず基盤を整えることが大事やで。

　理想の追求を大切にしたいからこそ
　マズローの欲求5段階説を思い出せ、
　いう話なんよ。

　----------

　正直、金儲けに全集中してれば今頃、
　ケタ違いの金持ちになってたと思ってる。

　ただ実際は、人生/人間/社会/文化等を
　学ぶ時間を相当天引きしてきて、

　仕事メインの生活ではなかったからな。


　そういう目標設定をして、
　その道筋どおりに生きてきたという意味で、

　これまでの足跡を全肯定してる。

　----------

　仕事や資産構築と、それ以外の営みには
　順序を付けたほうがいいよ。

　不安定は不安や焦りを招くから、
　まずは経済的基盤を優先して整えるべき。

　その後、人生かけて
　探求したいテーマに進むと良い。


　順番が逆になると経済的困窮から
　歩みの途中で行き詰まる可能性が高くなる。

　人生、生き永らえてこそ。

　----------

　最初は大それたビジネスプランとか
　立てなくていいよ。

　月に1～2万円、
　稼ぐところからのスタートでOK。


　そこで身の回りの小さなニーズと
　課題解決の方法を発見/提供できれば、

　より深く掘る、あるいは周辺に手を広げることで
　10万20万円くらいは稼げるようになる。

　このあたりで商売の基本が見えてくる。

　----------

　お金儲けは一足飛びにはいかないよ。

　小さな経験を重ね、数々の失敗を繰り返しながら
　知識を増やし、知恵を身につけていくんだわ。


　時間はかかるけど、一度手に入れた
　無形の財産は失われないから、

　やるべきことをやってる人は時間とともに
　徐々に頭角を現し始めるよ。

　----------

　年商や従業員数、利益を誇る
　経営者と出会うと、

　ビジネス初級者は
　それだけで圧倒されることがあるよな。

　でも、それと人間性や人の価値とは
　全く関係ないからな。


　そんなものを唯一の尺度としてしか
　人を判断できない人間があれば、

　その程度の人物に過ぎんのだから、
　あなたの方から願い下げればいい。

　----------

　感受性の豊かな人と付き合いたいと
　思ってるのよね。

　ワイも経営者やけど、金儲けしか興味のない
　経営者とは昔から一切、付き合いがないし、

　付き合いたいと考えたこともない。


　経営者である前に人でありたいからな。

　尊敬できる経営者との、
　若い頃からの付き合いを通して、

　そんな精神が涵養されたのよ。

　----------

　多動系こそチャンスありやで。

　上手くいかないのは時間管理システムに不備があり
　思いつくこと全てをやり切れんからよ。


　仕組みをアップデートし、思いついたことを
　全部やり切れるようになればブチ抜けられる。

　発想豊かで脳が常にゴチャゴチャしとる人は、
　出力のボトルネック解消さえできれば無敵やで。

　----------

　能力高いのに斜に構えてる人いるやん。
　あれ本当にもったいないよな。

　達観、超越のそぶりを外に現してると
　本人は思ってるんやろうけど、

　周りからすれば単に扱いづらい人に
　過ぎないんだわ。


　多少能力低くても、
　いつも笑顔で素直な人が可愛がられ、

　機会を得て、幸せになれる真実は
　もっと世界に知られるべき。

　----------

　斜に構え、ひねくれた目線で
　人や社会と向き合ってる人は、

　他人から面倒がられ、忌避されがちな
　姿勢自体を客観的に捉えられていないよな。


　その他の面でどれだけ有能でも、

　一番大切な土台が抜けているのだから
　根本的な意味で無能、

　といえそうやね。

　----------

　こちらから売り込むより、向こうから
　求められる方が100倍簡単に売れるだろ。

　楽ちんな上にストレスフリー。
　キャリアも恋愛も一緒。

　誘われる方がどう考えて楽なんよ。


　どうすれば売れるかという問いを封印し、
　どうすれば求められるかだけを考えよ、
　いうこと。

　営業や恋愛における
　コペルニクス的転回よ。

　----------

　ワイ、昔から短期記憶能力が
　絶望的に低くて、

　冗談抜きで
　３秒経ったら全部揮発する感じなのよ。


　致命的弱点と思ってたけど対策として、
　思いついたアイデアはその場で全てメモに残す。

　ついでに着想を膨らませた文章まで
　記述するようにしたら、

　学びの歩留まりが劇的に向上したんだわ。
　災い転じて福となす。

　----------

　バカにされたとき、

　「そうなんす、バカなんす」

　と認められるか否かで
　幸福度が決まるんよ。

　認められる人は幸福度高し。


　ムキになって
　反論せずにはいられない人は

　「半端なプライド×自信のなさ」

　が原因で、幸福度は爆下がり。

　強みを磨き、盤石の自信を築いた上で、
　柳のようにしなやかに、がポイントよ。

　----------

　誰かが昔、朝令暮改は善である、
　と言ってたけど、その通りやな。

　改むるにはばかることなかれ、
　という言葉もあるけど、

　「過去を改める＝改善」なんよ。
　だから毎日、朝令暮改が正解よ。

　君子豹変こそが超速進化の秘訣。

　反応を観察しながら、
　変えて変えて変えまくるんやで。

　一刻たりとも留まることなかれ。

　----------

　昔、仲良い先輩社長と

　「経営者は皆、ドＭ」

　という話で盛り上がったんよ。

　これはもう、間違いないな。
　キツければキツイほどテンション上がる。

　制約があればあるほど興奮する。


　縛ってくれる人がなければ
　セルフ緊縛までおっ始める。

　追い込まれることでポテンシャルを
　開放できる人は経営者に向いとるよ。

　----------

　蹴落とす必要なんかないのに、
　他人に意識を向けてる時点で
　その人の未来はお察しよ。

　目線が低すぎる。

　長く商売する中で
　そういう人にも出会ってきたけど、
　マジでただの一人も生き残っとらんな。

　貴重なエネルギーは100%、
　自分のために使った方が得するよ。

　----------

　能力高いが扱いにくい人より、

　底辺レベルでも好奇心と向上心のある、
　素直で一緒にいて居心地いい人間は、

　年長者に可愛がられ、引き上げられ、
　紹介され、任され、
　なんだかんだ能力上がってくるんだわ。

　前者は10年経っても変わらんから、

　ある時期を境に、バカにしてた奴に
　どんどん抜かれていくんだよ。

　----------

　カネで解決できる悩みがあるのは本当。

　そしてその数は思われているより、
　はるかに多いのよ。

　さらに、ことが起きる前に
　カネの力で備えも分厚くできる。


　だからといって金さえあれば
　幸せになれるわけではないので、

　自分なりの思想/哲学/道の確立が急がれる、
　そう思って生きてきた。

　----------

　嫉妬されるのは同類と思われてるから。
　ブチ抜ければ嫉妬されなくなるよ。

　同類認定からブチ抜けまでの過渡期が、
　足を引っ張られて一番大変。

　あちらの世界にどうしても行きたいなら、

　くだらん引力を振り切って

　「限界突破×サバイバー」

　方式で行くしかあらへんで。

　----------

　バカにされても笑顔で受け流し、内心

　「そもそもお前なんか、
　　ハナから相手にしとらんわ」

　という気位と目線の高さを持っておけばいい。


　ムダな精神エネルギーの損耗なく、
　投じるべき場所に全て叩き込むためにも

　顔で笑って心で憐憫＆黙殺が一番よ。

　そんな人間はどうせすぐに、
　目の前から消えていく。

　----------
　（ここまで）


　
■以上、

　人生やビジネスについて
　真面目に考えている人であれば、

　（さらには
　　一を聞いて十を知る人であれば）

　なにか気づいて貰えるところも
　あるのではないかと思い、

　紹介させてもらいました。

　【圧倒的な進化】
　
　を遂げるに決まってると
　思ってます。

　あなたもぜひ。


　今日も人生とビジネスを
　楽しもうやで！
