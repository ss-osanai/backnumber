＜今日のピークパフォーマンス方程式＞

 【  沈（ちん）するな！沈起こしは大変！  】

───────────────────────────────────

　

■歴は浅いですが、

　昨年から今年にかけて
　ヨット練習に力を入れてきました。

　今日は、そこからの学びのご紹介。



■ヨットは、いったん
　
　「沈」（「ちん」と読みます、転覆すること）
　
　すると、起こすのが大変です。


　練習中、何回も
　
　「沈（ちん）」
　
　しては、
　
　「沈（ちん）起こし」
　
　するたびに、つくづく思います。


　
■強い風に吹き倒されたら、後が大変です。

　前進の機会が失われ、

　いったん休止して、船を起こすまでの時間は、
　非常にロスが大きいのです。

　しかも強風で高い波の日は、
　なかなか船を起こせず、本当に萎えます。

　（特に寒い日、、）



■やり始めたばかりのときは、

　「沈起こし（ちんおこし）」

　のために、

　どれほどヒジやらヒザやら太ももやらが、
　青タン、赤タンだらけになったものか。



■けれども、強風、高波に耐え、

　何とか船を倒さず、
　ぎりぎりでも踏みとどまることができれば、

　そこから立て直し、
　再び帆走を続けることができるのです。



■こんな経験を何回も、
　というか、何十回、何百回も重ねていくと、

　「人生も同じようなものだ」
　
　と思えてくる不思議（笑）


　生きている間には、いろいろと、

　
　▼辛いこと

　▼大変なこと

　▼萎えそうになること

　▼折れそうになること


　があります。



■けれども、

　そんなときでも、
　ひとたび吹き倒されたら（折れてしまったら）

　再起するのが大変です。


　あたかも、強風で波の高い日に

　「沈（ちん）」

　を起こしてしまったようなもの。


　そこからヨットを起こして、
　さらに帆走を始めようと思っても、

　なかなか思い通りに起きあがってはくれません。


　＜こんな感じ、、↓（私じゃないっすよｗ）＞
　
　https://www.youtube.com/watch?v=2SQ114LoJ68



■人生においては
　こんなことにならぬよう（！？）

　どれだけ大変なことがあったとしても、
　決して船を倒さず、

　（＝心を折らず）

　心を前向きに持ち、
　行動し続ける必要があるのです。


　以上、ヨットを通じて学んだこと。
