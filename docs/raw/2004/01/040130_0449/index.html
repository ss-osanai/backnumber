＜　昨日の活動ダイジェスト　九鬼さんと久しぶりのランチ。　＞
───────────────────────────────────


■昨日は、和歌山の九鬼政人さんとランチ。
　朝、遅くまで寝ていたら急に電話がかかってきて、一緒にランチを
　とらないか？と。

　実は九鬼さん、仕事を完全に軌道に乗せ、セミリタイア中の経営者。
　今回は所用で東京に来られていたのです。いつも、電話は、直前に。
　でも、有難いお話なので、都合が付く限り、伺うことにしています。
　そしてご一緒させていただくと、斬新な発想を頂戴できるのです。


■ばたばたしてて貧乏なのをバタビンというそうですが、そんな状況とは
　無縁の、好きなことだけをして過ごす、という状態にあります。

　経営にはいくつか型のようなものがあるのでしょうが、経営者としての
　いくつかある、理想スタイルの一つなのかもしれません。

　１時間ほどでしたが、今度出される新刊書籍について。また、その他、
　いろいろと勉強になる話を聞かせていただきました。

　実はメルマガだされてます。どんな内容についてかは、こちらから。
　→ http://www.mag2.com/m/0000120044.htm
　

■その後、平野先生と、もうお一方と打ち合わせ。
　いつもの、京王プラザホテルにて。
　今後の方向性について、３人で盛り上がりました。

＜今日のピークパフォーマンス方程式＞

　【　　　夢中になる。　　　】

───────────────────────────────────


■小学生低学年の時は、すべてのことに対して夢中だった憶えが
　あります。

　毎日、夕方暗くなるまで、友達と近所に流れる淀川の河川敷で駈けずり
　まわって遊んだ記憶があります。

　それが、大人になるにしたがって「夢中になる」ことを醒めた目で
　捉える人が現れるようになってくるようです。


■でも、夢中になって、何が悪い、と私は思います。
　夢中になれる、というのは幸せなことです。

　後ろ向きな事にエネルギーを使わなくても済むから。
　夢中になっているときに、マイナスの発想は出てこないから。
　毎日、楽しく過ごすことができます。
　
　
■数日前に、このメルマガで書いた「ご安全に」の話。メールや、直接
　会った人や、楽天日記の掲示板や私書箱などに、たくさんの同意の
　ご反響を頂きました。

　肯定形に自分が染まると、否定形の文脈で生き方を捉えることは一切
　なくなります。
　徹底した、明るい未来志向、これが本当に明るい未来を呼び込みます。
　逆もまた真なり。

　否定的な、悲観的なことを考え続けると、不思議なもので（本当は
　不思議でもなんでもないのですが）現実もその通りになるものです。


■常に夢中、すなわち、夢の中にあれば、否定的な生き方には決して
　ならないはず。


■こういう話をすると、その夢が見つからないんだ、という人がいます。
　でも、夢が見つかるまで待っていても、おそらく夢が向こうから
　やってくることはないでしょう。


■まず、手始めに出来ることを、何か一つでもやっていくところから、
　小さな波及効果が始まり、やがては大きな渦を巻き起こしていく
　のです。

　平野先生から聞いたのですが、渦に近寄る人になるのではなく、
　どんなに小さくてもいいから自ら渦を作り出せ、という言葉。
　本当にその通りだと思います。

 http://www.amazon.co.jp/exec/obidos/ASIN/4106002213/2ndstagejp-22


■もう、何年も前にお亡くなりになられましたが、京都大学の名誉教授で
　あった高坂先生の書かれた名著です。

　日々の活動、行動に忙しく、近視眼的な視野で動き尽くめに動いている
　のが現代人。
　そしてますます、その流れが加速しているようにも思えます。
　かくいう私自身も反省しなければならない点だと思っているのですが。

　そんなときに、時空を超えて大局的な見地から物事を見る。
　そんな視点を時には持つ必要があるように思います。

　
■たとえば、今回、ご紹介する「文明が衰亡するとき」。
　文明の栄枯盛衰について、ローマ帝国、ヴェネツィア、（当時に
　おける）現代アメリカの苦悩、といった例を通し、描かれています。

　歴史は繰り返す、ではないけれど、数十年の単位で見えなかったものが
　数百年の単位で俯瞰すると、見えてくるものがあります。

　たとえば、現代日本。いかに不況だといわれても、ブランドショップの
　新規オープンと聞くと、アリが群がるが如くに、我先にと駆けつける。
　（このたとえ、失礼ならごめんなさい）こんな恵まれた時代、国家は
　そうそうありません。

　そして、こんな状態が永続したためしがあったかどうかは、
　ちょっと世界史をひもといてみればよい。

　
■戦後６０年、たぐい稀なる僥倖に恵まれていただけで、実は砂上の楼閣
　であるのかもしれません。いや、恐らくそうなのでしょう。

　そんなことを、考えながらこの本を読み進めておりました。
　たまに、こういう本を見返してみないと、世の中の本質というものが
　見えなくなってしまう、そんな気がします。


■恵まれている現在に感謝し、又いつ、何が起こるか分からない世の
　中であることを認識し、今を夢中にいきていきたいと思います。
