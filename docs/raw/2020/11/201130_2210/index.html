　今の私は自尊心が弱まっている、

　内心

　「自分ならこの程度のことは
　　できて当たり前だ」

　という根拠のない自信があり、

　根拠はなくとも結果をその思いに
　追いつかせるべく行動していました。

　あの頃に言われたような人間だったのか、

　結局今の状態に陥っているのは
　当然の結果なのかと考えていました。

　気に留めていないような言葉のはずが、

　自分が、出来ると思えるかどうか、
　本当に出来ると思えればできるし、

　心の底の方で出来ないと思っていれば
　出来ません。


　そういう意味では、
　人生は思い通りになります。


　東京に行ったことがなくても、
　新幹線で東京に行くことは出来ます。

　時刻表を見れば、
　東京に着く時間もわかります。


　エヴェレストに登頂することは
　可能でしょうか。

　もちろん、過去に登頂された人はいます。


　しかし、現段階では、
　体も鍛えていませんし、
　やり方もわかりません。

　予算がどれだけいるのかも、
　見当がつきません。

　よって、出来ないということになります。


　自分が目指したい目標を、
　いかに出来るとイメージするか。


　調べて計算して設計して、物理的に
　出来ると思えるのもオッケーですし、

　出来るという単なる思い込みでも
　良いと思います。

　心の底から、出来ると思えたなら、
　どんなことでも、必ず出来ると思います。


　しかし、やったことがないことは
　イメージできませんから、

　出来る方法、順序を
　勉強しないと進みません。

　脳は現実か非現実かを区別しません。


　スポーツのイメージトレーニングのように、

　自分で好き勝手に物語の主人公になって、
　どんどん出来る人になることも可能です。


　（伊賀崎 靖 さま）
