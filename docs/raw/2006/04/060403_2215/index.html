　【　ビジネスは売り手と買い手の関係性を育てる有機農法　】

───────────────────────────────────



■ビジネスの本質は、関係づくり。
　昨日はそんな話をしましたが、今日はその続きです。

　ビジネス系メルマガの範疇（はんちゅう）に入る「平成・進化論。」
　を読んでいらっしゃるあなたは、

　今まさにビジネスに携わっておられるか、少なくともビジネスの世界に
　ご関心のある方ばかりであろうことは、想像に難くありません。

　では、そんなあなたにご質問です。



■「ビジネスとはいったい、何ですか？」

　そう聞かれたら、あなたはどう答えられるでしょうか。



■「ビジネスとは金儲けである」

　「ビジネスとは自己実現である」

　「ビジネスとは社会貢献である」

　「ビジネスとは価値の創出である」


　これらは、どれも正しいでしょう。正解はないと思います。

　さまざまな定義が考えられる中で、わたしがひとつ
　お伝えしたいことは、昨日からお話ししている

 「ビジネスとは関係づくりである」ということなのです。



■世の中にはさまざまなビジネスがあります。

　安価な値段の商品、非常に利益率の低いビジネスがある一方で、
　高額な商品、高い利益率を生み出すビジネスも存在します。

　その違いを生み出すうえ生み出すものは何なのか？

　その一つは、


　【 売り手と買い手の関係性 】


　ではないのでしょうか。



■いってみればそれは有機栽培のごときものかもしれません。

　虫も寄りつかず、ほうっておいたら作物もとれない畑に
　化学肥料を大量投入して、痩せた（やせた）土地から強制的に収穫する
　のもひとつの農法ではあるでしょう。

　しかしそれでは、ただでさえ痩せた土地がますます衰えていきます。

　そこで有機農法では、落ち葉や動物の死骸、フンを畑に与えて
　土づくりから始める。

　有機農法にこだわる人は、
　作物をつくるのではなく、土をつくることに注力するのです。



■ビジネスもまた有機農法に徹することで「肥沃（ひよく）なビジネス」
　に育ちます。

　収穫をあせった畑では、一見、豊作なように見えても
　栄養の乏しい作物がとれるだけ。

　あなたとお客さまとの関係という肥沃な土壌から芽生えてくるもの、
　それがビジネスです。

　土づくりをおろそかにして収穫を急ぐのは得策とはいえません。



■場合によっては即席で栽培する必要性が生じることもあるの
　かもしれませんが、

　基本的には、ビジネスは有機農法でいくべきものです。

　絶対にやってはいけないのは、客を刈り取ると称して
　悪徳、悪質な焼き畑農法を展開すること。

　そのようなことはビジネスに携わる者としては禁じ手です。



■モノを売る前に、関係づくりこそ重要。

　その関係の中から、熟柿（じゅくし）がポトリ、ポトリと落ちるが
　ごとく、結果としてビジネスになっている、

　というのが美しい姿なのでしょう。
