＜今日のピークパフォーマンス方程式。＞

 【　飽きずにひたすらインプットしつづけるには　】

───────────────────────────────────



■何かを学び、習得するには一定の期間を要します。
　
　どんなジャンルの勉強でも、
　

　【 飽きずにひたすらインプットしつづけること 】


　が大切です。



■そこで効率を上げるために、
　期間を区切って１つのジャンルに集中するという方法が
　あります。

　「よし、むこう６ヶ月はマーケティングの勉強をしよう」

　「１年で心理学を学ぼう」など

　期間ごとの目標を立てるのです。



■ところが、皮肉なことに、

　意識を集中させようとして、
　毎日ひたすら同じジャンルに関するインプットをつづけると、

　かえって集中力が低下しやすいように感じるのは
　私だけでしょうか。



■どんな学びでも、まずは前提として
　

　【人の心はうつろいやすい】


　ということを理解しておく必要がありますね。


　集中力をそがれるのだとしたら、

　それはあなたの心がうつろい、興味・関心が弱まった表れ
　なのでしょう。



■人によって、学びたいジャンルはさまざまです。


　そして、同じ１人の人間でも、
　

　「心理学」「マーケティング」「文章術」「財務諸表の読み方」
　「キャリア形成」「資産運用」「コミュニケーション」「勉強法」
　「歴史」「哲学」「スピーチ」「ビジネスモデル」「特許」
　「ITリテラシー」「マネジメント」「世界経済」「創造力」
　「営業」「護身術」「リスク管理」「健康維持」「トレーニング」
　「水問題」「環境問題」「食料問題」「為替動向」「脳」


　などなど、いくつものジャンルに興味関心があるはずです。


　※ちなみにこれらはすべて、いま私の机の上に置いてある、

　　「近日中に読むべき本の山」にある本のジャンルを
　　ずらずらと挙げました。



■個人的な経験則ではありますが、

　（たとえば）４つのジャンルを３ヶ月ずつ集中して勉強するよりも、

　４つのジャンルの勉強に一度に着手し、
　「その時々の自分の心の状態に合った内容」を
　そのときの気分にあわせて勉強していったほうが、

　どんどんインプットしつづけることができるはずです。



■同じ12ヶ月でも、

　期間限定で集中しようとするより、

　つねに興味を失わないでインプットしつづけたときのほうが、
　学びの総量は飛躍的にグッと大きくなります。



■資格試験を受けるときなど、
　期限が決まっている場合は話が別ですが、

　ビジネスの現場で生かしたい学びは、
　欲張って一度にいくつも取り組み、つねに興味を失わずに
　インプットしつづけていきたいものです。


　今日もビジネスを楽しみましょう。
