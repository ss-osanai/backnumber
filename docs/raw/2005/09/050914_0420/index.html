■「平成・進化論。」はビジネス系メールマガジンであり、

　気になることがあり、取り上げさせていただきます。

　いままでも、現在も。

　わざわざ、少ない可処分時間の中から捻出して、

　でしたが、

　それでも、と思った候補者と政党に投票いたしました。

　が必然的に増大することとなります。


■国民が政治に無関心になればなるほど、投票率の低下を招き、

　それが、ある意図を持った、特定団体への利益誘導や、権力の集中、
　権限の集積につながり、

　投票所に足を運ばなければなりません。

　【　決断力は訓練することで上達する。 】

───────────────────────────────────


■きのうもお話ししましたが、
　
　私たちは毎日生きているだけで何十、何百という決断を
　迫られています。


■決断をしなければ、ものごとは進みません。

　決断をするにあたって優柔不断であるということは、
　すなわち、

　ものごとを進めることを知らず知らずのうちに引きのばして
　いることにほかなりません。


■なにかを決断したら、とうぜん、その次には行動が待って
　います。

　決断のスピードを速め、多くの決断をこなすということは、

　多くの行動を起こすことと同じなのです。


■この決断力、意識さえしていれば、鍛えることはそれほど
　むずかしいことではありません。

　いってみれば、一種のクセみたいなものです。

　決断力を鍛えるには、どんどん、決断の回数をふやす。
　前倒しで決断する。即断即決するのです。


■たとえば朝、目覚ましがなったら、眠くても起きるか、
　スヌーズ機能をつかうのか。

　まずこれが、一日の最初の決断項目であるわけですが、
　私の場合、この決断は

　「問答無用で、気合いで跳び起きる」

　です。


■まずは、そんな小さなことから即断する習慣をつける。

　そこからはじめてみられたらいいのです。

　そんなことすらできないのに、もっと大きな決断ができるはず
　もありません。


■どんな小さなことでもいいですから、決断をふやして
　いきましょう。

　決断をふやすということは、結果的に行動をふやすことにも
　なります。

　行動すれば、うまくいくこともあるし、いかないことも
　あります。

　なら、うまくいったものは継続すればいいし、失敗したなら
　二度とその決断をしなければいいだけのこと。


■このように、決断の回数をこなすほどに、経験が積み重なり、
　ミスを犯す可能性がどんどん低くなっていきます。

　「決断の精度」があがっていくのです。


■最初は、いくら失敗してもいい。

　まずは、小さな決断をふやし、小さいことを即断即決してみては
　いかがでしょうか。

　そうやって鍛えた決断力が、やがてあなたに大きな結果を
　もたらすことになります。


■目の前の小さなことを、やるか、やらないのか、瞬時に見分け、
　決断したなら躊躇せずに飛び込みましょう。

　それが実は、ピークパフォーマンスを発揮するための最高の近道
　なのでしょう。
