　いつもお伝えしていることですが、
　こちらの音源を聴くことなしに、

　せっかく貴重な時間を割いて、
　メルマガをお読み頂いても、

　【効果は半減どころか、十分の一以下】
　
　と思っています。


　今回の企画をきっかけとして、
　大勢の方にお聴き頂くことができたこと、

　とても嬉しく思っております。



■話は少し変わりますが、


　数日前、メルマガでご紹介した
　パリ駐在のＫさんのブログ記事（↓）

　http://blog.livedoor.jp/katonobuyukidesu/archives/33156016.html
　　

　に上記音源をお聴き下さった感想が
　記されていたので、勝手に（汗）、

　一部を抜粋して、ご紹介いたします。


　（ここから）
　---------------------------------------------
　

　七年前くらいのことだと思います。
　

　身銭を切って、自分としては高額でしたが、

　当時（別記事で音源について紹介した）鮒谷周史さんが
　行なっていた3日間セミナーに参加しました。


　その中で２時間程度の時間を取り、
　自分のありたい姿を書きまくったのですが、

　「目標設定」について学び直している中で、

　その時のことを思い出し、数年ぶりに
　自分が書いた目標を読み返してみたのです。


　実にびっくり！！

　そこには、海外で働くことも、３つ星レストランやら
　５つ星レストランに慣れるということも、

　ブルゴーニュやボルドーの視察を複数回すること、
　自由に使えるお金の額等々、

　ここ数年で実現して来たことが
　相当な割合で書いてあったのです。


　当時の加藤は東京にいましたし、

　今とは違う職業だったので、
　いわゆる逆算思考で書いたわけではありません。

　こうなったらいいなぁという願望を
　精一杯のところまで膨らませて書いただけのこと。


　前に目標を見たのは2015年のことでしたから、
　すっかり忘れておりましたが、

　一度設定した目標は
　潜在意識に刻まれているのでしょうか。

　不思議なもので、
　多くが叶っていたというわけです。


　みなさん、海外で働きたかったら、
　ガチで目標設定するといいかもしれません！
　
　（なんだか怪しい感じに
　　聞こえちゃいそうですが笑）。


　でも、結果論的に言えば、書けば叶うという
　よく聞く言葉は事実なのでしょう。


　---------------------------------------------
　（ここまで）


　まさにＫさんがお書き下さった通り、

　目標を設定するとは、
　こういうことなんですよね。

　書いた当人も信じられないことが
　時間をおいて、次々と実現していくのです。


　（私（鮒谷）もまったく同じ経験をしました。
　　そして今も、リアルタイムで継続中、、）

　

■音源をお聴きくださった方は
　すでにお分かりの通り、

　
　私たちは毎日、

　▼今、起きるか、二度寝するか

　に始まり、


　▼昼ごはんは食べるのか、食べないのか、
　　食べるとすれば何を食べるのか

　▼通勤・通学のスキマ時間に本を読むのか、
　　スマホゲームをするのか

　▼お客さまに価値を提供しようと仕事に
　　向き合うのか、惰性で作業に取り組むのか

　▼気になる言葉を見つけたときに
　　その場で調べるのか、面倒だとスルーするのか

　等々、

　このレベルでの意思決定は、
　おそらく日々、何十どころではなく、

　【時々刻々、最低でも一日、
　　数百個の単位で】

　行っているはずです。



■そして恐ろしいのは、

　【目標を立てていない人は、自分が惰性で
　　意思決定したという自覚もなく流されている】

　という現実。

　そしてその事実にも気づいていないのです。



■こうして目標を持たぬ人は、

　「桃が上流から下流に、
　　ただ流れ流されていくが如くに」

　目の前に迫られている意思決定を、
　無自覚に、楽な方へ、楽な方へと行うから、

　時間の経過とともに

　【人生が「詰む」方向に向かっていく】
　
　のも、むべなるかな。



■反対に、
　明確に未来の目標を立てている人は、

　【現在の、時々刻々の意思決定によって
　　人生がどんどん枝分かれしていく】

　ことがハッキリと分かっているから、
　真剣に決断し、真剣に行動します。


　【意思決定のたびに人生の分岐点にいる】

　ことを強制的に意識させられるのですから、
　当然です。



■もし私（鮒谷）が間違ったことを
　言っていたらごめんなさい。

　でも、そんなに違ったことを言っている
　とは思えないんですよね。


　仮に、私たちが毎日、
　300個の意思決定をしているとして、

　そのうちの幾ばくかでも、
　こうすれば目標に近づけるかも、

　と思う方向に決断を下し、
　行動をすればどうなると思いますか。
　

　
■一つ一つは微細な変化しかもたらさなくとも、

　何十回、何千回、何万回と
　ごくごく小さな意思決定を重ねていくうちに、

　あたかもタイタニックのように
　すぐには方向転換できない私たちの人生が、

　ゆっくりと、やがては急激に、
　変化・変容し始めるときが来るのです。



■時間をかけて、こうした強烈な変化を
　もたらしてくれるのが、

　【正しく目標を設定することから
　　生まれる力】

　です。


　上述のＫさんは、

　そこまでハッキリと目標を意識し続けられた
　わけでもないかもしれませんが、

　それでさえ「無意識」が目標を
　自動追尾し続け、

　やがては「撃墜（目標達成）」された、
　といって良いでしょう。



■私自身も含め、
　こうした事例を無数に見てきたからこそ、

　【目標を設定せずして、
　　人生を変えられるはずなどない】

　と強調したいのです。


　だからこそ、以下の音源を、
　しつこくしつこく、ご案内してきたつもりです。
