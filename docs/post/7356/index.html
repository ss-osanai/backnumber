7356号 そんなにいうなら、お前がやってみろ

■どんな仕事にも「顧客」が存在します。


　顧客は社外に存在するのみならず、

　社内においても、関連する部署や
　担当者を顧客だと思えば

　（思えばというよりも
　　実際にそうなのですが）

　人との接し方が変わる人も
　あるかもしれませんね。



■仕事を通して、
　有形・無形の財に恵まれたいのなら、

　（上記定義における）顧客の期待を
　つねに超えようとする、

　意志と努力が
　必要になってくるものです。



■その結果、

　お客さまの期待を超え続けられれば、
　評価され、評判も上がり、

　さまざまな形で報われる機会も
　増えていくでしょう。



■一方、期待を超えるまでには
　至らないけれども

　「期待通り」

　の価値を提供し続けたときには、


　ひどいことにはならないけれども、
　役務の提供者側も受益者側も

　「ある種の退屈（ルーティン）」

　に陥りがち。



■もちろん、
　それは悪いことではありませんが、

　「仕事において、
　　最も楽しく、美味しい果実」

　は得られません。



■最後に、提供価値が

　「顧客の期待値以下」

　ということになると、

　顧客サイドから見ると
　それぞれの程度において

　幻滅、軽蔑、さらには怒り、

　といった感情が
　引き起こされないとも限らない。



■そう考えると、

　多くの人命の次に大切だと
　考えているお金を頂戴するのは、

　思っている以上に大変なことですし、
　気が引き締まりませんか。



■まして、自分で商売を
　始めるということになれば

　「24時間、365日、
　　そんな環境に身を置き続ける」

　ことになるわけですから、
　決して甘いものではありません。



■ただし、

　きちんと役務提供ができれば
　それと引き換えに提供される対価は、

　一般に、より大きなものが
　見込めることとなるでしょう。


　リスクは取るけれども、その分、
　リターンも大きく取れるのです。



■いずれにしても、

　「お客様の期待を超え続ける」

　この思いは、仕事をする上で、
　絶対に意識する必要があるとともに、


　その意識を持つだけでなく、
　現実に成果を渡せなければ、

　お客さまのほうから
　黙って離れていかれるという、

　厳しい環境でもあります。



■補足すると、

　とはいえ、すべての仕事がそのような
　仕事である、

　あるいは、そうでなければならない、
　などというつもりもありません。

　（皆が皆、そういう仕事をしたいわけでも
　　ないでしょう）



■したがって、９割方の仕事は

　「やるべきことを

　（一定水準以上でもなければ
　　以下でもなく）

　　それなりにやっていたら、
　　それで良し」

　なのかもしれません。



■しかし、

　残り１割の世界
　（プロフェッショナルの世界）

　で楽しみたいし、
　大きな報いも手に入れたい、

　と思うのであれば、

　「お客様の期待を超え続ける精神と、
　　そのための実力を身につける」

　ことは絶対です。



■もし自分が顧客の立場であれば、
　当然そう思いますよね。


　いつも期待を超える価値を
　渡してくれる人、

　いつも期待通りの成果を
　返してくれる人、

　いつも期待以下の結果しか
　戻してくれない人、


　誰を贔屓にし、
　誰とは程よい距離を取ろうとし、

　誰に返金交渉をしようとするか、
　（あるいはクレームを入れるか）

　ということ。



■こんな風に偉そうなことを
　言っていても

　「じゃあ、
　　お前（鮒谷）はできているのか？」

　と尋ねられたら、
　はなはだ怪しいところではあります。



■とはいえ、それでも20年に渡って
　会社を潰すことなく、

　毎年、それなりの利益を
　計上しつづけられているということは、


　完全に基準を超えられているわけでは
　ないけれども、

　まったくの未達、
　ということもなかったのではないか、

　という少なからぬ自負もあります。



■何度でも繰り返しますが、
　商売やっている人はわかると思いますが、

　売上げを上げるというのは
　本当に大変なこと。


　20年やってきたから簡単になった、
　なんて、口が裂けても言えません。

　やっぱり、いつも大変です。



■こんなことを書くのはいかがと思いつつ、
　勢い余って書いてしまいますが、

　自分で商売やったことのない人は

　この苦しみを理解せず（できず）に
　好き勝手いわれるわけですが、

　「そんなにいうなら
　　お前がやってみろよ」

　と思わないでもありません（笑）



■それはともかく、

　顧客全員にとまでは言わないにしても
　それなりの割合のお客さまに、

　期待を上回る水準の結果を
　残せるようになって

　「仕事人として、ようやく一人前」

　だと私（鮒谷）は考えています。



■ところで、

　期待を超える成果を出すには
　何をどのように心がければよいか、

　について、


　私（鮒谷）がそれなりの年数をかけて
　必死に学び、実践し、編み出した

　【再現性の高い方程式】

　があります。

　ご提供しているわけですが、

　名前はふざけていますが、
　内容はガチのマジの本物です。
