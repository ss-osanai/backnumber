　今回の音源を聴かせていただいて、

　「過去に許せないがあってはいけない」

　ということです。


　過去の思い出したくない、
　無かったことにしたい出来事を

　そのまま負の歴史にするのではなく、
　そのことがあったからこそ

　現在の自分が形成されて、
　さらに未来の自分の物語を編むことに
　繋がっている。

　過去に起こった出来事には何一つとして、
　無駄なことはないとする。

　今の自分にとって、
　本当に救われる、力強いものでした。
