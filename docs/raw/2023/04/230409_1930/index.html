　話はそれからだ！


　【“子どもの人気者”となった
　　小島よしおの現在。

　　「来年、消える芸人」から
　　YouTuberへ華麗に転身】

　https://meetscareer.tenshoku.mynavi.jp/entry/20230406-kojima



■この記事の冒頭に

　---------------
　
　全方位から「消える」と言われて。
　「武器」をつくるために必死だったあの頃

　---------------

　とありますが、

　完全に自分と重ね合わせながら
　読みましたよ。



■2003年にメルマガを開始して、
　
　「たまたま時流に乗って」

　毎日、毎週、毎月、
　みるみる読者数が増えていきました。



■当時は

　「メルマガ広告ビジネス」

　つまり、
　
　このメルマガに広告掲載枠を用意し、
　それをクライアントに販売する、


　そんなビジネスを行っていたわけですが
　創刊後、２年半後くらい後には

　「まぐまぐランキング１位」

　になったんですよね。



■広告費はおおむね、
　読者数に比例しますから、

　最初は右肩上がりで
　ランキング１位になった頃には、

　月商（≒月収）が
　2000万円を超えました。



■通帳記帳しにいくと、

　「ジー、ジー、ジ、、、」

　の音が延々、止まらないんですよね。


　記帳される金額は、

　数万円から十数万円、
　ないし数十万円。

　ときにまとめての受注だと
　100万ないし200万ほどの振り込みが、

　日に何件もあるわけです。

　メルマガ執筆から配信まで
　原価はタダですから、

　まるまる利益です。



■でも、

　そういうマーケットだと
　他の人にも明らかになってくると、

　当然のことながら、徐々に
　メルマガビジネスへの新規参入も、

　増えていくこととなりました。



■初期の頃は、事業環境としては

　「ぬるかった」

　のですが、

　こうして新規参入者が増えるほど
　当然ですが

　「どんどんきつくなっていく」

　わけですよ。

　部数も増えにくくなるわけです。



■あるいは、既に能力も実績も経験も
　ある人であれば、

　そうした状態になっても
　焦りは生まれないのかもしれませんが、


　こちらは、なんといっても

　「ダメリーマン」

　上がりですからね。



■無能で実績も経験もない人間が、
　勤務先の倒産を機に、

　シャ乱Ｑの『上・京・物・語』を
　ヘビロテしながら、

　逃げるようにして東京に出てきた（マジ）
　だけですから、


　たまたま一山当たったからといって
　それが実力だなんて思えず、

　（小島よしおさんと同じく）

　【一発屋の自覚】

　はつねにありましたね。



■一発屋であれば、

　次の瞬間には消えていても
　まったく不思議ではない、

　ということですから、

　そこには不安とか恐怖とか
　緊張感とか焦りしかないわけです。



■そのときの感情を、
　小島よしおさんの

　---------------

　2008年から10年ごろには週刊誌の
　「来年、消える芸人ランキング」で
　3年連続1位でしたから。

　ほかにも、

　「占い師に聞いた
　　来年消える芸人ランキング」

　「女子高生に聞いた～」

　「子どもたちに聞いた～」

　などでも1位になって、
　全方位から「消える」と言われていました。

　当然、焦るし、それにあらがおうと必死に
　仕事をしているところがあったと思います。

　---------------

　という言葉を見たときに
　リアルに思い出しましたよ。


　当時、私（鮒谷）も
　「２ちゃんねる」でスレッドが立ち、

　全方位から

　「ぽっと出」「消える」
　「受け売り」「つまらん」

　と酷評されていたものでした、、、



■あるいは

　
　---------------

　いろんなバラエティ番組に出演しながらも、
　常に実力不足を感じていました。

　特に『クイズ！ヘキサゴンII』
　（フジテレビ、2011年に終了）という番組では
　何もできなくて、無力感でいっぱいでしたね。

　周りはすごい先輩たちばかりで、
　何かしら自分の特徴を生かして笑いをとるんです。

　でも、僕には何もなくて。

　無理やり前に出てはスベって周りに助けて
　もらうみたいなことしかできなかった。

　---------------


　これなんかも、当時を思い出して
　胸が痛いですね、、、


　
■とにかく人前で話をすることが苦手で、

　大勢の人の前に立つだけで
　赤面するし、動悸はするのに、
　

　にもかかわらず、ただメルマガの部数が
　多いというだけで（！）

　突然、「話をしてください」なんて
　振られることもあり、

　しどろもどろ、頭が真っ白になり、
　めちゃくちゃになってしまったり、、、
　


■そんなこともあり、

　毎日、ふるい落とされないように
　必死で生きていました。

　（今もそうですが）



■激化する一方のビジネス環境で
　消されないよう、追い落とされぬよう、

　来る日も来る日も
　
　「自分らしさとは？」
　「自分の強みはどこにあるか？」
　
　「飽きられないためには
　　どうすれば？
　「読み続けてもらうためには
　　なにが必要か？」

　「新たなキラーコンテンツを」
　「新しい能力開発を」

　など自問自答を続け、


　新しい学びの対象を見つけては
　すぐに飛びつき、

　なにか思いついたらすぐ実験、
　を繰り返していたものです。



■当時のことを覚えてくださっている
　読者さんであれば、

　当時の私の試行錯誤（迷走とも）、
　ならびに軌道修正、

　を重ねている姿を覚えている方も
　あるのではないでしょうか。


　※昨日の「年間プログラム」に
　　参加下さった方は

　　【軌道修正は手品のように】

　　のくだりを思い出してください。
　　


■小島よしおさんの振り返りを読み、

　たしかに取り組んできた
　具体的なことは

　（分野が違うのだから当然ですが）
　全く異なるけれども、


　どれほどの勢いであれこれ学んだとか、
　どんな思いで新しい挑戦をしたかとか、
　どんな失敗をしでかしたかとか、

　【深いところで通底する話】

　が随所に出ていて、驚きました。
　
　と同時に、
　とても共感したんですよね。


　あ、この文章に書かれていること、
　あのときとまったく同じだと。



■たとえば


　---------------
　　
　最初は手探りでした。

　絶対に笑ってもらえると思ってやったことが、
　意外にウケなかったりして。

　---------------

　ネタとネタの間に暗転を入れると
　子どもが怖がって泣き出してしまったり、
　予期せぬことがたくさん起こりました。

　---------------

　（中略）

　でも、今さら止められない。

　オラの音だけが響き渡る会場で
　ポールダンスを踊り続ける、
　地獄の時間でしたね。

　---------------


　こうしたことは私（鮒谷）にも
　頻繁にありました。

　あれこれ試すんですが
　なかなか

　「これが当たる！」

　みたいなものが出ないのです。

　スベる、はずす、刺さらない。
　それが続くと「迷走」になる。



■あと、以下の箇所も印象に残りました。

　　
　---------------

　──例えば、これまでどんな下手を
　　　こいてきましたか？


　小島：

　いろいろなことに手を出してみては、
　自分に合わなくてやめることは多いですね。

　ポールダンス、太鼓、カポエイラ……。

　それからオンラインサロンもやってみましたけど、
　おそらく芸能界最短で閉鎖しています。


　──それでも、とりあえずやってみようという
　　　バイタリティはすごいと思います。

　　　それに、挑戦したことがすぐには実らなくても、
　　　将来的に何かしらの形で
　　　役に立つこともあるのでは？


　小島：

　そうですね。失敗したとしても、
　自分に合うか合わないかは
　チャレンジしてみないと分かりません。

　だから、あまり深く考えずにいろんなことを
　経験しておいたほうがいいのかなと。


　資格取得なども、その一つかもしれません。

　もともとは「消える、消える」と言われてた
　時期に、形に残るものが欲しくなって
　いろいろな資格に手を出しました。

　でも、そこから資格の勉強をする楽しさを知り、
　中には仕事につながったものもあります。


　子ども向けの資格をたくさん取っているのも、

　少しでもライブの質や親御さんたちに対する
　説得力が上がればという思いからでした。
　

　──ちなみに、どんな資格を
　　　お持ちなのでしょうか？


　小島：

　キッズコーディネーショントレーナー、
　ジュニア野菜ソムリエ、
　パーソナルフィットネストレーナー、
　ジュニア・アスリートフードマイスター、
　日本漢字能力検定準1級、
　日本さかな検定3級、
　ダイエット検定1級、
　ナマハゲ伝導士

　……などですね。

　---------------


　
■いやー、素敵すぎる。素晴らしすぎる。

　そうした試行錯誤の末、
　小島よしおさんは現在、

　---------------

　「保護者500人が選ぶ
　　教育系YouTuber人気ランキング」
　　（CyberOwl調べ）で1位を獲得

　---------------

　という風に、見事で華麗な転身を
　遂げられているそうです。



■この記事は

　「小島よしおさんの履歴」

　であるとともに、

　私たちが自らのキャリアについて
　考える際の、極めて有用な教科書、

　ともなり得るのではないかと
　思ったのでご紹介した次第。



■仕事やキャリアや事業で

　「このままだとまずい」

　と思われている方は、

　この記事で記されていることを
　自分に置き換えて、

　必死に自分なりの戦略を考え、
　試行回数を増やすこと、

　を考えてみるとよいのでは。



■少なくない人が

　「AIに駆逐される＝消える」

　ことすら予見される時代にあっては
　一人ひとりが
　
　【自分は一発屋】

　の自覚を持ち、

　次の瞬間には消されているかも、
　という焦りや不安や緊張を力に変えて、

　「小島よしおばりの行動力」

　で自らのキャリアを切り開いていく
　べきではないでしょうか。



■特に

　「何が当たるか分かったら
　　取り組もう」

　ではなく

　「何が当たるか分からないから
　　手当り次第に挑戦しよう」

　という姿勢が大切だと思われます。



■小島よしおさんも、おそらく

　「何が当たるか分からないけれども
　　役に立つかもしれないから取り組もう」

　と考えて、

　---------------
　
　キッズコーディネーショントレーナー、
　ジュニア野菜ソムリエ、
　パーソナルフィットネストレーナー、
　ジュニア・アスリートフードマイスター、
　日本漢字能力検定準1級、
　日本さかな検定3級、
　ダイエット検定1級、
　ナマハゲ伝導士、、、

　---------------
　　
　といった各種資格を
　取られたのだと思います。



■思うに、

　これらの資格が今のキャリアに
　直接つながっている、

　というよりは、

　こうした貪欲な学びに対する姿勢が
　人生を切り開く土台になったのでは、

　と思われます。



■必死に学び、
　挑戦を繰り返しているうちに

　【必ず突破口は開かれる】

　のでしょう。


　私（鮒谷）も
　自身の履歴を振り返りながら、

　分野も取り組んできた対象も
　まるで異なるけれども、

　【通ってきた過程は全く同じ】

　と感慨深く、
　記事を読ませて頂きました。



■、、、そんなわけで、

　もしあなたが今ののままでは
　「消えるかも」という恐怖があるのなら、

　【新しい「武器」をつくるのに必死】

　になられてみても
　良いのではないでしょうか。



■その取り組みが直接役立つこともあれば、
　間接的に役立つこともあるでしょう。

　大事なのはどんなことでもいいので
　挑戦し続けること。

　かならず突破口は開かれます。



■どんな分野を学んだらいいか、
　さしあたって具体的な対象がない、

　ということであれば、


　先日来、ご案内してきた
　
　【複利効果で
　　人生にブレークスルーを！

　「友だち100人プロジェクト」
　　究極実践セミナー】

　へのご参加を検討いただいても
　よいかもしれません。


　自分に期待し、前向きに学び、
　行動し続けることによってのみ、

　【華麗なる転身】

　を図れるのではないでしょうか。
