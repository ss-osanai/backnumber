【 目標なき人生は「閑古鳥の鳴く、回転寿司屋さんの寿司」のようになる 】

───────────────────────────────────



■先日の日経新聞夕刊（2018年1月18日（木）付）を
　読んでいたら、

　「私のリーダー論」

　というタイトルで、

　ファーストリテイリング 代表取締役会長兼社長の
　柳井正さんのインタビュー記事が出ていました。



■読み進めていくうちに
　以下のような文章が目に留まりました。


　（引用ここから）
　-------------------------------------

　「日本人はほとんどそうだろうけど、
　毎日努力したら、ある程度成功すると思うでしょう。


　でもね、努力してても、
　努力の方向性が違うとダメ。

　成功しないの（笑）。
　同じ所を回ってるだけ。


　人生をかけて何がしたいか決まらない限り、
　ビジネスはうまくいかないと気づいた」

　-------------------------------------
　（引用ここまで）


　このあと

　「目標が如何に大切か」

　について柳井さんが力説しているのですが、
　本当にそうですね（自戒を込めつつ）。



■だいたい

　「目標を定めてもいないのに、
　　毎日を一生懸命に」

　なんてことは原理的に存在しませんよね。


　目標があるから、一生懸命になれるのです。
　当たり前の話です。

　
　目指す方向性も決まっていないのに

　「一生懸命、頑張ります！」

　とか、愚の骨頂。というか、あり得ない。

　どこを目指して頑張ってるんですか、という話。



■私（鮒谷）は

　「目標設定＆達成音源３部作」

　において、


　実際に私が掲げている

　（たとえば、そのうちの一つは
　　圧巻の言語化能力を身につける」という）

　目標を聴かれた方も多いと思います。



■この目的が実現されているかどうか、
　についてはもちろん、

　（今は）お寒い状況ですが、
　
　こうした方向の目標を始めて掲げた
　15年前と比べると、

　その方向性に進化しているのは明らかです。

　（あくまで自己評価ではありますが）



■あるいは

　「自信と誇りを兼ね備えた、
　　圧巻の存在になりたい」

　とも掲げて生きてきました。

　これも未だ、お寒い状況ではありますが、
　それでもなお、
　
　掲げた目標に近づいているのは明白です。

　（あくまで自己評価ではありますが）
　
　　　　

■こうした目標を掲げて生きているから、

　いろんな外敵（トラブル等）＆内敵（怠惰等）に
　襲われつつも

　「時々刻々が意思決定の連続」

　となり、

　「時々刻々が引き締まる」

　ようになるのです。


　
■上の例でいえば、

　「言語化能力を高める（という目標の）ために」

　今の一瞬、一瞬において、


　目の前の、この本に目を通すか否か、
　閃いた名文句を記録するか否か、

　記憶したい言い回しを何度も頭の中で
　反芻し、定着させようとするか否か、

　といった戦いを繰り広げることとなるのです。



■あるいは、

　「圧巻の存在になる（という目標の）ために」
　
　今の一瞬、一瞬において、

　楽に流されてしまうか否か、
　きちんと自分を躾けようと努めるか否か、

　といった戦いを繰り広げているのです。



■もちろん、

　こうした内的な戦いの末、
　導き出された時々刻々の意思決定は

　「怠け心や先送り癖に完敗」

　してしまっていることだって頻繁にあるし、


　反対に

　「それらに打ち克ち、連戦連勝」

　ということだってあるのです。

　（まれですが 泣）



■こうして勝ったり負けたりを
　繰り返しながらも、

　「徐々に目標に近づいていく
　　（あるいは、そのための努力が行える）」
　
　のは、ひとえに

　「目標があるから」

　です。


　目標がなければどうなるか。
　
　ハッキリ言いましょう。百戦百敗です。
　これでは何者にもなれません。

　こうした（目標なき）長期間に渡る怠惰の後、
　我と我が身に残るのは
　
　「悔悟の念」

　のみでありましょう。
　


■反対に、目標を立てている私（私）は、
　少なくとも

　「十回戦ったら、一回くらいは勝つ」
　
　のだから、

　この差は小さなように見えて
　長期の時間軸で見たときにデカいです。
　


■仮に今日、

　「微細な100回の意思決定」

　の機会があったとき、

　10勝しているか、0勝（全敗）してしまっているか、
　くらいの差は現れるのです。



■だからこそ

　「（「向こう側の世界」に意識を留め続けるための）
　　橋頭堡たる目標」

　を掲げずして、努力しようとしても、
　目指す方向がハッキリしていないのだから、

　そりゃ、柳井さんの言うとおり、


　（引用ここから）
　-------------------------------------

　「日本人はほとんどそうだろうけど、
　毎日努力したら、ある程度成功すると思うでしょう。


　でもね、努力してても、
　努力の方向性が違うとダメ。

　成功しないの（笑）。
　同じ所を回ってるだけ。


　人生をかけて何がしたいか決まらない限り、
　ビジネスはうまくいかないと気づいた」

　-------------------------------------
　（引用ここまで）


　ということになりますよね。



■お互い、くれぐれも、

　「閑古鳥の鳴く、回転寿司屋さんの寿司」

　のように


　「同じところをぐるぐる回リ続ける
　　（そして段々、干からびていく）」

　といったことのないよう、

　努力の方向性を決定づける、
　目標の設定に意識を向けてまいりましょう。



　※今日のメルマガを読んで

　　「確かにそうだ」
　　「こりゃ、ヤバイ！」

　　と思われた方は、まずは以下の音源を聴く
　　ところから始めて頂ければと。


　　同じところをぐるぐる周り、
　　すんでのところで干からびかけた

　　「鮒寿し（！？）」

　　が奇跡の復活を遂げた物語が収録されています。
