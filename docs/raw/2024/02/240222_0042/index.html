■おはようございます、鮒谷です。


　Twitter、開始後およそ10日間。
　メルマガとは全く別ルールですね。
　
　毎日、細かく微修正を重ねてますが、
　踊り場を少しだけ抜け出せそうな。


　＜鮒谷Twitter＞
　https://twitter.com/shinkaron

　※「いいね」押して頂けると喜びます

＜今日のピークパフォーマンス方程式＞

■怠惰を求めて、勤勉に行き着く
　（商売も、文章も、恋愛も、セックスもみな同じ）

───────────────────


■わけあって、
　今年の４月末日までに、

　一定程度の水準で回る

　【Twitterの運用システム】

　を稼働させなければなりません。



■５月以降、新しい挑戦が始まる
　（可処分時間の７割ほどを投入予定）のため、

　Twitter投稿やシステム改善に
　割ける時間をほとんど取れなくなるから、

　なんですけどね。



■ここでいう

　「一定程度の水準」

　とは、

　現時点における、私（鮒谷）中の
　イメージとしては、

　一投稿について、おおむね

　「5000インプレッション
　　※たまに万超え」　　

　くらいな感じ。


　※インプレッション数とは、

　　ユーザー画面（タイムライン上）に
　　ツイートが表示された回数のこと。


　
■平常運転（一日、数通の投稿）で
　この状態を実現できれば

　「最低限のシステム化は
　　完了した」
　
　といえるのではないかと。

　遅くとも４月末までに
　その水準を目指そうと。



■そんなことも考えながら
　先ほどこんな投稿もしました。


　----------
　
　毎日、まじめに
　３つの改善を重ねている人は、

　ゼロ改善の人よりも一見、
　勤勉に見えますよね。

　でも本当は、未来の自分に楽をさせたい、
　怠惰なだけの人かもしれません。


　一方、いつも、ぼーっとしている人は
　死ぬまで楽をしたくないという意味で、

　本質的には勤勉なのかもしれません。

　あなたはどっち？

　----------



■ちょっと煽り気味の投稿ですが、

　Twitterのメディア特性的に
　これくらいまで書かないと
　
　（これくらい書いたところで）

　たいした反応も
　起こらないんですよね。



■だから、雑な言辞を弄したり、
　過激な方向に走ったりしがちで、

　その気持ちもTwitterやり始めて
　分かるようになりました。

　もちろん私（鮒谷）は自制して、
　せいぜいこの程度にしておきますが。



■それはさておき、
　Twitterにおいて、私（鮒谷）は

　「一日３改善」

　どころか、


　（外から見ているだけだと
　　まるでわからないと思いますが）

　細かいところで、この10日ほど
　少なく見積もっても
　　
　「一日、数十以上の改善」

　を繰り返しています。


　※進化のプロセスが気になる方は、
　　過去投稿にさかのぼってご覧ください。

　＜鮒谷Twitter＞
　https://twitter.com/shinkaron　



■10日前に、

　長年、寝かせていたアカウントを
　超久しぶりに叩き起こしたので、

　最初は投稿を重ねても
　一投稿のインプレッションは、

　「数十から100」

　をなかなか超えなかったのですが、
　やがて、

　「数百」

　くらいまでは、
　なんとかいくようになりました。



■そこで数日ていどの
　「踊り場」がありましたが、

　このところ、投稿後、
　丸一日が経過した時点で

　「おおむね
　　1000インプレッション以上」

　を安定的に出せるようになりました。
　（まだまだ予断は許しません）



■現状、フォロワー数の増加よりも、

　インプレッション数の増加のほうが
　テンション上がるので、

　方向を転換して、当面は
　インプレッションを最重要指標として、

　追いかけていくことにしました。

　（朝令暮改は善である）

　

■こんなことを
　毎日、繰り返していれば、

　2月末までに、あと8日あるので、
　
　「平均インプレッションを
　　2000超」
　
　まで持っていけるのではないか、
　（いや、持っていく！）

　そんな気持ちで徹底的な
　超速のPDCAを回しているところ。



■そんな最中に
　投稿ネタがまた降りてきたので、

　執筆＆Twitter仕様に編集して
　以下を投稿してみました。


　----------
　
　【センスとは】


　１、膨大なトライアル・アンド・エラーが
　　　センスの源流。

　　　動きもせずに
　　　センスが身につくはずもない

　２、大量の蓄積データが
　　　センスへと昇華される

　３、商売も、文章も、恋愛も、
　　　セックスもみな同じ


　好きでなければ実験を続けられず、
　上達しない。

　好きこそものの上手なれ、
　と言われる理由。

　----------



■わざわざ、最後に

　----------　
　
　３、商売も、文章も、恋愛も、
　　　セックスもみな同じ

　----------　

　と、

　自分の経験不足を棚に上げて（笑）
　内角高め、ギリギリの危険球を投げて
　
　「これは、バズんじゃね？」

　と思ったのですが、、、
　（テストしたということです）



■まるでバズらないどころか、
　インプレッション数も大して上がらず。

　予想通りに動いてくれず、
　なかなかに難しいものですね。



■じつのところ、以下の文章

　----------
　
　３、商売も、文章も、恋愛も、
　　　セックスもみな同じ

　好きでなければ実験を続けられず、
　上達しない。

　好きこそものの上手なれ、
　と言われる理由。

　----------

　に対するコメントとして、


　「セックスは好きなんだけど
　　ぜんぜん上達しねーけど」

　みたいなのを期待したんですが
　無風状態（笑）


　真面目かよ。まじ難しいわ。



■毎度の投稿において
　こんな風な仮説検証を繰り返し、

　「ほとんど外して、たまに当たる。
　　まれに長打やホームラン」
　
　というゲームを
　楽しんでいるんですけどね。



■昔、ハマっていたパチンコや
　パチスロよりも、

　金がかからない上に、
　うまく跳ねれば将来、

　さらなる安泰と安寧の老後（？）が
　もたされるわけですから、

　【健全きわまりない、
　　実利を兼ねた娯楽】

　とも言えそうです。



■そんなわけで、
　私（鮒谷）は究極の怠け者だからこそ、

　誰よりも勤勉に試行錯誤を
　重ねていこうと思います。



■そういえば麻雀漫画、
　『勝負師伝説 哲也』でも

　【怠惰を求めて、勤勉に行き着く】

　って、名言がありました。
　


■怠惰きわまりなかった私（鮒谷）が
　いつの間にか

　「けっこう勤勉」
　
　になったのも、
　そういうことなのかもしれません。
　

　※怠惰だった私が勤勉に
　　進化を求めているプロセスが気になる方は、

　　過去投稿にさかのぼって
　　ご覧ください。

　　＜鮒谷Twitter＞
　　https://twitter.com/shinkaron　



■今日も人生とビジネスを
　楽しみましょう！
