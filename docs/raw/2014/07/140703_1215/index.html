【鮒谷の、回顧ならびに懺悔】３日間セミナーを止めるに至った思いと、未来への決意

2014/07/03 12:15 
■こんにちは、鮒谷です。
　いつもありがとうございます。


　これまでずっとメルマガで書いてきたとおり、

　振り返ってみれば、これまで３日間セミナーを
　何度も開催してきました。

　初めてのセミナー開催は
　今からもう４年近くも前のことになります。

　そして直近のゴールデンウィークに開催した
　当面、最後となるセミナーは２ヶ月前。



■この間、全29期、
　400名以上の方とご縁を頂いて参りました。

　それらの方に向けて、実に３日間×29回、
　つまり87日、約3ヶ月間もの間、

　セミナーやっていたことになりますね。
　（我ながら、よく続けてきたものです）



■さらにセミナー後、引き続きご縁のあった方々への
　のべ800回を越える個別コンサルティングなどを通して、
　
　大勢の、


　▼劇的に変わった人

　▼大きく変化を遂げた人

　▼ゆっくり変化を遂げつつある人

　▼これから変化が約束されていると思われる人

　▼変化の兆しがなかなか見られない人


　といった方々と接してきました。


　
■この過程において、

　「人はどのようにして変わることを決意し、
　　行動し、現実の人生を作り変えていくのか」

　という変化変容について、
　深く考える機会が一層、増えました。


　増えた、というよりは、

　「朝から晩まで考え続けるようになった」

　といったほうが適切かもしれませんね。



■ここで話は大きく変わりますが、


　何度もメルマガで書いてきたし、
　セミナーでもお話ししてきた通り、

　「ドラッカー５つの質問」
　
　の中に、

　「われわれの顧客は誰か」
　
　「顧客の価値は何か」

　という問いがあります。


　極めてシンプルな質問ですが、
　深く考えようとすればするほど、

　その奥行きの、とてつもなく深く、広いことに
　驚かされます。



■また、たまたま先日読んだある本の帯には
　
　「『何を売るか』ではなく
　　　　『顧客のために何ができるか』」

　という言葉が記されていたのですが、


　これもドラッカー５つの質問とほとんど同義で、

　「顧客が、本当は、
　　どのような価値を求めているのか」

　について深く思いを致す機会を与えてくれる
　問いでありました。



■当たり前ですが、

　セミナーにお越し下さった皆さんは、
　安くもない金額をお支払い下さったわけですが、
　
　この金額で、

　「鮒谷の３日間という時間を買った」

　のでもなければ、

　「セミナーに参加する権利を購入した」

　わけでもありません。


　あくまで私の話す時間を通し、
　あるいはセミナーに参加することを通し、

　それを契機として、

　「人生のコントロール権を取り戻す、
　　そのための『きっかけ（と場）』を掴む」

　という価値を購入しようとお申込み下さったのだと
　思います。



■特に個別コンサルティングの現場において、

　過去１年半の間、
　それこそ最近では月間60から70件に及ぶ、
　
　１対１の対話を通して深くニーズを探っていくと、


　「本当の自分はもっとできるはずなのに、
　　どうしても身体が動かない」


　「さまざまな要因から来る
　　精神的不安、圧迫、焦燥感から逃れたい」


　「やりたい気持ちはあるのだけれど、
　　具体的なやり方がわからない」


　「はじめの一歩を踏み出すきっかけがなく、場もなく、
　　日常の引力に負け続け、後悔の日々を送っている」


　そんな悩みや苦しみをどうにかして解決したい！

　という、お客さまの真の欲求が強く、強く、
　伝わってくるようになりました。　



■こうして対話を繰り返すうち、
　３日間セミナーの開催を一旦、
　
　「廃棄」

　することによってしか、

　「より大きな価値を
　　お客さまに提供することはできないのでは」
　
　と考えることも増えてきました。



■つまり、

　
　この何年かの間に私（鮒谷）が考えてきた、

　より再現性の高いプログラムをお届けすることに
　よってしか、
　

　（「新しいパラダイムを提供する」

　　「人生やビジネスにおいて、　
　　　成果を出すための知識を提供する」

　　のは、もちろんのこととして）


　それらをクリアした上で、

　「自分を変えて、
　　取り巻く外部環境も180度、変える」

　という難事業のお手伝いを満足のいく形で
　果たすことはできないのでは、
　
　と考えるようになったのです。


　
■つまるところ、


　お客さまが真に求める結果にコミットして、
　
　より大きな価値をお渡しすることに
　自らの誇りをかけて取り組みたい、

　そんな欲求（野望）に目覚めてしまった
　わけなのです。


　もはやビジネス云々といった話ではなく

　「自らの存在理由、存在価値をかけて」

　というレベルで。



■コンテンツを伝えてくれる人は無数にいるし、
　新しいパラダイムを見せてくれる人も無数にあるでしょう。


　しかし、

　本当に変化のお手伝いをしようと思ったなら、
　ただセミナーさえしていればいい、

　というものではないことに気付き、
　
　その思いがもはや無視できないまでに
　大きくなってきたのです。



■こうして、本当に変わっていただくために
　必要なもの（ご用意すべきもの）を長らく考えている中で、

　「以下の８つ」

　が一つ残らず必要なことにも気づきました。

　（ひとつ欠けるごとに再現性が減少していくように
　　思われます）
　

　＜そのリストは以下↓の通り＞
　----------------------------------------------------------


　１、日常と引き離された（まとまった）時間、
　　　ならびに空間を確保し、
　　
　　　自身のビジネスやキャリア・その他について
　　　深く思索する時間が強制的に「天引き」される場を作る

　
　２、新たなパラダイムを提供する

　　　（新しい視点・見え方・世界観・認識。
　　　　別の言葉で言うと、驚き・発見・目からうろこ）
　

　３、目標設定と発表の機会を作る
　　　
　　　（目標なくして、行動なし。逆算から入る人生）

　
　４、歩き出すきっかけの提供、
　　　ならびに歩みを止めたときに再起動を促す場

　　　（目標実現に向けたコミットメントを行い、
　　　　一歩踏み出すための場、
　　　　もしくはドンと背中を押される場）

　
　５、「センス」の伝達
　　　コンテンツではなく、コンテクストを身体で理解する

　　　（長期間、センスある人と関わることによってのみ、
　　　　初めてセンスは身につけられる。
　　　　
　　　　知識でセンスは身につかない）

　
　６、圧倒的なコンテンツの質と量

　　　（成果を出すための、具体的極まりない、
　　　　圧倒的なボリュームのノウハウ・スキル・テクニック・知識）

　
　７、長期的な関わり

　　　（学びの共同体・仲間・場。
　　　　→一般的に、変容は長期（年単位）の時間軸で起きるから
　　　　　「場」なくして変容することは絶対にない）

　
　８、１から７をバックアップ、あるいは下支えするための
　　　様々な継続的サービス提供

　　　（メールマガジン、限定セミナー、
　　　　その他様々なメディアやツール・場などを用意し、
　　　　日常の引力から切り離された新しい世界への水先案内を行う

　　　　今後、随時、企画していく）
　

　----------------------------------------------------------


　今日は誌面の関係もあり、詳しくは述べませんが、
　以上のような理由から、

　これらすべてを一つのパッケージとして包含した形で、
　新たなプログラムをスタートさせることといたします。


　
■実のところ、３日間セミナーでは、

　わずか３日という時間しかなかったということも
　ありますが、

　（コンサルなどでお付き合いさせていただいた方は
　　ともかく）

　参加下さった全員に対して、
　継続したフォローをしきれなかった、


　あるいは、上に挙げた項目のうち、
　ごく一部しか提供できなかった、

　という強い反省があります。



■過去を真摯に振り返り、反省を活かし、

　「顧客価値の最大化」

　に向けて、

　現在の自分にできる最善、最高は
　どのようなものであるべきか、
　
　を改めて考え抜いた末の、今回のプログラムです。



■一言でいえば、

　決して悪くはなかったであろう（と思っている）
　３日間セミナーは

　「GOOD」

　ではあったかもしれないけれども、
　決して

　「GREAT」

　ではなかったことを潔く認め、
　新しいチャレンジを行うべきである、

　と決意したということ。



■そのようなわけで、

　今回、はじめて新プログラムの募集を
　募ることにいたしましたが、
　
　最初から大々的に募集はいたしません。


　（ご興味をお持ち下さり、
　　参加下さる方がいらっしゃれば）

　その方々との、極めて少ない人数から始め、

　まずは小さく渦をまき、
　やがてこれを大きな渦に変えていく、

　そんな理想を実現できればと考えております。



■繰り返しになりますが、このプラットフォームは、
　ただ、人数が集まればいい、といったものではなく、

　「極めて再現性高く、
　　（私自身も含め）人が変わるための場」

　として機能させることを目指しています。


　今回は、その集まりの「０期生」の募集です。



■なぜ「０期」かというと、

　（コミュニティの趣旨は以上の通りで、
　　一切のブレはありませんが）

　まだ年間プログラムの名称も定まっておらず、


　今後の具体的な方向については、
　日々の実践ならびに振り返り、

　そして参加者との対話を通じて、

　「より良いものに変えていく」
　
　という形でスタートすることになるので、

　「１期」

　ではないだろう、ということで。



■一番最初から（０期生として）参加しておくと、
　もしかすると将来、
　　
　「一番最初から参加していたんですか！？」

　と畏敬の念を持って接せられたり、
　という特典があるかもしれませんし（それはないか）、
　　
　
　しかしながら、今回は初回ゆえ、

　「おそらく極めて少人数での集まりとなる」

　ことだけは間違いありません。

　
　（なにせ年間のプログラムなのに
　　開始９日前に告知しているということもあり 汗）



■繰り返しますが、
　上述の通り、まだプログラムの正式名称もありません。

　（実は複数のプランがあり、決めかねています。
　  そのためにご案内が遅れてしまいました。
　　　
　　結局、まずは正式名称なしでスタートすることに
　　いたしましたが）


　コミュニティの文化もこれから作っていきます。
　
　単なるコンテンツ伝達の場ではないからこそ、
　これは非常に大事な問題であると位置づけています。

　（だから、最初から大人数を集めません）


　速戦即決の３日間セミナーのようなものでは
　ありませんので、

　ルールも徐々に整備していきます。



■そんな孵卵器に入れられたヒナのような
　プログラムですが、

　「そんな状態でも参加したい」

　あるいは

　「そんな状態だからこそ参加したい」

　という方にのみ、
　今回は参加いただければと思います。


　※ルールや文化は参加者の変化を最大化させる目的で
　　必要に応じ、どんどん作り変えていく予定です。

　　その旨、あらかじめ同意頂ける方のみ
　　ご参加ください。



■気になるプログラムの流れとしては、
　今のところ、


　＜第１部＞

　▼午前10時から午後４時

　　セミナー形式で、
　　圧倒的な量と密度でコンテンツをお届けいたします


　＜第２部＞

　午後５時から９時

　リラックスした環境で放談＆質疑応答

　※むしろ２部のほうが価値があるかも


　という流れ、つまり、

　＜朝10時から夜9時まで、ほぼ、びっしりと＞

　お話しさせていただくことになります（現時点での予定）。


　※毎月１度、土曜日開催（全12回）となります。

　　初回は「7月12日（土曜日）」スタートです。


　　次回以降は、

　　「8月9日、9月13日、10月11日、11月15日、12月6日
　　　（いずれも土曜日）」　　　

　　を予定しています。

　　（来年1月以降は、後日発表）
　　


■受講料については、いろいろと考えた結果、
　

　【 年間受講料　88万円（＋税）】


　とさせていただきました。


　10時間ほど話し続ける本セミナー、
　１日あたりの金額で換算してみても、

　なんと

　「個別コンサル１時間」

　の単価以下。

　以前行っていた３日間セミナーの時間単価と比較しても、
　圧倒的に安価です。



■ということで、

　
　「特定の人しか参加できない高額なものではなく、
　　リーズナブル、

　　しかしながら、
　　一定の覚悟がなければ入れない」


　（この金額では参加に相応の覚悟は必要でしょう。
　　またそういう覚悟を持った人の集う場にしたい）

　そんなコミュニティを形成するために
　妥当な金額は、

　と検討に検討を重ね、
　この金額とさせていただいた次第です。



■時間単価で考えても、
　３日間セミナーより大幅に安価です。


　本当は、正直に告白すると、

　「安価過ぎて、他サービスの価格体系との
　　整合性が取れず、この値段では売りたくない」

　というのが本音です。


　けれども、

　「再現性高く、変化のお手伝いをする」

　という目的を考えたとき、

　「然るべき思いや志を持った、一定以上の方々が
　　集い、語らい、学び、刺激し合う場を作る」

　のは必須。



■だからこそ、参加していただきやすいよう

　「あえて、この値段設定」

　をさせて頂きました。


　※ただし、今回のお申込み状況を見ながら、
　
　　次回のご案内以降、価格変更（値上げ）させて頂く可能性は、
　　なきにしもあらず、

　　です。予め御了承ください。



■この金額が高いか安いかのご判断は、

　
　当プログラムに参加することによって、

　（あなたの、10年、20年、あるいはひょっとすると
　　30年、ないし50年以上に及ぶ将来（人生））

　において期待される成果、波及効果を、

　現在価値に割り戻して妥当な金額かどうか、
　で検討いただければと思います。



■時の経つのは早いもので、

　起業12年目を迎え、日々配信してきたメルマガも
　あと50日ほどで、4000号に至ろうとしています。


　この間、何千万円ものコストをかけ、
　人生をかけて（１万時間どころではない時間）を投下し、
　学んできたことを、
　
　惜しみなく、
　お客さまの期待される成果にコミットして
　お伝えする、

　これだけはお約束いたします。

　（これ以上はしつこくなるので、言いません）



■ということで、

　ゴリゴリ営業するつもりは、
　今回も、これからも絶対にありません。
　
　自信の現れと思ってください。


　あとはご自身の自由な意志でお申込みのご決定を
　頂ければと思います。


　お申し込みをお待ちしております。
