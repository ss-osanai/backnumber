【 頼むから潰れてくれ 】

───────────────────────────────────



■テレアポ営業をしてくる会社に対しては

　「頼むから潰れてくれ」

　と思っています。


　こちらは全く望んでもいないのに、
　一方的に電話をしてきて、

　【人的資源や精神的資源を強奪】

　していくのはもはや、
　犯罪行為であるとすら考えているからです。



■あらゆる業種から、
　営業電話がかかってきますが、

　なかでも、

　SEOやウェブマーケティングを
　謳う会社からかかってくる電話は、

　【まるで、漫画】
　【もはや、喜劇】

　としか思えません。



■人さまに提案して、
　カネをもらおうとするくらいに、
　
　マーケティングやSEO（検索エンジン対策）に
　自信満々であるのなら、

　そもそもテレアポする必要など
　どこにあるのでしょう。

　得意のネット集客で、ガンガン、ネットから
　クライアントを集めてきたらいいのにね。



■テレアポをしていることそのものが、

　【提供サービスの品質の悪さを証明】

　していると言ったら言い過ぎですかね。
　

　そういうことに気づき、ツッコみ、

　テレアポからの面談、さらには受注にまで
　至る会社など一社もなくなれば、

　そもそも電話をかけようなんていう
　動機も生まれないのでしょう。



■けれども、

　中に騙されて契約する（そして泣きを見る）
　会社が後を絶たないから

　【SEO会社のテレアポ営業】

　などという矛盾が、

　いつまでたっても社会に残り続けて
　いるのではないでしょうか。



■それはともかくとして、
　改めてここで私（鮒谷）が強調したいのは、

　【テレアポ（や飛び込み営業）】

　という行為は、

　相手の貴重＆希少な時間資源や認知能力を
　強奪している以外のなにものでもない、

　ということです。



■仮に、たとえ提供しているものが
　素晴らしい商品、サービスであったとしてさえ、
　
　【無理矢理、相手の懐に押し入って
　　求めてもいない100人に迷惑をかけたけど、
　　
　　最終的に求めていた１人に
　　価値をお届けできたから良かった良かった】

　などという独りよがりが
　許されてよいのでしょうか。

　自分さえよければ、
　迷惑を被る人があったとしても目を瞑る、

　そんな姿勢はいかがなものかと思います。



■昔からそうでしたが、特に近年は

　【時間や認知能力の価値】

　は、高騰する一方。


　あなたも私も、

　昔よりはるかに、時間が大切だという
　認識になっているのではと思いますし、

　望んでもいない営業に集中力を奪われることなど
　ゼロになってもらいたいと願っているはず。



■こうした価値の増大し続けている資源を
　容赦なく奪い取ろうとする輩に鉄槌を、

　と声を大にして言いたいところ。


　あるいは、そこまでは言わずとも、
　少なくとも彼ら彼女らを反面教師として

　【以て他山の石とせよ】

　の精神を常に持ち、
　
　誰も幸せにせず、迷惑をかけるばかりの
　仕組みの中で仕事をするのではなく、


　【本当に必要としている人に、
　　本当に必要な価値が提供される環境】

　を作ることに、
　お互い精力を傾けたいものです。


　
■新年度を迎え、
　テレアポ＆飛び込みが激増中。


　私は人一倍、自分の時間と認知能力を
　大切にしているつもりなので、

　その希少資源を、傍若無人な連中の
　配慮なきセールスに奪われることに、

　強烈な不快感と憤りを覚えています。



■ハッキリ言って

　【一方的に時間を奪われる】

　イコール

　【命を奪われている】

　イコール

　【殺されている】

　のと同じですからね。

　怒るなと言われて、
　怒らないほうがおかしいですよ。

　なんなら本気で損害賠償請求まで
　したいくらい。



■もはや、

　【そういう時代じゃない】

　んだと、いつ彼ら彼女ら（の勤める会社、
　経営者、上司）は、気づくんですかね。


　【頼むから潰れてくれ】
