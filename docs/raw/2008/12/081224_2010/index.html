 【　苦悩、葛藤をこそ、よろこぶ　】

───────────────────────────────────



■生きていくなかで、人間は様々な「苦悩」「葛藤」に
　遭遇するものです。


　みんなと仲良くやりたいのに、仲良くなれない、

　時間がもっと欲しいと切望しても、時間に追われている、
　
　お金があればとおもいながら、お金がない、

　義理と人情の間で葛藤している、

　なんらかの劣等感をぬぐい去れない、

　
　などなど。


　本来ありたいと思っている、理想の自分と現実との
　落差が苦悩となったり、心理的葛藤となって、
　我と、我が身を責め立てます。
　


■しかしながら、これはある時、気がついたのですが、

　この「苦悩」「葛藤」のあることをあるがままに受け入れ、

　その悩みをこそ、強力な力に変えて人生の難所を乗り越え、

　新しい世界に出た人がたくさんいるのです。



■逆に言うとそのような苦しみがなければ、
　この人はここまでいかなかったであろう。

　そんな人がたくさんあるのです。

　かくいう私も、思い起こせば、苦悩続きの毎日でした。

　そして、今もなお、あります。



■しかし昔の私と今の私が根本的に異なるのは、
　
　苦しいと感じているなら、その感情はその環境から抜け出す
　ために後押しをしてくれている、

　と捉えられるようになったこと。

　こうなると苦しいほど、ありがたいことになります。


　なぜなら、その思いが強いほど、
　自分が今の環境を抜け出さなければならない強烈な
　動機付けになってくれるから。



■おかげさまで、

　そうやって幾多の苦悩を乗り越えてきましたが、
　そのたびに自分に自信がついてきました。

　とても越えられないような壁に見えた苦難でも
　「必ず越えてみせる」と誓って、そして行動すれば
　扉は開けるのだと確信が持てるようにもなったのです。



■そう考えると、

　「苦しい」

　という感情が、自分を後押ししてくれたことと
　なるでしょうし、

　幸か不幸か、ぬるま湯に浸かって、そのように感じることが
　なかったなら、いまの私は存在しえなかったこととおもいます。

　

■苦しみの感情と、そこから抜け出したい、という強い欲求が
　自分を行動に駆り立ててくれたのですから、

　その苦しみに対しても、振り返ってみると、

　感謝とか、懐かしさとか、ある種の感慨をもって
　振り返ることすらできるようになったのです。

　

　あなたの人生とビジネスの参考になれば幸いです。
