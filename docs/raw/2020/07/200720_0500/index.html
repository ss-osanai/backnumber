【すぐに取り出せない意思決定基準は、意思決定基準ではない】

2020/07/20 05:00 
■おはようございます。


　ふと、思いついた言葉があるので
　忘却の彼方に消える前に、

　せっかくなので言語に落として、
　お伝えいたします。
　


■それは

　【すぐに取り出せない意思決定基準は
　　意思決定基準ではない】

　という言葉。


　
■みんな、一生懸命、勉強しますよね。

　
　そして

　「あ、これは使えそう！」

　と思ったり、
　感銘を受けた言葉について

　「うん、うん」

　と頷き、いっときの満足を得る。



■でも、そうしたことを、

　何百回、何千回、重ねて、
　その場限りの満足を得るだけのことに、

　果たして何の意味があるのかと
　問題提起したいのです。



■知的好奇心が満たされたとか、
　ちょっとイイ話を聞いたとか、
　
　そうした

　「娯楽としての学び」

　と理解した上で、趣味として

　「お勉強」

　していると自覚の上で
　行っているのであれば、

　別に構わないでしょう。



■しかし問題は

　「当人は一生懸命、時間をかけて、
　　勉強しているつもり」

　になってはいるけれども、


　そうして触れてきた
　幾多の価値ある言葉が、

　いつでも取り出せる状況に
　なっていないこと。



■私（鮒谷）にとっての
　学びの定義は


　「この状況においては、
　　このように振る舞うことが
　　最善の結果をもたらすであろう」

　と思われる

　
　-----------------------

　１、質の高い意思決定基準を
　　　手に入れて

　-----------------------

　さらにそれを
　
　-----------------------

　２、いつでも自在に取り出せるように
　　　つねに整備しておくこと
　
　-----------------------

　の２つを兼ね備えた状況を
　保つための営み。



■分かりやすく例を挙げるなら、


　「自分の商品を、誰かに販売する」

　というセールスの状況における
　幾多の期待する結果

　（成約率を高めるとか、
　　売上金額を引き上げるとか）

　をもたらしてくれる

　【質の高い（上記の可能性を高める）
　　意思決定基準】

　を学び（インストールし）、


　しかもそれを臨機応変、自由自在に
　駆使することができるようになって

　【初めて学んだ意味が生まれる】

　ということです。

　

■状況の変化に応じて、
　リアルタイムに

　【質の高い意思決定基準を発動して
　　行動を変化させられる】

　ことができてこその、
　真の学びである、

　とも言えるでしょう。



■だからこそ

　-----------------------

　１、質の高い意思決定基準を
　　　手に入れて

　-----------------------

　さらにそれを
　
　-----------------------

　２、いつでも自在に取り出せるように
　　　つねに整備しておくこと
　
　-----------------------

　ことが大事なのです。



■逆にいうと

　【（必要なときに）取り出せない
　　意思決定基準は意思決定基準ではない】

　とも言えますね。


　また「意思決定基準」とは
　言葉を換えると

　【自問自答をもたらす問い】

　です。



■例を挙げるなら
　
　-----------------------

　「北斗セールス音源」
　https://shinkaron.buyshop.jp/items/28741845

　-----------------------

　をお聴き下さった方であれば
　お分かりの通り、


　ピンポイントで顧客の心を射抜き、
　
　摩擦なく、エレガントに
　セールスを成功させるには

　「お客さま自身が気づかれていない
　　真のニーズを特定し、指摘して差し上げる」

　ことが決定的に重要です。



■こうした話（意思決定基準）を

　「読んだことがある」
　「聞いたことがある」
　「知っている」

　というだけでは実は意味がありません。



■セールスの現場において、

　営業している当人自身が
　（理想的には）自覚することすらなく、


　「どうすれば、お客さま自身が
　　気づかれていない真のニーズを特定し、
　　指摘して差し上げられるか」

　という問いを回し続け
　（＝その答えを探し続け）

　ている状態がキープされていてこそ、
　
　
　学びの真の目的である

　【質の高い意思決定基準を発動して
　　行動を変化させられる】

　ことが実現できている、

　といって良いでしょう。



■この状況がもたらされない学びは、
　冒頭に記したような

　「娯楽としての学び」
　「趣味としての学び」

　に過ぎません。


　なぜならば

　【（必要なときに）取り出せない
　　意思決定基準は意思決定基準ではない】

　からです。

　これでは学びに意味が
　生まれません。



■何度でも繰り返しますが、
　大事なのは

　【質の高い意思決定基準を手に入れ、
　　常時、そこから問いが生まれている状態】
　
　を作り出すことです。

　それが正しい学びです。



■ここまでお読み下さったあなたは
　こうした話すらも

　「一度聞いて、なるほど、と思った
　　だけでは真の学びに昇華されない」

　ということを
　お分かりのことでしょう。
