■昨日は、日本有数のマーケティングコンサルタントでいらっしゃる
　日本経営教育研究所の石原明先生と御茶ノ水でランチ。


■石原先生は日本トップレベルのコンサルティングフィーをチャージして
　いらっしゃる方で、その戦略発想など、前々からずっと直接教えを請いたい
　と思っていたのですが、なにせコンサルフィーも一回、数十万から百万の
　単位、なんて聞いていたこともあり、なかなかそういう機会も持てなかった
　のですが、、、


■昨年12月にご一緒に講演させていただく機会があっててから、お客様を
　ご紹介頂いたり、さまざまにお世話になっており、今回もランチのお誘いを
　頂いたので、勿論、速攻アポ入れいたしました。

　限られた時間でしたが「これ幸い」といろんなことをお聞きしてしまい
　ました。貧乏性なもので。（笑）


■著名な急成長有力ベンチャーなどをクライアントに持ち、いずれも
　常識を覆すほど大きな成果をあげていらっしゃる秘訣などをお聞き
　できて良かったです。

　石原先生ご自身のビジネスモデルや今後の展開などもお聞かせいただき、
　呆然。さすがです。（でも内容は内緒…すみません）


■まずは先日ご紹介くださったクライアント様への仕事は、そこそこに
　結果が出ていたということのご連絡を頂き、一安心。

　これをご縁にこれからもいろんなところでご一緒に仕事をさせて頂け
　そうな予感も！？そうなるよう、期待値以上の結果を常に出し続けて
　いきたいところです。

　石原先生、今後とも宜しくお願いいたします。

　「検索会議2005」に参加。

　素晴らしい会でした。（マウスやらバッグやら時計やら、大量のお土産も
　頂きました）


■たまたまですが隣の席には元リクルート、様々にご活躍されている

　いらっしゃったり、

　ブログのコンサルティングやコーチングの世界でご活躍されている松田充弘
　さん、などとも久しぶりの再会もできました。


■今回のイベントは、一週間ほど前、夜中の4時頃に弘中さんから電話が
　かかってきて紹介してもらったのですが、（既に残枠がゼロだったところを
　ご好意でご用意頂いたようで…）司会をされていた百式の田口さんを、弘中
　さんから紹介してもらったりで、新しい出会いも生まれました。

＜今日のピークパフォーマンス方程式。＞

　【　売ってから友達になるのではなく、友達が買って下される。　】

───────────────────────────────────


■最近考えていて、

 「たぶん、こういうことではなかろうか」、と思ったことがあります。
　
　
■どちらがよい、どちらが悪いということはありませんが、
　私の場合は、友達がモノを買ってくださるというケースが非常に多いです。
　
　非常に多いというよりも、それが圧倒的に大多数です。
　
　
■このメールマガジンも、実は、友達づくりのためのメールマガジンで
　あることに、あなたは気が付いておられたでしょうか。
　

■毎日せっせと書いているのは、お友達をつくるためなんです。

　別にそれがビジネスにつながらなくてもいいんです。そのために行っている
　わけでもないですし。

　ただ、それが結果としてビジネスにつながるケースは非常に多いです。

　
■皆さんも小学校、中学校、高校、大学でたくさんの友達をつくって
　こられたでしょう。
　
　そして社会人になったなら職場や地域のコミュニティーなど、
　いろんな形で友人関係を持っていらっしゃると思いますけれども、
　
　
■けれども友人関係というものは、たとえばクラスに数十人の人がいても、
　本当に仲良くなるのはせいぜい数人くらいのもの。
　
　全員と仲良くなるということはまずあり得ないことです。
　
　
■これは当然のことで、一人一人に個性や考え方があります。

　それは良い悪いではなく、それぞれが十人十色の価値観で生きている
　ということ。
　
　そんな中で、全員と、友達になるということはあり得ないわけですし、
　まして全員と親友になることも不可能です。
　
　
■いろんなコミュニティーとの接触を重ねる中で、

 「この人とは合いそうだ」とか、
 「この人とは難しそうだ」とか、

　そういうものは言葉や態度でだんだんとお互いに認識されていって、
　自然発生的に友人関係というものが生まれてきます。


■せっかくだったら、自分と趣味や価値観や嗜好が会う人と常に一緒に
　接触していきたいと思いますが、
　
　通常は、その速度は遅々としたもので、なおかつ広がりがなかったり
　するもの。
　

■私がこのメールマガジンを発行しているのは、実は、そのプロセスを
　人為的に加速させるつもりでやっているのです。
　
　ビジネスはあとから付いてくる副産物です。
　それが主ではありません。あくまで従。


■一体、それはどういうことなのか。
　明日詳述したいと思います。
