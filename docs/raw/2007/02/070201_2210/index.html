■恥をかきたくないから行動に移せない、と言う人がいます。

　しかしよくいわれることですが、恥をかかないと成長できないのです。

　なのに年を重ねるにつれ、だんだん恥をかきづらくなってきます。
　行動に移しづらくなってくるのです。



■今、大きな恥をかいても、それはやがて得がたい経験、学びとなり、
　また、なんとかしなければ、と向上の元にもなります。



■恥とは影と一緒ではないかと思います。

　影は太陽の方へ向かうとついてくるし、
　太陽とは逆の方向へ行くと、逃げていきます。

　恥から逃げようとすればするほど、恥まみれになってしまい、
　理想とはかけはなれた人生になってしまうかもしれません。

　どうせ恥をかくのであれば、
　こちらから積極果敢に恥をかきにいった方がいいのです。

　（わたしなど、毎日、恥まみれです。でも覚悟の上）
　


■どうしてこのようなことを思ったのかと言いますと、
　最近、現在活躍しているアーティストの昔のコンサートDVDを
　仕事の間、流していることが多いのですが、

　10年前、20年前のDVDを見ていると、
　登場するアーティストの現代とのギャップに、
　見ているこっちが恥ずかしくなってしまいます。
　


■じつは、最近見てほっとした（？）のは、TMネットワークの
　初期の頃のコンサートDVD。

　改めてみてみると、なんとも不思議な踊りをしているものです。


　【今日のお勧めDVD　DECADE(TM NETWORK 1984-1994)】
　
　 ⇒ http://amazon.co.jp/o/ASIN/B00005G3T2/2ndstagejp-22/ref=nosim



■実はこのDVD、ご本人たちが一番見るに耐えないのかもしれません！？

　その心境がなんとなく察せられます。

　なぜわかるのかと言いますと、わたしも３年前に
　配信したメルマガなんて、今振り返ってみるととても見るに
　耐えないものだから。

　だからあえてバックナンバーは見ないようにしています。
　　


■あの頃の小室哲哉さんがなければ、今の小室哲哉さんは
　いなかったかもしれません。

　そしてこれは　TMN　⇒　t.k.　だけではなく、全ての人において
　言えること。

　アーティストだけでなく、俳優、作家、スポーツ選手、
　政治家、教員、ビジネスパーソンなど。
　
　行動を起こすから恥をかき、そして認知をされるのです。

　逆に言うと恥をかかなければ、成長を遂げ、脱皮することもできない
　ということ。



■もし、TMNが、将来これが映像コンテンツとして残ったときに
　「とっても恥ずかしいことになるかもしれない」、
　
　との恐れや危惧をもって、やっぱりコンサートはやめようなんて思ったら、

　今のt.k.は存在していないだろうし、t.k.が存在しなければ、

　内田有紀、hitomi、dos、天方直実、TRF、安室奈美恵、
　華原朋美、globe、観月ありさ、篠原涼子、鈴木あみ、トーコ

　さらには、今のJ-POPも様変わりしていたかもしれません。
　そんなふうに聞くと、あなたも少しは安心しませんか？
