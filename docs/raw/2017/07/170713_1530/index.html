【鮒谷がギャンブルのためのカネを借りていた話】
短い時間軸で生きている人はギャンブル中毒者と一緒

───────────────────────────────────



■昨日は、

  成果を出すうえで
　必要不可欠の心構えであると思われる

　【継続の力の効能】

　について、

　「１万時間の法則」

　などの話を交えながら、
　お伝えいたしました。

　今日も続けます。



■こうした継続の持つ力を見誤り、

　横着して、一足飛びに
　ウルトラＣ的な成果を求めたい、

　という心的態度がひとたび、
　身についてしまうと


　【 毎日の取り組み → 時間経過による蓄積 → 複利効果】

　のコンボから生まれるパワーを
　享受できず、


　時間という何よりの希少資源を、

　成果なきまま、日一日と
　取り崩していくこととなるでしょう。



■不可逆的な、この流れは
　さらなる焦りを生じさせるので、

　より一層のミラクル、ウルトラＣを
　求めようとすることとなります。


　まるで、ギャンブルで負け続きの人が、
　負けが込むほどに


　▼いちかばちか
　▼一攫千金
　▼毒を食らわば皿まで
　▼捨て鉢になる
　▼自棄（やけ）のやんぱち
　▼自暴自棄


　という真理状態に陥るようなもの。



■こうしてますます

　【時間を味方につける、という
　　成果を求める者の王道から遠ざかっていく】

　こととなるのです。


　横着の精神は、こんな風にして、
　人生を破滅に向かわせかねないもの、

　なのです。



■一攫千金系の話（＝うまい話）、
　
　たとえば投機や、期待値の低い投資や、詐欺で
　やられてしまう人は、

　例外なく、

　【意識している時間軸が短い】

　ということが言えるでしょう。



■人生を短いスパンでしか考えられない人は、
　節操なく、

　「すぐに結果が出る」

　という甘言に釣られては、
　貴重な人生の残り時間を浪費し続け、

　焦りが募るから、ますます

　「すぐに結果が出る」

　という謳い文句に飛びつき、

　ジリ貧からドカ貧へと、
　真っ逆さまに転落していってしまうのです。
　


■あえて言ってしまえば、

　【ギャンブル中毒患者】

　と心理的には同じ状況にあるといって
　差し支えないでしょう。


　なぜ、そのようなことを断言できるかというと、
　私（鮒谷）自身、

　【高校→大学→社会人初期の、
　　およそ10年もの間、ギャンブル中毒であり】

　かつ

　【社会人スタート時につまづき、

　　性急に成果を求め、一攫千金的なあれこれに
　　惹かれては、時間とカネを浪費し続けた】

　という経験があるために、
　その類似性がよく分かるのです。



■以下、本邦初公開ですが、

　実は私（鮒谷）は、社会人になってからも、
　ギャンブルのためのカネ

　（＝一攫千金狙いのカネ）

　を友達から借りていたのです。
　　
　特にＡ君にはお世話になりました、
　というか、迷惑をかけました。

　言いたくなかった、書きたくなかった、
　本当の話です。



■これはちょっと極端な例かもしれませんが、

　【ジリ貧状態を抜け出すための
　　確度の低い、起死回生の奇策】

　をついつい求めたくなる心境が
　強ければ強い人ほど、

　
　【出来る限り、意識する時間軸を長く取り
　　（できるならば、老後から死ぬそのときまで）、

　　その上で人生設計を策定し直す】

　ことが重要であると、

　以上のような経験からも、確信を持って、
　お勧めしておきたいと思います。



■そんなこんなを経て、
　今は、上に記したような
　
　▼いちかばちか
　▼一攫千金
　▼毒を食らわば皿まで
　▼捨て鉢になる
　▼自棄（やけ）のやんぱち
　▼自暴自棄

　といった

　【ギャンブル依存的な、
　　思考習慣、行動習慣、生活習慣】

　から、

　【長期継続を主とする、
　　思考習慣、行動習慣、生活習慣】

　へと上書きインストールし、


　「３分間、待つのだぞ」

　ならぬ

　「３年間、待つのだぞ」

　（＆「じっと我慢の子であった」）

　的精神を自らに打ち立てられれば、


　自身の専門とする分野において一定の成果を残し、
　評価されることは約束されたも同然、

　といってもよいのではないかと
　考えているわけです。
