■今日は個別コンサル２件

───────────────────────────────────



■こんにちは、鮒谷です。

　
　今日は個別コンサルを２件。

　明日から大阪出張です。

　大阪の皆さま、よろしくお願いいたします！

　
　
■そういえば、先のセミナーに参加下さった方の
　ブログから、このサイト↓

　http://www.hal.ac.jp/

　
　を知りました。


　以前、こんなことを書いたことがありますが、

　＜3498号 量産型ザクでは勝負に勝てない＞
　
　http://www.2nd-stage.jp/backnumber/ppf_full/2013/04/3498.html


　量産機には、なりたくないものです。



■話は変わりまして、

　
　少し前になりますが、
　個別コンサルをさせて頂いた中村さんより、

　以下のようなご感想を頂戴いたしました。


　（ここから）
　--------------------------------------------------------------


　鮒谷さま、


　先日は貴重な時間を有難うございます。

　憧れ！？の、『たん熊』に行くことができ、
　未だにやや興奮気味です。


　「社長を目指しましょう！」

　1年ぶりくらいにお会いした鮒谷さんより、
　いきなりの先制パンチ…

　そして、久々に出会った眼のキラメキ…
　（本当にキラ～ンという言葉が聞こえてきそう笑）

　まさに子供が悪だくみを思いついた時のようなそのお顔に、
　あぁ、最近何か物足りなかったのはこれだっ、と、
　勝手に納得していました。

　（かなり失礼なこと言ってますね、すいません）


　今回のテーマは、

　1、社長
　2、いい人

　この2点に尽きると言っても過言ではありません。


　確かに字面だけ見ると、
　「え！？、それだけ！？？」となるかもしれません。

　でも、それだけで十分なのです。

　まず1に関しては、
　鮒谷さんのお力添えをお借りし、
　ある行動を起こすことを決めました。

　2に関しては、
　鮒谷さんのメルマガを読んでいる皆さまには、
　お馴染みの言葉ではないでしょうか。

　いい人の『いい』の類義語としては、

　・圧巻
　・圧倒
　・超越

　あたりでしょうか。

　とにかく他を圧倒する（寄せ付けない）分野を持ち、
　オンリーワン、ナンバーワンの存在になること。

　そうなることで他者から見たその人は、


　　　凄い人　≒　立派な人　≒　いい人


　となるのです。

　（会食中にはハッキリと言えませんでしたが）

　鮒谷さんは紛れもなく『いい人』であり、
　私もその位置をどん欲に目指します。


　1に関しての第1歩目を早速動かします。

　歴史的な？スタートを切るその場に、
　鮒谷さんに立ち会って頂けましたら、
　こんなに嬉しいことはありません。


　全ては『キラ～ン』から始まっております。

　是非皆さまにも『それ』を味わって頂きたいです。


　【百聞は一見にしかず】


　　　　　　　　　　　　　　　　　　　（中村 朋正 様）

　
　--------------------------------------------------------------
　（ここまで）


　中村さん、先日はありがとうございました！

　またご紹介させて頂くのが遅くなりましたが、
　長文のご感想もありがとうございます。


　高い目標を掲げた瞬間から、
　見え方や行動が変わってきたのではないでしょうか。

　例の決起大会（？）についても
　よろしくお願いいたします！




■それでは、

　「今日のピークパフォーマンス方程式」

　（ビジネスにおいて、
　　最高の結果を発揮するための方程式）

　をどうぞ。

＜今日のピークパフォーマンス方程式＞

【  「手ぶらセールス」」こそが最強のセールス  】

───────────────────────────────────



■個別コンサルにおいて、


　（多分に個人的な偏見も交えつつ）

　営業関連のお悩みを持つお客さまに
　よくお伝えしているのが、

　「プレゼン資料なし（もしくは必要最小限の資料）で
　　モノを売るのが最強の営業」

　ということ。



■私（鮒谷）も経営者という立場上、

　いろいろな商品、サービスを売り込もうと
　営業されるわけですが、

　「あ、これはダメな営業
　　（＝とっとと、お引き取りを願う必要のある営業）」

　というのは会って、ものの２分もすれば分かります。


　それは訪問したと思ったら世間話もそこそこに

　（場合によっては世間話ゼロで）

　おもむろにカラフルで分厚いプレゼン資料を取り出し、
　１ページ目から、得々と説明しようとする人。



■新入社員がやるのならまだ分からないでもありませんが、

　そろそろ中堅どころに差し掛かろうかという人でも
　こんな営業スタイルをとっている人があるのです。

　はっきり言って、これでは話になりません。


　一言でいえば、

　「（営業職に就くものにとって最も必要な）
　　想像力の著しい欠如」

　という致命的欠陥があるのです。


　そんな状態のまま、営業していて
　モノが売れるはずもありません。　



■そういう人は、とりあえず、

　元リクルートのトップセールス、
　現在、セレブレイン社社長の高城幸司さんの


　【今日のお勧め本　仕事の９割は世間話】

　 http://amazon.co.jp/o/ASIN/4532261597/2ndstagejp-22/ref=nosim


　この本などを読まれるとよいのではないでしょうか。

　レビューは厳し目ですが、

　（お世話になってきた人だから、
　　というのを抜きにして）

　個人的には凄くいい本だと思いましたけどね。　



■話はそれましたが、こちらは別に、

　先方の会社の所在地とか、資本金とか、従業員数とか、
　メインバングとか、設立以来の履歴とか、

　何の興味もありません。


　まず一番に聞きたいのは、

　その商品あるいはサービスを購入することが
　自社にとってどんなメリットや利益があるのか、

　その一点です。


　その疑問や不安がクリアになったとき、
　さらなる情報として上に記したような情報もチェックし、
　
　「じゃあ、まずは小さく取引を始めてみましょうか」

　ということはあるかもしれません。



■しかし、まだ見込み客の興味も関心も
　引き出せていない段階にもかかわらず、

　こうしたことを縷々（るる）述べられて

　「それではお願いします！」

　なんていう経営者があるはずもありません。

　
　ちょっと考えればすぐにわかることなのに、

　なぜ少なからぬ人が、

　わざわざ、誰も興味関心を示さない、
　つまらない、くだらない、
　
　マスターベーション的なプレゼンを行い、
　見込み客を退屈させるのでしょうか。


　高い時間給の意思決定権者が
　わざわざ時間を取ってくれているのだから、

　そのことを理解しようとしなければなりません。



■繰り返しますが、営業において最も大切なのは

　１、お客さまが求めているものを理解しようとする
　　　気持ち（＝押し売りは最悪）

　です。


　次に

　２、お客さまの欲求を実現させる商品、サービスを
　　　提供しているのであるということを、

　　　根拠や、事例とともにできるだけ早い段階で伝える


　ことに注力しなければなりません。



■この流れを忠実になぞるのであれば、
　
　カラフルで見栄えの良いプレゼン資料を
　１ページ目から順番に読み上げていくような営業は

　絶対に行わない（行えない）はずです。


　こんな営業スタイルを私は

　「紙芝居営業」

　と名付けているのですが、

　相手の目を見ることなく、ひたすらプレゼン資料を
　読み上げていく、

　そんな紙芝居営業は即刻やめる必要があるでしょう。


　
■紙芝居営業は、
　
　時間を大切にする顧客に対して
　絶対にやってはいけない迷惑行為、
　
　と肝に銘じるべきでありましょう。


　そもそもそんな営業なのであれば

　「（現段階での）ペッパー」

　君に任せればいい。


　＜Pepperがコーヒーマシン売り場で接客＞

　https://www.youtube.com/watch?v=xlKNPums81w


　そんなことを考えれば考えるほど、

　顧客の要求に合わせて融通無碍、臨機応変、当意即妙で
　伝える内容を変えていく、

　「手ぶらセールス」

　こそが最強のセールスだと思われるのです。



■念のため言っておきますが

　「手ぶらセールス」

　は決して、

　「手抜きセールス」

　ではありません（笑）


　意図と確信を持った上での

　「手ぶらセールス」

　なのです。



■少し偏った考え方、と思われるかもしれませんが、

　あながち間違ったことを言っていないと考えている
　理由は、


　私（鮒谷）自身、独立してから、

　ただの一度もプレゼン資料を作ったこともなければ、
　それを読み上げて営業したこともないから。


　それでも数十万から数百万程度のものまで、
　さまざまな商品やサービスを売りまくってきました。

　個人向けも法人向けもあんまり関係ありません。
　（ただし中小オーナー企業に限りますが）


　以上述べてきたことを参考にしていただくも、

　既読スルー（笑）にしていただくも、
　すべてあなたのご自由です。

　押し売りするつもりはありません。

　してお届けするのが、鮒谷の個別コンサルティング。


■経営戦略、ビジネスモデル設計、マーケティング
  情報発信、営業といったビジネス直結の話から、

  キャリア相談、独立起業に当たっての心構えや実践方法、
  何らかの決意の後押し、事業計画相談、

　などなど、経営、ビジネス、キャリアに関わるご相談、
　承っております。

　鮒谷の時間を買うことではありません。

　「より良き人生、より良き未来」を買うことです。
