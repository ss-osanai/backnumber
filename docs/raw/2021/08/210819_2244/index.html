　「深い学び」

　（ここから）
　------------------------


　いつもお世話になっております。　


　このたびは音源を提供いただき
　ありがとうございました。　

　今回の音源も、大いに学びになり、
　気持ちを新たにする機縁となりました。　


　量げいこということで、
　とにかく毎日行動し、

　改善を積み重ねていくことの
　重要性について話をされる音源と思って
　聞きはじめましたが、


　変わりたいと思って行動を起こしても、
　目標がしっかりしていないと、

　ただ体を動かすだけになってしまい、
　そこに意義を見出せずに
　現状に日和ってしまう。

　とはいえ、

　自分の心を燃やし続ける目標を
　決めるというのは簡単ではない。


　かといって、現状の自分から離れた

　「なりたい自分」

　を思い描いて踏み出そうとしていると、
　いつまでも踏み出せないまま
　現状に取り込まれてしまう。


　試行錯誤、煩悶、あきらめきれんと
　思いこませて、

　やはり行動から入るしかない、
　そうすることで、

　1ミリでも積み上げて、
　昨日の自分から伸びていく。


　何か、聞いているだけで、自分が
　コンサルを受けているような気持ちになる、

　疑似的に双方向のやりとりがなされている
　音源のように感じました。　


　振り返りますと、

　昨年の７月に、
　音源学習法の音源を購入して以来、

　いくつか音源を提供をいただきました。


　日々の通勤の行き帰り、
　休日に出かけるときなどは、
　頭がよほど疲弊していない限りは、

　それまで聞いていた音楽に替えて、
　鮒谷さんの音源を繰り返し聞くようになり、

　日々、気を入れていただくように
　なりました。　


　個人的には、30代後半ぐらいから、

　「自分のいる組織も含めて、
　　このまま安定的な世界が
　　長続きするはずがない」

　という意識を持ち、

　組織内の先達が歩んできた道や、
　見てきた景色とは違うロールモデルを、

　後進に示す存在になりたいと思いながらも
　組織内でそれなりに処遇され、

　自分の裁量で楽しい業務を
　切り回すことができたことで、
　現状の自分の立ち位置に日和ってしまい、

　「今のままでもなんとかなるかも」

　的な思いが支配し、


　自分の目指すところは
　ぼやっとしたままで、

　ただ、現状に満足していない自分に
　満足しているという状態に陥り、

　未来の実現に向けて十分な資源を
　配分できずにおりました。　


　昨年後半は、鮒谷さんの力のある言葉の
　後押しにより、

　日記をつけるようになり、
　今年に入ってからはブログを書くように
　なりました。

　
　（中略）


　自分としても

　「変わったな」

　と思うのは、

　先日、職場で待機がかかっていたときに、
　周囲の人がスマホいじって
　時間をつぶしていたのに対し、

　自分は未来に向けてどう踏み出すべきかを
　考え、言葉にしようとしていた。


　おそらく、以前の自分であれば、
　周囲の人と同じように、
　手持ち無沙汰にしていたことでしょう。


　そもそも、こうした待機の時間を

　「必要経費」

　と考えず、

　自分の貴重な時間を奪われることになると
　考えるようになった点でも、
　変わったとは思います。


　年齢的なことを考えると、
　試行錯誤に時間をかけている場合ではないと
　焦る気持ちもなくはないのですが、

　だからといってあきらめれば
　そこで終わりであり、

　いろいろな意味でこれまでにない
　ロールモデルを自分が作るつもりで、
　日々積み上げていきたいと思います。　


　一つヒントになったのが、
　鮒谷さんのおっしゃった

　「サイレント・マジョリティーの
　　代弁者になる」

　ということです。


　何か、ものを言えない人々の代弁者となり、

　静かな世論とのつなぎ役となることで
　自分の処を得る、

　そして後進にとってもロールモデルとなる、
　そこを目指しつつ、

　ただ組織の中でも機嫌よく生きられるように、
　うまくアピールしていく。


　完全に全振りすることは難しいですので、
　自分の人生を目標に向けて、
　少しずつ押していきます。　


　あと、組織にいると年度単位でものを
　考えてしまうクロック周波数が
　身についてしまい、

　それだと変化のスピードが
　ものすごく遅くなってしまいます。


　自分の中にあるクロック周波数を
　引き上げ、月単位、週単位、日単位で
　どんどん試行錯誤していく癖をつけていく、

　それにより、自分を未来に
　押し出していくことを加速させていく、


　ただやみくも動いても
　足場を積み上げることになりませんので、

　「河原にある石を一つ一つ拾って磨くように
　　（これは早稲田メンタルクリニックの
　　　益田院長がユーチューブの中で話していた
　　　ことの転用ですが）」

　自分の持っている石を一つ一つ磨く中で、

　強みを見出し、
　微差を積み上げていきたいと思います。　


　今回の音源、

　いろいろな角度から聞き直すことができる、
　ファセット面の多い宝石のような感じで、

　１感想がうまくまとまりませんでした。

　今後とも、どうぞよろしくお願いいたします。


　（Ｍ・Ａ さま）
