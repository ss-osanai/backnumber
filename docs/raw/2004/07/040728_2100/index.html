頭の回転が速くなる「速聴」が手軽に取り組めるようになりました。
 
すでに100万人に指示されている加速学習法「速聴」はご存知ですか？
今までは音質が劣悪なものだったり、高額な「速聴」しかありませんでした。
 
ところが、「速聴」システムに惚れ込んだ、東京の吉井さまが「速聴」の商標
を持つ会社と交渉して、テープタイプの速聴機の販売権を取得され、高音質の
速聴機を格安で提供できるようになりました。
吉井さまは、販売チャンネルを中間マージンの必要がない、インターネット
販売に絞るとのことです。今なら、サイトオープンに伴い、価格の割引を
おこなっています。
 
「注文が殺到すると生産が追いつかない可能性がある」ので,
今すぐコチラで確認してくださいネ。→　http://www.ja-control.com/

＜今日のピークパフォーマンス方程式。＞

　　【　　ブランドを作り出すことによって生じるメリット。　　】

───────────────────────────────────


■放談会での内容についてちょっとだけ、お伝えしたいと思います。
　→ご参加下さった方、怒らないでくださいね。。。
　　復習ということでお読み下さい。


■ブランドを作り出すということは、追いかける側になるか、追いかけられる側
　になるか、100％真逆の状況を生み出します。


■自分が商品を売っているとすると、


　▼ブランドのあるとき⇒お客様が押し寄せてきて、お断りするのが大変
　　　　　　　　　　　　営業コストはゼロ
　　　　　　　　　　　　それどころか口コミでお客様がさらに増える善循環

　▼ブランドのないとき⇒営業活動で疲弊する
　　　　　　　　　　　　莫大な営業コスト
　　　　　　　　　　　　待っていても口コミは永遠に起こらない

　

■あなたはどちらの商品を売りたいですか？

　ブランドがあると営業面で得をする。人材獲得面で得をする。提携
　パートナーを見つける上で得をする。そして税金のかからない無形資産
　ということで税金がかからないからお得（補足してくださった高松の若松親分、
　ありがとうございます！）。


■ブランドと商売は切っても切り離せない関係にあります。

　とくに今の時代は商品供給が過剰になってきていて、存在しない商品が
　ないくらいの過当競争の時代。

　昔は商品を作るだけで売れたそうなんですが、、、（古き良き時代。私は
　そんな恵まれた時代は未体験）いまはそんなわけにはいきません。


■過当競争の中で選んでもらえるかどうか、は一にブランドがあるか、ないか。

　ジャンル１位のブランドが総取りして、あとは残りの小さなパイをさらに
　小さく分け合うのみ。


■ブランドになったかどうか、は模倣される存在になったかどうか、で
　ある種の判断を図ることができます。

　偽ヴィトン、えせグッチ、ローロレックス（ロレックスではありません！）
　反対向きのワニ（ラコステのワニが逆を向いている！）、リーボッコ
　（リーボックではありません！）などなどなど、、、

　が生まれるのは、ブランドの信用力を借りようとしているわけです。


■なにをもってブランドといえるか、様々な定義があると思いますが、
　まずは一番になりたいと思う当該ジャンルにおいて、模倣される存在になる
　ことを目指してみるのはいかがでしょうか？
