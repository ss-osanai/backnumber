7395号 笑わない鮒谷も思わず吹き出した、AI生成の『弱者を導くハゲたおっさん』画像

■社会において、

　「生粋（きっすい）の強者」

　といえる人は、せいぜい、
　１～３％くらいしかいないのでは、

　というのが、
　私（鮒谷）の昔からの見立てです。



■強者は特段、

　生きていく上での戦略を考えなくても
　現代社会に適応し、

　勝手に成果を出しては、
　楽しく生きているようです。



■しかし、私を含めた、

　どうにも社会に適合させられない部分を
　持つ者（＝弱者）は、

　放っておいたら、脇へ、脇へと
　追いやられて苦しむことになるので、


　そうした人間が生きていき、
　さらには幸せになるためには

　【（弱者の）生存戦略】

　を立てて、実行し、
　生き延びなければなりません。



■自然界でいうならば、

　ライオンとかトラは「王者」ですから
　生きていくのに苦労はなさそうですが、

　小動物や小さな昆虫なんかは
　生き残るために必死で策を練っています。

　あれと同じです。



■私（鮒谷）も自分や自社を、

　つねに小動物や小さな虫に擬して
　生存戦略を考えているし、


　次の瞬間には、

　「環境変化でお陀仏となる」
　「食物連鎖の上位に食われる」

　そんな恐怖に突き動かされつつ、
　毎日を生きてきました。



■長い間、そんな日々を
　生き延びてきて分かったのは、

　自然界と同じで、とはいえ、

　「弱者は弱者なりの生きていく方策が
　　ないわけではない」

　ということ。



■あきらめたらお終いですが、

　あきらめない限り、
　試行錯誤を行う機会が得られるので、


　そこで、資源にならないような
　資源まで総動員しながら、それらを

　【新結合
　（異質のもの同士の組み合わせ）】

　させることで、

　オンリーワンや、
　ニッチな世界のナンバーワン、

　といった立場を目指せます。



■さらには、その戦略がハマれば、

　「弱者が強者を食う
　（ジャイアント・キリング）」

　といった逆転現象さえ
　起こせるのですから、

　社会不適合者にとって、
　なんとも夢のある話です。



■そんなことを考えながら、

　私は「同類」であるところの弱者に
　勇気と希望を提供する存在たらん、

　と思い立ち、

　そのイメージを、

　ドラクロワの
　『民衆を導く自由の女神』

　を参考にして生成するように、と
　AIに指示して出力させました。



■そうして出てきたのが以下の作品。

　なかなか良くできていると思うので
　ぜひご笑覧ください。


　出力された画像を見て、
　思わず吹き出してしまいました。

　【弱者を導くハゲたおっさん】
　https://www.pixiv.net/artworks/113371619



■今日も人生とビジネスを
　楽しみましょう！

　■世の中は圧倒的に社会不適合＝弱者、の
　　比率の方が高いが、それは自然界と同じ。

　■しかし弱者にも生き残るための術は
　　残されている。

　■手元にある、資源ならざる資源も含めて
　　棚卸ししては、組み合わせることで、

　　想像もつかぬような結合が、生存のための
　　イノベーションにつながることがある。
