■（あと２日）　
　「越えられない壁」の乗り越え方

───────────────────


■時間の経過とともに
　どんどん良くなっていく、

　そんな人生を送るための秘訣を
　本日はお伝えいたします。



■そのための取り組みは
　大きく３つのパートに分かれます。

　---------------

　１、破滅可能性をできる限り抑える
　　　（守りを固める）

　２、期待する成果を得るために挑戦を続ける
　　　（積極的に攻めていく）

　その上で

　３、上記を通して実現された、
　　　確実な成果が見込める「守り＆攻め」を
　　　生活や仕事にシステム組み込む

　---------------

　この３つ。



■これにより、

　人生における攻守がシステム化され、
　すなわち再現性が約束された形で、

　高い安全性と効率、ならびに効果が
　同時に実現することとなります。



■ところで、これらはすべて、
　アイデア（着想）から始まります。

　アイデアならびに前提となる仮説に
　基づき、具体的なプランを立案する。


　そのプランの実行と
　振り返りを重ねつつ、

　システムが構築、さらには強化され、
　やがて安定稼働が始まります。



■こうしてひとたび、システムが
　人生や生活、ビジネスに組み込まれれば、

　そこから先は
　日々のルーティンを繰り返すだけで、

　安全性が確保されつつ、高い成果が
　安定的にもたらされるようになる。



■ここにおいて、
　各種の「余剰」資源が生まれます。

　時間であったり、お金であったり、
　認知能力であったり。

　

■もし、あなたが望むならば、

　これらの余剰資源を、
　さらなる守りと攻めのシステム化に
　投資することが可能となりますし、

　逆に、リタイアを選ぶ道も
　開かれることでしょう。



■こうした複数の選択が可能になると、

　人生は一段と豊かで楽しく、
　自由になるに違いありません。


　しかし、多くの人にとって、
　その世界に到達するための
　
　「越えられない壁」

　として立ちはだかるのが、
　以下に記すポイントです。



■守りと攻めの計画は、
　それぞれ具体的なプランが必要です。


　ただ、プラン立案のヒントを
　得るためには、

　様々な場所に足を運び、多くの人と
　接触する必要がありますが、

　これが億劫になることが
　しばしばあるわけです。



■さらに、プランは立てられても、

　実現にはプラン立案の
　100倍、1000倍の時間や努力が
　要求されるわけですが、


　これまた実行が面倒になり、

　「企画はしたけれども
　　やりきれない」

　と挫折する人が後を絶ちません。



■このような「越えられない壁」が、
　未来へと進む者の前に立ちはだかる。

　これを越える唯一の方法は、

　【何があっても、
　　粛々と行動し続けられる力】

　を身につけること。



■そう聞くと、

　「それができないから
　　苦労しているんだよ」

　と反論したくなるかもしれません。


　しかし、もし、
　この巨大な壁を越え、

　アイデアを得て、プランを形にし、
　大量行動を容易にする方法があるならば、

　あなたはそれを知りたいとは
　思われないでしょうか。
