＜今日のピークパフォーマンス方程式。＞

　【　学びが必要な時代だから、「何を学ばないか」にこそ細心の注意を　】

───────────────────────────────────



■社内での立場が上がったとき。

　会社を移ったとき。ある業界を出たとき。

　独立起業したとき。

　あるいは海外でのビジネスを始めるとき。

　ステージが移るたび、そのステージごとに

　「ある特定の能力を磨くか、捨てるか」

　の選択が必要になります。



■前にも書いたかもしれませんが、たとえば、

　会社員時代には必要だったかもしれないワードやエクセルの能力、
　あるいはパワーポイントをキレイにつくるといった能力は、

　経営者になってからも、きっとあったほうがよいのでしょうが、
　わたしはばっさりと切り捨てました。

　それは、もっとうまくできる誰かに委任することによって
　手を煩わさずに、そのような資料を制作を依頼することが
　できるから。

　わたしは、もっと経営者として、あるいはそれ以外の立場として
　やらなければならないことがたくさんあるわけです。



■ステージが変わったとき、その立ち位置に応じて、

　今までの延長線上ではなく、いちどゼロリセットをして


　「次に何が必要なのか」

　「どんな能力が求められることになるのか」


　を見極めて、意識してその力を身につける必要があります。



■みなさんも、きっと今までそのようにされて自分自身を
　作りかえてこられたのかもしれませんし、
　
　わたし自身もそうして自己変革を起こし、またそれにある程度
　成功してきたように思います。



■これだけ世の中が激しく動いており、取り巻く環境が
　変わってきているのですから、

　今までのように１つの自分を作り上げ、それを一方向に対して
　磨きあげていればそれでよし、ということにはなりません。

　突然、それまでのキャリアの断絶、あるいは超越が生まれる
　ことがあるのです。

　それは自分が意図した場合もありましょうし、外部的要因によって
　ムリヤリそのキャリアが引き裂かれることもあります。



■そうなったときに、


　「今まで通りやっていてよいことは残し、
　　そうでないものは切り捨て、新たに必要になったものを身に付ける」


　という発想は非常に重要です。



■現在は、学校を卒業してからも学び続ける姿勢が大切と言われます。

　この「平成・進化論。」上でも、広告も含めていろいろな学びの
　情報を出させていただいておりますし、

 「平成・進化論。」に限らず、世間一般を見渡しても
 「能力を身に付けよ、学べよ」の大合唱です。



■しかしながら物理的に、すべてを学ぶ時間はありません。

　ならば何を学んで、何を学ばないか？

　これを明確にしておかなければ、結局「あれもやりました、
　これもやりました」とつまみ食いに終わり、しかしながら、
　なんらの結果にもつながらなかった、ということにも
　なりかねません。



■自分は、次のフェーズにおいてどういうポジションを取りに
　行きたいのか？

　この質問を自分に投げかけ、そのために有効なことはやる。
　有効でないと思える学びは思い切って切り捨てる。

　そういう決断力が重要です。



■わたしは社会人になってから、日系企業から外資系に、
　直販セールスからチャネルセールスに、

　会社づとめから起業へ、そこから複数会社の経営へと
　変遷（へんせん）してきました。

　そしていままた、新しいステージに進みたいと
　思って、そのための勉強を重ねているところです。



■人間はどんどんキャリアチェンジしていきます。
　そのたびに、その仕事において求められる能力が変わります。

　そのときの立ち位置、あるいはこれから移ろうとする
　立ち位置をよく見て、

　「そこではどんな能力を必要とされているのか」

　を吟味し、取捨選択していかなければならないのです。



■「学びが大切」と聞くと、ついついいろんな学びを
　したくなります。

　そこを焦らず、騒がず、腰を落ち着け、「次はここへ行きたい」と
　思ったならば、そのジャンルの学びに集中して自分を
　つくりかえる必要があるのです。

【編集後記】


■学びたいこと全てを学ぼうと思っても現実的に不可能です。
　
　ならば、次のステージで最も必要となる資質、あるいは知恵、知識
　はなにか、を常に自分に投げかけて、学ぶ対象を吟味する必要が
　あるわけです。


■たとえば、本もただ、たくさん読めばいいというものではありません。

　「読みたくても読まない本」を選択し、決断する必要もあるでしょう。

　あるいは「読み始めても、途中で合わないと思ったら、すぐに
　本を閉じる勇気」を如何にもたなければならないこともあるでしょう。

　時間は有限です。常にこのことを意識して、次の段階に行くための
　学びだけにフォーカスしていきたいものです。
