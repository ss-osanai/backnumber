■ちょっと面白い記事を見つけました。

　一ヶ月前くらい、「平成・進化論。」上に、

　【全てのものがコモディティ（日用品）化する現代】

　というテーマで数話にわたって執筆いたしました。


■毎回、多くのご感想をいただくのですが

　（同時にたくさんのご質問も頂きます。

　ただ、個別のご質問や返信を求められるメールにはお答えする時間が
　物理的にとれませんので、申し訳ございませんがご了承下さい）

　このテーマのときには、非常に大きな反響を頂きました。


■そのとき書かせていただいたことと近いテーマが、
　本日発売のハーバード・ビジネス・レビューに特集が組まれていたのです。

　タイトルもそのものずばり、

　【「コモディティ化」時代のマーケティング戦略】

　というもの。（こっちははるかに学術的ですが）


■いくつかの面白い記事があったのですが、

　【「脱」ライフサイクルの市場戦略】

　の中の、

　「リバース」
　「ブレイクアウェイ」
　「ステルス」

　という３つのポジショニング戦略、

　といった記事が興味深く読めました。

　S字カーブを人為的に前後に移動させる、といったらよいのでしょうか。


■さまざまな商品が市場に送り出されても、あっというまに陳腐化し、
　コモディティ化する速度がますます加速しているのを肌で感じている
　からこそ、

　【継続した脱コモディティ戦略】
　【旬を伸ばすためのマーケティング施策】

　のための考え方を常に捜し求めていたところです。

　そんな私のニーズにジャストミート。


■書店で手にとってみられて、興味があれば、ご一読されてみてはいかが
　でしょうか。


■来週のセミナーの準備と称して、空き時間は事務所に引きこもって
　本や雑誌ばかり読んでます。

　まずい、そろそろまとめにかからないと。。。

　
■そんなことを思いながらも、昨日は空き時間、終日事務所にこもって
　コーヒーに飲みながら本を読んでました。

　そこだけを切り出して見たら、とっても優雅な生活なんですが。。
　

■それでは今日の方程式をどうぞ。

＜今日のピークパフォーマンス方程式。＞

　【　常識発想を超えたところに大きな飛躍がある。 】

───────────────────────────────────


■一昨日から私のビジネスモデルを例にとってお話ししています。


■私のメールマガジン広告のビジネスの特徴を要約しますと、

　（繰り返しになり申し訳ありませんが、新しく読み始められる方も
　　いらっしゃいますので、書かせていただきます）


　▼無借金経営である。

　▼原則前金でのお取引き。（潤沢なキャッシュフロー）

　▼一切の値引きには応じない。値引き要請があってもお引取り願う。

　▼クライアントの分散。（ビジネスとしての安定性の確保）

　▼非常に高い利益率。

　▼利益率100％。

　▼リピート・紹介により営業をする必要がまったくない。

　▼効果を出すことによってお客さんに喜んでいただける。

　▼広告依頼とともに情報商材の寄贈、セミナーへのご招待をいただく。

　▼それらを生み出す時間（メルマガ執筆にかける時間）は
　　1日1時間から1時間半程度。

　▼在庫なし。

　▼時間・場所に一切、依存しないビジネス。

　▼時間が経過するごとに、よりビジネス基盤が厚くなるモデル
　　（狩猟型ではなく、農耕型）


■こういう形でビジネスをやっているケースは、世の中のビジネスの
　一般的な常識からはあり得ない、

　あるいは極めてまれなモデルではないかと思います。


■しかしそういうビジネスを組み立てたいと、常識の枠を飛び越え、

　その理想像の達成を熱望して、そこにアプローチをかけていたからこそ
　現実のかたちにすることができるわけです。


■もちろん私は、今のビジネスの形態がこれこそベストだとは思って
　おりません。また、ベターであるとも思っていません。

　それどころか至らないところばかりで、もっと美しく、もっと芸術的な
　ビジネスを組み立てたいと思い、今は青写真を描き、その準備をしている
　最中です。


■ビジネスの設計図の段階で、

　「こんなものを生み出せるはずはない」だとか、

　「できるはずがない」

　「こんな商売は不可能だ」

　と思えるような時でも、その制限事項をすべて撤廃してあげることが
　非常に重要なのではないでしょうか。


■常識発想を超えたところに大きな飛躍がある。

　いまの状況からの積み上げではなくして、「思い込み」を抜け出し、
　真に得たい結果から逆算して今、なにをなすべきかを考える。


■欲しい状況を想像し、それを得ようとして動いていけば、
　きっとあなたに途方もない結果をもたらしてくれるであろう、

　そんなことを考えたりもしているのです。


■現在から未来、への志向ではなくして、未来から現在へと回帰させる
　ような思考方法、

　が大きなパフォーマンスをあげるためには必要ではないかと
　最近は考えています。

　
　【制限を取り払って真に得たい結果から
　　　　　　現在にさかのぼる思考が偉大なブレークスルーを生み出す】


　ということ。
