■こんにちは、鮒谷です。

　今一度、改めてこの「平成・進化論。」のポジションについてお伝えを
　したいと思います。

　
■現在、このメールマガジンの購読対象者として想定しているのは、
　ビジネスパーソン（現在の所属されている業界、業種、職種などは問いません）
　の中でも意識が高く、向上心があり、勤勉なトップ10～20％、もしくはその
　仲間入りを目指している、

　【勉強熱心なあなた】

　です。

　それ以外の読者さまは、おそらくこのメールマガジンを読まれていないか、
　間違って購読されても、すぐに解除されていらっしゃることと思います。


■そんな勉強熱心で忙しいあなたに成り代わって、あらゆるセミナーや勉強会、
　交流会に足を運び、大量の本を読み、身体にかけて実践して生の情報を得て、
　それを咀嚼し、わかりやすく伝達する、というのが私の使命だと思っています。


■だからこそ、他のだれより勉強しなければならないと思っています。

　勉強した知識や学んだノウハウについても、うまく伝達することができる
　スキル（文章力、編集力、構成力、その他読者さまを惹きつける力、などなど）
　についても日々、研鑽をしていかなければならないと思っています。


■更にどこかで聞いてきたような、そんな情報を焼きなおしただけの机上の空論
　を読者の皆様にお伝えしていても仕方ありません。

　読者さまはみなさま賢明な方ばかりなので、そんなことをすればすぐに
　見透かされてしまいます。


■ということで、様々に勉強したことを自分でも年商数億円規模の事業を経営し、
　そこでしかるべき利益も上げ、その中で自分が学んできたことを生かし、
　皆様にフィードバックしております。

　そして今後もそのようにしていきたいとおもっております。


■その中で「平成・進化論。」に掲載されている「メールマガジン広告」に
　ついても改めて確認させていただきます。

　このメールマガジンでは、メールマガジンのヘッダー広告、フッター広告
　（上部の広告と下部の広告）ならびに号外広告、をクライアントさまから
　頂き、掲載させていただいております。


■わたしはこの「広告」はメールマガジンにおける一つのコンテンツだと
　思っております。

　冒頭でものべたように日常業務などで忙しいあなたに代わって、情報収集を
　して、それを出来る限り、読みやすく、吸収しやすいように編集して伝達
　するのが私の役目ですら、そのご提供情報の一環としての
　
　「メルマガ広告」

　だと思っております。


■メールマガジン広告により、今まで知らなかったような教材やセミナー、
　あるいは勉強会、サービスなどと出会い、そこから人生の転機が生まれた、
　新しいビジネスの機会が発生した、といった事例は枚挙に暇がありません。


■現に私も、そのような教材やセミナーに育てられたと本当に感謝しています。

　だからこそ、「平成・進化論。」の中でも「広告枠」は主要なコンテンツだと
　思っているのです。


■と同時に、その掲載内容のクオリティにも継続して注意を払っていき、
　今まで以上によりよい情報を読者の皆様にお届けさせていただかねば
　ならないと思っております。


■ここ数ヶ月、いろいろの出来事があり、「平成・進化論。」が単に個人の
　所有する一媒体、といったレベルを超え、大げさに言うと一つのメディアに
　成長した、という実感を持っております。

　一つの大きなメディアだからこそ、個人発行の一メルマガ、というレベルでは
　ない自覚を今、改めて要求されているのかもしれません。


■このメールマガジン広告について、今までも掲載に当たり簡単な審査等を
　行い、「平成・進化論。」読者にふさわしくない、と思った商品や
　サービスについてはあらかじめ、ご掲載をお断りさせていただくことも
　多々ありました。

　
■ただ、大手法人と比べて、一人でやっておりますもので、事前の審査、調査
　等に予算、時間を潤沢にかけられるものではございませんので、
　良質コンテンツの広告を、と思って配信させていただいておりますが、全てに
　おいて万全のチェックをさせて頂いているとは申し上げることができません。

　させていただければと思います。

　それは、

　【ご掲載商品を実際に購入したけれども、掲載情報と内容に食い違いがある】
　
　などクレームについての情報がもしあれば、お教えいただきたいということ
　です。

■頂いたメール全てについてお返事をする時間的な余裕は申し訳ありませんが、

　より品質の高い商品の広告が掲載されるように鋭意、継続的に努力させて
　頂きます。


■一歩づつの改善ではありますが、「平成・進化論。」はビジネス系メルマガ
　においてのリーディングマガジンの一誌との自負、自覚を持っておりますので、
　業界？全体として、良質のコンテンツのみを広告としてご提供できるような

■このような取り組みをしていくことで、「平成・進化論。」の媒体としての

＜今日のピークパフォーマンス方程式。＞

　【　大事を成すには、遠くを見て近くを見よ。　】

───────────────────────────────────


■今の自分にとっては夢物語といいましょうか、想像できないような結果を
　つかもうとするときには、注意しなければならないことがあります。

　遠くを見て、近くを見ることです。


■例えて言うなら車の運転と同じことです。

　車の運転は、ずっと先の方を見ていないといけません。信号は赤になって
　いないか、この先に渋滞は発生していないか。

　それと同時に、突然横から子供が飛び出してこないか、うしろから自転車が
　近づいて来ていないかといった、ごくごく近くの視野にも気を配らなければ
　なりません。


■ビジネスを展開して行くにあたって、10年先、20年先の目標はぜひとも必要
　です。

　しかし、「今日何をするか」ということも、おろそかにしてはなりません。

　政治家になる。ビルのオーナーになる。ポルシェに乗る。なんでもいいです
　が、そのような将来の目標だけを立てて、今何もしていないというのは、
　遠くばかり見て近くを見ていないのです。


■逆に、長期目標につながらないような目先のことばかりでもいけません。

　例えば人からの、相手にとっても自分にとってもメリットを生み出さない
　頼まれごとですとか、不急不要のこと、行っても行かなくてもいいような
　飲み会であるとか。

　そういうことばかりを、いくら一生懸命やっているつもりでも、長期ビジョ
　ンが明確でなければただ流されてしまっているだけで、気がついたら全然
　違う場所にたどり着いているということがあり得ます。

　ですから長期目標と短期目標を、50対50で見ていなければなりません。


■何かを成し遂げなければならない時も同じで、小さいことばかりにとらわれ
　ていては全体で何が起きているかが見えずに、見当はずれの方向にことが
　進んでしまいます。

　逆に、夢物語ばかり描いていて具体的な小さな目標に落とし込んで行かな
　ければいつまでたってもどこへも進めません。


■「こうしたい」という目標なり夢があり、それを着実に自分の方に引き出し
　てくるためには、自分の方向性やビジョンを確認しながら、目先のリスクを
　取り除き、毎時毎秒、歩みを進めていくという慎重かつ着実な動きが重要
　です。

　このときに、遠くだけを見ていても、近くだけを見ていても事故を起こしま
　す。


■長期目標と短期目標を繰り返し繰り返し確認することによって、長期の
　ビジョンがブレることなく、かつ、一歩一歩間違いなく歩みを進めることが
　できるようになるのです。
