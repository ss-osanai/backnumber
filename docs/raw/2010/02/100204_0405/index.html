＜今日のピークパフォーマンス方程式。＞

 【　好きなものしか深掘りしない　】

───────────────────────────────────



■特に20代くらいで、

　「自分の強みが何かわからない」

　と思われる方は、もしかしたら、

　それはまだ強みとして
　喧伝できるほどのものを持っていない、

　と思ったほうがいいのかもしれません。

　ならば、今は多くのことにチャレンジされてみては
　いかがでしょうか。



■そのとき大切なのは、
　自分の興味・関心のあるものにのみ、取り組むこと。


　よく私の親友が話してくれることですが、

　【 Must 】

　や

　【 Should 】

　で何かをやっていても意味がないよ、


　「～しなければならない」

　「～すべきだ」

　とおもって行動を起こしても、


　【 Want 】
　
　すなわち

　「～したい」

　で動いている人に勝つことはできない、

　と、その友人は言うのです。
　まったくその通りだとおもいます。



■いま興味のあるものが「Ａ」というものしか
　ないとしたならば、

　その「Ａ」を深掘りしていく。

　そうしたなら、その「Ａ」に関連した「Ｂ」や「Ｃ」と
　いった事柄に必ず行きつきます。

　「Ａ」が楽しいなら、
　関連する「Ｂ」「Ｃ」にもいずれきっと関心を覚えるはずで、

　そうしたら、その「Ｂ」「Ｃ」も掘り始める。



■すると今度は、

　「Ｂ」に関係した「Ｄ」「Ｅ」、

　「Ｃ」と隣接する「Ｆ」「Ｇ」という分野に
　行き当たるでしょう。



■だから、たったひとつ「Ａ」さえとことん深掘りして
　いけば、強みとなりうる分野がどんどん広がっていきます。

　そのためにはまず、最初に取りかかる「Ａ」を
　どこどこまでも好きであること。

　これが重要な条件となるのです。



■今の私であれば、この「Ａ」に該当することの一つが
　日経新聞に昭和３１年から連載されている

　「私の履歴書」

　をイチから順番に読むこと。



■実利とか関係なく、純粋に読むのが楽しくて仕方ないから
　読み進めているのです。

　それが結果として、新たな興味・関心分野をもたらして
　くれています。


　さらにいずれ、それらのものがどういう形でかは
　まったく分かりませんが、
　
　「私固有の強み」

　として発現してくるのを5年後、10年後、20年後を楽しみに
　しながら、地道に種まき（＝趣味道楽に没頭）していけば
　よいのです。



■冒頭の言葉でいうと、


　【 Must 】【 Should 】

　ではなく

　【 Want 】


　でやっていることだから、そこに揺らぎはありません。
