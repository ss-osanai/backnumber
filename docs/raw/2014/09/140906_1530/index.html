＜今日のピークパフォーマンス方程式＞

 【  お客さまから躊躇なくお金を頂けるようになるために  】

───────────────────────────────────



■コンサルその他を通して、

　起業を志望されている方から
　よく聞く「悩み」があります。

　それは、

　「お客さまから、
　　躊躇なくお金を頂くことができない」

　というもの。



■そういった方々に対し、お答えの一つとして、

　（特に）コンサルタントやコーチ、士業など、
　専門知識を生業とする人においては、


　「少なくとも、
　　過去に払ってきたお金までは請求できる」

　と考えてみてはどうか、
　
　
　とお話しすることがあります。



■更に、

　「金額」もさることながら、
　「時間」も投資してきているわけですから、

　過去に投資してきた

　「金額×時間」

　に見合う金額までは、
　ためらいなく頂戴してもよい、

　という許しを自らに与えることが大切だと
　お伝えしています。



■かくいう私も、

　過去、時間をかけて累計では、山手線内に、
　そこそこデカイ一戸建ての家を建てられる程の、

　セミナー、講演、海外視察、会食、書籍購入、
　その他経験、　

　といったものへの投資を行なってきました。


　（だから今は、
　　やっすい賃貸マンション暮らしです泣

　　でも、それでいいんです！）



■話を戻しますが、

　ただそれだけの金額を投資したということ
　のみならず、

　「膨大な時間をかけてきた」
　
　という自負があるからこそ、

　しかるべきお金を、正当な対価だと思って
　請求させて頂くことができるし、


　それが妥当であると思って下さる方が、
　お客さまになって下さっている、
　
　そんな理解をしています。



■裏を返せば、

　自身のサービスを購入するしないを判断するに
　あたっての判断材料は事前に、大量に提供しますが、

　その上で購入するに及ばず（その価値もない）、
　と思われる方は、

　そもそもお客さまになり得ないし、
　また、無理やりお客さまとするべきでもない、
　
　と思っている、ということでもあります。



■もちろん、

　学習や経験にお金をかければさえ良い、
　というものではありませんが、

　（あえて生々しい表現を取らせて頂くなら）

　「金銭解決」で済ませられるものがあれば、
　「金銭解決」で済ました方が手っ取り早く、

　「良質の学びと経験」

　を手に入れられる、
　その事実は否定すべきではない、
　
　と個人的には考えています。



■そんな風に考えてきたからこそ、私も、

　学びと経験において「金銭解決」が
　可能なものに対しては、

　躊躇なく、前のめりに、
　全力で投資をしてきたつもりです。



■そういった経緯から、


　最低でも、
　
　「過去にかけてきた『お金ならびに時間』を
　　回収するのに必要な金額」
　
　程度は、請求してもバチがあたらないだろう、
　そんな風に考えているのです。


　商売にあたっては、
　こういったメンタルの部分はあなどれません。



■冒頭、起業志望者の、

　「躊躇なくお金をお客さまに請求できない」

　という悩みをご紹介いたしましたが、
　
　
　その場合、少なくない方において、


　▼そもそも、提供しようとしている価値が
　　お金を頂くレベルには到達していない


　ということもありますが、
　その解決は究極的には、

　自分が投資してきた有形・無形のものの
　累積から生まれる自信、

　からしか生まれないのではないか、
　と思っているのです。　



■いずれにしても、

　「しかるべき時間とコストをかけて、
　　学びと経験を積んできた」

　という

　「自負・自覚」

　が、高額のチャージへの許しを
　自他に与えてくれることもある、


　そんなことを今日はお伝えして筆を置きたいと
　思います。
